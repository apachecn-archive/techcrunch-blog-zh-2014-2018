# Twitter 被指控再次回避英国退出欧盟僵尸网络问题 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/01/26/twitter-accused-of-dodging-brexit-botnet-questions-again/>

Twitter 再次被指控回避议会委员会的问题，该委员会正在调查英国 2016 年英国退出欧盟公投期间俄罗斯机器人的活动。

在昨天发给 Twitter 首席执行官杰克·多西的一封信中，DCMS 委员会主席达米安·科林斯写道:“恐怕有一些悬而未决的问题……Twitter 还没有回答，而一些进一步的问题来自你最近的一封信。”

在 Twitter 上周五发出的[信函](https://web.archive.org/web/20221025223057/https://www.parliament.uk/documents/commons-committees/culture-media-and-sport/180119%20Nick%20pickles%20Twitter%20to%20Chair.pdf)中，该公司表示，它现在已经对支持城市大学[去年 10 月份的一项研究](https://web.archive.org/web/20221025223057/http://journals.sagepub.com/doi/10.1177/0894439317734157)的数据集进行了分析(该研究发现了大约 13，500 个虚假 Twitter 账户的僵尸网络，这些账户广泛发布了关于英国退出欧盟公投的推文，并在投票后不久消失)。

据说这些账户中有 1%是“在俄罗斯注册的”。

但是 Twitter 的信没有说太多其他的。

“虽然城市大学发现的许多账户违反了 Twitter 关于垃圾邮件的规定，但目前我们没有足够有力的证据，使我们能够最终将它们与俄罗斯或互联网研究机构(之前发现的俄罗斯 trollfarm)联系起来，”它写道。

Twitter 继续表示，在该研究发表之前，总共有 6508 个账户已经被暂停(根据研究本身，我们已经知道这一点)，并表示超过 99%的暂停“具体涉及违反我们的垃圾邮件政策”。

所以这意味着这些账户中的大部分都在向其他 Twitter 用户发送垃圾邮件。因此，由于反垃圾邮件措施，来自这些账户的推文不会很明显。

“在剩余的账户中，大约 44.2%被永久停用，”它继续说道，但没有确切解释它们被关闭的原因。“其中，1093 个账户在被删除前被 Twitter 标记为垃圾邮件或低质量，这将导致他们的推文被隐藏在所有用户的搜索中，不会以任何方式对热门话题做出贡献。

“正如我们在上一封信中所说，这些防御行为对使用我们公共 API 的研究人员来说是不可见的；然而，它们是我们解决这些问题的主动技术方法的重要组成部分。”

Twitter 的信件作者、英国公共政策负责人尼克皮克尔斯(Nick Pickles)补充道，“城市大学发现的极少数账户仍在 Twitter 上活跃，目前没有违反我们的规定”。

他没有说有多小。

TL；dr 这个英国退出欧盟僵尸网络的一小部分实际上仍然存在于 Twitter.com。

虽然 Twitter 的信有两页，第二页指向牛津互联网研究所的研究人员于 2017 年 12 月进行的一项英国退出欧盟机器人研究，也依赖于 Twitter 的公共流媒体 API 的数据，Twitter 称“几乎没有发现与俄罗斯来源链接的证据”——字面意思是*就在*对“使用我们公共 API 的研究人员”进行的研究拉屎之后——柯林斯显然对如此缓慢提供的情报的数量和质量都不感兴趣。

切入正题，他要求 Twitter 具体说明有多少账户“是由俄罗斯的机构控制的，即使它们没有在那里注册”。

他还想知道:“有多少账户具有已被确认与俄罗斯有关联的账户的特征，即使你尚未最终确定这种关联存在。”

他指出，Twitter 仍然没有告诉委员会 13493 个可疑的机器人账户是“合法用户还是机器人；谁控制了这些账户，公投期间他们的活动有哪些受众，谁从这些账户中删除了推文”。

这么多问题，仍然缺乏有力的答案。

“我担心，无论这些问题是什么，无法获得直接的答案只会增加人们对这些问题的担忧，而不是让人们放心，”柯林斯补充道。

我们联系 Twitter 寻求对他的信的回应，但该公司拒绝提供公开声明。

[上周](https://web.archive.org/web/20221025223057/https://beta.techcrunch.com/2018/01/18/facebook-agrees-to-take-a-deeper-look-into-russian-brexit-meddling/)，在科林斯指责推特和脸书基本上忽视了他的信息请求后，脸书写信给委员会，表示将更彻底地调查该事件的历史数据——尽管后续行动会有多全面仍有待观察。(脸书也表示，这一过程将需要“几周时间”，但没有给出明确的最后期限)。

这两家公司上个月也披露了一些信息，以回应选举委员会的一项平行调查，该调查正在调查英国退出欧盟投票前后的数字支出——但随后他们只是披露了俄罗斯实体针对英国退出欧盟的付费广告的细节(分别表示为: [~$1k](https://web.archive.org/web/20221025223057/https://beta.techcrunch.com/2017/12/13/twitter-says-russians-spent-1k-on-six-brexit-related-ads/) 和 [~$1](https://web.archive.org/web/20221025223057/https://beta.techcrunch.com/2017/12/13/facebooks-brexit-probe-unearths-three-russian-bought-immigration-ads/) )。

因此，他们没有尝试扩大搜索范围，寻找俄罗斯支持的非付费内容，这些内容可以在他们的平台上自由创作和传播。

到目前为止，柯林斯一直在推特上对这个问题进行最严厉的批评，但他警告说，如果他们继续阻挠他的调查，他们可能会面临制裁。

DCMS 委员会将于下个月前往华盛顿参加一个公开的证据会议，脸书和推特代表已被邀请参加。

它显然希望，与华盛顿的接近——以及最近这些公司因美国选举相关的虚假信息而受到美国议员质询的记忆——可能会让它们感到羞愧，从而展开更令人愉快的合作。

与此同时，英国情报和安全委员会能够从国内间谍机构获取闭门证据，在去年的[年度报告](https://web.archive.org/web/20221025223057/https://b1cba9b3-a-5e6631fd-s-sites.googlegroups.com/a/independent.gov.uk/isc/files/2016-2017_ISC_AR.pdf)中讨论了来自国家行为者的安全威胁。

尽管该报告没有明确指出英国退出欧盟是俄罗斯干预的明确目标，但它确实引起了人们对俄罗斯活跃的网络活动的担忧，并警告说，选举和公投可能是虚假信息攻击的目标。

“国家行为者非常有能力实施先进的网络攻击；然而，他们对这些方法的使用在历史上一直受到活动被揭露后会产生的外交和地缘政治后果的限制。最近俄罗斯的网络活动似乎表明，情况可能不再如此，”该委员会写道，并指出，对民主党全国委员会和约翰·波德斯塔(John Podesta)电子邮件的黑客攻击表明，俄罗斯正在采取“更厚颜无耻的方法来开展网络活动”。

它从英国 GCHQ 和 MI5 间谍机构获得的证据在报告中进行了编辑，包括在讨论英国政治制度安全的一节中。

该委员会在这里写道，敌对外国和恐怖组织的网络攻击可能“可能包括在合法的政治和时事网站上植入虚假信息，或以其他方式干扰政党和机构的在线存在”。

另一份来自 GCHQ 的经过编辑的证据详述了该机构是如何“已经对围绕数据完整性的风险保持警惕”的。

ISC 继续推测，这种国家攻击可能有各种各样的动机，包括:

*   总体上破坏联合王国政治进程的完整性，以期在英国人民和更广泛的世界的眼中削弱联合王国政府；
*   通过破坏或支持特定的竞选活动来破坏特定的选举或公民投票，使敌对行为者偏向的一方获得补偿利益；
*   以符合敌对国家外交政策目标的方式毒害关于敏感政治问题的公开言论；或者
*   在政党关于选民的敏感数据的情况下，获取大部分英国人口的政治倾向和其他特征，从而确定可能会因敌对行动者的利益而遭受颠覆或政治极端主义的人

“国家行为者的高能力与日益厚颜无耻的方法相结合，使得确保英国控制关键国家基础设施的系统的安全变得越来越重要。检测和应对高端网络活动必须仍然是政府的首要任务，”它补充道。

在相关新闻中，本周英国政府宣布计划建立一个专门的国家安全单位来打击国家领导的虚假信息运动。
