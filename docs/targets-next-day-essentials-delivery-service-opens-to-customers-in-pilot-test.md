# Target 的次日必需品交付服务在试点测试中向客户开放 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/06/27/targets-next-day-essentials-delivery-service-opens-to-customers-in-pilot-test/>

Target 今天早上宣布向消费者开放一项名为 Target Restock 的新的次日送货上门服务。该公司在 5 月份表示，这项服务已经上线，并在面向消费者的试点计划于今年夏天到来之前，与公司员工进行了测试。这项服务最初面向明尼阿波利斯地区的客户，该地区靠近塔吉特百货公司总部。

Target Restock 旨在与[亚马逊的 Prime Pantry、](https://web.archive.org/web/20221210070037/https://www.amazon.com/gp/pantry/info)竞争，因为它还为在线购物者提供了在网上装满一箱日常必需品的能力——其中许多物品原本是禁止单独运输的，如洗涤剂、苏打水、狗粮等。

有了 Prime Pantry，顾客可以以 5.99 美元的统一价格装满一个高达 45 磅的箱子，否则 Prime 会员可以免费运输，只要它至少有五个合格的项目。然而，这些箱子不能用于快速运输，由于它们的尺寸和重量，它们使用地面运输。

与此同时，Target Restock 将允许在线购物者以每箱 4.99 美元的价格在一个箱子里装满最多 45 磅的类似商品。然而，它目前只提供给 REDcard 持有者，尽管他们可以在结账时选择任何支付方式。随着服务扩展到测试之外，非持卡人可能会在未来某个时候被邀请进来。

为了进行网上购物，顾客还必须拥有 Target.com 的账户。

在员工试用期间，Target Restock 提供了大约 8，000 多个 SKU 用于发货。随着零售商向客户开放该服务，现在可用的选择超过 10，000 个 SKU。

要使用这项服务，消费者可以访问新网站[target.com/restock](https://web.archive.org/web/20221210070037/http://target.com/restock)，在那里他们可以按类别、品牌浏览产品，或者在验证他们的邮政编码后进行搜索。当他们填满他们的进货箱时，屏幕顶部会出现一个灰色条，显示箱子被使用的百分比，以及还剩多少空间。

与亚马逊的 Prime Pantry 不同，Target Restock 从他们的仓库履行订单(至少现在……随着[以 137 亿美元收购全食超市](https://web.archive.org/web/20221210070037/https://beta.techcrunch.com/2017/06/16/report-amazon-is-gobbling-whole-foods-for-a-reported-13-7-billion/)，这种情况很容易改变), Target Restock 从附近的商店履行订单。这也是它实现次日送达承诺的方式。

Target 并不是唯一一家希望利用其实体业务帮助自己在电子商务中更好地竞争的零售商。沃尔玛也在本月宣布[开始测试使用店员进行最后一英里的配送。](https://web.archive.org/web/20221210070037/https://beta.techcrunch.com/2017/06/01/walmart-begins-testing-using-store-staff-for-last-mile-deliveries/)

在沃尔玛的例子中，运送到商店的订单将由员工在下班回家的路上用自己的车交付给顾客。沃尔玛正在测试自己的应用程序，以高效地发送这些快递，允许员工选择通过处理一些离他们日常通勤路程不远的快递来赚取额外的钱。

然而，塔吉特表示，它将从 UPS 开始，与各种快递供应商一起测试重新进货。

Target 首席信息和数字官迈克·麦克纳马拉(Mike McNamara)在一份声明[中说:“Target Restock 的目的是让 Target 的运营更容易，并帮助我们的客人在繁忙的生活中节省时间。“我们期待看到我们家乡市场的客人对这一新产品的反应。”](https://web.archive.org/web/20221210070037/https://corporate.target.com//article/2017/06/twin-cities-pilot-restock)

Target Restock 目前在双子城(明尼阿波利斯-圣保罗地区)有售。该公司拒绝透露预计测试该服务的时间，但表示试点的目标是微调整体体验，并测试未来将出现的其他增强功能。
