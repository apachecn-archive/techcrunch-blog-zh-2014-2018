YC 支持的 Livecoding.tv 是 Twitch.tv For Coding | TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/06/29/yc-backed-livecoding-tv-is-the-twitch-tv-for-coding/

Y Combinator 当前班级中的初创公司 Livecoding.tv 今天推出，旨在帮助程序员实时向同行学习。

现场编码的工作方式非常简单。开发人员将他们自己编码的实时视频流式传输，观看的用户可以提问或给出反馈。

自 2 月份推出测试版以来，Livecoding 已经有来自 162 个国家的 40，000 人注册。用户使用多种语言进行交流，包括葡萄牙语、俄语和德语，以及编码语言，包括 C#、Python 和 PHP。

从专业程序员到构建自己第一个 Java 游戏的 11 岁孩子，任何想对编码感兴趣的人都可以这样做。

与 Periscope 或 Meerkat 类似，您可以关注个人用户，并在他们开始新的直播流时收到通知。

对于专业程序员来说，Livecoding 是一个寻找测试用户或获得新应用反馈的好地方。例如，著名的程序员和物理学家史蒂夫·沃尔夫勒姆上周在 Livecoding 上通过一个吸引了 4000 多名观众的演示来推广他的新编程语言。

对于经验不足的程序员来说，网上有大量的编码教程，但与专业的程序员交流并实时观察他们的进度可能更有教育意义。

Livecoding 联合创始人杰米·格林(Jamie Green)表示:“这种互动是我们区别于你可以在 YouTube 上观看的常规在线教程的一个关键因素。

“那些教程往往是经过精心编辑的，它们采取最直接的方式来解决它们试图解决的问题，但在现实生活中，这不是工程师解决问题的方式，”他说。

初学者也可以发布他们自己的流，以寻求关于他们正在构建的内容的提示和评论。

“有些人说，这有助于他们集中注意力，更好地思考，如果他们知道他们有一个观众，他们就不会每五分钟就切换到脸书，而分心，”格林说。

Livecoding 并不局限于构建一个应用或游戏。格林告诉我，一名程序员用这个平台安装了一台 3D 打印机，另一名程序员则编写了播放现场音乐的代码。格林说，Livecoding 最终可能会扩展到编码以外的新领域。

“现场教育是未来的关键，”格林说。“像设计这样的垂直行业参与这种模式可能会非常有趣。”

目前，Livecoding.tv 对用户完全免费。格林表示，该网站永远不会有广告，但未来可能会对更多结构化内容收取订阅费。
