# SanDisk 的 1TB SD 卡比我的电脑还大

更高的分辨率意味着更大的视频和图像，但幸运的是，闪存容量似乎与此保持同步。SanDisk 今天发布了一款新的 SDXC 卡，它有 1TB 的可用空间，这基本上意味着你可以把它放在相机里，并且基本上忘记它甚至有存储容量。

新的 1TB 容量目前只是一个原型，这意味着还没有定价或上市日期，但该公司已经推出了 800 美元的 512GB 版本，因此无论何时登陆都可能很贵。但是，与大多数家用电脑的基本配置相比，你还可以获得更大的存储空间，因此溢价是意料之中的。

我自己的 5K iMac 只有 256GB 的存储，真的不知道对此作何感想。我确实知道，我花了大约三年时间用我的佳能 6D 拍摄我们的 Disrupt 事件、产品评论等来填充 256GB 的 SD 卡，所以以此为基准，即使有糟糕的数据管理实践，这也应该基本上能让你坚持到生命的尽头。
