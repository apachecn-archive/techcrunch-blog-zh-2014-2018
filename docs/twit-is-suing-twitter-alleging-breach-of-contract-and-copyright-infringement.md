# TWiT 起诉 Twitter，指控其违反合同和商标侵权 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/01/16/twit-is-suing-twitter-alleging-breach-of-contract-and-copyright-infringement/>

TWiT，在技术上的官方名称是 This Week，正在起诉 Twitter。音视频媒体平台指控违反书面合同、违反口头协议、故意干扰预期经济优势、商标侵权。

诉讼称，据说 Twitter 的联合创始人伊万·威廉姆斯曾告诉 Leo Laporte Twitter 只是一个基于文本的微博客服务。

诉讼称，威廉姆斯还承认，TWITTER 意识到了 Twitter 和 TWiT mark 之间的冲突。“当时，Williams 代表 TWITTER 承认，在市场上使用 Twitter 可能会引起混淆，并且已经出现了实际混淆的情况。”

因为 Twitter 和 TWiT 相对不同，Laporte 和 Williams 同意允许商标共存，“条件是每家公司继续其独特的分销平台，”TWiT 在诉状中写道。

正如诉讼所称，Twitter 上发生的事情——140 个字符的短消息——与其平台上产生的音频和视频非常不同。然而，诉讼称，2009 年，拉波特担心 Twitter 会入侵 Twitter 的音频和视频。诉讼称，当时拉波特联系了威廉姆斯，威廉姆斯告诉拉波特“我们不会在 Twitter 品牌下扩展到音频或视频领域”。

然而，2017 年 5 月， [Twitter 宣布计划将原创优质视频内容引入其平台](https://web.archive.org/web/20221207101050/https://blog.twitter.com/marketing/en_us/topics/product-news/2017/new-premium-video-content-coming-to-twitter.html)。7 月，Twitter 的律师要求 Twitter 停止扩大 Twitter 商标的使用。

“从那时起，双方一直在进行沟通，目的是非正式地解决这一争端，”诉讼称。“这些努力并没有解决争端，Twitter 违反与原告的协议，继续向 Twitter 的业务扩张，驳斥其陈述和承诺，侵犯原告的知识产权，所有这些都伤害了原告。”

Twitter 正在寻求救济，包括一项初步和永久的禁令，命令 Twitter 停止在音频和视频内容的分发中使用该标志，以及“从非法行为中获得的任何和所有利润。”

TWiT 和 Twitter 都没有立即发表评论。
