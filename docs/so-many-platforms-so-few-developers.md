# 如此多的平台，如此少的开发者 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/09/06/so-many-platforms-so-few-developers/>

今天，当我在 TechCrunch Disrupt Hackathon 上走动时，我与许多决定在活动上设立桌子的供应商进行了交谈，原因只有一个:吸引开发人员使用他们的 API 并在他们的平台上构建应用程序。

事实上，我和十几个供应商谈过，包括 Evernote、Twitter、Microsoft Azure、Weather Underground、Pearson(图书出版商)、GoGo InFlight 和许多其他供应商。我甚至与一位医院代表交谈过，他在那里是因为他们已经建立了一个开源医院管理平台，你猜对了，他们希望吸引开发人员加入他们，并在此基础上构建应用程序。

尽管有如此多的平台，但要让它们都达到必要的临界质量以获得它们渴望的开发者的关注是极其困难的。老掉牙的说法当然适用。你建了不代表他们会来。事实上，在本周早些时候的 [BoxWorks](https://web.archive.org/web/20230129100013/http://www.boxworks2014.com/) 客户会议上，来自 Scale Venture Partners 的 [Rory O'Driscoll 说，当然每个人都想成为一个平台，但这并不意味着它总是一个现实的目标。](https://web.archive.org/web/20230129100013/http://www.scalevp.com/biography/rory-odriscoll)

“只有当你达到临界质量时，第三方才会介入。你得做点别的，成为一个平台。作为一个平台，你还没有完全成型。”以 Box 为例，O'Driscoll 指出，企业文件同步和共享在今天可能是一种商品，但它在 2009 年被认为是非常创新的，Box 设法让很多人使用该产品。反过来，这引起了开发者的注意，正是因为他们相信他们可以通过在 Box 上构建来赚钱。

今天，Box 声称有 47，000 名开发人员在其平台上开发应用程序。

如果没有足够的数量，不管这个数字是多少，都只有少数人会来，因此公司会通过出现在 TechCrunch Disrupt Hackathon 这样的活动中来主动吸引开发人员，在这些活动中，他们拥有大量受其支配的开发人员受众。他们正在尝试一些策略，包括为在他们的平台上编写最有创意的程序的开发者提供现金和商品。许多供应商也在举办免费的 API 研讨会，帮助对它们感兴趣的开发人员入门或学习技巧和诀窍，使构建应用程序变得更容易。

一些公司，如 Twitter、Yammer 和 Evernote，当然已经达到了这个规模，并且公司很高兴地在这个平台上进行建设，原因正是 O'Driscoll 概述的。他们知道，如果他们在那里建设，他们有巨大的潜在受众，所以时间和资源的投资是值得的。

Yammer 更进一步，与一家名为[炼金术士加速器](https://web.archive.org/web/20230129100013/http://www.alchemistaccelerator.com/)的创业孵化器合作。参与者可以获得 28K 美元，从 Yammer 和其他地方获得额外的资助者、客户和导师，以帮助他们开发产品。至少其中一些项目最终会构建在 Yammer 之上(尽管这不是必需的)。

正如来自 Jafco Ventures 的[Dana Evans 在 Box VC 小组会议上所说，如今每个技术公司都面临的问题之一是市场上缺乏编程人才，这导致了围绕招聘的激烈竞争，进而吸引他们到每个供应商都在努力推动的平台。](https://web.archive.org/web/20230129100013/http://www.jafco.com/team-dana.php)

“我从未见过对开发人才的竞争像今天这样激烈，”她说。这意味着你有一个有限的供应和巨大的需求。对于平台玩家来说，这意味着他们必须比以往更加努力地吸引开发者在他们的产品上构建工具。

套用一句古老的[威利·尼尔森的歌曲](https://web.archive.org/web/20230129100013/http://www.metrolyrics.com/mama-dont-let-your-babies-grow-up-to-be-cowboys-lyrics-willie-nelson.html)，“妈妈们，你可能想让你的宝宝长大成为开发者，”因为对他们的注意力有疯狂的竞争，而这是一个很好的地方。在这种背景下，今天的黑客马拉松参与者都是赢家。

*注意:这篇文章最初说炼金术士加速器是 Yammer 的一部分。它实际上是一家独立的公司，在 Yammer 的办公室之外。*
