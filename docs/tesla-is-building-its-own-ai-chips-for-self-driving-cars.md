# 特斯拉正在为自动驾驶汽车打造自己的人工智能芯片

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/08/01/tesla-is-building-its-own-ai-chips-for-self-driving-cars/>

“在过去的 2-3 年里，我们基本上一直处于半隐形模式，”埃隆·马斯克在今天的财报电话会议上说。“我想大概是时候泄露秘密了…”

问题中的猫:特斯拉电脑。也被称为“硬件 3”，这是一款特斯拉制造的硬件，旨在被替换到 Model S、X 和 3 中，以进行提高这些汽车自动驾驶能力所需的所有数字处理。

到目前为止，特斯拉一直依赖于 Nvidia 的驱动平台。那么为什么现在要换呢？

特斯拉表示，通过内部制造，它能够为了效率而专注于自己的需求。

Hardware 3 项目负责人皮特·班农(Pete Bannon)表示:“我们受益于[……]了解我们的神经网络是什么样子，以及它们在未来会是什么样子。班农还指出，硬件升级应该在明年开始推出。

“关键是，”埃隆补充道，“能够在基本的裸机水平上运行神经网络。你必须在电路本身进行这些计算，而不是在某种模拟模式下，这是 GPU 或 CPU 的工作方式。你想用那里的内存做大量的(计算)。”

根据埃隆的说法，最终的结果相当引人注目:他说，尽管特斯拉在英伟达硬件上运行的计算机视觉软件每秒处理约 200 帧，但其专用芯片每秒能够处理 2000 帧，“具有完全的冗余和故障转移”

此外，正如人工智能分析师王济民指出的那样，这让特斯拉对自己的未来有了更多的控制权:

通过拥有自己的芯片，特斯拉可以按照自己的速度满足自己的需求。如果他们突然意识到硬件缺少什么，他们就不会等待别人来制造它。这绝不是一项微不足道的任务——但如果他们能够在不倾家荡产的情况下完成这项任务(埃隆表示，这与他们目前的硬件成本相同)，它可能会成为一项重要的优势。

至于他们将如何把芯片装入现有的特斯拉，埃隆说:“我们使关掉计算机变得容易，这就是所有需要做的。你拿出一台电脑，插上另一台。所有连接器都兼容。”
