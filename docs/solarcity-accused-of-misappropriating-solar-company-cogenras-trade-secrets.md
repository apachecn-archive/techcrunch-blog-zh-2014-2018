# SolarCity 被指控盗用太阳能公司 Cogenra 的商业机密

太阳能公司 cogena 正在起诉埃隆·马斯克的 [SolarCity](https://web.archive.org/web/20230304004642/http://www.solarcity.com/) 使用 shingling 技术，该公司声称 SolarCity 从 cogena 那里获取了技术，并用于创造一个打破世界纪录的太阳能电池板。

Cogenra 表示，它在 2010 年至 2014 年期间与 Silevo 和 SolarCity 分享了“最珍贵和机密的商业秘密、制造工艺和其他知识产权”。

Silevo 是 SolarCity 的子公司，诉状称这些商业秘密帮助 SolarCity 制造了自己的太阳能电池。

“只是通过盗用 co gena 的专有技术，包括其商业秘密和其他知识产权，SolarCity 和 Silevo 后来才能够宣布他们创造了太阳能电池板能效的新世界纪录，”co gena 在诉讼中说。

诉状还称，Cogenra 在 2014 年开始收购更大的太阳能公司，据[采访彭博](https://web.archive.org/web/20230304004642/http://www.bloomberg.com/news/articles/2016-09-26/musk-controlled-solarcity-sued-by-cogenra-and-khosla-ventures-itkp0ypx)的消息人士称，SolarCity 是其中一家被视为潜在买家的公司。据称，可能的收购让 SolarCity 获得了机密信息。

然而，SolarCity 称这起诉讼“毫无价值”，并表示整件事始于它提醒 SunPower 注意一名前 SolarCity 员工，该员工非法下载了该公司的机密信息，最近加入了 Cogenra 的母公司 SunPower，担任高级销售经理。

SolarCity 告诉 TechCrunch 说:“SunPower 子公司 Cogenra 没有承担责任并确保归还我们被盗用的商业机密，而是通过提起毫无根据的诉讼，跑到法庭上转移人们对其行为的注意力。”“co gena 的诉状未能确定 co gena 拥有的任何实际商业秘密，更不用说 SolarCity 据称盗用的商业秘密了。我们相信法院最终会驳回 Cogenra 的指控，这些指控没有事实和法律依据。”

SolarCity 还坚称，其技术与 Cogenra 无关，许多像他们一样的公司已经开发了自己的太阳能技术，并提出向 Cogenra 展示其开发技术的证据，而不是从 Cogenra 获得的任何信息。

SolarCity 的发言人说:“这些事实对 SunPower 没有帮助，所以 Cogenra 起诉时没有费心去了解它们。”

从 2009 年到 2015 年拥有 co genra 80%股份的 Khosla Ventures 也在这起案件中起诉 SolarCity。科斯拉和 SunPower 都称 SolarCity 的声明是虚假的。
