# 智能手表上个月大幅降低了传统手表的销量

NPD 的一项调查显示，苹果手表已经将手表销量推至 2008 年以来的最低点。NPD 发现，手表制造商 7 月份卖出了价值 3.75 亿美元的手表，比去年下降了 11%。

这是自 2008 年金融危机导致类似下降以来的最大降幅。

在这一点上，苹果与此有关的说法是似是而非的。然而，我认为，从 Pebble、摩托罗拉、Fitbit 到苹果，所有智能手表都在严重冲击时尚手表市场。

对于制表商来说，夏季传统上是一段难熬的日子。孩子们已经毕业了，所以没有角色可以买。在这休闲的几个月里，假期很少，除了一些免税的旅游贸易，人们很少有机会学到新东西。

此外，在过去一年里，面对令人困惑的消费者需求，该行业一直相当低调。斯沃琪的 [Sistem 51 已经在低端市场获得了一定的恶名，但其他制造商已经将自己定价过高，脱离了时尚界，迫使他们在周期性的假日市场工作。](https://web.archive.org/web/20230207060808/http://www.wristwatchreview.com/2015/04/16/swatch-sistem51-update/)

最终，手表行业面临绝境。当面临化石和 Fitbit 之间的选择时，我怀疑一般懂技术的买家会购买健身带。虽然这应该会让手表爱好者感到震惊，但这就是现状，很快就会成为新的常态。
