# 优步计划 2020 年在达拉斯和迪拜进行按需垂直起降演示飞行

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/04/25/uber-targets-2020-for-on-demand-vtol-demo-flights-in-dallas-and-dubai/>

今天，优步在垂直起降产业大会上宣布了新的合作伙伴关系，从而使其垂直起降(VTOL)计划更加具体化。合作伙伴是老牌航空公司和新兴垂直起降公司，包括 Aurora Flight Sciences、 Embraer 、Bell Helicopter、 Pistrel Aircraft、Mooney 和 ChargePoint、，最初将专注于两个市场，包括美国的达拉斯和国外的迪拜。

合作伙伴将为优步运营其所谓的 Elevate Network 提供各种技术要求，Elevate Network 是一种计划中的按需空中叫车服务，旨在将人口稠密的城市地区的行程从通常需要 30 分钟缩短到大约 5 分钟。这个想法听起来像科幻小说，因为它确实是；这基本上是一个让飞行出租车成为现实的计划。

优步之前已经在一份白皮书中详细讨论了 Elevate 的提议，但该公司现在正在概述一些将采取的步骤，以使其远大的愿景成为更切实的东西——并且它也在为结果设定时间表，目标是将 2020 年迪拜世博会作为首次优步 Elevate 网络演示的发射台。根据今天透露的计划，达拉斯也将有一个演示网络准备在同一时间推出。

[![](img/1d446d8f83ed18ad91ae7eaaa0e1fc26.png)](https://web.archive.org/web/20230324160732/https://techcrunch.com/wp-content/uploads/2017/04/dubai.png)

迪拜已经对一些先进的交通项目表达了浓厚的兴趣，包括其他飞行汽车和 Hyperloop，并有能力提供财政资源来支持其雄心。达拉斯和沃斯堡的市长分别支持这个项目，并分享了使达拉斯-沃斯堡地区成为美国第一个按需飞行汽车服务的目标。

达拉斯也是优步正在进行的 Elevate 会议的主办城市，在该会议上，来自今天在该倡议中列出的所有合作伙伴的发言者将介绍他们关于电动垂直起降飞行器的想法，以及如何在城市中安全、有效和经济地使用它们。

优步表示，它已经与迪拜道路和运输管理局合作，该机构为旨在找出城市中路线走向的研究提供资金，以及如何优化交通流量，以及向乘客收取多少费用。ChargePoint 将通过专为优步垂直起降网络设计的垂直起降充电器为整个企业供电。优步将与达拉斯地区的土地开发商 Hillwood 合作，在达拉斯创建垂直起降机场。

[![](img/04035a5fdfb8caa3c85017f0157359b4.png)](https://web.archive.org/web/20230324160732/https://techcrunch.com/wp-content/uploads/2017/04/dubai-flight-arc-001.jpeg)

Embraer、Bell Helicopter 和 Aurora Flight Sciences 已经签署了一项大规模制造垂直起降飞机的协议，这种飞机的短程飞行范围是为城市内的旅行设计的。优步将与这些飞机制造商合作，设计用于管理和部署垂直起降飞机的软件和硬件，以及当地交通部门和空域监管机构接入该系统的方法。这将是关键，因为当涉及管制空域时，优步规避监管的典型策略可能不会奏效。
