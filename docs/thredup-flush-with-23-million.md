# 拥有 2300 万美元

紧随在线寄售竞争对手 [Threadflip 的 B 轮](https://web.archive.org/web/20221001121719/https://beta.techcrunch.com/2014/07/20/threadflip-raises-13-million-in-series-b-plans-to-grow-workforce-by-40/) , [thredUP](https://web.archive.org/web/20221001121719/http://www.thredup.com/) 刚刚向[美国证券交易委员会](https://web.archive.org/web/20221001121719/http://www.sec.gov/Archives/edgar/data/1484778/000148477814000004/xslFormDX01/primary_doc.xml)提交了一份 2300 万美元的 D 轮融资申请。

根据文件，ThredUP 正在寻求总计 2500 万美元的股权融资。我们已经联系过威胁要申请，但是还没有回音。

高地资本合伙人早在 2012 年就领导了该公司的 C 轮融资。其他投资者包括[三一投资](https://web.archive.org/web/20221001121719/http://www.crunchbase.com/organization/trinity-ventures)和[红点投资](https://web.archive.org/web/20221001121719/http://www.crunchbase.com/organization/redpoint-ventures)。这份文件中提到了之前的三位投资者，以及 Clinkle 和网飞的运营总监安迪·伦迪奇、ThredUP 的联合创始人兼首席执行官詹姆斯·莱因哈特和来自 advanced Ventures 的格雷格·贝蒂内利。

这将使公司的资金总额超过 4600 万美元。这一数字是竞争对手 2300 万美元、posh mark 1600 万美元和 thread flip 2100 万美元的两倍多。
