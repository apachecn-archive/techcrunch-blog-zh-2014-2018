# “不性感的管道系统”，集成数据和医疗保健系统的未来

> 原文：<https://web.archive.org/web/http://techcrunch.com/2015/12/06/unsexy-plumbing-integrated-data-and-the-future-of-the-healthcare-system/>

杰森·t·安德鲁撰稿人

像许多其他经济部门一样，医疗保健正在被技术所改变。这些变化在医疗保健的前端最为明显，医疗支付解决方案、数据分析工具、远程医疗、可穿戴设备和其他产品和服务正在满足企业和消费者的需求。

但同样重要的是在幕后，在医疗系统的后端基础设施中正在发生的事情。这种被称为“不性感的管道”使得前端进步成为可能。对于技术创新者来说，这种管道也带来了无数的机会:降低成本，简化流程，塑造一个触及所有美国人的行业的未来。

尽管它很重要，但管道工作不是特别好。医疗保健信息在很大程度上仍然是孤立的、未集成的，导致效率降低、成本增加和效果变差。根据 MeriTalk 对 IT 领导者的[调查](https://web.archive.org/web/20230319231927/https://fcw.com/articles/2015/08/17/leaky-data-integration.aspx)，由于政府卫生和公共服务机构难以管理不同的数据集，糟糕的数据集成每年造成 3420 亿美元的利益损失。

咨询公司埃森哲的另一项调查显示，只有 46%的医生认为他们的电子健康记录(EHR)系统改善了患者的治疗效果，低于 2012 年的 58%。此外，76%的医生认为他们的电子病历互操作性差对医疗产生了负面影响。在另一个例子中，由电子健康倡议组织(eHealth Initiative)开展的一项针对责任医疗组织(ACOs)领导人的调查[发现，访问外部数据，然后整合这些数据，是他们的第一(78%)和第二(62%)最紧迫的挑战。](https://web.archive.org/web/20230319231927/http://www.ihealthbeat.org/articles/2015/9/9/acos-still-struggle-to-access-outofnetwork-data-survey-finds)

为什么数据集成没有发生得更快或更具变革性？遗留技术是罪魁祸首之一。现有法规也导致了数据互操作性的滞后。例如，根据州和地区卫生官员协会最近的一项研究，“与隐私相关的障碍可能会阻碍[健康信息交流]，一个州的政策和做法可能会与另一个州或联邦法律相冲突。”然而，数据集成背后的动力正在增长。

## 政府在数据整合方面正取得(一些)进展

在公共层面，健康信息交易所，如加州综合数据交易所(Cal INDEX)和纽约州健康信息网络(SHIN-NY)，正在努力收集和共享健康数据，目标是降低成本、提高效率并最终提高医疗质量。

医疗保险和医疗补助服务中心(CMS)建立的有意义使用标准是政府在促进数据整合中的作用的另一个例子。通过向医疗保健提供商和组织提供支付激励，CMS 鼓励医疗保健信息交换和合格 EHR 的实施和“有意义的使用”,以及其他目标。虽然现在评估该计划的全面影响还为时过早，但激励措施最终应该会扩大互操作数据的使用，这反过来将有助于改善护理和结果。

> 数据集成背后的动力正在增长。

患者保护与平价医疗法案法案下的新法规也推动了集成数据的增长。比如，ACA 的“卡迪拉克税”(Cadillac tax)将于 2018 年开始实施，将对超过特定门槛的工人健康福利征税。这将鼓励一些雇主将员工转移到公共交易所，在那里他们可以购买自己的保险。

然而，其他雇主更愿意继续使用他们的传统供应商——如果他们能够找到符合成本效益的计划和福利来满足员工的需求。有了整合的数据，运营商和经纪人将能够向雇主展示计划之间的比较，计划是否避税，以及覆盖范围的差异可能如何影响工人。

此外，ACA 规定保险公司的支出中只有 15-20%可以用于管理。由于数据集成减轻了管理负担，保险公司有了另一个推动集成的理由。对电子记录的更多依赖也应该有助于减少每年因使用纸质记录而导致的浪费和低效率而造成的高达 4000 亿美元的损失。

## 私营部门介入，数据整合开始起步

然而，数据整合越来越不是由政府机构或法规驱动，而是由私营部门驱动。将数据集成与创新的基于云的解决方案和先进的 API 相结合，该领域的公司正在创造产品和服务，以提高护理质量，为消费者提供更多选择，并加强企业的品牌和底线。

例如，IBM 的沃森健康云正在一个安全的云中心收集超过 5000 万人的健康记录，从而展示了“大数据”的潜力。通过获取这些信息，医生将能够识别疾病发展的模式，了解哪些治疗方法在人群中有效，并为类似情况的患者提供更好的结果。

Fitbit、Jawbone 和 Apple Watch 等可穿戴设备，以及苹果 ResearchKit 等基于智能手机的应用程序，作为大数据的来源有着巨大的潜力。根据 2014 年普华永道[的调查](https://web.archive.org/web/20230319231927/http://www.pwc.com/us/en/press-releases/2014/wearable-technology-future.html)，超过 20%的美国人拥有可穿戴设备，每天产生大量的健康数据。越来越多的技术人员开始关注获取这些数据并将其集成到电子健康记录中。

一旦数据存储在 EHR 中，医生就可以为设备佩戴者提供更好的护理。上传到沃森健康云等平台后，患者的信息与数千或数百万其他人的信息相结合，可以帮助研究人员更清楚地了解人口健康。

> 医疗系统后端的不性感管道从未如此重要。

很少有人会认为电子健康记录的实施是顺利的。然而，电子记录相对于纸质记录的明显优势，以及有意义的使用激励计划，导致了电子病历使用的爆炸性增长:根据[美国医院协会](https://web.archive.org/web/20230319231927/http://dashboard.healthit.gov/quickstats/pages/FIG-Hospital-EHR-Adoption.php)的数据，从 2008 年到 2014 年，采用电子病历的医院比例从 9%上升到 76%。

未来的 EHR 将为医疗机构内部以及外部系统提供更加无缝的连接。数据将变得更加容易获得和有用，输入它将花费医生更少的时间，它将不仅支持病人的护理，而且支持更大的公共卫生倡议。

在保险行业，软件工具现在将以前分散在不同运营商的大型健康计划信息数据集整合在一起。因此，代理商和经纪人可以更轻松地创建更多定制报价，并提供更好的方案选择。由于数据的即时可用性，过去需要几周时间的报价流程现在只需几秒钟。

与此同时，Castlight、ClearCost Health 和 HealthSparq 等公司正在使用集成数据来帮助企业客户的员工比较不同提供商的程序成本，就何时何地寻求医疗保健做出更明智的选择，并了解从第一次看医生到最终出院的费用。

随着这些和其他创新依赖于集成数据，医疗系统后端的不性感管道从未如此重要。能够正确整合数据，并创造出满足医生、患者、消费者、企业和其他关键受众需求的产品和服务的技术公司，将是未来几年医疗保健行业真正的变革者。
