# Stripe 与 Intuit 合作，帮助按需员工跟踪他们的财务状况

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/01/26/stripe-intuit-on-demand-workers/>

初创公司 Stripe 有助于推动大量新服务的支付，为日益增长的“按需经济”提供了赚钱的途径。与此同时，为小企业和个体户制作会计和税务软件的 Intuit 希望帮助这些员工更好地跟踪他们的财务状况。

因此，Stripe 和 Intuit 合作开发一种产品是有意义的，这种产品可以立即识别按需员工收到的收入，并帮助他们优化税收，注销与工作相关的费用。

通过这种合作关系，在通过 Stripe 进行支付的点播平台工作的人将能够轻松地与 Intuit 的新 [QuickBooks 在线自营软件](https://web.archive.org/web/20230328203551/http://selfemployed.intuit.com/)连接。一旦完成，QuickBooks 产品将能够立即将付款确认为收入，从而能够帮助员工跟踪他们的财务状况，尤其是他们的纳税义务。

据 Intuit 自营解决方案副总裁兼总经理亚历克斯·克里斯称，大多数 1099 名员工通常不知道他们的月收入、季收入或年收入。这部分是因为他们没有获得与受薪员工相同的收入报表——相反，他们从他们工作的按需平台获得不定期的支付。

他们也没有很好地跟踪他们可以从税收中扣除的费用，但这是另一个问题。

为了迎合不断增长的 1099 名员工的需求，Intuit 免费提供了自己的 QuickBooks 软件。每月 7.99 美元，它还提供一些付费功能，如允许用户将他们的银行账户和信用卡连接到该软件，以便跟踪和分类收入和支出。

希望通过这样做，QuickBooks 可以提供更多的员工财务信息。例如，能够估计他们需要支付多少季度税和年终税。这可以消除收到巨额意外税单或因季度税贡献不足而被罚款的可能性。

为了简化跟踪收入的过程，Stripe integration 将免费提供给那些从公司获得收入并将其用于支付的员工。据 Stripe 战略合作伙伴负责人克里斯蒂娜·科尔多瓦(Cristina Cordova)说，这包括 Lyft、Sidecar、Summon、Flywheel、Handy、Homejoy 和 Washio 等公司。

因为这些公司雇佣工人作为承包商，他们不能提供税务建议或福利。但是，通过在 QuickBooks 这样的产品中直接跟踪收入，这些员工通常可以更好地管理他们的财务。

无论如何，值得注意的是，Stripe 合作伙伴关系将帮助一些按需工人，但不是其他人。例如，优步司机不会受益，因为优步不使用条纹。但是这些员工仍然可以支付 QuickBooks 来连接他们的银行账户。

不管怎么说，这不就是免费增值商业模式的美妙之处吗？
