# WTF 是众筹？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/09/04/wtf-is-crowdfunding/>

如今，你不可能抓住一只喝醉的美洲虎的尾巴不去参加六个众筹活动。看看一些活动，它似乎可能是最终的快速致富方案:你想出一个轻率的想法，制作一个漂亮的视频，然后繁荣。让它下雨。除了运行一个成功的 Kickstarter 或 Indiegogo 活动要比这复杂得多。在这个视频中，我们仔细看看一些常见的陷阱。

创建活动只是战斗的一部分——按下红色的“上线”按钮是大量工作的高潮，但很多人没有意识到的是，创建活动只是一个漫长过程的开始。在某种程度上，众筹运动本身就是迷你创业公司。

对于创业公司来说，众筹可能会很可怕。你没有在相对舒适的“秘密”模式下工作，而是向全世界展示了你的实力。您的客户知道您计划做什么，这限制了您在这一点上的敏捷程度。你的竞争对手肯定会密切关注这场运动。见鬼，他们通常会用一美元来支持你的竞选活动，以获得支持者的最新消息。这绝对是一门艺术。

你必须想出一个商业模式，包括谁是你的客户，为什么他们可能想要支持你的活动，以及你将如何接触他们。你必须弄清楚你将如何设计、迭代并真正销售你的项目。

众筹页面本身是一个相当大的项目，尤其是如果你想创作一个真正唤醒人们想象力的视频。

最后，一旦竞选本身的激情消退，是时候兑现你的承诺了。在那一点上，你*真的*希望你之前做的所有计算都是正确的。有很多众筹活动最终会提供 50 美元的支持者奖励，而公司只需花费 60 美元就能完成。

我们查看了数百个众筹活动，在上面的视频中，我们强调了许多初创公司都会犯的一些错误。但是不用担心；我们不会为你破坏这一切。你自己还会犯很多错误。希望它能让你朝着正确的方向思考，让你的众筹获得成功。
