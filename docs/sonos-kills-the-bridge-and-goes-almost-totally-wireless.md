# Sonos 摧毁了桥梁，几乎完全无线化

无线扬声器公司 Sonos 已经更新了它的整个流媒体系统，允许几乎无线的扬声器连接。直到今天，将整个家庭连接到同一个扬声器网络的唯一方法是使用一种叫做网桥的东西，一个 50 美元的盒子，通过以太网直接连接到你的路由器。然而，现在系统可以直接连接到您的无线路由器，完全绕过网桥。

为什么这是一件大事？ [Sonos](https://web.archive.org/web/20221006015706/http://sonos.com/) 长期以来一直是各地书呆子的首选家庭音响系统。能够在不同的房间播放不同的曲目(或为整个家庭音乐打开派对模式)已经从一个需要昂贵的控制器和桥接器的过程发展到一个使用商用硬件和您自己的无线网络的系统。由于专有系统和架构，大多数硬件公司变得更加孤立和难以使用，而 Sonos 却使其扬声器变得简单得多。

通过降低成本和硬件要求，Sonos 展示了硬件行业中一些最智能的行为。面对过多的平板电脑和手机，他们停止了原来的(优秀的)遥控器，转而支持 iOS 和 Android。现在，他们通过在某些情况下取消桥梁，进一步降低了成本和准入门槛。

虽然不是每种情况都允许完全无线流媒体，但 Sonos 让无线连接所有扬声器变得异常简单。
