# 算法问责的需要

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/09/08/the-need-for-algorithmic-accountability/>

赫曼特·塔内贾撰稿人

More posts by this contributor

新一代的公司对社会产生了如此大的影响，以至于它们基本上变成了公用事业。谷歌、亚马逊、脸书和优步近乎垄断，为我们的现代生活提供不可或缺的服务，就像一个世纪前的电力、电话和交通系统一样。

但上个世纪的公用事业和今天的公用事业有一个关键区别:今天的公用事业不是简单地提供服务，而是使用算法来塑造我们的整个生活体验。从我们看到的新闻和我们行驶的路线，到我们为商品和服务支付的价格，算法影响着它们。

有了算法，公司可以针对每个变量进行优化。优步在高峰时间高效地派遣司机，脸书向用户展示最相关的故事，亚马逊可以在最佳时间推荐合适的产品以推动购买。尽管有其积极的一面，但算法没有优化的地方是做正确的*事情或者显示任何数量的透明度。*

从外部来看，人们无法理解这些软件驱动的黑盒内部是如何做出决策的，也无法知道这些公司是公开还是无意中表现不佳。管理内部运作的算法强烈倾向于做最有利、最高效和最有效的事情，即使这意味着将一些人排除在信息或服务之外，或者为放大社会偏见敞开大门。

这必须改变，这些公司自己必须带头在其服务中建立算法问责制。这不仅仅是像 Airbnb 今年早些时候所做的那样承诺做更多的事情来促进访问的公平性，还延伸到创造一定程度的自动化透明度，让消费者、媒体、技术社区或公共官员看到正在发生的事情。

这个行业不应该依赖政府来扮演这个角色。由立法者和官僚撰写的老派法规过于繁琐，无法跟上技术的发展，也过于“愚蠢”，无法及时监控算法。政府主导的监管会加重科技公司的负担，减缓创新。主要例子:20 世纪的电力公司以及监管如何扼杀了几十年的投资和创新。

我们最不需要的就是公司变得像高度监管的电力公司一样缺乏创新。在上世纪初，监管给相互竞争的标准和不可靠的装备带来了秩序。但在很短的时间内，监管机构为该行业规定了一种“成本加成”的商业模式。几十年来，这意味着电力公司没有动力去追求更清洁、更分散的能源，或者投资任何创新。监管助长了加剧气候变化问题的心态，而我们现在才刚刚开始在这方面取得进展。

> 软件定义的监管比监管团队实施的监管更能监控软件驱动的行业。

为了避免类似的命运，这些新的公用事业公司(世界上的谷歌、亚马逊和优步)必须主动将算法问责纳入其系统，忠实而透明地充当自己的监督者，否则最终将面临繁重的监管。

对支配我们生活的“黑盒”算法的担忧一直在蔓延。纽约大学信息法律研究所[主办了一场关于算法问责制的会议，指出:“学者、利益相关者和政策制定者质疑管理算法决策的现有机制的充分性，并努力应对算法力量的崛起在透明度、公平性和平等待遇方面带来的新挑战。”耶鲁法学院的信息社会项目也在研究这个问题。“算法建模可能有偏见或局限性，算法的](https://web.archive.org/web/20230404032432/http://www.law.nyu.edu/centers/ili/algorithmsconference)[用途在许多关键领域](https://web.archive.org/web/20230404032432/http://isp.yale.edu/event/unlocking-black-box-promise-and-limits-algorithmic-accountability-professions)仍然不透明，”该小组总结道。

软件定义的监管比监管团队实施的监管更能监控软件驱动的行业。算法可以全天候监视脸书和优步等新兴公用事业公司，看到人类可能永远抓不住的细节和模式。如果监管出租车的交通官员希望确保优步不会避开某些社区或偏向其服务，他们应该编写一个算法来寻找这种行为。

让算法时刻关注人类在这些公司中的角色也同样重要。我们想要能够辨别 Airbnb 主机是否非法拒绝少数族裔或者脸书的人工编辑是否压制保守派新闻头条的技术。

看门狗算法可以像开源软件一样——对任何人开放检查。通过这种方式，编码人员可以看到看门狗算法是否在监控正确的事情，同时公司保留私有的专有算法和数据。

软件定义的监管还有另一个好处，或许是一个巨大的好处。它还将向我们展示一条通往更高效政府的道路。世界上的法律逻辑和法规可以编码到软件中，智能传感器可以提供从空气和水质量、交通流量和车管所排队等一切的实时监控。监管者定义规则，技术专家创建软件来实施规则，然后人工智能和人工智能帮助完善政策的迭代。这将导致地方、国家和全球各级政府更加高效和有效。

尽管如此，我们现在需要一个关于如何使这一工作的对话。科技公司和政策制定者需要走到一起，分享想法——这在这个软件无所不包的时代太难得了。很明显，沿着这条路走下去，我们将会产生监管社交网络、搜索和其他关键服务的监管机制，让它们变得像电力公司一样乏味和官僚。这对科技行业、经济和社会来说都是糟糕的。
