# 电子商务搜索开发商 Twiggle 将阿里巴巴列为投资者 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/05/31/twiggle-alibaba/>

总部位于特拉维夫的初创公司 Twiggle 正在开发一款新的电子商务搜索引擎，该公司今天宣布，已将[阿里巴巴集团](https://web.archive.org/web/20221225215946/http://www.alibabagroup.com/en/global/home)作为新的投资者。阿里巴巴的融资是 Twiggle A 轮融资的第二部分。

阿里巴巴的出资额没有披露，但 TechCrunch 获悉，这使得 Twiggle 自成立以来筹集的资金总额不到 2，000 万美元。上个月宣布了 Twiggle 首轮融资的第一笔资金[，金额为 1250 万美元，之前已经筹集了 220 万美元的种子资金。这意味着阿里巴巴投资了大约 530 万美元。阿里巴巴没有置评。](https://web.archive.org/web/20221225215946/https://techcrunch.com/2016/04/07/twiggle/)

Twiggle 的其他支持者包括互联网巨头 Naspers 和雅虎日本，它将利用这笔资金来壮大其研发团队。该公司的目标是开发一个电子商务搜索引擎，提供比当前选项更准确的结果，同时允许购物者用会话语言编写查询(例如，“我想要一台不会产生太多噪音的双门冰箱，它也是白色的，是新的，有很好的评论，物有所值”，而不是“双门冰箱”)。它正在开发桌面、移动和基于语音的搜索产品。

Twiggle 由首席执行官阿米尔·柯尼斯伯格(Amir Konigsberg)和首席技术官阿迪·阿维多(Adi Avidor)于 2013 年创立，前者是谷歌在以色列的首批员工之一，后者曾是谷歌以色列的首席软件工程师。它已经在今年夏天达成了整合到几个主要电子商务网站的交易，但尚未透露其客户的确切身份。然而，有理由假设，它们将包括其投资者阿里巴巴和雅虎日本以及 Naspers 投资组合的其他公司(包括 Flipkart、Avenida 和 OLX)拥有的平台。

其中一些企业相互竞争。当在一封电子邮件中被问及 Twiggle 在利用不同网站的数据构建知识图谱时，将如何处理可能的利益冲突时，柯尼斯伯格回答说，“我们非常清楚公司对其数据隐私的重视，并在确保最高透明度的同时，最大限度地尊重这些信息。”

除了来自阿里巴巴的资金，Twiggle 还表示，它已经将 Udi Manber 加入董事会，并将 Noa Ganot 任命为产品副总裁。科尼斯伯格说，曼伯曾担任雅虎首席科学家、谷歌工程副总裁和亚马逊搜索子公司 A9.com 的首席执行官，是“我们行业的传奇人物”。加诺曾是易贝以色列公司的产品负责人。
