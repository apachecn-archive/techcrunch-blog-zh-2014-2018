# 再见 TC，你好 GV 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/06/04/so-long-tc-and-hello-gv/>

我努力思考了这篇文章，但是时间并没有让它变得更容易写。在 TechCrunch 工作了五年半之后，我将作为运营合伙人加入谷歌风投公司。

我准备好迎接新的挑战。当人们问我最喜欢我工作的哪一点时，我的回答是一样的:“我很幸运能够讲述创始人的故事。”正是这种讲述企业家故事的热情让我加入了谷歌风投。

我现在不能透露很多细节，但我们有一个令人兴奋的愿景，创造一些风投公司从未开发过的东西。对我来说，这是一个从零开始帮助建立一些东西并以新的方式延伸自己的机会。今年夏天晚些时候，在我有几周时间安顿下来并发展这个新角色之后，再回来看看。我期待着尽快分享更多！

在过去的五年里，比尔·马里斯、大卫·克莱恩和他们的团队报道了谷歌风险投资，很明显，他们对投资的未来有着真正独特的看法，以及他们如何帮助每个企业家实现创建十亿美元以上公司的梦想。

我还想补充一点，GV 已经能够吸引一些前技术专家，包括 MG Siegler 和 Laura Boychenko(所以我不会觉得离我在山景城和南方公园的 GV 办公室的旧居太远)。

五年半前，我给迈克尔·阿灵顿发了封邮件，请求在 TechCrunch 实习。我写道，作为一名新闻学院的毕业生，我知道新闻业的前沿是网络，我会免费工作(这可能不是写给潜在新雇主的正确之举，但也不错)。

一个月后，TechCrunch 首席执行官希瑟·哈德(Heather Harde)回复说，他们实际上可以付钱给我(万岁！)我想在一月份开始。三周后，我来到阿林顿在阿泽顿的家，迎接我的第一个技术人员是他心爱的甜蜜巧克力实验室 Laguna。我的第一个想法是，这是一个好兆头。

的确如此。一月份的那一天是一段奇妙旅程的开始，这段旅程带给我的职业满足感、终身友谊和总体幸福感超过了我以前做过的任何一份工作。直到我开始为 TechCrunch 写博客，我才明白“我爱我的工作”这句话的含义。

TechCrunch 是我开始写作和视频生涯的一个神奇的地方，我很幸运能与才华横溢的作家和编辑一起工作。当然，有段时间我可能想辞职，但那些日子也教会了我什么样的工作值得去争取:一份给你自由和机会去披露重大收购和融资消息的工作；采访一些我们行业中最优秀、最聪明的 CEO、创始人和风投；最重要的是，讲述创始人的故事，以及他们如何用自己的想法改变世界。

而且我知道，目前的编辑团队，在亚历克西娅 Tsotsis 和马修·潘扎里诺的强有力领导下，将继续独家新闻、突发新闻和撰写有见地的文章。

感谢迈克尔、希瑟和 Erick Schonfeld 在我身上下注(还付钱让我去实习！)并感谢 Eric Eldon、Alexia 和 Matthew 的领导、支持和友谊。

最后，我想说的是创业者、读者和整个 TechCrunch 社区，正是他们让我们蓬勃发展。感谢你们在过去五年半的时间里，成为我的共鸣板、镜子和灵感。有时我们会有分歧，但你一直在让我成为一个更好的作家，也帮我找到了真正的激情。
