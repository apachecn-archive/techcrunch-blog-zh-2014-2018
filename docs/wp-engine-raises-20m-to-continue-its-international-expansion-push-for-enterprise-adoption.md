# WP Engine 筹集 2300 万美元继续其国际扩张，推动企业采用

市场上领先的托管 WordPress 平台之一 WP Engine 今天宣布获得由[北桥成长股权](https://web.archive.org/web/20230212160757/http://www.northbridge.com/growth-equity)领投的 2300 万美元后续融资。这使该公司迄今为止的资金总额达到 4120 万美元。

WP 引擎团队表示，它将利用新流入的现金推动其国际增长战略，并加快企业对其产品的采用(这通常意味着销售和营销团队的扩张)。这家成立于 2010 年的公司表示，它已经在 128 个国家拥有大约 30 万个网站和应用程序。它的一些标志性企业客户包括帕布斯特酿酒公司、AMD 公司和优步公司。

WP Engine 首席执行官希瑟·布鲁纳(Heather Brunner)今天在一份声明中表示:“去年 WP Engine 取得了惊人的增长:我们的团队规模扩大了一倍，开设了伦敦和圣安东尼奥办事处，并通过我们的实验室推出了创新的新产品。”“这笔资金将帮助我们保持这一势头，并增强我们投资于卓越客户体验、新创新和更好地服务于不断增长的全球客户的能力。”

鉴于 WordPress 的受欢迎程度，它现在为[超过 20%的互联网网站提供支持，围绕它的生态系统也吸引了风险投资者的浓厚兴趣也就不足为奇了。我们最近看到了对](https://web.archive.org/web/20230212160757/http://w3techs.com/technologies/details/cm-wordpress/all/all)[飞轮](https://web.archive.org/web/20230212160757/https://getflywheel.com/layout/announcing-the-flywheel-seed-funding/)(专注于设计师)[万神殿](https://web.archive.org/web/20230212160757/https://techcrunch.com/2014/05/15/pantheon-raises-21-5m-series-b-round-to-expand-its-web-hosting-platform/)(最初是 Drupal 平台，但现在已经扩展到 WordPress)等公司的投资。

当其他人试图专注于特定的利基市场时，WP Engine 正在追逐博主、营销人员、中小型企业和企业的广阔市场。它为自己的平台带来了几个前沿的特性，包括对 HHVM 的支持，脸书的(明显更快的)PHP 引擎替换。
