# MSI Trident 3 Arctic 将疯狂的游戏能力融入一个小盒子中 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/10/30/the-msi-trident-3-arctic-packs-crazy-gaming-power-into-a-tiny-case/>

最重要的是:这台电脑很瘦。微星三叉戟 3 北极是一个令人惊讶的小包装游戏装备。在测试过程中，他们发现这款电脑能够运行最新的虚拟现实硬件和游戏，尽管这款微型电脑缺乏标准机箱中的传统冷却功能。微星三叉戟 3 北极版是一款游戏主机杀手。

## 规格:

*   Windows Home 10
*   英特尔酷睿 i7-7700 3.6GHz 8M 高速缓存
*   H110 芯片组
*   微星 GTX 1070 8GB GDDR5
*   SO-DIMM DDR4 2133 MHz 16GB (8GB*2)
*   13.63 英寸 x 2.83 英寸 x 9.15 英寸
*   6.9 磅

## 回顾:

游戏公司长期以来一直提供不同外形的电脑。一般情况下，机箱越大，计算机越强大。这仍然是与 MSI Trident 3 Arctic 相关的情况。这并不是 MSI 提供的最强大或配置最好的计算机。相反，该公司将一个有能力的系统打包成一个 Xbox One 大小的包。基本上，这是一台围绕 MSI GTX 1070 构建的计算机，对我来说很好。

案例本身是有趣的部分。它有点小，最好垂直摆放。如果坐在它的旁边，冷却风扇似乎在挣扎，尽管我没有注意到任何图形退化。这个设计引人注目。看起来如果 Nvidia Shield 是一台计算机。和白色。

尽管总体目标显然是制造一台小型计算机，但整个系统中有大量的端口。前面板配备了常见的 USB 和音频端口，而背面的输入和输出几乎与 Trident 更大的兄弟姐妹一样多。

[![](img/658612ef382156c65efc1ccd80b28353.png)](https://web.archive.org/web/20230209045211/https://techcrunch.com/wp-content/uploads/2017/10/msi-trident-1-of-2.jpg)

尽管该系统体积小巧，但仍可升级。用户将能够更换包括 GPU、内存、存储驱动器在内的组件，除了处理器。该系统配有定制的 MSI 主板，并将最新的 Kaby Lake CPU 集成到系统中，CPU 直接集成到主板中。

不过，大小是有权衡的。330 瓦的电源是外置的，因此用户将不得不处理一个变得有点热的大功率块。

这个系统符合它的规格。英特尔酷睿 i7-7700 处理器的主频为 3.6Ghz，微星 GTX 1070 处理器的主频为 8GB，在测试过程中，性能从来不是问题。

系统发出咕噜声——当我说咕噜声时，我的意思是它在负载下运行平稳，尽管有一种可以听到的嗡嗡声，因为风扇尽最大努力让酷睿 i7 和 1070 尽可能保持冷却。不管内容是标准游戏还是虚拟现实耳机，我发现 Trident 3 Arctic 可以轻松处理一切，我可以很好地说这台电脑可以处理任何你可以扔给它的游戏。

尽管它并不完美。这是一个你可能不希望放在卧室或客厅的系统。尽管我的测试电脑还没那么旧，但在游戏过程中风扇一直开着，而且随着时间的推移，它们可能会变得更吵。这台电脑显然是为客厅设计的，对我来说，经过一点测试后，三叉戟北极 3 对安静的客厅来说太大声了。

值不值得？不，当然不是。对于游戏玩家来说，预装电脑很少是花钱的最佳方式。如果省钱是优先考虑的事情，一个类似的系统(尽管是一个更大的箱子)可以比 Trident 3 Arctic 的 1449 美元的建议零售价低 300-500 美元。Corsair、Zotac 和 VoodooPC 提供小型电脑，但这款来自 MSI 的电脑要便宜几百美元，部分原因是 GTX 1070 而不是 1080。尽管如此，大多数图形评论认为 1070 是 1080 的有力替代品，因此游戏玩家应该明智地节省资金。

由于成本原因，我很难推荐预建的系统，但 MSI 的这个系统有点不同。它太小了，以至于你的客人会认为它是一台游戏机，而不是一台羽翼丰满的游戏电脑。事实就是如此。这是一款完整的游戏电脑，价格比其直接竞争对手更低，在某些情况下，甚至比其他小型电脑还要小。

微星三叉戟 3 北极是强大的小野兽。如果你可以处理一点风扇噪音，并需要尽可能最小的游戏电脑，这可能是你的电脑。
