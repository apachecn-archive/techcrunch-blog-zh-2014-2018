# 当支付处理成为商品 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/08/08/what-happens-when-payment-processing-becomes-a-commodity/>

**编者按:** *克里斯多佛·奥·赫恩斯是挪威管理咨询公司 Core Group 的合伙人。克里斯托弗目前受聘于挪威最大的金融机构之一 Sparebank1 此分析基于公开来源，与任何客户约定无关。*

今年早些时候，银行业讨论的一个大主题是发布的[千禧一代颠覆指数](https://web.archive.org/web/20221207011040/http://www.millennialdisruptionindex.com/)，该指数指出，千禧一代认为银行无关紧要，与其他 B2C 行业相比，传统零售银行业务面临最高的颠覆风险。[埃森哲(Accenture)的《2020 年银行业报告》](https://web.archive.org/web/20221207011040/http://www.accenture.com/SiteCollectionDocuments/PDF/Accenture-Banking-2020-POV.pdf)证实了这一点，并将其与 20 年前电信业面临的挑战相提并论，称到 2020 年，非银行业务将占据现有银行收入的三分之一。

随着移动钱包、点对点支付、小额贷款和各种个人金融工具的兴起，银行业面临着来自科技行业的新一代竞争对手。值得注意的例子包括易贝，它与 PayPal 在支付领域已经有很长时间了，现在正在通过将苹果的指纹识别器整合到 PayPal 来加强其在移动支付领域的立足点。

谷歌试图通过停产的谷歌收银台和谷歌钱包进入支付领域。亚马逊通过收购 GoPago 瞄准了移动支付领域[，并计划推出](https://web.archive.org/web/20221207011040/https://beta.techcrunch.com/2013/12/21/amazon-bought-gopagos-mobile-payment-tech-and-productengineering-team-doublebeam-bought-the-pos-business/)[类似的支付处理解决方案](https://web.archive.org/web/20221207011040/http://9to5mac.com/2014/07/27/amazon-apparently-set-to-launch-square-competitor-next-month-as-it-preps-biometric-payment-tools/)来挑战 Square。另一方面，脸书瞄准了 5000 多亿美元的全球汇款市场。

这些挑战者的共同点是，大多数公司都瞄准了支付处理领域。美国银行和 Capital One 都表示不要担心，因为这已经是旧闻了，只会扰乱银行价值链的外围部分。但是，与任何形式的价值链中断一样，真正的影响是支付处理成为商品的可能性。

支付处理商品化的催化剂是加密货币和新支付协议的引入，如[比特币](https://web.archive.org/web/20221207011040/https://en.wikipedia.org/wiki/Bitcoin)和 [Ripple](https://web.archive.org/web/20221207011040/http://en.wikipedia.org/wiki/Ripple_(payment_protocol)) ，这使得清算变得过时，并大大降低了商户的交易成本。相比之下，根据交易规模的不同，通过 Visa/Mastercard/PayPal 支付的交易成本在 3-5%之间。另一方面，随着不断努力降低比特币社区的交易费用，比特币的交易成本低至 1%。

为了加快发展 [Bitpay](https://web.archive.org/web/20221207011040/http://www.crunchbase.com/organization/bitpay) 最近宣布取消启动计划中的交易费，向接受比特币的商家提供[免费无限支付处理](https://web.archive.org/web/20221207011040/http://gigaom.com/2014/07/30/bitpay-cuts-transaction-fees-on-starter-plan-making-bitcoin-payment-processing-free-and-unlimited-for-merchants/)。随着[易贝考虑将比特币整合到 PayPal](https://web.archive.org/web/20221207011040/http://www.businessinsider.com/ebay-is-considering-integrating-bitcoin-into-paypal-2014-5) 以及苹果将 [Blockchain 等比特币钱包重新纳入 App Store](https://web.archive.org/web/20221207011040/https://beta.techcrunch.com/2014/07/28/blockchain-bitcoin-wallet-is-back-in-the-apple-app-store/) 加密货币作为默认支付方式，对于寻求经济高效解决方案的逐利商家来说，这是一个诱人的选择。这通过数字钱包和颠覆性支付平台将挑战者置于银行业和零售业之间的某个最佳位置。

支付处理的商品化将需要新的商业模式，在这种模式下，现金不再是王道，但对通过交易收集的信息进行分析是数字支付处理领域的新竞争优势。 [Ovum 的报告“忠诚度和基于位置的支付服务”](https://web.archive.org/web/20221207011040/http://payments.banking-business-review.com/news/analytics-and-loyalty-will-drive-mobile-payments-growth-says-ovum-280714-4328183)预测忠诚度和分析是移动支付的主要增长动力。它表示，这将需要支付提供商、商家和第三方分析供应商(如 Oracle 和 IBM)之间建立新的合作关系，[后者已经与 Monitise](https://web.archive.org/web/20221207011040/http://www.zdnet.com/ibm-monitise-partner-to-boost-mobile-payment-solutions-7000031806/) 建立了合作关系，以便为金融服务行业提供基于云的移动解决方案。

星巴克是结合忠诚度计划和支付的商业潜力的一个例子，它报告说，从第三季度开始，移动支付占美国销售额的 15%以上，并且正在考虑向其他商家出售其软件。

与金融危机前相比，这些变化与行业融合导致的行业复杂性增加、[资本要求法规](https://web.archive.org/web/20221207011040/http://en.wikipedia.org/wiki/Bank_regulation)、行业整合和[股本回报率(ROE)下降](https://web.archive.org/web/20221207011040/https://www.bcgperspectives.com/content/articles/financial_institutions_sales_channels_distribution_2020/)相结合，在未来几年为银行带来了巨大挑战，并创造了一场完美风暴，似乎有利于新进入者和技术行业内在的试验能力和尝试失败的意愿。

尽管有企业家精神和对陈旧的遗留系统的独立性，进入壁垒正在上升。Glen Fossella 在接受[美国银行家](https://web.archive.org/web/20221207011040/http://www.americanbanker.com/bankthink/why-banks-will-win-the-payments-war-1069019-1.html)的采访时表示，监管复杂性和合规性要求将成为新进入者越来越大的负担，以至于现任者收购初创公司或与之合作既快又便宜。Square 付出了惨痛的代价，在佛罗里达州因无照经营被罚款 50.7 万美元。他还建议银行投资初创公司，收购和聘用企业家或从科技公司聘用人才。

在为数字支付的兴起做准备方面，Visa 是一个领先的例子。Visa 首席执行官 Charles W. Scharf 表示，领先的支付网络正在依靠新的数字服务来推动未来的增长，尽管 Visa 第三季度全球支付收入增长了 11%。其中一项举措是创建 Visa 数字解决方案，提供多种产品。

这包括推出 Visa Checkout，V.me 的继任者，Visa 云支付解决方案，以及更令人兴奋的是，用数字令牌取代传统信用卡号码的[令牌化服务](https://web.archive.org/web/20221207011040/http://digitaltransactions.net/news/story/Tokenization-Plays-the-Central-Role-in-Visa_s-New-Cloud-Payment-Suite)。Visa 还开放了与开发者和技术公司在这些新服务上的合作。此外，Visa 透露了[对 LoopPay](https://web.archive.org/web/20221207011040/http://thefinancialbrand.com/41211/looppay-visa-mobile-payments-wearables/) 的投资，这是一种在大多数销售点终端接受的移动支付解决方案。

通过这些举措，Visa 为支付生态系统中的现有企业树立了一个领先的榜样，展示了一种尝试失败的心态，以及通过反复的移动钱包举措、投资于有前途的初创企业的意愿以及与合作伙伴合作鼓励开放式创新的长期承诺。银行和金融机构应该将 Visa 视为领先的榜样，以确保在数字支付生态系统中的地位。

面对现实吧。千禧一代永远不会认为银行很酷。但银行代表着消费者的安全，通过采用新技术和获得新创新，有可能让银行业变得不那么无聊。
