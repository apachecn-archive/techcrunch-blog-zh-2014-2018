# 英国初创公司呼吁政治家改善签证、宽带、技能、监管 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/09/03/uk-startups-call-on-politicians-to-improve-visas-broadband-skills-regulation/>

在下届大选之前，超过 150 家英国科技创业公司和投资者支持一项新的“创业宣言”(在 [PDF](https://web.archive.org/web/20221007041350/http://www.coadec.com/wp-content/uploads/2014/09/Startup-Manifesto.pdf) 或 [Slideshare](https://web.archive.org/web/20221007041350/http://www.slideshare.net/Coadec/startup-manifesto) 中)，旨在游说英国主要政党制定政策，以建立一个科技驱动的社会。

非党派的 24 点创业宣言，由非盈利的[数字经济联盟](https://web.archive.org/web/20221007041350/http://coadec.com/)(被称为 Coadec)发布，要求政策制定者——除其他事项外——投资超高速宽带，放宽雇佣国际人才的规定，包括简化企业家和毕业生的签证，改善学校的编码教学。

宣言的支持者包括英国科技界的名人，包括 King、TransferWise、SwiftKey、Lovestruck、Funding Circle、MOO、Index Ventures、Balderton Capital、Passion Capital、Techstars London、Seedcamp 和 Accel Partners。

该宣言由 Coadec 执行董事盖伊·莱文(前财政大臣乔治·奥斯本(George Osborne)议员的经济顾问，在野时)撰写，迄今为止已收到政治家们令人鼓舞的反应。

Chuka Umunna 议员(影子商业大臣)说:“我们希望看到更多的人创业、领导和从事商业活动，并创造高技能、薪酬更高的工作岗位。英国蓬勃发展的数字经济在应对这两大挑战方面发挥着巨大的作用，这就是为什么 Coadec 的宣言正确地强调了数字创业的重要性以及培养数字技能的必要性。”

马修·汉考克议员(商业、创新和技能部国务部长)评论说:“毫无疑问，还有更多事情要做，我们将仔细研究任何建议，使英国成为世界上最适合创业和发展的地方。”

它主要呼吁:

为企业家减税，包括减少志愿教授编程的初创公司的国民保险缴款(NICs ),

对被颠覆性技术淘汰的人进行再培训，而不是通过保护性监管阻碍创新，

通过恢复毕业后工作签证，允许风投代表初创公司赞助员工，向欧盟以外的科技人才敞开大门，

通过为比特币创建一个法律框架，以及将反洗钱程序所需的身份检查数字化，支持英国的金融科技公司。

罗宾·克莱因(Index Ventures 合伙人)表示:“过去几年，英国在成为全球最佳创业和运营公司的地方方面取得了显著进步。然而，还有许多工作要做。这份宣言充满了实用的想法，总的来说，将朝着英国的目标迈出更有意义的步伐。”

达米安·金梅尔曼(DueDil 创始人兼首席执行官)说:“如果被采纳，这些建议会给我们在竞争激烈的全球经济中所需要的优势。”

乔恩·布拉德福德(伦敦 Techstars 医学博士):“在过去的四年里，政府做了大量的工作来培育创业生态系统……但现在是一个重要的关头:如果我们要继续这条增长之路，我们需要巩固迄今为止取得的成就。”

TransferWise 的联合创始人 Taavet Hinrikus 说:“十年内，技术将改变我们所知道的世界。比如，银行业已经在发生变化，而伦敦似乎——至少目前——走在了前面。然而，下一届政府可以做更多的事情来确保这一点。”

Tom Raffe(screen burn 创始人)说:“学校的计算是我特别感兴趣的一个问题，因为我们需要继续培养一流的开发人员，以便在全球范围内完成。为了规划未来，有必要彻底审查学校的信息和通信技术教育。”

Coadec 由谷歌、TechHub、iHorizon 和 Intuit 赞助。

> **创业宣言推荐:**
> 
> **改善融资渠道**
> 1。承诺为下届议会保留 SEIS 和 EIS 税收减免
> 2。恢复对企业风险资本的税收减免。取消企业家减免上限，降低股权门槛
> 
> **提升获得天赋**
> 4。恢复 STEM 毕业生的读后工作签证
> 5。让创业公司更容易从海外招聘员工。改革研究生企业家签证
> 7。向获得天使投资人和众筹平台资金的人开放企业家签证
> 8。在政府数字服务的支持下审查和简化签证流程
> 9。投资帮助教师讲授新的计算课程
> 10。为个人和创业公司提供激励，帮助培训教师和学生。推广免费工具，包括 Moocs、Codecademy 和 code clubs 等第三部门倡议
> 
> **建设世界一流的数字基础设施，增加可负担得起的办公空间的供应**
> 12。继续投资超高速连接，提高数字基础设施的目标水平。审查规划系统和物业法规，以增加初创企业负担得起的办公空间的供应
> 
> **把法律法规带入 21 世纪**
> 14。将支持颠覆性创新作为政府政策，创造一个人和企业都能适应的环境
> 15。鼓励未经许可的创新
> 16。向信息专员办公室投入更多资源。创建一个数据保护框架，让消费者充满信心，同时不会扼杀创新
> 18。继续支持金融科技创新。审查有关协作消费和“共享经济”的法规
> 
> **用数字政府解锁创新**
> 20。让创业公司更容易卖给政府。承诺保持和扩大政府数字服务
> 22。创建安全在线身份认证标准
> 23。加快向“政府即平台”的发展，包括为政府服务发布 API
> 24。进一步开放数据，包括所有政府服务的实时绩效仪表板
