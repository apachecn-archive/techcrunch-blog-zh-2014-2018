# 数以千计的 xHamster 登录凭证出现在网上

色情网站 xHamster 的成员今天应该改变他们的密码，因为网上出现了一组近 38 万个用户名、电子邮件和糟糕的散列密码。

仅限订阅的漏洞通知网站 [LeakBase](https://web.archive.org/web/20230118085139/https://leakbase.pw/) 公布了一组登录凭证，据[主板报道](https://web.archive.org/web/20230118085139/http://motherboard.vice.com/read/hackers-are-trading-hundreds-of-thousands-of-xhamster-porn-account-details)这些凭证正在网上交易。尚不清楚该数据库的确切来源，但它只包含 xHamster 1200 万注册用户中的一小部分信息。虽然 xHamster 不要求观众在网站上注册，但注册的观众可以发表评论并制作视频播放列表。

尽管如此，泄露的信息仍有可能让用户感到尴尬——其中几个账户与美国军队和其他政府电子邮件地址相关联。如果 xHamster 的用户在其他网站上重复使用他们的密码，那么他们在这些网站上的帐户也有被盗用的风险。

“所有 xHamster 用户的密码都经过适当加密，因此几乎不可能被黑掉。因此，所有的密码都是安全的，用户数据也是安全的。

但 LeakBase 的发言人不同意这种说法，他告诉 TechCrunch，这些密码是用 MD5 算法散列的，这被认为是不安全的。这位发言人说:“MD5 散列值很小，很容易被破解。”“他们认为哈希是安全的，这一事实是一个明显的例子，说明即使在今天，公司也存在安全缺陷。”

LeakBase 向 TechCrunch 提供了一组 60 个用户的凭据，以核实违规行为，这些电子邮件看起来确实与注册的 xHamster 账户相符。
