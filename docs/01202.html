优步与 11 个合作伙伴一起开放其 API，包括 OpenTable、猫途鹰和联合航空 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2014/08/20/uber-api-part-deux/

我们告诉你它就要来了，它就在这里:上周，我的同事 Josh Constine 报告说优步正在准备发布一个 API ，这将使其按需打车服务能够集成到许多第三方应用程序中。今天，该公司宣布其 API 将对应用开发者开放，已经有 11 个合作伙伴签约。

对优步来说，其 API 的引入旨在增加覆盖面，并让它出现在大量新的潜在用户面前。该公司遍布全球近 50 个国家的 150 个城市，但还有无数用户可能没有尝试过它的服务。在这些用户已经安装的其他应用中显示是向他们介绍获得优步点播的简单方法。

这并不是优步第一次向第三方提供其服务:今年 5 月，它与谷歌地图整合，使该应用的用户能够立即请求其驱动程序之一。但今天的声明代表了一个更加雄心勃勃的愿景，即它可以在其他应用程序中广泛分布。

为此，优步正在向任何希望找到有趣的方法将服务嵌入应用程序的开发者开放其 API。这意味着任何可以向优步发送目的地地址的应用程序都将能够显示取车时间和费用估计，而用户不必离开他们的应用程序。对于一小部分精选合作伙伴，优步还将允许用户无需打开自己的应用程序就可以申请汽车。

这不仅仅是向第三方开发者提供 API，而是激励他们使用它。为了让尽可能多的应用程序使用它的服务，它推出了一个联盟项目，为那些让人们使用它的服务的开发者提供免费积分。

虽然它试图吸引尽可能多的应用程序开发者，但 start 优步正在与 11 个 API 合作伙伴一起推出，这些合作伙伴已经承诺向他们的用户介绍优步。这些应用包括 Expensify、Hinge、Hyatt Hotels & Resorts、Momento、OpenTable、Starbucks、Tempo、Time Out、猫途鹰、TripCase 和 United Airlines。

“我们认为这对我们来说意义重大，”商业埃米尔的优步·SVP 在电话采访中告诉我，然后重复道，“这对我们来说意义重大。”

他认为，总的来说，可能有大约 2 亿用户下载了这些初始启动应用程序。为了让乘客上车，优步还将向通过这些应用注册的新用户提供高达 30 美元的奖励。

对于优步及其合作伙伴来说，API 和在第三方应用上提供的能力都是为了简单。这是为了能够为用餐者、旅行者，甚至只是与陌生人约会的人提供无缝的端到端体验。

优步将与其发布合作伙伴一起发布各种不同的使用案例。使用 OpenTable 这样的应用程序预订晚餐的用户将能够预订一辆车带他们去那里。或者那些乘坐联合航空公司航班的人会有车送他们去机场，如果他们住在凯悦酒店，优步会带他们去那里。当然，将优步与 Tempo 这样的智能日历应用捆绑在一起，可以确保用户按时参加会议。

优步的发布合作伙伴已经同意将优步作为他们的独家地面运输选择，未来的开发者也将被禁止使用竞争性的 API。尽管如此，迈克尔相信优步是目前唯一一个有足够规模来打击它所拥有的全球伙伴关系的服务。

“我们在 150 多个城市和近 50 个国家……(我们)每天每时每刻都在接触每个城市的每一寸土地，”他说，这种规模很重要，尤其是当你与联合航空这样的航空公司打交道时，它服务于 40 或 50 个不同的国家，或者与星巴克这样的品牌打交道，它在世界各地都有业务。

有了新的十亿美元和银行里的变化，难怪优步想要出现在更多的地方和更多的应用程序上。毕竟，它试图让尽可能多的用户使用这项服务，而最好的方法是通过人们已经在使用的应用程序，以及它自己的应用程序。
