# Zendesk 通过收购 扩展到 CRM 领域

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/10/zendesk-expands-into-crm-with-base-acquisition/>

Zendesk 主要局限于客户服务场景，但这似乎还不够。如果您想真正了解交互背后的客户，您需要一个客户记录系统来配合客户服务组件。为了满足这一需求，Zendesk 宣布它正在收购[基地](https://web.archive.org/web/20230320003404/https://getbase.com/)，一家已经筹集了超过 5000 万美元的初创公司。

两家公司没有透露收购价格，但 Zendesk 报告称，此次收购不会对收入产生重大影响。

尽管 Base 在 CRM 领域的知名度可能不如 Salesforce、微软(Microsoft)或甲骨文(Oracle)，但它已经创建了一个复杂的销售队伍自动化平台，并拥有自己的人工智能基础。首席执行官简自豪·什米洛维奇声称，他的公司的人工智能可以与更富有的竞争对手[竞争，当它在 2016 年](https://web.archive.org/web/20230320003404/https://techcrunch.com/2016/10/04/base-wants-to-compete-with-the-big-boys-in-the-crm-artificial-intelligence-game/)发布时，可以为销售人员提供如何更加成功的有意义的规范性建议。

Zendesk 首席执行官 Mikkel Svane 当然看到了将 Base 这样的公司加入他的平台的价值。他在一份声明中说:“我们希望为销售做 Zendesk 已经为客户服务做的事情:为销售人员提供围绕他们和他们所服务的客户构建的工具。”

密切关注这一市场的 CRM Essentials 公司的所有者 Brent Leary 说，如果客户数据的核心包括客户服务、CRM 和营销，Base 就会给 Zendesk 增加一个缺少的部分。

Leary 告诉 TechCrunch:“Zendesk 在客户服务方面有很好的地位，但现在为了加强他们在寻找集成平台的中端市场/企业客户中的地位，Base 为他们的难题增加了一个强大的移动销售人员自动化部分。”

正如他指出的那样，我们已经看到 HubSpot 在 HubSpot 应用程序上做出了类似的举动，而最近[被出售给 Accel-KKR](https://web.archive.org/web/20230320003404/https://www.bizjournals.com/sanjose/news/2018/08/15/sugarcrm-funding-investment-growth-accel-kkr.html) 的 SugarCRM 也可以购物，因为它的新主人更有钱了。“这几乎就像一个 CRM 企业软件饥饿游戏正在进行，”他开玩笑说。但他指出，我们应该期待更多的整合，因为这些公司试图收购其平台的缺失部分，以提供更完整的解决方案。

曾在 Zendesk 担任产品组合高级副总裁的马特·皮尔斯将领导基础团队向前发展。

Base 成立于 2009 年，拥有 5000 多名客户。值得指出的是，Base 已经在公司的应用程序市场上出售，所以这里有一些重叠，但公司打算尝试将现有客户转移到 Base，当然。

Zendesk 表示将继续支持所有基础客户。此外，Base 的 125 名员工已被邀请加入 Zendesk，因此不会在这里放血。
