微软证实 Windows 10 将于今年夏天发布，并透露与小米的测试

微软今天证实，Windows 10 将于今年夏天在 190 个国家和 111 种语言中发布。然而，令人沮丧的是，该公司并没有透露其备受期待的新平台的实际发布日期，但今天发布了一条有趣的新信息。

该公司此前表示，打算向“数百万”用户提供 Windows 7 或更早版本的免费升级，而中国——盗版的代名词——是该计划的一个重点。微软负责操作系统的执行副总裁特里·迈尔森(Terry Myerson)在中国深圳的一次活动中透露，微软将与中国三家最知名的软件公司——联想、腾讯和奇虎 360——合作，为他们的集体客户群提供免费升级。这无疑涵盖了大量的潜在用户。

但是，除此之外，微软还与小米——没错，就是快速增长的安卓手机公司——合作，这将让小米 4 的部分用户有机会“帮助测试 Windows 10，并为其今年晚些时候的未来发布做出贡献”

小米有一个强大的用户社区——该公司每周都会推送软件更新，并积极征求用户反馈——微软正在利用这一点来播种 Windows 10 技术预览版，以获得初步反馈。

小米的一位发言人告诉我们，这是“一个完全由微软领导的实验性项目”，我们正在试图确定更多关于这种合作关系性质的信息。这很糟糕，但微软和小米对这一联盟含糊其辞，尽管这是今天发布的低调消息，但它可能为 Windows 10 在移动设备上的未来计划提供了一个有趣的迹象。
