# 观看 Ehang 的无人驾驶客机飞行

像许多参加 2016 年 CES 的人一样，[我们被 Ehang 184 带到了](https://web.archive.org/web/20221018005759/https://beta.techcrunch.com/2016/01/06/the-ehang-184-is-a-human-sized-drone-taking-off-at-ces/)。如果没有别的，无人驾驶客机是所有智能手机和巨型电视的一个很好的突破。也就是说，大型四轴飞行器只不过是一个巨大的纸张重量和一些愚蠢的合成视频。

然而，今天早些时候，该公司展示了一些令人印象深刻的无人机搭载真人的视频。在与实地测试视频相关的一份新闻稿中，该公司表示，它已经测试了 40 名乘客，包括其首席执行官胡华智和中国广州市副市长王东，该公司在广州进行了测试。

视频

在该公司成立的四年左右时间里，据其统计，该公司总共进行了“数千次试飞”——但这段视频代表了该飞行器在运行中的第一批清晰证据。

没有消息表明该公司将于何时推出这款产品。这可能在一定程度上取决于它计划销售的国家的监管。去年这个时候，[迪拜宣布计划](https://web.archive.org/web/20221018005759/https://beta.techcrunch.com/2017/02/14/dubai-plans-to-introduce-flying-drone-taxis-as-early-as-this-summer/)使用这种飞行器作为出租车服务，努力帮助减少拥堵。当时，它乐观地争取“最早在(去年)夏天”推出。

当然，当进入城市上空的小型飞行器时，最大限度的谨慎是件好事。胡在新闻稿中说:“进行载人试飞使我们能够证明我们的飞行器的安全性和稳定性。”。“我们做的不是极限运动，所以每位乘客的安全永远是第一位的。”

很公平。
