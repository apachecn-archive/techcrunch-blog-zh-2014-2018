苹果可能正在组建一个增强现实团队

苹果可能会在增强现实上下大赌注。根据新彭博的报道，该公司可能正在为iPhone开发增强现实功能，然后为眼镜开发，这是一种可以与智能手机配对的新设备。

虽然假想的眼镜还远未准备好，但看看苹果在幕后的工作仍然很有趣。它首先从一个团队开始。

迈克·洛克威尔显然正在领导这个新的增强现实团队。他之前在杜比公司工作，研究新技术和硬件。去年春天，罗克韦尔组建了一个经验丰富的工程师团队，负责增强现实。

据彭博称，该小组成员包括:

  • 科迪·怀特(Cody White)，前亚马逊首席工程师，曾在亚马逊的CryEngine游戏引擎 fork 工作
  • Duncan McRoberts， Meta 的前软件开发总监
  • 前Oculus研究员尤里·彼得罗夫
  • Avi Bar-Zeev，前亚马逊和微软工程师，从事HoloLens等工作
  • 为好莱坞大片制作特效的各种工程师

在苹果公司从事其他项目的硬件和软件工程师也加入了这个团队。苹果还为这些项目收购了 Metaio 和 Flyby Media 。总的来说，数百名工程师现在可能正在研究增强现实。

所有这些听起来很酷，但它并没有真正告诉我们苹果为用户准备了什么。在彭博的报告中，这方面的细节很少。

苹果可以从相机应用程序中的基本增强现实功能开始，如自拍滤镜和实时物体检测，从面部开始。换句话说，你可以期待iOS很快内置Snapchat风格的过滤器。虽然这不是增强现实本身，但该公司也可以超越肖像模式，让你在拍照后调整图像的焦点。这就是我们目前所知的全部。

如果苹果真的对增强现实感兴趣，那应该还有比这更重要的事情。彭博谈到了新的增强现实眼镜，这样你就可以向你周围的世界添加虚拟数据，但听起来该公司本身还不知道它想用这种新设备做什么。
