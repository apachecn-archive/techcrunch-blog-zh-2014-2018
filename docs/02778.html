微软将向树莓派开发者免费提供 Windows 10 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/02/02/windows-10-free-for-pi-devs/?ncid=rss

微软宣布它将在今年晚些时候向 Raspberry Pi 微型电脑的用户免费提供其最新版本的桌面操作系统 Windows 10——因为它试图跟上并保持与物联网发展的相关性。

Windows 物联网集团总经理 Kevin Dallas 在微软的 Windows 博客上写道:“我们将创客社区视为智能互联设备的惊人创新来源，智能互联设备代表了下一波计算的基础，我们很高兴能够成为这个社区的一部分。”。

树莓派基金会今天发布了其低成本微处理器 Pi 2 的下一代——增加了一个四核芯片和两倍的内存，同时保留了 35 美元的价格——用他们的话说，将制造商友好的硬件“牢牢地带入了个人电脑领域”。

35 美元的“入门级个人电脑”价位对微软来说既是一个扩大其操作系统使用的机会，也是一个威胁——考虑到用户可能会转向免费的 Linux 替代品。

向制造商赠送 Windows 10 是试图跟上 Pi 进军 PC 领域的步伐的一种方式，同时也能产生善意、参与度，并——微软希望——推动开发者社区采用其服务，随着更多类型的设备成为联网设备，开发者社区正在扩大。

在他的博客中，达拉斯指出，微软去年为物联网创建了 Windows 开发者计划，并为英特尔的伽利略板添加了对 Windows 的支持，伽利略板是另一种与 Arduino 开发硬件和软件兼容的单板计算机(如 Pi)。

他今天说，通过免费访问其开发工具来支持 Pi 是“下一步”，并补充说:“Raspberry Pi 2 是一个令人惊讶的强大设备，它为大量的人和技能水平打开了计算和编程的世界。

“我们很高兴与 Raspberry Pi 基金会合作，并分享 Windows 10 将支持 Raspberry Pi 2。我们将在未来几个月分享更多关于我们物联网 Windows 10 计划的细节。”

Redmond 上个月发布了 Windows 10，同时还有另一个旨在推动普及的免费服务:Windows 7、Windows 8.1 或 Windows Phone 8.1 的用户有资格在 Windows 10 的第一年内免费升级。
