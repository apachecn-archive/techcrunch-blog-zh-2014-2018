# 时代公司在 Speakr 的帮助下推出了自己的影响者营销网络

时代公司(Time Inc .)正在以更大的规模进入影响者营销业务，推出了一种名为 Time Inc. Connect 的新产品。

这个影响者网络将由初创公司[speaker](https://web.archive.org/web/20221207094319/http://speakr.com/)提供“动力”；正如时代公司数字业务发展和业务运营高级副总裁 Regina Buckley 向我解释的那样，这意味着时代公司 Connect 将结合 Speakr 的技术和独特的影响者群体。

“我们选择 Speakr 的原因是，他们代表营销人员处理和使用分析的方式非常聪明，”Buckley 说。“他们在自己的领域有优势，他们在我们的领域也给了我们优势。”

Speakr 首席执行官 Marco Hansell 补充说，Connect network“结合了我们所做的所有工作中的最佳成果”，具有帮助营销人员确定最适合他们活动的影响者(根据他们发布内容的语气和受众的人口统计数据等因素)，以及预测特定活动的覆盖范围和参与度等功能。

有影响力的人不是 real Time Inc .的编辑或记者，但该公司表示，他们已经过编辑团队的审查——最初的阵容包括《T4》的 Nguyen 和《我的健康美食》的 Jacey Duprie 以及《迪奥的少女》的 Jacey Duprie。

因此，举例来说，影响者可以与时代公司团队合作，为时代公司的资产创建原生广告，然后通过他们自己的网站和社交媒体帐户推广信息。

需要说明的是，这并不是该公司第一次在营销活动中与博客和社交媒体影响者合作——巴克利说，时代公司“随着影响者对营销人员越来越重要，不同的出版物已经开始自然地将他们作为内容创作者。”但她说，通过推出新的网络，该公司正在集中这一过程，并“创建一个卓越的中心。”

巴克利说，即使它进入了新的广告类型(内容和广告之间的界限会变得更加模糊)，该公司在这个过程中仍然保持着“政教分离”。它不会是“100%独立的”，但该网络将由编辑部的一名影响者经理监督，他将在影响者和时代公司的编辑之间进行协调。
