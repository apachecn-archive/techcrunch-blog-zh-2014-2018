# 这些黑客组装了一个假肢 Nerf 枪 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/03/09/these-hackers-put-together-a-prosthetic-nerf-gun/>

如果你少了一只手，你怎么能参加一场 Nerf 枪战？匠心独运的修补匠们通过组装一个可以用手臂肌肉控制的假肢神经枪解决了这个问题。

换句话说，尼古拉斯·于切特成为了《最终幻想 7》或《超级男人》中的[巴雷特·华莱士](https://web.archive.org/web/20221025222851/https://en.wikipedia.org/wiki/Barret_Wallace)。这是它的样子:

让我们更仔细地看看这个设备。特别是，Hackerloop 必须找到一种方法，用另一种射击手势取代 Nerf 枪上的扳机。

基枪是一把没有手柄的[群火 Nerf 爆能枪](https://web.archive.org/web/20221025222851/http://nerf.wikia.com/wiki/Swarmfire)。多亏了一些 3D 打印技术，Huchet 可以把这个设备作为右臂的假肢——这是一个定制的外壳。

Nerf 枪然后连接到一个类似 Arduino 的微控制器，该微控制器根据需要激活枪。最后，休特在他的肘部附近戴着三个电极。如果他收缩肌肉，电极就会向微控制器发送电活动。

如果电压达到某一水平，微控制器就发射 Nerf 枪。当然，于歇在巴黎的街道上玩着它。相当整洁！

过去，Hackerloop 曾从事过其他创造性的黑客工作。该团队用纸和泡沫在“Up”建造了一个房子的复制品，并将其发送到巴黎上空，用树莓 Pi 在 Instagram 上发布照片。

他们还开发了 [Nosulus Rift](https://web.archive.org/web/20221025222851/https://mashable.com/2016/08/16/vr-fart-simulator-south-park-nosulus-rift/#lo.uEDr3F8qM) ，这是一款 VR 屁模拟器，用于推广育碧的《南方公园》游戏(破碎但完整)。每次你在电子游戏里放屁，诺瑟卢斯裂缝就会散发出放屁的味道。
