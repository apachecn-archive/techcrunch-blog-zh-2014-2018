# 启动你的猜测引擎，苹果将停止其雷电显示器

苹果公司今天宣布，它将停止生产 Thunderbolt 显示器，这是一种大型外部显示器，许多人用来连接 MacBooks 或其他 MAC 电脑，以获得额外的屏幕空间。这很可能会加剧人们的猜测(这种猜测一直在持续),即苹果将很快推出 4K 或 5K 版本的显示器。

目前 Mac 的显示屏为 5K，可以扩展到现有的 Thunderbolt 显示屏(分辨率更低)，后者的分辨率为 2560×1440——但我可以根据个人经验告诉你，从可用性的角度来看，分辨率的差异很糟糕。
