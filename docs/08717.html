三星 Galaxy Note 7 评论

原文：https://web.archive.org/web/https://techcrunch.com/2016/08/16/samsung-galaxy-note-7-review/?ncid=rss&sr_share=twitter

原来是一场赌博。它很大，很奇怪，在所有的东西中，它是一个触控笔——一个被遗忘的(在当时是笨拙的)生产力硬件，似乎随着物理键盘而消亡。

令许多人惊讶的是，这也是一次泰坦尼克号的撞击。这一策略很快成为移动领域最容易识别的设备之一，在头两年就售出了 5000 万台。

三星的 Note 系列虽然不是旗舰产品(这一荣誉属于更为主流的兄弟产品 Galaxy S)，但它无疑是全球最具标志性的手机之一，拥有足够强大的追随者，可以重新定义整个行业接近屏幕尺寸的方式。第一代 5.3 英寸显示屏将“phablet”这个词从类似的科技术语转变为主流产品类别。

可以预见，经过几代人的努力，屏幕尺寸逐渐增大，先是 5.3，然后是 5.5，然后是 5.7，每年都会进行相当健康的硬件检修。上个月在一些联播的国际活动中宣布，最新的 Note 标志着手机回归到更大的 Galaxy 文件夹，该公司完全跳过 Galaxy Note 6，转而支持 Galaxy S7，试图将手机与 Galaxy S7 和 S7 Edge 统一为一个庞大的、充满空间的 70 多岁家庭。

很合适，真的。从许多方面来说，新手机感觉像是这两个系列的巅峰之作，是 Note 的外形、Galaxy S 的设计语言和顶级规格的交汇点——还有一些关键的新功能。Note 7 是一款真正的“厨房水槽”设备，旨在成为每个人的一切，从自拍社交网络到严肃的商界人士。再一次，它泰然自若地做到了。

自上一代以来，屏幕尺寸一直保持不变，实际尺寸也显著缩小。但是，这张纸条再次表明，一切都是为了做大。

很容易想象插播银河系列是某个更大的、手写笔包装的兄弟，却没有带来任何有意义的创新。但是，正如该公司 8 月份花哨的多国发布活动愉快地提醒我们的那样，该通知从未有过折中的办法。即使有标准的发射前泄漏电池，该公司仍设法提供一些惊喜，如虹膜扫描解锁和更大的电池。

当然，当你把每件事都做大时，清单上总是包括成本。随着起价徘徊在 800 美元左右，可以肯定地说，Note 7 不是为每个人设计的。

一瞥

  • 5.7 英寸 2560 x 1440 显示器
  • 64GB 存储(可通过 microSD 扩展)，4GB 内存
  • 2.15GHz、1.6GHz 四核处理器
  • 依赖于运营商的定价

职业选手

  • 漂亮的边缘对边缘曲线显示器
  • 改进的电池
  • 额外的安全功能

Cons

引人注目的容貌

Note 7 是个 looker，朴素简单。按照简化的命名惯例，这款新手机将平板手机的外形与 Galaxy 系列的外观完美结合。这一次最激进的美学偏离是采用了该家族 Galaxy Edge 侧的曲面显示屏。结果是边到边的显示，使得侧挡板在该过程中几乎不存在。这对于在 5.7 英寸显示屏上弹出的较小设备来说是一个不错的效果，同时带来了面板菜单和订阅通知等标准的额外屏幕功能。然而，最重要的是，在已经很大很漂亮的屏幕上看起来真的很好看。

该设备的后部被弯曲以匹配，这创造了一个很好的对称性。更重要的是，它有助于该公司正在进行的生产线任务:创造一个大屏幕设备，而不是感觉笨重。事实上，尽管 Galaxy Note 7 有一个大的 ole 屏幕，但拿在手中并不觉得巨大。事实上，我可以轻松地用手指握住这个装置。假设你没有画这些东西，它也可以很舒服地放进你的裤子口袋里。

不仅后部的形状与前部相似，而且由相同的材料制成——康宁的新大猩猩玻璃 5，该公司声称它可以在 80%的时间里承受 1.6 米的坠落。老实说，这次我真的没有勇气去冒险。有点烦人的是，它还使背部成为主要的指纹磁铁。我的用了一点就模糊了。还好我是那种总是带着羞耻的怪人。

双锥形设计形成了一对相当窄的侧面，左侧是音量按钮，右侧是电源按钮。三星发给我的设备的前玻璃和侧边带之间有一个小间隙，夹住了一些散落的棉绒。有点难看，但没什么一点压缩空气应该清理不了。

顶部是一个插槽，将 SIM 卡和 microSD 卡放在一个托盘上。在底部，你会发现 USB-C 端口、S 笔插槽和耳机插孔——后者在新闻发布会上引发了其他公司(摩托罗拉，很可能还有苹果)放弃端口以寻求更薄的手机。

这是一个有点不同的线条到一个坚实的设计语言的完美高潮，这是三星在一个外观相似的手机和 iPhone 粉丝的世界中独一无二的。我真希望公司能送来一个珊瑚蓝色的版本。新的颜色真是令人眼花缭乱，尽管黑色的也不错。

只是不要叫它手写笔

当然，Note 系列不仅仅是一个巨大的屏幕。另一个很大的区别是那个小 s 笔。从一开始，公司就明确表示不希望你把这个东西叫做“手写笔”。很公平。三星的内置外设肯定比你的传统塑料版本做得更多。

这一次，该公司继续缩小笔尖，从 1.6 毫米缩小到 0.7 毫米，大约只有一支圆珠笔的大小。更重要的是，它已经被重新调整，所以用户不会心不在焉地把它放在后面，这在以前版本的设备上造成了相当大的混乱。

S Pen 还学会了一些关键的新招。最重要的是无需唤醒手机就能在屏幕上书写。拔出手写笔并开始书写，在黑色背景上留下白色文本笔记，如果您在“设置”中启用了该功能并准备好应对随之而来的电池电量不足，这些笔记可以固定在“始终显示”显示屏上以供快速参考。

对于更注重社交媒体的人来说，这个杀手级应用很可能是创建快速动画 gif 的能力。这是一个非常简单的过程——只需启动一个视频，快进到您想要开始记录的点，然后拔出 S Pen 并单击按钮，它将生成一个六秒钟的 GIF 文件，可以通过所有标准方式共享。这并不比用 Giphy 的在线工具创作东西更容易，但这是一种快速分享你用笔记拍摄的视频的简短版本的方法。

Scroll Capture 可以让你快速保存一个长故事或大图像，而 Glance 可以在角落里将一个应用程序的小预览拖到另一个应用程序的顶部，在相对有限的空间内更容易在两者之间切换。将鼠标悬停在 S Pen 上可以放大部分屏幕，还可以用来将文本(甚至是图像)翻译成不同的语言。

哦，和手机本身一样，S Pen 也是防水防尘的。事实上，如果你迫切需要在淋浴或倾盆大雨时记下一些深刻的想法，在潮湿的屏幕上写东西是可行的。

视听俱乐部

抛开新奇的曲线不谈，屏幕有点停滞不前。但实际上，这并不是什么大问题。同样，5.7 英寸的屏幕提供了 2560 x 1440 的 QHD 分辨率，分辨率为 518PPI。它很大，很明亮，很漂亮。颜色真的很流行，视频看起来很棒。曲面显示器也对这些视角产生了影响。

在这方面，与前几代产品相比，最大的变化是增加了 HDR 视频，它承诺改善兼容服务的明暗平衡，如亚马逊 Prime。

与许多高端智能手机一样，扬声器仍然感觉像是事后想起的东西，被置于 USB-C 端口和触控笔之间底部的一个小格栅中。三星无疑指望人们通过耳机或同步到外部扬声器来收听。因此，扬声器并不适合像 YouTube 视频这样的短脉冲，这对于一个 800 多美元的硬件来说有点糟糕。

这款相机这次没有大幅改进，但有一些关键的增加，包括更快的自动对焦和改进的弱光拍摄，使已经非常稳定的相机产品更加完善。

勇气和荣耀

Note 的内部已经升级到与 Galaxy S7 相匹配，因为一起处理的家庭，某些东西，某些东西。这里指的是三星 Exynos 8890(四个 2.3GHz 内核，四个 1.6GHz 内核)，比 Note 5 的三星 Exynos 7420(四个 2.1GHz 内核，四个 1.5GHz 内核)有一个不错但不算大的提升。这也得益于从 3gb 到 4GB 内存的大幅提升。

你可能不会注意到性能的巨大提升，但升级肯定会有助于三星增加到其生态系统中的无数新任务，从 S Health 到 Oculus team-up、Gear VR 等多媒体产品。

从规格角度来看，最值得注意的升级是增加了 3500 毫安时电池，这是 Note 5 的 3000 的可靠升级。自然，无线充电还是车载的，如果升级到充电板的话。它可以通过无线垫或新的可逆 USB-C 端口快速充电。

在存储方面，该公司已经将选项减少到一个(至少在美国)。这次基线增加了一倍，达到 64GB。对于大多数用户来说，这可能已经足够了，但为了以防万一，这次你可以通过顶部的 microSD 插槽将容量扩展到 256GB，这应该足够让我们大多数人在长途飞机旅行中享受娱乐了。

锁定

诚然，三星没有像黑莓那样将自己的声誉押在安全上，但随着黑客和移动恶意软件继续占据优势，这个问题显然越来越令人担忧。Knox 再次出现，以保持文件安全，并加入了一些额外的隐私措施，最值得注意的是虹膜扫描仪，它将指纹传感，模式，密码和 PIN 加入到解锁手机的方法列表中。

对于大多数用户来说，虹膜扫描是一件新奇的事情，但它很酷。在锁屏上轻轻一点，手机就会开始寻找一双眼睛。一旦对齐，它几乎立即解锁，识别用户眼中的独特图案。事实上，我们在拍摄这个过程的视频时遇到了一些麻烦，因为它进行得太快了。

同样新的是增加了一个 Knox 保护的安全文件夹，即使手机已经解锁，也需要另一次安全登录。对于大多数用户来说，它可能不会是一个日常功能，但对于那些与家人共享设备的人来说，它肯定会派上用场，因为……你可能想放在家里的任何东西。

记笔记

所以，嗯，这里还有一件更重要的事情要讨论。事实上，三星不想在 Note 7 发布会上讨论定价可能是有原因的。价格取决于你在哪里购买它(以及他们选择捆绑什么)，但无论如何，手机都要花你的钱。

美国电话电报公司的起价是 879 美元——或者每月 29.34 美元，如果你喜欢合同的话。在 T-Mobile，它的价格是 849 美元——预付 69 美元，每月 32.5 美元。同样，合同。不管是哪种情况，全力以赴的坏处是你肯定要为你在这里得到的付出代价。如果你想要最大、最亮、最闪亮的手机，它会让你付出代价。

但是，根据预算，它可能是值得的价格。Note 7 结合了 Note 和 Galaxy S 系列的一些最佳功能，同时又增加了更多功能，这一点令人钦佩。就像虹膜扫描这样的一些功能而言，这可能感觉有些过头了，但从整体来看，毫无疑问，新笔记本有很多功能可以提供。
