# YouTube 正在将 Red 扩展到更多的国家

据 YouTube 首席执行官苏珊·沃西基称，YouTube Red 将把其订阅服务扩展到多达 100 个国家。

Wojcicki 在加州亨廷顿海滩举行的 Recode Code 媒体会议上表示，YouTube 将寻求将其红色服务扩展到“更多国家”。

订阅服务[于 2015 年 10 月首次推出](https://web.archive.org/web/20221025222905/https://beta.techcrunch.com/2015/10/21/youtube-red/)，以 9.99 美元的价格免费观看，旨在取代谷歌音乐密钥，为用户提供一种简单的观看或收听 YouTube 视频和播放音乐的方式——作为现有 YouTube 和谷歌账户的一个层级。

该公司表示，将与人们在该服务上收听或观看的内容的版权所有者分享订阅收入。据我们报道，该公司已经与独立创作者、唱片公司、电视网络和电影工作室签署了该计划。

没有登录 YouTube Red 的创作者在无广告的 [YouTube 上隐藏了他们的视频。](https://web.archive.org/web/20221025222905/https://beta.techcrunch.com/2015/10/21/an-offer-creators-cant-refuse/)

随着 YouTube 不再将 Red 用作音乐服务，并开始对原创剧集和电影进行分层独家访问，其中包括 YouTube 明星 PewDiePie 和[当下的有毒影响者](https://web.archive.org/web/20221025222905/https://beta.techcrunch.com/2018/02/09/youtube-suspends-ads-on-logan-pauls-channels-after-recent-pattern-of-behavior-in-videos/)罗根·保罗。

这两个人都引起了争议，凸显了 YouTube 在一些最受欢迎的艺人的内容方面存在的许多问题。

随着 YouTube 继续推出其优质内容，问题仍然是它是否会继续依赖 YouTube 制作的名人，如 PewDiePie 和 Paul，这对该公司的广告客户来说可能是灾难性的，或者试图以 Hulu 或网飞的模式制作更多优质内容。

随着它走向全球，这些内容问题以及随之而来的广告竞争可能只会变得更加棘手。尤其是像联合利华这样的主要广告商开始威胁要撤掉他们的业务。

Wojcicki 说，YouTube 将吸引 10，000 人来审查内容……该公司尚未为他们的招聘设定时间表，但正如持续的保罗争议所示，时间正在流逝。
