# Vector 是成年人的

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/08/08/vector-is-cozmo-for-grownups/>

上周，当 Anki 邀请我们去办公室看看他们的最新作品时，我们不知道会期待什么。自该公司由卡内基梅隆大学的三名毕业生创立以来的八年里，它提供了一些截然不同的智能玩具，从其 Drive smart 汽车到 2016 年的 Cozmo 机器人。

事实是，该公司的最新产品看起来和它的上一个非常相似。事实上，唯一立竿见影的变化是一层深色的油漆，类似于 iMac 升级到 iMac Pro 时的颜色。事实上，这可能是一个非常可靠的类比，不仅仅是颜色的变化，所以让我们坚持下去。

简单来说，Vector 就是成年人的 Cozmo。在许多方面，新的“机器人”是建立在从 Cozmo 吸取的经验教训的基础上，加上更先进的内部结构。Vector 有大约 700 个部件，是其前身的两倍，而它的大脑是一个更先进的骁龙处理器。Vector 面部还具有更高分辨率的显示屏，使其能够表达由该公司雇佣的前皮克斯和梦工厂动画制作人重新设计的更微妙的情感。

那么，所有这些到底意味着什么呢？该公司当然有一些宏伟的目标。Anki 认为，它有能力为用户提供通向下一代家用机器人的大门。很自然，在公司的新闻发布会上，R2-D2 和杰森一家的罗西都被随意地提到了名字。

关于 Vector 的真相更加谦虚，尽管该产品看起来确实比最近离开的 Kuri 更像是一个主流解决方案。首先，它的售价不到一半，为 250 美元。请注意，这仍然比 Cozmo 贵 70 美元。然而，与 Cozmo 不同，Vector 不需要持续的智能手机连接——只需要 Wi-Fi——这就是为什么拥有更先进的车载计算系统很重要的一部分原因。

![](img/96ab32b76a09f0918f6c7c57fb20ec81.png)

除此之外，Vector 还获得了一个 120 度视野的高清摄像头，因此它可以兼作漫游安全设备(该功能将于明年推出)，以及四个麦克风，这使它可以通过“嘿 Vector”发出 Alexa 风格的命令。像 Cozmo 一样，当你进行目光接触时，机器人也会主动交谈，所以你不必每次都召唤它。然而，该公司没有选择 Alexa 或谷歌助手，而是使用第三方知识图来构建自己的系统，因此 Vector 不会因为推出另一个助手而破坏角色。

和 Cozmo 一样，我怀疑该公司通过软件更新的方式为机器人做了很多计划。然而，就目前而言，我还没有看到太多让我相信 Vector 比稍微先进一点的 Cozmo 好得多的东西——这可能会让这个昂贵的机器人玩具对严肃的成年人很难销售。更广泛的目标应该包括一个机器人，它可以超越桌面的有限范围，以便更好地在家里巡逻。

iRobot 即将推出的 Roomba 智能家居计划是家庭机器人技术的一个潜在发展方向，尽管 Anki 认为，通过从一开始就在个性方面投入大量资金，它在这方面占据了上风。也许人们真的需要人与机器人的联系来真正将他们的生活托付给这样一个设备，尽管个性的完全缺乏肯定没有伤害亚马逊的 Echo 或 Google Home。

但是 Anki 在这里得到了好处。这都是非常早期的阶段，当公司设法在全球销售【几十万】时，我们早期对 Cozmo 是一个利基产品的怀疑被搁置。启动 [作为 Kickstarter 活动](https://web.archive.org/web/20230226174528/https://go.anki.com/2OyH5eh) 也应该有助于 Anki 评估对产品的早期兴趣，并从那里扩大规模。

我仍然像以往一样怀疑 Vector 是未来的主流家用机器人——它看起来更像是一块垫脚石，真的。当然，Anki 在不断借鉴过去模式的工作上是透明的。至少，该公司有兴趣、收入和资金来避免最近 Sphero 和 Kuri 的陷阱。
