# 女性使用 Pinterest，但她们不运行它 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/07/24/women-use-pinterest-but-they-dont-run-it/>

根据 Pinterest 周四发布的一份报告，Pinterest 的员工中只有 40%是女性，尽管该公司的大多数用户都是女性，[。](https://web.archive.org/web/20221204164107/http://engineering.pinterest.com/post/92753543099/diversity-and-inclusion-at-pinterest)

[Pinterest 的](https://web.archive.org/web/20221204164107/http://www.crunchbase.com/organization/pinterest)比率似乎优于其他发布类似报告的科技公司——只有脸书、谷歌和推特的大约 30%的员工是女性。但是考虑到 Pinterest 的用户绝大多数是女性——接近 70%——她们在公司员工总数中所占的比例如此之低有点令人震惊。

尽管研究显示女性使用 Pinterest 的可能性是男性的四倍，而且 Pinterest 97%的脸书粉丝是女性，但 Pinterest 只有 19%的领导层是女性。在我看来，由于未能招募和雇佣更多女性，该公司可能未能获得女性的观点，从而改善他们的用户体验。

谈到多样性，Pinterest 的数据似乎比我们在其他大型科技公司看到的糟糕数据还要糟糕。只有 2%的员工是西班牙裔，1%是黑人，相比之下，谷歌分别有 3%和 2%。2013 年，AdWeek [称 Pinterest](https://web.archive.org/web/20221204164107/http://adage.com/article/digital/pew-study-blacks-index-twitter-whites-pinterest/239810/) 为“最白的社交平台”，报道称 18%的白人互联网用户拥有 Pinterest 账户，而只有 8%的黑人和 10%的西班牙裔互联网用户使用该网站。考虑到这些数字，这并不奇怪。

在这篇文章中，Pinterest 发布了我们从这些科技公司那里不断听到的相同的陈词滥调。

Pinterest 的软件工程师兼技术负责人 Tracy Chou 写道:“我们离目标还很远，但我们正在努力。”

我厌倦了这种回应。如果你只看技术职位，21%的员工是女性，这比我们看到的数字要好，在 Twitter 和谷歌，这一数字分别为 10%和 17%。尽管 Pinterest 似乎在招聘方面做得更好(32%的科技实习生是女性)，但很明显，重大改变需要很长时间。

但更令人不安的是，尽管 Pinterest 概述了改善性别差距的措施，但该公司似乎并没有采取行动来纠正其糟糕的多样性数字。“虽然我们在公司性别多元化方面取得了一些进展，但我们在代表不同种族方面做得不够好，我们专注于做得更好，”周写道。

但是为什么不呢？有了这些数字，只要看看 Pinterest 的办公室就应该很清楚，多样性是一个问题，并且已经影响到谁在使用它的产品。

这些科技公司多样性报告不断出现，透明度很重要，因为它引发了关于这些问题的对话。但是，如果不采取任何措施加以纠正，那也没关系。

希望让这些公司的环境更加包容的努力也将继续。
