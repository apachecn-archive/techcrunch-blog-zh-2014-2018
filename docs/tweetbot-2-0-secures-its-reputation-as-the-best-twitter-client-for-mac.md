# Tweetbot 2.0 确保了其作为 Mac TechCrunch 最佳 Twitter 客户端的声誉

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/06/04/tweetbot-2-0-secures-its-reputation-as-the-best-twitter-client-for-mac/>

来自双人开发工作室 Tapbots 的第三方 Twitter 客户端 [Tweetbot](https://web.archive.org/web/20230127141259/http://tapbots.com/tweetbot/mac/) 长期以来一直是对第一方客户端不满意的 iOS 和 Mac 用户的非官方默认选项，这可以追溯到 Twitter 决定取缔非自己制作的客户端之前。Mac 版本于 2012 年推出，但其外观和感觉并没有改变，以符合苹果新的桌面软件美学，这种美学首先在 Mavericks 上发生了变化，然后在 Yosemite 上再次发生了变化。现在，随着 Tweetbot 2.0 的推出，它有了全新的外观和性能改进。

更新对现有用户是免费的，新版本[对新用户的零售价为 12.99 美元](https://web.archive.org/web/20230127141259/https://itunes.apple.com/us/app/tweetbot-for-twitter/id557168941?mt=12&at=11l4Hu)，但两者对你所得到的都显得过于慷慨。我是上一版本 Tweetbot(最近一次统计是 1.6.2)的铁杆用户，在视觉效果与 OS X 的其他部分及其干净、扁平的现代界面脱节很久之后。现在，外观有了很大的改善，与 iOS 版 Tweetbot 的最新更新更加一致，这款应用再次给人全新的感觉，并且仍然提供了当初使其成为该类别明确领导者的相同功能。

该应用程序的新功能是扁平的外观，以及更简单的调色板，包括深色的侧栏和浅色的主栏。账户资料图片现在用圆圈显示，与 Tweetbot 的 iOS 客户端上的外观保持一致；帐户切换器很容易导航，在侧边栏菜单之间切换的动画很微妙，很快，很棒。新的个人资料视图在信息和媒体方面都很丰富，总的来说，这是对推文本身内容的干净、简洁的呈现的敬意，尽管如此，这使得区分一条推文和另一条推文变得非常容易，并可以深入到任何特定的帖子以获取更多信息。

![Screen Shot 2015-06-04 at 8.19.23 AM](img/93894b569e20fe896268d07fdd29c7fb.png)

如前所述，你仍然可以做所有 Tweetbot 提供的其他人没有做的伟大事情，包括粒度静音，如果你想一次看到多个信息流，可以弹出到其他列。像让搜索栏取代顶部菜单栏这样的调整，而不是添加新的一行，是将 chrome 保持在最低限度以获得更好的整体消费体验的一种奇妙方式。

界面左下角的新列管理器图标也使管理多个视图变得更加容易，这意味着它可能是 Tweetdeck 等工具的更好竞争对手，在我看来，Tweetdeck 提供了糟糕的用户体验，但无论好坏，它都是许多社交媒体经理的首选平台。

这个版本仍然缺少的一点是支持今年早些时候 Twitter 推出的新的原生 rt 和评论格式，但它仍然是迄今为止你在 Mac 上可以获得的最好的 Twitter 体验，值得每一分钱。见鬼，我愿意花 13 美元(顺便说一句，比以前少了 2 美元)来增加内嵌的 Instagram 图像显示，而 Tweetbot 2.0 提供的远不止这些。另外，看看这个伟大的新图标:
