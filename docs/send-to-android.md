# 微软将它的短格式电子邮件应用程序 Send 引入安卓系统

今年夏天早些时候，微软[宣布了](https://web.archive.org/web/20221006150145/https://beta.techcrunch.com/2015/07/22/microsoft-introduces-send-a-short-form-email-app-that-works-more-like-instant-messaging/#.gkfheo:ZGB1) Send，这是一个针对 Office 365 用户的项目，旨在将电子邮件和即时消息的精华结合在一起。当时，这项服务只能在 iPhone 上使用，但是从今天开始[，Android 用户也可以](https://web.archive.org/web/20221006150145/https://blogs.office.com/2015/09/16/expanding-the-availability-of-send/)[尝试一下](https://web.archive.org/web/20221006150145/https://play.google.com/store/apps/details?id=com.microsoft.flow.android)。

通过此次更新，微软还将 Send 的可用性扩展到了美国和加拿大以外的地区。这项服务现已在英国、巴西和丹麦推出。

这些都很棒，但是 Send 用户真正期待的是发送动画 gif 的能力。不要害怕，Send now 也可以让你通过悲伤的动画猫形象来表达你最深的绝望情绪。毕竟，“有时一个简单的 GIF 比文字能表达的更多，”微软说。

虽然这不是一家公司第一次试图将即时通讯和电子邮件结合起来(甚至 Gmail 在早期也有这种感觉)，但 Send 似乎是一个有趣的实验。这些天来，我的绝大多数电子邮件回复似乎都只有两句话，毕竟，Gmail 和其他电子邮件仍然像处理其他电子邮件一样处理这些短信。

遗憾的是，Send 目前只支持拥有企业和学校账户的 Office 365 用户。不过，你的电子邮件收件人不一定是 Office 365 用户。他们只会将这些消息视为普通电子邮件。

你现在可以从谷歌 Play 商店[这里](https://web.archive.org/web/20221006150145/https://play.google.com/store/apps/details?id=com.microsoft.flow.android)下载安卓应用。
