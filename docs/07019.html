多亏了微软，小团队现在可以免费获得 Xamarin 的 IDE 和核心工具

微软上个月收购了跨平台 iOS、Android 和 OS X 开发服务 Xamarin ，这项新收购已经成为本周微软年度 Build 开发者大会的关注焦点。该公司在今天的主题演讲中宣布，核心的 Xamarin 工具现已作为免费的 Visual Studio 社区版的一部分提供，而OS X的 Xamarin Studio 现已作为独立的社区版提供(MSDN 的订户也将获得它)。

Xamarin 更高级的企业服务正在整合到 Visual Studio Enterprise 和 Professional 订阅服务中，无需额外费用。

Xamarin 之前在订阅的基础上销售其工具，从面向独立开发者的 25 美元/月的低价选项到 999 美元/年的企业账户和 1899 美元的企业版。

Visual Studio Community Edition 和 xa marin Studio Community Edition 的使用仅限于学术用户、开源开发人员和最多拥有五名开发人员的小型团队。社区版将包括完整的 Xamarin Studio IDE，但一些高级功能(如嵌入式程序集、Xamarin Inspector 预览和 Xamarin Profiler 预览)只能通过 Visual Studio Enterprise 订阅获得。

Xamarin 测试云允许开发人员在真实设备上远程测试他们的应用程序，它将仍然是一个独立的产品。这项服务的价格也没有变化，尽管 Visual Studio 企业客户将获得 25%的折扣。

作为此次声明的一部分，微软还表示，它将在未来几个月内在麻省理工学院开源许可下提供 Xamarin SDK。这些工具，包括运行时、库和命令行工具，将移到。微软去年宣布的. NET 基金会。微软也将很快在麻省理工学院的许可下重新发布 Mono。

从很多方面来说，这是微软合乎逻辑的一步。Visual Studio Community edition 已经取得了相当大的成功，将相同的公式应用于 Xamarin 是有意义的。这一举动也是一个明确的信号，表明微软非常重视跨平台开发，而不仅仅是谈论一个大游戏。

我预计 Xamarin 和 Visual Studio 之间的界限将会随着时间的推移而消失，微软将会把 Xamarin IDE 的大部分功能整合到 Visual Studio 中。甚至在收购之前，微软和 Xamarin 经常密切合作，毕竟已经有了 Visual Studio 的 Xamarin 插件。
