多亏了微软，小团队现在可以免费获得Xamarin的IDE和核心工具

微软上个月收购了跨平台iOS、Android和OS X开发服务 Xamarin ，这项新收购已经成为本周微软年度Build开发者大会的关注焦点。该公司在今天的主题演讲中宣布，核心的Xamarin工具现已作为免费的Visual Studio社区版的一部分提供，而OS X的Xamarin Studio现已作为独立的社区版提供(MSDN的订户也将获得它)。

Xamarin更高级的企业服务正在整合到Visual Studio Enterprise和Professional订阅服务中，无需额外费用。

Xamarin之前在订阅的基础上销售其工具，从面向独立开发者的25美元/月的低价选项到999美元/年的企业账户和1899美元的企业版。

Visual Studio Community Edition和xa marin Studio Community Edition的使用仅限于学术用户、开源开发人员和最多拥有五名开发人员的小型团队。社区版将包括完整的Xamarin Studio IDE，但一些高级功能(如嵌入式程序集、Xamarin Inspector预览和Xamarin Profiler预览)只能通过Visual Studio Enterprise订阅获得。

Xamarin测试云允许开发人员在真实设备上远程测试他们的应用程序，它将仍然是一个独立的产品。这项服务的价格也没有变化，尽管Visual Studio企业客户将获得25%的折扣。

作为此次声明的一部分，微软还表示，它将在未来几个月内在麻省理工学院开源许可下提供Xamarin SDK。这些工具，包括运行时、库和命令行工具，将移到。微软去年宣布的. NET基金会。微软也将很快在麻省理工学院的许可下重新发布Mono。

从很多方面来说，这是微软合乎逻辑的一步。Visual Studio Community edition已经取得了相当大的成功，将相同的公式应用于Xamarin是有意义的。这一举动也是一个明确的信号，表明微软非常重视跨平台开发，而不仅仅是谈论一个大游戏。

我预计Xamarin和Visual Studio之间的界限将会随着时间的推移而消失，微软将会把Xamarin IDE的大部分功能整合到Visual Studio中。甚至在收购之前，微软和Xamarin经常密切合作，毕竟已经有了Visual Studio 的Xamarin插件。
