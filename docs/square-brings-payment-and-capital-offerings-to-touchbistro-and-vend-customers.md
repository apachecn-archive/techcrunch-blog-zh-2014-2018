# Square 为 TouchBistro 和 Vend 客户带来支付和资本产品

随着与 Vend 和 TouchBistro 的两项新合作，Square 正在将曾经被视为敌人的销售点公司转变为盟友。Square 现在将向 [Vend](https://web.archive.org/web/20230129103747/https://www.vendhq.com/) 和 [TouchBistro](https://web.archive.org/web/20230129103747/http://www.touchbistro.com/) 的餐厅、零售和其他企业主客户提供 Square 平台的完全访问权限，包括其硬件、支付基础设施和金融服务产品(如小型企业贷款)。

曾几何时，当 Square 让事情变得更加封闭时，像多伦多的 TouchBistro 和新西兰的 Vend 这样的公司更直接地与它自己的产品竞争。Square 的 POS 系统和 Register 应用程序是其两个新合作伙伴提供的解决方案的替代方案。然而，自从 Square 开始提供对其个人服务的访问，包括其会计和库存工具，以及在 3 月份首次推出其公共 API 后，零售商使用更多混合解决方案的机会就出现了。

允许 Square 与 Vend 和 TouchBistro 等公司合作的新方法更具商业意义。这意味着 TouchBistro 和 Vend 的客户可以选择一系列新的硬件选项，包括使用 Square 读卡器(在美国，可以处理非接触式和 Apple Pay 交易)，对于 Square 来说，它为其最重要的新产品类别之一提供了另一个新的载体(Square 已经以类似的能力与餐厅管理软件提供商 Upserve 合作):面向小企业的金融服务。

Square Capital 是 Square 业务中快速增长的一部分，在贷款发放方面同比增长 123%。Square 未来增长的另一个关键数据是其总支付量，同比增长 42%，但如果其支付系统可以被其他 POS 平台上的零售商使用，这一数字还会增长很多，就像现在 Vend 和 TouchBistro 客户可以使用的那样。

使用 TouchBistro 和 Vend 的企业的新集成今天从有限的群体开始，但计划在未来几周内将其推广到所有现有和未来与这些 POS 公司合作的餐馆和零售商。
