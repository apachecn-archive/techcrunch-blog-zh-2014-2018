# Twitter 的新按钮让你可以接受来自你的网站 TechCrunch 的私人信息

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/08/24/twitters-new-button-lets-you-accept-private-messages-from-your-website/>

Twitter 今天继续推进其对信息功能的日益重视，推出了一个新的网站按钮，允许访问者直接向个人或公司发送私人信息。该功能现在是几个可用的网站按钮之一，此前该公司最近测试了品牌“客户支持档案”的变化，[鼓励用户在该公司的 Twitter 账户上直接发送消息](https://web.archive.org/web/20221219020202/https://techcrunch.com/2016/08/05/twitters-latest-test-encourages-users-to-direct-message-brands-not-tweet-at-them/)，而不是发推文。

在这种情况下，一个新的“消息”按钮出现在手机上的个人资料中——占据了过去“推文到”和“消息”并排存在的全部空间。早期的测试者包括苹果、优步、Beats、动视等知名公司。

与此同时，今天的更新是为了让从网上发送商业信息变得更容易。

该公司已经提供了关注、分享、提及和标签按钮，但没有 DM 按钮。因此，在某种程度上，这只是 Twitter 完善其产品提供更加全面。

然而，该按钮的出现正值脸书将其聊天应用 Messenger 作为消费者与企业互动的默认方式。

社交网络[最近重新设计了它的商业页面](https://web.archive.org/web/20221219020202/https://techcrunch.com/2016/08/05/facebook-rolls-out-redesigned-business-pages/)，在页面顶部放置了一个“消息”按钮，假设有问题的企业使用聊天。今年早些时候，[它发布了一套工具，使企业更容易通过 Messenger 与客户联系，](https://web.archive.org/web/20221219020202/https://techcrunch.com/2016/04/07/facebook-pushes-businesses-to-messenger-with-new-tools-for-pages-including-snapchat-like-scannable-codes/)包括 Messenger 链接和用于发起聊天的可扫描 Messenger 代码、基于页面名称的企业用户名和 Messenger 问候语等。

与此同时，Twitter 更为人所知的是，当出现问题时，消费者可以去那里投诉——经常发布愤怒的推文，并附上该品牌的@username。通过提供更多的工具，让客户可以将这种情绪带到私人聊天中，企业可以继续将 Twitter 作为其营销、消费者拓展和支持战略的一部分，而不是将所有的通信转移到脸书。

今天下午，在推特[宣布](https://web.archive.org/web/20221219020202/https://twitter.com/TwitterDev/status/768470500146286592)新按钮后不久，一些早期测试者报告说他们在尝试使用时收到了错误信息。显然，[新按钮](https://web.archive.org/web/20221219020202/https://publish.twitter.com/)还没有完全准备好，但应该在不久的将来就可以使用了。

**更新**:功能正常，上面的客户没有正确填写表格。
