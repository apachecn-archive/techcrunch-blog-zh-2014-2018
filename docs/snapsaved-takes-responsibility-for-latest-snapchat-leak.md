# Snapsaved 为最新的 Snapchat 泄露事件负责

上周，有消息称，一个第三方应用程序(用于保存人们的照片)被黑客攻击后，大约 20 万张 Snapchat 照片泄露。这被称为突然打开。

Snapchat 很快对此事做出了回应，称照片是从第三方应用程序中窃取的，但 Snapchat 的服务器从未遭到破坏。事实证明，Snapsaved.com 站出来承担泄漏的责任。

在该公司脸书页面上发布的一份更新中，Snapsaved 声称对此负全责，并进一步澄清了被黑客攻击的内容和提供的信息。

声明称，黑客从未接触过用户名等任何个人信息，尽管“据(Snapsaved)所知”，他们确实接触过 500MB 的图像

该公司还对[指控](https://web.archive.org/web/20221024141727/http://pastebin.com/cJcTbNz8)其可能故意向黑客提供图像数据库的访问权限做出回应，称其在发现漏洞后立即关闭了整个 Snapsaved 网站和数据库。

声明称，大多数受影响的用户是瑞典人、挪威人和美国人。

你可以阅读下面的[全文解释](https://web.archive.org/web/20221024141727/https://www.facebook.com/Snapsaved/posts/666573263457078):

> 我想详细说明一下最近关于 Snapsaved.com 的事件,
> Snapsaved.com 是一个用来保存 SnapChat 的网站，确切地说是应用程序 snapsave。
> 作为对最近媒体事件和 http://pastebin.com/cJcTbNz8,声明的回应，我想告知公众，snapsaved.com 遭到了黑客攻击，发帖者所指的词典索引从未公开过。我们的 Apache 服务器配置错误。
> SnapChat 没有遭到黑客攻击，这些图片也不是来自他们的数据库。
> Snapsaved 一直在努力打击儿童色情，我们甚至向瑞典和挪威当局举报了我们的一些
> 用户。我们一发现系统漏洞，就立即删除了整个网站和与之相关的数据库
> 。据我们所知，这次入侵已经影响了数据库中 500MB 的图片和 0 个人信息。最近关于 snappening 的谣言是一个骗局。黑客没有足够的信息来实现他创建一个可搜索数据库的声明。
> 我们的用户必须同意他们通过 SnapSaved.com 接收的所有内容，正如我们提到的，我们试图尽可能经常地清理数据库中不合适的图像。
> 我们的大多数用户是瑞典人、挪威人和美国人。我代表 snapsaved.com 真诚地道歉，我们从没想过会发生这种事。我们不希望对 SnapChat 或其用户造成任何伤害，我们只希望提供一种独特的服务。

我们已经联系了 Snapsaved 以获取更多信息，如果收到回复，我们将及时更新。
