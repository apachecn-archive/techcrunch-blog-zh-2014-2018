# 细分市场计划帮助公司遵守 GDPR 数据隐私要求

[Segment](https://web.archive.org/web/20221005120407/https://segment.com/) 是一家初创公司，帮助公司从各种孤立的来源收集客户数据，并将这些数据放在一个可用的记录中。今天，它宣布增加了一个新工具，为明年五月即将到来的[欧盟 GDPR 隐私指南](https://web.archive.org/web/20221005120407/http://www.eugdpr.org/)做准备。

新工具使停止收集数据或从系统中完全删除一个人的信息成为一件简单的事情，这一过程被称为“被遗忘的权利”。

Segment 通过提供一组 API 而声名鹊起，这些 API 使其能够从各种来源(如您的 CRM 工具、客户服务应用程序和网站)收集有关客户的数据，并将这些数据整合到客户的单一视图中，这是客户信息业务中每个公司的目标。

在你被数据收集的角度吓到之前，要意识到这家公司收集这些数据并不是为了出售，而是为了让你与之做生意的公司更清楚地了解你与他们在整个公司的互动。这意味着，如果你在网上订购了东西，然后走进商店，店员可以查看你的客户记录，看到你所有的互动，这对于大多数公司来说并不容易做到。

“我们对客户数据的看法是，消费者想要个性化的体验，所以公司需要了解他们的客户，”Segment 联合创始人兼首席执行官 Peter Reinhardt 告诉 TechCrunch。但他明白，顾客不希望他们的互动被卖给第三方，他把这比作杂货商和银行家闲聊你买了什么。

事实上，将这些数据移动到需要的地方非常困难。“如今，以客户为中心的业务，尤其是在数字渠道中，实际上是数据基础设施和细分市场的问题，这有助于公司解决这个问题，”Reinhardt 解释道。

另一方面，它现在也可以帮助你停止收集信息。它增加了一个新的 API，可以让你停止收集个人数据。根据 GDPR 规则，公司有义务遵守这一要求。更重要的是，如果个人希望你从你的记录中删除他们所有的客户数据，你也可以这样做。

如果您想一想 Segment 是做什么的，在一个地方收集关于客户的所有数据，这大大简化了对删除请求采取行动的能力，因为 Segment 不是必须在多个系统中查找客户信息并检查每个系统，而是以自动化的方式为您完成所有这些工作。

Segment 成立于 2011 年，2012 年发布首款产品。它已经筹集了超过 1 . 08 亿美元，包括 7 月份由 Y Combinator 牵头的 6400 万美元的 C 轮融资。它目前有 180 名员工和[15000 名客户](https://web.archive.org/web/20221005120407/https://segment.com/customers)，包括今晚酒店、Atlassian、Crate & Barrel、Draft Kings 和 New Relic。
