# Sphero 收购音乐教育初创公司

# Sphero 收购了一家音乐教育初创公司

很难说 Sphero 的[转向教育](https://web.archive.org/web/20221209205156/https://techcrunch.com/2018/01/21/sphero-lays-off-dozens-as-it-shifts-focus-to-education/)在这些早期阶段进展如何，但它最近获得了资金注入，并已经开始收购新的创业公司。BB-8 制造商今天早上宣布，它已经获得了 spec drus——同在博尔德的初创公司[是 Kickstarter 的一个成功故事](https://web.archive.org/web/20221209205156/https://www.kickstarter.com/projects/364756202/specdrums-music-at-your-fingertips)，让用户通过应用程序连接戒指来创作音乐。

乍一看，这是一个奇怪的组合，但 Sphero 显然将该公司的可穿戴技术视为其新发现的 STEAM 教育重点的有力补充。

“我们坚信玩耍是一位强大的老师。Sphero 首席执行官 Paul Berberian 在一份与新闻相关的新闻稿中表示:“随着 Specdrums 的加入，我们正在加强我们产品路线图中的‘A’。“凭借 Sphero 的基础设施和 Specdrums 创始人已经完成的基础工作，我们相信有一个巨大的机会来继续激发课堂内外的好奇心。”

Specdrums 将如何融入更大的公司仍有待观察，但目前，Sphero 承诺将在今年年底或明年年初重新推出该公司的首款音乐产品。在该公司 2017 年众筹活动结束后，首次发行的股票已经售罄。

对 Sphero 来说，今年开局不利，在该公司的迪士尼品牌机器人玩具供过于求，未能保持其早先在《星球大战》中的成功后，解雇了数十名员工。当时，该公司承诺加倍投资教育，并希望通过最近的一轮 1200 万美元的融资来实现这一目标。

然而，此次收购的条款尚未披露。
