# 通过 Dialpad-Amazon Echo 集成，Alexa 可以为你打电话

基于云的商业通信平台 Dialpad 本周宣布了新的亚马逊 Echo 集成，使你能够通过询问 Alexa 来执行许多商业电话任务。

例如，你可以问，“Alexa，我在 Dialpad 上有消息吗？”或者" Alexa 用拨号盘给 Liz Green 打电话"

Dialpad 将这种集成视为使电话功能随处可用的大趋势的一部分。Dialpad 的联合创始人兼工程副总裁布莱恩·彼得森(Brian Peterson)告诉 TechCrunch，它首先在云端拥有一个电话号码，一个云电话平台和一套 API，允许该公司将包括 Echo 在内的任何设备转化为“一部电话”。

这种灵活性使他们能够将电脑、平板电脑或手机变成公司电话，让员工无论身在何处都可以访问公司电话系统，即使他们不在办公室。Peterson 说，这种方法实际上是通过允许移动员工从任何地方访问公司电话系统来扩展他们的工作范围。

“事实上，我们谈论的是随处工作的员工，更多的人远程工作。他们不是坐在朝九晚五的办公桌前。无论他们在哪里，他们都可以控制(他们的业务)电话系统，”他说。

根据 Peterson 的说法，Echo 和类似的设备是合乎逻辑的下一步。能够连接到像 Alexa 这样的系统，真正消除了使用电话系统的摩擦。你不必拿起听筒，拿出手机，甚至打开笔记本电脑。你只需说出命令，它就会通过回声连接你。

他说，这种无需物理设备就能访问电话系统的能力是一个重大飞跃，显示了通过基于云的系统提供电话服务的力量。

你可以让 Alexa 在你的联系人列表中拨打电话，或将当前来电者添加到你的联系人列表中，转接或保留电话，录制电话并检查语音邮件信息。您还可以通过您办公室的拨号盘系统要求有关电话活动的报告。

该公司是由两名前谷歌工程师创办的，他们帮助开发了谷歌语音。他们认识到，如果你能把电话系统放在云端，你就能像处理电子邮件一样处理电话功能，并在任何地方访问它。

“我可以从任何地方访问我的电子邮件。Dialpad 的营销副总裁摩根·诺曼(Morgan Norman)解释说:“问题是我无法从任何地方使用(办公室)电话。随着物联网扩展到一系列联网设备，他们可以将手机功能扩展到任何地方，包括汽车、联网电视或任何其他联网设备。

该公司计划今年晚些时候在 Google Home 上提供这一功能。
