# Y Combinator 支持租户特纳为房东开发管理出租房产的软件

随着美国住房拥有率回到 1993 年以来的最低点，在可预见的未来，租金看起来将在 T2 的住房存量中占据更大的份额。

在全国 4300 万套出租房屋中，这是一个分散的市场，有数百万个人业主和房东。

[Tenant Turner](https://web.archive.org/web/20221206224723/http://tenantturner.com/) 是一家 Y Combinator 公司，成立于 2008 年金融和房地产危机之后。联合创始人詹姆斯·巴雷特(James Barrett)拥有一栋独栋住宅，但他需要为不断壮大的家庭搬到一个更大的地方。不幸的是，弗吉尼亚州里士满周围的房地产市场疲软，他无法卖掉它。所以他决定租出去。

“我是一个偶然的房东。我在 2005 年市场高峰期买了一栋联排别墅，然后我需要一个更大的房子，我们真的卖不出去。在处理转租、寻找租户以及遵循公平住房的法律程序方面，这成了一项真正的投资，”巴雷特说。“市场上真的什么都没有。”

因此，他、克里斯·斯图尔特和布兰登·安德森成立了租户特纳，帮助房东寻找和选择合格的租户，同时遵守公平住房条例等法律。他们已经从 Y Combinator、Lighthouse Labs、Triad LC、Altron Capital、Duke Dodson、Doug White、Daniil Kleyman 和 Jim Cheng 那里筹集了 50 万美元。

Tenant Turner 会自动列出排名前 20 的房源。他们还通过核实收入、提取信用报告和背景调查以及检查租赁历史来对租户进行资格预审。

他们还会问租户一些基本的问题，以筛选出不合适的候选人，比如那些有宠物的人。然后，他们通过短信和电子邮件提醒来协调看房事宜。

它们的价格从每个租赁列表 119 美元开始，根据购买的列表积分数量逐渐降低。

到目前为止，该公司在网站上有大约 200 个房源，从独栋住宅到公寓和城市住宅都有。
