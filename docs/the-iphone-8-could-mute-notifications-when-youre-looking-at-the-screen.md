# 当你看着屏幕时，iPhone 8 可以静音通知

上周，苹果公司意外发布了 HomePod 固件的早期版本。[古伊列梅·兰博](https://web.archive.org/web/20221208195827/https://twitter.com/_inside)一直在挖掘图书馆文件，寻找暗示未来产品和功能的字符串和参考文献。特别是，他刚刚注意到，下一代 iPhone 将能够检测到你何时在看屏幕，这样它就可以让你的通知静音。

我查看了 HomePod 的固件，发现对这个“supportsAttenuatingTonesForAttentionDetected”类有相同的引用。

基于之前的泄露，苹果一直在研究一些神秘的[人脸检测技术](https://web.archive.org/web/20221208195827/https://beta.techcrunch.com/2017/08/07/iphone-8-could-scan-your-face-even-if-your-phone-is-on-a-table/)。这项代号为“珍珠 ID”的技术应该完全取代触控 ID 传感器。

似乎下一代旗舰 iPhone 将配备两个前置摄像头和一个红外传感器。这样，即使你在黑暗中，即使你没有直视手机，你也可以用你的脸解锁手机。

今天的消息有点令人惊讶，因为这意味着你的 iPhone 将随时跟踪你的脸。如果你在看手机，苹果就是这样让通知和震动静音的，因为横幅就足够了。

但苹果可以将面部追踪用于许多不同的事情。例如，你可以通过查看锁定屏幕上的通知来解锁你的 iPhone。当你准备解锁时，你的 iPhone 已经知道你在看你的手机，所以应该是即时的。

同样，密码管理器、银行应用程序、文件管理应用程序和其他敏感应用程序一直在使用触控 ID 作为额外的安全层。如果你的手机一直知道你在看它，应用程序可以完全跳过这个屏幕。或者，如果你不看手机，iOS 会自动锁定。

有传言称，苹果公司希望将指纹传感器嵌入显示器本身。但报道称，苹果无法用这项技术生产数千万台设备。这就是为什么苹果正在改变现状，专注于面部识别。

苹果[应该会在九月份发布](https://web.archive.org/web/20221208195827/https://beta.techcrunch.com/2017/02/19/the-next-iphone-could-have-a-bigger-screen-and-more-battery/)三款不同的 iPhone。该公司应该宣布两款更强大的手机，看起来或多或少像 iPhone 7 和 iPhone 7 Plus。

但最有趣的设备是一款全新的超级高端手机——它可以被称为 iPhone 8、iPhone Pro 或其他不同的东西。据称，它将有一个更高的屏幕，将完全占据设备的正面，除了顶部的扬声器、摄像头和传感器。看看苹果如何为这款新设备调整 iOS 将会很有趣。
