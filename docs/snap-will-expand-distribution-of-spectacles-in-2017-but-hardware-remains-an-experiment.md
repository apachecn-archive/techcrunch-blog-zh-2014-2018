# Snap 计划在 2017 年扩大眼镜销售

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/02/02/snap-will-expand-distribution-of-spectacles-in-2017-but-hardware-remains-an-experiment/>

Snap，Inc】刚刚发布了其 IPO ，有很多东西可以带走，但从硬件的角度来看，没有太多的礼物——这实际上本身就说明了问题。S-1 文件中提到了 Spectacles，但这并不意味着 Snap 认为自己是一家非常依赖硬件产品或渠道的相机公司。但这仍有可能改变，因为它表示计划在 2017 年扩大眼镜的分销。

在一份分解业务各个方面的介绍性说明中，Snap 确实提到了其视频捕捉太阳镜 Spectacles，称其为“重新发明相机的最新努力，但在紧接着的一个关于其总体战略和机遇的章节中，强调了软件产品的广泛可用性是一项优势。

它还谈到 Spectacles 作为一个例子，介绍了“新的和未经验证的产品”，涉及“我们以前很少或根本没有开发或操作经验”的技术，并表示 Spectacles 迄今为止只通过弹出式菜单销售，“没有为公司创造大量收入”。

Spectacles 听起来仍然非常像 Snap 的试验阶段，因为它说该公司“已经并预计将继续进行重大投资来开发和推出 Spectacles，我们还不能确定用户是否会在未来购买或使用 Spectacles。”

但有迹象表明，实验进展顺利，并可能在未来一年大幅扩张。听起来，我们可能会看到更多传统的分销模式取代弹出模式，从字里行间解读。

“我们预计与眼镜相关的生产和运营成本将在不久的将来超过相关收入，”该文件称。“2017 年，随着我们对产品需求的评估，我们预计将继续在库存、营销、分销和产品创新方面进行大量投资。此外，我们计划大幅拓宽眼镜的分销渠道，这将增加我们的成本和整体财务风险。”

同样值得注意的是，根据 Snap 的说法，眼镜是由 FDA 监管的，它将眼镜列为一个风险因素:“FDA 和其他州及外国监管机构监管眼镜。”

我们可能会开发未来的产品，这些产品被 FDA 监管为医疗设备，”它补充说，为其他充当眼镜或具有医疗意义的硬件设备敞开了大门。

Snap 还列出了“圆形视频”，这是一种圆形的录制媒体格式，可以让 Spectacles 拍摄的视频以垂直或横向模式观看，这是其作为相机公司发展的关键一步:

“圆形视频代表了我们对移动视频的看法的演变，以及超越我们第一个垂直视频格式的进展。”
