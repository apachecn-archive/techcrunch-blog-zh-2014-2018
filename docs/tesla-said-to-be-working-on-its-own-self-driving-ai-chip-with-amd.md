# 特斯拉表示正在与 AMD 合作开发自己的自动驾驶人工智能芯片

据报道，特斯拉正在与 AMD 合作开发自己的人工智能处理器，旨在用于其自动驾驶系统。特斯拉已经与英伟达建立了关系，英伟达的 GPU 为其自动驾驶系统提供动力，但美国消费者新闻与商业频道报道的这款新的内部芯片[可能会减少其对第三方人工智能处理硬件的依赖。](https://web.archive.org/web/20230130100805/https://www.cnbc.com/2017/09/20/tesla-building-an-ai-chip-for-its-cars-with-amd-globalfoundries.html)

AMD 将帮助特斯拉开发芯片。据报道，这家电动汽车制造商显然已经收到了第一批用于测试的处理器原型样品。制造完全为自动驾驶设计的特定芯片将在许多方面帮助特斯拉，包括对其供应链的一个关键方面获得更多控制，以及提高整体系统效率和性能。

特斯拉声称，其当前一代车辆拥有所有的硬件和计算能力，未来可以通过软件更新实现完全自动驾驶。但该公司也一直在迭代其硬件，避开传统汽车制造商的传统车型年升级。

特斯拉的自动驾驶业务由吉姆·凯勒(Jim Keller)负责，他之前在 AMD 工作了很长时间，然后在苹果公司帮助设计了苹果 A4 和 A5 内部处理器。凯勒在该公司的关键角色可能会影响该汽车制造商走向拥有自己的处理器设计，以更紧密地协调其自动驾驶软件和计算硬件组件。
