SirionLabs 在美国建立据点，扩展其 NLP 合同管理软件

向企业提供供应商管理软件的初创公司 SirionLabs 正在增加其美国总部。该公司最初成立于印度，已从红杉资本印度公司、Canopy Ventures 和 Qualgro VC 筹集了 1695 万美元，用于从合同中提取数据，以确保透明度和问责制。

美国业务的建立代表着公司发展计划的战略转变。去年， SirionLabs 告诉 TechCrunch ，它正在寻求从硅谷招聘员工，并将他们重新安置到印度。虽然这家初创公司在英国、德国、丹麦和新加坡设有办公室，但它在建立一个永久性的美国团队方面进展缓慢。新总部将位于加利福尼亚州的都柏林。

该公司的平台 Sirion 目前被 BP 和 Vestas 等公司用来管理服务提供商，并增强传统上管理供应商关系的人员。这家初创公司预计在未来一年内使用自然语言处理来分析超过 80 亿美元的总合同价值。

SirionLabs 的首席执行官 Ajay Agrawal 告诉 TechCrunch，给定合同价值的 2-3%用于管理供应商关系。在斯坦福大学做了一段时间的 EIR 后，Agrawal 决定投入时间和资源来部分自动化这个过程。

“鉴于发票做法的多样性，我们必须想出一种形式来检测差异，”Agrawal 说。“合同告诉你应该发生什么，业绩数据告诉你应该发生什么。”

使用 NLP 分析和尽职调查合同的想法并不新鲜。SirionLabs 成立于 2012 年，像 Seal Software 这样的竞争对手可以追溯到更早的 2010 年。需要标记的潜在差异的长尾在一个没有风险胃的市场中已经成为采纳的一个障碍。这就是为什么 Sirion 要求双方亲自同意共同的结果，让人类而不是机器承担最终责任。
