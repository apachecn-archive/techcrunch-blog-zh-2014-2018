这个 DIY 的奇怪的东西灯套件会让你忘记倒钩

你和你的朋友们都爱上了同一个头部流血的陌生女孩吗？她喜欢华夫饼吗？你害怕怪物吗？那我们为你准备了万圣节装饰品。这个 DIY LED 灯套件允许你(或维诺娜·赖德)使用一个 Arduino 板和一些巧妙的布线与被颠倒的孩子交流。

这些计划可作为指令使用，主要由一串可寻址 LED 灯和一个 Arduino 板组成。灯光可以单独控制，Arduino 板只需在正确的时间发送正确的信号，这样你就可以告诉你的妈妈“怪物来了”和“发送多力多滋”

遗憾的是，代码有点复杂，但这是构建一种随机单词生成器的巨大努力。然而，因为整个事情非常简单，所以非常值得深入研究。我见过的大多数版本都包含许多 led，这些可寻址 led 的大量焊接使事情变得简单多了。最好的事？你可以将你的 Arduino 连接到互联网，让人们给你发送信息，最终让你发疯，因为你等待你的儿子解释他是如何被一个怪物吸进花园小屋的。
