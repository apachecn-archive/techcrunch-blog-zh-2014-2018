# 英国政府对共享经济的“独立”审查将由共享经济初创公司首席执行官领导

如今，英国政府全力支持所谓的共享经济，Airbnb 和 Zipcar 等公司就是其缩影，其公开目标是让英国成为“共享经济的全球中心”，并确保它能够与硅谷竞争。为了做到这一点，它已经下令进行一次“独立的”审查，以调查政策和监管问题，以及如何在英国创造一个共享经济可以发挥其潜力的环境。

然而，奇怪的是，这项审查是由 Debbie Wosskow 领导的，她是度假换房初创公司“T2 爱换房”(Love Home Swap)的首席执行官，该公司在合作消费/共享经济中发挥着重要作用。委婉地说，这并没有给我留下特别独立的印象。虽然，由于审查的职权范围已经预先倾向于将共享经济视为一个*的好东西*，也许任何关于由谁来领导审查的疑虑都有点悬而未决。

为了捍卫英国政府的任命选择，沃斯科负责“欧洲合作消费”(Collaborative Consumption Europe)，这是一个分享经济的网络和季度活动，因此她对这个领域和相关问题的了解远远超出了她作为一家分享经济初创公司首席执行官的角色。

我还被告知，她是在无偿的基础上领导这项审查的，所以英国纳税人的成本应该是最小的。这或许也无妨，因为可以说，这里正在进行的工作通常会由游说团体承担——比如最近推出的[英国创业宣言](https://web.archive.org/web/20221209122015/https://beta.techcrunch.com/2014/09/03/uk-startups-call-on-politicians-to-improve-visas-broadband-skills-regulation/)。但是我跑题了…

审查本身一旦结束，将根据以下职权范围提出建议:

*   挑战并定义共享经济的概念
*   探索共享经济给英国带来的潜在好处，以及它可能给传统行业带来的任何风险
*   了解企业在共享经济中面临的主要问题，例如保险政策对 Airbnb 等新公司的作用
*   理解共享经济公司面临的监管负担
*   了解数字信任的障碍
*   了解共享经济如何在英国发挥潜力。

然而，正如 GigaOm 的大卫·迈耶指出的那样，工人权利等棘手问题因其被忽略而引人注目——尽管，不管怎样，如果你是一家在共享经济领域运营的初创公司，你可能会欢迎今天英国政府的支持。
