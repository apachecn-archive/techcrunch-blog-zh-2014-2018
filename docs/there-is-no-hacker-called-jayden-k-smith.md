# 没有叫“袁晓超·k·史密斯”的黑客

作为一项公共服务，我张贴这个帖子，以便当你的父母给你发消息警告一个名叫[“袁晓超·k·史密斯”](https://web.archive.org/web/20221207150221/https://www.kaspersky.com/blog/jayden-smith-hoax/17462/)的黑客“将系统连接到你的脸书账户”时，你可以很容易地用一篇有说服力的新闻来反驳。

袁晓超·k·史密斯的信息是一个骗局。没有黑客“有系统”连接到你的脸书帐户，如果你按照与骗局相关的指示，并向你所有的联系人发送消息，那么你就上当了。不要这样做。现在停止。

我再说一遍，没有叫袁晓超·k·史密斯的黑客。他不会黑你的脸书。唯一有点危险的袁晓超·史密斯是贾登·史密斯在[不切实际的小丑萨尔·瓦肯诺的](https://web.archive.org/web/20221207150221/https://www.pinterest.com/pin/384987468120333476/)臀部上的纹身，尽管如此，它看起来还是很令人愉快。
