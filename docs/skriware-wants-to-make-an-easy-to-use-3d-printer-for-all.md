# Skriware 希望为所有人制造一台易于使用的 3D 打印机

首先，我鼓励你看下面的视频。通过将一个对英语的模糊误解与一个听起来像为动作片的预告片配音的人混在一起，你创造了我在 Kickstarter 视频作品中见过的最好的例子之一。但是我们不要吹毛求疵。Skriware 是一款波兰 3D 打印机，由一组想要改善我们 3D 打印方式的设计师和工程师创建。虽然你会说这个世界现在需要的是另一台 3D 打印机，就像我需要在头上打个洞一样，但 Skriware 希望让软件和硬件都更有用。

我们的目标是提供一种一键式系统，用于将印刷品从在线市场传输到打印机，然后打印机在没有太多实际人工交互的情况下进行打印。其他家用打印机在一定程度上已经做到了这一点，但一款售价 799 美元的打印机却没有做到这一点。

该团队能够通过降低打印头的复杂性和使用比其他系统需要更少热量的打印材料来降低价格。他们的模式是主要关注易用性，这是 3D 打印市场的一个可喜变化。

“我们设计 Skriware 打印机的目标是让它尽可能简单直观。这就是为什么我们推出了一款任何人都可以使用的设备——从沉迷于最新游戏玩偶的小学生到想要用他们最喜欢的超级英雄形状的饼干给孙子孙女们带来惊喜的奶奶！”他们写作。

这款打印机将于明年 4 月上市，看起来是一款坚固的欧洲制造的设备，内置了一些智能。虽然我不同意它需要这样一个热情洋溢的画外音艺术家，但我确实同意 3D 打印领域的变革时机已经成熟。
