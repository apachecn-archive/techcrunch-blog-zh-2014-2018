现在，您可以使用Duet Display TechCrunch在Mac上添加虚拟触摸条

原文：https://web.archive.org/web/https://techcrunch.com/2016/12/20/you-can-now-add-a-virtual-touch-bar-to-your-mac-with-duet-display/

如果你有一个iPad，你就有了一个非常有能力的使用 Duet Display 的电脑外接显示器，现在你也可以体验苹果的新Touch Bar，而不必拿起一台新电脑。这与在你的电脑中内置一个不同，但这是一个很好的方式来玩Touch Bar功能，并看看它如何与苹果原生应用程序和其他支持它的软件(如 Photoshop )中的现有工作流程配合。

Duet Display的更新还增加了对iPad键盘输入的支持，这意味着如果你有一个智能键盘或其他附件，如Logitech Create Keyboard for iPad，当你在运行Duet Display时将它连接到平板电脑上，你可以在Mac和PC版本上使用它进行输入。

不过，Touch Bar是这次更新中最值得注意的部分，它在使用中的效果令人难以置信，作为一个经常使用标准MacBook Pro硬件Touch Bar的人来说。它会根据你使用的应用程序进行上下文更新，就像真正的Touch Bar一样，它支持长按操作来控制亮度和音量，以及其他方便的功能，如输入预测和表情选择器。

Duet首席执行官兼创始人拉胡尔·万德表示，他的团队也在努力使用户在使用应用程序时可以定制触摸栏，用户可以指定上下文和全局快捷键。将Touch Bar的可用性扩展到台式机和其他目前无法使用新功能的人，应该有助于在为苹果最新输入设备建立支持方面普遍提高对开发者的吸引力。

您可以在Duet Display菜单中打开和关闭虚拟触摸条，默认情况下它是关闭的。这个功能也是Mac独有的，以防这一点不明显。键盘支持也非常方便(这意味着你可以将iPad用作iMac的第二个显示器，或者作为Mac mini的唯一显示器，而不需要另一个键盘)，可以跨平台工作。

Duet Display已经是iPad上最有用的应用程序之一，现在它是在新款MacBook Pro之外查看Touch Bar的唯一方式。此外，最近增加的功能，如苹果铅笔的压力感应，使得使用Duet Display的iPad成为比大多数专用外部显示器更好的外部显示器。如果你还没有使用过这个应用，这绝对是一个更新，让它更值得一试。
