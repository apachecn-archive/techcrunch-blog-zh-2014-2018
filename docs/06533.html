密歇根州弗林特市整个城市的铅中毒事件是可以预防的，不应该发生。

许多权威人士和行业专家都这样说过。然而，他们中的大多数人解释说，如果政府运作正常的话，环境保护机构会适当地与他们的上级沟通，问题会更快地被发现。

那些持更愤世嫉俗观点的人认为，安娜堡的政府对长期被围困的弗林特的大部分贫困居民的困境不太感兴趣，弗林特是铁锈地带崩溃的受害者。

我认为这些专家错了。开放 d ata 可以帮助我们，美国人民阻止下一个火石。更确切地说，实时，-可读，定期报告 o 笔 d 数据即从采集到分析的所有方式。

我们不能也不应该依赖政府来保证我们的安全。这不是起诉。政府是会犯错的，就像任何其他大型组织一样。但是 100 年前，还没有办法轻松访问、分析和监控政府活动。今天，没有任何借口不这样做。

在弗林特的案例中，如果州和联邦环境部门及时将原水检测结果放入一个类似 Socrata 、 Junar 或 CKAN 的平台，那么任何受过一点教育的公民都可以对结果进行快速分析，并自行判断是否出了问题。

开发人员也可以通过 API 访问这些数据，并将其传输到任意数量的数据分析平台，以发现异常样本、异常和其他危险信号。

同样重要的是，这种系统可以检测测试过程中可能显示操纵的变化。在燧石、的案例中，没有报告多项铅的阳性检测结果。如果这些额外的样本被包括在报告中，那么这些样本就会超过临界阈值，迫使政府进行更深入的调查，甚至可能停止供应。

让弗林特成为战斗口号。

一个智能分析系统，指向报告的数据，可能已经注意到样本量的急剧下降或发现异常。

不幸的是，今天许多政府数据以肮脏、不可读的格式提供给人民。通常情况下，它会晚我们一两年出现。想象一下，如果谷歌分析给你的数据有一两年的滞后时间。这不是一个完美的类比，但也差不远了。

我们已经证明，我们可以数字化并以非常及时的方式报告多种类型的关键模拟政府数据。查看你所在城市或邮政编码的实时犯罪地图，体验一下这个。犯罪数据是杂乱的数据，被人类收集。但是当报告结构被规范化和强制执行时， machine 可读性是自动的，没有痛苦的。

需要澄清的是，许多与我们生活相关的政府数据甚至都不是 T21 的数据。每年在火石采集的水样数量相对较少。如果你把过去 50 年里从美国每一个水系统采集的所有水样放入一个数据库，与网飞、亚马逊或任何其他主要网络资产的每月点击流分析数据库相比，它就相形见绌了。一位聪明的博主能够在一个下午的时间里分析纽约市所有的停车罚单数据，并找到该市“最贵”的消防栓。他选择的工具？微软 Excel。

我们已经看到人们自己动手收集和修正关键问题上的政府数据。布莱恩·伯格哈特经营着致命遭遇，这是一个非盈利的众包项目，由分散的志愿者团队负责记录美国公民在执法事件中丧生的每一个案例。在一个负责任的社会中，获得关于人们在这些事件中死亡频率的准确数据似乎是一项关键的政治职能。

我们不能也不应该依赖政府来保证我们的安全。

但是联邦调查局长期以来一直在努力从地方当局收集数据，而地方当局又不愿意自我报告和自我监督。Bergdorf 的团队将原始媒体报道转化成结构化的数据。你可以在的在线电子表格中看到它(或者用于你自己的分析)。

目前，谷歌数据表是最全面的警察死亡历史记录。它也是《华盛顿邮报》和《卫报》关于这个话题的重要媒体项目的基础。

在活跃的公民和活跃的媒体之间，在缺乏良好政府的情况下由人民创造的数据报道结构已经产生了影响。对话越来越多，许多人现在都在问，为什么在美国有成千上万的公民死于执法人员之手，而在欧洲或亚洲类似的发达社会中却没有发生这种事情。

将这一点与弗林特联系起来，全国各地的父母和社区活动家都对他们的环境状况非常感兴趣。即使是愤世嫉俗的人也能做出明显的经济关联，即在一个被铅中毒或其他环境问题所困扰的社区里，房屋将会失去重要的价值。

对于弗林特的父母来说，他们面临着最难以言喻的恐惧。他们的孩子不太可能从大学毕业，更有可能进监狱，也不太可能保住一份好工作，实现美国梦。幼年时期接触铅会导致智商永久下降。

密歇根州弗林特市整个城市的铅中毒是可以预防的，也不应该发生。

人们不禁会想，如果你把对话中的弗林特换成，比如说，帕洛阿尔托或诺伊山谷，技术圈将会大动干戈，不仅要求修复供水系统，还要求他们在测试数据一出现时就以一种他们容易分析的格式接收数据。换句话说，不要测试困在 PDF 中的数据或带有大量合并行和列的格式不良的电子表格，这需要花费大量时间来平整和清理。

这才是重点。对弗林特有好处的事情对帕洛阿尔托也有好处。开放 d ata 可以平等地帮助富人和穷人，年轻人和老年人，让我们让我们的政府更负责任，并为自己做出判断。对于政府来说，强制转向机- 可读、接近实时的 o 笔 d 数据将是一个巨大的变相祝福，因为我们终于可以将我们用来分析在线数据的强大工具应用于模拟世界。

先说清楚。开 d ata 不是万能的。它不会治愈病人。它不会自动修复火石里的管道。而且 o 笔 d ata 并不代表所有数据都要开。在许多情况下，人事档案必须保持关闭，私人记录也应如此。
