变色龙为更智能的产品教程筹集了 190 万美元

变色龙是一家初创公司，希望帮助其他初创公司和在线企业更好地向用户解释他们的产品。它今天宣布已经筹集了 190 万美元的种子资金，由 True Ventures 牵头。

联合创始人兼首席执行官普尔基特·阿格拉瓦尔(Pulkit Agrawal)表示，对许多公司来说，“用户加入真的很痛苦”——他们可以非常努力地工作，并花费大量营销资金来获得新用户，但当这些用户发现自己被产品弄糊涂时，他们就会立即消失。

在 Agrawal 看来，how-to 视频和其他解释者也不是最好的解决方案——如果有人试图记住视频中的一切，或者不断地在视频和产品本身之间来回点击，这就不好了。(并不是说他完全反对宣传片，你可以在下面看到。他只是不认为它们是解释如何使用特定网络应用程序细节的最佳方式。)

另一方面，使用 Chameleon，您可以在实际产品中包含教程。因此，如果有人在试用你的产品，你可以告诉他们，他们应该点击一个点来完成某个任务——然后他们就可以继续做下去。

一旦你在你的网站上添加了几行 JavaScript 代码，你就可以通过一个所见即所得编辑器来创建这些教程，它会作为侧边栏出现在你的网站上，不需要编码。这些教程还可以绑定到某些触发器，当用户进入特定页面或犯特定错误时就会出现。你可以测试不同的版本。

客户已经包括城市飞艇和 BloomReach 。

变色龙目前只适用于网络——agr awal 不排除最终创建一个移动版本，但他说，“这个问题确实存在于移动设备上。网络上的问题更大，解决方案也更容易。”他说，毕竟，得益于更小的屏幕尺寸，移动应用往往有一个更简单、更容易理解的界面。
