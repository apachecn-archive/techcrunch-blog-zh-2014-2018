# 由于缺乏轮椅无障碍设施，优步面临另一起诉讼

路透社报道，布鲁克林残疾人独立中心和其他残疾人权利组织正在起诉优步，声称优步违反了纽约市的人权法，因为它没有足够的轮椅无障碍汽车供需要的人使用。

该集体诉讼还包括大都会纽约残疾人行动组织和所有人的出租车运动作为原告，声称优步正在进行“普遍和持续的歧视”，因为优步的 58，000 多辆汽车中只有 200 辆左右可供使用轮椅的人使用。

在过去的几个月里，优步也被密西西比州杰克森市的[使用轮椅的人](https://web.archive.org/web/20221026011348/https://beta.techcrunch.com/2017/05/09/uber-lawsuit-people-with-disabilities/)和华盛顿特区的[平等权利中心](https://web.archive.org/web/20221026011348/https://beta.techcrunch.com/2017/06/28/equal-rights-center-sues-uber-for-denying-equal-access-to-people-who-use-wheelchairs/)起诉。去年，芝加哥的一个残疾人权利组织以类似的理由起诉优步。

优步确实为残疾人提供服务，但问题是没有足够的车辆为身体残疾的人提供平等的机会。

优步在 2014 年首次推出了一项面向残疾人的服务，[推出了优步接入](https://web.archive.org/web/20221026011348/https://newsroom.uber.com/us-california/uberaccess-sd/)。通过优步访问，乘客可以请求 UberASSIST 和 UberWAV，这意味着一辆轮椅无障碍汽车将接他们。值得注意的是，优步接入并非在优步的所有市场都可用，优步接入的可用性似乎取决于优步与商业提供商合作的能力。

在英国，优步一直在扩大无障碍服务。今年，[优步将其接入服务扩展到了英国的另外七个市场，包括利物浦、利兹、伍尔弗汉普顿、布拉德福德等。上个月，](https://web.archive.org/web/20221026011348/https://thenextweb.com/insider/2017/07/14/ubers-wheelchair-accessible-uberaccess-service-launches-four-new-uk-markets/#.tnw_qM50LuZM)[优步与残疾慈善组织“天才儿童”合作，为使用轮椅的人提供免费乘车去投票站的服务。](https://web.archive.org/web/20221026011348/https://www.theverge.com/2017/6/6/15745372/uber-wheelchair-users-access-free-rides-vote-polling-stations)

这起纽约州案件的原告最终希望优步出台一项计划，确保残疾人能够“完全平等地”获得其服务。

“优步的技术为包括残疾人在内的所有骑手提供了可靠的交通选择，并使残疾人能够以新的方式获得收入，”优步发言人在给 TechCrunch 的一份声明中说。“虽然还有更多的工作要做，但我们将继续倡导一种解决方案，为那些需要轮椅无障碍车辆的人提供负担得起的可靠交通工具。”
