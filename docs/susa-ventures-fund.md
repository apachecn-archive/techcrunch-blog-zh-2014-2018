# Susa Ventures 筹集了 2500 万美元的种子基金来支持专注于数据的初创公司

初创公司 Susa Ventures 表示，它已经筹集了 2500 万美元来资助“以数据为中心的创始人和企业”

苏萨有四个运营合伙人——[伊娃·何](www.linkedin.com/in/evaho1)(之前是位置数据公司 Factual 的营销和运营副总裁，在此之前是谷歌和 YouTube 的高级产品营销经理)[利奥·波洛维茨](www.linkedin.com/in/lpolovets)(LinkedIn 的早期工程师)[查德·拜尔斯](www.linkedin.com/in/chadabyers)(Integrate.com 平台的前高级总监)[塞思·伯曼](www.linkedin.com/pub/seth-berman/1/649/31a)(天使投资人，历峰集团的前营销人员)。

在宣布他们新基金的博客文章中，合伙人写道:

> 我们以四个朋友的身份创立了 Susa，我们有着共同的信念，尽管有大肆宣传，尽管有众多公司成立，尽管有令人捧腹的首字母缩写词，但数据革命并没有得到充分的重视。我们几乎没有触及关注利用数据做出关键业务决策、自动化流程和通知新产品创造将如何颠覆现有企业并创造新行业的表面。

这篇文章接着指出了卫生、银行/金融和教育，因为该公司认为这些以数据为中心的方法在一些领域有很大的机会。事实上，Susa 表示，它已经进行了 17 项投资，包括泽法健康、Namo Media ( [最近被 Twitter](https://web.archive.org/web/20230130005952/https://techcrunch.com/2014/06/05/twitter-buys-namo-media-to-expand-its-native-ads-offering/) 收购)、Whisper 和 Frontback。
