# YC 支持的 Sketchbox 是一款无需代码设计 AR/VR 环境的工具

设计 VR 和 AR 产品是一项挑战，并不特别适合这种现实。

Unity 和 Epic Games 在调整工作流程方面取得了很大进展，允许用户以非常精确的方式编辑虚拟现实内容，但与此同时，它们仍然是为技术熟练的人量身定制的非常技术性的程序。

今天， [Sketchbox](https://web.archive.org/web/20221207214152/https://www.sketchbox3d.com/) 从 Y Combinator 的最新创业公司类别中推出，帮助设计师在 VR 中创建虚拟和增强现实体验，而无需键入一行代码。

Sketchbox 的目标是给设计师一个复杂的工具来制作和改进 VR 环境，而不需要他们有工程背景和对桌面 3D 建模工具的深刻理解。该应用似乎特别倾向于对快速 AR/VR 原型感兴趣的设计师。

Sketchbox 联合创始人乔·康诺利(Joe Connolly)告诉 TechCrunch，“现在我们真正关注的是在桌面设计工具方面没有 3D 专业知识的设计师。“因为对很多公司来说，很难找到一个既是工程师又是设计师的人。”

该软件不仅仅是粗略的草图，你还可以导入 3D 模型，加入自定义图形，获得你需要的细节。虚拟现实允许用户将头伸向最细微的细节，同时他们还可以将设计的环境缩小到豌豆大小。

因此，设计师必须以不同于其他媒介的方式来考虑大图片和小图片，并且从 VR 内部开始做这些事情要容易得多。

对于许多不同的公司和工作室来说，虚拟现实有一种魅力，但也有一种知识，即创造它的成本很高，观众很少。即使创建一个概念证明也需要大量的资源，但是一个设计师向一个团队展示十几页潦草的草图或一个立体模型仍然不能很好地了解他们脑子里在想什么。

Sketchbox 用户可以发送项目文件或引导人们通过演示模式来展示他们的思维过程。从那里，可以进行注释，文件可以导出并发送到 Unity 或上传到 Sketchfab。Sketchbox 旨在为设计师提供一个巨大的空间，让他们将大脑投入到 3D 空间中，然后提炼、组织和展示他们对产品或项目的确切规划。

该软件可以在 HTC Vive 和 Oculus Rift 上使用，你可以在这里下载免费试用。
