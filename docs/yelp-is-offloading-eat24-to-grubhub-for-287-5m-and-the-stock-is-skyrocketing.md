# Yelp 正以 2.875 亿美元的价格将 Eat24 出售给 GrubHub，股价正在飙升

今天有两条来自 Yelp 的重磅新闻，这家公司出现在你谷歌搜索餐馆的第一位:首先，GrubHub 以 2.875 亿美元的大合作伙伴身份从该公司收购了食品配送服务 Eat24 第二，该公司报告了第二季度的收益……它们看起来相当不错。

让我们来解决第一个问题。这对 Yelp 来说可能是一个福音，因为它的甜蜜点一直是，真的，以试图收集餐馆信息的用户为中心的内容基础。由于食品配送方面的巨大竞争，Eat24 可能是该公司的用户行为不匹配——尽管在理论上，这个概念很有意义，因为人们可能会搜索午餐并决定只订购它。这样的话，对于 Yelp 来说，让其他具有专业知识的人来处理食品配送的运营组件，而他们专注于自己的核心专业知识，这可能是一笔更好的交易。

天哪，华尔街喜欢这笔交易。该公司收入的增长、可观的增长以及这笔巨额拟议中的交易正在推动股价飙升，消息传出后，股价上涨了近 20%。客观来看，Yelp 的股价今年下跌了约 17%。该公司似乎有点停滞不前——虽然它基本上是一种特定请求的关键资源，但它还没有想出如何打破这种局面。(我们在这里相当宽松地使用了“利基”这个词，因为它仍然是一个巨大的市场。)

在收益方面，Yelp 带来了 2.089 亿美元的收入，同比增长 20%。毫不奇怪，其广告收入在第二季度占了绝大部分，达到 1.866 亿美元。华尔街预计该公司将带来约 2.05 亿美元的收入。

Eat24 交易仍受制于惯例成交条件。但看起来这两家公司的结合——它们基本上都在各自擅长的领域运营，同时交叉授粉——可能最终会重新点燃 Yelp 的增长。Yelp 的股票在过去两年中上涨了 20%，但这种单日涨幅似乎是一个强烈的信号，表明 Yelp 突然不得不像一家不同类型的公司一样被评估。
