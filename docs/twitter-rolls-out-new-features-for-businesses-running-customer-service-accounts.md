# Twitter 为运营客户服务账户的企业推出新功能

Twitter 今天推出了一系列新功能，旨在帮助用户通过他们的官方 Twitter 帐户更好地与提供客户支持的企业联系。现在，这些企业将能够在他们的个人资料上清楚地显示他们的账户是否提供客户服务，以及这些账户在哪些时间是活跃的。

企业现在可以通过 Twitter 仪表板网站上的新客户支持设置页面来表明是否提供服务。一旦启用，企业的 Twitter 个人资料将显示它“提供支持”。此选项还将打开帐户接收来自任何人的直接消息的功能。换句话说，企业将不再需要请求客户跟踪他们，以便他们可以发送私人消息。

![screen-shot-2016-08-23-at-11-37-37](img/8cfdbbf52f7dff0ff8f65ed4ea837900.png)

这种“提供支持”的细节也将在人们搜索账户时显示出来，例如，包括他们何时在推文中提到该公司，或开始键入直接消息。

此外，企业现在还可以选择在其个人资料上显示客户服务的可用时间，这将有助于设定何时可以收到回复的预期。

当客户访问这些客户服务 Twitter 帐户时，他们还会看到一个新的、更突出的按钮，可以开始与有问题的企业直接联系。

TechCrunch 此前曾报道过更大的直接信息按钮的存在，当时该公司正在野外测试这一功能。像@AppleSupport、@优步 _Support、@BeatsSupport、@ATVIAssist (Activision Support)等账户都是早期的测试者。在 Twitter 上非常活跃的 T-Mobile 也采用了这些新功能。

![img_3729](img/57d7d4b07ea26389686cd3336f105ce7.png)

正如我们当时指出的那样，更大的直接消息按钮的推出——它占据了过去“推文到”和“消息”按钮并排放置的全部空间——鼓励用户开始与企业进行私人对话，而不是公开推文。这可能会将沮丧的客户在 Twitter 上发表的一些更负面的评论转移到企业的私人频道。

这些变化也让 Twitter 能够更好地与脸书竞争，后者在过去的向其页面用户推出了一个功能，向脸书用户展示该业务如何对客户查询做出[响应](https://web.archive.org/web/20230131004524/https://techcrunch.com/2015/12/08/facebook-page-response-time/)。(然而，最近的页面重新设计似乎暂时取消了这些信息文本。)

Twitter 此前推出了其他客户服务功能，包括直接消息链接和客户反馈卡。
