# 蒂姆·阿姆斯特朗宣誓就职，点击此处阅读备忘录全文

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/12/tim-armstrong-is-out-at-oath/?via=indexdotco>

Verizon Communications 今天证实，Guru Gowrappan 将取代蒂姆·阿姆斯特朗担任 Oath(TechCrunch 的所有者)的首席执行官，这证实了整个夏天都在流传的谣言。

这是一个巨大的变化:Armstrong 曾是 Oath 的首席执行官，在此之前是 AOL(Oath 的前身，在与雅虎合并之前)的首席执行官，在至少 10 年的时间里，他管理着这家公司，经历了起起落落，包括以 44 亿美元的价格出售给威瑞森，以 48 亿美元的价格收购雅虎，以及将两家公司合并。

在那段时间里，他还帮助一家境况不佳、日渐老化的拨号上网公司重新定位为一家媒体公司，并在它的下面配备了一个广告技术引擎。

在今天的一份内部备忘录中，阿姆斯特朗没有回避一直在流传的谣言，但也试图在他认为该公司正在走的道路上，将 Oath 和 AOL 本身进行比较。

他写道，“虽然已经有并将会有关于这些变化的猜测，但有几个重要信号不应被淹没在噪音中。”“首先，世界已经数字化，很少有公司像我们这样同时拥有移动和数字资产。再加上世界上最好的移动公司威瑞森的支持，Oath 拥有一切成功的要素。第二点是，事情需要时间。美国在线扭亏为盈用了两年半多的时间；Oath 刚刚成立一年多，第一年用于整合雅虎拍卖后的资产，并解决数据泄露相关问题。“誓言要充分发挥潜力还需要时间。”

 **Gowrappan 将宣誓承担作为首席执行官的所有管理责任，并向威瑞森首席执行官卫翰思报告。Armstrong 将继续留在公司指导管理过渡工作，并将向威瑞森董事长 Lowell McAdam 报告。

“我非常感谢蒂姆对业务的许多贡献，我相信他在未来几个月的支持将继续加强 Oath 的市场潜力，”Vestberg 在一份内部备忘录中说。维斯特伯格和阿姆斯特朗的备忘录都在下面。

很难理解阿姆斯特朗的临别备忘录。今年早些时候，我(也就是英格丽德)直截了当地问阿姆斯特朗，关于他离开的报道是否属实。他直截了当地回应说，他没有离开誓言的计划。

但鉴于过去几周的事态发展——有报道暗示，他一直在试图说服威瑞森让他完全脱离誓言，并将其作为一个独立的实体来运营——暗示着也许他没有说谎:也许他并没有计划离开誓言；但相反，他一直试图奠定计划离开威瑞森，并与他宣誓。

现在阿姆斯特朗走了，誓言还在。具有讽刺意味的是，从长远来看，也很难猜测誓言在威瑞森的地位。这家较大的电信公司现在由从爱立信加盟的卫翰思领导，他本质上是一个网络人——当然，这也是威瑞森更广泛业务的核心。在这方面，Oath 以及威瑞森收购美国在线(AOL)和雅虎(Yahoo)时设想的更广泛的媒体/OTT 游戏，如果没有这一战略的策划者领导，现在似乎就不那么清晰了。

值得注意的是，阿姆斯特朗将继续留在公司，而不是向维斯特伯格报告，而是向麦克亚当报告。

到目前为止，这个消息似乎并没有让投资者感到不安:在经历了过去几个月的普遍下跌后，该公司的股票交易略有上升。

Gowrappan 是从阿里巴巴集团加入 oath 的，他将负责运营管理，因为在威瑞森将美国在线和雅虎合并后，这两家公司的整合即将完成。
