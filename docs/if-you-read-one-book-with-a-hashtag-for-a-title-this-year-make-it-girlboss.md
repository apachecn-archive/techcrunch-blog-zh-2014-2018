# 如果你今年读了一本带有标题标签的书，那就把它变成#GIRLBOSS 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/05/11/if-you-read-one-book-with-a-hashtag-for-a-title-this-year-make-it-girlboss/>

励志故事是平庸的，你不认识的人的个人轶事是乏味的，大多数商业书籍包含大约一张明信片长度的智慧，分布在数百页上。

索菲亚·阿莫鲁索最近[出版的书#GIRLBOSS](https://web.archive.org/web/20230320054513/http://www.amazon.com/GIRLBOSS-Sophia-Amoruso/dp/039916927X) 设法避免了这些陷阱，融合了她的个人生活、职业生活和一系列可教的时刻，形成了一些相当可读的东西。*(上周在 TechCrunch Disrupt NY 活动上，索菲亚·阿莫鲁索(Sophia Amoruso)与我的老板亚历克西亚·特索西斯(Alexia Tsotsis)坐在一起，我偶然看到了这本书，并在回家的飞机上阅读了我从扬声器包里收到的副本。你可以很快穿过它。)*

阿莫鲁索并不是从顶层开始的。她的故事是一系列的不幸和冒险，促使她在易贝开了一家出售复古服装的商店。这家名为 Nasty Gal 的商店迄今为止已经筹集了 4900 万美元，目前的收入已经达到了 9 位数。

从在学校挣扎到因入店行窃差点被捕，再到经营一家价值远超 1 亿美元的公司，这并不是一个简单的故事。这本书虽然是针对年轻人的，但在谈到你可能已经学到的实用建议时，它拖了一些章节。跳过不适合你的部分。

### “梦想远大，从小处着手”

阿莫鲁索不是天生的大亨。事实上，她声称，如果她知道自己现在在“讨厌的女孩”忙着出生的时候做什么，她就不会继续经营这家公司了:“我从来没有创业过。我在易贝开了一家商店，最后开始做生意。如果我知道事情会变成这样，我绝不会这么做。”

鉴于阿莫鲁索看起来很开心，并在 Nasty Gal 负责管理自己的职责，她的评论暗示，过去的自己会在接下来的大量工作中犹豫不决，而不是说如果她可以回到过去重新生活，她就不会创建这家公司。

教训:你带着你拥有的军队去打仗，所以如果你今天还没有完全准备好成为一家大公司的首席执行官，不要担心。你还在成长。

在#GIRLBOSS 中无孔不入的是工作的观念。这项工作，而不是应该避免的事情，应该被接受。这对每个人都适用。

对年轻人来说:

> “我们这一代的很多人似乎都不明白，你必须通过努力才能获得提升。入门级的工作就是入门级的工作，这意味着你不会去做最有趣和最有创意的项目。我听过很多 20 多岁的人抱怨他们的工作，因为他们“能提供的东西太多了”，但最重要的是，你必须做好你该做的工作。”

对于更有经验的人:

> “你想知道我最讨厌哪四个字吗？那不是我的工作。[……]在一个理想的世界里，你永远不必做低于你职位的事情，但这不是一个理想的世界，也永远不会是。"

当 Nasty Gal 刚刚开始成长时，Amoruso 花时间在 MySpace 上发送垃圾邮件——嘿，这很有效——积极浏览房地产销售，寻找新的销售项目，同时处理库存和履行事宜。她是一个人的公司。但是，“运气在她的成功中扮演了什么角色？”是一个公平的问题。

### 运气

所有的成功都是运气吗？财富不是二元的，但阿莫鲁索对细节的关注——贴上恰到好处的标签，为每件商品建模，跟踪什么畅销，什么滞销，以更好地定制产品组合，等等——才是真正的魔力。

她对运气概念的评论将“运气是准备与机会的结合”这一陈词滥调又向前推进了一步:

> “我讨厌运气这个概念，尤其是当人们试图把它应用到我身上的时候。是的，这是真的:成千上万的企业倒闭了。我的成功了。这仅仅是因为我“运气好”吗？我不这么认为。
> 
> 我讨厌运气，因为它意味着没有责任。它暗示你可以什么都不做，然后像踩进人行道上的一堆狗屎一样轻松地踏入成功。[……]运气让我们相信，无论发生什么，无论是好是坏，都不是我们的功劳，也不是我们的错。这就是为什么我不买运气。

我所知道的最成功的首席执行官往往会筋疲力尽，焦虑不安，而不是那种经常出现在杂志封面上的快乐的“是的，我们在杀人”类型。阿莫鲁索目前的地位是基于多年的汗水和单调的劳动。

### 拥抱自己

一旦你获得了一定程度的成功，贬低自己总是件棘手的事情——你能设法让自己听起来不那么卑微吗？阿莫鲁索基本上成功了。

#GIRLBOSS 的个人组成部分的核心弧是看着年轻的阿莫鲁索一次又一次地摔倒，幅度越来越大，直到她基本上决定在商店行窃探险后恢复健康，最终她看着她的“非法生活方式迅速消失在远处。”

阿莫鲁索仍然不太想融入社会(“来自郊区，我总是把舒适等同于无聊……”)，但她把自己的生活推上了一条轨道，让讨厌的盖尔走到了尽头。

因此，事实证明，在墙上涂抹便便、穿孔、低分、搭便车、和嬉皮士一起闲逛、做一连串没有前途的工作、连续盗窃并不会让你不适合过为自己做好事的生活，是的，如果你对这类事情感兴趣，你找到的公司。

生活不易，生意更难。但是不管你今天在哪里，你都可以推进你的职业生涯，或者建立一些可能会失败也可能不会失败的东西。振作起来，那是好消息。
