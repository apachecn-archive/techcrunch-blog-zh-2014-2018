TrendKite 为 PR Analytics 筹集了 320 万美元，聘请达奇斯集团的 Erik Huddleston 担任首席执行官

承诺帮助品牌和机构衡量公关效果的初创公司 TrendKite 已经筹集了 320 万美元的首轮融资。它还宣布它有了一位新的首席执行官——Erik Huddleston，社交监测公司 Dachis Group 的前首席技术官兼产品执行副总裁。

哈德斯顿告诉我，当达奇斯集团在二月份被 Sprinklr 收购时，他“完全致力于”留在董事会。

然而，他已经是 TrendKite 的顾问，在收购宣布后不久，这家初创公司的创始人带他去吃饭，并建议他加入担任首席执行官。用他的话来说，这是“一个小小的埋伏”显然这很有效。

这些创始人将继续留在公司——前首席执行官马特·艾利森(Matt Allison)现在是首席产品官，而他的联合创始人 AJ·布鲁诺(AJ Bruno)和帕特里克·布兰宁(Patrick Brannen)继续分别担任总裁和首席技术官。

Huddleston 表示，尽管像 Sprinklr 这样的公司代表了从 IT 到营销的更广泛的支出转移，特别是向以数据为中心的营销工具的转移，但这些工具中的大部分都专注于广告，而“完全忽略了免费媒体(即新闻报道和消费者提及，公司不付费)。”

TrendKite 表示，它可以为公司提供他们需要的公关数据，其功能包括定制仪表板(跟踪关于您公司的最受欢迎的文章、顶级影响者的提及以及与竞争对手相比的总体情绪和声音份额)、电子邮件新闻提醒和可共享的 PDF 报告。

哈德斯顿还称赞了创始人的远见，以及他们在行业和销售方面的经验:“他们唯一真正缺少的是产品部分。”他并不是在批评产品本身(他回忆起曾与“赞不绝口”的客户交谈过)，只是暗示公司需要一个有强大产品背景的人担任高层。

该公司的新融资由 Mercury Fund 和 Silverton Partners 牵头。继去年秋天的 120 万美元之后，这笔资金将用于扩大公司的数据基础设施和销售团队。
