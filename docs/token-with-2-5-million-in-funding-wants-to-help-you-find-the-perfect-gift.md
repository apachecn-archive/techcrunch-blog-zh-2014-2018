# Token 拥有 250 万美元的资金，希望帮助您找到完美的礼物 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/04/26/token-with-2-5-million-in-funding-wants-to-help-you-find-the-perfect-gift/>

没有什么比送别人一份完美的礼物更好的了。因此，送错礼物是世界上最糟糕的感觉之一，这是很自然的，因为你不知疲倦地寻找却仍然没有得到。

这就是[令牌](https://web.archive.org/web/20221207014001/https://token.ai/)发挥作用的地方。

这家初创公司今天从 Human Ventures startup studio 公开推出，希望在人工智能的帮助下接受送礼的挑战。

它是这样工作的:

用户下载该应用程序，输入一些关于礼物接收者的信息，包括风格、他们与送礼者的关系、送礼场合和预算。然后，Token 通过其数据库过滤这些信息，该数据库包含 1000 多家品牌和零售合作伙伴的数千种产品。

机器学习缩小了选择范围，然后人类从更窄的礼物池中寻找，将少数最终选择发送给送礼者。

但是 Token 的真正特别之处在于它为客户提供了端到端的服务。Token 让用户直接从应用程序购买，处理礼物包装、运输，并将礼物与真正的手写便条配对，而不是将用户推到一些外部网站。

Token 的礼物推荐是免费的。该公司通过向在该平台上购买礼物的用户收取基于百分比的服务费来赚钱，同时在产品售出时从零售商和品牌那里收取佣金。

虽然有少数初创公司希望改变我们送礼物的方式，但没有一家公司提供端到端的服务来为用户闭环。

首席执行官乔纳森·贾维斯表示，Token 的优势之一是其机器学习算法。

“这种类型的技术以前没有，这将有助于我们以一种深思熟虑和创造性的方式将服务扩展到数百万用户，”贾维斯说。“在送礼的过程中，存在着巨大的抛弃。但是 Token 会提醒您发送礼物、选择合适的物品、处理递送和礼物包装。到目前为止，还没有人完成过送礼的整个过程。”

贾维斯还提到，仅在美国，送礼就是一个 1310 亿美元的市场。在 beta 测试期间，Token 在客户间的平均转化率为 30 %,这对电子商务来说是非常高的。

Token 获得了 250 万美元的种子资金，由前期风险投资公司牵头，慢风险投资公司和人类风险投资公司参与其中，它似乎准备好迎接挑战，确保不断给予的礼物:盈利能力。
