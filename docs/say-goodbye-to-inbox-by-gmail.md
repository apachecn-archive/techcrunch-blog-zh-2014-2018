# Gmail TechCrunch 向收件箱说再见

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/12/say-goodbye-to-inbox-by-gmail/>

随着新的 Gmail 的推出，事情已经明朗，但今天谷歌正式宣布:该公司的 Gmail 实验性电子邮件客户端 Gmail 的收件箱将于 2019 年 3 月底关闭。

谷歌表示，它做出这一改变是为了将重点“完全放在 Gmail 上”尽管这很有道理，但看到收件箱驶进夕阳中还是令人遗憾，因为它开创了许多现在已经成为新版 Gmail 一部分的功能。

![](img/8221a853490792177db6c0194ed6fb5d.png)我很想看到谷歌继续尝试收件箱。毕竟，这是公司开始收件箱项目的原因之一。毕竟，很难对一个拥有 10 亿用户的服务进行激进的实验。然而今天，Google now 似乎也愿意在 Gmail 中尝试新事物。例如，Smart Compose 在新的 Gmail 中首次亮相(许多权威人士正确地将此解读为收件箱在砧板上的标志)。

虽然新的 Gmail 现在拥有收件箱的大部分功能，但其中一个非常缺乏的功能是旅行捆绑包。这项有用的功能可以自动将您所有的航班、酒店、活动和汽车预订组合成一个包，是 Inbox 的最佳功能之一。我们的理解是，谷歌计划在明年年初将这一功能引入 Gmail 希望在收件箱关闭之前。

所以你有它。Gmail 的收件箱将在六个月后关闭，但如果谷歌在几年后恢复这个想法，尝试一些其他的电子邮件实验，我不会感到惊讶。在那之前，这里是谷歌从收件箱转移到 Gmail 的指南。
