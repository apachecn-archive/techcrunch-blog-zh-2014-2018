# 涉及 Dyn DDoS 攻击的网络摄像头被召回

Dyn 表示，上周它发现了“数十万个”独特的 IP 地址，这些地址涉及对其托管 DNS 服务的大规模僵尸网络 DDoS 攻击，这些攻击使 Twitter、亚马逊和其他网站的许多用户陷入瘫痪。这些设备中至少有一部分现在需要召回，中国电子公司杭州熊迈召回了使用其部件的网络摄像头，这些部件被确定为构成了所涉及设备的很大一部分。

安全专家指出，这些网络摄像头容易受到攻击，并被包含在用于淹没 Dyn DNS 的 Mirai 未来组合僵尸网络中，因为它们的默认密码很容易被猜到，这使得攻击者可以更容易地破解他们的登录信息，并将它们纳入僵尸网络。

熊迈在向 BBC 提交的一份[声明中否认其设备代表了攻击中使用的大多数设备，事实上，似乎有可能涉及来自大量不同制造商的物联网硬件。尽管如此，熊迈已经对所有使用其电路板和其他组件的网络摄像头进行了召回，这代表了相当数量的设备，因为熊迈为许多公司供货。](https://web.archive.org/web/20230324084955/http://www.bbc.com/news/technology-37750798)

该公司还指出，用户不更改他们的默认密码也是一个原因，事实上，今天早上[美国物联网消费者研究](https://web.archive.org/web/20230324084955/https://techcrunch.com/2016/10/24/americans-uneasy-with-iot-devices-like-those-used-in-dyn-ddos-attack-survey-finds/)的一份报告显示，人们不更改他们在这些设备上的默认登录凭据是非常常见的。
