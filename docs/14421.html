研究发现，Reddit 对其毒性最大的子内容的有争议的禁令实际上起了作用

原文：https://web.archive.org/web/https://techcrunch.com/2017/09/11/study-finds-reddits-controversial-ban-of-its-most-toxic-subreddits-actually-worked/

似乎就在几天前，Reddit 终于禁止了一些最可恶和最可悲的子网站，包括 r/coontown 和 r/fatpeoplehate。当时，这一举动被一些人嘲笑为毫无意义，类似于将罪犯从一个社区赶走，却给另一个社区带来麻烦。但一项新的研究表明，至少对 Reddit 来说，它有着持久的积极影响。

对网上仇恨言论的监管已经成为许多火焰战争的爆发点，尤其是在过去几个月，因为白人民族主义者、新纳粹分子和其他持有令人憎恶但严格来说相当合法的观点的人正在努力应对被互联网最大平台多次禁止的问题。

这种做法导致 StormFront 等网站寻求在品牌外主机和小型社交网络等惨淡的港口寻求庇护，宣传他们对某些类型的自由言论被他人“审查”的容忍。这是一个反对禁止麻烦用户或社区的例子:他们会去别的地方，所以为什么要麻烦呢？

佐治亚理工学院的研究人员认真对待这个问题，因为直到有人真正调查这种禁令是有益的、有害的还是两者兼而有之，这都是猜测。因此，他们采用了 Reddit 数据的主要语料库(由 PushShift.io 编译)和检查了仇恨言论及其提供者到底发生了什么，并将上述两个子数据库作为案例研究。

本质上，他们观察了构成 CT 和 FPH(他们这样称呼)的数千名用户，并量化了他们仇恨言论的使用情况。然后，他们将封禁前的数据与封禁后的相同用户进行比较:他们发表了多少仇恨言论，他们“迁移”到了哪里(例如，重复的子街道，相关的子街道，等等)。)以及“入侵”的子群是否因此经历了仇恨言论的激增。对照组是通过观察没有被禁止的相似子群的活动而建立的。

他们发现在 Reddit 这样的网站上减少不必要活动的策略令人鼓舞:

  • 禁令实施后，同一批用户发表的仇恨言论减少了 80-90%。
  • 被禁社区的成员离开 Reddit 的比率明显高于对照组。
  • 移民是常见的，既有类似的亚种族(即公开的种族主义者)，也有不太相关的亚种族(r/The_Donald)。
  • 然而，在这些社区中，仇恨言论并没有可靠地增加，尽管入侵者遇到并测试新规则和版主时会有轻微的颠簸。

总的来说，研究人员得出结论，该禁令在它所要做的事情上是相当有效的:

对于我们的研究问题框定的“工作”的定义，禁令对 Reddit 有效。它在用户层面和社区层面都取得了成功。通过禁止从事种族主义和肥胖羞辱的子网站，Reddit 能够减少这种行为在网站上的流行。

当然，事情并没有那么简单。很自然地，许多之前在 CT 上散布种族歧视言论的用户转到了 Gab 或 Voat，在那里他们的行为得到了自豪的培养。但是 Reddit 上的禁令并不是为了消除种族歧视；是为了在站台上劝阻。为此，它实现了自己的目标(我问过 Reddit 对这项研究及其结论的看法)。类似的策略可能适用于其他平台。

对于像 Reddit 甚至谷歌或脸书这样的主要平台来说，如何打击种族主义和仇恨的问题实在是太多了。他们能希望做的最好的事情就是在它出现的时间和地点打击它。尽管这看起来无效，但它对 Reddit 有效，在其他地方也可能有效:偏执很容易，珍惜它的人很懒。让它变得困难，许多人可能会发现在网上骚扰、羞辱和辱骂那些与自己不同的人是不值得的。
