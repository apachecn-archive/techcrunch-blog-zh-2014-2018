# 你现在可以在 iOS 设备上安装屏幕亮度应用 f.lux，而无需越狱

更新:嗯，真快。F.lux 的开发者[已经撤下了下载链接](https://web.archive.org/web/20221007034255/https://beta.techcrunch.com/2015/11/12/f-uxd-apple-says-screen-brightness-app-violates-developer-agreement/)，称苹果称该应用是基于 iOS 开发者协议的意志。

几个月前，当苹果合并他们的 iOS 和 Mac 开发者计划时，该公司也悄悄地取消了你必须是注册开发者才能将应用程序从 Xcode 加载到你的 iOS 设备的要求。

虽然苹果的意图可能是消除新应用程序开发者的财务障碍，但一个副作用是，现在任何人都可以下载开源 iOS 应用程序并将其安装到他们的设备上。

[F.lux](https://web.archive.org/web/20221007034255/https://justgetflux.com/) ，这个调节电脑屏幕色温以减少夜间使用时眼睛疲劳的流行程序，刚刚通过发布 f.lux 的开源 iOS 版本利用了这一变通办法。以前，你必须越狱你的设备才能使用屏幕亮度软件。

那么为什么 f.lux 的开发者不能直接把他们的应用提交到应用商店呢？

因为自动调整你的 iOS 设备的屏幕温度需要应用程序不断在后台运行，以及使用私有 API……这两者都违反了 App Store 的规则。

苹果的立场是有道理的，特别是因为私有 API 已经被认为会偷偷摸摸地做一些事情，比如把用户信息发送回开发者的服务器。也就是说，f.lux 已经存在了五年多，是一款非常值得信赖的开源软件，尤其是在开发者社区中。

你可以从[应用的网站](https://web.archive.org/web/20221007034255/https://justgetflux.com/sideload/)下载 Xcode 项目，也可以查看如何将它加载到你的 iDevice 上的完整说明。

这肯定不像从 App Store 下载一个应用程序那么容易(你仍然必须处理 Xcode 不断令人沮丧的预置描述文件)，但至少就苹果而言，这比越狱更合适。
