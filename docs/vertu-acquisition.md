# 豪华手机制造商 Vertu 再次易主

据报道，流亡的土耳其商人哈坎·乌赞刚刚为 Vertu 公司支付了大约 10，000 部均价的手机。虽然这很难说，真的，这将是最糟糕的投资。

根据[昨天来自*每日电讯报*T5 的报道，一个神秘的土耳其商业家族的成员 Uzan 已经花费了大约 5000 万英镑(6100 万美元)收购了这家总部位于英国的淫秽花哨的 blinged 手机制造商。这标志着 Vertu 自 2012 年从诺基亚分离出来后第三次易主。最近，它由总部位于中国的高丁控股有限公司(Godin Holdings)所有，该公司于 2015 年底收购了该公司。](https://web.archive.org/web/20230407104416/http://www.telegraph.co.uk/business/2017/03/12/turkish-exile-snaps-smartphone-maker-vertu-50m/)

不难看出为什么公司像许多镶有钻石的烫手山芋一样把公司传来传去。Vertu 的手机平均售价约为 6000 美元，一些型号的售价是这个数字的几倍。这种定价可能是一个很好的指标，可以准确地表明一个人要烧多少钱，但它并不能真正说明你在内部得到了什么。

从纯硬件的角度来看，这些设备与上一代手机没什么两样——用花哨的材料和标识装饰起来，再加上“礼宾”客户支持服务。这可能是该公司在 2014 年公布销售额 1.1 亿英镑亏损 5300 万英镑的很大一部分原因——这是该公司最后一次公开提供此类信息。

Uzan 对该报道做出回应，告诉《每日电讯报》“Vertu 是一个强大的品牌，拥有公认的市场定位。我期待着与团队合作，并提供投资，使 Vertu 实现其全部潜力。”这不完全是一个统治世界的计划，而是承认 Vertu 产品的市场潜力相当有限。

除了为什么有人想成为众多所有者中的最新一员这一简单问题之外，这次收购之所以引人注目，是因为乌赞人与前 Vertu 所有者诺基亚之间的紧张关系。这个家族已经受到了这家曾经的智能手机巨头的诉讼，因为它贷款成立了土耳其电信运营商 Telsim。

诺基亚和摩托罗拉指控这个家庭欺诈，最终获得了数十亿美元的未付贷款。这是这个家庭不断下滑的一部分，自然也包括与唐纳德·特朗普(Donald Trump)在特朗普大厦(Trump Tower)公寓的争吵。这可能意味着总统不会很快将他心爱的安卓手机升级到 Vertu。
