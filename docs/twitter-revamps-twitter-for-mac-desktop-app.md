# Twitter 改进了 Mac 桌面应用的 Twitter

今天，在 Twitter 的开发者大会 Twitter Flight 上，该公司宣布对 Mac 版 Twitter 客户端进行了改进和更新。

尽管几个月后才会发布(TBD 时间)，Twitter 解释说更新的功能正在被集成到 Mac 桌面应用中。其中一些功能包括在线 Vine 和视频播放、群组直接消息、直接消息中的照片和在线回复。

此外，Twitter for Mac 将在通知中心显示亮点，让您了解最新的互动/参与情况。
