# 参议院投票 67-32 改革国家安全局的电话记录项目

今天，参议院以 67 票对 32 票通过了美国自由法案。该法案现在将被提交到奥巴马总统的办公桌上，标志着自爱德华·斯诺登爆料以来，国会首次采取行动改革国家安全局(NSA)的监控项目。

该法案在众议院获得了压倒性的支持，但在参议院，它引发了一场有争议的长达数周的辩论，分裂了共和党。参议院多数党领袖米奇·麦康奈尔试图阻止改革法案的努力失败了，并导致《爱国者法案》中的一些条款到期。在即将到期的项目中，批量元数据收集项目可能是爱德华·斯诺登披露的项目中最有争议的。

为了更详细地了解美国自由法案的内容，[点击这里](https://web.archive.org/web/20230129072725/https://techcrunch.com/2015/05/13/surveillance-reform-bill-clears-house-only-to-face-obstacles-in-senate/#.zvmch4:y9mK)。该法案将迫使电话公司而不是政府机构保留美国人的电话记录 18 个月。情报官员将通过请求 FISA 法院的许可来获取这些记录。

这项法律的通过并不令人惊讶。该法案及其在关于美国政府监控努力的更大辩论中的地位，特别是当它们涉及美国公民时，分裂了政党。参议院多数党领袖米奇·麦康奈尔(Mitch McConnell)提议对《爱国者法案》(Patriot Act)进行为期五年的“彻底”重新授权。大多数人反对重新授权本身。

除了未能通过彻底的重新授权，麦康奈尔推动修改《自由法》的努力也失败了。他允许对三项修正案进行投票，这将大大削弱该法案的条款。参议院民主党人和众议院共和党人坚决反对任何会延长批量元数据计划失效时间的修正案。

预计每一个有哪怕半分政治脉搏的人和实体都会做出反应。

参议院今天的投票是自 911 袭击以来国会为削减国家情报机构所采取的最重要的行动。为了避免国家大规模元数据收集计划的长期失效，许多人批评《自由法案》是一种仓促的尝试。但实际上，这是改革国会在大约 14 年前通过《爱国者法案》时仓促行动的第一步。

过去几天的事件表明，我们的立法者有能力改革这些项目。希望这不是结束。
