# Tinder 点击喜剧演员惠特妮·卡明发起“反应”

继[脸书](https://web.archive.org/web/20221203103028/https://beta.techcrunch.com/2016/02/24/facebook-reactions/)和 iMessage 之后， [Tinder](https://web.archive.org/web/20221203103028/https://beta.techcrunch.com/topic/company/tinder/) 今天推出了[反应](https://web.archive.org/web/20221203103028/http://blog.gotinder.com/tinder-launches-reactions/)，让 Tinder 用户互相发送自定义动画响应。

这些反应包括红心、翻白眼、鼓掌、扔马提尼酒等。这使得 Tinder 用户更容易随意使用该应用程序，而不必实际参与对话。

去年 1 月，Tinder [推出了 Giphy 支持和更大的表情符号](https://web.archive.org/web/20221203103028/https://beta.techcrunch.com/2016/01/28/tinder-makes-messaging-more-fun-with-gifs-bigger-emoji-finally-lets-you-upload-photos-directly/)。反应似乎是这一过程的自然延伸，让已经轻如鸿毛的 Tinder 体验变得更加随意和简单。

但这些反应似乎是 Tinder 一项更大计划的一部分，该计划旨在为女性提供所有她们需要的工具来对付混球。

演员兼喜剧演员惠特妮·卡明正在帮助 Tinder 推出一部名为“The Menprovement Initiative”的视频系列，由 J.J. Adler 执导。

【YouTube https://www.youtube.com/watch?v=s3WwQkItiUM&w=560&h=315]

该视频展示了 Tinder 的真实女性员工与惠特妮·卡明互动，以改善男性。也就是说，Tinder 并没有赋予女性权利的悠久历史，尤其是它自己的女性员工。

Tinder 提供了清晰的关于不良行为的社区指南，并为用户提供了报告恶意行为者和匹配后向左滑动以快速逃离不想要的对话的工具。也就是说，将品牌反应作为女性应对混球的一种方式似乎有点乏善可陈。

用飞溅的马丁尼酒反应来回应不会阻止你收到不想要的信息，事实上甚至可能鼓励那个人发更多的信息。因为这些反应很可爱，而且觉得很轻浮。这就像 Tinder 为用户创造了一种方法，让他们在数字街上向打电话的猫眨眼。

Tinder 的反应可能会让 Tinder 各方之间的对话变得更容易，但这似乎并不符合所谓的 Menprovement 倡议。
