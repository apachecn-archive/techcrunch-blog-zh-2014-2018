# Shiftgig 再筹集 2000 万美元，将小时工与公开工作联系起来 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/01/17/shiftgig-raises-20m-more-to-connect-hourly-workers-with-open-jobs/>

像优步、Lyft 和 Postmates 这样的按需服务的兴起，正在催生服务业中新的自由职业者大军，他们不局限于单一的就业地点，可以选择更适合自己的工作时间。现在，一种类似的灵活性也在小时工领域找到了出路。

一家初创公司为小时工搭建了一个移动平台，帮助他们在当地寻找员工来填补他们的时间表缺口(Shiftgig= [轮班工作](https://web.archive.org/web/20230202203432/https://en.wikipedia.org/wiki/Shift_work) + [零工经济](https://web.archive.org/web/20230202203432/http://www.investopedia.com/terms/g/gig-economy.asp))——今天宣布，它已经筹集了 2000 万美元来扩大其市场规模。这一轮 C 轮融资包括新的和现有的支持者——DRW 风险投资公司、FJ 实验室、GGV 资本公司、KDWC 风险投资公司和威廉·布莱尔的一家子公司——使 Shiftgig 的融资总额达到 5600 万美元。该公司没有透露其估值，但消息人士告诉我们，约为 1.5 亿美元。

目前，这使得 Shiftgig 在增长融资方面遥遥领先于两个著名的竞争对手。同一领域的其他公司包括[work op](https://web.archive.org/web/20230202203432/http://workpop.com/)和 [Wonolo](https://web.archive.org/web/20230202203432/http://wonolo.com/) ，这两家公司迄今为止都筹集了 1000 万美元以下的资金(参见此处的[和此处的](https://web.archive.org/web/20230202203432/https://www.crunchbase.com/organization/workpop)和)。

在“中国脸书”人人网带领 Shiftgig 上一轮融资 2200 万美元大约一年后，Shiftgig 获得了融资。自那以后，该公司一直在“稳步增长”，用首席执行官兼联合创始人埃迪·楼(Eddie Lou)的话说。如今，约有 1500 家企业和 15000 名员工在使用该平台，其客户遍布美国 50 个州，但主要集中在 12 个城市。(楼继伟说，该计划“肯定”会使用一些资金来扩大规模。)

楼说，如今，Shiftgig 与广泛的商业客户合作，主要是在食品服务、酒店、零售、物流、仓库和体验营销领域。楼说，有趣的是，优步也接触过它，想使用这个平台，但目前它的员工中没有司机。

这个想法是，企业可以连接到 Shiftgig 平台来填补任何数量的班次。该市场既适合需要填补一个或几个班次的小企业，也适合寻找 200 人的大型活动。在所有情况下，企业为每一个班次支付费用，Shiftgig 收取一定比例的费用，这取决于订单的大小和客户的规模。

在按需经济中，人们对员工的背景和信用调查有时有多彻底进行了一些审查。以 Shiftgig 为例，该公司通过自己的一套资格和审查检查，这些检查也随着时间的推移而变化，因为企业在每次轮班工作后对员工(在 Shiftgig 的市场上称为“专家”)进行评级，最终导致员工的可靠性得分。

除了这些支票之外，Shiftgig 还处理专家的每周付款以及他们的福利:娄说，该公司符合患者保护与平价医疗法案的规定，并投保了适当的工人赔偿政策。

现在有很多基础设施，但并不总是这样。娄告诉我，Shiftgig 成立于 2012 年，是一家纯网络社交网络，由娄与杰夫·皮耶塔(现任首席增长官)和肖恩·卡西(现任首席数据官)共同创立，旨在将服务业的小企业与寻求全职和兼职工作的人联系起来。“我们可以免费使用，”他说，这有助于平台“指数级”扩展(免费有个习惯就是这样。)在巅峰时期，它将 140 万求职者与 3 万多家小企业联系起来。

但在 2013 年末，该公司停止了病毒式增长，因此决定增加收入。他说:“我们与 25 家企业进行了交谈，许多企业希望短期聘用合格且经过审查的候选人和/或工人。”。“通过倾听客户的需求，我们将业务转向按需市场，将企业与寻求短期工作的人联系起来。这一次，我们利用智能手机而不是网站来开发新产品。”

新公式显然对客户和投资者都有很大吸引力。

“Shiftgig 是世界上最大、最好的临时员工创业公司，”FJ 实验室的联合创始人 Fabrice Grinda 在给我们的一封电子邮件中说。“随着雇主和雇员都寻求更大的灵活性，越来越多的工作被‘物化’，这显然是在朝着历史的方向前进。”

“通过赢得利用其技术与小时工联系的企业客户，Shiftgig 团队继续执行其计划，”GGV 资本管理合伙人杰夫·理查兹在一份声明中说。“该公司正在多个垂直领域进行扩张，并证明一个平台可以成为赢家通吃的平台。”
