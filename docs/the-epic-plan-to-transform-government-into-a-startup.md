# 将政府转变为创业公司的宏伟计划

[Josh Thompson](https://web.archive.org/web/20221209234747/https://www.joshfornewyork.com/)

是教育倡导者和纽约市长候选人。

我们有机会改变政治，让它变得干净和没有裙带关系。作为一个公民，这是你的机会，你参与这一进程和创建你的政府的机会。我们可以创造新一代的领导，用千年经济的精神取代坦慕尼大厅的裙带资本主义，让政治变得可及、容易和随需应变。

关键是从特斯拉和优步等世界领先的科技公司吸取经验教训，并将其应用于政府服务:现在是改变政治的时候了。

不要问政府能为你做什么，问问埃隆·马斯克如果在政府会做什么。如果你是一个世界领先的企业家，你会如何设计一个新的公务员制度？每一个领导者都可以利用这些步骤来倾听他们的选民，让伟大的事情发生。

## 第一步:公务员训练营

训练营已经遍布各地来训练程序员。让我们建立新兵训练营来训练公务员。随着训练营的兴起，创业社区蓬勃发展。编码训练营帮助初出茅庐的技术人员花 10 周时间学习新技能，然后他们可以用这些新技能找到新工作。我们可以为初露头角的政客提供公开培训课程，讲授竞选公职的技术细节，以及他们获胜后该做些什么。它目前是一个封闭的过程。它们可以是关于如何组建一个委员会、组织和开展一项活动等的公开研讨会。

例如，在四个月的时间里，致力于培养下一代领导人的新政治组织 [Run for America](https://web.archive.org/web/20221209234747/https://www.runfor.us/) 负责管理合著者乔什·汤普森的团队，并教授如何竞选公职的培训课程。这消除了恐吓，减轻了令人生畏的官僚作风，这种作风经常阻碍竞选活动的开展。像这样的组织开创了招募新一代竞选公职的先例。

## 第二步:社区>顾问

听听你的社区的答案，避开咨询师！我们需要真正的社区驱动的解决方案。政治家应该向风险资本家学习，开放办公时间。创业公司通过倾听潜在客户的意见和进行客户开发来解决问题。风险资本家通过接触初创公司来获得交易。政治家应该让他们自己、他们的委员和他们的工作人员可以亲自和在线访问。他们应该在 Reddit 上主持会议，回复推特和脸书的帖子，与他们的社区互动。

> 最好的解决方案来自于倾听你的决定所影响的人。

例如，在康涅狄格州的布里奇波特，前市长比尔·芬奇(Bill Finch)每周举行一次自带午餐，邀请公众讨论他们认为哪些措施有效，哪些措施无效。这有助于他们有效地引导政策。最好的解决方案来自于倾听你的决定所影响的人。然后，在你的团队中加入当地社区关心你的人。

当汤普森发起他的运动时，许多咨询公司带着他们的固定信息和固定的“解决方案”来了。汤普森整个夏天都在五个区举办社区活动和市政厅活动，他发现纽约人民为关键问题带来了更好的解决方案，并最终成为这场运动的真实声音。

## 第三步:宣布你的队伍

当你启动一家创业公司并向投资者推销时，最重要的部分是你的团队。“初始团队在早期创业公司中的重要性不可低估。天使投资人兼企业家大卫·塞古拉说:“风险资本家和经验丰富的天使投资人都希望创始人能够通过他们组建的初始团队证明他们的可信度——我们认为这对于评估创始人的判断力和建立关系的技能至关重要。”

目前，当人们竞选公职时，他们做出承诺，直到当选后才宣布他们的团队。通常他们甚至在当选前都不会选择自己的团队。让我们改变一下——在竞选之前宣布你的团队，这样人们就知道谁支持你。

现任者通常采用一种指向他们的团队和他们已经完成的事情的概念。新的候选人应该采取这种策略，并在选举前宣布他们的团队。与您的团队一起开展活动；把他们带到你的社区前，并以他们为荣。

## 步骤 4:透明管理

用创业公司的话来说，这就是向董事会公开你将如何处置投资者的资本，然后对结果保持诚实。在政治上，这是仆人式领导。最优秀的初创公司首席执行官会向全世界大声而清晰地做出承诺。

例如，埃隆·马斯克说过一句名言，“我认为有一个反馈回路非常重要，在这个回路中，你会不断思考你已经做了什么，以及如何才能做得更好。”对自己和自己的意图保持透明是创业公司的重要资产。当你筹集资金时，你通常会准备一份财务计划和一套幻灯片。这些是你的计划，你的承诺。然后，你的董事会让你负责。最优秀的创始人会开诚布公地解决问题，直面问题，最终走向成功。

> 我们的国家需要努力工作、致力于国家成功的领导人。

如果你向董事会做出了承诺，但却没有兑现，你可以发展，去做一些不同的事情，或者你可以辞职，去找一个更好的领导(或者你的董事会会替你做)。

政客们通常会兑现一系列承诺，但从来不会兑现。为了有一个成功的反馈循环，你需要过程透明。致力于目标，而不是终身政治。坚守你的承诺，向你的选民展示你能实现你的承诺，他们会用选票回报你。彭博市长有一个竞选承诺指数，跟踪每个承诺的结果，他在很多方面都取得了成功。

彭博市长的前高级传播和媒体顾问 Arick Wierson 评论道:“彭博政府建立在将私营部门的概念，如问责制和衡量，引入政府业务中所有大大小小的事情。”这位前市长发言人接着说，“每个人都记得彭博最大的政治斗争，比如争夺学校控制权、世贸中心遗址的重建以及他的禁烟运动；然而，他对纽约市最大的长期影响很可能是对纽约市 25 万名市政雇员的文化，他们学会了适应我们从彭博带来的现代管理技术。”

当情况变得艰难时，埃隆·马斯克承诺将他的每一分钱都捐给特斯拉。请你们的政治家也这样做。这种治理的透明度和精神气质应该决定个人是否有权竞选连任。就像初创公司的说法一样，如果你未能实现多个目标，你的董事会可以决定你不再经营这家公司。这应该转化为政府。

## 第五步:穿过过道

在创业的世界里，为了给你的客户创造最好的产品，接触不同类型的人和组织是非常重要的。你经常会看到乍一看似乎是竞争对手的公司之间的合作关系。在 API 和平台的世界里，几乎每个人都相互交互。实现目标的系统和方式的多样性是关键。这在政治领域同样重要。

为了服务你的选民，并为当地社区最重要的问题找到真正的解决方案，你必须能够跨越通道。与政治家和各种政治团体的组织合作，创造最好的解决方案，让更多的人参与到地方政治中来。人们希望走向一种更关心问题和政策而不是“政党政治”的政治气候。政客们必须在他们的竞选中反映出这种后党派心态。

不管你在我国最近的选举中忠诚与否，事实是将近 50%的合格登记选民没有投票。现在，我们比以往任何时候都更需要一种新的领导形式，一种打破党派界限、把公民的需求放在第一位、让另一半民众参与进来的领导形式。没有民主党或共和党的方式来收拾垃圾，建造学校和修复我们的道路。

我们生活在一个神奇的国家。我们的国家需要努力工作、致力于国家成功的领导人。这些技术创造了数十亿美元的价值，通过创业为所有人提供了平等的机会，并为推动我们日常生活的创新奠定了基础。让我们把这些最好的技术带给政府，看着我们的国家腾飞。
