# 由于利润率提高，特斯拉降低了 Model X 的基本价格

特斯拉降低了 Model X SUV 的入门级价格，从 8.25 万美元降至 7.95 万美元。据路透社报道，3000 美元的价格下降是电动汽车利润率提高的结果，该公司实现这一目标要归功于制造过程中实现的“效率”。

特斯拉的 Model X 和 Model S 的销售业绩是该公司上周财报期间的主要关注领域，因为分析师预计，Model 3 及其有吸引力的价格可能会降低对特斯拉产品线中更昂贵的现有车辆的需求。据该公司称，事实证明，在特斯拉 7 月底推出 splash Model 3 之后，对 Model S 和 Model X 的需求都有所增长。

在特斯拉本周的收益电话会议上，马斯克表示，虽然很难预测未来，但他们看到 Model 3 的推出提振了需求，并且没有理由认为它不会继续下去。如果 Model 3 最终为特斯拉的高端汽车带来持久的光环效应，那基本上是该汽车制造商可能要求的最好结果，特别是因为它还需要一段时间才能以足够的产量满足 Model 3 的现有需求。

X 75D 车型的降价应该有助于进一步刺激需求。然而，下一季度的结果将是真正值得关注的，因为我们将有更多的时间和距离 Model 3 的大新闻发布会，来评估光环效应是否会持续下去。
