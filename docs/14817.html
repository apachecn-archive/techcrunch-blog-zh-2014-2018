谷歌取消了 Pixel 的耳机插孔，为无边框手机奠定了基础

在迈向后耳机插孔世界的过程中，又一张多米诺骨牌倒下了。今天，谷歌成为最新一家放弃曾经无处不在的手机端口的公司，尽管它几乎没有花任何时间致力于自己勇敢的举动。在对移动领域进行快速调查后，放弃插孔的驱动动机非常明显:像素正在转向边到边显示。

活动结束后，谷歌产品总监马里奥·奎罗斯(Mario Queiroz)告诉 TechCrunch:“(放弃千斤顶的)主要原因是为未来建立一条机械设计道路。“我们希望显示屏越来越接近边缘。我们的团队说，如果我们要进行转变，那就让我们尽快实现，而不是推迟。去年可能太早了。现在市场上有更多的手机。”

根据奎罗斯的说法，这一举动“绝对”是该公司决定推出自己的 AirPod 竞争对手 Pixel Buds 的驱动因素。该公司基本上通过放弃千斤顶在市场上创造了一个机会，并且它能够直接介入硬件，这也有助于使 Assistant 变得更加普遍，同时将 Translate 变成现实世界版本的道格拉斯·亚当斯 Babelfish 。

边框不可避免的死亡也是该公司为更大的 Pixel XL 转向 P-有机发光二极管的一大驱动力。“这是一项新技术，但我们真的对能够将屏幕包裹在侧面的可能性感到兴奋，”产品管理副总裁 Brian Rakowski 解释说。我们将它设计成一种握起来仍然很舒服的方式，你不一定要拿着显示器。它让你能够更靠近一点，所以你可以开始看到曲线，而不是更像一个块状的手机。”

当然，快速浏览一下新的像素就可以清楚地看到，谷歌这次没有选择这个方向，不像苹果、三星和 LG 这样的公司。那么，为什么不全力以赴迎接无边框的未来呢？这一举措在一定程度上被包含在屏幕两侧的前置扬声器的决定所否决。

根据奎罗斯的说法，该公司也只是想给用户一个拿手机的地方。“我非常努力争取的一件事是边带，这样你就不会觉得你在抓着显示器，”他解释道。凭借 Pixel 的 2 U11 灵感活动边缘功能，这也意味着有更多的地方可以挤压。
