# Slack 的联合创始人获得了年度创始人奖

如果你的团队在线合作，你几乎肯定听说过 Slack。它于 2013 年年中推出，很快成为团队沟通的顶级品牌之一。

这一成功让 Slack 的联合创始人斯图尔特·巴特菲尔德、埃里克·科斯特洛、卡尔·亨德森和塞尔吉·莫拉乔夫获得了年度创始人奖。

Slack 称自己为“21 世纪的团队沟通”，这很符合它。与 Hipchat 等竞争对手不同，它的核心是基本的聊天室，但凭借坚如磐石的 Mac/iOS/Android 客户端、出色的 API 和多层可定制性，它很快成为 Rdio、Heroku、Soundcloud 等公司的首选工具。

和之前的许多创业公司一样，Slack 是从一个不太成功的项目中诞生的。在构建现已不存在的 MMO *故障*时，开发团队发现现有的通信工具没有一个能满足他们的需求。他们建立了自己的……随着 Glitch 关闭大门(引用“有限的观众吸引力”)，Slack 涌现出来。一年多后，Slack 在[以 11.2 亿美元的估值筹集了 1.2 亿美元。](https://web.archive.org/web/20230116112655/https://techcrunch.com/2014/10/31/slack-confirms-120m-fundraise-led-by-google-ventures-and-kpcb-at-1-12b-valuation/)

在创建 Slack 之前，这四位联合创始人在创建 Flickr 的过程中都扮演了重要角色。

这一类别的其他候选人包括 Stripe 联合创始人约翰和帕特里克·科利森，Theranos 创始人伊丽莎白·霍姆斯，WhatsApp 联合创始人布莱恩·阿克顿和简·库姆，以及 Zendesk 联合创始人米克尔·斯万，亚历山大·阿加西普尔和莫滕·普里姆达尔。
