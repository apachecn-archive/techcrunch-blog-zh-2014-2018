# Visa 对信用卡处理初创公司 Marqeta TechCrunch 进行战略投资

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/07/26/visa-marqeta-25m/>

自 2010 年以来， [Marqeta](https://web.archive.org/web/20221207142843/https://www.marqeta.com/) 已经悄然确立了自己的地位，成为那些希望向员工、合作伙伴和客户等发行虚拟和实体借记卡和信用卡的企业的基础技术提供商。在寻求国际扩张并在全球范围内签约客户的同时，该公司获得了由支付巨头 Visa 牵头的 2500 万美元战略投资。

除非你是金融科技极客，否则你可能从未听说过 Marqeta。但它是那些一直在幕后使企业能够处理信用卡和借记卡支付以及发行自己的信用卡和借记卡的公司之一。

通过采用以开发人员为中心的方法来处理卡，Marqeta 建立了一个技术堆栈，第三方可以使用它来解决各种问题。因此，Marqeta 已经能够签约许多你*听说过的公司作为客户——包括 DoorDash、Instacart、Kabbage 和 Square 等公司。*

当像 DoorDash 或 Instacart 这样的按需公司希望向其承包商发放借记卡，以支付他们代表用户购买的商品时，它会使用 Marqeta。这样做可以让这些企业进行与商品成本相匹配的“即时”支付，这样承包商就不会拿着预付费借记卡到处跑了。

与此同时，像 Affirm 这样的公司可能会使用 Marqeta 来创建虚拟卡号，以便于在线供应商的销售点进行支付。如果你够大方，你可以用 Marqeta 发行一张品牌借记卡，就像这样:

该公司还使市场平台能够支付自雇员工的资金，并允许其他企业建立虚拟和实体的公司费用卡。也就是说，Marqeta 通过构建一个灵活的技术堆栈，使信用卡和借记卡处理民主化，以多种方式帮助了其他公司。

迄今为止，它也是在很少融资的情况下做到这一点的。自 2010 年成立以来，总部位于奥克兰的 Marqeta 在与 Visa 建立战略关系之前，仅筹集了 4600 万美元的外部资金。在那段时间里，由于“非常可观的收入”，Marqeta 已经发展到 100 多名员工，首席执行官杰森·加德纳(Jason Gardner)在电话交谈中告诉我。

该公司或许可以继续用其收入为运营提供资金，但与 Visa 合作的机会太多了，不能错过。虽然 Visa 领投了这一轮，但它也包括了之前 Marqeta 投资者 Commerce Ventures、83 North、Granite Ventures 和 CommerzVentures GmbH 的参与，以及新投资者 CreditEase，世界上最大的替代贷款机构。

Visa 不经常投资，但它在支持不断增长的金融科技公司方面有着良好的记录，如 Chain、DocuSign、Klarna、LoopPay(被三星收购)、Square 和 Stripe。当它投资时，Visa 这样做的目的是打开大门，帮助公司显著增长。

Gardner 表示，由于与 Marqeta 有关，Visa 准备在三个重要领域帮助该公司:首先，这家全球支付公司将通过向 Visa 合作伙伴和客户推广其以开发者为中心的产品，立即为 Marqeta 提供更大的分销。鉴于其全球足迹，Visa 也将能够帮助 Marqeta 进行国际扩张。最后，Gardner 看到了 Visa 和 Marqeta 从技术交流中获益的机会。

“我们在 Marqeta 这里做 Visa 可以使用的东西，Visa 也在做我们可以使用的东西，”他告诉我。不过，最重要的是，融资和战略伙伴关系是为了能够利用一个机会，进入一个高速增长的阶段。

“如果你看看全球发生的交易数量，天空是极限，”加德纳说。“他们希望建立在现代平台上，通过这种合作关系，我们可以实现这一点。”
