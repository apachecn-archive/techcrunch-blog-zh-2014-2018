# 这个 BB-8 完全由乐高零件制成，实际上像 BB-8 一样滚动

[https://web.archive.org/web/20221207012705if_/https://www.youtube.com/embed/QKCkOAETxBE?feature=oembed](https://web.archive.org/web/20221207012705if_/https://www.youtube.com/embed/QKCkOAETxBE?feature=oembed)

准备好你的半定期提醒，有些人只是方式，*方式*太聪明了？

这些人设法做了一个 BB-8 模型…完全用乐高零件做的。它真的会滚动。它的头部实际上停留在顶部，就像 BB-8 的头部一样。

这里的一个关键点是，它的创造者只声称它是由 100%真正的乐高*部件*制成的，而不是 100%的乐高*积木*。由于一把长方形的乐高积木通常不会滚动得太好，建造者重新利用了早已退役的《星球大战》飞梭赛车布景中的圆形塔图因模型。添加一些从其他套件中剥离的零件、磁铁和轮子，这款功能惊人的 BB-8 就组装在一起了。

他们甚至为 BB-8 建造了一个小基站，当你转动轮子时，它可以在适当的位置滚动。

嫉妒？想要一个自己的吗？好消息！创作者们在乐高创意上为它设立了一个项目页面，在这里，乐高承诺调查任何达到 10，000 名支持者的项目。该项目上线不到一周，就已经有超过 1000 名粉丝，它还有一年多的时间来吸引更多粉丝。我猜不会花那么长时间。

时机合适的话(也就是在圣诞节或第八集上映前后)，你就会知道这个东西会卖得很好。
