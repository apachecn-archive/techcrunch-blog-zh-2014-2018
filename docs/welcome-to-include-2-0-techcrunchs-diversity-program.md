# 欢迎加入 2.0，TechCrunch 的多元化计划

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/08/27/welcome-to-include-2-0-techcrunchs-diversity-program/>

2014 年 6 月，TechCrunch [推出了 Include](https://web.archive.org/web/20221005191528/https://beta.techcrunch.com/2014/06/03/announcing-include-techcrunchs-diversity-partner-program/) ，这是一项旨在引导 TechCrunch 资源提高创业社区多样性的计划。

第一年，我们的方法很简单。头条新闻是一笔 5 万美元的赠款，用于支持一个与 Include 的使命保持一致的组织。去年的获奖者是[黑人女孩代码](https://web.archive.org/web/20221005191528/http://www.blackgirlscode.org/)，这是一个由金伯利·布莱恩特运营的非常棒的非营利组织，通过研讨会和课外项目向有色人种女孩介绍技术。Include 还与其他几个团体合作，包括 Mission Bit、拉丁裔创业联盟和奥克兰数字艺术和扫盲中心，为成员提供免费参加 TechCrunch 活动的机会。

在加入 Include 的第一年，我们学到了很多东西，并受益于结识了几十名申请资助的人以及 [Include 董事会](https://web.archive.org/web/20221005191528/https://beta.techcrunch.com/2014/06/03/announcing-include-techcrunchs-diversity-partner-program/)和我们的非营利合作伙伴。外卖？为了让我们的活动和社区更具包容性，TechCrunch 还有很多工作要做。

今天，我们将宣布 Include 2.0，这是一个扩展的系列计划，将深化我们在包容性方面的努力，并将其扩展到硅谷以外。以下是主要举措:

1) **包括拨款。再次向致力于增加科技多样性的非营利组织开放申请。TechCrunch 将再次分发 5 万美元奖金，但在获胜者之间分配奖金，并将积极征集来自伦敦和纽约以及硅谷的申请人，这与举办破坏性活动的三个城市一致。申请将于 9 月初开始。**

 **2) **包括办公时间。最好的介绍是热情的介绍，但如果你没有合适的关系网，这在风险投资或媒体领域并不总是触手可及。TechCrunch 将通过为符合条件的创始人安排与风投和 TechCrunch 作家的办公时间，来帮助促进这些热情的介绍。**

 **3) **包含事件。** Include 将举办多场活动，旨在为参加 Disrupt 或其他 TechCrunch 活动的人数不足的群体建立社区。继今年春天在纽约成功举办“科技女性”( Women in Tech)活动之后，今年晚些时候，Disrupt 将在旧金山和伦敦举办类似的聚会。

4) **包含在 TC 上。** Include 将在 TechCrunch 上有一个永久的家。该页面将作为个人和非营利组织寻找资源的资源，包括 TechCrunch 和其他资源，这将有助于他们推动自己的想法和职业发展。它还将跟踪我们自己在 Include 2.0 方面的进展。

上述举措的具体公告将在未来几周和几个月内发布，但现在，请关注博客。如果你对赞助机会感兴趣，想要合作或为团队提供反馈，请发送电子邮件至 include@beta.techcrunch.com。****
