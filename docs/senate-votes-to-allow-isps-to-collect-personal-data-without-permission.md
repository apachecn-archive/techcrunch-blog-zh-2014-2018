# 参议院投票允许互联网服务提供商未经许可收集个人数据

联邦通信委员会去年制定的宽带隐私条例(T1)和 T2 昨晚激烈辩论的宽带隐私条例(T3)在参议院今早投票废除后岌岌可危。除其他外，这些规则要求互联网服务提供商获得消费者的许可，才能使用他们通过服务获得的某些敏感数据，如浏览历史记录。

听起来是个坏主意，对吧？确实是。[昨晚](https://web.archive.org/web/20230326172931/https://techcrunch.com/2017/03/22/senate-debates-legislative-rollback-of-fccs-broadband-privacy-rules/)我在一篇文章中详细解释了原因，很多参议员，包括马萨诸塞州的埃德·马基，他领导了 1996 年电信法案的制定，强烈反对这项决议。

参议院以 50 票对 48 票通过了第 34 号法案，该法案将删除这些规定，并根据国会审查法案的授权，阻止类似的规定被颁布。现在，该法案正等待众议院批准。

FCC 专员 Mignon Clyburn 和 FTC 专员 Terrell McSweeny 在一份联合声明中表示:“如果总统签署，这项法律将废除 FCC 广泛支持的宽带隐私框架，并取消有线电视和宽带提供商在出售敏感的个人信息之前向客户提供选择的要求。”

Nathan White 是 Access Now 的高级立法经理，他说:“这项决议是为了大公司的利益，而不是普通人的权利和公民自由。众议院现在必须站出来支持消费者，反对 CRA 抛弃互联网隐私保护的决议。”
