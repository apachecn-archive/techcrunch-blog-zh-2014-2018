# Minifree Libreboot T400 和 freedom 一样免费

Libreboot T400 看起来不怎么样。它基本上是一个翻新的联想 Thinkpad，带有传统的联想/IBM 指针 nubbin 和一个小触摸板。这是一台普通的黑色笔记本电脑，就像分配给旅途中的小隔间战士的任何一台手提电脑一样熟悉。但是，在引擎盖下，你有一台为自由而战的机器。

T400 运行自由开放的 [BIOS](https://web.archive.org/web/20230326104136/https://libreboot.org/) 和[trisqel GNU/Linux](https://web.archive.org/web/20230326104136/https://trisquel.info/)操作系统 Libreboot。这两个工具都应该使 Libreboot T400 尽可能地安全，不会被篡改。"你的 Libreboot T400 服从你，而不是其他人！"写它的创造者，似乎也是如此。

它是如何工作的？你应该花 300 美元买一台安装了 Linux 的翻新 Thinkpad 吗？这取决于你想做什么。我测试的模型是低端的，有足够的速度和性能，但 Trisquel 倾向于停滞不前，而安全浏览器，“一个无品牌的基于 Mozilla 的浏览器，从不推荐非自由软件”，有点太锁定了。我能够解决我遇到的一些问题，但这绝对不适合胆小的人。

也就是说，你将得到一台几乎完全打开的计算机。这款 14.1 英寸的机器运行英特尔酷睿 2 双核 P8400 处理器，启动内存为 4GB，硬盘空间为 160GB。这大约需要 257 美元外加运费，包括一块电池和一个美国充电器。

一旦你有了 T400，你基本上运行一个完全干净的机器。它运行一个免费的操作系统，配有开放的驱动程序和应用程序，Libreboot 确保你的机器上没有被锁定的软件。你可以很容易地在你自己的计算机上重新创建这个包，但是我怀疑你，像我一样，最终会遇到一个不能完全用自由软件解决的问题。因此有动力让 Minifree 为你工作。

如果你是隐私、安全和开放标准的斗士，那么这款笔记本电脑正适合你。谢天谢地，它出奇的便宜，而且相当结实，所以你不仅可以把它贴在男人身上，还可以买几个这样的东西，在必要的时候扔给他。

桌面上普通 Linux 的时代——而不是像这样以安全、自由设备的形式——可能还会到来。虽然现在安装一个 Linux 实例很简单(也很有趣),但我怀疑是否有人会在日常使用的笔记本电脑上直接安装。但是你可以用不到一部手机的价格，使用 T400 之类的东西，当你的计算体验涉及到你不支持(许多)公司利益时，你会感到安全可靠。无论从哪方面来说，它都不是一台完美的笔记本电脑，但如果你在寻找除了你自己没有人能控制的东西，它就是你想要的东西。
