# 社交音乐初创公司 Smule 为亚洲扩张筹集了 5400 万美元

[Smule](https://web.archive.org/web/20230404165324/http://smule.com/) ，星之制造者！卡拉 ok 和其他社交音乐应用,[昨日宣布，在中国公司腾讯的带领下，该公司已经募集了 5400 万美元。](https://web.archive.org/web/20230404165324/http://blog.smule.com/smule-fuels-more-music-in-the-world-with-54-million-capital-raise-led-by-tencent)

亚当斯街合作伙伴、贝塞尔风险投资公司、洪门、富兰克林邓普顿和沙斯塔风险投资公司也参与了融资。路透社[报道](https://web.archive.org/web/20230404165324/http://www.reuters.com/article/us-fundraising-smule-idUSKBN1840ZO)本轮估值为 6.04 亿美元，成立 Smule 是为了在亚洲扩张，并有可能上市。

成立于 2008 年的 Smule 应用程序允许用户与他人分享不同类型的音乐表演，该公司表示，它现在每月有 5200 万活跃用户。首席执行官杰夫·史密斯表示，他的愿景是“围绕音乐建立一个社交网络”，同时帮助推广音乐。

与此同时，腾讯是广受欢迎的微信应用的制造商，它也一直在投资于[音乐](https://web.archive.org/web/20230404165324/https://www.wsj.com/articles/tencent-to-buy-majority-stake-in-china-music-corp-creating-streaming-giant-1468470851)和[用户生成的内容](https://web.archive.org/web/20230404165324/https://techcrunch.com/2017/01/04/tencent-ookbee-u-thailand/)。
