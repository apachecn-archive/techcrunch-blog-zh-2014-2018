# WhatsApp 推出 Mac 和 Windows 桌面应用

[WhatsApp](https://web.archive.org/web/20230210170724/https://www.whatsapp.com/),[声称拥有 10 亿用户的脸书旗下的消息服务](https://web.archive.org/web/20230210170724/https://blog.whatsapp.com/616/One-billion),[推出了 Mac 和 Windows 桌面客户端](https://web.archive.org/web/20230210170724/https://www.whatsapp.com/download/)。此次发布是在 WhatsApp 发布其第一款网络应用大约 15 个月后。

已经在网络浏览器中使用 WhatsApp 的人会发现，该软件并没有太大的不同。公司[在一份声明](https://web.archive.org/web/20230210170724/https://blog.whatsapp.com/10000621/Introducing-WhatsApps-desktop-app)中表示，“我们的桌面应用只是你手机的延伸”，所有信息在设备间同步。

WhatsApp[在印度、巴西和南非等国家的成功](https://web.archive.org/web/20230210170724/http://www.wired.com/2016/01/whatsapp-is-nearing-a-billion-users-now-its-time-to-find-the-money/)当然是由智能手机在这些市场的高普及率推动的，但为高级用户——尤其是依赖 WhatsApp 进行工作交流的人——提供桌面选项有助于它与其他消息服务竞争，如 iMessenger、微信和 Skype。

新桌面客户端的推出，以及默认的端到端加密，正值 WhatsApp 测试 B2C 账户之际，这将为其在放弃 99 美分的年费后带来急需的新收入来源。
