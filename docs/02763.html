自由剪辑打破了咖啡荚 DRM，因为 Java 想要自由，伙计

最新的 Keurig K-Cup 机器有一个肮脏的秘密:内置在许多人都知道和喜欢的小研磨咖啡盒中，是一种邪恶的 DRM 形式。一种只在真正的杯子上发现的特殊墨水(因使用而损坏)确保你只能使用“真正的”K 杯，从而防止你制作自己的盗版杯子。

但是自由只是坚持的另一个词，Gourmet-Coffee.com 的所有者罗杰斯家族公司正在用自由剪辑进行反击。

他们的自由夹适合你的 Keurig，欺骗 DRM 传感器认为每个杯子都是授权的。这家人写道:

￼It 通过视觉识别盖子上的特殊墨水来做到这一点。任何没有这种“特殊”墨水的杯子都会被机器拒绝，从而确保了 Keurig 的市场主导地位。当其他公司正在迅速采用这种特殊的墨水到他们的杯子上时，我们罗杰斯家族公司相信你选择任何选项的权利是必要的。

这就是为什么我们为 Keurig 2.0 啤酒厂开发了这款易于安装的“自由夹”。只需将夹子放入您的新啤酒杯，它就会将所有 K 杯型豆荚视为“授权 K 杯”。这个夹子是我们送给你的礼物。现在去自由地酿造吧。

用自由酿造！Keurig 在他们的机器上强制使用 DRM 是愚蠢的，这样的漏洞是对他们限制用户决定的明显反应。这些豆荚本来就非常浪费，几年来一直是黑客的目标。视频已经出现告诉你如何重新装满自己的杯子，现在像 Freedom Clip 这样的工具可以确保严厉的咖啡数字版权管理不再存在。

K-Cup 专利于 2012 年到期，让 Keurig 陷入困境。该公司没有拥抱制造商并提供低价许可证，而是决定像对待罪犯一样对待客户。这从来都行不通，尤其是当你的顾客都在喝星巴克的时候，更是难以实施。你可以从 Gourmet-Coffee.com 的人那里免费得到你的自由剪辑，再一次，把它粘在那个人身上。
