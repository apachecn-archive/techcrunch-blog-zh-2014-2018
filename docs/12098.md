# Stripe 收购了独立黑客，这是一个面向企业家的知识共享社区

![](img/80d65a8e1984822112b4b8ac9a95e40c.png)

[Stripe](https://web.archive.org/web/20221006215216/https://stripe.com/) 收购了 [Indie Hackers](https://web.archive.org/web/20221006215216/https://www.indiehackers.com/) ，这是一个专注于帮助创业者在保持独立性的同时实现盈利的网站和社区。收购的消息通过独立黑客创始人考特兰·艾伦的博客文章发布[，并很快被](https://web.archive.org/web/20221006215216/https://www.indiehackers.com/blog/acquired-by-stripe) [Stripe 的推特账户](https://web.archive.org/web/20221006215216/https://twitter.com/stripe/status/851843213161668608)证实。

艾伦写道:“去年 7 月，当我开始创建独立黑客时，没有多少地方能让盈利的在线业务受到关注。我想创建一个社区，成功的创始人可以在这里分享他们有价值的故事和见解，有抱负的企业家可以在这里获得灵感和建议。”

但是为什么 Stripe 想要独立黑客这样的东西呢？Stripe 的联合创始人帕特里克·科利森在一篇黑客新闻中谈到了这个话题:

> 我是 Stripe 的帕特里克。我想快速插话强调，我们收购独立黑客的目标只是为了确保网站尽可能成功。我们希望的优势是更多的公司开始创业，并且更加成功。我们已经看到很大一部分新的互联网公司选择 Stripe 我们主要希望独立黑客能帮助我们增加总数，而不是增加我们的份额。(我们的产品要做后期。)

自不到一年前成立以来，独立黑客制作了几十个采访独立创始人的节目(“独立”通常指的是钱来自客户，而不是风投或雇主),这些人想出了如何每月获得数百到数万美元的稳定收入。它还主持一个类似的播客。

目前还没有关于交易条款的消息——但艾伦表示，独立黑客将在 Stripe 下照常运营，并指出他的新头衔实际上是“Stripe 的独立黑客”
