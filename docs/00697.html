Twitter 现在支持在线和移动 gif 动画

原文：https://web.archive.org/web/https://techcrunch.com/2014/06/18/twitter-now-supports-animated-gifs-online-and-on-mobile/

嗯，终于。Twitter 今天宣布了对其社交网络的一个微小但超级棒的改变:它现在将支持一直流行的动画 GIF 格式。该公司表示，这一变化从今天起在 Twitter.com 网站、安卓和 iPhone 上生效。

该公司通过 tweet (当然)宣布了这一消息，展示了动画 GIF 的样子。

那张 GIF 本身就在取笑正在进行的关于“GIF”这个词的发音的争论。有些人，比如这种格式的发明者史蒂夫·怀特，说它应该像花生酱一样发音为“JIF”。但是我们中的大部分人习惯于用一个硬的“G”来表示“GIF”，不喜欢改变。

在 Twitter 的演示 GIF 中，有人开始键入带有“J”的单词“GIF”，作为对知情人的一个内部玩笑。

该公司似乎也在向其 API 推出 GIF 支持，这将允许第三方开发者在其客户端应用程序中提供类似的支持。

如今，随着动画 gif 格式越来越受欢迎，包括 Tumblr 和甚至 Pinterest 在内的许多社交网络都增加了对动画 gif 的支持。然而，脸书和推特是很大的例外。(不过，也有变通办法，比如 Giphy 的网址缩写器。)

值得注意的是，脸书做出了一个非常明智的决定，不支持 gif，因为该公司认为这会使新闻订阅过于混乱，而且它已经在向 memes 宣战。相反，目前只有原生上传的脸书视频、Instagram 视频和付费视频广告可以在脸书自动播放。Twitter 支持 gif 的举动是否会影响脸书的决定还有待观察。

随着这一变化在整个网站上线，每个人都开始试水，Twitter 也有可能被 gif 淹没。但从 Twitter 成为一个更加以富媒体为中心的服务的更大目标来看，这也是有意义的，因为它不再仅以 140 个字符的文本更新而闻名。
