# 打字写手把你的 iPad 变成打字机(不，它不是汤姆·汉克斯创造的)

几周前，影星汤姆·汉克斯的打字机应用程序 Hanx Writer[蹿升至苹果应用商店](https://web.archive.org/web/20221006173421/https://beta.techcrunch.com/2014/08/18/tom-hanks-typewriter-app-shoots-to-the-top-of-the-app-store/)的首位。我猜一定有什么在酝酿中，因为[不是最近推出的唯一一款 iPad 输入打字机的应用](https://web.archive.org/web/20221006173421/http://www.latimes.com/books/jacketcopy/la-et-jc-tale-of-the-dueling-typewriter-apps-20140826-story.html)。

还有[打字作家](https://web.archive.org/web/20221006173421/https://itunes.apple.com/us/app/typing-writer/id892249870?mt=8)，这是一款由文学网站 [The Rumpus](https://web.archive.org/web/20221006173421/http://www.therumpus.net/) 的创始人斯蒂芬·埃利奥特(Stephen Elliott)与伊莱·霍洛维茨(Eli Horowitz)、克里斯·英(Chris Ying)和拉塞尔·奎因(Russell Quinn)共同开发的应用。

几周前，埃利奥特在 TechCrunch 纽约办公室停下来展示了这款应用程序——我还没有使用过这两款应用程序，但从所写的内容来看，Hanx Writer 应用程序强调对打字机声音等事物的怀旧，而 Typing Writer 可能更像是一款实用的写作工具，强调在写作时向前移动，而不是不断后退和修补。(为此，它还附有吉姆·谢泼德、梅丽莎·费博斯、艾米·布里尔和里克·穆迪等作家的初稿。)

作为披露，我还应该提到，艾略特是我的一位老写作老师——就在我们拍摄这部视频之后，我们决定合作拍摄一部短片。

反正打字写手 1.99 美元[你可以在这里](https://web.archive.org/web/20221006173421/https://itunes.apple.com/us/app/typing-writer/id892249870?mt=8)下载应用。
