# 当众筹失败时，支持者就无路可走了 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/11/19/when-crowdfunding-fails-the-backers-are-left-with-no-way-out/>

几年前[一个有进取心的程序员](https://web.archive.org/web/20221210053217/https://beta.techcrunch.com/2012/05/24/failure-is-not-an-option-why-kickstarter-hides-failed-projects/)挖掘了 [Kickstarter 的](https://web.archive.org/web/20221210053217/https://beta.techcrunch.com/tag/kickstarter)数据来计算到底有多少失败的项目。然而，由于编码中的一个技巧，没有一个出现。这个系统被描述为将最好的项目带到前台的努力，当 Kickstarter 很小并且项目通常很小时，它工作得很好。然而现在，失败是巨大的，潜在的支持者灾难只会越来越大。

以[最冷的冷却器](https://web.archive.org/web/20221210053217/https://www.kickstarter.com/projects/ryangrepper/coolest-cooler-21st-century-cooler-thats-actually/description)为例。这个 chimera——一个内置搅拌机的冷却器(由于系统本身的质量问题，我们自己的 Matt Burns 没有审查这个项目)——最初筹集了 1300 万美元，但由于本应供应搅拌机电机的工厂罢工而未能发货。该公司随后开始在亚马逊上销售该产品，尽管支持者在等待他们的重型冷却器，并在商业网站上发布尖刻的评论。

￼This 冷却器看起来太棒了！我也很高兴有一天我会收到一个，还有盘子、刀子和砧板！我被告知这个冷却器的所有属性；蓝牙扬声器，地面冲击的车轮，以及它的平滑外观即将问世！今天早些时候，基于 kicksatrter 上的通知，我提醒人们不要购买这件物品，我想知道他们付钱给谁给了这么多虚假的五星评论！！我是靠山，不能等机会被拧！！！！我不知道怎么会有人对他们刚刚订购的产品发表评论，但最酷似乎是这方面的冠军！！我的冷藏箱呢，瑞恩！！？？

欧洲加入了众筹失败俱乐部，推出了 Zano T8，这是一款小型无人机，本应与其他人协同飞行。这个项目筹集了 300 万美元却破产了。“我们对 Zano 项目的结果非常失望，我们想借此机会感谢在这个困难时期支持我们的所有人，特别是我们忠诚的员工，他们的承诺超出了所有人的预期，”该公司写道。粉丝们则创作了一些优秀的恶搞视频。

再加上[、价值 100 万美元的 dragonfly 失败了](https://web.archive.org/web/20221210053217/https://beta.techcrunch.com/2015/11/07/another-1-million-crowdfunded-gadget-company-collapses/)和[从未发货的低成本 3D 打印机](https://web.archive.org/web/20221210053217/https://beta.techcrunch.com/2015/10/14/a-1-5m-kickstarter-project-fails-leaving-most-backers-without-their-3d-printer/#.yismjja:jfCq)，你就有了一连串的大额崩溃。

普遍的反应是“买家当心”，这总是正确的。通过保证你不预购，这是一个常见的误解，而且羽翼未丰的硬件制造商往往没有意识到与创建一个经验不足的复杂产品相关的危险。

硬件很硬。与软件或创意项目不同，这个世界对硬件制造商并不友好，他们最终会面临巨大的 R&D 成本、制造合同和运输费用。虽然硬件创业公司复兴仍在全力进行——成功项目的例子远多于失败项目——但很少有人意识到为什么微硬件制造业花了这么长时间才起步。几十年来，唯一有技术和资金生产硬件的公司是像三星、LG 和索尼这样的老牌公司，由于笨拙的品牌稀释，它们不再是曾经的强者。再加上即时制造和小型合同建筑商，你就有了制造混乱和与之相关的现任失败的方法。

最终，我们无法得到我们的冷却器、我们的蜻蜓和我们的 3D 打印机，这太糟糕了。但是，当过于乐观的创始人遇到过于兴奋的支持者时，就会酿成灾难。然而，作为一个已经在这个领域工作了十年的人，我很高兴这些产品存在于第一位，包括所有的缺点。所以，购买者要小心，因为没有硬件项目，我们都只能[回到一个油漆干燥的薄膜](https://web.archive.org/web/20221210053217/https://www.kickstarter.com/projects/charlielyne/make-the-censors-watch-paint-drying?ref=video)——这个项目虽然愚蠢，但更符合我们真正应该从 Kickstarter 上期待的，不是天上掉馅饼的硬件项目。选择权在你。
