微软推出人工智能和入门级软件开发课程

微软今天在其面向开发者的在线教育项目中推出了两门新课程:一门入门级软件开发课程和一门面向更高级开发者的人工智能课程，这些开发者希望扩展他们的机器学习知识。

众所周知，没有足够的数据科学家和机器学习开发人员来满足当前的需求。因此，许多大公司开始向现有员工教授这些学科的基础知识就不足为奇了；从今天开始，任何人都可以参加微软首次为其员工开发的人工智能课程。

微软人工智能专业程序在 edX.org 是免费的，尽管你也可以选择付费购买证书。每个课程为期三个月，在季度初开始。不出所料，这里有点侧重于 Azure 和微软的认知服务(你需要一个 Azure 账户)，但除此之外，课程与你运行的操作系统无关。

整个项目由 10 门课程组成，从人工智能简介(T1)和数据科学的 Python(T3)到人工智能开发人员的职业道德(T5)课程，以及大量与培训模型相关的实践工作。10 门必修课中的大部分需要大约 8 到 16 个小时来完成。

除了人工智能课程，微软今天还宣布了一个面向入门级软件开发人员的类似项目。这个基于 edX 的项目包括 13 门课程，教授学生软件开发的基础知识，重点是 Python 和 JavaScript。然而，可能比仅仅教授这些语言更重要的是，该课程还着眼于基础知识，如数据结构，以及如何使用 GitHub 和其他工具专业地编写代码。

这两个新课程加入了微软所谓的“专业项目”中越来越多的类似项目(为什么他们不干脆称之为微软学院(Microsoft Academy)超出了我的能力，但我不是营销人员……)。这些现有课程从前端开发课程到面向云管理员的课程和面向 IT 支持专业人员的课程。
