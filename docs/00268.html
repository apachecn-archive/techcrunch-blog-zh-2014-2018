Kantar: Windows Phone在Android价格竞争中“口吃”, iPhone 5S是苹果的缓冲器

原文：https://web.archive.org/web/http://techcrunch.com/2014/04/27/kantar-windows-phone-stutters-amid-android-price-competition-iphone-5s-is-apples-buffer/?ncid=rss

上周五，微软终于控制了诺基亚设备业务的新主人，但它作为移动硬件制造商的新地位带来的挑战不亚于诺基亚在过去几年中遇到的挑战。

根据WPP市场研究部门 Kantar Worldpanel ComTech 的最新12周数据，在截至3月底的12周内，Windows Phone在欧洲五大市场(英国、法国、西班牙、意大利和德国)的智能手机销量中占8.1%，Android占70.7%，iOS占19.2%。

欧洲似乎是Windows Phone目前最好的市场。在美国，Windows Phone占销售额的5.3%，而在澳大利亚，这一比例不到6%；在中国，这一比例为1%；而在日本，不到1%。

这是怎么回事？简言之，就欧洲、美国和亚洲主要市场的智能手机销量而言，运行诺基亚和其他设备的Windows Phone操作系统仍远远排在第三位。

这有双重效果。首先，在规模经济的游戏中，微软会发现它更具挑战性。其次，在消费者和开发者之间的思想共享游戏中，这意味着Windows Phone设备对许多人来说仍然不是必需品，无论是在设备所有权方面，还是在构建新应用和其他服务的平台方面。

Windows Phone面临的问题是Android手机价格和销量的双重压力，这两个因素都在很大程度上将Windows Phone挡在智能手机热潮之外。这一繁荣使得英国等市场的智能手机普及率达到71%，智能手机占移动销售的88%。对于那些与消费者和运营商有联系的人来说，这是一个机会。

Kantar董事Dominic Sunnebo写道，Windows面临的主要挑战是，作为Windows Phone的主要和主导OEM厂商，诺基亚在很大程度上将其反击战略建立在以入门级手机吸引智能手机新用户的基础上。问题是这些设备还没有足够好地与Android阵营竞争。

他写道:“Windows今年开局不利，因为其入门级诺基亚机型面临来自低端摩托罗拉、LG和三星安卓智能手机的激烈竞争。”

甚至在被微软收购之前，诺基亚的设备命运就已经与雷德蒙的命运密不可分。

“诺基亚现在真的是Windows的主流，所以你看到的任何Windows趋势都是由诺基亚推动的，”Sunnebo在一次采访中告诉我。“基本上，诺基亚开始通过其入门级Lumia 520取得一些真正的进展，特别是在吸引首次购买智能手机的用户和从Symbian过渡的用户方面……可以说，它拥有最好的入门级设备。

“现在，我们开始看到在市场的这一端出现了一些真正引人注目的竞争，来自摩托罗拉(Moto G)和三星(大幅打折的S3和迷你机型)等公司正在蚕食诺基亚的黄金领地。”

他还指出，崭露头角的中国品牌向本土市场以外扩张的举动——想想小米、Oppo和一加——现在也将使突破性挑战变得更加困难。

“消费者越来越意识到，高端和低端之间的差别变得非常细微，”他表示。“这可能会给诺基亚带来优势(它制造出很棒的入门级设备)，但它肯定不会是唯一一家以中低价位推出优秀规格设备的公司。”

有趣的是，苹果继续以自己的高端方式对抗安卓巨头。Sunnebo表示，其在欧洲、日本和澳大利亚的销售出现反弹，主要是因为高端产品5S的“强劲表现”。

但“反弹”和“强劲表现”是相对而言的:Android和iOS之间的差距继续扩大，在Kantar调查的五个市场中，Android的销售额超过了70%。

然而，在特定市场，苹果继续击败安卓。在日本，苹果仍然是占主导地位的智能手机品牌，销售额接近57%。Kantar指出，具体来说，它在NTT DoCoMo上占据了42%的智能手机销售份额，在KDDI AU上占据了59%的份额，在软银上占据了81%的份额。

为什么？苹果已经找到了设计、4G接入和设备可靠性的正确组合，引起了日本消费者的共鸣。“日本对苹果的热爱没有消退的迹象，”Sunnebo指出。他还指出，这对iPad产生了连锁效应，近四分之一的日本iPhone用户也拥有苹果平板电脑。

尽管苹果在iPad和iPhone的尺寸上保持了相当大的差距，但事实上在亚洲，真正的问题是越来越多的设备出现在两者之间。该公司指出，在中国，3月份屏幕大于5英寸的设备占智能手机销量的40%。

“很明显，泡沫确实正在改变中国消费者使用智能手机的方式，”孙尼波写道。“现在超过五分之一的平板电脑用户每天都看手机电视，有一半的用户至少一个月看一次，而这还没有普及4G网络的保障。随着中国4G基础设施的扩张，对数据的需求将达到前所未有的水平，从而为运营商通过更大的数据包大幅提升收入铺平道路。”

虽然苹果在吸引新的智能手机购买者方面，已经对旧机型和存储空间较小的机型采取了涓滴式的策略，但问题在于，它是会选择同时避开这一趋势，还是会对iPhablet采取自己的策略。
