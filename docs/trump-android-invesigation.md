# 加州议员提议对特朗普不安全的安卓手机进行调查

还记得新任总统川普[放弃](https://web.archive.org/web/20230129223149/https://techcrunch.com/2017/01/20/trumpdroid/)的不安全安卓手机吗，但显然[并没有放弃](https://web.archive.org/web/20230129223149/https://techcrunch.com/2017/01/25/trumpdroid-2-0/)？在这方面，事情似乎变得沉默了，因为世界花了一些时间来关注 2017 年疯狂政治的其余部分。

然而，今天，这个故事再次浮出水面，加州众议员泰德·刘(Ted Lieu)提议[对报道特朗普仍在使用他的不安全安卓手机(显然是旧的三星 Galaxy S3)的故事进行调查](https://web.archive.org/web/20230129223149/https://www.engadget.com/2017/02/17/ted-lieu-trump-android-phone/?sr_source=Twitter)，高管安全见鬼去吧。

众议员 Lieu [已经要求](https://web.archive.org/web/20230129223149/https://lieu.house.gov/media-center/press-releases/congressman-lieu-requests-house-oversight-investigation-trump-s-reckless)众议院监督和政府改革委员会举行一次调查听证会，调查总统“因严重未能全面实施常识性安全措施而危及国家安全”的报道。

一份与字母[相关的新闻稿还提到了关于特朗普在 Mar-a-Lago 餐桌上谈论核战略的报道。Lieu 在信中补充说，“网络安全专家普遍认为，尽管特勤局一再警告，总统仍在使用的普通安卓智能手机很容易被黑客攻击，”称其为“对国家安全的严重冒犯。”](https://web.archive.org/web/20230129223149/https://lieu.house.gov/sites/lieu.house.gov/files/2017-02-15%20Letter%20to%20Chaffetz%20-%20Hearing%20on%20Trump%20Phone.pdf)

这封由 15 名国会议员联署的信还提到了一个问题，这个问题应该是任何在选举期间关注过的人都熟悉的，即白宫工作人员涉嫌使用不安全的电子邮件账户。“我们将提醒国会共和党主席对国务卿克林顿使用私人电子邮件的担忧，”信中说，并补充说，“需要举行公开听证会，以确保此类通信的安全以及联邦法律要求的记录保存程序的充分性。"
