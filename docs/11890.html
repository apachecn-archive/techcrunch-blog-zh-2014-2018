微软的 Windows 10 Creators 更新将于 4 月 11 日发布 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/03/29/microsofts-windows-10-creators-update-will-launch-april-11/

经过几个月的戏弄，微软终于准备好向用户发布 Windows 10 Creators Update(T1)，这是其桌面操作系统的下一次重大升级。免费更新将于 4 月 11 日在全球范围内推出。这个过程通常需要几个星期，但用户也可以从他们的 Windows 设置中强制更新。

顾名思义，更新的重点是“创作者”。微软在这里对这个主题进行了非常广泛的解释，但这次发布的亮点是对即将到来的以 Windows 为中心的混合现实和虚拟现实耳机的支持(特别是对开发者来说)，由于新的专用游戏模式和内置的光束流，以及像 Paint 3D 这样的新创意工具，对游戏的支持更好。

此次更新还包括新的夜光模式以帮助您更好地睡眠，父母可以为孩子设置的屏幕时间限制，以及对 Windows Hello 安全功能的更新。

本月早些时候，当我与 Windows 总经理 Aaron Woodman 交谈时，他指出，他在过去几年中看到的是回归微软和 Windows 的根源，他认为这次更新是另一个例子。在他看来，此次发布的三大亮点是 Windows 混合现实(曾被称为 Windows 全息)、新的游戏功能和对 Edge 浏览器的更新。

事实上，尽管取笑微软的浏览器努力一直很流行，早期的 Edge 版本肯定有一些可用性问题，但它现在已经成为一个值得尊敬的竞争对手。伍德曼指出，微软希望首先“解决基本问题”，如性能和安全性，现在已经在其他领域开展工作，如标签管理和 Cortana 的集成(实际上效果很好)。

通过这次更新，该公司正在做一些有趣的事情，即将电子书带到 Windows Store，并将在 Edge 中显示。起初，这似乎是一个奇怪的举动。我们都已经习惯于使用专门的应用程序甚至设备来阅读电子书。然而，伍德曼认为，虽然这在移动设备上是正确的，但在 PC 上，浏览器是人们消费文本的默认场所。

该更新将于 4 月 11 日开始推出。它需要多长时间才能到达每台电脑上还有待观察，微软将根据它得到的反馈调整这个过程。

除了宣布此次更新的发布日期，微软今天还宣布将把 Surface Book 和 Surface Studio 硬件推向更多市场。Surface Book 现在可以在奥地利、中国、丹麦、芬兰、法国、德国、中国香港、日本、荷兰、挪威、瑞典、瑞士和英国预订。它将于 4 月 20 日发货。

Surface Studio(和 Dial)现在可以在澳大利亚、加拿大和新西兰预购。它也将于 4 月 20 日发货。
