特斯拉实现 Model 3 生产目标 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/07/01/tesla-model-3-production-goal/

特斯拉似乎在 6 月的最后一周生产了近 5000 辆 Model 3 电动汽车，距离首席执行官埃隆·马斯克最初计划在 2017 年底实现的目标仅几个小时。

根据路透社与两名工人的谈话，第 5000 辆 Model 3 于周日早上从装配线上下线。来自工人的社交媒体帖子证实了这份报告。马斯克周日在推特上说，特斯拉在七天内生产了 7000 辆汽车。这条推文没有说明这是不是 Model S、Model X 和 Model 3 车辆的组合。

几个月来，特斯拉的目标一直是每周生产 5000 辆 3s——但没有成功。在 2017 年 5 月致股东的一封信中，特斯拉表示，其生产设施的准备工作正在进行中，以支持在 2017 年的某个时候将 Model 3 的产量提升至每周 5000 辆。

但自 2017 年 7 月马斯克向员工分发第一批车辆的引人注目的活动以来，该公司未能实现 Model 3 的任何生产和交付目标。特斯拉一直在生产瓶颈中挣扎，一度不得不回归手工制造汽车。

Model 3 的生产问题最早出现在 2017 年 10 月初，当时特斯拉报告称，第三季度仅生产了 260 辆新电动汽车，仅交付了 220 辆。

马斯克多次将 Model 3 称为“生产地狱”，最近几个月他加大了努力，希望证明该公司可以成为电动汽车的大规模生产商。马斯克最近指示该公司在其加利福尼亚州弗里蒙特的工厂建立一个巨大的帐篷结构，以容纳 Model 3 的另一条装配线。

特斯拉通常会在季度结束后几天报告生产和交付数据。第二季度截至 6 月 30 日。

该公司仍有很多事情要向客户和股东证明，包括证明它可以持续达到每周 5000 辆的产量，并生产没有装配和完工问题的汽车，新车主已经注意到了这个问题。

特斯拉最近向美国和加拿大的所有预订者开放了 Model 3 订单。截至周四，数以万计的预订者能够登录特斯拉网站，并在该公司的在线设计工作室完成订单。目前，预订者有三个版本的 Model 3 可供选择:远程 310 英里后轮驱动，远程双电机全轮驱动或性能变体。当客户做出选择时，将向他们显示交付估计，交付估计将取决于他们的订单日期和所选的车辆配置。

难以捉摸的标准电池 35，000 美元的变种还不是客户的选择。
