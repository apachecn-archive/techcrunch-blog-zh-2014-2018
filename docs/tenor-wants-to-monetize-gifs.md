# 创业公司分享 gif 能赚钱吗？特诺想知道。

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/04/12/tenor-wants-to-monetize-gifs/>

在过去的几年里，gif 已经成为多种沟通渠道中一种普遍的视觉沟通和自我表达形式:它们已经进入了我们的文本、我们的推文、我们的 Messenger 聊天，甚至进入了我们的 Slack 频道。

但是像大多数新的媒体和传播形式一样，gif 到目前为止一直在努力寻找一种商业模式来匹配它们的无处不在。GIF 分享初创公司 [Tenor](https://web.archive.org/web/20230321060326/https://www.tenor.co/) 希望通过与媒体公司、品牌和广告商合作，帮助他们建立和分发用户会喜欢的 GIF，并希望在此过程中赚点钱，从而改变这种状况。

Tenor ( [neé Riffsy](https://web.archive.org/web/20230321060326/https://techcrunch.com/2016/08/09/gif-keyboard-maker-riffsy-gets-a-new-name-tenor-and-opens-up-an-sdk/) )制造了一款[流行的移动键盘](https://web.archive.org/web/20230321060326/https://itunes.apple.com/app/apple-store/id917932200?mt=8%3Fapp%3Dmessages)，让用户可以在他们可能正在使用的任何应用程序中轻松搜索和分享 gif。它还有一个 SDK，使第三方开发者能够将 GIF 共享添加到自己的应用程序中。到目前为止，这一功能已经被苹果的 iMessage、Facebook Messenger、脸书评论、WhatsApp 和 Twitter 等应用和服务所采用。

由于它自己的键盘和与其他应用程序的集成，Tenor 每月拥有超过 2 亿用户，每天处理超过 2 亿次 gif 搜索请求。凭借这一规模，该公司正寻求利用积极的用户受众来赚钱，这些用户正在搜索应用程序，以最好地表达他们此刻可能感受到的任何情绪。

但是你是怎么做到的呢？好吧，如果你是 Tenor，你会尝试寻找那些渴望与消费者建立联系的新方式的媒体公司和品牌，他们希望将自己的商品和服务与顾客与朋友分享的 gif 图片相关的情感联系起来。

为了帮助实现这一目标，该公司聘请媒体高管杰森·克雷布斯(Jason Krebs)担任首席商务官。Krebs 最近是迪士尼制造商工作室的销售主管，在过去的几十年里一直从事数字媒体工作。一直以来，他都在帮助品牌和广告商了解他们可以用有限的广告费获得大量受众，这一切都要归功于新的媒体传播形式。

在推销了阳光下的每一种新型广告单元后，从数字显示到在线视频，再到像震颤视频、康泰纳仕互动和纽约时报网站这样的有影响力的营销，Krebs 现在的任务是让营销人员理解 gif。

但是克雷布斯也有一些好的方面。首先，我们在前面几段谈到了规模问题。

他还得到了男高音的“洞察力引擎”的支持。这些工具让任何人——不仅仅是营销人员！—获取用户如何与公司的 GIF 搜索引擎交互以及他们正在共享什么的实时数据。

从理论上讲，这一工具将让品牌更好地了解用户在任何特定时间所关心的东西。利用其数据，品牌将能够向 Tenor 付费，在该应用的搜索结果中优先显示它们制作的 gif。希望如果这些 gif 足够吸引人，消费者会与他们的朋友分享这些品牌信息。

对营销人员来说，这是一个相当新颖的信息。“没有人会把展示广告的截图分享给他们的朋友，”男高音首席执行官大卫·麦金托什告诉我，这可能是今年最低调的说法。但是有了 Tenor，人们还是可以分享广告。
