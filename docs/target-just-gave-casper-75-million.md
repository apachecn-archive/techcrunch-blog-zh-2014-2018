# Target 刚刚给了 Casper 7500 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/05/25/target-just-gave-casper-75-million/>

# 塔吉特刚刚给了卡斯帕 7500 万美元

据报道，Target 已经向 Casper 投资了 7500 万美元，这是一家直接面向消费者的床垫初创公司，你可能在所有这些播客中都有所耳闻。据最先报道这一消息的 Recode 的消息人士称，这笔资金是最终应该超过 1 亿美元的一大笔资金。

塔吉特公司已经证实给了卡斯帕一大笔钱，但不愿证实具体数额(叹气)。~~我们已经向 Target 寻求更多信息，如果有回音，我们会及时更新。*更新:* *没有，目标不想评论。*~~

据称，这笔投资发生在收购谈判失败之后。今年早些时候，Target 曾出价 10 亿美元收购 Casper，但显然 Casper 对交易条款并不感兴趣。据接近此事的 TechCrunch 消息人士称，这家快速增长的床垫初创公司已经是一家默默无闻的独角兽，TechCrunch 最近获得的一份 IOData 报告显示，该公司在 2014 年至 2017 年期间占所有床垫盒销售额的 39%。

现在，塔吉特将成为主要投资者。Casper 的现有投资者 Lerer Hippeau Ventures、IVP 和 NEA 也为这轮融资注入了一些现金。

对于零售巨头来说，这笔融资可能是一个不错的选择，因为实体商店一直在努力应对在线竞争。Target 在一份声明中基本上证实了这一点(如果你能越过公关术语)，称与 Casper 已经建立的在线品牌合作将使其受益:

> “战略合作伙伴关系使 Casper 能够接触到一个成熟的零售品牌，并让 Target 有机会与一个关注未来的数字品牌合作，该品牌正在探索一个对我们的客人有意义的领域——睡眠和健康。我们期待着一起探索未来。”

Target 还希望通过与 Casper 等品牌合作，以及最近与 Bevel 和名人风格博客 WhoWhatWear 合作，吸引更年轻、懂数字的客户，以吸引新一代在线购物者。

对于 Casper 来说，这笔资金给了它一些跑道，让它能够继续在已经拥挤的(但显然[相当有利可图的](https://web.archive.org/web/20230325133012/https://techcrunch.com/2017/04/13/purple-a-utah-mattress-startup-youve-probably-never-heard-of-is-on-track-to-unicorndom/))在线床垫市场上占据主导地位，因为它要与 Tuft & Needle 和 Purple 等初创公司竞争，这些公司都拥有自己的制造设施，并在成本低得多的地区运营，以保持低运营成本。
