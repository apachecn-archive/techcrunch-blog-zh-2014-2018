iPhone 版尤利西斯让我考虑在手机上写文章 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/03/13/ulysses-for-iphone-makes-me-consider-writing-articles-on-my-phone/

我每天在手机上写几百字。起初它只是电子邮件和短信，但现在它也是各种消息应用程序中各种大小和形状的松散对话、笔记和无数消息。然而，有一件事我至今没有做——在手机上写 TechCrunch 的帖子。但这也在改变。

本周早些时候，小型独立开发商店 The Soulmen 为 iPhone 发布了尤利西斯。尤利西斯是 OS X 和 iPad 上一款流行的写作应用。我以前在 Mac 上玩过这个应用程序。但是我真的不能认真考虑使用它，因为它缺少了一个元素——一个 iPhone 应用程序。

尤利西斯是一个 Markdown 文本编辑器，用于小型和大型项目。它适用于博客文章和小说之间的任何内容。而尤利西斯也是我们在 iOS 上需要的那种 app。这是一款售价 25 美元的昂贵应用，提供了几乎所有桌面版的功能。换句话说，做正经工作，商业模式清晰，才是正经 app。当然，不同版本的应用程序之间的一切都保持同步。

这是理解尤利西斯如何脱颖而出的关键。我现在可以在我的手机上开始起草，在我的 iPad 上编辑，在我的 Mac 上进行最后的检查，然后在这个网站上发布。过去我已经在我的 iPhone 上用 Simplenote 写了一些草稿。但这是一个更加简化的过程。

Ulysses 是一款精致的 iOS 应用，可以让你以不同的格式写作、组织和导出你的作品。这是一个设计良好、功能多样的应用程序，如果你每天都写，值得一试。它与许多其他应用程序竞争，如 iOS 上的 Editorial 和 1Writer ，以及 OS X 上的 Scrivener

我想让尤利西斯更进一步，让它与我们的 CMS——WordPress 无缝结合。我从 OS X 的 Ulysses 开始，只需点击几下，我现在就可以运行一个 Ruby 脚本，将我的草稿发送到我们的后端，这样我只需审阅它，调整几个选项，然后发布它。

但是 iOS 呢？你不能在 iOS 上这么做，对吗？受到 iOS 自动化向导 Federico Viticci 的启发，我试图在 iOS 上重新创作我的 OS X 脚本。果然，自动化应用工作流程让你做得很好。我现在可以从尤利西斯发布到 WordPress，并使用这个工作流程自动填充标题、标签和摘录。另一个工作流程让我从 iOS 上传图片到 WordPress。

我已经用这个过程写了一些帖子，它比我预期的更有用。我已经花了很多时间在我的 iPhone 上打字，我在 iPhone 上打字的速度几乎和在 Mac 上一样快。但很多时候，用手机写要方便得多。其实这个帖子是我手机写的。
