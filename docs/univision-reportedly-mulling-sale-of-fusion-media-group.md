# 据报道，Univision 正在考虑出售融合媒体集团

根据*华尔街日报*， [Univision 正在考虑](https://web.archive.org/web/20230325032637/https://www.wsj.com/articles/univision-eyes-sale-of-fusion-media-group-1530914852)出售品牌组合，包括 Gizmodo、Deadspin、The Root、Lifehacker 等热门网站和一大块 *The Onion* 。

据报道，去年，这家西班牙语广播公司试图以 2 亿美元的价格出售该公司 20%的股份，但最终未能找到任何买家，原因是潜在投资者“胆小如鼠”。

所有这些都发生在 Univision [以 1.35 亿美元收购 Gawker Media](https://web.archive.org/web/20230325032637/https://techcrunch.com/2016/08/16/unvision-acquires-gawker/) 不到两年后，该品牌因彼得·泰尔提供部分资金的胡克·霍根诉讼而被拖垮。该品牌随后被并入 Fusion Media Group，这是四个月前 Univision 收购迪士尼在 Fusion TV 的股份时创建的。

此举在很大程度上被视为一家广播公司吸引年轻观众的尝试，否则该公司很难适应当前的趋势。但媒体从来都不容易，尽管 Univision 拥有许多强大的资产，但这次收购并没有完全改变 Univision 的命运。

上个月，Univision 宣布前媒体总经理 Vince Sandusky 将接替 Randy Falco 担任首席执行官，后者在公司放弃 IPO 计划后不久辞职。四月，作为正在进行的重组的一部分，公司解雇了 150 名员工。

Univision 尚未对此事发表评论。
