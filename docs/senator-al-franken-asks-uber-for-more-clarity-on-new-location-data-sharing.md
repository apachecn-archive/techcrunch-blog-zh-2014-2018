# 参议员艾尔·弗兰肯要求优步在新的位置数据共享上更加清晰

美国参议员艾尔·弗兰肯(D-Minnesota)给优步首席执行官特拉维斯·卡兰尼克写了一封公开信，要求该高管就其最近对用户位置数据收集的变化提供更多透明度。这封信认为，消费者应该对他们的个人数据有更直接的控制权，但是，如果做不到这一点，他们至少应该对他们的数据如何以及何时被书面使用有一个全面而清晰的了解。

这封信是对 11 月份产品更新的直接回应，该更新将位置服务权限选项从“使用应用程序时”更改为简单的“总是”或“从不”，优步表示，这一更改是必要的，因为它希望不仅能够在旅行期间收集位置信息，而且能够在旅行完成后的五分钟内收集位置信息，以帮助它改善接送位置识别，以及改善安全和欺诈防范措施。

弗兰肯表示，理想情况下，优步应该在应用内实现用户对何时以及如何共享数据的控制，以此绕过苹果在 iOS 系统级位置服务设置的限制，在使用应用时，除了“开”、“关”和“开”之外，该设置不允许太多渐变

如果做不到这一点，参议员希望优步对其隐私声明进行一些修改，他在信中详细阐述了这些内容，我在下面全文转载，以供参考:

> 1.  收集位置信息:在“我们通过您使用我们的服务收集的信息”下，优步的隐私声明规定，如果允许，优步“也可以在应用程序在前台或后台运行时收集(用户)设备的精确位置。”这一部分应该更新，以明确说明优步仅在用户与应用程序交互时、用户正在旅行时以及司机结束旅行后的五分钟内收集精确的位置信息。
> 2.  位置信息的使用:在“信息的使用”下，优步的隐私声明详细说明了公司可能如何使用其收集的用户信息。此部分应更新，以反映出行后位置数据仅用于应用程序更新的声明理由，包括改善接送、增强安全性和防止欺诈。
> 3.  旅行相关位置数据指南:优步的隐私声明应包括网站的旅行相关位置数据页面上提供的相关信息，包括如何禁用位置跟踪的详细信息，或提供一个页面链接，以确保用户可以轻松访问和全面了解优步对其位置数据的收集、处理和共享。
> 4.  用户通知:在“声明的变更”下，优步的隐私声明规定，当公司对处理用户个人信息的方式或声明做出重大变更时，将通知用户。为了确保用户对任何此类更改提供有意义的同意，包括这里建议的更改，优步应该通过应用程序和电子邮件通知用户。

Franken 在信中公开表示，他相信优步的话，因为它试图改善最终用户的服务和整体客户体验。但他也很好地要求在优步自己面向用户的政策中书面保证这一点，这是一种让它对其声称的位置数据变化的合理性负责的方式。
