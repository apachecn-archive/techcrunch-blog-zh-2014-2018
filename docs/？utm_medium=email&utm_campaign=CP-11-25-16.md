# 支付提供商 Stripe 以 90 亿美元的估值再融资 1.5 亿美元 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/11/25/payments-provider-stripe-has-raised-another-150-at-a-9b-valuation/?utm_medium=email&utm_campaign=CP-11-25-16>

让网站和应用通过 API 和几行代码整合支付服务的公司 Stripe 已经筹集了新一轮资金，D 轮融资 1.5 亿美元，前期估值 90 亿美元；92 亿美元的后期资金。

当许多人质疑创业界最有价值的所谓“独角兽”是否被过度膨胀时，这对该公司来说是一个重大的进步。从 2015 年夏天与 Visa 合作的一轮融资开始，[之前的估值为 50 亿美元](https://web.archive.org/web/20221005193733/https://beta.techcrunch.com/2015/07/28/stripe-5-billion/)。

除了今天的这轮融资，Stripe 还从摩根大通(J.P. Morgan Chase & Co .)、高盛集团(Goldman Sachs Group Inc .)、摩根士丹利(Morgan Stanley)和巴克莱银行(Barclays PLC)获得了高达 2.5 亿美元的循环信贷。这里的想法是，在利率较低的时候，这扩大了 Stripe 的借款能力，但没有扩大其债务。没有使用的义务。

这笔资金的消息来得正是时候。我们正处于假日购物季的[第一天，像 Stripe 这样从其平台上的每笔交易中抽取佣金的公司，将有望获得今年最大的收入。仅感恩节一天，预计 2016 年销售额将首次突破 20 亿美元。](https://web.archive.org/web/20221005193733/https://beta.techcrunch.com/2016/11/24/2016-thanksgiving-on-track-to-break-2b-in-online-sales-for-the-first-time/)

有些人可能会觉得这条新闻在新闻较少的假日周末出现有点奇怪——WSJ[在今天早些时候率先报道了这条新闻。](https://web.archive.org/web/20221005193733/http://www.wsj.com/articles/stripes-valuation-nearly-doubles-to-9-2-billion-1480075201)

但对我来说，这非常有意义:Stripe 最终会想要上市，或者被更大的商业(或者科技，嗯，谷歌？)庞然大物，这是它的说法，在网购的关键时刻:我们在这里，我们来找你了。

这一轮由两家风投公司领导:之前的投资者 [General Catalyst](https://web.archive.org/web/20221005193733/http://generalcatalyst.com/) 和新的支持者“[Capital g](https://web.archive.org/web/20221005193733/https://www.capitalg.com/)”——谷歌资本后谷歌重组其搜索、移动和其他业务的新名称，以新的 Alphabet 品牌命名。这轮投资还包括红杉资本(Sequoia Capital)和 Stripe 没有提到的其他投资者。

这使得总部位于旧金山的 Stripe 公司(由爱尔兰兄弟帕特里克和约翰·科利森(分别是其首席执行官和总裁)于 2010 年创立)的资金总额达到约 4.6 亿美元[。](https://web.archive.org/web/20221005193733/https://www.crunchbase.com/organization/stripe)

尽管 Stripe 主要围绕支付服务开展业务，但它也一直在开发许多其他功能，以实现将自己定位为金融服务平台的更大抱负。

这些工具包括[整合美国以外的业务](https://web.archive.org/web/20221005193733/https://beta.techcrunch.com/2016/02/24/stripe-expands-startup-tools-with-atlas-a-way-for-global-companies-to-incorporate-in-delaware/)；[欺诈防范](https://web.archive.org/web/20221005193733/https://beta.techcrunch.com/2016/10/19/stripe-launches-radar-to-tackle-e-commerce-fraud-with-machine-learning/)工具；以及使用其平台的商家和市场加速支付的方式。该平台帮助该公司增加了利润(仅 Stripe 支付服务的交易利润非常低)，并增加了与客户的接触点。

为此，Stripe 表示，这笔资金将用于进一步发展，提供更多的开发工具、收购和其他服务，以帮助企业家运营他们的业务。该公司表示，它还将增加招聘。

虽然 Stripe 没有说明下一步可能会推出什么，但当该公司在 10 月底推出防欺诈工具 Radar 时，约翰·科利森暗示，可能会有更多产品推出，以改善买家和卖家的保护，建立对电子商务的更多信任。

“这是一个积极发展的领域，我们想做的事情有很多，”他说。“我们没有排除(将它作为一项独立服务推出)的可能性，但我们想先看看人们如何使用它，哪些有用，哪些没用。”

它还更普遍地采取了类似苹果的方式来实现在线商务的概念，并在互联网上经营企业:它看到了通过简化当前复杂的事情来与 PayPal 等公司竞争的机会。

科利森上个月告诉我，该公司的使命是“消除业务增长的复杂性……我们将把这视为 Stripe 未来产品供应的主要部分。这是我们看待世界的方式。是什么让企业日子不好过，为什么它们没有更快地发展？我们该如何改变这种状况？”

该公司在全球拥有多少用户，或者通过 Stripe 平台处理的收入或资金总额方面仍然没有那么透明，只是说这些用户现在遍布 110 个国家，Stripe 已经为一半的美国互联网用户处理了至少一笔支付。

换句话说，它有一个相当广泛的客户名单，这些客户使用它的一种或多种服务，其中包括大众市场的互联网品牌。其中一些客户是 SAP，梅西百货，Missguided，通用电气，阿迪达斯，DocuSign，Slack，Medium，每日邮报，Yelp，纳斯达克，联合国儿童基金会“和两个总统竞选。”
