Accel任命前Facebooker用户Nir Blumberger为风险合伙人，专注于以色列

风险投资公司Accel的伦敦办公室任命了一个新的风险投资伙伴，专注于以色列。Nir Blumberger从脸书加入，他在那里是一名企业发展执行官，我们被告知，他参与了M&A和虚拟/增强现实、消息和视频等领域的战略。他将常驻特拉维夫，让Accel在以色列这个布隆伯格出生和成长的国家有一个更加正式的存在。

在加入这家社交网络巨头之前，他曾在欧洲和以色列的麦肯锡公司工作。Blumberger拥有斯坦福大学的MBA学位，之前在以色列跨学科中心学习计算机科学，在那里他参加了Zell创业计划。他还联合创立了两家科技创业公司，并参与了奇点大学的研究生课程。

与此同时，尽管布隆伯格的任命标志着Accel在以色列加倍下注——我被告知，这是这家风投公司30多年历史中第一次有人常驻以色列，专门负责以色列的投资——但这在该国的投资活动中并不新鲜。

目前和过去的投资包括企业安全公司Aorato ( 被微软收购)和上市公司Imperva、全球服务市场Fiverr和数据治理解决方案提供商Varonis Systems。最近，Accel领投了1200万美元的B轮投资 E8存储，这是一家成立于特拉维夫的早期企业存储公司。

Accel的合伙人菲利普·博特里(Philippe Botteri)在一份声明中表示:“以色列是一个充满活力的创新中心，Accel长期以来一直坚信该地区涌现出的伟大创意和企业家。我们看到以色列涌现出越来越多令人兴奋的公司和技术，这是我们扩大业务的绝佳时机。我们很高兴有来自脸书的Nir加入我们的团队。他来自以色列和硅谷的世界级技术和商业背景使他非常适合Accel。”
