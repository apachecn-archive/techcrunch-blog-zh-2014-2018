三星可能会收回自己的软件，在 Galaxy S6 上拥抱微软

我们已经听到报道称，三星可能会在下一款旗舰产品中突破 TouchWiz，但来自基本准确的 SamMobile 今天的一份新报告称它还将从 Galaxy S6 中删除大部分内部预装软件，转而提供一系列预装的微软智能手机应用程序。如果是真的，这是迄今为止最确定的迹象之一，表明三星正在以一种真正发挥其优势、最小化其劣势的方式重新思考其移动战略。

根据该报告，三星将减少 TouchWiz 的性能影响，并在某些方面更接近 stock Lollipop。但大新闻是，它将移除“所有”预装应用，这大概意味着像 S Voice、S Health、S Note 等应用。这些仍将可用，并将提供更丰富多彩的重新设计，但它们将作为可选下载驻留在三星 Galaxy 应用商店中，而不是你从设备上开箱获得的东西(永远无法真正删除)。

不过，三星不会完全取消预装软件:相反，它将提供微软改进后的生产力应用套件，包括 OneNote、OneDrive、Office Mobile(包括不确定期限的 365 免费订阅)和 Skype。鉴于微软在为其表面上的竞争对手的移动平台重新设计软件产品方面的成功，这应该比向用户提供三星一般不起眼的同类产品更有益。

对微软来说，这是一种立即获得 Windows Phone 永远无法实现的影响力的方式，至少在不久的将来不会。随着该公司重新关注其软件和服务部门，做这种事情将使其提高新一代用户的意识。即使最终它想让用户回到 Windows Phone 平台，吸引现有用户也是为了实现其短期目标。如果它的移动操作系统最终无处可去，至少他们与用户建立了关系，他们可以在此基础上建立一个替代的收入战略。

三星将在 3 月 1 日巴塞罗那世界移动通信大会开幕前的一场特别活动中揭示所有细节，我们将尽快公布所有细节。
