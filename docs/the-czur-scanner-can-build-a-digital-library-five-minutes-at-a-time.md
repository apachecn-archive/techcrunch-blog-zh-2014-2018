# Czur 扫描仪可以一次建立五分钟的数字图书馆

当西哥特人烧毁亚历山大图书馆时，你可以打赌老托勒密一世会喜欢在他富丽堂皇的大理石装饰的大客厅里有一台 Czur(发音为“凯撒”)扫描仪。这

> Czur

基本上是一个类固醇的书籍扫描仪，它可以让你在大约五分钟内扫描 300 页的书——或者无价的古代知识卷轴。

Czur 是一种相机/扫描仪的混合体，可以让你把一本书放在主扫描板上，然后翻页。它可以拍摄每一页的单独照片，对它们进行处理，甚至可以取出错误的手拍照片。早鸟价格是 189 美元，这对于像这样的图书扫描仪来说是相当不错的。您还可以将其他物体放在高分辨率相机下，HDMI 端口可以让您将图像直接发送到投影仪。脚踏板可以让你远程触发扫描仪，使用该软件，普通人一分钟可以扫描大约 80 页。

该团队已经超过了他们 20，000 美元的目标，尽管他们的营销材料有点粗糙，但你已经了解了该系统的大致情况，尤其是考虑到它可以扫描书籍，同时适当地展平页面，并获得清晰的图像和 OCR 输出。

Czur 也是批发图书盗版机吗？也许吧！但如果你从事的是非小说或研究时间较长，这可能会有很大的帮助，而且对于对进入数字未来感兴趣的小型图书馆来说，这可能是天赐之物。该团队预计在 1 月份发布。
