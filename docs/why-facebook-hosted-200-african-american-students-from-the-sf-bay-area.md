# 为什么脸书接待了来自旧金山湾区的 200 多名非裔美国学生 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/02/29/why-facebook-hosted-200-african-american-students-from-the-sf-bay-area/>

就在马克·扎克伯格要求员工停止划掉黑人的命也是命标语的内部备忘录泄露一天后，脸书接待了来自旧金山湾区的 200 多名非裔美国学生。作为黑人历史月的一部分，上周五为期一天的脸书实地考察活动已经筹备了数周，由员工资源小组 Black@FB 和脸书的社区参与团队牵头。它包括参观门洛帕克校园，职业小组，介绍[脸书最近推出的 TechPrep 计划](https://web.archive.org/web/20221210032912/https://beta.techcrunch.com/2015/10/20/facebook-launches-tech-prep/)，以帮助增加技术领域中代表性不足的人的管道，关于实习的信息会议，以及其他活动。

脸书全球多元化总监 Maxine Williams 告诉我:“当多元化运作良好时，如果你有一个可扩展的系统，就永远不会有成功或失败的问题。”。“所以这就是我们的目标。这是一件美妙的事情，当有一个多元化的事件，而我与它无关。我与此事无关。”

尽管是 Williams 创建了 TechPrep，但社区参与团队已经接管了它。Williams 接着说，这个项目“不应该归多元化团队所有。”相反，Tech Prep 以及由多元化团队发起的其他计划应该深深植根于脸书的所有团队和整体文化中。

“其中一个基础是，通过多元化努力和其他人的努力，我们已经让整个公司更加关注根本问题，”威廉姆斯说。“你可以在这里继续修剪树枝，但由于这些问题发生在根部，你永远也不会得到一棵健康的树。我们关注的根本问题是什么——是什么导致一些人的机会减少，而另一些人的机会增加？是什么原因导致某些人才的渠道更加畅通，而其他人的渠道却越来越少呢？——激励人们去做更多这类活动。”

该活动的最终目标是向学生传授科技领域的机会，以及揭穿在科技公司工作需要知道如何编码的神话。在周五来脸书之前，圣伊格内修斯学院预科大三学生亨利·戴维斯从未参观过科技公司。他也没有意识到除了编程之外，科技公司还有如此多的工作机会。

戴维斯说:“这比想象中的要广泛得多，这是科技公司的一个危险的耻辱，因为它会把人吓跑。”。“有很多像我一样的人，他们在 STEM 领域不是特别有天赋，他们看着科技行业，说，‘这是一个非常以 STEM 为导向的地方’，但它不是 STEM 专属的，这是人们缺乏适当信息了解的一件事。”

琳达·乔丹(Linda Jordan)是旧金山联合学区(San Francisco Unified School District)教会高中(Mission High School)的家长和社区协调员，她帮助将一些学生带到了脸书，其中一些学生对图形设计感兴趣，一些学生已经知道如何编程。

“我希望今天在场的学生能够认识到，科技行业并不是一个不需要他们的行业，”乔丹说。不过，值得注意的是，科技行业并不是一个多样化的地方。例如，脸书只有 2%的黑人。展望未来，乔丹表示，她希望与学生们就实习、大学申请和选择专业进行后续对话。

QuinSi Dominguez 是 Mission 高中的一名三年级学生，他是已经知道如何编写代码的学生之一。Dominguez 告诉我她对从事平面设计和市场营销感兴趣，她能够用 Python 和 HTML 编写代码。Dominguez 设想自己有一天会在脸书全职工作，所以她渴望更多地了解在这样一家有影响力的科技公司工作会是什么样子。她六年级的时候曾经去过一次脸书，但是这次旅行对她来说有更重要的意义。

“现在我长大了，我需要了解这些东西，比如实习机会，他们想如何帮助社区，并试图推动他们去高中实习，”多明格斯说。“也许有一天我可能会在这里，甚至比我想象的还要早。”
