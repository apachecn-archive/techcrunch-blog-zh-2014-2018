三星永久封杀 Galaxy Note 7

原文：https://web.archive.org/web/https://techcrunch.com/2016/10/11/samsung-kills-the-galaxy-note-7-for-good/

由于对缺陷电池的安全担忧，三星将永久停止生产和销售 Galaxy Note 7 智能手机，包括该公司随后在召回原始设备后发送给客户的替换手机。

《T2 华尔街日报》早些时候报道了这一消息，三星的一位发言人证实，三星将停止生产这款设备，并告诉 TechCrunch:“我们可以确认三星已经永久停产 Galaxy Note7 的报道。”

该公司没有提供任何解释其决定的附加声明，但在向韩国监管机构提交的一份关于停止生产的文件中，该公司称原因是“客户安全”。

昨天，随着替换手机出现问题的报告增多，三星要求运营商和经销商停止销售和更换 Note 7 设备。Note 7 用户可以获得全额退款或不同三星智能手机的退款，以及 25 美元的礼券或其他附加产品，以减轻处理两轮召回的麻烦。

9 月初宣布了首批召回约 250 万部流通中的 Note 7s。

该公司此前还表示，正在调整 Note 7 的生产，用它的话说，“采取进一步措施确保质量和安全问题”。

然而，在这种情况下，鉴于越来越多的公关噩梦，即使是更换的 Note 7 也有明显的电池爆炸风险——再加上该型号在机场和世界各地的航班上被检查为安全隐患——三星决定拔掉插头，让 Note 7 脱离苦海也就不足为奇了。

除了与爆炸电池有关的对三星 Galaxy 品牌的损害，撤回 Note 7 的成本也不会便宜。路透社援引分析师的估计称，永久停止 Note 7 销售可能会让该公司损失高达 170 亿美元。

与此同时，该公司的股价也因 Note 7 倪星的消息而受到重大打击:今天下跌了 8%，这是自 2008 年 10 月以来的最大百分比跌幅，投资者将该公司的价值抹去了约 190 亿美元。

更新:三星现已向 TechCrunch 提供以下声明:

为了消费者的安全，我们停止了 Galaxy Note7 的销售和换货，并因此决定停止生产。
