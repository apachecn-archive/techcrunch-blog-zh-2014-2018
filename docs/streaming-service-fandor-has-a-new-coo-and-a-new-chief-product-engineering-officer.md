# 流媒体服务 Fandor 有了新的首席运营官和新的首席产品工程官

[Fandor](https://web.archive.org/web/20230328183314/https://www.fandor.com/) 扩大了其管理团队，将法律总顾问兼首席运营官 Felice Oper 提升为首席运营官，并任命 TS Ramakrishnan 为首席产品工程官。

奥普于 2015 年加入公司，大约在同一时间[拉里·艾登接任首席执行官](https://web.archive.org/web/20230328183314/http://variety.com/2015/digital/exec-shuffle-people-news/larry-aidem-fandor-ceo-1201601566/)。之前曾在 HBO 与 Oper 合作的 Aidem 说，她在 Fandor 的角色已经有机地扩展了，她不仅处理公司的法律事务，还策划了一些交易，如《T4》与 MoviePass 的新合作关系。

“她不仅仅是拒绝和袖手旁观——这是一些律师认为他们的工作，”Aidem 说。“她知道如何从两方面看待这件事。从 GC 到分销，再到产品和工程，这是一个非常自然的过程，不仅使我们的分销成为可能，也使广告和赞助成为可能。”

至于 Ramakrishnan，他以前的职务包括雅虎移动产品主管和脸书产品工程副总裁，Aidem 说他们通过克里斯·凯利联系在一起，这位早期的脸书高管投资了 Fandor，目前担任其董事长。Aidem 和 Oper 都称赞了 Ramakrishnan 的产品洞察力和“互动印章”。

这些任命正值更广泛的娱乐业格局发生变化之际，网飞及其竞争对手最引人注目的支出都投向了电视剧。艾登说，相比之下，范多尔将继续“纯粹以电影为中心”，拥有“最大的独立电影供应”

[![Felice Oper](img/13bca168c607c5f1b8c8dc37e15dccbc.png)](https://web.archive.org/web/20230328183314/https://techcrunch.com/2018/02/16/streaming-service-fandor-has-a-new-coo-and-a-new-chief-product-engineering-officer/felice-oper-hs/) 而 Fandor 本身正处于重启之中，这将由 Oper 和 Ramakrishnan 来指导。

“这将是渐进的，”奥珀说。“不过，这将是激进的。”

这个想法是把一个纯粹的订阅服务，所有内容都在付费墙后面，变成一个结合免费和付费内容的混合体。Fandor 已经通过其关键帧博客和视频文章朝着这个方向前进，但 Oper 表示，该计划最终也将提供免费电影，并创造一种产品，在这种产品中，一切都“以一种你在其他任何地方都找不到的方式集成到一个空间中”

“新的电梯广告是‘电影爱好者的潘多拉’，”Aidem 补充道。“你可以吃点心，了解我们。简短的形式是非常战略性的设计……成为一个将点点滴滴联系起来的地方——特别是对于那些可能看过*敦刻尔克*但不知道的千禧一代来说，“哦，这就是制作*纪念物*、*声望*和*蝙蝠侠*电影的同一个人。”"
