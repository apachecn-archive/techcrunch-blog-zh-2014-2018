# 你可以虚拟地居住在丰田新的人形机器人

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/11/21/you-can-virtually-inhabit-toyotas-new-humanoid-robot/>

【YouTube https://www.youtube.com/watch?v=uPI0-14kEZI？ecver=2]

丰田有一个新的第三代人形机器人，有着迷人的名字“T-HR3”，旨在成为人类有益和安全的助手。它还具有所谓的“主操纵系统”，这实际上是一个虚拟现实驱动的远程操作平台，人类可以使用它来让 T-HR3 反映它的运动。是的，有点像韩国电影《环太平洋》中的积家。

T-HR3 旨在以助手的身份在一系列不同的潜在用途中为人类工作，包括家庭护理、医院、建筑工地、受灾难影响的地区，甚至丰田公司表示，在遥远的外太空。这个机器人有点像一个矮个子，手臂很长，光滑的白色面板覆盖着它的机械部分，光学传感器在头部。

主操纵系统中的操作员有手臂和腿部覆盖物，将他们的运动传递给机器人，允许操作员使用全范围的运动来原地行走、操纵手臂，甚至通过直接转换他们的自然动作来抓握。由于他们戴着头戴式显示器(视频中是 HTC Vive ),操作员也可以从机器人的角度看。

同步的向上运动也有适当的安全措施，以确保机器人的运动不会干扰操作员，这样你就不会在坐在椅子上时意外地(或故意地，我猜)用机器人打自己。

远程操作、灵巧的人形机器人在几乎所有人类活动中都有很大的潜力。如果我们碰巧需要抵御异次元世界的入侵者，至少现在我们有了一条实现它的途径。
