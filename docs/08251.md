# Mentat 将代表你申请工作，并保证你得到面试机会

你知道现在不喜欢自己工作的人比喜欢自己工作的人多吗？

虽然曾经有一段时间，绝大多数美国人对他们目前的工作很满意，但现在有 50%到 70%的员工对他们目前的职位不满意。

结合这一统计数据和超过 90%的千禧一代预计每三年换一次工作的事实[，这听起来是进入就业行业的好时机。](https://web.archive.org/web/20221208131941/http://www.forbes.com/sites/jeannemeister/2012/08/14/job-hopping-is-the-new-normal-for-millennials-three-ways-to-prevent-a-human-resource-nightmare/#12f577985508)

虽然工作列表网站绝对不缺，但 Y Combinator 2016 年夏季推出的公司 [Mentat](https://web.archive.org/web/20221208131941/https://thementat.com/) ，做的事情有点不同。

该公司专门帮助那些知道自己想要一份新工作，但要么没有足够的组织能力来迈出这一步，要么实在太忙而没有时间去寻找和申请工作的人。

为了帮助这些人，Mentat 将接管并优化他们的 LinkedIn 和简历，然后逐字写求职信，并让他们的一名顾问为你手工申请工作。

[gallery ids="1353269，1353267，1353268，1353270" orderby="rand"]

该公司还为你配备了一名专职的现实生活中的顾问，在这个过程中，你可以通过电话、聊天和电子邮件尽可能多地与他交谈。该公司有大约 50 名顾问，他们拥有不同行业的专业知识，因此他们可以为你想从事的特定行业提供最佳建议。

那么这项技术是从哪里来的呢？Mentat 表示，他们正在使用机器学习来为求职者匹配潜在的工作，并为他们匹配具有类似技能的顾问。

整个套餐的价格是 249 美元，Mentat 会给你退款保证，你至少会得到一次面试机会。他们也有一个免费层，类似于传统的工作网站，让你找到并申请不同的工作。

但是如果你得到了一份你都不申请的工作，这算吗？Mentat 的观点是，大多数实际的工作申请只需要你反复阅读你的简历，最终是浪费时间。他们处理这种繁重的工作，这样你就可以专注于面试，这是一个人的实际个性和技能开始发挥作用的地方。

有趣的是，Mentat 正在与教育机构合作，为学生购买他们的产品。在今天的市场上，找工作真的是学生们唯一关心的事情，对于未来的学校来说，能够保证他们的毕业生在毕业后获得面试机会可能是一个优势。
