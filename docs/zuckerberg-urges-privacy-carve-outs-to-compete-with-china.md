# 扎克伯格呼吁隐私创业与中国竞争

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/04/10/zuckerberg-urges-privacy-carve-outs-to-compete-with-china/>

脸书的创始人上个月表示，该公司愿意接受监管。但今天，美国参议院问他希望(和不希望)看到什么样的立法变革来解决剑桥分析数据丑闻所暴露的问题。

扎克伯格对此的回应——以及关于他对欧洲隐私法规的看法的另一个问题——最详细地显示了他希望美国的数据处理和隐私规则如何演变，包括直接呼吁监管例外，以避免美国落后于中国竞争对手。

扎克伯格列出了“几条原则”，他说他认为这些原则将“有助于讨论并有可能编纂成法律”，扎克伯格首先倡导“一套简单实用的方法来解释你在用数据做什么”，显示出通过适用于所有参与者的便捷通用标准来卸载棘手的隐私泄露问题的意愿。

“当一件事只写在一份长长的法律文件中时，很难说人们完全理解了它，”他补充道。“这种东西需要以人们能够真正理解的方式来实现。”

然后，他大谈“让人们完全控制”他们分享的内容——声称这是“脸书最重要的原则”。

“你在脸书分享的每一条内容都是你自己的，你完全可以控制谁能看到它，以及你如何分享它——你可以随时删除它，”他说，但没有提到该公司在历史上的其他时期(T1)离这一原则有多远。

“我认为这种控制是很重要的，我认为应该适用于每一项服务，”他继续说道，并毫不隐晦地请求其他平台不要像脸书的平台那样泄露数据(从而堵住任何竞争漏洞，这些漏洞可能会因脸书在面临重大丑闻的情况下收紧开发者对数据的访问而出现)。

在回应立法改革问题时，他最后也是最有争议的一点是关于他所谓的“授权创新”。

“其中一些用例非常敏感，比如人脸识别，”他小心翼翼地说道。“我认为，在面部识别等敏感功能获得特别同意的情况下，达成平衡非常重要。但是不要——而是我们仍然需要这样做，以便美国公司能够在这些领域进行创新。

“否则，我们将落后于中国的竞争对手和世界各地的其他对手，他们对不同的新功能有不同的制度。”

扎克伯格没有说他具体想到的是哪些中国竞争对手。但本周早些时候，电子商务巨头阿里巴巴宣布了对面部识别软件业务的另一项重大投资，领投了总部位于香港的商汤科技(SenseTime)的 6 亿美元 C 轮融资(T2)，这是一个可能的竞争对手。

在会议的稍后阶段，扎克伯格还被直接问到欧洲隐私法规是否应该适用于美国。在这里，他再次亮出了自己的底牌——[再次拒绝确认](https://web.archive.org/web/20221208163132/https://techcrunch.com/2018/04/04/facebook-gdpr-wont-be-universal/)脸书是否会对北美用户实施“完全相同的规定”，因为一些[消费者团体一直呼吁这样做](https://web.archive.org/web/20221208163132/https://techcrunch.com/2018/04/09/facebook-urged-to-make-gdpr-its-baseline-standard-globally/)。

“不管我们是否实施完全相同的监管——我想它会有所不同，因为我们在美国和其他国家的敏感性有些不同——我们致力于推出控制和肯定同意，以及对 GDPR 所需的面部识别等敏感类型技术的特殊控制，我们正在世界各地这样做，”他重申。

“所以我认为，我们是否应该在美国采取类似的措施当然值得讨论，但我今天想说的是，无论监管结果如何，我们都将继续实施(同样的控制和肯定同意)。”

鉴于这是脸书第三次拒绝确认 GDPR 将普遍适用，看起来很明显，北美用户将获得一定程度的二级隐私，而不是国际用户——除非或直到美国立法者强行提高公司和整个行业的标准。

这也许是意料之中的事。但对于脸书来说，在重大数据丑闻中，这仍然是一个棘手的公关信息——因此扎克伯格试图将其重新定义为国内与国外“敏感性”的问题。

北美脸书用户是否会接受他对经济舱隐私标准与世界其他地区的重新包装，仅仅是一点当地风味，还有待观察。
