BitTorrent 发明者宣布生态友好型比特币竞争对手 Chia

一次比特币交易浪费的电力相当于一个美国家庭一周的电力，传奇编码师布拉姆·科恩想要解决这个问题。科恩创办了一家名为 Chia Network 的新公司，该公司将推出一种基于时间和存储证明的加密货币，而不是比特币的烧电工作证明。
