这是英伟达全新的 750，000 平方英尺建筑 TechCrunch 的首次亮相

原文：https://web.archive.org/web/https://techcrunch.com/2018/05/29/this-is-the-first-look-at-nvidias-wild-new-750000-sq-ft-building/

这是第一次看到 Nvidia 的新的 75 万平方英尺的建筑

Nvidia 正在准备一个新的，巨大的建筑在加州圣克拉拉，这是它。它被称为旅行者，将比 Nvidia 刚刚完成建造的大楼大 25 万平方英尺。就像另一个建筑 Endeavor 一样，Voyager 将分享同样狂野、独特的 Nvidia 设计。

Nvidia 在 2017 年开设了第一座建筑 Endeavor，感觉就像一个人可以进入 Nvidia GPU。到处都是绿色的尖角。三角形天窗布满屋顶，也是三角形。这只是两层楼，由于当地的分区法律，似乎新的建筑将遵循同样的趋势。

旅行者号占地 750，000 平方英尺，位于圣克拉拉的奋进号旁边。这使得这两栋大楼的总面积达到 125 万平方英尺，比苹果在库比蒂诺的新总部的一半还少一点。Nvidia 告诉我，该公司仍在规划大楼的人员配置，但预计大楼将容纳越来越多的工程师团队。该公司在全球拥有 11，500 名员工，其中 5，000 名在圣克拉拉。

Nvidia 正在使用 Gensler 设计这座建筑，它将由敌无双建造，第一座建筑就是由同一家公司设计的。预计建设将于下月开始，耗时三年半。

我问英伟达为什么这些建筑以著名的航天器命名。回应很有意思。Endeavor 和 Voyager 有“En”和“V”的初始音，因此 Nvidia 的初始音也是如此，而这些名称也暗示了一种推动前沿的感觉。俗气？有点，但还是很聪明，我买了。
