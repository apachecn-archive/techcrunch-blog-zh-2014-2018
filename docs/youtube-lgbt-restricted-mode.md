# YouTube 回应关于其受限模式审查 LGBT 视频的投诉

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/03/19/youtube-lgbt-restricted-mode/>

在担心其受限模式隐藏了 LGBT 内容的视频后，YouTube 声明“对任何混淆感到遗憾。”

在其 YouTube Creators 账户的一条推文中，该公司表示:

> “我们非常自豪能够在我们的平台上代表 LGBTQ+的声音——他们是 YouTube 的重要组成部分。受限模式的目的是过滤掉成熟的内容，以满足一小部分想要更有限体验的用户。LGBTQ+视频在限制模式下可用，但讨论更敏感问题的视频可能不会。我们对由此造成的任何混乱感到抱歉，并正在调查您的问题。我们感谢您的反馈和热情，让 YouTube 成为一个包容、多元和充满活力的社区。”

YouTube 的一位发言人也向 TechCrunch 发送了一份录音声明，称“受限模式是一个可选功能，由一小部分希望获得更有限的 YouTube 体验的用户使用。包含 LGBT 主题的视频在受限模式下可用，但是，一些涉及健康、政治和性等主题的视频可能不会向选择使用该功能的用户和机构显示。”

[更新:在最初有点不温不火的回应后，YouTube 已经[就受限模式的缺点](https://web.archive.org/web/20230307220134/https://youtube-creators.googleblog.com/2017/03/restricted-mode-how-it-works-and-what.html)发表了更直接的道歉:“底线是这个功能没有按照它应该的方式工作。我们很抱歉，我们会解决这个问题。”]

上周四，当 YouTube 上的 Rowan Ellis 发布了一个关于这个问题的视频，标题为[“YouTube 是反 LGBT 的？”](https://web.archive.org/web/20230307220134/https://www.youtube.com/embed/Zr6pS07mbJc)

此外，标签#YouTubeIsOverParty 和#YouTubeRestricted 周末开始在 Twitter 上流行，其他用户报告说，他们关于 LGBT 主题的视频被隐藏在限制模式下。

根据 YouTube 的说法，[受限模式依靠](https://web.archive.org/web/20230307220134/https://support.google.com/youtube/answer/174084?co=GENIE.Platform%3DDesktop&hl=en)“社区标记、年龄限制和其他信号来识别和过滤掉潜在的不当内容。”父母、学校、图书馆和其他机构经常使用它来阻止儿童找到含有色情、图形暴力和其他令人反感的内容的视频。

然而，涉及同性关系的视频也被隐藏了(Gizmodo 指出，包括[一对女同性恋夫妇的婚礼](https://web.archive.org/web/20230307220134/http://gizmodo.com/youtubes-restricted-mode-is-hiding-some-lgbt-content-1793382337))。埃利斯在 YouTube 上的个人简介显示，她拥有约 22600 名订户，专注于“从女权主义者和酷儿的角度看流行文化”，她声称自己的约 40 个视频已被过滤出限制模式。

埃利斯在关于这个问题的视频中说，即使审查是一个意外，它也有一个有害的影响，因为它使 LGBT 儿童更难找到支持:“我认为研究为什么 LGBT 内容被认为是不合适的，这真的很重要。这远远不止是 YouTube 可能犯的一个错误，他们会引起人们的注意，并在以后予以纠正。”

“YouTube 是唯一一个同性恋和跨性别青年，同性恋青年，双性恋青年，泛性青年，无性青年，这些孩子中的任何一个，都有办法进入社区，有办法了解知识，有办法感觉他们并不孤独，”她补充说。
