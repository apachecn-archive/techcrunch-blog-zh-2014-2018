# 工作流自动化初创公司 Workato 宣布投资 1000 万美元启动 A 轮融资

对于这家初创公司来说，这是一个重要的日子，因为它宣布了一笔 1，000 万美元的 A 轮融资，以及其工作流自动化平台“图灵”的最新发布

这轮融资由 Storm Ventures 牵头，战略投资者 Salesforce Ventures 和 Workday Ventures 参与其中。这家成立四年的公司现在已经筹集了 1600 万美元的资金。

Workato 通过尽可能自动化各种 SaaS 应用程序和 API 之间的连接，帮助公司简化工作流集成。您可以看到为什么 Salesforce 和 Workday 认为在这种情况下投资他们是合适的。

SaaS 应用程序的吸引力之一是自助服务的能力，但当涉及到在应用程序之间建立连接或工作流时，事情就变得有点复杂了。Workato 的首席执行官 Vijay Tella 说，通常情况下，这需要去一趟 IT 部门，以创建跨工具的简单连接。

他说，这不仅仅是终端用户寻求独立的问题。It 部门还希望为营销、销售、财务和其他部门的知识工作者提供一种工具，让他们自己构建不需要开发人员技能的集成。

该公司创建了 Workato，通过尽可能自动化和建议跨工具的逻辑流程，使这些类型的最终用户能够更容易地构建工作流。该解决方案依赖于底层的机器学习算法来以交互方式驱动这些建议。Tella 说，随着用户调整这些食谱以适应他们的需求，系统会随着时间的推移学习并提供更完整的食谱。

Tella 说，在图灵测试期间，55%的食谱是自动编写的，并以建议的形式呈现给用户，用户可以通过做出一系列选择来接受或调整。

由于没有一个过程是完全可靠的，所以当一个配方由于某种原因在跨系统建立连接时中断时，该公司正在使用机器学习来进行自我纠正(或者至少提供可能的解决方案)。这包括用简单的语言而不是代码向用户显示配方流程，这应该使他们能够修复和重新运行。

Workato 声称 78%的客户在第一周就开始使用他们的产品。这有助于 Workato 使用 Salesforce 和 Workday(惊喜，惊喜)以及 Zendesk、Slack 和许多其他公司的典型集成，预先打包了许多现成的配方。

该公司报告说，它的产品正在被超过 21，000 个组织使用，包括 Box、IBM、Cisco、Ideo 和 Credo。
