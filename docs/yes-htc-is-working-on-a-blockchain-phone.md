# 是的，HTC 正在开发一款“区块链手机”

在发布最新旗舰产品的几周前，HTC 刚刚发布了另一款硬件。虽然这家台湾公司近年来整合了许多移动产品，但它今天在 2018 年区块链共识大会上宣布，其即将推出的 [Exodus 手机](https://web.archive.org/web/20230325190824/https://www.htcexodus.com/#reserve-now)采用了每个人最喜欢的科技流行语。

那么，到底是什么让一部手机成为区块链手机呢？主要是安全和加密货币支持。根据 HTC 的 Exodus 登录页面，“我们的愿景是通过创造世界上第一款致力于分散应用和安全的手机来扩展区块链生态系统。随着 HTC Exodus 的发布，我们现在可以让这成为现实。”

Exodus 将支持比特币和以太坊等，这得益于通用钱包、安全硬件和去中心化应用。据 The Next Web 报道，HTC 还概述了创建一个本地区块链网络的计划，由此加密货币可以在 Exodus 用户之间进行交易。自然，用户也将能够使用加密货币购买手机本身。然而，价格和发布日期尚未透露。

除此之外并没有太多的信息，但 HTC 显然正在努力制造轰动，因为其数字在不断下滑的智能手机市场中的整体比例已经缩水。然而，即使对加密货币领域的认识和兴趣迅速增加，也很难想象《出埃及记》会引起多大轰动。
