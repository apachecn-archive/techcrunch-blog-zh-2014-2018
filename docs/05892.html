多亏了谷歌，你的手机现在可以兼作光剑

每次我看到别的东西在推《星球大战:原力觉醒》，我都会翻白眼。然后我试着做一些我认为“好吧，这很聪明”的事情迪士尼与人类所知的每一个品牌合作来推广这部电影，这部电影昨晚在洛杉矶首映。迪士尼的合作伙伴之一是谷歌。

如今，来自迪士尼、谷歌、卢卡斯影业和工业光魔公司的一款名为“光剑逃生”的新体验在各地的 Chrome 浏览器上热卖。它的承诺是通过网络将你带入生活，让你“沉浸”在电影中。如果你有 18 号开幕式的门票，我会坚持购买，因为那将是一次终极体验。

所以事情是这样的。你在一艘歼星舰里，你的手机被用作光剑。所以，是的，这涉及到你的浏览器和你的手机(任何有网络接入和 Chrome 的手机都可以)。你开始做其他的事情——与你电脑上弹出的帝国冲锋队战斗。

在你的桌面上点击g.co/lightsaber，你会得到一个唯一的网址来进入你手机上的 Chrome 版本。你必须“校准”，这只需要一秒钟，然后你进入训练模式。保持电脑和手机的声音，以获得最佳体验。当你训练的时候，你用你的手机军刀挡住冲锋队员的射击。很整洁。

[gallery ids="1251774，1251776"]

一旦你偏转了足够多的子弹，你就进入飞船，战斗才真正开始。不会毁了你的，试试吧。

你看到的 3D 图形是用 WebGL 构建的，3D 渲染在你的电脑上非常快。你的手机和电脑之间的实时通信要归功于 WebRTC 和 WebSockets，我没有发现任何延迟。不需要插件。是的，谷歌通过这次经历展示了它的网络能力。

如果你非常了解谷歌是如何做到这一点的，那就去看看它的开发者案例研究。如果你已经看够了《星球大战》,也许可以把这本书传给你的朋友。我保证，一旦你看到它在他们的屏幕和手机上亮起，你就会想亲自尝试一下。
