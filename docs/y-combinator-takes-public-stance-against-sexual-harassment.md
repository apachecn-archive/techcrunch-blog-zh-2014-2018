# Y Combinator 公开反对性骚扰 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/08/12/y-combinator-takes-public-stance-against-sexual-harassment/>

在下周的演示日之前， [Y Combinator](https://web.archive.org/web/20230129095705/http://www.crunchbase.com/organization/y-combinator) 在周二的一篇博客文章中提醒[的投资者，它对针对创始人的性骚扰采取零容忍政策。](https://web.archive.org/web/20230129095705/http://blog.ycombinator.com/a-reminder-to-investors)

“想都别想去做，” [Y Combinator 的杰西卡·利文斯顿](https://web.archive.org/web/20230129095705/http://www.ycombinator.com/partners/)写道。“我会查清楚的。Y Combinator 不会继续和你合作。”

这篇文章之前，围绕女性创始人在为初创公司寻找投资者时经常发现的性挑逗和歧视的报道越来越多。《连线》杂志刊登了经历偏见的女性创始人的故事，以标题[提醒读者，“这就是科技丑陋的性别问题的真实面貌。”](https://web.archive.org/web/20230129095705/http://www.wired.com/2014/07/gender-gap/)随后，[《福布斯》公布了一位女性创始人的匿名账户](https://web.archive.org/web/20230129095705/http://www.forbes.com/sites/jeffbercovici/2014/08/07/what-its-like-raising-money-as-a-woman-in-tech/)，她说，如果女性想要向前一步，她们最好“武装起来”

一些人甚至在 Secret 上讨论这个问题，其中有一个帖子专门问哪些男性风险投资家对女性创始人进行不适当的性挑逗，并“诱骗她们约会”《秘密邮报》受到了多项诽谤和中伤的指控，评论者认为这不是披露犯有这种行为的风险资本家姓名的合适平台。

Secret 是硅谷关于口交和最新解雇事件的匿名八卦网站，不应该成为严肃报道此类冒犯和不当行为的地方。但创业者报告自己的成就这一事实突显出，在不损害自身声誉的情况下，女性创业者几乎没有其他选择。

他们没有人力资源部门，对这种指控的反弹的恐惧使得他们甚至难以与商业伙伴和其他投资者交谈，特别是考虑到投资者与初出茅庐的企业家相比所拥有的相对权力。

一个年轻公司的女性创始人直接陷入了有钱(主要是男性)看门人的黑暗深渊，可能依赖于继续容忍这种歧视行为的投资公司提供的资金。

自从《福布斯》杂志的匿名文章和《秘密》的帖子发表后，一些人认为这个问题不可能得到解决，除非女性开始公开自己的名字。但责任应该落在行为不当的投资者身上，让他们改变自己的行为，也落在袖手旁观的同事身上，让他们说点什么。那些被骚扰的人不应该因为想要隐私而受到责备。 [Re/code 创始人卡拉·斯威舍](https://web.archive.org/web/20230129095705/http://recode.net/author/kara-swisher/)在[说得最好，我上个月参加了一个实习生交流活动](https://web.archive.org/web/20230129095705/https://techcrunch.com/2014/07/23/tech-leaders-tell-interns-what-they-wish-they-knew-at-age-20/)，她提醒主要由男性组成的小组和观众，这不仅仅是“由女性来决定他们应该做什么”投资者，尤其是男性投资者，也需要在这个问题上参与进来。

在最近的媒体关注之前，从投资人到创始人的性骚扰仍然是禁忌。Y Combinator 总裁 Sam Altman 告诉 Valleywag，公司过去已经采取了行动，没有邀请涉嫌违规的投资者参加演示日，也没有给他们推荐人。但通过最终公开谈论这个问题，Y Combinator 为其他加速器和投资者采取类似的零容忍政策铺平了道路。

“几乎所有我们认识的投资者都是完全正直和专业的，但即使是一个不恰当的事件也太多了，”利文斯顿写道。
