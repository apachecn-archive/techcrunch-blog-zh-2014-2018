Spire 的船只跟踪卫星数据使得从太空监控船只变得更加容易

Spire Global 是一家数据分析提供商，它使用一组轨道纳米卫星提供的信息来跟踪公海上的船只，这项服务对海运公司和其他客户来说非常有价值。现在，由于 Spire 用于收集数据的太空和地面技术的普遍改进，它正在为其面向开发者的工具添加两种新产品，使跟踪海上船只变得更加容易。

Spire 通过简化处理大量关于海上船只的数据库条目的过程来工作，企业使用这些条目来决定他们自己的公司如何运营。船舶的位置、目的地和方向会对物流和航运公司如何规划其日常运营，以及包括港口在内的相邻行业如何经营自己的业务产生巨大影响。

Spire 正在为其产品增加两种新产品，包括 Sense Vessels，它可以使用一个 API 调用，随时提供大约 30 万艘海上船只的最新信息。另一款名为 Predict 的新产品将使用机器学习来帮助预测船只在未来的位置，基于包括当前路线、天气和已知舱单数据在内的大量输入。

该公司表示，它可以提供这些新功能，这要归功于它可以接入的地面监测站数量的增加，星座中添加的新卫星，以及向现有轨道卫星群传输的软件升级。Spire 对其星座的策略是追求可承受的容量，通过软件不断更新以保持其能力的提高。由于纳米卫星技术的兴起，这成为可能，这使得在一个曾经由数百万和数十亿美元的硬件主导的市场上发射多颗廉价卫星成为可行的选择。

Spire 目前运营着 40 颗轨道卫星，并计划在未来几年内通过更多的发射、更多的地面站和更多的机器学习改进来推出更多的网络和数据分析改进。
