苹果的 Face ID 在发布时将只支持每部 iPhone 一个注册面孔

苹果即将推出的 FaceID 技术将支持每台设备只注册一张脸，至少在最初是这样。这已经被在本周的 iPhone 发布会上与苹果交谈过的多位人士报告过，并被 TechCrunch 单独证实，但值得注意的是，这可能会改变用户对该设备的一些行为。

Face ID 只支持一张脸，这在表面上很有意义(抱歉)，因为不像触控 ID，用户只有一张脸用于解锁。触控 ID 表面上允许注册多个手指，因为你自己有相当多的手指，根据你如何使用设备或放在哪个口袋里，一个可能比另一个更方便使用。

问题是，人们还经常注册他们重要的其他人，甚至是孩子的指纹，这取决于他们如何使用他们的设备。这款手机最多可以存储五个指纹，这使得在你的手机上保存几个你自己的和其他重要的人的指纹变得相对容易，以防万一。

一部手机，一张脸的政策显然不会支持这种安排:如果你想让你爱的人进入，你需要对着镜头微笑，或者分享一个备用密码，这样他们就可以进入。当然，密码选项只是比注册的触控 ID 指纹稍微不方便，但它仍然有所不同，特别是如果有人只是偶尔访问你的设备，可能不会因此将 pin 提交到肌肉存储器。

我认为，如果苹果最终将 Face ID 产品转移到 iPads 等更常见的共享设备上，这将是一个更大的问题。不过，取决于你如何使用手机，这可能是你选择 iPhone 8 还是 iPhone X 的另一个决定性因素——或者，当然，两者都不是。
