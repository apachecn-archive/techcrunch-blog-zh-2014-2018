微软收购大数据初创公司 Metanautix | TechCrunch

微软今天宣布已经收购了 Metanautix ，这是一家专注于大数据的初创公司，于 2014 年走出秘密。该公司得到了 700 万美元的资本支持，其中包括来自知名风险投资公司红杉资本的现金。

交易条款没有披露。

根据 TechCrunch 之前对 Metanautix 的报道，该公司在 2012 年成立时，外部资金占大多数。当该公司公开发布时，这份出版物报道说，它有“六个测试客户在使用[它的]软件。”尽管如此，它作为独立商店的经营已经结束。

微软打算将 Metanautix 并入其“数据平台，包括 SQL Server 和 Cortana 分析套件。”就目前而言，这个软件巨头在公开场合对这些说明以外的细节轻描淡写。

在这方面，Metanautix 说得更少，它自己的帖子只有 175 个字。它说的话比你想象的还要少，但确实为其首席执行官找到了感谢员工的空间:“这是一段激动人心的旅程，但在我们开始下一个激动人心的篇章之前，我想花点时间感谢我们出色的团队，是他们让这一切成为可能。]"

仅仅筹集了 700 万美元，而且大部分都是在过去几年里筹集的，不难猜测该公司在出售时现金不足，进入了比推出时更具挑战性的筹资环境。

无论如何，微软都有新的大数据血液进入新的一年。
