原文：https://web.archive.org/web/https://techcrunch.com/2018/03/12/former-docker-ceo-ben-golub-joins-storj-as-executive-chairman-and-interim-ceo/amp/

图片来源:大卫·保罗·莫里斯/彭博

去年五月，Docker 首席执行官本·戈卢布在执掌集装箱化先锋四年后离职。从那以后，我们再也没有听到他的消息，但是今天，分布式对象存储服务 Storj Labs 宣布 Golub 将加入其团队，担任临时首席执行官和执行主席。

显然并不缺乏对象存储服务，但 Storj Labs 采取了一种有些不同的方法。该公司自 2014 年成立以来，一直强调安全性和分散存储是其主要优势。不过，除此之外，Storj 还在区块链技术上下了一笔早期赌注，去年，Storj 以 3000 万美元的 ICO 和迁移到项目技术上，把所有赌注都押在了以太坊上。

Storj 背后的基本理念是，任何人都可以向 Storj 提供硬盘空间，以换取 Storj 的令牌。目前，该服务拥有 70，000 名客户，他们在 150，000 个节点上存储数据，这些节点由数万名该公司所谓的“农民”(相对于“矿工”)运营。总的来说，这些节点现在存储超过 60pb 的数据。同样值得注意的是，包括流行的 FileZilla FTP 客户端在内的许多第三方服务现在都支持它，Storj 在 2016 年加入了微软 Azure 的区块链即服务生态系统，以帮助将这项技术引入更多企业。

“对许多人来说很明显，我们正处于计算领域新的重大转变的尖端，由完全去中心化的互联网和新的去中心化的信任和安全模型驱动，由区块链和分布式总账驱动，”Golub 今天在一篇博客中写道。“当我第一次与 Storj 交谈时，我对这个团队很感兴趣，因为他们已经构建了一个强大的平台和充满激情的社区来为这种新的、分散的互联网提供存储层，还有希望使用这一新层的开发人员。”

Golub 还将 Storj 的状态比作 Docker 的早期。当 ras Docker 谈到为互联网建立一个基础设施层时，Storj 想要建立一个存储层。这绝对是一个雄心勃勃的任务，这一概念是否会成功还有待观察。不过目前，Golub 希望专注于为 Storj 打造可持续发展的业务。
