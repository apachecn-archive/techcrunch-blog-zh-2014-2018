# 风险投资的多样性灾难 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/07/30/venture-capitals-diversity-disaster/>

硅谷缺乏多样性的问题触及了该行业的方方面面——创业精神、大型科技公司的人口统计数据和风险投资。虽然科技公司在过去几年里在促进多样性和包容性方面更加谨慎，并取得了一些进展，但风险投资行业在这方面仍然非常缺乏。

只有 1%的风险资本家是拉丁裔，只有 3%是黑人。毫不奇怪，白人占风险投资行业的 70%，根据 T2 平等风险投资公司的合伙人理查德·科比最近的分析。与[Kerby 2016 年的分析](https://web.archive.org/web/20221025230217/https://techcrunch.com/2015/03/10/why-are-there-so-few-black-investors/)相比，女性现在占风险投资行业的 18%，而当时仅为 11%。在交叉层面上，黑人和拉丁裔女性在风险投资行业中所占比例为零。

与此同时，40%的风投去了斯坦福或哈佛。

“就两所学校！，" [Kerby 写在介质上](https://web.archive.org/web/20221025230217/https://medium.com/@kerby/where-did-you-go-to-school-bde54d846188)。“那是为什么呢？每个人都希望与自己最相似的人一起工作，教育、性别和种族是让人们在他人身上找到相似之处的属性。”

他补充说，“这个行业 82%是男性，近 60%是白人男性，40%来自两个学术机构，难怪这个行业感觉如此孤立，不太像精英管理，而是更像一个镜像政府。”
