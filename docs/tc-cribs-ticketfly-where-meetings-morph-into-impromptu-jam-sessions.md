# TC 婴儿床:Ticketfly，在这里会议演变成即兴即兴即兴即兴即兴会议

在这一集的 Cribs，我们前往 [Ticketfly](https://web.archive.org/web/20230130235707/http://www.ticketfly.com/) ，一个为各种现场活动运行售票平台的初创公司。

正如你在上面嵌入的视频中所看到的，去 Ticketfly 总部本身就是一种活动。从卢浮宫到以前的纽约俱乐部 CBGB，这个地方装饰着各种著名的场所，乐器也触手可及。

[gallery link = " file " ids = " 1064252，1064251，1064255，1064257，1064303，1064261，1064284，1064286"]

由于 Ticketfly 的工作人员中有很大一部分是音乐家，这些乐器不仅仅是坐在那里收集尘埃。与 Ticketfly 首席执行官安德鲁·德累斯顿(Andrew Dreskin)的会面演变成一场正式的即兴表演并不罕见——正如我们在巡演结束时发现的那样。任何以演奏铃鼓结束的会议都是一次很好的会议。
