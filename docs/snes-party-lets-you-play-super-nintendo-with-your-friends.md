# SNES.party 让你和朋友一起玩超级任天堂

紧随精彩的 [NES.party](https://web.archive.org/web/20230201214502/http://nes.party/) 而来的是 Haukur Rosinkranz 的 [SNES.party](https://web.archive.org/web/20230201214502/http://snes.party/) ，一个让你和所有伙伴一起玩超级任天堂的网站。

Rosinkranz 是冰岛人，但现在住在柏林。一年前，他在试验 WebRTC 和 WebSockets 时创建了 NES.party，并更新了软件以支持 SNES。

“我制作它的原因很简单，因为我发现 Chrome 中的 RTC 实现已经变得多么先进，所以想用它做点什么，”他说。“当我发现可以通过网络传输视频元素时，我就知道我必须做一些很酷的事情，于是我想到了流式仿真器的想法。”

他说，他花了六个月的时间来开发这款应用，又花了一个月的时间来添加 NES 支持。

“很难说花了多长时间，因为我基本上为需要一个或多个参与者之间实时通信的 web 应用程序创建了自己的框架，”他说。他是一名自由程序员。

这是一个聪明的小技巧，可以给你原本沉闷的一天增添一点乐趣。感觉像是回到了过去？过来[，我们一起玩](https://web.archive.org/web/20230201214502/https://snes.party/t5475e)！
