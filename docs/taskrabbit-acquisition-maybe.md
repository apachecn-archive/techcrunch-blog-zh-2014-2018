# 零工经济坚定的 TaskRabbit 正在考虑出售

所谓的“共享经济”或“零工经济”的最早和最著名的创业公司之一，正在评估出售自己的可能性。据 Recode 报道，自由职业市场 [TaskRabbit](https://web.archive.org/web/20221208011605/https://www.taskrabbit.com/) 承认，在收到一个潜在战略买家的意向后，它正在考虑出售。

TaskRabbit 于 2008 年初推出，旨在为当地有空闲时间的兼职人员匹配各种类型的零工。在早期，这项业务在一个非常松散的市场模式下运营——用户可以列出他们需要完成的工作以及他们愿意为这些服务支付的价格，所谓的“任务兔”可以选择接受或不接受这些工作。

这在当时是一个非常新颖的概念，但没过多久，许多其他初创公司也突然提供了类似的功能。随着时间的推移，这些零工经济公司开始围绕特定类型的工作来定位自己，许多公司试图成为清洁服务或家居装修的*。*

又过了几年，在对零工经济产生了一些最初的兴奋之后，后续风险资本枯竭了，TaskRabbit 的许多竞争对手要么倒闭，要么被廉价收购。

所以我们在这里。在从 Shasta Ventures、Founders Fund、First Round、Floodgate、CollabFund 和其他投资者那里筹集了 3800 万美元之后，其他投资者没有兴趣继续向 TaskRabbit 这样的公司投资。

那现在怎么办？所有迹象表明，杂工服务、轻型房屋翻新和家具组装最终将成为一项价值数十亿美元的业务。因此，如果有一个或多个买家感兴趣，TaskRabbit 探索出售的可能性是有道理的。

唯一的问题是这些买家可能是谁，以及他们可能愿意为这项业务支付多少钱。
