# 斯沃琪重申将打造一款具有支付功能的智能手表 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/05/28/swatch-reafirms-its-building-a-smartwatch-with-payment-features/>

# 斯沃琪重申，它正在打造一款具有支付功能的智能手表

斯沃琪仍想参与智能手表游戏。据首席执行官尼克·海耶克称，这款手表将支持互联网，并具有远程支付功能，尽管具体细节仍有待商榷。哈耶克在最近的一次股东会议上表示，其即将推出的智能手表将在瑞士和“一个大国”推出。

尚不清楚斯沃琪瞄准的是哪个大市场。正如《华尔街日报》报道的那样，斯沃琪已经与中国银联达成了支付协议，而中国是斯沃琪最大的市场。

根据[过去对 Hayek](https://web.archive.org/web/20221007231049/http://www.bloomberg.com/news/articles/2015-02-05/swatch-plans-smartwatch-to-compete-with-apple-watch-s-debut) 的采访，这款手表将不需要充电，并将支持 NFC。但是，至少在 2 月份该公司宣布其计划时，智能手表将无法与 iOS 设备兼容；它是为 Windows 手机和 Android 设备设计的。

斯沃琪集团是世界上最大的制表集团，为斯沃琪、天梭、浪琴和欧米茄手表生产时计。苹果手表和智能手表总体上对其业务构成了威胁。据报道，去年该公司对现代智能手表没有任何兴趣，但今年早些时候改变了态度。

斯沃琪已经尝试了许多有趣的——尽管不太受欢迎——智能手表，包括液晶手表，其特点是所谓的[互联网时间](https://web.archive.org/web/20221007231049/http://www.timeanddate.com/time/internettime.html),这是一个统一全球时区的不可靠系统，以及前面提到的 SPOT 手表。SPOT 是一个基于调频的传输系统，它将数据推送到一系列大型液晶手表上，为早期加入这项服务的少数客户提供短信、体育比分和天气预报。

袖手旁观可能会损害斯沃琪在智能手表市场产生有意义影响的机会。在这一点上，斯沃琪正在与老牌公司竞争，其中许多公司已经在使用第二代或第三代设备。
