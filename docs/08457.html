《白噪音 7》已经过时，希望它能像 Instagram 一样发出宁静的声音

原文：https://web.archive.org/web/https://techcrunch.com/2016/07/28/white-noise-7-is-out-and-wants-to-be-like-instagram-for-restful-sounds/

2008 年，软件开发人员托德·摩尔(Todd Moore)为他现在著名的白噪声应用程序编写代码时，对他来说这是一个附带项目，也是苹果应用商店上线的首批移动应用之一。

该应用程序很快攀升至健康与健身类别的首位，多年来一直是首选。摩尔说，在这一点上，白噪声应用程序每天有 50 万到 100 万活跃用户，并且已经安装了数千万次。

他说，这些年来，他已经记不清不同平台上的独特安装。该应用程序现在可以在 iOS 和 Android 设备上使用，包括连接的电视、智能手机、平板电脑以及 Mac 和 Windows 电脑。有用户付费下载的高级专业版本。

自 2008 年以来，软件开发商出身的 bootstrap 企业家创办了一家小型企业, TMSOFT LLC，支持白噪声，并开发其他免费移动应用和音频内容。

该公司的其他一些创作包括一款受欢迎的移动益智游戏，名为强迫性游戏，以及一款名为盒扇的白噪声游戏，该游戏在 Spotify 上获得了约 500 万次旋转。

这位首席执行官说，今天，TMSOFT 发布了其旗舰应用《白噪声》(White Noise)的第 7 版，并引入了一些主要的新功能，这是受到许多应用用户的启发，他们会要求摩尔公司的音频工程师为他们录制怀旧的声音。

该应用程序现在允许用户上传他们生活和旅行中的声音，在白噪声市场上分享它们，并从这些声音片段中创建完美的循环，不间断且无缝，因此比粗略的音频循环更可能帮助人们睡眠或集中注意力。

该应用程序受广告支持。在白噪声市场上传声音的用户是免费的，并不是为了出售他们的作品，因为他们可能会在 Shutterstock 、 Pond5 或 Unstock.io. 等市场上出售照片或视频

摩尔说，“我只是认为人们已经太习惯于在 Instagram、YouTube 或脸书上免费获取内容，因此没有理由对这种事情收费。”

White Noise 7 还支持西班牙语，这可能有助于该公司在美国以外甚至美国国内的新市场扩大用户群。

摩尔推荐了一些新的用户生成的声音在平台上测试，包括水流过汉斯·穆尔的水雕的声音，科罗拉多的蒸汽火车的声音，还有他个人最喜欢的睡觉的声音，由 TMSOFT 的声音工程师创造的 T4 帐篷雨的声音。

这位首席执行官说,《白噪音 beta 测试中录制的声音来自世界各地，包括美国、百慕大和萨尔瓦多等。

这款应用面临着竞争，至少是间接的竞争，来自过多的小型模仿者，以及 Spotify、SoundCloud 和 Pandora 等流媒体音乐服务，用户可以在这些服务中找到大量冥想的声音。

但这些都没有自动工程工具，旨在使声音循环无缝，和实际的白噪音。
