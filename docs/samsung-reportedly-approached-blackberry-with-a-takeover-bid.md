# 据报道，三星试图收购黑莓(更新:黑莓否认)

**更新:**黑莓已经回复[官方否认](https://web.archive.org/web/20230330233614/http://www.marketwired.com/press-release/blackberry-responds-to-media-report-nasdaq-bbry-1983083.htm?hootPostID=51e4918cbbf7a0700c00cec669b04b7d)收购谈判，但有点矛盾的是，它还声称不对谣言或猜测发表评论。完整声明如下:

> 黑莓有限公司(纳斯达克股票代码:BBRY)(TSX 股票代码:BB)(以下简称“黑莓”)了解今天发表的某些媒体报道，这些报道涉及三星可能提出收购黑莓。黑莓尚未与三星就任何可能的收购要约进行讨论。黑莓的政策是不评论谣言或猜测，因此它不打算进一步评论。

据路透社今日[报道，三星已经向黑莓提出了潜在的收购要约。这家韩国公司出价高达 75 亿美元收购这家加拿大公司，特别是对这家较小公司的专利组合感兴趣。](https://web.archive.org/web/20230330233614/http://uk.reuters.com/article/2015/01/14/us-blackberry-m-a-samsung-exclusive-idUKKBN0KN2FW20150114)

三星的出价介于每股 13.35 美元至 15.49 美元之间，相当于新闻发布时该公司交易价值的 38%至 60%。据路透社(Reuters)报道，这一对话显然仍在进行，上周该公司高管和可能交易的顾问举行了一次会议。

三星的兴趣似乎主要在知识产权方面，这可能会浇灭那些希望这家韩国公司也提供面向企业的键盘运动智能手机的人的希望。然而，三星过去对企业市场表现出了明显的兴趣，推出了适用于 Android 的 KNOX 安全数据服务，然后与谷歌合作，将其部分功能直接整合到 Android 5.0 中。

据推测，收购黑莓还将为三星提供该公司前景广阔的企业服务部门，包括 BES 多平台 MDM 解决方案、黑莓信使(越来越侧重于企业)等。如前所述，知识产权可能是主要的推动因素，但三星不可能忽视为自己提供更大企业影响力的捷径的价值。

如果三星收购黑莓的交易成功，硬件部门可能是风险最大的，但三星也有更多的自由继续在这一领域进行实验，如果它认为这样做有好处的话。这不是黑莓第一次表示对包括三星在内的追求者感兴趣，但听起来谈判可能会比过去更深入。
