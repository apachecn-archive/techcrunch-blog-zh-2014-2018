哈迪·帕托维是一名科技投资者和企业家，也是西雅图非营利组织Code.org的联合创始人兼首席执行官。 有钱的巴顿格伦·凯尔曼埃德·拉佐夫斯卡阿里·斯坦伯格 促成了这篇文章的写作。

在过去的十年里，西雅图的科技领域发生了巨大的变化。

在20世纪90年代和21世纪初，镇上唯一的大型软件公司是微软和亚马逊。有一段时间，大多数西雅图的初创公司只有两种可行的退出策略，这甚至成了常识:上市，或者被微软收购。

这导致了一个不平衡的创业生态系统，只有极少数的科技巨头和大量相对较小的创业公司，中间几乎没有。

除非你的初创公司明确地走上了IPO之路，否则替代成功战略的缺乏会对初创公司生态系统造成抑制效应。无论你是在寻找投资者还是试图招募明星人才，都很难找到一家唯一的退路是“被微软收购”的初创公司。

当然，当一个人投资一家初创公司时，他的目标是获得彻底的成功。但有相当有吸引力的后备选择仍然很有吸引力，尤其是对于那些害怕跳槽到新公司的新员工来说。

最近，作为一种有吸引力的退路策略, acqui-hire 的兴起，激励了更多的硅谷企业家和工程师尝试创业。但西雅图一直缺少同样的激励，因为历史上大多数科技初创公司的潜在收购者非常少。

在过去的10年里，西雅图的这种动态发生了翻天覆地的变化，这是由两种力量造成的。

变化的第一部分是新一代大型西雅图科技公司的崛起——这些公司的规模仍小于当地的两大巨头微软和亚马逊，但足以填补科技生态系统的中间层。

这一群体包括上市公司，如 Expedia 、 Zillow 、 Tableau 、 Zulily ，以及超大型收购公司，如 PopCap Games 、 Isilon 、 Big Fish Games 或 Bluekai 。除了Adobe和Real等老牌公司，西雅图本土的科技行业现在已经有相当多的公司，不仅估值达到1000亿美元，而且估值在100亿美元、10亿美元或1亿美元之间。

谷歌(Google)在这座城市设立分支机构十年后，西雅图见证了硅谷公司建立第二个工程办公室的爆炸式增长。

第二股力量是西雅图市区硅谷工程办公室的不断增加。谷歌是第一批在西雅图开设工程办公室的硅谷大公司之一，事实上谷歌现在有两个工程办公室——一个在西雅图市中心，一个在华盛顿州的柯克兰郊区。

自从谷歌第一次扩展到西雅图以来，许多团队生产力和工程生产力工具已经出现，并使团队更容易跨越地理界限一起工作——无论是Atlassian的 HipChat ，最近的 Slack ，还是像 GitHub 这样的工具。

谷歌(Google)在这座城市设立分支机构十年后，西雅图见证了硅谷公司建立第二个工程办公室的爆炸式增长。西雅图现在是谷歌、脸书、苹果、 Twitter 、 Salesforce 、易贝、 Dropbox 、优步、 SpaceX 、 Taser 、 Palantir 、 Groupon 、、 Pivotal Labs ，还有很多其他的(请随意在评论中列出我错过的)。

甚至中国的阿里巴巴也开设了西雅图工程办公室，据传将成为中国电子商务巨头的美国总部。许多这样的办公室是最近两年才出现的(苹果、Dropbox、优步、Twitter、Hulu、SpaceX)。

许多这样的办公室正计划雇佣成千上万的工程师。这些西雅图前哨中的一些甚至已经开始收购较小的初创公司，而不需要搬迁到湾区——例如当谷歌收购Picnik 时，或者当 Salesforce收购Thinkfuse 时——为西雅图的最新企业家提供了一种新的退出策略。

在这一点上，西雅图的技术场景与七年前明显不同。

有一种误解认为，科技公司为了降低成本会扩张到硅谷以外——因为软件工程师对薪水的要求太高。事实显然不是这样——事实上，西雅图软件工程的平均工资仅次于湾区。更重要的因素是，硅谷根本没有足够多的高素质、经验丰富的软件工程师来满足这些公司的雄心和增长潜力。当他们寻求扩张以开设新的工程办公室时，西雅图是第一站，原因有三:

(1)西雅图拥有最多的软件工程师，部分原因是微软在20世纪80年代和90年代最初的招聘能力。

事实上，华盛顿州是最普遍的工作职业是“软件开发人员”的四个州之一(其他三个是弗吉尼亚、科罗拉多和犹他)。根据劳工统计局的数据，西雅图-贝尔维尤-埃弗雷特大都市地区的软件开发人员比其他任何科技地区都多，例如，比圣何塞-桑尼维尔-圣克拉拉大都市地区都多。

(2)华盛顿大学是全美排名前十的计算机科学专业之一，希望留在该地区的毕业生源源不断。波士顿一家初创公司最近对LinkedIn数据的分析发现，在过去两年里，UW大学毕业生中，将自己的职业列为“软件工程师”的人有90%居住在西雅图！。《T4时报》广泛报道了UW CSE在2012年对当地科技行业的影响。

(3)西雅图和旧金山在同一个时区，所以安排会议不需要跨海岸协调。由于阿拉斯加航空公司，西雅图和圣何塞或旧金山之间的旅行是小菜一碟，每天单程有8和9个直达航班。(这些航班在业内被称为“书呆子鸟”，因为都是科技行业的乘客)

在这一点上，西雅图的技术场景明显不同于7年前，即2008年，当时Redfin的Glenn Kelman写道，西雅图将成为“下一个硅谷”，或者纽约时报的John Markoff写道，人才和资本以及该地区三大公司(微软、亚马逊和谷歌)的分支机构大量涌入。七年后，生态系统发生了变化。

这里聚集了几乎所有科技界的大腕，无论他们是本土的，还是开设第二家店铺的。“婴儿”创业公司已经成长起来，导致西雅图每年都有多起科技公司IPO。

毫无疑问，西雅图仍然比硅谷小。事实上，Expedia和Zillow的联合创始人Rich Barton最近将西雅图描述为“明星四分卫(硅谷)的金发邋遢小弟”

私下里，他补充道:“做那个小弟弟真的很棒…他的女朋友把你的头发弄乱，说你可爱，等等。显然，资金和人才沿着硅谷和西雅图之间的高带宽路线流动。”事实上，下一批西雅图的IPO，如Redfin或Apptio，在硅谷和西雅图都有投资者，这种情况越来越普遍。

唯一没变的是。它仍然比加利福尼亚多云得多。或者像艾伦人工智能研究所的柳文欢·埃齐奥尼比较在西雅图和在三藩市工作时所说的那样:“人们不会因为太多的阳光而分心。”
