在过去的几个月里，我花了很多时间把耳机绑在脸上。我不知道这是极大地丰富了我的生活，还是让它变得一团糟，但我可以报告说，游戏至少开始变得非常有趣，视频内容也越来越好。

PlayStation VR今天发布，消费者有三个很好的选择来接受高质量的VR，但是，首先，你应该买一个吗？

如果你是一个铁杆游戏玩家，是的。如果你是一个长期的技术早期采用者，是的。如果你是一个看火人节或者总是谈论火人节的人，是的。对于其他人，我相当怀疑。基本上，不要为这些事情节省你的津贴，一个Gear VR应该比你多。

我想我在过去几个月中得到的一个启示是，这些PC驱动的耳机的更新周期可能会更符合游戏主机(可能3-4年)，而不是像移动电话那样的东西。

对于消费者来说，个人投资是如此之高，由于快速发展的光学/传感器，耳机中的技术飞跃将是如此重要，以至于迭代之间可能不会有太多的兼容性。换句话说，如果你觉得你有兴趣，但打算等到第二轮这些高性能耳机，请准备好等到2019年或2020年。

如果你还在阅读，并觉得可能会购买虚拟现实，这里有一个主要耳机的快速纲要。

PlayStation VR

+ $499耳机+摄像头+控制器
+ 需要PS4系统
+ 使用传统Move运动控制器
+ 显示分辨率:1080×960有机发光二极管(每只眼睛)
+ 显示刷新率:120Hz

HTC Vive

+++需要一台结实的PC
+steam VR OS
+带运动控制器的房间级跟踪
+ 显示分辨率:1200×1080有机发光二极管(每只眼睛)
+ 显示刷新率:90Hz

Oculus裂缝

++599美元的耳机和Xbox One控制器、
++199美元的触控控制器(12月6日上市)
+ 需要一台结实的PC
+Oculus Home OS
+显示分辨率:1200×1080 AMOLED(每只眼睛)
+ 显示刷新率:900

哪一个？

好了，这些是耳机。现在你应该买哪一个？实际上，我会在这里给你一个真实的答案。

如果你不想买很贵的电脑，那就买PS VR吧。这肯定不是最好的耳机，但价格合适，你不用担心定期升级系统来跟上GPU的脚步。此外，索尼与游戏工作室的现有关系可能会导致一些AAA级游戏首先登陆该平台。

如果你是一个VR的死忠，个人投资确保虚拟现实仍然是一个开放的平台，购买HTC Vive。虽然Oculus和PlayStation都是独占的，但HTC正在购买Valve的SteamVR系统，希望确保一个内容可以在耳机上播放。

但最终，如果你愿意花钱，就买Oculus Rift吧。Vive可能是目前最好的整体体验，它的跟踪非常棒，但新的触摸控制器(将于12月6日推出)是其中最好的，我在OC3上看到的令人敬畏的Oculus独家游戏数量令人震惊。一些新技术将允许你在一台不太强大的电脑上运行Rift。此外，Oculus与其母公司脸书的密切合作将确保这家虚拟现实巨头保持在非游戏社交应用的前沿，这无疑将增长其网络潜力。Xbox One控制器时代导致了乏善可陈的VR内容体验，但耳机的最佳时代仍在前方。

现在为时尚早，但也不算太早。你不会真的在这些耳机中出错——它们在功能、复杂性和价格上都比谷歌Cardboard高出一大截——但基于你的需求，以上可能会帮助你找到最佳体验。
