Stripe 向美国初创公司开放 Atlas 项目

原文：https://web.archive.org/web/https://techcrunch.com/2017/04/06/stripe-atlas-us/

一年多前， Stripe 推出了它的 Atlas 项目，作为帮助国际企业家在美国注册和建立业务所需的一切的一种方式。现在，这家支付初创公司正在开放该项目，作为一种促进已经在美国的创始人创建公司的方式。

Stripe Atlas 旨在为初创公司提供必要的法律、财务和运营工具。通过与硅谷银行合作进行融资，与 Orrick 合作进行法律公司事务，与 PWC 合作进行税务指导，与 AWS 合作进行云服务，Stripe 计划为公司提供了一种快速组建特拉华州公司的方式，并建立了他们需要的所有银行、税务和技术基础设施。

Stripe 推出 Atlas 作为仅限受邀者的实验，部分是为了让更多的国际公司使用其支付 API。成为 Stripe 客户并不是使用 Atlas 的必要条件——毕竟，处于组建阶段的公司不太可能需要支付帮助。但 Stripe 非常专注于 Atlas 是国际企业家的事情，在过去的一年里，来自全球 124 个国家的数千家公司已经注册。

然后有趣的事情发生了:美国的企业家也开始要求利用 Atlas。Atlas 项目负责人泰勒·弗朗西斯说，他们不仅仅是初出茅庐的新企业家。在要求加入 Atlas 的美国企业家中，62%的受访者之前已经创办了一家公司。

换句话说，在经历过这一过程之后，他们只是在寻找一条更简化的公司成立之路。Stripe 决定迎合他们的需求，并向美国和海外的初创公司开放 Atlas。

该项目仍将只接受邀请，部分原因是 Atlas 仍然专注于高增长的初创公司，或那些有潜力成为初创公司的公司。正如弗朗西斯所承认的，其目的不是为小企业或个体企业家建立一个特拉华州的公司。“如果你要开一家新的瑜伽馆，那可能不适合你，”他说。

在向美国初创公司开放的同时，Stripe 团队正在向 Atlas 用户添加新功能。特别是，该公司开设了一个详细的操作指南和模板部分，以及一个企业家分享知识的在线论坛。

它还扩大了合作伙伴计划，将 Y Combinator 等加速器以及众筹平台公司 Kickstarter 和 Indiegogo 纳入其中。除此之外，它在世界各地的加速器和风险公司中已经有了几十个合作伙伴。
