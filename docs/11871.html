Waze 凭借“提前订购”和 Dunkin' Donuts 在食品行业占有一席之地

Waze 将帮助你更快地获得咖啡、甜甜圈和早餐食品，新的“提前订购”功能可以让你做到这一点:提前订购 Dunkin' Donuts 食品，当你到达餐厅时再取。Dunkin 称自己是 Waze 努力的“第一合作伙伴”，这意味着如果这项工作成功，它可能会与其他食品供应商合作。

Waze 的提前订购功能要求用户在他们的 iPhone 或 Android 设备上也有 Dunkin' mobile 应用程序的更新版本，该版本通过其现有的移动订购功能直接连接到 Waze 应用程序。

为什么从邓肯甜甜圈开始？嗯，Waze 和咖啡/糖果供应商是老朋友了，事实证明，Waze 绘制了 Dunkin 的所有 dunk 目的地(这就是他们所谓的 Dunkin' Donuts 商店位置；或者至少，这是 2012 年应用程序中我对他们的称呼。Waze 的应用内广告也是以 Dunkin' Donuts 作为第一个合作伙伴的。

通过应用程序提前订购已经成为咖啡销售商星巴克的诅咒和祝福，星巴克正在对其零售模式进行重大改变，以应对这一功能的流行。将它与人们已经在使用的导航软件结合起来，从一个地方到另一个地方，似乎是一种进一步激发兴趣的方式；当我打开我的导航应用程序并把它作为一个选项时，我更有可能决定是的，我确实想要一杯咖啡。

这也是 Waze 对优步进行狡猾、间接报复的另一种方式；在某些非常特殊的情况下，谷歌旗下公司的拼车服务是打车服务的竞争对手，而订餐服务可能适合点播食品领域的类似利基市场，UberEATS 也在该领域开展业务。
