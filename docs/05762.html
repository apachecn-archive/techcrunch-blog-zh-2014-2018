Sylaps 允许您通过链接邀请他人加入视频通话

见见 sy laps T1，这是一家新成立的 IP 语音公司，它可以让你在不给出电话号码的情况下开始打电话。Sylaps 就是要尽可能无缝地与陌生人打电话。

这家初创公司拥有适用于所有主要平台的桌面和移动应用——OS X、Windows、Linux、iOS 和 Android。你可以创建一个对话，并从应用程序中邀请某人。不使用 Sylaps 的人会收到一个链接，点击它就可以从浏览器加入通话。

Sylaps 依赖于 WebRTC，这就是该公司如何在所有平台上工作。这是一个开放的标准，无需安装插件或应用程序，就能在你的浏览器中自然运行。

您可以使用 Sylaps 与一个或多个用户进行音频通话和视频通话。在通话窗口旁边有一个聊天窗口，这样你就可以交换链接。你也可以发送文件。

但最棒的是，文件共享和聊天也可以在你的手机浏览器上进行。您可以在所有平台上获得完整的体验，从而全面减少摩擦。

这家初创公司还详细介绍了其安全特性。你可以使用 Sylaps 自由交谈，因为公司不会试图找出你在说什么。特别地，媒体通信被加密。

Sylaps 与 Skype 和 Google Hangouts 等通信巨头竞争。电话会议公司已经尝试了链接方法，但是 Sylaps 也能很好地处理一对一的对话。这家初创公司圆滑的方法使其在竞争中脱颖而出。创业公司不会试图做所有的事情，但它会做好几件事。
