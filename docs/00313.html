社交营销公司 Unified 收购分析初创公司 Awe.sm

创造了所谓的“社交操作平台”的 Unified 宣布收购了分析初创公司 Awe.sm 。

Unified 提供在社交媒体上购买广告的工具，以及监控其他广告购买系统的工具。今年早些时候,增加了商业智能数据,用于跟踪公司社交广告支出的更广泛趋势。

与此同时，Awe.sm 最初是作为开发人员跟踪社交分享的工具(即使你从未使用过该产品，你也很有可能见过 awe . sm——缩短链接)，但它的重点已经转移到广告商。

Unified 表示，通过集成 Awe.sm 的技术，它将能够提供关于营销活动效果的更详细数据，特别是在社交媒体以外。例如，一位发言人告诉我，Unified 已经在脸书、Twitter 和 LinkedIn(移动和社交)上提供了行动跟踪和投资回报衡量，但通过收购，它可以为所有数字广告添加类似的数据，包括搜索和显示，以及离线广告。

该发言人表示，由于此次收购，Unified 还将能够提供一个开放的 API 框架，可供合作伙伴和开发者使用。

“将 awe.sm 的社交归因技术与 Unified 的记录系统相结合，使我们能够为营销人员提供前所未有的关于社交分享给他们的业务带来的价值的见解，”Unified 联合创始人兼首席执行官谢尔登·欧文在收购发布会上说。

Awe.sm 团队的五名成员将加入纽约和旧金山的统一办公室，使该公司的总人数超过 90 人。Unified 表示，Awe.sm 的客户应该不会遇到任何服务中断，合并后的公司将在未来几个月推出使用 Awe.sm 技术的新产品。

这笔交易的财务条款没有披露。Awe.sm 已经从前期风险投资、Foundry Group 和其他公司筹集了 530 万美元。

去年，Unified 收购了脸书分析公司 PageLever。

更新:这篇的统一博文提供了更多的细节，包括 API 方面:

Unified 现在还提供了一个开放的 API 平台，使我们能够为客户提供对其付费和自有媒体数据的前所未有的可见性，以及跨渠道的社交测量，即基础架构即服务。这将使品牌、代理商、技术供应商、媒体出版商、内容制作者和有影响力的人能够将多代社会测量构建到其产品的基础中。
