# Windows 8 开始菜单可能要到 2015 年才会上市

准备好进入[开始菜单](https://web.archive.org/web/20221006082131/https://beta.techcrunch.com/2014/04/02/microsoft-will-soon-bring-back-the-start-menu-in-windows-8-1/)了吗？据玛丽·乔·弗利报道，[微软要到 2015 年才会发布修改后的开始菜单，所以你可能要穿上你的等待裤了。](https://web.archive.org/web/20221006082131/http://www.zdnet.com/no-microsoft-start-menu-for-windows-8-until-2015-sources-7000030100/)

微软在今年早些时候的 Build 大会上演示了即将到来的开始菜单，但该产品展示得很简短，很可能只是一个模型。因此，开始菜单不会出现在 Windows 8.1 的秋季更新中并不令人惊讶。

令微软失望的是，个人电脑用户似乎对这种工具有很强的市场需求，它的加入可能会刺激那些观望的人对新设备的需求，他们正在等待购买一台与他们目前拥有的电脑更匹配的电脑。

2013 年近乎直线下降的个人电脑市场已经有所稳定。市场应该达到每年 3 亿台左右的平衡，尽管英特尔和微软等现有公司希望扭转这一趋势。随着时间的推移，随着敞篷车和其他类似平板电脑的设备变得越来越普遍，个人电脑的确切定义正在变得模糊，这可能会扭曲一种消极的趋势——主要是偶然的。

在今年的销售周期中，预计 Windows 8.1 将继续保持其一贯的风格。
