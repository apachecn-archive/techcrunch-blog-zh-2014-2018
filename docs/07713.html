聊天机器人今年经常出现在新闻中。首先是 Tay ，微软的种族主义私生子，她无法控制自己。然后是脸书的信使机器人，据大多数人所说，开始并不顺利。而现在又有来自 Viv 的消息，乍一看好像是 Siri 2.0。不出意外，它是由 Siri 的创始人打造的。

真正令人惊讶的是，尽管有这么多宣传，但很难找到一个真正好的聊天机器人产品。当然，我们可以争论什么是好产品的定义，但是用最简单的话来说，一个好的产品应该有三个特点:(1)简单易用；(2)它在 99%的时间里工作良好；(3)无论你想让它做什么，它都能消除或减少摩擦。

所以回到这个问题——为什么聊天机器人很烂？我的意思是，我能想到的没有一个聊天机器人赢得了广泛的赞誉，或者更重要的是，它证明了聊天机器人实际上比一个应用程序更容易使用。那么这是为什么呢？让我们仔细看看。

试图做太多事情的聊天机器人通常会失败

Viv 和 Siri 都深受其害；他们的目标是涵盖一切，并在这个过程中妥协的质量。为了理解为什么大范围是有问题的，我们需要首先理解聊天机器人是如何工作的。

聊天机器人基本上有两个部分:“大脑”和“身体”

第一部分是“大脑”为了理解这个大脑是什么，想象一辆汽车。在汽车内部，你有一个发动机，使汽车移动。与发动机相连的是你可以根据自己的需求定制的车身——你可以拥有一辆四轮车、一辆运动型跑车或普通轿车。同一个引擎可以有多种不同的用途，这取决于你安装的车身。

聊天机器人也是这样工作的。“大脑”就像这个巨大的神经系统，它接收人类说的话，并将其转化为可操作的代码。在一个非常狭窄的聊天机器人中，人类只能说这么多。

例如，如果有人制作了一个聊天机器人来帮助你预订航班，你可以想象一个人可能会对聊天机器人说 100-200 句话。所以，为这种需求建造一个大脑是相当容易的。我可以在几周内做到这一点，并且非常确定我会有 99%的准确率来回应人类。但是现在想象一个范围更广的聊天机器人——比如 Siri。你可以对它说任何话。

对于 Siri 来说，正确理解一切是非常困难的。这需要巨大的工程努力，并且需要时间。如果一个婴儿需要几年才能学会一门语言，那么人类也肯定需要几年才能人工创造一门语言。然后，他们也会失败，因为他们永远无法完全复制人类行为的非理性方面。就像你永远无法预测一个六岁的孩子会说什么。但是你能预测一个 40 岁的人会说什么。

聊天机器人的第二部分是“身体”正如我所说，这是“容易”的一部分。它需要构建和定制，但它实际上只是一个庞大的知识体系。编译它可能需要时间，这取决于行业。对于像通过聊天机器人进行旅游预订这样的事情，你需要的知识并不丰富。我想象你可以把你的聊天机器人插入猫途鹰的 API，然后交付一个非常健壮的产品。

很难找到一个聊天机器人是真正好的产品。

但对于金融服务领域的任何事情(这是我工作的世界)来说，编纂这些知识的障碍和时间是巨大的。大多数金融服务网站不习惯像真人一样说话。聊天机器人天生就需要像人类一样说话。它们需要简单易懂。将银行术语/网站翻译成聊天机器人友好的术语既费时又费力。很难，因为金融产品在默认情况下是复杂的，因此用简单、通俗的术语来描述它们是相当困难的。

建造一个真正智能的聊天机器人真的很难

听起来可能很奇怪，大多数聊天机器人实际上并不智能。嗯，至少不是那些被使用的。你如何知道聊天机器人是否有智能？这里有两个决定性的特征:(1)它能够“照顾自己”，并随着时间的推移而变得更好；(2)它让你可以说或问任何你想问的事情。

让我们以微软聊天机器人 Tay 为例。是的，它很聪明。当人们对它说话时，它试图学习。而且，理论上，它应该随着时间的推移变得更好——更好的是，它开始看到模式，而不必被“告知”它们。一个不智能的 app 呢？大多数 Facebook Messenger 机器人，例如 CNN 聊天机器人，并不智能。为什么？因为你只能沿着它们上面的许多预设路径中的一条走下去。

最有可能做对的应用——实际上做他们应该做的事情而不会搞砸——是不聪明的应用，因为你可以走的“路”的数量是有限的。

聊天机器人在演示中看起来很酷，但 UI 实际上相当笨重

这个很有意思。你看到的大多数正在演示的聊天机器人，例如 Viv，完全掩盖了这个微小但重要的细节。问题是，对于大多数好的应用程序，你实际上输入的很少。刷一刷，摸一摸，就完事了。对于大多数聊天机器人来说，情况并非如此。

聊天机器人非常适合解释事情，因为它们可以不停地问你是否明白。

大多数聊天机器人还不支持语音，这意味着你必须输入所有内容。毫不奇怪，这意味着需要更多的时间来完成任务——所以实际上使用聊天机器人比使用应用程序效率更低！这完全违反直觉。当你考虑到聊天机器人不理解你在交流什么而导致的延迟时，效率会变得更差——因为现在你必须重新输入你的问题，并希望聊天机器人能理解。

那么聊天机器人注定要失败吗？

一点也不。但这确实表明聊天机器人仍处于早期阶段。10 到 15 年前，当应用程序开始接管世界时，好处是显而易见的。与尝试使用移动网络相比，在手机上使用应用程序做事情更快、更容易。

今天，同样的好处——速度和效率——在我使用聊天机器人和移动网络时并不明显。即使他们做到了，缺乏准确性意味着我永远不确定我是否真的可以信任聊天机器人来完成工作。

我个人认为，非智能聊天机器人将是聊天机器人的第一次出色使用。将它们用于客户服务——向客户回放常见问题的答案。并用它们来解释复杂事物是如何运作的。比如银行产品。想象一下，一个聊天机器人可以帮助你——第一次买房的人——了解你在购买第一栋房子时需要知道的一切。那会有多大价值？这将是惊人的和巨大的。

聊天机器人非常适合解释事情，因为它们可以不断问你是否明白，如果你不明白，它们可以不断深入解释。我认为聊天机器人也是长表格的一个很好的替代品。它们比表格更人性化，更可口。所以，如果你是一个想建立聊天机器人的企业家，有很多地方可以分层。

关于 Viv 的最后一点说明

有些人有点搞不清楚 Viv 是什么。明确一点，不是 Siri。它不是一个完整的聊天机器人。它只是一个大脑。Viv 是一个大脑。它依赖于第三方 API 的输入来为您提供内容。Viv 试图成为默认的操作系统或网络，所有其他聊天机器人或聊天机器人 API 都在其上运行。

如果可行的话，Viv 会变得像应用商店一样。因此，从这个意义上说，他们将与 Facebook Messenger 或 Slack 或社交网络竞争，这些网络也试图成为聊天机器人的默认平台。

这就是未来的样子吗？我个人不确定。我不确定拥有一个像 Viv 这样的集中式聊天机器人操作系统是否有价值。如果聊天机器人是新的应用程序，那么我们不会期望在应用程序商店中找到它们，就像我们今天找到我们的应用程序一样吗？也许是，也许不是。只有时间能证明一切。
