# Spiri 的 200 英里电动汽车原型是为按需拼车设计的 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/09/20/spiris-prototype-200-mile-ev-is-made-for-on-demand-carpooling/>

丹麦初创公司 [Spiri 在 8 月下旬披露了他们的按需拼车服务](https://web.archive.org/web/20221207081343/https://beta.techcrunch.com/2016/08/26/spiris-on-demand-carpooling-service-will-use-custom-evs-and-let-drivers-ride-free/)，现在准备展示其第一辆原型车。该公司决定从头开始制造自己的全电动汽车，使用来自阿斯顿·马丁(Aston Martin)、特斯拉(Tesla)等公司的人才组成的工程团队，以达到最适合频繁、多乘客城市导航需求的最终产品。

Spiri 强调这在很大程度上仍然是一个原型，前特斯拉副总裁兼 Spiri 董事会成员彼得·卡尔松在发布会的新闻稿中称其为“基本上是一个轮子上的实验室”。这个想法是在决定最终 Spiri 汽车上路后的外观之前，通过测试调整包括操控、内部客舱空间、范围和安全选项在内的功能。

目前的精神汽车设计仅重约 992 磅。，这比特斯拉 Model S 的 4597 磅要轻得多。，甚至是 2635 磅的宝马 i3。Spiri 首席执行官斯特凡·霍尔姆·尼尔森在一份声明中表示，这家初创公司通过基本上抛弃标准汽车上“对一个人的日常通勤和旅行毫无价值”的一切来实现这一重量。这意味着没有任何内部车身面板，大约四分之一的零件用于制造普通的消费汽车。甚至没有油漆。此外，该车的车身完全由碳纤维制成，碳纤维是一种轻质但坚固的材料，电动汽车制造商通常使用这种材料来节省宝贵的能源以提高效率。

然而，Spiri 的车载技术确实比普通汽车多一点，因为它一直连接到云，以确保汽车在使用过程中受到安全监控。乘客也通过 Spiri 应用程序进行注册，并使用初创公司的算法与其他人进行匹配，以最大限度地提高拼车路线的效率和速度。

Spiri 的超轻电动汽车将有助于优化服务定价，以实现用户费用(该公司表示，用户费用将类似于公共汽车票的费用)，以及让客户成为汽车司机的计划，让他们免费使用汽车，以换取他们的服务。最终，Spiri 表示，它将汽车自动化视为消除对其商业模式中这一部分的需求的一种方式，但目前该公司最大的障碍仍然是获得高质量、负责任和安全的司机，同时还依赖于其普通用户群的无薪成员。

这家初创公司的目标是在 2017 年的某个时候在五个发射国家之一的城市开始服务，包括英国、德国、丹麦、挪威和瑞典。
