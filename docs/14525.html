特斯拉申请移动电池交换装置专利

特斯拉仍在探索让更换车辆电池组更容易、更快的选项。这项在五月份申请的专利揭示了一种选择，它允许技术人员在不到 15 分钟的时间内更换包装。

这并不是第一个这样的想法。早在 2014 年特斯拉就曾短暂使用过一个自动系统，该系统能在不到 90 秒的时间内更换 Model S 电池组。随着增压器网络的推出，这个想法似乎被搁置了。

该专利中描述的系统可以安装在服务站中或者集成到移动钻机中。型号 S 或 X 将驱动坡道或电梯，技术人员将协助机器交换包装。

该专利指出，这是为 S 型和 X 型车辆，虽然它可以有更广泛的应用。Elon Musk 此前表示，如果特斯拉推行移动电池交换计划，它将支持商用车，就像特斯拉即将推出的半卡车平台。
