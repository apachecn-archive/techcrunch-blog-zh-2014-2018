DigitalOcean 以同样的价格为您提供更多内存和存储空间| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/01/16/digitalocean-gives-you-more-ram-and-storage-for-the-same-price/

云托管公司 DigitalOcean 今天推出了一些新的价格产品。特别是，该公司还在升级其基本的 5 美元的 droplet 实例，这是几年来第一次有了更好的规格。

五年前，DigitalOcean 提供了一些革命性的东西。在今天之前，你可以每月支付 5 美元，获得一台 20GB 固态硬盘存储、512MB 内存和 1 个 CPU 内核的服务器。但是虚拟 Linux 服务器已经成为一个拥挤的空间，有大量的竞争者出现并调整他们的价格。

在 Linode 上，你可以每月支付 5 美元获得 1GB 内存、20GB 固态硬盘存储和 1 个 CPU 内核。在 Scaleway 上，你可以支付 3.65 美元(€2.99 英镑)购买 2GB 内存、50GB 固态硬盘存储和 2 个 CPU 内核。你会明白，数字海洋正在调整其产品阵容以保持竞争力。

每月 5 美元，你现在可以获得 1GB 的内存和 25GB 的固态硬盘存储，而 CPU 性能保持不变。所有标准的 droplets 现在以同样的价格获得了两倍的内存和更多的存储空间。您支付的费用越高，获得的存储空间就越多。

除了每月 160 美元的型号之外，从 40 美元起，你可以以同样的价格获得更多的 CPU 虚拟核心。有两种新的昂贵的液滴，包括一种新的顶级液滴。每月支付 960 美元，你就可以获得 192GB 的内存、32 个内核和 3840GB 的存储空间。

有趣的是，现在有更多每月 15 美元的选择。您可以获得 1 核 3GB 内存、2 核 2GB 内存或 3 核 1GB 内存。最后，优化的 droplets 获得了 33%的内存和存储升级。DigitalOcean 有一个新的定价页面，有更多详细信息。

该公司还表示，按秒计费即将推出。亚马逊网络服务和谷歌计算引擎已经提供按秒计费。

和往常一样，你可以在 DigitalOcean 的管理面板上切换到另一个 droplet 模式。这种灵活性是 DigitalOcean 成功背后的一个关键特征。因此，如果你是 DigitalOcean 的客户，是时候利用这些价格变化了。
