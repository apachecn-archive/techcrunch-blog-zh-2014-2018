Path 的“位置”功能在出售给 Daum Kakao 后关闭

Path 宣布今天已经将旗舰通讯应用出售给韩国 Daum Kakao，但该公司保留了其业务中一个有趣的部分:Places 通讯服务。这是根据 Daum Kakao 的说法，该公司向 TechCrunch 证实，它没有收购该资产。

更新:看起来 Places 正前往死亡池。Path 向 TechCrunch 证实，它“目前不准备投资为该功能开发新的应用程序”，因此它已被“禁用”。完整的声明在这篇文章的底部。

概括来说，Path Talk 是该公司去年发布的独立消息应用，让 Path 用户能够在没有社交网络干扰的情况下进行交流。(就像脸书有信使一样。)当 Path 增加了 Places 时，服务变得更加有趣，这一功能允许 Path Talk 用户在应用程序中与当地的餐馆和商店进行交流——要么通过与员工直接交谈，要么通过 Path 员工团队充当联络员。

Path Talk Places 从未成为主流，但它在用户中拥有忠实的追随者，其中许多人可能担心这项服务被出售。Daum Kakao 表示，“几乎所有你喜欢的关于 Path 和 Path Talk 的东西都将保持原样，”但在这笔交易宣布后，当 Places 功能停止与 Path Talk 合作时，事情看起来很糟糕。然而，那是因为 Daum Kakao 拥有 Path Talk 和 Path 拥有 Places，并且它们没有一起工作。

https://twitter.com/ZacDavies/status/604142946804359170

Path 业务已经转向成为一个多应用工作室(T1)，它最近发布了 GIF 创作应用 Kong(T3)，这表明我们可能会看到 Places 作为一个新的独立服务由 Path 管理。特别是自从 Path 首席执行官大卫·莫兰(David Morin)一直对服务非常乐观——这符合他的“应用新功能”哲学——而通信基础设施，通常是商业消费者服务中最困难的部分，已经在美国、澳大利亚和欧洲的许多地方就位。

我们已经联系了 Path，请其对 Places 的未来发表评论，但这项服务的粉丝们有理由保持乐观。

更新:以下是 Path 的声明:

Daum Kakao 获得了路径对话，而不是路径位置。因此，我们不再有路径位置的家。我们已经决定，我们目前不准备投资建立一个新的应用程序的功能。我们可能会在未来重新评估，因为我们仍然拥有知识产权。我们只收到了一些关于路径位置的消费者询问，并通过@Pathservice 传达了这一变化。
