IBM 的云增加了对 Nvidia 迄今最快的 GPU 的支持 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/04/05/ibms-cloud-adds-support-for-nvidias-fastest-gpus-yet/?ncid=rss

IBM 今天宣布，其 Bluemix 云上的用户很快就可以在他们的裸机服务器上添加两张 Nvidia 特斯拉 P100 加速卡。该公司表示，这项功能将于本月晚些时候推出，届时，IBM 将有可能成为第一家为这些芯片提供支持的主要云提供商，这些芯片可以提供高达 4.7 万亿次双精度性能和 16g 内存。

不过，谷歌仍有可能在市场上击败 IBM。去年年底，谷歌也宣布将在今年年初支持 Nvidia 的最新 GPU，但我们还没有听说该公司计划何时推出这一功能。我们要求谷歌更新时间表，但还没有得到回复。

当然，AWS 也提供 GPU 支持，其机器可以配备多达 16 个 GPU(这些是较旧的 K80 卡，尽管其中 16 个显然也提供了大量原始计算能力)。微软的 Azure 提供了类似的设置，支持多达 4 个 Nvidia 稍旧的 GPU。

英伟达特斯拉 P100 GPU 是英伟达迄今最快的 GPU，并针对深度学习应用进行了优化。然而，除了机器学习之外，现在还有大量其他高性能计算应用程序利用 GPU 来加快处理速度。

“随着人工智能时代的到来，对我们在云中的 GPU 加速计算平台的需求继续激增，”Nvidia 加速计算总经理 Ian Buck 在今天的声明中表示。“这些新的 IBM 云产品将为用户提供几乎即时的访问迄今为止最强大的 GPU 技术的机会，使他们能够创建应用程序来解决曾经无法解决的复杂问题。”

所有这些将最新最快的 GPU 带到云的热潮表明，这些大型云平台现在正在竞相为客户提供运行深度学习和人工智能应用程序的最快机器。毕竟，提供基本的云计算功能正在迅速成为一种商品，但至少在目前，主要参与者仍然可以通过提供对 GPU 的更好支持来区分他们的服务——当然，直到这也再次成为赌注。
