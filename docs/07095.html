聊天应用 Line 在泰国和日本遇到了麻烦，这是它的两个主要市场

总部位于日本的消息应用今天开始进军聊天机器人，在这一过程中击败了脸书，但对该公司来说，这是充满挑战的一周，据传该公司正在寻求在其最重要的两个市场进行 IPO。

首先，在本土，Line 正因其游戏处理虚拟货币的方式接受当局调查。路透社报道日本监管者拜访了该公司的办公室，调查其是否无视处理客户资金的规则。

游戏代币在日本非常受欢迎，日本是世界上最赚钱的移动游戏市场，法规要求公司将其客户未使用的存款的一半存放在法律事务局，如果他们的总额超过 90，000 美元。调查人员正在调查 Line 游戏“打开宝箱的钥匙”，Line 告诉路透社，它没有存入未使用的积分，因为它们不是货币，而是线积分。

与此同时，在泰国，Line 正在接受调查，因为它允许模仿该国王室的贴纸出现。泰国的冒犯君主罪法律禁止侮辱君主，最高可判 15 年监禁。泰国军政府通过 2014 年的军事政变控制了这个国家，最近利用法律起诉一名诽谤国王的狗和对脸书发表评论的人，以及其他案件。

泰国用户可以通过其社区获得贴纸，该社区允许任何人制作贴纸并出售给 Line 的 2.15 亿活跃用户。Line 通常会在内容上市前对其进行审查，因此考虑到这一特定标签集的法律后果，不清楚为什么在本案中没有这样做。

泰国科技犯罪镇压部门的副主任告诉路透社，警方正在“调查这些贴纸来自哪里，是谁做的。”在 TechCrunch 收到的以下声明中，Line 发表了道歉。

Line Corporation 意识到这种文化敏感的贴纸可能会引起泰国用户的不适。有问题的贴纸已从 Line 贴纸店下架。

当我们认真对待我们的用户问题并考虑每个国家的文化因素时，我们将继续改善我们的 LINE Creators 市场。

我们对此造成的任何不便表示歉意，并感谢您的理解。

泰国是 Line 的重要市场。这款即时通讯应用被认为在中国拥有超过 3000 万用户，规模与脸书相当。泰国、日本、台湾和印度尼西亚代表了该公司活跃用户群的三分之一。

Line 最近在泰国测试了一些新服务，但它在该国也遇到了争议。泰国警方先前要求访问用户数据，而的政客先前声称已被批准。Line 一再否认提供过此类信息。去年，引入了一个端到端的信息加密系统，它声称该系统使得部分用户数据甚至连我自己都无法访问。(Line 的加密不如 WhatsApp 最近宣布的安全措施强大，后者完全加密所有用户数据。)
