# Circle 推出其 stable coin 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/26/circle-launches-its-stablecoin/?utm_source=dlvr.it&utm_medium=tumblr>

当 [Circle](https://web.archive.org/web/20230129223616/https://www.circle.com/) 筹集到其[1.1 亿美元融资时](https://web.archive.org/web/20230129223616/https://techcrunch.com/2018/05/16/circle-raises-110-million-or-13300-btc/)，该公司利用这个机会谈论其稳定的硬币——美元硬币，简称 USDC。现在你可以在 Circle Trade 和 Circle ' s exchange[Poloniex](https://web.archive.org/web/20230129223616/https://techcrunch.com/2018/02/26/circle-acquires-cryptocurrency-exchange-poloniex/ "Circle acquires cryptocurrency exchange Poloniex")上买卖和发送美元硬币。

但是什么是稳定币呢？顾名思义，1 USDC 值 1 美元。与传统的加密货币不同，你可以确定 USDC 的价值不会疯狂波动。

有多种原因可以解释为什么你想使用 stablecoins。首先，如果你想做空加密货币而不套现，你可以将你的比特币或乙醚转换成 USDC。这样，将来再购买加密货币就更容易了。

第二，如果你想避开传统的金融机构，你可以不通过银行把 USDC 送给其他人。发送 USDC 就像发送任何其他令牌一样——你只需要告诉你的收件人拿一个钱包，并询问他们的钱包地址。

第三，我相信很多人会使用 stablecoins 来避免税收问题。藏一堆代币比一大笔电汇打到你的银行对账单上容易多了。

许多生活在遭受恶性通胀或长期通胀之苦的国家(如委内瑞拉或土耳其)的人，也可以依靠 USDC 来转换他们的部分储蓄。这样你就不用在另一个国家开银行账户了。

USDC 是一个 ERC-20 令牌，这意味着如果你正在运行一个交换或钱包，很容易添加对 USDC 的支持。但 Circle 希望确保发行者不只是在银行账户中没有实际美元的情况下印刷钞票。

多家公司合作创建了[中心](https://web.archive.org/web/20230129223616/https://www.centre.io/)，这是一个将围绕稳定资本和治理制定政策的财团。如果你想发行 USDC，你必须遵守一系列的规则。特别是，你必须发送月度审计报告，证明你的存款和发行的代币一样多。

多家公司已经宣布，他们将很快开始交易 USDC，如 DigiFinex、CoinEx、KuCoin、OKCoin、Coinplug 和 XDAEX。在钱包方面，BitGo、Cobo、比特币基地钱包、CoolWallet S、Elph、imToken、Ledger、Status 和 Trust 将很快添加原生 USDC 支持。
