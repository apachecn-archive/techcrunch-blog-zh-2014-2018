# 书籍的下一步是什么？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/02/20/whats-next-for-books/>

我喜欢数字阅读器编辑内特·霍夫菲尔德。他是为数不多的几个关于出版的博主之一，他既不讨好这个行业，也不画蛇添足。他基本上相信书是伟大的，出版大概是注定的，写作真的很重要。

这就是为什么我很高兴他浮出水面，揭穿了[奇普·麦格雷戈](https://web.archive.org/web/20230220161744/http://www.macgregorliterary.com/blog/publishing-predictions-will-happen-2017/)的说法，他是一位代理人，相信随着主流出版商最终采取行动，我们将看到更多的书籍直接发布到移动设备上，并远离独立出版。虽然麦格雷戈有时是对的——电子书将更经常在手机上阅读——但他也大错特错。

他的第一个错误？他相信巴诺会创造迷你畅销书商店。他写道:

> B￼arnes 和诺布尔将开设一些只储存畅销书的迷你商店。我对此没有任何内部消息，但随着亚马逊开设实体店，B&N 必须采取措施，试图抢占更多的市场份额。

找不到更好的词来形容，B&N 已经死了。他们的战略是开大型商店，从棋盘游戏到填充动物玩具(还有一些书)，什么都有，但这种战略已经失败了。除非你想免费喝杯咖啡、看本杂志，否则没有理由去 B&N。也就是说，电子书的反弹给了独立书店新的腿，它摧毁了大型零售商，但我绝对可以看到一个连锁的*小型*书店咖啡馆，可以储存新的和二手的书籍，大量的儿童书籍，供父母为他们的孩子阅读，还有一些咖啡。我只是不认为美国会带头冲锋陷阵。

此外，他认为独立作者将回归大牌出版商。他们不会的。正如麦格雷戈在他自己的帖子中所写的，帕累托法则假设 80%的收入来自 20%的作家。考虑到大多数出版商做出决策所依据的不完全信息，相信他们发现那 20%充其量是愚蠢的。

相反，你会看到更多的长尾作者选择并留在独立领域。埃利奥特·佩普给人的印象是，他在独立制作方面比在主流出版商那里获得了更多的成功，而且还不止这些。他还怀疑，99 美分的超低定价将在独立游戏中消失，这也有点牵强；99 美分的定价仍然是提高销售额和亚马逊排名的聪明方法，放弃这种奇怪的调整可能会导致大多数独立作家的死亡。

简而言之，我们在几件事情上达成了一致——纯手机版可能会变得更受欢迎，基督教小说和其他利基市场正在消失(“这些天只有少数几家出版社仍在收购基督教小说，其中一些正在转向为更广泛的信仰者创作高质量的文学或女性故事，或者一系列悬疑小说，而不是只针对信徒的明确的宗教故事，”他写道。)我不同意他对 B&N 的乐观态度，也不同意他的假设，即印度群岛会掉头向出版商索要支票。在传统出版业之外，事情进展得相当顺利，而且，只要你愿意尝试新事物，你就可以成功，而不会让一本营销不足的书掉进现代出版实践的黑洞。
