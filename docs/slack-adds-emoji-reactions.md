# Slack 增加表情符号反应👍💥

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/07/09/slack-adds-emoji-reactions/>

Slack 的新表情符号反应是我在互联网上最喜欢的东西，至少今天是这样。

我渴望的不仅仅是一个“喜欢”当有人分享家庭中的损失，疾病降临到他们身上，或者只是想得到人生决策的反馈时，有时“喜欢”并不能完成工作。至少没有专一或者个性。

Slack 是 TechCrunch comms 有史以来最棒的事情，今天发布了一个更新，了解了非常真实的斗争。

准备好表情符号的反应。

Slack 团队在今天的一篇博客文章中解释说，许多用户要求能够“喜欢”评论。更进一步，Slack 意识到有时一个“喜欢”是不合适的，并尝试了一些潜在的响应按钮。

来自博文:

> 我们争论我们应该使用什么样的标志——心形、星形、竖起大拇指、复选标记。但后来我们意识到，所有这些都已经在默认的表情设置中了…还有舞者、火花、猴子、微笑、哭泣、鬼脸和便便。
> 
> 表情符号反应诞生了。

这是一个特性的起源故事，我个人希望这个特性能渗透到整个互联网领域。

有了这次更新，用户将能够在他们自己的空闲消息中添加表情按钮，其他用户可以点击这些按钮来表达……嗯，任何事情。你可以对简单的“喜欢”反应竖起大拇指，或者用“+”将人添加到列表中，或者使用两个(或更多)不同的表情符号进行投票。可能性确实是无穷无尽的。

反应可以添加到基本的消息和文件中，有一个特殊的新的最近提及标签，这样你就可以在一个地方看到你所有的反应。

Slack 还增加了一个新的表情符号选择器，这样找到完美的表情符号就不会花费太多时间。类别已经被简化，现在可以滚动，该团队还增加了更好的搜索功能，以便用户可以键入他们想要的内容。
