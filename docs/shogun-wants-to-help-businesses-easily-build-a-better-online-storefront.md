# 幕府希望帮助企业轻松建立更好的在线店面 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/03/05/shogun-wants-to-help-businesses-easily-build-a-better-online-storefront/>

Finbarr Taylor 和 Nick Raushenbush 已经开始了一个小的副业项目，在他们进行下一个计划时，这个项目赚的钱刚好够维持网站的正常运转。

然后，几个月后，[幕府将军](https://web.archive.org/web/20221209054944/https://getshogun.com/)——一个帮助小企业为 Shopify 等网站建立店面的工具——开始多赚一点。几个月后，它从一顿丰盛的晚餐变成了一笔可观的存款，然后足以支付一份真正的薪水。那时，泰勒和劳申布什开始了幕府将军的副业项目，他们意识到虽然它需要一段时间才能运行，但人们正在注册和使用它，并实际上为它付费。这就是为什么两人决定开始全职经营幕府将军，该公司现在正寻求扩展到为试图上网的企业提供通用店面以外的领域。幕府正从 Y Combinator 2018 冬季班出来。

“它开始占用更多的空闲时间，每天早上和晚上我都在做更多的支持，”泰勒说。这件事意义重大，以至于我们说，‘这不仅仅是一项副业。我们认为这实际上是一家初创公司。当我们接受[Y Combinator]采访时，这是一个非常引人注目的创业公司从死亡中重新振作的案例。"

幕府将军归结为一个易于使用的页面编辑器，允许小企业建立一个在线店面。它直接插入 Shopify 和其他电子商务服务，让非技术所有者或创始人以简单而吸引消费者的方式在线购买产品。不只是通过一个平台，比如说亚马逊，像这样的工具——来自泰勒和劳申布什与需要编辑页面的非技术人员合作的挫折——将允许这些企业做一些更定制的事情。当然，这确实意味着它将与像 Webflow 这样的[设计服务竞争，但 Taylor 说它正试图将自己定位于非技术用户，而不是更多以开发者为中心的工具。](https://web.archive.org/web/20221209054944/https://techcrunch.com/2014/03/11/responsive-website-builder-webflow-raises-1-5-million-from-khosla-tim-draper-others/)

虽然幕府基本上赚到了足够的钱来支付账单——这一点上，继续滚动可能是有意义的，也许还会接受一些新的融资——但两人仍然决定通过 Y Combinator 获得泰勒所说的“路线修正”的建议泰勒和劳申布什并不只是推进他们认为最好的想法，而是希望进入公司的网络，告诉他们，不，这是一个愚蠢的想法，而是尝试这个。

幕府本质上是一个页面构建器，这也是两人开始向代理商推销它的最初目标(Raushenbush 在视频创意方面也有一些经验)。它普遍受到欢迎，但它的反馈是，这项服务对 Shopify 这样的网站来说很棒，对其他网站来说就没什么了。两人只是让灯亮着，继续探索其他东西，但这个用例比预期的要大得多，并且有条不紊地成长为实际上可以演变成真正业务的东西。去年 4 月，两人决定全职从事这项工作，并开始关注 Shopify 以外的领域。

“这是在所有不同的层面上，”泰勒说。“它是为一些企业家、一些刚刚起步的人，甚至一些已经建立了企业的人准备的。他们想定制他们的店面。他们唯一的选择就是雇佣一个开发者来制作定制页面。我们进来说，他们可以一键安装这个应用程序，他们可以拖放真正漂亮、定制、无缝的页面，与他们的电子商务商店的其余部分完美集成。”
