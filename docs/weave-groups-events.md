# Weave 将群组和活动添加到它的“Tinder For Networking”应用中

专业社交应用程序 [Weave](https://web.archive.org/web/20221007040338/http://weave.in/) 希望通过双重选择过程，帮助专业人士找到他们想要会见的其他人。现在，这款应用增加了一项新功能，让参加活动或参加专业组织的用户更容易找到彼此。

[Weave](https://web.archive.org/web/20221007040338/http://www.crunchbase.com/organization/weave-2) 采用了 [Tinder](https://web.archive.org/web/20221007040338/http://www.crunchbase.com/organization/tinder) 向左或向右滑动的模式来指示你想要会见的人，并将其应用于专业社交。像 Tinder 一样，它考虑了用户的位置，以确保被匹配的人实际上能够喝咖啡或以其他方式相互见面。

你只需下载应用程序，将其链接到他们的 LinkedIn 账户，它就会使用那里的数据生成一个个人资料页面。完成后，你可以去镇上左右滑动，然后给你共同感兴趣的人发信息。这是一个非常有用的小工具，我曾用它来会见一些投资者和创始人，否则我不会遇到他们。好玩！

根据 Weave 联合创始人布莱恩·马(Brian Ma)的说法，该应用每周产生超过 1000 场比赛，主要是在早期的技术中心。迄今为止，它主要用于求职和招聘，但也用于联合创始人约会、筹款和业务发展。

唯一的限制是你只能和相对亲近的人联系在一起。所以对于像活动这样的事情来说，由于人们在一起的时间是有限的，所以它是非常有用的。这对专业组织来说就更没用了，因为成员可能在不同的城市，只是偶尔会有交集。

随着 Weave 的最新更新，该应用程序希望解决其中的一些问题。现在，Weave 将使参加活动或部分校友网络或其他组织的用户能够接触到分享这些共同联系的其他人。这将允许与会者在活动开始前进行联系，并在现场会面。它还允许相似组织中的人即使不在同一个城市也可以联系，但可以计划在他们在一起时见面。

但是，人们不能选择加入任何网络或活动…相反，他们必须输入私人邀请代码才能被添加并与该群中的其他人互动。该应用程序还添加了特定的群组，以组织和突出显示应用程序上的成员。

Weave 由 Decide.com 创始人 [Brian Ma](https://web.archive.org/web/20221007040338/http://www.crunchbase.com/person/brian-ma) 和 Buzzlabs 早期员工 [Elpizo Choi](https://web.archive.org/web/20221007040338/http://www.crunchbase.com/person/elpizo-choi) 创建，目前在 [Y Combinator](https://web.archive.org/web/20221007040338/http://www.crunchbase.com/organization/y-combinator) 批中。该应用程序是自筹资金的，但随着其他人在几周后的 YC 演示日，它可能会很快寻求资金。
