# UnifyID 希望彻底埋葬密码 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/09/12/unifyid-wants-to-bury-the-password-once-and-for-all/>

如今，老式密码很少受到黑客的尊重。事实上，对他们来说，这几乎不是一个减速带。黑客可以找到你的密码，甚至你的“安全”问题的答案，在互联网黑市上出售。今年 TechCrunch Disrupt SF 战地竞赛的参与者 UnifyID 看到了一个无可救药的系统，他们认为他们有办法修复它。

联合创始人库尔特·萨默维尔说，今天的方法的问题是，它使用一个秘密系统，你透露秘密就可以进入程序或服务。在他看来，这种技术是不可持续或不可扩展的，他的公司有一种更好的方法——隐式认证。

有了 UnifyID，系统不再提供密码，而是开始了解你是谁，你使用的设备，你去的地方，你一天中与你互动的传感器，甚至你走路的方式和打字的节奏。当你开始建立一个合理的个人资料时，软件可以根据是你的可能性计算出一个分数。

这项技术被设计成知道是你，因为你是独一无二的，你有不同的行为方式，让你成为你自己。

第一款产品是 Chrome 浏览器的扩展，带有 iOS 移动应用程序，将于本周在旧金山的 [TechCrunch Disrupt](https://web.archive.org/web/20221206152125/https://beta.techcrunch.com/event-info/disrupt-sf-2016/) 进行私下测试。(一款 Android 应用将在未来某个时间推出)。你安装浏览器扩展和移动应用程序，它就开始了解你和你的行为。你去一个网站，只要它确信是你，它就用你的 Unify ID 让你登录，而不是询问用户名和密码。如果有问题(当你第一次开始使用该产品时，更有可能会有问题)，它会向你的手机发送一个挑战，例如询问你的触控 ID。

如果你担心一家公司收集这么多你的数据，他们会说用户完全控制他们的数据。数据大部分存储在本地设备上，而不是后端云服务器上，为了确保该公司无法获取它收集的任何数据，这些数据都是加密的。即使他们有出售数据的意图，他们也不能，因为他们没有任何可操作的数据。

最初的产品是免费的，作为该技术的一种展示，但该公司计划向公司出售嵌入这项技术的能力，该公司的种子期未披露。

你可能认为其他公司也尝试过类似的方法，你可能是对的。谷歌一直在为 Android 开发一个类似的系统，但是联合创始人 John Whaley 说他们在 UnifyID 上所做的和谷歌正在做的有一些关键的不同。

“他们的技术专注于单一设备和平台:Android，并且完全是设备本地的。他说:“根据我们的经验，一旦你将来自多种设备的传感器数据结合起来，准确性和安全性就会大大提高。

更重要的是，他认为四大巨头——微软、谷歌、脸书和苹果——可能会尝试建立类似的系统，但不是为所有人建立一个单一的身份系统，而是为他们各自的平台建立，Whaley 认为这是一个根本问题——他的公司希望解决这个问题。

“对我们来说，其他玩家认识到这个机会当然是一种证明。像我们这样的小型创业公司面临的最大挑战之一是让市场了解身份认证的未来，”Whaley 说。这些公司有一个大喇叭来让公众支持新的认证类型，但他相信像他这样的公司仍然有创新的空间。

“尽管如此，我认为创新的空间已经成熟，一项新的足够酷的技术可以产生重大影响。”
