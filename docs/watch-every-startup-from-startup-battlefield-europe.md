# 观看初创公司欧洲战场 TechCrunch 的每一家初创公司

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/05/24/watch-every-startup-from-startup-battlefield-europe/>

TechCrunch 在巴黎举办了有史以来第一次创业战场。今天早上，15 家创业公司争夺梦寐以求的最佳展示奖。

他们都在三个不同的尊敬的评委面前投球。投资者和技术领导者花了一些时间问他们一些棘手的问题，并了解他们在做什么。今天晚些时候，[决赛选手](https://web.archive.org/web/20221025222219/https://techcrunch.com/2018/05/24/startup-battle-europe-finalists/)将在大舞台上面对一批全新的评委。

现在，来认识一下参加欧洲创业战场的 15 家创业公司。

## 怀斯巴特

Wisebatt 希望通过为硬件工程师提供专利模拟和协作平台来降低他们的 R&D 成本。

## 翁利

Wingly 是一个连接私人飞行员和乘客的航班共享平台，用于分担一次航班的费用。

# 走小路

Walk With Path 的可穿戴设备可以帮助帕金森患者更加自信地行走，避免摔倒。

# Wakeo

Wakeo 是一个 SaaS 平台，它使用机器学习和卫星数据来帮助行业领导者优化他们的供应链。

# 瓦拉尼达

Varanida 是一个网络扩展，允许用户选择他们想看广告的时间。

# 塔波利

Tapoly 为自由职业者、承包商和中小企业提供按需保险。

# 今日状态

StatusToday 是一个人工智能驱动的员工洞察平台，可以简化团队管理。

# 匙叶草属植物

Statice 的软件保护一家公司的私人数据，同时提供了一种共享这些数据的途径。

# 完全原创

only Original 是一个女性鞋类品牌，让顾客能够在线设计自己的鞋子。

# Mapify

Mapify 是一个社交旅行平台，旨在为规划交通、娱乐和住房提供单一渠道。

# IOV

IOV 为区块链和钱包用户提供了一个通用协议。

# 格洛威

Glowee 是一种可持续的生活光源，由废物和可重复使用的生物质提供动力。他们的任务是扰乱我们生产和消耗光的方式。

# 德罗瓦

Drova 是一种分散的游戏租赁服务，使客户能够在世界各地租赁游戏/应用程序，而不必购买游戏主机。

# 双重性

BIMlosophy 是一个平台，旨在为建筑经理提供支付工人工资所需的软件，而不必购买许可证。

# 防水布

Anorak 是一个使用机器学习为寻求人寿保险的人量身定制建议的平台。
