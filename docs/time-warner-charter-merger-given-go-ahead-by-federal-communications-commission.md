# 联邦通信委员会批准时代华纳与查特的合并

时代华纳和 Charter Communications 的拟议合并将获得政府监管机构的批准，他们希望合并后的公司遵守旨在防止其抑制流媒体视频服务增长的限制。

这笔 550 亿美元的交易将创造美国第二大宽带提供商(康卡斯特是领导者)，以及仅次于康卡斯特和美国电话电报公司的第三大有线电视公司

在合并后的七年时间里，时代华纳章程将不得不遵守联邦通信委员会和美国司法部设定的几项条件，以防止其限制用户使用流媒体视频平台，包括网飞、Hulu 和 YouTube。

首先，这要求时代华纳宪章不得根据数据使用情况收费或设置数据上限。其次，它将无法迫使提供大量数据(如流媒体视频所需的数据)的科技公司支付 it 互联费用。最后，时代华纳宪章不能与媒体公司签署内容协议，阻止他们向流媒体视频服务提供相同的节目和电影。

FCC 主席汤姆·惠勒(Tom Wheeler)在一份声明中表示:“所有这三项为期七年的条件都将通过使(在线视频分销商)受益来帮助消费者。”。“这些条件的累积影响将为互联网上提供的新形式的视频节目服务提供额外的保护。”

《华尔街日报》[报道](https://web.archive.org/web/20220930231915/http://www.wsj.com/articles/regulators-recommend-approval-of-charter-time-warner-cable-deal-1461611989)称，该特许还将被要求确保 200 万新家庭获得宽带接入，即使在市场中这意味着它将不得不与其他有线电视公司竞争。

尽管来自监管者和消费者的抱怨，像时代华纳和康卡斯特这样的公司已经[试图强加数据限制或额外收费](https://web.archive.org/web/20220930231915/http://www.bloomberg.com/news/articles/2016-01-28/comcast-draws-customer-ire-by-putting-netflix-addicts-on-a-meter)，使得用户很难观看视频。互联网服务提供商声称，这使得对不使用大量数据的人来说定价更加公平，但这也有利于保护他们的付费电视业务，因为越来越多的观众拒绝有线电视和卫星服务，成为插队者。

然而，时代华纳-Charter 的交易得到了网飞的支持，它之前反对康卡斯特和时代华纳的合并提议，该提议在去年联邦通信委员会和美国司法部提出反垄断问题后失败。

在去年 7 月提交给 FCC 的一份文件中，网飞[表示，Charter](https://web.archive.org/web/20220930231915/http://www.bloomberg.com/news/articles/2015-07-15/netflix-will-support-charter-acquisition-of-time-warner-cable) 采用的一项新政策阻止其向科技公司收取更快互联网连接的费用，这将“有助于促进整个互联网生态系统的持续创新”
