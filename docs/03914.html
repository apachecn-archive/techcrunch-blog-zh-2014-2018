雷电 3 使用可逆 USB Type-C，可能是最终的端口

英特尔在今年的 Computex 大会上公布了其新的雷电 3 规格，它可能是最符合苹果对 MacBook 未来的单一 I/O 愿景的端口:雷电 3 使用可逆的 USB-C 连接器，并包括 USB 3.1 支持，以及将 Thunderbolt 2 的数据传输速度提高 100%，达到 40Gbps，并支持同时使用最多两个 60Hz 的 4K 显示器。

例如，USB-C 的使用意味着该连接器实际上可以取代明年新款 MacBook 上的单个连接器，因为它还能够使用 USB 功率传输规范传输高达 100W 的功率。对 USB 3.1 的支持意味着它将与当前的 USB-C 电缆一起工作，用于连接附件和适配器，以及支持 Thunderbolt 的更高带宽和附加的显示器连接选项。

这很可能是苹果正在寻找的连接器，以帮助创造一个在精神上类似于当前 12 英寸 MacBook 的 MacBook Pro 它可以为仍然作为日常设备的标准连接端口的专业用户提供显著的优势，特别是随着越来越多的制造商开始使用 USB-C 和 USB 3.1 作为外部硬盘驱动器和闪存卷等设备的默认连接标准。

英特尔表示，它将在今年晚些时候推出新的连接器，2016 年全面量产。毫无疑问，如果事情按计划进行，我们明年也会看到 MacBooks 配备这种连接器——可能还有移动设备，因为可逆端口对 USB-C 的使用也比当前的 Thunderbolt 连接器规格节省了宝贵的组件空间。
