# Square M&A 负责人里希·加尔格加入 Twitter，负责公司发展

Twitter[带来了](https://web.archive.org/web/20230322160428/https://twitter.com/rishigarg/status/465890165861670912)公司发展的新负责人——[里希·加尔格](https://web.archive.org/web/20230322160428/http://www.crunchbase.com/person/rishi-garg)。加格从 Square 加入该公司，在那里他负责支付公司的企业发展。

在过去的几年里，Twitter 只[增加了](https://web.archive.org/web/20230322160428/https://techcrunch.com/2013/10/04/twitters-ma-has-ballooned-from-52-2m-last-year-to-over-417-5m-in-2013/)它的收购支出，所以 Garg 应该在他领导并购的新角色中保持忙碌。当然，他将致力于公司的其他战略合作伙伴关系。

加格于 2012 年初加入 Square，根据他在 LinkedIn 上的个人资料，除了处理 M&A 的交易，他还帮助与 T-Mobile、威瑞森、安吉名单、奥巴马代表美国以及罗姆尼代表总统建立了合作关系。他曾是通用催化剂公司的 EIR，也是 MTV 的战略、业务发展和企业发展总监。
