# Stripe 公司的帕特里克·科利森表示不会很快进行 IPO

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/04/07/stripes-patrick-collison-says-no-to-an-ipo-any-time-soon/>

可以想象，Stripe 公司的首席执行官帕特里克科利森经常被问到他的位于旧金山的支付公司是否计划在短时间内上市。去年，在 General Catalyst Partners 和 CapitalG 的联合牵头下，这家成立七年的公司为 T2 筹集了 1.5 亿美元，给这家初创公司估值 92 亿美元，几乎是 2015 年估值的两倍。

据《T4》和《财经》杂志报道，他们一共筹集了 4.4 亿美元。

不过，别指望 IPO 这么快，科利森在今天早些时候的旧金山上市大会上明确表示。尽管行业观察人士喜欢注意到拿枪指着的亚马逊持续取得的似乎不可阻挡的成功，这家公司于 1997 年上市，成立仅三年后，但科利森认为，亚马逊从一开始就比许多人似乎回忆起来的更加艰难。

他告诉采访者和产品发布会(Launch)创始人贾森卡拉卡尼斯:“如果你看看亚马逊(作为一家上市公司)的日常历史，你会发现，他们头几年的日子非常不平静。”“现在是 2017 年。。。这一切看起来都很好。但他们刚开始日子并不好过，我认为这是整个行业面临的一个更广泛的结构性挑战。”

当然，最近几年，对于上市公司较难承担风险这一点，支持和反对的声音层出不穷，一派认为只要还有资金可用，尽可能保持私有状态就是明智之举，另一派则认为上市可以为公司提供现金，使它们可以通过[收购其他公司。](https://web.archive.org/web/20230110103919/https://www.strictlyvc.com/2015/02/17/keith-rabois-startups-go-public-already/)

这是一个论点，科利森先发制人地争辩说，事实上，他今天说，“上市当然不会阻止你采取一个真正的长期的时间视野，但它确实使它更加困难。

“从我们的角度来看，当你达到某个稳定点，某个稳定点，或者至少完成了最初设定的目标，上市才有意义。但事实并非如此。我们刚刚起步。因为我们和初创公司合作，它们本身也有很长的发展轨迹，所以在建设基础设施时，也必须涉及很长的时间跨度。”

对于 Stripe 和许多其他独角兽公司来说，剩下的一个问题是，那些可能渴望获得流动性但却不能的员工会怎么样。毫无疑问，这也是科利森和他的公司将越来越多地处理的事情，尤其是当其他知名的“独角兽”测试公开市场时。

然而，在这里，科利森在台上有点不太愿意，他承认员工流动性是一个“全行业”的问题，但避免回答他是否赞成二次销售，允许他们出售一定比例的股份。

他说:“让(这一决定)具有挑战性的部分原因是，有太多的支持者——不仅仅是硅谷的支持者，还有监管者，尤其是证交会等，对吗？，”他说。“我和那些花了很长时间分析这个问题并真正深入研究监管细节的人一起度过了一段时间，我认为你开始在许多地方看到一种寻找更好模式的真正欲望，因为最终，在某种程度上，公开市场使围绕创新的真正长期愿景变得更加困难，这对整个国家来说真的很糟糕。这对投资者也不利。”

他没说的:他不是员工提前套现的粉丝，看起来是这样。至少， [EquityZen](https://web.archive.org/web/20230110103919/https://equityzen.com/) 的联合创始人 Shriram Bhashyam 表示，他的理解是 Stripe“有一些限制，需要董事会批准员工转让他们的股份。”EquityZen 是一个让初创公司员工出售非公开股票薪酬的平台。

一位熟悉该公司制度的消息人士称，这些限制对新员工来说是最难规避的。据这位人士称，早期 Stripe 的员工被允许出售一些股票，但该公司随后修改了章程，取缔了这种做法。
