# 凭借 360 万美元的新资金，YotaScale 为企业优化了云计算

毕业于炼金术士企业加速器的 YotaScale 今天宣布了一轮 360 万美元的风险投资，投资者包括[工程资本](https://web.archive.org/web/20221007191520/http://engineeringcapital.com/)、[佩利翁风险投资](https://web.archive.org/web/20221007191520/http://pelionvp.com/)以及天使投资人乔斯林·戈德费恩、蒂莫西·周和罗伯特·戴克斯。这家初创公司采用机器学习来帮助平衡企业云计算的性能、可用性和成本。竞争对手 [CloudHealth Technologies](https://web.archive.org/web/20221007191520/https://www.cloudhealthtech.com/) 和 [Cloudability](https://web.archive.org/web/20221007191520/https://www.cloudability.com/) 在这个热门领域总共筹集了 8000 万美元。

云计算已经迅速成为几乎每个行业的业务不可或缺的一部分。但是快速的创新步伐使得监控不断发展的云基础设施变得困难。YotaScale 没有把责任推给人类，而是自己实现了绩效管理的自动化。

该公司梳理了无数的云数据，以确保公司的基础设施针对其首要业务优先级进行了优化。这些优先级可能非常简单，比如最小化成本，也可能非常复杂，涉及多个具有不同最终目标的项目。

“任何人都可以做简单的事情，告诉你你的机器利用率低，你应该关闭它，”YotaScale 的首席执行官 Asim Razzaq 解释说。

Razzaq 的系统能够将使用数据与账单和日志数据结合起来。这些信息作为基线异常检测的基础。虽然这听起来不像是很多数据，但这足以推断出资源消耗和 CPU 利用率之类的事情。

但是异常检测的棘手部分是定义正常，因为正常是高度上下文相关的。对于一家电子商务公司来说，黑色星期五的使用量激增可能并不奇怪。在这一点上，YotaScale 不仅仅关注历史数据，它实际上还做出了前瞻性的预测。这使得数据波动的背景化成为可能。系统将预期绩效与实际绩效进行比较，而不是标记每一个变化。

不同类型的云基础设施数据是在不同的时间间隔创建的；有些每小时一次，有些每天一次，等等。挑战在于如何优化这种差异。集成机器学习技术用于提高分析的准确性，并管理捕获数据的多个维度。回归模型作为基础，其他半监督模型用于特定用途。

使用 YotaScale，像 Apigee 和 Zenefits 这样的企业可以理想地依靠机器来管理他们的云计算需求，减轻云和 DevOps 团队的负担。更不用说，当涉及到实时分析时，机器具有相当强的计算优势。
