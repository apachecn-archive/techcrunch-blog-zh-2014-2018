"要让风险投资者对我感兴趣，我需要增长多快？"

这是首席执行官们经常问我们的问题，我们意识到“视情况而定”并不是最可行的答案。相反，我们提出了一个简单的模型，让我们能够对这个问题给出一个明确的数字答案。

该模型允许我们在一家SaaS公司生命周期的每一个阶段确定一个增长率，低于这个增长率，该公司就没有一个清晰的、可进行风险投资的轨道。我们称这些增长率的图表为门多萨增长线。对于非棒球迷来说，门多萨线是一个棒球术语，指击球率低于该线的击球手不值得大联盟雇佣。

我们承认这个门多萨线模型的大量简化，并认识到在分析中找出漏洞的几种方法，但我们相信，对于所有的决策启发法，简化的好处是值得牺牲精度的。

门多萨线仅仅基于两个断言。

第一，大多数风险投资者更喜欢投资那些至少有机会成为独立上市公司的公司(这并不是说大多数都能实现这一目标)。看看成为上市公司的现实低门槛，这意味着在IPO时的运行率收入(ARR)为1亿美元，同时在第二年仍以25%或更高的速度增长。

第二，在大多数时候，增长率只会下降，但下降的方式平均来说是可以预测的。对于一家一流的SaaS公司来说，任何一年的增长率都在该公司前一年增长率的80%到85%之间。我们称之为增长持续性(这个假设从大约1000万美元开始成立)。根据我们最近的估计，我们在本文中使用82%的增长持续性估计。

仅使用这两个假设，看下面的图表，它显示了两个1000万美元的ARR run rate软件公司的轨迹。

A公司去年增长了120 %,销售额从450万美元增加到1000万美元。对明年增长率的预测略低于100%(98%，反映了82%增长持续性的假设)。该公司以1000万美元的运行速度几乎翻了一番，如果增长率按照增长持续性数学继续稳步下降，五年后将达到1.33亿美元的运行速度，未来增长率仍将达到36%。A公司正在走向上市。

B公司去年增长了94 %,从520万美元增长到1000万美元，并计划来年增长77 %,再次反映了同样的增长持续性计算。该公司将需要六年时间在ARR中突破1亿美元，此时该公司的未来增长率仅为23%。这正好等于或略低于软件公司上市的门槛，因此收入为1000万美元、未来增长率为77%的B公司处于门多萨线。

这种分析是普遍真理吗？

这是一个启发，而不是物理定律。换句话说，这是一般情况下运行良好的经验法则，当然也有例外。为了快速直观地说明这条规则有多有用，请看下图，图中用绿色显示了门多萨线。单个点代表21家成功的SaaS IPO在IPO前几年的收入和未来增长率，当时它们是私有的，收入低于1亿美元。很明显，大多数点都在这条线以上，这强调了一个事实，即成功的IPO表现出的增长轨迹高于门多萨线。同样值得注意的是，IPO时低于1亿美元的点甚至更少。这意味着，在IPO成为可能之前，增长必须存在。

诊断不是死亡——改变是可能的

好消息是，轨迹可以改变，这样一家低于门多萨线4000万美元的公司可以逆转轨迹，在8000万美元时变得引人注目。但这并不容易。

我们与许多公司合作，改进他们的上市(GTM)模式。令人惊讶的是，关键变量(如流失率和销售效率)的微小变化可以将增长率提高至关键的5%至10%，达到1000万美元，这在复合增长的世界中非常重要。有时，这就像在一个资金匮乏的行业中运用资金一样简单；更常见的情况是，需要对GTM模型进行重组才能使其工作。

例如，我们投资了ExactTarget，当时它的收入为9000万美元，未来增长率为38%。该公司曾试图上市，但在2009年初的金融危机中，规模较小的IPO并不在计划之中。相反，该公司筹集了风险资本，然后在首席执行官斯科特·多尔西(Scott Dorsey)的领导下，继续重组GTM模型，更加关注企业客户，并增加电子邮件以外的其他产品。一年后，该公司将其增长率提高到50%以上，这意味着增长持续率超过100%，并在极其成功的首次公开募股中以2亿美元的速度上市，然后被Salesforce以25亿美元的价格收购。

我们很幸运地在Box看到了类似的故事，当我们投资时，它的运营成本为660万美元，增长率为110%——回想起来，非常接近门多萨线。首席执行官Aaron Levie完全清楚他需要做些什么来加速，那就是坚持不懈地为产品添加企业功能并雇佣企业软件销售代表。)，但两年后，它达到了3000万美元，并以140%的速度增长，远远高于门多萨线，并锁定了一个将他们带到5亿美元的轨迹。

因此，诊断显然不是死亡，而是精神错乱在重复过去，并期望结果会有所不同。改变是艰难的。在超过五年的44家上市公司中，只有30%的公司有一年的增长再加速(增长持续性大于100%)，只有10%的公司有两年或两年以上的增长再加速。在我们的私人投资组合中，情况是一样的。只有35%的人至少经历了一年的增长再加速，但只有不到10%的人能够经历两年或两年以上的增长再加速。

这表明，你当前的增长率乘以对增长持续性的估计，就是你最有可能的未来。要改变未来，提升发展轨迹，需要积极主动地克服所有企业都面临的惰性。这可能意味着新的高管、新产品或不断变化的商业模式。

门多萨线被含蓄地理解

在任何人写下来之前，市场参与者很早就“知道”一条经验法则，这是非常典型的。在20多年的风险投资生涯中，我还没有听到任何人明确提出这个概念，但投资者心里清楚。它表现在一些公司的融资完成得很快，估值很高，条款非常干净。还有其他一些交易完成了，但进展缓慢，而且往往有一定的结构，所有这些交易都是为了在缺乏清晰的IPO轨迹的情况下确保最低回报。

人们谈论的往往是团队、过程和投资者，但在表面之下，现实是市场是有效的，有逃逸速度的公司和没有逃逸速度的公司之间存在价值差异的阶梯函数。

增长确实存在门多萨线。
