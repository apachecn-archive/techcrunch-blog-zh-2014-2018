# T-Mobile 悄悄透露政府数据需求上升 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/08/27/t-mobile-quietly-reveals-uptick-in-government-data-demands/>

T-Mobile 透露，它从政府收到的数据需求数量有所上升。

这家手机巨头在 8 月 14 日悄悄发布了其 2017 年透明度报告，显示与去年相比，其回应的总体数据需求数量增加了 12%。

报告称，该公司回应了 219，377 份传票，比 2017 年增长了 11%。这些要求是由联邦机构发布的，不需要任何司法监督。该公司还回应了 55，372 份法院命令，上升了 13%，以及 27，203 份认股权证，上升了 19%。

但是窃听命令的数量——允许警察实时监听电话——比前一年下降了一半。

T-Mobile 的一位发言人告诉 TechCrunch，这些数字反映了“全面法律需求的典型增长”，并且这些增长“与过去几年一致”。

尽管结果显示了更多的客户数据请求，但透明报告没有说明有多少客户受到了影响。

截至第二季度财报，T-Mobile 拥有 7700 万用户。

几家科技公司开始公布自谷歌 2010 年发布首份报告以来，它们收到了多少份政府索取客户数据的请求。但直到 2013 年爱德华·斯诺登(Edward Snowden)的爆料揭露了美国国家安全局(National Security Agency)的大规模监控后，科技公司和电信公司才开始定期发布透明度报告，被视为是为了反驳有关公司帮助政府进行间谍活动的破坏性指控。

两年后的 2015 年，T-Mobile 成为[最后一家发布透明度报告的主要手机运营商。](https://web.archive.org/web/20221208015712/https://www.zdnet.com/article/t-mobile-quietly-issues-first-transparency-report/)

该公司还表示，它回应了执法部门对客户历史手机站点数据的 64，266 次请求。这些数据成为今年早些时候[美国诉卡彭特案](https://web.archive.org/web/20221208015712/https://techcrunch.com/2017/08/15/countless-advocates-and-companies-advise-supreme-court-to-protect-phone-location-data-under-the-4th-amendment/)的焦点，在该案中，最高法院裁定执法部门必须获得历史手机和位置数据的授权。这一数字预计将在 2018 年报告年度下降，因为获得法院签署的逮捕令的新门槛更高了。

T-Mobile 还表示，它收到了 46，395 份跟踪客户实时位置的请求，以及 4，855 份对信号塔转储的授权和命令，警方可以使用这些授权和命令来获取特定时间段内连接到信号塔的所有附近设备的信息。

但 2017 年收到的国家安全请求数量有所下降。

全年，联邦特工用来秘密获取通话记录的国家安全信函数量和秘密外国情报监视法院批准的命令数量都低于 1000 项。

科技公司和电信公司在如何秘密报告需要客户数据的机密订单数量方面受到高度限制，只能报告他们收到的请求范围。

自 2015 年《自由法案》签署成为法律以来，司法部开始允许公司在更窄的范围内报告。
