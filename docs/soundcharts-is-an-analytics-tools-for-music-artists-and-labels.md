# Soundcharts 是音乐艺术家和唱片公司的分析工具 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/09/12/soundcharts-is-an-analytics-tools-for-music-artists-and-labels/>

法国初创公司 [Soundcharts](https://web.archive.org/web/20221005225046/https://soundcharts.com/) 正在为音乐艺术家打造一款类似于[的 App Annie](https://web.archive.org/web/20221005225046/https://www.appannie.com/en/) 。这项服务为你提供大量数据，包括世界各地的电台正在播放什么，Spotify 播放列表中流行什么等等。该公司刚刚从[阿尔文资本](https://web.archive.org/web/20221005225046/https://www.crunchbase.com/organization/alven-capital)以及[基马风险投资](https://web.archive.org/web/20221005225046/https://www.crunchbase.com/organization/kima-ventures)和[全球创始人资本](https://web.archive.org/web/20221005225046/https://www.crunchbase.com/organization/global-founders-capital)筹集了 310 万美元(€265 万英镑)。

许多大的音乐公司已经与公司合作，以获得关于电台 airplay 的见解。例如，他们向尼尔森支付大量资金来获取这些数据。

Soundcharts 希望公平竞争，增加更多数据。许多人不收听收音机来寻找新音乐。他们在 Spotify 上订阅流行的播放列表，在脸书等网站上关注艺术家。如果你想保持领先，播放时间是不够的，至少不再是了。

“我们想成为音乐的彭博金融，”创始人兼首席执行官大卫·韦斯菲尔德告诉我。“或者像 App Annie 为应用程序所做的那样。”

Soundcharts 在世界各地都有服务器，可以同时监听数百个电台。这些服务器使用 Shazam 等音乐识别技术将音频信号转化为结构化数据。

通过这种方式，你可以按国家、电台、艺术家或歌曲进行浏览，并查看谁在播放什么。你可以将你的艺人与你的竞争对手进行比较，你可以发现什么是流行的等等。也是节日程序员的好工具。

Soundcharts 还使用 Spotify 的 API 来索引所有公共播放列表并跟踪变化。这样，当一首 Chvrches 歌曲被添加到官方 Spotify 播放列表时，您可以在 Soundcharts 中看到它。如果你正在寻找共同的趋势，这项服务还可以跟踪播放列表的增长。你会是第一个知道 dubstep 是否卷土重来的人。

最后，Soundcharts 收录了脸书喜欢的歌曲，按国家给你分类，并告诉你增长情况。有许多社交媒体分析服务，但这是一个让 Soundcharts 成为一站式商店的好方法。每个人都可以访问所有数据，所以不仅限于你自己的艺术家。

迄今为止，已有 250 家公司签约，包括为环球音乐、索尼音乐和华纳音乐工作的员工，以及独立唱片公司、出版商和管理公司。这些公司需要定期付费才能获得音乐图表。
