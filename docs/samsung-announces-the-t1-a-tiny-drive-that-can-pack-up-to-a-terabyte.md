# 三星发布了 T1，这是一款可以存储 1tb 的微型硬盘

今天下午在拉斯维加斯，三星宣布了一款新的存储产品 SSD T1，它可以在一个名片大小的设备中存储高达 1tb 的内容，重量约为 1 盎司。

SSD T1 也有更小的 256 和 500 版本。这款设备的零售价在 179 美元到 599 美元之间，取决于你需要多大的存储容量。T1 本月发货。

在一个云存储无处不在且免费的时代，你可能会觉得奇怪，三星正在努力销售似乎是类固醇的 USB 记忆棒。但通过 Wi-Fi 网络上传内容可能会很费力，LTE 网络更糟糕。所以有时候当涉及到大量数据时，老派可能是更好的选择。

(关于低估一辆装满硬盘的汽车在高速公路上飞驰的带宽的老笑话在这里很贴切。)

看看创意行业之外的市场是否觉得 T1 有吸引力，这将是一件有趣的事情，创意行业经常处理需要移动和共享的大型文件。
