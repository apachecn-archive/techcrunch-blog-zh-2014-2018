# TC AppleCast 17: WWDC 2015 预演盛典！

苹果 WWDC 2015 主题演讲还有几天就要开始了，所以我们来讨论一下在这个特别的演讲上你能期待看到什么。基本上，这将是一个完整的乐透 iOS 9 和 OS X 10.11，加上苹果手表和手表操作系统的健康堆积。除此之外，可能还会有一些惊喜。

我们将详细介绍我们期望看到的关于 Proactive、Home 和 HomeKit、苹果地图交通方向以及传闻中的 iTunes 流媒体音乐服务的信息。我们还得到了一些关于 Watch OS 版本 1.1(或 2.0，无论下一个主要版本是什么)以及苹果将如何在可穿戴设备上测试原生应用的异常预测。

除了 WWDC，我们还谈一谈[雷电 3](https://web.archive.org/web/20221007042417/https://beta.techcrunch.com/2015/06/02/thunderbolt-3-uses-reversible-usb-type-c-and-could-be-the-ultimate-port/) ，英特尔高带宽输入/输出标准的最新更新。下一个版本听起来可能是苹果新一代快速轻便的笔记本电脑 MAC 所需的唯一端口，这为即将到来的假期提供了一个非常有趣的可能性。
