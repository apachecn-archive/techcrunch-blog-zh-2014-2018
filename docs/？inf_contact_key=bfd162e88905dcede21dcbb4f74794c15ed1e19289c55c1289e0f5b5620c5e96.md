# 优步联合创始人为 Expa 创办名为 Haus TechCrunch 的新房地产公司

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/07/28/uber-cofounder-launches-new-real-estate-venture-for-expa-called-haus/?inf_contact_key=bfd162e88905dcede21dcbb4f74794c15ed1e19289c55c1289e0f5b5620c5e96>

优步联合创始人兼 Expa 工作室首席执行官 Garrett Camp 今天公布了 Expa 的最新项目 Haus。Haus 是工作室的第一部房地产剧，完全专注于住宅地产的买卖，数字化并组织买家的报价，以便所有各方(买家、卖家和代理商)在谈判过程中更加透明和直接。

Camp 说，有很多创业公司专注于发现财产。但是，技术还没有真正触及的问题是，做出和接受官方提议。

豪斯将整个体验数字化，让卖家将他们的清单放在平台上，买家和他们的代理人都可以发布他们的报价，修改它们，并查看该房产的其他报价的匿名版本。

目前，房地产经纪人通过传真、电子邮件或直接面对面的方式向卖方代理人发送报价。另一方面，卖家的代理人正试图在电子表格、电子邮件或纸笔中合理地汇编多个报价。

这似乎很简单——写下所有最好的报价并选择最高的数字，对吗？但是，除了实际的整体价格之外，单个报价中还有许多因素，包括买家的财务安全、交易的额外条款、预付现金的数量等等。

房地产行业不一定以透明著称。虽然卖家的代理人被要求向他们展示桌面上的每一个报价，但买家很难了解某一特定商品的实际动向。

豪斯的目标是通过向卖家和其他感兴趣的买家展示每个报价及其独特的细微差别来消除这种不透明性。

但是这种透明度也有潜在的缺点。

例如，当卖家收到几个好的房屋报价时，卖家代理会在明天中午之前询问最好的和最后的报价。如果买家能够看到所有其他报价的条款，就像他们在使用 Haus 时一样，那么随着买家进入竞价大战，这可能会推高房屋的价格。更不用说，不道德的卖家或代理人可以通过[抬价](https://web.archive.org/web/20230129223230/http://www.ocregister.com/articles/privately-630075-bids-properties.html)人为抬高房价，因为报价没有法律约束力。

虽然豪斯帮助卖家衡量对他们房产的需求，但它也可能会赶走那些不想在已经很贵的房产上参与竞购战的潜在买家。或者，从另一方面来说，这可能会做空卖方，卖方本可以从买方那里获得高得多的最佳和最终报价，但看到该报价下降到仅次于最佳报价。

“我们认为开放将创造一个更有效的市场，报价的数量和价格最终将取决于需求，”豪斯总经理萨拉·哈姆说。“竞价战是当今房地产过程中常见的、几乎被接受的一部分。但通过我们的方法，买家知道他们的立场。买家会知道他们需要提供什么来使他们的报价具有竞争力，但他们也不会与自己讨价还价。”

也有人担心这种完全取代代理的做法，尽管 Camp 认为这是对经纪人向客户提供的服务的补充，在缓慢而耗时的过程(汇编报价)中提高了效率。此外，大多数人仍然需要经纪人的专业知识来营销房屋，寻找房屋，了解财产的真正价值，当然，起草文件。

虽然在透明度方面可能会有一些问题，但在汇编和向买家展示报价时，豪斯无疑提供了更高的效率。坎普将它比作过去的城市汽车产业，这个产业(你可能已经知道了)被坎普之前的创业公司，一个叫做[优步](https://web.archive.org/web/20230129223230/https://techcrunch.com/topic/company/uber/)的小公司摧毁了。

Camp 说:“收集报价和提交报价是一个非常手工的过程，城市汽车公司会拿起电话，在板上写下票价，然后派下一个有空的司机去那个地方。”“代理使用一个平台来自动协调所有这些信息，似乎效率更高。”

豪斯目前没有收取任何费用，但一旦公司明白如何通过节省谈判过程中的时间为房地产经纪人创造额外的业务，他们最终会从经纪人费用中抽走。
