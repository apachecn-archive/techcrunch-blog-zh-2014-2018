合格的创始人 Katelyn Gleason 计划颠覆价值数十亿美元的医疗结算行业

医疗账单是一个很大程度上尚未开发且利润丰厚的行业，到 2020 年，全球潜在收入将达到 550 亿美元。但它的内部运作仍然非常模糊——大多数时候不清楚某样东西要花多少钱，有时你甚至要到几个月后才会收到账单(可能数额巨大)。创始人 Katelyn Gleason 希望在你带着她的新医疗账单初创公司 Eligible 走进医生办公室之前，更容易知道你在支付什么和多少。

格里森已经通过 Y Combinator 两次(一次是为 DrChrono，一次是为 Eligible)，并从投资者财团筹集了 2500 万美元，这些投资者包括 Angellist 的 Naval Ravikant，Reddit 的 Steve Huffman，Dropbox 的创始人 Drew Houston 和其他人。但是，与健康创业领域的许多创始人不同，她并非从医学领域起步，也从未上过商学院。

相反，她是纽约市的一名舞台演员，在州立大学做销售代表和服务员。2010 年，当电子健康记录公司 DrChrono 首次进入科技领域时，格里森很快就成为了这家公司的首席销售代表，每周工作 90 多个小时。

尽管早期混乱不堪，格里森还是想推出自己的产品，并认为有必要改革医疗收费制度。虽然电子医疗索赔处理并不新鲜，但该系统一直停留在无云、数据共享前的时代。

这意味着当病人在医生的办公室，医生推荐某个专家时，没有办法知道病人的医疗保健是否涵盖了推荐的专家，或者最终的账单是多少。也没有办法知道在同一个保险计划中是否有更具成本效益的治疗方案。你基本上被蒙在鼓里，这可能会让医生收不到所有欠款，让病人背上巨额医疗债务。

Eligible 与 Cleveland Clinic 或 One Medical 和 Quest Labs 等医疗系统相结合，不仅会告诉您手术费用，还会告诉您您的共付额，并为您提供一种无需等待数月即可获得账单总额的支付方式。根据 Gleason 的说法，医生每月使用 Eligible 的收入通常会增加 50 %,因为他们不必在病人离开办公室后收集账单。

Simplee 和 iVinci 等其他初创公司也在努力实现医疗计费的现代化，但 Eligible 实际上为许多面向消费者的初创公司构建了主干，并让他们使用其 API 为患者解锁定价数据，就像电子商务公司可能使用 Stripe 进行支付处理一样。像 Change Healthcare(以前的 Emdeon)这样的传统票据交换所也或多或少地服务于相同的目的，但运行在更古老的技术上，以创建计费工作流。

Gleason 告诉我们，Eligible 过去一直是盈利的，但现在专注于扩大规模，它从每笔支付中赚钱，每笔交易收取 5 美分，大部分收入来自保险公司。

Gleason 还表示，Eligible 目前每月交易量约为 1400 万笔，预计到今年年底每月交易量将达到 5000 万笔。与此同时，她正忙于数据科学和工程部门的招聘，并且，随着健康技术领域的强劲趋势，该公司刚刚发布了一款名为 Ellie 的人工智能机器人，让客户更容易就他们的账单提出问题。
