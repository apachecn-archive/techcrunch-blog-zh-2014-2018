# 黑莓 Passport 是一款平板电脑，配有毫无意义的硬件键盘

就在你认为你可能已经知道黑莓在做什么(在企业服务和物联网等新兴市场的强大推动力)的时候，他们去做了一些像黑莓 Passport 这样的事情。

这款设备今天在加拿大智能手机制造商的收益电话会议上曝光(通过 [MobileSyrup](https://web.archive.org/web/20230129224032/http://mobilesyrup.com/2014/06/19/blackberry-passport-formerly-known-as-windermere-coming-in-september/) ),它将于今年 9 月推出，随后官方将公布更多关于规格和定价的细节。

黑莓 Passport 有一个 4.5 英寸的正方形显示屏，无可否认地达到了令人印象深刻的 1440×1440 分辨率，但其他一切都很疯狂，并不令人印象深刻。它在那个巨大的方形显示屏下配备了一个 QWERTY 硬件键盘，据传它是触摸感应手势交互的，而且它的设备宽度超过了庞大的 Galaxy Note 3。

老实说，我不知道该对这个设备说些什么。值得称赞的是，这不是一本剧本，但它是更好的东西吗？黑莓显然致力于其核心受众对硬件键盘的热爱，但在基本上是平板电脑的东西上安装一个键盘似乎是诠释这种渴望的错误方式。

至少现在他们与富士康在所有新硬件上合作，他们将抵消制造这个不可思议的怪物的一些成本。我也想给 lulz 买一个。
