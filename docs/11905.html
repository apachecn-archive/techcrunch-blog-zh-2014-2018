百度的人工智能团队教会了一个虚拟代理人，就像人类教他们的孩子一样

百度的人工智能研究团队已经实现了一个重要的里程碑:通过首先通过正负强化来教授 it 语言，来教授“生活”在 2D 环境中的虚拟代理如何使用自然语言命令导航其世界。根据科学家的说法，特别令人兴奋的是，智能体最终发展了一种“零射击学习能力”，这本质上意味着人工智能智能体发展了一种基本的语法意识。

你可能不记得了，因为它发生在你还是婴儿的时候，但这基本上是父母在孩子很小的时候教他们的。你给他们看图像，重复单词，最终，通过足够的积极强化，孩子可以将这些单词与这些图像联系起来，瞧，它知道事物的名称。

对人工智能来说，推广使用以前使用过的技能的能力是不小的成就。这个模型表明，系统可以以类似于人类的方式学习和应用保留的知识，即使只是在简单的、类似视频游戏的 2D 环境的有限领域中。实际上，这很不像我:我永远也学不会。
