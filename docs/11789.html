Twitter 收购 TweetDeck 已经六年了，但该公司最终可能会给客户带来新功能。

在今天发给一些高级用户的调查中，Twitter 询问他们最希望看到 TweetDeck 的“高级”版本增加哪些功能。之后，受访者被问及他们是否愿意为一个功能更全的客户端版本付费，否则这个版本会被遗忘。

TweetDeck 被忽视的简史

早在 2011 年，Twitter 就以 4000 万美元收购了 TweetDeck，当时对于开发移动和桌面客户端与 Twitter 竞争的开发者来说，这尤其令人担忧。就在两个月前，Twitter 当时的平台负责人 Ryan Sarver 警告开发者停止开发 Twitter 客户端，转而专注于补充应用和服务。

在这方面，TweetDeck 被收购可能比关闭其 API 访问并慢慢死去更好。但是，那些希望 TweetDeck 能够成为面向高级用户的更强大的 Twitter 客户端的人最终会大失所望。

虽然 Twitter 从未正式关闭 TweetDeck，但该产品在过去几年中因缺乏开发而萎靡不振。它在 2012 年末进行了一次小更新，但从那以后，它的功能慢慢消失，或者跟不上 Twitter 官方客户端增加的功能。先是移动 TweetDeck 客户端消失，后来原生 Windows 客户端停产。

然而，尽管缺乏内部支持，TweetDeck 在网络及其 Mac 桌面应用上保持着相当热情的用户基础。而且，就像凤凰涅槃一样，Twitter 正威胁要让这个客户端成为超级用户的付费订阅客户端。

对有价值的功能进行排名

参与调查的 TweetDeck 用户被问到，如果添加到客户端，他们可能会发现哪些功能最有帮助。这包括高级图表、关注者分析、突发新闻提醒、移动和桌面管理、改变配色方案和多账户切换的能力。

调查受访者还被问及他们愿意为客户支付多少钱或多少钱。安德鲁·塔瓦尼公开分享了他在调查中的问题，当被问及他是否愿意每月支付 19.99 美元时，TechCrunch 受众开发总监特拉维斯·伯纳德被问及他是否愿意支付 4.99 美元购买该应用的高级版本。很可能 Twitter 正在测试各种价格，以观察用户的反应。

一些人认为这项调查表明 Twitter 正在考虑开发广告之外的新收入来源，这一收入来源在最近几个季度已经开始滞后。但是推特上说事实并非如此。

调查的消息传出后，Twitter 就其计划发表了以下声明:

“我们正在进行这项调查，以评估人们对 TweetdDeck 新的、更强大版本的兴趣。我们定期进行用户研究，以收集人们对 Twitter 体验的反馈，并更好地为我们的产品投资决策提供信息，我们正在探索几种方法，让 TweetDeck 对专业人士更有价值。”
