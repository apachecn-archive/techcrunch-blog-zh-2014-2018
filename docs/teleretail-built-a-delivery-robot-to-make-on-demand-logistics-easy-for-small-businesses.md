# Teleretail 制造了一个送货机器人，使小型企业的按需物流变得容易

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/05/15/teleretail-built-a-delivery-robot-to-make-on-demand-logistics-easy-for-small-businesses/>

送货机器人被视为新奇事物、就业杀手和餐馆吸引附近顾客的噱头。这个物流领域的最新参与者是一家名为 [Teleretail AG](https://web.archive.org/web/20230316161016/https://teleretail.com/) 的瑞士初创公司。该公司对个人送货设备进行了改造——其机器仍处于原型阶段，旨在覆盖长达 50 英里的长距离，并为农村和郊区市场以及城市居民提供服务。

像其同行和竞争对手一样，包括[星际飞船技术](https://web.archive.org/web/20230316161016/https://techcrunch.com/2017/01/12/daimler-leads-17-2m-round-in-rolling-delivery-drone-startup-starship-technologies/)或[大理石](https://web.archive.org/web/20230316161016/https://techcrunch.com/2017/04/12/marble-and-yelp-eat24-start-robot-food-delivery-in-san-francisco/)等，Teleretail 采用了一些为自动驾驶汽车开发的相同系统，包括传感器、计算机视觉系统和 GPS 辅助导航系统。这家初创公司已经从赞助商那里获得了一些用于建造原型的组件和设备，比如博世、英伟达和激光雷达制造商。无论它在哪里测试机器人，你都能看到它们贴在机器上的标签。

Teleretail 的首席执行官 Thorsten Scholl 说，该公司的部分资金来自赠款，包括来自欧洲航天局的 200 万美元的可用资金。但到目前为止，它还没有筹集到任何种子资金或风险资本。他说，航天局对该公司利用卫星数据帮助地面系统导航以从零售商和居民那里取货或送货的方式感兴趣。

为了创造收入，Scholl 说，“我们认为自己是一个将(机器人)应用于本地物流的消费者应用程序。我们可以设想一种基于订阅的模式，在这种模式下，消费者可以订阅送货服务，或者商店可以订阅该服务，以便提供员工无法亲自提供的送货服务。”

Teleretail 的主要目标是让主要街道商店和小企业能够在便利性方面与资金更雄厚的电子商务巨头竞争，如亚马逊。“我们也有很大的动力来帮助老年人在他们变老时生活得更好，更独立，”他说。“这种技术有助于他们派出自己的火星探测车去送干洗的衣服或买杂货。”

最近，该公司在达拉斯举行的 [AUVSI Xponential](https://web.archive.org/web/20230316161016/http://www.xponential.org/xponential2017/public/enter.aspx) 会议上展示了其技术，其中展示了各种各样的自动驾驶汽车和无人机。接下来，Teleretail 打算让其机器人与瑞士和美国各城市的早期试点客户和合作伙伴合作，这些客户和合作伙伴很早就授权对个人送货设备进行测试和商业使用。

Scholl 说，“华盛顿特区、加州的桑尼维尔和爱达荷州从 7 月开始允许这样做。瑞士的一些山村也是如此。我们正在探索其他可能性，并试图帮助监管机构找出解决这一问题的办法。”

最终，这位首席执行官认为，这种设备不仅可以在城市人行道和人行横道上运行，还可以在街道上运行，甚至可能在高速公路上运行。这也是 Teleretail 开发小型车辆以全天候覆盖长距离的原因之一。

目前，用户将不得不走出家门，从远程零售机器人那里取回快递。但这位首席执行官说，公司希望开发一种机器人，它可以进入室内，不仅进行最后一英里的送货，还可以以某种方式进行最后一米的送货。然而，即使对于资金雄厚的机器人公司来说，爬楼梯仍然是一个相当大的挑战。
