# 塞拉里昂刚刚举行了第一次基于区块链的选举

塞拉利昂公民于 3 月 7 日前往投票站，但这一次有所不同:塞拉利昂使用一种在实际应用中尚属首次的技术记录了区块链 70%的投票率。

这项技术是由 Agora 公司的 Leonardo Gammar 发明的，它将选票匿名存储在一个不变的分类账上，从而提供了对选举结果的即时访问。

Gammar 说:“Agora 的区块链记录了匿名投票/选票，任何感兴趣的人都可以公开查看、计算和验证。”"这是政府选举首次使用区块链技术."

“塞拉利昂希望在一次有争议的选举中创造一个信任选民的环境，特别是考虑到选举后公众将如何看待这次选举。通过使用区块链作为一种手段，不断记录选票和结果，该国希望创造选举的合法性，并减少反对党的影响，”他说。

这为什么有意思？虽然这仅仅是一个概念验证——这不是一个完整的投票记录，而是捕捉到了看似可以接受的多数票——但看到这项技术在塞拉利昂这个约有 740 万人口的国家得到实施是令人着迷的。最终目标是通过减少纸质选票以及减少投票过程中的腐败来降低投票成本。

对于 Gammar 来说，他看到了去中心化系统的价值。

“我们是世界上唯一一家建立了全功能区块链投票平台的公司。其他电子投票机是“阻止盒”,越来越容易受到安全攻击。出于这个原因，许多美国州和外国国家一直在转向纸张，”他说。“如果你认为从现在起 50 年后大多数国家将使用某种形式的数字投票，那么区块链是唯一能够为未来提供端到端可验证和完全透明的投票解决方案的技术。”

一个国家的一次选举还不是一场运动。然而，Gammar 和他的团队计划将他们的产品推广到其他非洲国家，并最终推广到世界其他地方。
