# 优步加强打击人口贩运

优步与旨在全球范围内打击人口贩运的 Polaris 合作，授权其司机打击这种现代形式的奴隶制。据国际劳工组织统计，[全球估计有 2090 万人口贩运受害者。](https://web.archive.org/web/20221025223621/http://www.ilo.org/global/about-the-ilo/newsroom/news/WCMS_182109/lang--en/index.htm)

[该合作项目](https://web.archive.org/web/20221025223621/https://www.uber.com/info/human-trafficking-education/)将帮助教育优步司机关于人口贩运的一些迹象，并授权他们向国家人口贩运热线报告任何可疑的人口贩运案件。一项努力是通过优步应用程序为司机提供北极星热线电话号码和关于人口贩运的教育信息。

优步优步安全通信负责人特雷西·布里登在一次新闻发布会上说:“司机在帮助识别并最终防止人口贩运方面具有独特的地位。”。“我们将与我们的国家合作伙伴共同努力，利用我们的创新和技术以及我们全球社区的范围和规模，致力于帮助预防和提高认识，并为社区英雄赋权。在我们服务的城市里，我们可以一起帮助瓦解和终结人口贩卖。”

优步还与 Thorn 组织结成伙伴关系，支持其在建设保护儿童免受性虐待的技术方面的工作。

Thorn 首席执行官 Julie Cordua 在一份声明中说:“我们最近的幸存者观察报告发现，在 T4，三分之二的儿童性交易幸存者在遭受虐待期间从来不知道他们可以获得帮助资源。”。“与优步合作是一个机会，可以确保我们从各个方面解决这个问题，今天我们授权司机为处于危机中的人提供帮助。当我们一起努力时，我们可以建立一个每个孩子都能成为孩子的世界。”
