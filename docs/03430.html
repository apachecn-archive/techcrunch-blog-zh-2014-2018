通过购买 Verold TechCrunch，Box 获得 3D 观看体验

原文：https://web.archive.org/web/https://techcrunch.com/2015/04/14/box-gains-3d-viewing-with-verold-purchase/?ncid=rss

Box 今天在首席执行官 Aaron Levie 的博客中宣布，该公司已经收购了 3D 浏览器技术制造商 Verold

Levie 在一份声明中说:“借助 Verold 的技术，我们正在将我们世界级的内容预览和协作平台扩展到交互式 3D 内容，为制造、工程、消费产品、媒体和娱乐领域开辟所有新的用例。”

Verold 允许公司使用基于 WebGL 的技术将 3D 动画嵌入网页。该技术基于 OpenGL ES 2.0，并使用 HTML 5 显示。所有这些听起来都像是 Box 想要的技术。

正如 Levie 在他的博客文章中所写的那样，能够为建筑和零售等许多行业共享 3D 渲染可能会改变他们的游戏规则，并通过其 API 提供一种在 Box 平台上展示这种技术的方法，最终可能会产生一些有趣的结果。

“结合 Verold 创新的 3D 转换和查看技术，我们将解锁强大的新方法，用于预览、注释和协作关键任务信息，”Levie 在他的帖子中写道。

很有可能，这一收购的根源与工业用盒(Box For Industries)有很大关系，这是自去年秋天在 BoxWorks 用户大会上宣布以来一直在构建的计划。Box 正在创建一系列垂直行业，它正在寻找一套特定的工具来应用于每一个垂直行业。

这种方法让该公司得以进入医疗保健、政府和金融服务等行业，而在此之前，它可能很难在这些行业站稳脚跟。通过与合作伙伴合作，他们可以在 Box 开发平台上构建特定于行业的工具，这将打开全新的机会领域。

在 2 月底，它宣布了一个新的金融服务垂直行业，旨在吸引金融服务公司转向云，这些公司以前可能不愿意这样做。

它今天购买的 3D 工具是吸引不同垂直行业不情愿的公司的另一种方式。它应该非常适合这种方法，因为它可以使 Box 接近某些行业，如建筑，并提供一套独特的在线查看功能，无论是在办公室还是在网站上。

对于零售商来说，它可以通过提供产品的交互式 3D 视图来改变销售商品的方式，甚至可以让你放大或缩小以获得更好的外观。

上个月在接受 TechCrunch 采访时，Levie 暗示 Box 将会进行这类购买，但他没有透露具体细节。

“你将继续看到对那些为我们打开新大门的长期技术的投资，”他说。“围绕内容管理和协作，我们(仍然)需要开发很多技术。”

Box 于 1 月上市，在此过程中赚了 1.75 亿美元，这笔现金使 Box 能够进行这些类型的战略收购。

一旦交易完成，Verold 的员工将加入 Box，并开始努力将他们公司的技术融入 Box 的工作流程。没有披露任何条款。维罗德已经筹集了 160 万美元。
