脸书限制API，在丑闻中砍掉旧的Instagram平台

原文：https://web.archive.org/web/https://techcrunch.com/2018/04/04/facebook-instagram-api-shut-down/?

首席技术官迈克·斯科洛普夫告诉TechCrunch，脸书正在进入一个艰难的转型期，在剑桥分析公司惨败之后，它不会在数据隐私问题上冒险。这就是为什么它把Instagram API的关闭时间提前了。这极大地限制了从脸书的活动、群组和页面APIs plus脸书登录获得的数据或要求获得批准。脸书还关闭了通过电子邮件或用户名进行的搜索，并在发现恶意行为者利用这些来收集人们的数据后，改变了其账户恢复系统。“鉴于我们所看到的活动的规模和复杂程度，我们相信脸书的大多数人都可以通过这种方式获得公众形象，”Schroepfer写道。

Instagram将立即关闭其原定于7月31日弃用的部分旧平台API。 TechCrunch首先报道，由于API调用限制的突然降低，开发者的Instagram应用在周末被打破。Instagram拒绝置评，导致开发者感到沮丧，因为他们分析人们的粉丝并帮助他们增加受众的应用程序停止了工作。

现在，Instagram发言人告诉TechCrunch，“从今天开始，Instagram正在加速之前宣布的对Instagram API平台的弃用”，作为脸书保护人们数据工作的一部分。用于关注者列表、关系和评论公共内容的API将立即停止运行。2018年12月11日对公共内容阅读API的弃用和2020年对基本个人资料信息API的弃用将如期进行，但现在已经对它们实施了速率限制降低。

脸书承认，多达8700万用户的数据被剑桥分析公司不正当获取，比早期估计的5000万有所增加。这些用户将在他们的新闻提要上看到一个警告，告诉他们发生了什么，他们应该做什么，并看到移除他们授予权限的其他应用程序的选项。脸书首席执行官马克·扎克伯格计划在下午1:00的太平洋电话会议上就今天的公告接受提问。

关于脸书API，以下是更改的缩略版本及其含义:

  • 未来，活动API 将需要获得批准才能使用，开发者将不再能够调出来宾名单或在活动墙上发帖。这可能会破坏一些活动发现和票务应用程序。
  • 群组API 将需要脸书和一个群组管理员的批准，开发者将无法获得成员列表或与帖子相关的姓名和照片。这将限制群组管理应用程序只对有信誉的开发者开放，并防止封闭群组中的单个非管理员成员授予开发者访问权限。
  • Pages API 将只对提供“有用服务”的开发者开放，未来所有的访问都需要脸书的批准。这可能会严重限制页面管理应用程序安排帖子或审核评论。
  • 脸书登录使用将需要更严格的审查过程，应用程序将无法获取用户的个人信息或活动，此外，如果超过3个月未使用，他们将失去访问权限。不过，大多数登录应用程序应该仍然可以工作，因为实际上很少有人需要你的宗教信仰或视频观看活动，尽管一些专业应用程序可能无法在没有你的工作历史的情况下运行
  • 通过电话号码或电子邮件进行搜索将不再有效，因为脸书表示，它发现恶意行为者正在利用他们将一条信息与某人的身份进行配对，并循环使用IP地址以避免被脸书屏蔽。这可能会让那些拥有相似名字的国家的人们更难找到彼此。在所有的变化中，这可能是对用户体验损害最大的。
  • 账户恢复将不再立即显示用户的身份，当有人提交他们的电子邮件或电话号码，以同样防止刮擦。该功能仍然有效，但可能会更加混乱。脸书认为，所有用户的数据都可能被搜索和账户恢复技巧窃取了。

Schroepfer说，脸书的目标是锁定一切，审查一切，然后找出哪些开发人员应该访问，以及是否应该恢复任何功能。这些声明引发了人们的疑问，为什么脸书要在剑桥分析公司丑闻之后才认真对待数据隐私问题。你可以预期，当马克·扎克伯格4月10日来作证时，众议院能源和商业委员会可能会问这个问题。

脸书首席技术官迈克·斯科洛普夫

脸书改革其API的大胆行动表明，它愿意将用户置于开发者之上——至少有一次是在公众反对和内部冲突的推动下。平台whiplash可能会让开发人员担心将来会在脸书上进行开发。但是，如果脸书不支持数据隐私，如果未来外部开发人员侵犯隐私的行为被曝光，它将无法辩护。

Schroepfer告诉我，脸书非常认真地对待自己的责任，该公司对允许这种情况发生感到不安。至少他看起来很认真。上周，我写道，如果脸书想要稳定其员工日益低落的士气，它需要做出重大的忏悔和谦卑之举。这些彻底的改变是合格的，并且可以作为脸书团队的号召。他们不是坐在那里抱着头，而是有一个路线图来解决问题。

尽管如此，鉴于公众对API和平台缺乏了解，脸书可能很难重新获得被一个月来关于社交网络隐私疏忽的野蛮头条打破的信任。从长远来看，这种观点的恶化可能会让用户不愿在脸书上分享太多内容。但考虑到它作为一个在网上用你的身份登录的无处不在的工具的作用，我们滚动其提要和检查其通知的强迫性欲望，以及缺乏可行的社交网络替代品，脸书最终可能会看到反弹。希望这不会导致一切照旧。
