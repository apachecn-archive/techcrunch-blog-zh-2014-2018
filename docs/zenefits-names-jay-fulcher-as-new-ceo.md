# Zenefits 任命 Jay Fulcher 为新任首席执行官

Zenefits 正式有了新老板。《华尔街日报》最先报道称，Ooyala 和敏捷软件公司的前首席执行官杰伊·富尔彻将接替大卫·萨克斯的职位。富尔彻是过去 12 个月中担任 Zenefits 首席执行官的第三个人。

去年 2 月， [Zenefits 时任首席执行官 Parker Conrad 因公司合规和监管问题离开了公司](https://web.archive.org/web/20221201074937/https://beta.techcrunch.com/2016/02/08/zenefits-ceo-parker-conrad-out-amid-compliance-concerns/)。为了让 Zenefits 重回正轨，当时的首席运营官[大卫·萨克斯接任首席执行官](https://web.archive.org/web/20221201074937/https://beta.techcrunch.com/2016/02/08/zenefits-ceo-parker-conrad-out-amid-compliance-concerns/)。然后，去年 12 月，萨克斯说他将离职，并积极寻找替代者。

萨克斯在 TechCrunch 获得的一封写给 Zenefits 员工的电子邮件中写道，对富尔彻的任命显然是“一年前的本周开始的有计划过渡的最后一步”。他接着说，他感到很幸运能够将权力移交给 Fuclher，“一位在人力资源和 SaaS 都有丰富经验的经验丰富的领导者，他的精力、经验和领导力完全符合公司现在需要的。”

Fulcher 在 TechCrunch 获得的一封给员工的电子邮件中写道，他将自己的新角色称为“难得的机会”。他也认识到，在 Zenefits 的最后一年是“艰难的”，充满了变化。

“但是，尽管如此，我真的对这个机会感到兴奋，”富尔切写道。“我加入 Zenefits 是因为我相信这家公司可以成为行业中的一支创新和颠覆性力量，并有望产生巨大影响。”
