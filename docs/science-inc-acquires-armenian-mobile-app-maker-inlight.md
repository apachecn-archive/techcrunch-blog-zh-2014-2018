# 科学公司收购亚美尼亚移动应用制造商 Inlight

看起来科学公司对移动开发越来越认真了——创业工作室宣布它已经收购了一家位于亚美尼亚埃里温的名为 Inlight 的八人公司。

该公司开发了[同名的 iOS 应用](https://web.archive.org/web/20230129095733/https://itunes.apple.com/us/app/inlight-app/id789613452?mt=8)，这是一项面向女性的社交新闻阅读服务。[科学](https://web.archive.org/web/20230129095733/http://www.crunchbase.com/organization/science)首席执行官[迈克·琼斯](https://web.archive.org/web/20230129095733/http://www.crunchbase.com/person/mike-jones)和首席商务官[彼得·范](https://web.archive.org/web/20230129095733/http://www.crunchbase.com/person/peter-pham)(上图来自 TechCrunch Disrupt)都告诉我，虽然该公司将继续支持该应用程序，但收购的真正目的是雇佣一个移动开发团队来支持科学公司。

这是该公司称自己为工作室，而不是另一个孵化器或加速器的原因之一。它不仅为公司提供资金和指导，还雇佣了一个可以为其公司制造产品和提供其他服务的团队(它投资其中一些公司，并直接拥有其他公司)。在这一点上，Pham 说，Science 已经建立了“一个启动电子商务公司的代码库”，例如，该领域的任何科学公司都可以“内置所有的分析和优化功能”

科学公司包括[美元刮脸俱乐部](https://web.archive.org/web/20230129095733/http://www.crunchbase.com/organization/dollar-shave-club)， [DogVacay](https://web.archive.org/web/20230129095733/http://www.crunchbase.com/organization/dogvacay) ，以及[通过收购](https://web.archive.org/web/20230129095733/http://dealbook.nytimes.com/2014/05/08/delicious-social-site-is-sold-by-youtube-founders/?_php=true&_type=blogs&_r=0)，[美味](https://web.archive.org/web/20230129095733/http://www.crunchbase.com/organization/delicious)。

尽管公司总部设在圣莫尼卡，但它在印度也有一个开发团队。琼斯说，缺少的是一个专门专注于移动的团队。

“我对移动的部分信念是，我们需要非常迅速地进行大量押注，”他说。“我们的信念是，为了做到这一点，我们真的需要一个一流的快速开发团队。”

Pham 补充说，拥有这个团队可以帮助科学“更多地用纯粹的社交产品进行实验”，这是一个尚未成为大焦点的领域(尽管琼斯曾担任 Myspace 的前首席执行官)。他说，这对“可能不是核心移动公司”的科学公司也很有用——他们需要建立一个移动应用程序，但聘请全职移动开发人员不一定有意义。

交易的财务条款没有披露。
