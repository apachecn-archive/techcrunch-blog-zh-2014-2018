# 这可能是法拉第未来对特斯拉 Model X 的回应

据 Electrek 报道，【Faraday Future 正在洛杉矶驾驶一辆真正即将生产的电动汽车的伪装版。该网站报道称，偷拍的一辆被掩盖的 SUV 实际上是一辆测试车，来自加州的特斯拉潜在竞争对手，由中国消费电子产品公司 LeEco 支持。

这款车看起来像一款跨界风格的 SUV，类似于特斯拉 Model X，这与之前的报道一致，即法拉第未来正在寻求推出一款 10 万美元以上的豪华运动型多功能车跨界车，这将基本上打入特斯拉汽车已经涉足的市场。

该 SUV 将是其第一款量产车，此前该公司在今年的 CES 上推出了一款远不实用但更炫的 [FFZER01 概念车。FFZER01 实际上看起来像任天堂的 F-Zero 未来科幻赛车游戏系列中的东西，理论上有超过 1000 马力和不到三秒的 0-60 英里/小时时间。](https://web.archive.org/web/20221007213546/http://www.autoblog.com/2016/01/05/faraday-future-ffzer01-ces-official/)

Faraday 的概念电动赛车马戏团表演与你用来买杂货的东西相去甚远，这就是为什么这辆测试车如此有趣。毕竟，在成为一家可行的电动汽车公司方面，法拉第还有很多事情要证明。
