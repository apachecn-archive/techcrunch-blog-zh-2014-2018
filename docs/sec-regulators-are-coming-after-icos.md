# 美国证券交易委员会监管机构正在追查 ICOs

看起来，首次发行硬币的简写形式, [ICOs](https://web.archive.org/web/20230406074002/https://techcrunch.com/2017/05/23/wtf-is-an-ico/) 即将接受更多的审查。

美国证券交易委员会得出结论，数字货币融资事件将作为证券进行监管，这意味着未注册的产品可能会受到刑事处罚。这个决定是在星期二[宣布的。](https://web.archive.org/web/20230406074002/https://www.sec.gov/news/press-release/2017-131)

为了得出调查结果，监管机构评估了一个由“道”促成的产品，该产品导致黑客窃取。[报告](https://web.archive.org/web/20230406074002/https://www.sec.gov/news/press-release/2017-131)总结道，“分布式账本或区块链科技证券的发行人必须登记此类证券的报价和销售，除非适用有效的豁免。”美国证券交易委员会表示，其报告旨在提醒“投资者注意投资欺诈的危险信号，新技术可能被用来实施可能不符合联邦证券法的投资计划。”

美国证券交易委员会不会在这一特殊事件中提出指控，但表示这将作为未来发行的一个警告。那些“参与未注册发行的人也可能因违反证券法而承担责任。”

这对许多将 ico 作为融资替代方式的初创公司来说是一个打击。最近几个月出现了一波此类产品，人们通过比特币、以太坊或其他加密货币投资商业创意。

但是和所有的创业公司一样，这些投资都有风险。ico 的不透明性意味着没有足够的监督企业如何处理收益。许多硬币在二级市场交易，这提供了短期流动性。

尽管许多 ico 都是规模较小的未知公司，但艰难的融资环境已经导致一些风险投资支持的初创公司筹集足够的资金，以进入下一步。消息应用程序 Kik 是最近一个显著的例子。

由于预计到 SEC 的打击，一些初创公司已经禁止美国投资者进入。

最近几个月，比特币和以太坊价格飙升，但由于担心货币分裂，本周价格下跌。
