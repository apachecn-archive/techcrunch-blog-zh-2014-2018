# Spotify 现在集成了亚马逊 Echo...如果你是高级用户 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/02/04/spotify-now-integrates-with-amazon-echo-if-youre-a-premium-user/>

对于谦逊的人来说， [Amazon Echo](https://web.archive.org/web/20230204072018/http://www.amazon.com/echo) 看起来像一个新奇的立体声扬声器，因此音乐是亚马逊的互联语音激活人工智能设备中更受欢迎的应用程序之一也就不足为奇了。从今天开始，Echo 上的音乐得到了提升，T2 Spotify 和亚马逊 Echo 之间有了新的直接整合。

亚马逊 Alexa 总监托尼·里德(Toni Reid)在一份声明中表示:“现在，从 Spotify 播放你最喜欢的音乐就像问 Alexa 一样简单。”“音乐是亚马逊 Echo 上最受欢迎的功能之一，Spotify 是最受欢迎的服务之一，所以我们很高兴今天能把它带给我们的客户。”

Spotify 高级订阅用户也有一个亚马逊 Echo，现在可以命令 Echo 的虚拟助手 Alexa“播放 Spotify”，并调用他们的 Spotify 播放列表，以及流媒体服务上的艺术家或流派。

“我们非常高兴 Spotify 高级用户现在可以在亚马逊 Echo 上听他们最喜欢的音乐，”Spotify 全球硬件负责人伊恩·盖勒在一份声明中说。“创造创新体验是我们的核心使命。我们知道我们的用户会喜欢在家里用他们的声音控制音乐。”

Spotify 的一位发言人说，这项服务最初将只在美国出售，亚马逊 Echo 将在美国出售，没有关于进一步推出的更多细节。这是 Echo 客户要求最多的新增功能之一，Spotify 是一项全球性服务，所以我猜测，当亚马逊将 Echo 扩展到其他市场时，Spotify 的整合也会随之而来。

亚马逊 Echo 还与 Spotify Connect 集成在一起:Spotify 用户可以从他们的应用程序中选择 Echo 来切换控制。

把 Spotify 和 Amazon Echo 放在一起是相互营销的一个例子。

对于亚马逊 Echo 来说，这可能会推动产品的更多销售。使用这款设备听音乐有一种天然的亲近感，如果你是那种投资 Echo 的消费者，你也很有可能会播放数字音乐。

亚马逊 Echo 已经与其他几个流媒体服务整合——亚马逊音乐、Prime 音乐、iHeartRadio、Pandora 和 TuneIn。但 Spotify 是全球市场的领导者，拥有 1 亿用户，其中付费用户超过 2800 万。能够在设备上播放 Spotify 可能会成为更多 Echo 购买的引爆点，或者至少有助于让它更具主流吸引力。

这也符合亚马逊对 Echo 的整体战略。就像 Kindle 平板电脑、电子阅读器和其他硬件一样，亚马逊迄今尚未公布其 180 美元 Echo 设备的任何销售数据。但它一直在积极增加更多集成，以提高设备的实用性——并可能有助于为运行 Alexa 和潜在的许多其他服务的机器学习算法提供信息。

这些功能包括非常重要的操作，如订购披萨、创建购物和其他待办事项列表、访问新闻和其他信息、设置闹钟等等。

对于 Spotify 来说，Echo 流媒体是一项高级功能的事实很有意思:它实际上给了为高级服务付费的消费者另一个好处。如果你已经是 Echo 的用户，你可以在设备上播放 Spotify 的事实可能会成为你最终决定从免费 Spotify 用户转向付费用户的原因。

如今，Spotify 的高级层每月收费 9.99 美元，为用户提供无广告收听、更高的音频质量、离线播放和其他功能。

这也符合 Spotify 更广泛的战略:就活跃用户而言，该公司可能是当今最大的流媒体音乐提供商，但它面临着来自苹果和谷歌等公司以及 Deezer 等其他纯流媒体公司的激烈竞争——顺便提一下，Deezer 今天在与华为的新协议中取得了自己的硬件进展，将其服务嵌入其新的 Honor 手机。

但是流媒体音乐在某种程度上是一种商品，在基本目录方面几乎没有什么区别。因此，Spotify 可以确保其用户保持忠诚或迁移到其服务之上的一种方式是通过其围绕目录建立的服务——无论是播放列表、独家功能还是简单的便利因素。

在这种情况下，流行硬件的普及是必须的。
