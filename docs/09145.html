Thalmic Labs 从亚马逊 Alexa 基金和更多 TechCrunch 融资 1.2 亿美元，推出新产品

原文：https://web.archive.org/web/https://techcrunch.com/2016/09/19/thalmic-labs-teases-new-products-with-120m-raise-from-amazon-alexa-fund-and-more/

Thalmic 实验室从亚马逊 Alexa 基金筹集了 1.2 亿美元及更多资金，用于开发新产品

滑铁卢的 Thalmic Labs 已经从包括英特尔投资、亚马逊 Alexa 基金和加拿大富达投资在内的投资者那里筹集了 1.2 亿美元的新一轮 B 轮投资。新的资金将大大增加该公司的资金，并应帮助它实现其目标，扩大其产品线，超越目前提供的原始 Myo 臂章。

Thalmic 的 Myo 是一种连接带，可以从佩戴者的手臂上获取电脉冲，并将其转化为计算平台的输入。它已经被用于一些事情，比如在传统的 2D 台式电脑上跳过音乐播放器中的曲目，控制虚拟现实游戏中的虚拟角色，甚至帮助 T2 控制和治疗使用假肢。

Myo 于 2012 年首次原型化，并于 2014 年向公众发布。Thalmic 在发布前管理了自己的预购活动。

Thalmic 的 Myo 技术走在了时代的前面；它开创的交互模型并不真正适合日常计算，而且它对于消费无人机市场来说仍处于相对早期的阶段，这是另一个帮助 Myo 产生早期兴奋的演示。两年后的今天，消费者虚拟和混合现实的引入帮助 Myo 产生了更适合消费者使用其技术的需求。此外，Thalmic 与医学科学的合作，以及创新系统，如用于手语的手势输入系统，完善了该平台的优点。

本月早些时候，Thalmic 还开设了一个旧金山办事处，并宣布 Tara Kriese 为其新的 CMO。Kriese 从三星电子美国公司来到 Thalmic，在那里她负责三星 VR 产品的营销。该公司表示，旧金山的扩张和新的资金意味着 Thalmic 正在寻求在业务的各个方面进行招聘。

至于 Thalmic 即将推出的产品，该公司没有具体说明它们是什么；相反，它只是指出，他们“在管道中”，它将“很快分享更多。”
