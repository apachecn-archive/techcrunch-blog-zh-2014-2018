# 这个面部识别系统可以追踪你欣赏电影的方式

随着电影制作变得像艺术一样科学，电影制作者需要更好的方法来衡量观众的反应。他们喜欢吗？具体多少钱？在第 42 分钟？加州理工学院和迪士尼研究中心的一个系统使用面部表情跟踪神经网络来学习和预测观众的反应，或许为新一代尼尔森收视率奠定了基础。

刚刚在夏威夷举行的 IEEE 计算机视觉和模式识别会议上展示的研究项目展示了一种新方法，通过这种方法，可以可靠且相对简单地实时跟踪剧院中的面部表情。

它使用了所谓的因子分解变分自动编码器——我甚至不打算解释它的数学原理，但它比现有的方法更好地捕捉了复杂事物的本质，如运动中的人脸。

该团队通过记录数百名观看电影的观众(当然是迪士尼的观众)收集了大量面部数据。一台红外高清摄像机捕捉每个人的动作和面部，由此产生的数据——约 1600 万个数据点——被输入神经网络。

一旦完成训练，该团队就将系统设置为实时观看观众镜头，并试图预测给定人脸在不同时刻的表情。他们发现，可以说，对观众热身需要大约 10 分钟，之后可靠地预测了笑声和微笑(似乎还没有跟踪到哭泣或惊吓)。

当然，这只是这种技术的一个应用——它可以应用于其他情况，如监控人群，或实时解释复杂的视觉数据。

“理解人类行为是开发展示更大行为和社会智能的人工智能系统的基础，”加州理工学院的宋轶·岳[在新闻发布会](https://web.archive.org/web/20230320060241/http://www.caltech.edu/news/neural-networks-model-audience-reactions-movies-79098)上说。“例如，开发人工智能系统来帮助监控和照顾老年人，依赖于能够从他们的肢体语言中获取线索。毕竟，人们并不总是明确地说他们不开心或有一些问题。”
