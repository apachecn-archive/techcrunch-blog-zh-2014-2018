# 3D 打印的 Peek 相机有助于诊断发展中地区的眼病

Peek 或便携式眼部检查套件应用程序是一种基于智能手机的眼部问题诊断系统。它使用 3D 打印的附件，允许眼科医生使用一个应用程序和一个小型相机覆盖图在现场进行详细而完整的眼部检查。该系统由英国眼科医生开发，最近在 Indiegogo 上的点击量达到了 130，475 英镑，目前正在 T2 接受医生的预购。

Peek 可以使用智能手机的高质量摄像头观察视网膜，查看白内障，并提供视力测试以及颜色和对比度测试。该项目由 Andrew Bastawrous 博士和 Stewart Jordan 领导，由 Kate Tarling 和 Mario Giardini 博士设计。它旨在帮助全世界 3900 万无法获得基本光学保健的人。在 TED 演讲中，Bastawrous 博士描述了他帮助发展中地区获得更好的眼部护理的研究，包括通过白内障摘除和处方镜片恢复视力。Peek 拍摄的医学图像可以远程发送给医生，为患者诊断和建议治疗方法。

创作者注意到，全世界 80%的失明是可以预防的，但是在这个领域解决眼睛问题所必需的工具是笨重、笨拙和难以操作的。通过将一块塑料与廉价的智能手机配对，眼科医生能够解决世界任何地方的眼部护理问题。有趣的是，该系统还允许你通过在屏幕上显示越来越小的字母和数字来进行视力测试。它也通过类似的方法解决色盲问题。

这显然不是为了消费者，但它将使医生和受过训练的人员能够在实地进行真实和可用的眼睛测试。通过将眼科办公室的工具放入您的口袋，并提供一种非常简单和廉价的方法来拍摄眼睛内部的精彩照片，医生可以访问有需要的位置，并在飞行中帮助人们。该项目得到了充分的资助，应该很快就会开始运输应用程序和 3D 打印的附件。
