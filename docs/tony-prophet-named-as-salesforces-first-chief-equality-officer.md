# Tony Prophet 被任命为 Salesforce 的首位首席平等官

曾在微软和惠普工作过的资深科技高管托尼·预言家被任命为 Salesforce 的首位首席平等官。

在加入 [Salesforce](https://web.archive.org/web/20230323181500/http://www.salesforce.com/) 之前，Prophet 曾担任微软公司教育营销副总裁。在微软时，这位高管发起了许多倡议，包括微软的黑人和 BlackLight，这是一个为黑人营销人员赋权的组织。在他工作了八年半的惠普公司，Prophet 领导了惠普打印个人系统业务的运营。

在一份声明中，Salesforce 将 Prophet 描述为“人权和社会正义的捍卫者”除了在微软的工作，Prophet 还帮助旧金山湾区的非营利组织，致力于儿童保健、低收入青少年和艾滋病毒阳性妇女的教育机会。

Salesforce 首席执行官马克·贝尼奥夫(Marc Benioff)和 Prophet 的关系可以追溯到 Prophet 在惠普工作期间。在惠普工作期间，Prophet 致力于为 UCSF 儿童医院筹集资金(这是一个与贝尼奥夫的心息息相关的项目)，并于同年在罗恩·康韦家的一个圣诞派对上结识。

Prophet 甚至是 2014 年贝尼奥夫[梦想力量](https://web.archive.org/web/20230323181500/https://www.salesforce.com/dreamforce/DF16/)会议的特邀发言人。

Prophet 将他的母亲，Benioff，罗恩·康韦和 Laura Arrillaga-Andreessen 列为个人偶像，但在专业上——特别是在惠普——Prophet 说他最自豪的是三项社会正义倡议。首先是关注惠普产品中冲突矿物的使用；第二，惠普供应链中员工的工作生活——包括级联式同伴教学；最后，Prophet 强调了对环境的关注。

Salesforce 一直站在实现平等(不仅仅是多元化和包容性)以及每个企业都需要解决的青年教育和医疗保健问题的最前沿。
