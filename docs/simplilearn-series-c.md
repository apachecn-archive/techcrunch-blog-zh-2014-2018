# Simplilearn 为专业培训和认证筹集了 1500 万美元

Simplilearn 是一家帮助客户获得 PMP(项目管理)和安卓开发等领域认证的公司，已经在 C 轮融资中筹集了 1500 万美元。

该公司提供在线课程和课堂研讨会，并表示有 200 多门课程和 40 项认证。自 2010 年推出以来，已有超过 40 万人通过了 Simplilearn 课程的认证。

首席执行官 Krisha Kumar 表示，最受欢迎的课程包括大数据和分析、数字营销、项目管理和各种网络编程语言。

但是，在线教育中肯定有很多竞争，甚至在专业培训中，我们也涵盖了几种不同的方法——例如，[我最近写了一篇关于 Grovo](https://web.archive.org/web/20230207190925/https://techcrunch.com/2015/02/18/grovo-series-b/) 的文章，它可以制作简短的培训视频。库马尔说，Simplilearn 之所以脱颖而出，是因为它专注于“对职业发展有直接影响的短期认证课程”。

他补充说，2015 年的目标包括培训 100 万新用户，并将课程目录增加一倍。

[Simplilearn 现在已经筹集了 2700 万美元的资金。新一轮由 Mayfield Fund 领投，过去的投资者 Kalaari Capital 和 Helion Venture Partners 参与了投资。](https://web.archive.org/web/20230207190925/http://venturebeat.com/2013/09/17/simplilearn-raises-10m-so-professionals-dont-get-left-in-the-dust/)

“Simplilearn 在过去几年中取得了令人印象深刻的增长，我们相信这一势头将会持续下去，因为越来越多的人寻求专业认证以取得成功，”Mayfield 董事总经理 Navin Chaddha 在资金发布中表示。“我们相信，额外的资本将使克里希纳和他的团队发展得更快，超越市场。”
