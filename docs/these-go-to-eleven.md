# 新的 Razer Blade 使笔记本电脑上的高分辨率游戏不那么容易实现

![Razer Blade](img/d97dc259487e2a8ee3e6742291df8176.png)

随着 2015 年 Razer Blade 的首次亮相，Razer 再次证明了它能够在一个狭小的空间内压缩大量的能量，这是该公司旗舰笔记本电脑系列的最新产品，为游戏市场带来了苹果般的旋转。

去年，Razer 将其已经光滑的 Blade 变成了 Retina MacBook Pro 的竞争对手，其 14 英寸屏幕的每一个维度的像素数量都增加了一倍，分辨率达到了惊人的 3200 x 1800。这不太像 4K，但与大多数人习惯的靠窗户的屏幕相比，这已经很大了。

唯一的问题是，在那个分辨率下玩现代视频游戏需要一个巨大的装备。花了 2199 美元购买刀片服务器的人通常不得不降低纹理质量、着色器或其他设置，以获得不错的帧速率。就是这样，或者调低分辨率，浪费花哨的显示屏。

这在今年的刀片式服务器上不会成为太大的问题。图形芯片从去年的 Nvidia Geforce GTX 870M 升级到新的 970M，在原始功率和性能功耗比方面轻松击败了其前代产品。它不会以笔记本电脑的原生分辨率玩每一个高设置的新游戏(这是不公平的)，但大多数游戏看起来绝对令人惊叹。

Razer 全面提高了规格，以便不成为新 GPU 的瓶颈。随着内存从 8 GB 增加到 16 GB，去年的 CPU 已经换成了通常为更大的笔记本电脑保留的 Haswell Core i7。这可能不会对笔记本电脑的电池寿命产生奇迹，但它应该会为 CPU 密集型游戏和应用程序带来更好的体验。

对于那些预算稍微紧张的人，Razer 还提供了一款 8 GB 内存和非触摸 1080p 屏幕的机型，售价为 1999 美元。在我最近测试的游戏笔记本电脑上，我很少受益于触摸屏，使用高端型号的 CPU/GPU，你可以在 1080p 的最大设置下玩大多数游戏，即使只有一半的 RAM。
