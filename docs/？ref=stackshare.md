# 开发者社交网络 StackShare 从 e.Ventures  融资 520 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/01/09/developer-social-network-stackshare-raises-5-2-million-from-e-ventures/?ref=stackshare>

Web 开发是复杂的。几年前曾经相对统一的 Linux、Apache、MySQL 和 PHP 的[灯栈](https://web.archive.org/web/20230316231734/https://en.wikipedia.org/wiki/LAMP_%28software_bundle%29)已经被几乎无限的变化所取代。多个云提供商、数十个数据库、数百个开发工具，这还不算图书馆。[节点包管理器 NPM](https://web.archive.org/web/20230316231734/http://npmjs.com/) 列出了[大约 50 万个可以单独导入的包](https://web.archive.org/web/20230316231734/https://www.linux.com/news/event/Nodejs/2016/state-union-npm)。

对于软件开发人员来说，选择是令人眼花缭乱的，而且网上经过审查的资源有限，无法了解哪些组合效果好(尤其是在生产中)，哪些不好。随着技术步伐的加快，这越来越意味着工程师们花费更多的时间试图跟上最新技术的步伐，而不是真正制造出可投入生产的产品。

输入[栈共享](https://web.archive.org/web/20230316231734/https://stackshare.io/news)。该社区最初于 2014 年作为一个不起眼的 WordPress 博客网站推出，现在已经发展到超过 20 万名开发者，他们分享他们的技术堆栈以及如何使用它们的建议。今天，该公司宣布了由 e.Ventures 牵头的 520 万美元的 A 轮融资。这是几个月前 TechCrunch 报道的 150 万美元种子轮融资的顶部。

当 StackShare 开始的时候，它是相当静态的，因为用户只会列出他们正在使用的技术。该公司的创始人兼首席执行官 Yonas Beshawred 开始在所有这些数据中看到了机会。“当我们看到它时，它是专业人员相互寻找和交流。”

随着开发人员技术的激增和成熟，工程师们越来越需要将现有的解决方案缝合在一起，而不是滚动他们自己的代码。“当你的老板问你要一条新的数据管道时，你不要打开文本编辑器，你要做研究，”贝肖瑞德说。他把这比作“玩乐高”——目标是利用现有的积木，创造出新的东西。

StackShare 越来越希望成为一个地方，工程师们可以在那里获得可行的建议，在出现这类研究问题时选择什么技术。我们的目标不仅仅是知道 Airbnb 正在使用 React，而是知道 Airbnb 的工程团队中谁在实际编写它，以及他们从生产中运行它中学到了什么。

Beshawred 将他正在收集的数据描述为将人与技术联系起来的“技术图”。这个想法是，使用 StackShare 的搜索工具，一个创业团队可以与其他公司的经验丰富的工程师联系，这些工程师曾经使用过一个堆栈，可以帮助给出建议，说明哪些可行，哪些不可行。你可以搜索同时使用特定数据库、编程语言和前端库的公司，这些公司比你稍微大一点，而且学得更快。

StackShare 在其他大型开发者平台之间看到了一个机会。“当你看到最大的平台时，即使是那些专注于开发人员的平台，如 Github 和 StackOverflow，他们也不专注于技术，而是专注于代码。StackOverflow 必须是关于特定的一行代码。Github 也有同样的问题，它必须绑定到回购或拉取请求，”Beshawred 解释道。“我们所知道的工程师，世界已经超越了这一点。”

该公司去年年中宣布的一项新功能是名为 Stack News 的产品。开发人员经常在他们的代码中包含几十个产品和库，要跟上每个产品和库的最新消息几乎是不可能的。Stack News 可以查看用户使用的技术，并向他们发送每日或每周简讯，记录有关该技术的任何变化或新思维。

从长远来看，该公司认为招聘是其商业模式的一部分，也是供应商的一个明显的机会。[该公司开发了 Stack Match](https://web.archive.org/web/20230316231734/https://stackshare.io/posts/introducing-stack-match) 工具，让招聘人员能够找到完全符合他们需求的工程师。Beshawred 表示“如果你是招聘人员，我们可以为 B 轮创业公司搜索在 AWS 上使用过 Go 的人。如今，你在其他任何地方都做不到这一点。”对于工程师来说，StackShare 可以成为技术简历，很难在像微软的 LinkedIn 产品这样的网站上展示。

除了 e. Ventures，A 轮还包括 Cervin Ventures 和天使投资人 Nick Rockwell、Aston Motes、Dave Johnson 和比尔·史密斯。
