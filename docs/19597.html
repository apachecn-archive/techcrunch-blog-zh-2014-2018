微软取消更多密码 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/09/24/microsoft-wants-to-help-you-do-away-with-more-passwords/

我们都有数不清或者记不住的密码。除非你只有一个，在这种情况下，请给自己多拿几个。但是光靠密码远远不是保护我们账户安全的理想方式。这就是为什么大多数企业现在押注双因素身份认证来确保网络安全。然而，微软想加入完全废除密码的行列。

正如该公司今天在其 Ignite 大会上宣布的那样，它现在将支持成千上万的 Azure Active Directory 连接应用程序通过其 Microsoft Authenticator 应用程序进行无密码登录。“没有哪家公司能比微软让企业消除更多的密码，”该公司在今天的公告中自豪地写道。

该公司过去曾就此写过文章，现在已经为 Windows 10 用户提供了一个版本。对于 Azure Active Directory，Windows Authenticator 应用程序本质上复制了 Windows Hello 的功能，它允许用户使用他们的指纹、面部或 PIN 来登录他们的企业应用程序。这里的总体思想是，您仍然提供两个身份验证因素:您拥有的东西(您的手机)和您拥有的东西(您的指纹或面部)。

这里的就是个人账户的样子。企业帐户的流程非常相似。

对于企业用户，微软还在 Ignite 上发布了一些额外的安全公告。其中第一个是推出微软安全评分，Office 365 安全评分的进化。该服务为组织提供了一份安全状况报告卡，使他们能够快速评估需要做出哪些更改。这些变化可能是打开多因素身份验证或关闭电子邮件转发。

基于这一主题，微软今天还推出了微软威胁保护，这是一个面向微软 365 用户的新解决方案。由于微软 365 结合了一系列跨越在线和离线的服务，该公司现在可以提供同样的安全解决方案。威胁防护查看电子邮件帐户、电脑、文档和用户的基础设施，以检测和缓解攻击。

“网络安全是数字时代的核心挑战，”微软公司安全副总裁罗布·莱弗茨今天写道。“没有它，像隐私权这样最基本的人权就无法存在。每天，组织都会从其核心业务任务中抽出宝贵的时间和资源来防御网络攻击并从中恢复。他们操作数十种复杂的互不关联的工具，但是这些工具之间的差距仍然存在，威胁仍然存在。他们的安全团队很难跟上，熟练的专业知识也很匮乏。”
