自主机器人和玩游戏的人工智能——12月5日至6日在Disrupt London举行的精彩演示TechCrunch

伦敦TechCrunch Disrupt于12月5日至6日举行。抢票这里。除了扬声器和面板，我们还将展示一些令人惊叹的科技公司的演示。

第一个是波士顿动力公司。是的，伙计们，扰乱伦敦的代表们将有机会近距离亲自看到那些神奇的BD机器人中的一个，几乎真的是真人(如果他们有任何血肉的话，那是真的)。

首席技术官和波士顿动力的创始人马克·雷伯特将展示他的团队创造的一个令人惊叹的机器人，但你必须来看看是哪一个…

从1986年到1995年，Raibert是麻省理工学院电子工程和计算机科学的教授，也是人工智能实验室的成员。Raibert的研究致力于研究动态移动的系统，包括物理机器人和动画生物。Raibert在麻省理工学院的实验室Leg Lab因其在动态移动系统方面的工作而闻名，包括腿式机器人、模拟机械装置和动画人物。

腿部实验室创造了一系列实验室机器人，包括一条腿的跳跃者、两足跑步者、一只四足动物和两只类似袋鼠的机器人。总的来说，这些机器人沿着简单的路径行进，主动保持平衡，爬简单的楼梯，快速奔跑(13.1英里/小时)，用几种步态奔跑，并做基本的体操动作。

几年前，两个机器人(和三个学生)与肖恩·康纳利和韦斯利·斯奈普斯一起出现在《旭日东升》中。Leg Lab还制作了《奔跑吧》( On The Run ),这是一部计算机生成的卡通片，其中所有角色都使用模拟和控制技术制作了动画。波士顿动力公司在自动化角色和基于物理的动态模拟方面的工作是麻省理工学院Raibert小组研究的成果。

Raibert于1973年获得东北大学电气工程学士学位，并于1977年获得麻省理工学院博士学位。他的博士论文题为“状态空间模型的运动控制和学习”，使用机器人技术来模拟生物行为。从1977年到1980年，他在喷气推进实验室和加州理工学院研究机器人传感和控制。1981-1986年，他在卡内基梅隆大学计算机科学系和机器人研究所工作。他是麻省理工学院出版社出版的平衡的腿式机器人的作者，是《国际机器人研究杂志》的编辑委员会成员，是两期专门讨论腿式系统的《IJRR》的客座编辑，并且是AAAI的研究员。

与此同时，克里斯·布斯(Chris Boos)也将一同出席，他是领先的人工智能公司阿拉戈的创始人兼首席执行官，该公司通过智能自动化帮助企业实现IT流程自动化。

克里斯让他的团队让他们的平台玩一种特定的游戏，这种游戏实际上比围棋更复杂，围棋是DeepMind用AlphaGo征服的中国游戏。但是你应该买一张票，看看有哪些游戏，以及它们是如何进行的。

克里斯在阿拉戈监督战略和产品开发，帮助该公司建立在人工智能的前沿，并继续进行人工智能的研究，同时指导阿拉戈的技术在商业上的应用。作为图论和决策系统方面的专家，Chris在苏黎世联邦理工学院和达姆施塔特技术大学学习计算机科学，在美国和欧洲的机构从事研究，并于2003年获得了约翰·肯尼迪国家领导奖。

除了参与公司事务，Chris还活跃在R&D，并担任多个董事会成员，通过天使投资支持欧洲的初创企业。你可以在这里跟随克里斯对未来和AI 的想法。
