Stereolabs 的深度感应相机帮助机器人、无人机和汽车看到

Stereolabs 的 ZED 立体摄像机是一种类似 Kinect 的传感器，允许开发人员将深度传感功能内置到他们的设备中。比如你想在机器人或无人机中内置避障功能。为此，您需要对机器周围的情况有一个基本的了解。最简单的方法(假设你不想使用雷达或激光雷达)是使用立体摄像机。

这款 449 美元的相机通过标准的 USB 3.0 端口连接，然后 Stereolabs 使用主机上的 GPU 来渲染实时深度图。根据使用情况，您可以在每秒 15 帧的高分辨率 2.2k 图像之间进行选择，或者选择具有更高帧速率的较低分辨率(对于 1280×480 视频高达 120fps)。

该公司将在今天的 CES 上在我们的硬件战场推出 ZEDfu。Stereolabs 创始人兼首席执行官 Cecile Schmollgruber 告诉我们，ZEDfu 是一款 3D 扫描应用程序，它将相机数据与位置数据相结合，在任何给定的环境下创建 3D 地图。

“最容易实现的用途是 3D 扫描。事实上，由于这个原因，更新还包括一个 3D 扫描应用程序，”Schmollgruber 告诉我。“但能够实时绘制环境地图也为自主导航、增强现实和其他新兴技术铺平了道路。”

Stereolabs 表示，与类似产品相比，它的相机技术让开发者更容易开始深度制图。例如，相机校准(和重新校准)对于 ZED 来说不是问题，该团队认为其计算算法比竞争对手更有效。Schmollgruber 告诉我，她认为立体视觉比其他一些传感技术更有优势，因为它比激光雷达系统更灵活，更便宜。

该公司目前为许多平台提供 SDK，包括 Windows 和 Linux。不过，更有趣的是，它现在还提供与 Oculus Rift 的第三方集成，例如虚拟现实/增强现实应用，以及在 Matlab 和专业机器人和计算机视觉平台如 OpenCV 和 ROS 中使用 ZED 数据的网关。

Schmollgruber 告诉我，Stereolabs 也在与 NVIDIA 等合作伙伴合作，将其技术应用到汽车中。她说:“英伟达已经将其强大的 Tegra 芯片集成到顶级汽车制造商中，zed 相机针对该芯片进行了优化，使汽车制造商更容易测试 ZED。”
