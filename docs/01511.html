音乐协作套件 Splice 从 True 和 DJs 筹集了 450 万美元，推出了公开测试版 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2014/09/17/splice-music-collaboration/

GroupMe 的联合创始人史蒂夫·马托奇为他的音乐制作存储和合作初创公司 Splice 找到了一些强大的盟友，该公司今天发布了其公测版和可混音音乐播放器。

瑞典浩室黑手党的超级巨星 DJ tisto 和 Steve Angello、经理人 Scooter Braun 以及经纪公司 WME 和 AM Only 投资了 Splice 新的 450 万美元 A 轮融资，由 True Ventures 领投，Union Square Ventures 参与。经过 10 个月的私人测试，任何使用 Ableton、Logic 或 FL Studio 制作套件的音乐人现在都可以免费注册 Splice 的公共测试。Splice 自动保存和同步歌曲的每一处更改，让合作者轻松地注释和共享正在进行的工作。

为了让听众了解 Splice 能做什么，今天首次推出了 DNA 播放器，用该软件制作果酱。它将组成一首歌曲的每个单独的轨道可视化，让人们静音或独奏某些乐器，并留下 SoundCloud 风格的定时评论。DNA 播放器以方平的“尖叫”和提斯托的标签“音乐自由”的 J-Trick 首次亮相，你可以在这里或下面的嵌入中查看。

Martocci 告诉我，在没有拼接的情况下创作音乐是一个不断管理文件保存的过程，并且担心你的作品没有备份或不能更改。与其他音乐人合作也是一种痛苦，因为很难保持同步，知道谁改变了什么，或者为什么。这就是为什么他认为 Splice 是一个远远超越 Dropbox 的重要产品。“这不仅仅是知道他们将文件存储在哪里。你什么都不用想，”他告诉我。最终，Splice 将收取订阅费，但目前是免费的。

自最初推出以来，Splice 一直在与顶级艺术家签约，获得他们的反馈，并举办活动。为了展示它的潜力，Splice 一直在举行聚会，每个音乐家花 10 分钟创作一首歌，然后将它传递给下一个制作人。马托西自豪地说，“你不会相信这些人合作无间是多么令人惊奇。”

现在的焦点是让 Splice 成为制作音乐的默认方式。450 万美元的首轮融资包括 DJ Richie Hawtin 的 Plus 股票基金 LP，使 Splice 的总融资达到 725 万美元。Martocci 告诉我“我们对这个公开测试版感到兴奋的一大部分是更多的反馈来完善我们的平台。我们希望扩大我们的艺术家网络，成为音乐创作过程的核心部分。这轮融资和相关投资者将帮助我们做到这一点。”让 DJ、经理和代理人出现在排行榜上，为产品做宣传应该会大有帮助。

最大的挑战将是让出了名的顽固艺术家改变他们的创作过程。“使用 GroupMe 或我以前开发的任何应用程序，你都有自己的做事方式。但有了这个，你就像是在用他们的方式制作音乐，并把它和云连接起来，”马托奇承认道。“我们必须在我们想要的做事方式和他们的做事方式之间找到平衡。”

即使你不是音乐制作人，你也可以用 DNA 播放器做你想做的事情。每个底鼓、声线、激光枪和“摇摆”都可以打开或关闭，查看来自创作者和其他听众的注释，并留下自己的注释。

到今年年底，计划将有更多的创作工具集成到 Splice 中。马托奇笑着说:“一旦这些合作的障碍消失，我认为这将是一个音乐的新世界。”
