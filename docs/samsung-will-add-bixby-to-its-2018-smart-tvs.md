# 三星将在其 2018 年智能电视中添加 Bixby

![](img/8ae5aa00ed7c6793f561577ac1f03aaf.png)

三星的 bix by——该公司对 Siri 和谷歌助手等的回应——在 2017 年推出时，开始有点艰难，但他们正在稳步前进。

该公司刚刚在 CES 上宣布，Bixby 将于 2018 年集成到其所有新的美国智能电视中。

[![](img/80a22dd5b0788dc81b0daee502a281ff.png)](https://web.archive.org/web/20221006164056/https://beta.techcrunch.com/tag/smart-home-at-ces-2018/)

它可以让你做所有你期待的语音助手的基本工作，从询问天气问题到用你的声音寻找新的电视节目。但这一切都将与三星的 SmartThings hub 应用程序挂钩，让你可以通过电视控制无数的物联网设备。

至少根据舞台演示，它看起来像是通过遥控器上的麦克风处理语音输入，就像 Apple TV 上的 Siri 或 Fire TV 上的 Alexa 一样。按下一个按钮，与你的遥控器对话，你就设置好了。

我仍然喜欢我家周围的各种语音助手暂时成为独立的扬声器，特别是当所有相关公司都在解决问题并重复这个概念时。如果我决定从一个助手换到另一个助手，更换一个 50 美元的扬声器要比更换一个 1500 美元的电视容易得多。
