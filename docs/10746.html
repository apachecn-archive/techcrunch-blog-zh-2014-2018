2017 年推出的所有 Chromebooks 都将兼容安卓应用

2017 年新推出的所有 Chromebooks 都将支持开箱即用的 Android 应用。不需要更新。用户将可以把 Chromebook 带回家，打开它并立即访问谷歌 Play 商店。

这条消息来自谷歌兼容安卓应用的 Chromebooks 列表中的一行文字。“2017 年及以后推出的所有 chrome book 以及下面列出的 chrome book 在未来都将与 Android 应用程序兼容，”它说。

去年，谷歌终于让 Chromebook 与 Android 应用兼容，但迄今为止，支持仅限于部分型号，要求购物者查阅列表，以确定哪款 chrome book 支持谷歌 Play 商店。这一举措消除了购物的模糊性:购买新的 Chromebook，并知道可以安装 Android 应用程序。

Android 应用的加入极大地扩大了 Chromebooks 的吸引力。通过访问 Play Store，Chromebooks 获得了一个与 Windows 或 macOS 一样深入的生态系统。
