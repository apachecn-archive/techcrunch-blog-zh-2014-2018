如何在您的 Raspberry Pi 2 TechCrunch 上安装 Windows 10 IoT

由于针对多种单板计算机的 Windows 10 的发布，修补者们现在意识到了将 Windows 作为一种可用且令人惊讶的替代开源操作系统(如 Raspbian)来运行的可能性。但是如何在 RaspPi 上运行 Windows 呢？为什么呢？

首先，我邀请你看看微软令人耳目一新的完整的 GitHub 页面，在那里他们提供了在 Raspberry Pi、Arduino、Galileo 和 MinnowBoard 上安装 Windows 10 的说明。该页面提供了获取官方“Windows 嵌入式物联网”映像以用于这些设备的说明。你也可以在这里下载。

给 OSX 用户的一个提示:我原本希望包括如何使用 OS X 或 Linux 为 RaspPi2 闪存 SD 卡的说明，但遗憾的是，微软的 FFU 图像文件需要特定的 Windows 文件处理软件。我试图用显而易见的方法(改变扩展名)和不显而易见的方法(到处寻找答案)将它转换成一个图像文件，但通常这是一个我无法解决的问题。当我找到解决方法时，我会更新这篇文章。

更新–成功！要将 FFU 文件转换成 IMG 文件，只需像平常一样抓取这个 Python 脚本并安装生成的图像。

￼ffu2img 是一个简单的 python 脚本，可以将微软的 ffu 格式转换成标准的原始 img 格式。IMG 文件是您从 dd 获得的，而 FFU 涉及一系列确认、验证、签名甚至某种形式的重复数据消除。Raspberry Pi 2 Windows 10 映像不使用重复数据删除功能，否则写起来会更烦人。我跳过了该格式的所有块管理特性，只输出原始块。

以下是如何将 Windows 10 物联网安装到您的树莓 Pi 上。

1.在微软连接创建一个账户。

2.下载包含 flash.ffu 的文件Windows _ IoT _ Core _ rpi 2 _ build . zip，将该文件放在可访问的地方。

3.将一张空白的 8GB Class 10 SD 卡插入计算机。

4.打开命令提示符，键入:
diskpart
list disk
exit

这将允许您找到您的 SD 卡的驱动器号。在下一步中使用它。

5.最后，按照微软自己的说明在 Windows 10 中刷新 SD 卡。

使用管理员命令提示符，通过运行以下命令将映像应用到您的 SD 卡(确保用您在上一步中找到的值替换 PhysicalDriveN，例如，如果您的 SD 卡是 3 号磁盘，请使用/ApplyDrive:\\。\PhysicalDrive3 below):

dism.exe /Apply-Image /ImageFile:flash.ffu /ApplyDrive:\\.\PhysicalDriveN /SkipPlatformCheck

6.安全取出 SD 卡。

该卡现在可以在您的 Raspberry Pi 2 中启动了。

你为什么想这么做？正如微软指出的那样，“Windows 10 物联网核心是面向低成本、小尺寸设备的新 Windows 10 版本，将对制造商和商业设备制造商‘免费’提供。”这意味着您可以在自己的项目中轻松刷新和使用 Windows 的精简版本。有趣的是，这个版本的 Windows 对于类似 Arduino 和 Rasbian 平台的用户来说会非常熟悉。

不要期望很快就能在这个东西上玩 Far Cry 把它想象成一个通用的应用平台，连接到简单的设备，如基于 Arduino 的继电器和 led。事实上，用户界面非常有限，除非你自己编写一些应用程序。

 

然而，微软将它推向市场是一个令人惊讶的友好举动，我认为即使是最顽固的开源粉丝也会同意，让更多人在嵌入式项目中使用 Raspberry Pi 是一个崇高的目标——无论是什么操作系统。
