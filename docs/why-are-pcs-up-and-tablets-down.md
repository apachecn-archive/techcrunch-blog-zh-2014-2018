iPads 刚问世时，被誉为个人电脑的末日。最终，普通用户可以用一种便宜可靠的计算设备来代替复杂古怪的个人电脑。经过几年 iOS 和 Android 平板电脑的强劲增长以及 PC 销量的相应下降，情况突然相反了:PC 销量上升，而[平板电脑销量“崩溃”。](https://web.archive.org/web/20230226101949/http://recode.net/2014/07/30/exclusive-interview-best-buy-ceo-says-tablet-sales-are-crashing-sees-hope-for-pcs/)发生了什么？

鉴于平板电脑除了在尺寸、屏幕分辨率和处理器速度上有相对表面的变化外，几乎没有什么改进，平板电脑的放缓不应令人惊讶。平板电脑的最初市场现在已经饱和:祖父母和孩子都有，人们购买它们作为 Sonos 控制器之类的东西，许多家庭都有它们用来阅读。想要平板电脑的人已经有了，不需要升级，因为它们已经足够完成分配给他们的任务了。

企业和消费者都在再次购买个人电脑，Mac 的销量也在逐年上升。由于 Windows XP 不再受支持，企业尤其被迫升级旧电脑。购买新 PC 时，选择 PC 还是平板电脑的主要驱动因素是相当明显的:如果您经常创建任何类型的内容，您需要一个键盘、一个更大的屏幕和(对于大多数企业)Microsoft Office。

## **用“超级平板电脑”重新点燃平板电脑的增长**

为了让平板电脑类别继续增长，平板电脑需要超越 Chris Dixon 所说的“[玩具阶段](https://web.archive.org/web/20230226101949/http://cdixon.org/2010/01/03/the-next-big-thing-will-start-out-looking-like-a-toy/)”，变得更像个人电脑。平板电脑演变为超级平板电脑所需的功能直接来自 PC playbook:至少 13 英寸屏幕、64 位处理器、2GB 内存、256GB 驱动器、真正的键盘、真正的文件系统以及具有窗口和真正多任务处理能力的改进操作系统。超级平板电脑的外形可以从笔记本电脑到一体化台式机，如 iMac。7 英寸和 9 英寸的小型超级平板电脑可以接入更大的屏幕和键盘。

从 Sun 公司的 Sun Ray 到甲骨文公司的网络计算机，再到微软的 Windows CE，计算机行业充斥着简化个人电脑的失败尝试。但这一次，其实不一样。大规模生产的 64 位 ARM 芯片的力量，智能手机和平板电脑生产的规模经济，以及最重要的是，iOS 和 Android 应用程序的庞大生态系统，终于使这样的“网络计算机”成为可能。

## **企业需要超级平板电脑**

作为 CBS Interactive 的前首席信息官，我会为我们的员工成批购买这种超级平板电脑，他们中的绝大多数人主要只使用网络浏览器和微软 Office。当然，总会有一些高级用户，如开发人员和视频编辑人员，需要一台功能齐全的电脑。一款升级版的平板电脑确实会增加企业销量，就像蒂姆·库克希望 iPad 增加销量一样……但代价是牺牲了 MacBooks。

在一个企业中管理个人电脑的成本是巨大的，Gartner 估计一台笔记本电脑的总拥有成本可能高达 9000 美元(T4)。个人电脑价格昂贵，容易出故障，容易损坏，容易感染病毒和恶意软件。仅仅使用了一会儿，许多电脑就容易经常死机和崩溃。

个人电脑如此容易出故障，以至于 service now——一家致力于帮助 IT 组织跟踪服务台票证的公司——价值超过 80 亿美元。一些组织受够了有问题的电脑，他们正在使用昂贵而繁琐的桌面虚拟化，在这种情况下，电脑环境在服务器上受到严格控制，并传输到客户端。

虽然 MAC 电脑比 Windows 要好一些，但我建议你站在任何公司服务台或苹果天才吧旁边，观察和学习，如果你认为它们没有问题的话。

超级平板电脑给企业带来的主要好处是系统管理和可替换性。智能手机和平板电脑如此简单且易于管理，以至于它们通常由 IT 组织的经济高效的电话团队来处理，而不是更昂贵的 PC 技术人员，这些技术人员通常被小问题淹没，无法专注于解决更复杂的问题。IT 和最终用户都可以调配和更新应用，而不会引起冲突或问题。如果设备丢失，很容易远程擦除数据，并为新设备提供所有相同的设置。

像 BYOD(自带设备)这样的项目只是强调了这样一个事实，即智能手机和平板电脑非常容易管理，企业可以放心地让员工自己选择设备。用户也获得了巨大的好处，包括即时启动、长电池寿命、简单性，以及从 iTunes 和 Play 应用商店访问大量应用程序。

## **为什么三巨头不能推出超级平板电脑？**

苹果前高管 Jean-Louis Gassée 早就指出，苹果正在逐渐[融合 Mac OS X 和 iOS](https://web.archive.org/web/20230226101949/http://www.mondaynote.com/2012/02/19/apple%E2%80%99s-grand-user-experience-unification/) ，并且很可能用 ARM 处理器取代[英特尔处理器。然而，苹果坚定地保持平板电脑和个人电脑之间的分离，并通过其新的连续性功能弥合这一鸿沟。虽然微软愿意在桌面体验中加入触摸界面，但可以理解的是，在体验完美之前，苹果不会这么做。](https://web.archive.org/web/20230226101949/http://www.mondaynote.com/2014/08/03/macintel-the-end-is-nigh/)

然而，苹果将不得不在不久的某个时候进行这种转换，因为用户越来越期望每个屏幕都可以触摸。蒂姆·库克声称他不害怕蚕食业务，但苹果似乎不愿蚕食其不断增长的 200 亿美元的 Mac 业务。

谷歌的 Chromebook 本质上是一台只能运行网络应用的 PC。正如许多评论家困惑的那样，谷歌应该专注于桌面版本的 Android，而不是 Chrome OS。市场已经决定，它需要智能手机和平板电脑上的原生应用程序，因此很明显，用户也会希望他们的 PC 替代品上有原生应用程序。

Android 凭借其庞大的应用商店和开发者社区拥有巨大的优势。Chrome 操作系统有一个固有的缺陷——为什么要和 Windows 竞争，而微软自己正在超越 Windows？基于 ARM 芯片的新一代 Chrome book 非常符合超级平板电脑的规格——因为 Chrome OS 的限制，没有任何应用程序。硬件没错，操作系统不对。

微软凭借其适用于 iPad 和 Android 的 Office 365，实际上在超级平板电脑世界中处于非常有利的位置，因为作为一种订阅产品，它可以在制造商兑现硬件本身的微薄利润后很长时间内获得收入。这是微软在 Mac 上比苹果赚更多钱的机会，就像微软在 20 世纪 90 年代做的那样。微软已经放弃了在低端硬件上靠 Windows 赚钱的 T2，并在塞特亚·纳德拉的领导下，围绕设备和服务，开始了后 Windows 时代的 T4。

微软的 Surface Pro 3 在某种程度上是一次勇敢的尝试，将个人电脑重塑为超级平板电脑；然而，它很贵，屏幕小，键盘不够标准，功耗很高的英特尔处理器，以及管理 Windows 的所有令人头疼的问题。具有讽刺意味的是，备受争议的搭载 Windows RT 的 Surface 2 是朝着正确方向迈出的一步，但其 32 位 ARM 处理器动力不足，Windows 商店中的应用也不多。微软巧合地在 20 世纪 90 年代末提供了 Windows CE 设备，实际上非常接近超级平板电脑，但就像 Windows touch 平板电脑一样，它们进入市场太早了。

围绕制造、分销和维护个人电脑的生态系统是巨大的，苹果和个人电脑公司不愿意瓜分他们的销售额是可以理解的。联想提供了一款 [10 英寸的安卓笔记本](https://web.archive.org/web/20230226101949/http://www.zdnet.com/lenovo-confirms-ideapad-a10-android-laptop-after-leaking-manual-online-7000021872/)，据报道，惠普[也将很快推出一款](https://web.archive.org/web/20230226101949/http://www.zdnet.com/hp-slatebook-14-android-touchscreen-laptop-revealed-in-hp-video-7000028833/)，但为了不与笔记本竞争，这些笔记本故意做得很小且动力不足。这种真空为索尼这样已经退出个人电脑业务但继续销售智能手机和平板电脑的公司提供了机会。

据报道，特别是三星[正在寻求关闭其个人电脑业务](https://web.archive.org/web/20230226101949/http://thenextweb.com/asia/2013/06/25/samsung-reportedly-shutting-down-desktop-pc-business-amid-declining-demand/)，并且在智能手机销售放缓的情况下，必须评估如何发展其平板电脑业务。在与微软的下一场专利冲突中，三星可能会提供 Office 365 捆绑销售，以换取免版税的设备销售。

## **企业遗留网站滞留**

一个有趣的附带说明是，大型企业通常运行许多不能在现代 web 浏览器上工作的遗留 web 应用程序，一些遗留 web 应用程序只能在像 Internet Explorer 6 这样的老式浏览器上工作。这些应用程序中有许多是在互联网的第一波浪潮中构建的，以实现“员工自助服务”，自那个时代以来一直没有人使用过。也许转向更简单、更便宜的 PC 替代品将最终改变成本/收益等式，这样这些 web 应用程序将最终升级或被 SaaS 解决方案取代。

我们希望苹果、谷歌和微软能够很快进入超级平板电脑的未来，在这个未来，大多数企业和消费者将能够像管理他们的手机和平板电脑一样轻松地管理和定制他们的个人电脑……而我们这些技术人员可以离开我们的兼职技术支持工作。
