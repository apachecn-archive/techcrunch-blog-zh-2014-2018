# Target 在 50 家商店开展 Beacon 测试，今年晚些时候将扩展到全国 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/08/05/target-launches-beacon-test-in-50-stores-with-expanded-rollout-later-this-year/>

美国第二大百货零售商塔吉特百货(Target)今天上午宣布，将在全国 50 家门店开始测试 beacon 技术。该公司表示，借助 beacon 技术，它将能够直接向消费者的智能手机发送交易信息和建议，前提是他们选择接收这些提醒。

最初，信标技术将与那些在手机上安装了最新版本的目标 iPhone 应用程序并打开了蓝牙的用户一起工作。

该应用程序将提示顾客“选择加入”，在商店里与零售商分享他们的位置，并允许它向他们的手机发送推送通知。该公司表示，它将限制这些通知，以免让客户不知所措。每次购物行程只会发送两次推送通知。

与此同时，这些推荐可能会以推送提醒和应用内更新的形式出现在目标应用的“Target Run”页面上，该页面的设计类似于一个社交媒体新闻源，提供交易、产品精选、Pinterest 上的热门商品等。

Target 最初在芝加哥、丹佛、明尼阿波利斯、纽约市、匹兹堡、波特兰、旧金山和西雅图的 50 家商店测试这项技术。塔吉特百货告诉我们，选择这些商店是因为它们代表了不同市场的商店。塔吉特表示，它将根据消费者反馈和其他数据，继续调整和改进服务，并在今年晚些时候“广泛”推广到其他商店。

我们听说这将在圣诞节前在全国推广。

Android 支持也在进行中。

零售行业的许多企业已经开始测试 beacon 技术，该技术使用蓝牙与购物者的手机在他们的位置附近进行连接。例如，房地产开发商 Simon [去年秋天表示，将为近 240 个商场和购物中心安装信标。梅西百货公司在九月](https://web.archive.org/web/20221022091516/http://www.rfidjournal.com/articles/view?12144)[也推出了](https://web.archive.org/web/20221022091516/http://www.zdnet.com/article/macys-rolls-out-retails-largest-beacon-installation/)一个涉及 4000 台设备的大型信标装置。 [GameStop 表示，它将在 Q2 的 36 家商店试行](https://web.archive.org/web/20221022091516/https://www.internetretailer.com/2015/01/14/gamestop-will-pilot-geo-fencing-and-beacons-36-stores-q2)地理围栏和信标，Lord & Taylor stores [去年推出了信标](https://web.archive.org/web/20221022091516/https://www.umbel.com/blog/mobile/15-companies-using-beacon-technology/)。

信标让零售商有机会进行有针对性的营销，唤醒安装在消费者手机上的应用程序，以便向他们发送个性化信息。到目前为止，消费者已经在一定程度上接受了这项技术，可能是因为他们对它有更好的控制——他们可以选择是否加入，或者干脆关闭蓝牙来禁用警报。

根据营销平台提供商 Swirl 2014 年 11 月的一项研究[, 73%的消费者甚至表示，信标活动增加了他们在访问期间购买的可能性。](https://web.archive.org/web/20221022091516/https://www.swirl.com/swirl-releases-results-retail-store-beacon-marketing-campaigns/)

在 Target 的案例中，该公司表示，该技术不仅将用于触发交易警报，如来自 Cartwheel(其移动优惠券服务)的新报价，还将向购物者提供信息和建议，例如关于 Pinterest 上附近服装商品趋势的警报。

在未来的版本中，Target 将引入其他 beacon-powered 功能，例如当你在商店中移动时，可以重新排序你的购物清单——类似于当你选择不同的方向时，地图应用程序如何为你重新选择路线。另一个即将推出的功能将允许购物者从他们的手机上请求目标员工的帮助。

Target 表示，它确实会从移动应用程序中收集数据，以了解购物趋势和偏好，但信标数据是针对店内位置的，将用于增强购物体验。

虽然目前市场上有几家信标硬件供应商，包括 Estimote、BlueCats、Bluesense、高通的 Gimbal、Gelo 等，但 TechCrunch 知道 Target 正在与 Estimote 合作安装。Target 不愿证实这一点，称正在与“少数”供应商合作。从技术上来说，这是真的，因为完整的展示包括其他内容，如用于映射的 Point Inside，但 Estimote 是硬件信标提供者。

[周三晚些时候，Estimote 也发表了自己的博客文章](https://web.archive.org/web/20221022091516/http://blog.estimote.com/post/125942350690/apple-google-and-the-state-of-the-enterprise)，描述信标市场的状况。虽然它没有把自己定位为 Target 的硬件提供商(它隶属于 NDA)，但它确实提到了 Target 的新推出，作为 beacon 堆栈现在如何为企业做好准备的一个例子。
