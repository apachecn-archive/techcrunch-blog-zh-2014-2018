# Skype For Web Beta 现已向所有美国和英国用户开放

[公司今天宣布](https://web.archive.org/web/20221006225100/http://blogs.skype.com/2015/06/05/skype-for-web-beta-is-now-available-to-everyone-in-the-us-and-uk/), Skype 基于网络的客户端，以及即时消息和视频/音频通话功能，现已对所有用户开放测试。这款基于网络的流行聊天和通话工具于去年年底问世，基于邀请系统的[访问权限，但现在每个人都可以加入，只要你居住在美国或英国](https://web.archive.org/web/20221006225100/https://beta.techcrunch.com/2014/11/14/skype-web-beta/)

Skype web beta 可用于 Internet Explorer、Chrome、Safari 和 Firefox，但不能在 Chromebooks 上运行。语音和视频通话仅在列出的浏览器上可用，包括 Windows 和 OS X，并且需要安装浏览器插件才能运行这些功能。

与本地客户端相比，网上 Skype 有一些优势:例如，当你不在家，使用一台不熟悉的电脑时。你也可以不安装原生应用程序，但不管你的动机如何，拥有一个功能齐全的网络版 Skype 绝对不是一件坏事。

缺乏对 Chrome OS 的支持有点令人失望，因为这是将这些设备用作功能更全的电脑替代品的主要障碍之一。

只需前往 Skype.com开始使用基于网络的客户端。
