# YouTube 升级评论系统，给予创作者更多控制权 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/11/03/youtube-upgrades-its-comments-system-to-give-creators-more-control/>

YouTube 今天推出了对其评论系统的升级，目标是让创作者更好地控制哪些评论会出现在提要中，以及更好地与观众和粉丝互动的能力。除了能够将评论钉在提要顶部并保留不适当的评论以供审查之外，创作者还将在回复网站上的评论时突出显示他们自己的用户名，他们将能够通过“关注”(喜欢)他们希望脱颖而出的评论来显示他们的粉丝的关注。

YouTube 表示，新功能的想法是让创作者更好地通过其服务与社区联系，同时让与粉丝的互动更加个性化。

这些变化发生在 YouTube 最近试图将其不断增长的视频平台转变为更类似于社交网络的东西之后。9 月，[该公司推出了 YouTube 社区，](https://web.archive.org/web/20221208005352/https://beta.techcrunch.com/2016/09/13/youtube-gets-its-own-social-network-with-the-launch-of-youtube-community/)这是创作者频道上的一个专用标签，他们可以在这里发布新闻和其他基于文本的内容，以及图像和 gif，以吸引粉丝。然而，视频评论仍然是大量粉丝与创作者互动的地方。

根据 YouTube 的声明，创作者现在可以使用新的锁定功能将任何评论提升到提要的顶部。YouTube 告诉我们，他们一次只能发布一条评论。

这只是 YouTube 解决其评论系统问题的一种方式，该系统多年来名声不佳。同样在 9 月份，[公司推出了一个新项目](https://web.archive.org/web/20221208005352/https://beta.techcrunch.com/2016/09/21/youtube-enlists-volunteers-to-moderate-its-site-via-a-new-youtube-heroes-program/)，它将允许可信的版主帮助审查其网站上的评论，通过标记不适当的评论和执行其他任务来赚取积分和提前获得新功能。

但在此之前， [YouTube 还赋予创作者指定特定人员在他们自己的频道上调节评论](https://web.archive.org/web/20221208005352/https://youtube-creators.googleblog.com/2016/06/youtube-vidcon-new-support-new-features.html)的直接能力，包括从创作者的视频中删除公共评论的能力。

此外，自 2013 年以来[有一个黑名单功能](https://web.archive.org/web/20221208005352/https://support.google.com/youtube/answer/6109622?hl=en)，允许视频创作者保留包含特定单词和短语的评论，以便在上线前进行审查。

![screen-shot-2016-11-03-at-10-23-51-am](img/7a1faf9cb1cda472ed00b2418803bdf4.png)

今天，YouTube 也在测试版中引入了一个新的评论评论功能。[创作者可以选择加入测试这个新系统](https://web.archive.org/web/20221208005352/https://docs.google.com/forms/d/e/1FAIpQLSdQGVVERLoawGUcAi_N_X09s9bPBhZmBvs-YcDthZxWJsSydg/viewform)，这将涉及一个新的算法，该算法将自动检测潜在的不当评论，并阻止它们进行最终审查。然后，创建者可以阅读这些标记的评论，并根据需要选择批准、隐藏甚至报告它们。

该公司警告说，仍在开发中的系统可能会出现误报，也可能会错过本应捕捉到的评论。然而，随着时间的推移，算法会变得更好，因为它会从所采取的行动中学习。

**红心&创建者用户名**

今天推出的另外两个新功能——hearts 和 creator 用户名——旨在帮助 YouTube 的视频明星更好地在评论区与他们的社区建立联系。红心是一种简单的，普遍理解的，在社交网络上表达欣赏的方式，而用户名功能只是突出了创建者的用户名，所以它很突出。

用户名将是彩色的，这样粉丝就可以很容易地看到创建者何时参与到一个线程中。该公司正在努力使颜色可定制，以便他们可以匹配频道的颜色，但这还没有上线。与此同时，经过验证的创建者的名字旁边也会有一个勾号，类似于其他社交网络上经过验证的账户，如 Twitter 和脸书。

考虑到这些功能实际上是当今网络上任何社交网络或社区网站的基础，YouTube 花了这么长时间才推出这些功能有点令人惊讶。但社交并不是谷歌擅长的领域。

过去，该公司希望通过[将 YouTube 连接到其更大的社交网站 Google+，](https://web.archive.org/web/20221208005352/https://beta.techcrunch.com/2013/09/24/youtube-announces-a-new-google-powered-commenting-system/)可以鼓励评论区更负责任的行为，因为[用户将不得不使用他们的真实身份](https://web.archive.org/web/20221208005352/https://beta.techcrunch.com/2014/07/15/3-years-later-google-drops-its-dumb-real-name-rule-and-apologizes/)发帖。

但是这个系统遭到了强烈反对，不仅仅是因为强制迁移到 Google+,还因为它不承认 YouTube 是一个用户经常希望拥有独立身份的地方，与他们更公开的谷歌账户和网络存在无关。

今天的新功能远没有 Google+的举动那么惊天动地，而是代表了一种更现实的评论管理观点——就像网络上的其他地方一样，自动化系统结合人工审核似乎效果最好。
