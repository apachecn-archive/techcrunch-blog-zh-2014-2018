# 原型制造商 Tempo Automation 获得 800 万美元用于升级工厂

毫无疑问，我们正处于硬件初创公司的复兴时期。众筹、低成本组件、快速原型制作、可扩展制造——这些明星完美结合，为创新硬件解决方案创造了黄金时代——但障碍依然存在。

[Tempo Automation](https://web.archive.org/web/20221226005101/http://www.tempoautomation.com/) 是一家总部位于旧金山的初创公司，围绕消除最后几个障碍这一令人钦佩的目标而建。该公司的使命宣言说得最好，“我们正在创造一个电子开发可以和软件开发一样快的世界。”

该公司网站上的其他大部分语言也同样冠冕堂皇，但 Tempo Automation 实际上做的相对简单，为小批量原型制造提供快速周转，将设计文件转化为物理板，周转时间为三天。

“目前用于电子制造行业的工艺是为大批量生产而优化的，”联合创始人沙希德·萨马拉告诉 TechCrunch。“如果你正在生产一百万部苹果手机，那就太好了。当您构建几页时，就像使用印刷机打印几页一样。我们正在从头开始构建一个自动化流程，该流程针对低容量进行了优化。”

该公司[刚刚获得了由莱克丝资本(Lux Capital)牵头的 800 万美元的 A 系列融资回合](https://web.archive.org/web/20221226005101/https://medium.com/tempo-automation/tempo-automation-announces-8m-series-a-funding-to-build-software-enabled-electronics-factory-2643c1bc53dc#.73wfzm8wm)，以及软银、AME 和博尔特，以帮助改善其目前已经产能的制造设施。根据今天发布的新闻稿，“我们的产品很受欢迎，在几个月的时间里，从早期的硬件初创公司到你听说过的大型消费电子公司，我们的工厂产能都被客户耗尽了。”

该公司的主要重点是初创企业，特别是新的物联网制造商，尽管一些大公司显然已经加入进来，尽管 Tempo 不能说是哪家。
