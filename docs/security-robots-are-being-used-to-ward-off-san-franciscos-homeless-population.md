# 安全机器人被用来保护旧金山无家可归的人

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/12/13/security-robots-are-being-used-to-ward-off-san-franciscos-homeless-population/>

如果用机器人代替人来阻止无家可归者在商业场所外扎营，情况会更糟吗？

一个这样的机器人警察最近接管了旧金山 SPCA 的外面，这是一个位于该市任务区的动物保护和宠物收养诊所，以阻止无家可归者在那里闲逛——这导致一些人非常不安。

硅谷游戏开发商和国会候选人陈伶俐·吴昨天在推特上表达了她对这一举动的失望，她说:“我很抱歉这么坦率，但这绝对让我感到厌恶，因为我是一个经历过无家可归的人。”

旧金山的无家可归问题既棘手又复杂。看到财富和特权的过剩与市场街上 Twitter 总部外的悲惨境遇并存，人们可能会感到震惊。

然而，这座城市的无家可归者也与更高的犯罪率、暴力率和有时的精神病发作率联系在一起，导致许多人认为旧金山没有妥善处理的安全问题。

一个月前，S.F. SPCA 推出了安全初创公司 Knightscope 名为 K9 的机器人装置的使用，理由是同样的安全问题。

“整个夏天，我们的避难所被闯入过两次。里面被破坏，财产和现金捐赠被盗，”SPCA 基金会发言人 Krista Maloney 告诉 TechCrunch。“此外，许多工作人员和志愿者投诉天黑后下班时在我们的停车场遇到汽车损坏和骚扰。我们目前雇佣了保安，但我们的校园很大，他们一次只能在一个区域。”

K9 单位也比人类便宜。一个机器人的使用成本是 6 美元一小时，相比之下，支付一名保安的平均工资[是 16 美元一小时](https://web.archive.org/web/20230325191645/https://www.indeed.com/salaries/Security-Guard-Salaries,-San-Francisco-Bay-Area-CA)。

“不幸的是，在过去的一年里，我们不得不花费大量的资金来确保我们校园里的人们以及我们所照顾的动物的安全，”马洛尼说。

而且，根据 SPCA 和骑士镜的说法，部署机器人后犯罪率下降了。

然而，K9 部队也有自己的困难。SPCA 告诉 TechCrunch，它被打翻了，有人在 3 月初在上面涂了烧烤酱。另一位推特用户[提到](https://web.archive.org/web/20230325191645/https://twitter.com/TysonKallberg/status/928675069533945856)看到便便被涂在车上，尽管 SPCA 发言人无法证实此事。

值得一提的是，过去许多机器人都曾被人类当作目标。今年早些时候，SPCA 雇佣的一个相同型号的 K9 机器人被一个醉汉残忍地袭击，撞倒了在硅谷停车场巡逻的 400 磅重的机器人。

SPCA 雇佣 bot 警察面临的另一个问题是公共人行道的使用。K9 小组正在商店周围的几个区域巡逻，包括人类行走的人行道，这引起了行人和倡导组织 Walk SF 的愤怒，该组织此前提出了一项[法案](https://web.archive.org/web/20230325191645/http://www.slate.com/blogs/future_tense/2017/05/17/san_francisco_supervisor_moves_to_ban_sidewalk_delivery_robots.html)在全市禁止送餐机器人。

“我们在人行道上看到更多类型的机器人，希望看到城市走在这前面，”步行旧金山政策和项目主任凯西·迪鲁卡说。

上周,[市下令](https://web.archive.org/web/20230325191645/https://www.bizjournals.com/sanfrancisco/news/2017/12/08/security-robot-homeless-spca-mission-san-francisco.html)SPCA 皇家空军完全停止使用这些安保机器人，否则将面临每天 1000 美元的罚款，原因是未经许可在公共道路上操作。

S.F. SPCA 表示，他们已经移走了机器人，并正在通过许可程序。自从机器人被移走后，它已经经历了“两次破坏行为”。

但是，撇开许可证和公共使用人行道不谈，机器人似乎不仅仅可以阻止无家可归者的营地。是的，它可以监视周围地区并报告犯罪，但它也可以用来提醒警察和社会工作者无家可归者似乎有所增加的地区，或者寻找任何可能面临暴力或精神病发作并需要干预的人。

Knightscope 机器人配备了四个摄像头，能够每分钟读取 300 多个车牌。他们可以四处走动并监视某个区域，注意名单上不应该在那里的人。

SPCA 旧金山警察局已经表示，使用机器人警察后，犯罪率有所下降。同样的话，如果它增加了人类保安的使用，但如上所述，人力成本更高。他们也无法全天候监控或立即将所见上传至云端。

此外，机器人不会消失。虽然还不清楚旧金山市议会将拿出什么解决方案来处理未来人行道上这种类型的机器人的增加，但我们不可避免地会看到更多这种机器人。

这是一个古老的人类对机器的争论。但是机器通常会赢。
