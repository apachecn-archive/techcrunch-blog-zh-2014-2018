# Twitter 与聊天应用 Lookup 合作，让印度用户可以用 140 个字符购物

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/12/23/twitter-lookup-india/>

Twitter】宣布与 Lookup 合作，Lookup 是一款连接消费者和零售商的聊天应用，它将允许用户使用 140 个字符进行购物和查找信息。

Lookup 最近[从包括 Twitter 联合创始人比兹·斯通在内的一系列投资者那里筹集了 250 万美元](https://web.archive.org/web/20230322162948/https://techcrunch.com/2015/10/14/lookup-a-chat-app-that-connects-retailers-and-customers-in-india-raises-2-5m)，它使用聊天来消除本地购物的网络浏览、多个应用程序等的痛苦，并且——重要的是——以最小的难度将线下零售商带到线上。商店只需获得一个帐户，就可以开始通过“查找聊天”接收客户的订单和查询。

这种搭配不像去年宣布的 [Twitter 的购买按钮](https://web.archive.org/web/20230322162948/https://techcrunch.com/2014/09/08/twitter-commerce-buy-now/)那样复杂，它直接通过 Stripe 处理交易。相反，Twitter 将充当通信管道，就像 Lookup 应用程序本身一样。订单和支付是分开和离线完成的。目前，Lookup 处理连接商家和客户的重要步骤——这要感谢 Twitter APIs 及其对合作伙伴的监控选项。

在这方面，这种合作关系将 Twitter 变成了一种个人助理服务，很像正在美国试用的 Facebook Messenger M..事实上，Lookup 创始人兼首席执行官 Deepak Ravindran 告诉 TechCrunch，Twitter 对这种合作关系感兴趣，因为它作为礼宾服务的吸引力，这可能会提高它在巨大的印度市场的受欢迎程度。(据估计，印度只有 17%的社交网络用户使用 Twitter，不到总人口的 2%。)

Lookup 处理各种各样的查询——比如检查商店的可用库存或营业时间——但 Ravindran 告诉我们，一周后，Twitter 合作伙伴关系到目前为止很受欢迎，可以从当地商店购买商品。

“基本上我们就像一个‘我现在就想要’的按钮，”他解释道。有高端产品的订单，如 iPads 和 Fibits，但“大多数询盘都是 5-10 美元以下的低价订单。"

Lookup 积极推动商业角度，为圣诞节前的所有订单提供免费送货服务。因此，如果有任何最后一分钟的礼物要求，这可能是值得一试。

最终，Lookup 应用程序可能会继续成为这项服务的主要门户，但加入 Twitter 有助于推广这项服务，并提供替代选择。不过，圣诞节过后，兴趣可能会下降。Ravindran 补充说，该公司正在调查其他渠道合作伙伴关系。

“未来机器会比人类更聪明，但这需要时间。我们想在印度获胜，这是我们的机会。我们希望接触每一个渠道来增加用户，并最终成为线下的谷歌，”他说。

就像它与 Zipdial 的合作关系一样，[它最终收购了未接来电服务](https://web.archive.org/web/20230322162948/https://techcrunch.com/2015/01/19/twitter-confirms-acquistion-of-indias-missed-call-marketing-platform-zipdial/)，与 Lookup 的这种合作关系很有趣，因为它将其服务超越了通信协议，几乎纳入了一个可以部署或集成服务的平台。这种合作或类似的合作似乎不太可能出现在西方市场。但这表明 Twitter 正在加大对印度的关注，在印度十几亿人口中，脸书和脸书旗下的 WhatsApp 是主要的社交和移动用户。
