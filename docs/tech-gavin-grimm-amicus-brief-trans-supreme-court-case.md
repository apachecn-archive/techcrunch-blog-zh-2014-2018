# 科技公司在即将到来的斯科特案件中提交了一份支持跨性别青年的简报

当最高法院下个月接手这个问题时，科技公司正准备为变性人的权利而战。正如 Axios 报道的那样，以苹果为首的一些知名公司将提交一份法庭辩护状，支持 17 岁的主要原告，跨性别的弗吉尼亚高中生[加文·格林](https://web.archive.org/web/20230129094029/https://www.nytimes.com/2017/02/23/us/gavin-grimm-transgender-rights-bathroom.html)。

据报道，其他将签约的公司包括微软、Salesforce、IBM、Tumblr、Slack、Yelp、Box、Affirm、Ebay、GitHub 和 Box。预计在案件开始前会有更多人加入辩护状。新闻发布的前一天，一波科技领袖[发表公开声明](https://web.archive.org/web/20230129094029/https://techcrunch.com/2017/02/23/tech-companies-trump-transgender-school-guidelines/)支持保护跨性别学生。
