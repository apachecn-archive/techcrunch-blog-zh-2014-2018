# 虚空给迪士尼度假区带来了基于位置的星球大战虚拟现实体验

基于位置的虚拟现实初创公司 [The Void](https://web.archive.org/web/20230130100805/https://www.thevoid.com/) 今天宣布，它将与卢卡斯影业和 ILMxLAB 合作，为迪士尼度假区带来星球大战品牌的体验。新体验“星球大战:帝国的秘密”将于今年假期晚些时候在迪斯尼乐园度假区的迪斯尼市区和华特·迪士尼世界度假区的迪士尼之泉上映。

这家总部位于犹他州的初创公司最出名的是其《捉鬼敢死队》(ghost busters)VR 体验，该体验现已在四个地方推出，包括犹他州、纽约、迪拜和加拿大。

不要与常规的、普通的、令人敬畏的虚拟现实混淆，虚空将其新的星球大战体验称为“超现实”除了上面的预告片，我们还没有得到很多关于这些虚拟现实体验的信息，尽管看起来肯定会有一些枪战。

虚空所做的是非常独特的；该公司利用自己的硬件以及物理空间和 VR 软件的深度集成，使用户可以伸出手触摸环境游戏对象，如墙壁或栏杆，并实际感受它们。这有点像你用消费级 VR 头戴设备所能做的事情和那些 4D 电影院的混合，在那里你有时可以闻到屏幕上发生的事情或感觉到温度的变化。

与迪士尼和虚空的战略伙伴关系并不令人惊讶。上个月，这家初创公司宣布，它在当前一批 2017 年迪士尼加速器中。

对于基于位置的虚拟现实初创公司来说，在迪士尼乐园获得体验肯定是他们的圣杯。虚空一直致力于以一种不依赖于拱廊的复兴或说服电影院转换自由空间的方式来商品化 VR 体验。
