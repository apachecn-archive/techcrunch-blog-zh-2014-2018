# 奥米德·柯德斯塔尼离开谷歌，加入 Twitter，担任执行董事长 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2015/10/14/twitter-names-googles-omid-kordestani-as-its-new-executive-chairman/>

然而，随着社交网络继续其巨大的转变努力，Twitter 正在发生更多的变化。在社交网络宣布对其[新任首席执行官杰克·多西](https://web.archive.org/web/20230130004110/https://techcrunch.com/2015/10/05/twitter-names-jack-dorsey-ceo-adam-bain-becomes-coo-dick-costolo-steps-down-from-board/)领导下的[大规模裁员](https://web.archive.org/web/20230130004110/https://techcrunch.com/2015/10/13/twitter-is-laying-off-8-percent-of-the-company/)一天后，它现在任命了一位新的执行董事长:奥米德·柯德斯塔尼，从谷歌挖走了他，他曾是该公司的[首席商务官](https://web.archive.org/web/20230130004110/https://en.wikipedia.org/wiki/Omid_Kordestani)。

Kordestani 是谷歌的老员工了，他在谷歌担任首席商务官刚刚一年多，在尼科什·阿罗拉离职去软银后接替他。然而，最近，他似乎从公司的核心位置退了一步。当谷歌在今年夏天将 Alphabet 分拆时，他的首席商务官职位被取消，他悄悄地成为了 Alphabet 和谷歌的顾问，但没有具体的头衔。

这意味着谷歌不太可能宣布 Kordestani 的替代者。谷歌拒绝就是否会以任何方式取代 Kordestani 置评。

Kordestani 被任命为 Twitter 的董事长具有象征意义。Kordestani 自 1999 年以来一直在谷歌工作——显然是第 11 号员工——他被认为是该公司第一批创收活动的掌舵人，也是让该公司首次实现盈利的人。

这里有一个有趣的转折:随着过去几个月关于谁将成为 Twitter 掌门人的猜测越来越多，我们听说 Twitter 已经将名单缩减到两个人:多尔西和谷歌的某个人。一个非常有趣的巧合是，长期在谷歌工作的 Kordestani 今天将接任董事长一职。

不出所料，这个消息是杰克在推特上爆料的:

截至发稿时，Kordestani 自 2012 年开设账户以来共发了 11 条推文，[注明](https://web.archive.org/web/20230130004110/https://twitter.com/omidkordestani/status/654281177835016192):“很高兴能与 [~~@~~ **杰克**](https://web.archive.org/web/20230130004110/https://twitter.com/jack) 、Twitter 板、 [~~@~~ **亚当班**](https://web.archive.org/web/20230130004110/https://twitter.com/adambain) 、 [~~@~~ **安东尼诺托**](https://web.archive.org/web/20230130004110/https://twitter.com/anthonynoto) 和团队的其他成员一起成长【18】"

作为一个在搜索巨头中既担任执行销售又担任首席执行官顾问的人，Kordestani 将在公司的关键时刻为 Twitter 带来一些强大的商业和业务印章。

谷歌在关于 Omid 离开公司的官方声明中也强调了这一想法。

“我们为 Omid 感到兴奋，也为 Twitter 感到高兴。Omid 在建立谷歌业务方面发挥了重要作用，并帮助我们在整个行业发展了真正密切、长期的合作关系，”一位发言人说。"我们祝愿他在未来的岁月里一切顺利。"

自去年上市以来，Twitter 的股票和业务一直处于过山车状态。动荡和增长乏力促使迪克·科斯特罗辞去首席执行官一职，同时还有许多其他高管离职。一些人猜测 Kordestani 现在的前雇主可能会收购它，其他人更普遍地质疑该公司的方向。现在是 Twitter 加倍努力推动更多增长的时候了，并向市场表明它不仅仅是一个利基社交网络，而是一个大型媒体企业。

这则新闻为上周首席执行官公告中悬而未决的一个细节画上了句号:Twitter 曾表示，随着多尔西永久升任首席执行官，将会是[寻找一位新的董事长](https://web.archive.org/web/20230130004110/http://www.reuters.com/article/2015/10/05/us-twitter-ceo-dorsey-idUSKCN0RZ16520151005)，多尔西曾以非执行董事的身份担任这一职务，同时也管理 Square(他目前仍在担任这一职务)。

Twitter 的股票在昨天的新闻中有所上涨，在今天早上的交易中又有小幅上涨。
