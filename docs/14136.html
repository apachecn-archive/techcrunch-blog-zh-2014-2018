Galaxy Note 8 的笔拥有新的笔功能，包括翻译

三星的 Galaxy Note 8 配备了 S Pen，与之前的版本一样，但它比过去的版本多了一点。新的 S Pen 的亮点可能是它的实时消息功能，它可以从手写和涂鸦中创建可共享的动画，这样你就可以在任何地方发送和发布消息。

Note 8 S 笔还具有翻译功能，这意味着你可以通过悬停用笔突出显示文本，然后它会提供一组以你的主要语言翻译的文本。这是一个很酷的功能，在方便性方面超过了其他设备上的类似产品，并且由于其笔记功能，对于经常是 Note 8 粉丝的商业用户来说很好。

除了翻译，它还使用相同的机制转换货币，作为一个经常与美国货币打交道的加拿大人来说，这是另一个节省时间的好方法。这两个都是非常棒的面向旅行的特性。

另一个很酷的新 S Pen 三星改进了 100 页容量的永远在线显示屏笔记功能，让你可以在显示屏关闭更长时间的情况下书写，这对于电池保护和在做其他事情如接电话时使用这一功能非常重要。

新的 S Pen 与 Note 8 一起装运，并装入其右侧的“下巴”。这已经成为 Note 系列的一个销售功能有一段时间了，实际上，我对三星在这次更新中以智能的方式迭代其手写笔印象深刻。
