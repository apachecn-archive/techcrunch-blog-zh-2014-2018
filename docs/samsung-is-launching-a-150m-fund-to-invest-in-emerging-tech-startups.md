# 三星将推出一只 1.5 亿美元的基金，投资新兴科技初创公司

今天在 CES 2017 上，三星[宣布](https://web.archive.org/web/20230225042256/https://news.samsung.com/global/samsung-next-formerly-samsung-global-innovation-center-announces-150m-fund-to-expand-global-support-of-early-stage-startups-in-emerging-tech)成立[三星 NEXT 基金](https://web.archive.org/web/20230225042256/http://samsungnext.com/)，这是一个 1.5 亿美元的风险基金，目标是在新兴技术领域工作的早期创业公司。

具体来说，他们将寻找专注于虚拟现实、人工智能、物联网和“其他新的前沿技术”的公司。

该基金最早将进行种子前期投资，最晚将进行 B 轮投资。

到目前为止，该基金已经投资了 10 家初创公司:Converge Industries、Dashbot、Entry Point VR、Filament、Intezer、LiquidSky、Otto Radio、2Sens、SafeDK 和 Virtru。这些公司从[虚拟现实动画工作室](https://web.archive.org/web/20230225042256/http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.baobabstudios.com%2F%23_blank&esheet=51487815&newsitemid=20170104005561&lan=en-US&anchor=baobabstudios.com&index=1&md5=3c040d5b2666ee4d69cf48e2ca1fcea3)到[家用电器的水监测传感器](https://web.archive.org/web/20230225042256/http://www.lotik.io/#_blank)——这表明 NEXT 的投资范围将会非常广泛。

领导该基金的部门三星 NEXT 曾被称为三星全球创新中心。三星内部的部门负责识别有前途的新技术和初创公司，这些技术和公司可能会让三星受益。

NEXT 今天还宣布在特拉维夫设立新的办事处，他们目前在山景城、纽约、旧金山和韩国都设有办事处。
