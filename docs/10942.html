任天堂报告利润为5.69亿美元，因为神奇宝贝游戏销售有了一个良好的开端

任天堂仅从2016年风靡一时的游戏《神奇宝贝Go》、中获得一小部分收入，但正如该公司最新的财务报告所证明的，它的受欢迎程度无疑提升了它自己的神奇宝贝游戏。任天堂报告称，其收入为1743亿日元(15亿美元)，利润为647亿日元(5.69亿美元)。

这是继上一季度5710万美元亏损后的又一次亏损。收入同比下降22%，但任天堂超越了分析师的预期，将5月份的年度财务报告预期利润从500亿日元(4.4亿美元)提高到900亿日元(7.91亿美元)。

该公司专注于其3DS的神奇宝贝太阳和神奇宝贝月亮两个系列，并表示自11月发布以来，这两个系列的总销售额为1469万英镑。《超级马里奥制造》是第一版，最近一个季度超过了100万大关。任天堂表示，这反过来推动了硬件销售，尤其是海外销售——任天堂3DS销量为645万部，同比增长10%，销量跃升20%，达到4678万部。

任天堂也在12月发布了备受期待的第一款移动马里奥游戏。《马里奥快跑》(Mario Run for iOS)第一季度的下载量达到创纪录的4000万次，但任天堂没有进一步透露该游戏从那时起的表现，以及票房收入。游戏将于3月登陆安卓系统，所以也许我们会在下一季度获得更多的颜色。另一个标题，火焰徽章英雄，将会是Android-第一次当它在2月2日掉落，但是第三个标题——动物穿越——已经被延迟了。

任天堂的下一款旗舰游戏机Switch 也即将推出，这可能导致Wii U销量下降。任天堂确实表示，这款游戏机的表现“符合我们的预期”，上个季度售出了76万台。这比去年同期下降了75%，游戏销售额也下降了45%，达到1248万。任天堂还表示，来自amiibo角色销售的收入保持稳定。总体而言，由于提供的“可下载内容相对较少”,过去几年数字业务的收入没有变化。
