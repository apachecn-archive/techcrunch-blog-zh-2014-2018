# 一切的分拆技术危机

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/04/18/the-unbundling-of-everything/>

**编者按:** *[乔·麦肯](https://web.archive.org/web/20221207200700/https://twitter.com/joemccann)是 [NodeSource](https://web.archive.org/web/20221207200700/https://twitter.com/nodesource) 的联合创始人兼首席执行官。他是一名黑客、修补匠、建设者和破坏者，拥有超过 13 年的网络、移动和软件开发经验。*

我们一直生活在一个捆绑的世界里。ESPN 与 Nickelodeon 打包在一起，医疗保健与雇主捆绑在一起，学习在大学和学位项目中进行。我们在如此多的捆绑产品和服务的包围中成长，以至于很容易忽视分拆所带来的灵活性和价值。

捆绑销售的出现有几个原因。一种情况是，公司试图强迫消费者购买他们并不真正想要的东西，把它和他们喜欢的东西包装在一起——比如只包含一首好歌的专辑。但是，当单个产品或服务的交易成本(货币或其他)太高，不足以证明单独购买这些产品时，捆绑销售是有意义的。

随着基于互联网的技术降低了交易的低效率，我们有了新的机会来放弃不必要的捆绑，转而支持选择和灵活性。从娱乐到工作再到企业技术，这种颠覆性的影响无处不在。

## 无处不在的分拆

想想电视；我们现在可以[只挑选我们想要的频道](https://web.archive.org/web/20221207200700/https://beta.techcrunch.com/2015/03/10/one-small-step-for-unbundling-one-big-leap-for-hbo/)、节目，甚至单个剧集，而不是花钱购买臃肿的有线电视包。今年，我们将达到一个关键的转折点，有线电视用户的数量将被互联网用户的数量所取代，这支持了一个观点，即当有选择时，消费者更喜欢点菜。

松绑也正在改变一些非常个人化的东西:我们管理自己健康的方式。在过去，失去工作通常意味着失去健康保险，但是[医疗与脱钩的趋势已经很明显了。随着这种情况的发生，病人，而不是雇主，正在成为顾客，医疗保健正在变得像一种消费者服务——给个人更多的选择和更多的控制。](https://web.archive.org/web/20221207200700/http://avc.com/2015/01/what-is-going-to-happen/)

高等教育为一大笔综合费用提供了一系列不同的价值主张——学费、住房、体育等等。但是单点教育提供了一个可行的替代传统的、极其昂贵的大学经历的方法，它简化了学习，而没有完整学位课程的成本。无论是[可汗学院](https://web.archive.org/web/20221207200700/https://www.khanacademy.org/)还是[总会](https://web.archive.org/web/20221207200700/https://generalassemb.ly/)都只要求学生一次承诺一门课程。哈佛和耶鲁现在提供在线认证，这表明即使是顶尖的机构也看到了非捆绑学习的价值。

## 从单片到微服务—越小越好

分拆也在重塑科技世界，将庞大的结构分解成更小、更灵活、更创新的单元。从宏观层面来看，我们正处于科技公司内部一场伟大的分拆的早期阶段。易贝和贝宝已经友好地分道扬镳。戴尔私有化以变得更加灵活和积极，而赛门铁克拆分成两家上市公司。

反过来，这种大规模的解耦在企业中更细粒度的层面上影响解绑，这一趋势在解决老化的代码库和笨拙的软件架构所导致的问题方面被证明是至关重要的。维护传统技术的困难意味着公司积累了大量的技术债务，使得有才华的开发人员更难完成工作，结果扼杀了创新。

对于大型企业来说，松绑是对抗惰性和停滞的关键对策，也是对抗更敏捷的竞争对手的宝贵策略——创新型初创公司不会被同样的遗留技术债务和随之而来的工程官僚主义所困扰。

从单一的遗留系统中分离出来也增强了公司吸引和留住人才的能力。许多开发人员会回避用数百万行代码来维护 Java 应用的想法，但这是依赖 Java 和. Net 等传统技术的公司的现实。仅在 2014 年，PayPal 就雇佣了 400 多名 JavaScript 开发人员，他们中没有一个人被请来处理技术债务。

## 每个公司都是科技公司

我们看到了打破不同行业单一架构的切实价值。对于消费者来说，松绑意味着灵活性和选择——只为你真正想要的产品和服务付费。但是对于企业来说，松绑是创新的关键驱动力。

现实是，今天，每个公司都是技术公司。不管你的业务是什么，技术要么给你优势，要么给你的竞争对手优势。通过远离宏观服务和笨重的传统技术，企业可以像初创公司一样灵活和灵敏。
