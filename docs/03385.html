视频 Imgur vid me 获得 320 万美元| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2015/04/07/vidme-an-imgur-for-video-grabs-3-2-million/

该公司今天上午宣布，视频分享应用 Vidme 已经筹集了 320 万美元的新资金，这是一个非常重要的视频应用，因为它与 Reddit 社区也有着密切的联系。种子轮的投资者包括的马克·苏斯特(Mark Suster)、的 Reddit 联合创始人亚历克西斯·奥哈尼安(Alexis oh anian)、首轮、小写资本、 SV 天使、Mucker Capital、Launchpad 等。

这项视频服务是 Bit Kitchen 的三个项目之一，Bit Kitchen 是一个产品实验室，由洛杉矶 SocialEngine 的前创始人兼首席执行官亚历克斯·本泽和前 SocialEngine 的首席运营官·沃伦·谢弗创立。该团队正在整合项目，并将其发布到野外，以观察用户的反应，但最终是 Vidme 取得了成功。

到 2014 年 1 月，该网站在第一个月就积累了 140，000 个独特的照片，这在一定程度上要归功于 Reddit 社区的成员，他们使用该网站作为一种快速简便的方式来分享视频和帖子。Benzer 说，如今，Vidme 每月的用户数已经超过 3000 万。

这位联合创始人表示，投资者主要被 Vidme 吸引，因为这项服务在视频领域提供了一个有趣的差异化优势，并已显示出早期的吸引力。此外，他指出，今天对数字视频内容的需求是巨大的，市场正在快速增长。

Benzer 解释说:“人们在手机上消费内容，这是一个浪潮式的转变，投资者希望资助一个团队，这个团队整天都在思考这个问题，并想出创新的方法来重新想象发现体验和创作体验。”。

例如，与 Vimeo 或 YouTube 等替代产品相比，这项服务的吸引力在于其简单性。你所要做的就是浏览到一个视频文件或 GIF，从你的电脑上拖放一个或者直接粘贴一个 URL。与此同时，Vidme 的移动应用程序可以让你直接从手机上的图库中上传文件。

视频上传后，你可以将 Vidme 链接分享到社交网络和其他地方，包括发布在 Reddit 上。

Benzer 说，Vidme 的大部分增长是由一步视频上传的效用推动的。

然而，Vidme 值得注意的一点是，它不要求用户注册他们的社交档案、电子邮件地址或电话号码(尽管脸书、Twitter 和电子邮件是可选的)。相反，你只需要在注册时选择一个用户名和密码。但是你不必创建一个账户来上传——你也可以匿名上传。

然而，这些视频与您的设备相关联，因此如果您愿意，您可以稍后将它们拉下来。

最近，Vidme 在其网络上推出了额外的发现功能，包括其自己的“首页”，展示每天更新的热门视频。我们被告知，该公司现在看到了 Vidme 有点像视频杂志的潜力。而且，受 Reddit 的启发，它甚至为其各个类别添加了版主。

到目前为止，Vidme 用户已经向网站上传了广泛的内容，包括有点新闻性质的直播镜头、个人视频和 Vidme 分类到几十个类别的一系列其他内容——其中一些内容会让人想起你在 Reddit 或 Imgur 上找到的内容。默认网络包括像动画，牛逼，Aww，无人机，搞笑，树，WTF 等类别。

但在其他地方，甚至有更多的类别，从娱乐到科技到体育到《我的世界》等等。一些用户在 Vidme 上分享他们的 Snapchat 故事，而另一些用户则用它来收集 gif。但大体上，这些视频本质上是娱乐性的。

“我们正在建设一个新的娱乐平台，一个对创作者友好的社区，一个非常精英化的地方，让内容被发现并上升到顶端，”谢弗说。

这位联合创始人暗示，该公司现在正在投资创作工具，包括一个新的移动应用程序，允许用户创建“独特的视频”。他说，该公司还在开发其 API，允许第三方网站和其他开发者在 Vidme 的基础上进行开发。例如，一些潜在的用例是其他平台的应用程序，或者集成到其他网站中。

最终，该公司希望通过原生广告产生收入，而不是前滚动，但这是未来的路。在短期内，该团队专注于通过更好的应用程序来改善其在 Android 上的体验，更像当前的 iOS 版本。

Vidme 是洛杉矶的一个六人小团队。其 iOS 应用程序于本周早些时候更新，可在 iTunes 上下载。
