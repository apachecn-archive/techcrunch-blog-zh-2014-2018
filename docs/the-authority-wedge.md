# 作者身份楔

More posts by this contributor

**编者按:** *Tadhg Kelly 是游戏行业顾问，自由设计师，领先设计博客* [*的创建者，什么游戏有*](https://web.archive.org/web/20230328073543/http://www.whatgamesare.com/) *。你可以在 Twitter 上关注他* [*这里*](https://web.archive.org/web/20230328073543/https://twitter.com/tiedtiger) *。*

“权威”是一个值得思考的有趣词汇。一方面，它唤起了独裁者和镇压的形象。但它也随着作者身份而分解。一种认可、未指明的接受、作者身份而非作者身份的印章。通过出现在官方的旗帜下，看起来比外面所有的 10，000 个选择更好。如果是网飞付费的，肯定比在 YouTube 上自制的好。如果它以 TED 演讲的形式出现，那它肯定比录制在某人个人 Tumblr 上的演讲要好。如果它出版了，可能会比只是制作出来的更好。

在过去的几年里，这个想法被重新提出来，这难道不有趣吗？根据各种关于互联网将如何改变一切的文化理论，这并不是真的应该的。然而，从音乐到视频游戏，从书籍到博客，对权威的偏好情绪到处都在重复。即使在这里。

这里有一个简单的例子:我可以写这篇文章并发布到许多网站上，但因为它在 TechCrunch 上，所以它会传播得更远，比其他地方更受重视。公关人员会在几个星期后给我发电子邮件，说他们喜欢它，并问我是否有兴趣报道他们的产品或服务？(第 n 次我不是记者)。为什么？“上 TechCrunch”让我看起来像个作家。我是吗？我不太确定。不管怎样，我的文本都是一样的。但是因为它在绿色的旗帜下，人们对它有不同的看法。没有人可以幸免，我不行，你也不行。亲爱的读者，如果一件东西看起来是由别人出售的，你往往会对它做出更积极的反应。其他人也一样。

在互联网走过第三个十年之际，这是一个重要的教训，因为它解释了平台业务运营方式的深刻转变。这么想吧:大部分显而易见的平台机会都没了。从搜索引擎到空余房间共享，似乎许多让它们起步的效率之痒现在都被抹去了。有一些更好的解决方案(例如拍卖)仍然成熟，但不多。这导致平台模式向边缘推进，以寻找新的领域(手表、虚拟现实、信息、视频直播等)，但平台业务也开始向创作业务推进。这在某些方面是一种进化，在其他方面是一种逆转。

一个很好的例子是网飞、亚马逊和 Hulu 正在制作的节目。根据平台的效率模型，很多硅谷人认为他们不应该做，但当然应该做。如果只有老电影和去年的电视剧，网飞不会继续吸引人们的注意力。它需要只有超胆侠才能带来的正面关注。如果 PlayStation 只有 Steam 上已经有的独立游戏，那它就不会成功。它需要它的*血统*。尽管很乱，独家内容还是很吸引人。

另一个例子是在所谓的平等主义平台上宣扬关键和独家内容。举例来说，苹果的应用程序商店真的是所有关于获得特色和进入排行榜前 25 名。如果你没有达到这个目标，那么你需要花费大量的广告费来通过其他方式获得安装，通常这是行不通的。选择可能是有的，但是大多数时候用户没有动力做出选择。他们看起来没有权力。

实质上，克里斯·安德森、塞思·戈丁、克莱·舍基和许多其他早期的思想形成者似乎高估了各种市场中无限选择的力量。人们总是假设利基市场会定义主流市场，主流市场会跟随，但这种假设只对了一半。相反，我们有越来越头重脚轻的单一网络，我们越来越少听到从网站起家的艺术家的故事。事实证明，我们的选择经济并不是普遍适用的，通常是由平台的新颖性而不是持久的兴趣所驱动的。

在我的整个空间(游戏)中，我看到同样的故事发生在那些觉得他们没有办法得到关注的开发者身上，我也从其他领域的人那里听到了同样的事情。马库斯·佩尔森并没有催生一波新的富有的印度人。阿曼达·帕尔默的成功很大程度上是阿曼达·帕尔默的。休豪伊在 Kindle 上做得很好，但他不是剑尖。并不是说我对这些创造者有什么意见，我在这里谈论的更多的是宏观而不是分配责任。每一个都很大程度上与平台经济学的新奇阶段有关，而经常被预测的曲线应该跟随它们？只是没有实现。

断言所有事物的经济学都是曲线型的似乎有些言过其实。相反，我们看到的是，随着新平台的新奇感逐渐消失，熟悉度逐渐提高，权威的楔子正在形成。我们将看到它在那些仍处于新奇阶段的平台上发生，从 YouTubers 到 live streamers，从绿光游戏到 Kindle Scout。观众越来越喜欢转向熟悉的权威形式，而忽略其他形式。当然，他们的权威来源现在可能是网上，而不是电视上，但在许多方面，他们的功能还是和以前一样。我们看到许多不同的声音聚集成更小的集合，以及一组无限提供的观众，他们只关心选择第一。

那么，这让创造者们何去何从？赤脚和怀孕。
