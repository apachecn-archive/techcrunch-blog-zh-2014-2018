# 东京工业大学为日本最快的人工智能超级计算机开发英伟达

英伟达的业务越来越多地成为人工智能业务，其最新的合作伙伴关系符合这一新角色。这家图形处理制造商正在向东京工业大学提供 GPU，这将为其新的人工智能超级计算机提供动力，一旦完成，这将是日本同类计算机中最快的。

使用 Pascal 处理架构的 NVIDIA Tesla P100 GPU 将用于创建该集群，该集群将被称为 TSUBAME3.0，并将以两倍的性能能力取代 TSUBAME2.5。不过，不要为 TSUBAME2.5 感到太难过——它仍将被积极使用，将其能力添加到 TSUBAME3.0 的预计 47 petaflops 中，总共达到 64.3 petaflops 你需要一大堆 iPhones 才能与之匹配(就像非常非常多)。

TSUBAME3.0 的目标是在今年夏天之前完成开发和处理，届时其技术将被用于东京学术机构的教育和高科技研究。它也可用于私营部门承包，该学校表示，它迫不及待地开始教授新的虚拟大脑。
