在我们生活的世界里，由于技术和信息的获取，人们可以培养专业知识，并为以前需要正规学校教育的项目或行业做出贡献。第一个很好的例子是大约 10 年前，我们看到“公民新闻”从博客和社交媒体的兴起中诞生。

今天的运动是我们称之为“公民发展”的东西，这意味着发展不再仅仅是关于开发者，而是关于社区以及信息和创新流的动态。

在过去的几年里，当今社会的几种情况使得公民发展蓬勃发展。

负担得起的互联网接入和高性能设备已导致数字素养的普遍提高。这意味着许多普通市民可以清楚地表达对创新数字服务的需求，这种服务可以帮助他们的日常生活。由于可以使用低代码工具和在线教育，他们实际上可以参与到将想法付诸实践的过程中。

此外，随着对数据和设备的访问变得更加统一，社区中的阶级差距正在消失。一个非洲的孩子(理想情况下)可以和一个加利福尼亚的孩子一样使用同样的数字资源。这为公民参与和社区间合作带来了许多机会，减少了文化和语言障碍。

这种日益增加的一致性也将有助于形成更强大、更协同的社交网络，其目的不仅仅是寻找和联系朋友。例如，想想看，美国的一个小额融资课程可以帮助非洲的马赛部落与加州的一所学校建立联系，在美国出售他们美丽的珠宝，避免中间商。这两个组可以访问同一个数字平台，可以相互连接，并在每一端不同地使用该平台来实现共同的目标。

另一个需要注意的重要的、稍微更技术性的条件是 API 及其互操作性的标准化。这是推动公民参与和公民发展的核心开放数据政策之一。

它是关于围绕开发者建立一个社区，每个成员将放大核心服务的价值。

例如，在过去五年中，政府开放了他们的数据，允许公民为他们的社区建立有用的应用程序和服务。像美国政府的Data.gov和麻省理工学院的数据中心这样的计划是核心资产如何变得可供公民为他们的社区贡献集体智能生态系统的很好的例子。美国政府甚至最近成立了首席技术官办公室，由梅根·史密斯管理，她正在打造一个互联互通的生态系统。

随着大多数以消费者为中心的互联网服务(如 Spotify 、 Yelp 和优步)纷纷效仿，并向大型开发者群体开放其数据，现在轮到公民将商业和政府数据源连接在一起，为他们的社区创建有用服务的混搭。世界各地已经有很多这样的例子。例如，在以色列，有一些完全由公民开发的可靠应用程序可以提醒他们的社区火箭袭击的威胁。

阻碍公民参与和发展的一个关键因素是缺乏直观的体验设计工具。随着发展所需的数据变得越来越容易获得，这些工具将在运动的进程中发挥关键作用。

回想一下早期的网络时代，那时构建网页只是开发人员的事情。随着越来越多的工具可供他人使用，首先是像微软 Front Page 这样的可视化开发工具，然后是像 WordPress 这样的博客和平台，每个人都获得了在线发布内容的权力。接下来，我们看到像 Twitter 这样的统一平台出现，允许数百万人在网上表达他们的观点。

类似的类比也适用于公民参与，移动设备处于最前沿——只不过现在你需要工具(特别是在移动设备上)，让你远远超越发布内容，并融入隐私管理、上下文感知、传感器集成等概念。

也就是说，公民发展和参与持续进步和成功的真正关键是拥有将不同群体(如设计师、开发商、艺术家和政府官员)聚集在一个社区中的工具。这是我认为公民发展成为我们新标准的最大障碍，几乎任何人都可以将想法带到桌面上，并与他们需要的所有人联系，使之成为现实。

所以在某种意义上，公民发展不仅仅是开发者的事。它是关于围绕开发者建立一个社区，每个成员将放大核心服务的价值。公民发展有能力将设计师、开发商、政府和企业聚集在一起，使他们能够解决关键问题并合作创新。只要我们都继续推动开发运动蓬勃发展所需的工具，我们将看到它在未来几年继续聚集动力。
