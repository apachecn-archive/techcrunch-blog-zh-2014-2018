# SAP 将社交和学习融入新工具 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2016/01/25/sap-melds-social-and-learning-in-new-tool/>

多年来，我们已经有了社交工具和正式的学习管理系统，但我们从未真正有过将这两者结合起来的东西——直到今天，SAP 宣布了一个名为 SAP Jam Collaboration for learning 的新系统。

新工具旨在将企业学习的各个方面整合到一个更加社会化的环境中。它使用 SAP Jam 作为一个平台，结合了用户内容、指导、社区和正式的学习系统，也称为学习管理系统或简称 LMSs。

虽然学习和员工参与通常处于高管关注的边缘，但 SAP 高级副总裁兼协作和社区总经理 Sameer Patel 表示，随着公司被迫应对市场上的破坏性力量，高管需要更加关注公司的知识管理和共享学习。

帕特尔说:“该系统旨在为每一个问题汇集最优秀的人才。”。Patel 解释说，SAP 正试图通过融合社交、学习和员工参与来实现这一目标，因此对于每个感兴趣的主题，你都可以将实践经验与正式的课堂培训和同伴指导相结合。

后一点在这里尤其重要。“从有经验的员工那里可以学到很多东西，但你如何将这些带到正式的学习经历中呢？你需要找到专家，拓宽话题，不仅仅是通过上课，还要从拥有宝贵经验的个人那里寻找，”他解释道。

实际上，这包括创建链接到 LMS 的学习社区，同时让人们围绕它进行协作和共享。这可能意味着提供相关的课程材料，或者提供分享用户生成的专家视频的方式，或者确定可以作为导师的人。它还包括让注册课程和查看人们对课程的评价变得更简单。

根据美国培训与发展协会的一项研究，不管质量如何，发展培训每小时要花费超过 1700 美元。然而，这种培训往往可以在一个大组织内部进行，员工无法知道外面有什么。通过将它与 SAP Jam 等社交平台相结合，它为员工可能会使用的社交社区内部的培训提供了一个结构。SAP 相信您可以通过这种方法降低总体成本并提高兴趣

在许多方面，它试图带回知识管理的概念。在 20 世纪 90 年代，公司试图(并且经常失败)创建概括人们经历的数据库。最近，企业 2.0 协作工具应该在社区中公开这些信息。在某种程度上，这两种方法都没有奏效。在第一种情况下，要求员工以一种明确的方式分享他们所知道的东西实在是太过分了，而在第二种情况下，数据可能会在社交流量的冲击下丢失。

这个系统所做的是结合这些世界的精华，让人们在一个特定的学习讨论的背景下接触导师和专家，展现他们的专业知识，这样用户就不必明确地寻找它。

当你把它与正式的类和用户生成的内容结合起来时，它开始填充一个比这些部分单独完成的要大得多的画面。当然，证据将会在实践中，以及它在过去控制知识和学习的尝试上有多好的改进。如果它像描述的那样工作，那么它可以解决一个根深蒂固的问题，这个问题围绕着企业中的表面知识。
