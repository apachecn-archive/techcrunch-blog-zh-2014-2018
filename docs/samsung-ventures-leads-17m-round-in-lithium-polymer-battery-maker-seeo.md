# 三星风投领投锂聚合物电池制造商 Seeo 万美元

总部位于加州海沃德的锂聚合物电池公司 Seeo 已经完成了由新投资者三星风险投资公司(Samsung Ventures)牵头的 1700 万美元融资，三星风险投资公司是三星集团的投资部门。现有投资者 Khosla Ventures 和 GSR Ventures 也参与其中。这使 Seeo 迄今筹集的资金总额达到 4060 万美元。

Seeo 的可充电锂电池是使用一种专有的不易燃聚合物电解质制造的，它称之为“DryLyte”，它声称这种电解质比传统的锂离子电池更安全。T2 公司表示，他们的电池体积更小，但无论外界条件如何，如温度如何，都能保持同样的性能。Seeo 目前正在开发用于电动和混合动力汽车的电池；电网；数据中心等电信基础设施；和消费电子产品。

这并不是三星风投的第一笔清洁技术投资。今年早些时候，它[领导了对 Distech Controls](https://web.archive.org/web/20221005141806/http://www.distech-controls.com/News_Events/PR/20130522_PR_DC_Welcomes_Samsung_Ventures.html) 的 3700 万美元投资，该公司开发管理建筑能效的技术。由于三星集团的垂直行业从消费电子到工业工程，投资那些可能让它获得更便宜、更环保的能源的公司是有意义的。
