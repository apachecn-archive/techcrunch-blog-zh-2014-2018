# 利用流行消息工具 TechCrunch 的组织覆盖图，对 Slack 发布进行分类

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/03/06/sortd-for-slack-launches-with-an-organizational-overlay-for-the-popular-messaging-tool/>

至少从 2014 年开始， [Sortd](https://web.archive.org/web/20221206205213/http://www.sortd.com/) 的团队已经开发出工具来组织来自消息服务的不和谐声音。现在，该公司又开始了这项工作，这一次[希望将 Slack](https://web.archive.org/web/20221206205213/http://www.sortd.com/sortd-for-slack?code=39583) 转变为 Sortd 声称的更完美的群组信息和工作流服务。

像 Gmail 的[工具](https://web.archive.org/web/20221206205213/http://lifehacker.com/how-i-finally-organized-my-messy-inbox-with-sortd-1754956174)一样，organizational Slack integration 被设计成一个工作流助手和任务管理工具。使用它，渠道领导可以分配任务，管理团队优先级，并有选择地发布状态更新通知。

Sortd 的首席执行官韦恩·西尔伯曼(Wayne Silberman)称该工具是 Slack 内部 Asana 或 Trello 的竞争对手，而不是需要走出 Slack。

“我们认为它是 Slack 用户的终极待办事项列表和项目管理工具，”西尔伯曼写道。

该工具包括启动一个新的 Slack launcher，可以从 Sortd 的网站下载。该应用程序将 Slack 环境与任务管理器结合在一起，它们集成在一个屏幕上。

用户将下载新的 Slack launcher，并将其与 Gmail 帐户(Sortd 要求你注册两个帐户)和 Slack 帐户(尽管 Silberman 保证他的软件跟踪的唯一信息是权限令牌)相链接。

西尔伯曼说，对于许多人来说，组织和任务管理功能可以成为今年早些时候 Slack 推出的[线程解决方案](https://web.archive.org/web/20221206205213/https://slackhq.com/threaded-messaging-comes-to-slack-417ffba054bd?gi=36b2659e609e)的替代方案(但仍然很笨重)。在 Sortd 中，任何人都可以评论任务和更新任务注释。

Sortd for Slack 为每个 Slack 通道提供了一个可视的项目板，用于跟踪分配和完成的项目，每个团队成员都可以使用，无需任何额外的配置。该软件包还包括一个独立于频道的私人任务板，因此用户可以看到他们的个人任务。

还有不同的视图，包括:日常使用的侧边栏，包含多个讨论和优先级列表的扩展视图以及全屏模式。可以根据优先级对任务进行重新排序，并且可以将松弛消息拖放到待办事项列表中。

任何团队成员都可以将可分配的任务添加到项目列表中，并且可以公开或私下查看这些任务。最后，该集成为用户提供了日常任务的待办列表视图，以及按日期、组或时间以及在聊天中查看任务的能力；未完成的分配任务显示为屏幕侧边的项目。

Sortd 的管理层明确表示，新产品不计入 Slack 集成，它不是一个机器人或使用 Slack API。

事实上，如果有什么不同的话，西尔伯曼的新项目应该是对 Slack 的一种挑战。与 Trello 不同，Sortd 没有为卡片提供封面图片的能力，它没有针对任务的统一活动轨迹，并且移动应用程序缺乏评论功能。

这意味着它将成为从[特雷罗](https://web.archive.org/web/20221206205213/https://trello.com/)、[阿萨纳](https://web.archive.org/web/20221206205213/https://asana.com/)和[神童列表](https://web.archive.org/web/20221206205213/https://www.wunderlist.com/)获得的 Slack 集成的竞争对手，以及像[凯伯](https://web.archive.org/web/20221206205213/https://kyber.me/)和[待办机器人](https://web.archive.org/web/20221206205213/https://www.todobot.io/)这样的 Slack 机器人。
