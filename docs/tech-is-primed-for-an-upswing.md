# 科技股正准备上涨 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/08/23/tech-is-primed-for-an-upswing/>

本·希珀斯撰稿人

More posts by this contributor

过去的 15 个月里，围绕着技术部门的走向，人们产生了许多困惑。企业家们经常问我，现在是否是创业的好时机，我会投资当前的市场吗？是啊是啊。

没完没了地谈论底部已经跌出、估值过高(确实如此)和公司倒闭，导致种子和风险投资交易的完成和公司的启动出现了轻微的干旱。那个时代已经过去了。

戏剧创造了页面浏览量，但市场的现实很简单:利率处于历史最低水平。资金在以非常吸引人的速度流动。如果有什么时候可以开一家公司，那就是现在。此时此刻，找工作的工程师越来越多，投简历的企业家越来越少。时机很理想。

更重要的是，风险公司正在筹集和结束新一轮融资。这些基金中的大多数必须在四年内部署他们的资本，所以有大量的资金需要找到归宿。

种子基金正以正常的速度投资，尽管估值已回升至正常规模，但交易正在结束。

> 这是制造产品的理想时机。

我不是提倡为了更多的应用而构建应用。那些不会得到资助的人。我会敦促企业家在有资金的时候为强有力的想法筹集资金，甚至如果它的估值不是你认为可以接受的。(解决估值差异的方法是在估值较低时拿得较少，将成本保持在最低水平，并在下次融资时补足差额。)

有些领域比其他领域更容易起步。app 很棒，但是已经有人做了，所以准入门槛很低。消费者和更大的市场已经达到了一定的饱和门槛。随着这些市场的成熟，正如我们在其他市场看到的那样，企业家必须解决更困难的问题才能取得成功。毕竟，解决难题的企业提供价值。

在这种情况下，几乎所有东西都需要固件和软件以及硬件来驱动。这些市场远没有饱和，对投资者更有吸引力。企业家应该在核心互联网之外思考。[我已经暗示了一些伟大的想法](https://web.archive.org/web/20230217020558/https://techcrunch.com/2016/06/25/the-hungry-consumer-and-the-software-pivot/)，这些想法是发现和探索的最佳时机，大多与 ARM 芯片架构有关。

作为一名创业者，现在是创业的最佳时机。别管博客，透过树看到森林。请注意，更大的景观描绘了一个非常不同的画面。这是制造产品的理想时机——各方面的市场都是最佳时机。
