# Yandex 将其机器学习智能转化为一种新的 B2B 产品，Yandex 数据工厂 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/12/09/yandex-turns-its-machine-learning-smarts-into-a-new-product-for-all-yandex-data-factory/>

俄罗斯搜索巨头[Yandex](https://web.archive.org/web/20221007180603/http://www.yandex.ru/)——刚刚[进行了一项收购](https://web.archive.org/web/20221007180603/https://beta.techcrunch.com/2014/12/09/yandex-buys-sovetnik-to-expand-its-e-commerce-business-with-better-recommendations/)以扩大其电子商务业务——也推出了一款新产品，这将使其更深入地进入 B2B 服务领域。今天，它正在推出 [Yandex 数据工厂](https://web.archive.org/web/20221007180603/http://www.yandexdatafactory.com/)，这是一个新的业务部门，基于它为自己的服务在内部开发的机器学习，名为 [MatrixNet](https://web.archive.org/web/20221007180603/http://company.yandex.com/technologies/matrixnet.xml) ，现在已经开发成自己的产品。

YDF 将基于 SaaS 的订阅商业模式，该公司表示，已经为客户开发了大约 20 个项目。

YDF 是上市公司 Yandex 更大野心的一部分，该野心还包括将其业务扩展到主要在俄罗斯的边界及其核心搜索广告业务之外。这个部门将在阿姆斯特丹和莫斯科有两个总部，很明显，这项服务不是在俄罗斯，而是在巴黎正在进行的 le 网络会议上正式推出。

YDF 不是凭空捏造的。自今年早些时候以来，该公司一直在一对一的基础上与银行、俄罗斯[道路管理机构](https://web.archive.org/web/20221007180603/http://company.yandex.com/technologies/traffic_jams_technology.xml)和其他机构合作，以将其[机器学习技术](https://web.archive.org/web/20221007180603/http://company.yandex.com/technologies/searcharch.xml)和人工智能应用于这些第三方的不同挑战和任务——有效地使用该公司已经用于自己的搜索、音乐推荐、语音和图像识别的相同技术。

这包括与 CERN 的合作，Yandex“训练”MatrixNet 在大型强子对撞机(LHCb)记录的事件中搜索特定的粒子碰撞。

“这个项目的成功让我们有理由相信它可以在其他应用领域重复，”该公司在一篇博客文章中写道。“任何产生大量数据并专注于业务目标的行业都可以受益于我们的专业知识和我们基于 MatrixNet 的技术:个性化的搜索建议、推荐或搜索结果、图像或语音识别、道路交通监控和预测、自动翻译的词形预测和排序、针对受众的人口统计分析。”

但实际上这项工作可以追溯到更早的今年。2012 年，Yandex[还入股了一家地理技术公司](https://web.archive.org/web/20221007180603/https://beta.techcrunch.com/2012/06/22/tech-meets-geo-tech-russias-search-giant-yandex-takes-1m-25-stake-in-seismotech/)，地震技术公司，作为其应用其计算架构和算法解决其他问题的战略的一部分。

YDF 与谷歌在自己的企业服务领域以及通过谷歌大脑(Google Brain)等项目开发的东西没有什么不同(离开公司的前谷歌员工也在[试图开发](https://web.archive.org/web/20221007180603/https://beta.techcrunch.com/2014/11/14/scaled-inference-lands-8m-from-khosla-to-build-a-cloud-based-ai-platform-for-all/))。

如今，Yandex 和 Google [等公司的大部分收入来自搜索广告](https://web.archive.org/web/20221007180603/https://beta.techcrunch.com/2014/12/08/in-86b-of-online-media-sales-in-h1-google-leads-chinese-sites-grow-and-ads-take-lions-share/),但为了实现多样化，并抵消该业务未来可能出现的任何下滑，他们需要继续开发和投资这样的新领域，因此我们很可能会看到更多这样的事情发生。
