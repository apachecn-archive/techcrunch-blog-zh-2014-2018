Skype，振作起来 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/03/10/skype-get-your-shit-together/

Skype 是 TechCrunch 新闻编辑室工作流程的一部分。这是个人作者交流、分享、合作等的标准方式。我们使用其他软件作为一个团队工作，但对于一对一的聊天，Skype 是我们的障碍。

不好的一面是，它就是不太好用。这已经成了办公室里的一个笑话:用 Skype 发给你的——那么，你明天会收到？

Skype 有什么问题？它不能在不同设备之间正确地同步信息，所以如果你在工作时在 Mac 上使用 Skype，而在家里使用 PC，那就糟了。文件传输仍然非常慢。信息通常在一段时间内不会出现在收件人的机器上，导致混乱，有时还会挫伤自尊心。还有 Skype 群聊，我们中的一些人直到第二天才可以进入。

引用我的同事瑞安·劳勒的话，Skype“只有一份工作！”—消息传递——而 it“连这点都做不好。”

你知道什么最棒吗？创建一个 Skype 房间，以便一组 TechCrunch 作者可以一起工作，然后一个或多个作者告诉他们“看不到它”。所以你坐在那里想:什么？为什么？创建一个团体真的有这么难吗？

但是，嘿，你正试图覆盖一个大的，突发新闻故事，所以你只能好奇几秒钟。然后你只是耸耸肩，试图让每个人都知道房间里发生了什么，并希望在你打电话的时候不要漏掉一条重要的信息。

微软于 2011 年以 85 亿美元收购了 Skype。现在是 2015 年，Skype 仍然是一个脆弱的体验。对于一个如此受全球欢迎和使用的工具来说，这是一个奇怪的情况。微软最近在 Skype 品牌上下了双倍的赌注，将其企业聊天服务转移到“Skype for Business”的名下。鉴于消费者 Skype 存在的问题，您几乎会怀疑这一选择的有效性。
