Entelo 凭借价值 2000 万美元的 C 轮投资升级其人工智能游戏

人工智能招聘软件领域的冠军争夺战已经开始。随着过去几周 Workey 和 T2 Mya 都进行了几轮谈判，现在是一些玩家以增长资本的形式寻求优势的最佳时机。这似乎正是一个六岁的太空玩家恩特洛正在做的事情。该公司今天宣布 2000 万美元的 C 轮融资，由美国风险投资伙伴领投，并有电池风险投资、沙斯塔风险投资和关联风险投资参与。

Entelo 抓取互联网，自动生成潜在招聘前景的简介。然后，该公司努力将潜在客户与其企业客户相匹配，以发现和招募顶尖人才。与 LinkedIn 不同，Entelo 目前不允许个人创建自己的账户。相反，除了允许任何人随时请求删除其个人资料的退出控制之外，所有操作都在后台进行。

Entelo 的首席执行官 Jon Bischke 告诉我，他对公司的首要任务是改善幕后的匹配过程。要实现这一点，Entelo 需要从 GitHub 等非传统来源收集额外的非结构化数据，并实施额外的机器学习功能，以使企业能够快速识别和锁定最佳候选人。

Entelo 面临着来自市场两侧的竞争——更年轻的人工智能优先的初创公司和 LinkedIn 等传统公司。就目前而言，企业应该采取“除了厨房水槽以外的一切”的方法来获得招聘优势，但这种心态可能不会永远持续下去。

Bischke 认为他的定位最能让 Entelo 在尘埃落定时茁壮成长。他在一次采访中告诉我，如果没有足够的数据，许多人工智能优先的人力资源初创公司将会陷入困境。在等式的另一边，LinkedIn 还有很多工作要做，以在潜在的分散注意力的收购中保持创新。

迄今为止，Entelo 已经为其平台签约了 600 多家公司。这些企业包括脸书、通用电气、诺斯罗普·格鲁曼和塔吉特百货。该公司将寻求雇用更多的数据科学家，并加入其销售团队。
