# 智能手机维修初创公司 Save 被 Remade Group  收购

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/04/07/smartphone-repairing-startup-save-gets-picked-up-by-remade-group/>

# 智能手机维修启动保存被翻拍组拿起

上次我[写](https://web.archive.org/web/20230319023522/https://techcrunch.com/2015/09/07/save-secures-16-7-million-to-repair-your-smartphone-in-no-time/)关于[拯救](https://web.archive.org/web/20230319023522/https://www.save.co/)的时候，这家法国初创公司刚刚筹集了 1670 万美元，以证明你可以扩大智能手机维修服务的规模。从那以后，情况就像坐过山车一样，公司不得不[宣布破产。在这个过程之后，](https://web.archive.org/web/20230319023522/https://medium.com/@damienmorin/save-2016-c2aff5d910ee)[重组后的集团](https://web.archive.org/web/20230319023522/http://remadeinfrance.com/)正在收拾 Save 的业务和员工。

虽然这家初创公司发展非常迅速，但很难跟上它的步伐。有一些物流问题和一些国际扩张举措没有奏效。

我听过一些关于拯救的疯狂董事会故事。没有人同意该做什么。一些人想解雇管理团队，另一些人想解雇董事会成员，而这家初创公司想从德国投资者那里筹集更多资金。最终，一切都泡汤了，Save 最终申请破产。

为在 137 家商店工作的 500 名员工支付工资是相当昂贵的(其中大部分在法国商场)。Save 不得不解雇部分团队，专注于公司最赚钱的领域，优化一切。

第三方公司可以竞标……拯救拯救。通常，法院会考虑所有的提议，选择对员工和公司未来最有利的一个。

***更新:** Damien Morin 告诉我，该业务实际上并不是由破产导致的收购。这是一笔现金加股票的交易，因此股东们也能拿回一部分钱。管理团队将留在公司。“这几乎是一个合并，”莫林说。*

Remade Group 是一家销售翻新智能手机的法国公司。[根据](https://web.archive.org/web/20230319023522/https://medium.com/@damienmorin/save-rejoint-remade-group-dfd58056bd40)Save 首席执行官达米恩·莫兰的说法，Remade Group 正在偿还债务(**更新:** Remade Group 没有直接偿还债务，Save 正在用收购款偿还债务)，Save 品牌仍将继续运营。你仍然可以在你最喜欢的购物中心找到便利店。并且挽救员工仍将有工作。

储蓄增长过快，跟不上其疯狂的增长率。或许这家初创企业本可以发展得更慢一点。但是，当你筹集到 1670 万美元时，赌注突然变得更大，因为每个人都希望你退出。

很容易回顾过去，找出问题所在。所以我不会因为他们雄心勃勃的目标而责怪 Save。
