# 到目前为止，网络中立还没有破坏互联网 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/06/15/so-far-net-neutrality-hasnt-broken-the-internet/>

FCC 的网络中立规则于上周五生效，TechCrunch 很满意地报告说，更广泛的技术领域并没有崩溃。

上周，一家法院驳回了暂停执行该规则的请求，并按照预期设定了该命令。联邦通信委员会主席汤姆·惠勒在一份声明中称新规定是“互联网消费者和创新者的巨大胜利”今年早些时候，惠勒在接受 TechCrunch 采访时表示，新的“开放互联网保护”是“任何人都想象不到的”最“强”的，这也是对这一观点的回应。

威瑞森——本出版物的未来家园——一家从本质上反对网络中立的公司的股票在下跌的市场中下跌了一小部分。投资者似乎没有在网络中立法案颁布后立即对其进行太多的负面定价。

威瑞森点有一定的讽刺意味。威瑞森对导致新规则的网络中立提起了最后一次诉讼。标题 II 在很大程度上是由于 ISP 的行为。

对网络中立性的法律挑战正在加速进行，可能会在 2015 年底结束。加快庭审时间表的关键是市场需要这一问题的确定性。如果联邦通信委员会的规则被推翻，事情就变了。如果该机构在法庭上胜诉，事情不会改变。

在联邦通信委员会赢得搁置其规则的胜利后，惠勒表示，该决定“给予宽带提供商确定性和经济激励，以建设快速和有竞争力的宽带网络。”互联网服务提供商宁愿有另一种方式。

除了法律威胁，网络中立的另一个潜在威胁是:新政府的 FCC 改变了规则。这一事实给当前的总统选举周期增加了另一个难题——谁赢了，谁就能维持或改变网络中立政策的方向。

当然，国会采取行动仍然是可能的。

技术利益集团 CALinnovates 的迈克·蒙哥马利(Mike Montgomery)告诉 TechCrunch，如果白宫的政党发生变化，事情可能会迅速转变:

> 一位共和党总统肯定会优先任命一位新的 FCC 主席，而这位新主席很可能会将开放互联网秩序作为她或他的第一项任务。[……]新总统任命新的联邦通信委员会主席将改变委员会的权力平衡，将 3 比 2 的民主党多数变成民主党少数，从而提供选票，要么完全推翻第二章的实施，要么彻底忍耐，导致理论上的“狂野西部”，缺乏任何明确的道路规则，这将为消费者、创业公司和更大的商业社区以及投资者创造一个噩梦般的场景。

蒙哥马利说，如果网络中立规则在法庭上失败，它可能会导致“快车道，阻塞和节流将被重新摆在桌面上。”

简而言之，正如我们所料，我们到了。欢迎回到网络中立。
