# 想去特斯拉工作？埃隆·马斯克转向 Twitter 招聘工程师

埃隆·马斯克正在寻找一些优秀的工程师。特斯拉首席执行官今晚在推特上宣布，特斯拉正在扩大其自动驾驶汽车部门，并需要一些新的人才。哦，他说他要去做一些采访，所以也有。

要应用马斯克，只需给公司发电子邮件，并附上一个你工作实例的链接。值得注意的是，马斯克表示，该公司正在寻找软件工程师，不需要汽车方面的经验。毕竟，车辆正变得越来越软件化而非硬件化。

特斯拉的这个部门负责该公司所谓的自动驾驶系统，即将其 Model S 轿车变成准自动驾驶汽车的系统。Autopilot 不会完全控制汽车，但确实允许车辆在高速公路交通和道路危险中行驶，不像谷歌的自动驾驶汽车那样完全自主。

一名特斯拉代表在自动驾驶仪发布后告诉 TechCrunch，这可能是由于当时的照明条件；如果汽车的人工智能不是 100%相信变道是安全的，它就不会。让一个司机接手总比自动转向它不知何故错过的东西要好，对吧？

埃隆·马斯克说，他认为完全自动化将在 3 年内实现；自动驾驶只是一个很大的第一步。但可能会比那更早。特斯拉显然准备加大开发力度。
