WhatsApp打击第三方应用，暂时禁止其用户使用其服务TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/01/21/whatsapp-cracks-down-on-third-party-apps-temporarily-bans-their-users-from-its-service/

WhatsApp，手机(现在还有 web ！消息客户端脸书以现在大约220亿美元收购了，开始让使用第三方客户端访问其服务的客户更加困难。虽然官方应用是访问WhatsApp最受欢迎的方式，但提供差异化功能集的替代产品也已经出现了一段时间。然而，这种情况可能不会持续太久——WhatsApp已经开始禁止这些应用，并暂时阻止被发现使用这些应用的用户使用其服务。

第三方应用之一WhatsApp Plus表示，它收到了WhatsApp的“停止通知”，因此将关闭这项服务。

在Google+ 上，WhatsApp Plus的共同所有者穆尼布·阿尔·里法伊博士表示，他们有义务“移除所有下载链接”，并删除他们的在线社区。

“这是一段有趣的旅程，但它已经走到了尽头，”他写道。“[WhatsApp Plus开发者]Rafa lenes和我以及所有whatsapp+团队对你们的支持致以最深切的问候。”

Google+上的WhatsApp Plus社区拥有70万名成员，这表明该应用拥有相当大的用户群。

据报道，另一家公司的客户端WhatsAppMD的用户也被暂时禁止使用WhatsApp。

与此同时，在WhatsApp网站上，该公司发布了一个 FAQ 更详细地解释了禁令，以及它将如何影响用户。从本质上讲，WhatsApp明确表示，不支持WhatsApp Plus等第三方客户端，可能不如官方应用安全:

为什么我被禁止使用WhatsApp Plus，如何解除禁令？

WhatsApp Plus是一款不是由WhatsApp开发的应用，也没有得到WhatsApp的授权。WhatsApp Plus的开发者与WhatsApp没有任何关系，我们也不支持WhatsApp Plus。请注意，WhatsApp Plus包含WhatsApp不能保证安全的源代码，并且您的私人信息可能会在您不知情或未授权的情况下被传递给第三方。

请卸载您的应用程序，并从我们的网站或Google Play安装WhatsApp的授权版本。然后，你就能使用WhatsApp了。

据博客 9to5Google 称，WhatsApp+等第三方应用的用户收到消息称，他们被禁止使用该服务，因为他们在试图使用他们的首选客户端时违反了WhatsApp的服务条款。

然而，禁令是暂时的，只有24小时。倒计时显示还有多少时间。

我们现在知道，WhatsApp服务条款中提到的问题部分是5C部分，其中包括一项声明，即您不得“试图获得对服务或其相关系统或网络的未经授权的访问。”

禁止用户，不仅仅是应用

在这种情况下，追踪用户本身是一种有趣的策略，惩罚他们在官方渠道之外使用WhatsApp的服务。WhatsApp这样做实际上充分说明了它的服务对最终用户来说是多么重要和被大量使用。它知道被禁止的用户会很快回来。许多其他公司不会因为这样的事情而冒险疏远他们的用户群。

尽管如此，鉴于这类应用可能给母公司带来的问题，WhatsApp禁止第三方客户端的决定并不令人惊讶。

在WhatsApp的案例中，第三方应用是丢失信息的最大来源，但历史表明，风险并不总是止于此。

例如，去年的大规模Snapchat黑客攻击被称为“Snappening”是由于第三方应用程序的不安全性造成的。虽然Snapchat在技术上并没有因数据泄露而犯错，这次数据泄露暴露了Snapchat用户的有伤风化的照片，但它在当时受到了严重的公关打击。事实上，Snapchat至少应该受到一点指责——它允许逆向工程其私有API的应用程序蓬勃发展，而不是关闭它们或以任何方式阻止它们的使用。

脸书旗下的WhatsApp显然采取了不同的方式。

我们还听说，WhatsApp正计划在未来几个月内积极追踪其他未经授权的应用。敬请关注。

我们的目标始终是让WhatsApp的用户快速安全地使用它——这是我们最重要的事情。第三方在WhatsApp上建立了未经授权的功能，给人们带来了包括丢失信息在内的问题。这违背了我们努力给予人们的体验，我们不会让它继续下去。从今天开始，我们将对未经授权的应用采取积极行动，并提醒使用它们的人。我们将把这些人转移到WhatsApp上，这样他们就可以继续安全可靠地与朋友和家人交流。
