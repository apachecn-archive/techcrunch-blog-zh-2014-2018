# 三星在美国泄露其 Note 7 更换计划的细节

如果你有一部 Galaxy Note 7，你可能已经听说了召回事件。但是每个地区的具体情况都不一样，三星刚刚发布了在这个自由的国度里你可以选择的细节。

你想做的第一件事就是去你购买设备的地方——他们会在那里进行交易。然后你可以选择你要找的交易类型。

当然，你可以“最早在下周”拿到一部新的 Note 7——大概是在不易燃电池到货并被调换后。

但你也可以买一台普通的 Galaxy S7 或 S7 Edge——都是固态设备——并获得两者之间的差价退款。配件也将被替换，所以如果你买了一个保护套或屏幕保护套，你不会失去任何现金。

除了一台新设备，每个参与交换的人都将获得一张价值 25 美元的礼品卡或信用卡。

如果你有问题，检查以确保你的手机没有着火，然后拨打 1-800-SAMSUNG，他们应该可以帮助你。
