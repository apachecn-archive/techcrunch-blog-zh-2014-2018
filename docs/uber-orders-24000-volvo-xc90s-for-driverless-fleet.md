# 优步为无人驾驶车队订购多达 24，000 辆沃尔沃 XC90s

据彭博新闻报道，优步已经与汽车制造商沃尔沃达成协议，将在 2019 年至 2021 年间购买 24000 辆 XC90 SUVs，以组建一支自动驾驶汽车车队。XC90 是优步最新一代自动驾驶测试车的基础，该车具有传感器和自动驾驶计算能力，由优步在购买 XC90 车辆后安装。

据《金融时报》报道，这笔交易据说价值[约 14 亿美元](https://web.archive.org/web/20221205234345/https://www.ft.com/content/36f071b0-cd64-11e7-b781-794ce08b24dc)，根据基本型消费者定价，XC90 在美国的起价为 46，900 美元。优步已经在亚利桑那州、旧金山和匹兹堡测试 XC90，车上有安全驾驶员，帮助完善和改进他们的软件。优步还与沃尔沃合作，共同开发自动驾驶和一款可用于自动驾驶的汽车，双方去年都承诺进行投资。

优步新的 XC90s 车队将比现有的测试车辆走得更远，因为它们将包含冗余的制动和转向系统，这将允许它们在没有人类安全驾驶员的情况下运行。2.4 万辆车型也可能会根据优步的需求有所变化。作为交易的一部分，优步还将有权从其他原始设备制造商那里订购车辆，以帮助其车队。

无人驾驶的竞争对手 Alphabet 的自动驾驶汽车部门 Waymo 最近宣布，计划很快向消费者开放自己的自动驾驶打车服务，这可能是优步加速其计划的动力之一。不过，两家公司都没有给出日常用户何时能够以非测试身份访问这些服务的时间表。

一位优步发言人证实，24，000 的数字提供了一个总体框架，但并不代表实际确认的汽车数量将被购买。
