# 史蒂夫·沃兹尼亚克讨论他的首届硅谷动漫展

上周，我和苹果联合创始人史蒂夫·沃兹尼亚克坐在一起，听他讲述他的最新创作，硅谷动漫展。

这项活动将于本周末在圣荷西会议中心举行，届时将有迈克尔·J·福克斯、威廉·夏特纳和斯坦·李等名人出席。沃兹和他的团队预计整个周末将有近 30，000 人参加。

大会还将展示一些技术元素，如 Oculus 展示区和应用初创公司可以展示其技术的区域。
