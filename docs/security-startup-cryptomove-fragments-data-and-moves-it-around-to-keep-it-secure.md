# 安全初创公司 CryptoMove 将数据分段并四处移动，以确保数据安全

[CryptoMove](https://web.archive.org/web/20221225200100/https://www.cryptomove.com/) ，一家今天从 stealth 崛起的新安全初创公司，对数据安全有着不同的看法。而不是简单的加密，监控或者记录；早期创业公司将数据分成碎片，并不断移动，这使得黑客几乎不可能对数据做任何事情，如果他们得到其中的一部分。

该公司今天迎来了重要的一天，它带着第一款测试版产品悄然亮相，从[炼金术士加速器](https://web.archive.org/web/20221225200100/http://alchemistaccelerator.com/)创业孵化器毕业，出席炼金术士演示日，并正式宣布获得 150 万美元的种子阶段融资，其中包括蒂姆·德雷珀的[德雷珀联合风险投资基金](https://web.archive.org/web/20221225200100/http://www.draper.vc/)。

这家由父子创始团队运营的初创公司使用了他们开发的所谓“积极防御”方法。它的工作方式是将数据分解成块，不断地移动它们，然后在有人需要时将数据放回一起。他们声称恢复没有任何明显的延迟，并且他们可以通过制作数据片段的副本来确保数据可以安全地恢复，因此如果一个节点出现故障，数据仍然可以恢复。

公司首席执行官 Mike Burshteyn 表示，虽然已经有工具在文件级别实现了这种安全性，但他的公司与这种方法的区别在于，他们在基础架构级别安装了他们的解决方案，让您可以保护数据，无论数据位于何处。更重要的是，这些碎片在不断地变异、移动和变化。他解释说，即使你能收集到碎片，碎片不断变化的性质也构成了另一个黑客屏障。

Burshteyn 说:“公司进行数据碎片化有很长的历史，但没有人(像我们一样)持续移动碎片化的数据。”

实际上是他的父亲鲍里斯·伯什泰恩开发了这项技术。Mike 曾在 Perkins Coie 担任网络犯罪和数据保护律师，他和他的父亲一起帮助 Boris 开发了这项技术，并将其转化为他们今天宣布的测试产品。

这个团队支撑了公司几年。今天，该公司有 5 名员工，包括创始团队。他们计划用种子资金将产品推向市场。

网络安全就像一场象棋比赛，正如我们在过去几年里从民主党全国委员会、T2、索尼、T4、OPM 和摩根大通所看到的高调数据泄露事件中看到的那样，黑客们一次又一次地找到了突破当前防御的方法。

需要时间来证明 CryptoMove 的方法是否可以大规模使用，但如果可以，Mike 相信这可能是积极防御趋势的开始。他和他的投资者当然希望如此。
