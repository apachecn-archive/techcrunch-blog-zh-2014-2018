前奥巴马竞选团队，Threadless CTO 着眼于移动商务与新的创业谦虚

哈珀·里德(Harper Reed)在 2012 年作为首席技术官领导了奥巴马的连任竞选，现在他又回到了创业游戏中。

他专注于移动商务，让小型零售商可以更容易地开发免费的移动应用程序或在智能手机上创造美好的购物体验。在谷歌执行董事长埃里克·施密特(Eric Schmidt)、Greylock Partners、500 Startups 和 Base Ventures 的支持下，这家新初创公司被命名为谦虚。该公司有几位来自奥巴马连任竞选的工程师，以及里德和首席技术官迪伦·理查德，他们曾在 Threadless 工作，并创建了& Barrel。

“我的整个职业生涯都围绕着商业，”里德说。“奥巴马竞选团队因在网上筹集大量资金而闻名。我的问题是，你如何通过手机和电子邮件更好地进行转化。”

基本上，谦虚帮助小企业以移动应用程序的形式快速创建商店。有一键购买，定价包括税和运费，轻松编辑订单，这样买家就不必联系客户服务和保存的信息，这样客户就不必每次都重新输入他们的信息。

“这个产品是关于一个我们可以很好解决的问题，”他说。“我们只是专注于这一小块移动业务，即原生移动交易。”

sm 谦的产品不是 Shopify 或 Magento 的竞争对手，这两家公司也帮助零售商将商店搬到网上。它集成了两者。Android、ApplePay 和 PayPal 支持即将推出。

该公司以免费增值模式运营。这是免费的，或者你可以每月支付 200 美元来购买推送通知、电子邮件购买、提供促销代码的能力以及没有 Modest 的品牌。
