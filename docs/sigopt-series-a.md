# 优化初创公司 SigOpt 融资 660 万美元

SigOpt 是一家旨在“[优化一切](https://web.archive.org/web/20221209130132/https://beta.techcrunch.com/2015/02/12/sigopt-launch/)的公司，今天宣布它已经筹集了 660 万美元的首轮融资。

当初创公司谈论优化时，他们通常意味着尝试一些不同的变化，然后选择获得最佳结果的变化。对于 SigOpt，这更像是一个持续的过程，技术可以查看您的结果，并建议接下来测试什么。

例如，SigOpt 可能会测试一组广告，然后基于这些初始结果，使用相同的创意资产来创建新的版本进行实验。

这不仅有助于改善网站和广告，也有助于实体产品 SigOpt 的首批客户之一使用该技术测试剃须膏的不同化学组合。

“对我们来说，定义完美客户的东西是生产中影响商业价值的某种复杂模型或流程，”联合创始人兼首席执行官斯科特·克拉克说。“在这种情况下，我们可以微调该模型或流程，以增加从中产生的业务价值。不一定需要一个专家数据科学家团队，他们只需要能够调整它。”

SigOpt 的许多早期采用者来自包装消费品和金融服务行业。克拉克说，随着时间的推移，该计划将提供更多的自助产品。

首轮融资由安德森·霍洛维茨基金牵头，[基金还牵头了 SigOpt 的种子轮](https://web.archive.org/web/20221209130132/https://beta.techcrunch.com/2015/06/16/sigopt-seed-funding/)。另一位之前的投资者 Data Collective 参与了新的融资，SV Angel、斯坦福大学和 Blumberg Capital 也参与了融资。
