# Twitter 收购了一家帮助广告商与社交媒体名人合作的初创公司 Niche

Twitter 刚刚[宣布](https://web.archive.org/web/20230207125734/https://blog.twitter.com/2015/welcoming-niche-to-the-twitter-team)已经收购了社交营销初创公司[利基](https://web.archive.org/web/20230207125734/http://www.niche.co/)。

由 Rob Fishman 和 Darren Lachtman 创建的 Niche 帮助广告商在 Vine 和其他平台上与有影响力的人合作。(随机连接:拉克特曼是 TechCrunch 作家乔希·孔斯蒂内(Josh Constine)的表弟。)它标榜自己是一个营销人员可以“在一个地方找到世界上所有的创造者”的地方

我第一次写这家公司是在 2013 年，[公司进行了一轮小规模种子投资](https://web.archive.org/web/20230207125734/https://techcrunch.com/2013/11/22/niche-funding/)，投资者包括大卫·蒂施(Box Group)、迈克尔·卡桑(MediaLink)、克里斯·阿尔特切克(PolicyMic)、布莱恩·戈德堡(Bleacher Report/Bustle)和约翰·奥尔德曼(John Alderman)。随后[筹集了由 SoftTech 领投的 250 万美元 A 轮融资。](https://web.archive.org/web/20230207125734/http://fortune.com/2014/05/15/niche-raises-2-5-million-to-help-internet-celebrities-get-paid/?utm_content=buffer25e9f&utm_medium=social&utm_source=twitter.com&utm_campaign=buffer/)

我还采访了菲什曼和拉赫曼，以及 Vine star Robby Ayala(她管理利基的战略合作伙伴)，今年夏天[在 AOL 的舞台上](https://web.archive.org/web/20230207125734/https://techcrunch.com/2014/08/24/niche-aol-build/)，菲什曼解释说，该公司“不是一个人才机构”，而是一个技术提供商，“在一个活动接一个活动的基础上”与创作者联系

Niche 向社交媒体创作者提供免费分析，以便他们可以看到什么是有效的，什么是无效的，同时允许品牌浏览排行榜，并与顶级创作者合作开展营销活动。

Twitter 表示，它将很快分享关于两家公司整合的更多细节，与此同时，它指出:

> 在过去的一年里，Niche 一直是 Twitter 和 Vine 的重要合作伙伴。该公司通过开发领先的技术，包括免费的跨平台分析，以及将创意社区与世界上最大的品牌直接联系起来，帮助推动了创作者经济。

小生[说](https://web.archive.org/web/20230207125734/http://blog.niche.co/post/110747993803/twitter-is-acquiring-niche)现在和超过 6000 个创作者合作。收购后，它计划“继续我们对创作者的承诺:更多的工具，更多的平台，甚至更多的机会与世界上最好的广告商合作。”

财务条款没有透露，但是 [Recode 报道](https://web.archive.org/web/20230207125734/http://recode.net/2015/02/11/twitter-buys-niche-a-social-media-talent-agency/)价格是“至少 3000 万美元”

**更新:**我们从一位了解财务状况的消息人士处获悉，如果加上保留方案，价格可能会达到 5000 万美元。
