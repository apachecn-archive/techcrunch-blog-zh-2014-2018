# 脸书不顾抱怨自动播放声音的 7 个原因

如果你对脸书开始自动播放带有默认声音的视频感到不安，至少它不会暂停或播放你已经通过 Spotify 等应用程序听到的音乐。脸书今天向 TechCrunch 证实，如果你已经在你的设备上听了一些东西，声音将不会在脸书的视频上自动播放。你必须点击声音图标来打开音频，或者点击以全屏显示视频，这两种操作都会暂停你的其他音乐应用程序。

一些用户肯定会对脸书[昨天宣布的自动播放音频的变化感到恼火。这可能会导致他们在公共场合无意中发出手机声音。这可能会导致尴尬的局面，用户在工作、上课、吃饭或谈话时试图偷偷浏览脸书被抓。幸运的是，那些讨厌它的人可以在他们的脸书设置中关闭自动播放声音。](https://web.archive.org/web/20221209125424/http://newsroom.fb.com/news/2017/02/new-ways-to-watch-facebook-video/)

但是，围绕一个中心主题，脸书想要这样做有很多原因:视频是脸书的未来，它将扫清让这些内容更具观赏性的任何障碍。

1.  **一个简单的开关**——脸书本质上是将音频控制推迟到设备的物理静音开关，这在某种程度上比必须正确配置每个视频的开关和开关更容易混淆音频。一个你不能大声说话的地方？让你的整个手机处于振动状态。
2.  测试人员喜欢它–脸书说最初的测试收到了积极的反馈。如果默认打开音频会减少应用的使用，那就不值得了，所以这个改变听起来可能比实际体验更糟糕。
3.  **Snapchat 正在做**——脸书最大的竞争对手 Snapchat 已经默认开启声音，这使得它成为人们观看视频和企业购买视频广告的最受欢迎的地方。
4.  **视频广告客户**——说到这个，音频对于推动广告的情感反应至关重要。Snap 表示，超过 60%的视频广告是开着声音观看的。视频广告正成为脸书最大的赚钱工具之一，由于今年广告量已经达到极限，没有空间再塞进去，它需要让每个视频广告更令人难忘。
5.  减少对字幕的需求——专业出版商现在经常在他们所有的脸书视频上加上花哨、风格化的字幕，让他们在没有声音的情况下也能理解。这可能会分散视觉效果，但也不是正常人可以对他们的剪辑做的事情。用户制作的视频因此成为二等公民，与脸书把“朋友和家人放在第一位”的目标相冲突
6.  **视频配乐的潜力**——脸书在[与唱片公司](https://web.archive.org/web/20221209125424/https://beta.techcrunch.com/2017/02/13/facebook-soundtracks/)的谈判中更加努力，以达成一项许可协议，允许用户将有版权的音乐作为配乐添加到他们的视频中。这将防止恼人的侵犯版权行为，并使无聊的手机播放片段更像史诗音乐视频。
7.  **适应无线耳塞**——苹果的 AirPods 很棒，而且[你可以一整天都戴着的无线耳塞](https://web.archive.org/web/20221209125424/https://beta.techcrunch.com/2016/09/08/snackable-audio/)将继续流行。这将让更多的人可以开着声音观看脸书的视频，甚至是在公共场合。

会不会有部分用户被气死？当然可以。但是，脸书是在打一个精心计算的赌，认为这些好处会超过少数直言不讳的人的抱怨。请记住，在新闻订阅成为脸书最受欢迎的产品之前，人们就对它的推出提出了抗议。
