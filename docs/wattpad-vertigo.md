# DC 的《迷魂记》成为 Wattpad 上的第一家漫画出版商

漫画迷们请注意:社交阅读应用 [Wattpad](https://web.archive.org/web/20221209093020/http://wattpad.com/) 为你提供了一种与你最喜欢的出版商和作家联系的新方式。

嗯，最初是一个出版商和四个作家——出版商是[眩晕](https://web.archive.org/web/20221209093020/https://www.wattpad.com/user/VertigoComics)，它已经招募了*幸存者俱乐部*作家[劳伦·布克斯](https://web.archive.org/web/20221209093020/https://www.wattpad.com/user/LaurenBeukes)和[戴尔·哈尔沃森](https://web.archive.org/web/20221209093020/https://www.wattpad.com/user/DaleHalvorsen)今天发布他们的简介，并分享他们最喜欢的[恐怖电影](https://web.archive.org/web/20221209093020/https://www.wattpad.com/169452887-my-favorite-horror-movies)的[列表](https://web.archive.org/web/20221209093020/https://www.wattpad.com/169453784-my-favorite-horror-movies-the-best-of-horror)。盖尔·西蒙和霍利·布莱克预计也将在未来几个月加入。

《迷魂记》因《睡魔》、《传道者》、《寓言》、《迷魂记》等系列而闻名，是 DC 娱乐公司针对成人读者的原创作品。(这是一个有点过渡期，新的编辑领导和[一系列雄心勃勃的新标题](https://web.archive.org/web/20221209093020/http://comicsalliance.com/vertigo-new-titles-2015/)。)

在 Vertigo，作家往往与角色或概念一样具有巨大的吸引力，因此作为 Wattpad 上的第一家漫画出版商是有意义的，这一切都是为了连接读者和作家。但是，如果推出进展顺利，我们可能会看到更多的漫画出现在网站上。
