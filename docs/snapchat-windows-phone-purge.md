# 微软从 Windows Phone 商店移除所有第三方 Snapchat 应用

[Snapchat 在 11 月](https://web.archive.org/web/20230326025955/https://techcrunch.com/2014/11/11/snapchat-wants-its-users-who-still-use-third-party-apps-to-give-them-up-already/)确认将打击所有第三方应用，现在斧头落在了 Windows Phone 应用商店上，正如 [Windows Central 报道](https://web.archive.org/web/20230326025955/http://www.windowscentral.com/6snap-removed-store-along-all-third-party-snapchat-windows-phone-apps)的那样，该商店不再包含任何第三方应用。

这家美国公司没有 Windows Phone 的官方应用程序，因此这意味着任何拥有运行微软操作系统的手机的人都不能下载 Snapchat 兼容的应用程序。那些已经拥有一个应用程序的用户，比如最受欢迎的 6snap，也因为使用第三方应用程序而被锁定了账户。这是 Snapchat 上个月承诺实施的另一项安全措施。

[开发 6snap 的 Rudy Huyn](https://web.archive.org/web/20230326025955/https://twitter.com/RudyHuyn/status/546281229637914624) 在推特上表示，Snapchat 已经指示微软移除这些应用，但我们联系了两家公司以证实这一点。

更新:Snapchat 的一名代表证实这些应用已经下架，但指出这一举动影响了“个位数百分比”的用户群。

我还要指出的是，这影响了 Snapchat 社区的个位数百分比。

Huyn 表示，他将“现在与微软密切合作，说服 Snapchat 改变主意”，但美国公司对第三方应用程序采取零容忍态度，因为[两家第三方应用程序被指责](https://web.archive.org/web/20230326025955/https://techcrunch.com/2014/10/13/snapsaved-takes-responsibility-for-latest-snapchat-leak/)泄露了数百万用户的 id 和密码。

然而，对第三方的禁令可能不会持续太久。[我们知道 Snapchat 正在努力构建一个移动消息平台](https://web.archive.org/web/20230326025955/https://techcrunch.com/2014/12/16/snapchat-emails-not-so-ephemeral/)，这可能意味着它最终会发布一个官方 API，允许第三方连接到它的服务，同时为用户隐私提供更多保护。

也就是说，[该公司最近推出了视频广告](https://web.archive.org/web/20230326025955/https://techcrunch.com/2014/10/18/snapchat-ads-are-probably-cooler-than-facebooks-but-still-not-cool/),因此保持其服务在所有设备上不受第三方应用程序的影响，使其能够绝对控制向用户群投放广告，从而可能最大限度地提高其对广告商的价值。

对于 Windows Phone 上的 Snapchat 粉丝来说，这些都没有多大意义，他们从来没有过官方应用，现在甚至没有非官方应用。这就是在今天 iOS-Android 主导的世界中成为“第三”平台的问题。
