沃尔玛的简单文本可能是沃尔玛版的魔术

关于黑客马拉松的一件非常酷的事情是，你知道人们真正使用的是什么类型的技术。我们都想知道这些事情中的一个会有什么热门“趋势”。在 Disrupt 黑客马拉松上，我们学到了很多。一个早期的趋势是短信。惊喜！

还有魔术，脸书 M ，符还有现在的沃尔玛简文。后者是你所期待的——在沃尔玛购物的助手，沃尔玛是一家大得离谱的商店，有成吨的商品。你只想知道他们有没有你想要的糖果，在哪个通道。这是可以预见的，因为所有沃尔玛商店的布局都差不多。

多亏了 Twilio ，你只需发短信“嗨”到 415-214-8445，就可以询问你感兴趣的商品了:

需要比助手所能给予的更多的帮助？发送“聊天”文本，您将被连接到一个实时客户支持代理，以回答更多细微的问题。借助沃尔玛商店的地图，你还可以看到你要去的通道。

我和这个团队聊了聊，这个团队由黑客克拉拉和瑞秋曹以及他们高中的老朋友德文·芬泽组成。

Rachel 告诉我,“我们开发了这个应用程序，因为我们总是对商店感到失望，尤其是沃尔玛。这是一个如此巨大的商店。”克拉拉补充道:“我想买热奇多，然后马上离开。”

一样。

关于为什么短信对于个人助理来说是一种有趣的媒介，而不是沉重的本地应用程序，克拉拉说:“大型商店的人口统计是我们的灵感，大多数用户不知道如何下载应用程序，或者他们不知道如何找到应用程序。此外，要求他们使用他们的数据计划是其中的一大部分。我们真的想要一些东西来帮助他们找到他们需要的东西，这将驱使他们使用该应用程序。”

我感到震惊的是，沃尔玛实验室没有测试像这样的东西，但如果这三个公司继续努力，用人工智能让它变得更聪明，并利用沃尔玛 API 的更多部分，它实际上可以变成可持续的东西。Devin 告诉我“(短信)这是一个强大的界面，是人们越来越习惯的东西。这是应用程序的入口。”

或许是沃尔玛官方应用的入口？
