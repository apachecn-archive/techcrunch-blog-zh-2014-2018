# 这个独特的真空萃取咖啡机是哥伦比亚自己的 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/11/this-unique-vacuum-extraction-coffee-maker-is-colombias-own/>

如果你今天早上喝了一杯美味的咖啡，那么这些咖啡豆很有可能来自哥伦比亚，这个国家几个世纪以来一直以种植和销售咖啡豆而闻名。但是这个国家还没有生产出任何咖啡机——至少到目前为止。FrankOne 是一个聪明的设备，它将多功能真空抽取技术放在一个紧凑的单杯外形中。

当然，[它必须首先达到它的 Kickstarter 目标](https://web.archive.org/web/20221025222754/https://www.kickstarter.com/projects/edumana/frankone-the-first-one-touch-specialty-coffee-brew)。FrankOne 的设计者 Eduardo Umañ解释说，有一天他和一位哥伦比亚烘焙师聊天时产生了这个想法。

乌玛娜在一封电子邮件中告诉我，“他当时正在用‘反向法式压榨’的方法煮咖啡，我觉得我可以改进这种方法。”。“过了一段时间，我非常好奇，想试试高真空煮出的咖啡是什么味道。我做了一些简单的实验，对浓郁香甜的味道感到非常惊讶。一件事接着一件事，我最终设计了一款新产品。”

在操作上，FrankOne 最接近你可能在高档咖啡店看到的大型玻璃真空滴头。[这种有趣而古老的方法](https://web.archive.org/web/20221025222754/https://en.wikipedia.org/wiki/Vacuum_coffee_maker)利用沸腾产生的气压迫使水通过一根管子进入咖啡渣，然后当水冷却时，冲泡好的咖啡通过气压的变化被拉回过滤器。你看到的虹吸管很优雅，但不太方便。

“他们以非常不同的方式实施相似的原则，”乌玛尼亚说。他的设备不需要这种忽冷忽热的舞蹈；你把这个圆柱形的装置，大约“一个大汉堡”的大小，放在附带的玻璃瓶上，然后加入咖啡粉。倒入热水，盖上盖子，等一会儿，让油之类的东西提取出来。然后点击顶部的按钮，让它做自己的事情。

一个泵从玻璃瓶中抽出空气，通过金属过滤器将咖啡吸出。大约一分钟后，这个过程就完成了，剩下的就是 Umañ所说的苦克莉玛。结果就是一杯香甜清澈的咖啡。它的味道(我没有尝试过)可能最接近 AeroPress，而不是 drip，因为主动力推动(实际上是拉动)水通过地面。

该设备可以接受各种研磨和数量的咖啡，制作不同的杯子——这在法式压榨中是不可能的，也不建议用 pourover 或 espresso。它肯定比气动压力机小得多。

它确实靠电池运行，但每次充电 150 杯，你可能不必太担心它。而且没有定制过滤器之类的诱饵和开关——你只需清洗它，它就可以再次运行。

顺便说一句，我不得不再次向 Umañ确认没有哥伦比亚人发明过咖啡机。这个行业在那里是如此古老和重要，以至于这似乎是不可能的。

“难以置信吧？”他写道。“作为哥伦比亚人，为第一个哥伦比亚设计的咖啡冲泡机工作对我来说也非常有意义。也许通过这个项目，我们可以通过咖啡创新为这个国家带来一些急需的经济发展；它就长在我们的后院，我们可以比现在做得更多！”

Umañ和他的公司 Frank de Paula(以哥伦比亚第二任总统的名字命名，他在 19 世纪开始在哥伦比亚做咖啡出口生意)正在寻找 12 万美元来资助这个设备。《早起的鸟儿》售价 50 美元，很可能是个便宜货，至少如果你像我一样是个咖啡爱好者——我就收集这样的东西。每个人都需要一个爱好，对吗？当它以零售的形式出现时，它会花费一个账单，所以如果你喜欢这个想法，为你自己省下几块钱，并承诺支持好的设计。
