# Tiles 是一个易于使用的 3D 设计程序，可以让您打印自己的对象

Cadigo 的 Tiles 是一个面向儿童的 3D 建模程序，它让孩子们将块、墙和像素放入环境中，然后输出对象进行 3D 打印。设计非常简单，你基本上在 2D 创建一个形状，然后将其“挤压”成 3D，并在构建时添加功能。

该软件的早期“创始人”版售价 25 美元，提供免费更新、更大的画布和更多区块选项。较便宜的版本会缩小您可以创建的对象的大小和形状。该公司正在寻求筹集 3 万美元，如果他们达到目标，将在 PC、Mac、Linux、iOS 和 Android 上提供该软件。

要创建物体，你只需在虚拟空间中放置乐高式积木。一旦你建立了你的模型，你可以导出到一个 STL 文件，在 [Makerbot](https://web.archive.org/web/20221006234557/https://beta.techcrunch.com/tag/Makerbot) 或类似的打印机上打印。针对非常年轻的人，Tiles 提供了许多建筑风格，包括通过丢弃砖块进行添加建筑，以及让您从更大的砖块中移除材料的“芯片方式”。不像像 [Printcraft](https://web.archive.org/web/20221006234557/http://www.printcraft.org/) 这样的工具，它将游戏世界翻译成 STL 文件，这是一个真正的 CAD 工具，尽管它的复杂性大大降低了。

Tiles 由 3D 模型市场 [Cadigo](https://web.archive.org/web/20221006234557/http://www.cadigo.com/) 创建，是该公司的第一个软件产品。他们还允许用户上传对象到他们的图书馆，这样你就可以与世界分享你的块状创作。像 TinkerCAD 一样，Tiles 让非常没有经验的人尝试 3D 建模，一旦他们变得很好，就可以升级到更复杂的工具。如果有一件事阻碍了普通的 3D 爱好者，那就是对 3D 设计和瓷砖的理解正在尽最大努力弥补这一点。
