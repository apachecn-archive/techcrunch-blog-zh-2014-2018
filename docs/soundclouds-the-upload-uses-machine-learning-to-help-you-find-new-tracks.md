# SoundCloud 的“上传”使用机器学习来帮助你找到新曲目 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/05/03/soundclouds-the-upload-uses-machine-learning-to-help-you-find-new-tracks/>

SoundCloud 在音乐流媒体服务领域声名鹊起，因为创作者自己上传音乐和其他录音——在这个过程中获得了“音频的 YouTube”的绰号。现在，该公司推出了一个新的探索功能，该功能基于这一想法，并突出了正在增加的内容。这种所谓的上传将向你推荐最近几天上传到 SoundCloud 的新音乐，它的建议基于你过去在 SoundCloud 上听过的音乐，并使用机器学习来确定你接下来可能喜欢听的音乐。

SoundCloud 在宣布新服务的[博客文章](https://web.archive.org/web/20230130011329/https://blog.soundcloud.com/2017/05/03/introducing-upload-new-soundcloud-selected/)中指出，你可以通过在网上的“发现”标签导航，或者通过在 SoundCloud 的 iOS 和 Android 应用上搜索来获得新功能。

上传是 SoundCloud 一年前推出的另一项探索功能的扩展，名为[建议曲目](https://web.archive.org/web/20230130011329/https://techcrunch.com/2016/06/22/soundcloud-improves-music-discovery-with-new-suggested-tracks-feature/)，它也使用了类似的算法，但涉及了 SoundCloud 更广泛的目录。如果建议曲目就像《发现周刊》,上传就像 Spotify 的[发布雷达](https://web.archive.org/web/20230130011329/https://techcrunch.com/2016/08/05/spotify-release-radar/),同样旨在向用户建议新的发布。

发现和推荐工具——也被 Apple Music 等其他平台使用——帮助用户了解在大型开放式音乐平台上听什么，是这些流媒体服务如何扩大用户群和参与度的核心部分。

他们尤其重要，因为他们试图超越早期音乐爱好者的核心听众。更多漫不经心的听众可能会被有数百万首歌曲可供他们听的声音所吸引，但当真正要弄清楚播放什么时，他们可能会沉默不语。

对于 SoundCloud 来说尤其如此，该公司为其高级 Go+用户(1.2 亿免费和 Go 用户)宣传了超过 1.5 亿首音乐曲目，以及一个更加开放的分类，在最流行的情况下，同一首歌的多个版本由不同的相同的人制作。(相比之下，Spotify 有大约 3000 万首歌曲。)

我们已经问过了，但是 SoundCloud 没有透露，只是说每周有多少首歌曲被上传。

然而，鉴于 SoundCloud 实际上比其他许多服务拥有更多的歌曲，我有点惊讶它没有做出更多努力来推出更多这样的发现服务。除了上传和建议曲目，SoundCloud 上的其他机器学习服务包括关注谁、电台(曲目和艺术家电台)和图表。

更普遍地说，SoundCloud 目前正处于一个有点困难的时期。该公司表示，它拥有 1.75 亿用户，虽然它试图将更多用户转化为付费用户(最便宜的 SoundCloud Go 起价为 4.99 美元)，但它也试图扩大用户群，以推动其广告模式，并继续使该平台成为创作者展示其作品的有吸引力的地方。

该公司不时成为 Spotify、谷歌和 Twitter 等公司的收购目标，但据我们所知，在它能够以投资者对其的估值(上一轮股权融资为[7 亿美元](https://web.archive.org/web/20230130011329/https://techcrunch.com/2014/01/25/soundcloud-raises-60-million-at-700-million-valuation/)，尽管那是在 2014 年)或更高的价格达成交易之前，它需要做好准备。

据我们所知，SoundCloud 仍在试图完成新一轮股权融资，因此获得更多服务来提高参与度可能也有助于为此做好准备。与此同时，SoundCloud 一直在安排信用额度——最近一次是在 3 月份宣布的，金额为[7000 万美元。](https://web.archive.org/web/20230130011329/https://techcrunch.com/2017/03/23/soundcloud-confirms-new-70m-credit-line-after-failing-to-close-100m-round/)
