男性发起变革是根除科技兄弟文化的重要一步

原文：https://web.archive.org/web/https://techcrunch.com/2014/08/09/men-initiating-change-an-important-step-to-amending-techs-bro-culture/?ncid=rss

泰勒·惠特尼撰稿人

编者按: 泰勒·惠特尼是安妮塔·博格研究所的总裁兼首席执行官，也是格蕾丝·赫柏计算机界女性庆典的联合创始人。联系、激励和指导女性技术专家是她的激情所在。

随着科技公司开始对自己负责，关于科技行业女性的话题正在发生转变。最近的一些举措，如谷歌、脸书、领英和雅虎发布员工多样性数据，显示出为女性技术人员做出真正改变的强烈承诺，但这种情绪并不是全行业的。

我们经常听到技术领导层表示，他们希望在技术岗位上雇佣更多女性，但他们继续将 STEM 管道中缺乏女性作为原因。推迟问责不会影响真正的系统性变革。

我们陷入了先有鸡还是先有蛋的僵局。事实是，初创公司闻名的科技程序员文化是女性不愿从事该行业的一个主要原因。

在 HBO 的《硅谷》(Silicon Valley)中，一个经常击中离家太近而被认为是讽刺的节目——一个流行的笑话是应用程序 Nip Alert，一个角色创造的帮助定位大胸女性的应用程序。当然，这个应用程序是基于科技行业的真实事件，这些事件每天都会出现，但往往会被掩盖。

当技术会议中包含性别歧视或揭露初创公司创始人的厌女行为变得司空见惯时，这是初创公司兄弟会文化失控的危险信号。面对最近的这些事件，科技行业的“男孩就是男孩”心态正在对 STEM 管道的多样性产生影响。

面对来自科技行业的所有负面报道，难怪年轻女性不会蜂拥成为下一个目标。聪明的年轻女性通常对加入兄弟会环境不感兴趣，这种环境以消极的方式塑造她们，并经常贬低她们。人们认为缺乏在科技职业中发展的机会，这可能会让年轻女性无法进入计算机科学和工程领域。

有一种方法可以实现文化变革，它从技术领域的人开始。女性将继续发出声音，推动变革，但同样重要的是，男性也要站出来说话。这包括各级技术人员、领导层的男性，尤其是风险投资公司的男性。

初创公司以科技博客文化而闻名，这是女性不愿从事该行业的主要原因。

就目前的情况而言，最近科技领域的许多争议遭到了风险投资公司的沉默和不作为。当朱莉·安·霍瓦特谈到她在 GitHub 面临的 T2 骚扰时，该公司风投公司唯一的回应是马克·安德森支持辞职创始人的 T4 孤独推特。没有向霍瓦特道歉，也没有表示支持改善妇女的工作环境。同样，当 RadiumOne 、古尔巴斯克·查哈尔CEO 因家庭暴力和殴打女友两项轻罪被定罪时，RadiumOne 董事会旷日持久的不作为只导致他在媒体风暴太大时解雇。

即使法律没有被违反，女性的缺席也没有成为风投的一个问题，比如在 2013 年首次公开募股前，推特董事会中缺少女性。面对此类事件保持沉默，尤其是那些掌握着财权的人，向考虑从事该领域职业的女性技术专家和年轻女性传达了一个可怕的信息。一种颂扬男孩天才创始人并鼓励兄弟会环境的文化，尽管有所有的警告信号，但并不是一个大多数聪明的年轻女性认为自己能够成功，并引导她们追求其他职业的地方。

在安妮塔·博格学院，我们经常看到处于领导地位的男性走上前来参与谈话。在过去的 20 年里，我们一直在举办格蕾丝·赫柏计算机界女性庆典，我们经常邀请许多忠诚的董事会成员，包括谷歌的 T2·阿兰·尤斯塔斯、脸书的 T4·迈克·斯科洛普夫、英特尔的贾斯汀·拉特纳和微软的里克·拉希德。今年，我们邀请了微软首席执行官塞特亚·纳德拉作为我们的主题演讲之一。这些人明白，改变科技文化不仅仅是女性的问题，也是一个创新问题。

现在不是自满的时候。科技行业的男性可以采取很多行动来表达他们的支持。最重要的行动是大声反对不平等和不作为，大声支持女性技术专家，尤其是当你看到不适当或居高临下的行动或信息时。男性对女性在科技领域的支持将产生更大的草根效应，从而带来更具包容性和创新性的科技文化，让各方都能茁壮成长。
