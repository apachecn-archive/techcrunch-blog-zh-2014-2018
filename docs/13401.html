隐形独角兽:35家白手起家的大公司

原文：https://web.archive.org/web/https://techcrunch.com/2017/07/01/invisible-unicorns-35-big-companies-that-started-with-little-or-no-money/

约瑟夫·弗莱厄蒂撰稿人

More posts by this contributor

风险投资是一剂猛药，而且有可能过度依赖风险投资，但对于大多数创始人来说，这是一个香槟问题。投资者更经常听到的问题是“我如何让风投支持我的创业公司？”这些创始人并不担心资本过剩会让他们的首次公开募股前景变得更加棘手——他们正争先恐后地让某人，任何人，签署他们的第一份投资意向书。

创业者们普遍认为风险投资是成功的前兆。风险投资是最成功的科技创业公司的共同特征，但不是先决条件，尤其是在早期阶段。

企业家可以用很少甚至没有资本证明自己。资本不会让你的公司有洞察力。如果你不能创造性地把1美元变成10美元，你为什么期望能够把100万美元变成1000万美元？

为了帮助说明初创公司如何前进，这里有35个公司的例子，它们以几千美元起步，甚至只是血汗资本，后来成为我所说的“高效创业”的典范。

这些公司中有许多后来获得了数十亿美元的估值，有些甚至获得了数十亿美元的收入，但没有一家是从被认为是种子期以外的任何事情开始的。这些创业公司中的大多数都从风投那里筹集了资金，但前提是他们必须确定，无论有没有投资者的电汇，他们都会成功。即使是现在，他们中的许多人还不广为人知——他们是科技行业的隐形独角兽。

因此，在匆忙安排与投资者的会面之前，先看看这些故事。它们与许多创始人所持的以风险投资为中心的观点形成了抗衡，并提供了考虑融资的替代方式。

以下是对这些公司的简要描述(按照它们共享的方法分类)和故事链接，在那里你可以读到更多关于它们的信息。记住，接受风险投资应该是一种选择，而不是强迫。这些公司展示了它是如何做到的。

想出办法，然后要钱

如果你能为客户解决一个真正的问题，并为此收费，那么你在大多数行业都不需要风险投资。以下是思考这个问题的三种方式:

自动化您的工作流程

构建有用产品的最简单方法是自动化日常工作流程的某个部分。这将确保你已经得到了对你正在建设的东西的已证实的需求，以及你的项目已经存在的资金来源。

2000年，联合创始人兼首席执行官Ben Chestnut经营着一家设计咨询公司，有一批客户想要电子邮件简讯。唯一的问题是他讨厌设计它们。因此，为了让他的团队免于单调乏味，他决定开发一个工具来简化这个过程。4亿美元的run rate业务MailChimp 诞生了。

琳达:20世纪90年代末，琳达·温曼开始是一名需要工具来指导网页设计师的教师。书店里的商品平淡无奇，所以她开始制作培训影片，以更好地教育她的学生。她的公司帮助软件开发人员和设计人员提高技能。她花了20年时间建立了一个内容库和技术资产，其规模足以吸引LinkedIn支付15亿美元收购该公司。

从资本高效型产品开始

许多企业家对行业领导者进行正面攻击，通常以失败告终。在硬件的情况下尤其如此。这些斗志昂扬的初创公司没有试图与苹果这样的公司竞争，而是填补了RadioShack留下的空白，建立了值得尊重和效仿的业务。

AdaFruit Industries:Limor Fried在麻省理工学院上学时，通过组装由现成零件组成的DIY套件，开始了她的DIY电子商务帝国。弗里德销售的是在电子商店找到的相同的积木，但也制作了古怪的内容，使得焊接一个复制品太空入侵者橱柜的前景看起来合理。现在她有85名员工，年收入3300万美元。

与AdaFruit相似，Nathan Seidle在他的宿舍里开始了SparkFun，向一群想要探索新奇的传感器和系统的工程师出售电子设备和古怪的组件。现在，他的电子商务帝国拥有154名员工，年收入达3200万美元。

解决现有问题并利用现有商业模式

创业公司在商业模式上不一定要特别创新。在更现代的技术平台上构建一个更好的捕鼠器，或者使用UX层，就足够了。随后的公司都没有重新发明轮子，但最终都创造了真正的价值。

Braintree Payments :在网上进行货币交易，而不被诈骗者诈骗，是网络上最古老的问题之一。交易各方都乐意为卓越的体验支付公平的“税”。Braintree建立了一个更好的技术解决方案，并依靠这些交易的收益生存了四年，然后在两轮风险投资中筹集了6900万美元，这是在一次8亿美元的收购之前。

ShopifyShopify:Shopify的创始人在为滑雪爱好者创建电子商务网站时，一直在寻找购物车解决方案。由于找不到，他们决定自己动手，在当时炙手可热的Ruby on Rails框架上构建一个定制的解决方案。事实证明，对于更多的人来说，这是一个完美的解决方案，创始人用他们创造的收入独立经营了六年。他们最终从风投那里筹集了资金，后来又进行了IPO，获得了10亿美元的估值。

自力更生规则

许多企业家把时间浪费在“扮演首席执行官”上，精心制定战略，为他们的企业可能成为的样子绘制一个梦想中的组织结构图。不要那样做。相反，想想你今天能做些什么，只用你现有的资源来推进这个想法。

多亏了像Birchbox这样的先驱，给业余美容师送几盒化妆品已经成为一个增长行业。YouTube明星Michelle Phan没有先发优势，但她利用了她的网络名人(800多万YouTube订户)、与化妆品品牌的关系以及<50万美元的种子资金，建立了一个订阅箱初创公司，在获得1亿美元风投之前创造了1.5亿美元的收入。

资本不会让你的公司有洞察力。

乔恩·奥林格是一名专业软件开发人员和业余摄影师。他综合了这一系列技能，并使用个人照片库中的3万张照片，创建了一个股票照片服务，目前价值20亿美元。他的资本效率得到了回报，并最终使他成为真正白手起家的亿万富翁。

SimpliSafe :人们对尝试启动硬件业务的想法嗤之以鼻，但SimpliSafe的Chad Laurans做到了。他从朋友和家人那里筹集了少量资金，然后花了8年时间建立了一个自安装安全业务，为了省钱，他自己焊接了第一个原型。 八年后 ，该业务拥有数十万客户，收入数亿，获得红杉5700万美元风投。

每个人的钱都是绿色的

资金并不总是一次到位几百万美元。创始人可以从补助金、孵化器和天使投资人，甚至是预售中凑钱。最精明的企业家设计他们的商业模式，以便他们在交付产品之前收取货款，把客户变成增长资本的来源。

艰难的搅局者(Tough Mudder):田径企业家威尔·迪安(Will Dean)将7000美元的积蓄变成了一家年收入超过1亿美元的公司。秘密是预售注册比赛，然后使用这些资金作为营运资金来建设电气化障碍课程，使艰难的Mudder成为一个全球现象。

CoolMiniOrNot:CoolMiniOrNot最初是一个网站，极客们可以在这里展示他们绘制龙雕像的能力。最终，该网站的创始人决定利用Kickstarter作为一个渠道来设计和发行他们自己的游戏。他们已经开展了 27次Kickstarter活动，筹集了35，943，270 百万美元的非摊薄资金。游戏开始。

卖！卖！卖！

通常最好的资金来源是客户，销售有两个好处。首先，你让收银机马上响。其次，你很快就能了解什么能引起顾客的共鸣，并能利用这些见解来完善你的产品。

Scentsy:dnvb很时髦，但它们过度依赖twee发布视频和脸书广告来推动收入。Scentsy在旧货交易会上卖蜡烛，因为他们买不起广告。这并不迷人，但它确实给了创始人一个坚实的基础，让他们的信息引起了买家的共鸣——现在他们的年收入超过了5 . 45亿美元。

CarGurus :这款应用利用数据分析来帮助客户找到二手车的最佳交易，但该公司的首席执行官将其每年5000万美元的收入和盈利能力归功于在公司生命周期的早期雇佣了一个销售团队。该公司350名员工中，有近一半在忙着打销售电话，而不是写软件。

LootCrate :在筹集机构资本之前，LootCrate拥有超过60万名客户和1亿美元的收入。他们如此高效的部分原因是该公司从成立的第一个周末就开始向顾客收费。创始人在一次黑客马拉松上，建立了一个登陆页面，收集订单，并用这些资金购买将装满包裹的极客商品。

吝啬营销

创业营销人员可能不想浪费时间在无法衡量的品牌营销上。高效的企业家需要活动能立即产生附加效果。

Wayfair :这家家居用品电子商务公司在运营的第一个月就实现了盈利，因为他们跳过了品牌广告，买下了数百个与常见搜索词完全匹配的域名。这种模式开启了十年的盈利增长，直到他们最终在上市前不久筹集了价值1.65亿美元的A轮融资，目前市值超过40亿美元。

如果你不能创造性地把1美元变成10美元，你为什么期望能够把100万美元变成1000万美元？

反人类卡片:仅从Kickstarter获得15700美元的资金，反人类卡片团队就建立了一个业务，第一年的总收入超过1200万美元。他们还通过一系列精明的营销噱头来维持自己的品牌，销售牛粪、切割毕加索、挖一个代表后特朗普时代美国厌倦的大洞、然后销售特朗普“bug out”包以及简单地要钱。这些促销活动运营起来并不便宜，但它们赚的钱足以支付成本，同时还能获得不成比例的免费媒体。

当病毒式营销被附加到商业模式中时，它理所当然地被摒弃，但是当它被恰当地整合到商业模式中时，它可以成为一个强大的驱动力。搭配超高效转换率优化(CRO)，它可以是无与伦比的。GoFundMe的创始人能够利用这两股力量将一项业务提升到价值约6亿美元的水平。

效率>资本

初创公司通常以他们筹集了多少资金来衡量。更重要的是要问那些公司利用资本的效率如何。效率并不意味着吝啬，而是要找到那些围绕技术或商业模式经营业务的企业家，这种技术或商业模式本质上更能有效地增加资本。

将莫奈和梅洛结合起来的想法已经有一段时间了，但PaintNite的创始人希望让这种模式更具成本效益。当他们的竞争对手依赖缓慢而昂贵的特许销售模式时，PaintNite将艺术教师与现有的想在工作日出售葡萄酒的酒吧配对，并创建了一项业务，在筹集风险资本的前一年实现了3000万美元的收入。

鱼很多:这个交友网站成立于2003年，在接下来的十年里，在功能和审美方面没有发生巨大变化。其他网站有更多的功能，更华丽的图片和大量的风险投资，但是PoF是免费的，它把大部分资源花在了打击垃圾账户上。和Craigslist一样，许多鱼最大的资产是它作为一个储备丰富的池塘的名声。该公司随着时间的推移不断更新产品，但从来不需要大规模的资本注入。最终，该公司以5.75亿美元的价格售出。

《我的世界》背后的共济会成员从未筹集过任何风险资本，在卖给微软之前，他们只雇佣了50个人，却赚了近10亿美元的利润。这家瑞典工作室从未卷入Zynga启发的社交垃圾邮件和掠夺性微交易等时尚。《我的世界》通过向用户收取固定费用实现了增长，导致了25亿美元的收购。

命运偏爱“无聊”的人

无聊不是一种价值判断。许多最令人印象深刻、最成功的公司在没有资本的情况下成功发展，它们是通过解决尖锐的(尽管有些枯燥)问题而繁荣起来的。如果你解决了一个难题，客户会很乐意资助它。

  • SurveyMonkey 成立于20世纪90年代的网络泡沫时期，虽然它不像Kosmo这样具有破坏性，但它更持久。它挺过了网络泡沫，稳步成长为九位数的运营率，在开始运营11年后仅筹集了1亿美元。
  • Protolabs 为塑料注射成型做的事情和Vistaprint为名片做的事情一样，目前价值12亿美元。
  • Cvent ，价值13亿美元，构建活动管理工具 Textura ，以6.63亿美元收购，负责施工管理——这两个市场通常都不被视为热门或时尚市场。
  • Grasshopper 是一家电话网络公司，拥有15万客户，年收入超过3000万美元，但账面上没有风投，最终被Citrix收购。
  • 史诗由朱迪思·福克纳于1979年创立；总部位于威斯康星州的电子医疗记录提供商可能是当今最大的自举软件公司。
  • eClinicalWorks 成立于1999年，当时的口号是“快速做大”，许多同时代的公司都倒闭了。通过专注于擅长管理临床数据的枯燥但有利可图的工作，该公司幸存下来，现在雇用了4000多名员工，年收入达到3.2亿美元。
  • Unity 通过专注于游戏开发的所有非性感方面，如跨平台兼容性和“凹凸贴图”，成为了移动游戏行业的中坚力量他们几年没有筹集资金，但现在的估值超过15亿美元，比大多数品牌游戏创业公司更成功。
  • GitHub 摆脱了版本控制的痛苦，并在融资前成为科技生态系统的重要组成部分。
  • Qualtrics最初是犹他州一个地下室里为学校和企业管理调查的工具，现在拥有1000名员工，每年获利1亿美元。

无法找到的人有福了

有时筹集资金几乎是不可能的。我们看到营收数千万、增长率达到三位数并拥有其他优势的公司甚至很难筹集到少量资金。幸运的是，尽管存在明显的劣势，但这些初创公司最终往往会胜出。

Atlassian :在硅谷、纽约市、洛杉矶或波士顿之外创业的好处之一是没有多少风险投资。这听起来像是诅咒；毕竟，没有资本怎么可能有帮助呢？这可能是塞翁失马，焉知非福。

这种隔离阻止你做白日梦，想象你会用数百万美元做什么，并迫使你让你的付费客户开心。总部位于澳大利亚的Atlassian一路高歌猛进，市值达到40亿美元。如果它更容易获得资金，他们可能会发现自己在追逐低质量的增长，并在找到如何有效扩大规模之前就破产了。

你不需要获得投资者的许可就可以创建和扩大一家初创公司。

资金效率高的公司有一个奇怪的特点，那就是它们的第一轮融资往往数额惊人，看起来更像是首次公开募股的收益。Campaign Monitor就是这种情况，它的第一轮资金达到了2.5亿美元。总部位于悉尼的Campaign Monitor不容易获得风险投资，所以他们启动了这项业务，并开发了一种独特的技术，为迪士尼、可口可乐和Buzzfeed等公司提供卓越的电子邮件分析。时间会证明筹集2.5亿美元是对公司有益还是有害，但这肯定是对他们迄今为止所取得进展的一种验证。

交易柜台:尽管创始人杰夫·格林对如何推动程序化广告行业有着独特的见解，但他在现代广告技术的融资周期后期才开始设立交易柜台。这种市场资本过剩，加上投资者被表现不佳的投资者所伤害，使得公司生命周期中的每一轮融资都是一场斗争。格林是一位完美的初创公司首席执行官，在公司成立的头六年里，他只筹集了2640万美元的风险资本，并将其转变为在纳斯达克上市的价值10亿美元的企业。怎么会？接受资本减少的限制，专注于回报最高的活动，建立一种由想法而不是资本注入推动的创新文化。(披露:方正集体是交易台的投资者。)

风投并不完美，即使是最优秀的风投也会错过看似确定无疑的想法。令人震惊的是，听到创始人谈论他们如何无法向投资者推销一个后来成为10亿美元企业的想法是多么普遍。AppLovin的创始人Adam Foroughi以14亿美元的价格卖掉了他在T2的公司，但他发现很难筹集到风险资本，即使收入可观。“我找不到任何人以我认为合理的起点估值(也许400万美元或500万美元)给我们投资，到第一年运营结束时，我们开始盈利，每月收入超过100万美元。”其余的，正如他们所说，都是历史了。

要点:避免围绕风险投资设计你的业务

太多的创始人从第一天开始就把他们的企业定位于风险投资。创业公司过去常常想出办法，然后要钱。今天，他们要钱来解决问题。除了药物研发或航空硬件，这通常是错误的决定。事实上，在没有资源的情况下取得进展是让风投对你的公司感兴趣的最好方法。上述公司选择长时间不融资，但当它们融资时，它们可以选择投资者并设定条款。

我们的建议是不要试图永远自举一家企业。从苹果到Zappos，风险投资几乎为每一家大型科技公司提供了动力。只要记住你不需要一分钱就可以开始。你不需要获得投资者的许可就可以创建和扩大一家初创公司。所以下次风投告诉你他们“通过了”，记住这三条原则:

  • 没有资金也可以启动一个科技型企业。
  • 用很少的资金迅速扩大科技企业的规模是可行的。
