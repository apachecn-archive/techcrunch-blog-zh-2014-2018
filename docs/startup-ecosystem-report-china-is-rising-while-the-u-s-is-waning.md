# 创业生态系统报告:中国正在崛起，而美国正在衰落

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/04/20/startup-ecosystem-report-china-is-rising-while-the-u-s-is-waning/>

创业是一场赌博，但通过观察它们所处的生态系统，有可能更好地理解为什么有些公司会兴旺，而更多的公司会消亡。这就是成立八年的初创公司 Genome(T1)的使命，该公司由一群研究人员和企业家组成，他们每年采访世界各地的数千名创始人和投资者，以更好地了解他们经营的地区正在发生什么变化，以及哪些东西顽固地保持不变。

更大的目标是找出如何帮助更多的创业公司取得成功，该机构今年在 [Crunchbase](https://web.archive.org/web/20230316231228/https://www.crunchbase.com/) 和 [Dealroom](https://web.archive.org/web/20230316231228/https://dealroom.co/) 等合作伙伴的帮助下调查了 1 万名创始人，得出了一些或许应该引起美国人关注的数据。也就是说，在众多科技领域，中国看起来有望超越美国的主导地位。想想看:2014 年，只有 14%的所谓独角兽位于中国。从去年年初到今天，这一比例已飙升至 35%，而在美国，本土独角兽的数量已从全球总数的 61%降至 41%。

你可能会说，投资者只是给中国的初创公司过高的估值，就像在美国发生的那样，我们部分相信这是真的。但从专利来看，中国显然也“志在必得”，去年在中国注册的人工智能相关申请和加密及区块链相关专利分别是中国的四倍和三倍。随着如此多的科技行业现在专注于深度技术，这是值得注意的。事实上，尽管[我们厌恶](https://web.archive.org/web/20230316231228/https://techcrunch.com/2018/01/19/moritz-sabotages-sequoia-again/)著名风险投资家迈克尔·莫里茨在《金融时报》一月份的专栏文章中建议[美国公司效仿中国](https://web.archive.org/web/20230316231228/https://www.ft.com/content/42daca9e-facc-11e7-9bfc-052cbba03425)，但他的基本呼吁可能，嗯，以其自身的方式是有先见之明的。

创业公司还应该知道什么？根据初创公司 Genome 的研究结果，除了人工智能、区块链和机器人制造的崛起，还有一些子行业也在明显下滑，其中包括毫不奇怪的 adtech，该公司在过去五年中的资金下降了约 35%。毫无疑问，这与脸书和谷歌日益增长的主导地位直接相关，根据股权研究公司 Pivotal 的数据，这两家公司去年占据了美国所有数字广告的 73%。

该研究的作者指出，这并不意味着 adtech 初创公司已经被煮熟了。相反，衰落的子行业通常是“成熟的”，但可以通过新技术复兴。在这种情况下，尽管对 adtech 的资助有所下降，但虚拟现实和增强现实很可能在某个时候为该行业注入一些新的增长。也许吧。

无论哪种方式，对我们来说，这份报告最有趣的方面——也确实值得仔细研究——是它能够通过与世界各地如此多的人交谈而建立的联系。例如，它阐述了斯德哥尔摩这个相对较小的创业生态系统如何能够以有意义的速度产生规模可观的创业公司，而芝加哥的生态系统表面上要大三倍。(答案是:斯德哥尔摩的初创企业创始人显然与全球七大生态系统有着更好的联系。)

同样非常有趣的是，该报告关于女性创始人的发现，她们与地区创始人建立了更多的关系，并且比男性同事在当地的关系更密切——除了与投资者的关系。这对女性创始人和投资者来说都是坏消息，因为本地人脉与更好的创业表现相关。
