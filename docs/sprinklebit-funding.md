# 社交投资初创公司 SprinkleBit 融资 1000 万美元

在朋友和专家的指导下帮助你投资的初创公司 SprinkleBit 已经从 GTC 获得了 1000 万美元的新资金。

利用大众的智慧在股票市场上做得更好并不是一个新的想法[，但是 SprinkleBit 提供了一个独特的工具，叫做](https://web.archive.org/web/20230129104052/https://techcrunch.com/2015/11/24/steal-business-from-the-rich-banks/)[价值预测指数](https://web.archive.org/web/20230129104052/http://blog.sprinklebit.com/sprinklebit-vpi-explanation-infographic/)，或 VPI——每只股票都根据社区是否认为其价格将会上涨来评分，每个用户都根据他们预测未来价格的准确程度来评分。(VPI 实际上是基于 CEO 亚历山大·沃林在加州大学圣迭戈分校的本科生研究。)

虽然 VPI 可能是 SprinkleBit 方法中最有趣的事情，但这家初创公司建立了一个完整的社交平台，用户可以与 SprinkleBucks 进行无风险交易，并关注其他用户的交易。然后，如果他们想投入实际资金，他们可以切换到 SprinkleBit 的经纪模式，真实地买卖股票。(沃林说，随着社交投资和真实交易的结合，他的团队必须做大量的工作来确保 SprinkleBit 符合 FINRA 的规定。)

还有一个教育部分——该公司提供一套名为 SprinkleBit 大学的在线投资教程。

你可以免费使用 SprinkleBit 的社交端，但它对交易收取 8 美元的佣金。你也可以每月支付 9.95 美元购买 SprinkleBit Premium，这将佣金降至 5 美元，并包括额外的数据和工具。

该服务于 2012 年首次推出，目前拥有约 1 万名用户。沃林说，该公司现在只准备做一个严重的营销推动，因为它终于准备好“开始登上大众。”

至于股市更广泛的动荡可能会如何影响使用，他说，“我认为这是一个好时机，因为当市场行情好时，一切都是贪婪，任何平台都会赚钱……现在，当你必须实际知道哪些股票将上涨时，你需要一个工具(像这样)。”

SprinkleBit 总共筹集了 1350 万美元。它还在将总部从圣地亚哥迁至纽约。
