# 《超级马里奥快跑》现已在 App Store 上架

顾名思义,“超级马里奥快跑”就是向前冲的动力。这是一个简单的物理问题——你无法阻止一个马里奥兄弟，你只能希望用一个放置得当的暂停块来暂时遏制他。因此，胖水管工[在 App Store](https://web.archive.org/web/20221208101259/https://itunes.apple.com/us/app/super-mario-run/id1145275343?mt=8) 中有点操之过急就不足为奇了。

任天堂期待已久的标题定于今天上午 10 点太平洋时间发布，但设法偷偷提前了一点。这款单手操作的平台游戏现在可以在 iPhone 和 iPad 上下载。正如之前宣布的那样，这款游戏价格高达 10 美元，不过任天堂将推出三个级别，免费提供世界末日城堡的预览。

这款游戏标志着该公司的一个关键转变，该公司长期以来一直避开移动平台，转而支持自己的硬件。9 月，马里奥的创造者宫本茂在旧金山的 iPhone 7 发布会上登台，宣布了这个角色的智能手机首款，此前 Niantic 设计的 Pokemon Go 获得了巨大的惊喜成功。

这款游戏提供了一种有趣的经典模式，设计用于在肖像模式下单手玩。你可以看看我们早期的《T2》，标题是《T3》，以及对美国任天堂总裁雷吉·菲尔斯·艾梅的采访。
