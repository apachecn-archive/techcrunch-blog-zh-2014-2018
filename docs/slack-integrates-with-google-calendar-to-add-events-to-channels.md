# Slack 与 Google Calendar 集成，将事件添加到频道中

流行的办公通讯工具 Slack T1 现在将与谷歌日历 T3 整合，让事件自动在 Slack 频道中发布提醒。

将谷歌账户链接到 Slack 后，你可以选择任何日历，并指示它发布到某些 Slack 频道。例如，您可以让您公司的开发截止日期日历中的事件在产品截止日期前两天向#dev 频道发布提醒。

或者，以一种更轻松的方式，您可以让您公司的生日或假日日历自动发布到#general，这样您的整个团队就可以一起庆祝了。

![1 - in Slack screenshot](img/7d5110ad501a0d854bcb0487b978db7b.png)

该集成还将允许每天或每周的日历摘要发送给每个订阅某个 Slack 频道的人。

考虑到许多人已经使用 Slack 来共享日历事件，这种集成只是让 Slack 进一步自动化用户现有的工作流程。诸如此类的好处将使 Slack 对企业更加有用，并帮助 Slack 进一步推动其收入模式(部分基于员工使用该服务的频率)。

Slack 在获取新用户方面如此成功的部分原因是，它提供了与其他企业服务和应用程序的数十种不同集成。考虑到谷歌日历是企业使用的最受欢迎的日历服务之一，这种新的集成可能是考虑 Slack 的公司已经在使用谷歌应用程序的一个主要卖点。

对于那些已经在 Slack 上的人来说，这种整合可能会让他们更加依赖这个平台。Slack 是在与电子邮件正面交锋的前提下开始的。通过在 Slack 中添加日历通知，该公司将能够让更多的用户留在应用程序中，而不是他们的收件箱中。

新功能将于今天上线，并可在您团队的集成页面中启用。
