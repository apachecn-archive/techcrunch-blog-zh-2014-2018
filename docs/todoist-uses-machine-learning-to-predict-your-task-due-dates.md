# Todoist 使用机器学习来预测您的任务截止日期 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/11/16/todoist-uses-machine-learning-to-predict-your-task-due-dates/>

当谈到用待办事项清单来计划你的一天时，如果你有点过于乐观，请举手。如果你像我一样，很有可能你写下了太多的任务，并且总是在一天结束时以未完成的任务结束。流行的任务管理服务 [Todoist](https://web.archive.org/web/20221209210417/https://todoist.com/) 想帮助你重新安排你的任务，并使用机器学习来平衡工作量。

智能日程安排是一项正在进行的工作，随着时间的推移会变得更好，但它听起来像是一个对强烈嗜酒用户有前途的功能。“我们创建了两个神经网络，可以帮助我们的用户进行日程安排和重新安排。我们认为这是人工智能的一种非常酷的用法，”Todoist 创始人兼首席执行官 Amir Salihefendic 告诉我。

任务管理服务是很棒的，直到它们变成了太多的工作。很多服务需要你勾选很多框，输入很多信息，只是为了管理一个待办事项列表。这就是为什么一切可以加速这些服务的东西都是必不可少的。

在 Todoist 中，过期部分旁边有一个新的“重新安排”按钮。该服务基于许多不同的数据点，智能地为所有这些过期任务建议新的到期日期。它也适用于未计划的任务。

首先，Todoist 向你学习。对于常规任务，比如跑腿，Todoist 会记住你通常什么时候完成这些任务，并把它们分配到同一天。你也可能有一大堆即将到来的任务，所以 Todoist 会确保所有的任务被平均分配，这样你就可以真正地完成任务。

Todoist 也知道你什么时候停止完成任务。因此，如果已经晚了，服务不会建议将到期任务重新安排到今天。

第二，Todoist 也从整个 Todoist 用户群中学习。许多用户写下相同的任务。“修复关键错误”可能是一个常见的错误，服务知道应该尽快完成，而不是下周。

最后，Todoist 试图区分工作日和周末，因为人们倾向于根据环境完成不同的任务。

用户可以随时编辑或拒绝建议的截止日期，这样您就可以控制自己的待办事项列表。除了所有平台上新的“重新计划”按钮，当你重新计划任务时，Todoist 还可以为单个任务建议一个截止日期。

现在，如果公司能为你做实际工作，那就太好了。
