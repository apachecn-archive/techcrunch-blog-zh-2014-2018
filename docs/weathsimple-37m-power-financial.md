# 机器人顾问 Wealthsimple 从 Power Financial  再融资 3700 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/05/11/weathsimple-37m-power-financial/>

总部位于多伦多的投资平台 wealth simple 管理着 30，000 多名客户和 7.5 亿美元的资产，在美国和美国市场继续增长。为了支持这一增长，该公司从 Power Financial 额外筹集了 3，700 万美元(5，000 万加元)的资金。

像其他类似的平台一样，Wealthsimple 提供自动化技术，允许其用户“自动驾驶投资”。也就是说，用户提供关于其财务状况和风险承受能力的数据，Wealthsimple 创建了一个个性化、低成本和多样化的投资组合，以后可以根据需要重新平衡。

但是，当其他所谓的机器人顾问专注于迎合高端市场时，Wealthsimple 正试图向所有用户提供相同类型的工具和建议，而不管他们必须投资多少。

Wealthsimple 对开户没有最低金额要求，并对管理的所有资产收取 0.5%的费用。它也不收取任何交易、账户转账或再平衡费用。

Wealthsimple 的首席执行官迈克尔·凯臣(Michael Katchen)告诉我:“进入是这个行业的一个主要问题。“我们试图为任何资产规模的人提供优质服务。”

但 Wealthsimple 不仅仅是提供一个管理客户资产的技术平台，它还通过一个品牌的[在线杂志](https://web.archive.org/web/20230307173317/https://www.wealthsimple.com/en-us/magazine)提供教育，并让用户访问其自己的财务顾问团队。

虽然 Wealthsimple 在加拿大开始运营，但它已经扩展到美国，接受像 Wealthfront 和 Betterment 这样的机器人顾问。为了继续扩张，该公司又从加拿大最大的金融机构之一[电力金融](https://web.archive.org/web/20230307173317/https://www.powerfinancial.com/en/)获得了 3700 万美元。

在这一轮融资中，Power Financial 的投资基本上翻了一番，使其对 Wealthsimple 的投资总额达到 7400 万美元。“我们与 Power 的第一笔交易是建立长期合作关系，如果我们需要额外的资金，他们会投入更多资金，”Katchen 说。

拥有像 Power Financial 这样的大型金融公司的支持对 Wealthsimple 来说有巨大的好处，尤其是在它寻找新市场的时候。据 Katchen 称，该战略合作伙伴在全球金融服务公司中拥有约 1.5 万亿美元的资产。

虽然有一个大的战略投资者可能会限制其收购的选择，但 Katchen 说 Wealthsimple 的目标是让企业上市。“我们正试图建立一些持久和长期的东西，”他说。
