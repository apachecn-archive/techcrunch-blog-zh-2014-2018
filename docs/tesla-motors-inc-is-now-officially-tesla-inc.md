# 特斯拉汽车公司现正式更名为特斯拉公司。

特斯拉已正式将其名称从之前的正式公司名称特斯拉汽车公司(Tesla Motors，Inc ),更名为特斯拉公司(Tesla，Inc)。更名可能反映了其业务的扩张性质，现在它拥有 SolarCity，并为家庭和商业用途制造电池组，特斯拉首席执行官埃隆·马斯克(Elon Musk)此前表示，该公司的这些方面将对其整体业务越来越重要。

特斯拉公司在 2016 年底收购了 SolarCity，这是一家由马斯克的堂兄弟创立的太阳能公司，他还担任董事会主席。特斯拉去年还推出了第二代家用 Powerwall 电池存储设备，以及其 [Powerpack 2.0 商用太阳能存储解决方案](https://web.archive.org/web/20230227162257/https://techcrunch.com/2016/10/27/tesla-powerpack-2-has-twice-the-energy-density-began-shipping-in-september/)。该公司还透露计划[制造看起来像普通屋顶瓦片](https://web.archive.org/web/20230227162257/https://techcrunch.com/2016/10/28/these-are-teslas-stunning-new-solar-roof-tiles-for-homes/)的太阳能屋顶板，计划今年晚些时候开始向客户发货。

该公司去年将网站网址从 Teslamotors.com 改为 Tesla.com，并在其官方“关于”页面上将自己列为“不仅仅是一家汽车制造商，也是一家专注于能源创新的技术和设计公司”，因此官方名称的改变姗姗来迟。
