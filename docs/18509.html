NTSB | TechCrunch 称，在佛罗里达州发生致命车祸后，特斯拉 Model S 电池两次重燃

原文：https://web.archive.org/web/https://techcrunch.com/2018/06/26/tesla-model-s-battery-re-ignited-twice-after-fatal-florida-crash-says-ntsb/amp/

图片来源: NTSB

美国国家运输安全委员会在周二发布的初步报告中表示，2014 款特斯拉 Model S 在 5 月份佛罗里达州发生致命高速碰撞后，电池再次着火两次。

NTSB 的报道称，在 18 岁的司机失去对车辆的控制并沿着劳德代尔堡的海风大道撞毁之前，电动车以每小时 116 英里的速度行驶三秒钟。司机和一名前座乘客死亡。第三名乘客被弹出，幸免于难。随着撞车事故的展开，汽车突然燃烧起来。劳德代尔堡消防和救援部门使用 200 至 300 加仑的水和泡沫扑灭了车辆火灾。报道称，消防队员还向碎片喷洒了水和泡沫，其中包括从车上分离下来的一小部分锂离子高压电池。

当车辆装载完毕准备离开现场时，蓄电池再次着火。大火很快被扑灭，NTSB 说。然而，当 Model S 到达储存场时，电池再次点燃。当地消防部门对储料场做出反应，扑灭了大火。

特斯拉拒绝对 NTSB 的报告置评。

NTSB 表示，调查将继续进行，其中还将包括检查用于扑灭电池起火以及在撞车后移走和存放汽车的程序。特斯拉确实向急救人员提供了碰撞后如何处理特斯拉汽车的信息。

最近几个月还发生了另外两起涉及特斯拉电池的火灾。山景城消防部门告诉当地电视台 KTVU 说，在加州一辆特斯拉 Model X 发生致命车祸六天后，这辆车的电池重新点燃了三四次。本月早些时候，女演员玛丽·麦克科马克在推特上发布了一段视频显示她丈夫的特斯拉突然起火。特斯拉表示正在调查这一极不寻常的事件。

特斯拉一直声称，电池组的设计以及安装在汽车地板上的加固结构有许多保护措施，可以防止它着火。例如，电池组包括成组的数千个单元，并容纳在单独的模块中。每个模块之间以及电池组和乘客舱之间都有防火墙。这种包装还被设计成，如果发生火灾，它的蔓延速度应该比在燃气汽车里要慢。

在劳德代尔堡发生致命车祸后，特斯拉发布了一项新功能，可以选择性地限制特斯拉汽车的加速度，并允许车主将最高速度限制在 90 英里/小时。该功能只能通过四位数的 pin 码来启用或禁用。如果车辆接近设定的最大速度，则向车主的移动设备发送通知。
