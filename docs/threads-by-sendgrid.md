# SendGrid 推出 Threads，这是一种面向营销人员的触发式电子邮件服务

SendGrid 首席执行官 Sameer Dohlakia 说他认为公司的新产品 SendGrid 的 Threads 是“反垃圾邮件”

基本上，Threads 允许客户设置基于用户活动发送的自动电子邮件，因此每当客户在您的移动应用程序中查看特定网页、使用特定功能或执行特定操作时，您都可以发送电子邮件。

对于像我这样没有花太多时间考虑电子邮件发送和电子邮件营销的人来说，这听起来可能类似于 SendGrid 和其他公司已经提供的服务。

然而，Dholakia 表示，如果你想用普通的营销工具发送这类有针对性的电子邮件，你必须要求你的开发者拿出一份符合某些参数的客户名单，然后创建一个专门针对这些用户的定制活动。有了线程，你不用写任何代码就可以设置规则和触发器，这样那些邮件就会自动发送。

Dholakia 还表示，Threads 允许 SendGrid(据说它每月已经发送 200 亿封电子邮件)创建一个直接针对营销人员的产品。

“它打开了一大条更以营销为中心的电子邮件，通常没有通过 SendGrid 管道，”他说。

作为营销人员如何使用线索的例子，他说，他们可以针对那些开始注册但没有完成注册过程的用户，或注册但没有验证电子邮件地址的用户，或信用卡被视为无效的用户，创建电子邮件活动。在每种情况下，营销人员都可以创建“各种复杂的工作流程和序列”，根据消费者的行为传递不同的信息。

他甚至认为，这可能会为消费者带来更多相关的电子邮件(因此有了“反垃圾邮件”的评论):“每当你将这样一个强大的工具放在营销人员手中，他们就会利用自己的创造力，根据一些触发的行为，根据他们应该如何定制他们的信息来构思许多微片段。它们不是基础广泛、让人感觉像垃圾邮件的电子邮件。”
