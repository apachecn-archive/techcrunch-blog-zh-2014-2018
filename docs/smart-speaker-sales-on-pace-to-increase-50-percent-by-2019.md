# 到 2019 年，智能音箱销量将增长 50%

似乎亚马逊在 2014 年末发布第一款 Echo 时并不知道自己手上有什么。人工智能驱动的扬声器为消费电子产品的下一个重要时刻奠定了基础。这些设备帮助了主流消费者人工智能，并为联网家庭产品的大规模采用打开了大门。

来自 NPD 的新[数字，自然地，没有显示出该类别的任何衰退迹象。据该公司称，这些设备将在 2016-2017 年至 2018-2019 年期间实现 50%的美元增长。该类别预计明年将增加 16 亿美元。](https://web.archive.org/web/20221207202243/https://www.npd.com/wps/portal/npd/us/news/press-releases/2018/us-voice-activated-speaker-sales-to-see-50-percent-growth-by-2019-according-to-npd-forecast/)

Echo 系列在过去四年中发展迅速，亚马逊增加了像 Spot 和 Show 这样的最畅销的点和屏幕产品。与此同时，谷歌一直在用自己的家用产品紧追不舍。该公司最近还增加了由 LG、联想和 JBL 设计的三种“智能显示器”。

一个新的高端类别也已经兴起，由苹果首次进入该领域的 HomePod 引领。谷歌也推出了类似的 Home Max，三星也将紧随其后推出 Galaxy Home(看起来有点像三脚架上的 HomePod)。

正如上述所有参与者无疑都希望的那样，智能扬声器的销售似乎也在推动智能家居产品的销售，据该公司称，19%的美国消费者计划在明年购买一台智能扬声器。
