三星 Galaxy S9/S9+:下一代旗舰产品有什么值得期待的

原文：https://web.archive.org/web/https://techcrunch.com/2018/02/10/what-to-expect-from-samsungs-next-galaxy-flagships/

又到了一年中的这个时候。三星准备在全球最大的移动会议上发布一些闪亮的新高端智能手机。Android 粉丝准备好欢呼了。

两周之后，TechCrunch 团队将抵达巴塞罗纳的 MWC，为您带来所有的新闻。但是如果你想知道 Sammy 在官方银河拆箱之前做了什么，请继续阅读…

S9 和 S9+未包装

虽然大多数主要的 Android 智能手机制造商都没有在 2018 年 MWC 发布旗舰产品——或许是感受到了中国智能手机市场萎缩带来的压力——但三星肯定没有。今年没有。

按市场份额计算，这家全球最大的智能手机制造商预计将在展会上展示 Galaxy S9 和 S9+。

事实上，在 MWC 之前的新闻发布会邀请函中有一个相当大的线索——一个紫色数字“9”的形状…

三星的时机意味着 S9 和平板电脑大小的 S9+将比去年的 S8/S8+提前一个月推出，去年的 S8/S8+转而在 MWC 之后推出。

一些人认为，在苹果通过在秋季拆箱 iPhone X (和 iPhone 8/8Plus)跳过一个 iPhone 数字后，三星觉得有必要将 S9 的发布时间提前一个月。尽管这一理论并不真正站得住脚，但鉴于三星多年来一直在 MWC 发布会前夕推出新的 Galaxy 旗舰产品——并且一直如此，直到 2017 年。

去年是反常的一年。这种跳动可以解释为在 Note 7 召回后落后于其通常的发布时间表——以及在出现如此高调的问题，呃，爆炸电池后，随后的紧迫需要花费时间来改变其产品安全流程。

三星显然希望将所有这些乱七八糟的事情抛诸脑后。还有什么比回到 MWC 发布 S9 前的全球舞台更能传达“一切照旧”的信息呢？

对于三星来说，在 2018 年 MWC 上抢占硬件风头的事情看起来非常好:就全球市场份额而言，其主要安卓手机挑战者华为预计不会推出太多产品，因为它已经宣布了自己在 3 月下旬在巴黎举行的新闻发布会。

而诺基亚品牌的新贵 HMD 不能——当然——希望连续两次拨动怀旧的心弦，今年再玩一次复古手机游戏。

聚焦相机功能

当然，三星希望其新智能手机凭借自身优势吸引眼球。作为引人注目的升级，它吸引了人们对相机的明显关注。

从很多方面来看，这都是一种奇怪的高端智能手机营销信息。不仅仅是因为图形形状中对电影摄影的微妙暗示。而是因为在过去十年里，智能手机摄像头已经获得了太多的工程关注。以及溢价有多高。

一个真正重新想象的智能手机摄像头必须拥有真正的超能力——比如能够穿透墙壁。这也是非常奇怪和令人不安的。令人高兴的是，没有人指望 S9 能够做到这一点。

苹果的 iPhone X 更好地解释了三星的玩笑，即 S9 相机将被“重新想象”，因为 Cupertino 的顶级 iPhone 装有专用深度传感器，用于通过相机镜头为增强现实体验提供动力——例如面罩和可以跟踪面部表情的动画表情符号。

iPhone X 还采用了一种新的生物认证方法，该方法依赖于使用相同的原深感摄像头单元捕捉面部生物特征。

因此，三星试图在传感硬件方面做得更多，以追赶苹果在这方面的领先优势似乎是可能的。

也就是说，从泄露的设备图片来看——由可信的智能手机泄密者埃文·布拉斯(Evan Blass)获得(见下文)——S9/S9+似乎没有在去年的 S8/S8+ 之上安装任何额外的传感器硬件。

上个月三星确实为其最新的智能手机芯片组制造了一些噪音，明确吹捧硅的潜力，为苹果在 iPhone X 上所做的类似体验提供动力——写道，“通过深度传感”，芯片组可以用于“3D 扫描用户的面部，用于混合面部检测”。所以，嗯，[此处插入思考表情符号脸]。

另一种可能性是:三星可以使用一种结合多种现有生物识别技术(即 S8 的面部+虹膜扫描系统)的工程解决方案，试图提升其与苹果 FaceID 的竞争。这已经是传闻中的。

鉴于苹果尚未在所有 iPhones 上推广原深感摄像头，这种方法对 S9 可能最有意义。事实上，iPhone X 的传感器包装缺口仍然只有 iPhone X。相关的 iOS 功能也是如此，比如 Animoji 和 FaceID。

鉴于苹果公司的溢价门控，三星可能正在寻找一个机会，为其 S9 旗舰产品开发一些“animojish”的华丽而有趣的相机功能——即使其 FaceID 竞争对手尚未准备好推出。

(而且——更刺激的是——一个 Blass 消息来源声称 S9 将包括自拍模式，带有“有点像 animoji 的动画头像”。)

苹果还利用 iPhone X 上一次重大传感器升级的机会，放弃了 home 键，转而在设备上使用更具手势的用户界面。这在某种程度上是不幸的，因为它将 iPhone 用户界面分成了两部分。(Cupertino 可能会在未来再次统一。)

三星在取消 home 键方面走在了前面，去年的 S8 去掉了物理键以最大化屏幕空间。尽管它没有完全采用基于滑动的导航。相反，它在接近边缘的显示屏底部添加了一个带有触觉反馈的虚拟触敏按钮。

因此，看看三星是否会决定完全消除 S9 的可用性支柱将会很有意思。事实上，有一些传言说，一个新的，只有 S9 的用户界面即将推出。

另一方面，与界面惯例的重大突破确实需要比三星在管道中出现的更激进的硬件升级。所以我们不能指望这里会有任何过于彻底的界面变化。

看，没有缺口！

Blass 上个月底得到了上面泄露的 S9 和 S9+的图像。此后，他又发了一些帖子(见下文)。

看到这些，一个直接的收获是 S9/S9+上没有凹槽。凹口是异形传感器单元，它不幸地咬了 iPhone X 的屏幕一口。

事实上，泄露的 S9 图像上的传感器配置看起来与 S8 相同。因此，如果三星正在将更多的传感硬件挤压到手机顶部的细长空间中，显然和不会这么做。

(需要说明的是，iPhone X 的原深感摄像头单元包含:一个红外摄像头；泛光照明器；近程传感器；环境光传感器；点投影仪；以及一个扬声器和麦克风。而 S8 的一组前传感器包括一个 SVC LED 接近传感器(检测器)&光传感器；接近传感器(发射器)和虹膜 LED——后者为虹膜扫描生物特征提供动力。)

S8 和 S9 之间的视觉设计一致性很大程度上表明，三星还没有传感硬件来直接挑战 iPhone X 的原深感摄像头的能力。

该公司自己的公关指出，其上述顶级芯片组硬件也需要深度感测硬件，才能支持 3D 人脸扫描“用于混合人脸检测”(这将实现“逼真的人脸跟踪过滤器，以及在用人脸解锁设备时更强的安全性”，正如三星所销售的那样)。

因此，除非它设法从根本上小型化 S9 上必要的深度传感硬件，将其缩小到与 S8 几乎相同的形状，并且在这个时候，它也在重组其智能手机流程，专注于安全问题，否则类似的 FaceID 风格的面部解锁功能似乎不太可能即将推出。

虽然三星可能仍然设法利用它已经能够烘烤的传感器来鼓动一些敌意。

所以，准备好开始你关于 S9 的“隐形缺口”的笑话吧。

另一个引人注目的设计要点是 S9 和 S8 的外观没有什么新的东西。除非你能把紫/紫丁香染成紫红色。

从设计角度来看，基本上都是相同的弧形屏幕边缘——不管你喜欢还是讨厌！—和所有。

说到更多相同的东西，我们估计三星不会做苹果，会在 S9/S9+上保留 3.5 毫米耳机插孔。

为什么？因为为什么要在这个月看竞争对手的礼物，毫无意义地浪费一个意想不到的竞争优势。勇气见鬼去吧。

黏糊糊的手指

接着，Blass 还拿到了 S9/S9+和相关组件的一些后置照片，其中显示了一个指纹读取器位于后置摄像头正下方的新位置。这对于尴尬的 S8 相机侧置来说肯定是一个受欢迎的调整。

因此，根据你的观点，三星正在采取“蛋糕和吃它”的生物识别方法，而苹果根本没有为 iPhone X 用户提供使用指纹生物识别的选项(他们可以选择注册一个强大的深度映射面部生物识别，或者完全没有生物识别认证)。

或者三星对自己的面部生物认证系统的稳健性并不完全有信心——之前被轻易骗过。因此，保留指纹扫描仪是有帮助的，因为它为不习惯该公司虹膜或面部扫描系统的用户提供了另一种选择。

至少在安全方面，苹果似乎正在让 iPhone X 的专用传感硬件也算。(除非你恰好有一个一模一样的邪恶双胞胎。)因此，三星保留指纹识别器也符合 S9 更多是权宜之计、迭代升级而非智能手机战略重大变革的理念。

从好的方面来说，至少这些手机不会强迫你面对解锁，如果你不想的话。

想模糊就模糊

Blass 泄露的图片的另一个要点是:S9+与 S9 相比确实有一个非常明显的摄像头硬件差异——它配备了两个后置摄像头。终于来了！

这符合广泛报道的谣言三星最终将双摄像头添加到其旗舰智能手机中——最初将硬件功能引入其高端平板电脑 Galaxy Note 8 。

与 Note 8 一样，S9+的双镜头将用于增强摄影深度效果，如散景(在令人愉快的模糊背景下，主体清晰可见)，因为这两个镜头可以收集立体数据。

以及增强弱光摄影——智能手机相机的一个长期挑战，相机传感器必须挤在这么小的空间里。

在 Note 8 上，三星还将双摄像头用于其他东西——比如照片功能，可以捕捉相框以外的其他图像。

这里的底线是它在进行必要的追赶。早在 2016 年，苹果就在 iPhone 系列中引入了双摄像头，在 iPhone 7 Plus 上引入了。所以三星肯定需要缩小差距。

它上个月在推特上发布的 S9 invite 的视频版本通过淡出荣耀的模糊来强调散景。该动画还暗示了一个超级慢动作视频捕捉功能——另一个被广泛报道的谣言我们期待它会被证实。

三星措辞奇怪地声称 S9 的发布将“改变你体验一切的方式”，这可能是暗指相机驱动的 AR 功能或暗示——正如广为流传的一样——S9 也将有机械可变光圈。(或者，好吧，这只是一些可怕的过度公关。)

可变光圈有什么意义？它允许相机通过控制进入镜头的光量在不同的焦距之间切换——字面上是通过扩大或缩小它进入的孔。

这又通过更好地适应不同的拍摄条件而允许对照片/视频的外观进行更好的控制。因此，再次承诺改善智能手机照片/视频，包括在弱光条件下。

但是，正如所有预期的功能一样，我们谈论的是“受欢迎的改进”和“值得拥有的增强”。不是带 x 光视觉的智能手机。

不要太激动——现在还不要

总而言之，我们期待三星为 S9/S9+推出一些不错的额外产品。但它的下一代 Galaxys 看起来更像是在追赶——做一些常规的增强(当然，也期待处理器和电池升级)——而不是为了智能手机的名声。

但是——但是！—如果你渴望在 2018 年对三星智能手机进行更彻底的升级，那么，其他传言 都有。尽管 2018 年 MWC 很可能不会成为萨米最终推出其长期以来被搁置的& D-works 可折叠智能手机的事件(尽管该公司确实在上个月表示，它计划在 2018 年发布可折叠手机)。如果是这样的话，那么，三星确实一直在保持粉末非常干燥。

我们相当肯定，该公司也不会在巴塞罗那推出其计划中的 iPhone X 杀手。不过，今年晚些时候，它可能会在这方面再做一件事。
