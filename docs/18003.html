TiVo 将 Alexa 语音控制添加到其 DVRs TechCrunch 中

原文：https://web.archive.org/web/https://techcrunch.com/2018/05/14/tivo-adds-alexa-voice-control-to-its-dvrs/

TiVo 的 DVR 得到了 Alexa 的支持。该公司宣布其 DVR 系列，包括系列 4 (Premiere)、5 (Roamio)和 6 (Bolt)盒子，如去年秋天推出的 TiVo Bolt VOX,将获得对亚马逊虚拟助手 Alexa 的支持。TiVo 表示，该助手将能够做诸如更换频道、跳过广告、向前或向后跳转、启动网飞等应用程序等事情。

该公司并不是唯一一家增加了对 Alexa 支持的第三方 DVR 制造商。

得益于开发工具，如视频技能应用编程接口(Video Skill API)，其他有线电视和卫星电视公司、流媒体服务和内容提供商现在也可以在他们的设备和应用程序中添加语音控制。例如，Dish 去年秋天成为第一家与 Alexa 整合免提电视的美国付费电视提供商。其他与亚马逊合作的公司包括 DirecTV 和 TechCrunch 的母公司威瑞森。

亚马逊的视频技能 API 在 3 月份更新了以包括对 DVR 录制的支持，允许用户通过他们的声音设置和管理他们的 DVR 录制内容——这是 TiVo 可能会在以后添加的东西，因为它还没有上线。在亚马逊宣布的时候，它还说 TiVo 是使用视频技能 API 开发体验的公司之一。

此外，TiVo 在 1 月份的 CES 上宣布了加入智能家居集成的计划，包括通过 Alexa 和谷歌助手进行语音控制。

根据 TiVo 的说法，Alexa 将会让它的客户用 TiVo 遥控器做很多他们现在可以做的事情。例如，你可以通过说“Alexa，看 CBS”或“Alexa，去福克斯”这样的话来要求 Alexa 更换频道。你也可以在 TiVo 上启动应用程序，比如说“Alexa，打开网飞”。"

但 TiVo 的实现与其他 DVR 制造商不同的地方在于它如何利用 Alexa 来控制其设备的独特功能，比如跳过广告——TiVo 称之为 SkipMode。

TiVo 说，这可以通过告诉 Alexa“跳过广告”来实现，它加入了其他播放相关的技能，如向后跳 8 秒(“Alexa，后退”)、暂停和播放、快进和倒带。

TiVo 产品管理高级总监安德鲁·海曼(Andrew Heymann)在一份关于新功能的声明中说:“有了远场语音控制，我们的客户生活变得更加自由了。”。“他们可以利用 TiVo 的超酷功能继续欣赏喜爱的节目，即使他们在准备晚餐时手太脏而无法使用遥控器，或者在锻炼时无法使用遥控器。生活突然变得容易多了，”他补充道。

定期报道 TiVo 的 Dave Zatz 报道说，本周末在支持的设备上弹出了 Alexa 新功能的占位符屏幕。屏幕上显示 Alexa“即将推出”，并将推出软件版本为 20.7.4 或更高版本的 TiVo 零售设备。它还指出，首次展示预计将于 6 月 1 日完成。

将 Alexa 添加到 TiVo 的盒子中也值得注意，因为 TiVo 本身已经开发了自己的语音控制功能。其较新的 BOLT VOX 和 Mini VOX 是该公司第一批包含语音遥控器的 DVR，提供与 Alexa 类似的功能。

然而，TiVo 单独出售遥控器，这限制了其语音控制产品面向消费者的覆盖范围。有了 Alexa，该公司能够追逐那些已经拥有亚马逊 Echo 设备的不断增长的市场。
