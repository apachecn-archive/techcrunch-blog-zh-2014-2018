# 立即观看 Jane 上线 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/09/13/see-jane-go-live-today/>

针对女性司机和乘客的打车服务是一个规模虽小但正在增长的利基市场。[见简就走](https://web.archive.org/web/20221210052503/https://seejanego.co/)今天正式上线运行；司机和乘客可以在 iOS 或 Android 上下载该应用程序以开始使用。 [SafeHer](https://web.archive.org/web/20221210052503/http://www.safeher.com/) 将于今年秋天在波士顿推出，并计划在不久后扩展到四个城市。

在今年早些时候报道 [See Jane Go](https://web.archive.org/web/20221210052503/https://beta.techcrunch.com/2016/06/21/see-jane-go-is-bringing-women-only-ride-hailing-to-the-west-coast/) 和 [SafeHer](https://web.archive.org/web/20221210052503/https://beta.techcrunch.com/2016/04/08/chariot-for-women-is-a-new-ride-sharing-service-for-women-only/) (女性打车服务，简称战车)时，我发现女性在使用优步和 Lyft 等较大的打车服务时，无论是司机还是乘客都感到不舒服。对许多女性来说，搭一个随机的男人的车或搭一个随机的男人的车似乎不太安全。

See Jane Go 通过为女性和那些认为自己是女性的人设计服务来解决这个问题。首席执行官金伯利·图宁(Kimberly Toonen)在早些时候的一次采访中说，“所有司机都要提交一份申请，附上他们的驾驶执照，我们用它来验证性别。无论他们自己认为那里有什么“都足够好了”。男人可以在为他们担保的女人的陪伴下使用这项服务。

骑手可以在 See Jane Go 应用中将车手标记为收藏夹，反之亦然。该服务的算法会更频繁地将收藏夹配对在一起。这培养的不仅仅是友谊；该公司的目标之一是为使用该服务的女性创建一个网络空间。Toonen 说:“我们试图鼓励一个来回——共享资源，网络和建议。

尽管这项服务目前只在加州的奥兰治县上线，但它鼓励全国的女性注册。宣布推出的新闻稿指出，该公司的“积极计划在全国范围内扩张”，拥有大量潜在用户的城市将有助于制定这些扩张计划。
