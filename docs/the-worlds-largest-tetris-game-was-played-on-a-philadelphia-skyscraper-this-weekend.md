# 这个周末,“世界上最大的俄罗斯方块游戏”在费城的一栋摩天大楼上进行

【YouTube http://www.youtube.com/watch?v=A6P1sskPc4w&w=640&h=360]

周六晚上，费城一座 29 层的摩天大楼举办了据称是“世界上最大的俄罗斯方块游戏”，也可能是世界上最大的视频游戏。

德雷克塞尔大学计算机科学教授[黎先春](https://web.archive.org/web/20230401145141/http://www.pages.drexel.edu/~fjl24/)带领的一个团队黑进了费城市中心 [Cira 中心](https://web.archive.org/web/20230401145141/http://en.wikipedia.org/wiki/Cira_Centre)大楼的 led 照明系统，创造了一个夸张的俄罗斯方块游戏，可以在该市正在进行的科技周期间玩。

《T4 时报》采访了李和他的团队，并制作了一份精彩的视频报告，你可以在这里观看。不幸的是，该视频不能嵌入到其他网站，但你也可以在上面嵌入的视频中看到游戏的运行。

李告诉《纽约时报》:

> “人们会认为这是一场游戏，但我认为这是一个公共装饰品。科技让我们彼此隔离。我希望我们能在一起，一起玩。”

这是李的团队连续第二年侵入 Cira 中心的 1400 多个 LED 灯，以创建大型版本的经典街机游戏。2013 年 4 月，李博士的团队在摩天大楼上首次推出了一款乒乓游戏，并获得了世界上最大的建筑视频游戏显示器吉尼斯纪录。李开复告诉《NYT 》,上周末的俄罗斯方块游戏的尺寸是乒乓显示屏的两倍，所以他们期待创造另一项纪录。

一个非常大的视频游戏不是火箭科学或世界和平。但如今，有时感觉技术就是钱、指标和"[出口](https://web.archive.org/web/20230401145141/http://www.investopedia.com/terms/e/exitstrategy.asp)"因此，很高兴看到人们以一种更有趣的方式玩科技——这种方式很可能让他们第一次爱上它。
