# Snapchat 引入了多达 16 人的群组，以及新的创意工具

Snapchat 增加了一些新功能，[包括 Groups](https://web.archive.org/web/20221207102536/https://www.snap.com/en-US/news/post/happy-holidays/) ，这是一个新的群聊功能，可以让多达 16 个朋友共享对话和媒体。默认情况下，群聊会在 24 小时后自动删除，并且群内发送的快照只能由群中的每个成员打开一次，并且只能重放一次，就像常规快照一样。

新的群组功能还包括一种与群组内任何人单独聊天的简单方式，当他们在聊天中活跃时，您可以点击任何成员的名字，然后进入一对一的对话。聊天中支持的所有功能都支持分组，包括贴纸、投票和视频笔记、Bitmoji 等等。

Snapchat 还推出了新的创意工具，用于合成您的快照。有剪刀可以让你剪下快照的一部分，并把它做成自己的贴纸，然后可以在贴纸抽屉里找到。这将首先在 iOS 上推出，随后将推出 Android 版本。

还有一个新功能是添加了 Paintbrush，这是一个可以在 memories 中的任何快照上使用的功能，它的工作方式类似于 Prisma 等艺术滤镜应用程序的工作方式。还有一个新功能是 Shazam 支持，内置在 Snapchat 摄像头中，让你可以识别应用程序中正在播放的音乐。
