# CES 应该走向创新城市

一年一度的科技和创新盛宴 CES 已经不再受欢迎。自 20 世纪 70 年代末以来，拉斯维加斯一直是这项活动的举办地，不管是好是坏，这座城市都幸存了下来，并为每年涌入这项活动的技术专家提供服务，他们都是为了观看最新最棒的产品。但是两件事同时发生，使得拉斯维加斯成为最后一个发现创新的地方。

首先，拉斯维加斯的基础设施，虽然看起来对会议观众的大规模涌入有弹性，但在跟上 CES 方面有困难。许多大厅里的大量人群受到了[多次停电](https://web.archive.org/web/20221025223652/https://beta.techcrunch.com/2018/01/10/the-day-the-lights-went-out-at-ces/)的欢迎——这对于一场科技盛会来说是一个讽刺的场景。价值百万美元的展位最终陷入黑暗，谷歌的一个展位被雨水淹没，因为对实际天气没有做好准备。穿梭系统根本跟不上拥挤的人群，而贫血的单轨铁路——对公共交通的冒犯——在白天高峰时毫无用处。再加上多个大型场馆和一个无法应对无数非本地人试图从 A 点到 Z 点的拥挤的出租车系统，你就有了一个烂摊子。

第二，像 CES 这样的大型展会的原因正在发生变化。大多数业务都是通过电子方式完成的，甚至是通过算法完成的，为什么每个人都需要在一个地方？虽然在赌场呆上一周很不错，但也许是时候来一场规模更小、更有针对性的表演了，或者干脆不要了？

或许，到最后，是时候把 CES 搬到一个现代化的城市了？

至于把与会者送到哪里，我没有具体的答案——丹佛？旧金山？洛杉矶？迪拜？柏林？——也许真正的答案是拉斯维加斯解决其非常现实且非常困难的交通、电力和数据问题，以便创新能够在这个陌生的沙漠绿洲蓬勃发展。就目前的情况来看，这座想要招待成千上万技术专家的城市并不太科技化，而且它的便利设施——更多地针对单身派对，而不是局域网派对——不足，甚至严重缺乏。

也许下一个大事件不会是物理上的。毕竟，举办一年一度的科技节的原因已经改变了。CES 曾经是前瞻性的:一月份在那里发布的产品几个月后就上架了。当前的模式是颠倒的:最酷的东西一年前在网上推出，并在 CES 上展示。4 月份完成的众筹项目在 1 月份的 CES 上作为一种亮相派对出现。但是人们真的还在乎吗？众筹者关心的是运送产品，而不是在一个充满敌意的城市花上几千美元买一个摊位。

要想让 CES 再次惊艳，还需要做出很多改变。也许为时已晚。但是，当拉斯维加斯跟不上未来的发展时，它对世界没有任何好处，并严重损害了自己的形象。
