# 多亏了 3D 打印技术，没有前腿的吉娃娃狗 TurboRoo 又能走路了

有什么可爱的动物是 3D 打印救不了的？看看 TurboRoo，一只出生时没有前腿的小吉娃娃，多亏了一只 Makerbot 和一名来自 T2 的 3D 设计师 3D yn T3，它被赋予了新的生活“皮带”。

[3d yn 总裁 Mark Deadrick](https://web.archive.org/web/20221007215932/https://www.linkedin.com/in/markdeadrick) 在网上看到 [TurboRoo 对轮子的呼吁](https://web.archive.org/web/20221007215932/http://www.youcaring.com/pet-expenses/wheels-for-turboroo-/209928)并设计了一款小型轮式推车，从[网上照片](https://web.archive.org/web/20221007215932/https://twitter.com/RooTurbo)估算尺寸。他用亮橙色打印了模型，装上了一些滚轮，然后把手推车送给了 TurboRoo 的主人。现在，这只小狗狗正骑着一套免费的、完全可改装的超级腿跑着。

为什么这很酷？因为在 3D 打印之前，TurboRoo 的所有者必须以高昂的成本用现成的管道、布料和其他材料建造一些东西。然而现在，手推车可以根据 TR 的身体定制，随意再版，甚至可以由其他设计师进行修改。最重要的是，他们可以在几乎没有任何花费和时间的情况下制造多个手推车。

这并不是 3D 打印第一次让动物和人类生活得更好。我首先想到的是现在帮助残疾儿童抓握物体的机械手。在*后生动物*方面，[设计师在一月份为鸭子](https://web.archive.org/web/20221007215932/https://beta.techcrunch.com/2014/01/22/designers-delay-ducks-death-with-3d-printed-limb/)造了一条可爱的腿，目前[有企鹅和其他带有 3D 打印喙的家禽](https://web.archive.org/web/20221007215932/http://3dprint.com/1720/3d-printed-beak-to-save-a-polish-penguins-life/)。但是没有什么比看到一只小狗飞奔来温暖我的心脏更好的了。
