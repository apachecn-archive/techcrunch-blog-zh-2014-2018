谷歌推出新的机器学习平台

谷歌今天在旧金山的下一届谷歌云平台用户大会上宣布了一个面向开发者的新机器学习平台。正如谷歌董事长埃里克·施密特在今天的主题演讲中强调的那样，谷歌认为机器学习是“下一步”借助这个新平台，谷歌将使开发人员更容易使用谷歌已经用于支持收件箱智能回复等功能的一些机器学习智能。

该服务现已推出有限的预览版。

“谷歌的主要应用程序使用云机器学习，包括照片(图片搜索)，谷歌应用程序(语音搜索)，翻译和收件箱(智能回复)，”该公司说。“我们的平台现已成为云服务，可为您的业务应用带来无与伦比的规模和速度。”

谷歌的云机器学习平台基本上由两部分组成:一部分允许开发者从自己的数据中建立机器学习模型，另一部分为开发者提供预先训练好的模型。

为了训练这些机器学习模型(这需要相当多的计算能力)，开发人员可以从工具中获取数据，如谷歌云数据流、谷歌大查询、谷歌云数据平台、谷歌云存储和谷歌云数据实验室。

“云机器学习将处理从数据摄取到预测的所有事情，”该公司表示。“结果是:现在任何应用程序都可以利用同样的深度学习技术来支持谷歌的许多服务。”

预先训练的模型包括现有的 API，如谷歌翻译 API 和云视觉 API，但也包括新的服务，如谷歌云语音 API (你可以在这里了解更多信息)。云语音 API 支持谷歌自己的语音搜索和语音应用。它可以为 80 多种语言进行语音到文本的转换。

谷歌在今天的主题演讲中强调，它希望将其内部开发的技术带给开发者，并尽可能使其易于使用。与此同时，该公司还开源了 Tensorflow 等工具，以允许社区使用其内部工具，根据自己的使用情况进行调整，并对其进行改进。
