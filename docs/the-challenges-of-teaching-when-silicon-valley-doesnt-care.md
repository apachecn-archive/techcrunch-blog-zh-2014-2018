# 当硅谷不在乎时教学面临的挑战 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/07/11/the-challenges-of-teaching-when-silicon-valley-doesnt-care/>

在过去的几周里，我一直在疯狂地写作。部分原因是，我觉得绝大多数教育工作者和企业家终于意识到，硅谷一开始就没有正确对待我们的技术和课堂方法。

当然，一些内容比以前更容易获得，这对于获取和平等绝对是一个了不起的进步。然而，尽管创业公司大肆宣传，教育的日常工作并没有改变多少。虽然硅谷声称它将彻底改革教育，但现实是，它充其量只是增强了经典模式，而不是取代它。

我感兴趣的另一个原因更加个人化:我目前正在教我的第一届学生。在这个案例中，36 名韩国大学生来自当地一所顶尖大学，他们正在学习创业和企业家精神课程。

令我惊讶的是，我对教育学生的实际教学挑战毫无准备。在这次旅行之前，我曾问过一位教授关于教学的问题，我得到的回答是“你一辈子都是学生，所以教学应该很容易。”说到建议，你真的是物有所值。

不幸的是，这种想法不仅是常春藤联盟学校的精英教授的通病，也是那些认为自己将彻底改变课堂的技术专家的通病。教学被认为是教育中容易的部分，好像还有其他更重要的东西。我们知道，老师更好的学生比老师更差的学生表现更好，所以很明显，教学很重要。

我过去一周的经历很能说明问题。当我同意教授这门关于创业的课程时，我的挑战几乎立刻就开始了。我该教什么？我的课程应该如何组织？我每天要安排五个小时的课，持续两周，我不能只给学生讲内容，并期望他们理解正在发生的事情，尤其是在对学习的期望(可以接受的)较低的夏天。

所以我做了 21 世纪任何人都会做的事，我搜索了谷歌。正是在这里，我突然意识到我们的教育思想是多么的基础。

什么在课堂上有效？不同的学生应该如何进行不同的教育？当我在谷歌上搜索时，当然有很多创业和创业课程的例子，但关于哪些课程真正成功的信息却很少。

其他问题也冒了出来。我如何开展正确的课堂活动和作业，以达到我想要的学习效果？我应该为暑期速成课程和为整个学期设计的课程使用相同的课程结构吗？

我希望有一种更加以数据为中心的方法，某种课程的 A/B 测试，这样我就可以了解哪些课程或教学方法是最成功的。鉴于在线教育的成功，对内容进行随机实验似乎很容易实现。然而，这样的工作才刚刚开始。

鉴于硅谷对消费产品和数据科学的过度关注，我本以为这里会有更多的行动。我相信 MOOCs 从他们的课程中获得了一些见解，这很好。然而，这些见解很少与我分享，我是一个希望为我的学生建立正确的教育体验的老师。

那么教育技术的问题不一定是我们的方法，而是我们的焦点。虽然数据对教育至关重要，但我们长期以来一直在强调错误的消费者类型。我们一直认为学生是教育的“最终”消费者——他们是接受教育的人，也经常为教育付费。

然而，这个市场还有另外一面，那就是教育者。尽管学生取得了所有的技术进步，但教育工作者只收到了少数有用的工具来帮助更好地管理他们的教室和学习过程。相比完全取代教育体验，转变教师的“革命性”尝试要少得多。

我不认为我们会让人类脱离教育。教育从根本上来说是一个社会过程，即使我们用计算机取代了这个过程中的一些元素，人类仍然有空间以有效的方式帮助他们的学生。

创业公司应该考虑如何重建授课的体验。即使我的学生没有提问，我怎样才能得到重要问题的提示？我怎样才能知道哪种材料可能是最有效的，甚至在我教它之前？

这些都是数据科学问题，所有这些都是大师级教师在其职业生涯中自然而然学到的。我希望看到的是一个能够彻底转变教师发展的创业公司。如果我们能让一个老师只用一半甚至十分之一的时间就能让他们变得几乎同样有效，我们为人类发展所做的可能比任何创业公司都要多。

毫无疑问，建立一个洞察力的图书馆，并最终开发一个关于教育的计算机化的直觉是巨大的想法。它甚至可能需要我们完全弄清楚大脑是如何工作的，这也许是最重要的(也是最困难的！)21 世纪的研究项目。但这不是一个不可能解决的问题，这似乎是下一代教育创业公司需要解决的正确的“触及”挑战。

对于教育科技创业公司来说，这是一个重新考虑未来的激动人心的时刻。我们几乎不应该忘记学生——但他们需要在整个企业中被视为平等的。只有当我们制造出能够彻底改变市场双方体验的工具时，我们才会看到我们所希望的科技教育进步。
