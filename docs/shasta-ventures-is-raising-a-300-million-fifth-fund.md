# Shasta Ventures 正在筹集 3 亿美元的第五只基金

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/06/29/shasta-ventures-is-raising-a-300-million-fifth-fund/>

# 沙斯塔风险投资公司正在筹集 3 亿美元的第五只基金

根据一份新的 [SEC 文件](https://web.archive.org/web/20230327065524/https://www.sec.gov/Archives/edgar/data/1678104/000167810416000001/xslFormDX01/primary_doc.xml)，Shasta Ventures 正在为第五只基金筹集高达 3 亿美元的资金，该文件列出了董事总经理 Jason Pressman 和 Rob Coneybeer (Coneybeer 在 2004 年与董事总经理 Tod Francis 和 Ravi Mohan 共同创立了 Shasta；普雷斯曼次年加入)。

沙斯塔似乎仍在坚持用新基金编织。当其投资组合公司 Nest Labs 在 2014 年初以 32 亿美元的价格出售给谷歌时，它看到了巨大的回报。(Coneybeer 在这里采访了记者[关于 Nest 赌注的坚持是如何得到回报的。)尽管如此，其最新基金的目标金额与该公司两年前](https://web.archive.org/web/20230327065524/http://www.strictlyvc.com/2014/01/13/nest-investor-shasta-ventures-persistence-pays/)[收盘时的金额相同。](https://web.archive.org/web/20230327065524/https://techcrunch.com/2014/06/18/shasta-ventures-fourth-fund/)

其第三只基金于 2011 年关闭，规模也大致相同:2.65 亿美元。

专注于早期机会的沙斯塔投资于企业、消费者和互联硬件公司。

除了 Nest，沙斯塔的许多高调投资组合公司包括个人金融服务 Mint([2009 年卖给 Intuit](https://web.archive.org/web/20230327065524/https://techcrunch.com/2009/09/13/intuit-to-acquire-former-techcrunch50-winner-mint-for-170-million/))； [Airware](https://web.archive.org/web/20230327065524/http://www.airware.com/) ，无人机操作系统启动； [Eero](https://web.archive.org/web/20230327065524/https://www.crunchbase.com/organization/eero) (下一代 Wi-Fi 路由器公司)；以及剃须刀订阅服务[美元剃须俱乐部](https://web.archive.org/web/20230327065524/https://try.dollarshaveclub.com/try-the-club/)。
