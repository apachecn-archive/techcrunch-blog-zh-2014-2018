特斯拉 Model S P100D 的 0-60 英里/小时时间为 2.389 秒，Ludicrous+更新

【YouTube https://www.youtube.com/watch?v=fUAZcD0gqDg&w=680]

特斯拉的 Model S P100D 已经可以媲美尚未发布的法拉第未来 FF91 的最高 0 到 60 英里/小时的时间，这是使用任何驾驶员现在都可以使用的软件的库存汽车。该汽车制造商去年年底推出的 Ludicrous+软件复活节彩蛋更新使 0-60 英里/小时的时间仅为 2.389 秒，比特斯拉自己声称的标准加速度测量距离的当前最快时间快了一根头发。

根据特斯拉赛车频道(Tesla Racing Channel)的说法，测试中使用的 P100D 没有以任何方式进行修改，无论是在软件还是硬件方面，该频道向其 YouTube 频道提交了运行结果。 Electrek 指出,这意味着一辆目前在街上合法生产的特斯拉汽车已经可以打破法拉第未来声称的 FF91 的 2.3 秒时间，这一时间也是通过修改内饰来实现的，旨在减轻重量，使概念车比最终的出货版本快一点。

特斯拉尚未对法拉第未来做出任何直接回应，法拉第未来在舞台上召唤了它的电动竞争对手，甚至在拉斯维加斯的汽车发布会上推出了 Model S 和 Model X 进行计时测试。但马斯克指出，根据 Motor Trend 的测试标准，他认为理论上 2.3 秒的时间对于配备 Ludicrous+更新的库存 Model S 来说是可能的，来自第三方的演示无疑使这一目标看起来触手可及。

大多数人永远不会使用这个功能，它可能不太适合你想在车上安装的东西，比如轮胎。但这是电动汽车制造商选择的一项措施，看起来很有趣。此外，FF 仍然不得不发运那辆它曾声称拥有令人印象深刻的加速性能的汽车，当它发运时，FF91 的价格似乎越来越接近 30 万美元。现在那是可笑的+。
