# 特斯拉亏损超出预期，但坚持盈利目标

特斯拉报告第二季度亏损超出预期，但坚持对下半年盈利和现金流的积极预测。

特斯拉[报告](https://web.archive.org/web/20221205133006/http://ir.teslamotors.com/static-files/7235e525-db16-470c-8dce-9ecac0ad7712) 季度亏损 7.175 亿美元，去年同期亏损 3.364 亿美元。特斯拉历史上只有两个盈利季度，最后一个是在 2016 年报告的。这是该公司连续第七个季度亏损。

经一次性项目调整后，特斯拉亏损 5.2 亿美元，或每股 3.06 美元，去年同期为 2.2 亿美元，或每股 1.33 美元。

该公司第二季度收益中有一些亮点，这些亮点是在周三收盘后公布的。特斯拉约 7.4 亿美元的负自由现金流低于预期。该公司第二季度末拥有 22 亿美元现金。

特斯拉还报告第二季度销售额高于预期，达到 40 亿美元，比去年同期的 28 亿美元增长 46%，主要是由于 Model 3 的交付。特斯拉报告第一季度销售额为 34 亿美元。

根据公认会计准则，特斯拉汽车公司的毛利率上升至 20.6%。该公司的非 GAAP 汽车毛利率增加到 21%。

特斯拉在第二季度财报中表示，自 6 月最后一周首次实现目标以来，该公司已多次实现约 5000 辆 Model 3 汽车的周产量目标。

特斯拉表示，现在的目标是到 8 月底每周生产 6，000 辆 Model 3 汽车，预计未来几个季度的产量将超过每周 6，000 辆，同时保持额外的资本支出有限。

该公司表示，将通过提高现有生产线的利用率和进行选择性改进来解决瓶颈问题，而不是建立全新的重复生产线，来实现这些生产目标。

该公司在其股东信中表示:“我们的目标是尽快将 Model 3s 的产量提高至每周 1 万辆。”。该公司表示，预计将在 2019 年某个时候达到这一速度。
