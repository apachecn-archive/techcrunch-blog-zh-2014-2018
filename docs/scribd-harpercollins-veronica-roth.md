# Scribd 将在发布日获得维罗尼卡·罗斯的“命运分割”

Scribd 正着眼于更大、更新的游戏。

维罗尼卡·罗斯的系列的第二卷*命运分割线*的电子书和有声读物版本将于明天在 Scribd 上上线。这不是一项独家交易——你可以在书店和网上零售商那里买到*The Fates Divide*——但 Scribd 将是你在发布日可以找到电子书的唯一订阅服务。

2013 年，Roth 的出版商 HarperCollins 是五大出版商中第一个与这项服务合作的。去年，它开始在 Scribd 上提供一些更新的“热门”图书。

因此，即使你没有急切地等待该系列的下一本书，值得注意的是，HarperCollins 愿意在 Scribd 上推出主要书籍，而不仅仅是将这项服务视为旧书籍的一种货币化方式。

罗斯最为人所知的是他的青少年小说系列*的作者。她的新书也引起了轰动，第一卷首次登上了《纽约时报》青少年畅销书排行榜的榜首。[根据评论者](https://web.archive.org/web/20221025222504/http://www.chicagotribune.com/lifestyles/books/sc-carve-the-mark-veronica-roth-books-0118-20170114-story.html) , *雕刻印记*类似于*分歧者*的反乌托邦科幻小说和青少年浪漫爱情的结合——但是当动作转移到一个遥远的星球时，带有一点*星球大战*的味道。*

在公告中，哈珀柯林斯数字业务发展高级总监亚当·西尔弗曼说:

> 我们相信 Scribd 的订阅模式是一个优秀的零售平台，可以在这个平台上推出新的游戏，并为像 *The Fates Divide* 这样的大型发布活动制造声势。在我们与 Scribd 合作的五年中，我们已经看到在这项服务上提供我们的书籍的价值越来越大——他们精彩的推荐引擎和营销努力是难以超越的——随着我们的合作关系继续发展，我们很高兴能与 Scribd 合作，在未来推广更多我们最好的书籍。

Scribd 表示，它现在拥有超过 75 万付费用户。它最近也回归了一种模式，在这种模式下，那些订阅者可以阅读和收听无限数量的书籍(尽管对于最贪婪的读者来说有一些限制)。
