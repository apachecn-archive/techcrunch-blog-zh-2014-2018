# Stampery 现在允许你使用区块链和你的真实身份认证文件

在[参加](https://web.archive.org/web/20230104094623/https://techcrunch.com/2015/09/22/stampery-leverages-the-blockchain-to-certify-all-your-documents/)Disrupt SF 的战地比赛后不久， [Stampery](https://web.archive.org/web/20230104094623/https://stampery.co/) 从 [Draper Associates](https://web.archive.org/web/20230104094623/https://www.crunchbase.com/organization/draper-associates-2) 处筹集了 60 万美元，同时 [Boost VC](https://web.archive.org/web/20230104094623/https://www.crunchbase.com/organization/boostfunder) 、 [Blockchain Capital 和](https://web.archive.org/web/20230104094623/https://www.crunchbase.com/organization/crypto-currency-partners) [Di -Ann Eisnor](https://web.archive.org/web/20230104094623/https://www.crunchbase.com/person/di-ann-eisnor) 也参加了比赛。

提醒一下，Stampery 允许您通过向您的个人 Stampery 电子邮件地址发送电子邮件附件来认证任何文件。你也可以使用该公司的网站，通过 API 将 Stampery 集成到你的产品中，或者直接在你的 Dropbox 帐户中认证你的文档。Stampery 也在开发其他服务，比如 Box。

该公司计划通过利用比特币的区块链来取代公证人。Stampery 为您所有的敏感文件出具具有法律约束力的证明。如果您需要证明您稍后正在查看未经修改的文档，您可以通过探索区块链来证明该文档的存在性、完整性和所有权。

像优秀的老公证人一样，Stampery 可以帮助你处理知识产权案件、遗嘱、誓言、合同等等。Stampery 希望通过使认证文件变得更加容易，这将成为一种习惯。你可以在电脑上做任何事情，你不必亲自去见公证人。

最近，该公司更新了其 API，允许任何人在区块链无限量的文件上盖章。Stampery 还介绍了一种证明文档作者真实身份的新方法。该公司将政府 ID 与区块链关联起来，以便以后检索该 ID，并使用真实的、经过验证的身份对文档进行公证。

这个功能让人想起 [ShoCard](https://web.archive.org/web/20230104094623/https://techcrunch.com/2015/05/05/shocard-is-a-digital-identity-card-on-the-blockchain/) 。这是 Stampery 产品中缺失的一个重要部分。

使用区块链使入侵 Stampery 变得更加困难，因为校样存储在区块链——你不能修改区块链，因为它是一个分布式数据库。Stampery 的好处还在于，任何独立的第三方都可以验证公司的样张。50 年后 Stampery 不存在也没关系。只要区块链还在，Stampery 校样就会继续存在。
