# 安全摄像头金丝雀增加了包裹检测和亚马逊 Alexa 的新技能

Canary 一直在通过软件更新的方式稳步升级其安全摄像头系列，增加了新功能，如[双向通话](https://web.archive.org/web/20230117181140/https://techcrunch.com/2017/08/14/canary-finally-brings-two-way-talk-to-its-security-cameras/)和[人物检测](https://web.archive.org/web/20230117181140/https://techcrunch.com/2017/12/14/canary-brings-person-detection-to-its-cameras-to-help-avoid-false-positives/)。今天，它准备再次这样做，宣布另一项人工智能驱动的活动检测功能，称为包裹检测，也是亚马逊 Alexa 的一项技能。Alexa 集成允许您通过说话将视频从您的金丝雀传输到您的 Echo Spot、Echo Show、Fire TV 或 Fire TV Stick。

新的包裹检测功能使用了与去年 12 月推出的早期人员检测功能相同的人工智能和机器学习技术。但是这一次，这项技术被用来提醒用户包裹已经到达他们家门口。

不幸的是，这一新增功能只提供给[金丝雀的室内/室外相机 Canary Flex](https://web.archive.org/web/20230117181140/https://techcrunch.com/2016/12/22/canary-flex-2/) 的用户，而不是原来的金丝雀。它将在 2018 年上半年通过软件更新推出，但只针对金丝雀的[付费会员](https://web.archive.org/web/20230117181140/https://canary.is/membership/)。该公司在去年秋天推出了 9.99 美元/月的会员选项，这使得它对一些过去免费的服务收费，如夜间模式，这让它的用户非常懊恼。(金丝雀在听取了顾客的反馈后，于 12 月重新推出了它。)

随着时间的推移，金丝雀应该为会员选项增加更多的价值，以说服人们升级。

![](img/193ad2a3d416341388504781fdcd4f81.png)

与此同时，Canary 的亚马逊 Alexa 集成将更加广泛。

此次发布遵循了其他第三方集成，包括与谷歌助手和 Wink 的集成，并将对所有 Canary 设备和用户开放，而不仅仅是其付费用户。

该公司的新 Alexa 技能将允许客户说类似“Alexa，让我看看客厅”的话，以开始从支持的设备上播放视频。能够支持最受欢迎的语音扬声器平台对于任何联网相机制造商来说都是至关重要的，特别是考虑到亚马逊自己已经凭借其价格实惠(如果有些基础的话)的云摄像头进入了这一领域。

Canary 表示，Alexa 集成将在“未来几个月”推出，但没有提供更具体的时间框架或日期。
