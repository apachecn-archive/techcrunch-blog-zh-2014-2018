WordPress.com开源并获得桌面应用TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/11/23/wordpress-com-goes-open-source-and-gets-a-desktop-app/

WordPress的完全托管版本WordPress.com今天接受了有史以来最大的一次更新。代号为的Calypso ，Automattic从零开始改写了WordPress.com——引擎盖下的一切都是新的。以下是重大变化。

首先，WordPress.com现在已经完全脱离了WordPress的核心。WordPress.com现在是一个与WordPress核心交互的管理界面，就像任何其他第三方界面和应用程序一样。它使用REST API来获取你的文章，发布新文章，上传照片等等。

第二，WordPress.com背后的团队换了一个全新的堆栈。开发人员没有使用PHP和MySQL，而是使用JavaScript和API调用来构建一切。这意味着当你访问网站时，服务器会分发一个完全可用的WordPress客户端，它大部分运行在你的浏览器中。

这是一个单页面应用程序，这意味着当你与界面交互时，你将得到很少的加载屏幕。它应该在你的手机和平板电脑上也能很好地工作——一切都很灵敏。如果你使用的是WordPress管理后端，你仍然可以直接进入你的后端。但是在WordPress.com，如果你正在使用一个托管的WordPress.com博客，一个自带Jetpack插件的WordPress或者一个WordPress VIP网站(比如TechCrunch ),你还有另外一个选择。

最后，一切都是开源的，并且在 GitHub 上。只要遵守GNU通用公共许可证版本2，您就可以查看代码、派生代码和重用代码。

但是团队并没有就此止步。你也可以下载一个新的 Mac应用来访问WordPress.com。在许多方面，这个应用程序的工作方式类似于Slack桌面应用程序。它利用了网络技术和桌面功能，所以你得到的东西或多或少与WordPress.com网站上的完全一样，但有一些好处，比如通知。Windows和Linux应用程序正在开发中。

我下载了应用程序，玩了几分钟。如果你熟悉WordPress.com的界面，你会有宾至如归的感觉，因为它看起来完全一样。但是在Dock中有一个应用程序图标总是很好的。

那么，为什么WordPress.com背后的公司Automattic要经历这个痛苦的重写过程呢？WordPress.com现在的感觉和工作就像一个现代的网络应用程序。它又回到了游戏中对抗新人，比如中型。

虽然编辑器缺乏WordPress超级用户(包括TechCrunch作家)使用的许多功能，但WordPress是一个干净、高效的写作界面，应该会吸引许多今天在媒体上写作的人。

今天25%的网络运行在WordPress 上。这不是一个小壮举，WordPress不是对抗大公司的年轻、活跃的初创公司——它是一个网络巨头。通过今天的举动，Automattic证明了它仍然意识到自己的环境和潜在威胁。对于WordPress的未来来说，这是一个令人鼓舞的信号。
