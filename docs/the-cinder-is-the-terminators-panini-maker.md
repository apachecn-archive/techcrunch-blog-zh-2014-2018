# 煤渣是终结者的帕尼尼制造者

如果你想烤焦，就跟我来。两年前[第一次看到的](https://web.archive.org/web/20221025222320/https://beta.techcrunch.com/2015/04/25/slideshow-the-cinder-sensing-cooker-wont-burn-your-food-to-a-crisp/)煤渣，现在正在 Indiegogo 上销售，我有了一个独特的机会尝试这个聪明的真空烤肉炉。

基本上煤渣可以完美地烹饪肉类。你把一大块肉放在内部浅盘上，设定温度，然后合上沉重的盖子。盖子就像一种真空密封，可以保存果汁，而食槽可以让脂肪排出，进入一个小收集器。最后，当肉熟透时，它会完全烤焦肉。煤渣的价格是 499 美元，用于 7 月份的早鸟和船只。

那么它是如何工作的呢？我用鸡肉和牛排测试了它，发现它很容易使用，尽管有点笨重。整个包装大约 13 英寸×17 英寸×7 英寸，带有一个大而重的顶板。它不适合存放在小公寓里，但如果你有很多柜台空间，它就很好用。

它是如何烹饪的？非常好。鸡肉做得很好，非常多汁，而且很结实，特别是经过了一次很好的烤焦。我还用它做了一些牛排，得到了一块烤得很好的干肉，可以做一些很好的墨西哥玉米卷。

![](img/59d9254090f1adf46b68f66236329022.png)

要澄清的是，我没有用一块好肉。我做的裙边牛排像钉子一样坚韧，得到的肉比我放进机器里的厚片要好。它花了大约 25 分钟——你可以在这里[阅读完整的食谱](https://web.archive.org/web/20221025222320/https://blog.cindercooks.com/skirt-steak-fajitas-precision-cooked-2bcf99a1b921)——虽然看起来有点过火，但烤焦了，内部的肉也很好。简而言之，它做了一块很好的牛排，没有什么大惊小怪的。

![](img/9541b43e7f57630d0d83051a82868015.png)

我喜欢 sous vide 牛排和煤渣，虽然肉不是在真正的真空中，但模拟 sous vide 的体验相当好。如果你做很多肉，不想经历取出铸铁来烹饪和烤焦厚牛排的繁琐程序，我怀疑煤渣会很好。它很大而且很笨重，所以如果你打算把它放在你的厨房里，给它一个荣耀的位置，不要期望移动它。

这个东西的样式、可用性和构建质量都是一流的，尤其是对于一个独立的硬件项目来说。我向任何需要肉的人推荐这种做法，虽然你可以在这种动物中过度烹饪，但我怀疑只要稍微计划一下，你就会发现自己每次都有一块烹饪得很好的烤焦的肉。
