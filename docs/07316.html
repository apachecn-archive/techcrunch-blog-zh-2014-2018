印度数字支付公司 TranServ 获得 1500 万美元用于推出新的金融产品

TranServ 是印度几家实现支付数字化的公司之一，它今天表示，它已经获得了 1500 万美元的 C 轮融资，用于开发新的金融产品。这一轮由 IDFC Spice 基金和 Micromax Informatics 牵头。回归投资人 Nirvana 和 Faering Capital India 也参与其中。

Micromax 是印度市场份额最大的智能手机制造商之一，也是这家初创公司的战略合作伙伴之一。这两家公司最近与 Visa 签署了一项协议，提供移动支付服务，包括 Micromax 设备上基于 NFC 的非接触式支付。

TranServ 声称总共有 1000 万用户。它通过其 API 每月处理约 1500 万美元的交易，这些 API 允许商家将支付工具集成到自己的软件中，客户包括 BookMyShow 、 PayU 、 Ixigo 和 Askmebazaar 。

该公司计划推出新产品，包括小额贷款服务，其 API 堆栈的新非支付功能，以及仍向员工发放纸质优惠券的公司的在线餐券。它还将继续开发其移动钱包应用程序。

TranServ 的联合创始人兼首席执行官 Anish Williams 表示，Udio 是印度首批点对点交易应用之一，它允许用户分摊账单并相互汇款。它还可以链接到大约一百万个线下商家接受的预付费借记卡。

几个因素推动了印度对数字支付的需求，包括印度电子商务行业的快速增长和 T2 智能手机的普及。然而，该国的信用卡普及率仍然很低，这意味着公司需要帮助消费者找到在线支付的替代方式。

通过提供可以链接到银行账户或在实体店充值的移动钱包等工具，初创公司已经介入其中，包括 Mobikwik 和 Paytm 。

TranServ 联合创始人兼首席执行官 Anish Williams 希望他的公司推出新功能和产品以吸引客户，并对优质服务收费的竞争战略，将通过避免折扣和返现交易等昂贵的促销工具，让这家初创公司“保持财务纪律”。
