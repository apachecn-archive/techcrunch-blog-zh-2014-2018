# 走进 Syfy 的“魔术师”——推广魔术厅

Syfy 的奇幻秀 *[魔术师](https://web.archive.org/web/20230331015012/http://www.syfy.com/themagicians)* 今晚拉开了第二季的序幕，但在接下来的几天里，纽约市居民也有机会探索与该节目相关的现实世界装置。

上周，我参观了[魔法馆](https://web.archive.org/web/20230331015012/http://www.hallofmagicnyc.com/)，那里的每个房间都提供了不同类型的奇幻主题体验。其中一些只是提供了有趣的噱头，比如拍照的能力，让它看起来像你在漂浮。其他一些则更令人印象深刻——比如一个战斗室，你可以指着不同的物体，看它们对你的神奇能量做出反应，或者一个教室，你可以和你的朋友进行智力竞赛，把球推过桌子。

自然，实际上是技术推动了这些体验——比如可以告诉你手指向哪里的 Kinect 传感器，或者在推球智力竞赛中使用的 [Muse](https://web.archive.org/web/20230331015012/http://www.choosemuse.com/) 头带和磁铁。创造了魔法大厅的 Mash Studio 的埃里克·弗莱明(Eric Fleming)称赞 tech 允许他的团队“以如此低的成本和如此快的速度做这么多疯狂的事情”，尽管他补充道，“我们不只是为了技术而做华而不实的技术。我们从不从技术开始，而是从我们想要创造的体验开始。”

NBC 环球的 Alexandra Shapiro 补充说，虽然与展览相比，魔术式装置大厅只会吸引一小部分观众(特别是因为它只开放到 1 月 29 日)，但 Syfy 认为这样的促销活动是与粉丝真正联系的一种方式。

“如果做得正确，如果做得真实，如果你注意细节……伟大会得到回报，”夏皮罗说。“反过来，粉丝们也爱它。这种互惠，这种动力，这种互动就是粉丝圈的建立方式。”
