# Sendwithus 筹集 230 万美元帮助营销人员优化电子邮件 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/09/25/sendwithus-seed-funding/>

电子邮件营销初创公司 [Sendwithus](https://web.archive.org/web/20221208075230/https://www.sendwithus.com/) 宣布，它已经筹集了 230 万美元的额外种子资金。

这是在交易几个月后发布的融资公告之一。联合创始人马特·哈里斯告诉我，这家由 Y Combinator 孵化的公司“就在 Y Combinator 启动之前”筹集了初始种子资金，然后在 3 月的演示日筹集了额外资金[。](https://web.archive.org/web/20221208075230/https://beta.techcrunch.com/2014/03/25/yc-w14-batch-1/)

Sendwithus 团队希望现在宣布资助，部分是为了突出其新功能和迄今为止所取得的成功。当[在今年早些时候写关于公司](https://web.archive.org/web/20221208075230/https://beta.techcrunch.com/2014/02/07/sendwithus-y-combinator/)的文章时，它提供了 A/B 测试等工具来优化交易电子邮件——即用于特定目的并与特定事件相关联的电子邮件，如“重置你的密码”和“感谢注册！”(该服务集成了 SendGrid 和 Mailgun 等电子邮件服务。)

哈里斯承认，专注于事务性电子邮件听起来“超级无聊”。然而，每个公司都会发送这些邮件，客户更有可能会打开它们，而这些邮件是“大多数营销人员不会接触到的，甚至想都不会想的。”Sendwithus 允许营销人员测试这些信息的不同变体，而无需通过开发者。他还说，在较大的公司，发送这些电子邮件“成为管理、生产和自动化的真正痛苦…所以我们有效地做的是将这些系统产品化，比这些公司自己做的更好。”

最近，Sendwithus 增加了细分功能，允许企业深入客户数据库中的特定群体，例如收到激活电子邮件但没有激活的用户。

哈里斯说:“最终结果是，这家公司不费吹灰之力就能向特定的客户群发送真正个性化的电子邮件。”。

显然，该公司在过去八个月中每月收入增长 30%，该产品已被用于优化 2.5 亿封电子邮件。

与此同时，这笔资金由 Baseline Ventures 的 Steve Anderson 牵头，参与方包括 SV Angel、Maiden Lane、Garry Tan、Alexis Ohanian、Paul Buchheit、Kevin Hale (Tan、Ohanian、Buchheit 和 Hale 都是 YC 的合伙人)、Scott Banister 等人。Sendiwthus 现在已经筹集了 260 万美元。
