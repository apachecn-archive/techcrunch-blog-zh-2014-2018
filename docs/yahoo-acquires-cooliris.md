# 雅虎收购摄影初创公司 Cooliris

照片应用程序制造商 Cooliris 刚刚宣布被雅虎收购。

通往这个出口的路很长。Cooliris 成立于 2006 年，最初因创建了一个用于浏览照片和其他媒体内容的“ [3D 墙](https://web.archive.org/web/20230305215528/http://venturebeat.com/2008/09/24/embeddable-wall-brings-cooliris-to-any-website/)而闻名。

它还创建了一个名为 Adjitsu 的移动广告平台，并于 2012 年将其出售给新加坡电信的 Amobee 部门~~[。](https://web.archive.org/web/20230305215528/https://techcrunch.com/2012/05/08/singtels-amobee-buys-adjitsu-to-take-mobile-ads-into-the-third-dimension/)~~

最近，该公司将重点转移到一个移动应用程序(也叫 Cooliris，界面让人想起旧的 3D 墙)，允许用户浏览来自脸书、Flickr 和 Dropbox 等服务的照片。该团队去年告诉我，由于与人人网、Yandex 和百度等公司的合作，他们在亚洲的增长尤为迅速。

今年夏天，Cooliris 还推出了一款名为 BeamIt 的图片信息应用。

该公司已经从包括 Kleiner Perkins cau field Byers、德国电信的 T-Venture、Westly Group 和 DAG Ventures 在内的投资者那里筹集了总计 2760 万美元的资金。

该公司在其网站上写道:

> 雅虎有一个清晰的愿景，并坚定不移地致力于使移动成为一种直观和轻松的体验。
> 
> 这使得雅虎成为 Cooliris 的完美合作伙伴，我们很高兴能够走到一起，为全球观众带来不可或缺的产品。

在首席执行官玛丽莎·梅耶尔(Marissa Mayer)的领导下，雅虎确实谈了很多关于其对移动业务的关注，就在上个月，它收购了移动通讯应用 MessageMe。

Cooliris 的收购公告没有太具体地说明雅虎是否计划对现有产品做些什么，或者这更像是一次人才收购——它只是说，Cooliris 的移动应用程序和 BeamIt Messenger 将“暂时”继续运营。

更新:以下是雅虎的声明:

> 我们抓到了库里斯。为了打造令人振奋的产品、增加参与度并最终增加收入，一切都始于拥有最优秀的人才来帮助我们加快增长领域的转型。因此，我们正专注于与我们的关键增长领域一致的收购:搜索、通信、数字杂志和视频。我们非常高兴地欢迎来自 Cooliris 的 17 名员工来到我们核心沟通团队所在的桑尼维尔。在不久的将来，Cooliris 产品不会有任何改变。
