# 红杉支持 Simplisafe 进军家庭安全市场 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2014/04/21/sequoia-backs-simplisafe-in-a-play-for-the-home-security-market/>

TechCrunch 了解到，红杉资本(Sequoia Capita)已经对 DIY 家庭安全公司 [Simplisafe](https://web.archive.org/web/20230131011221/http://www.crunchbase.com/company/simplisafe) 进行了少量投资，以进军家庭安全和自动化市场。

红杉资本的投资紧随[谷歌 32 亿美元收购 Nest Labs](https://web.archive.org/web/20230131011221/https://techcrunch.com/2014/02/06/googles-3-2-billion-acquisition-of-nest-gets-the-ftc-green-light/) 之后，这显示了本土市场是多么有利可图。这笔交易为投资者带来了丰厚的回报，包括凯鹏华盈(Kleiner Perkins cau field)&、拜尔斯(Byers)、[光速风险投资伙伴](https://web.archive.org/web/20230131011221/http://www.crunchbase.com/financial-organization/lightspeed-venture-partners)、[世代投资管理](https://web.archive.org/web/20230131011221/http://www.crunchbase.com/financial-organization/generation-investment-management)、[文洛克](https://web.archive.org/web/20230131011221/http://www.crunchbase.com/financial-organization/venrock)和谷歌自己的[谷歌风险投资](https://web.archive.org/web/20230131011221/http://www.crunchbase.com/financial-organization/google-ventures)。

对 Nest 的收购也标志着大型科技公司首次努力打开家庭自动化的大门。投资者认为这可能不会是最后一次。

甚至在 Nest 表现出企业对家庭自动化的兴趣之前——包括供暖和制冷的恒温器控制、家庭安全以及其他类型的远程监控和管理——投资者已经开始支持一系列解决这个问题的创业公司。

现在红杉拥有了自己的投资组合公司来应对市场。

据了解投资的多个消息来源称，该公司在一场激烈的竞购战后，通过投资银行 Raymond James 处理的一个过程，投资了 Simplisafe。红杉资本拒绝对本文置评。

Simplisafe 的开箱即用家庭安全系统[获得了好评](https://web.archive.org/web/20230131011221/http://www.apartmenttherapy.com/simplisafe-home-security-syste-99778)，并且已经安装在数千个家庭中。

尽管 Simplisafe 通过新的红杉融资获得了巨大的推动，但它并不是唯一一家吸引了大牌风险投资的进入者。今年早些时候，Khosla Ventures 向 Canary 投资了 1000 万美元，这是另一个高概念、设计良好的安全系统。根据来自 [CrunchBase](https://web.archive.org/web/20230131011221/http://www.crunchbase.com/) 的数据，来自投资者的资本持续流入。

一些投资者可能已经下了早期赌注，但 DIY 家庭安全和自动化市场仍然敞开着，支持尚未筹集任何机构投资的私营公司的新机会比比皆是。

例如，总部位于芝加哥的公司 [Scout](https://web.archive.org/web/20230131011221/https://www.scoutalarm.com/) 已经在 Kickstarter 上成功完成了一项活动；FrontPoint Security 为家庭提供一整套自动化安全技术；LiveWatch Security 还为房主和公寓居民提供了新的安全技术，如果他们住在一个城堡多于家的地方，他们会感到更加舒适。

“这个领域有很多投资，”Lemnos 实验室创始人杰里米·康拉德说。“事实是，每个人都在关注 ADT，以及他们如何为一项 20 年来从未改变的服务带来数十亿美元的收入。这与 Nest 在恒温器领域追赶霍尼韦尔的方式非常相似。”
