# Xage 推出指纹识别技术保护工业物联网设备

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/05/14/xage-introduces-fingerprinting-to-protect-industrial-iot-devices/>

随着石油和天然气等老派行业越来越多地将石油平台等实体网络化，它们变得更容易受到黑客攻击，而这在它们独立时是不可能的。这需要一种新的安全方法，去年[推出的安全初创公司](https://web.archive.org/web/20230225042901/https://techcrunch.com/2017/12/14/xage-emerges-from-stealth-with-a-blockchain-based-iot-security-solution/)认为，结合区块链的“指纹识别”概念，它有了答案。

Xage 的首席执行官邓肯·格里特伍德解释说:“每个人的指纹都试图反映尽可能多的关于设备或控制器的信息。”它们通过存储来自网络上每个设备和控制器的配置数据来实现这一点。这包括硬件类型、其上安装的软件、CPU ID、存储 ID 等等。

如果有人试图将恶意软件注入这些控制器中的一个，指纹识别将会注意到变化并关闭它，直到人类技术人员能够确定这是不是合法的变化。

## 区块链向何处去？

你可能想知道区块链是从哪里来的，但是想象一下这些指纹的蜜罐存储在传统的数据库中。如果数据库遭到破坏，这将意味着黑客可以访问一家公司的整个指纹库，完全否定了这一想法。这就是区块链出现的原因。

Greatwood 表示，防止这种情况发生有多种目的。首先，它带走了那个集中的蜜罐。它还提供了一种认证手段，使得在没有明确许可的情况下不可能插入假指纹。

但他说，Xage 采取了一项与区块链无关的预防措施，以允许对控制器进行合法更新。“我们有一个保存在云中的系统的数字副本(双胞胎)，所以如果有人改变软件或计划在设备或控制器上改变它，我们将在更新控制器之前预先计算新的指纹，”他说。这将使他们能够了解何时有经批准的更新发生，而不是外部威胁代理试图模仿更新。

## 政府机关彼此之间的相互制衡

通过这种方式，他们检查每个指纹的有效性，并对每一步进行检查和平衡。如果更新后的指纹与云副本匹配，他们就可以合理地确信它是真实的。他说，如果没有，他们认为指纹可能已经被黑客攻击，并关闭它，供客户进一步调查。

虽然这听起来像是保护这种基础设施的一种复杂方式，但 Greatwood 指出，这些设备和控制器在配置方面往往相当简单，不像管理工作站网络的安全性那样复杂，因为黑客可能会有许多接入点。

具有讽刺意味的是，这些公司正在将其设备联网以简化维护，但在这样做的同时，他们也产生了一系列新的问题。“这是一个非常有趣的问题。他们正在采用物联网，所以他们不必做[这么多]卡车翻滚。他们想要这种网络能力，但黑客攻击的风险更大，因为只需要一次黑客攻击就可以访问数千个控制器，”他解释道。

如果你认为他们可能夸大了石油钻井平台和其他工业目标被黑客攻击的实际问题，国土安全部 3 月份发布的一份报告表明，能源部门近年来一直是国家黑客感兴趣的领域。
