特斯拉回应 Model 3 生产问题的最新报告 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/01/25/tesla-responds-to-fresh-reports-of-model-3-production-issues/?ncid=mobilerecirc_recent

特斯拉 Model 3 EV 的延迟是众所周知的，包括该公司自己的财报和季度交付数据。周四来自美国消费者新闻与商业频道的一份最新报告称，Model 3 的生产比看起来还要推迟，电池制造方面仍存在一些具体问题。

美国消费者新闻与商业频道还暗示，该公司“还没有接近”大规模生产其 3.5 万美元的 trim level Model 3 所需的电池。特斯拉最初对新闻媒体做出了回应，指出他们之前对特斯拉首席执行官埃隆·马斯克和首席技术官 JB·斯特劳贝尔关于其当前延迟和生产瓶颈的评论，但现在该公司发布了一份更广泛的声明，断然驳斥了新报告中的说法。

特斯拉特别反对其质量控制人员缺乏必要经验的说法，或其偷工减料可能导致与电池相关的危险。特斯拉的声明全文如下:

这是一篇极其误传和误导的文章。绝对明确的是，我们正在按照本月早些时候提供的先前预测实现 Model 3 生产率的提高。正如已经被很好地记录的那样，在我们达到全面生产之前，按照定义，生产过程的一些要素将更加手工化。这是 Elon 和 JB 在我们的第三季度收益电话会议上广泛讨论的事情，它对我们生产的电池的质量或安全性没有影响。正如我们在第四季度发布的交付报告中所指出的，在第四季度，“我们在解决 Model 3 生产瓶颈方面取得了重大进展，我们的生产率在本季度末大幅提高。”

此外，正如制造业中经常出现的情况，生产过程的某些部分需要具有工程或制造经验的员工的专业知识，而其他部分则不需要。近年来，我们在内华达州创造了数千个新的高质量工作岗位。随着我们继续扩大 Gigafactory 1 和 ramp Model 3 的生产，我们能够为数千名新员工传授新技能，其中许多人在加入特斯拉之前没有任何制造经验。模块生产线上的新员工接受广泛的培训，包括安全培训，并了解适当的电池间距的重要性，以便他们能够识别生产过程中的此类问题。更广泛地说，电池生产——尤其是模块生产线——由我们的顶级工程人才和特斯拉的许多最高层领导监督。

最后，暗示特斯拉会交付带有危险电池的汽车是绝对不准确的，违背了所有证据，也脱离了现实。根据未透露姓名的匿名消息来源提出如此多的建议是不负责任的，这些消息来源没有提供任何此类证据，并且显然没有完全了解特斯拉汽车中所有电池都要接受的广泛测试。与安全记录良好的 Model S 和 Model X 一样，我们对 Model 3 电池保持严格的质量和过程控制方法。更重要的是，据我们所知，在 Model 3 生产的六个月中，没有一个与 Model 3 电池相关的安全问题。

至于关于 Model 3 电池中电池接触的断言，这是极其误导的，并且显示出对我们的电池如何工作的基本知识的完全缺乏。特斯拉汽车中的每个电池都有数千个电池单元，其中绝大多数与相邻的电池单元具有相同的电压。假设，即使两个具有相同电势的电池接触，也绝对不会产生安全或其他影响——就好像两个中性金属片接触一样。尽管如此，所有型号 3 电池模块的电池位置在制造过程中都要测量两次，以验证过程控制和出厂零件的质量。相反，如果在生产过程中的任何一点上，电池处于不同的电压电位，它们就不能相互电连接。在生产过程中，我们进行了三种不同的测试，以确保型号 3 模块中的电池数量正确。此外，电池位置的长期可靠性是通过测试验证的，包括冲击和振动、高温和高湿测试，以及贯穿整个设计和通过生产取样的热循环耐久性测试。所有这些测试都是为了防止触摸电池安装在我们的任何车辆上，包括 Model 3。最后，我们的模块设计的安全方面将继续发挥作用，即使在接触细胞的情况下，所以提出的担忧是毫无根据的。

尽管我们有制造道路上最安全车辆的历史，Model S 和 Model X 不仅在整体上，而且在每个子类别中都获得了五星安全评级，但这些错误的说法仍然存在。虽然还没有经过 NHTSA 的测试，但 Model 3 的设计和内部测试具有相同的效果。NHTSA 测试的数据显示，S 型和 X 型是 NHTSA 测试历史上受伤概率最低的两种车型。此外，在数十亿英里的实际驾驶中，特斯拉的车辆发生火灾的可能性大约比传统汽油车低五倍。鉴于这些事实，建议像特斯拉这样致力于安全的公司允许未经测试或不安全的电池进入我们的车辆是荒谬的。

就分析师、股东和利益相关者的压力而言，特斯拉的 Model 3 生产可能是其最大的问题。这家汽车制造商大规模交付最实惠的电动汽车的能力，以及在适当的时候这样做的能力，可能是目前任何人怀疑其长期成功的最大原因，因此可以理解的是，它对围绕产品进展的报告的准确性很敏感。
