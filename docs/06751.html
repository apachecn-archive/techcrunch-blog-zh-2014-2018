人工智能是最近最热门的话题之一，最近的技术进步使人工智能比我们大多数人想象的更接近现实。

去年，史蒂芬·霍金、埃隆·马斯克和 1000 多名人工智能和机器人研究人员签署了一封公开信，就人工智能在武器开发中的使用发出警告，这一主题真正受到了关注。接下来的一个月， BAE 系统公司发布了有史以来最先进的无人驾驶飞行器 Tara NIS；目前有 40 个国家致力于在武器开发中部署人工智能。

国防工业中的那些人不是唯一参与军备竞赛以创造高级人工智能的人。科技巨头脸书、谷歌、微软和 IBM 都在参与各种人工智能计划，并在开发数字个人助理方面展开竞争，如脸书的 M、微软的 Cortana 和苹果的 Siri。

马克·扎克伯格甚至想创造出他自己版本的《钢铁侠》中的贾维斯来管理他的家。在今年的达沃斯世界经济论坛上，有人指出人工智能正在迎来第四次工业革命，正如我们所知，这将改变社会，到 2020 年将损失 500 万个工作岗位。

机器人不再局限于传统的蓝领工作、全自动流水线和高频交易算法。白领工作自动化的时机已经成熟，机器人正在取代金融行业的银行出纳员、抵押贷款经纪人和信贷员。这些例子遵循严格重复的基于规则的例程，并且机器可以很容易地执行它们，而无需任何人类交互。

然而，最近的一项发展是人工智能新时代的开始，在这个时代中，人工智能可以执行复杂的任务，并且必须不再依赖预编程的规则进行决策。像 Betterment 和 Wealthfront 这样的机器人顾问服务越来越受欢迎，对冲基金行业正在推出完全没有人类互动的人工智能控制的基金。其中一个基金的联席主管预测没有人类投资经理能打败计算机的时代将会到来。但是 AI 在没有任何人类交互的情况下，怎么可能自主操作呢？

通往真正人工智能的道路不是由单一学科铺就的。

机器学习是人工智能背后的基础之一，由 Arthur Samuel 在 1959 年定义为让计算机在没有明确编程的情况下学习和行动的科学。这项技术是自动驾驶汽车、 IBM Watson 和语音及图像识别开发中不可或缺的一部分，也是解决我们一些最具挑战性的任务的一部分，比如理解人类基因组。

机器学习植根于统计模式识别，是许多日常应用和服务的基础，如垃圾邮件过滤器和网络搜索算法。机器学习的基本方面是让计算机程序从例子中学习。为了加速机器学习的发展，谷歌在 GitHub 上发布了其机器学习系统 TensorFlow，这导致微软紧随其后。

深度学习将机器学习的概念更加深入(双关语)，可以对由许多层组成的复杂非线性关系进行建模。深度学习经常与人工/深度神经网络一起被交替提及，后者可以被视为一种受生物启发的编程范式，使计算机能够从观察数据中学习。深度学习被认为是我们应用于神经网络学习的技术。

量子计算是人工智能发展中最新最热的。谷歌表示，他们与美国宇航局合作开发了一台比传统计算机快 1 亿倍的量子计算机。理论上，D-Wave 2X 可以在几秒钟内完成一个问题的计算，而这个问题可能需要数字计算机 10，000 年才能完成。然而，谷歌表示，量子计算可能不适合深度学习。

传统计算机依赖于 1 或 0 位，而量子计算机基于量子位，可以保持叠加，同时为 1 和 0。这种状态使得量子计算机能够以指数速度处理数据。虽然量子计算可能不适合深度学习，但它可能会彻底改变物流、投资策略以及能源生产和消费的优化领域。

通往真正人工智能的道路不是由单一学科铺成的，而是由专业主题、技术和理论的集合，它们相互作用，创造出某种形式的智能。
