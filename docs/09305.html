雅虎开放其色情探测神经网络 TechCrunch

有没有想过互联网上的东西怎么会被标上 NSFW？是雅虎。雅虎做到了——用他们特制的，经过色情训练的，检测色情的神经网络。现在你也可以了，因为系统背后的团队已经把它开源了。我想你可以说他们到了岔路口。

当然，我开玩笑说:雅虎的算法并没有做到这一切。事实上，探测 NSFW 图像是一个众所周知的难题。套用一句名言，当你看到它的时候你就知道了，但是你——承认吧——当你做出这样的分类时，你有一生的时间来观看色情作品。

然而，机器是完全无辜的——或者曾经是，直到雅虎通过在数以千计的色情图片上训练图像识别引擎而败坏了它们。他们永远无法回到我们从他们那里偷走的优雅状态——但另一方面，你的在线搜索不太可能随机包含性敏感区。

不过，说真的，卷积神经网络是一种优秀的图像分类工具，过去几年的研究已经一次又一次地证明了这一点。在接受了特定类型图像的数据库训练后，他们的算法会锁定特定的模式——如果是狗，他们会学习看到尾巴、鼻子和口鼻。如果是汽车，他们会认出轮子，门把手，格栅。如果是色情，那就发挥你的想象力。

结果是一个系统可以梳理大量不同的图像，并给每一个图像打分，从 0 到 1，说明它认为特定图像的 NSFW 程度。这在很多情况下都是有用的，不仅仅是审查方面。令人兴奋的图像在其位置上是很好的，但能够筛选出来是有用的，例如，当处理直接来自网络的大型数据集时。

这种系统还可以检查你的电子邮件和信息，而无需任何人工干预或(一些人会认为)任何侵犯隐私的行为，并在图像可能是 NSFW 时警告你(当同事试图恶作剧时，这很方便)。

你需要提供你自己的色情片来训练模特，但是如果你已经读到这里，我怀疑这将是一个问题。在雅虎博客上有更多细节，在 GitHub 上可以下载模型。
