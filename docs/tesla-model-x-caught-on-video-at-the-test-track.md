# 特斯拉 X 型在测试跑道上被视频捕捉到

[YouTube https://www.youtube.com/watch?v = xtf qvqn1n 5 p 4]

阿拉梅达机场试验跑道上，一辆新型特斯拉被摄像机捕捉到(通过[electreek](https://web.archive.org/web/20230320020401/http://electrek.co/2015/01/30/model-x-testing-video-at-alameda-airport-shows-side-mirrors-and-larger-backside/))。这款车看起来很像普锐斯，或者是一款跨界多功能车，很有可能是特斯拉已经研发了多年的 X 型 SUV，或者像其他 T2 人所说的 T3 型，这是大众市场 3 型的原型，未来计划以 3.5 万美元的价格投放市场。

除了一辆车以相对较高的速度直行和穿过一些宽阔的环路，视频中没有显示什么，还有一辆伪装的车身，车身有一个宽阔的后端，看起来司机的座位安排相当高，这确实表明这是一辆 X 型测试车。在设计、车身形状和外观尺寸方面，它与埃隆·马斯克(Elon Musk)于 2013 年发布的特斯拉 X 型原型机也有许多共同之处。

[YouTube https://www.youtube.com/watch?v =那边 2eMsPHU]

我认为这很可能是 X 型，因为上面提到的几点，也因为特斯拉首席执行官埃隆·马斯克本月早些时候说，X 型有望在 2015 年下半年发布，这意味着它应该像这部间谍视频中的车辆一样，做好上路准备并最终设计。这意味着电动汽车迷们该兴奋起来了，他们想要更多的载客量。
