# 科技泡沫在硅谷之外意味着什么

它是一个地方，一种思维模式，一个昵称，一个转喻，一个创新中心，一个电视节目和一个衡量所有技术摇篮的标准。硅谷引以为豪的是，它拥有一种挑战简单分类的元地位，但这无法将它从即将到来的泡沫中拯救出来。根据像[马克·库班](https://web.archive.org/web/20230130021050/http://blogmaverick.com/2015/03/04/why-this-tech-bubble-is-worse-than-the-tech-bubble-of-2000/)、[克里斯·萨卡](https://web.archive.org/web/20230130021050/http://www.bloomberg.com/news/articles/2015-06-11/uber-investor-chris-sacca-predicts-technology-industry-decline)、[比尔·格利](https://web.archive.org/web/20230130021050/http://fortune.com/2015/03/15/bill-gurley-predicts-dead-unicorns-in-startup-land-this-year/)、[马尼什·戈亚尔和巴拉特·拉姆纳尼](https://web.archive.org/web/20230130021050/https://techcrunch.com/2015/05/24/who-will-be-hurt-most-when-the-tech-bubble-burst-not-vcs/#.vag8gc:cyZh)这样的投资者的说法，泡沫的存在已经不再有疑问——问题只是它什么时候会破裂以及大屠杀会有多严重。

迄今为止，科技股泡沫的讨论就像硅谷一样自我参照。它围绕着投资者已经投入数十亿的 117 只“独角兽”展开。根据 Sacca 的说法，无论所有的愚蠢行为是由 Gurley 所说的“FOMO”(害怕错过)，还是“傲慢、错误的决策或对后果的天真”所驱动，泡沫远远超出了 117 家公司、一个山谷及其投资者的焦虑。

一旦破裂，科技泡沫将会像末日小行星的冲击波一样波及整个科技世界。大多数技术中心根本不能自给自足*；他们总是被称为“[插入国家]的硅谷”这一事实应该是一个提示，即他们只是硅谷主干的分支。甚至像以色列、中国和印度这样多产的中心也依赖硅谷通过他们的动脉输送资本。*

对于我们这些“世界其他地方”的局外人来说，硅谷的重要性不仅仅在于其投资者、高科技公司和人才的财富。价值不仅仅是数十亿美元如此自由地流动，以至于 HBO 的*硅谷*的编剧，无论他们问了多少人，[都找不到一个理由](https://web.archive.org/web/20230130021050/http://www.bloomberg.com/news/articles/2015-05-21/hbo-s-silicon-valley-creators-q-a-don-t-burst-the-tech-bubble)为什么像花衣魔笛手*这样的公司*不能获得 1000 万到 1500 万美元的资金。

不，真正的价值是*信任*和*信任*。硅谷是信誉的经纪人。在迪拜，这里并不缺少资金。只是这里没有人敢在游戏早期向一家科技初创公司投入大量资金，除非它得到了硅谷的批准。

让我给你一个真实的例子。今年早些时候，我的公司 [Fetchr](https://web.archive.org/web/20230130021050/http://fetchr.us/) 获得了美国投资者的估价，比迪拜一位顶级投资者给出的价格高出了*六倍。从迪拜投资者那里融资不值得。*

在当地的商业文化中，一些投资者认为，如果一家初创公司需要引入资本，创始人一定没有能力和知识来发展公司。筹款是软弱的表现。我看到当地投资者拒绝发放股票期权，并坚持新员工必须从创始人的股票池中提成。

然而，在新企业联合(NEA)和其他美国风险投资公司全面投资我们之后，阿联酋的投资者相信了该公司的潜力。硅谷的光芒成了我们自己地盘上客户和资本的灯塔。我们获得的信誉至少和实际资本一样有价值。

> 一旦破裂，科技泡沫将会像末日小行星的冲击波一样波及整个科技世界。

这就是硅谷以外的泡沫特别可怕的原因。在数千英里之外，跨越多个时区和文化断层，随着美国投资者对外国技术望而却步，硅谷的错误可能会在怀疑的浪潮中把我们一扫而光。如果没有外部资本的流入和信誉，迪拜羽翼未丰的科技社区将倒退 30 年。

我并不是想淡化美国泡沫的后果——这将是痛苦的——但我确实认为创新将会相对较快地反弹，就像互联网泡沫之后一样。硅谷有一种强大的创新文化，即使许多初创公司破产，这种文化也不会消失。在更脆弱的经济体中，尽管出现了本地风险投资行业，但泡沫的后果可能更严重，持续时间可能更长。

以西班牙为例，西班牙各年龄组的失业率为 23.78%，25 岁以下人口的失业率为惊人的 49.30)。规模虽小但不断增长的技术领域是一个年轻人可以有效地创造自己的工作机会的领域，前提是他们能够获得资本。事实上，根据[风险观察研究](https://web.archive.org/web/20230130021050/http://viewdoc.co/#54b44d1a1f0b8)的数据，2014 年西班牙的 IT 投资价值超过 3 亿美元，比 2013 年增长了 30%。

这里有一个问题:总投资额的 32.6%来自美国，另外 26.3%来自国外。2015 年 1 月至 3 月间，仅美国就占了 51.3%的交易量，而西班牙仅占 17.3%。

如果泡沫破裂，美国投资者退出，像西班牙这样的科技社区如何继续繁荣？加入西班牙一半失业青年的行列，对于那些在伊比利亚半岛无望成为企业家的精通技术的人来说，不是一个有吸引力的选择。

许多人将最终在欧洲大公司打卡上下班，在那里“创新”是一个笑话。许多人将离开西班牙，夺走这个国家的人才和未来能够提供就业、培训、指导和可持续经济价值的公司。

西班牙的 3 亿美元融资与 2014 年风险公司支付给美国公司的 480 亿美元相比微不足道(相比之下，全球风险投资活动的总量为 860 亿美元)。西班牙当然不是唯一依赖美国风险投资的外国科技中心。根据研究公司 PitchBook 的数据，2014 年，308 只风险基金中有 239 只位于北美。

这导致了一个巨大的未知数:如果泡沫切断了资本流动，对西班牙、尼日利亚甚至希腊等新兴科技中心会有什么更广泛的影响？这些地方已经存在失业和政治紧张的动荡局面。受伤的科技界会支持更激进的政党吗？这些国家的“人才外流”会让当地的技术创新陷入昏迷吗？科技的崩溃会拖垮其他垂直行业吗？

硅谷是创新的中心，因为它为全球科技生态系统注入了新的资本和信誉。硅谷的创业公司是一件日常的、乏味的事情；在迪拜这样的地方创业是古怪和令人不安的，至少在硅谷投资者称赞它之前是如此。我在迪拜经营着一家初创公司，它之所以成为可能，完全是因为美国顶级风投赋予我们的信誉。

我无法告诉你泡沫的确切影响。但我知道，作为一个生活在“世界其他地方”的企业家，这大约超过 117 只独角兽和一个山谷。如果泡沫破裂，无论是由傲慢、无知还是看不见的力量造成的，全球技术社区都将分担痛苦。
