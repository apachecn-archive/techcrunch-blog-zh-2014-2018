华为 P20 不会进入美国市场。

认识一下华为 P20。这是一部相当不错的手机。我摆弄了一下它，我可以确认它确实是一款坚固的旗舰产品，具有一些合适的夸张功能——你说呢？三个后置摄像头？！但是如果你住在美国，所有这些都是没有意义的。

当然，华为在美国销售手机遇到了很多问题。事实上，就在我摆弄 P20 的时候，有消息称百思买计划停止销售该公司的手机——这有点像在开幕日之前发现你的首发投手需要做汤米·约翰手术。只是，你知道，更多的国际间谍之类的。

这次，该公司没有处理被运营商和零售商拒绝的繁琐程序，而是简单地明确表示，新旗舰不会在这里上市——不会通过任何官方渠道。老实说，对于华为来说，最好的办法可能是专注于那些长期储备其手机的国家——从本周早些时候的 FCC 报告来看，这种情况在好转之前会变得更糟。

对于世界上的其他地方来说，这里有很多值得喜欢的东西。P20 看起来像是 iPhone X 和 HTC 最新闪亮金属手机的结合体。它有 5.8 英寸的显示屏(P20 Pro 为 6.1 英寸)和一些疯狂的相机规格，包括三个后置镜头，包括一个 800 万像素的长焦镜头和 4000 万像素的长焦镜头。)RGP，再加上内置色温传感器。仍然有一个前置指纹传感器和一些奇怪的选择，比如 2D 面部解锁功能，可以弥补深度感应的不足。

截至发稿时，还没有定价或供货情况，除了在美国，后者只是一张有一条线穿过的红色大圆圈的图片。
