# 谷歌免费向研究人员提供 1000 个云处理器集群

在谷歌 I/O 结束时，该公司推出了一项新计划，让研究人员免费获得该公司最先进的机器学习技术。TensorFlow 研究云项目将被称为基于应用的项目，对任何从事研究的人开放，而不仅仅是学术界成员。

如果被接受，研究人员将获得 1000 个云 TPU 的集群进行训练和推理。每个云 TPU 将提供 180 万亿次浮点运算性能和 64GB 内存。计算时间将根据特定的批准项目进行分配和限制。

作为交换，谷歌要求用户在同行评议的出版物和开源代码中分享他们的研究。如果你不喜欢这种程度的开放，谷歌还计划推出一个面向内部、私营部门工作的云 TPU 阿尔法项目。

该项目的申请尚未开放，但谷歌正在指导感兴趣的各方填写一份表示兴趣的[表格](https://web.archive.org/web/20221020161951/https://services.google.com/fb/forms/tpusignup/)。该问卷要求提供以下基本信息:典型训练集的规模、模型通常需要的训练时间、您喜欢的训练模型平台以及您经常使用的硬件。

最终，将在滚动的基础上做出决定。即使你被拒绝了，公司也会鼓励所有申请者在新项目和需求出现时重新申请。
