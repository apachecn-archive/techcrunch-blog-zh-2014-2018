# 三星 Galaxy S8 新闻发布会现场直播

今天是三星的重头戏。距离该公司发布其最后一款旗舰产品已经有一年多的时间了，在这期间，该公司发生了很多事情。在经历了几个月的召回和一些[离奇的高管丑闻](https://web.archive.org/web/20221207073527/https://beta.techcrunch.com/2017/02/16/samsung-vice-chairman-arrested-on-bribery-charges/)之后，现在是该公司需要掸去灰尘，向世界展示当初是什么让它成为如此强大的智能手机巨头的时候了。

从过去几个月的泄露来看，不太可能有太多惊喜——至少就[Galaxy S8](https://web.archive.org/web/20221207073527/https://beta.techcrunch.com/2017/03/25/samsung-galaxy-s8-3/)和(据传更大的兄弟)Galaxy S8+而言。自从 2 月的世界移动通信大会以来，我们已经看到了从各个角度泄露的手机照片、规格表和一些快速视频。

这款手机看起来让该公司更接近其边缘到边缘显示的梦想，甚至在这个过程中放弃了物理 home 键。该公司已经提供了关于其即将推出的 Bixby 智能助手的大量信息，该智能助手以一些很酷的技巧进入了一个拥挤的市场。

但是三星从来不是专注于单一产品的公司。该公司在今天的活动中可能还有更多锦囊妙计，特别是考虑到它希望在警惕的公众中产生多少好感，这些公众已经习惯了在登机前的公告中听到它的名字。
