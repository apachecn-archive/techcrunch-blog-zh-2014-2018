Opera的Max数据节省应用程序现在预装在来自14家原始设备制造商的手机上

原文：https://web.archive.org/web/http://techcrunch.com/2015/11/08/operas-max-data-savings-app-now-comes-pre-installed-on-phones-from-14-oems/?ncid=rss

Opera的Max数据节省应用现在已经预装在来自14家原始设备制造商的手机上

Opera 今天宣布其 Opera Max 数据管理应用程序现已集成到14家原始设备制造商的手机中。其中包括之前宣布的合作伙伴，如三星和小米，以及三个新宣布的合作伙伴:宏碁、海信和TWZ。由于这些合作关系，Opera现在相信，到2017年，其Max软件将安装在超过1亿部Android手机上。

Opera Max本质上是一个代理服务器，它拦截来自你手机的所有流量(加密连接除外)，然后通过Opera的服务器进行路由，这些服务器对流量应用了许多节省带宽的技术。这包括压缩图像和视频，以及优化基本的HTML和CSS。

如果你有一个无限制的数据计划，或者订阅的数据超过你一个月内通常使用的数据，Opera Max可能不适合你。不过，如果你使用的是计量套餐，这项服务很有可能会帮你省下不少钱。Opera表示，其服务可以将数据消耗降低多达50%。对于在Instagram、YouTube和网飞等应用上花费大量时间的用户来说，可以节省高达60%的时间。

“许多用户对使用移动数据持谨慎态度，担心花费过多或超出数据上限。Opera Max的产品经理Sergey Lossev在今天的公告中表示:“我们看到原始设备制造商对此做出了回应，并通过在他们的设备上提供数据优化解决方案来降低移动互联网接入的壁垒。”。

Opera Max现在预装在宏碁、Cherry Mobile、Evercoss、Fly、海信、Mobiistar、Micromax、Oppo、Prestigio、三星、Symphony、TWZ、Tecno和小米的手机上。
