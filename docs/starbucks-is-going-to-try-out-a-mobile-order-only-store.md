# 星巴克将尝试一种移动订单商店

星巴克在 2015 年推出了移动点餐系统，在某些方面，它已经成为自己成功的受害者。热门地点的顾客渴望使用移动订购系统来选择他们的选择并提前付款，希望避免排队——但他们无论如何都不得不等待，因为虚拟队列根据地点的不同与真实队列一样大或更大。现在，它正在寻找让移动订购更好工作的方法，为了实现这一目标，它将在自己的西雅图总部内尝试一个专门为移动订单客户服务的地点。

路透社报道称，从下周开始，这家位于西雅图的公司的两个内部咖啡馆之一将变成专门的移动点餐和支付场所。据报道，大楼员工的所有移动订单(包括 5000 人)都将被路由到新的地点，它将采用不同的设计，有一个突出的取货窗口，也可以让咖啡师看到订单的准备情况。

2015 年 9 月，星巴克在美国各地的应用程序中添加了提前订购和取货选项，自那以来，这一功能一直受到用户的欢迎。该功能允许用户在应用程序中浏览星巴克菜单，选择位置，提前支付订单，并收到关于何时可以提货的估计。根据地点的不同，订单会被放在指定的取货点，或者像店内订单一样，由咖啡师上门取货。
