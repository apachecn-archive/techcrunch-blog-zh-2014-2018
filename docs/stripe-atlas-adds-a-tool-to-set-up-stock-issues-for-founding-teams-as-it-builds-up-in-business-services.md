# Stripe Atlas 在构建其业务服务 TechCrunch 时增加了一个为创始团队设置股票发行的工具

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/10/31/stripe-atlas-adds-a-tool-to-set-up-stock-issues-for-founding-teams-as-it-builds-up-in-business-services/>

Stripe Atlas 由支付公司 Stripe [于去年](https://web.archive.org/web/20221209164730/https://beta.techcrunch.com/2016/02/24/stripe-expands-startup-tools-with-atlas-a-way-for-global-companies-to-incorporate-in-delaware/)推出，旨在帮助小企业在美国建立一个合法的公司企业实体。现在，来自 125 多个国家的“数千名”企业家正在使用 Atlas，Stripe 正在通过一项新功能扩展它，因为它专注于成为一个创业服务平台。注册 Atlas(售价 500 美元)的公司现在也可以用它来建立法律文书工作，并向创始团队发行股票。

这项基础服务是与律师事务所 Orrick 合作创建的，包括生成、审查、签署和存储股票发行文件的模板，基于初创公司世界的“标准条款”——通常包括四年的授权和一年的悬崖。任何更加定制的服务都需要更高的服务级别(以及额外的费用)。

该功能目前也仅限于创始团队，尽管 Stripe Atlas 项目负责人泰勒·弗朗西斯(Taylor Francis)指出，Stripe 可能会考虑的一项更新是将其扩展到其他员工。股票期权是人们选择在初创公司工作时谈论最多的方面之一——希望如果它起飞了，你的银行余额也会增加。因此，将选择权扩大到更广泛的员工群体，可能是一个显而易见的细节，可以添加到 Stripe 的组合中。

“我们可以看到另一个方向，”弗朗西斯在一次采访中说。

这项新服务出现在 Stripe 的关键时刻。截至 2016 年 11 月的上一轮融资，该公司的[估值为 90 亿美元](https://web.archive.org/web/20221209164730/https://beta.techcrunch.com/2016/11/25/payments-provider-stripe-has-raised-another-150-at-a-9b-valuation/)，尽管其首席执行官帕特里克·科利森表示该公司[不会很快进行 IPO](https://web.archive.org/web/20221209164730/https://beta.techcrunch.com/2017/04/07/stripes-patrick-collison-says-no-to-an-ipo-any-time-soon/) ，但在实现这一天的道路上，它将寻求更多方式来多元化和扩展其商业模式，以填补这一巨额估值。将自己定位为商业服务平台是 Stripe 实现这一目标的一种方式。

Stripe 推出 Atlas 背后的一些逻辑是这样的:这是一种有效的方式，让那些可能尚未使用 Stripe 基于 API 的支付服务核心产品的企业加入进来，但希望他们在使用时会转向 Stripe。

弗朗西斯说:“我们希望(使用 Atlas)意味着他们今后会更加成功，如果他们随后使用我们的一些其他产品(如支付 API)，这对他们和我们都有好处。”

弗朗西斯说，Atlas 本身的加入费用为 500 美元，其中很大一部分用于提交表格和相关法律工作的费用，所以目前看来，似乎没有太多关注将其作为 Stripe 的收入来源。

但从长远来看，Atlas 本身有潜力成为一个平台——特别是因为并非所有使用它的公司都一定是产生大量支付的商业公司。增加股票发行服务是测试更多功能的一种方式，Francis 证实将会有更多功能推出。

“我们今天没有分享的任何东西，但我们思考这个问题的方式是，有一大堆障碍挡在创始人的路上，”他说。“我们将继续研究这份清单，看看我们能解决多少问题。”

这是近几个月来 Stripe Atlas 和 Stripe 的其他几项增强的基础，旨在成为其基本支付产品之外的更多服务平台。今年 4 月，Stripe [向美国公司](https://web.archive.org/web/20221209164730/https://beta.techcrunch.com/2017/04/06/stripe-atlas-us/)开放了 Atlas(它最初是作为美国以外公司的服务推出的)——“自那次推出 以来，我们看到 Atlas 用户的数量激增，”弗朗西斯说——本月早些时候，该公司 推出了一种内容游戏， [Stripe Atlas Guides](https://web.archive.org/web/20221209164730/https://stripe.com/atlas/guides) ，开始回答创始人可能会有的各种问题。

如果你想知道我们还能从 Atlas 看到什么产品，你可以使用指南作为指南。一位发言人告诉我:“它们很大程度上预示着条纹地图集的未来。”。

本月，Stripe 还推出了一项名为 [Elements](https://web.archive.org/web/20221209164730/https://beta.techcrunch.com/2017/10/03/elements-stripes-new-check-out-toolkit-aims-to-boost-e-commerce-sales-completions/) 的服务，旨在帮助顾客打造个性化的结账体验，即 la Shopify。

除了股票发行工具，Stripe 还增加了一些较小的东西，包括创业公司在合并后需要跟踪的其他实用项目的清单，如在特拉华州获得会计师和获得代理的自动化服务(所有的创业公司都是通过 Atlas 注册的)。它说，它还致力于自动化清单上的更多任务。
