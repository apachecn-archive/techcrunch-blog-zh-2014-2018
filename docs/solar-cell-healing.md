# 科学家发现一种用强光治愈某些太阳能电池的方法

有时候在科学上，答案一直就在你的眼前。比如说，太阳能技术的局限性问题可以用光来解决。

在过去的几年里，钙钛矿化合物作为太阳能电池基础的潜力引起了越来越多的兴趣——尽管一些固有的缺陷限制了它们的效率。然而，麻省理工学院今天早上宣布的新研究[揭示了化合物局限性的答案可以在最方便的地方找到:非常强烈的光线。](https://web.archive.org/web/20230130013137/http://news.mit.edu/2016/light-heal-defects-solar-cells-0524)

根据该学校的说法，对用有机-无机金属卤化物钙钛矿产生的薄膜施加强照明具有从材料中清除缺陷的效果，这反过来可以增加太阳能电池产生中的光捕获效率。这种修复还可以使这种材料成为更有效的发光体，从而创造出新的发光二极管和激光器。

这个修复仍然有一些问题需要克服，最大的问题是如何在一段时间内保持它的效果，使它值得制造商去做。这项研究正值各公司寻求在年内将该材料的各种版本推向市场。
