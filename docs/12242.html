苹果发布了一点代码，让你可以把现场照片放在你的网站上

原文：https://web.archive.org/web/https://techcrunch.com/2017/04/20/apple-releases-a-bit-of-code-to-let-you-put-live-photos-on-your-sites/?utm_source=shootr&utm_medium=referral

苹果发布了一些代码，让你可以在网站上放实时照片

苹果的现场照片很有趣——它们绝对为我捕捉到了一些标准剧照可能会错过的意想不到的神奇时刻。但在大多数情况下，它们存在于你的手机上，并停留在你的应用程序的范围内。看到它们漂浮在网络上，尤其是在桌面浏览器上，总是一件稀罕事。

去年，Tumblr 通过在网页版中添加实时照片并发布一个工具让其他人也这样做，稍微推倒了那堵墙。然而，当涉及到苹果官方认可的方法时，却没有任何发现。

这种情况在今天早上悄然改变了，苹果用一个新工具更新了它的开发者门户网站，他们称之为 LivePhotosKit。这是一个 JavaScript API，专门为您的网站提供实时照片回放支持。

我们还不能在这里嵌入现场照片(或者在这方面比 GIF 格式更好的 WebM，因此顶部是超级大块的 GIF)，但你可以在这里看到一个工作示例。

它看起来很流畅，但是苹果在这里触发回放的示例机制…看起来并不是非常直观。

我点击了吗？我悬停在图像上吗？就在我滚动的时候开始播放了一秒钟…是那个键吗？

原来，诀窍是将鼠标光标悬停在右上角的小“实时”框上。一旦你想明白了，就很容易记住，但是有大约 20 秒钟，我确信有什么东西坏了。

苹果新 API 的文档表明，它应该可以与大多数主流浏览器兼容，无论是移动浏览器还是其他浏览器。
