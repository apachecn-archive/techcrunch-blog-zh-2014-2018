# 测试给了 iPhone X 显示屏最高荣誉，但相机只是竞争 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/11/06/tests-give-iphone-x-display-top-honors-but-camera-is-merely-competitive/>

对最近发布的 iPhone X 的实验室测试使苹果的新旗舰在显示屏和摄像头方面达到了最高质量水平，但只有在前一个类别中，它才真正将竞争对手甩在身后。当然，如果你的屏幕不能很好地显示图像，那么再好的图像又有什么意义呢？

苹果并不倾向于制造自己的显示屏；但是，虽然 LG、夏普和三星(就 iPhone X 而言)理应因制造这些产品而受到赞扬，但苹果并不只是将它们从货架上抢走。大量的金钱和时间花费在定制和调整手机上，手机在出厂前都要单独校准，以适应制造过程中的变化。

[DisplayMate 的一系列测试](https://web.archive.org/web/20221025222443/http://www.displaymate.com/iPhoneX_ShootOut_1a.htm)旨在测试显示器的绝对色彩准确度、亮度和其他客观指标。以这些标准来衡量，最新款的 iPhone 甚至胜过了其母公司三星的最新款有机发光二极管显示器。

OLEDs 自然在许多类别中表现出色，从对比度到色彩准确度，苹果的软件强调了这些优势。特别是它的色彩准确度是 DisplayMate 测试过的最好的。而且方便的是，它会根据内容切换到正确的色彩配置文件或色域，这意味着您不会看到通过 Adobe 或 DCI-P3 的镜头显示的打算在 sRGB 显示的图像。

[![](img/e1224fcc75874a60d94dfbcb26f2e27b.png)](https://web.archive.org/web/20221025222443/https://beta.techcrunch.com/wp-content/uploads/2017/11/refcolors.png)

iPhone X 几乎囊括了整个扩展的色域，在任何领域都没有弱点。

如果这对你来说没有任何意义，不要担心——关键是你不需要意识到这一点，相反，你可以简单地确保照片、电影、游戏等将完全按照它们应该的样子被看到。尽管如此，你可能还是想在显示选项上花点时间，因为自动白平衡可能会让观众对这种东西很敏感(比如我)。

显示技术的一个可能被认为是横向的变化是转向菱形子像素。如你所知，数字显示器中的每个像素通常由许多子像素组成:不同数量和形状的红色、蓝色和绿色，以不同程度发光，形成我们所感知的颜色。

对于液晶显示器，这通常采用 RGB 网格的形式，通常有一个由红色、绿色、蓝色组成的正方形，然后可能是另一个绿色子像素，或类似的东西。这工作得很好，但导致某些模式的锯齿，或像素化。不同的子像素布局产生不同的混叠图案。

iPhone X 的子像素布局与之前的所有 iPhone 不同，像素是菱形的，排列在对角对称的网格中，而不是矩形和矩形网格中:

[![](img/911d651e2b5c2bd2d68af44987fe1f17.png)](https://web.archive.org/web/20221025222443/https://beta.techcrunch.com/wp-content/uploads/2017/11/iphonex_diamond_sub-pixels_1a.jpg)

这是有机发光二极管亚像素的超级特写。

现在，自从超过 300 PPI 屏幕出现以来，混叠已经不像以前那样是个问题了。但是有些走样比其他走样更好，iPhone X(以及其他菱形或五边形排列的手机)显示的走样类型对于垂直和水平线条来说并不理想。

这张对比照片拍摄于 [iMore 对手机](https://web.archive.org/web/20221025222443/https://www.imore.com/iphone-x-review)的评测，说明了这一点:

[![](img/a0092ff84bb476882b00634ecb8b0ac5.png)](https://web.archive.org/web/20221025222443/https://beta.techcrunch.com/wp-content/uploads/2017/11/imore_closeup.jpg)

如果你想看到不同之处，一定要以全尺寸观看。

在对角线和圆边上，菱形图案使曲线更加自然，没有阶梯。但是在水平和垂直的直线上，你最终会得到一个锯齿图案。

也就是说，如果你通过显微镜看手机。虽然锯齿混叠在最初的[Galaxy S](https://web.archive.org/web/20221025222443/https://beta.techcrunch.com/2010/07/26/review-samsung-galaxy-s-aka-att-captivate-t-mobile-vibrant-verizon-fascinate-sprint-epic-4g/)上是一个问题，但我们已经取得了很大的进步，像素间距现在小了很多，使得图案虽然还在，但不太明显。(我也是在没有看到现实生活中的事情的情况下这么说的，据我所知，到目前为止还没有人抱怨过。)

## 相机与最好的竞争

DxOMark 今年已经用一套新的以移动为重点的测试测试了所有的旗舰产品，虽然这些半合成的指标应该总是有所保留，但这些人知道他们在做什么，当然是不可再生的像素窥视者。

iPhone X [在静态照片方面超过了之前的高分](https://web.archive.org/web/20221025222443/https://www.dxomark.com/apple-iphone-x-top-performer-stills/)，以微弱优势击败了 Galaxy Note 8 和华为 Mate 10 Pro 它也比 iPhone 8 Plus 好，iPhone 8 Plus 本身[曾短暂地成为一个高水位标志](https://web.archive.org/web/20221025222443/https://beta.techcrunch.com/2017/09/22/iphone-8-and-8-plus-cameras-get-top-marks-from-testing-outfit-dxomark/)。所以很优秀，[正如我们的评测发现的](https://web.archive.org/web/20221025222443/https://beta.techcrunch.com/2017/10/31/review-the-iphone-x-goes-to-disneyland/)。

正如你可能期待的那样，一部拥有梦幻屏幕的手机，色彩和对比度表现得尤为出色。然而，像其他苹果设备一样，它的快门延迟经常比竞争对手更长——特别是 Pixel 2，它为自动对焦速度和精度设立了新的标准。

它在极低的光线下失分，在这方面它也优于 Pixel 2，并且它的闪光灯人像似乎经常曝光不足。这也是它在视频方面失分的地方:噪音和曝光不足是其 1080p/30 视频的特点。

不过，看起来在良好的条件下，iPhone X 与其前辈和竞争对手一样无可指责。
