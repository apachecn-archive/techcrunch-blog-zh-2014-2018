SAP将以每股129美元的价格收购费用软件制造商Concur

原文：https://web.archive.org/web/https://techcrunch.com/2014/09/18/sap-to-acquire-expense-software-maker-concur-for-129-per-share/

SAP将以每股129美元收购费用软件制造商Concur

企业软件公司SAP刚刚宣布已经同意收购Concur Technologies，该公司是23，000家企业(包括TechCrunch及其母公司AOL)使用的费用和差旅管理工具的制造商。

该交易的价格为每股129美元，比Concur 9月17日(昨天)的收盘价高出20%。这使得Concur的企业价值达到83亿美元。SAP表示，将通过€70亿美元的信贷机制为此次收购提供资金。

“收购Concur符合我们对业务网络的不懈关注，”SAP首席执行官Bill McDermott在收购发布会上表示。“我们正采取大胆举措，在公司内部和公司之间创新业务的未来。随着Ariba [ 于2012年被收购，Fieldglass [ 于今年被收购，加上Concur，SAP是无可争议的商业网络公司。”

本月早些时候，彭博报道称，Concur正在与包括SAP在内的公司讨论可能的收购。

该收购预计将于今年第四季度或2015年第一季度完成。在这笔交易之后，SAP表示，合并后的公司将拥有5000万“云端”用户。为了说明双方的一些机会，报告指出，大多数SAP客户不使用Concur，而只有30%的Concur客户使用SAP。

Concur成立于1993年，早在2011年就收购了旅游初创公司TripIt。新闻稿还称，该公司目前的营收运行率为7亿美元。
