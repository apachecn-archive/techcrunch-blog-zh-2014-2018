# 沃尔沃将从 2019 年开始仅生产电动和混合动力汽车

# 从 2019 年开始，沃尔沃将只生产电动和混合动力汽车

该公司宣布，沃尔沃完全不再使用传统发动机和纯燃气汽车。到 2019 年，沃尔沃集团打算在所有新车型上只提供全电动或混合动力发动机，使其成为第一家承诺只使用替代传动系统的汽车制造商。

鉴于从制造和性能角度来看电动和混合动力汽车的优势，以及该行业在自动驾驶汽车技术上的巨额投资，纯内燃机汽车的终结似乎是最终不可避免的。但对于沃尔沃来说，承诺在两年后完全转向混合动力和电动仍然是迄今为止最强烈的信号，我们已经看到纯内燃机的日子可能会越来越早，而不是越来越晚。

沃尔沃已经制定了到 2025 年销售 100 万辆电动和混合动力汽车的目标，现在这个目标似乎触手可及，因为它将在新车方面销售所有汽车。沃尔沃还宣布，将在 2019 年至 2021 年期间推出五款新的电动和混合动力汽车，其中两款将由 Polestar 制造，该公司最近宣布，Polestar 将成为自己的子公司，销售高性能电动汽车，可能与高端特斯拉车型竞争。

制造电动汽车的部分成本优势是应对日益严格的汽车排放要求，这些要求在大多数主要国际市场都将变得更加严格，包括沃尔沃的母公司吉利的总部所在地中国。随着产能和制造工艺的提高，电动汽车零部件和电池的生产成本也越来越低。

沃尔沃开始转向电动汽车，将混合动力汽车作为过渡性的权宜之计，这是一个明智而积极的举措，旨在宣称自己在未来市场的领导地位。许多公司谈论他们的工作，并致力于替代传动系统，但这真的是把你的钱用在你的嘴上。
