# SoundCloud 通过新的建议曲目功能改进音乐发现 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/06/22/soundcloud-improves-music-discovery-with-new-suggested-tracks-feature/>

SoundCloud 正在推出一项新功能，旨在帮助用户根据他们的活动和收听历史找到他们喜欢的新歌曲和艺术家。这项名为“建议曲目”的新增功能本质上是 SoundCloud 试图为其个人用户提供个性化服务，以保持他们的参与，并在应用程序中收听更多音乐和音频内容。

今天，流媒体音乐服务仅仅在价格、用户体验和排他性方面进行竞争是不够的，它们还在不断努力超越对方，成为向用户介绍他们喜欢的新音乐的更好选择。

如果做得好，这可能是该服务的一个巨大卖点——例如，Spotify 的 Discover Weekly 播放列表自去年 7 月推出以来已有[4000 万独立用户](https://web.archive.org/web/20221007095151/https://news.spotify.com/us/2016/05/25/discover-weekly-reaches-nearly-5-billion-tracks-streamed-since-launch/)，50 亿首歌曲被播放，该公司[最近](https://web.archive.org/web/20221007095151/https://beta.techcrunch.com/2016/05/25/playlists-not-blogs/)吹捧道。

在 SoundCloud 的例子中，它的“建议曲目”部分将由用户在该服务上的收听历史以及你的“喜欢”来驱动然而，与 Spotify 广受欢迎的 Discover Weekly 播放列表不同的是，当你使用这项服务时，SoundCloud 的算法会不断更新这些建议。也就是说，当您继续听音乐并与服务交互时，此部分会刷新，而不是播放列表。

这使得推荐曲目更类似于 Spotify 的 Discover 部分，它在横幅下提供建议，横幅上写着:“因为你听了……”，后面是艺术家的名字。

像 Spotify 一样，SoundCloud 的建议也将明确地与你刚刚演奏或喜欢的特定歌曲或艺术家联系起来，甚至会以类似的介绍作为开头(“例如，因为你演奏了…”或“因为你喜欢…”)。其他项目的列表，用缩略图显示，然后可以在下面滚动。

要使用这一新功能，你需要进入 SoundCloud 网站上的“发现”标签或应用程序中的搜索部分。如果你已经在 SoundCloud 上听了音乐，你会收到一些建议。

如果没有，你需要开始流媒体音乐和音频，根据你的兴趣来训练应用程序。

建议曲目现已在 iOS、Android 和 web 上发布。SoundCloud 表示，它将在未来几周推出其他音乐发现功能，这暗示着它可能会推出自己的个性化播放列表。
