# Spotify 收购照片聚合网站 CrowdAlbum，为艺术家打造更多营销工具

Spotify[进行了另一项收购](https://web.archive.org/web/20230404080349/https://press.spotify.com/us/2016/04/27/spotify-acquires-crowdalbum/),为使用 Spotify 流媒体平台的艺术家提供工具，以增加他们的听众和整体业务。它已经收购了 [CrowdAlbum](https://web.archive.org/web/20230404080349/http://crowdalbum.com/) ，这是一家整合社交媒体上分享的音乐活动照片和视频的公司，以创建它所称的“视觉历史”

有时这些照片包含了很多辛酸，比如这张给[王子](https://web.archive.org/web/20230404080349/https://www.crowdalbum.com/album/570f4fdd7574690c2500017e/Prince_20160414)的照片。

交易条款尚未披露。CrowdAlbum 成立于 2013 年，总部位于旧金山，仅筹集了 10 万美元。无论如何，这是一个小但有影响力的公司:它当时与大约 1000 名艺术家和场地合作伙伴合作，但他们包括[迪普](https://web.archive.org/web/20230404080349/https://twitter.com/diplo/status/677248893516709888)、[可爱的死亡出租车](https://web.archive.org/web/20230404080349/http://www.deathcabforcutie.com/)、[李尔·韦恩](https://web.archive.org/web/20230404080349/https://www.facebook.com/LilWayne/posts/10150677952944959?comment_id=10150677992939959&comment_tracking=%7B%22tn%22%3A%22R%22%7D)和[辐射男孩](https://web.archive.org/web/20230404080349/http://falloutboy.com/photos)。

Spotify 表示，它将把 CrowdAlbum 的工具添加到它为使用 Spotify 作为营销平台的艺术家提供的越来越多的工具中。它还在这方面进行了其他收购，特别是今年早些时候收购了 Cord 项目和声波。Spotify 已经推出的产品包括 [Spotify Fan Insights](https://web.archive.org/web/20230404080349/https://www.spotifyartists.com/introducing-spotify-fan-insights/) 和[音乐会](https://web.archive.org/web/20230404080349/https://news.spotify.com/us/2015/11/12/never-miss-another-show-with-personalized-concerts/)。

建立更多像这样的 B2B 服务对 Spotify 来说是明智的，这既是为了吸引更多的音乐人到这个平台上来——也是为了抵消围绕 Spotify 的一些迷因，因为它是艺术家收入损失的温床——也是为了与其他也向艺术家提供工具的流媒体竞争对手竞争。其他公司如 Pandora 和 Soundcloud 也提供分析和其他工具，如消息服务，以帮助音乐人查看他们的音乐是如何被听到的，控制他们的内容是如何被分享的，并与粉丝交流。

Spotify 产品副总裁查理·赫尔曼(Charlie Hellman)在一份声明中表示:“我们每天都在努力寻找加强艺术家和粉丝之间联系的方法。”。“收购 CrowdAlbum 是 Spotify 投资的最新方式，旨在帮助艺术家在 Spotify 内外找到并吸引他们的观众，尤其是围绕着日益重要的巡演业务。”
