随着新广告出版商的加入，Appsfire 成为一个可靠的替代移动广告网络 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2014/09/30/with-new-ad-publishers-on-board-appsfire-becomes-a-solid-alternative-mobile-advertising-network/

广告技术初创公司 Appsfire 在移动设备上提供一些最好的互动广告 格式。然而，在移动领域，脸书、谷歌和现在的 Twitter 似乎是占据压倒性优势的广告网络。为了反驳这种想法，Appsfire 刚刚宣布了一些首批广告出版商，看起来这家初创公司正在成为一个可靠的替代网络。

当我第一次报道的新单元时，我称赞了产品设计，并怀疑像 Appsfire 这样的小型创业公司是否能够说服主要的广告商和出版商。对于后一部分，初创公司 just 宣布正在与 Opera Media Works、Cut the Rope 开发商 Zeptolab、Viadeo、Cheetah Mobile、Taringa、Jogatina、NaturalMotion、TF1、Lagardère Interactive 和 La Place Media(代表法国一些最重要的出版商，如法国 Télévisions Publicité和 Figaro Medias)合作，在各自的应用中显示 Appsfire 广告。

他们中的一些人在法国比在世界上其他地方更有名，但是 Appsfire 的广告将通过这些合作伙伴接触到非常多的观众。我相信这些出版商与多个广告网络合作，但这对初创公司来说无疑是朝着正确方向迈出的一步。

现在，所有这些都没有回答另一个重要的问题 Appsfire 是否与足够多的广告商合作，它能否填补其库存？联合创始人兼首席执行官欧里尔·奥哈扬告诉我，该公司与《纽约时报》、优步、腾讯等签署了广告合同。现在的库存是 5 个月前的五倍。

Appsfire 管理自己的库存。它不从现有的广告网络购买广告。换句话说，它是一个“真正的广告网络”，而不仅仅是广告商和应用发行商之间的另一个中介。随着 Appsfire 开发新的广告单元，一切都在其内部 SDK 和 API 之上运行。

“我们也向我们的出版商支付高额报酬的原因之一是，我们设法利用我们过去在应用推荐引擎方面的工作，根据显示广告的应用，为正确的用户找到正确的应用广告。它让我们限制库存浪费，”Ohayon 说。

当然，它的互动广告单元可能比标准横幅广告更有效。迭代一个技术产品与一直签下新的广告商和出版商有很大不同。既然 Appsfire 有足够的广告单元来说服移动广告商和出版商，它需要证明运行 Appsfire 广告比脸书广告更有效率。这是一条漫长而多风的路，但绝对是一个有趣的挑战。

披露:Appsfire 联合创始人兼首席执行官奥里埃尔·奥哈永曾负责法国 TechCrunch。我当时不在 TechCrunch 工作，所以我们从来没有合作过。
