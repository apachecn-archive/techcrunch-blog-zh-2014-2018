Curve 现在允许您“回到过去”并追溯切换您支付的卡 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/07/03/back-to-the-future/

手机钱包和一卡通应用程序 Curve T1 今天推出了一个漂亮的新功能，可以让你追溯切换你用来支付的卡。

该选项被大胆地称为“金融时间旅行”，据我所知，这是用户首先要求的，正在申请专利，它给你两周的时间来改变你与 Curve 关联的银行和信用卡对特定购买收取的费用。

因此，举例来说，如果您不小心将一份工作午餐的费用记到了您个人活期账户的借记卡上，而不是费用账户的信用卡上，您可以简单地“回到过去”, Curve 会撤销或退还费用，并从正确的卡上扣除。

为了理解为什么这很重要，而且可以说是非常需要的，这里有一个对 Curve 核心产品的回顾:移动钱包可以让你把你的借记卡和信用卡链接到一个应用程序和附带的由万事达卡支持的 Curve 卡。使用 Curve 应用程序，您只需选择您希望 Curve 卡作为线下或线上支付渠道的卡，这意味着您只需随身携带一张实体卡。

然而，问题是，如果你用 Curve 支付了一些东西，却忘记了它当前被设置为收费的卡。或者，更糟糕的是，你在一个手机信号差的地区，因此无法在购物前使用 Curve 应用程序切换卡。新功能缓解了这两种情况。

但是，如果你曾经与 Curve 的联合创始人兼首席执行官沙查尔·比亚里克(Shachar Bialick)相处过，这并不奇怪，他的讲话速度每小时 100 英里，并且总是提前三步思考，这里也有一个更大的愿景。“回到过去”和追溯性地改变付款来源的能力可能在将来提供推荐不同的信贷提供者的机会。

例如，如果 Curve spot 发现你刚刚购买了一件高价商品，比如一台新电视机，并用信用卡支付，它可以与银行或其他信贷来源合作，提供另一种利息较低的“预先批准的”贷款，并相应地调整收费。

这将与该公司即将推出的曲线功能很好地结合起来，该功能被称为“曲线连接”(Curve Connect)，该应用将开始连接到其他金融科技或数字金融服务，从而为金融科技融合战略的路线图提供基础。从长远来看，这家初创公司希望成为你管理所有资金的平台，本质上是重新捆绑金融服务，让你拥有控制权，在这里或那里建立合作关系。

与此同时，Curve 最近宣布，它的注册人数已超过 5 万人，Curve 应用程序已处理了 5000 万笔付款，但它没有透露每月活跃用户数。我也知道这家初创公司的首轮融资将在几周内披露。该公司已经有了一些著名的投资者，包括 Seedcamp，Taavet Hinrikus，他是汇款初创公司 TransferWise 的联合创始人，以及 challenger bank Tandem 的 Ricky Knox。
