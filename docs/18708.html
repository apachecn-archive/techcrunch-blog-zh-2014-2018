终极软件以 3 亿美元收购 people doc TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/07/17/ultimate-software-is-acquiring-peopledoc-for-300-million/

上市公司终极软件以 3 亿美元的现金和股票收购法国初创公司 PeopleDoc 。该交易预计将于 2018 年第三季度完成。这两家公司都做人力资源解决方案。

终极软件已经存在一段时间了。它于 1998 年上市，并于 2002 年转向软件即服务解决方案，该解决方案名为 UltiPro。它可以让你管理所有人力资源的事情，从工资到福利，时间管理，入职，绩效管理等等。

PeopleDoc 是一家年轻的法国创业公司，已经从筹集了超过 5000 万美元的资金。顾名思义，PeopleDoc 可以让您将所有与您相关的人力资源文档集中在一个位置。它们可能来自多个来源和系统，它们都在那里。

这家初创公司还致力于一个入职解决方案和其他工具，以尽可能自动化人力资源流程。例如，您可以使用 PeopleDoc 与人力资源团队进行沟通，并通知他们发生了变化。

Ultimate Software 有大约 4，100 个客户，代表大约 3，800 万名员工。所以很明显公司在追求大客户。每个客户平均雇佣 9200 人。

PeopleDoc 拥有 1000 名客户，为 400 万名员工提供服务。虽然 PeopleDoc 比 Ultimate Software 小得多，但对这家初创公司来说，这是一笔值得注意的收购。

Ultimate Software 表示，它计划在收购结束时花费 7500 万美元现金。PeopleDoc 股东一年后将再获得 5000 万美元。

最后，Ultimate Software 将在剩余的收购中花费约 1.75 亿美元购买股票。该公司在股票市场的表现令人难以置信的好，在过去的十年里持续上涨。

收购背后有两个原因。首先，Ultimate Software 主要面向美国客户。随着今天的收购，终极软件将能够说服新的国际客户，特别是在欧洲。

第二，PeopleDoc 将继续作为子公司运营，因为这两家公司做的事情并不完全相同。事实上，Ultimate Software 明年将开始向自己的客户分发 PeopleDoc 的服务。
