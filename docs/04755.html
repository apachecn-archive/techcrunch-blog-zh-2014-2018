Twitter 关闭了在 30 个国家追踪政客被删除推文的服务 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/08/24/twitter-shuts-down-services-that-tracked-politicians-deleted-tweets-in-30-countries/

为了保持社交媒体上公开透明的声明记录，追踪政客和其他外交官删除的推文的两个服务 Politwoops 及其姐妹网站 Diplotwoops 现已被禁止访问 Twitter。据开放国家基金会周日发布的一篇帖子称，Twitter 于周五撤销了对 Diplotwoops 以及全球 30 个国家所有剩余 Politwoops 网站的 API 访问。在此之前，推特在 5 月份以违反服务条款为由关闭了美国版的 Politwoops。

实际上，剩下的账户用了这么长时间才被暂停有点奇怪。

允许美国版本的 Politwoops 运行的代码是由荷兰组织开放州基金会在三年前首先开发的。然后这个组织与美国的阳光基金会共享代码，该组织同样致力于使政府更加透明。

跟踪服务允许包括记者在内的组织和个人，通过监控他们的账户中被删除的推文，来跟踪政治家和外交官在 Twitter 上说了什么。

正如 TechCrunch 早在 2012 年解释的那样，当 Politwoops 首次上线时，该服务不仅突出显示了被删除的内容，还显示了政治家及其工作人员花了多长时间进行删除。

在某些情况下，被删除的推文只是那些有错别字或跑题的推文，但在其他时候，它们有助于揭示政客们想要回避的公共评论。

例如，在美国关闭 Politwoops 的时候，Ars Technica 在其对该事件的报道中指出，该服务发现政客删除了他们对 Bowe Bergdahl 的赞美，Bowe Berg Dahl 是一名在阿富汗被扣为人质的士兵，后来他被指控犯有逃兵罪。

在最初关闭时，Twitter 提供了一份声明，解释其在此事上的决定，指出该服务违反了其开发者政策:

“我们强烈支持 Sunlight 增加政治透明度的使命，并使用公民技术和公开数据让政府对选民负责，但保留已删除的推文违反了我们的开发者协议。尊重所有账户的用户隐私是我们的首要任务，无论用户是匿名用户还是国会议员。”

然而，尽管今年早些时候暂停了美国版的 Politwoops，该版本专注于美国议员被删除的推文，但开放州基金会继续在 30 个国家运行 Politwoops，包括欧洲议会。

此外，在 API 被撤销之前，该服务还支持以下国家:阿根廷、澳大利亚、加拿大、智利、克罗地亚、丹麦、葡萄牙、埃及、爱沙尼亚、法国、希腊、印度、爱尔兰、意大利、韩国、马其顿、挪威、比利时、英国、德国、荷兰、瑞典、西班牙和瑞士、突尼斯、土耳其和梵蒂冈。

根据开放州基金会的帖子，Twitter 表示其暂停访问 Politwoops 的决定是因为它不区分用户。或者，换句话说，推特上的每个人——即使他们是像政治家一样的公众人物——都有权表达自己，而不用担心他们的推文会成为永久记录。当然，这一观点与许多人——包括开放州基金会(Open State Foundation)或阳光基金会(Sunlight Foundation)等组织的人——对政治舞台上公众人物言论的看法有些冲突。

正如阳光基金会主席克里斯托弗·盖茨今年早些时候所说，“国会议员不会也不应该像普通公民一样对隐私抱有同样的期望。权力只有在透明的情况下才能被问责。”

“开放国家基金会”主任阿尔扬·法塞德现在也表达了同样的观点。

“当选政客公开发表的言论是有公开记录的。即使推特被删除，它也是议会历史的一部分，”他说。“这些推文曾经被发布，后来被删除。任何人都应该知道政治家在公开场合所说的话。这与打字错误无关，但这是一个独特的见解，表明当选政客的信息可以在没有通知的情况下改变。”

该组织表示，它将继续探索其他方式，以保持选举产生的政治家的公共信息在这一事件后可见，但没有进一步详细说明。

我们已经要求 Twitter 对此事发表评论，如果他们提供声明，我们将会更新。

El Fassed 还向我们提供了周五通过 Twitter 发送给他的组织的完整信息。全文如下:

你可能知道，2015 年 6 月 3 日，Twitter 暂停了美国主要 Politwoops 应用程序的 API 访问，因为它显示了违反我们开发者协议和政策的已删除推文。

这一决定是经过深思熟虑的内部审议和对若干因素的仔细考虑后作出的。最终，Twitter 的决定是由公司的核心价值观“捍卫和尊重用户的声音”所引导的。无论出于何种原因，删除一个人的推文的能力一直是 Twitter 服务的一个长期功能。想象一下，如果推特是不可改变的，不可撤销的，那会是多么令人伤脑筋——甚至可怕——的事情？没有一个用户比另一个用户更值得拥有这种能力。的确，删除一条推文是用户心声的表达。
经审查，您的应用程序被发现违反了 Twitter 的开发者协议和政策，特别是:

3.B.i .尽一切合理努力做以下事情，前提是当 Twitter 要求时，您必须立即采取此类行动:删除 Twitter 报告为已删除或过期的内容。"

Twitter 努力为我们平台上的所有合作伙伴保持一致的政策执行，因此我们暂停 API 访问与 http://politwoops.co.uk/countries 相关的密钥

一些人认为，Twitter 在这件事上的决定不仅仅是上传开发者政策的简单愿望，而且也是关于该公司——该公司正在努力扩大其用户群——希望确保知名人士不会对使用其服务感到疏远。毕竟，如果政客们担心他们在 Twitter 上说的一切都可能以这种方式存档，他们可能在未来不太可能使用该网站。
