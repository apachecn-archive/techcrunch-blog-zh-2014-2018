# 三星 Galaxy Note 4 遭遇#GapGate

新的一天，新的一扇门。据报道，这一次，三星 Galaxy Note 4，这款吸引 Android 人群的最新平板电脑，在屏幕和外壳壁之间有一个薄间隙。这个问题于今天在韩国报道，似乎影响了一小部分设备。

根据这篇文章，两张纸或一张名片可以恰好放入缝隙中。这可能会将灰尘和液体带入设备。

该公司刚刚在韩国发布了 Galaxy Note 4，首批 3 万台销售一空。三星韩国公司似乎意识到了这个问题~~，并将在美国上市前解决这个问题~~将会忽略这个问题，至少根据 [TrustedReviews](https://web.archive.org/web/20230129101025/http://www.trustedreviews.com/news/samsung-galaxy-note-4-launch-hit-by-build-quality-complaints#55GsPMEbbPesLfd7.99) 报道:

￼"The 报告的问题不会影响 Galaxy Note 4 的功能或质量，”三星官方发言人告诉我们。“我们向客户保证，所有 Galaxy Note 4 设备都符合我们严格的制造和质量控制标准。”
