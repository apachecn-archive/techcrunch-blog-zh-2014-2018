Google Cloud Launcher 允许开发人员快速部署 120 多个流行的开源软件包 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/03/26/google-cloud-launcher-lets-developers-quickly-deploy-over-120-popular-open-source-packages/amp/

谷歌和 Bitnami 正在发布谷歌云平台的一项新功能，这使得开发者可以更容易地将 120 多种流行的开源应用部署到其云平台上。谷歌和 Bitnami 总共打包了超过 120 个应用程序，包括 WordPress、Drupal、Redis、MongoDB、Gitlab 和 Django，以及 Ruby、LAMP apps、Puppet 和其他应用程序的基础设施栈。

正如 Google 所说，建立一个基于 VM 的解决方案通常意味着你必须花相当多的时间来配置你的服务的不同组件。使用 Cloud Launcher，开发人员只需选择他们想要运行的应用程序或服务，只需点击几下，一切就都启动并运行了。

“开发者应该把大部分时间花在设计和编写代码上，”谷歌产品经理 Varun Talwar 在今天的公告中写道。“花在查找和部署库、修复依赖关系、解决版本问题和配置工具上的时间与工作无关。”

正如谷歌在今天的公告中指出的那样，该团队也正在努力将许多这些软件包与该公司的云监控服务相集成。

值得注意的是，谷歌已经为一系列流行的应用程序提供了有点类似的“点击部署”服务，如 Casandra 和 MongoDB 数据库、Drupal、Joomla 和各种开发栈。据我所知，这项新服务现在将现有功能与 Bitnami 的软件包结合在一起。
