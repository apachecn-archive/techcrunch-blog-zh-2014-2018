# 苹果公司首席执行官蒂姆·库克表示，科技公司在气候变化问题上不应妥协

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/09/22/apple-ceo-tim-cook-on-climate-change/?ncid=txtlnkusaolp00000058>

苹果公司首席执行官蒂姆·库克今天在纽约气候周与联合国气候变化框架公约执行秘书克里斯蒂娜·菲格雷斯坐下来讨论苹果公司对气候变化的担忧，以及该公司正在采取什么措施来应对这种情况。库克总结了他的公司的努力，他说苹果需要成为“池塘中激起涟漪的鹅卵石之一”，指的是激励科技行业领导者做出更广泛的努力来改善环境实践。

他在采访中说，“你看到的是根本原因，你不接受经济和环境之间有所取舍。”。“如果你创新，并且把标准设得很高，你会找到一种两全其美的方法。你必须双管齐下，因为不解决环境问题的长期后果是巨大的。”

库克说，这种不妥协的态度是苹果在产品设计和开发过程中已经有过大量实践的东西，这也是为什么在这里应用它是有意义的。

“这就是苹果公司的每个人都不接受你必须选择这个或那个的原因，”如果我们对我们的产品采取这种方法，我们永远不会做出很大的妥协。事实是你不能妥协。"

作为苹果环境政策领导力的一个例子，他引用了目前正在库比蒂诺建设的新总部，库克表示，他预计这将是地球上最绿色的建筑之一。他还提到了使用可再生能源发电的数据中心，以及他们在供应链上为保持合作伙伴的一致性所做的努力。库克表示，这需要真实的、动手的工作。

“这是肮脏和细致的工作，”他谈到他们的供应链审计和合作伙伴责任计划。它卷起你的袖子。不是深奥和理论，是实实在在的工作和实实在在的项目。"

库克关于如何改善科技行业现状及其对气候变化形势改善的总体贡献的一般性建议，可以最好地总结为呼吁提高透明度。库克没有直接点名任何人，但仍然设法挑战苹果的竞争对手，让他们在让环境影响成为公共记录的一部分方面做得更多。

“我认为公司必须向消费者传达他们在卖什么，而且他们必须传达他们的全部足迹，而不仅仅是他们好看的那一部分，而是全部，”他说。“我认为如果你这样做，我是一个乐观主义者，我认为如果你这样做，消费者是聪明的，世界上绝大多数人都想做正确的事情，所以我认为(透明度)将推动人们期望的消费者行为。如果有足够多的公司开始这么做，我认为消费者会用他们的钱投票。”

苹果自己的环保努力最近赢得了绿色和平组织的大量赞誉，目前它在主要科技公司中排名第一，特别是在可再生能源的使用方面。
