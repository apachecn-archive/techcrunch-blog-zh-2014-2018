企业版电脑双倍下注 TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2014/12/09/box-doubles-down-on-the-enterprise/?ncid=rss

博克斯对企业号加倍下注

Box 是一家为大型企业提供文档存储和协作工具的公司，它今天宣布了一项名为 Box Trust 的新计划，该计划为其平台带来了 19 个新合作伙伴，为其客户群提供了更多的安全选项。

Trust 项目加入了一个名为“企业移动管理盒子”的新应用程序，它将帮助大公司更好地管理他们的移动 IT 设置。数据和应用的移动管理是企业 IT 团队仍在努力应对的重要领域。

Box 的新闻紧随其竞争对手 Dropbox 宣布其 Dropbox for Business API 产品的一系列合作伙伴之后。这两家公司都非常专注于增加他们在企业市场的市场份额——大公司拥有大量资金，因此会成为强大的长期客户。具体来说，Box 一直热衷于向通用电气(General Electric)的客户群强调其超过 30 万个座位的部署。

Box Trust 包括像 Palo Alto Networks 这样的合作伙伴，以及为大公司提供企业级安全工具的其他大公司。

该计划是 Box 成为企业领先云存储提供商的工作的一部分。作为先行者，该公司经历了来自 Dropbox 等年轻公司以及微软和谷歌等老牌公司的日益激烈的竞争。与无限的基于云的存储的市场平均水平不同是其利基的新计算。

因此，Box 必须确保它的平台上有许多专注于安全的公司，这样任何拥有大量用户的公司都可以以自己选择的方式动摇安全。在一篇博客文章中，Box 首席执行官 Aaron Levie 用以下术语描述了信任工作:“我们正在与合作伙伴的蓬勃发展的生态系统一起建立一个安全模型，以提供所有新的工具和服务来保护企业中的内容。”

对这种情绪的考验将是市场反应。新的信托工作会带来更多的客户吗？如果是这样的话，Box 拖延已久的 IPO 可能会在明年第一季度末之前胎死腹中。如果不是，那么，成长是有代价的。
