# Yandex 通过人工智能、浏览器和启动器合作项目吸引智能手机制造商

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/10/12/yandex-woos-smartphone-makers-with-ai-browser-and-launcher-partnership-programs/>

在俄罗斯成功提起针对谷歌将服务与安卓操作系统捆绑销售的高调反垄断诉讼后，俄罗斯国家搜索巨头 [Yandex](https://web.archive.org/web/20230317210203/http://yandex.com/) 正试图将其在移动领域的竞争力提升一个档次。

为了在目前的核心市场俄罗斯之外发展业务，该公司推出了一个新的合作伙伴计划，与智能手机制造商合作，将提供由 Yandex Zen 人工智能引擎驱动的 Android 启动器和浏览器，作为预装服务，取代通常与谷歌 Android OS 捆绑在一起的产品。

![zen-feed](img/f692f0db86d68c245803d8726a37b4c5.png)除了浏览网页或你的应用程序等基本功能，Yandex 的服务还将提供内容流，包括文章、新闻、视频、图像的提醒，“以及以无限订阅的形式匹配个人兴趣的其他故事”

该计划的早期采用者包括一些较小的手机制造商和一些知名度较高的公司:Fly、LAVA、Mobile TeleSystems、Multilaser 和 Posh Mobile 还有 Wileyfox 和中兴，他们将在不同程度上定制浏览器和启动器。

(例如，中兴将进行白标上市，你永远不会知道幕后黑手是 Yandex 其他人正在使用 Yandex 品牌。)

总之，Yandex 国际业务发展副总裁 Artem Fokin 告诉我，新设备将在欧洲、印度、拉丁美洲和非洲销售，覆盖全球约 70%的互联网人口。

根据 Gartner 的数据，瞄准较小的手机制造商是一个有趣的策略:单独来看，它们可能不会对苹果或三星这样的公司构成威胁，但合在一起，它们占所有智能手机销售额的一半不到。

考虑到大规模生产智能手机的经济性，以及需要产品与众不同，他们也是最有可能尝试这种产品的候选人。

“设备制造商经常不得不在以巨大的管理成本开发内部软件和牺牲产品差异化和额外收入之间做出选择。有些 OEM 厂商会预装尽可能多的 app。不幸的是，这对用户体验有负面影响，从长远来看，对他们的品牌不利，”福金在一份声明中说。“我们的软件和人工智能技术旨在撼动市场，帮助我们的合作伙伴克服这些挑战，减少客户流失，提高利润率。”

Yandex 在这里更大的商业动力是显而易见的:该公司是俄罗斯和周边国家最大的搜索引擎，但增长空间有限，尤其是在被谷歌垄断的更广泛的全球搜索市场。

与此同时，Yandex——与谷歌类似——建立了大量技术来支持其在俄罗斯市场的服务，如地图、新闻提醒等，既增加了搜索收入，又利用这种优势在其他领域拓展业务。现在，鉴于它在全球桌面搜索领域基本上输给了谷歌，Yandex 希望它能为其他平台开发其他产品，这些平台仍有增长和变化的空间。

这并不是 Yandex 第一次尝试利用移动和智能手机使用的增长来发展其在俄罗斯本土市场以外的业务。回到 [2014 年，它发布了 Yandex。套件](https://web.archive.org/web/20230317210203/https://techcrunch.com/2014/02/19/yandex-kit/)，一个完整的固件更新，本质上取代了谷歌在 Android 手机上的整个移动应用程序组合。签约使用它的原始设备制造商包括华为这样的公司，但正如福金向我指出的那样，这在很大程度上是一项没有扩展到俄罗斯市场以外的努力。

今天的发布会有点不同，因为 Yandex 不打算在智能手机上做全套(至少现在还没有)。它有意只推出两项服务——一个浏览器和一个启动器——在俄罗斯等少数核心市场之外，它甚至没有捆绑 Yandex 搜索等服务。

“之前我们专注于国内市场，这是我们分发 Yandex 搜索的方式，”Fokin 说。“现在我们已经把它变成了一个全球性的产品，从搜索中分离出来。例如，如果它安装在巴西，默认搜索是谷歌，因为这是该市场的预期。”

Fokin 告诉我，Yandex Search 被排除在外的另一个原因是，在大多数市场，Yandex 没有像在本土市场那样的基础设施来赚钱。相反，其合作项目背后的商业模式将基于“以最不引人注目的方式做广告”，他说。

他的意思是没有弹出窗口，并且大部分内容以“原生内容”的形式出现在更大的内容提要中。如果用户点击一个链接，Yandex 和原始设备制造商将分享收入。除了说这是“收入的一个重要部分”和“一个公平的份额”之外，没有说双方有多少百分比

鉴于移动互联网浏览器和智能手机发射器已经存在了一段时间，你可能会想，Yandex 在这里做的事情到底会有什么不同或更有吸引力。

福金告诉我，这就是人工智能引擎 Yandex Zen 的用武之地:机器学习将跟踪你如何滚动和暂停某些链接以阅读它们，通过你的浏览习惯跟踪你感兴趣的内容，并将其与 Yandex 更大的搜索索引算法相结合，以开发出你可能想看的更准确的图片，无论是在内容推荐还是广告方面。

他说，在这些产品的早期实施中，人们已经在 Yandex 的浏览器和启动器上平均花费了大约 20 分钟，这个数字还在增长。

他补充说，Yandex 的人工智能还支持该公司在浏览器中内置的广告拦截功能。默认情况下，什么都不会被屏蔽，但如果用户决定禁止或屏蔽侵入性广告，Yandex 会了解这一点，并在未来减少同样的服务，还会向出版商提供数据，以便他们知道哪些广告不起作用。

他说，所有这些都是内部建造的。“自 1997 年以来，我们一直在为 Yandex 本身制造人工智能和推荐产品，”他说。"我们认为这是市场上最好的产品之一。"

Fokin 没有透露 Yandex 可能在合作项目中包括哪些其他 Zen-powered 服务，只是指出他们暂时将继续专注于内容，在推荐和发现领域提供产品。
