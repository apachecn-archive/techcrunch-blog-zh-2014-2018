# 智能玩具制造商 Elemental Path 开始接受 IBM Watson 驱动的恐龙的预购订单

Elemental Path 现在接受其智能玩具产品的预购。

这家智能玩具制造商今年在 Kickstarter 上首次亮相，推出了一系列名为 CogniToys 的玩具，或与玩耍的孩子一起学习和成长的玩具。这些玩具，包括一只 3D 打印的恐龙，由 IBM 的沃森超级计算机提供动力，以帮助每个玩具倾听和参与其孩子的主人，并帮助他们学习新的东西。

该公司表示，这些玩具可以在几秒钟内回答儿童的问题，而且儿童越是使用玩具来了解这个世界，它们就会变得越聪明。

Elemental Path Kickstarter 活动超过了最初的 5 万美元目标，在 2015 年 3 月从 2256 名支持者那里筹集了 27.5 万美元。这家聪明的玩具制造商很快就开始生产恐龙玩具——增加了不同的颜色和原型，粉丝可以投票决定。

该公司还在后端推出了一个家长面板，允许父母访问一个安全的基于网络的仪表板，以监控他们孩子的 Dino 进度。仪表板的未来版本将允许增加父母控制，例如设置就寝时间或通过玩具向孩子发送消息的能力。

smart Dinos 将只在有限的时间内小批量提供预购。Elemental Path 随后计划在 2016 年假期前停止其第一版产品的订单。
