SmartThings支持即将登陆亚马逊Echo TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/08/21/smartthings-support-is-coming-to-amazon-echo/

亚马逊似乎计划与SmartThings合作，通过语音驱动的个人助理设备亚马逊Echo来实现各种家庭自动化功能。

根据SmartThings博客的谷歌网络缓存，亚马逊Echo将很快能够通过对Alexa的简单呼喊为家中任何SmartThings兼容设备供电。这包括插入SmartThings电源插座的非SmartThings产品，以及该公司的开关和灯。

基于一个演示视频(它被设置为私人，但我们猜到了密码:“Alexa”)，同时拥有Echo和任何SmartThings产品的用户将能够通过名称要求Alexa打开各种设备，甚至调暗灯光。

这篇博文已经被撤下，所以还不知道什么时候会正式发布。

亚马逊Echo于2014年11月在首次亮相，仅限注册邀请的Prime会员使用。该设备的形状像一个薄纸巾卷，插在电源插座上，连接到WiFi，以满足任何和所有的家庭需求。

通过一个永远在线的麦克风，Echo可以监听一个命令词(“Alexa”)，并能够执行各种谷歌搜索来回答查询，向你的亚马逊购物清单添加商品，甚至讲笑话。

今年6月，Echo以179美元的价格向美国的任何人提供，几天后亚马逊为开发者开放了Alexa API ，以便与语音驱动的物联网设备集成，用于其他尚未发现的用例。

就众所周知的智能家居而言，亚马逊最初利用Wink中枢和兼容设备为家中的不同功能提供动力。

但根据SmartThings现在发布的博客文章，亚马逊似乎正在扩大Echo用户的智能家居选择。

以下是这篇博文的内容:

我们很高兴地宣布，SmartThings现在与Amazon Echo兼容！

对于那些不熟悉的人来说，Amazon Echo是一个连接互联网的扬声器，可以对你的命令和问题做出反应。(就当是你高大黝黑帅气的私人助理吧。)

通过以“Alexa”开始句子，你可以播放音乐，听新闻，获得预报，现在…控制你的SmartThings兼容的灯，开关和任何插入SmartThings电源插座的东西。
