# LG G6 有一个内置的正反面功能

哦，天哪，我错过了[前锋](https://web.archive.org/web/20221207134427/https://beta.techcrunch.com/tag/frontback/)。这个移动应用程序创建了一个有趣的照片格式，有两张方形的图片来分享你面前的东西和一张自拍。虽然你仍然可以[下载](https://web.archive.org/web/20221207134427/https://beta.techcrunch.com/2015/08/03/frontback-is-back-with-a-new-team/)这款应用，但它已经停滞了几年。但是 LG 在新 G6 的[中实现了一个名为“比赛镜头”的前后功能](https://web.archive.org/web/20221207134427/https://beta.techcrunch.com/2017/02/26/lg-g6-reveal/)

LG 手机最大的新特点是[更高的显示屏](https://web.archive.org/web/20221207134427/https://beta.techcrunch.com/2017/02/26/lg-proves-that-taller-screens-are-the-future-of-smartphones/)。该公司已经用 2:1 显示屏取代了 16:9 显示屏。正好是两个方块叠在一起。

LG 想展示你可以用这种不同的屏幕格式做什么。于是有了新的方形相机应用。例如，你可以拍摄一张方形照片，并在取景器下方立即以全尺寸预览。如果你想确保你得到了完美的 Instagram 照片，这很好。

但我更感兴趣的是比赛镜头功能。在这种模式下，你可以自拍，拍下你正在看的东西。这是两个完美的正方形，一个在另一个上面。基本都是前后脚。

一方面，我很兴奋能再次在手机上拥有某种前后功能。另一方面，它说了一些关于新媒体格式的事情。

许多创业公司试图创造一种新的格式，从 Vine 到 Mindie，Frontback 等等。但围绕它建立一个创业公司就够了吗？在许多情况下，格式可能很有趣，但很难将它们转化为业务。只有 Instagram 和 Snapchat 变得可持续。无数其他人失败了。

然后它们会出现在韩国手机上…
