# 拓扑让你在购买使用 AR 的眼镜之前进行尝试

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/10/02/topology-lets-you-try-before-you-buy-glasses-using-ar-in-an-app/>

[拓扑眼镜](https://web.archive.org/web/20221025235636/https://www.topologyeyewear.com/)是一款增强现实应用，通过对你的面部进行 3D 扫描，提供定制眼镜。

大多数眼镜都是批量生产的，以适应一个假定对称的头部。问题是，我们大多数人都没有那么对称。iOS 上的拓扑应用程序使用面部扫描技术来考虑耳朵可能伸得更远或坐得不均匀，鼻子更宽或更薄以及其他面部因素，这些因素可能会使大多数眼镜不太合适。

连续创业家和创始人埃里克·瓦拉迪(Eric Varady)在旧金山码头附近购买一双适合自己脸型的鞋子时想出了这个主意。他看到一副非常昂贵的镜架，但仍然不符合他的要求，这时他意识到他可以在家里创造一个更简单的解决方案。

当然，在家购买眼镜已经有几种选择了。Warby Parker 通过一个平台主导了这里的市场，该平台允许消费者选择各种各样的眼镜，它将运往任何地方，并允许你在购买前亲自试穿。

拓扑学让你可以使用增强现实技术，在各种不同的镜框风格和颜色中看到自己的样子，而不是运送无数的眼镜给你。但是，如上所述，真正的秘密是通过对你的脸进行双耳 3D 扫描来定制。

它的工作原理是首先通过 Topology 应用程序拍摄视频自拍，该应用程序旨在完美捕捉你的面部测量数据，然后创建定制的眼镜，而没有桥梁滑动或框架不均匀的麻烦。在那里，你可以试戴任何数量的眼镜，调整镜片的高度、宽度和其他一些功能，以获得你想要的外观和适合度。

然而，尽管如此，价格并不便宜。每副拓扑眼镜的起价为 495 美元，最高可达 800 美元，具体价格取决于附加产品。根据一份光学实验室消费者报告，与沃比·帕克处方眼镜的价格相比，沃比·帕克处方眼镜的起价为[95 美元、](https://web.archive.org/web/20221025235636/https://www.warbyparker.com/)或一副普通的成品眼镜，通常为[150 至 400 美元](https://web.archive.org/web/20221025235636/https://opticianworks.com/the-consumers-guide-10-11-15/)。

Varady 认为，消费者将愿意支付更高的价格，以获得一个简单的定制适合的远程订单。

他告诉 TechCrunch:“是的，我们很贵，但实际上我们正在市场上追逐那些已经在眼镜和镜片上花费那么多甚至更多的人。”“以前从来没有人定制过，它们非常适合你。”

拓扑刚刚在 7 月推出，并表示它已经收到了“几百个”订单它现在开始了一场数字营销活动，以增加订单和产品的知名度，并从 OPI 指甲油的创始人乔治·谢弗(George Schaeffer)那里获得了约 400 万美元的初始种子资金，以帮助其发展。

我亲自测试了一下，发现这款应用是为增强现实软件的有趣用途而设计的，并且有大量的帧样式可供选择。我特别喜欢让你玩镜头高度和长度的功能。

至于有多合身？你将不得不为你自己的脸作出判断。
