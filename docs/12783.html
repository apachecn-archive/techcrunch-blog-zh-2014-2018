谷歌在今年的 I/O 大会上谈到的 Android 超越手机的一个领域是汽车，而汽车中的 Android 是它扩大其车载存在的主要方式。其 Android Auto 软件的扩展提供了与车辆信息娱乐和气候控制系统的更深入集成，并将 Assistant 置于车内和车外更多语音控制的前端和中央。

当然，该系统提供谷歌地图导航，正如在沃尔沃 V90 上演示的那样，该车配备了专门用于演示目的的安卓系统。但它也提供了方便的访问您的客舱空调和热量，你可以要求沃尔沃简单地冷却或加热东西来打开或关闭空气。基于 Android 的操作系统还可以控制窗户、天窗等。

您还可以在舒适的汽车驾驶舱中通过语音命令使用与 Assistant 兼容的智能家居设备。我们的体验向导，谷歌高级产品经理 Haris Ramic，向我们展示了如何使用 Hue 灯泡和 Nest learning 恒温器来实现这一点，这些灯泡和恒温器安装在模拟你的房子的木头背景上，用于演示目的。

基本上，汽车上的 Android 意味着你可以随身携带大量的移动体验，现在 Assistant 正在改进跨设备功能，谷歌正在与开发者一起推动它，这对消费者来说应该是一个非常好的消息。

我们还需要等待一段时间，才能让这种技术应用到你可以实际购买的汽车上:沃尔沃表示，它将在两年内的某个时候首次亮相新车，但它没有透露更多具体信息。
