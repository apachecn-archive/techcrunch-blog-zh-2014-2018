当然，微软正在开发自己版本的 Chromebook

Windows Central 已经得到了一份强调 Windows 10 Cloud 硬件性能的内部文件，这是一个传闻中的操作系统衍生产品，预计将于下个月在纽约举行的大型教育活动上发布。可以预见的是，幻灯片将操作系统与谷歌 Chromebooks 进行了正面交锋。

对于微软来说，这个类别是显而易见的，因为该公司正在努力在 K-12 教育领域站稳脚跟，该领域完全由谷歌基于云的产品主导。

微软在 2000 年代末看到了这一类别的一个高峰，这要归功于短暂但充满活力的上网本热潮，它帮助教育工作者实现了长期以来希望的学生一对一设备的承诺，但随着平板电脑的出现，这种情况开始发生变化。然而，近年来，Chromebooks 通过结合廉价的硬件和为学校和学区做出许多购买决定的 IT 部门的控制，完全主导了这场对话。

微软最近的主要教育活动一直在 Surface 方面，尽管 iPads 似乎在为不介意为设备支付额外费用的学校赢得市场方面做得更好。据传，该公司将在发布会上推出 Surface Phone 或新的 Surface Pro，但我们能够证实并非如此。

微软无疑暗示了其回归低成本教育产品的目标。今年 1 月，该公司讨论了针对教育的 Intune，它将基于云的管理引入了面向学校的 189 美元的 Windows 10。将于 5 月 2 日在 T2 举行的活动几乎肯定会详细介绍它认为能够在电池寿命、成本和大规模设备管理等方面与 Chromebooks 抗衡的设备。

虽然该公司最近在教育方面运气不佳，但鉴于微软仍受欢迎的生产力选项，可能仍有重返的需求。谷歌在基于云的 G 套件方面取得了很大进展，但在专业人士中，Office 等产品仍然不受欢迎。对于一个像 Windows 一样提供一个几乎无处不在的生态系统版本的系统，也有一些东西要说。

但即使有这些因素在起作用，如果微软希望在教育领域有所作为，它将不得不提供比 Windows 版 Chromebook 更多的东西。
