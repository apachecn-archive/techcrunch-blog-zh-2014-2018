Terpon 正在为直播色情制作虚拟现实网络摄像头

原文：https://web.archive.org/web/https://techcrunch.com/2017/02/21/terpon-is-making-vr-webcams-just-for-live-streaming-porn/

一家瑞士初创公司，Terpon SA，正在专门为成人娱乐业制作虚拟现实网络摄像头。这些相机不能购买，但客户可以以每月约 30 美元的价格租用，这个价格包括 24 小时的技术支持。Terpon 首席执行官 Jean-Claude Artonne 表示，他希望让世界各地的成年表演者能够创作身临其境的优质内容，并通过他们选择的任何直播平台出售给粉丝。

这位首席执行官表示，虽然虚拟现实和 3d 内容是蓬勃发展的趋势，但关注直播是关键。“由于网络上的免费内容，成人娱乐业中的录制内容已经死亡或即将死亡。该行业正在增长的部分是直播。但是这些网站的大多数访问者只消费免费的东西。如果你给他们 3d 内容，他们会为不同的东西付费。尤其是如果你已经有了一个虚拟现实耳机，你就会致力于在网上寻找一些新的、有趣的体验。”

该公司的 3K 分辨率 Hermes 相机和 4K 分辨率 Artemis 相机从理论上看，就像一张大眼睛指向拍摄对象的脸。工业设计背后的想法是让表演者对粉丝的注视有一种自然的感觉。

根据公司声明，“Terpon 的 3D-VR 相机具有 200 度视野的立体同步模块。该相机可传输 2D 或 3D 视频，通过 USB 即插即用，因此可以在 PC 或 Mac 上工作。”该相机与任何提供 API 的网络摄像头平台兼容，或者可以与表演者自己的网站或聊天应用程序一起使用。

Terpon 首席执行官让-克洛德·阿东。

阿东解释了为什么公司不直接出售相机，而是出租。“大多数 cam 女孩生活在每月收入 300 至 500 美元的国家。对于想在罗马尼亚或哥伦比亚从事这类业务的人来说，拥有符合预算的东西非常重要，按地理位置来看，这两个国家是 livecam 成人娱乐的最大提供商。”为了使其产品在业内广为人知，Terpon 本月将向有影响力的表演者赠送 1000 台 beta 版相机。

该公司已经从一家天使投资者那里筹集了一笔未披露金额的外部资金，该公司无权透露该天使投资者的名字。Terpon 打算筹集更多的种子或风险资本来推动其相机的工业生产和营销，但 Artonne 承认传统投资者可能不热衷于涉足成人娱乐。

也就是说，Artonne 指出，许多成为主流的技术首先通过成人娱乐获得了牵引力，包括 VHS 磁带、DVD 和网络摄像头。Artonne 说，“我们不打算长期局限于成人娱乐内容创作者。但是制造满足他们需求的产品是很重要的。拥有一家完全致力于这个市场的公司，将有助于我们在未来为通才市场创造一些东西。”

Terpon 面临着许多中国网络摄像头制造商的竞争，这些制造商瞄准了成人娱乐业，但也有尼康、三星和联想等强大的硬件公司，为虚拟现实时代创造了网络摄像头。
