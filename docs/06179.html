Axway 收购移动应用开发平台 Appcelerator TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2016/01/17/axway-acquires-mobile-app-development-platform-appcelerator/?ncid=rss

已获取 Appcelerator。该移动应用开发平台的客户中包括 T-Mobile、PayPal 和 GameStop 等公司，该平台已被 B2B 企业公司 Axway 以全现金交易收购。

很有可能， Axway 并不是一家你熟悉的公司，但它是一家总部位于亚利桑那州斯科茨代尔的大型上市公司，提供软件和服务帮助企业处理数据流。它是更大的总部位于法国的 Sopra Steria 集团的子公司。Axway 目前拥有约 11，000 名客户和约 2，000 名员工。2015 年第三季度，€的收入为 6590 万英镑。

Appcelerator 拥有几百名客户，它最出名的可能还是用于编写移动应用程序的 Titanium 框架。在这次收购之前，该公司筹集了大约 8790 万美元。它的一些投资者包括 Accel，Mayfield Fund，Red Hat，EDB 风险投资公司，Rembrandt Venture Partners，Storm Ventures，Union Grove Venture Partners 和 Translink Captial。

虽然 Appcelerator now Axway 都不愿意透露收购的价格，但 Appcelerator 首席执行官兼联合创始人 Jeff Haynie 告诉我，交易已经完成，钱在银行里。他指出，这笔交易对投资者和 Appcelerator 的员工都有好处。

正如海妮告诉我的那样，这些年来，他的公司有很多追求者。“当 Axway 接近我们时，我们很兴奋，”他告诉我——主要是因为 Axway 在该领域的市场领导者形象，以及它的产品组合和 Appcelerator 的产品组合几乎没有重叠的事实。“这桩婚姻的好处之一是没有重叠，”他说。“他们在移动性和 API 方面与我们完全不同。”他还指出，像 Axway 现有的 API 网关和管理工具等工具也在 Appcelerator 的路线图中。“从产品路线图来看，这是一个直接的战略优势，”Haynie 说。

Axway 首席执行官 Jean-Marc Lazzari 在另一次采访中对此表示赞同。他还强调，两家公司已经分享了一些客户，但因为它们过去的销售方式不同 Appcelerator 专注于直接向开发人员销售，而 Axway 以更传统的亲力亲为的企业方式销售——它们经常与这些公司内部的不同群体进行对话。

那么，这次收购的真正目的是为企业提供一个端到端的解决方案，从接收数据到在移动设备上提供数据。到目前为止，Axway 的投资组合中明显缺少移动业务。使用 Axway 的服务，他们能够通过 API 提供他们的数据，但这是该公司迄今为止能够为其客户提供的最大限度。

对于 Axway 来说，这次收购也是为了让其客户转向双模 IT——也就是说，为他们提供其产品传统上提供的稳定性和安全性，以及变得更加敏捷和快速以保持竞争力的能力。

Axway 和 Appcelerator 显然是非常不同的公司，拉扎里和海涅都承认这一点。但拉扎里也强调，Axway 仍然相对较小且灵活，约有 2000 名员工，他自己也经营过初创公司。“我们不想拥抱[Appcelerator]至死，”拉扎里告诉我。“我们进行了长时间的讨论，我以前也做过类似的事情——从一家初创公司发展成一家更大的实体。”

拉扎里还指出，这次收购将使 Axway 在硅谷的存在。“我们可能会将 Appcelerator 用作一种创新中心或数字实验室，”他说，但他和 Haynie 都指出，Appcelerator 的产品不会消失。“Titanium 和 Appcelerator 这两个品牌都是强势品牌，我们会保留它们，”他说。“我们无意不尝试最大限度地利用我们正在收购的品牌。”
