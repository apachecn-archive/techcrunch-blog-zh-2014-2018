# 优步跨越 50 亿美元的旅行里程碑

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/06/29/uber-crosses-the-5-billion-trip-milestone-amid-ongoing-issues/>

# 在持续的问题中，优步跨越了 50 亿美元的里程碑

优步骑手已经使用该公司今天宣布的服务进行了 50 亿次旅行，比 2016 年创下的 20 亿次里程碑增加了一倍多。5 月份实际上突破了 50 亿大关。优步说，从格林威治时间 5 月 20 日早上 7:29:06 开始，有 156 次出行，将打车服务推向了边缘。

对于优步来说，这是一次很大的旅行，考虑到其首席执行官刚刚辞职，对其文化的独立调查结果发现了很多问题，并且它正在处理越来越多的法律问题，它可能正在庆祝云下的牵引力。然而，目前还很难说这对优步的实际业务有多大影响；该公司一直在宣传 2017 年乘客人数的增长，尽管最近的第三方统计数据包括应用下载量表明，Lyft 等竞争对手开始获得优步的市场领先地位。

优步的第 50 亿次旅行不仅仅是让这家陷入困境的公司享受一次好消息；该公司还奖励了所有 156 名提供乘车服务的司机，并向他们发放了 500 美元的奖金。骑手——只要回顾你的历史，找出你是否是其中的一部分，你的回报就是你帮助建立这个参与统计的知识。
