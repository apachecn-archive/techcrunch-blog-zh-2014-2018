# Equifax 表示将免除 30 天的信用冻结费

为了回应公众对其持续拙劣回应的愤怒，Equifax 在 Twitter 上声明将免除 30 天的信用冻结费。由于有如此多的个人数据在网上流传，信用冻结是那些受 Equifax 违规事件影响的人唯一可以用来保护自己的事情之一。考虑到社会安全号码是终身有效的，Equifax 现有的一年免费信用监测服务只是在伤口上撒盐。

昨天，夏威夷州参议员 Brian Schatz 驳斥了该公司迄今为止的努力，称其“[不足，原因有几个](https://web.archive.org/web/20221005100451/https://beta.techcrunch.com/2017/09/11/senator-says-equifax-should-offer-customers-free-credit-security-freezes/?ncid=rss)”，呼吁 Equifax 覆盖 tri-bureau 信用冻结，并提供无限制的免费信用监控。

一个免费的单一局信用冻结的问题是，你真的需要做所有三大信用局。否则，潜在的身份窃贼可以以你的名义在只使用 TransUnion 或 Experian 的贷款机构开立信用账户。Equifax 很清楚这一点，但似乎更感兴趣的是在现有信贷服务短暂的免费期结束后从这些服务中赚钱，而不是将这些服务交给愤怒的消费者，让他们支付其他地方或更长时间的费用。
