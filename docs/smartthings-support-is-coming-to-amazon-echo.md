# SmartThings 支持即将登陆亚马逊 Echo 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/08/21/smartthings-support-is-coming-to-amazon-echo/>

亚马逊似乎计划与 SmartThings 合作，通过语音驱动的个人助理设备亚马逊 Echo 来实现各种家庭自动化功能。

根据 SmartThings 博客的谷歌网络缓存，亚马逊 Echo 将很快能够通过对 Alexa 的简单呼喊为家中任何 SmartThings 兼容设备供电。这包括插入 SmartThings 电源插座的非 SmartThings 产品，以及该公司的开关和灯。

基于一个[演示视频](https://web.archive.org/web/20221208225345/https://vimeo.com/136672350)(它被设置为私人，但我们猜到了密码:“Alexa”)，同时拥有 Echo 和任何 SmartThings 产品的用户将能够通过名称要求 Alexa 打开各种设备，甚至调暗灯光。

这篇博文已经被撤下，所以还不知道什么时候会正式发布。

亚马逊 Echo 于 2014 年 11 月在[首次亮相，仅限注册邀请的 Prime 会员使用。该设备的形状像一个薄纸巾卷，插在电源插座上，连接到 WiFi，以满足任何和所有的家庭需求。](https://web.archive.org/web/20221208225345/https://beta.techcrunch.com/2014/11/06/amazon-echo/)

通过一个永远在线的麦克风，Echo 可以监听一个命令词(“Alexa”)，并能够执行各种谷歌搜索来回答查询，向你的亚马逊购物清单添加商品，甚至讲笑话。

今年 6 月，Echo 以 179 美元的价格[向美国的任何人](https://web.archive.org/web/20221208225345/https://beta.techcrunch.com/2015/06/23/amazon-echo-us-wide/)提供，几天后[亚马逊为开发者开放了 Alexa API](https://web.archive.org/web/20221208225345/https://beta.techcrunch.com/2015/06/25/amazon-unbundles-alexa-virtual-assistant-from-echo-with-new-dev-tools/) ，以便与语音驱动的物联网设备集成，用于其他尚未发现的用例。

就众所周知的智能家居而言，亚马逊最初[利用 Wink 中枢和兼容设备为家中的不同功能提供动力。](https://web.archive.org/web/20221208225345/http://www.cnet.com/news/wink-syncs-up-with-amazon-echo/)

但根据 SmartThings 现在发布的博客文章，亚马逊似乎正在扩大 Echo 用户的智能家居选择。

以下是这篇博文的内容:

> 我们很高兴地宣布，SmartThings 现在与 Amazon Echo 兼容！
> 
> 对于那些不熟悉的人来说，Amazon Echo 是一个连接互联网的扬声器，可以对你的命令和问题做出反应。(就当是你高大黝黑帅气的私人助理吧。)
> 
> 通过以“Alexa”开始句子，你可以播放音乐，听新闻，获得预报，现在…控制你的 SmartThings 兼容的灯，开关和任何插入 SmartThings 电源插座的东西。
