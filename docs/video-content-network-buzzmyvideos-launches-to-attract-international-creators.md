# 视频内容网络 BuzzMyVideos 推出吸引国际创作者

BuzzMyVideos 是一个新的内容网络，专注于国际在线视频内容推广，允许视频制作者通过多渠道网络和创作者计划发展和货币化他们的视频内容。它现在推出了测试版。这家初创公司迄今已吸引了 5000 多名视频制作人(他们为自己提供的工具付费)，部分原因是其创始人保拉·马里诺内和本古·阿塔默都是 YouTube 的前领导人，所以他们知道自己的东西。

主要在六个主要垂直行业运营；美妆、娱乐、美食、音乐、科技和游戏，BuzzMyVideos 声称其网络上已有超过 23 亿次观看和 1350 万订户。

这家总部位于伦敦的公司瞄准了国际视频创作社区，因为 YouTube 上超过 80%的浏览量来自美国以外，而 2016 年预计 1630 亿美元的数字广告支出中的 70%也将来自美国以外。因此，它已本地化为英语，俄语，土耳其语和意大利语，并计划在其他地区。

现在，这个领域有一些竞争对手。全屏[专注于多垂直领域的技术和品牌](https://web.archive.org/web/20221007151654/https://beta.techcrunch.com/2013/07/31/fullscreen-gives-video-creators-the-tools-to-collaborate-and-make-more-money-on-youtube/)。

[Maker Studios](https://web.archive.org/web/20221007151654/http://www.crunchbase.com/organization/maker-studios) 开始于内容制作，但也充当 MCN(多渠道网络)和聚合渠道。还有更多。

然而，BuzzMyVideos 认为它有优势，因为它的技术平台旨在增加观众和将内容货币化；创始人对这个行业了如指掌；以及将视频本地化为其他语言。
