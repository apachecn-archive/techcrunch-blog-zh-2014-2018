# Swarm 现在允许你给朋友的朋友发消息

Swarm 刚刚发布了一个更新，带来了朋友的朋友的朋友的能力信息。

这意味着，无论何时你在应用程序中遇到一个和你有共同好友的用户，你都可以直接给他们发送信息。

在接收端，您会收到通知，消息来自一个朋友的朋友，并可以选择接受或阻止该消息。用户还会被提示阻止所有朋友的朋友发来的消息。

Swarm 在三月份引入了[直接消息](https://web.archive.org/web/20221005145753/https://beta.techcrunch.com/2015/03/12/swarm-launches-direct-messaging-ditches-awkward-public-plans/)来改变最初的“公共计划”功能，这使得一切都变得[尴尬。](https://web.archive.org/web/20221005145753/http://www.urbandictionary.com/define.php?term=Awkward+Sauce)

有了直接消息，用户可以简单地挑选他们想一起出去玩的附近的朋友，而不是向他们的整个社交圈宣布。这经常会让用户得不到回应(因为没有人被特别邀请)，或者从他们一开始就不想看到的人那里得到回应。

直接消息解决了这个问题，并使 Swarm 成为一个真正的社交网络，提供真实世界的价值(与你的朋友见面的能力)。

另一方面，朋友的朋友的消息传递更具有探索性质，让用户发现新的潜在朋友。

Swarm 最近得到了 Foursquare(该公司)的大量关注，贴纸书的更新和即将发布的[承诺公开市长职位，就像旧的 Foursquare 一样。](https://web.archive.org/web/20221005145753/https://beta.techcrunch.com/2015/05/04/swarm-2-3-feels-like-the-old-foursquare-looks-way-better/)
