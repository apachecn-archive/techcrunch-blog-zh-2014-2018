# 优步赢得 2014 年最佳整体创业奖

几十年后，当我们回顾优步的历史时，2014 年将会被铭记，因为这一年它确立了自己作为一支不可忽视的全球力量的地位。因此，该公司赢得今年的最佳整体创业奖也就不足为奇了。

在过去的四年里，优步迅速扩张，在全球 250 多个城市提供经济、可靠的交通服务。但 2014 年是它真正从一个斗志昂扬的新贵转变为一台运送人们的大型机器的一年。

2014 年最佳整体创业类别的竞争非常激烈。优步与 GoPro 竞争，后者推出了[系列新产品](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/09/29/the-newest-gopro-models-are-more-approachable-than-ever/)，[于 2014 年上市](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/06/27/gopros-14-6-day-2-rise-puts-it-up-50-from-its-ipo-price/)。Stripe [在 2014 年初加入了独角兽俱乐部](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/01/22/with-an-eye-to-international-growth-payments-company-stripe-raises-80m-from-peter-thiel-khosla-and-sequoia-at-a-1-25b-valuation/)，到年底[的估值翻了一番](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/12/02/stripe-3-5-billion/)，因为它让支付再次变得性感起来。Tinder 面临性骚扰诉讼[并且](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/09/08/tinder-and-iac-settle-sexual-harassment-suit-with-early-employee-whitney-wolfe/)[失去了首席执行官](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/11/04/tinders-sean-rad-demoted-from-ceo-will-serve-as-president-of-board/)，但是继续显示出巨大的增长。

与此同时，亚军 Snapchat 继续[筹集新资金](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/12/31/snapchat-485m/)，并用[新功能](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/06/21/snapchat-our-story/)和[货币化战略](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/11/23/snapchat-sponsored-our-story/)证明其 100 亿美元的巨额估值是合理的。

但是优步去年筹集了不止一轮，而是两轮巨大的 [融资](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/12/04/uber-confirms-new-1-2b-funding-round-looks-to-asia-for-expansion/)，总计 26 亿美元，最终使公司估值达到 400 亿美元。(今年早些时候，优步为 T4 追加了 16 亿美元的可转换债券。)

也许比增加其庞大的战争资金更重要的是优步为建立新的关系和改变其形象所做的工作。该公司推出了一项积极的 API 战略，让用户能够从多个第三方应用程序按需欢呼其汽车。该公司还雇佣了奥巴马的前助手大卫·普劳夫来领导其竞选活动并保持其汽车在路上行驶。

与此同时，优步在 2014 年面临许多挑战。除了与当地监管机构和立法者之间常见的地区性冲突之外，它还面临着一系列损害其形象的丑闻。

今年年初，一名休班的优步司机因[一名年轻女孩的死亡](https://web.archive.org/web/20221224151340/https://techcrunch.com/2014/01/02/should-car-services-provide-insurance-whenever-their-driver-app-is-open/)而被捕，这让人们对该公司在这些案件中的保险政策和责任产生了质疑。《商业周刊》的 SVP 关于调查记者和评论家的评论引发了对优步隐私政策的调查。此外，包括德里强奸案在内的几起事件引发了人们对该系统在让司机上路前对其进行筛查的能力的质疑。

尽管如此，优步仍在继续增长，而且没有任何放缓的迹象。正因如此，我们自豪地宣布优步成为 2014 年最佳整体创业公司。
