# 微软将面向企业客户的 OneDrive 存储从每用户 25GB 提升至 1TB

在云存储战争中，容量是一种武器，而且是一种正在迅速贬值的武器。简而言之:提供云存储的公司正在争先恐后地增加存储的千兆字节的价值，因为存储的边际美元价格正在迅速下降到零。

微软今天更加明确地表明了这一点[宣布](https://web.archive.org/web/20230112063531/http://blogs.office.com/2014/04/28/thinking-outside-the-box/)它将把为企业和 Office 365 ProPlus 产品提供的 OneDrive 提升到 1tb，而前者的容量为 25g。

在其博客文章中，微软几乎点名批评 Dropbox 和 box 对其云产品的竞争不足:

> 文件同步和共享解决方案代表了让人们保持同步和快速响应的关键功能。有几种解决方案可供选择，其中许多来自专门专注于这个市场的公司。有些人来自消费者世界，对企业软件市场和交付企业级云服务的要求并不熟悉。其他人专注于企业，但只是作为单点解决方案。很少有人准备好满足企业不断变化的需求，这些企业正在寻找一种整体和全面的方法来满足其员工的全部需求，因为他们过着云优先、移动优先的工作方式。

想想被甩的窗帘。Dropbox 尤其是[目前正在企业内部努力发展。微软不希望这种情况发生。(值得称赞的是，Dropbox 最近的产品实际上设计得非常漂亮。看看他们能在企业中得到什么样的发挥将是有趣的。外面有怀疑者](https://web.archive.org/web/20230112063531/https://techcrunch.com/2014/04/09/dropbox-hits-275m-users-and-launches-business-product-to-all/)。)

这里的金融动态非常有趣。正如我[在讨论盒子 S-1 时写的](https://web.archive.org/web/20230112063531/https://techcrunch.com/2014/03/24/a-look-inside-box-competition-product-plans-and-unit-economics/):

> 大量的竞争和每千兆字节费用的下降放大了这里的(财务)压力。这降低了 Box 对其价值主张的一个关键部分的定价，因为谷歌和微软等更大、更有钱的玩家打得你死我活。谷歌和微软正在表演的舞蹈与 Box 提供的不同，但这是新的现实:没有编辑和协作工具的云存储是没有意义的，没有云存储的编辑和协作工具也是如此。

微软刚刚将最近存储的千兆字节的费用削减到零以上的十分之一。因此，存储本身不再是一个优势。对于像 Dropbox 和 box 这样长期收取存储费用的公司来说，这很难做到。

谷歌和雅虎也帮助降低了存储成本。雅虎在 Flickr 上提供 1tb 的存储空间，谷歌最近在 T2 降低了云存储的价格。

Dropbox 和 box 必须提升价值，值得称赞的是，[正在努力](https://web.archive.org/web/20230112063531/https://www.google.com/search?q=box+boxnotes+wilhelm&oq=box+boxnotes+wilhelm&aqs=chrome..69i57.6216j0j1&sourceid=chrome&ie=UTF-8)到[这样做](https://web.archive.org/web/20230112063531/https://techcrunch.com/2014/03/26/box-unveils-first-standalone-product-and-new-api-pricing-at-inaugural-dev-conference/)。但微软不想放弃这块空间，因为这样做会直接影响其未来的 Office 现金流。这不是它想要剥离的东西。
