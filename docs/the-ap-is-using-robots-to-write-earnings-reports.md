# 美联社使用机器人撰写收益报告 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/07/01/the-ap-is-using-robots-to-write-earnings-reports/>

美联社将从 7 月份开始使用“自动化技术”即机器人来发布新闻，概述数千家公司的季度收益报告。

据美联社官方博客上的一篇文章称，这将使美联社每个季度覆盖大约 4400 家公司的收益报告，而不是通常的 300 家左右。这不会减少任何工作岗位，而是允许美联社记者围绕这些数字的含义写更多的分析。无论如何，收益报告大多是固定的文本，由一些重要的数字组成，并以简短的分析结束。

提供该技术的公司 [Automated Insights](https://web.archive.org/web/20230120191715/http://automatedinsights.com/) ，最近完成了一轮 550 万美元的 B 轮融资，史蒂夫·凯斯、三星和美联社参与了融资。

从一个记者的角度来看，我必须说这是非常令人兴奋的。财报季对所有媒体出版物来说都是一个紧张的时期，因为它涉及到多家公司连续几周每天发布数据的不明确时间表。这些公司的收益报告也没有标准——每家公司都以不同的格式发布季度财务摘要。

事实上，对于其他美国记者来说，美联社一直是一个比收益报告本身更标准化的获取数据的地方。

这意味着通常要花费相当多的时间和精力，以一种全面的、有背景的方式将这些数字汇集在一起。此外，围绕收益报告的分析性报道可以更快地发表，因为作者不再专注于在数据发布后立即对其进行分析和概括。

美联社已经将自动化技术用于体育相关的数据传输，但这是这家媒体机构第一次使用机器人来撰写真实的故事。根据帖子，每个故事将在 150 到 300 字之间。
