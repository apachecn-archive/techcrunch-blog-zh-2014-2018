# YC 支持的 Rigetti Computing 筹集 250 万美元创建商业量子系统 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/08/11/yc-backed-rigetti-computing-raises-2-5m-to-create-commercial-quantum-systems/>

![hitchhiker's guide supercomputer](img/2857e8c09fea3fd3050a6e4a1b830b17.png)

量子计算曾经局限于研究实验室和科幻小说，现在终于到了这样一个地步，研究人员不仅仅是试图找出可能性，而是着眼于如何将这项技术实际应用到商业硬件中。

[Rigetti Computing](https://web.archive.org/web/20230214191851/http://www.rigetti.com/) ，目前 Y Combinator 批次的成员，希望成为领导围绕量子计算创建生态系统的公司之一。该公司由 IBM 和耶鲁大学的一名前高级研究员创建，计划通过迭代、模拟驱动的原型开发过程为该领域带来持续的性能改进。

正如创始人兼首席执行官[查德·里格特](https://web.archive.org/web/20230214191851/http://www.crunchbase.com/person/chad-rigetti) i 解释的那样，该公司有一个工程问题清单，它可以在未来几年内解决，一旦解决，将带来可预测的性能和成本改善。Rigetti 希望处理的许多问题甚至不涉及他们硬件的实际量子方面，这就是为什么这家初创公司能够使用 [Ansys](https://web.archive.org/web/20230214191851/http://www.ansys.com/) 模拟软件来快速测试变化，而不必在每次迭代中都建立新的电路。

早在 4 月份， [Rigetti](https://web.archive.org/web/20230214191851/http://www.crunchbase.com/organization/rigetti-computing) 就能够将其基于模拟的测试的早期结果转化为 250 万美元的种子资金，这些资金来自 AME 云风险投资公司、Morado Ventures、Susa Ventures 和 Tim Draper 等。它希望利用这笔资金将它在伯克利的小办公室变成一个灵活的量子计算实验室，雇佣该领域的一些顶级科学家——里盖蒂说，该公司“非常注意组建团队”，从“与我们的技术相关的领域中前 1%或 2%的博士毕业生中招聘。”虽然 Rigetti 承认从顶级研究实验室和其他公司招聘人员，但他拒绝透露细节，因为这些信息可能会透露哪些研究领域正在影响他们的工作。

接下来几年的原型开发将决定 Rigetti Computing 能否制造出足够可靠、可扩展且价格合理(与竞争对手相比)的处理器，以投入商业部署。理论上，量子计算机执行某些运算的速度比传统处理器快得多；该技术的常见应用包括在分子水平上模拟化学反应，以创造药物和新材料，更先进的人工智能和密码学。

尽管存在声称由量子处理器驱动的机器，但仍然没有任何重大突破，为量子系统构建的软件将传统计算抛在脑后。根据里盖蒂的说法，有两个原因可能会很快改变:“第一个是中小型量子计算机新应用的快速发展；第二个是量子机器的计算能力随系统规模呈指数增长的方式。”

Rigetti Computing 的命运取决于利用这两种趋势。为量子计算机构建软件比简单地改变 API 以将应用从 PC 移植到智能手机要复杂得多，因此初创公司越早将硬件送到潜在客户手中，让他们能够舒适地使用它，就越有可能有人会构建出显示可量化结果的软件。如果他们能够达到这一点，天空基本上是极限:理论上扩大量子计算机会带来性能的指数级改善，这只会使更多的应用成为可能。
