# Spotify 收购 Cord 项目和声波，以扩展消息和社交功能

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/01/20/spotify-buys-cord-project-and-soundwave/>

就在流媒体竞争对手 Deezer [宣布向€注入 1 亿美元现金](https://web.archive.org/web/20230404091932/https://techcrunch.com/2016/01/20/deezer-100/)的同一天， [Spotify](https://web.archive.org/web/20230404091932/http://www.spotify.com/) 确认了两项新的收购，Cord Project 和声波。这两家初创公司分别位于纽约和都柏林，专注于社交和通讯。这些收购表明 Spotify 可能会提升其社交功能，特别是在通讯等领域。

但这不包括语音信息，而语音信息是 Cord Project 开发的技术之一。“我们对 Cord 团队给 Spotify 带来的人才感到兴奋，特别是在设计和音频体验方面，”Spotify 说。“此次收购并非专门针对语音信息，也没有进入该领域的计划。”

交易的条款尚未披露，但可能会比较保守。声波得到了像马克·库班这样的人的支持，筹集了不到 300 万美元；由前谷歌员工共同创立的 Cord 项目筹集了[不到 200 万美元](https://web.archive.org/web/20230404091932/https://www.crunchbase.com/organization/cord-project#/entity)的资金，支持者包括 Lerer Hippeau 和 modification。

Spotify 没有评论它计划如何使用它今天获得的技术和人才，只是在一份简短的声明中指出，它将提高其产品和工程能力。

“Spotify 专注于提供创新和迷人的音乐发现体验，以取悦我们数百万的用户，”Spotify 的产品副总裁 Shiva Rajaraman 在一份声明中说。“收购 Cord Project 和声波让我们有机会将两个才华横溢、志同道合的团队带入 Spotify 大家庭。我们很高兴欢迎他们加入，并期待他们为塑造音乐参与的未来做出贡献。”

需要说明的是，Spotify [已经有了一种让人们在应用程序中相互发送信息的方式](https://web.archive.org/web/20230404091932/http://www.hypebot.com/hypebot/2015/08/have-a-song-suggestion-for-a-playlist-how-to-message-a-spotify-user.html)，但声波和 Cord Project 将给公司带来的是专注于如何开发创新方式将通信插入平台的团队，这些平台可能不仅仅专注于这一目的。

Spotify 对此感兴趣的原因是，它可以帮助这项服务成为其 7500 万用户的平台。正如我们上周所写的，当 Spotify [宣布与 Genius](https://web.archive.org/web/20230404091932/https://techcrunch.com/2016/01/12/spotify-teams-with-genius-to-annotate-music-tracks/) 达成协议时——注释公司将提供歌词、观察和艺术家的想法，这些将在音乐播放时弹出——Spotify 可以使用这样的功能，使其应用程序不只是一个背景工具，而是更多的人们可以在听音乐时积极观看的东西。

消息功能也是如此。如果你独自一人听音乐，有一个与朋友讨论歌曲的信息选项——或者只是碰巧在同一时间听音乐的其他人——可以为体验增加一个不同的维度。这也可以追溯到非常古老的音乐作为社会媒介的概念。

添加更多功能来增加 Spotify 的功能也有助于盈利，要么通过收取使用这些功能的费用，要么将它们包括在优质层中；或者将它们用作潜在广告的载体，或者帮助提高参与度指标以帮助在其他地方做广告。对于专注于音频广告的 Spotify 来说，拥有更多的视觉特性可能有助于推动其他形式的广告。

我们[曾将声波](https://web.archive.org/web/20230404091932/https://techcrunch.com/2014/07/17/soundwave-just-got-watercooler/)描述为为音乐迷开发一个“水冷却器”，通过跟踪你听了什么，你的朋友听了什么，然后使用算法基于此推荐音乐，同时还提供一个信息平台来讨论你的播放列表。

有趣的是，声波背后的诀窍是通过大数据分析，从多个流媒体网站(包括 Spotify 在内总共约 20 个服务)中提取信息。这项服务获得了约 150 万用户，并获得了苹果等公司的多项赞誉。目前，Spotify 还没有说它打算如何处理现有的服务。“我们正在评估下一步措施，但现在没有什么可以宣布的，”Spotify 发言人告诉 TechCrunch。

一年多以前，当我们[尝试 Cord 项目的时候](https://web.archive.org/web/20230404091932/https://techcrunch.com/2014/12/15/cord-project-launches-voice-messaging-app-that-borrows-the-good-parts-of-texting/)，联合创始人兼首席执行官 Thomas Gayno 说，他的目标是将语音通话的优点(情感和细微差别)与文本的优点(速度和异步通信)结合起来。

盖诺表示，Cord 项目团队的七名成员中有五名将加入 Spotify——盖诺本人、他的联合创始人杰夫·巴克斯特和三名高级工程师。(除了在 Cord 的工作时间，Gayno 和 Baxter 还在谷歌的创意实验室一起工作。)根据收购公告，该团队“将(在 Spotify)建立一个产品团队，专注于创造引人注目的内容体验。”

Cord 应用程序以及这家初创公司推出的其他应用程序将被关闭，尽管 Gayno 承诺用户将能够在未来几周内下载他们“最情绪化的信息”。
