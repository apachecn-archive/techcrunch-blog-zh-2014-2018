# 为什么金·卡戴珊会出现在科技会议上？！

因为她是 2014 年最受欢迎的应用之一的联合创始人。因为她今年准备从一个移动应用程序中获得 2 亿美元现金。因为仅从应用程序每天就能获得 70 万美元的收入[，她仍然比大多数初创公司的创始人更成功，至少从收入的角度来看是这样。](https://web.archive.org/web/20221225211713/http://money.cnn.com/2014/07/30/technology/kardashian-app-game/)

昨天，金·卡戴珊[在 Re/Code 大会的舞台上](https://web.archive.org/web/20221225211713/https://techcrunch.com/2014/10/27/kim-kardashian-west-on-oversharing-adopting-new-platforms-and-blackberry/)接受了采访，除了承认她对黑莓的热爱，她没有透露太多关于她的游戏“金·卡戴珊好莱坞”，这就是为什么我们想听听她的意见。我们报道了她的演讲，因为我们过去已经报道过比赛了。地狱之门在我们的评论区打开了。

像大多数评论系统一样，脸书的评论很糟糕:要么你在一篇报道上得不到任何评论，因为没有一个初创公司的员工愿意在关于竞争对手的文章上署名，等等。或者，如果一个帖子真的爆炸了，你会从杰克·卡莫迪那里得到这首诗，“加州惩教改造部的惩教官”和“顶级评论者”…

![Screen Shot 2014-10-28 at 10.13.18 AM](img/927ffcaec581c7e29d190e8982940535.png)

“当她所有的魅力都消失了”！

卡戴珊不在 Re/Code 是因为她的女性魅力——不管那是什么。她在那里是因为卡拉·斯威舍认为她作为一个移动增长故事足够鼓舞人心，可以搬上舞台。没有人会邀请玛丽亚·凯莉或者那些应用程序日渐式微的名人参加他们的技术会议。在 Path 和 Foursquare 努力在美国获得牵引力的世界里，金·卡戴珊和 GLU mobile 真正实现了吸引眼球、用户和金钱的壮举。老实说，我们希望她参加我们在*的*会议(这个邀请仍然有效)。

这个世界是残酷无情的。如果你能用你所拥有的一切成功，那么你就有更大的力量。

不希望金·卡戴珊出现在科技会议舞台上的人也是不希望我出现在科技会议舞台上的人。(更可怕的是，我在加州监狱系统工作。)所以，金，我们要么 a)回到学校拿我们的计算机学位，要么 b)现在我们已经在这里了，希望，只是也许，我们对技术的热爱和深深的好奇心有一天将足以证明我们的座位。

*地狱之门开启*
