# TechCrunch 创客:Lemp 酿酒厂是圣路易斯的后起之秀

像莱姆普酿酒厂这样的地方很少见。这座工厂建于 1840 年，位于密苏里州的圣路易斯，曾经在彻罗基街上喷出芳香的啤酒花蒸汽，而工人们将一桶桶啤酒滚进建筑群下的天然洞穴。现在，除了一小群专注的制造商正在慢慢地将啤酒厂变成我见过的最独特的工作空间之一之外，它空无一人。

我们和圣路易斯制造公司的人一起参观了 T2 工厂，他们带我们参观了可以容纳数百人的空地板和在零散的口袋中涌现的小作坊。我们确信没有鬼，但蝙蝠有时会穿过敞开的房间。

Lemp 啤酒厂是我们为 TechCrunch 制作者参观过的最原始、最有趣的地方之一，在重金属凝视下爬上爬下，想象一旦一些企业家将一个死气沉沉的啤酒厂变成一个活跃的创作中心，这个地方会是什么样子，这是一件非常有趣的事情。
