# 该网站将为被指控性骚扰的男性道歉

你是一个对一群人进行性骚扰，现在需要一个道歉来制服一群愤怒的暴徒的名人吗？看看[名人变态道歉生成器](https://web.archive.org/web/20221007004512/https://apologygenerator.com/)就知道了。

这个桌面应用程序打包了最近几个不道歉的所有常见元素，这些元素是在其他变态伙伴的黑暗过去曝光时出现的。在一种可能被描述为更诚实的方法中，你会躲避、指责他人或暗示这是多么正常，就像在“更衣室谈话”中一样，在不承认任何事情的情况下说你感觉不好，并用一个很好的开场白来帮助你逃避指责，所有这些都在一个易读的段落中。

以这个随机生成的道歉为例:

> 作为一个生长在不同时代的人，我现在感到非常内疚，因为我的所作所为被公之于众。我当时喝醉了，当然现在我意识到我的行为是错误的。总之，我不会改变我的任何行动或行为。

另一个很好的例子:

> 作为女儿的父亲，骚扰是完全不可接受的——尤其是当人们发现这件事的时候。这只是男生更衣室的谈话，当然现在我意识到我的行为是错误的。总之，我会等 2-3 年再出现在电影和电视中，只是希望你们都忘记这一点。

我们似乎每天都听到身居要职的人被抓的消息。就在上周，艾尔·弗兰肯、查理·罗斯和格伦·思拉什加入了越来越多被指控犯有可怕性骚扰行为的人的行列。

虽然有些男人似乎对他们的行为真诚地感到悲伤，但许多人一次又一次地求助于这些空洞的所谓道歉策略。现在，你甚至不必为自己考虑如何给出一个非道歉。你会有一个现成的——当你的时间到来时——假装你对那些尊敬你、信任你、依赖你的关系、事业和生计的人所做的没有错。
