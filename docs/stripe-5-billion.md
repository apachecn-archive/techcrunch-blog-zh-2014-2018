# Stripe 以 50 亿美元的估值获得新的融资，并与 Visa 合作

支付处理公司 [Stripe](https://web.archive.org/web/20230130021212/https://stripe.com/) 在最近一轮融资后，市值达到 50 亿美元。除了增加更多的资金，这家成立五年的公司还与 Visa 签署了一项协议，双方将在一系列项目上合作，包括数字交易和安全。

参与此轮投资的投资者包括 KPCB、Visa、美国运通和红杉。Stripe 没有证实它到底筹集了多少资金——除了它不到 1 亿美元之外——但它的新估值比去年 12 月[结束最近一轮 7000 万美元融资](https://web.archive.org/web/20230130021212/https://techcrunch.com/2014/12/02/stripe-3-5-billion/)时的 35 亿美元估值有了很大的提升。(顺便提一下，这个估值是之前的两倍。)

该公司一段时间以来一直专注于扩大其国际足迹，这是它与 Visa 合作的另一个原因。首席执行官兼联合创始人帕特里克·科利森[告诉纽约时报](https://web.archive.org/web/20230130021212/http://www.nytimes.com/2015/07/28/technology/stripe-digital-payments-start-up-raises-new-funding-and-partners-with-visa.html?_r=0)，Stripe 将依靠支付巨头的国际影响力和专业知识来扩大其目前在 20 个国家的覆盖范围。
