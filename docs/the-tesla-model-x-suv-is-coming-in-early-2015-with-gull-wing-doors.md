# 特斯拉 X 型 SUV 将于 2015 年初推出，采用鸥翼车门

特斯拉 Model X [的预订者今天收到通知](https://web.archive.org/web/20230324023401/http://www.teslamotors.com/forum/forums/model-x-update-email-tesla),电动 SUV 将于明年初进入生产线。最重要的是，特斯拉在这封电子邮件中确认，鸥翼车门将成为量产车型，所有车型都将采用全轮驱动，并将有一个第三排座位的选项。这些都是重大的更新，但它们不应该让人感到意外。因为你知道泰斯拉。

特斯拉是承诺不足和兑现过度的大师。一般来说，在汽车领域，汽车公司会在概念车中加入一些不切实际的功能，比如鸥翼门，然后将它们从生产模型中移除。但不是特斯拉。在 Model S 上，该公司展示了后向座椅，很少有汽车专家预计这种选择会出现在量产车型上。然而他们是。现在，在 Model X 上，鸥翼式车门是概念车的标志性特征，令人惊讶的是，它也将成为量产车的标志性特征。

没有宣布的是 Model X 将于何时发货，只说面向预订者的车型将于 2015 年初生产。从历史上看，特斯拉一直采取保守的制造方法，因为该公司在 Roadster 和 Model S 首次上线时要精简得多。尽管如此，让制造工厂满负荷运转需要时间，因此 Model X 可能要到 2015 年晚些时候才能上市。但是鸥翼门！
