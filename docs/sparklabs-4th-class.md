# 韩国 SparkLabs 命名其第四家创业公司 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/09/17/sparklabs-4th-class/>

韩国的 [SparkLabs](https://web.archive.org/web/20221007040442/http://www.sparklabs.co.kr/kr/html/home.html) 加速器已经命名了它的第四个入口——包括它的第一个物联网创业公司(一家制造联网花盆的公司)；它的第一家来自中国的创业公司，一家移动医疗创业公司；以及营销自动化领域的一家美国企业。

这延续了该项目近年来与 2012 年首次招生相比更具国际性的特点，当时的招生对象是希望打入海外市场的韩国公司。

去年 SparkLabs 的一批创业公司中有更多非韩国创业公司(总共九家中有四家)，而这次只有两家。然而，联合创始人 Bernard Moon 告诉 TechCrunch，总体而言，今年孵化器有更多的国际申请——注意到来自 20 多个国家的初创公司申请参加。他补充说，SparkLabs 总共收到了大约 220 份申请。

2014 年共有 10 家创业公司。每个团队都将通过 SparkLabs 为期三个月的指导计划，每个团队将获得 25，000 美元的资金，以换取高达 6%的股权。

就业务阶段而言，与之前的入选者一样，2014 年的入选者既有仅有 alpha 产品的初创企业，也有已经推出产品并筹集了数百万美元资金的更成熟的企业。

SparkLabs 在 4 月份举行了第三次[演示日](https://web.archive.org/web/20221007040442/https://beta.techcrunch.com/2014/04/08/sparklabs-korea-demoday3/)，其中包括制造可穿戴技术、医疗设备和手机游戏的初创公司。第四批涵盖物联网、医疗保健和社会企业领域的企业。

今年的这一批中出现了更多的硬件线程，三家初创公司——open 21、N.Thing 和 mobi doo——都在构建某种形式的物理套件，而不是纯粹的软件。

以下是 spark lab 2014 年十大创业公司的完整名单:

**HUD 技术** : HUD 开发了一种新技术，只需使用平面图就能呈现建筑物的 3D 图像。他们利用算法建模而不是其他方法，如使用全景相机，来生成完整建筑的图像。

创始人是一位连续创业者，他为智能手机、电视和其他电子产品创造了一种潜在的革命性触摸传感器。不需要 Open21 电场围绕设备或电器的物理按钮。

《树木星球》:树木星球是一个虚拟的植树游戏，在这个世界上种植了真实的树木。迄今为止，已在 9 个国家的 46 个森林中种植了超过 472，000 棵树。

灯塔家庭医生(Beacon Family Doctor):一家位于中国成都的初创公司，计划改善该地区的医疗服务。他们有一个移动应用程序，将城市家庭与医生联系起来，以获得一次性或长期的医疗服务。

**Vengine** : Vengine 正在建立一个网络平台，为初创公司提供招聘解决方案，旨在解决缺乏信任、信息不足和简单的问题，同时建立最佳实践标准并鼓励公司文化。

**Stayes** :该公司将未出租的公寓与在韩长期居留的外国人联系起来。与 Airbnb 不同， [Stayes](https://web.archive.org/web/20221007040442/http://www.stayes.com/) 专注于商务旅客。

**n . thing**:[n . thing](https://web.archive.org/web/20221007040442/http://www.nthing.net/)寻求将人们与生活中的绿色事物联系起来。他们的主要产品是 Planty，一种联网花盆。他们有一个免费的应用程序，用智能园艺日记记录你的“绿色生活”。

**Onnuri DMC** :移动广告新数据管理平台。他们的产品 CrossTarget 试图解决跨平台移动重定向的难题。

**Mobidoo** : Mobidoo 向消费者和零售店提供“简单方便”的移动忠诚度印章服务——旨在用智能手机上的电子印章取代纸质忠诚度卡和墨水印章。

**BuyFi** : [BuyFi](https://web.archive.org/web/20221007040442/http://www.buyfi.com/) 分析客户的信用卡购买行为，自动触发一系列‘一次设置，一劳永逸’的营销计划，将一次性访客转化为多次访问的忠诚客户。目前，该公司通过其支付处理器合作伙伴关系为 10，000 多家商户提供服务。
