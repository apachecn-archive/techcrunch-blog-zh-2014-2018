克雷格·奈特菲尔德撰稿人

有句谚语说，作为初创公司的首席执行官，你的第一份工作是确保你的公司永远不会缺钱。在为一家成长中的公司融资时，风险债务可以成为风险资本的一个很好的补充。已经有很多文章帮助创业者思考风险投资，但是风险债务有点像黑箱。

这就是为什么我们与我们在[**Columbia Lake Partners**](https://web.archive.org/web/20220925031707/http://www.clpgrowth.com/)的朋友合作，这是一家领先的欧洲风险债务基金，共同编写了一份[白皮书](https://web.archive.org/web/20220925031707/https://www.bvp.com/BessemerGuidetoVentureDebt)，帮助初创公司以一种深思熟虑的方式处理风险债务。以下是你在考虑风险债务时应该问的十个问题:

**1)** **为什么要创业负债？**

风险债务由银行或专门的风险债务基金提供，作为股权融资的补充。它通常是一笔一次性的钱，可以在一段时间内偿还，可以作为一种比股权稀释更少的有吸引力的企业融资方式。它有许多优点——它不需要为企业设定估值，风险贷款人不占董事会席位，并且几乎没有治理要求。但像任何贷款一样，风险债务需要支付利息，并且需要在一段时间内偿还。

**2)** **什么时候应该筹集风险债务？**

风险债务没有“一刀切”的方法，但有四种常见的使用案例:

a)扩展业务的现金跑道，以达到公司的下一个里程碑(例如，你已经筹集了 1000 万美元的股权，但认为你可能需要 1500 万美元才能达到下一个重要里程碑)

b)防止需要一轮过桥或一轮下跌来度过艰难时期

c)为大型资本支出、收购提供资金，或为盈利提供桥梁

d)在执行公司计划所需时间比预期长的情况下，充当保险

大多数公司在股权融资后立即筹集风险债务，此时最容易获得，你所有的尽职调查材料都是新的，你的业务也有发展势头。

**3)** **什么时候应该避免风险债务？**

如果出现以下任何一种情况，使用风险债务是一个坏主意:

a)你认为你无法通过再融资或增加股本来偿还债务

b)贷款的条款或契约过于苛刻

c)你的风险投资者不支持

**4)** **筹集风险债务时，我应该考虑哪些条款？**

以下是需要考虑的关键事项:

a)尺寸

b)期限(什么时候需要偿还？)

c)价格(费用和利率是多少？)

d)契约(财务和非财务要求是什么？)

e)分期偿还(贷款何时开始偿还？)

银行的利率是最低的，所以如果你想要额外的资金，先和银行谈谈，然后再去找风险债务基金。

使用风险债务时，你可以做的最有影响力的事情之一是尽量推迟你提取贷款的时间。这让你可以推迟开始偿还贷款的时间。这不会马上变得明显，但是这个小小的改变会对你的现金跑道和债务成本产生巨大的影响。我们已经创建了一个方便的[风险债务模型](https://web.archive.org/web/20220925031707/https://bvp.box.com/BVPVentureDebtModel)来帮助你评估延迟提取的价值和风险债务的有效成本。

**5)** **这些术语的“市场”是什么？**

| **期限** | **市场利率** |
| 利率 | 银行:prime(目前 3.5%)+0-4%；基金:优质+ 4-9% |
| 持续时间 | 3-4 年 |
| [会计]分期偿还 | 提取贷款后 12 个月内 |
| 成交费 | 0%-1% |
| 保证 | <1% dilution; Funds: ~1% dilution |

**银行:**

我应该如何回应圣约？

非金融契约很常见。谨慎对待财务契约，它们是必须通过的财务测试，否则你的公司将会违约。

**7)** **如果我不能还款会怎么样？**

当偿还风险贷款的时候，你有三个选择:

a) *偿还*:用你资产负债表上的现金或者投资者的股权来偿还。

b) *再融资*:寻找另一个愿意为贷款再融资的贷款人。

c) *重组*:与贷款人协商还款计划。

如果事情进展顺利，还款或再融资很容易。但是如果你的公司陷入困境，你需要和你的贷款人一起重组债务，避免违约。一旦你违约，贷方有能力取消你公司的赎回权。

我如何决定与哪家风险贷款机构合作？

选择贷款人时，最重要的是研究他们过去的表现。如果事情变糟，你希望有一家贷款机构与你一起重组贷款，而不是作为重组的一部分，强迫出售或苛刻的条款。你应该打电话给贷款人，看看他们在与其他公司的艰难时期表现如何。

**9)** **风险贷款人典型的尽职过程是怎样的？**

风险贷款尽职调查类似于风险资本尽职调查流程，但强度较低。准备好提供你在风险投资融资中提供的大部分信息。

**10)** **运行风险债务筹资流程的最佳方式是什么？**

就像风险资本世界一样，你需要与许多不同的银行和风险债务基金进行沟通。我们建议聘请一位对什么是“市场”有感觉的风险律师，他将能够建议你获得尽可能好的条款。就像股权融资一样，确保你有几匹马参加，这样你就可以谈判出最好的条件。

风险债务不会适合所有公司，但如果结构合理，它可能是一种稀释性较小的方式，可以为你通往下一个里程碑的旅程提供资金。
