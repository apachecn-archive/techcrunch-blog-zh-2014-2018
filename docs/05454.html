DJI 与 Canonical 合作推出无人机嵌入式计算机

无人机制造商 DJI 和 Canonical ，Ubuntu Linux 发行版背后的公司实体，今天宣布推出 Manifold，这是一款小型嵌入式计算机，为构建无人机应用程序进行了优化。

歧管只适合 DJI 的 Matrice 100 平台，所以不要指望在你的幻影无人机上安装这个。3300 美元的 Matrice 100 本质上是 DJI 的飞行开发平台，能够携带硬件，如歧管和可定制的传感器。

在内部，该歧管采用四核 ARM Cortex A-15 处理器和基于 NVIDIA Kepler 的 GPU。GPU 显然不是为了渲染图形，而是为了利用其图像处理和并行计算能力。

DJI 认为，这将实现“新的人工智能应用，如计算机视觉和深度学习”该计算机还具有标准的 USB 和以太网端口，用于连接红外摄像机、大气研究设备、测量设备和其他传感器。还有一个 HDMI 端口用于连接显示器。

“凭借流形，我们正在进入一个更智能、更快速、更强大的空中平台的新时代。Manifold 为空中和地面技术开辟了智能合作解决复杂问题的途径，”DJI 战略伙伴关系总监 Michael Perry 在今天的一份声明中表示。“我们很高兴看到我们的开发人员社区将为这个新平台开发出什么样的应用。”

鉴于与 Canonical 的合作关系，计算机将运行 Ubuntu 也就不足为奇了(在这种情况下， 14.04 长期支持版本)。
