# Eko 核心是数百年来听诊器的数字升级

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/09/02/the-eko-core-is-a-digital-upgrade-for-the-centuries-old-stethoscope/>

再过几个月，听诊器将迎来它的 200 岁生日。这是 1816 年的一项医学突破，今天它仍然是几乎每个医生就诊的一部分，也是医学本身的象征。

然而听诊器在过去的 200 年里并没有太大的变化。与 17 世纪没有什么不同，听心跳一直是一个手工过程，完全依靠医生的耳朵来检测异常情况。

直到今天。这一切都要归功于总部位于伯克利的创业公司 [Eko Devices](https://web.archive.org/web/20230227002611/https://ekodevices.com/) ，该公司刚刚成为获得 FDA 二级医疗设备许可的最年轻团队。联合创始人 Connor Landgraf、Jason Bellet 和 Tyler Crouch 都是加州大学伯克利分校的应届毕业生，他们为这个有数百年历史的工具添加了数字元素。

他们的解决方案被称为 Eko Core，是一个连接到典型听诊器的适配器，并将心跳数据传输到云，为医生提供了一个全新的信息分析层。

“听到一分钟的心脏杂音是一个令人难以置信的挑战，尤其是对心率高的患者来说，”兰德格拉夫说。“心脏病专家说，这几乎就像一只音乐耳朵，你必须通过五年或十年的练习才能学会。”

借助 Eko Core，医生可以在移动设备上看到波形心跳，并听到放大的声音。视觉和听觉数据都可以被记录下来，并在医生和医院之间轻松共享。

对于医生来说，这消除了检测杂音、瓣膜问题和动脉堵塞的大量猜测。

对于 70%的怀疑有心脏杂音的儿科患者来说，他们不必转诊到心脏病专家那里，Eko 将能够在可避免的超声心动图检查中节省数千美元。

“医生对自己在很多情况下使用听诊器的能力没有信心，所以在没有必要时，他们经常让人们去找心脏病专家，”兰德格拉夫说。

Eko 正在斯坦福医院进行试点，所有住院医生都将使用 Eko 核心设备作为训练工具。今天，该公司向公众发布这款设备，售价 199 美元，或 299 美元，包括听诊器。

在接下来的几个月里，该团队将继续开发一种算法，分析所有 Eko 设备收集的数据，以便实时匹配心跳和条件。兰德格拉夫说，医生喜欢称之为“心跳的沙赞”。

兰德格拉夫说:“通过非侵入性工具将病人与医生联系起来，以了解人们内心的想法，这将是非常强大的。现在，你可以给病人插入导管，以了解心脏内部的压力，但这是非常侵入性和低效的。"

条件检测功能定于明年在 Q1 推出，兰德格拉夫暗示，Eko 最终将推出更多产品，以无创方式让医生更好地了解心脏。

自 Eko 于 2013 年成立以来，该公司已经从斯坦福的 StartX 基金、FOUNDER.org 创始人迈克尔·鲍姆和 Shazam 的联合创始人等人那里筹集了 280 万美元。
