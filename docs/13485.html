你可以给 SFMOMA 发短信，它会回复你艺术点播

原文：https://web.archive.org/web/https://techcrunch.com/2017/07/10/you-can-text-sfmoma-and-it-will-respond-with-art-on-demand/

你可以给 SFMOMA 发短信，它会回复你点播的艺术作品

给艺术爱好者一个很酷的小技巧。你可以给旧金山现代艺术博物馆发短信，号码是 572-51，它会回复你一件收藏品。

发送给我 SFMOMA 可以让你发送一个关键词，并得到一件与该词匹配的艺术品。你可以发送颜色、主题、心情甚至表情符号。

这项服务由 SFMOMA 的收藏 API 提供支持，该 API 拥有收藏中所有 35，000 多件艺术品的数据。当它找到与你的请求相匹配的东西时，它会回复一张该作品的图片、艺术家、标题和日期。如果有多条与您发送的关键字匹配，每次您发送该关键字时，号码都会随机发送一条。

SFMOMA 于去年推出，最初是一个标准的 10 位号码，但最近改用了一个特殊的 5 位号码，就像其他高流量短信服务一样，用于处理高需求。

号码是 572-51，你现在就可以发短信来查看一些艺术品。只需说“发送给我”，然后输入关键字，就可以将一件艺术品发送回来。这项服务是完全免费的，但是如果出于某种原因你不能无限制地发短信，你当然还是要支付自己的短信费。

博物馆注意到，他们今天的收藏如此之多，以至于在任何给定的时间，只有大约 5%的收藏可以在画廊中展出。另一方面，发送给我 SFMOMA 可以访问整个收藏——这意味着使用这项服务，你实际上可以看到比实际进入博物馆更多的艺术作品。

仅供参考，这项服务可能有点慢，因为这个数字今天受到了所有的关注，但对我们大多数 TechCrunch 的人来说，它一直非常可靠。
