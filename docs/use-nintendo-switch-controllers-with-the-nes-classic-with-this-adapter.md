# 使用任天堂 Switch 控制器与 NES 经典与此适配器

任天堂 Switch 的控制器虽然有缺点，但也很灵活——他们使用的最新硬件是 NES 经典，配有 8bitdo 的 17 美元复古接收器，现在这个小加密狗有了新的固件更新。这款无线配件已经可以让你将一系列蓝牙控制器连接到你的 NES Classic 上，包括 8bitdo 自己的 NES30 控制器复制品，以及 PlayStation 4、Wii U Pro 等等。

新固件支持连接两个单独的 Joy-Con 的一半，以及 Switch Pro 控制器。它还提供了版本的复古接收器，插入到原来的 NES 和 SNES 游戏机，让你使用你的开关输入设备与您的老式游戏机，太。

这对于整合控制器来说是个好消息——如果你碰巧拥有多个游戏系统，这些东西就会疯狂地复制——尤其是在 Joy-Cons 和 Pro 控制器也可以与 T2 的 Mac、PC 和 Android 设备配合使用，几乎不需要或根本不需要特殊软件的消息传出之后，这一点就更棒了。

当然，任天堂的 Switch 也有不少控制器的问题:很多用户都遇到了连接问题，特别是他们游戏机附带的左 Joy-Con。他们建议把它放在几乎所有无线设备都无法触及的范围之外，这在很多情况下几乎是不可能的。而且有消息称，只有通过硬件修复，它才有可能真正被 T4 解决。

不过，如果能在紧要关头为这些控制器提供额外的便携式 NES 经典配件，还是很有用的。
