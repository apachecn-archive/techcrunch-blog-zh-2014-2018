# 据报道，三名优步安全经理今天辞职

路透社今天早些时候报道，三名高级安全经理今天从优步辞职。辞职的三人中的一个，Pooja Ashok，是前首席安全官 Joe Sullivan 的办公室主任。沙利文上周因试图隐瞒影响约 5700 万名乘客和 60 万名司机的安全漏洞而被解雇。

另外两个辞职的是高级安全工程师普里特维·拉伊和杰夫·琼斯。阿肖克和琼斯都计划留在优步直到一月份，以帮助过渡。

优步的安全措施最近受到了严密的审查。上个月，新任首席执行官 Dara Khosrowshahi 披露公司[未能报告 2016 年的大规模数据泄露。](https://web.archive.org/web/20221025222300/https://beta.techcrunch.com/2017/11/21/uber-data-breach-from-2016-affected-57-million-riders-and-drivers/)本周早些时候，在法庭上，优步的首席法律官提出了该公司在 Khosrowshahi 之前收集竞争对手信息的安全措施。在一场关于其与 Alphabet 旗下 Waymo 在自动驾驶汽车技术上的法律战的听证会上，对优步策略的洞察浮出水面。

优步发言人向 TechCrunch 证实了这些离职，称他们与公司正在进行的调查无关。
