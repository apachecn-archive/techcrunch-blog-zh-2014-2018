以下是特斯拉售价3.5万美元的Model 3 TechCrunch的14张实际照片

原文：https://web.archive.org/web/https://techcrunch.com/2016/04/01/here-are-14-actual-pictures-of-teslas-35000-model-3/?ncid=rss

奇怪的是，在今晚的Model 3发布会上，特斯拉有一个奇怪的规定:不允许使用“大”相机。拍照手机还可以，但如果它有一个大变焦镜头，它就不应该进来。

为什么？我们不能百分百确定。我问了周围的人，我听到的最接近的答案是，汽车还没有完全完成，所以更好的细节可能会发生变化。黑暗、拥挤的房间和微型手机相机传感器构成了…一个相当具有挑战性的摄影环境

因此，除了特斯拉自己发布的照片(大部分似乎是渲染图)，Model 3的好照片可能很难获得。那么，好吧，这里是:14张真实的汽车照片，是我和几个同事在活动中拍摄的。

这通常是我们的幻灯片/图库的事情之一……但是，嗯，图库加载的图片很小，这有点偏离主题。尽情享受吧！

(只是想多了解一下车？这是我们关于实际公告本身的帖子

红色！

虽然顶部的银色汽车是整个展示的中心舞台，但他们只有一辆红色汽车在右边。

这款车显然借鉴了S型车的许多设计语言，但它不仅仅是一个小型克隆。特别是前端，完全不同。从这个角度来看，前面实际上让我稍微想起了接穗FRS/斯巴鲁BRZ(这真的是一件好事，那辆车的前面非常漂亮)

车轮！

稍后你会在这组照片中看到一些不同的细节，所以这可能是特斯拉尚未完全确定的细节之一——或者，很有可能，会有多种选择。

前端

他们会保留那个几乎没有烧烤的鼻子吗？我当然希望如此；本人看着惊艳。几乎有点超现实；就像有人在用粘土制作汽车模型时忘记了雕刻烤架，后退一步，说“哦，嘿，那看起来真的很不错。”

注意门把手

像之前的许多特斯拉一样，Model 3的门把手在车辆行驶时会缩进车身。不过，这款原型车的把手明显比S型或X型的把手更具曲线感。

【当前预订】

就在发布会结束后，前台屏幕变成了一个实时计数器，显示已经预订的数量。当埃隆登上舞台时，它已经达到了11万辆左右——这还是在有人看到这辆车之前。大约40分钟后，人数达到了15万(这时他们关闭了计数器。)

作为参考，这大约是特斯拉2015年全年交付汽车数量(50，580辆)的3倍。

屋顶

很难准确地拍到屋顶，但它很美。它本质上是一个大玻璃窗格，只被两个支架分开:一个在前排座椅遮阳板的位置，另一个在司机/乘客头枕的正上方。否则，它更多的是天窗而不是屋顶。就像有人拿走了Model X已经该死的华丽全景车顶，一路拖到后面。我想这在白天，或者在繁星点点的夜晚是相当疯狂的。

来自前方

看到司机和乘客之间的东西了吗？这是一个很大的触摸屏，类似于Model S中的触摸屏……但它现在是15英寸的横向(宽)屏幕，而不是17英寸的纵向屏幕。它突出了一点从仪表板比发现在S；目前还不清楚最终设计中是否会有所改变。
