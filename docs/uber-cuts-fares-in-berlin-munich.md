# 由于无视德国范围内的禁令，优步在柏林和慕尼黑吹嘘降价 30%

德国法院发布针对按需打车服务优步的全国性禁令几天后，该公司宣布削减两个主要城市的车费:柏林和慕尼黑。

优步与德国司机的收入分成仍为 20%归其自己，80%归其“合作伙伴”，因为它指的是那些通过将自己的汽车变成优步用户可以通过应用程序叫到的私人出租车来为其服务提供动力的人。

尽管法兰克福法院上周晚些时候颁布了临时禁令，但优步已经发誓要继续在德国运营。本周早些时候，优步告诉 TechCrunch，它将对禁令提起上诉。一旦任何上诉听证会开始，禁令将被取消，直到一个坚定的法律结果。

显然，该公司还决定，当其服务受到攻击时，最好的防线是通过降价来刺激额外的乘客需求。优步表示，柏林和慕尼黑的票价“平均降低了 30%”——并声称，对于一些用户来说，现在去优步可能比自己开车更便宜。

它将用户引向一个[在线工具](https://web.archive.org/web/20230320174926/http://form.jotformeu.com/form/42392929256363)，在那里他们可以计算自己开车的相对成本，以及同样的预算可以购买多少次优步 Pop(其最便宜的服务级别)旅行。

如果优步的降价在德国引发司机的愤怒，可能会适得其反。该公司在德国市场已经受到出租车行业的持续攻击，并被法院裁定为非法。优步降价在其运营的多个市场引发了司机抗议。有时，该公司甚至会补贴降价对司机的影响，例如，继续向他们支付原票价的一定比例(即，而不是折扣金额)。

优步似乎没有在柏林和慕尼黑提供这种补贴。公司发言人向 TechCrunch 证实，市场上的收入分成仍然是 20:80。

对德国的优步司机来说，另一个潜在的棘手问题是，那些继续提供这项服务的人可能会面临 25 万€的罚款——因为目前针对这项服务的全国性禁令已经生效。

优步的一位发言人声称情况并非如此，称禁令针对的是公司优步，而不是其非员工“合作伙伴”，并补充说公司“支持其合作伙伴”。
