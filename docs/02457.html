你可以在棍子上得到几乎任何东西。实际上，我不骗你，你可以在一根棍子上得到油炸黄油。很快，多亏了英特尔，你甚至可以在棒上获得完整的 Windows 8.1 PC。

基于 Windows 8.1 的计算棒包含一个四核 Atom 处理器，32g 存储空间，2g 内存，以及 wi-fi 和蓝牙支持。今年年初，它将在 T4 的零售价为 149 美元。一个独立的 Linux 版本将包含一半的内存，三分之一的存储空间，售价降低到 89 美元。

这是它最棒的地方:

英特尔正在努力缩小其计算产品的尺寸。其新的 Quark SE SOC(片上系统)可能会帮助可穿戴革命不再是一个梦想，而是一个实用的现实。英特尔在智能手机和平板电脑领域姗姗来迟——其每个季度的收益都说明了这一点——但当涉及到更小的机器时，该公司似乎决心不落后。
