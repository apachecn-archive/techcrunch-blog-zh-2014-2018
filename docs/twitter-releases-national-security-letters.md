# Twitter 发布国家安全信件

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/01/27/twitter-releases-national-security-letters/>

# 推特发布国家安全信件

在过去的八个月里，科技公司慢慢透露，他们收到了联邦调查局的国家安全信函，迫使这些公司秘密向政府披露用户数据。今天，Twitter 宣布收到两封国家安全信函，一封在 2015 年[发出，一封在 2016 年](https://web.archive.org/web/20230406043025/https://g.twimg.com/blog/blog/attachments/Redacted-NSL-15-418313-Twitter.pdf)[发出，加入了雅虎](https://web.archive.org/web/20230406043025/https://g.twimg.com/blog/blog/attachments/Redacted-NSL-16-422732-Twitter.pdf)、 [Cloudflare](https://web.archive.org/web/20230406043025/https://techcrunch.com/2017/01/11/cloudflare-explains-how-fbi-gag-order-impacted-business/) 和[谷歌](https://web.archive.org/web/20230406043025/https://techcrunch.com/2016/12/13/google-national-security-letters/)的行列。

NSLs 带有禁言令，禁止 Twitter 向公众或目标用户透露政府的要求。联邦调查局最近取消了这些禁令，首次允许 Twitter 承认 NSLs。

虽然 Twitter 已经披露了这两封信，并通知了目标用户，但很可能该公司还收到了其他 nsl，但仍被禁止讨论。TechCrunch 最近[透露](https://web.archive.org/web/20230406043025/https://techcrunch.com/2017/01/13/cloudflare-and-credo-are-still-gagged-from-talking-about-national-security-letters/)虽然 Cloudflare 和 CREDO Mobile 已经发布了几个 nsl，但他们收到了其他仍然保密的 nsl。Twitter 目前正在起诉司法部，试图更公开地谈论对用户数据的秘密请求。

在新公布的 NSLs 中，FBI 要求 Twitter 交出两名用户的“姓名、地址、服务年限和电子通信交易记录”。Twitter 副总法律顾问伊丽莎白·班克(Elizabeth Banker)表示，该公司提供了“非常有限的一组数据”来回应这些请求，但没有明确说明 Twitter 到底提供了什么样的数据。

“Twitter 仍然不满意对我们更自由地谈论我们可能收到的国家安全请求的权利的限制，”银行家在博客中写道。“我们希望有一个有意义的机会来挑战政府的限制，当‘分类’阻止对公众重要的问题发表言论时。”
