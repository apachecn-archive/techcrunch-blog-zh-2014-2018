# 待办事项应用 Wunderlist 首次推出 API，早期集成 Slack、Sunrise、HipChat 等技术

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/05/13/to-do-app-wunderlist-debuts-an-api-early-integrations-of-slack-sunrise-hipchat-and-more/>

总部位于柏林的待办事项应用 Wunderlist 目前拥有近 1300 万个免费和付费账户。今天，它增加了一个新功能，希望能让这款应用程序更有用、更常用:它终于[打开了一个 API](https://web.archive.org/web/20221209205221/https://www.wunderlist.com/blog/building-with-wunderlist-introducing-our-public-api/) ，这样人们就可以在其他应用程序中创建和修改神童列表任务，并在他们在这些应用程序中时获得神童列表任务的提醒。

该公司告诉我，第一波 API 合作伙伴是 Sunrise、Slack、OneNote、HipChat、Zapier 和 Scanbot，他们自 1 月份以来一直在玩 API 的封闭测试版。

Wunderlist 宣布 API 将于今年 3 月在[发布，但事实上这是一个漫长的过程。我实际上早在 2013 年就听说过 API 计划，当时该公司](https://web.archive.org/web/20221209205221/https://www.wunderlist.com/blog/wunderlist-in-2015-from-our-founder-and-ceo-christian-reber/)[筹集了由红杉(Sequoia)牵头的 1900 万美元](https://web.archive.org/web/20221209205221/https://beta.techcrunch.com/2013/11/12/to-do-app-wunderlist-confirms-19m-series-b-and-expands-to-the-us-while-sequoia-heads-into-germany/)(这是传奇风投对柏林初创公司的第一笔投资，意义重大)。当时，我被告知 API 将“很快”发布。

事实上，团队不得不重新构建整个产品，以使 API 达到他们想要的位置(我很想看看那份长达数年的待办事项清单)。随着 2014 年 7 月[Wunderlist 3](https://web.archive.org/web/20221209205221/https://beta.techcrunch.com/2014/07/31/here-comes-wunderlist-3/)的推出，这一切终于发生了。

制作 Wunderlist 的初创公司 6Wunderkinder 迄今已获得近 2400 万美元的投资，其他投资者包括 Earlybird 和 Atomico。该公司表示，目前不会再筹集更多资金，尽管，或许是基于其规模、团队和针对免费和付费用户的产品的优势，我们已经听说有几个“显而易见的候选人”接触过他们。

事实上，有两个消息来源更进一步，称微软是一个潜在的强有力的竞争者。(有趣的是，这里的两个早期 API 合作伙伴是微软的 Sunrise 和 OneNote。)

“在这个话题上没什么可分享的，”一位神童的发言人在回答我关于 M&A 活动的问题时说。

无论 Wunderlist 是否被收购，添加诸如 API 之类的功能是区别于其他待办应用的重要方式。

我们真的是指海洋。根据扎皮尔的统计，至少有 40 款可信的待办应用值得一提。就增长而言，Wunderlist 形容自己目前在同类应用中“表现突出”，尽管另一个待办应用 Any.do 以 1200 万用户的自有应用和 400 万用户的轻量级 Cal 应用紧随[之后。](https://web.archive.org/web/20221209205221/https://beta.techcrunch.com/2015/04/16/to-do-app-any-do-3-0-adds-collaboration-zoom-and-list-sorting-among-150-new-features/)

Wunderlist 告诉我，下一步将是一个集成中心——这听起来不像你会在 Slack 这样的产品中得到的类似功能，让用户可以非常容易地链接他们选择的任何应用程序，以馈入 Wunderlist。(注意，Wunderlist 也已经在集成领域取得了一些进展:10 月， [Dropbox](https://web.archive.org/web/20221209205221/https://beta.techcrunch.com/2014/10/01/to-do-app-wunderlist-links-up-with-dropbox-its-first-integration/) 成为了它的第一个集成合作伙伴。)

API 和未来集成中心都将对所有神童用户免费，尽管也有潜在的盈利计划。具体来说，这家初创公司正在寻求围绕某些更专注于业务的集成建立一个围栏，这些集成可以与其 [Pro](https://web.archive.org/web/20221209205221/https://www.wunderlist.com/pro/) 和 [Business](https://web.archive.org/web/20221209205221/https://www.wunderlist.com/business/) 层一起出售，Pro 的起价为每月 4.99 美元，价格根据相关业务的规模而浮动。
