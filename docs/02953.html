Twitter 的困境 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/02/22/twitters-dilemma/?ncid=rss

Twitter 是两件事情。这是一个概念——世界上的每个人都实时相连——回想起来是如此明显，以至于无法想象它不存在。这也是一个经历了艰难时期才实现这一概念的产品。

它的许多战斗都与说服市场这是一个可持续的业务，也是一个可用的产品有关——找到平衡。

该公司最近的收益超过了预期，证明亚当贝恩(Adam Bain)领导下的收入部门继续为 Twitter 的业务提供提振。股票上涨了百分之几。活跃用户的增长，虽然不完全是世界上最热门的材料，但在过去的一年中可以测量到 20%。市场似乎对科斯特洛和他的公司发出的声音做出了回应。但是产品呢？

过去几周，该公司一如既往地推出了各种新功能。但这些功能的细节让我对 Twitter 产品的决策过程感到好奇。

期待已久的唾手可得的果实，如群发消息功能、推文翻译和 Twitter 上的原生视频，都已于最近发布。因为它能够在用户离开平台时“捕捉”有趣的推文。对于一些用户来说，更受欢迎的新版主页正在预览中。

取悦大众。看似简单的产品，却花了太长时间才提供给用户。为什么是现在？

固化

作为一家上市公司需要果断，但这一点并不总是显而易见，因为 Twitter 喜欢实验，有时会对推出哪些产品做出令人困惑的决定。这种对承诺的恐惧影响了关于 Twitter 未来的故事类型。

我与新任命的产品副总裁 Kevin Weil 和工程副总裁 Alex Roetter 讨论了 Twitter 目前如何做出产品决策。

“对于使用 Twitter 的用户来说，他们能够建立一个账户并关注正确的人，我们一遍又一遍地听到它为他们提供了真正不相称的价值，”罗特说。“凯文和我真正关注的是为我们的用户打造产品，这样每个人都更容易进入那种状态。”

这可能包括为新用户通过算法生成的时间表，TechCrunch 去年 9 月报道了这一时间表，目前正在积极测试中。

Twitter 的新主页

罗特还指出，Twitter 应该利用自己掌握的数据，为用户做更多的工作。他讲述了一个关于地震的轶事，凌晨 3 点他被地震惊醒，寻找新闻。

“半小时前，我在 Twitter 上发现了这件事，然后我才发现其他地方也在报道这件事，但我做了很多工作。我必须知道正确的搜索词之类的东西。我真正想要的是，一旦 Twitter 知道了我感兴趣的事情，我的手机就会向我发送推送通知。”

罗特尔说，Twitter 正在考虑如何开发产品，让用户无需做任何工作就能在 Twitter 上分享内容。

“不是每个人都会有一个账户和一个精选的关注者列表。可能有人通过联合、注销或网站体验 Twitter，或者他们实际上可能登录并做我们的核心用户今天做的事情。”

视频

韦尔锁定了视频，这一产品已经引起了一些混乱，甚至在 Twitter 的员工中。很久以前，当 Twitter 从其官方应用中撤出视频分享服务时，人们认为它会很快获得原生视频，而不是更晚。有些人认为 Vine 会是这方面的自然选择，但 Twitter 建立了自己的引擎。可能会支持品牌和股东们所渴望的广告形式和本土广告产品。

“人们在他们的时间线中遇到的来自 NFL 和 NBA 的剪辑。它们被大量转发和分享……你可以在手机上做的任何事情，你也应该可以在 Twitter 上做。视频是其中的一个主要部分，”韦尔说。他说，即使在第一周，结果也很“好”，但不愿透露具体数字。

消息传递

在我的直接网络中，对集团 DMs 的反应相当积极。我目前正在和一些我在任何其他空间都没有交集的人进行对话。这一直是 Twitter 消息传递改进的吸引点。

Twitter 上的公共和私人之间有一层独特的薄膜——较低的表面张力有助于从公共场合私下开始对话。

韦尔同意，你在 Twitter 上的这种联系“图表”是使信息传递成为一个引人注目的焦点领域的原因。

“DMs 的神奇之处在于 Twitter 上的图表。事实是，在 Twitter 上，你不仅仅是和你的朋友联系在一起。你与你关心的人、事业、品牌和产品联系在一起。Weil 说:“群发直接消息可以让你接触到这些用户，私下进行交谈，这是你在其他地方无法做到的。

如果你对自己说“我迫不及待地想与品牌建立联系”，那么请继续关注，更多关于这方面的内容。

但是，如果消息传递对 Twitter 的未来如此重要——我认为现在如此，将来也是如此——为什么要花这么长时间才能发布呢？

“我们一直在做的很多事情都集中在 Twitter 的公开部分。你已经看到我们在 Twitter 内容的丰富性方面取得了巨大的进步。韦尔说:“我认为，甚至在 12 个月或 18 个月前，推文都不是 140 个字符。“现在，我们非常兴奋能够重新专注于私人信息。你会看到这种关注继续。我们刚刚开始使用 DMs。”

流程

Roetter 指出了一些工程团队的变化，以说明为什么他们现在能够发布 DM 改进。

“我花大量时间做的一件事就是让团队尽可能快地前进。这意味着在高层次上对什么是重要的有一个非常清晰的愿景，但留下一堆细节，以便团队可以找出答案，确保我们在团队中有合适的人才……雇用世界一流的人，然后确保每个人都有工具，以便他们可以非常快速和轻松地做他们职业生涯中最好的工作。我花了很多时间在这上面。我认为，我们向 DM 团队展示了，通过专注于这些事情，他们真的能够获得一些我们以前没有获得的东西。”

在过去的一年里，Twitter 的产品团队确实发生了变化。迈克尔·西佩于 2014 年初离开，当时 Twitter 带来了谷歌老将丹尼尔·格拉夫。目前还不清楚为什么 Graf 只持续了六个月，但 Graf 以数据为中心的背景和 Twitter 上的工程问题老鼠窝都被各种来源引用给我。不管那里到底发生了什么，Twitter 在去年 10 月用 Weil 取代了 Graf，从那以后，产品发布开始相当有规律地进行。

我使用几个分析包来跟踪我的 Twitter 使用情况，这些分析包告诉我，我越来越少使用第三方应用程序来访问 Twitter。我认为，这说明了一个事实，即 Twitter 增加的功能真的很有用。有人可能会冷嘲热讽，指出 Twitter 已经选择不支持其第三方 API 的许多功能，但它完全有权这样做。如果它想快速行动，同时又不担心失去自己产品以外的支持，那么它就有临床意义。

Twitter 目前发布和计划发布的产品类型已经发生了转变。两年前看似紧迫的用户登机流程问题终于得到了解决。

“如果我们正在开发利用 Twitter 这些独特方面的产品，那么我们正在开发的产品将对我们的用户产生巨大的意义。这将是一次很棒的经历。作为 Twitter 的一部分，感觉会很自然。这真的是我们的理念。”韦尔在回答一项关于他们如何挑选需求的调查时说。

“在过去六个月左右的时间里，Alex 和我一直专注于帮助团队，消除障碍，调整目标，并帮助每个人提高效率。我们有很多好主意。我认为，你已经看到，对执行节奏的关注现在随着过去几个月来许多伟大的发布而显现出来。”

关联引擎

新用户的即时时间表和“你不在的时候”功能是 Twitter 试图向外行人展示其价值的例证。我认识的大多数人都不属于传统的“全天每隔几分钟查看一次 Twitter”的核心群体，这个群体推动了关于产品以及什么对用户来说是“正确的”的讨论。他们中的许多人一天会查看服务几次——我敢打赌这是大多数人使用它的方式。这使得“当你不在的时候”功能不仅很好，而且对于证明更值得打开应用程序是不可或缺的。

“我们花了很多时间做研究，并真正试图了解我们的用户，”罗特说。“我们学到的一件事就是我之前提到的，当你得到 Twitter 时，它提供了不成比例的价值。我们非常专注于确保我们能让每个人都进入一个地方，在那里他们可以看到所有的价值和所有只存在于 Twitter 上的精彩内容。这就是我们走上即时时间线之路的原因，它允许你以一种更简单的方式注册 Twitter，而不必预先做任何工作，然后也可以在你离开的时候注册。你看到了 Twitter 的优点——你不必在完全正确的时刻查看你的手机。

“当然，这些对我们来说还为时过早。我们还没有看到我们最终将看到的所有收益，但迄今为止他们做得相当好。”

就在发布收益之前，Twitter 宣布从 Nest 聘请 Yoky Matsuoka 担任其技术和分析副总裁。松冈受到高度重视，她的聘用被许多聪明人标记为 Twitter 的重要收获。

罗特称松冈是“各种意义上的世界级领导者”，并表示她的聘用绝对与分析有关，但也与整个“相关性基础设施”有关。也就是说，Twitter 上的人们到底对什么感兴趣，以及如何让更多的人感兴趣。

“我们有所有这些很棒的内容，但今天……你实际上看到的是当你查看手机时的功能，你碰巧关注了谁，如果你知道正确的标签，诸如此类的事情。如果你转而转向组织我们的内容，并在正确的时间以正确的格式向所有正确的人提供独特的 Twitter 内容，无论是登录、注销、在你的手机上、在辛迪加中，无论是什么，那么需要进行一系列相关的工作，”罗特尔说。“Yoky 在许多不同方面都有丰富的经验。我们将在这方面投入巨资，这样我们就可以非常轻松地为用户提供高度组织化的内容体验。”

这种利用数学来帮助 Twitter 计算出其数十亿条内容中的哪些片段与每个用户相关、在什么位置和什么时间的推销，我已经听说了一段时间，但看到他们在该领域高调招聘令人鼓舞。

在提到投入到这种相关性机制中的努力时，Weil 很快插话，指出 Twitter 的实时方面对 Twitter 仍然“至关重要”。

“当我们谈论组织内容和相关性时，实时信号仍然是 Twitter 拥有的最独特的东西之一，其他任何人都没有，”Weil 说。“这对我们仍然至关重要。你不会看到 Twitter 不再专注于及时和引人注目的内容。”

Twitter 的实时性绝对是其最重要的独特 DNA。互联网的其他主要纤维都不是这样工作的，移除它会留下一个必须被填充的整体。

但毫无疑问，定制的算法时间线将成为 Twitter 未来的主要焦点。

“这不是一个放之四海而皆准的经验。这是一种体验和机会，可以适应我们拥有的不同类型的用户，”韦尔说。“我们为全新用户组织内容的方式，比如我们的即时时间线，可能与我们为几周后第一次回到 Twitter 的用户所做的不同，可能与每天使用 Twitter 一次的用户不同，也可能与每天使用 Twitter 十次的用户不同。”

本地

Twitter 最大的未开发领域之一是有组织的本地信息。Twitter 的群体思维非常强大——看看最近在迪拜码头火炬塔燃烧的大火，这是一个 Twitter 比任何当地新闻媒体更快、更丰富地描绘突发事件的例子。然而，Twitter 几乎没有在产品中真正利用这种能力。

“总的来说，当你考虑组织内容时，围绕你组织内容是一个非常有吸引力的机会，”韦尔说。“亚历克斯的[地震]例子完全是关于本地的。我周围发生了一些事情。正在发生地震。我想了解更多。我们绝对认为那里有机会。一般来说，事件是另一个将 Twitter 和本地或地理信号结合起来产生真正令人惊叹的体验的好例子。将来我们会对此有更多的说法。”

Weil 和 Roetter 无法分享产品的确切数字或未来计划，这是可以理解的。但是我很好奇，所以我自己做了一些调查。以下是 Twitter 在未来一年可以关注的一些事情。

2015 年的推特

据消息人士称，Twitter 的新面貌正在形成。目前正在预览的主页——带有用户和活动的网格——预示着新的外观可能需要什么。2015 年，改变 Twitter 外观和工作方式的工作正在进行中。

Twitter 未来一年的路线图中的一个要点是身份，以大写的“I”开头。Twitter 处于一个独特的位置，因为它的主要用例是公开的。这使得它在某些方面比脸书的公私混合模式或谷歌登录服务的“用户名和密码”身份更适合在线身份。如果 Twitter 找到了利用这一点的方法，那么它可以找到另一种方法来确保它在用户日常习惯中的持续地位，并设法使其保持不变。

这里有很多竞争，甚至在“社交空间”之外。苹果的 ID 和支付野心不太可能停留在“支付”按钮上。但是也有很多好处。与你一天查看两次的东西相比，在网络上跟随你(并进入其他应用程序)的服务更难放弃。

最终，视频仍将是焦点。在取消应用程序中现有的视频服务几年后，Twitter 开始着手支持核心媒体三足鼎立的第三条腿。用户实时直播流媒体事件的潜力是显而易见的——去年美国和海外的抗议事件凸显了这一点。

不仅仅是用户生成的内容。几个月前，我在推特上发布了一个链接，链接到一部可以在推特时间轴上观看的长篇电影。这告诉我。当你把整部电影直接放在 Twitter 上，点击一下就可以购买，会发生什么？像在消息应用程序中购买一包贴纸一样简单的 PPV 活动直播流怎么样？有趣的潜力。

说到消息传递，我们听说 Twitter 目前正在与 DMs 合作的一些工作包括允许您直接在 Twitter 上向品牌发送消息，以获得问题的答案，访问财务帐户等。反向渠道不仅仅是关于 Twitter 的人，它也是关于连接人和它最大的居民。

推特的平衡之举

有时(相当多)Twitter 选择推出功能和产品的方式让人感觉精神分裂。这并不奇怪，因为公司现在有两个主人。它的用户和股东。虽然这些利益有时可能是一致的，但对于一家上市公司来说，取悦哪一方更重要是毫无疑问的。

这导致了外界，甚至内部对 Twitter 方向信心的看法不稳定。

在外部，有很多关于首席执行官迪克·科斯特罗被取代的流言蜚语，阿丽克西亚在这里做了完美的剖析。虽然我听过科斯特洛用许多五颜六色的贬义词来形容，但“无能”从来不在其中。当然，这并没有阻止员工行使他们抱怨的制度权利，也没有员工像工程师一样察觉到弱点。

最近，Twitter 的秘密巢穴变得如此拥挤，以至于科斯特洛在 Twitter 内部留言板上称其为“粪坑”。有消息称，从那以后，这里已经有点像鬼城了。

我不是金融奇才，所以我不能告诉你 Twitter 上市的时机是正确的还是太早了(T4)。它有现金，它用这些现金进行收购和招聘，并留住人才。但是这样做绝对排除了很多选择，并且使得弄清楚需要什么变得更加困难。尽管 Twitter 感觉已经存在了一段时间，但在许多方面它仍处于妊娠期。

我也不相信市场理解，或者永远不会理解 Twitter 存在的原因。这让 Twitter 的领导层，包括其产品团队，陷入了不得不继续执行产品变戏法以服务于两个主人的不幸境地。

我在过去曾提出过这样的例子【that Twitter 允许自己被一个由测试数据伪造的黄铜鼻环牵着鼻子走。在数据驱动的文化中做决策的危险在于，你忘记了消费产品的目的不是为自己服务，而是为用户服务。

最近的产品决策似乎显示出对如何平衡 Twitter 困境的更多思考。市场是否会承受这一点，或者是否有办法真正找到一个平衡点，还有待观察。但是也有一些令人高兴的迹象。我们将会看到消息产品是否真的像韦尔所说的那样受到关注，以及视频和入职培训是否会继续得到完善。

不幸的是，并不是人们喜欢 Twitter 的所有东西都受到了皇家待遇，这让我想到了这个:

马修:我的最后一个问题是一个很受欢迎的问题，那就是“苹果的推特是怎么回事？”凯文:请继续关注。你用它吗？马修:是啊。我试着…它需要被修理。仅此而已。凯文:请继续关注。
