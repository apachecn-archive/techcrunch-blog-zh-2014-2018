# Quantcast 说，特斯拉司机正在使用他们的仪表板浏览器来了解新闻

特斯拉车主究竟在用他们的车内网络浏览器做什么？这是广告测量和定位公司 [Quantcast](https://web.archive.org/web/20221204152523/http://www.quantcast.com/) 试图用一些刚刚发布的数据来回答的问题。

显然，特斯拉 Model S(T2 现在有超过 25，000 辆在路上)包括[一个带网络浏览器的 17 英寸触摸屏](https://web.archive.org/web/20221204152523/http://www.techradar.com/us/news/car-tech/the-new-tesla-s-has-the-most-insane-in-car-touchscreen-multimedia-system-ever-1185159)，是的，它在你开车的时候也能工作。

为了了解人们可能在汽车中实际进行的浏览，Quantcast 使用浏览器标识符来隔离特斯拉在 30 天内对 1 亿个网站的访问。该公司专注于页面浏览量超过 100 次的房产，在此期间跟踪了特斯拉的 46.3 万条 PV，并表示新闻网站占了异常大的流量(54%)——仅 Drudge Report 就占了特斯拉所有页面浏览量的 10%，财经新闻网站合起来占了 13%。在新闻类中，本地网站占总访问量的 26%。

Quantcast 还发现，早上 7 点至下午 6 点之间的使用相对一致，这似乎令人惊讶——在一篇博客文章中，该公司想知道，“特斯拉车主只是整天开车吗？”嗯，在那段时间里有一些不同，中午的娱乐使用量更高，上下班时间的新闻使用量更高。Quantcast 表示，这是为了“以后的分析”而保留这些司机是否是当天浏览的同一批司机的问题。

随着越来越多的联网汽车上市，这告诉我们更广泛的使用模式是什么？该公司总结道:

> 尽管特斯拉 Model S 是第一款配备标准网络浏览器的大规模量产汽车，但尚不清楚 Model S 的浏览行为能否预示联网汽车作为媒体平台的未来用途。随着汽车制造商开始提供谷歌和苹果的原生应用环境，联网汽车将会发展。当前的浏览活动似乎更能反映特斯拉车主的情况，其中一半人住在加州，他们自我选择拥有更高的收入，并且是早期的技术采纳者。我们可以说的是，特斯拉车主正在使用他们的内置浏览器来获取新闻、金融等信息，尽管他们手头可能有智能手机。特斯拉司机已经验证了车内浏览体验的价值，而不仅仅是寻找目的地——这是联网汽车是一个值得关注的新媒体平台的早期指标。
