英国间谍机构支持的网络安全孵化器选出前七家初创公司

原文：https://web.archive.org/web/https://techcrunch.com/2017/01/11/uk-spy-agency-backed-cyber-security-incubator-picks-first-seven-startups/?ncid=rss

一家由 GCHQ 支持的初创企业孵化器在英国开门迎客，宣布首批七家初创企业参与为期三个月的加速器计划(团队列表见下文)。

政府在 9 月宣布了 GCHQ 网络加速器倡议，作为国内信号情报网络安全机构之间的合作；英国文化、媒体和体育部(DCMS)；以及(Telefonica 的)Wayra UK accelerator initiative，它正在运行这个项目。

近年来，英国政府一直将网络安全作为优先事项，政府承诺19 亿英镑用于到 2020 年的国家网络安全战略，而将为一系列倡议提供资金；包括加强情报机构的网络安全人员；建立一个国家网络安全中心，集中专业知识并以身作则；扩大政府对安全技术的采购——包括投资安全初创公司。

GCHQ 支持的加速器是在国内推动网络安全理念和创新的战略的一部分。它位于一个新的切尔滕纳姆网络创新中心内，将选定的创业团队置于机构工作人员和专业知识的轻松覆盖范围内，正如政府所说，“允许他们扩展能力，改进想法和设计尖端产品，以超越当前和新兴的威胁”。

两年前，当时的财政大臣乔治·奥斯本说，政府的目标是从切尔滕纳姆的 GCHQ 情报局总部，通过布里斯托尔和巴斯到埃克塞特，创建一个他称之为“网络卓越之弧”的区域，目的是“使西南地区成为网络安全服务的世界领导者”。第二个中心计划于今年晚些时候在伦敦开业，DCMS 将在未来 5 年内出资 5000 万英镑建设这两个中心。

GCHQ 没有投资加速创业公司，但 Wayra UK 的一位发言人告诉 TechCrunch，该机构将提供导师——一旦该计划结束，团队可能能够继续与该机构建立合同关系，通过既定的采购流程，和/或邀请“更深入的研究机会”，以及由于他们建立的联系而与现有的主要 GCHQ 供应商建立伙伴关系。

参与的初创企业还将获得 Wayra 提供的财政资助(5，000 英镑), Wayra 目前也不接受股权，并获得工作场所和 IT 设备。

入选第一批的七支队伍是:

  • 反间谍公司 CounterCraft ，旨在保护拥有网络安全欺骗平台的大型组织，用诱饵电脑、虚假数据和虚假身份欺骗对手
  • Cyberowl ，一个网络攻击预警系统，结合了“高级安全分析和启发式方法”
  • Cybersmart ，一个自动化实施、认证和遵守网络安全标准的平台
  • future escaper，一个集体智能平台，提供数据可视化，以便理解复杂、不确定或不稳定的问题
  • 球形防御，一种银行 API 入侵检测系统，通过建立正常通信的基线，使用深度学习来检测黑客攻击企图
  • StatusToday 一个由人工智能驱动的智能平台，旨在理解人类在工作场所的行为，提高抵御内部攻击的安全性，并检测无意中的错误
  • Verimuchme ，一个保护、验证和重新使用个人信息的数字钱包和交换平台。通过基于身份的加密技术，个人和企业与个人信息联系在一起，这些信息可以一次又一次地以数字方式共享

在接受 TechCrunch 采访时，Countercraft 的联合创始人和 CMO·丹·布雷特表示，这个位于圣塞巴斯蒂安的团队希望通过该计划在英国建立强大的联系。

“我们正在设计一种防御产品，让大型组织能够防御现有的最先进的攻击，在哪里能比 GCHQ 更好地获得关于公司可能面临的问题的建议。我们的目标是提供一种新型的防御层，提高英国和全球公司的安全性。我们相信这将是让我们的产品变得更好的一大步，”他补充道。

目前还不清楚 GCHQ 网络加速器是否会在这个时候运行第二个程序。Wayra 的发言人表示，该项目将在第一个 3 个月后进行审查，然后再决定接下来会发生什么。
