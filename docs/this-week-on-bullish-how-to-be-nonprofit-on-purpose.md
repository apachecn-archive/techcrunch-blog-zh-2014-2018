# 本周看涨:如何有目的地成为非营利组织

大家好，欢迎回到[看涨](https://web.archive.org/web/20230212130659/https://techcrunch.com/video/bullish/)，这是 TechCrunch 的第一个脱口秀节目，也是一次跑步尝试，看看我能在常规的 10 分钟内出汗多少。

本周，我们深入探讨了一个离我们内心既近又远的话题:非营利组织及其在技术世界中的影响。这个概念贴近我们的内心，因为非营利组织对我们经济和政治的大部分至关重要。这个问题远离我们的心，因为在硅谷，赤裸裸的贪婪几乎被视为一种美德。

这就是 2015 年。

为了帮助我们深入了解这个问题，优秀的[乔伊斯·金](https://web.archive.org/web/20230212130659/https://twitter.com/joyce)来到了演播室。金目前正在开发 [Stellar](https://web.archive.org/web/20230212130659/https://www.stellar.org/) ，这是一种价值转移的数字方法，希望帮助全球非精英人士转移他们的资金，而无需支付高昂的费用。

恒星发展基金会是一个非盈利性的组织。我非常好奇这会对它的运营、筹资和招聘产生怎样的影响。在这个时代，你能成为一家非盈利性的商店，同时还能吸引你所需要的人才来推动技术进步吗？

与此同时，利他主义在我们当前的环境中处于什么位置呢？它还有空间吗？让我们找出答案。
