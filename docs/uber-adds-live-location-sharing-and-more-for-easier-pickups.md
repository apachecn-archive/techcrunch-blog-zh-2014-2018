# 优步增加了实时位置共享等功能，方便取货

优步对其骑手应用程序进行了一些更新，旨在使繁忙的假日旅游季节的取车过程更加容易。最大的变化可能是你现在可以在应用程序中进行实时位置共享，让你暂时与你的司机分享你的确切位置——所有这些都在你的主优步屏幕的控制之下。这将有助于避免你的司机不确定在哪里与你会合的情况。

要使用此功能，您只需通过地图显示右下角的点击图标打开“实时位置”功能。蓝色图标表示它是活跃的，灰色版本意味着你没有分享这些额外的信息。您可以随时点击图标在两种模式之间切换。

优步也在扩大其信标计划，该计划使用司机挡风玻璃上的硬件设备向乘客显示相应的门，以帮助他们更容易找到汽车。在最初的有限试点后，它不会在纽约市、旧金山和芝加哥推出，所以它将在一年中最繁忙的乘车时间向更多的乘客和司机提供。优步于去年 12 月首次推出 Beacon，首先在迈阿密、丹佛、纳什维尔和纽卡斯推出。

其他骑手的变化包括简化了为其他骑手请求搭车的流程(不再需要你首先指定一个不同于你自己的地点来为其他人请求搭车)，以及一个新的应用内礼物流程，可以轻松地直接从应用本身向其他人提供优步积分。
