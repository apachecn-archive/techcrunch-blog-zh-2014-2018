# PocketBeagle 是市场上最小的、名字最古怪的单板计算机

你需要一个大约几个硬币大小的 Linux 机器吗？伙计，我有东西给你。它叫做 [BeagleBoard PocketBeagle](https://web.archive.org/web/20221006224236/http://beagleboard.org/pocket) ，是你花 25 美元就能买到的最小的 Linux 电脑。

我是单板电脑的超级粉丝，原因很简单:在我的童年，实现这些东西的技术根本不存在。此外，像这样的一个小套件——包括 512MB RAM、微型 Octavo Systems OSD 3358 1 GHz ARM Cortex-A8 处理器和 72 个扩展引脚，包括电源和 I/O——是业余爱好电子产品力量的真实证明。大约十年前，把这样的东西降到 25 美元是不可能的。现在已经司空见惯了。

PocketBeagle 是完全开源的，它包括一个板载启动 ROM，可以让你将这个东西嵌入到几乎任何需要一点 Linux 大脑的项目中。
