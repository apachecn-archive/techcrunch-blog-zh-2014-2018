# 谷歌地图收购地图分析和可视化初创公司 Urban Engines

这与其说是一项收购，不如说是欢迎回归。Urban Engines 由一群前谷歌员工在 2014 年成立，得到了谷歌风投和埃里克·施密特的支持。这家初创公司的成立是为了帮助解决城市拥堵问题，改善用户通勤状况。

去年 2 月，该公司推出了一款以自己命名的应用，为移动地图带来了一些关键功能，其中包括快速计算、增强现实地图模式以及在北美主要发布城市波士顿、芝加哥、洛杉矶、纽约、波特兰、旧金山、西雅图、多伦多、温哥华和华盛顿特区离线使用地图的能力

Urban Engines 使用术语“移动物联网”来描述其整体构想，利用城市通勤者收集的分析来帮助控制城市交通。现在，该公司正在将其数据收集和平台带回山景城母舰。

在一篇概述公司历史的博客文章中，Urban Engines 宣布它将加入谷歌地图团队，“我们很高兴能够联合各方力量，帮助组织更好地了解世界是如何运动的。”

关于该公司将如何融入更大的谷歌生态系统，没有太多信息，但肯定它可以为谷歌地图应用程序带来很多东西，以及有助于支持地图团队底层基础设施的数据分析。
