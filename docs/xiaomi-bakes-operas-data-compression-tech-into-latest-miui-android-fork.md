# 小米将 Opera 的数据压缩技术烘焙到最新版本的 MIUI Android 中

小米最出名的可能是其价格有竞争力的智能手机，这些手机在中国的市场份额被三星和其他安卓制造商所占据，但它也有自己的安卓版本。今天，这家中国公司发布了其 MIUI 平台的新版本，该平台最初是在 2010 年小米开始销售手机之前创建的，现在拥有 1.5 亿用户。

MIUI 7 全球版的突出特点——将于 8 月 24 日和[推出测试版，看起来仍与苹果的 iOS](https://web.archive.org/web/20221209193051/https://beta.techcrunch.com/2014/08/18/xiaomis-new-miui-6-wont-help-it-fend-off-copycat-accusations/) 非常相似——可能是与 Opera 的合作，该公司声称，将把浏览器和其他应用程序的移动数据使用量压缩多达 50%。该功能利用了 Opera 的 Max 技术——现在可用于 YouTube 和网飞视频——并被植入 MIUI 7 的“数据保护”功能中。同样在性能下提交的，小米声称其软件的最新版本可以帮助应用程序运行速度提高 30%，同时消耗的电池寿命减少 10%。

这是唯一的主要增加，这肯定比以前的 MIUI 更新感觉更迭代。在设计方面，小米推出了许多新的调整，据称这使其软件比以往任何时候都更具可塑性。“Showtime”允许用户为每个联系人分配一个五秒钟的循环视频，当他们打电话时播放，一个每日图像功能在锁屏上显示不同的照片，而一个新的“XXL”字体大小是为那些抱怨手机上的文本不够大的人准备的。

作为一名家长，如果小米的“婴儿相册”功能像该公司声称的那样有效，我会印象深刻。该功能使用面部识别将孩子的照片整理到专用相册中。MIUI 的其他新增功能包括四个新的系统 UI 设计，以及与小米的 Mi Band 可穿戴设备的更紧密集成。

在另一个展示中，[小米优先考虑印度](https://web.archive.org/web/20221209193051/https://beta.techcrunch.com/2015/04/07/xiaomi-is-coming-for-you-india/)作为[其领先的全球](https://web.archive.org/web/20221209193051/https://beta.techcrunch.com/2015/04/26/xiaomi-ratan-tata-investment/)(理解:不是中国)市场，全球 MIUI 发布会在新德里举行——中国版 MIUI 于本月早些时候发布——许多新功能将首先向印度用户推出。当然，小米[现在正在印度实地生产智能手机](https://web.archive.org/web/20221209193051/https://beta.techcrunch.com/2015/08/10/xiaomi-is-now-making-smartphones-in-india/)，今年早些时候[小米 4i 成为其第一款为印度消费者设计的设备](https://web.archive.org/web/20221209193051/https://beta.techcrunch.com/2015/04/23/xiaomi-mi4i/)。
