福特的 AppLink 将很快支持第三方导航应用

与谷歌地图等现代智能手机应用相比，大多数汽车的内置导航系统往往有点笨重。看起来福特是第一批承认这一点的汽车制造商之一。下一个版本的 AppLink ，其连接智能手机应用程序和同步信息娱乐系统的系统，将允许第三方导航应用程序将其地图从手机投影到汽车的内置屏幕上。

福特将在今年晚些时候推出 SYNC 的最新版本(和改进很多的),但 AppLink 3.0 将在晚些时候推出(SYNC 可以通过 WiFi 和连接的智能手机更新)。

该公司已经与阿里巴巴合作，将其导航和音乐服务引入车载屏幕。有可能，其他公司也会加入进来，尽管值得注意的是，福特必须明确地将应用程序列入白名单才能在 AppLink 上运行。不过，没有理由认为该公司会阻止谷歌、微软、TeleNav 或 here 在其平台上提供地图服务。

AppLink 目前主要专注于音频应用，但通过 Glympse，它长期以来也提供位置共享服务。

正如福特指出的，AppLink 3.0 将使用 Genivi 联盟的开源 SmartDeviceLink 服务及其 API 来实现这一功能。Genivi 联盟中的成员除了福特，还包括宝马、本田、日产、雷诺、沃尔沃和约翰迪尔，以及英特尔、高通和英伟达等芯片制造商和大量售后制造商。本月早些时候，Genivi 联盟也宣布将提供开源中间件来支持 Android Auto 集成到汽车信息娱乐产品中。
