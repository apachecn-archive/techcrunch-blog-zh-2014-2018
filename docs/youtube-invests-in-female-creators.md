# YouTube 投资女性创作者 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/03/02/youtube-invests-in-female-creators/>

YouTube [今天早上宣布了两个项目，旨在资助和突出女性在视频网络中的声音。人们看到该公司与联合国建立了为期一年的合作伙伴关系，任命顶级女性创作者作为该组织可持续发展行动运动的第一批变革大使，他们将在那里倡导性别平等。第二个是投资 50 多个视频，由 YouTube 新的全球 YouTube Spaces 计划资助，该计划将女性放在镜头前和镜头后。](https://web.archive.org/web/20221226025355/http://youtube-global.blogspot.com/2016/03/celebrating-female-voices-on-youtube.html)

随着联合国运动的开展，YouTube 宣布视频创作者[英格丽·尼尔森](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/missglamorazzi)、[杰姬·艾纳](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/lilpumpkinpie05)、[于亚](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/lady16makeup)、[塔蒂·费雷拉](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/AcidGirlTestosterona)、[海拉·加扎尔](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/HaylaTV)、[路易斯·彭特兰](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/Sprinkleofglitter)和[吉田千香](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/cyoshida1231)为变革大使。作为项目参与者，他们将与联合国合作，通过发布促进性别平等的视频，充当榜样并“激励和激活”他们的粉丝社区。

与此同时，来自世界各地的六名女性创作者，包括[安娜·阿卡娜](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/AnnaAkana)(美国)[阿列克西斯·弗莱明](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/MadeYewLook)(美国)[朱莉娅·维加·法利亚](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/joutjoutprazer)(巴西)[Em·福特](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/mypaleskinblog)(英国)[尼拉姆·法鲁克](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/daaruum)(德国)和[库马·米基](https://web.archive.org/web/20221226025355/https://www.youtube.com/user/kumamikicraft)(日本)，最近在 YouTube 的全球空间担任创意总监，这些空间是该公司在全球各地设立的专用制作设施这些女性在洛杉矶、伦敦、东京、纽约、圣保罗和柏林的 YouTube 空间指导其他人，努力制作更多由女性主导的视频。该公司表示，这些视频是在作为该计划一部分的特别设计的场景中拍摄的。

这些努力产生的视频将包括女性在劳动力中的主题，向历史上的女性领导人致敬，以及其他原创的脚本内容。YouTube 指出，主题还可能包括健康、科学、工程、音乐和喜剧。

作为这一就职计划的一部分，创建的一些视频包括 YouTube 上几位大牌的视频，如 Michelle Phan、GloZell、Ali Brustofski、Flavia Calina、Olhos de Nuvens、Ochikeron、mirellativegal 等。

该公司表示，这些视频将从今天开始在 YouTube Spaces 频道以及创作者自己的频道上发布，就在 3 月 8 日国际妇女节到来之前。
