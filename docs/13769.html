特斯拉专注于 Model 3 发布时的生产挑战| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2017/07/28/tesla-is-focused-on-model-3-production-challenges-at-launch/

在第一批交付给客户之前，特斯拉比以往任何时候都透露了更多关于其全新 Model 3 的信息。我们知道定价和可用选项的细节，以及特斯拉计划如何扩大到每年生产 50 万辆汽车的年运行率，希望到 2018 年底。

“我们在 Model 3 上面临的主要挑战并不是真正的产品，”特斯拉首席执行官埃隆·马斯克在会前的新闻发布会上指出，尽管如此，他们还是花了很多时间来进行正确的设计。“我们将经历 6 个月的生产地狱。这将非常棒，但生产这款车将是一个相当大的挑战。”

马斯克在关于这款车的最初评论中反复强调了生产挑战，事实上，他指出，像这款车这样复杂的东西的制造过程必然会出现在 S 曲线上，而且“汽车中有 10，000 个独特的部件”，生产速度最终必然只会“与最慢的部件一样快”。

为了让生产变得更容易，特斯拉对车辆做出了一些具体的决定，包括汽车的内部，这是汽车制造商在今天之前没有讨论过的事情。

“我们有意选择一个非常简单的内部结构，只有一个屏幕，”马斯克解释道，展示了一个旨在最大限度地增加可用内部容积，同时尽可能保持车辆外部最小的客舱。

例如，第二排座椅后面没有顶盖，首席设计师 Franz von Holzhausen 解释说，这是为了给乘客开辟可用空间。

“你的头后面什么也没有，所以感觉就像是一个漂亮、宽敞、开放的后座，”他说。“内部空间对我们来说很重要，它是功能的一部分。”

这部分是因为 Model 3 在设计时考虑到了自主性的长远观点。马斯克解释说，其产品包括与 Model S 和 Model X 相同的传感器硬件，尽管其他功能和设计元素被大幅削减，以帮助提高制造效率。自动化程度的提高是有所帮助的一个因素，但还有很多事情需要改进。

马斯克表示，特斯拉可以在目前制造 5 万辆 Model S 汽车的相同空间内制造约 25 万辆 Model 3 汽车，通过简化流程，制造能力增加了五倍。这包括最初坚持单电机后驱动设计，以及使用更多的钢材。大约 50%的车辆是钢，而另一半是铝。相比之下，Model S 几乎全是铝，这在生产中更难使用。

尽管如此，马斯克继续回到制造和提升生产流程所代表的挑战。

“我们面临的最大挑战是 S 曲线制造，”他说。" S 曲线部分基本上是我们正在经历的地狱."

他说，这是特斯拉跑车最糟糕的一次，尽管它使用了莲花底盘。他说，在 Model S 上稍微好一点，但仍然“见鬼”，在 Model X 上，它几乎和在 Model S 上一样困难，这不应该是因为它设法重复使用了 X 上的许多组件。

马斯克说，最终，特斯拉有信心实现这个 S 曲线，但考虑到各种因素，包括来自供应链的“不可抗力”风险，包括多达 30%的零部件来自北美以外，很难预测它将采取什么形状。

尽管如此，马斯克似乎对 2018 年底达到每年 50 万辆的目标相对有信心，这将大大有助于解决被压抑的需求。关于需求，马斯克表示，虽然在衡量真实需求时，预订数量是一个错误的指标(因为特斯拉会尽一切努力取消销售)，但目前预订量超过 50 万辆。

“如果我们做任何事情，甚至不按要求刹车，它会疯掉，”马斯克说。“但我们现在还不必这么做。”
