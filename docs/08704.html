原文：https://web.archive.org/web/https://techcrunch.com/2016/08/15/surface-pro-bowl/

微软用巨大的NFL标志装饰其Surface Pro型号外壳

当你生产NFL的官方平板电脑时，这些就是你得到的额外津贴。除了一船的宣传视频和在美国最受欢迎的运动中的一些严肃的副业时间，微软还展示了一些对Surface Pro的官方认可的NFL球队类型封面的喜爱。

在2016年橄榄球赛季之前，该公司发布了一系列商务平板电脑的键盘盖，上面印有所有32支NFL球队的巨大标志。微软表示，在保罗·艾伦(Paul Allen)的定制Surface 3海鹰封面在Twitter 上大受欢迎后，它选择全面推广这一品牌——尽管Twitter上的回复也不出所料地变成了一个客户支持论坛，供人们抱怨他们对平板电脑的问题。

微软Surface特别版NFL Type Covers(确实有点拗口)售价高达160美元(比非品牌款高出30美元)。他们现在可以通过微软购买，并将在本月底登陆百思买和亚马逊，在赛季正式开始前给你大约一周时间。
