苹果收购了数字报刊亭，因为它对来自可靠来源的内容加倍下注

原文：https://web.archive.org/web/https://techcrunch.com/2018/03/12/apple-acquires-digital-newsstand-texture-as-it-doubles-down-on-content-from-trusted-sources/?ncid=mobilerecirc_recent

随着围绕假新闻以及脸书等聚合器在传播假新闻中扮演的角色的辩论继续进行，苹果正在进行一项收购，这可能有助于其奠定可信信息提供商的地位。这家iPhone制造商正在收购被称为“杂志出版界的网飞”的杂志虚拟报摊 Texture ，读者可以以每月9.99美元的价格获得约200种杂志。

“我们很高兴Texture将加入苹果，以及来自许多世界领先出版商的令人印象深刻的杂志目录，”苹果互联网软件和服务高级副总裁Eddy Cue在一份声明中说。“我们致力于从可靠的来源获得高质量的新闻，并允许杂志继续为用户制作设计精美、引人入胜的故事。”

据我们所知，Texture——以前称为Next Issue——将继续保持原样运营，没有任何变化——这意味着它将继续为iOS、Android、亚马逊Kindle Fire和Windows 8和10提供应用程序。苹果正在收购整个公司，包括员工，预计交易将很快完成。

“我很高兴Next Issue Media及其屡获殊荣的纹理应用程序被苹果收购，”Next Issue Media / Texture的首席执行官约翰·拉夫林在一份声明中说。“Texture团队及其目前的所有者Condé Nast、Hearst、Meredith、Rogers Media和KKR对这一进展感到无比高兴和激动。我们想象不出比这种服务更好的归宿和未来。”

苹果已经在Texture专注的出版和媒体领域进行了大量收购，其中包括Spotify/Pandora的竞争对手Beats for Apple Music，以及 BookLamp ，我们在宣布收购消息时将其称为“书籍的潘多拉”。Texture并不是苹果在杂志出版领域的第一次收购:它在2014年收购了Prss ，以帮助建立苹果新闻。

这笔交易的财务条款尚未披露，Texture也从未披露其估值。

在2015年更名之前，Texture(当时被称为Next Issue Media)是康泰纳仕、赫斯特、梅雷迪斯、新闻集团、罗杰斯通信公司和时代公司的合资企业。该公司似乎已经从出版商那里筹集了至少9000万美元、4000万美元，另外从KKR和其他投资者那里筹集了5000万美元，其他投资者包括BuzzFeed、Vox Media和Imgur。

纹理，在它下一期之前，从2010年就有了。它是在其他一些数字报刊亭上市的时候推出的。

利用应用程序的新普及，以及这些应用程序将成为消费者阅读报纸和杂志的主要方式的信念，出版商还认为，这可能是他们更好地将内容货币化的一个关键方式，此前他们错过了在线内容付费墙和从在线广告中获得巨大利益的机会，而谷歌等大型聚合器在这些领域获得了巨大的回报。

目前还不清楚Texture有多少用户。2014年的一篇文章估计这一数字为15万，而2016年的这篇文章指出这一数字在“几十万”之间，预计未来几年将增长50%。该公司还与Sprint等第三方达成协议，将该服务与订阅捆绑在一起，作为一种甜头，让消费者选择他们的服务，而不是其他移动运营商的服务。

也不清楚苹果打算如何长期处理纹理。

我们可以看到该产品的一个领域是苹果新闻，苹果已经提供了各种第三方内容的访问。更广泛地说，该公司一直专注于在其他媒体上播放更大规模的优质内容，对音乐、视频和播客进行了大量投资。纹理用阅读材料填充了视觉的范围。

当然，苹果曾经有过自己的“报摊”——特别是它自己的本地应用程序。这项服务从未真正起飞，苹果最终扼杀了该产品，并将其并入苹果新闻。

与众不同的一个原因是，它将本质上把许多杂志纳入一种格式，而不是提供一个本质上不同的杂志和报纸应用程序的市场，从用户和出版商开发者的角度来看，这似乎是最初的报刊亭的一个痛点。

(我不得不说，当时是谁规定了这种格式，这一点从来都不清楚:很可能是苹果和2011年报刊亭首次出现时的技术限制，尽管也有其他应用程序可以解决这个问题。同样可能是出版商认为，从零开始建设，类似于他们自己的垂直整合印刷机构，才是前进的方向。)

但从另一个意义上来说，这次收购对苹果这样的公司来说也只是简单的桌面股权。亚马逊去年推出了自己的订阅服务，谷歌当然也通过Google Play提供各种报摊，所以这也是为了保持和确保它也能继续提供所有设备所有者越来越想要和期望的东西。

苹果的提示今天也走上了奥斯汀SXSW的舞台，他谈到了Texture在苹果新闻工作中的地位以及该公司更广泛的媒体和内容战略。
