谷歌正在使用光束技术将印度农村与互联网连接起来

在宣布计划在印度推出后，谷歌正准备使用光束将地球上的农村地区联网。

Alphabet 的 X——该公司前身为谷歌 X——正在与印度安得拉邦的一家电信运营商合作，该邦拥有超过 5000 万人口，使用自由空间光通信(FSOC)，这是一种利用光束提供长距离高速、大容量连接的技术。

现在，合作伙伴 AP State FiberNet 将从 1 月份开始引入 2000 条 FSOC 链路，为其在该州的网络主干提供额外支持。该公司表示，X 项目旨在“支持成千上万人的主要接入点的关键缺口，如手机信号塔和 WiFi 热点”。

这家美国公司补充说，该倡议与 T2 政府的一项倡议相结合，到 2019 年将有 1200 万家庭接入互联网。

在 X 利用 FSOC 为的 Loon connectivity 项目传递信息之后，该项目利用热气球在偏远地区传递互联网连接，这一想法应运而生。领导这个项目的巴里斯·厄克曼解释说，这个团队后来意识到它本身也可以成为一个独立的机会。

x 将部署一个团队到安得拉邦进行推广工作。Erkmen 说，它也愿意与其他组织进行交易。
