# Twitter 首席执行官多尔西向开发者道歉，称他想“重置”关系 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/10/21/twitter-ceo-dorsey-apologizes-to-developers-says-he-wants-to-reset-relations/>

几年前[Twitter 的](https://web.archive.org/web/20221001174743/https://beta.techcrunch.com/2012/09/05/twitter-officially-launches-v1-1-of-its-api-developers-have-until-march-5-2013-to-switch/)[史诗般的破产](https://web.archive.org/web/20221001174743/https://beta.techcrunch.com/2012/08/16/twitter-api-client-apps/) 是该公司坎坷历史上最大的挫折之一，所以现在，当这家社交媒体公司在新任首席执行官杰克·多西的领导下努力重新启动业务时，它以一个关键信息开始了它的大型开发者活动:为 Twitter 过去的行为道歉，并承诺努力做得更好。

在提到 Twitter 最初向开发者示好的那些年里，随着 Twitter 的发展，它砍掉了某些功能，并试图找到自己的业务和平台时，多尔西说:“我们与开发者的关系变得令人困惑，不可预测。我们今天来找你，为造成的混乱道歉。”

最大的问题是:这次开发者能信任 Twitter 吗？

(h/t [@Selviano](https://web.archive.org/web/20221001174743/https://twitter.com/selviano/status/656881818344402944) )

多尔西继续谈论公司如何翻开新的一页，现在希望“确保我们与我们的开发人员有良好的关系…我们正在满足和服务每个人的需求…我们需要与我们的开发者社区，与这个房间里的每一个人进行更好的对话…我们不能孤军奋战。我们需要你的帮助。”

当然，Twitter 已经不像以前那样风光了。就在今天，该公司被摩根士丹利(Morgan Stanley)降级，理由是参与度和用户增长有限，以及对该公司广告业务的担忧:降级导致该公司股票在早盘交易中下跌近 7%。

因此，毫不奇怪，多尔西决定将这种关系拉回一个更好的时期，当时 Twitter 有开发者的支持，而不是敌对的关系。

展望未来，该公司表示将改善与开发者的沟通。“我们想确保我们与我们的开发者有一个很好的关系，一个开放和诚实的关系，”他说。

他甚至指出了一些引起 Twitter 愤怒的应用程序，以及一些更具争议的行为。多尔西说:“我们有责任继续支持像 Politwoops 这样希望实现透明度的组织。”。(回想一下 [Twitter 曾经因为违反 ToS 教而关闭了](https://web.archive.org/web/20221001174743/https://beta.techcrunch.com/2015/08/24/twitter-shuts-down-services-that-tracked-politicians-deleted-tweets-in-30-countries/)这个跟踪政客们被删除的推文的账户。)

他以一个邀请结束:“把你想在 Twitter 上看到的东西推给我们。多尔西说:“在带有#helloworld 标签的推特上，我们将收集所有这些信息和输入，随着时间的推移做出决策，以确保这个平台是你引以为豪的东西，并且会越来越多地使用。”“这不会在一夜之间发生，但我相信我们会做出正确的决定。”

让我们看看 Twitter 是否会跟进。
