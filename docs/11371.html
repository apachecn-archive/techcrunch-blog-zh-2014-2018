三星将推出大型 Galaxy S8 手机

三星即将发布一款屏幕比 Note 7 更大的 S8。根据这份泄露的规格页，S8+将拥有 6.2 英寸的曲面屏。那大约是一本电话簿的大小。

这款手机显然被称为 S8+,据说是 S8 的变体——有点像苹果公司提供的 iPhone 和 iPhone Plus。一向富有创造力的三星显然是在用加号，而不是拼出这个单词。

泄露的规格表也证实了 S8+将像短命的 Note 7 一样配备虹膜扫描仪。该说明书似乎还暗示这款手机将有一个耳机插孔，因为它指出这款手机将配备“AKG 调谐的耳机”

三星有可能在未来几周发布这款手机，最早可能在本周末的世界移动通信大会上发布。
