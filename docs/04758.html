大数据供应商 BlueData 宣布 2000 万美元融资并与英特尔合作 TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2015/08/25/big-data-vendor-bluedata-announces-20m-round-and-intel-collaboration/?sr_share=facebook

帮助客户生成虚拟化大数据集群的公司 BlueData 今天宣布了 2000 万美元的 C 轮融资以及与英特尔的合作伙伴关系。

英特尔投资领投了这一轮。参与的还有 Amplify Partners、Atlantic Bridge、Ignition Partners，以及一位选择匿名的战略投资者。作为交易的一部分，英特尔高级副总裁兼软件和服务集团总经理道格·费舍尔将加入 BlueData 的董事会。

BlueData 的创始人都来自 VMware。BlueData 的目标是让 Hadoop 集群像虚拟机一样简单，公司创始人兼首席执行官 Kumar Sreekanti 告诉 TechCrunch。

“企业需要高效、敏捷地实施其内部大数据解决方案。速度和创新在大数据分析中至关重要，基础设施不能减慢这一速度。”斯里坎蒂在博客文章中写道。

该产品本质上是 Hadoop 或 Spark 即服务，但它不是驻留在云中，而是驻留在客户硬件上。它结合了云的灵活性、速度和多租户，以及在数据中心的裸机上运行的性能。

更重要的是，它可以与任何版本的 Hadoop 或 Apache Spark 兼容。该公司还计划在即将发布的版本中支持 Docker 容器。

与英特尔的交易将涉及几个层面，包括两家公司的工程师一起工作。BlueData 也将提前获得新的硬件版本。这个想法是 BlueData 将致力于在英特尔架构上优化其产品，特别是英特尔至强芯片。

两家公司还将在联合营销和销售项目上合作，英特尔将为 BlueData 提供其渴望的大型企业客户的入口。

对于 BlueData 这样目前拥有 35 名员工的小公司来说，这种合作关系有可能成为巨大的推动力。该协议是英特尔支持大数据公司持续努力的一部分，紧随去年与 BlueData 合作伙伴 Cloudera 的类似协议。

这并不是英特尔第一次对 BlueData 表现出兴趣。也促成了公司的 A 轮和 B 轮融资。

英特尔本周一直在传播财富。就在昨天，它宣布了一项大数据计划。作为其中的一部分，向 OpenStack 公司 Mirantis 投入了大量资金和资源。

自 2012 年推出以来，BlueData 已经在三轮(包括今天的公告)中筹集了相当适度的 3900 万美元——这次它并不指望获得大笔资金。

“我们并不想要一大笔钱。现在还为时过早，”Sreekanti 说。

Sreekanti 不愿透露它有多少客户，但在公司网站上列出了几家，包括 Comcast、Orange 和 Symantec。
