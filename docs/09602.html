任天堂 Switch 是 Nvidia SHIELD tech crunch 的精神继承者

原文：https://web.archive.org/web/https://techcrunch.com/2016/10/21/nintendo-switch-is-the-spiritual-successor-to-the-nvidia-shield/?sr_share=facebook

任天堂展示了它的新游戏机,开关(在公开上市前代号为 NX)。一家大型游戏制造商之所以将 Switch 与其他任何游戏机区别开来，是因为它能够从家庭游戏机快速过渡到便携式游戏机，这要归功于主要游戏硬件的平板电脑形式和模块化的“Joy-Con”控制器设计。

交换机是一种可行的产品吗？嗯，在 2017 年 3 月发布之前，我们不会了解全部情况，但早期迹象是好的——似乎有很多人对媒体报道发布会和粉丝的社交媒体反应感兴趣。Switch 也有一些前身，我们可以从中寻找使用它们的迹象，并获得这些迹象，这些迹象提供了很多振作起来的理由。

该交换机使用了一款定制设计的 Nvidia Tegra 处理器，这是该公司用来为自己的 SHIELD 系列移动游戏硬件供电的功能强大的移动处理器。SHIELD 设备一直使用安卓作为操作系统，并专注于移动优先或个人电脑(和一些控制台)端口到移动平台的游戏体验，但 SHIELD 产品系列和 Switch 之间分享了在家或在旅途中同样舒适的设备的理念。

关于游戏玩家想要什么样的混合家庭/户外游戏设备，Nvidia 已经努力学习了很多课程，SHIELD 系列的各种变化反映了这一点，包括最初的 SHIELD 便携版，本质上是一款移植到硬件控制器上的 Android 手机(没有手机功能)，在概念上最接近 Switch 的 SHIELD 平板电脑，以及仍然是 Android 电视的杰出明星的 SHIELD 机顶盒。

在整个发展过程中，SHIELD 一直执行良好，并兑现了一款设备的承诺，这款设备可以填补个人电脑和更多寻求移动体验的核心游戏机玩家在移动领域的空白。现在，任天堂 Switch 看起来准备从这些更利基的成功中吸取教训，并将其带给更多的受众。

Switch 无缝的客厅/便携式过渡看起来像是 SHIELD 试图实现的下一个合乎逻辑的步骤，控制台后面的 Tegra 技术看起来也更能实现真正的控制台级游戏体验，而不是升级的移动体验。此外，任天堂将比英伟达自己的硬件拥有巨大优势，这要归功于它的生态系统和构建自上而下用户体验的经验，尤其是在整个操作系统中专注于游戏。

需要说明的是，Nvidia 和任天堂之间的关系似乎是一种非常直接的供应商/被供应商安排——但 SHIELD 一直让人觉得是一种更伟大的承诺，任天堂对混合移动游戏愿景的具体化可能正是如此。
