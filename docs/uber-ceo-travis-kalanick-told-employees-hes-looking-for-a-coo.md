# 优步想雇一个首席运营官

# 优步首席执行官特拉维斯·卡兰尼克告诉员工，他正在寻找一个首席运营官

优步首席执行官特拉维斯·卡兰尼克今天上午通知员工，公司正在积极寻找首席运营官。此前[有报道称，卡兰尼克计划任命一名二把手高管](https://web.archive.org/web/20221209131653/https://www.theinformation.com/uber-ceo-facing-multiple-controversies-seeks-no-2-exec?shared=c507b1)。

“今天早上，我告诉优步团队，我们正在积极寻找首席运营官:一个可以与我一起谱写我们旅程的下一章的同行，”卡兰尼克今天在优步的博客上写道。

优步的 2017 年迄今一直动荡不安。它充斥着性骚扰和性别歧视的指控，[两名](https://web.archive.org/web/20221209131653/https://beta.techcrunch.com/2017/03/03/ubers-vp-of-product-and-growth-has-left-the-company/) [高管离开](https://web.archive.org/web/20221209131653/https://beta.techcrunch.com/2017/02/27/ubers-svp-amit-singhal-leaves-company-because-he-didnt-disclose-a-sexual-harassment-allegation/)公司，以及卡兰尼克因车费问题与一名优步司机发生口角的泄露视频。卡兰尼克为他在视频中的行为道歉，[承认他需要领导的帮助](https://web.archive.org/web/20221209131653/https://beta.techcrunch.com/2017/02/28/travis-kalanick-apologizes-for-blowing-up-at-uber-driver-who-complained-about-drop-in-pay/)，并且他打算得到帮助。看起来这正是他正在做的。
