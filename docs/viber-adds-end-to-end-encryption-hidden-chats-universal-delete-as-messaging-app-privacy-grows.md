# 随着消息应用隐私浪潮的增长，Viber 增加了端到端加密和隐藏聊天功能

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/04/19/viber-adds-end-to-end-encryption-hidden-chats-universal-delete-as-messaging-app-privacy-grows/>

继 WhatsApp 在其平台上添加[端到端加密](https://web.archive.org/web/20230326022647/https://techcrunch.com/2016/04/05/whatsapp-completes-end-to-end-encryption-rollout/)之后，又一家大型消息公司加入了应用程序扩展隐私功能的浪潮。[Viber](https://web.archive.org/web/20230326022647/http://www.viber.com/)——一款拥有 7.11 亿多用户的消息应用——今天推出了对其平台上所有消息和通话的端到端加密，包括群聊(你可以与多达 200 人聊天)，以及在现有的扩展删除功能之外“隐藏”你账户上的聊天内容的方法。

该公司成立于以色列，T4 于 2014 年被日本乐天收购。该公司表示，新服务将在未来几周内在全球范围内推出，从今天开始在 Viber 研发中心所在的四个国家推出:巴西、白俄罗斯、以色列和泰国。

新的隐私功能将在 Android、iOS、PC 和 Mac 桌面上运行，加密来自最新的应用更新 (6.0)和应用的重新认证(通过二维码)以打开该功能。

今天，几乎每个[大型消息应用](https://web.archive.org/web/20230326022647/https://www.eff.org/secure-messaging-scorecard)都在传输中对你的消息进行加密，但更安全的功能，如端到端加密，意味着服务提供商根本无法在任何时候访问消息。Viber 的首席运营官迈克尔·什米洛夫(Michael Shmilov)告诉 TechCrunch，该公司多年来一直在研究端到端加密功能

因此，Viber 直到现在才推出它——只是逐步推出而不是全面推出，并且仅在 WhatsApp 的新闻发布后几周——这既是一个时代的标志(这些天来用户越来越想要和要求的东西,)也是一个标志，表明今天的消息应用领域仍然竞争激烈，每个人都在确保没有竞争对手拥有比它更多的功能。(更多信息请见下文。)

该公司表示，Viber 的加密将具有不同的安全级别，这将在屏幕右侧以彩色编码锁的形式出现。

灰色锁表示对话(语音或文本)完全加密。你可以轻敲锁来确认这一点。

用户还可以选择绿色锁，这意味着您可以在每次对话之前对用户进行额外的身份验证，或者持续监控用户以确保用户得到验证。如果您正在与可能与其他人(比如孩子或伴侣)共享一个帐户的人交谈，此选项非常有用。

与此同时，如果受信任的用户没有通过他/她那端的重新认证，就会出现一个红色的锁。Viber 指出，这可能是因为设备发生了变化，或者因为有人试图通过中间人攻击来访问数据，这意味着用户需要重新验证安全和加密才能重新生效。

除了加密，最新版本的应用程序还增加了一些其他隐私功能。隐藏聊天将使用户能够从他们的使用日志中“隐藏”某些对话，只有当你知道一个指定的四位数 PIN 时才能访问。为什么要在应用程序中隐藏聊天内容？

同样，这又回到了多人可能访问你的设备，并随后访问你的 Viber 账户的想法。一种情况可能是你的孩子在你的 iPhone 上玩游戏，然后切换到消息传递。但另一种可能是对其他人隐藏更露骨的东西。顾客小心上当。

与此同时，增强的删除功能已经在应用程序中存在了一段时间，但它是该公司的一部分，是用户不仅可以在自己这边，还可以在收件人的手机上删除对话的一种方式。你可以把这看作是 Viber 对短暂消息传递的回应，但是采用了更加人工的方法。

## 即将到来的机器人和企业服务

添加更多你在竞争对手服务中看到的功能的想法是目前所有消息应用的一个大主题，这些应用都在竞争成为消费者的首选，这些消费者正在平衡大量应用，但最终只使用少数有规律的应用。

在这个主题上，什米洛夫说，Viber 将在未来几周和几个月内增加更多的功能，以保持与 Messenger 和其他人一样的水平，将他们的服务从围墙花园转变为更广泛的平台。这将包括一项面向企业的服务，在这项服务中，公司将能够使用 Viber 与他们的客户进行交流，作为他们 CRM 计划的一部分；同时也是一个支持的平台—没错！Viber 上的机器人。这是在 Viber 扩展了许多其他服务之后出现的，例如[公共和群聊](https://web.archive.org/web/20230326022647/https://techcrunch.com/2015/07/28/messaging-app-viber-buys-nextpeer-to-add-social-features-to-its-gaming-portal/)，以及[游戏](https://web.archive.org/web/20230326022647/https://techcrunch.com/2015/02/25/viber-games-worldwide/)。

Shmilov 说，企业服务——听起来很像 WhatsApp 的[商业账户功能——将首先出现，这将是 Rakuen 希望与其所有者 Rakuten 整合的关键方式之一，其市场上的商家将能够恢复到 Viber，以便在购买前或售后支持方面与客户进行对话。“我们正在与乐天合作，”他说，“与在平台上销售的商家互动。几个月后，我们将与合作伙伴一起推出这一产品，与他们的 CRM 软件合作，但也直接与公司合作。”](https://web.archive.org/web/20230326022647/https://techcrunch.com/2016/01/18/whatsapp-free-business/)

虽然 WhatsApp 和 Twitter 等其他公司也希望通过为客户服务的企业账户来扩展他们的服务，但这与其说是一个效仿的功能，不如说是一个很有意义的功能:在这种情况下，如果你是一家企业，你不会希望为了互动而要求你的客户下载一个新的消息应用程序；您希望在客户已经在使用的平台上进行对话。从这个意义上说，聪明的公司将使在所有主要平台上发言成为可能——就像出版商给用户一系列小工具，让他们在各种社交网络上分享故事。

至于机器人，这些听起来非常类似于[脸书现在在 Messenger](https://web.archive.org/web/20230326022647/https://techcrunch.com/2016/04/12/agents-on-messenger/) (带有[打嗝](https://web.archive.org/web/20230326022647/https://techcrunch.com/2016/04/13/facebooks-new-chatbots-still-need-work/))中提供的东西，它们很可能在第三季度开始走向市场。虽然脸书的 Messenger 基于大量机器学习和人工智能，这些都是该公司通过收购 Wit.ai 而在内部构建的，但看起来 Viber 选择了一种 API 来运行机器人，但这种方式可能会整合其他助手，而不是在内部构建技术。

“有很多基于人工智能的解决方案，如谷歌和微软的 Cortana，”什米洛夫说。“我们正在开发一个强大的 API，它将在 Viber 的基础上实现基于 bot 的解决方案。开发者将能够使用这些 API 来集成他们想要的东西。已经有一些有趣的场景出现了，”他谈到该公司正在进行的内部测试。"这仅仅是信息传递工作的开始."
