微软收购移动商务智能服务Datazen TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/04/14/microsoft-acquires-datazen/?ncid=rss

微软今天宣布已经收购了 Datazen ，一家移动商务智能和数据可视化服务公司。

大约三年前推出的Datazen允许企业从Microsoft Excel中的数据以及其他云和企业数据库来源创建移动仪表板。要运行Datazen，企业需要运行Windows Server、IIS和。以及Datazen的服务器软件。

该公司提供Windows、Windows Phone、iOS、Android和网络应用。根据今天的公告，其所有产品将暂时保持可用，该团队将继续致力于Datazen的服务器和客户端产品。

从今天开始，微软SQL Server企业版客户(2008版及以上)已经可以免费下载Datazen的服务器软件。

Datazen团队今天写道:“Datazen的定位是为需要内部实施解决方案的客户提供移动BI解决方案。“因此，它将补充微软基于云的商业分析服务Power BI。”

微软可能会将Datazen的技术和专业知识整合到其Power BI产品中，但目前尚不清楚这一整合的路线图是什么样的。如果微软最近的收购是某种迹象的话，那么该公司有可能会将Datazen的一些移动应用程序重新命名为iOS或Android的Power BI。

微软云和企业主管Kamal Hathi在今天的公告中写道:“此次收购加速了我们的Power BI战略，以帮助组织创建一种数据文化，利用易于使用、易于访问的工具从任何地方、任何设备上的数据中提取最大价值。”他还指出，SQL Server客户“喜欢Datazen，因为它针对SQL Server Analysis Services和整个微软平台进行了优化[…]”

随着企业创建的数据量持续快速增长，微软最近将相当多的注意力放在了商业智能平台上。今天的收购正好符合这一趋势。
