# Udacity 计划打造自己的开源自动驾驶汽车

据 Thrun 称，巴斯蒂安·特龙在线教育初创公司 Udacity 最近创建了一个自动驾驶汽车工程 nanodegree，并在 Disrupt today Thrun 的舞台上透露，该公司打算建造自己的自动驾驶汽车作为该计划的一部分，并且还打算开源该技术，以便“任何人”都可以尝试建造自己的自动驾驶汽车。

众包车辆计划最终将为学校服务，而不是一个产品。数据的开源应该有助于其他项目的发展，并将包括驱动数据和更多内容，以促进其他人的项目。

Thrun 还表示，无人驾驶汽车项目将像谷歌一样悄悄地开始测试，并在出现问题时，将使用训练有素的安全驾驶员。

其他公司正在开源自动驾驶技术，包括 Comma.ai，该公司发布了一个驾驶数据集，用于创建其第一个功能性自动驾驶车辆附加原型，但听起来 Thrun 的计划是开源整个东西，因为它不是 Udacity 业务的核心。

Thrun 还透露了其自动驾驶纳米学位项目的官方合作伙伴数量[，包括梅赛德斯等。](https://web.archive.org/web/20221007202554/https://beta.techcrunch.com/2016/09/13/mercedes-benz-partners-with-udacity-on-an-autonomous-driving-nanodegree/)
