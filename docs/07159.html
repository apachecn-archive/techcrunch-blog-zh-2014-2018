Telegram增强其bot平台| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/04/12/telegram-beefs-up-its-bot-platform/

今天晚些时候，在f8开发者大会上，脸书有望在其Messenger平台上发布一项机器人大推进计划。如此令人惊讶的是，竞争对手消息平台Telegram借此机会提醒人们，早在2015年6月就推出了自己的机器人平台——今天宣布了该平台的 2.0更新，这是自推出以来最大的更新。

什么是机器人？在这里，该术语指的是位于消息平台上的自动聊天机器人，允许用户通过熟悉的聊天界面与第三方软件程序进行交互，以轻松执行某些操作——更重要的是，无需离开消息平台即可完成这些操作——比如搜索特定的URL以在聊天中分享，或者询问关于天气或新闻的信息。例如，TechCrunch在Telegram 上有自己的新闻机器人。)用户只需要ping正在讨论的机器人，就可以接入它所插入的外部服务。

许多机器人可以有效地被认为是一种替代搜索机制，嵌入到现有平台中，并利用特定数据的缓存——取代用户去执行一系列外部搜索以检索他们想要的任何第三方数据的需要。

Telegram的 bot 2.0 更新带来了一系列新功能，以构建其bot平台的效用——包括开发人员可以构建能够发送Telegram上支持的任何类型内容的bot，如文档、MP3、视频、贴纸、动画和联系人；支持基于位置的服务；以及基于用户电话号码与其他服务深度整合的能力。

它还增加了内嵌键盘，可以根据特定机器人的信息进行构建——因此用户可以点击预先确定的按钮来响应机器人，而不需要键入特定的指令。

Telegram以前支持机器人提供定制回复键盘的能力，例如其贴纸评级机器人提供1星、2星等按钮，而内嵌键盘则更进一步——不发送消息到聊天，但支持后台按钮，如回拨按钮和URL按钮。该平台现在还允许机器人编辑自己的信息。

Telegram创始人帕维尔·杜罗夫解释说:“由于消息编辑和带回调功能的嵌入式键盘，开发人员现在可以在他们的机器人中构建交互式用户界面，而不必让用户在聊天中看到多条很快过时的消息(我们以前有时会看到这些消息)。”

Telegram构建了四个样本机器人来展示新功能: @music ，一个用于搜索和分享古典音乐的机器人； @youtube ，搜索分享youtube视频的机器人； @foursquare ，一个寻找附近餐馆和地方与朋友分享地址的机器人；和 @sticker ，一个使用表情符号作为贴纸搜索起点的寻找贴纸的机器人。(所以，如果你像我一样，发现自己过度使用了笑的眼泪表情符号，这是一种快速找到替代品的方法……)Telegram用户需要更新到最新版本的应用程序，才能与新的机器人互动。

“这些样本机器人只是开始。我们的Bot API 2.0允许开发人员为强大的Bot创建流畅且易于使用的界面。我们将在未来几个月看到许多新的机器人，毕竟，2016年是机器人之年，”它在宣布更新的博客帖子中补充道。

Telegram上的机器人有多受欢迎？这家初创公司声称，大量用户现在经常与机器人聊天。“去年机器人的采用主要是由高级用户推动的，但现在机器人已经成为主流。Telegram每月有超过20%的用户观看机器人生成的内容。

早在2月份，这款即时通讯应用宣布其平台的月活跃用户已达1亿，并表示每天增加35万新用户，每天产生约150亿封邮件。很明显，现在大约有2000万Telegram用户每月观看机器人生成的内容。

这家初创公司如何看待脸书推出自己的机器人大行动？“我们很高兴其他平台正在迎头赶上，因为这对生态系统有好处，”它表示。

同样在今天，Telegram宣布对应用程序中的分享和预览进行调整，包括一个新的低摩擦视频播放器(点击更少的按钮开始观看内容)；更容易使用贴纸包预览；支持在iOS上对gif和bot内容进行点击和保持预览；以及Telegram Android应用程序的重大重新设计。
