# 据报道，优步竞争对手 Grab 正从软银和滴滴融资 20 亿美元

Grab 是优步在东南亚的竞争对手，在七个国家开展业务，据报道，它正在与总部位于日本的软银和总部位于中国的滴滴出行进行谈判，以筹集高达 20 亿美元的资金，[华尔街日报报道，援引消息来源](https://web.archive.org/web/20221025222818/https://www.wsj.com/articles/uber-rival-grab-in-talks-for-up-to-2-billion-from-softbank-chinas-didi-1500032435)。据报道，这笔资金对 Grab 的估值超过 50 亿美元，这将使其成为东南亚最有价值的初创公司。

软银是电信巨头，因投资除优步以外的任何打车公司而闻名。软银在 2014 年首次投资 Grab，投资金额为[2.5 亿美元。](https://web.archive.org/web/20221025222818/https://beta.techcrunch.com/2014/12/03/softbank-invests-250m-in-grabtaxi-ubers-archrival-in-southeast-asia/)这笔投资是在软银[向印度的 Ola 打车初创公司](https://web.archive.org/web/20221025222818/https://beta.techcrunch.com/2014/10/27/olacabs-softbank-india/)投资 2.1 亿美元几个月后进行的。软银还投资了优步在巴西的竞争对手 99。

国际打车市场正在升温。去年，优步将其中国子公司出售给滴滴，本周早些时候，优步表示将与 Yandex 合并。滑行。

随着优步继续被美国的众多丑闻、诉讼和高管离职所困扰，或许国际市场上的公司将此视为一个开发天然气的机会。

Grab 发言人拒绝对任何“谣言或猜测”发表评论。我已经联系了滴滴和软银，如果有回音，我会更新这个故事。
