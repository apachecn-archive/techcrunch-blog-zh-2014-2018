# (某些)物联网 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/05/19/the-internet-of-some-things/>

**编者按:** *[比尔·布里格斯](https://web.archive.org/web/20221209104029/http://www2.deloitte.com/us/en/profiles/wbriggs.html)是德勤咨询公司的 CTO。*

2015 年，现代淘金热席卷了科技行业。最近的一项研究发现，75%接受调查的高管表示，物联网(IoT)计划目前正在他们的组织内进行。也就是说，仍然有一个问题许多公司没有回答——更多的数据总是一件好事吗？

不幸的是，无数组织可能永远也不会挖掘到数据驱动的洞察力的母矿，这种洞察力可能会改变他们的业务。从装配线到送货卡车，再到收发室的邮资计算器，各种设备中都嵌入了传感器，这些传感器产生的万亿字节的数据可能会淹没公司。而且不仅仅是来自他们自己的资产——还来自他们的合作伙伴、周围环境，甚至来自行为和偏好日益量化的客户、潜在客户和员工的“数字废气”。

面对如此大量的数据涌入，许多公司都不知道是什么袭击了他们。战略应该关注有限的场景——在这些场景中，一小群智能或互联设备可以用新的方式解决旧问题，或者解决只有在物联网兴起的情况下才能实现的全新问题。

解决方案不在于*万物*的互联网，而在于*某些**事物的互联网，这可能包括对新的“智能”资产的全新投资。或者更有可能的是，利用已经闲置的现有传感器网络和具有嵌入式智能的设备。德勤预测，2015 年，无线物联网设备的出货量将达到 10 亿台，比 2014 年增长 60%，装机量将达到 28 亿台。*

 *对于希望开始为未来世界做准备的公司，请考虑以下三点:

## 小心“连接”

与流行观点相反，并不是每个设备、物体或人都需要支持 Wi-Fi 或嵌入其他时髦的小玩意。随着我们的连接，我们也提出了潜在的安全和隐私问题，特别是在处理物理安全有限的远程控制资产以防止不适当的访问时。

保持安全、警惕和弹性——不仅仅是设备本身，还有设备之间代表它们如何通信、决策和采取行动的接缝。网络应该是一门从构思到推广的学科，有助于达到可接受的风险点。

## 当心“大数据”效应

物联网工作可以借鉴围绕大数据的炒作周期。两者都建立在技术进步的基础上，而技术进步无疑推动了业务的发展。但是，由于范围和潜在影响的巨大，在试图解决大问题时，进展可能会停滞不前。

大多数组织不需要大规模的统一物联网理论——绝对不需要，可能永远也不需要。相反，关注小球:有目的的具体问题。

## 把你的眼睛放在终点线上

没有可衡量的结果，战略是没有意义的。在开始收集更多数据和更多见解的任务之前，请确保您所促成的事情能够与改进的业务流程或影响明确联系起来。

在今年的【2015 年技术趋势中，我们讨论了物联网如何超越预定义的工作流、控制脚本和操作程序。

它需要具备以下能力:

*   利用专有数据和技术，整合来自众多全球制造商的不同类型设备之间的信息流。
*   对物理对象和低级别事件进行分析和管理，以检测信号并预测影响。
*   编排这些信号和对象，以完成复杂的事件或端到端的业务流程。
*   保护和监控整个系统的设备、连接和信息交换。

有了这些功能，公司可以将物联网从简单的信息收集提升到使用一系列设备和信号为企业做点什么。它将关注点从新奇的互联和智能对象转移到业务流程和模型转换。

也许更重要的是，它可以帮助战略家、it 项目负责人和执行决策者完善他们对物联网的方法和理解，以创造更大的价值。正如许多目前正在实施物联网计划的组织所发现的那样，目标不应该是创造万物互联；应该是构建一个“某些事物的网络”经过刻意的选择和有目的的部署。

物联网淘金热继续快速发展，由机器驱动，这些机器将我们对世界的认识与人类对有意识地观察和记录正在发生的事情的依赖相分离。但是机器自动化只是搭建了舞台。真正的影响，无论是商业还是公民，将来自于数据和相关传感器、事物和人的结合，这样生活可以过得更好，工作可以以不同的方式进行，竞争规则可以重新构建。*
