Tomu 是一个指甲大小的计算机，易于吞咽

我是单板电脑的超级粉丝，尤其是如果它们小到可以吞咽的话。这就是我喜欢 Tomu 的原因。这个非常小的 ARM 处理器本质上是通过 USB 端口与您的计算机接口，包含两个 led 和两个按钮。一旦插上电源，这台小电脑就可以模拟硬盘或鼠标，发送 MIDI 数据，甚至快速闪烁。

Tomu 运行 Silicon Labs Happy Gecko EFM 32hg 309，还可以充当通用第二因子安全令牌。它是完全开源的，所有代码都在他们的 GitHub 上。

我花 30 美元买了一个，摆弄了几个小时。这些程序非常简单，你可以加载各种工具，包括一个聪明的小鼠标移动器——也许是为了模拟应用程序的鼠标使用——和一个快速闪烁灯光的小应用程序。否则，你可以用它把你的 USB 集线器变成电脑的开关。它肯定不是一台完全成熟的计算机——显然，它的 I/O 选项有限——但对于那些想做一点开源计算的人来说，它是一个可爱的小工具。

一个问题？真的真的很小。我会做更多的工作，但我已经失去了它，当我清理一张桌子，以便我能更好地看到它。事情是这样的。
