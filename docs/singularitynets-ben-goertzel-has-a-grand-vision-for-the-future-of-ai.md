# 奇点网的 Ben Goertzel 对人工智能的未来有一个宏伟的愿景

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/12/22/singularitynets-ben-goertzel-has-a-grand-vision-for-the-future-of-ai/>

奇点网(SingularityNET)是一个雄心勃勃的项目，旨在为人工智能创造一个去中心化的市场，该项目通过代币销售筹集了大量资金。在向公众开放销售后的大约 60 秒钟内，所有可用的代币(AGI 代币)销售一空，使总筹款达到 3600 万美元。

然而，在今天这个时代，一家在 ICO 中筹集大量资金的初创公司并不真正令人感兴趣，至少对许多人来说是这样。这是当今疯狂的、不受监管的、秘密的世界的一部分。但是*有趣的是*奇点网实际上计划成为什么样子。

[首席执行官
和创始人【Ben Goertzel 博士有一个宏伟的愿景。](https://web.archive.org/web/20230404124004/https://en.wikipedia.org/wiki/Ben_Goertzel)

奇点网将人工智能和区块链结合在一起，为人工智能创造了一个去中心化的开放市场。这意味着它可以让任何人将人工智能货币化，允许公司、组织和开发者大规模买卖人工智能算法，从而降低成本并增加人工智能的能力。最终，该计划是将汉森机器人公司的“索菲亚”机器人真正接入奇点网的网络，为其大脑提供动力。如果这听起来像一部科幻电影，那是因为很多科幻电影都有这样的情节。

在一次采访中，我问他这一切将如何运作:

“专有市场是存在的，比如亚马逊网络服务。我们在这里创造的是一个去中心化的市场，更像 BitTorrent。没有中央独裁者决定什么进入那里。任何人都可以将人工智能放在网上，包装在我们的 API 中，向网络公布，任何需要人工智能服务的企业都可以请求它。

“那么你就需要一个好的信誉体系，给最好的 ai 定一个高评级。我们需要区块链让我们以对等的、分散的方式来做这件事。P2P 软件是可靠的，不容易被黑客攻击，并涉及支付，但它仍然需要一个分布式账本。BitTorrent 不涉及支付或身份管理或高安全性，这就是为什么我们在区块链这么做。

“人工智能处理对计算时间很敏感。所以我们选择了 OpenCog 平台，它有两个作用。一个是，它是我们将用来在网络中建立一些初始人工智能服务的许多人工智能工具之一，就像苹果公司出售自己的应用程序一样。我们会在那里放一些很酷的工具。另一个角色更接近基础设施。

“我们希望让系统的人工智能层独立于我们正在使用的区块链。我们的原型使用以太坊，但这太慢了。所以我们希望能够轻松替换掉区块链技术。因此，智能合约需要以一种抽象的方式来表达，这种方式独立于我们正在使用的区块链。而 OpenCog 的逻辑语言很好地解决了这个问题。

“如果市场上的人工智能没有用，它的评级就会降低。但在这种市场中，有些东西不一定对每个人都有用。例如，你可以有一个只对非洲咖啡农有用的算法。我的意思是，亚马逊云不提供这种服务，因为咖啡农在它要解决的问题列表中排名不高。但对某些人来说还是有价值的。比如咖啡农！

“这就像拥有 Elance，但人工智能是在做一项服务，而不是一个人。

它可以扩展吗？无论如何，人工智能算法不是大科技公司发明的，而是学者和学生发明的。因此，如果你有办法让发明者将他们的算法包装在一个像样的 API 中，那么你就可以让创造者受益，而不是让他们必须创建一个初创公司并将其出售给一家大公司。

“他们可以不把代码放在 GitHub 上，而是放在 SingularityNet 上，企业可以找到这些代码，然后他们可以从中获得报酬。在最终用户方面，我们需要让企业养成将他们的代码连接到 SingularityNet APIs 的习惯，而不是亚马逊的 API 或初创公司，后者有一个令人讨厌的习惯，即当初创公司被收购时就会消失。

“我自己的动机很大程度上是为了在像汉森机器人这样的各种软件背后获得更高水平的人工智能，将其作为奇点网的展示平台。随着这变得更加智能，我们将把 Sofia 的大脑指向奇点网络，将它从 Hanson 云移到奇点网络，例如使用计算机视觉。到明年 8 月，我们应该会有一些实质性的改进。我们受益于世界上任何人改善索菲亚思想的能力。我们不能雇佣所有这些爱好者，但我们可以给他们一个简单的方法来帮助索菲亚。

“例如，一个孩子可以通过将他们的计算机视觉人工智能投入我们的市场来赚取数百万美元。”

这是一个非常非常大的愿景。我只能说，小心亚马逊…
