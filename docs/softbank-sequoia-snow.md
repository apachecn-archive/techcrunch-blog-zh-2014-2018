# 曾经是 Snapchat 克隆版的自拍应用 Snow 从软银和红杉中国融资 5000 万美元

我们已经有一段时间没有听到 Snow 的消息了，Snow 是亚洲的 Snapchat 克隆应用程序，[脸书曾经试图收购](https://web.archive.org/web/20230330164052/https://techcrunch.com/2016/10/31/facebook-bid-for-snow/)，但今天它背后的公司已经从软银和红杉中国获得了 5000 万美元的投资。

Snow 是由 Naver 创立的，这家韩国公司开发了流行的消息应用 Line，由于专注于本地化的过滤器、贴纸和功能，它在日本、韩国、中国和亚洲其他市场都很受欢迎。更不用说 Snapchat 在亚洲市场有名的不努力。

然而，自从我们上一次写它以来，Snow 应用程序已经有了显著的变化。它不再是快照克隆。

上周取消的一项重大更新删除了 Snow 的用户对用户通信功能，并将其变成了一个专用的自拍相机应用程序。在没有聊天的情况下，该应用程序在滤镜、贴纸、增强现实(AR)和其他自拍相关功能上加倍努力，以制作可以导出到社交网络或聊天组的照片和其他媒体。

例如，Snow 用户现在可以录制一段视频，配上包括查理·普斯在内的艺术家的音乐。有一系列常见的照片过滤器，还有一个 GIF 制作工具和类似 Instagram 的回旋镖功能。

Snow 计划利用这笔新投资开发其增强现实和面部识别技术。其应用商店列表显示，它正在与中国独角兽商汤科技合作进行面部识别。

该公司还计划在中国建立合作伙伴关系，并实现服务本地化。除了 Snow，Snow Corp 还拥有从 Line 收购的相机应用 Foodie 和 B612，因此它们也可能作为独立应用在中国推广，尽管它们背后的技术也与核心的 Snow 应用共享。

Snow 的一名代表告诉 TechCrunch，该应用程序在 iOS 和 Android 上的下载量现已超过 2 亿次。该公司没有公布每个市场的具体数据，但表示中国是其最大的市场。在 2017 年 1 月，[我们报道了 Snow 拥有 4000-5000 万月活跃用户](https://web.archive.org/web/20230330164052/https://techcrunch.com/2017/01/04/snow-the-asian-snapchat-clone-facebook-tried-to-buy-claims-40-50m-monthly-users/)，但目前没有 MAU 方面的进一步更新。

软银——这是软银集团，而不是愿景基金——红杉资本通过这笔交易收购了斯诺中国业务部门 20%的股份。 [Line 是 Snow 的其他支持者](https://web.archive.org/web/20230330164052/https://techcrunch.com/2016/09/29/line-investment-snow-45-million/)、[通过两笔投资](https://web.archive.org/web/20230330164052/https://techcrunch.com/2017/02/26/chat-app-line-doubles-its-stake-in-snapchat-clone-snow/)。
