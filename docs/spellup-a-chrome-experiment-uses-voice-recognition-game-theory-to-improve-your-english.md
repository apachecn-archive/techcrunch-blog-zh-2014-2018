# Chrome 的一个实验项目 Spell Up 利用语音识别和游戏来提高你的英语水平

对于在扩展[测试版](https://web.archive.org/web/20230316050112/http://www.slate.com/articles/news_and_politics/recycled/2009/07/why_did_it_take_google_so_long_to_take_gmail_out_of_beta.html)和[临时项目](https://web.archive.org/web/20230316050112/http://www.google.com/doodles/)中推出服务并不陌生，谷歌今日[公布了其最新的 Chrome“实验”](https://web.archive.org/web/20230316050112/http://googleblog.blogspot.ca/2014/05/speak-and-learn-with-spell-up-our.html):[拼写](https://web.archive.org/web/20230316050112/https://spellup.withgoogle.com/)，这是一款基于浏览器的游戏，使用语音识别和语音合成来帮助人们更好地学习英语拼写。

用户可以从他们想要的任何水平开始，重点是建立单词的“塔”(因此是“拼写”的“上”)。浏览器向用户朗读单词，用户必须将单词拼写回语音。围绕这一点有各种各样的变化，比如猜字游戏、解读单词、填补缺失的字母空白和正确发音(从我玩游戏的地方来看，“正确地”似乎是女王的英语)。

用户通过定义得到线索，你通过正确拼写建立信用，当你不知所措时，你可以用它来帮助自己。

这个项目是谷歌伦敦创意实验室的 Xavier Barrade 的创意，利用了 Chrome 在语音方面的一些最新进展。

去年，谷歌在 Chrome 上增加了对[网络语音 API](https://web.archive.org/web/20230316050112/https://techcrunch.com/2013/01/14/chrome-25-beta-now-supports-the-web-speech-api-will-let-developers-build-voice-commands-into-their-web-apps/) 的支持，今年它又在此基础上增加了[语音合成](https://web.archive.org/web/20230316050112/https://techcrunch.com/2014/01/16/chrome-33-beta-introduces-custom-elements-and-support-for-speech-synthesis-api/)。这两者结合在一起，让开发人员可以创建用户可以对输入数据说话的应用程序，而应用程序本身可以使用不断变化的数据库来制作口头回复——这两者都可以在拼写中使用。

换句话说，这是一个关于如何将漂亮的谷歌工具用于教育(和游戏)的有趣项目！)因为这是一种展示浏览器技术创新的方式，以及与本地应用环境相比，它已经走了多远。就其目标人群而言，这也是一个有趣的想法:年轻人，可能还有更多国际用户。

正如 Barrade 所描述的那样，Spell Up 是与游戏设计师和教师合作开发的——他们和开发者一起，已经形成了一种非正式的教育应用三巨头。它似乎主要是为桌面上的 Chrome 和安卓设备设计的。如果你在 iPhone 或 iPad 上使用它，语音部分就会消失，你需要输入你的答案。

目前，该应用程序看起来仅限于英语，但它是否会扩展到其他语言将是一件很酷的事情——在大多数传统的英语教育系统中，这往往是一个不足。

(在 Mac 笔记本电脑上)玩了一圈，我发现这款应用相当有趣，只是识别和发布我说出的字母有点慢，而且不止一次地，它似乎完全把一个字母错当成了另一个字母。但我有一种感觉，我的孩子会喜欢这个——一个上网的合法借口，一个我不介意他们使用的借口。
