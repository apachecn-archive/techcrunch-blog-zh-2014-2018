# YouTube 电视现在可以在 Firefox 上运行

YouTube 电视在一个不是由谷歌开发的浏览器中找到了生命。

在之前被 Chrome 锁定后，直播电视服务[现在在 Firefox 上运行](https://web.archive.org/web/20221208075408/https://support.google.com/youtubetv/answer/7129767?co=GENIE.Platform%3DDesktop&oco=1)。由[yourtech 首先发现的新发展解释说](https://web.archive.org/web/20221208075408/https://yourtechexplained.com/2018/04/05/youtube-tv-now-works-using-firefox-web-browser/)并不是一个巨大的冲击——谷歌曾承诺将支持其他浏览器，但很高兴看到他们信守承诺(尽管如果你是忠实的 Safari 或 Edge 用户，你仍然不太走运)。

每月 40 美元的流媒体服务让用户可以观看和录制 NBA 电视、MLB 网络、喜剧中心、MTV 和 CNN 等频道的内容。这仍然是一个有点昂贵的销售，但随着这项服务继续增加更多的网络，更多的功能和更多的观看选择，它慢慢成为一个更好的购买者，他们希望在自己的设备上获得直播电视。
