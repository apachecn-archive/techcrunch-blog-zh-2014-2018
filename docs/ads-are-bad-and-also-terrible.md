# 广告不好，也很糟糕 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/08/20/ads-are-bad-and-also-terrible/>

我时常发现自己被迫在没有 uBlock 或类似功能的浏览器上使用网络；每一次，我都在想“人们怎么会这样生活？”未被广告屏蔽的网络是一个悲惨的自动播放视频和歇斯底里的行动呼吁的粪坑，加载缓慢，难以查看。在你的手机上情况更糟，广告会消耗你的电池寿命和高达 75%的数据。

值得称赞的是，苹果支持手机 Safari 的广告拦截器。但是谷歌呢？是啊，没那么多。你可以运行单独的浏览器，比如[勇者](https://web.archive.org/web/20230129061928/https://brave.com/downloads.html)。你可以通过代理服务器运行，或者重置你的域名系统，使用类似于 [Optimal](https://web.archive.org/web/20230129061928/https://optimal.com/) 的广告拦截器。但你不能只给 Android 版 Chrome 添加一个广告拦截器。令人惊讶的是，一家拥有谷歌技术实力的公司还没有做到这一点。

…或者这并不令人惊讶。毕竟，谷歌的大部分收入来自展示和搜索广告。或许与此相关的是，苹果也让在 iOS 上更改 DNS 设置变得相当容易:无论是通过 Wi-Fi 设置还是通过编程方式，这在 Android 上都是一个更大的麻烦。

根本问题是用户讨厌广告，但网站需要广告费。广告拦截领域最有趣的举措是我上面提到的两个，勇敢和最优，因为这两个都试图解决这个根本问题。勇敢者用不会追踪或惹恼用户的“好广告”取代“坏广告”，着眼于用小额支付取代广告收入。 [Optimal](https://web.archive.org/web/20230129061928/http://optimal.com/our-product/) 希望用订阅费取代广告收入，订阅费在 Optimal 用户访问的所有网站上共享。

广告屏蔽已经成为一个足够大的问题，引起了脸书的注意:这个社交巨头现在正在[屏蔽广告拦截器](https://web.archive.org/web/20230129061928/http://newsroom.fb.com/news/2016/08/a-new-way-to-control-the-ads-you-see-on-facebook-and-an-update-on-ad-blocking/)，同时承诺让用户更好地看到广告。正如乔希·孔斯蒂内[所说](https://web.archive.org/web/20230129061928/https://techcrunch.com/2016/08/09/facebook-will-bypass-web-adblockers-but-offer-ad-targeting-opt-outs/)，“脸书认为如果它能让广告不间断、快速、安全，人们就不会介意。”

现在，这引起了很多有趣的怀疑:

事实上，一场军备竞赛[随之而来](https://web.archive.org/web/20230129061928/https://techcrunch.com/2016/08/11/friendblock/)。但我认为脸书在这一点上至少说对了一半。今天的广告生态系统面临两个基本问题:

1.  广告不好。
2.  广告太可怕了。

不良广告会耗尽你的电池和数据。他们是弹出式菜单和弹出式菜单。他们是追踪者和超级警察。他们是 Outbrain 和 Taboola clickbait，在其他体面媒体发表的文章底部。

但是即使是不坏的广告仍然很糟糕。以这种或那种方式通过广告获得报酬的人，最终不可避免地合理化广告在某种程度上对人们有益，只要广告符合他们的人口统计和兴趣，他们就喜欢看，等等，令人作呕。

我必须强调这不是真的。人们*讨厌*广告；只是程度问题。我们讨厌商业广告。我们讨厌展示广告。我们讨厌广告牌。我们和他们生活在一起，因为我们别无选择，但是我们讨厌他们——广告拦截器告诉我们，在网上，我们有其他选择。我们根本不用看他们。用“没那么糟糕的广告”代替“糟糕的广告”并没有让广告的基本概念变得更容易接受。

当然，这条规则也有罕见的例外。电影预告片是广告，但我们并不讨厌它们——恰恰相反。我们并不讨厌搜索广告，因为它们只在我们主动寻找某样东西的时候出现:这是谷歌是一台巨大的赚钱机器的一个主要原因。同样，出于同样的原因，我们也不讨厌亚马逊上“买了这个的人也买了”的广告。

(即使是原本很吸引人的广告——比如苹果公司华丽的“iPhone 6S 拍照”海报——也会变得令人讨厌；不是因为它们本身，而是因为它们是不断积累的广告背景的一部分，是我们整个公共领域不断交易化的一部分。没有一个广告客户有任何削减的动机，所以广告的整体压垮了我们，以至于我们在某种程度上对它感到麻木和厌恶。称之为文化共有的悲剧吧。这正是脸书想要阻止的——但是尽管其初衷是好的，我担心这口井已经被其他地方的广告毒害了。)

清除不良在线广告会让人们忘记广告是可怕的，并退回到我们习得的无助状态吗？我希望不会。我认为，基于订阅的自动小额支付解决方案是一种更好的方式，Brave 和 Optimal 正在朝这个方向努力。一方面，我对我们能很快从当前的世界到达那个更好的世界并不乐观；但另一方面，从长远来看，做你的客户讨厌的事情永远是一个糟糕的策略。

现在这是一个有争议的问题。从短期来看，重要的是苹果、Brave、Optimal、脸书和许多其他公司都在反复强调“不良广告”越来越不可接受。我希望，在线广告生态系统正走向毁灭性的崩溃。继续使用不良广告的组织最终将会为此遭受损失，有些迟早会死掉。希望能快点。
