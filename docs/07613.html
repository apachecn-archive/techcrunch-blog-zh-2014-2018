Tally 为应用程序筹集了 1500 万美元，使信用卡更便宜、更易于管理| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2016/05/19/tally-raises-15-million-for-app-to-make-credit-cards-less-expensive-easier-to-manage/

总部位于旧金山的 Tally Technologies 在首轮风险投资中筹集了 1500 万美元，用于推出一款应用程序，该应用程序承诺帮助人们保持良好的信用，同时避免费用、收费和其他与信用卡相关的痛苦。沙斯塔风险投资领投首轮融资，该公司的早期支持者牛仔风险投资和 AITV 也加入了进来。硅谷银行也投资了。

根据 Tally 首席执行官兼联合创始人杰森·布朗(Jason Brown)的说法，该公司已经秘密运营了一年多，并在测试客户中测试了大约三个月的服务。

Brown 解释说，Tally 应用程序是这样工作的:用户可以将他们所有的个人信用卡扫描到 Tally 中，进行简短的信用评分检查，然后授权初创公司从 Tally 发放的信用额度中支付这些账单。

“大多数美国成年人都有多张个人信用卡，”布朗说，“平均每人 3.7 张。”至少 10 个美国家庭中有 4 个家庭的信用卡上有余额，产生滞纳金并支付利息。

这家拥有八名员工的初创公司帮助客户避免所有这些滞纳金和其他费用，并为他们提供低于所有信用卡平均年利率的年利率。如果客户没有付清他们的全部理货余额，该公司在较低的年利率上赚钱，但他们再次承诺，该金额将低于客户向其他银行支付的金额。

早些时候，Cowboy Ventures 领投了该公司 200 万美元的种子资金。牛仔风险投资公司的艾琳·李和沙斯塔风险投资公司的董事总经理肖恩·弗林是 Tally 的董事。弗林表示，他预计这家初创公司将利用其 A 轮融资来使其服务广为人知，并通过分阶段推出来吸引用户。

Tally Technologies Inc .联合创始人(L-R)贾斯珀·普拉茨和杰森·布朗。

Tally 还需要筹集机构资本，以提供信贷额度，因此它需要证明其客户具有良好的信用风险，并在金融业发展合作伙伴关系。

作为一家公司，Shasta 避免投资另类或点对点贷款平台，如 Lending Club 和 Prosper，这些平台现在正受到审查，面临严重的增长挑战。

弗林表示，Tally 似乎不同于其他非银行金融服务公司，因为这家初创公司的服务对人们有着持续而广泛的吸引力，而不仅仅是在他们需要贷款来完成像上大学或买房这样的重大生活事件时。

Flynn 说:“Tally 解决了客户管理多张信用卡、产生费用以及不知道先付哪一张的问题。”"他们的解决方案很巧妙，可以适用于很多人。"

然而，布朗说，Tally 受益于另类贷款创业公司早期的成功。

“投资界已经习惯了非银行实体发放贷款资产，”布朗说。“金融机构已经习惯于通过 API 和其他技术和系统与初创公司合作。”
