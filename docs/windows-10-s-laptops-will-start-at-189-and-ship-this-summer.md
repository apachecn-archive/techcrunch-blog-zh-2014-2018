# Windows 10 S 笔记本电脑的起价为 189 美元，将于今年夏天上市

微软对 Chrome OS 的回答叫做 [Windows 10 S](https://web.archive.org/web/20221020054333/https://beta.techcrunch.com/2017/05/02/windows-10-s-will-fuel-microsofts-chromebook-competitors/) 。这个新的操作系统是 Windows 10 的精简和安全版本。它运行沙盒应用程序，不需要昂贵的硬件。

这也是微软大放异彩的地方，因为它可以与所有主要的 PC OEMs 厂商谈判，说服他们生产 Windows 10 S 设备。该公司宣布，宏碁、华硕、戴尔、富士通、惠普、三星和东芝都在开发 Windows 10 S 设备。

入门级 Windows 10 S 设备的起价为 189 美元，这应该有助于说服学校为学生购买大量笔记本电脑。

[gallery ids="1484308，1484327，1484309，1484324，1484325，1484326"]

这些设备附带一年免费订阅的[《我的世界》:教育版](https://web.archive.org/web/20221020054333/https://education.minecraft.net/)。但这还不是全部。Office 365 在线应用也将对学生和教师免费。

Windows 10 S 设备将于今年夏天发货，正好赶上新学年。如果学校想将旧设备升级到 Windows 10 S，将可以免费下载 Windows 10 S。让我们看看这是否足以在教育领域与谷歌竞争。
