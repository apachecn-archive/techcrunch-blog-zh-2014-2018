# 飞轮为其以设计师为中心的 WordPress 托管平台筹集了 120 万美元

开源软件 WordPress 运行着互联网上所有网站中的一大块，所以毫不奇怪，这个平台周围也涌现出了一个巨大的生态系统。如今，几乎任何你能想到的领域都有专门的 WordPress 服务。

位于奥马哈的[飞轮](https://web.archive.org/web/20221208073612/http://getflywheel.com/)就是其中之一。它专注于设计师、自由职业者和创意机构，他们需要一个可管理的 WordPress 平台来建立和管理他们客户的网站。今天，该公司宣布它已经筹集了 120 万美元的资金。

这轮融资由总部位于奥马哈的[亚麻子资本](https://web.archive.org/web/20221208073612/http://linseedcapital.com/)牵头，参与方包括芝加哥的[莱特银行](https://web.archive.org/web/20221208073612/http://www.lightbank.com/)和[海德公园风险投资伙伴](https://web.archive.org/web/20221208073612/http://www.hydeparkvp.com/)，以及总部位于底特律的[拉德洛风险投资](https://web.archive.org/web/20221208073612/http://ludlowventures.com/)。

飞轮告诉我，它计划利用这笔新资金专注于产品开发。“在不久的将来，我们将推出集结地和加盟/推荐项目，”飞轮公司首席执行官兼联合创始人达斯丁·戴维森本周早些时候告诉我。“除此之外，还有 API 和第三方集成，以及其他功能来改善设计师和代理商在 WordPress 上建立网站的工作流程。”

这项服务已经将重点放在确保其工作流程对用户来说尽可能简单明了。创建一个新的网站只需要几秒钟，一旦它启动并运行，这项服务就可以很容易地添加合作者，给客户开账单，以及做设计师和自由职业者在从事 WordPress 项目时需要处理的大多数其他事情。Flywheel 的一个很好的特点是，用户只在网站上线后才为其付费——在这一点上，很可能是客户会支付托管费用。

飞轮的[定价](https://web.archive.org/web/20221208073612/http://getflywheel.com/pricing/)主要基于一个网站每月的访问量，起价为每月 15 美元，最多 5000 次访问。从今天开始，这项服务还向用户提供批量定价。每月支付 100 美元，他们可以运行 10 个网站，总访问量达 15 万次，或者每月支付 250 美元，运行 25 个网站，访问量达 50 万次。
