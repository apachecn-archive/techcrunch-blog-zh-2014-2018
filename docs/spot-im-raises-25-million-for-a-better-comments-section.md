# 斑点。IM 为一个更好的评论部分筹集了 2500 万美元

社交媒体对在线出版商来说已经变得至关重要。它带来了大量的流量，并帮助品牌建立了追随者。

但这也意味着很多互动发生在另一个平台上，比如 Twitter 或脸书，而不是真正的新闻网站。

总部位于纽约的 Spot。我看到了一个机会，可以帮助从时代公司到 NBC 的企业建立一个更好的评论区，让人们在自己的网站上停留更长时间。TechCrunch 的姐妹出版物《赫芬顿邮报》和 Engadget 也是它的客户。

这家初创公司正在筹集由 Insight Venture Partners 牵头的 2500 万美元 C 轮融资，以扩大业务规模。俄罗斯亿万富翁罗曼阿布拉莫维奇与 Altair VC 和 Norma Investments 一起参与。

[斑点。IM](https://web.archive.org/web/20230405124211/http://www.spot.im/) 首席执行官 Nadav Shoval 告诉 TechCrunch，他的“梦想是真正在开放网络上建立社区。”他说点点。IM“提供了像社交网络一样的技术。”

他说那个地方。IM 平台让用户在网站上停留的时间更长，从而为出版商带来更多的广告收入。斑点。IM 还为该页面建立了额外的广告单元。他们没有向媒体公司收取服务费，而是制定了一种收入分成模式。

还有其他的评论平台，比如 Disqus，但是 Shoval 相信 Spot。IM 是最好的版主。斑点。IM 使用机器学习来删除非法评论和垃圾邮件。

他说那个地方。IM 有一些新的大项目正在进行中，并将在 R&D 和招聘上花费资金。“我们将推出一个重要的新平台，”他说。

最终，他希望当场。IM 将帮助出版商赚更多的钱。斑点。Shoval 说，IM“真的想拯救媒体”。
