Crate.io 筹集了 1100 万美元，并推出了其托管物联网数据平台。TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/06/19/crate-io-raises-11m-and-launches-its-hosted-iot-data-platform/?utm_source=dlvr.it&utm_medium=twitter

Crate.io ，我们 Disrupt Europe 2014 创业战场竞赛的获胜者，今天宣布它已经筹集了 1100 万美元的 A 轮融资。此外，该公司今天还推出了“Crate 机器学习平台”，这是一种新的托管解决方案，适用于希望使用该公司基于 SQL 的数据库平台处理物联网数据的企业。

新一轮融资由 Zetta Venture Partners 和 Deutsche Invest Equity 牵头，Chalfen Ventures、Momenta Partners 和 Charlie Songhurst 参与。包括 Draper Espirit、Vito Ventures 和 Docker 创始人 Solomon Hykes 在内的现有投资者也参与其中。

Crate 的联合创始人兼首席执行官 Christian Lutz 告诉我，在过去一年左右的时间里，该公司的付费客户大幅增加，现在总计约有 30 家。这也使得 Crate 的年收入超过了 100 万美元。他将这家初创公司目前的成功归因于其对机器数据的重新关注，这是该团队在首次推出产品时并没有真正关注的事情。

卢茨告诉我，也正是这种专注让筹款变得更加容易。“与众不同的是对机器数据的强烈关注——结合交付销售，”他说。事实上，Crate 现在也有一些知名的参考客户，包括 Skyhigh Networks 和 ALPLA ，这是一家包装制造商，你可能从未听说过，但它为美国市场生产几乎所有的可口可乐和联合利华瓶子(以及其他一些你可能在家里拥有的瓶子)。

不出所料，该公司现在有 30 多名员工，计划利用新的资金扩大其营销和销售工作，以及扩大其核心工程团队。

谈论工程。凭借其机器平台，Crate 今天还推出了其首个托管产品，该产品位于微软的 Azure 平台上。这并不是一个很大的惊喜，原因有二:a)Crate 的许多工业客户已经在 Azure 上下了赌注，b) Crate 是柏林微软增长加速器 2017 届的一部分。新平台的重点是为企业提供从物联网设备摄取大量数据的单一解决方案。该平台支持实时分析，并允许用户设置自己的规则，以在必要时触发工作流和警报。平台本身处理所有的扩展(由流行的 Kubernetes 容器编排工具处理)，以及备份、归档和通常的基于角色的安全功能。

Crate 今天还发布了其开源产品的 3.0 版本。虽然该公司的商业重点显然是企业的增值功能，但它也继续积极开发开源版本，Lutz 指出，这个新版本对某些类型的查询提供了 100 倍的性能提升。
