# 特斯拉 Semi 首次在船上使用电池进行生产货运

对于一个熟悉的客户来说，特斯拉正在把它的新电动 Semi 投入使用，这是它作为“生产”车辆的首次运行。特斯拉本身就是客户，因为卡车配备了拖车，装载着刚从 Gigafactory 装配线上卸下来的电池组，准备运往位于加利福尼亚的特斯拉弗里蒙特汽车厂。

埃隆·马斯克(Elon Musk)分享了一幅卡车在工作时的照片，有哑光黑色版，也有银色版的航空发动机罩，这大概与他们在官方发布会上展示的相同(至少颜色相同)。这些卡车附有全尺寸拖车，这不是他们在之前的目击中所配备的。

从 Gigafactory 到 Fremont 工厂大约需要四个半小时的路程，行程约 260 至 270 英里。这并不算是一段很长的路程，但对许多客户来说，这可能仍然非常有用，为此，特斯拉自己必须定期用自己的运输车队进行这一行程。

特斯拉称之为“生产”运行，但其实际生产这些的目标似乎仍然是 2019 年，可能的交付时间框架更像是 2020 年。尽管如此，随着公司的发展，这次货物运输应该会为公司提供有价值的信息，尤其是现在公司有大量的客户预订单需要处理。
