# Spotify 为跑步者推出了新功能，包括根据你的速度变化的歌曲

Spotify 今天的新闻发布会不仅仅是添加视频和其他内容类型。该公司还透露，它正在增加新功能，以改善任何人在跑步时听应用程序的体验。

为此，首席产品官 Gustav sder strm 表示，Spotify 现在将使用手机上的传感器来检测你跑步的速度。然后，当你点击 Spotify 中的 running man 按钮时，应用程序会根据你的速度选择正确的曲目。如果你想改变节奏，索德斯特罗姆说，“我们会找到合适的音乐，一次又一次。”

更令人印象深刻的是，他说 Spotify 正在与音乐家(包括在活动中短暂露面的 Tiesto)合作，为跑步创作特定的曲目。毕竟，他指出，当他的团队与跑步者交谈时，他们说当合适的歌曲响起时，他们“感觉像个英雄”但这种效果只持续了大约一分半钟，然后就消失了——至少在下一首歌出现之前是如此。

![Spotify NYC Running Playlists](img/09a0aa7888708304ff364934386ad516.png)

所以 Sö derströ想知道为什么这种高水平不能持续整个过程。为了实现这一点，这些新曲目实际上会调整中期宋。这不是“简单的节拍拉伸”，他说:“作品本身似乎会重新排列，以适应你目前的节奏。”到目前为止，Spotify 已经创造了六种这样的“体验”，他说“每一种都有自己的特点。”

Sö derströ说，所有这些功能都是与跑步者和健身专家协商开发的。是的，这意味着实际上要进行大量的运行，这就是为什么他吹嘘说，“我现在可能有最合适的产品和设计团队。”

该公司还宣布与耐克和 RunKeeper 合作，在今年晚些时候整合新的 Spotify 跑步体验。
