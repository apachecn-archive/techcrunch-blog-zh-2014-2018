# 原创内容播客:“致所有我爱过的男孩”是一部迷人的高中爱情片

虽然好莱坞对浪漫喜剧的兴趣似乎正在消退，但网飞一直在收拾残局。就在几个月前， [it 发布了极其有趣的*设置了*T5。现在我们有*给所有我在*之前爱过的男孩，这是一部根据韩珍妮的青少年小说改编的高中爱情片。](https://web.archive.org/web/20221209040401/https://techcrunch.com/2018/06/25/original-content-set-it-up/)

*致所有男孩*讲述了劳拉·珍·柯维(陈东兰饰演)的故事，一个十几岁的女孩给她所有的暗恋对象写了情书，但从未寄出——直到电影开始，这些情书神秘地落入了名义上的男孩手中。

自然，这导致了强烈的屈辱和尴尬，特别是当劳拉·珍不顾一切地隐藏她对姐姐的前任乔什(伊瑟尔·布罗萨德饰)的感情，以至于她同意假装和她的前任约会(？)碾压彼得(诺亚·森特尼奥)。

在最新一集的[原创内容播客](https://web.archive.org/web/20221209040401/https://itunes.apple.com/us/podcast/original-content/id1271763098?mt=2)中，我们的同事 Taylor Nakagawa 加入了我们，一起回顾这部电影。泰勒并没有完全被说服——毕竟，根据上面简单的情节描述，你可以猜到接下来会发生什么。但是你的固定主持人安东尼和乔丹还是喜欢这部电影，尤其是这部电影非常迷人的主角。

我们还讨论了*疯狂富有的亚洲人*，一部少有的登上大银幕的好莱坞浪漫喜剧，以及[制片人如何拒绝了来自网飞的邀请](https://web.archive.org/web/20221209040401/https://www.hollywoodreporter.com/features/crazy-rich-asians-story-behind-rom-com-1130965)。我们还报道了本周的流媒体新闻，包括网飞与黑人创作者肯尼亚·巴里斯的独家协议，以及亚马逊正在就收购连锁影院 T21 进行谈判的报道。
