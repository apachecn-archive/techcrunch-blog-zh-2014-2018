# 近 30 年后，微软的开发者部门主管离开公司

负责监管微软开发者工具和服务的高管 s·索马塞格(S. Somasegar)已经宣布，他将在 27 年后离开公司。在一系列推文中，通常被称为他的昵称 Soma 的 Somasegar 说，他计划寻求其他职业机会。

在获得一份内部备忘录后，ZDNet 首先爆料了 Somasegar 离职的消息。在其中，微软云和企业执行副总裁斯科特·格思里(Scott Guthrie)表示，微软希望在未来几周内宣布一项过渡计划。微软发言人告诉 TechCrunch，该公司目前不予置评。

Somasegar 对微软的追随者来说是一个熟悉的名字，因为作为微软开发部门的公司副总裁，他负责工程和营销开发工具和服务，包括 Visual Studio、Team Foundation 和编程语言。在 1989 年加入微软后，索马塞格[在加入开发者部门之前贡献了八个 Windows 版本](//web.archive.org/web/20221006035221/https://www.reddit.com/r/IAmA/comments/1qjney/%E2%80%9C)。

他还监管微软旗下的网站，如面向开发者的社区 [MSDN](//web.archive.org/web/20221006035221/https://blogs.msdn.com/%E2%80%9C) ，以及新闻门户 [TechNet](//web.archive.org/web/20221006035221/https://technet.microsoft.com/en-us/%E2%80%9C) 。
