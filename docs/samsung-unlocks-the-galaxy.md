# 三星的 Galaxy S7 和 S7 Edge 现已在美国解锁销售

美国部分三星粉丝的等待结束了，他们一直在等待该公司的旗舰获得无运营商待遇。智能手机巨头[今天](https://web.archive.org/web/20221025223337/http://www.businesswire.com/news/home/20160630005315/en/Samsung-Introduces-Unlocked-Galaxy-S7-Galaxy-S7)宣布，它将通过自己的网站提供 [Galaxy S7 和 S7 Edge](https://web.archive.org/web/20221025223337/https://beta.techcrunch.com/2016/02/22/every-angle-of-the-new-samsung-galaxy-s7-and-s7-edge/) 的解锁版本，以及亚马逊、百思买和塔吉特等知名零售商。

解锁版手机可以在你能想到的大多数 GSM 和 CDMA 运营商上运行，包括美国电话电报公司、T-Mobile、威瑞森和 Sprint(还有一些你可能想不到的)。这些手机与该公司一直提供的合同补贴相同，包括所有的快速充电和诺克斯防水防尘功能，这些功能使它们首次获得成功。

从 3 月开始，你可以通过我们的[双评来刷新你的记忆。](https://web.archive.org/web/20221025223337/https://beta.techcrunch.com/2016/03/08/review-samsung-galaxy-s7-and-s7-edge/)

Galaxy S7 和 S7 Edge 自然会比运营商品牌的版本花费更多，尽管你无疑已经知道这一点。它们现在可以通过上述渠道购买，价格分别为 670 美元和 770 美元。
