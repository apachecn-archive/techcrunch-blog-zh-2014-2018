# 以太坊下跌后，一个强大的采矿芯片表面的谣言

来自比特大陆的新 ASIC 采矿钻机的传言已将以太坊价格推低至 585 美元的一周高点以下。在加密货币世界中，ASIC——或专用集成电路——是设计师为挖掘单一货币的特定目的而创造的芯片。例如，早期的比特币专用集成电路推动了采用率的上升，然后在一些人看来，将比特币开采集中在少数人手里，从而挫败了铁杆加密货币粉丝的分散化风气。

据美国消费者新闻与商业频道报道，分析师克里斯托弗·魅兰访问了中国，在那里他发现了一种专用于以太坊采矿的新 ASIC 芯片的传言。

> “在我们上周的亚洲之旅中，我们确认比特大陆已经开发出了用于采矿以太坊的 ASIC(专用集成电路)，并为 2018 年第二季度的出货做好了供应链准备，”分析师克里斯托弗·魅兰周一在给客户的一份报告中写道。“虽然比特大陆可能是最大的 ASIC 供应商(目前占比特币挖矿 ASIC 的 70-80%)，也是第一个将该产品推向市场的供应商，但我们已经了解到至少有三家其他公司正在研究以太坊 ASIC，它们都处于不同的开发阶段。”

历史上，用户使用 GPU 挖掘以太坊，这反过来导致游戏和图形的 GPU 不可用。然而，ASIC 将完全改变采矿等式，导致一定程度的集中化，因为大型参与者(包括比特大陆)为临时矿工创造了更高的准入门槛。

“以太币是 GPU 采矿中最赚钱的硬币之一，”TryMining.com 的创始人米哈伊尔·阿瓦迪说。“这将对市场产生很大影响。不了解这些比特大陆机器的哈希能力，我们无法判断它是否会让 GPU 过时。”

“这可以看作是对网络的攻击。这是一个集权问题，”他说。

阿瓦迪指出，加密货币爱好者一直在争论 ASICs 及其对市场的影响。一些人预计会转向更多可开采的硬币，包括 Monero 和 ZCash。

“如果只有一家以太坊 ASIC 制造商，那就糟了，”他说。“但随着三星和其他几家公司进入这一领域，这种情况不会持续太久。”

还有人担心 ICO 的推出和基于以太坊的智能合约令牌的实际效用。“Cypherium 的首席执行官 Sky Guo 说:“随着人们对技术越来越现实，ETH 的价格正在变得稳定。”。“人们正在寻找更高质量的区块链项目。我相信，随着围绕监管的恐慌开始消退，ETH 的价格将很快反弹。”
