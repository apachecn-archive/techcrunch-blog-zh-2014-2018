# SEC 批准股权众筹，因此任何人都可以投资私人公司

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/03/27/sec-rule-change-gives-startups-an-a-for-capital-formation/>

克里斯·泰瑞尔撰稿人

Chris Tyrrell 是 OfferBoard Group 的首席执行官，也是行业贸易协会 CFIRA 的主席。

**编者按:** *Chris Tyrrell 是 [OfferBoard Group](//web.archive.org/web/20230326121807/https://www.offerboard.com/) 的首席执行官，也是行业贸易协会 [CFIRA](https://web.archive.org/web/20230326121807/http://www.cfira.org/) 的主席。*

本周早些时候，美国证券交易委员会(SEC)宣布了一套实施《就业法案》第四章的新规则。

这些变化影响了小型公开募股的监管，俗称“监管 A+”。

这些最终规则的发布进一步推进了 2012 年法律的核心原则和目标之一:创造一个新兴企业可以有效筹集公共资本的环境。

SEC 委员周三批准的规则将企业在监管 A 发行中可以筹集的资本金额上限从 500 万美元提高到 5000 万美元，分为两个“等级”——12 个月内最多 2000 万美元(第一级)和 12 个月内最多 5000 万美元(第二级)。融资少于 2000 万美元的公司将在 1 级和 2 级之间做出选择。

这是公开发行，很像传统的首次公开募股，但发行公司的监管负担更低，无论是在股票出售时还是出售后。

> 这些最终规则的发布进一步推进了 2012 年法律的核心原则和目标之一:创造一个新兴企业可以有效筹集公共资本的环境。

某些符合股东限制或收入和公众持股上限的监管 A+公司将避开 SEC 的标准报告制度——表格 10-K、10-Q、8-K 和代理声明，此类公司还可以免除许多繁重的“上市公司”规则，包括萨班斯-奥克斯利法案(SOX)和更深奥的负担，如与披露“冲突矿物”使用情况有关的规则。

每一个投资者，不管是合格的还是不合格的，都可以参与这些发行。

对于努力筹集资金、但缺乏高净值投资者或机构渠道的中小企业来说，这是个好消息。Reg A 将向所有人开放更多成长型公司的产品，这将增加资本形成，增加就业机会，促进经济增长。

新 Reg A 规则中最有争议和最重要的条款之一是，根据被称为“蓝天”法律的州规则，联邦政府对这些产品的州审查有优先权。

> 本周的变化旨在吹掉这种很少使用的资本形成工具上的灰尘，并让它脱离监管的泥沼。

以前，每家公司都必须在发行其证券的每个州单独注册其 Reg A 发行。这种管理负担是在 SEC 要求注册之外的额外负担，导致 Reg A 发行非常少。现在，根据 Reg A 的第 2 层，只需要 SEC 审查，这大大减轻了企业的管理负担。

在起草过程中，SEC 一直关注新规则将如何保护投资者——特别是未经认证的投资者——同时保持足够简单和精简以鼓励资本形成。

一些人将《就业法案》规则制定的延迟归因于解决这些潜在对立利益的艰巨任务。最终规则在这方面取得了平衡，要求公司在首次出售前几周向 SEC 提交其 Reg A 发行文件。这让州监管机构有时间利用这些文件来发现潜在的问题。

Reg A 发行并不新鲜——这种小额发行豁免自 1933 年《证券法》首次成为法律以来就一直存在。但近年来，SEC 批准的发行数量逐渐减少，从 1998 年的 57 只减少到 2011 年的 1 只。

本周的变化旨在吹掉这种很少使用的资本形成工具上的灰尘，并让它脱离监管的泥沼。5 月 25 日，这些规定将开始生效，全国各地的新兴成长型企业将拥有一个新的“A+”工具。

下表比较了 Reg A 下一级和二级资本的一些规定:

**一级和二级法规对比**

|  | 一级 | 二级 |
| 最大发行规模 | 两千万美元 | 五千万美元 |
| 投资者类型 | 认证和非认证 | 认证和非认证 |
| 个人投资限额 | 没有。 | 自报的年收入或净资产中较高者的 10% |
| 广告/一般征集 | 没有限制。* | 没有限制。* |
| 财务披露 | 会计师审查的财务 | 经审计的财务状况；符合 S-X 法规 |
| 州优先权 | 没有；国家协调审查与美国证券交易委员会备案 | 是 |

*任何征集的内容必须符合“试水”要求，或者采用初步招股说明书的形式。
