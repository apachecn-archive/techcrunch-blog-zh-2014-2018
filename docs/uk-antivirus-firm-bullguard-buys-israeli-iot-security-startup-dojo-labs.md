# 英国防病毒公司 BullGuard 收购以色列物联网安全初创公司 Dojo-Labs

英国反病毒制造商 BullGuard 正在收购以色列初创公司 [Dojo-Labs](https://web.archive.org/web/20230318233016/http://www.dojo-labs.com/) ，以将其安全产品组合扩展到物联网领域。交易条款尚未披露。

去年 11 月，我们报道了 [Dojo-Labs](https://web.archive.org/web/20230318233016/https://www.dojo-labs.com/) [，当时它秘密发布了一款名为 Dojo 的卵石形消费物联网安全设备，旨在监控网络流量，标记和阻止家庭网络上连接设备的异常行为。](https://web.archive.org/web/20230318233016/https://techcrunch.com/2015/11/19/dojo-labs/)

当时，这家初创公司的目标是在 2016 年 3 月推出其安全设备，并接受预购。八个月的发货时间已经过去了——对于硬件初创公司来说，这是一个非常常见的故事。但该团队现在将利用更大的母公司的资源，在年底前将他们的小发明推向市场。

“我们的当务之急是尽快将 Dojo 产品推向市场，”BullGuard 的一位女发言人告诉 TechCrunch。

“Dojo 专注于根据用户反馈改善用户体验和产品功能。(发货)的时间表是今年年底，”她补充道。

这位发言人证实，Dojo 的创始人将在收购后留任，但补充说品牌战略尚未完全成型。

Dojo 的 pebble 形状的物联网小发明和移动通讯应用程序 UI 与 BullGuard 销售的传统矩形盒装杀毒软件相比，具有更加平易近人和消费者友好的风格，因此这两种风格如何融合还有待观察。

“我们正在制定我们的品牌战略和定位，”她说。“我们正在联合两家拥有共同愿景的公司，以保护消费者及其数据和设备。这意味着客户将在其所有联网设备上获得最高级别的保护。最有可能的是，我们会有一个品牌——bull guard。”

上个月，捷克杀毒软件制造商 AVG 被捷克安全公司 Avast 收购，部分原因是为了扩大其产品组合，为物联网提供安全产品。

预计将会看到更多的安全公司达成协议，帮助他们应对由家庭联网设备激增造成的[安全噩梦](https://web.archive.org/web/20230318233016/https://techcrunch.com/2016/05/09/the-internet-of-things-is-security-nightmare-warns-eff/)，即使计算继续从最初打造反病毒帝国的桌面转移。
