# “明日之星”机器人可以奔跑、翻转和爬行 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/07/13/the-rising-star-robot-can-run-flip-and-crawl/>

# 新星机器人能跑，能翻，能爬

内盖夫本古里安大学的研究人员制造了一个聪明的小机器人，它可以翻滚，翻转，甚至像乌龟一样在崎岖的地形上爬行。该机器人使用轮子、一组星形滚轮和巧妙铰接的手臂，以各种速度行驶。

这个名为[后起之秀](https://web.archive.org/web/20221208100205/https://spectrum.ieee.org/automaton/robotics/robotics-hardware/sprawling-wheel-leg-robot-crawls-and-climbs)的机器人使用轮子和辐条“轮子”以大约每秒一米的速度滚动，当它发现泥土或沙子时，它可以折叠自己并向前拉。它还可以使自己非常瘦，以便通过狭窄的地方，甚至可以在摔倒时翻转。

研究人员说，一个加重的“头部”可以在机器人行进时保持平衡，使它能够爬上和越过陡峭的表面，甚至可以偷偷穿过管道或墙壁之间。

“明日之星”是该大学在 2013 年展示的[蔓延调节自主机器人](https://web.archive.org/web/20221208100205/https://spectrum.ieee.org/automaton/robotics/diy/watch-this-sprawltuned-insect-bot-skitter-all-over-the-place)的升级版。这个新版本更有能力，而且由于它的“whegs”和乌龟步态，也更酷。
