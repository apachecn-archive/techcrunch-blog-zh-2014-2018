# 以下是纽约科技之星 2017 年夏季班的 12 家创业公司

纽约科技之星最新一期的 12 家创业公司将接管[今晚的纽约科技聚会。](https://web.archive.org/web/20221209172908/https://www.meetup.com/ny-tech/?_cookie-check=BnMYOVAcBzqu_Cv2)

这个“社区演示日”是常务董事 Alex Iskold 试图重新思考传统加速器演示日的一部分。(不要担心，在 Techstars 的投资者周期间，初创公司也将通过一对一的会议与投资者见面。)

我最近参观了 Techstars 的办公室，在那里我有机会与大多数创始人进行了短暂的会面，他们从事从结肠镜检查到抹茶到科隆的各种工作。(遗憾的是，我没有尝试抹茶或古龙水。)下面是他们的行动概要:

*   *   [CottageClass](https://web.archive.org/web/20221209172908/http://cottageclass.com/) 是一个连接家长和“微型学校”教师的市场，微型学校基本上是合作性的家庭学校项目。首席执行官 Manisha Snoyer 曾经是一名教师，她认为这种方法是“越来越不灵活，拥挤”的公立学校系统的一种替代方法。该公司称已经为 455 个家庭提供服务。
    *   [Gather](https://web.archive.org/web/20221209172908/http://gatherdata.co/) 提供了一个自助服务平台，帮助公司管理来自零散来源的数据。这个想法是，企业应该能够在不雇佣庞大的数据工程师团队的情况下，构建自己的数据基础设施。
    *   [Geopipe](https://web.archive.org/web/20221209172908/https://geopi.pe/) 构建真实世界的 3D 模型。首席科学官托马斯·迪克森(Thomas Dickerson)表示，这项技术最初是为游戏而建，但 Geopipe 现在专注于建筑，将其作为第一个市场。显然，这家初创公司的模型允许你做用[纹理映射](https://web.archive.org/web/20221209172908/https://en.wikipedia.org/wiki/Texture_mapping)无法完成的事情，例如调整灯光以代表一天中的不同时间。
    *   Hawthorne 正在创造个性化的古龙水，并计划扩展到其他个人护理产品。

[![matchaful](img/f13ad0564f436ff70083504af8383dcc.png)](https://web.archive.org/web/20221209172908/https://beta.techcrunch.com/2017/09/28/techstars-nyc-summer-2017/matchaful/)

*   *   [Matchaful](https://web.archive.org/web/20221209172908/http://matchaful.com/) 向咖啡馆、办公室和直接向消费者提供有机抹茶和抹茶制作设备。首席执行官 Hannah Habes 告诉我，她加入 Techstars 是因为她想探索如何改善人们对抹茶的体验，首先是教育和内容，最终是新设备。
    *   从维拉·布拉德利开始，obses 正在为零售商创造虚拟现实体验。首席执行官 Neha Singh 认为，随着实体零售业努力发展，虚拟现实为企业提供了一种扩展商店的方式。通过痴迷，他们可以创造展示和炫耀他们在商店本身没有空间的库存。当然，顾客可以在虚拟现实体验中进行购买。
    *   鹦鹉已经建立了一个云电话系统，集成了其他工具，如 Shopify 和 Magento。首席执行官科林·莫雷利回忆说，他在一次黑客马拉松上建造了一个自动聊天机器人，并意识到没有简单的方法将其连接到现有的电话系统。因此，通过 API 使电话系统具有适应性和可扩展性的目标。
    *   Phonewagon 提供电话跟踪软件，帮助企业确定电话何时由特定的营销活动驱动。
    *   SVRF 是一个 360 度全方位内容的搜索引擎。目标是更容易找到 VR 和 AR 内容。首席执行官索菲亚·多明格斯说，SVRF 目前抓取了一个特定的网站列表，但它也增加了一个按钮，供用户提交自己的内容。还有一些独特的内容——[这种字面上无法形容的体验](https://web.archive.org/web/20221209172908/https://www.svrf.com/search/Vitalik)，例如，灵感来自[维塔利克·布特林在 Disrupt](https://web.archive.org/web/20221209172908/https://beta.techcrunch.com/2017/09/18/vitalik-buterin-explains-ethereum-in-his-own-words-at-disrupt-sf/) 上令人惊叹的 t 恤。

[![Vitalik 360](img/54e173dad090b5f62a2ce5dba4b44aa4.png)](https://web.archive.org/web/20221209172908/https://beta.techcrunch.com/2017/09/28/techstars-nyc-summer-2017/unnamed-502/)

*   TypeLead 正在开发 [Eta](https://web.archive.org/web/20221209172908/http://eta-lang.org/) ，一种面向企业应用的编程语言。(首席执行官 Jyothsna Patnam 将其描述为“ [Haskell](https://web.archive.org/web/20221209172908/https://www.haskell.org/) 移植到 Java 虚拟机。”)Eta 是一种开源语言，拥有 20，000 名活跃的开发人员，但 TypeLead 计划随着时间的推移添加商业产品和服务。
*   处女座为结肠镜检查提供视频记录。首席执行官 Matt Schwartz 表示，虽然大多数结肠镜检查目前都没有记录(除了少数静止图像)，但视频可以揭示重要的细节。随着时间的推移，他希望增加更多的自动化来帮助诊断。
*   [至关重要](https://web.archive.org/web/20221209172908/http://vitally.io/)是一款面向 B2B 企业的分析产品。首席执行官杰米·戴维森表示，大多数分析产品都是为面向消费者的公司设计的，所以他们专注于一组不同的指标——即消费者如何与你的网站或应用程序互动。另一方面，重要的是，衡量企业与每个客户的关系，以及这种关系如何推动收入。
