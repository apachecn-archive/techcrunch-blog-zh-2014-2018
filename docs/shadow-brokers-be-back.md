# 所谓的美国国家安全局黑客集团影子经纪人发布新的利用宝库

影子经纪人，去年发布的据称被国家安全局使用的黑客攻击的幕后组织，已经丢失了另一批文件。在今天[媒体的一篇文章](https://web.archive.org/web/20221209073641/https://medium.com/@shadowbrokerss/dont-forget-your-base-867d304a94b1)中，黑客组织提供了一个密码，可以免费访问他们之前试图拍卖的文件。

影子经纪人在去年 8 月首次崭露头角，当时他们泄露了与 [NSA 和包含](https://web.archive.org/web/20221209073641/https://beta.techcrunch.com/2016/08/19/snowden-docs-link-nsa-to-equation-group-hackers/)[主要防火墙产品漏洞的方程式组](https://web.archive.org/web/20221209073641/https://beta.techcrunch.com/2016/08/17/cisco-and-fortinet-say-vulnerabilities-disclosed-in-nsa-hack-are-legit/)相关的漏洞。该组织随后[发布了一份 IP 地址列表](https://web.archive.org/web/20221209073641/https://beta.techcrunch.com/2016/10/31/shadow-brokers-post-list-of-compromised-ip-addresses/)，声称这些 IP 地址被方程式小组泄露。

影子经纪人希望[拍卖另一组文件](https://web.archive.org/web/20221209073641/https://motherboard.vice.com/en_us/article/while-alleged-nsa-thief-sits-in-detention-shadow-brokers-post-messages)，但在尝试中没有引起太多兴趣——或比特币。在那次失败后，该组织在 1 月份发布了一条“[告别消息](https://web.archive.org/web/20221209073641/https://arstechnica.com/security/2017/01/nsa-leaking-shadow-brokers-lob-molotov-cocktail-before-exiting-world-stage/)，并泄露了一组新的与 Windows 相关的漏洞。

今天影子经纪人的泄露伴随着[冗长的媒体帖子](https://web.archive.org/web/20221209073641/https://medium.com/@shadowbrokerss/dont-forget-your-base-867d304a94b1)，该组织在帖子中表示，在对唐纳德·特朗普总统的领导失去信心后，它将公布这些文件作为一种“抗议形式”。该帖子声称特朗普似乎正在“放弃他的基础”，还提供了一系列关于总统如何“让美国再次伟大”的建议。
