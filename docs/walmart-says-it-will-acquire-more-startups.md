# 沃尔玛表示将收购更多初创公司

沃尔玛最近收购了一些初创公司，该公司表示，这种收购热潮将继续下去。

沃尔玛电子商务美国公司首席执行官马克·洛尔(Marc Lore)周二在拉古纳海滩举行的华尔街日报直播(WSJ D.Live)上表示，沃尔玛正在寻求收购技术、零售和数字本土品牌类别的企业。

这位 Jet.com 创始人是通过去年的一次收购加入沃尔玛的，当时沃尔玛斥资逾 30 亿美元收购了这家快速增长的初创公司。从那以后，它继续购买[的鞋子](https://web.archive.org/web/20221208201714/https://finance.yahoo.com/news/walmart-acquires-shoebuy-70m-164800758.html)、[的穆塞乔](https://web.archive.org/web/20221208201714/https://beta.techcrunch.com/2017/02/15/walmart-acquires-outdoor-retailer-moosejaw-for-51-million/)、[的倭黑猩猩](https://web.archive.org/web/20221208201714/https://beta.techcrunch.com/2017/06/16/walmart-to-buy-bonobos-for-310m-in-its-bigger-push-into-fashion-retail/)、包裹、干草针和[的 ModCloth](https://web.archive.org/web/20221208201714/https://beta.techcrunch.com/2017/03/17/confirmed-walmart-jet-modcloth/) 。

“专家定位优于大众，”关于这一战略的传说。“我们已经授权这些公司的领导基本上在整个实体中运营该类别。”

洛尔还谈到了与亚马逊的竞争，在这家电子商务巨头收购他的公司后，他在亚马逊工作了几年。

他说沃尔玛已经建立了在美国提供更快送货的能力。Lore 声称，它已经拥有为全国 87%的地区进行隔夜交货的基础设施，并且正在努力将库存分散到这些仓库位置。

Lore 说，预计沃尔玛将“在未来两年内击败它”。他说他“现在不会和任何人交换我们的位置。”
