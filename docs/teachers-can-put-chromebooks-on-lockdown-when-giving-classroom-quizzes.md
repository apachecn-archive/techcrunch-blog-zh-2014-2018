# 教师可以在进行课堂测验时将 Chromebooks 锁定

本周，科技界的重量级人物齐聚芝加哥，在国际教育技术协会会议上展示他们最新的教育技术产品。对谷歌来说，这意味着对教室的一些关键更新(T2)，这是一个免费的基于浏览器的教育软件，据其统计，目前全球有“超过 3000 万学生”在使用。

该应用程序正在进行一些调整，其中最值得一提的是，对测验的更多控制。谷歌表格测验现在有一个“锁定模式”，禁止学生在提交答案之前上网冲浪或打开应用程序——换句话说，这可以阻止他们在考试时作弊。

![](img/8442a03acc11797514d5673d021ab725.png)

有趣的是，这是该应用程序中第一个专属于托管 Chromebooks 的功能，即那些销售的唯一目的是在课堂上使用的设备。这标志着这款应用的一个变化，对于任何可以访问网络的机器来说，它都是开放的和平台不可知的。当然，这里的原因看起来很简单——将用户锁在其他应用程序之外需要比更标准化的功能更多的系统控制。

还有一个新的课堂作业页面，它被设计成教师和学生的首选目的地，在一个目的地组织问题和作业。除此之外，这些信息现在可以按主题或单元进行组织，而以前所有的信息都是按日期进行分类的。与此同时，新人页面。允许教师添加和删除同事教师、学生和监护人，同时对流和系统设置页面进行了其他调整。

微软还利用[机会宣布了](https://web.archive.org/web/20221025222500/https://educationblog.microsoft.com/2018/06/minecraft-update-aquatic-stem-lesson-plans/)来自 BBC 等合作伙伴的新教案，以及*《我的世界》:教育版*的水上 DLC。
