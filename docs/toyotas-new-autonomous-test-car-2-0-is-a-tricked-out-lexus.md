# 丰田新的自主测试车 2.0 是一辆精心设计的雷克萨斯

丰田的研究所(TRI)拥有一些汽车制造商最雄心勃勃的未来项目，R&D 组织周五首次推出了其新的第二代自主安全技术研究车。该车首次亮相于加利福尼亚州索诺马的丰田普锐斯挑战赛，这是一个硅谷论坛，旨在将企业家、机器人专家等与 TRI 聚集在一起。

新车辆也是第一个完全由 TRI 构建的自动驾驶测试平台，其预期目的是帮助研究小组测试和探索广泛的安全功能和自主技术。它的装载包括线控驱动接口，以及分层的激光雷达，雷达和相机传感器，以减少对预先填充的高清地图的依赖。它还被设计成相当模块化，这意味着当单个部件的升级可用并可以换入时，它可以即时改进。

丰田的第一辆测试车于 2013 年在 CES 上首次亮相，因此该团队有一些时间来改进。这款车所基于的新款雷克萨斯 LS 600hL 只是这种变化的一个明显例子。该车辆旨在帮助 TRI 开发其司机和监护人系统，分别为 5 级/4 级(完全自主)和非常先进的驾驶辅助功能。

Guardian 的目标是通过主动监控驾驶员辅助系统，在驾驶员需要干预的时候提供线索，同时在很大一部分时间内完全控制。根据 TRI 负责人 Gill Pratt 的说法，Guardian 系统具有更大的近期潜力，随着时间的推移，它可能会变得更加智能，具有大量即时救生的潜力，然后通过司机等方式实现完全自主，从而实现广泛部署。
