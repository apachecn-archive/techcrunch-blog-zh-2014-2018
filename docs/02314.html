Sinch 通过面向开发人员的 SMS API 走向全球

Sinch,新成立的 Rebtel 公司已经在全球范围内发布了其 SMS API，为全球 105 个不同国家的开发者提供了将 SMS 集成到他们的应用程序或网站的能力。

该产品最初是在 Rebtel 的保护伞下开发的，然后辛奇在今年 5 月获得了1200 万美元的资金来剥离这项业务。从那时起，该公司已经在一些国家测试推出了短信工具，今天标志着全球推出。

这个想法是，各种应用程序的开发者希望让他们的用户能够在应用程序内打电话或发送信息，无论他们是对品牌本身还是对另一个用户。然而，将这项技术嵌入现有的应用程序和网站被证明是困难的。除了消息传递本身的挑战之外，SMS 集成还需要处理终端以及后端操作管理。

像竞争对手 Twilio 和 Layer 一样，Sinch 通过 API 向应用程序或网站添加几行代码，就可以添加社交功能。不仅如此，Sinch 用户还可以通过新的 API 向他们的开发者提供应用内调用。

自 5 月份以来，Sinch 已经有 6000 名开发者注册，在该平台上进行了 40 亿分钟的通话。
