机器人革命已经酝酿了几十年，但历史上市场预期超过了技术准备。虽然工业和军事部门已经采用了许多高价机器人解决方案，但由于技术不够成熟和成本高，消费领域一直落后。

近年来，我们已经看到软件和硬件创新水平的加快，这正在推动消费者准备就绪和采用个人机器人的新可能性。

**设备、计算和带宽的激增。**不再需要昂贵的定制机载计算，机器人现在能够利用商用硬件、智能手机和云计算进行处理和存储。这对机器感知和学习的数据成本和可用性都有影响。在这种新范式下，机器人通常是一组商品化的传感器(和致动器)，它们利用云实现智能。Nest 就是一个例子。

**自然语言处理、语音、视觉、机器学习方面的进展。**新的硬件和算法，通过训练数据的语料库进行调整，使机器更具感知能力，并改善了与人类的界面。这是一个自我强化的循环:随着机器能够更好地理解现实世界，它们学习的速度也越来越快。这些新界面为消费者应用打开了市场，用户可以用近乎自然的语言和手势与机器进行交互。Siri、Kinect 和 [Wolfram Alpha](https://web.archive.org/web/20221225215130/http://www.wolframalpha.com/) 就是这样的例子。

**在线服务的 API 化。**机器利用多个在线数据源和服务的能力使它们能够快速整合价值，减少上市时间和成本。基于位置的数据、金融和天气以及越来越多的医疗保健数据都是例子。

在过去的几年里，大型企业进行了大量投资，帮助验证和重振机器人市场。我想到了 IBM 的沃森平台、谷歌的无人驾驶汽车以及最近在人工智能/机器学习领域的一系列收购。由于大多数突破性技术创新需要大型实体来奠定昂贵的基础，我们现在看到这种情况在人工智能领域发生。例如，沃森降低了创建创新的人工智能应用程序的障碍，这些应用程序处理大量非结构化数据以获得准确的答案。

未来 5 到 10 年的消费机器人会是什么样子？汽车会自动驾驶吗？家用机器人会帮助我们做日常家务吗？机器人最终会代表我们甚至与其他机器人进行互动和交易吗？我们预测以下发展将会形成。

### **重新定义机器人外形**

媒体普及的人形机器人不会是消费机器人的主导形式。人工智能设备将呈现多种形式，其外形将与其功能和用例更加匹配。更多的将是我们已经熟悉的日常系统中的嵌入式智能。我们可以预测的一件事是:告别扁平、长方形的设备；你好，外形多样性。

### **技术成本、生产率提高**

机器人技术将首先增强并最终取代高成本的人力。市场对这一进步的接受程度将取决于可靠性和安全性，并且会有临时的解决方案。例如，在无人驾驶汽车之前，市场上已经有了带平视显示器和自动感应刹车的汽车。

### **基于服务的货币化**

随着设备学习并开始预测消费者的需求，它们会提出更多建议，甚至代表消费者进行交易。在物联网世界，如果设备提供商从这些平台上提供的服务中获得一定的收入，他们将更愿意降低硬件和设备的价格。消费者将能够下载“应用”(付费或免费)，然后连接到云，通过订阅模式提供功能。

### **新的平台和生态系统**

最后，计算机产业结构将松散地映射到机器人领域。可能会有一个由大型供应商(如 Linux/Android)采用和支持的主流开放操作系统平台，以及一些特定于制造商的封闭操作系统(如 Windows)，一些“应用程序”商店(如 iTunes)，以及由大小公司提供的各种应用程序和基于 API 的服务。新一波的教育提供商、工程师和服务专业人员、安全提供商、金融/保险产品和法律框架也将出现。
