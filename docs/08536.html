这是三星Galaxy Note 7 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/08/02/samsung-galaxy-note-7/

这就是:这款新手机如此令人兴奋，以至于该公司跳过了一次完整的数字升级(三星Galaxy Note 6与捉鬼敢死队III 和弗雷迪对杰森对阿什一起被载入史册)。但是就像那些令人遗憾地被放弃的续集一样，创作者对这个系列有其他的计划。

就Note 7而言，这意味着三星Galaxy Note产品线的统一，将平板电脑与其70多岁的兄弟姐妹S7和S7 Edge并列；最值得注意的是，字里行间的美学一致性是放弃第六部的真正原因。

这意味着，正如所承诺的那样，5.7英寸智能手机将采用双边曲面显示屏，首次将几乎无边框的曲面引入生产线。该设备的后部也有了熟悉的曲率，这是该公司长期以来对该系列的使命声明的一部分:以越来越小的外形尺寸提供大显示屏。该公司承诺在这个过程中去掉所有“不必要的边”。

事实上，这个音符以一种很好的方式不断变小。这个最新版本非常舒适地握在手中，不再是昔日笨重邪恶的手机/平板电脑的混合体。这在很大程度上是因为该公司已经设法将手机的宽度从76.2毫米削减到73.9毫米。虽然这个版本实际上比前一个版本厚了几分之一毫米，但该公司已经设法将重量减少了几克，降至169克。

除了外形之外，显示屏几乎是一样的——这非常好，保持了2560 x 1440的四核高清，每英寸518像素。就像我说的，非常好。然后一些。自然，处理器已经升级到八核(2.3GHz四核+ 1.6GHz四核)骁龙——与S7上的处理器相同——再次具有跨设备奇偶校验。

存储现在是64GB标准——三星已经取消了32GB的变体，只剩下一个板载选项，但允许用户通过microSD插槽扩展到256GB。这里仍然有4GB的RAM，而电池已经得到了显着的升级，增加了17%，从3000毫安时增加到3500毫安时，该公司承诺，由于新的USB-C型充电器(手机还附带一个免费的适配器)，充电时间应该差不多。与此同时，后置摄像头为1200万像素，与S7相同。

除了标准的金色、黑色和银色之外，这款笔记本也有了新的颜色——该公司称之为“珊瑚”蓝，这是一种柔和的金属色，该公司称之为“不太男性化，也不太女性化”挺好的；肯定没有该公司一直在推广的深蓝色那么刺眼和工业化。

新款Note首次具备防水和防尘功能，S Pen的防护等级为IP68，这意味着，在现实世界中，该设备的防水深度可达1米——该公司非常乐意在手机首次亮相时展示这一事实，以及令人印象深刻的事实，即手写笔可以在潮湿的显示屏上书写，如果你不得不在倾盆大雨中记下一些重要的东西。它发生在我们最好的人身上。

手写笔本身也被缩小了。它现在重3克，有一个0.7毫米的尖端——大约是一支圆珠笔的大小。它的新功能包括能够将备忘录钉在永远在线的显示屏上，比如永久性的便笺簿，文本放大高达300%，并将鼠标悬停在文本上进行实时翻译(总共支持31种语言)。在绘图时，它现在能够像水彩画一样混合颜色，那些值得保存的绘图可以通过集成的Pen Up社交网络共享。

还有一些新的安全功能值得(咳咳)注意，即虹膜扫描屏幕解锁，它的速度令人印象深刻——事实上，我在拍摄静止图像时遇到了麻烦。三星Knox也包括在内，还有指纹扫描仪和新的“安全文件夹”，旨在为那些与不同用户(如家人)共享设备的人保护文件安全。

新纸币的预购从明天开始。它将于8月19日袭击像塔吉特、百思买和亚马逊这样的零售商。定价仍然是TBD，很可能会通过该公司的运营商合作伙伴(美国电话电报公司，斯普林特，T-Mobile，美国蜂窝和威瑞森)正式宣布。然而，我们确实知道，该公司将提供一系列不同的捆绑包。那些提前进入的人还将获得一个免费的Gear Fit2或256GB存储卡，每个人都将获得五个免费下载的游戏包之一。
