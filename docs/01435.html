Code.org推出Code Studio，这是一个教授儿童编程的工具集和课程

自从创立以来，Code.org的使命就是将编码融入全国学校的课程中。今天，这个非营利组织推出了 Code Studio ，这是一套工具和课程的组合，通过指导性的教案让幼儿园到高中的学生对编码背后的基本概念感兴趣。

Code Studio不是让孩子们学习Python或Java之类的语言(就像在大学或AP计算机科学课上一样)，而是通过操作逻辑块来教授编程中的基本概念，当这些逻辑块以特定顺序堆叠在一起时，可以在场景中移动角色或绘制形状。该界面的工作方式很像麻省理工学院的Scratch，尽管Code.org产品总监莫娜·阿克毛在一次谷歌聚会上告诉我，麻省理工学院的产品和Code Studio之间有一些关键的区别，其中主要是HTML5的使用(因此它可以在大多数浏览器中运行)和面向K-12学生的基于谜题的课程计划。

在实际的用户界面之外，Code.org能够利用围绕其事业的知名度(通过其“一小时的代码”活动)邀请马克·扎克伯格和比尔·盖茨等大牌人物拍摄视频片段介绍概念，并在课程中使用《愤怒的小鸟》等热门品牌的资产:

Code Studio的另一大优势是为教师提供了一个界面，用于监控学生的课程进度。这是一个自然的增加(看起来Code.org希望看到这种课程在学校出现)，允许教师管理他们自己的班级或图书管理员/计算机实验室工作人员协调各年级学生的课程:

虽然Code.org希望学校将向学生介绍Code Studio，但他们也觉得他们提供了一个环境，鼓励孩子们开始制作自己的游戏和应用程序，并在完成适合他们年级水平的课程后分享它们。为了鼓励这种分享，Code Studio允许用户创建一个到他们项目的链接，让朋友和家人通过短信和社交网络分享他们的成果。
