# 三星的 Gear 智能手表终于有了 iPhone 应用

三星今年在 CES 上的移动内容非常少，但这个硬件巨头确实在本周结束前偷偷溜进了一点移动新闻。

昨天，该公司终于放弃了期待已久的 iOS 版 Gear S 应用程序，该应用程序为其可穿戴设备系列带来了 iPhone 和 iPad 兼容性——即 Gear Fit 2 band，以及 Gear S2 和[新款 S3 智能手表](https://web.archive.org/web/20230109063927/https://techcrunch.com/2016/12/19/samsung-gear-s3-review/)。

就三星而言，它承诺了设备之间的广泛兼容性，包括标准通知、Find my Gear、健康数据同步以及将应用程序下载和安装到 Gear 设备上的能力。

这一到来标志着人们热切期待的承诺得到了兑现，但 iOS 应用程序尚未与三星的所有 Gear 可穿戴设备兼容，而且它已经开始在 iTunes 上获得褒贬不一的评论，用户抱怨功能集有限，包括无法使用三星支付，这是 Gear S3 的主要卖点之一。
