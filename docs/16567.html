Sylabs 推出 Singularity Pro，这是一个用于高性能计算的容器平台

开源奇点容器引擎背后的商业公司 Sylabs ，今天公布了其首款商业产品奇点 Pro 。

Sylabs 于 2015 年推出，旨在创建一个专门为科学和高性能计算用例设计的容器平台，创始人兼首席执行官 Gregory Kurtzer 表示，这两个领域在过去几年的容器化运动中落后了。(关于容器的解释，参见本文。)

Docker 成为开发人员首选的引擎容器，但 Kurtzer 表示，早期开发的容器解决方案侧重于微服务。他说，这本身没有什么错，但它忽略了一些依赖于处理作业而不是服务的计算类型，特别是高性能计算。

Kurtzer 并不仅仅是从开源芜菁卡车上摔下来的，他作为高性能计算架构师在美国能源部实验室工作了 20 多年，在那里他创建了开源企业 Linux 项目 CentOS 和 Warewulf ，他说这已经成为利用率最高的无状态 HPC 集群供应器。

2016 年 4 月，他决定将注意力转移到容器上，成立 Sylabs，并于 2016 年 4 月推出了第一个开源版本的 Singularity。即使在那时，他也有创造该产品商业版本的愿景。他将 Singularity 视为 HPC 环境的 Docker，并以与 Docker 相似的方式运营他的公司，以开源项目为先导，然后在此基础上建立商业业务——就像 Docker 所做的那样。

Kurtzer 现在希望将 Singularity 引入企业，不仅关注 HPC 商业市场，还关注其他高性能计算工作负载，如人工智能、机器学习、深度学习和高级分析。

“这些应用程序承载着需要类似 HPC 的资源的数据密集型工作负载，随着越来越多的公司利用数据来支持他们的业务，对适当容器化和支持这些工作流的需求大幅增长，”Kurtzer 在宣布企业产品的博客文章中写道。

尽管 Singularity 旨在处理不同类型的工作负载，但它仍然可以与容器编排工具一起工作，特别是 Kubernetes 和 Mesos ，并且它还兼容微软的 Azure Batch 工具和其他云工具。

Kurtzer 指出 Sylabs 目前有 12 名员工，运营资金数额不详。它是由 RStor 资助的，这是一家目前正在秘密运作的初创公司。
