两个新的Apple Watch应用程序，Knock和oneID，让你从手腕上解锁Mac(以及更多)

原文：https://web.archive.org/web/https://techcrunch.com/2015/04/22/two-new-apple-watch-apps-knock-and-oneid-let-you-unlock-your-mac-from-your-wrist-and-more/?ncid=rss

随着移动开发人员急于发布与苹果新的可穿戴设备Apple Watch兼容的应用程序，许多人仍然想知道，与智能手机相比，什么样的应用程序体验对手腕上的小屏幕更有意义。从某种程度上来说，Apple Watch真正闪光的地方是“隐形”应用程序——例如，用Apple Pay支付物品，或者用Apple Watch打开酒店房间或汽车的门。

今天，苹果手表推出了两款新应用，延续了这一趋势: Knock 和 oneID 都是首次推出的手表应用，只需轻轻一点，就可以锁定和解锁你的Mac电脑。在oneID的情况下，该应用程序也适用于网站。

为Apple Watch敲门

敲门，你可能还记得，于2013年末首次亮相，它是一款智能的iOS应用，让你在手机还在口袋里的时候就可以敲敲你的iPhone屏幕，以便解锁你的Mac台式机或笔记本电脑。该应用利用低能耗蓝牙技术来实现设备之间的连接，并专注于开发一种解决密码问题的解决方案——即人们讨厌密码，希望有更好的方法。

但要明确的是，Knock不会取代你的密码——它只是在你敲iPhone时，通过一种安全的加密连接(1024位RSA加密)为你输入密码。

在首次亮相时，曾在Square钱包团队工作的联合创始人威廉·亨德森和在Bonobos负责用户体验的乔恩·施洛斯伯格表示，他们的长期愿景是未来Knock可以解锁任何东西，而不仅仅是电脑。但是，他们没有像他们早先想象的那样，将应用程序带到网络浏览器上，而是将注意力转向了Apple Watch，称Knock和Apple Watch是“天作之合”。

今天，该公司推出了下一个主要版本的Knock iOS应用程序，即 Knock 2.0 。

在这个版本中，Knock可以与Apple Watch配合使用。这包括敲击扫视，这是一种从手表主钟面访问应用程序的快捷方式。而且当你打开你的Mac电脑时，该应用程序也会自动弹出。

还有一个新功能是“触摸解锁”，它为那些想要更多安全性的人增加了一个双因素身份验证选项。有了这个，应用程序可以使用iPhone内置的触控ID传感器——而不是只敲你的手机，应用程序可以要求PIN或你的指纹来解锁你的电脑。

与此同时，如果你有苹果手表，你可以跳过TouchID这一步，只要你自上次把手表戴在手腕上以来至少使用过一次TouchID。换句话说，只要你戴着苹果手表，你只需轻轻一点就能解锁电脑。

现有用户可以免费升级，在App Store上的头两天价格为3.99美元。价格将会增加到4.99美元。你可以在这里下载应用。

Apple Watch的oneID

一直在开发身份即服务平台的数字身份管理公司oneID今天也通过Apple Watch推出了自己的身份认证服务。在这种情况下，该应用程序不仅可以锁定和解锁您的Mac电脑，还可以用来登录您电脑上的网站。

与Knock一样，要锁定和解锁Mac电脑，需要首先安装oneID桌面应用程序才能工作。

要开始设置Apple Watch，您需要在iPhone上安装oneID应用程序，并在那里设置您的帐户。明确地说，这款移动应用程序的设计不仅仅是锁定和解锁您的计算机，它还可以让您安全地存储和管理您所有喜爱的网站和服务的帐户信息，并为那些需要额外安全性的网站打开双因素身份验证。

一旦在iPhone应用程序上设置和配置了您的帐户，您的规则和设置就会同步到Apple Watch。

虽然Knock只专注于锁定和解锁你的电脑，但oneID的Apple Watch应用程序旨在使其服务提供的所有东西更容易从你的手腕上访问。这包括为双因素身份验证、推送通知、解锁或锁定设备等提供快捷方式。

实际上，这意味着，当你靠近Mac电脑时，你可以启动oneID应用程序，然后使用苹果的“强制触摸”来选择你想要锁定或解锁的设备，方法是按下Apple Watch的屏幕。(该公司目前正在进行一项更新，当你进入Mac电脑的范围时，它也会弹出解锁请求。)

这类似于oneID iPhone应用程序目前的工作方式——你可以通过长按来解锁或锁定你的Mac，而不是“强制触摸”。

但oneID与Knock最大的区别在于它支持网站认证，这是通过浏览器扩展来实现的。目前，该扩展可用于 Chrome 和Firefox，但Safari支持即将推出。

这里的体验更加无缝。像任何其他密码管理器一样，oneID可以在网站上自动填充您的用户名和密码信息，并自动让您登录。不过，你的Apple Watch现在可以在需要的地方充当双因素认证机制。这是通过推送通知的方式发送到你的手机，你可以点击接受或拒绝。
