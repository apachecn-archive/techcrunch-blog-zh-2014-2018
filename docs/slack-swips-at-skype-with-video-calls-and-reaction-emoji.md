# Slack 通过视频通话和反应表情符号在 Skype 上滑动 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/12/13/slack-swips-at-skype-with-video-calls-and-reaction-emoji/>

拥有[125 万付费用户](https://web.archive.org/web/20230124110716/https://techcrunch.com/2016/10/20/slunk/)和总共 400 万付费用户的工作场所协作平台 Slack ，正在为其用于人们相互交流的工具增加一个关键功能，这可能有助于它从 Skype 和 Gmail 等其他服务中获得更多用户:这是[增加视频通话](https://web.archive.org/web/20230124110716/https://slackhq.com/slack-calls-now-with-100-more-video-50bf365a0d98#.jvfmsxc1l)，这是其今年早些时候发布的音频聊天服务的视听补充。Slack 表示，付费层的群组视频通话目前最多可以容纳 15 个人。

Slack 表示，视频通话功能将在未来几天内首先在 Slack for Mac 和 Windows 桌面系统以及谷歌 Chrome 上推出。手机用户将可以加入视频通话，但只能以音频形式体验。

这项服务的工作方式与它现有的语音通话功能相似，该功能是今年早些时候增加的，允许用户打开彼此的摄像头，并听到彼此的声音。但它还将推出一个与当今许多其他视频服务不同的特色:表情符号。

许多使用视频和音频会议与同事通话的人会打开静音按钮，这样他们随意的咳嗽、叹息或其他环境噪音就不会打断同事的独白。现在，如果他们想对事情做出快速反应，他们不必四处摸索取消静音按钮:相反，他们可以竖起大拇指，或碰拳头，或跳舞的香蕉，或任何目前合适的东西。

我们早在三月份就听说该公司将在今年的某个时候推出视频聊天，现在终于实现了。不清楚它是否建立在 Slack 在 2015 年收购 YC 公司 Screenhero 的技术基础上，当时 Slack 已经为人们使用视频聊天和相互分享屏幕建立了一种非常简单的方式。(我们已经问过了，并将在了解更多信息后更新。)

无论如何，Slack 表示，通过建立自己的视频聊天服务，它目前不打算取消使用其他服务的能力。Slack——建立在通过简单使用斜杠命令将数百个其他应用和服务集成到其更大平台的想法上——也可以与谷歌 Hangouts 和 Zoom 等服务配合使用，如果你喜欢它们而不是 Slack 的原生体验，你仍然可以[将它们设置为默认](https://web.archive.org/web/20230124110716/https://get.slack.help/hc/en-us/articles/216951758-Customize-your-team-s-call-settings)。

虽然视频通话将有助于 Slack 作为一个完整和独立的平台充实其产品，但它也可以在其他方面有所帮助:Slack 已经看到[在其仍然相当大的用户增长中出现了一些下降](https://web.archive.org/web/20230124110716/https://techcrunch.com/2016/10/20/slunk/)，我们听说除此之外还有一些用户保留方面的问题。与此同时，视频是更受欢迎的服务之一，通过让人们逗留更长时间来增加网站的“粘性”;看看它在这里是否也能奏效将会很有趣。

迄今为止，Slack 已经[筹集了](https://web.archive.org/web/20230124110716/https://www.crunchbase.com/organization/slack)近 5.4 亿美元，估值 38 亿美元。
