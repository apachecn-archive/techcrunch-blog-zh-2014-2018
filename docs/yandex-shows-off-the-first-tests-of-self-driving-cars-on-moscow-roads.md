# Yandex 展示了无人驾驶汽车在莫斯科道路上的首次测试

【YouTube https://www.youtube.com/watch?v=Bx08yRsR9ow&w=680&h=383]

俄罗斯搜索和互联网服务巨头 Yandex 一直在致力于其自动驾驶计划，其最新的成就是其自动驾驶测试车在莫斯科的公共自动驾驶试点，这是俄罗斯首都公共道路上的首次此类测试。

Yandex 还有一个额外的困难需要应对:他们最近的测试于去年 12 月开始，包括大雪后的几天，这对于任何自动驾驶汽车来说都是困难的条件，因为它们会模糊光学相机系统使用的路标和其他导向指示器，以及其他挑战。

【YouTube https://www.youtube.com/watch?v=7hUut7Hsgys&w=680&h=383]

正如你所看到的，即使在冬季条件下，汽车也能应对雪、冰、停放的汽车、交通甚至行人，这是自动驾驶汽车技术提供商在该技术被广泛采用之前需要解决的一个关键用例。

除了雪，Yandex 指出，莫斯科还为其 AVs 提供了一个很好的测试平台，因为包括高交通量在内的因素，以及行人密度加上城市跨季节的可变天气模式。
