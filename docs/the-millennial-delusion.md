# 千禧年错觉

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/06/27/the-millennial-delusion/>

对“千禧一代”的痴迷继续让我着迷。尽管是历史上最直言不讳的一代，人们——非常重要和有权力的人——声称他们不理解我们。我们显然没有意义，好像我们父母的行为和职业道路完全有意义。

甚至还有专门教企业如何与我们互动的咨询公司(我拒绝链接他们，如果你敢的话谷歌)。我希望我能开一家这样的公司，整天谈论自己是一份充满激情的工作，在这个过程中成为千禧一代的精髓。

或许是一个新千年一代。

关于这种人为的聚集，商业媒体的报道比任何其他媒体都多，然而争论似乎仍在无休止地进行。

那是因为真的没有争论，真的没有“千禧一代”这样的概念如果还不清楚的话，千禧一代的价值观就是美国的价值观，这一点可能在本周最高法院关于同性婚姻、T2 医疗保健和 T4 住房歧视的判决中更加明显，这些判决明显更符合千禧一代的想法，而不是婴儿潮一代。

千禧一代是我们想象中的虚构，是营销人员和其他人的错觉，他们认为我们社会的变化只适用于一小部分人，而不是整个人口。

他们完全错了。

正在发生的是，人们终于利用了我们在过去几十年中取得的所有技术进步，在这个世界上找到了以前缺乏的力量。我们现在都有能力选择自己的道路——我们自己的“激情事业”——并利用技术创造更美好的未来，而不仅仅是拥有足够资源的精英阶层。

不出所料，每个人似乎都在这么做。

我们随处可见科技对社会的影响。千禧一代被描述为比其他任何一代人都更有“社会意识”，但这是我们大量使用技术，特别是社交网络的结果。今天的人们比以往任何时候都更容易获得来自美国和世界各地的新闻和观点，其他地方的冲突或疾病与我们产生了以前不存在的情感共鸣，这并不奇怪。

以耐克和苹果等品牌为例，它们不得不应对海外工厂的恶劣条件。他们被迫做出改变和改善这些条件，不是因为法律的改变或政府的倡导，而仅仅是因为他们的客户开始有办法关注他们的行为。

作为社会进步的工具，我们智能手机中的相机可能比目前任何其他相机都更有影响力。有了 Periscope 和 Meerkat 这样的直播视频流应用，我们不只是在以后观看这些图像，而是实时感受一个场景的本质。你怎么能眼睁睁地看着伤害发生，却不想去改善它呢？这不是千年一遇的事情，而是人的一种冲动。

科技让我们对世界上正在发生的事情有了深刻的认识。这也是为什么我们这一代人拒绝加入传统的劳动力市场，而是自己铺路的原因之一。我们在这个世界上经历了太多的错误，也许是时候让我们每个人都尝试做些什么来让它变得更好一点了。如果无知是福，那么科技让我们永远渴望解决每一个问题。

当我们将这种情绪与美国对创业的持续推动结合起来时，千禧一代比前几代人更自由思考和独立就不足为奇了。爬上公司阶梯的想法看起来如此陌生，因为如此多的传奇公司在我们的有生之年已经不复存在，而且这个过程看起来与真正改善人类大相径庭。

技术降低了建立我们自己的公司和服务机构的成本，这也有所帮助。今天，任何人——不仅仅是千禧一代——都有这种替代选择，那就是无视所有人，走自己的路。世界的停滞已经被基于技术的灵活性所取代，这种灵活性由云和移动设备提供支持。

然而，授权是有代价的。如果这个国家曾经有过争论，那就是我们这个时代的伟大工程仍然需要重要的团队来建设。不是每个人都能成为创始人的。虽然我们有合作和面向社区的精神，但这并不一定意味着想要加入别人的创业公司或非营利组织。事实上，我们可能想开始我们自己的。

尽管如此，我们必须接受这样一个事实:这个世界不会围绕着几百家大公司转，而是数百万家小公司。来自“千禧一代”的最大挑战只是，我们以分散的方式思考世界，而企业仍然专注于集中。应该有一个品牌，而不是几百万个适应每个消费者的品牌。

令人欣慰的是，在过去十年中，我们已经看到了营销领域的巨大变化，以促进消费者与品牌的互动，但我们在产品和服务层面仍需要更多。

非营利部门和政府部门更能感受到这种集中化的冲动。红十字会的浪费危机是当今规模问题的一个完美例子。该组织怎么可能自下而上而不是自上而下地思考，并在实际使用其捐款时可能节省数亿美元？

当然，红十字会浪费的故事只是整个千年叙事的另一种复述。一组记者现在可以在当地进行调查，并在网上挑选文件来证明一些事情，这些事情在以前即使不是不可能，也是很困难的。技术赋予了捐赠者权力。

技术给了我们比以往更多的意识和选择。我们如何在丰富的选择中导航与千禧一代关系最密切，因为我们天生就理解这一点，但其他人也很快加入进来。这个新世界可能有所不同，但也更好，更加平等和安全。如果这就是技术能做的，那我们都该成为千禧一代了。
