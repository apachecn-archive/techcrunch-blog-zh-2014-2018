# Stitch Fix 已秘密申请上市 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/07/28/stitch-fix-has-confidentially-filed-for-an-ipo/>

消息人士告诉 TechCrunch，Stitch Fix 已经秘密申请首次公开募股，为不久的将来 IPO 市场的又一次重大考验做准备。

虽然我们不知道 Stitch Fix 将于何时上市，但我们知道一些公司正计划在劳动节和感恩节之间上市。Stitch Fix 最近才在 6 月份聘请了首席财务官保罗·易(Paul Yee)——所以他们可能会进一步推进这一计划。这标志着 S-1 申请程序的正式开始，该公司将正式披露其财务状况。根据 2012 年签署的《就业法案》,初创公司可以秘密申请 IPO，允许他们在正式公布公司内部运作之前试水。

Stitch Fix 也将标志着现代电子商务初创公司及其作为上市公司的生存能力面临的重大考验之一。Stitch Fix 收集关于你的个人风格的信息，并把衣服送到你家门口，让你选择买哪件，退哪件。这是一个竞争日益激烈的领域，其中最大的竞争可能是亚马逊(Amazon)最近推出的 Prime 衣柜。

Stitch Fix 的一名代表拒绝置评。

Stitch Fix 是采用先试后买方法的订阅式造型初创公司浪潮的一部分。Le Tote 在空间和出租跑道方面也很有竞争力，并推出了“无限”订阅。还有专门为职业女性设计的 MM. LaFleur、为加大码设计的 Dia & Co 和只卖珠宝的 Rocksbox。这种模式非常适合那些想要尝试最新款式、只保留自己喜欢的款式的忙碌人士。

但 Stitch Fix 作为上市公司可能面临的一个挑战是，可能很难预测收入。一些订阅业务，如我们看到的蓝色围裙，一直在努力留住客户。该文件还会告诉我们，用户是否购买了足够的商品来抵消客户的购买成本。

Stitch Fix 于 2011 年在首席执行官卡特里娜·莱克(Katrina Lake)位于马萨诸塞州的公寓里成立，最初专注于女装。一部分是时尚初创公司，一部分是科技初创公司，Stitch Fix 已经演变成一家大型电子商务初创公司，是 Lake 在激烈竞争中的一个巨大成功。PitchBook 的数据显示，上一轮 Stitch Fix 融资是在 2014 年 6 月，后估值为 3.0931 亿美元。这家初创公司已经从 Benchmark 等投资者那里筹集了 4250 万美元，人们普遍认为它将于今年进行 IPO。

事实上，这将是一次缝针测试。在经历了一个火热的开局之后，当我们走过 2017 年的中点时，消费类 IPO 在公开市场上的表现并不那么好。Snap 上市首日表现强劲，但此后股价不断创下历史新低。毫不奇怪，随着亚马逊即将进入餐包递送领域，蓝色围裙的销量也大幅下降。目前尚不清楚投资者是否会看到 Stitch Fix 在亚马逊的威胁下是否面临与蓝色围裙相同的困境。不过，Stitch Fix 有一个强大的品牌，作为一家不同类型的初创公司上市可能会更好。

最近，Recode 报道称，Stitch Fix 的[申请可能即将](https://web.archive.org/web/20221206045910/https://www.recode.net/2017/7/25/16019942/julie-bornstein-stitch-fix-ipo-coo-departure)提交，5 月[路透社报道称，该初创公司为其首次公开募股接洽了银行](https://web.archive.org/web/20221206045910/http://www.reuters.com/article/us-stitch-fix-ipo-idUSKBN18R36N)。据 Recode 报道，该公司的首席运营官也在 IPO 前离职。TechCrunch 现在可以确认该公司已经秘密提交了申请。

Stitch Fix 的下一步将是公开发布信息，15 天后开始“路演”，在公开亮相之前向投资者推介该公司。
