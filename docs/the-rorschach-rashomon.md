# 亲爱的美国，请你给爱德华·斯诺登颁发自由勋章好吗？

发生了什么事？你猜对了:每个人最喜欢的英雄/恶棍/恶魔/圣人，爱德华·斯诺登，一年前他在俄罗斯获得了庇护。本周，科技行业支持一项法案，该法案提议“[全面限制美国国家安全局的监控](https://web.archive.org/web/20221208075337/http://www.reuters.com/article/2014/07/29/us-usa-nsa-congress-idUSKBN0FY1EM20140729)”，并且“将代表自国会 13 年前通过《美国爱国者法案》以来，政府监控机构最重大的改革。”它实际上可能会通过——再次感谢斯诺登。

那么这个人什么时候得到他的奖章？

很多人，包括[黛安娜·范斯坦](https://web.archive.org/web/20221208075337/http://thehill.com/policy/defense/304573-sen-feinstein-snowdens-leaks-are-treason)和[约翰·克里](https://web.archive.org/web/20221208075337/http://www.politico.com/story/2014/05/edward-snowden-coward-john-kerry-msnbc-interview-nsa-107157.html)(还有[马克·安德森](https://web.archive.org/web/20221208075337/http://www.businessinsider.com/marc-andreessen-snowden-is-a-traitor-2014-6))，仍然认为斯诺登是叛徒。请注意，理论上，叛国罪[需要](https://web.archive.org/web/20221208075337/http://www.merriam-webster.com/dictionary/traitor)“帮助或支持敌人。”我不确定他们心目中的敌人是谁:俄罗斯？中国？爱德华·爱泼斯坦暗示斯诺登事件自始至终都是外国间谍活动，但此人本人声称他没有将任何秘密文件带到俄罗斯，也能够保护它们免受中国间谍的攻击。

地球人算不算敌人？

反斯诺登阵营普遍声称，他应该在系统内工作，揭发这件事，和/或应该回到美国面对随后的音乐——尽管此后变得明显的是，美国国家安全局对斯诺登试图在不公开的情况下表达他的担忧并没有完全公开。

撇开自我保护不谈，很明显，如果斯诺登返回并被关进切尔西·曼宁式的单人牢房，他在过去一年的工作效率会大大降低。(“[他被指控的法律不允许他在陪审团面前陈述案情](https://web.archive.org/web/20221208075337/https://beta.techcrunch.com/video/biggest-social-impact-edward-snowden-revelations-crunchies-awards-2013/518119838/)，”根据欧洲足联的埃沃·蒂姆代表斯诺登接受采访时说。相反，他可以在 TED 与谢尔盖·布林聊天，与丹尼尔·埃尔斯伯格一起出现在座谈会上，等等。，并通过远程呈现将他的信息发送出去。

但是你知道吗？这是一个有争议的问题。即使斯诺登一直是一个彻头彻尾的外国间谍，根据基于结果的分析，他仍然应该获得一枚奖章——因为，尽管我们有残余的冷战反共产主义下意识反应，但事实是，我们生活在一个对美国生活方式最大威胁是美国本身的时代。

(不，不是基地组织之流。当然，他们是一个问题，但他们在美国的非理性，偏执，仇外心理，对他们的大规模过度反应和类似的威胁面前苍白。你不得不怀疑，美国威权主义者认为他们应该有全权以国家安全的神圣名义为所欲为，因为一群疯狂的疯子在 13 年前走运了。又一个十年？又一个世纪？永远？)

本周[有一篇引人入胜的《外交政策》文章](https://web.archive.org/web/20221208075337/http://www.foreignpolicy.com/articles/2014/07/29/the_social_laboratory_singapore_surveillance_state)，讲述了新加坡试图利用大规模监控和古老的大数据来“构建一个更加和谐的社会”这本质上是支持 NSA 的人所支持的，即使他们没有意识到:

> 许多美国间谍前往新加坡亲自研究这个项目。他们不仅被新加坡对大规模监控的拥抱所吸引，还被这个国家民主和威权主义的奇怪组合所吸引……在新加坡，对居民和游客的电子监控无处不在，并被广泛接受……(政府)拥有法律权力，可以监控各种形式的电子通信，包括电话通话，这符合几部旨在防止恐怖主义、起诉毒品交易和阻止印刷“不良”材料的国内安全法

我敢肯定，新加坡的[无令状圆形监狱](https://web.archive.org/web/20221208075337/http://www.techinasia.com/singapore-advanced-surveillance-state-citizens-mind/)对那些去新加坡学习的间谍来说似乎很有吸引力。当警察被交给警察国家的工具时，他们很少提出异议，而支配或受益于一个系统的人很少反对其权力的任何扩张。但无处不在的监控先发制人地压制异议，不管这是不是它的意图；不管他的意图如何，爱德华·斯诺登的爆料帮助美国和整个西方暂时远离了滑坡。我期待着颁奖仪式。
