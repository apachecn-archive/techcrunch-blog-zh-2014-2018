# Sphero 聚焦教育，募集 1200 万美元

今年对 Sphero 来说是艰难的一年。总部位于科罗拉多州的玩具机器人初创公司以[数十人裁员](https://web.archive.org/web/20221206214449/https://techcrunch.com/2018/01/21/sphero-lays-off-dozens-as-it-shifts-focus-to-education/)开始了这一年，这是对其迪士尼品牌消费产品系列兴趣不温不火的结果。

然而，这里有一点好消息。据 Crunchbase 称，该公司又筹集了 1200 万美元，使其总额达到约 1.19 亿美元。最新一轮将帮助 BB-8 制造商成为一家教育第一的公司。

Sphero 在提供给 TechCrunch 的一份声明中表示:“最近一轮融资目前已筹集到 1200 万美元，我们预计在最终交易结束时，可能总共筹集到 2000 万美元。”资金已经/将来自现有和新的投资者，并将用于营运资金，因为我们参与了一个更大的战略，重点是娱乐和学习的交集。”

考虑到目前编码玩具的世界是多么拥挤，这是一个棘手的游戏，但 Sphero 长期以来一直在这个领域建立自己的游戏，同时推出更多以消费者为中心的产品。

在《原力觉醒 BB-8》的成功合作之后，该公司在迪士尼加速器方面的投入翻了两番，发布了基于蜘蛛侠和闪电麦昆的高科技玩具。

裁员消息披露后，该公司告诉我:“(教育)是我们实际上可以拥有的东西。”。“我们做得好的是那些我们可以 100%拥有的体验，从开始到走向市场。”
