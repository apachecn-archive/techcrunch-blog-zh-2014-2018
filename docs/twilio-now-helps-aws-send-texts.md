# Twilio 现在帮助 AWS 发送文本 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/07/05/twilio-now-helps-aws-send-texts/>

今天[宣布](https://web.archive.org/web/20230110105618/https://www.twilio.com/press/releases/release-aws-sns)与亚马逊 AWS 平台的新合作。该公司表示，它正在“帮助 AWS 通过[亚马逊简单通知服务(SNS)](https://web.archive.org/web/20230110105618/https://aws.amazon.com/sns/) 提供短信递送服务”，并指出 SNS 用户现在将受益于 Twilio 发送批量信息的经验。

那到底是什么意思？我们也不确定，就问了公司。下面是 Twilio 发言人告诉我们的:“我们不能讨论这一关系的具体细节，但可以说亚马逊社交网络利用了 Twilio 超级网络的优势，该网络可以将短信发送到全球 200 多个国家。"

据我们所知，SNS 消息服务现在至少部分依赖 Twilio 的网络为其部分用户发送消息。遗憾的是，目前还不清楚哪些信息会通过 Twilio 的网络传送。SNS 服务已经支持全球[发送短信](https://web.archive.org/web/20230110105618/http://aws.amazon.com/sns/faqs/)，所以大多数用户可能不会马上看到任何重大变化。

值得注意的是，亚马逊的社交网络目前不提供许多高级功能，例如短码(或长码)，社交网络使用一个代码池来发送消息，这意味着接收者不会总是看到来自使用社交网络的单个公司的相同号码。Twilio [确实提供了那些功能](https://web.archive.org/web/20230110105618/https://www.twilio.com/sms/shortcodes)(以及更多)，所以希望这次合作也意味着 SNS 将很快获得更先进的短信功能。

Twilio 首席执行官兼联合创始人杰夫·劳森(Jeff Lawson)在今天的公告中表示:“通过这次最新的合作，我们正在用可扩展的高效工具加强通信生态系统，确保提供传递信息的最佳选择。”当然，他也没有透露任何细节。

值得注意的是，亚马逊在 Twilio 上市前投资了该公司的 E 轮融资，Twilio 也广泛使用了 AWS 的云计算服务。

我们向亚马逊询问了关于此次合作的更多细节，一旦得到回复，我们将更新这篇文章。
