# 智能气候控制初创公司 Tado 为欧洲扩张再融资 1000 万€

谷歌旗下的 [Nest](https://web.archive.org/web/20221210000708/https://beta.techcrunch.com/2014/06/23/google-makes-its-nest-at-the-center-of-the-smart-home/) 的欧洲竞争对手 Tado 和当地的智能恒温器公司，如英国的[heat 吝啬鬼](https://web.archive.org/web/20221210000708/http://www.heatmiser.com/)或英国天然气公司的 [Hive](https://web.archive.org/web/20221210000708/https://www.hivehome.com/) ，今天补充了其资金，披露了新一轮€1000 万英镑(1360 万英镑)的投资。新的投资来自于现有的支持者[Target Partners](https://web.archive.org/web/20221210000708/http://www.crunchbase.com/organization/target-partners)和 [Shortcut Ventures](https://web.archive.org/web/20221210000708/http://www.crunchbase.com/organization/shortcut-ventures-gmbh) ，以及其他一些不知名的投资者，并将被这家德国初创公司用于将其智能供暖和空调产品推向更多的欧洲市场。

“有了新的资金，我们努力扩大我们在智能气候控制领域的领导地位，”联合创始人兼首席执行官[克里斯蒂安·迪尔曼](https://web.archive.org/web/20221210000708/http://www.crunchbase.com/person/christian-deilmann)在慕尼黑和旧金山之间的飞行途中告诉我。具体来说，他说该公司的第一款产品——Tado heating([—](https://web.archive.org/web/20221210000708/https://beta.techcrunch.com/2013/09/05/tado/)——将被带到北欧更多地区，增加在德国、奥地利、瑞士和英国的现有供应。

最近，这家初创公司通过在 [Kickstarter](https://web.archive.org/web/20221210000708/http://www.crunchbase.com/organization/kickstarter) 上的众筹活动推出了 [Tado Cooling](https://web.archive.org/web/20221210000708/https://beta.techcrunch.com/2014/05/12/air-connectivity/) 。此举旨在测试市场，特别是美国市场，帮助衡量对这种产品的需求，并将其与生产相匹配，而不是像许多硬件公司那样，测试资本本身。

“在国际层面上预测 Tado——一个全新的产品类别——的销量是非常困难的，”迪尔曼当时说。“Kickstarter 让我们有机会在收到订单后将生产规模扩大到合适的规模。”

作为这场运动的最新进展，迪尔曼告诉我，Kickstarter 的 50%的支持者(他们投入了 20 多万美元)来自美国，其次是新加坡和南欧，这些国家将是未来 Tado 降温的重点。

与此同时，Tado 在智能 AC 领域最接近的竞争对手可能是[senso](https://web.archive.org/web/20221210000708/http://www.sensibo.com/)，它目前正在 [IndieGoGo](https://web.archive.org/web/20221210000708/https://www.indiegogo.com/projects/sensibo-make-any-air-conditioner-smart) 上众筹。
