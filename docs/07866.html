Twitter 主席:即使是痛苦的政治推文也有一席之地

原文：https://web.archive.org/web/https://techcrunch.com/2016/06/13/twitter-chairman-even-painful-tweets-have-a-place/

Twitter 董事长奥米德·柯德斯塔尼表示，即使是最严厉、最痛苦的政治推文也在 Twitter 上有一席之地——不管我们喜欢与否。

Kordestani 在彭博技术大会上说:“你个人对此有何反应，阅读这些推文是痛苦的——这是我们的社会，你必须为所有这些声音留出空间。”“话语对民主很重要；不是坦克和军队在街上滚动，而是讨论在进行。”

这一评论是对美国总统选举期间许多人在推特上发表的大量反移民言论的质疑的回应。作为 Twitter 董事会主席，Kordestani 本人表示，他直接与 Twitter 的高管接触，帮助指导公司的未来。他说，他的角色是对公司“有帮助”，每周去那里几天，就“艰难的决定”和招聘等其他流程提供建议。

因此，很自然地，他不得不坚持公司的党派路线——这是一个公共话语可以自由发生的地方，这让许多 Twitter 观察者非常懊恼。自始至终，Twitter 都将自己视为围绕阿拉伯之春等国际事件和其他大规模事件(包括 2008 年总统大选)展开讨论的发射台。Twitter 也一直存在用户在平台上处理骚扰和威胁的问题。

“就我们而言，我们已经看到了这一点，它每天都在现场发生，”Kordestani 说。“看到他们之间、他们与我们的公民之间的对话，以及来自世界各地的评论，真是太棒了。我们的重点是在这方面做得越来越好。整合现场视频，利用 Periscope，参加这些活动，从媒体团队、moments 团队中带来我们的专业知识，展现正在发生的最佳对话。很大一部分是关于，这些参与的观众如何相互交流。我们已经看到候选人利用媒体相互评论。”

尽管如此，他仍有责任坚持公司和新任首席执行官杰克·多西的愿景。在董事会最终决定多尔西之前，Kordestani 曾一度被考虑担任首席执行官一职，此前他曾担任谷歌首席商务官。Kordestani 表示，在他长期任职于谷歌之后，他希望能够帮助多尔西和公司，这就是他担任董事长职务的原因。

一个特别有趣的注意是他对 Twitter 是否可能成为收购目标的立场，因为今天有消息称微软以 262 亿美元收购了另一个巨大的职业社交网络 LinkedIn。正如预期的那样，Kordestani 给出了一种不回答，但为潜在的讨论敞开了大门。(毕竟，一个“不”可能看起来是最终决定，如果 Twitter 真的决定出售自己，他以后还得自相矛盾。)

“作为董事会，你必须对股东负责，评估每一个机会，”Kordestani 说。“作为管理团队和领导团队，你每天都专注于做好工作。该公司的重点将是创新和完善产品。关注创作者和开发者。所有这些都是你必须每天都有信仰的东西。无论结果如何，无论是合作还是其他机会，我们都会进行评估。”
