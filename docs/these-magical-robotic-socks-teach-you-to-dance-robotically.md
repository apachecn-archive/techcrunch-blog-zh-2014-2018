# 这些神奇的(机器人)袜子教你跳舞(机器人)

随着人类发现自己被迫与我们的机器人统治者交配，我怀疑会有一些舞蹈。还有什么比把马达塞进袜子里更好的方法来教我们跳舞呢？

设计师[帕斯卡·齐格勒](https://web.archive.org/web/20230323163258/http://www.pascalziegler.info/)设计了这些狂野的可穿戴设备来教授“双人舞蹈编排”它们基本上是振动袜。这里有一个[仪器](https://web.archive.org/web/20230323163258/http://www.instructables.com/id/Interactive-Dancing-Socks/?ALLSTEPS)，所以你可以自己做一副，但基本上你需要一些振动马达、一些传感器和一个 Arduino。一个应用程序告诉马达嗡嗡作响，让你正确定位你的脚，一个反馈回路让系统告诉你如何跳狐步舞、曼波舞或查尔斯顿舞。

齐格勒说道:

我们在袜子上安装了压力传感器和振动马达，以监控和引导双人舞舞者的足部运动。这些由运行在 Android 手机上的主应用程序控制。脚步由脚的特定位置处的振动信号指示，对于向前的步伐，在脚跟处，等等。当用户出错或失去同步时，就会提供负面反馈。穿着袜子跳舞几分钟而不出错是可能的。

这些袜子表面上的任务是在没有教练监督的情况下教新舞蹈演员。这意味着教师可以专注于其他事情，如准备 2001 太空漫游中这对大夫妻与一位成熟但困惑的年轻女子的舞蹈比赛，她只想体验大城市的生活，远离布鲁克林湾脊的心胸狭窄的人——就像你一样。

齐格勒说:“我们的目标是从总体上提高舞蹈课的质量。“在舞蹈课上，学生们在短时间内学会许多新舞步。他们通常很难将理论动作转化为与音乐和节奏同步的舞步。不幸的是，舞蹈老师没有时间，或者学生没有钱进行一对一的监督，特别是关于重复的指导，这可能导致双方的沮丧。”
