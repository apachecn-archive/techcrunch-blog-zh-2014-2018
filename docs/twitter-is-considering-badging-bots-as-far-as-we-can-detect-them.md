# Twitter 正在考虑给机器人贴上标签——“就我们所能检测到的而言”

在今天的参议院情报委员会听证会上，副主席马克·华纳问 Twitter 首席执行官杰克·多西，如果用户在其平台上与机器人或人类交谈，他们是否应该有“知情权”——以帮助人们更好地浏览他们接触到的信息。

多尔西同意，围绕推文和账户的“更多背景”是好的，但当华纳问及 Twitter 是否应该有一项政策来识别平台上的机器人和人类时，他说 Twitter 正在积极考虑，尽管警告说“就我们所能检测到的而言”。

“我们当然可以通过我们的 API 给账户添加标签和上下文，”他继续说道。“自动化实际上是在编写我们网站的脚本，让它看起来像一个人类演员，这变得更加棘手。因此，只要我们能标记和识别这些自动化，我们就能标记它们——我认为这是有用的背景。”

“这是我们过去几个月一直在考虑的一个想法，”他补充道。“这确实是一个实施的问题，但我们对此很感兴趣。我们将沿着这条路线做一些事情。”

在会议稍后回答关于虚假账户的问题时，多尔西表示，Twitter 在识别其平台上的不真实活动方面取得了更大的成功，方法是使用深度学习和机器学习技术，这些技术专注于识别“行为模式”——因此查看网络上的账户行为——而不是试图实时识别特定账户的位置。

“与试图识别虚假档案的系统相比，我们在规模上有了更多的优势，”他补充道。

多尔西在推特上向委员会发布了他的开场白，尽管不是在他宣读的时候实时发布。这凸显了 Twitter 上任何未来“机器人与否”标签系统的挑战，它需要筛选脚本活动，以试图确定什么是人类，什么是机器。
