# 空客 Vahana 飞行出租车实际上是首次飞行

似乎就在昨天，空中客车公司的 Vahana 自主电动垂直起降(VTOL)飞行器[还只是一个画中的概念](https://web.archive.org/web/20221207202108/https://beta.techcrunch.com/2017/01/16/airbus-plans-to-test-self-driven-airborne-taxi-by-the-end-of-2017/)，但现在它实际上已经飞行了，在持续不到一分钟的全尺寸原型测试中，Vahana 飞机完全自动驾驶，并在离地 16 英尺的高度飞行。

Vahana VTOL 类似于一架复杂的直升机或地上无人机，取决于你的视角，由空中客车硅谷 skunkworks A 开发，旨在最终成为一种可以为客户提供服务并在城市内运输人员和货物的东西，减少交通流量，并在战略部署的发射台和着陆场之间进行短途旅行。

这种首次飞行显然与一种可行的商业客运无人机服务相去甚远，但成功的首次飞行，以及随后第二天的第二次成功飞行，是朝着正确方向迈出的一步。

接下来，Vahana 表示，它的目标将是从能够悬停车辆转向能够让它自己定向飞行，这显然是将人和东西从 A 点到 b 点的关键因素。
