# 红杉资本正寻求筹集 50 亿美元的投资基金

红杉资本正在寻求筹集 50 亿美元的投资基金。

这家传奇的硅谷风险公司以其出色的投资回报而闻名，目前正处于筹集下一只全球增长基金的早期阶段。

Recode 的卡拉·斯威舍[首先报道了基金](https://web.archive.org/web/20221006062807/https://www.recode.net/2017/12/15/16780158/softbank-wag-funding-deal)的消息。她听说红杉的目标是 50 到 60 亿美元。我们听说现在看起来更像是 50 亿美元。预计融资将于 2018 年第一季度开始。

相比之下，其 2015 年的上一只增长基金规模为 20 亿美元。它的一些后期投资公司包括 Airbnb、Stripe 和 23andMe。红杉资本还有一项重要的早期投资业务。

我们[最近报道，红杉正在进行要约收购，希望收购优步的股份](https://web.archive.org/web/20221006062807/https://beta.techcrunch.com/2017/11/28/sequoia-tencent-tpg-look-to-invest-in-uber-alongside-softbank/)。这些是将从现有投资者手中购买的折价股票。日本投资公司软银集团是这项努力的领导者。

虽然红杉潜在的 50 亿美元听起来可能是一大笔钱，但与软银为其愿景基金筹集的近 1000 亿美元相比就相形见绌了。这意味着后期创业公司将有多种大额支票选择。涌入高估值“独角兽”的资金推动风险投资金额创下历史新高。

多年来，红杉取得了许多成功。他们投资了即时通讯应用 WhatsApp，后者以 190 亿美元的价格被卖给了脸书。他们也在苹果和谷歌工作过。
