编者按: 汤姆·古德温是哈瓦斯媒体的战略和创新高级副总裁。

优质新闻内容提供商正处于进退两难的境地。

他们可能做出的第二糟糕的决定是将他们的内容免费提供给脸书托管，但最糟糕的决定可能是根本不提供。

这一切都有点像 2006 年的音乐行业，当时唱片公司正在与流媒体服务进行交易。这些服务提供了接触更多人的机会——将音乐放在那些可能永远不会为专辑付费的人手中——但同时也存在内容贬值的风险，这几乎本质上是因为获取变得更容易了。

像《纽约时报》这样的报纸就像许多传统公司一样，面临着非常不公平的现实，他们为另一个时代做了准备和投资。

像出租车车主，像 Borders 或 Blockbuster 这样的商店，或者像 Kodak 这样的制造商；新闻机构拥有复杂、昂贵的资产来制作和传播在数字时代丰富、被低估和无形的东西。

《纽约时报》现在在记者的薪水上花了大量的钱，写出世界级的内容，这些内容在一夜之间被印刷出来，并通过复杂的物流运作传播到全国各地。

新闻机构拥有复杂、昂贵的资产来制作和传播在数字时代丰富、被低估和无形的东西。

另一方面，脸书编写代码，在其服务器上托管几 Pb 的数据，然后在所有赚钱的地方(客户界面)开始赚钱(利用已知的最好的个人数据)。

诚然，这些公司中有许多犯了愚蠢的错误，优步正在扼杀出租车，因为整个行业在监管墙的保护下安于现状；柯达和百视达态度傲慢，打消了任何人都能超过他们的念头；诺基亚在苹果之前就做了智能手机，却没看出来是软件和体验，而不是硬件。

像旅行社和音乐行业这样的其他行业刚刚成为软件蚕食世界的受害者。

在这种环境下，新闻业的问题不再是是否应该改变，而是如何改变。

对于目前只有广告融资模式的媒体所有者来说，在脸书托管内容似乎并不是最困难的举措。从本地唯一的 Buzzfeed，到《卫报》或 BBC，这似乎是一种赚更多钱、获得更多眼球、最有可能了解更多内容行为的方式。

对其他人来说，事情没那么简单。

来自《纽约时报》或《图片报》的优质新闻可以通过脸书免费获得，进一步传播内容，但也破坏了它的整个订阅模式，这恰好代表了他们业务的主要增长，也可能是他们未来的最大希望。

那么《纽约时报》是将此视为获取订户的免费内容营销，还是增量广告收入？只有时间会，也许他们自己都不知道？

无论这是一个艰难还是容易的选择，这都将意味着新闻业艰难时期的延续。

新闻品牌之死。

我们已经习惯了由新闻标题顶部的数字制作和策划的消费者新闻。我们会买《泰晤士报》,浏览那份报纸，我们的关系就是品牌。

我们广泛阅读各种题材的书籍，其共同点是提供新闻。

在现代，头版现在是 Twitter 或脸书，我们在一篇文章的水平上阅读，主题，观点，作者成为品牌，新闻广播的重要性逐渐减弱。这篇文章更好地反映了 TechCrunch 的主机还是汤姆·古德温的写作？当你通过社交媒体看到这些时，情况会有什么变化？

我们在音乐和电视领域看到了同样的演变。我们曾经购买专辑，或者在周六晚上观看 NBC，现在我们倾向于喜欢特定的歌曲，享受《T2 绝命毒师》，而不去想频道。当内容脱离提供商时，提供商就失去了控制权。

世事变迁。

这种失控最终可能会被证明是致命的，脸书可能看起来玩得很好，免费提供内容，并通过 100%的广告收入，但这种权力平衡会保持下去吗？

脸书在保持品牌优势方面没有最好的记录，免费访问和太好而不能持久的东西变成了付费模式。当脸书成为我们通往世界这么多地方的门户网站时，新闻品牌成了混乱中更加绝望的一方。

那通向哪里？

这是新闻内容的一个有趣时代，新闻的生产、货币化和传播方式将会改变。

Native 会成为首选路线吗？微支付或捆绑最终会帮助品牌为有价值的内容提取资金吗？来自电子商务或商业合作的其他收入来源会像一些人认为的那样大吗？

有一点是肯定的，这是一个实验，但尽管交易可以取消，消费者行为不会遗忘，浮士德式的约定也无法取消。
