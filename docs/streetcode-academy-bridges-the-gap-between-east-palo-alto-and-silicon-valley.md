# 街道代码学院架起了东帕洛阿尔托和硅谷之间的桥梁

东帕洛阿尔托位于脸书和谷歌硅谷园区之间。尽管它们非常接近，但它们通常被认为是两个不同的世界。

[东帕洛阿尔托曾被称为美国的谋杀之都](https://web.archive.org/web/20230130230646/http://articles.latimes.com/1993-01-05/local/me-833_1_east-palo-alto)，其失业率是其西边邻居的两倍。[三分之一的居民高中以上学历。这就是位于东帕洛阿尔托的慈善组织 Live in Peace](https://web.archive.org/web/20230130230646/http://www.ci.east-palo-alto.ca.us/DocumentCenter/View/437)的一部分 StreetCode Academy 的用武之地。

StreetCode 的 Olatunde Sobomehin 告诉我:“StreetCode 开始回应我们已经知道的事情，即有许多高质量的想法，创新的想法被当前的技术生态系统遗漏了。”"我们也知道，东帕洛阿尔托有很多价值、创造力和创新。这种脱节——我们希望成为这种差距，站在这种差距中，连接这两种现实。”

这就是为什么“和平生活”试图通过黑客马拉松和最近的一个名为 [StreetCode](https://web.archive.org/web/20230130230646/http://www.liveinpeace.org/streetcode-academy/) 的为期四周的加速器，尝试许多不同的方法来弥合这两个世界。加速器在本月早些时候的演示日达到高潮，学生们展示了他们的作品和他们在项目期间学到的技能。

Live in Peace 的项目得到了硅谷的一些支持，脸书首席产品官 Chris Cox 最近向该组织投资了 100 万美元。但是 Sobomehin 希望与硅谷的人们建立更多的联系和伙伴关系。

Sobomehin 说:“我们希望和尽可能多的人合作。“我们希望与导师和行业人士合作。我们希望与公司合作。我们只想尽我们所能创造最好的硅谷，你知道，最公平的硅谷。我们认为我们可以向硅谷展示如何真正公平地建设技术，我们相信我们有一个如何实现包容性的模型。”

“和平生活”的编码项目已经为学生带来了显著的成果。就在一年前，David Chatman 通过 Live in Peace 提供的一个项目开始学习 HTML 和 CSS。
