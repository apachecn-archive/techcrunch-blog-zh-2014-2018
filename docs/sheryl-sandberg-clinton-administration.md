# 雪莉·桑德伯格驳斥了在希拉里政府中扮演角色的谣言

几个月来，有传言称，如果希拉里竞选下届美国总统成功，她将任命脸书首席运营官雪莉·桑德伯格为财政部长。

但是桑德伯格今天在互联网协会主办的[良性循环会议](https://web.archive.org/web/20221101085104/http://vc.internetassociation.org/#schedule)上驳斥了这些谣言。

“我会留在脸书，”桑德伯格在一次炉边谈话中被问及她是否会担任下一任财政部长或商务部长时说道。当《纽约时报》的主持人 Cecilia Kang 追问时，桑德伯格坚持说:“我真的会留在脸书。我很开心。”

桑德伯格在比尔·克林顿政府时期崭露头角，当时她是时任财政部长拉里·萨默斯的幕僚长。桑德伯格在 2008 年加入脸书之前离开华盛顿特区，在谷歌工作。鉴于她在财政部的历史，猜测她可能会阻碍任命是有道理的。

桑德伯格在总统竞选期间也一直与希拉里保持着密切的关系。她在 7 月份给了克林顿在 T4 的支持，维基解密今天早上公布了克林顿竞选经理约翰·波德斯塔的电子邮件，邮件显示桑德伯格去年会见了克林顿，讨论了“关于女性性别和领导力的研究”

虽然桑德伯格说她不会考虑在政府中任职，但她告诉良性循环的观众，她关心政策问题。“我认为我们在这里讨论的问题，围绕政策的问题，很重要。所以政府很重要。我在私营部门没有工作，认为政府不重要；确实如此。桑德伯格向她的共同小组成员、诚实公司的杰西卡·阿尔芭点了点头，她说:“让这个群体走到一起的问题是:是什么样的政策框架让像诚实公司这样的公司在这么短的时间内开始发展，并以他们现在的方式扩大规模。”
