特斯拉 Model S P100D 在新电机趋势测试中获得 2.28 秒 0-60 英里/小时的成绩

特斯拉的 P100D 变体 Model S 创下新低——0 至 60 英里/小时加速的创纪录低时间，这是由 Motor Trend 使用 Ludricous+模式的库存车辆进行的新测试。

新的赛道测试时间，通过 Electrek ，还包括 10.5 秒的四分之一英里，这两项都代表了短跑测量的新纪录。事实上，Model S P100D 成为 Trend 测试的第一款量产车，实现了 0 至 60 英里/小时的速度，而不是 2.3 秒。

这比新的 Ludicrous+模式(顶级特斯拉的车主可以通过最近的软件更新解锁的复活节彩蛋)最初承诺的 2.5 秒 0 到 60 英里/小时的时间快得多，甚至比粉丝运营的特斯拉赛车频道在 1 月份创下的 2.389 秒的时间还要快。

特斯拉首席执行官埃隆·马斯克确实表示，他认为在 Motor Trend 之前在 Twitter 上的测试条件下，2.3 秒的时间是可能的，这证明了这不仅是可能的，而且实际上也低估了汽车的能力。正如 Elektrek 指出的那样，这款车确实有更轻的蜘蛛车轮，这可能有所帮助，但它们很难被视为售后升级，因为它们是该公司奖励特斯拉车主传播爱心的推荐计划的一部分。
