麦迪逊·里德锁定 1610 万美元的 C 轮融资，以发展业务并进军电视业 TechCrunch

家用专业染发配送初创公司麦迪逊·里德告诉我们，他们已经获得了 1610 万美元的 C 轮融资，“扩大业务规模，投资扩大营销，特别是在电视上。”

考虑到如此多的公司——特别是在初创领域——似乎更喜欢数字营销策略，如在脸书和推特上投放广告来吸引观众，转向电视广告是很有趣的。

对于 18-24 岁的女性来说，传统的电视收视正在下降。像《明迪计划》和《新女孩》这样的节目比你定期安排的节目更有可能获得在线浏览量(这也是《明迪计划》本季度从福克斯(FOX)转到 Hulu(T3)的原因)。

麦迪逊·里德(Madison Reed)表示，自推出以来的前 18 个月，该公司一直在了解其客户群——其中一群女性希望用一套升级的沙龙式染发套装来遮盖自己的灰色，但她们选择在自己家里独处。

我看到了这家公司是多么的雄心勃勃，我期待着继续这种关系，以创新他们的产品，以及他们通过电视和我们的媒体渠道与观众联系的方式。萨姆·兰德曼，康卡斯特创业公司

电视广告可以帮助麦迪逊·里德(Madison Reed)达到它打算追求的人群——头发开始变白的那个年龄(30 多岁及以上)的女性。根据

several studies

多年来，女性看的电视直播最多。

电视推动的部分原因来自与康卡斯特风险投资公司的关系，康卡斯特风险投资公司是麦迪逊·里德最新一轮投资的新投资者。

康卡斯特风险投资公司的总经理萨姆·兰德曼在谈到这项投资时说:“我看到了这家公司是多么的雄心勃勃，我期待继续保持这种关系，以创新他们的产品，以及他们通过电视和我们的媒体渠道与观众建立联系的方式。”。

现有投资者 Norwest Venture Partners 和 True Ventures 加入了康卡斯特风险投资和谢伊风险投资。

虽然当地药店货架上有大量的居家上色套件，沙龙也有多种上色方案可供选择，但麦迪逊·里德承诺会获得与沙龙专业人士相同的效果，但有了套件，您可以在家自己使用。

另一家公司可以在太空领域与麦迪逊·里德竞争。 eSalon 最初有 1100 万美元的启动资金，并告诉 TechCrunch 这是一项可自我维持且盈利的业务(预计今年年底将盈利约 3000 万美元)。它实际上也做了与麦迪逊·里德相同的事情——为客户运送专业的染发剂。然而，eSalon 会根据您的头发个性化颜色搭配。

护发产品占全球美容市场的第三大份额，根据他们的指标，eSalon 和 Madison Reed 似乎都在向右成长。然而，在康卡斯特与麦迪逊·里德的投资中，对兰德曼来说最重要的是关系。兰德曼说:“我们与艾米·埃瑞特有着长期的合作关系，我们认为她是一位出色的首席执行官，有能力打造一个世界级的品牌。”。
