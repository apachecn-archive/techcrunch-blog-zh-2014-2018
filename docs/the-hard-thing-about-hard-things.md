# 谈论(技术多样性)很难

在过去的七年里，科技的多样性已经从“很好拥有”变成了“需要拥有”再变成了“迫切需要拥有”，这就是我们现在所处的状态。尽管我们已经取得了一些进展，无论是更多的女性和有色人种被考虑担任首席执行官，更多的我们[大声疾呼](https://web.archive.org/web/20230326200005/https://recode.net/2015/09/10/ellen-pao-speaks-i-am-now-moving-on/)我们有权被视为社区中有能力的成员，更多的我们[显示出回报](https://web.archive.org/web/20230326200005/http://fortune.com/2015/07/29/female-founders-better-vc/)，还是更多不同阶层的人[首先学习编码](https://web.archive.org/web/20230326200005/http://www.npr.org/sections/ed/2015/08/17/432278262/hacking-tech-s-diversity-problem-black-girls-code)，我们还有很长的路要走。

过去几年发生了实质性的变化:无论是苏欣德·辛格·卡西迪分发的调查，还是 Pinterest 工程师 Tracy Chou 在她自己的公司推动多元化报告和透明度，这在整个行业产生了连带效应。

或者更近一点，一位工程师 [Isis Wenger](https://web.archive.org/web/20230326200005/https://www.crunchbase.com/person/isis-anchalee) ，他把网络钓鱼变成了一场运动，给女性和有色人种提供了大量的工程师榜样，只要搜索标签[# ilooklikeaneineer](https://web.archive.org/web/20230326200005/https://twitter.com/search?q=%23Ilooklikeanengineer&src=typd)。

这就是为什么这三位将在两周后加入我们 Disrupt 的舞台，在一次炉边谈话中，我将[翘课](https://web.archive.org/web/20230326200005/https://techcrunch.com/2015/05/20/make-new-friends-keep-the-old/)来主持。(我开玩笑！)

有许多关于技术多样性的论点，但很少是数据驱动的或可操作的。以上三位专家小组成员的努力已经带来了实质性的变化。

虽然这个小组中有很多女性，但请放心，这不会是一个“科技领域的女性”小组:因为 a)科技领域的多样性显然超越了女性，b)我们已经尝试过一次，但没有成功。

从纯粹的资本主义角度来看，增加我们行业的多样性势在必行。只是生意好而已。我们生产的产品继续影响着更加多样化的市场，需要有不同观点的人来引导这种影响。我再也不想听到一位风险投资家说，他会向妻子或行政助理询问像 Pinterest 或 Snapchat 这样的产品。能够回答这个问题的人应该在房间里，坐在桌子旁边:**所以雇佣他们**。
