macOS 的下一个版本是 MAC OS Mojave TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/06/04/the-next-version-of-macos-is-macos-mojave/

苹果还没有完成 macOS。该公司在 WWDC 开发者大会上展示了 macOS 的下一个版本。有了 macOS Mojave，苹果将大山的隐喻抛在了身后。

苹果首席执行官蒂姆·库克表示:“今天，我们很高兴能够将 Mac 向前推进一大步。”

苹果软件工程高级副总裁克雷格·费德里吉(Craig Federighi)从黑暗模式开始。只需一个设置，您就可以反转所有事物的颜色。甚至连 Finder、日历或照片的背景都是黑色的，不仅仅是 Dock 或菜单栏。苹果不得不重做操作系统中所有的按钮和配色方案。如果你觉得白色文档太咄咄逼人，黑暗模式适合你。

在下一个版本的 macOS 中，壁纸和桌面将根据他们一天中的时间进行调整，从早上到下午和晚上。在桌面上，macOS 可以自动将你所有的文档按格式堆叠在桌面文件夹中。

在 Finder 中，有一个新的画廊视图。它的工作方式有点像旧的 Cover Flow 功能，但具有扁平的设计和元数据。例如，对于一个照片文件夹来说，这是非常有用的。在“元数据”栏中，有一些使用 Automator 的可自定操作。您可以创建 PDF，添加水印等。

Quick Look 也升级了类似 iOS 的标记。您可以在快速查看窗口中绘制照片、剪辑视频等。在截图工具中，您现在可以录制快速视频。

苹果正在扩大你的 Mac 和 iPhone 之间的连续性。例如，在 Keynote 中，您可以使用 iPhone 右键单击并添加照片。它会在你的 iPhone 上启动相机应用程序。

苹果新闻即将登陆 macOS。如果一个网络浏览器还不够，你现在可以有一个专门的应用程序，提供你的选择、你的网站和你的推荐。苹果电脑也将推出股票应用。语音备忘录也将用于 Mac 电脑的快速录音。

最后，Home 将可以在 Mac 上通过语音控制你的家，控制你的安全摄像头，触发你的联网灯等等。

苹果正在增加新的隐私保护。除了位置数据和地址簿访问，该公司还增加了摄像头、麦克风、信息历史、备份和 Safari 数据隐私弹出窗口。在 Safari 中，浏览器将屏蔽脸书按钮，并阻止这些小工具加载第三方 Javascripts。您需要授予明确的访问权限。Safari 还会让你看起来像一台装有默认插件、默认字体和所有默认东西的 Mac，以此来防止指纹识别。

正如所料，Mac 应用商店正在更新。这是真的，iOS 在这方面得到了所有的爱。去年，苹果重新设计了 iOS 上的应用商店，但 Mac 应用商店却停滞不前。它已经完全重新设计了左侧栏，这是一个看起来更像 iOS 应用商店的用户界面。将有编辑的内容，视频预览和新的类别。

苹果正在为数据科学家开发一种新工具。使用 Create ML，您可以训练模型，而不必成为机器学习专家。它允许您管理一个测试集，并使用您的本地机器进行小型项目。

“我们想让你先睹为快一个多年的项目，”Federighi 说。苹果基本上会让你在 Mac 上移植使用 UIKit 的 iOS 应用。这样，iOS 开发者将能够更快地移植他们的应用程序。这并不意味着它会看起来像一个 iOS 应用程序。新的苹果新闻、股票、家庭和语音备忘录都使用这个 macOS 版本的 UIKit。大按钮被类似 macOS 的菜单所取代，代码改动很小。

开发人员还不能使用 macOS 应用程序的 UIKit，但很明显，它可以在未来几年重振应用生态系统。
