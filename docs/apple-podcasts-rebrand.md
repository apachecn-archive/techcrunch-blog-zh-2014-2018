# 苹果公司从播客应用程序中去掉了 iTunes 的名字

苹果已经正式从其播客应用程序中移除 iTunes 品牌。这是一个很小的变化，除了该部门的营销负责人发了几条推文之外，该公司没有大肆宣传。但它确实简化了产品，使其与苹果音乐保持一致，并从更传统的 iTunes 系列中分离出来(至少在名称上)。

正如 [9 to 5 Mac 指出的](https://web.archive.org/web/20221208011541/https://9to5mac.com/2017/04/13/apple-rebrands-itunes-podcasts-apple-podcasts/)，更名可能是该公司对播客领域重新产生兴趣的一部分，符合 Eddy Cue 大约一个月前承诺的“新功能”。

现在肯定是苹果更全面投入这个领域的好时机，因为播客成为媒体叙事中越来越重要的一部分，通过像*这种美国生活*广受欢迎的 *S-Town* 这样的节目，它目前垄断了 iTunes 排行榜的前七名。

苹果当然有很好的条件继续拥有这一类别，尽管随着 Spotify 和谷歌等更多公司对这一领域表现出越来越大的兴趣，重新关注这一领域将受到欢迎。新品牌的一部分是一个新徽章，旨在通过播客页面推广该服务。

虽然苹果自然不会讨论其未来的计划，但就 iTunes 作为其自己的类别的未来而言，这一品牌也很有趣。音乐软件与 iPod 一起在重振苹果的命运中发挥了重要作用，但尽管该公司已基本脱离了独立的音乐播放器，iTunes 仍在其软件生态系统中发挥着重要作用。

该公司在收购 Beats 后选择了简单的 Apple Music 名称，已经朝着这个方向迈出了重要的一步。播客名称的改变似乎与该公司音频产品的精简相一致。
