# 维珍银河太空飞船二号在第二次成功试飞中达到 1.9 马赫

维珍银河正在庆祝太空飞船二号的第二次成功试飞，这是一艘以火箭为动力的客运飞船，有朝一日可能会把游客带到太空边缘。今天的测试将 VSS Unity 带到了 114，500 英尺和 1.9 马赫，或近 1，300 英里/小时，VSS Unity 是该级别中建造的第二艘飞船。

不到两个月前，Unity 的第一次动力飞行是，这也是维珍银河自 2014 年该公司之前的两级太空飞船“企业”号致命解体以来首次尝试动力飞行。

从那以后，很多东西都被重新制作了，但是维珍银河飞行风格的基础是一样的。一架相对传统的喷气式飞机，一架 WhiteKnightTwo 级飞机(在这种情况下是 VMS Eve)，载着“太空船二号”( Unity)上升到大约 45，000 英尺的地方。在那里，后者分离并启动火箭发动机，加速到高速和高空，然后滑翔到表面并着陆，这或多或少像其他飞机一样。

今天的试飞严格遵循了这些参数，尽管数据比 4 月份的飞行要高得多。火箭总共燃烧了 31 秒，飞船独特的用于减缓其下降速度的“羽毛”系统被成功部署。

Unity 的设计处理速度几乎是现在的两倍，但这些公差当然必须逐渐接近，各种系统在友好条件下进行测试，然后再进入敌对状态。

“很高兴看到我们美丽的宇宙飞船重返空中，并与带领我们一步一步走向太空的才华横溢的团队分享这一时刻，”维珍创始人 Richard Branson [在新闻稿](https://web.archive.org/web/20221210070704/https://www.virgingalactic.com/articles/Richard-Branson-Welcomes-VSS-Unity-Home-from-Second-Supersonic-Flight/)中说。“看到 Unity 以超音速向上飙升令人振奋，绝对令人惊叹。我们越来越接近实现我们的目标。恭喜全队！”
