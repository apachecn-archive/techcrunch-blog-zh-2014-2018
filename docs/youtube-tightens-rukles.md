# YouTube 收紧了创作者货币化和合作关系的规则

为了重新赢得广告商的信任，谷歌宣布对 YouTube 的盈利模式进行“艰难但必要”的改革。

首先，它为 YouTube 合作伙伴计划设置了更高的标准，这使得出版商可以通过广告赚钱。以前，他们需要总共 10，000 次浏览才能加入该计划。从今天开始，频道还需要在过去一年中拥有 1000 名订户和 4000 小时的观看时间。(目前，这些只是加入该计划的要求，但谷歌表示，它也将于 2 月 20 日开始对当前的合作伙伴适用这些要求。)

这可能会让营销人员确信，他们的广告不太可能在随机的、不可靠的频道上播放，但是正如谷歌的保罗·穆雷[写的](https://web.archive.org/web/20230318211244/https://adwords.googleblog.com/2018/01/a-new-approach-to-youtube-monetization.html)，“当然，规模本身并不足以决定一个频道是否适合做广告。”

所以除此之外，他还说:

> 我们将密切监控社区罢工、垃圾邮件和其他滥用标志等信号，以确保它们符合我们的政策。新的和现有的 YPP 频道都将根据这一严格的标准进行自动评估，如果我们发现一个频道反复或严重违反我们的社区准则，我们将从 YPP 删除该频道。像往常一样，如果该帐户已被发出三个社区准则罢工，我们将从 YouTube 上删除该用户的帐户和频道。

穆雷还描述了为更具排他性的谷歌首选项目计划的变化，该项目应该仅限于最好和最受欢迎的内容。直到他的“自杀森林”视频引起的争议使他在上周被赶出——这个故事暗示了谷歌方法的一些局限性。

展望未来，穆雷表示，该计划将提供“不仅是 YouTube 上最受欢迎的内容，也是最受审查的内容。”这意味着 Google Preferred 中的所有内容都应该手动管理，广告只在“经过验证符合我们广告友好准则的视频上”播放。(看来[那些新的内容版主](https://web.archive.org/web/20230318211244/https://techcrunch.com/2017/12/05/youtube-promises-to-increase-content-moderation-staff-to-over-10k-in-2018/)都要忙了。)

最后，Muret 表示，YouTube 将在未来几个月内引入一个新的“三层适宜性系统”，旨在让营销人员在更安全的环境中运行广告与接触更多观众之间的权衡中获得更多控制。
