# 一窥 Hyperloop One 疯狂快速的未来

Hyperloop One 到底发生了什么——这个未来交通系统计划以每小时 750 英里的速度将人类从 A 点送到 B 点？

初创公司[将名字](https://web.archive.org/web/20221005150505/https://beta.techcrunch.com/2016/05/10/hyperloop-technologies-becomes-hyperloop-one-pulls-in-80-million-and-announces-global-partners/)从 Hyperloop Transportation 改为 Hyperloop One，[今年春天开始在内华达沙漠测试](https://web.archive.org/web/20221005150505/http://www.digitaltrends.com/cool-tech/hyperloop-one-debuts-its-hyperloop-system/)；宣布[8000 万美元](https://web.archive.org/web/20221005150505/https://beta.techcrunch.com/2016/05/10/hyperloop-technologies-becomes-hyperloop-one-pulls-in-80-million-and-announces-global-partners/)B 轮融资；遭遇高管[领导层变动](https://web.archive.org/web/20221005150505/https://beta.techcrunch.com/2016/07/12/ousted-hyperloop-one-co-founder-brogan-bambrogan-is-suing-shervin-pishevar-claims-harrassment/)和前联合创始人提起诉讼；然后何[提起](https://web.archive.org/web/20221005150505/https://beta.techcrunch.com/2016/07/12/ousted-hyperloop-one-co-founder-brogan-bambrogan-is-suing-shervin-pishevar-claims-harrassment/)反诉；建立了一个示范工厂；并宣布可能在全球几个关键地区开展重大航运业务，包括与俄罗斯一家大型投资集团在莫斯科[和迪拜](https://web.archive.org/web/20221005150505/https://beta.techcrunch.com/2016/06/21/throughrussiawithlove/)[合作，与全球第三大航运公司 DP World 合作。](https://web.archive.org/web/20221005150505/https://beta.techcrunch.com/2016/08/15/dubai-hopes-to-hop-on-hyperloop-ones-global-shipping-train/)

哦，几周后，我们将在 Disrupt 的舞台上听到投资人兼 Hyperloop One 联合创始人 Shervin Pishevar 的发言。

很自然地，我们认为是时候亲自看看提议的 hyperloop 背后的技术离现实有多近了。

联合创始人(前 SpaceX 工程师)[乔希·吉格尔](https://web.archive.org/web/20221005150505/https://www.linkedin.com/in/josh-giegel-43505534)让我们先睹为快，看看这家初创公司在总部做些什么。我们所看到的东西让我们站在了一定会改变我们的世界和我们生活方式的边缘，如果我们能让它的系统加速运转的话。
