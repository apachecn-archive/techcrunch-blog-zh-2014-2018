智能电子表格服务 Blockspring 融资 340 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/07/29/smart-spreadsheet-service-blockspring-raises-3-4-million/

Blockspring 是一项连接 Excel 和 Google 电子表格以更容易导入数据的服务，它表示已经筹集了 340 万美元。

Blockspring 与 Bing Search 等服务建立了多种连接，可以自动在电子表格中填充数据，而不必编写特定的代码来导入和排序电子表格中的数据。该公司计划发布 Tableau 和 Slack 等工具的几个集成。

Blockspring 首席执行官保罗·卡森(Paul Katsen)表示:“除了创造服务，工程师们还必须做更多的事情——他们必须开发应用程序，并教会(企业员工)如何使用它们。”。“它实际上只是位于开发者和其他人之间。它让开发人员花更少的时间来尝试交付新的功能，并让业务人员以此为基础”

例如，Blockspring 用户可以快速从网上获取 pdf，并在电子表格中进行分析。这将包括搜索这些文章，下载每一篇文章，然后分析它们——例如搜索特定的单词。由于 Blockspring 中插入了 API，因此可以通过多种方式操作这些数据。

“想象一下任何一种网络服务——我们所做的只是标准化每一种服务。它们都不同且复杂，”Blockspring 首席执行官保罗·卡特森说。“每学一个新的，你都会想，我该怎么用这个。以谷歌为例，对于他们的服务以及其他 API，你可以获得所有这些集成，使你的 API 可以被全新的受众接受和消费。这为他们节省了数十万美元的工程成本，因为他们不必构建这种集成。”

他解释说，Katsen 是“坐在中间的人”，从事咨询工作，工程背景有限。他逐渐变得更加技术化，但他和他的两位联合创始人 Jason Tokoph 和 Donald Pinkus(他们是工程师)看到了一个机会，可以构建一个本质上位于开发人员和业务员工中间的东西，自动构建有用的工具，而不占用开发时间。

他说:“这有点像‘伙计，我希望我能自己解决它，但是我真的对学习如何编码没有兴趣’。“那样的人太多了。他们不想成为软件工程师；他们是营销人员和记者。有人力资源人员、招聘人员、销售人员，他们有自己真正喜欢的工具，也有非常具体的问题。”

Blockspring 的下一步，除了继续增加像 Slack 这样的新服务，就是为插入内部数据库的公司建立服务。一个更大的公司可能有一个巨大的内部数据库，它用来跟踪像线索这样的事情，Blockspring 将利用这一点，帮助业务方面的员工更容易地访问这些数据。

更新:增加了一个澄清，Blockspring 支持必应搜索，不支持谷歌搜索。
