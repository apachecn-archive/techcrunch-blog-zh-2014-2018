SiFive 获得 5060 万美元，帮助公司推出定制芯片设计 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/04/02/sifive-gets-50-6m-to-help-companies-get-their-custom-chip-designs-out-the-door/

随着下一代硅的竞赛如火如荼地进行，流入定制硅初创公司的风险资金瀑布已经显示出一些更灵活的硬件在日益变化的技术格局中的巨大潜力 Naveed Sherwani 希望为其他人开发这种潜力。

这是 si five T1 的前提，si five T1 是一家旨在帮助企业家或任何公司为他们的需求定制芯片的初创公司。但 SiFive 的目标不是从风险公司筹集数千万美元或建立一个大规模的生产系统，而是帮助开发人员快速获得硅片，这样他们就可以看看它是否真的基于一套基本的硬件和提供的知识产权工作，然后确定何时以及如何将其投入大规模生产。该公司首先提供模板，然后允许他们对最终在他们手中的 RISC-V 硅片进行一些修改。SiFive 今天表示，它已经在由 Sutter Hill Ventures、Spark Capital 和 Osage University Partners 牵头的一轮风险融资中筹集了 5060 万美元。

“我们的看法是，我们认为我们不应该依赖人们学习特殊语言和类似的东西来修改架构和增强架构，”舍瓦尼说。“我们相信可能会有一个高级界面，这就是我们正在构建的，它将允许人们利用现有的核心，将它们带入他们的设计空间，然后应用配置。移动这些配置，您可以修改核心，然后您可以获得新的修改核心。这就是我们采取的方法，我们不需要学习特殊的语言或成为专家，这就是我们呈现核心的方式。我们希望从经过验证的内核开始，并且这些修改不会导致变得不可验证。”

SiFive 基于一种称为 RISC-V 的硅设计框架。你可以认为它是一种开源模拟主要芯片制造公司的设计，但 RISC-V 芯片的目标是依靠自最初的硅芯片出自英特尔以来数十年的经验，开发一些不那么混乱的东西，同时仍能完成正确的任务。Sherwani 说 RISC-V 芯片有 50 多个指令集，而普通芯片将有 1000 多个。从本质上来说，他们不是英特尔的规模，所以这些公司可能拥有的效率并不存在。但 SiFive 希望服务于各种不同的需求，而不是大规模生产单一类型的硅。

寻求使用 SiFive 构建芯片的开发人员有两个流程。首先是原型流程，开发人员将有机会详细说明他们的芯片，并找出他们的具体需求。那里的目标是将一些东西送到开发者手中，他们可以用这些东西来展示他们的想法或技术，SiFive 与知识产权供应商和其他供应链合作伙伴合作——在这段时间里，开发者不为知识产权付费。一旦这个案例被证实(创业公司可能已经基于这个想法筹集了资金)，他们可以转向 SiFive 的生产流程，开始为 IP 和服务付费。随着越来越多的人对操作核心提出新的想法，还有一个潜在的市场因素。

“对于任何细分市场，都会有一些可用的模板，”舍瓦尼说。“我们将有一些工具和方法，在所有可用的各种模板中，显示什么最适合[该客户]。我们也有一个应用商店——我们期待那些设计了内核的人愿意分享它，因为他们不需要它是专有的。如果任何人使用这个模板，那么无论他们给它定什么样的价格，他们都可以从中赚到一些钱。市场的整个想法会让更多人兴奋起来。”

随着开发新的定制硅的热潮，SiFive 提供的服务可能变得越来越必要。但 SiFive 背后的赌注还有另一个因素:让芯片本身不那么模糊，并试图消除黑盒。这不一定会让它比旁边的那个更安全，但至少，这意味着当出现像英特尔的 Spectre 问题这样的重大安全缺陷时，开发人员社区可能会更宽容一些，因为黑盒更少了。

“所有这些复杂因素都存在，除非你具备所有这些专业知识，否则你无法制作芯片，”舍瓦尼说。“我们的愿景是，我们将整个芯片体验交付给该平台，人们可以登录。他们不需要团队，任何工具，他们不需要 FPGAs，因为所有这些都可以在网上获得。结果，成本下降了，因为这是一个共享经济，他们共享工具，这就是我们如何戏剧性地认为你可以以低得多的成本制造芯片。”

尽管有大量风险资金流入人工智能芯片领域——对这种硬件的外观有许多不同的解释——Sherwani 说，与 SiFive 合作的好处是能够迅速将想法适应不断变化的算法。开发人员已经验证了许多不同的工具和框架，但是一旦一块硅片投入生产，就不容易立即改变了。如果这些最佳实践或算法发生变化，开发者将有机会尽快重新评估和设计芯片。

随着越来越多的用例出现，定制芯片的想法将成为未来的一个大主题，定制硬件可能会变得更容易。已经有像 Mythic 和 SambaNova Systems 这样的初创公司，它们已经筹集了数千万美元，专门从事典型人工智能过程的快速计算。但这种技术现在出现在从自动驾驶汽车到冰箱的各种设备中，每个用例可能有不同的需求。英特尔和其他芯片设计公司可能无法触及每一个利基市场，一刀切(甚至更模块化的东西，如英特尔的 FPGA)可能无法触及每个最佳点。理论上，这是 SiFive 这样的公司可以填补的漏洞。
