# Tinder 投资 Vina，因为它正在向交友迈进

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/09/22/tinder-invests-in-vina-as-it-inches-toward-friend-finding/>

统治一切的约会应用 Tinder 今天宣布，它作为投资者参与了 Vina 的最新一轮融资。

社交网络 Vina 最近从格雷洛克、NEA 和野猫风险投资伙伴那里筹集了 140 万美元的种子资金[。据 Tinder 首席执行官肖恩·拉德(Sean Rad)称，Tinder 碰巧也参与了这轮融资，并投资了“可观的金额”。](https://web.archive.org/web/20230322160427/https://www.crunchbase.com/organization/vina#/entity)

尽管如此，双方都不愿意透露资金条款。

这标志着 Tinder 向浪漫关系之外的垂直领域迈出了重要的一步，尽管这不是我们第一次看到约会应用[走上交友之路](https://web.archive.org/web/20230322160427/https://techcrunch.com/2016/03/04/bumble-launches-bff-a-feature-to-find-new-friends/)“这一转变显然是向前迈出的下一步，因为 Tinder 在多年观察人们的数字约会后，能够处理大量关于人们如何决策的数据。

这种转变的证据昨天浮出水面，当时 Tinder 宣布推出 iMessage 应用程序 Tinder Stacks，让用户发送一堆 Tinder 风格的卡片(带有图片)，要求他们的朋友对任何事情进行刷卡/投票。

随着 Vina 的投资，Tinder 将在未来提供指导和潜在的合作。同样值得注意的是，Tinder 的大股东 Match Group 批准了 Tinder 的资金分配。

看到像 Tinder 这样的子公司参与一轮融资，而不是直接收购该公司或将其加入 Match Group 的投资组合，这很奇怪。

但似乎 Tinder 在这方面的作用更大。

![ditto](img/071206f206eb634822193418d16cb7a6.png)

作为一家公司，Tinder 可能会在未来几年内转变为一把雨伞，将相同的刷卡/匹配功能整合到少数其他垂直行业中。但是，作为一个品牌，很难把 Tinder 看作是一个约会/勾搭应用程序以外的东西。

出于这个原因，对 Vina 的投资(与 Tinder 生产的任何东西相比，它更多的是关于女性赋权和友谊)是向交友过渡的第一步，最终很可能会被收购。

“在我们的研究中，很明显，我们的许多用户想要一个功能与 Tinder 相同的产品，但可以帮助他们找到朋友，”Rad 说。“我们投资 Vina 是因为很长一段时间以来，我一直在思考这个领域，思考作为一个更广泛主题的社会介绍。”

目前，两家公司仍将是独立的实体。
