# 这个机器人展开 Twitter 的线程，把它们变成可读的博客文章

尽管用户对他们选择的平台恨之入骨，Twitter 偶尔真的会对热门话题进行深入探讨，提供实际的、有意义的见解。这样的事件几乎足以提醒我们，为什么我们一开始就把生命浪费在那里。

还记得我们以前在*博客*的时候吗？比如，我们用句子和段落来表达思想，而不是一系列的社交媒体骚动？以前也发生过！这有点像那个。

一个名为[的 Twitter 机器人线程阅读器应用](https://web.archive.org/web/20230315095314/https://threadreaderapp.com/) (@threadreaderapp)可以将你勉强可读的 Twitter 咆哮转换成普通文本，呈现在稀疏的普通页面上。你可能已经注意到这些“展开”的线——它们到处都在冒出来。

要让你选择的 tweetstorm 以博客的形式传递，你只需要回复一个带有@threadreaderapp 和关键字“unroll”的帖子虽然 Twitter 线程做的整个 1/347 编号的事情读起来有点奇怪，正常的文本块被缝合在一起，成为非常清晰的东西。

创造者还提供了一个 Chrome 扩展，将这个过程变成一系列简单的点击。(我们还没有测试过，所以和所有东西一样，下载风险自担。)真的感受到了？你可以[向 Patreon](https://web.archive.org/web/20230315095314/https://www.patreon.com/bePatron?u=8298707) 的项目捐款。
