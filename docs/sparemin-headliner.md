# SpareMin Headliner 将播客剪辑转化为可分享的视频

播客的好处之一是有机会深入讨论事情，而不必将所有内容压缩到 30 或 60 秒的一小段中。但这随之带来了一个挑战:在社交媒体上推广这些更长对话的最佳方式是什么？

这是我们在 TechCrunch 上争论过的事情，去年我们在那里推出了几个播客。(我有点偏颇，但我觉得他们都[都](https://web.archive.org/web/20230328183314/https://itunes.apple.com/us/podcast/equity/id1215439780?mt=2) [挺](https://web.archive.org/web/20230328183314/https://itunes.apple.com/us/podcast/original-content/id1271763098?mt=2) [好](https://web.archive.org/web/20230328183314/https://itunes.apple.com/us/podcast/ctrl-t/id1317432118?mt=2)。)根据 Digiday 最近的一篇文章[，这是一个许多出版商正在努力解决的问题。](https://web.archive.org/web/20230328183314/https://digiday.com/media/era-video-podcasts-challenged-break-social-media/)

spare min[的团队已经用](https://web.archive.org/web/20230328183314/https://www.sparemin.com/)[headlines](https://web.archive.org/web/20230328183314/https://headliner.sparemin.com/)解决了这个问题，这是一个基于浏览器的产品，用于将音频剪辑转换成视频。如果你的播客中有特别有新闻价值或有趣的片段，你可以在几秒钟内把它变成一个宣传视频。

我以前写过关于 SpareMin 的文章，在它开发了一个用于半随机电话呼叫的应用程序之后。联合创始人 Oliver Wellington 表示，他的团队基于公共广播电台 WNYC 去年发布的听力图生成器，创建了第一个版本的头条新闻作为实验。

但出版商立即表现出了兴趣，Wellington 说 SiriusXM、Gimlet、Tim Ferriss Show、雷德·霍夫曼的 Masters of Scale、BBC、Guardian、Comedy Central 和几十家 NPR 电视台都使用了这个免费工具。因此，现在这就是 SpareMin 集中精力的地方(尽管没有计划关闭 SpareMin 电话呼叫应用)。

[介绍斯帕莱明的头条](https://web.archive.org/web/20230328183314/https://vimeo.com/251531977)来自[斯帕莱明](https://web.archive.org/web/20230328183314/https://vimeo.com/user76501574)上 [Vimeo](https://web.archive.org/web/20230328183314/https://vimeo.com/) 。

SpareMin 的一大卖点就是简单。你上传一个音频文件，它会自动转录(头条新闻支持多种语言的转录，包括英语，西班牙语和德语)。然后，您可以使用头条新闻编辑器来创建一个幻灯片风格的视频，或者只是移动波形。头条新闻甚至可以通过使用抄本和从 Getty Images 中提取相关照片来组装视频本身。

头条绝对是早期产品。编辑器是免费的，它只在 Chrome 和 Firefox 中工作，它的主要输出是一个 MP4 视频文件。(惠灵顿表示，该团队正在努力支持更多的浏览器，并致力于直接导出到 YouTube 等平台的能力。)

与此同时，对于那些没有通过[我们对网飞的*欧扎克*](https://web.archive.org/web/20230328183314/https://techcrunch.com/2018/01/26/original-content-ozark/) 的讨论的人，我用头条新闻制作了[我的播客联合主持人达雷尔·埃瑟林顿的视频](https://web.archive.org/web/20230328183314/https://media.sparemin.com/embed-video/videos/08/25/cd/67/0825cd67-9cac-40cc-8e03-98656a8ebc7b_8304.mp4)来解释为什么他对观看一些有史以来最伟大的电视节目不感兴趣。
