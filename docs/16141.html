XPRIZE 入围 Cloud DX 的 Vitaliti 是一款严肃的健康可穿戴设备

许多可穿戴设备公司做出的健康声明充其量也是可疑的。诚然，像 Fitbit 和苹果这样的公司对整个事情越来越认真，参与大学研究并与保险公司合作，但总的来说，我肯定不会把我自己的幸福托付给他们任何人。

作为高通 Tricorder XPRIZE 的衍生产品， Vitaliti 被设计用于最近出院的病人，因此他们的主治医生可以继续远程监控他们。

该设备位于衣领周围，有点像你脖子上的轭，有两个电极附着在皮肤上，一个传感器位于耳朵里。它不是最舒适或视觉上最吸引人的小玩意，但你可以长时间佩戴它。Cloud DX 创始人兼首席执行官罗伯特·考尔(Robert Kaul)在 CES 上一次充电就穿了三天——尽管电极是连着的，因为拉斯维加斯会议中心的广场显然更喜欢人们穿着衬衫。

该可穿戴设备能够测量一些标准可穿戴设备，包括运动和脚步，以及更复杂的生命体征，如心电图、心率、血氧饱和度、呼吸、核心体温和血压，符合 XPRIZE 指南。它连接到一个移动应用程序，实时提供读数，并在有人到达危险区时发出警报。

Cloud DX 目前正在寻求美国食品和药物管理局(FDA)批准其作为医疗器械进行销售。该公司还为 HoloLens 开发了一个界面，可以实时将生命体征叠加到患者身上。
