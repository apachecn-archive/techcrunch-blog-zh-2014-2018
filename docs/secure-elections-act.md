# 一项新的两党参议院法案寻求在州一级加强美国选举安全

参议院两党议员小组提出了一项法案，旨在保护美国中期选举中的技术完整性。该法案被称为安全选举法案，由共和党参议员詹姆斯·兰克福德、苏珊·科林斯和林赛·格拉汉姆，以及民主党参议员艾米·克洛布查尔、卡玛拉·哈里斯和马丁·海因里希提出。

“虽然没有迹象表明俄罗斯人能够改变投票总数，但我们知道俄罗斯人多次试图破坏州选举系统或公共网站，”参议员柯林斯在该法案的一份声明中说。“我们的两党立法将通过确保地方投票官员拥有他们需要的信息和财政资源来保护他们的投票系统，从而加强我们选举过程的完整性。”

在类似的声明中，参与该法案的所有参议员都确认，在美国进一步保护和规范其选举系统之前，俄罗斯将继续对美国民主构成生存威胁。该法案将要求联邦机构及时与州和地方政府分享与选举有关的网络威胁，调整需要访问可能保密的相关信息的州官员的安全许可，给予各州拨款以实现选举系统的现代化，并创建一套网络安全指导方针来保护选举系统，包括投票机。

“当俄罗斯干涉 2016 年选举时，他们攻击了我们民主的核心，”参议员哈里斯说。“随着 2018 年选举即将到来，俄罗斯将再次回来干涉。”
