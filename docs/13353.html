Accel 领投 KRY 瑞典远程医疗初创公司 2280 万美元的首轮融资

瑞典数字医疗创业公司 KRY 完成了一轮 2000 万€(2280 万美元)的首轮融资，以发展其点播视频呼叫医生业务，包括拓展新市场。现有投资者 Index Ventures、Creandum 和 Project A 也参与了该轮投资。
