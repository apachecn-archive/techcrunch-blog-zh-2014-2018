Stripe 通过 Atlas 扩展创业工具，帮助外国公司在特拉华州注册

今天，Stripe 在为初创公司提供更多工具方面迈出了一大步，除了它已经提供的基本支付服务之外。该公司今天发布了 Atlas ，这是一个让初创公司更容易在美国成立的平台，特别是在特拉华州，其服务包括成立公司、发行股票、增加董事、设立银行账户和(当然)条纹支付账户。

atlas——今天发布的测试版仅限受邀者，是由该公司联合创始人兼首席执行官帕特里克·科利森在巴塞罗纳 MWC 的舞台上宣布的。

科利森说，Atlas 将从美国公司开始，但随着时间的推移，将提供“更多的中心”，以便企业可以在“对他们来说有意义的地方注册”。

为了鼓励注册，Stripe 表示，Atlas 将对前 100 名用户免费(这要么是一个低得离谱的数字，要么是一个迹象，表明 Stripe 没有像看待付费一样，将此视为一种规模游戏)。之后，Atlas 测试版的定价将为 500 美元，其中包括与组建公司和开设商业银行账户相关的所有费用。

它补充说，所有支付的价格都与 Stripe 的标准费用相同，即每次成功收费 2.9% + 30 美分。

Atlas 的想法是，它将帮助 Stripe 扩大与初创公司的接触点，以建立更大的支付业务——从本质上说，这将使该公司更容易向美国以外的数十亿在线企业出售支付服务..

这也将有助于 Stripe 更好地与 PayPal 等公司竞争，PayPal 也提供了一系列商业支付以外的服务，包括贷款、发票和店内支付选项。

最重要的是，这将有助于该公司增加每名客户的收入，同时解决创业公司在拓展业务方面面临的一个更大的难题，特别是在美国以外的地方。

“这些市场的初创公司可以在几天内成立并运行起来，”科利森今天在台上说。“他们可以做重要的事情，而不是努力克服进入壁垒。”

或者，正如一位使用 Atlas 的初创公司 Platzi 的创始人所说，“你不必成为硅谷的圈内人就能发展一家初创公司。”

作为 Atlas 的一部分，Stripe 已经宣布与几个组织建立合作关系。包括融资的硅谷银行，法律公司事务的奥瑞克，税务指导的普华永道，云服务的 AWS。使用 AWS 的公司可以获得 15000 美元的免费服务。

科利森表示，该公司在试探领先于此的初创公司时，已经看到了对 Atlas 的“大量需求”。该公司还将与他所描述的加速器、孵化器和其他机构的“全球网络”合作，以确定初创公司成为该计划的一部分。其中包括 500 Startups、Startupbootcamp、Telefonica 的 Wayra、风投 Atomico 等。你可以在这里看到目前为止的完整名单。

Stripe 迄今为止已经筹集了2 . 8 亿美元的资金，估值50 亿美元-一直在逐步扩大向用户提供的基本支付以外的服务种类，尽管迄今为止这些产品都专注于交易。

它们包括订阅计费、中继让客户在其他移动应用程序中购买产品、连接市场、以及比特币接受。该公司简单的基于 API 的支付服务也使其成为其他已经停止运营的支付公司的常规默认选项。

Stripe 还与美国运通等其他金融公司以及推特(T21)等第三方建立了合作关系，以拓展其业务。但 Atlas 是 Stripe 首次尝试建立支付之外的创业工具。
