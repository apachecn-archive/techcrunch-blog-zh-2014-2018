# 在明天比特币拆分之前，你应该做什么？

8 月 1 日即将到来，伟大的比特币“硬分叉”即将让不经意的加密货币观察者感到困惑。以下是你应该知道的。

**很简单，如果你使用的是明确支持比特币现金的交易所，或者你控制着自己的私钥，那么你就没问题。**如果你将比特币存放在不支持比特币现金的交易所，比如比特币基地、**，那么你应该立即转移你的比特币，除非你不想用比特币现金交易。不管怎样，保留你自己的私人钥匙是个好主意，所以这是一个从你过去可能用过的任何交易所中取出你的硬币的好时机。**

如果您需要更多详细信息，请前往 Coindesk 获取更多最新信息。
