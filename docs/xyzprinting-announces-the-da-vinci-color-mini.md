# XYZPrinting 宣布达芬奇彩色迷你

XYZPrinting 可能终于破解了彩色 3D 打印代码。他们的最新机器，价值 1599 美元的达芬奇彩色迷你打印机是一台全彩色打印机，使用三个 CMY 墨盒在挤出时给细丝染色，允许多达 1500 万种颜色组合。

该打印机目前可在 [Indiegogo 上预购，价格为 999 美元](https://web.archive.org/web/20221207080224/https://www.indiegogo.com/projects/da-vinci-color-mini-full-color-3d-printer-design)。

这台打印机可以制造 5.1 英寸×5.1 英寸×5.1 英寸大小的物体，并且可以打印 PLA 或 PETG。一个小墨盒染色的 3D 彩色喷墨解放军，因为它出来，创造真正丰富多彩的对象。

“桌面全彩 3D 打印来了。现在，消费者可以以低于市场价 30，000 美元的价格购买一台操作简单、价格实惠的紧凑型全彩色 3D 打印机。XYZprinting 的首席执行官沈旭晖说:“这是革命性的，因为我们让公众获得了以前只有行业专业人士才能获得的技术。

新系统面向教育和家庭市场，售价不到 1000 澳元，在价格方面达到了一个独特而重要的平衡点。虽然打印并不完美，但能够以更好的单色 3D 打印机的价格进行全彩色打印是非常令人印象深刻的。
