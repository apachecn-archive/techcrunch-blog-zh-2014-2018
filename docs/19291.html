Skype 取消了故事、曲线和夸张的色彩，重新进行了设计 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/09/03/skype-rolls-back-its-redesign-by-ditching-stories-squiggles-and-over-the-top-color/

就在 Skype 推出受 Snapchat 启发的丰富多彩的改造项目一年多后，该公司表示，它现在将重新专注于简单性，并在此过程中放弃故事。重新设计遭到了强烈反对。Skype 显然希望通过其更新吸引更年轻的人群，但这样做的话，它用没有人要求或需要的功能扰乱了用户体验。

其中之一是“亮点”，这是 Skype 自己对 Snapchat 或 Instagram 故事的一个特色。有了 Highlights，Skype 用户可以向上滑动以拉起智能手机的摄像头，然后拍摄照片或录制视频，这些照片或视频可以用键入或手写的文本以及 Skype 自己的贴纸进行装饰。这可以与 Skype 的个人用户、团体分享，或者发布到应用程序的精彩部分。

上图:手机版 Skype

该公司当时辩称，社交媒体上故事的增加意味着所有社交应用都会采用这种方式。因为这是人们现在习惯的互动方式，Skype 也需要在自己的应用程序中加入这一功能。

但事实证明，故事可能不像“新闻订阅”界面那样无处不在或受欢迎——有些地方有意义，有些地方没有意义。Skype 是后者。

在微软的声明中，微软承认它引入的改变不起作用。

Skype 和 Outlook 的设计总监彼得·斯基尔曼(Peter Skillman)写道:“打电话变得越来越难，亮点没有引起大多数用户的共鸣。

相反，该应用程序引入了一种更简单的导航模式，现在手机应用程序的底部只有三个按钮——聊天、通话和联系人。高光和抓拍都没了。(如果你真的使用了 Highlights，你必须在 9 月 30 日之前下载并保存它们，直到该功能被删除)。

已经有一些迹象表明，微软正计划撤回其设计变更。在 T2 关闭 Skype Classic (Skype 7)的计划遭到了用户的强烈抗议后，该公司最近宣布将继续保留这款应用。该公司当时表示，在强制升级到 Skype 8.0 之前，将收集更多的反馈，以了解人们的需求。

随着新的桌面版 Skype 的推出，该公司现在表示，它正在将聊天、通话、联系人和通知移到窗口的左上角，以便长期 Skype 用户更容易理解。

Skype 还淡化了应用程序中过度使用的颜色，并引入了 Skype“经典”蓝色主题，以调整对比度和可读性。它还去掉了一些更傻的装饰元素，比如被剪成曲线形状的通知，它承认这“不是完成事情的核心”(你觉得呢？)

下图:曲线

虽然 Skype 现在倾听用户的声音是件好事——它说它正在全球市场测试新的原型，并推出了一个用户之声网站——但令人担忧的是它事先没有做足够的倾听。如果有，它就不会发布一个失败的应用程序版本。

Skype 应该拥抱它的“经典”地位，而不是觉得有必要追赶 Snapchat 等青少年聊天应用程序或 stories 等社交媒体趋势。人们使用 Skype 来完成事情——给远方的朋友打电话、打工作电话，甚至录制播客。作为一个简单而稳定的语音和视频通话应用程序，可以随着时间的推移留住忠诚的用户，并吸引那些需要跨平台交流的人，而不是在其他地方找到的所有绒毛。

最新设计可用于 Android、iOS、OS X、Linux 和 Windows 7、8 和 8.1 操作系统的 Skype 版本(8.29)。
