# 自动驾驶初创公司 Aurora 将与英伟达在自动驾驶方面合作

相对自动驾驶行业的新人 Aurora 最近透露了很多早期的牵引力，现在我们知道它也在与 GPU 制造商 Nvidia 合作。英伟达首席执行官黄仁勋透露，他的公司将为 Aurora 提供他们计算平台的底层技术。

这家初创公司上周透露，Aurora 正在为大众汽车和现代汽车制造自动驾驶系统。大众汽车(Volkswagen)正在寻求制造可以在相对较短的时间内上路的测试车辆，最终两家汽车制造商的目标都是最早在 2021 年让无人驾驶车队上路。

Aurora 使用 Nvidia 的自动驾驶计算机硬件作为其系统的基础这一事实并不令人惊讶——Nvidia 今天透露，它正在与 320 多家公司合作进行自动驾驶，通过其 Drive PX 等产品，现在驱动 Xavier GPU 驱动的自动驾驶计算机。

Aurora 在该领域也不是没有经验，尽管它相对年轻:其创始人包括谷歌自动驾驶项目早期团队成员 Chris Urmson，特斯拉自动驾驶建筑师 Sterling Anderson 和优步 ATG 大学校友 Drew Bagnell 所有人都可能在过去与 Nvidia AV 技术合作过。
