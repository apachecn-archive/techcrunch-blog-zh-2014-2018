不久前，一家区块链的初创公司向我推销他们的分散式社交媒体应用，用户只需做他们在其他平台上已经做过的事情，比如发布更新、照片和视频，就可以赚钱。

如果他们几年前给我发这条信息，我会很感兴趣。但是在对这个空间进行了几年的观察后，就不是这样了。

几个区块链的应用声称可以让用户将各种资源货币化，无论是他们未使用的存储和 CPU 能力，还是他们每天生成的大量数据。

不管他们是否会成功兑现他们的承诺，这些项目都凸显了困扰中央互联网的一个问题。用户很少因为他们给脸书、谷歌和亚马逊等平台带来的巨大价值而得到回报。

区块链应用表明，当前服务的分散替代方案将让用户有机会从他们参与在线生态系统所产生的收入中获得公平的份额。这是一个诱人的提议，因为它不需要用户做比他们已经在做的事情更多的事情:发送电子邮件、浏览网站、观看广告、保持电脑开机……

但是你从互联网上的资源货币化中到底赚了多少？你的收入有多容易获得和可靠？以下是你需要知道的。

你能卖什么？

少数区块链平台使你能够将你未使用的存储空间、闲置的 CPU 周期和互联网带宽出租给那些需要的人。前提很简单:你在应用程序上列出你的资源以及你的支付条款，当其他人使用它们时，你就可以在应用程序的专有加密令牌中获得报酬。购买通过智能合同点对点安排、执行和支付，智能合同是运行在区块链上的代码，不需要中央应用服务器。

例子包括假人和 iExec，两个计算能力的分散市场。用户可以通过向希望在网络上运行应用程序的开发者和用户出租 CPU 周期，分别获得平台的专有加密货币 GNT 和 RLC 令牌。假人和 iExec 的目标是取代亚马逊和谷歌等集中式云提供商，在这种情况下，服务提供商设定费率并收取所有利润。

Storj 和 Filecoin 是两个分布式存储网络，用户可以通过与网络共享免费硬盘空间来获得加密令牌。这两个平台都旨在为各种应用程序提供基础架构，如 web 托管和流媒体服务。Gladius 是一个分散式内容交付网络(CDN)和 DDoS 缓解解决方案，它使用户能够利用其互联网带宽来提供网站内容和网络上运行的服务。

这些应用程序提供了一个很好的机会，在你的电脑闲置在家或办公室的时候，把它变成一份额外的收入。

其他区块链平台让你可以利用你的数据赚钱。一个例子是 Datum，一个分散的用户数据市场。Datum 使用户能够通过选择与其他组织共享来获得 DAT 令牌。该领域的其他参与者包括 Streamr，这是一个面向物联网(IoT)的实时数据共享平台。通过 Streamr，用户可以通过将其连接的设备生成的数据与需要它来执行其功能的其他设备以及使用它们进行分析和研究的公司共享，来获得 DATAcoin 令牌。

数据是一个巨大的市场，目前由谷歌和脸书等几家大公司主导。这些公司将用户数据储存在他们的围墙花园筒仓中，并利用这些数据获取巨额利润。区块链平台通过还给用户数据的所有权，给用户选择和权力来要求他们的市场份额。

Matchpool 是一个去中心化的社交网络，让用户能够从他们的团体和在线社区中赚钱。Matchpool 提供了相当于脸书群组的去中心化工具，并为管理员提供了通过设定会员费和内容访问费来赚取 GUP 代币的工具。还有 Brave，由 Mozilla 前首席执行官开发的基于区块链的浏览器。Brave 从网站上移除广告，转而让用户通过选择观看广告来获得基本注意力令牌(BAT)。

你挣多少钱？

很难衡量在区块链应用上的收益，因为大多数应用要么还没有推出，要么还处于早期阶段。我接触过的公司中，很少有能提供稳定数字或平均数字的。

此外，你在这些平台上分享的资源的价值经常受到供求关系的影响。例如，iExec 让用户决定他们的计算资源的价格，并且不从他们的收入中抽取任何提成。如果对分散的 CPU 能力有很大的需求，你将从加入网络中获得更多。

Storj 是一个分散式存储网络，拥有最准确的信息可供共享。该平台提供了一个公式来计算“农民”的月收入，“农民”是与网络共享免费存储空间的用户。Storj 对存储的每千兆字节数据收费 0.015 美元，对下载的每千兆字节收费 0.05 美元，其中 60%归农民所有。

有几个因素会影响最终收益，包括农民节点是存储数据的主副本还是镜像副本，他们参与网络的时间以及他们在正常运行时间、带宽和响应时间方面的表现。Storj Labs 的首席技术官 Philip Hutchins 说:“如果有人整个月存储 1TB 的数据，并且整个 TB 的数据在那个月被下载一次，他们可能会赚 39 美元。但根据该公司共享的网络数据，Storj farmer node 目前的月平均费用约为 2 美元。

Storj 还与 FileZilla、微软和其他公司建立了合作伙伴关系，在其网络上构建分散的应用程序，这可能会增加对 Storj 空间的需求。

在分散数据市场 Datum 上，该公司首席执行官罗杰·海恩尼(Roger Haenni)表示，用户选择打开每封促销电子邮件，可以获得 0.50 至 5 美元的 DAT 代币，但他没有透露收入是如何计算的细节。目前，该网络支持货币化电子邮件收件箱，但在未来，该公司计划为用户提供通过共享各类数据获得报酬的选项，如他们的手机收集的位置数据、他们使用的应用程序、服务和网站、他们的智能设备收集的数据等。

最后一点听起来有点侵犯用户隐私。“这些[数据]目前被各种广告网络的 cookies 广泛跟踪，”Haenni 解释说。“然而，用户不会被要求明确选择分享这些数据，也不会在这些数据货币化时获得报酬。”Datum 将提供一个机会来要求从他们的数据中赚钱。

Datum 网络目前有 80，000 名用户，自 12 月底 Datum 应用程序推出以来，用户已经收集了 150 万个 DAT 令牌，金额约为 75，000 美元。

Gladius 是一家分散式 CDN 公司，它为通过一个节点传输的每千兆字节带宽的数据提供 0.03 美元的 GLA 令牌(然而，公司的网站声明这是基于有利的市场条件的估计)。一个上传速度为 30 mbps 的互联网连接每天与网络共享 8 小时，可以为其所有者每月赚取约 49 美元。

成本和风险是什么？

在大多数情况下，你已经为你将在区块链上分享的资源付费了，无论是你的硬盘空间、你的 CPU 还是你的带宽(除非你是按流量计费的连接，在这种情况下分享是不明智的)。然而，你必须考虑到保持电脑运行的电费，这取决于你居住的地区。

社交和数据共享平台不会产生任何额外费用，但你将负责在共享数据和保护隐私之间保持平衡。

赚取加密代币的真正风险之一是不断的价格波动。你今天赚的钱可能会在一夜之间翻倍，或者以同样的方式减少一半。这意味着你必须在持有代币和兑现之间做出选择。

而且总是有诈骗和失败项目的风险，这些项目会吸收用户的资金和资源，但最终会消失并被冷落。

“T2 区块链 T3 之上的资源共享项目允许用户控制并从他们自己的数据中获利，这将是未来最有利可图、最成功的项目，”区块链专家、DigiByte 创始人贾里德·泰特说。然而，泰特也指出，目前的许多资源共享平台是公关项目，永远不会扩大规模。

“那里的大多数项目在五年内都不会存在。泰特说:“大多数项目甚至没有工作软件，只有一份白皮书和一些网站上的精美图片。一些用户仅仅通过检查市值来评估项目，泰特认为这是衡量一个项目长期生存能力的最糟糕的方式。他警告称:“如此多的市值被开发商预先埋下的地雷或欺骗性的硬币计数人为夸大了。”。

你如何处理流动性问题？

用户必须克服的另一个挑战是如何处理他们从货币化资源中获得的代币。例如，如果你通过出租你的免费硬盘空间获得 Storj 代币，你唯一能做的就是从其他用户那里租赁存储空间，这是没有意义的，因为你已经有了多余的存储空间。

一些平台具有多方面的经济，使得用户能够将他们获得的代币用于各种目的。例如，在分散式流媒体服务 Flixxo 中，用户可以通过共享免费磁盘空间和带宽来在网络上托管内容，从而获得 FLIXX 令牌。然后，他们可以使用获得的代币来消费平台上发布的视频。但这仍然是一个有限的用例，可能不是他们想用收入解决的问题。

数字货币和代币存在流动性问题。很少有零售商和在线服务接受比特币作为支付方式，接受其他加密货币的就更少了。用户经常必须找到一些匹配各种数字和法定货币的买家和卖家的在线交易。这一过程缓慢而复杂，涉及不同级别的费用。

一个替代方案是 Bancor，一个建立在以太坊区块链之上的分散流动性网络。在其自有令牌 BNT 的支持下，Bancor 使用户能够在其网络支持的令牌之间进行转换，而无需找到买方或卖方。例如，如果你通过在 iExec 上出租空闲的 CPU 时间获得了一定数量的 RLC 代币，你可以立即在 Bancor 上用它交换，比如说，MANA，这种代币可以让你在分散的土地上购买 VR 体验。T11】

Bancor 已经在其网络上列出了几十个令牌，并计划在未来添加更多令牌。

Bancor 的联合创始人 Galia Benartzi 表示:“这种数学流动性解决方案的目的是让代币的长尾出现，让任何用户生成的货币在第一天都是可行的，而无需为了上市而实现巨大的交易量，从而变得具有流动性。”“伟大的代币仍将崛起，糟糕的代币将失败，但所有人都有机会尝试。”
