# Pixel 2.0 是一个微型有机发光二极管屏幕，可以让你享受可穿戴的 Arduino 乐趣

微小的 Pixel 2.0 基本上是一个与 1.5 英寸 128×128 彩色有机发光二极管屏幕结合在一起的 Arduino 板。这意味着你可以将它放在一个可穿戴设备中，直接从 Arduino 板寻址屏幕，这是对当前 DIY 电子产品“将屏幕焊接到 Arduino 并希望它能工作”的一个改进。

这个小小的板在 Kickstarter 上的价格为[75 美元](https://web.archive.org/web/20230323173335/https://www.kickstarter.com/projects/rabidprototypes/pixel-20-the-arduino-compatible-smart-display/)，将于 6 月份发货。它是完全开源的，因此您可以深入研究原理图，构建自己的原理图，或者使用它做任何您想做的事情。板载 SD 卡插槽让您可以存储游戏和视频等数据，您可以使用 Arduino SDK 将图形编程到屏幕上。

Pixel 2.0 已经超过了 5000 美元的目标，所以它已经准备好了。

这是由波士顿的 [Rabid 原型](https://web.archive.org/web/20230323173335/http://rabidprototypes.com/)创造的 Pixel 的第二个版本。他们还制作了一个叫做中微子的漂亮的高速 Arduino 板，并为你的 DIY 游乐园提供灯、马达和其他小工具。我很想看到一个微型街机游戏在这些东西上运行，考虑到可接受的像素密度，听起来它可能会工作。
