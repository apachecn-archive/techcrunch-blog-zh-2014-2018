# 移动数据创业公司 Zeotap 将€融资 1200 万英镑

[Zeotap](https://web.archive.org/web/20230319064002/https://www.zeotap.com/) ，一家帮助[电信公司向广告商出售其数据的初创公司，](https://web.archive.org/web/20230319064002/https://techcrunch.com/2015/08/04/zeotap/)今天宣布，它已经在 B 轮融资中为€筹集了 1200 万英镑(不到 1300 万美元)。

联合创始人兼首席产品官 Projjol Banerjea 告诉我，虽然针对桌面用户的在线广告商有“体面的数据”，但移动端的情况不太稳定:“市场上可用的大部分数据往往质量相当差。”

具体来说，他说可用的数据往往不太可靠，因为它是基于“概率建模”通过使用来自电信公司的数据，Banerjea 说 Zeotap 拥有更广泛的数据——包括人口统计、位置和购买意图——这也更可靠，因为它是确定性的(这通常意味着它是由用户自己提供的)。

虽然 Zeotap 最初是与电信公司合作，但 Banerjea 表示，其他公司(根据资金公告，包括“一家领先的安全公司”和“一家顶级导航服务提供商”)也与他的团队接触，出售他们的数据。

至于隐私，Zeotap 说它不包括任何个人身份信息。Banerjea 还表示，该公司战略的一个关键因素是，它阻止广告商使用持久标识符存储数据。

[B 轮](https://web.archive.org/web/20230319064002/https://www.crunchbase.com/organization/zeotap#/entity)来自[新科创投](https://web.archive.org/web/20230319064002/https://www.crunchbase.com/organization/new-science-ventures#/entity)和位置服务提供商[这里的](https://web.archive.org/web/20230319064002/https://www.crunchbase.com/organization/here-inc#/entity)，以及之前的投资者 Capnamic Ventures 和 Iris Capital。Zeotap 现在已经筹集了超过 2000 万美元。

Zeotap 的战略投资者之一，Xaxis 的联合创始人马克·格雷特(Mark Grether)通过电子邮件表示，广告的核心原则之一是“永远不要直接分享数据”。

“一旦进入市场，它就不受控制，失去了价值，”格雷特说。“当我开始了解 zeotap 时，我重新发现了这个原则，并意识到该公司在收集最高质量的数据源并在整个生态系统中将其货币化方面处于有利地位。他们不仅巧妙地解决了数据安全和隐私问题，还在市场中占据中立地位，可以安全地向任何想要利用这些数据的人提供数据。”
