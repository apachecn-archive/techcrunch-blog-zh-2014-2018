# 新款 MacBook Pro 可能会配备触控 ID 传感器和有机发光二极管迷你屏幕

有传言称，苹果即将在未来几个月内为 MacBook Pro 更新视网膜显示屏。据消息灵通的 KGI 证券分析师 Ming-Chi Kuo 和 [9to5mac](https://web.archive.org/web/20230326070139/http://9to5mac.com/2016/05/23/apple-prepping-thinner-macbook-pros-with-oled-screen-above-keyboard-touch-id-for-q4/) 称，这应该不仅仅是一个规格的提高。

该公司希望保持两种屏幕尺寸，一种是 13 英寸的，另一种是功能更强大的 15 英寸。但这些笔记本电脑应该采用更薄更轻的设计，更接近 MacBook Air。

但如果你有一台 MacBook Pro，你就会知道笔记本电脑的厚度受到端口的限制。retina MacBook Pro 的厚度大致相当于 USB 端口、Thunderbolt 端口或 HDMI 端口的厚度——这三种端口的厚度大致相同。

那么苹果将如何解决这个问题呢？答案是 USB-C。苹果已经在配备 retina 显示屏的 12 英寸 MacBook 中使用这种更薄的 USB 标准。而且看起来公司并不想就此止步。USB 口大概要变成 USB-C 口了。但你也可以制造兼容 Thunderbolt 的 USB-C 端口，因此苹果也可以为 Thunderbolt 端口改用 USB-C 设计。同样，苹果可以利用 USB-C 端口来取消 MagSafe 连接器。最后，苹果可以取消 HDMI 端口，或者用迷你 HDMI 端口代替。

根据这些报道，苹果也在考虑在你的 Mac 电脑上安装一个触控 ID 传感器。很高兴你不用再输入密码了。但这也可能带来其他可能性，比如在 Mac 上使用 Apple Pay 进行网上购物。

最后添加的内容有点令人惊讶。苹果可能会在你的键盘上方增加一个小小的有机发光二极管触摸屏，这要归功于一个新的、更小的金属注射成型铰链。虽然苹果从未生产过带触摸屏的笔记本电脑，但这种窄窄的触摸屏可以提供快捷方式，并且可以根据你使用的应用程序而改变。这可能是新款 MacBook Pro 最令人惊讶的元素。

新的 MacBook Pro 将采用新的英特尔 Skylake 处理器。它也可以采用重新设计的键盘，看起来像[魔法键盘](https://web.archive.org/web/20230326070139/https://techcrunch.com/2015/10/13/force-touch-for-imac-users/)。

不幸的是，今天的报道称，你不应该指望在 2016 年第四季度之前购买新的 MacBook Pro。许多人期待在[苹果 WWDC](https://web.archive.org/web/20230326070139/https://techcrunch.com/2016/04/18/apples-wwdc-will-be-held-june-13-to-june-17/) 发布一个重大的声明，但是看起来它还没有准备好。
