# Siri 获得了语言翻译和更人性化的声音

Siri 越来越不傻，也越来越人性化了。这并不矛盾。

苹果的语音助手在女性和男性语调方面都变得更加清晰，同时 iOS 11 中的一些新技能将提高其整体性能。

苹果公司今天宣布，Siri 现在每月在超过 3.75 亿台设备上使用。

苹果将翻译功能引入 Siri，这样你就可以问语音助手如何用多种语言说出某个英语短语，包括中文、法语、德语、意大利语和西班牙语。

在过去几年里，随着亚马逊的 Alexa 语音助手和谷歌助手功能的增长，Siri 遇到了更多的竞争，并使 Siri 的人工智能智能的缺点变得更加明显。通过这些变化，苹果希望展示它仍然是语音助手的对手，尽管我们必须等到今年晚些时候才能看到自己。

Siri 现在将能够处理后续问题，这是谷歌助手很久以前就有的，并且现在将允许 Siri 回答更复杂的分层问题。

较少报道的事件标题是，苹果越来越认为 Siri 是一个通过 iOS 扩展的通用人工智能，而不仅仅是一个语音助手。例如，在 iMessages 中，iOS 上的 Siri 对你的环境变得更加敏感。如果有人问你“你在哪里？”在 iMessages 中，应用程序会弹出一个你当前 GPS 位置的窗口。
