英伟达首席执行官澄清其图形处理器“绝对”对 Meltdown 和 Spectre 免疫

英伟达(Nvidia)周二发布了一份安全公告，详细介绍了它对其驱动软件所做的更新，以解决谷歌 Project Zero 揭示的所谓的 Meltdown CPU 漏洞。该公告被一些媒体误解为承认 Nvidia 的 GPU 也受到了影响。

“我们的 GPU 是免疫的，它们不受这些安全问题的影响，”英伟达首席执行官黄仁勋在今天上午的新闻问答中说。“我们所做的是发布驱动程序更新，以修补 CPU 安全漏洞。我们和亚马逊、SAP、微软等公司一样修补 CPU 漏洞，因为我们也有软件。”

黄解释说，任何人运行任何类型的软件都必须为研究人员发现的 CPU 漏洞打补丁。他强调，英伟达绝不是在修补自己硬件中的任何问题。

“我绝对肯定我们的 GPU 没有受到影响，”黄毫不含糊地说。

NVIDIA也更新了它的安全公告，以表明目前就他们所知，它自己的硬件产品没有受到被披露的漏洞的影响。
