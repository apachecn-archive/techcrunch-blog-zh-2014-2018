# Twitter 增加了对基于应用的双因素认证的支持

Twitter 正在推出其平台安全更新，允许用户使用第三方认证应用程序为其 Twitter 帐户接收双重登录认证。Twitter 提供双因素认证已经有很长时间了，但它一直过度使用不太安全的基于短信的认证方法。

第三方应用程序支持意味着您可以使用 Google Authenticator、Authy 或 Duo Mobile 等工具来验证您的登录，而不是 SMS 您需要做的只是在桌面上使用您的移动 Twitter 应用程序、手机摄像头和唯一生成的二维码进行一次性验证，以建立应用程序和 Twitter 之间的连接。关于如何设置的更多信息，请查看关于主题的 [Twitter 官方支持文档。](https://web.archive.org/web/20230206173631/https://help.twitter.com/en/managing-your-account/two-factor-authentication)

如果你想知道你为什么要这样做，它比 SMS 更安全，因为基于 SMS 的 two factor 容易受到黑客的攻击，攻击者会接管你的移动帐户，例如通过向人工服务代理申请新的 SIM 卡。认证应用更难刷，即使你换了号码或无法使用你的手机，它们也能跟着你。
