# 有钱的创始人和法拉利的秘密

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/04/30/the-secret-about-rich-founders-and-ferraris/>

到目前为止，愤怒的机器正全力以赴关闭山谷亲爱的秘密。曾经飞得很高的应用程序，允许你匿名地与你的朋友和更广阔的世界分享细节，最终在它的成长过程中磕磕绊绊，[导致其创始人昨天关闭了它](https://web.archive.org/web/20230326215144/https://techcrunch.com/2015/04/29/psst/)。

CEO 开车离开时的解雇(现在不存在)(红色！)法拉利是新闻集团的绝佳素材，这些新闻集团关心创始人的问题以及初创公司在多大程度上是富人的游戏。现在，媒体，尤其是 Twitter 上，广泛要求联合创始人向投资者返还数百万美元，以示悔悟。

我说，去他们的，留下货物。

二次销售是当今风险投资领域的一个关键且常见的部分。在这部秘密制作的过程中，没有风险投资人受到伤害，坦白地说，我只能对风险投资人因自己的决策而赔钱感到愤怒(他们中有多少人拥有法拉利？其中有多少是红色的？)

二次销售很容易理解。当一家公司筹集资金时，这些资金会作为资产负债表上的现金直接进入公司，这就是为什么它被称为一级资本。次要是指创始人甚至其他员工以该轮价格出售他们在公司的部分个人股份。这些钱没有进入公司，而是进入了卖家的口袋。

在私募股权收购中，如果一个创始人自办了一家企业，并且仍然拥有全部或几乎全部的股票，那么投资于一家公司的大部分资金实际上可能是次级资金。此类交易必须平衡投资者的所有权愿望和将资金放入资产负债表以推动公司增长的需求。因此，要让一级和二级资本水平正确，以产生最高的投资回报，并确保创始人同意交易，这是一门复杂的艺术。

这类交易在高增长的初创企业领域肯定不太常见，但它们变得越来越受欢迎。就在过去几年里， [Snapchat 的创始人在第二阶段各拿出 1000 万美元](https://web.archive.org/web/20230326215144/https://techcrunch.com/2013/06/22/source-snapchat-snaps-up-80m-from-ivp-at-a-800m-valuation/),[four square 的创始人拿出 460 万美元](https://web.archive.org/web/20230326215144/https://techcrunch.com/2010/07/14/foursquare-founders-4-6-million/)。Groupon 和 Zynga 等其他公司也经历了这些第二轮融资，尽管往往是在后期阶段。

虽然投资者部分使用这一机制是为了获得额外的股权——特别是在社交网络初创公司的高价融资中——但这些二次销售的关键原因是激励一致。投资者认为，给创始人几百万美元会让他们更加努力地工作，取得更大的成就。

我看到人们有时将这些美元描述为让创始人生活得更舒适，这样他们就不必考虑租金或吃饭。这是不对的。这些美元是为了让创始人尽早感受到富有和成功，这样他们就能体会到如果他们的公司再涨 100 倍会是什么感觉。Byttow 买了一个(红色！)法拉利，还有【Snapchat 创始人埃文·斯皮格尔在二次出售(也是红色！).把它想象成可卡因——一旦你有了一次，你以后会想要更多。

对于最好的公司来说，二次出售的需求通常更多来自投资者，而不是创始人。很容易看出为什么。如果创始人真的相信他们公司的增长轨迹，他们会以低于他们认为的价值 99%的价格出售他们的股份。Speigel 两年前价值 1，000 万美元的次级股票现在可能价值约 2 亿美元。真是敲竹杠！

投资者依靠创始人的傲慢让这一切运转起来。当你在这些联盟打球时，没有人想成为那种在成功的社交创业中只赚了 500 万或 1000 万美元的人。投资者还试图阻止早期收购出售，因为创始人在收购中以 5000 万美元出售股票与在 IPO 中获得 20 亿美元或更多美元之间存在身份差异。

这让我们又回到了秘密。当投资者给创始人二级股权时，他们认为公司可能价值数十亿，但也明白它可能会破产。创始人二级会议是一种确保创始人要么将公司推向顶峰，要么拼命努力的手段。他们死了。这就是风险投资中的交易。我很难理解像 Index 和 Redpoint 这样成熟有经验的公司是如何在这一切中被愚弄的。

如果有什么值得愤怒的话，那就是创始人和初创企业早期员工的待遇严重不平等。创始人被认为是不可或缺的，因此需要被收购以确保他们坚持到底。这与初创公司员工的待遇完全相反，这些天来，他们看到自己的股票被公司完全锁定，几乎没有流动性选择。

如果我们要抱怨秘密和次要，抱怨更多的员工没有收到它。这种不平等是二级市场的真正秘密。)法拉利。
