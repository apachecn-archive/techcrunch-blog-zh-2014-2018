Yala 的新聊天机器人知道发布到社交网络的最佳时间

原文：https://web.archive.org/web/https://techcrunch.com/2016/09/08/yalas-new-chatbot-knows-the-best-time-to-post-to-social-networks/

需要同时向多个社交网络发布内容？一个名为 Yala 的新聊天机器人想要帮忙。这个机器人现在正在 Slack 上运行，Facebook Messenger 支持即将推出，它不仅可以为你发帖，还足够智能，知道什么时候发帖。也就是说，Yala 使用算法和机器学习来确定发布的最佳时间，以增加你的曝光率，并在你分享的项目上获得更好的参与度。

这个机器人的想法来自加里·莱维特，他曾在 2014 年将其电子邮件营销初创公司 Mad Mimi 出售给 GoDaddy 。后来，他试图通过网上店面服务松果进入电子商务领域，但他说他很难让它起步。这导致了雅拉的转向——这解决了莱维特在建造松果时面临的一个问题。

“我对 Buffer 及其不那么令人印象深刻的竞争对手不满意，因为多渠道发布需要时间，”Levitt 解释说，“尤其是在跨多个渠道安排、编辑、选择时间和作曲方面，”他说。

Buffer 作为后台，是一个社交媒体调度服务，还提供工具帮助你确定发布的最佳时间。

然而，对于 Yala 来说，这个想法是将类似的智能带到“离家更近的地方”，Levitt 说，意思是松弛。

为了确定发帖的最佳时间，Yala 会查看你的社交媒体发帖历史，以确定哪些时间和帖子获得了最多的分享、喜欢和点击。它还考虑了时区和其他观众行为等因素。此外，根据你自己的社交媒体参与度指标，机器学习方面使它能够在运行中改进并自动纠正自己。

Levitt 说，通过能够识别模式，它能够创建一个发布时间表，实时调整自己。

该团队在测试期间使用 Yala 时，看到了 10%的份额和帖子增长。(如果你好奇的话，Yala 是以色列的俚语，从阿拉伯语中借用过来，意思是“继续前进。”)

目前，该机器人与脸书和 Twitter 合作，并在 Slack 上运行。其他平台和社交网络正在开发中。

Yala 是一个由三个人组成的团队，总部设在纽约和以色列。
