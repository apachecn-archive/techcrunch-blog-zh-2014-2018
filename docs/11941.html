海洋生物如何启发软机器人的工业夹钳

原文：https://web.archive.org/web/https://techcrunch.com/2017/04/01/soft-robotics-grippers/

< p 级=【drop cap】>在马萨诸塞州剑桥软机器人的情人节。一名工作人员将心形棉花糖放在传送带上。一只机械臂把它们一个接一个地抓起来，轻轻地放在附近的盒子里。实际上，这并不是什么浪漫的庆祝。但这也算是一种胜利。

Soft Robotics 的 RL7 和其他夹钳可以可靠地执行一些我们每天做无数次但已经学会视为理所当然的物理任务。他们可以拿起一个柔韧的物体，并把它放在它需要去的地方，所有这些都不需要用计算机视觉系统或任何类型的预编程来识别物体。

除了粉红色的 Peeps，这家初创公司的实验室还散落着一系列奇怪的物品。一边是一袋袋的花生和各种各样的农产品，房间的另一边是仿制的毛皮玩具和塑料冷冻玩具。Soft Robotics 首席执行官卡尔·沃斯承认，该实验室有一个可怕的一元店的气质，每个对象都因其形状不一致而被选中。但他解释说，库存代表了真实世界产品的一个很好的横截面。

Soft Robotics 的夹钳成立于 2013 年，已经被制造商和零售商用于挑选和包装从巧克力到注塑零件到生披萨面团的一切东西。夹子看起来不像人手。相反，它们由卡通风格的四个橡胶般的亮蓝色手指组成，像章鱼抓住猎物一样咬住目标。

该公司的名字是对用于制造其夹钳的软材料和更大的机器人领域中的一个子类别的认可。沃斯将他对公司的灵感追溯到 2011 年与乔治·怀特塞德斯合著的一篇论文，乔治·怀特塞德斯现在是软机器人公司的董事会成员。

在书中，这位哈佛教授详细描述了一个四条腿的 X 形四足机器人，在《异形》续集中，它看起来注定要贴在一名毫无防备的乘客的脸上。“我们对一类独特的机器人感兴趣，”论文解释道。“也就是说，用不使用刚性骨架来提供机械强度的材料[……]制造的软机器人。这项工作的目标是展示一个只需要简单设计和控制就能产生机动性的软机器人。”

为了设计一个既能行走又能通过门下一厘米缝隙的柔软机器人，怀特塞德斯在人类部落之外寻找生物灵感。像他们之前的其他机器人专家一样，怀特塞德斯的团队在海洋中找到了灵感。

最终产品在互联网上流传开来，这得益于它令人印象深刻的能力，即在膨胀中通过起伏的四肢行走，以及它公认的令人毛骨悚然的外表。同年，Whitesides 的团队将该设计应用于其第一个柔性机器人手爪，这次形状像海星。

由钓鱼线和一根输送压缩空气的管子悬挂着，该团队展示了把手能够拿起一个未煮熟的鸡蛋而不压碎它。该团队在一只活老鼠身上做了同样的事情，展示了在正确的聚合物材料和气压下，手如何能够安全地抓住一个物体而不伤害它——这一行为需要在一个更刚性的夹子中安装一系列更复杂的计算机视觉和传感器。

“我们认为这是一项突破性的技术，可以在一个简单的合规外形中为机器人带来类似人类的灵活性，”Vause 说。“[传统的机器人手爪]必须知道物体的准确位置。它必须计算它的路径，所以有大量的数字计算叫做路径规划，然后手指上有传感器，所以它必须抓住它们，但不要太多。因此，解决这个问题可能会非常缓慢和繁琐。”

到 2014 年，软机器人拥有了第一个公开原型。2015 年底，该公司将其夹持器投入商业使用，将这些夹持器连接到 ABB 等公司的第三方工业机械臂的末端。

自推出以来，Soft Robotics 的夹持器已被证明在食品运输中最为成功，在这个行业中，产品很少坚持严格的尺寸和形状一致性。当您的设备在一天结束时可以简单地冲洗时，它也有助于食品行业。

“当我们开始时，我们认为我们将在大约三年的时间内研究食物，”沃斯说，“但拥有可以直接处理食物的可清洁机器人系统的能力是一个巨大的未满足的需求。所以，我们现在大约 80%的业务是食品，从烘焙到生产的一切。"

Soft Robotics 并不是第一家将这项以其命名的技术应用于生产和实践的公司。成立于 2012 年的帝国机器人公司推出了 Versaball，这是一款基于类似原理的手爪，达到了极简主义的极致。抓手由一个充满沙状颗粒物质的气球状球组成。空气被泵入和泵出物体以调节握力。它不是用来拾取食物的，而是像小金属零件一样，你需要将它们分类并送到工厂的装配点。

尽管有 Versaball 的正面报道，包括与吉米·法伦的深夜啤酒乒乓赛，帝国还是失败了。它设计的气球状抓手无法承受高重复的工业使用。该公司在 2016 年陷入沉寂。

然而，Soft Robotics 的灵感来自头足类动物的机器人似乎正在获得牵引力，这在很大程度上要归功于它的多功能性和最小化的设置。沃斯说，根据机器人在给定设置下将要处理的产品，安装人员需要选择连接哪种类型的橡胶手指，并调整用于驱动这些夹钳的气压。否则，机器人就是即插即用的。

该公司已经设法克服了将突破性技术从实验室带到工业环境中的商业应用的障碍。

“最大的挑战之一是，如果你考虑学术工作，你希望得到一些运行几次，”沃斯解释说。“你得到了数据，发表了论文，登上了《自然》杂志或某大期刊的封面。在工业中，尤其是机器人，这些机器需要 24/7 运行，因此，一个月要运行数百万个周期。采用一种能够经受住工业磨损的技术确实是最大的挑战。”
