# 微信计划向早期非洲创业公司投资 340 万美元

中国最大的通讯应用将向非洲初创公司投资 5000 万兰特(约 340 万美元)。[微信](https://web.archive.org/web/20221203090506/http://www.wechat.com/en/)在[的博客文章](https://web.archive.org/web/20221203090506/http://blog.wechat.com/2015/12/08/wechat-invests-r50-million-seed-fund-in-african-tech-startups/)中表示，将与开普敦科技战略公司 [Batstone](https://web.archive.org/web/20221203090506/http://www.batstones.com/) 合作，寻找有前途的早期公司。

今年早些时候，中国互联网巨头[腾讯](https://web.archive.org/web/20221203090506/http://www.tencent.com/index_e.shtml)旗下拥有超过 6 亿用户的微信[在非洲进行了第一笔投资](https://web.archive.org/web/20221203090506/https://beta.techcrunch.com/2015/02/20/m4jam-wechat-africa/)[，向](https://web.archive.org/web/20221203090506/https://beta.techcrunch.com/2015/02/20/m4jam-wechat-africa/)[M4JAM(Jams 的钱)](https://web.archive.org/web/20221203090506/https://www.m4jam.com/)投入了未披露的金额。

总部位于南非的 M4JAM 与 Taskrabbit 相似，它将人们与临时的“微工作”联系起来。然而，与 Taskrabbit 不同的是，M4JAM [在推出时只通过微信](https://web.archive.org/web/20221203090506/http://www.m4jam.com/wechat/guide)接触到用户(尽管它现在可以作为一项独立的服务提供)。几个月后，微信也宣布与 PicUp 合作，PicUp 的总部也在南非，只能通过微信访问[。](https://web.archive.org/web/20221203090506/http://www.picup.co.za/how-it-works.html)

在今天的帖子中，微信表示，其新基金将寻找像 M4JAM 和 PicUp 这样使用消息应用开展业务的其他公司，这反过来可能有助于它利用非洲迅速提高的智能手机普及率。

微信[在全球拥有用户](https://web.archive.org/web/20221203090506/http://indianexpress.com/article/technology/tech-news-technology/wechat-claims-549-million-users-globally/)，但它没有披露具体国家的用户数量。这项服务在中国无处不在，可以用来玩游戏、支付水电费、预订出租车和在线支付，但它很难在其他市场站稳脚跟，与 WhatsApp、Facebook Messenger 和 Viber 等应用程序竞争。这些消息平台都将非洲国家视为增长机遇(也有本土应用，如南非的 [Mxit](https://web.archive.org/web/20221203090506/http://get.mxit.com/) 和尼日利亚的 [Howfar](https://web.archive.org/web/20221203090506/http://www.howfar.ng/) )。

在智能手机普及率较低的国家，短信仍然占据主导地位，但在南非和尼日利亚等拥有更多智能手机的国家，短信应用正在迅速发展。

TechCrunch 已向微信询问有关该基金的更多细节。
