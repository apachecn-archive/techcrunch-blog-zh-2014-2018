联网汽车公司 Zubie 为“车联网”TechCrunch 推出 API

原文：https://web.archive.org/web/https://techcrunch.com/2015/07/15/connected-car-company-zubie-rolls-out-an-api-for-the-internet-of-cars/?ncid=rss

Zubie 是一家销售设备的公司，该设备允许司机接入他们汽车的车载计算机，然后将车辆诊断数据同步到他们的智能手机上，该公司今天推出了一个开放的 API，允许开发人员在这个平台上进行开发，从而扩大了其业务范围。该 API 被称为“锌”——或“车联网的 Zubie 它为合作伙伴提供了访问车辆诊断以及位置数据、行程活动和驾驶数据的能力，供他们在自己的应用中使用。

这些应用程序可以专注于消费者用例，例如那些旨在跟踪和改善燃油经济性的应用程序或社交旅行共享应用程序。或者它们可以针对企业，包括涉及车队管理的企业，需要将车辆数据绑定到费用或计费平台的企业，甚至是为拼车服务设计的企业，等等。

该公司表示，它已经看到了数百名开发者和合作伙伴的兴趣，他们希望在他们的应用程序中利用新的 API。在线服务预约预订平台 Openbay 和按需道路救援提供商 Urgent.ly 是消费者可能熟悉的两个名字；与此同时，在商业方面，ZinC API 正在为 Safeco 的青少年驾驶项目的部署提供动力，此外它还与 Progressive、AutoNation、GeekSquad 和 Telefonica 的 O2 Car Connection 等公司合作。

Zubie 现在是当今市场上许多联网汽车平台之一。其中许多公司最初是通过向消费者提供一种插入汽车车载计算机的加密狗来起步的，以便用户可以通过智能手机应用程序查看和跟踪汽车数据，以及油耗、通勤和旅行时间等信息。例如，这一领域的其他知名品牌包括 Dash 和 Automatic。

最近，这些公司中的许多公司也在试图扩展，不仅仅是一个汽车加密狗到应用程序的服务提供商，而是希望更多地充当第三方应用程序的平台。事实上，今年早些时候，竞争对手 Automatic 推出了自己的软件开发套件，旨在将其硬件转变为一个平台。

与此同时，有消息称，超过 20 家应用程序开发商在 Automatic 的基础上开发了应用程序，包括那些针对商业用户的应用程序，如与费用追踪器 Concur 的集成，以及更多有趣的社交应用程序，如 Unmooch，允许朋友分摊燃料费用。Automatic 随后在由战略投资者 USAA 领导的一轮新融资中筹集了 2400 万美元，这表明那些能够在企业应用程序中利用车辆数据的公司对联网汽车领域感兴趣。例如，USAA 在其保险业务中看到了联网汽车技术的一个用例。

同样，Zubie 还通过其应用程序和现在的 API 瞄准了消费者市场以外的更广泛的细分市场。该公司还与小型企业和其他垂直业务合作，包括汽车和保险行业的战略合作伙伴，以及电信运营商。
