# Thrillist 通过三个新的垂直市场扩大其覆盖范围

下面是惊悚媒体集团花费[今年早些时候筹集的 5400 万美元](https://web.archive.org/web/20221210070754/http://recode.net/2015/09/30/thrillist-and-jackthreads-raise-54-million-split-into-separate-companies/)的一种方式:在其网站上增加三个新的部分，一个是娱乐部分，一个是健康部分，一个是性和约会部分。

很自然，公司会雇佣一些新员工来支持这种扩张。纽约杂志《秃鹫》的前高级编辑约翰·塞勒斯(John Sellers)将负责娱乐垂直业务。MindBodyGreen 的前执行主编 Anthony Schneck 是新的健康编辑。以及《花花女郎》前主编妮可·考德威尔(！)，将担任性与约会编辑。

Thrillist 是由 Ben Lerer(见上图)和 Adam Rich 在十年前创建的，是一个男性生活方式网站，到目前为止主要关注食物、饮料和旅行。销售总裁埃里克·阿什曼说，新的部分是“我们已经在做的内容的演变。”

性和约会与食物和饮料有明显的联系，但更广泛地说，Ashman 说 Thrillist 旨在帮助读者“在日常生活中推动决策”虽然该公司目前专注于这三个产品，但它也在考虑明年增加更多的类别:“这肯定不是我们扩张的最后一次。”

是的，新的部分需要有本地相关的内容。毕竟，阿什曼说，“Thrillist 上的头号搜索词是早午餐。我们的北极星是早午餐。人们需要能够在周日早上来到 Thrillist，找到一个吃早午餐的好地方。”

该公司还宣布，它已经聘请了前视频节目和制作在 Bleacher Report 副总裁比尔·麦坎德斯，以扩大其视频制作工作。哦，它的技术网站 Supercompressor 现在是 Thrillist 网站的一部分。

“最终，我们的目标是为千禧一代以及更广泛的受众建立一个卓越的生活方式目的地，”阿什曼说。"我们用的短语是，想要过快乐生活的人."
