# Tumblr 将苹果的实时照片带到了网络上

[Tumblr](https://web.archive.org/web/20221128191704/https://www.crunchbase.com/organization/tumblr#/entity) 今天正在[将其服务的](https://web.archive.org/web/20221128191704/https://staff.tumblr.com/post/150686995095/baaaaaaaaaa)版本带入网络，同时也推出了[工具](https://web.archive.org/web/20221128191704/https://github.com/tumblr/laphs)，这将允许其他网络开发者也这样做。该公司发布了一个开源的 JavaScript 库，它称之为“laps”——或“Live Anywhere Photos”——它允许您在任何网站上支持 Live Photos。要激活网络上的实时照片，您可以使用点击并按住，而不是像在手机上那样按下按钮。

苹果的 Live Photos 是苹果 6s 和 6s Plus 推出的一种文件格式，是静止图像和短视频的组合，它们结合在一起创造了独特的播放体验。为了在网络上运行，Tumblr 会收到两个静止图像的副本。jpeg)和视频(。mov)在用户从他们的 iOS 设备上传一张实时照片后。它对。mov 文件转换为. mp4，使其在所有浏览器上正常工作。

然后，Tumblr 使用其[laps](https://web.archive.org/web/20221128191704/https://github.com/tumblr/laphs)解决方案，通过将。jpeg 和. mp4 文件来重新创建 Live Photo 播放效果。

Tumblr 已经是 Live Photos 的早期采用者，使用苹果手机的 3D 触摸屏，Live Photos 在被按下之前一直是静止的。然后它们会产生动画，给你的照片一种哈利·波特的感觉。

由于 Live Photos 与在 Tumblr 上非常流行的 gif 相似，该博客网站很快在手机上推出了对这种新格式的支持。事实上，它成为了第一个这样做的第三方服务——甚至比脸书快了几周，比谷歌照片快了几个月。它还支持通过手机上的 GIF Maker 功能将实时照片转换为 GIF。

现在，通过 [Laphs](https://web.archive.org/web/20221128191704/https://github.com/tumblr/laphs) ，Tumblr 甚至有可能在自己的网站之外推动实时照片的进一步采用。该公司还表示，正在努力将实时照片引入安卓系统。

Tumblr 网站上的实时照片标有实时照片图标——熟悉的同心圆。这是在有问题的图像的左上方找到的。

例如，在 Tumblr 关于新功能的公告上，你可以[看到一张现场照片。要查看动态的实时照片，只需用鼠标点击并按住即可。](https://web.archive.org/web/20221128191704/https://staff.tumblr.com/post/150686995095/baaaaaaaaaa)
