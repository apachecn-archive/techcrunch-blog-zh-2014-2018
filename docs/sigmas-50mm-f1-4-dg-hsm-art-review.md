# 适马 50 毫米 F1.4 DG HSM 艺术评论

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/07/07/sigmas-50mm-f1-4-dg-hsm-art-review/>

第三方高端相机配件制造商很难脱颖而出，吸引消费者的注意力。佳能和尼康为他们自己的相机系统制造镜头，这些镜头往往比大多数相机更好——部分原因是他们在内部制造镜头，拥有所有内在优势。但像适马这样的公司多年来一直在提供替代品，通常具有成本优势。

该公司最近走上了一条[的道路，重塑自己](https://web.archive.org/web/20221205071949/http://www.sigmaphoto.com/article/sigma-corporation-announces-reorganization-of-lens-lineup-new-products-and-quality-control)和它的镜头系列，专注于更高的质量和价值，然而，这种方法甚至让品牌忠实者都刮目相看。

[适马新的 50 毫米镜头](https://web.archive.org/web/20221205071949/http://www.sigmaphoto.com/product/50mm-f14-dg-hsm-a)拥有非常宽的最大光圈，为 F1.4，并设法低于你为佳能顶级 50 毫米 F1.2L 镜头支付的 1600 美元左右。这种方便的人像/旅行/几乎任何东西都可以的镜头可能便宜 600 美元左右，但在图像质量方面它也能与 L glass 相媲美。例如，适马的清晰度，特别是在更宽的光圈下，比佳能好得多，而且你可以在散景和巧妙的选择性对焦方面产生同样令人惊叹的视觉效果。

在我进入关于性能的更多细节之前，我应该解决适马 50 毫米的镜头结构。这块玻璃的硬件构建非常出色:外部结构使用传统金属和定制的热稳定复合材料，以提高处理各种温度的能力，并配有一个黄铜卡口底座，用于连接到您的相机。

它握在手中感觉很重，这是内部包含大量玻璃的外在指标，这也是图像质量优异的原因。附带的镜头盖扣合牢固，表面涂有橡胶，与镜头本身的整体哑光黑色外观相得益彰。说实话，你真的不能把一个更好看的镜头放在你的佳能 DSLR 上，不管有没有红环

当然，外表并不能构成镜头:或者说，图像的外观才是最重要的。适马 50 F1.4 在这方面也没有让人失望。九叶片光圈有助于创建平滑、圆润的散景，自动对焦马达安静且相当快，尽管这不是我用来捕捉动作和飞鸟的镜头。当然，由于其焦距和宽光圈，它更适合人像摄影和一般的步行摄影——但它完全适合这些任务，而佳能的低端 50 毫米 1.4 和更贵的 50 毫米 1.2 升则不适合。

当与我的佳能 6D 机身结合时，全画幅传感器和宽光圈意味着我可以照亮甚至是夜间的场景，而在图像噪声方面只需相对较小的拉伸或牺牲，或者不需要外部光源。因此，适马 50 毫米是街头摄影师寻找一个良好的焦距，也可以在所有光线条件下拍摄的绝佳选择。对于在家庭聚会或朋友聚会时拍摄的抓拍照片，以我的经验来看，很少有组合能与 50 毫米的全画幅机身相媲美。

> 适马已经从一个稳定可靠的预算选择转变为一个热门的制造商。

我已经养成了随身携带两个佳能变焦镜头的习惯，一个可以覆盖 24-70 的范围，另一个可以覆盖 70-200 的范围。这对于我拍摄活动、动手演示、视频采访和更多的工作场合都非常有用，也是一个沉重但方便的旅行工具包。但适马的 50 毫米 F1.4 让我想起了用质数拍摄的感觉，以及当你有能力用极限光圈伸展自己，当你必须创造性地使用双脚和框架，而不是依赖变焦来获得正确的镜头时，摄影会有多么有趣。

适马还有一个特殊的配件，可以搭配他们所有的新镜头，包括这个。dock 连接到您的计算机，使用适马的专用软件，您可以使用 50 毫米 F1.4 在四个不同的焦距范围内调节特定的自动对焦性能。这意味着，如果您发现它在近距离时正面对焦，但在远距离时背面对焦，您可以调整两者。

我的镜头开箱后自动对焦非常准确，所以我不需要使用这个工具。但它远比佳能高端数码单反相机内置的微对焦调节更方便、更灵活。它允许你调整多个焦距，而不仅仅是整个镜头。

底线是适马已经从一个稳定可靠的预算选择转变为一个热门的制造者；50 毫米 F1.4 DG HSM Art 是其最新的成功，但 Global Vision 系列的其他产品，包括 35 毫米 F1.4 DG HSM Art、18-135 F1.8 DC HSM 和 120-300 毫米 f 2.8DG OS HSM Sport 同样令人印象深刻。当然，他们仍然会对钱包征税，但就性价比而言，适马目前拥有最好的游戏。
