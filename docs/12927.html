Essential Home 会比另一个愚蠢的智能家居中心更重要吗？TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/05/30/will-essential-home-be-more-than-another-dumb-smart-home-hub/

下一场智能手机大战与屏幕、电池或相机无关。其实根本不是真的硬件的问题。这是一场对互联家庭核心的激烈争夺。苹果、谷歌、亚马逊和三星都以不同的方式插上了旗帜，但没有一家真正有机会从头开始打造一款以这种连接为中心的手机。

这个机会似乎是安迪·鲁宾推出新款手机的驱动力。虽然今天早上的新闻感觉像是一个正式的公告，但这家初创公司显然已经将联网家庭作为其愿景的一个关键组成部分。Essential 手机将与它自己的智能家居中心一起推出，当然，它的名字叫做 Essential Home 。

事实上，作为智能家居中枢的想法似乎是推出这款手机及其配套中枢的驱动力。在的一篇帖子中，Essential 的工程主管 Manuel Roman 写了关于 Home 的操作系统的文章，内容丰富，但又令人沮丧。

Ambient OS 提供了一组服务和抽象，支持开发和执行在您的家庭环境中运行的应用程序。有了环境操作系统，你的家就是电脑。Ambient OS 知道您家的物理布局、住在里面的人、与您家和里面的人相关的服务以及设备。

今天早上，鲁宾的一份 Wired 简介也同样广泛地展示了该公司的互联雄心，写道 Essential 的计划是“将为即将上线的数十亿部手机、手表、灯泡和烤箱供电的开放平台。”

当鲁宾今晚在 Code conference 上登台时，他有望阐明这些词的确切含义，但该公司的操作系统以及其 Echo 竞争对手如何旨在超越现有竞争对手的关键似乎有两个。首先，拥抱开源。第二，培养一个能够随着成长而学习的助手。

开源是鲁宾开发 Android 的基石，由于(目前)没有谷歌这样的企业霸主的干预，该公司似乎在这方面更加投入。亚马逊和谷歌都在推出自己的智能中枢游戏后拥抱了开发者，但 Essential 似乎希望从一开始就提供这种开放将使其对智能家居公司更具吸引力——类似于三星对 Bixby 的承诺(但尚未兑现)。从各方面来看，Bixby 仍然是一项正在进行中的工作。

另一个尚未实现的 Bixby 承诺:情境学习。同样，随着时间的推移，这将如何发展还有待观察。这是一个几乎每个人都在承诺的功能，但尚未完全实现，这是一个学习并能够根据用户习惯定制自己的智能助手。

罗曼写道:“环境操作系统的关键是人们应该始终处于控制之中的信念。”“因此，它不会通过预测你的需求来让你的家变得智能。相反，随着它向人们学习，它可以建议某些行为，但最终人们会决定是否使用它们。”

就像目前几乎所有的必需品一样，这些想法感觉像是天上掉馅饼。有太多的因素在起作用。首先，即使是谷歌似乎也很难打入这个市场，因为家庭采用率滞后。亚马逊对这一领域的锁定一直难以打破，其设备的激增产生了级联效应。还有一个问题是，苹果今年为 WWDC 准备了什么。在一个已经拥挤不堪、即将变得更加拥挤的市场中，一家初创公司能产生多大的影响？

Essential 要求用户购买一家新公司的新手机和智能家居中心，并承诺它会随着时间的推移而发展和变得更好。即使拥有像鲁宾那样坚实的记录，这也是一个很高的要求。充分利用家庭产品还需要一个已经连接良好的家庭，而不是一个已经在使用的智能家庭中心(或者至少是愿意购买另一个具有类似功能的产品)。)

如果想要获得成功，Ambient/Essential 必须证明它比另一个能够回答基本问题和打开电灯开关的非回声机器人更像 T2。即使鲁宾今晚让观众惊叹，这种发展也需要耐心和时间——即使是最大的初创公司也往往没有这两样东西。
