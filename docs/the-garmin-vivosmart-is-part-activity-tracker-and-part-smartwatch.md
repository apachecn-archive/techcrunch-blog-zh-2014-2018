# Garmin Vívosmart 既是活动追踪器，又是智能手表

Garmin 首次涉足活动追踪器是一次坚实的尝试，但[仍有许多不足之处](https://web.archive.org/web/20221007030216/https://beta.techcrunch.com/2014/03/26/garmins-vivofit-has-some-neat-features-but-how-does-it-compare-to-other-wearables/)。vívofit 只是没有太多的功能让它与众不同。但是该公司最新的活动追踪器 vívosmart(比方说 *veevo* -smart)就是这么做的。

vívosmart 集美观、活动跟踪与智能手机提醒和控制于一身——可惜有点贵。

这款售价 169 美元的手环配有隐藏式触摸感应有机发光二极管显示屏。这款腕带当然会追踪佩戴者的活动，而且由于它是一款 Garmin，因此可以连接到该公司的其他产品，包括心脏监测器和活动摄像头。

vívosmart 还可以作为连接的智能手机的通知设备，为收到的文本、电子邮件、电话、日历提醒等进行振动。

这款最新的活动追踪器似乎是 Garmin 健身产品的坚实补充。老实说，Garmin 应该跳过令人失望的 vivofit，早点推出这款有竞争力的设备。Garmin 预计 vívosmart 将于 11 月初上市，零售价为 169 美元，如果捆绑心率传感器，则为 199 美元。
