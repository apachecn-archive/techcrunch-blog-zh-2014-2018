Box 与人工智能的亲密关系预示着内容管理领域更广泛的转变 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/10/12/boxs-dalliance-with-ai-foretells-a-broader-shift-in-content-management/?ncid=rss

正如 Box 首席执行官 Aaron Levie 在本周的主题演讲中指出的那样，内容管理自 20 世纪 90 年代作为企业软件概念出现以来，一直是一个不断发展的领域。

在那些日子里，最先进的是网络驱动器。随着内容在整个组织中传播，我们看到了企业内容管理的兴起。后来文件共享工具发展起来了，最后云出现了，并真正允许公司和个人在更现代的移动环境中共享组织内外的内容。

在这一演变过程中， Box 推出了，真正打破了我们所知的企业内容管理，这在很大程度上是为了保护防火墙内的内容。正如列维在 2011 年左右的一次采访中告诉我的那样，当智能手机以及后来的 iPad 出现时，它真的改变了一切。突然间，内容在移动，内容管理需要转向保护移动中的内容。

当时只有一家内容管理供应商真正有能力做到这一点，那就是 Box，一家诞生于云的公司，它对移动-云连接的理解远胜于那些措手不及、被迫追赶的传统同行。

本周在该公司的客户会议 BoxWorks 上，Box 宣布了内容管理的下一个根本性转变，使用人工智能和机器学习从内容中提取意义。

解决新内容问题

这不仅改变了管理内容的整个方法，而且解决了一个基本的内容管理难题—您拥有的内容越多，管理起来就越困难。通过应用在更多内容上蓬勃发展的机器学习，Box 正在解决内容量问题。

Levie 当然看到了今天的声明的重要性及其对行业发展的整体影响。“就改变云内容管理的方向并开始想象我们如何使用智能处理信息和数据而言，这确实是一个重要的时刻。在许多方面，我们今天展示的确实是这项技术将如何被使用的非常早期的阶段，”Levie 告诉 TechCrunch

就改变云内容管理的方向和开始想象我们如何使用智能处理信息和数据而言，这确实是一个重要的时刻。 Aaron Levie，Box 首席执行官

他意识到这是目前最前沿的技术，但他表示，如果这项技术足够重要，公司总是试图尽早进入。“我们一直发现的一个总体策略是，在你认为会持续很长时间的趋势中处于早期总是好的。处于无关紧要的趋势的早期并没有那么有用，但毫无疑问，机器学习和人工智能将对非结构化信息和工作流产生影响，这种用例的效用对我们来说是显而易见的，”他说。

与巨人同行

Levie 说，Box 决定对人工智能采取广泛开放的方法，部分原因是它太早了，部分原因是有一系列大型科技公司在做重要的工作，他不想让他的公司依附于任何一种方法。

“如果你要绘制谷歌、微软、IBM、亚马逊和苹果在人工智能和机器学习领域的创新量，你会看到创新量的指数图。当你有一个类别的创新发生得如此之快，而且它仍然处于行业的早期，你希望能够利用所有这些能力和所有这些进步。”

他说，他们也不希望客户被束之高阁，或者只从这些大供应商那里获得少量创新。“这就是构建允许跨多个智能平台的互操作性和无缝体验的架构对我们如此重要的原因。我们希望确保我们可以将所有这些先进技术引入平台，并将它们无缝集成到一个地方，”Levie 解释道。

虽然云降低了成本并增加了创新，但 Levie 表示，人工智能和机器学习可能会通过改变我们的流程产生更大的影响。“我认为，在许多情况下，云是降低存储成本、提高这些平台的可扩展性以及改善协作和内容管理用户体验的一种方式，但在许多方面，它并没有从根本上改变我们处理信息的方式。这就是我认为机器学习有潜力影响内容管理的地方，因为我们实际上开始从我们的数据中获得新的见解。”

Levie 强调今天的声明只是一个起点。他完全相信人工智能和机器学习将对行业产生根本性的影响，他希望他的公司现在学习如何最好地将这项技术应用到 Box 的工具集。他认为，当你早期进入这样的领域时，会产生飞轮效应，你可以从客户那里学习，并在前进的过程中改进技术。

“我们从第一天就看到了云，我们认为人工智能和机器学习也是如此，这就是为什么我们希望在这个领域进行如此重大的投资，”Levie 解释道。
