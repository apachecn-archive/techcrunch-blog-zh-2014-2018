Skype 推出了自己的表情符号版本

Skype 今天推出了一种方法，将简短的电影片段引入其聊天服务，在通信平台内添加了类似于 GIF 的支持。

Skype 的新功能被称为“Mojis ”( T1 ),用该公司自己的话说，是“你最喜欢的电影和电视节目的短片”,可以在通过文本与该服务的其他用户交谈时使用。两名 TechCrunch 员工更新并重新下载了 Skype 来测试这项服务。两个建筑都没有包括它。

据微软称，Mojis 将包括《侏罗纪公园》和《完美沥青》等电影的样本。再说一次，作为一个有钱的在职者有一定的优势。

长期以来广受欢迎的消费产品 Skype 的优势已经被 WhatsApp 等面向普通用户的服务和 Slack for business 削弱；微软最近将其 Lync 企业产品更名为 Skype for Business，凸显了后者对这家总部位于雷德蒙德的软件商店的重要性。

没有能力给你带来一个现场演示，我可以说，微软创造了一个莫吉工具在行动的短视频。Mojis 可以让你在聊天过程中播放视频剪辑。Slack 是一个发展中的协作服务，以支持动画 gif 而闻名。这也许是 Skype 对这种攻击的回应。

一个不同的想法:微软以至少 19 美元的公司政变收购 Giphy，禁止其 API 中的 Slack，将初创公司烘焙成 Skype，并接管世界。

也许我看了太多的毒品。
