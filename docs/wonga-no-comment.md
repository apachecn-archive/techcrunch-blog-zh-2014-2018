# Wonga 投资者仍然不会对受到指责的发薪日贷款公司 TechCrunch 发表评论

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/10/21/wonga-no-comment/>

本月早些时候，发薪日贷款公司 Wonga 进行了 2 . 2 亿欧元的资产减记后，一个由欧洲顶级投资者组成的小组公开拒绝对该公司的问题发表评论。

这些投资者在伦敦参加了 TechCrunch Disrupt Europe 关于欧洲投资生态系统的讨论。两名小组成员来自直接投资 Wonga 的风险投资公司:Accel Partners 和 Balderton Capital。

这是这些公司[第二次在被问及关于 Wonga](https://web.archive.org/web/20221204211730/https://beta.techcrunch.com/2014/10/03/investors-stay-silent-after-wonga-fiasco/) 的问题时保持沉默——但他们的“无可奉告”第一次在舞台上被引出。

这里的背景是[本月早些时候](https://web.archive.org/web/20221204211730/http://www.fca.org.uk/news/wonga-major-changes-to-affordability-criteria) Wonga 同意注销 33 万客户的贷款，并免除另外 4.5 万客户的利息和费用。它这样做是因为承认其自动检查未能充分评估可负担性。自动检查是核心算法知识产权，最初让该业务吸引了一系列高调的技术投资者。说白了，就是借钱给那些没有收入来偿还贷款的人。

Wonga 的业务还因向欠款客户发送虚假律师函而受到金融行为监管局(FCA)的指责。它被要求为此再支付 260 万英镑的赔偿。

在与 TechCrunch 的 Ryan Lawler 的小组讨论中，Accel Partners 的 Philippe Botteri 拒绝就 Wonga 的问题发表评论。

Balderton Capital 的丹尼尔·沃特豪斯(Daniel Waterhouse)也没有直接回答劳勒的问题——包括投资该公司是不是一个坏主意，或者这个想法的执行是否有缺陷。相反，Waterhouse 对 Wonga 现在正在做的事情做了一个一般性的陈述，因为 FCA 已经介入。

“看，我认为 Wonga 已经很清楚，他们在业务的位置上犯了很多错误，”Waterhouse 说。“他们有一大笔贷款，他们说他们正在与 FCA 密切合作，向市场提供一种很好的产品。他们非常清楚过去发生了什么，他们现在在做什么，未来在做什么。”

Index Ventures 的索尔·克莱恩(Saul Klein)也是董事会成员之一，尽管 Index 本身并不是 Wonga 的投资者，但 Index 合伙人罗宾·克莱恩(Robin Klein)——索尔的父亲——曾担任 QuickBridge 董事会主席，直到 2013 年 11 月卸任。

“指数没有投资，”克莱恩说，接着点名到底是谁投资了发薪日贷款业务。“Accel 做了，Balderton 做了，Dawn Capital 做了，Oak 做了，Meritech 做了，但 Index 没有……让我们搞清楚到底是谁投资了什么。”

克莱恩没有提到罗宾克莱恩是 Wonga 董事会主席。

当被问及是否对问题所在有所了解时，克莱恩说:“我不知道。我不在船上，我不是投资者，我也不是投资者。”

“我不确定你说的哪里出了问题是什么意思，”他补充道。

激情资本的 Eileen Burbidge 也是小组成员——后来[在推特上发布了](https://web.archive.org/web/20221204211730/https://twitter.com/eileentso/status/524509321284304896/photo/1)关于劳勒的问题引起的反应。

她在讨论中表示:“我确信(投资者)没有进去，公司创始人也没有进去，他们说我们将试图利用这里的形势，并试图冒着社会或其他风险尽可能多地获取资金。”。

“我敢肯定，这一切都是为了提高本来就不完美的系统的效率。这可能会超过教育和监管，我认为他们正在尽最大努力尝试和纠正。这是我的观察。”

Passion 和 Burbidge 与 Wonga 没有直接投资或董事会关系。

该小组更乐于谈论的话题包括欧洲生态系统的规模和轨迹；与硅谷的差异；该地区有前途的技术；鉴于同一家公司的高联合投资率，欧洲的投资界是否是乱伦的；以及他们各自感兴趣的投资类型。

Burbidge 认为信息安全是她特别感兴趣的领域——尤其是它与人工智能和自然语言处理相结合的领域。

沃特豪斯指出，设计是欧洲的一个关键领域。“欧洲有趣的地方在于，我们融合了人才和设计技能——尤其是在斯堪的纳维亚和伦敦。它们融合在一起，创造出一些非常有趣的产品，”他在评论那些吸引他眼球的初创公司时指出。

博特里说，Accel 正在继续关注他所说的“基本趋势”，如云计算、大数据、云基础设施。他表示:“我们仍然看到电子商务实现的大量机会。”。

Botteri 感兴趣的第二个领域是新兴计算平台轨道上的初创公司，他说这将在未来 5 到 10 年内成熟。“我们正在看到新的计算平台的出现，”他指出，并以即将推出的苹果手表为例。“你戴在手腕上的将是一种新的计算设备，但我认为计算设备正在发展，对吧。它们可以在你的手腕上，可以在你的口袋里，现在你有了无人机，它可以在空中，你有了机器人。

“所有这些硬件驱动的计算平台……我认为这将推动大量创新。我们刚刚看到这种情况的出现。”

Index 的克莱恩指出，人工智能是 Index 感兴趣的投资领域。“我们看到谷歌去年收购了 Deep Mind，我们对一家名为 SwiftKey 的(智能键盘)公司进行了非常有趣的投资……如果你喜欢人工智能和来自欧洲的 deep tech，这是一个根本性的区别，”他说。

“当你看到人工智能和自然语言将对我们体验和浏览信息的方式产生多大的影响时，我对欧洲存在的一些深层技术人才感到非常非常兴奋，”克莱恩补充道。“这非常非常有趣，当我们开始看到这样的业务…真正使用数据和智能来帮助我们以新的方式处理信息。对我来说，这确实是过去最有价值的地方。”

在小组讨论的早些时候，克莱恩还提出了一个关于早期投资的观点，讨论了投资者需要非常关注他们所投资的初创公司。

“投资者需要把手弄脏，不管他们是不是经营者，这实际上是我们所说的 A 系列肌肉，”他说。“这是关于真正弄脏你的手，参与团队，容易接近，成为一种资源，拥有一个网络——这使他们有希望犯错误，因为我们在职业生涯中都犯过很多错误，希望我们可以传递好处。”
