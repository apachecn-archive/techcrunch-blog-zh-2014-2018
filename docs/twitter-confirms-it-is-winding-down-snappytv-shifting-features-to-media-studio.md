# Twitter 证实正在关闭 SnappyTV，将功能转移到 Media Studio 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/07/25/twitter-confirms-it-is-winding-down-snappytv-shifting-features-to-media-studio/>

一段时间以来，Twitter 一直在努力精简和简化其界面，使其社交媒体平台更容易被新用户和更随意的用户访问，它在服务后端也在做同样的事情。在最新的发展中，我们已经证实 Twitter 正准备关闭 SnappyTV，这是一项 it [在 2014 年](https://web.archive.org/web/20230403225650/https://techcrunch.com/2014/06/19/twitter-acquires-snappytv-to-expand-offerings-for-live-media-partners/)收购的服务，用于在 Twitter 和其他社交平台上快速剪辑、编辑和分享直播视频剪辑。

取而代之的是，Twitter 计划将 SnappyTV 的几个功能引入[媒体工作室](https://web.archive.org/web/20230403225650/https://business.twitter.com/en/help/troubleshooting/media-studio-faqs.html)，这是一个供企业和高级用户上传、安排和管理媒体推文的仪表板，看起来像是大约一年前推出的。

今天早些时候开始出现关于该公司计划终止 SnappyTV 的报道。巧合的是，我们的一个社交团队也在上周向 Twitter 询问了一些关于视频功能的问题，并在今天早上早些时候收到了 SnappyTV 详细信息的回复。

“我不知道你是否听说了，但我们正在关闭 SnappyTV，并将关键功能转移到媒体工作室，这样就有了一个创建、剪切、分享和货币化视频的目的地，”这位 Twitter 员工说(因为她不是发言人，所以我们不透露她的名字)。她补充说，很快会有更多的细节分享。Twitter 的发言人拒绝置评。

除了作为 SnappyTV 服务核心的关键“实时剪辑”功能，Twitter 还计划为 Media Studio 添加其他功能，这些功能既不是 Twitter Video 的一部分，也不是 SnappyTV 的一部分，例如可以包含 SRT 文件(视频字幕文件)。

不太清楚的是 Twitter 可能会改变什么。目前，SnappyTV 和 Media Studio 都是免费使用的，还不清楚 Twitter 是否计划在未来对其收费。同样，Twitter 是否会让用户继续创建可以在 Twitter 之外的其他地方发布的剪辑也不得而知。

正如最先报道 SnappyTV 将被关闭的谣言的 Digiday 指出的那样，这项服务的一个关键好处是它免费且使用快捷。创建视频剪辑的其他变通方法成本更高，或占用更多时间，或两者兼而有之。

这不是 Twitter 最近第一次削减产品。该公司还承诺削减一些广告产品，将 [Fabric 移交给谷歌](https://web.archive.org/web/20230403225650/https://techcrunch.com/2017/01/18/google-twitter-fabric/)，减少对[“购买”按钮](https://web.archive.org/web/20230403225650/https://techcrunch.com/2017/01/17/bye-buy-on-twitter/)的使用，降低其商业目标，关闭了 [lead generation 产品](https://web.archive.org/web/20230403225650/http://marketingland.com/twitter-eliminating-ads-requested-peoples-personal-information-200617)，在视频领域，当然还有 [Vine](https://web.archive.org/web/20230403225650/https://techcrunch.com/2016/12/16/vine-wont-be-removed-from-the-app-stores-will-instead-relaunch-as-vine-camera/) 的减少。

但是媒体——尤其是视频，更具体地说，[直播视频](https://web.archive.org/web/20230403225650/http://www.recode.net/2017/5/22/15676722/twitter-live-video-executive-todd-swidler) —已经被强调为 Twitter 希望发展的一个关键领域，因为它希望为其业务带来更多增长。视频不仅承载着更有价值的广告，而且它是一种被证明能够推动观众更多参与的媒体，这是 Twitter 希望看到增长的一个指标，以帮助提高网站的整体使用指标。

两天后，Twitter 将于 27 日发布季度业绩报告，届时可能会有更多的问题和讨论，涉及该公司在努力将自己定位为媒体播放器方面的进展。随着我们了解更多，我们将同时更新这篇文章。
