三星凭借Gear VR耳机TechCrunch步入虚拟现实世界

原文：https://web.archive.org/web/https://techcrunch.com/2014/09/03/samsung-gear-vr-galaxy-note-4/amp/

三星正在大胆地进入虚拟现实世界，但不是没有一款值得信赖的智能手机。该公司今天推出了三星Gear VR，这是一款插入Galaxy Note 4的耳机，可以创建与Oculus Rift类似的虚拟现实体验。

Gear VR本身充满了各种传感器，而Galaxy Note 4的处理器和电池处理所有繁重的工作。移除遮阳板后，用户可以将他们的Galaxy Note 4插入设备，2k显示屏将接管。

三星正在利用Oculus为该设备提供动力，该设备使用Oculus的VR商店为用户提供内容访问。Gear VR还将能够让用户观看音乐会，360度游览城市和地点等。

由于Gear VR分割了Galaxy Note 4的518ppi显示屏的分辨率，因此体验不是超高清的。然而，在我们不得不试用这款耳机的短暂时间内，它无疑为那些寻求更轻便选择的人提供了一种有趣的虚拟现实体验。

Oculus和三星在这方面共同努力，这使Oculus通过Oculus store在市场上进一步分销，并使三星获得一些信誉和既定内容，首次进入一个全新的市场。

在发布会上，Gear VR将会有一些特殊的内容，包括参观托尼·斯塔克的实验室(作为与漫威的《复仇者联盟:奥创时代》合作的一部分)。IMAX还提供预告片和电影样本，供在虚拟影院观看，以及为太阳马戏团(Cirque Du Soleil)和Zarkana等演出提供360度真人3D虚拟现实。

Vevo还提供超过100，000个高清音乐视频、现场音乐会等。
