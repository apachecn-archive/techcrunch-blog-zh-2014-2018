# WaitChatter 帮助您在等待即时消息回复时学习一门新语言

尽管在线交流的速度很快，但我们仍然要花很多时间等待:盯着即时消息窗口对于那些一天中大部分时间都呆在电脑前的人来说是很平常的事情。 [WaitChatter](https://web.archive.org/web/20221209140253/http://people.csail.mit.edu/ccai/waitchatter/) 希望利用这段待机时间推广一项重要技能，让用户在等待同事或朋友回复他们的谷歌聊天信息时，温习第二语言的词汇。

该应用程序来自麻省理工学院计算机科学和人工智能实验室的研究，该研究发现，平均每个人每天在通过即时消息交谈时会浪费 10 到 15 分钟的等待时间。这种认识导致了 WaitChatter 的开发，它使用 Google Chat(你现在必须从 Hangouts 返回到使用它，但这相当容易)扩展，在你的 IM 聊天窗口中提供快速词汇学习课程。

它不仅基于利用那些未被占用的时间，还基于人们倾向于认为专门的语言学习应用程序需要额外的时间投入，即使是为了方便和移动而设计的。

该应用的用户参与了一项试点测试，每天以大约四个的速度学习新单词，利用来自内置列表和用户自己的 IM 对话的单词。虽然这不完全是掌握一门语言并变得非常流利的方法，但它确实是那种便于积累需要大量记忆的小而有用的技能的方法。

WaitChatter 背后的研究团队正在寻求利用其他等待情况，例如当你让你的电子邮件信息加载或站在出租车排队时。至于 WaitChatter 本身，他们说它可以与 Google Translate 目前可以处理的任何字母语言一起工作，也应该能够移植到大多数其他流行的 IM 应用程序，包括 Skype 和 Facebook Messenger。
