# Waze 拼车瞄准更多美国城市和拉丁美洲进行扩张

Waze 的拼车共享功能可能很快就会出现在你附近的城市。谷歌旗下的众包导航应用首先在其祖国以色列试行拼车功能，然后在旧金山和湾区试行——现在它正寻求进一步推广，包括在更多的美国城市和拉丁美洲市场。

这是 Waze 的老板诺姆·巴丁在周三接受《华尔街日报》采访时透露的。此次扩张将大大加快 Waze 拼车雄心的步伐，迄今为止，该计划进展非常缓慢，并密切关注在有限的市场中逐步推出，慢慢增加新的领域，鼓励雇主和交通合作伙伴使用该服务。

Waze 拼车与包括 uberPOOL 在内的服务截然不同，因为谷歌拥有的服务并不寻求成为参与其中的司机有意义的创收机会。乘客为加入司机的通勤支付少量费用，但这主要是为了帮助司机分摊汽油费用，并可能在此过程中支付一部分车辆维护费用。

其结果是，乘客支付的费用比其他方式少得多——仅比公共交通费用高一点点。但这种权衡包括你能找到一辆车的保证更少，以及需要相对提前预订以确保你被覆盖。Waze 也纯粹是一个连接机制，这意味着它不会筛选司机——但它也不会从他们的收入中抽取任何东西。就目前而言，是这样的；巴丁告诉《华尔街日报》，如果这项服务流行起来，它可能会开始向乘客收取 15%的额外费用，并从中提成。

Waze 的拼车服务可能比优步等竞争对手的服务更有吸引力，这是因为一个重要原因:如果它真的起飞了，并继续专注于不是激励专业人士，而是连接现有的通勤者，它可以在减少道路上和停车场闲置的汽车数量方面做得更多，无论是在短期还是未来。专业的拼车服务，至少在短期内，可能会导致道路上的汽车数量激增，因为司机可以安全地驾驶车辆，特别是为了能够在平台上工作。

巴丁在采访中说，Waze Carpool 可能会将圣保罗作为一个扩张目标，但其他具体的增长市场尚不清楚。
