# 谷歌收购 Timeful，为谷歌应用程序带来智能调度

谷歌[收购了 Timeful](https://web.archive.org/web/20221209171552/http://gmailblog.blogspot.com/2015/05/time-is-on-your-sidewelcoming-timeful.html) ，这是一家初创公司，旨在通过帮助你解决更大的目标而无需你操心细节，从而使日程安排变得更容易。Timeful 的应用程序会分析你的时间表，并使用它来帮助你实现你心目中的特定目标。Timeful 创建的功能将被集成到谷歌应用程序中，该应用程序将保持可用，但 Timeful 团队正在将注意力转向谷歌的项目，因此继续支持不太可能。

Timeful 在 KPCB、Data Collective、Khosla Ventures 和阿什顿·库彻 A-Grade Ventures 的资助下，在 T2 悄然起步，并于去年 3 月在 7 月发布了一款公共应用。该应用程序通过对你的各种活动和时间需求进行优先排序来工作，并与所有主要平台的现有日历软件配合使用，如 iCal、Microsoft Exchange 或 Google Calendar。它利用你现有的时间表，添加你需要完成的事情，权衡这些事情有多重要，并为你找到时间来完成它们。

Timeful 的核心技术涉及机器学习，旨在消除结合调度和 GTD 任务管理的麻烦。在许多方面，它对日历和日程表的作用就像收件箱对电子邮件的作用一样。因此，它作为谷歌的收购目标是有意义的，因为它寻求提高应用程序的能力，以应对在线通信和时间管理的转变。

谷歌 Gmail 产品管理总监亚历克斯·高利(Alex Gawley)在宣布这一消息的博客文章中，就谷歌对 Timeful 的意图提供了以下信息:

> Timeful 团队建立了一个令人印象深刻的系统，通过了解您的时间表、习惯和需求来帮助您组织您的生活。你可以告诉 Timeful 你想每周锻炼三次，或者你需要在下周二之前打电话给银行，他们的系统将确保你在了解你的时间表和优先事项的基础上完成这项工作。我们对 Timeful 的技术可以应用于收件箱、日历等产品的所有方式感到兴奋，因此我们可以为您做更多的工作，让您专注于创意、娱乐和与您关心的人共度时光。
