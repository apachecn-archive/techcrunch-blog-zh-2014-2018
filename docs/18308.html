Zebra Medical Vision 获得 3000 万美元 C 轮投资，为放射学家创建基于人工智能的工具 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/06/07/zebra-medical-vision-gets-30m-series-c-to-create-ai-based-tools-for-radiologists/

以色列医疗成像初创公司 Zebra Medical Vision 利用机器和深度学习为放射科医生构建工具，已经筹集了 3000 万美元的 C 轮融资，由健康技术基金 aMoon Ventures 牵头，Aurum，Johnson&Johnson Innovation-JJDC 公司(该集团的风险投资部门)、山间健康和人工智能专家费-李非和理查德·索彻参与。现有投资者 Khosla Ventures、Nvidia、Marc Benioff、OurCrowd 和 Dolby Ventures 也回到了这轮融资。

斑马公司今天还宣布了它的 Textray 研究，它声称这是“迄今为止对胸部 x 光进行的最全面的人工智能研究。”Textray 正被用于开发一种新产品，该产品已经使用近 200 万张图像进行了训练，以识别 40 种临床发现。计划于明年发射，它将帮助放射科医生自动分析胸部 x 光片。

创始人兼首席执行官 Elad Benjamin 在 2014 年与 Eyal Toledano 和 Eyal Gura 一起推出了 Zebra，他在一封电子邮件中告诉 TechCrunch，其 C 轮资本将用于新员工，以加快其分析引擎和上市战略的发展。

胸部 x 光是最常见的射线照片之一，但也是放射科医生最难解读的照片之一。有几个研究小组专注于使用人工智能算法使过程更加准确和有效，包括来自谷歌、斯坦福和迪拜卫生局的团队。

Benjamin 说，Zebra 现在已经筹集了 5000 万美元，它通过从“整体产品角度”来看待它的算法，使它的方法与众不同。开发算法只是其中的一部分，”他补充道。“将它集成到工作流中，使它适应多个国家和医疗保健环境的需求，支持和更新它，并在全球范围内对它进行适当的监管，这些都需要极大的关注，而这正是算法所能带来的价值。”

他补充说，人工智能在医学成像和诊断领域成为主流“还需要几年时间”，但他相信它最终将成为放射学的一个关键组成部分。Zebra Medical Vision 已经推出了一套用于肺部、乳腺、肝脏、心血管和骨骼疾病诊断的算法，名为 AI1 ，医院每次扫描的费用为 1 美元。

在一份新闻声明中，aMoon 管理合伙人 Yair Schindel 博士表示:“我们很高兴与 Zebra 团队合作，该团队正在利用数据和机器学习的力量为医生和医疗保健系统提供工具，以大幅提高能力，同时改善患者护理。这项投资符合我们支持可扩展和可持续创新的愿景，这些创新将对全球医疗保健的基本方面产生有价值的影响。”
