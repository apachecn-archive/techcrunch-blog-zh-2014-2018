小米宣布在中国推出令人印象深刻的200美元智能手机和预付费移动服务

小米发布了其最新智能手机Mi 4c，这是其旗舰设备Mi 4的200美元预算版本。这还不是全部:这家中国公司在为中国客户推出两种预付费套餐后，还涉足了移动运营商服务。

与小米4月份专门为印度推出的智能手机Mi 4i一样，Mi 4c也是针对中国客户的。这非常令人印象深刻…至少在理论上是这样。

小米4i配备5英寸显示屏，采用64位处理器，小米声称这是第一款配备高通骁龙808的设备。还有一个1300万像素的后置摄像头和一个500万像素的“自拍”摄像头，因为前置摄像头越来越多地被提及。同样值得注意的是，Mi 4i包括新的USB Type-C和“快速充电2.0”，以帮助管理电池寿命。

基本款Mi 4c配有2GB内存和16GB存储空间，售价为1299元人民币(203美元)，增强版——3GB内存和32GB存储空间——售价为1499元人民币(235美元)。这两款机型都将搭载小米较旧的MI UI 6软件，但该公司表示，其新的MI UI版本7 的升级版本将很快推出。

小米以极具竞争力的价格发布令人印象深刻的设备而闻名——小米4c是另一个极具竞争力的产品，应该会担心三星和其他公司——但移动运营商服务完全是新事物。该公司是在中国获得MVNO(移动虚拟网络)牌照的一批非电信公司之一。这使得他们可以使用自己的品牌/名称转售电信公司的移动套餐，现在小米正在中国应用这一点。

小米的移动产品“Mi Mobile”有两个初始服务。有一种现收现付的资费，每分钟语音通话、短信或1MB数据收费0.10元人民币(不到0.02美元)，还有一种每月59元人民币(10美元)的套餐，给用户提供3GB的数据。第一个选项将于9月23日开始发售，第二个选项将于10月上线——这两个选项都将在小米的中国在线商店Mi.com发售。

移动服务风险投资是小米的一项有趣的新业务。该公司是中国最大的智能手机销售商，中国占了小米今年上半年售出的3470万部智能手机的大部分，但小米表示，它将达不到2015年的销售目标。因此，在推动销售可穿戴设备和智能家居设备的同时，进军移动服务是加强其在客户中的品牌并获得额外收入的一种方式。
