视频广告公司 Teads 收购 brain ient TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/09/12/teads-acquires-brainient/

视频广告公司 Teads 上个月通过债务融资筹集了 4700 万美元，以继续对其他广告科技初创公司的收购狂潮，它信守了自己的诺言。今天，它宣布收购总部位于英国和罗马尼亚的 Brainient ，这是一家由 Emi Gal 创立的互动视频广告初创公司，自从我报道以来，Emi Gal 几乎一直是伦敦科技界的常客。

成立于 2009 年的 Brainient 及其聪明的罗马尼亚工程团队突发奇想，通过使在线视频广告更加个性化和互动，来提高在线视频广告的参与度和表现。

前者指的是智能广告可以利用任何数据源，让广告符合用户的个人资料和偏好——包括社会演示、地理位置、使用的屏幕、一天中的时间、天气(或基本上任何其他类型的上下文数据)——而后者包括让视频广告“可购物”或提交视频表单以请求关于产品的进一步信息的能力。

Brainient studio 是一套工具，负责制作这种互动和数据驱动的视频广告，包括确保它们可以轻松地重新配置为跨格式和设备工作，并与不同的视频广告网络和出版商属性兼容。这正是 Teads 收购 Brainient 发挥作用的地方。

Teads 的首席执行官 Bertrand Quesada 在电话中告诉我，该计划是为了能够向公司的现有客户免费提供新命名的 Teads 工作室，以提高 Teads 视频广告主张的投资回报率。

这是因为这家欧洲公司专门从事所谓的“ outstream ”广告:位于视频内容之外的视频广告，比如放在两段文字之间的文章中。

这些广告已经有可能比传统的视频前置广告表现得更好，通过使广告本身融入数据和互动，这一想法不仅使 Teads 的客户能够围绕视频广告的位置和交付方式进行创新，而且还能在创意本身中进行创新。Brainient 一直声称其互动视频广告的表现优于标准广告。

在与 Brainient 的 Gal 的电话中，他听起来对退出持乐观态度，尽管他和 Quesada 都拒绝透露交易的条款。Brainient 总共筹集了大约 400 万美元的资金，包括 2012 年由总部位于布拉格的 Credo Ventures 牵头的180 万美元的首轮融资，参与方包括 Atlas Venture、Estag Capital、Sherry Coutu 和 Dave McClure 的 500 Startups。

一位接近 Brainient 投资者的消息人士也告诉我，他们对这家初创公司的退出所产生的回报非常满意，如果 Gal 继续创办另一家公司，他们会毫不犹豫地再次支持他。另外，另一个消息来源告诉我，这家初创公司之前已经接受了约 1000 万至 1200 万美元的报价，所以你就尽情享受吧。我还明白，尽管 Gal 和 Brainient 的另一位联合创始人兼首席技术官安德烈·巴拉甘(Andrei Baragan)是这家初创公司的最大股东，但他们不再持有多数股权。

与此同时，Gal 告诉我，他将搬到纽约市，帮助 Teads Studio 在美国市场销售更多产品(他说他早在 2012 年就迫切需要做这件事！)而且，一旦 Brainient 觉得自己已经成为欧洲的领导者，这一直是它的计划。他还说，借助 Teads 的资源，Brainient 计划将其在罗马尼亚的工程团队规模扩大一倍，并继续迭代产品。
