RetroPie 3 让你在新的 Pi 上玩旧游戏

RetroPie 项目一直是树莓派最棒的事情之一。这个软件套件将你的微型单板电脑变成一个复古游戏引擎，并为诸如 CupCade 这样令人兴奋的项目提供动力。RetroPie 包括从 Atari 2600 到 Game Boy Advance 的所有必需的模拟器。

新版本 3.0可在此下载它具有许多改进，包括使用手机使用虚拟游戏手柄，以及“许多可从仿真站访问的配置工具，包括设置 WiFi、配置编辑器、文件管理器、音频设置等。”他们还将超级马里奥战争作为默认游戏。

简而言之，RetroPie 项目变得更酷了，这是一个让你熟悉 Pi 的好方法。如果你在 RetroPie 上玩 Galaxian 游戏时闭上眼睛，把头倾斜到合适的位置，这种味道永远无法战胜当地街机/切片店的披萨和油腻硬币的味道，但你仍然可以听到你五年级的朋友抱怨说他们一直在杆位上撞车。
