# 优步教在特拉维夫被部分禁止

路透社报道，在交通公司、交通部和出租车司机工会之间的案件中，法院做出裁决后，特拉维夫地区法院禁止了优步在该市的两项服务。

法院特别禁止了该公司在特拉维夫的私家车服务 UberDay 和 UberNight。UberTaxi 使乘客能够通过有执照的出租车司机请求乘车，它不受影响。

在法庭上，代表该部和出租车司机工会的律师提出了对选择 UberDay 或 UberNight 的乘客的保险覆盖范围的担忧。这一裁决是在[交通部五月对优步发出起诉书后做出的，](https://web.archive.org/web/20230129221737/https://www.bloomberg.com/news/articles/2017-05-23/israel-indicts-uber-for-allegedly-offering-illegal-rides-j31ra8gg)指控优步在没有政府许可的情况下运营。

UberX 是一种让普通人付费驾驶自己的汽车并搭载乘客的服务，在特拉维夫已经被视为非法。这就是为什么优步推出了试点服务 UberNight 和 UberDay，要求乘客向私人司机支付费用报销费，包括燃料费。

“虽然我们正在结束我们的 UberNIGHT 和 UberDAY 试点，但我们将继续在以色列提供 UberTAXI，”优步发言人告诉 TechCrunch。“我们致力于与以色列当局合作，探索技术如何通过安全和负担得起的交通替代方案来改善我们的城市。”
