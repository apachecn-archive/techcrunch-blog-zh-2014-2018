GitHub 开源了其 Atom 文本编辑器

今年早些时候， GitHub 发布了一个的私有测试版，它是一个易于扩展的 Atom 文本编辑器。当时，它开源了编辑器的 80 个库和包，但编辑器本身仍然是闭源的。今天，经过 10 周的公开测试，它在麻省理工学院开源许可下提供了所有的编辑器，包括所有允许它支持不同编程语言的包和库。这包括 Atom 的包管理器，用于安装编辑器的第三方扩展，以及 Atom Shell，一个基于 Chromium 的桌面应用程序框架。

该公司告诉我们，该编辑器自 10 周前公开推出以来，已被下载超过 25 万次。从那时起，该公司已经发布了超过 40 个更新，社区已经发布了超过 800 个包来扩展 Atom 的核心特性。要扩展 Atom，开发者只需要懂 JavaScript 和 CSS 就可以了。

GitHub 表示，它将继续开发该应用程序，并为此成立了一个专门的团队，但它希望“开源社区的创造力、支持和热情”将使它变得更好，并允许团队更快地开发编辑器。

该公司仍然认为 Atom 是 1.0 之前的产品。目前，该团队将专注于发布在 Linux 和 Windows 上运行的版本(目前仅适用于 OS X)，以及稳定 API。
