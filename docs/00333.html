GitHub开源了其Atom文本编辑器

今年早些时候， GitHub 发布了一个的私有测试版，它是一个易于扩展的 Atom文本编辑器。当时，它开源了编辑器的80个库和包，但编辑器本身仍然是闭源的。今天，经过10周的公开测试，它在麻省理工学院开源许可下提供了所有的编辑器，包括所有允许它支持不同编程语言的包和库。这包括Atom的包管理器，用于安装编辑器的第三方扩展，以及Atom Shell，一个基于Chromium的桌面应用程序框架。

该公司告诉我们，该编辑器自10周前公开推出以来，已被下载超过25万次。从那时起，该公司已经发布了超过40个更新，社区已经发布了超过800个包来扩展Atom的核心特性。要扩展Atom，开发者只需要懂JavaScript和CSS就可以了。

GitHub表示，它将继续开发该应用程序，并为此成立了一个专门的团队，但它希望“开源社区的创造力、支持和热情”将使它变得更好，并允许团队更快地开发编辑器。

该公司仍然认为Atom是1.0之前的产品。目前，该团队将专注于发布在Linux和Windows上运行的版本(目前仅适用于OS X)，以及稳定API。
