脸书招募开发者用个人表情包超越 Snapchat 的过滤器

脸书每天会看到 3000 万条个人资料更新，但它无法构建每一个工具来保持你的照片和视频的趣味性。所以今天在全球向所有开发者推出它的个人资料表情包。这使得您可以轻松地将 Vine 视频、回旋镖 gif、美颜+增强自拍、MSQRD 面部滤镜、Lollicam 贴纸和 Flixel cinemagraphs(如上所示)等媒体添加为您的个人资料照片。

脸书在四月份的 F8 会议上与这六个合作伙伴一起发布了个人资料表情包。但是现在任何应用程序都可以申请添加 Profile Expression Kit SDK ，让你将他们的一个创作转化为你的个人资料图片或视频。作为交换，该应用程序将获得一个大声喊出来的链接，并在宣布你的新个人资料图像的新闻订阅帖子中下载，这是脸书最受欢迎的故事之一。

事实上，脸书的产品经理 Aigerim Shorman 告诉我“在脸书，个人资料是仅次于新闻订阅的第二大访问量平台。”这意味着使用 Profile Expression Kit 可以让应用程序获得大量曝光，考虑到应用程序商店的拥挤程度和应用程序安装广告的昂贵程度，这一点至关重要。

发布合作伙伴 Flixel 的首席营销官 Robert Lendvai 告诉我,“我们知道受邀加入 Profile Expression Kit 测试版对 Flixel 来说是一个巨大的机会，脸书没有让我们失望。Flixel 是唯一一家在 F8 登台亮相的初创公司，随后几天的媒体报道帮助我们完成了一轮融资，包括来自 500 家初创公司的投资。脸书已经帮助 Flixel 大规模地将电影放到了地图上。”

正如我在我的专题文章“脸书的下一个大平台:你的相机”中所讨论的，脸书经常依赖开发者提供它自己无法编码的更多定制体验。最初的应用平台让脸书充满了各种游戏，而 Open Graph 让你炫耀你的 Spotify 听力或你读过的书。

脸书不需要付钱给开发商来建造这些东西。相反，它只是悬着它最大的资源——用户。开发者渴望获得牵引力，如果脸书能让他们下载，他们会让它变得更有活力。

尽管 Snapchat 可能拥有最好的原生表情过滤器和镜头，但脸书现在正以其平台进行反击。通过众包有趣的方式来展示你的脸，它可以用比任何一家公司都多的创意工具来吸引青少年。
