# 优步董事会投票一致通过了霍尔德报告的所有建议

优步董事会一致投票决定实施前美国司法部长埃里克·霍尔德和他的法律合伙人塔米·阿尔巴兰委托撰写的报告中的所有建议。董事会周日开会讨论了这些建议，据报道，其中包括一项建议，即首席执行官特拉维斯·卡兰尼克的亲信优步·SVP·埃米尔·迈克尔被要求离开公司。

理事会的一名代表在发言中提出以下意见:

> 优步董事会今天会见了 Eric Holder 和 Tammy Albarrán。董事会一致投票通过了 Holder 报告中的所有建议。这些建议将于周二发布给员工。

自报告接近尾声以来，优步一直在泄露信息，上周出现了一些故事，包括一名优步高级管理人员在一名优步司机犯罪后获得了一名强奸受害者的医疗记录。

另一份报告称，在最近这些事情曝光后，卡兰尼克正在考虑至少暂时离开公司。据报道，董事会[在周日](https://web.archive.org/web/20221025232335/https://www.nytimes.com/2017/06/11/technology/uber-holder-report.html?mcubz=0)的会议上考虑了这一选项，以及霍尔德报告中的建议。

该公司将在周二了解将被采纳的建议的具体内容，很可能届时这些内容也会被公之于众，要么通过官方渠道，要么通过迄今为止一直在提供有关该报告的大部分信息的泄露网络。
