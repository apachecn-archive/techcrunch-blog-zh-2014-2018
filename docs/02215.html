德国众包软件测试公司增加云产品

Testbirds 今天推出了一款新的云测试产品，他们称之为测试变色龙。

到目前为止，这家德国公司一直以提供众包软件测试而闻名。他们有一个由 50，000 名测试人员组成的团队，他们将测试你的移动应用和软件，并向你报告测试结果。每个测试通常涉及 25-50 名测试人员，他们关注一个特定的问题，但是公司开始收到客户的来信，他们希望能够在更丰富的可能场景中进行测试。

该公司创造了 TestChameleon，这是一种自动化的测试方法，与基于人类的人群测试方法形成对比。使用 TestChameleon，您可以测试许多变量，包括浏览器、操作系统、Flash 和 Java 的类型和版本，以及其他一些变量。

客户可以选择在企业数据中心内使用公共云或私有云版本的服务。

为了测试桌面应用程序，客户可以上传可执行包(例如然后可以从云界面中选择它。对于自动化的 web 应用程序测试，他们将使用 TestChameleon API 来自动化这个过程。公司网站上有文档来帮助你。

一旦你选择了测试标准，Testbirds 就会在 TestChameleon 中创建一个虚拟机，并根据你选择的不同变量测试你的软件。如果您有一组典型的标准，您可以创建测试模板，然后选择预定义的模板，这样可以节省您每次进行相同选择的时间。

我看了一下该网站的预发布版本，该版本今天上线，界面简洁易用。你只需点击你想要的选项，Testbirds 就会根据你的选择为你创建虚拟测试机，并针对你测试的软件运行。当这个过程完成时，您会得到一个包含结果的报告。

定价公共云计划每月从€49 美元(61 美元)起，根据您的使用情况、虚拟机数量和运行测试所花费的时间分为三个等级。还有一个企业私有云版本，起价为每月 1399€(1741 美元)。

Testbirds 在 3 月份获得了€210 万英镑(290 万美元)的 A 轮，由 Seventure Partners 领投。他们还从德国政府获得了 35 万美元的额外赠款。
