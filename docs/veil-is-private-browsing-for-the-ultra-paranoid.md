# 面纱是超级偏执狂的私人浏览

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/02/23/veil-is-private-browsing-for-the-ultra-paranoid/>

如果你担心有人发现你的浏览器指向的是什么，有很多选项可以保密，只是难度和效果不同。Veil 比其他任何匿名浏览方式都更进一步，它不仅对潜在的攻击者，而且对你自己的操作系统屏蔽了你正在浏览的页面。

正如 Veil 背后的麻省理工学院研究人员在一篇概述该服务的新论文中解释的那样，问题在于，隐私浏览模式，即使是使用 Tor 和其他措施的模式，仍可能在设备本身的 RAM 或临时存储中留下你的历史记录。

当你访问一个页面时，即使是匿名的，这个页面和它的组件仍然需要被加载到内存中，显示和缓存，库或插件可能被访问或修改。浏览器可能会尝试删除这些跟踪，但成功与否取决于设置的方式。你的活动的一种幽灵版本可能存在于你的 RAM 中，即使它只是一些数据或时间戳的散列。

“根本问题在于(浏览器)收集这些信息，然后浏览器尽最大努力修复这些信息。但最终，无论浏览器如何努力，它仍然会收集这些信息，”麻省理工学院研究生弗兰克·王(Frank Wang)在新闻发布会上解释道，他是这篇论文的第一作者。“我们还不如一开始就不收集这些信息。”

Veil 通过他们所谓的“致盲服务器”来处理网站的交付，从而更进一步。你在网站上输入网址，网页就会从特殊的服务器上被检索出来，在传输过程中和你的浏览器缓存中被加密，只有在你浏览的时候才被解密。链接和网址本身是加密的，所以它们不能链接到请求的内容。

此外，它将不可见的垃圾代码注入到页面中，同时还“改变”内容(再次，不可见)，以便您可以在同一台计算机上加载它一千次，尽管对您来说它看起来是一样的，但任何结果数字指纹(如哈希、有效负载大小等)总是不同的。

这样，你的计算机就不会注册实际的 URL，不会缓存任何数据，留下的任何痕迹都不会与任何数据库匹配，甚至不会相互匹配。

通过研究人员提供的特殊编译器，可以将网页转换成兼容 Veil 的网页。它不应该破坏任何东西，尽管由于突变和额外的加密操作，它会增加使用的带宽和服务的请求。

但是等等，还有呢！对于那些担心这种程度的混淆还不够的人来说，有一个选项我一直认为是可能的，但从未见过执行:仅通过视觉数据浏览。

如果你愿意的话，Veil 将不会向你展示目标站点的实际代码，而只是截图给你看。你可以点击截图，它会记录下点击的位置，然后传输到实际页面，返回结果截图。

当然，这不是一个零信任的情况:致盲服务器将像 Tor 节点一样，由志愿者和组织运行，这些志愿者和组织可能会试图危害系统(一个网站也可以运行自己的系统)。但如果这个过程在数学上和程序上都是可验证的，你应该没问题。当然，安全研究人员会想要审计代码。
