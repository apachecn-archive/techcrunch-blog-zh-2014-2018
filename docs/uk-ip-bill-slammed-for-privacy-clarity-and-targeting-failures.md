# 英国监控权力法案抨击隐私、清晰和目标失败 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/02/09/uk-ip-bill-slammed-for-privacy-clarity-and-targeting-failures/>

英国议会的一个委员会正在审查提议的新监控立法，该委员会发布了关于[调查权力法案](https://web.archive.org/web/20221205164022/https://beta.techcrunch.com/tag/ip-bill/)草案的[报告](https://web.archive.org/web/20221205164022/https://b1cba9b3-a-5e6631fd-s-sites.googlegroups.com/a/independent.gov.uk/isc/files/20160209_ISC_Rpt_IPBill%28web%29.pdf?attachauth=ANoY7coglpZcXREFBQXSMlrki3778OHZZwkEbhSmuH0GpCDu9z3ct8o3xyLCZsz5pIX3F7d2iixYtSwwlp15S2j8YUXy6cqO7vKRVg9DMMy8aADflSKxgygMkLEoMh_T_c3zY5Ky9Mq0HG7ugjXQmuSvKO75u6tRWrsyNJDdL1qAq4UTi1uoZLeijnSiQYjQMreZNAHYqHSMvzlEnPECXf6t52tYdiHISZCWWVI4Bfi2AF9FmZoMzy2pDLLhNoWa0gIEfR6F4r_k&attredirects=0)——这让政府读起来很不舒服。

另一个委员会，科学技术委员会，已经警告过[该法案的语言](https://web.archive.org/web/20221205164022/https://beta.techcrunch.com/2016/02/01/uk-govt-must-clarify-its-position-on-end-to-end-encryption-says-parliamentary-committee/)不够清晰，包括加密。但是情报和安全委员会(ISC)的报告对一系列问题提出了批评，包括它认为不一致和不充分的隐私保护，以及过于广泛的干涉权力，包括国家黑客能力。

就其本身而言，政府声称扩大监控权力是必要的，以填补情报和执法机构的“能力差距”——尽管该委员会的部分批评是，政府未能为其寻求立法的一些更大的扩张和侵入性权力提供操作上的理由。ISC 成员能够审查机密证据，因此你可能会认为该委员会更符合安全部门对监控的观点。相反，它对立法草案持高度批评态度。

民权组织称 ISC 报告是对该法案草案的“严重打击”。“这些建议又一次被定义为过于宽泛和缺乏清晰度，”老大哥观察在报告发表后的一份声明中说。

“我们敦促内政部接受技术部门、民间社会，现在甚至是监督警方和情报机构监控能力的议会委员会对他们的提议提出的广泛批评，”隐私国际在另一份声明中补充道。“ISC 的报告明确要求对立法进行彻底的重新审议，将隐私推到了最前沿。”

在伴随其报告发布的[新闻声明](https://web.archive.org/web/20221205164022/https://b1cba9b3-a-5e6631fd-s-sites.googlegroups.com/a/independent.gov.uk/isc/files/20160209_ISC_Press_Release_IPBill.pdf?attachauth=ANoY7crP1M2NyMiR7ztzyhXezpuLd27ArvLNic2Z5-46EftQtqU0znsTPC38x4gM4K_8sG183ZkbCan_VXVzj8ZdD13SLz0bvXOBPc7aHJSVBSzTs-7ws4nQ8mb_K21_cUOa45gHq3rYWgkURg6WWjmZHshrCk2PLxKjiAWDC5pnBykQ1iKVt7B9vGCBEfunAQWTqZgTw3xtifvZiZEIRh9TkrsEu8k8j0l0fHudIe9J5EddRdzBkpVN4vsv77lm10A-bGED-Yob&attredirects=0)中，ISC 主席 Dominic Grieve 指出，该法案草案未能涵盖所有机构的侵入能力——这是 ISC 去年发布的[隐私和安全报告](https://web.archive.org/web/20221205164022/https://b1cba9b3-a-5e6631fd-s-sites.googlegroups.com/a/independent.gov.uk/isc/files/20150312_ISC_P%2BS%2BRpt%28web%29.pdf?attachauth=ANoY7cot0eYndJncwGfRncELKodDXF9qhTWmIfH43khdNjIBUYlQsJAlBnFFomoK9yLXib3rovuYj4ZjlNzhmX92iEJ2OBG_4yJ03q1q7TGGLOqtgR89yPtEc8Iq6uuMUmHUQEpxyo5DThKZvZiY0eH-fAoLpkiI-Xk_JsygQRhvZyY_n8t-1sQSOyxGjAy80FZO_WN436RzjDbJjl6HECqAdGJ_YKeavcYSWn57ea3XrKX4hiA5wXwox3Ma3i2stPlGq6jHlbfn&attredirects=0)的一项关键建议——是为所有各种监控权力创建所寻求的全面法律框架的“错过的机会”。

“从整体上看，该法案草案未能提供该领域急需的明确性，”ISC 认为。“审议中的问题无疑是复杂的，但是很明显，即使是那些从事立法工作的人也不总是清楚这些条款的意图是什么。该法案草案似乎缺乏足够的时间和准备。”

委员会注意到政府在这方面的时间表很紧，目前的紧急监视立法[DRIPA](https://web.archive.org/web/20221205164022/https://beta.techcrunch.com/2014/07/10/uk-fast-tracks-emergency-surveillance-law/)(2014 年末未经适当议会审查通过)将于今年年底到期，只留下一个小窗口来立法取代日落权力。有人建议，知识产权法案草案是一项紧急工作。

ISC 也有一些关于隐私的强有力的建议——以及一些对它所谓的政府“渐进方法”的重要批评。

“我们原本预计会发现普遍隐私保护贯穿始终，或者至少是立法前沿的总体声明，”它写道。「相反，条例草案采用一种相当零碎的方法，缺乏明确性，并削弱了与这些权力有关的保障措施的重要性。

“因此，我们建议新的立法包含一个全新的部分，专门用于总体隐私保护，这应该成为立法草案的主干，然后围绕它建立特殊权力。这将确保隐私是立法的一个组成部分，而不是一个附加物。”

该委员会建议对法案中包含的三个方面的权力进行“重大修改”:设备干扰(又名黑客权力)；大量个人数据集(又名“包含各种人的个人信息的大型数据库”，情报机构在调查过程中使用这些数据库来识别个人，在感兴趣的对象之间建立联系，并验证他们通过其他方式收集的信息)；和通信数据(也称为元数据)。

在黑客权力方面，它希望看到所有相关的行动都处于同一立法之下——“具有相同的授权程序和相同的保障”——而不是像目前这样，与《1994 年情报服务法》制度分开。

它还建议完全删除允许批量设备干扰(又名大规模黑客攻击)的条款，指出“我们没有得到足够令人信服的证据来说明为什么需要‘批量’设备干扰令”，并建议“有针对性的”设备干扰令“可以足够广泛地延伸，因此不需要单独的‘批量’令”。

该法案草案的批评者此前抨击了批量认股权证提供的过于广泛的能力，例如，他们指出，如果你不知道你到底在针对谁，就很难判断相称性。

该委员会还希望从立法中删除“一般类别”的批量个人数据集授权(类别在这种情况下是指整个类别或类型的数据，如旅行数据)——同样是因为它们的针对性不够，将不可避免地侵犯多人的隐私权。

“作为一般原则，委员会认为，集体授权应该保持在绝对最低水平，”报告写道。“在这种情况下，鉴于每个批量个人数据集可能包含大量个人的个人信息——其中大多数人不会引起有关机构的任何兴趣——委员会认为，每个数据集都具有足够的侵入性，因此需要有特定的搜查令。因此，我们建议从立法中删除集体个人数据集担保。”

在通信数据方面，该委员会抨击政府在审查这一数据时采用了“不一致和混乱”的方法。"委员会认为，必须对所有来文数据的审查适用同样的保障措施，无论这些数据是如何获得的。这一点必须在立法中明确规定:仅仅依靠政策和良好做法是不够的，”它指出。

委员会还提出了一系列额外的具体修正案，包括呼吁将目前司法专员对“紧急”逮捕令进行复核的五天宽限期缩短为两天；缩短“主题”权证的时间长度(从六个月缩短到一个月)；以及在请求批准某项特定活动的调查小组和批准该活动的机构内部人员之间“明确的分界线”,这只是多项建议调整中的三项。

“该法案草案要求在获取通信数据时进行这种划分，但这些机构不受这一要求的约束，”ISC 报告在后一点上指出。“虽然我们被告知，鉴于这些机构经常使用通信数据，这将造成不必要的负担和时间延迟，但我们仍然认为分离是一个重要的原则问题，并建议在提出立法之前重新考虑这一点。”

另一个委员会——联合特别委员会——将于周四公布该法案草案的报告，自该法案于秋季提交议会以来，该委员会已经从一系列证人那里收集了证据，包括内政大臣本人。
