Gracious Eloise 开放 API，将任何人的笔迹数字化 TechCrunch

我们的笔迹，虽然慢慢地从存在中消失，但却是我们更独特的东西之一。但是任何曾经坐下来写十几封“感谢”信的人都知道手写每一封信是多么耗时。

这就是优雅的艾洛伊塞的用武之地。

该公司开发了一种算法，可以将任何人的笔迹数字化，让用户快速打出他们的信息，然后将其翻译成他们自己独特的笔迹。

更重要的是:优雅的 Eloise 从不以完全相同的方式数字化一条信息。即使副本完全相同，但在字体或间距上总会有微小的差异，同时仍然保持了您特殊的手写“风格”。

该平台还提供了一些工具来整理你的笔迹或添加一些修饰。

与使用机械臂创建手写笔记的邦德不同，Gracious Eloise 是一个完全数字化的平台，使用软件将你的笔迹数字化。

当该公司刚成立时，将一个人的笔迹数字化需要大约 3 个小时，但该团队在过去几个月里调整了算法，将时间缩短到 10 分钟。

Gracious Eloise 没有提供面向消费者的产品，而是以 API 为目标开发人员，因为该公司认为这种类型的技术可以同时在多个垂直领域扩展。

Gracious Eloise 的早期重点是礼品登记、贺卡、电子商务、CRM、直邮和数字(例如，将您的聊天应用程序或电子邮件连接到 API，让您可以发送手写的数字消息)。

创始人兼首席执行官 Eloise Bune 表示，每年仅婚礼一项就有超过 2.5 亿封感谢信，8000 万封是为婴儿送礼会或与婴儿相关的礼物发送的。

Gracious Eloise 已获得 300 万美元的首轮融资，投资者包括 BBG 风险投资公司、百老汇天使投资公司、女性创始人基金、大橡树风险投资公司、蒙太奇风险投资公司和硅谷女孩风险投资公司以及现有投资者。
