# Tinder 创始人起诉母公司 Match 和 IAC，要求至少 500 万美元 2B

一群 Tinder 创始人和高管已经对母公司 Match Group 及其控股股东 IAC 提起诉讼。

诉讼中的原告包括 Tinder 的联合创始人肖恩·拉德(Sean Rad)、贾斯汀·马丁(Justin Mateen)和乔纳森·巴丁(Jonathan Badeen)——巴丁仍在 Tinder 工作，原告詹姆斯·金(该公司的财务副总裁)和罗塞特·帕姆巴基安(该公司的营销和传播副总裁)也是如此。

我们联系了 IAC 和 Pambakian，他们是我们在 Tinder 的主要联系人。如果有回音，我们会更新帖子。

该诉讼称，IAC 和 Match Group 操纵财务数据，以创造“虚假的低估值”(引用原告的新闻稿)，然后剥夺了 Rad、Mateen、Badeen 和其他人的股票期权。它指向[解除 Rad 首席执行官](https://web.archive.org/web/20230322160427/https://techcrunch.com/2016/12/08/sean-rad-steps-away-from-ceo-role-again-to-run-tinders-new-swipe-ventures/)的职务，以及其他管理层变动，这些举动旨在“允许被告控制 Tinder 的估值，并剥夺 Tinder 期权持有者参与公司未来成功的权利。”

该诉讼还指控，Match 首席执行官格雷格·布拉特(Greg Blatt)成为 Tinder 的首席执行官，他在该公司 2016 年的假日派对上抚摸并性骚扰帕姆巴基安，据称这导致该公司“粉饰”他的行为，足以让他完成对 Tinder 的估值及其与 Match Group 的合并，然后[宣布他的离职](https://web.archive.org/web/20230322160427/https://www.prnewswire.com/news-releases/match-group-names-mandy-ginsberg-to-succeed-greg-blatt-as-ceo-300497940.html)。

作为回应，原告要求“赔偿损失，金额将在审判中确定，但不少于 20 亿美元。”

拉德在新闻稿中说:“我们一直担心 IAC 的声誉，因为它无视他们的合同承诺，表现得好像规则不适用于他们一样。”“但我们从未想到他们会欺骗所有创建 Tinder 的人。Tinder 团队——特别是目前担任该公司高级领导的原告——在揭露 IAC/Match 系统侵犯员工权利方面表现出了巨大的力量。"

我们刚刚收到 IAC 和 Match Group 的联合声明。

> 诉状中的指控毫无根据，IAC 和 Match Group 打算对此进行有力的辩护。
> 
> 自 Tinder 成立以来，Match Group 已经向 Tinder 的创始人和员工支付了超过 10 亿美元的股权补偿。关于诉状中指控的问题，事实很简单:Match Group 和原告经历了一个严格的、由合同规定的估值过程，涉及两家独立的全球投资银行，拉德和他那群快乐的原告不喜欢这个结果。拉德先生(一年前被公司解雇)和马丁先生(已多年不在公司工作)可能不喜欢 Tinder 在他们各自离职后获得巨大成功的事实，但酸葡萄本身并不能构成诉讼。拉德先生有丰富的古怪公开声明的历史，这次诉讼只是其中的一系列。我们期待在法庭上捍卫我们的立场。
