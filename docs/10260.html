亚马逊的 AWS 在伦敦开设了首个英国地区

去年，亚马逊的 AWS 云计算服务宣布将于 2016 年底在英国启动一个数据中心。现在是 2016 年底，正如其承诺的那样，AWS 现在推出了“欧洲(伦敦)”地区，以补充其在 2007 年和 2014 年开设的爱尔兰和法兰克福的现有欧洲地区。

总的来说，AWS 现在为其用户提供了 16 个地区的选择，全球共有 42 个可用区。仅在一年前，这一数字还在的 11 个地区上升，而且在 2017 年，这一数字可能还会增加。伦敦的新区域将有两个可用区。

伦敦将提供 AWS 的所有核心服务(想想 EC2、S3、RDS 等。).值得注意的是，AWS 的一些更先进的工具，如 Lambda 和它最近宣布的机器学习服务还不会在伦敦上市。亚马逊的托管企业邮件、文件存储、共享和虚拟桌面服务组合也尚未在伦敦推出。

“随着 AWS 欧洲(伦敦)地区的推出，AWS 可以帮助更多的英国企业、公共部门和初创企业客户降低 IT 成本，满足数据本地化需求，并在关键的新领域开始快速转型，如大数据分析和物联网，”亚马逊首席技术官沃纳·威格尔今天在他的博客上写道。
