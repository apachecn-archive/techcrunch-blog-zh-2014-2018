Vivaldi 正迅速成为击败 TechCrunch 的替代浏览器

原文：https://web.archive.org/web/https://techcrunch.com/2015/03/08/vivaldi-chrome-alternative-tech-preview-2/

Opera 前 CEO Jón von Tetzchner 在一月份发布了基于 Chromium 的新浏览器 Vivaldi 的第一个预览版。现在，在其第二次技术预览版中，增加了大量新功能，该浏览器正在迅速成为 Chrome，Firefox，Safari 和 IE 的一个有价值的替代品——尤其是对以前的 Opera 用户来说。

Vivaldi 毫无歉意地将自己视为 Opera 最初雄心(根据 von Tetzchner 的说法，它在过去几年中淡化了这一雄心)的高级用户浏览器。第一个预览版引入了一些简洁的内置功能，比如标签堆叠、笔记和快捷命令(它的聚光灯般的浏览器键盘命令)。当时对我来说，交易的破坏者是丢失的书签栏。现在第二部预告片已经增加了这一点——以及一系列其他新功能——维瓦尔第绝对值得再看一眼。

这个技术预览版中的新功能还有按需图像加载(对我来说不是很有用，但当你使用非常慢的 WiFi 连接时可能值得一试)和快进快退。有了这个，Vivaldi 将自动计算出是否有一篇文章或搜索结果页面的第二页，并让你直接进入那里，而不必向下滚动到网站的底部。这在像 Google、Hacker News 甚至 TechCrunch 这样有明显第二页的网站上运行得很好。在更复杂的网站上，比如纽约时报主页，它的选择更有问题(它首先带我去询问房地产，然后，也许是为了让我笑一笑，去了 IE8 支持页面)。不出所料，Opera 在转向 WebKit 之前提供了类似的功能，Chrome 用户可以尝试这个插件来看看它是如何工作的。

另一个主要的新功能是空间导航。像快捷命令一样，它也是键盘快捷键爱好者的一个功能——但它也应该对不会使用鼠标的人非常有用。您只需按住 Shift 键，然后使用箭头键在链接之间导航。

我花了相当多的时间研究其他浏览器，比如 Opera T1 和 T2 Maxthon T3，以及各种定制版本的 Firefox(想想 T4 苍白的月亮 T5 和 T6 Waterfox T7)和 Chrome。他们都没有接近取代 Chrome 成为我的默认浏览器，但我认为 Vivaldi 有机会(现在更是如此，至少非正式地，它还可以处理许多现有的 Chrome 扩展。

我还有点担心在浏览器中内置一个电子邮件客户端会拖累它。我真的不需要在我的浏览器中安装电子邮件客户端，但许多以前的 Opera 用户都强烈要求在 Vivaldi 中安装这个功能，当我上次与 von Tetzchner 交谈时，他也是这个想法的粉丝。尽管如此，该团队显然有一些好的想法和技术能力来实现它们。

维瓦尔第现在值得一试，但我很确定在接下来的几个月里它只会变得更好。
