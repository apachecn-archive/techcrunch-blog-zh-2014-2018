Signal 通过一个独立的桌面应用程序摆脱了浏览器的限制

如果你是一个活跃的信号用户(为什么不是呢？)你可能已经知道桌面应用程序——或者什么被认为是桌面应用程序。它实际上是 Chrome 应用程序，需要谷歌的浏览器才能运行。好了，噩梦结束了，Signal 已经发布了一款独立的应用程序，不再依赖于价值数十亿美元的公司。

这款应用程序今天在 T2 推出，适用于 Windows 64 位系统 7 和更高版本、MacOS 10.9 和更高版本以及许多受欢迎的 Linux 版本。

已经安装了 Chrome 应用程序的用户可以在安装过程中移植他们的数据，所以他们可能想先登录旧版本，以确保所有数据都经过验证等等。

今年夏天，当我们在 Disrupt 与信号创造者莫邪·马林斯派克交谈时，他说有几个主要功能即将推出，但拒绝说出它们的名字；嗯，这里有一个！
