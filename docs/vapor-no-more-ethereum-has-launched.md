# 蒸汽不再:以太坊发布

我认为，周四是历史性的一天——尽管它是否会像牛郎星 8800 T1 的发布一样成为一个值得纪念的里程碑，或者成为 T2 CP/M T3 的一个模糊的注脚，还有待观察。周四是[以太坊项目](https://web.archive.org/web/20230316161018/https://www.ethereum.org/)终于[推出](https://web.archive.org/web/20230316161018/https://blog.ethereum.org/2015/07/30/ethereum-launches/)的日子，在喧嚣和愤怒之后，包括筹集了约 1000 万美元(比特币。)这是一个迷人的、雄心勃勃的项目，它不再是一个蒸汽器皿。

要解释以太坊，我们先回到比特币。到目前为止，人们普遍认为比特币是一种虚拟货币，通过加密技术及其分散的采矿网络的计算能力来保护。不太好理解的是，比特币不仅仅是比特币；每一笔比特币交易，其实都是用自定义脚本语言编写的程序。这种语言不是图灵完全的语言，也就是说，基本上，它不是一种成熟的编程语言。

(图灵完备并不是特别高的杠杠。[后记是图灵完备的](https://web.archive.org/web/20230316161018/https://en.wikipedia.org/wiki/PostScript#The_language)。 [TrueType 字体带有图灵完全脚本语言](https://web.archive.org/web/20230316161018/http://yahoo-security.tumblr.com/post/123981052855/font-parsing-vulnerabilities)。比特币的脚本语言不是因为这是中本聪出于安全/性能考虑做出的决定。)

以太坊和比特币一样，是建立在区块链上的，有一个分散的挖掘网络。它的硬币被称为“醚”；它通过预开采和向信徒/投资者出售大量乙醚筹集了数百万种子资金。但是它的脚本语言是图灵完整的和全功能的，这极大地扩展了它所支持的智能合同的种类。事实上，以太坊并不打算作为一种新的加密货币，而是作为一个运行在去中心化的区块链之上的大型虚拟机。

对我来说，比特币最有趣的两点是，它是完全去中心化的，它是可编程的货币。以太坊是这两种东西的兴奋剂。它被明确设计为“分散网络”，即提供由其整个网络维护的服务和内容，而不必依赖或信任任何特定的公司或服务器集群。(我这个学究想论证“互联网去中心化”既更准确又更有意义，但没关系。)

潜在的应用非常广泛。分散式网站。分散的[消息传递](https://web.archive.org/web/20230316161018/https://blog.ethereum.org/2015/06/21/ethereum-messaging-masses-including-fathers-via-infographic/)。一个有趣的名为[的预测/预警系统预测市场已经在以太坊上推出。我预计，最终，智能合约应用的整个类别*还根本不存在——比如*](https://web.archive.org/web/20230316161018/http://www.augur.net/)*[分散式自治系统](https://web.archive.org/web/20230316161018/https://blog.ethereum.org/2014/05/06/daos-dacs-das-and-more-an-incomplete-terminology-guide/)——因为基于区块链的虚拟机是一种全新的平台。*

…好吧，让我们停止空想，回到现实中来。这些都是非常早期的事情；这只是多阶段发射的第一个初步阶段。(对于你们这些铁杆加密货币迷来说，理论上将会以令人惊讶的从[工作证明](https://web.archive.org/web/20230316161018/https://en.wikipedia.org/wiki/Proof-of-work_system)到[利益证明](https://web.archive.org/web/20230316161018/https://en.wikipedia.org/wiki/Proof-of-stake)的转变而告终。)即使那样，表现也会*可怕*；这将是一个分散的虚拟机，但与你桌上的电脑，甚至是你口袋里的电脑相比，它又慢又弱，非常 T21。

我怀疑以太坊是向前迈出的重要一步，是一个重要的里程碑，但绝不是路的尽头。我的猜测是，它最终，或者甚至很快，会被其他基于其概念的侧链所取代。(不是新的观念；例如，交易对手[已经实施了](https://web.archive.org/web/20230316161018/http://counterparty.io/news/counterparty-recreates-ethereums-smart-contract-platform-on-bitcoin/)以太坊的智能合约。)

但也许我错了；也许以太坊是新的 DOS，而不是新的 CP/M。无论如何，祝贺它的创始人(也是泰尔的研究员)[维塔利克·布特林](https://web.archive.org/web/20230316161018/https://en.wikipedia.org/wiki/Vitalik_Buterin)和它的团队的其他成员。他们建造了[并运送了](https://web.archive.org/web/20230316161018/https://github.com/ethereum/)一些真正新的、非常雄心勃勃的、在智力上引人入胜的东西。这比大多数创业公司能说的要多得多。
