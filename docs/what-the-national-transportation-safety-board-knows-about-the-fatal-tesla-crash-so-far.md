# 到目前为止，NTSB 对致命的特斯拉事故了解多少

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/07/26/what-the-national-transportation-safety-board-knows-about-the-fatal-tesla-crash-so-far/>

美国国家运输安全委员会(NTSB)发布了一份关于 2015 年特斯拉 Model S 和 2014 年 Freightliner Cascadia 卡车在佛罗里达州 Williston 发生致命碰撞的初步报告。

不幸的是，这份报告没有产生重大的新见解。

该报告证实，特斯拉的司机在撞车前的时速是 74 英里，在一条限速 65 英里的高速公路上。它还证实了他正在使用特斯拉的“高级驾驶辅助功能”，也就是自动驾驶仪，包括“交通感知巡航控制”和“Autosteer 车道保持辅助”

NTSB 没有承诺何时发布最终报告，但指出通常需要一整年时间来完成数据分析，并回答此类调查中有关坠机可能原因的问题。

乔治·华盛顿大学法学院教授[韦恩·科恩](https://web.archive.org/web/20221013121930/https://www.law.gwu.edu/wayne-r-cohen)，也是人身伤害律师事务所[科恩&科恩](https://web.archive.org/web/20221013121930/http://www.cohenandcohen.net/)的创始合伙人，他说死亡报告通常由州警察调查，而不是联邦当局。

他说，佛罗里达州威利斯顿的特斯拉事故已经获得了联邦政府的干预，因为“我们所处的环境是，道路上的车辆使用越来越复杂的技术，人工智能超越了我们几十年来拥有的巡航控制和事件数据记录器。当涉及到民事和刑事责任时，这种技术的水域是混乱和未知的。”

也许最重要的是，调查结果将有助于为事故中受伤或遇难者的家属解答疑问。科恩建议，但他们也将帮助美国建立法律和立法框架，使这些车辆进入国内日常使用。

特斯拉尚未回应有关它如何与 NTSB、NHTSA 和佛罗里达州警方合作，以完成对这起致命车祸的各种调查的询问。

自从撞车事故发生以来,[消费者报告](https://web.archive.org/web/20221013121930/http://www.consumerreports.org/tesla/tesla-autopilot-too-much-autonomy-too-soon/)呼吁该公司“禁用”并重新命名其自动驾驶功能，直到它变得更加安全。

特斯拉对《消费者报告》以及所有批评者响亮地说“不”，并声明:

> “特斯拉正在不断推出经过数百万英里内部测试证明的增强功能，以确保由 Autopilot 支持的司机比没有辅助的司机更安全。随着技术的发展，我们将继续开发、验证和发布这些增强功能。”

特斯拉汽车公司首席执行官埃隆·马斯克也多次指出，特斯拉汽车公司的车辆已经在自动驾驶仪上行驶了 1.3 亿英里，其中一人确认死亡，这是一个比人类司机更好的安全记录。

上周，马斯克发表了一份针对特斯拉未来的[“总体规划”](https://web.archive.org/web/20221013121930/https://beta.techcrunch.com/gallery/dr-elon/)，其中包括让自动驾驶比传统驾驶安全 10 倍的目标。在该计划中，马斯克表示，按照目前的速度，该公司将在大约 5.5 年内看到特斯拉汽车驾驶自动驾驶汽车行驶 60 亿英里。这是他预计该技术将为“全球”主流批准做好准备的时间点。

当然，受到审查的不仅仅是特斯拉的技术。该公司与其他供应商合作。

例如，它已经使用 [Mobileye](https://web.archive.org/web/20221013121930/https://beta.techcrunch.com/2016/07/26/mobileye-wont-supply-tesla-self-driving-tech-beyond-current-eyeq3-processor/) 图像分析处理器来实现半自动驾驶。今天，Mobileye 宣布合作关系即将结束。
