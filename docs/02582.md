# 拿出你的刻录书，因为移动游戏《坏女孩》即将登陆 iOS

你有没有被莉贾娜·乔治伤害过？你现在可以在游戏中扮演她。名副其实的游戏公司 [So Much Drama](www.somuchdramastudios.com/,) 将电影《贱女孩》的改编版带到了 iOS 上。

类似的游戏如《T2》的《贱女高中对决》和《T4》的《贱女 DS》都存在，但这是第一次在 iPad 和 iPhone 上推出基于热门言情片的游戏。

![Burn Book 1](img/0622f7df565d9016571f516daba770e6.png)

玩家会发现，一组新的塑料试图通过拼凑破碎的春季狂欢头饰来接管北岸高中。您可以通过选择八个主要角色来恢复女孩世界的和平:凯蒂、雷吉娜、格雷琴、凯伦、詹妮斯、达米安、亚伦和凯文。

无论你是一个乐队极客还是一个规则辣妹，所有的角色都有自己独特的能力和派系来击败对手和提升水平。如果你选择邀请你的脸书朋友，你也可以和他们组队。

![cafeteria](img/d0041424f34a17fe846609c409403585.png)

这部电影的粉丝肯定会喜欢不同的播放模式，如“你不能和我们坐在一起”和“她甚至不去这里。”

《贱女孩》非常搞笑、残酷，而且可以被没完没了地引用；“我非常喜欢看这部电影，而且一遍又一遍地看，”多戏剧公司总裁杰夫·米德说。“这个游戏充满了古怪的幽默、夸张的高中权力斗争，以及从滑稽的台词、薄荷护足霜到，是的，烧伤书的一切。”

如此多的戏剧已经为露波制作了两个基于流行节目 RuPaul's Drag Race 的社交手机游戏。《贱女孩》将于今年年初上映。
