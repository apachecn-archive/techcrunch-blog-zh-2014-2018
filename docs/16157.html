我是一个长期的加密货币信徒，非常难过地看到这个空间退化为一个像不受监管的细价股市场一样有趣、充满新兴属性的东西。

但是仍然有一些真正有趣的、技术上创新的区块链项目，如果你去寻找的话:Blockstack、Foam、Mattereum 等等。我在 HappyFunCorp 的日常工作中，我们自己也在做一些，包括内部的和为几个客户做的(他们仍然处于秘密状态)。)

与此同时，世界上到处都是网络程序员，但真正缺乏加密货币开发者，这两个领域之间的鸿沟很难跨越。所以我想我应该把我从设计和构建我们自己的基于以太坊的项目中学到的东西，变成一个面向 web 开发人员的开源教程，希望鼓励更多实际的开发和进化，而不是盲目的猜测。

这么多钱进入这个领域是件好事，但除非人们真的用它来播下创新的种子，否则这不会有任何帮助，我们希望这将在某种程度上有助于实现这一点。
