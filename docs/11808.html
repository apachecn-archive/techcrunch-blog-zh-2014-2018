在特拉维斯·卡兰尼克的沙发上睡了几年后，这位创始人已经为他的创业公司 Mashape 筹集了 1800 万美元

对于 Mashape 的首席执行官 Augusto Marietti 来说，硅谷梦正在如想象中的那样展开。

2009 年，Marietti 和他的联合创始人 Marco palla dino——一年前在米兰成立了一家软件公司——了解并被接纳进入 TechCrunch50，一个已经演变成今天的巨型 TechCrunch Disrupt events 的沥青比赛。

随着活动的临近，优步首席执行官特拉维斯·卡兰尼克——当时是一名活跃的天使投资人，刚刚开始拼车服务的工作——在 Craigslist 上发布了一则广告，向外地来的创始人提供他在的家。三支队伍响应；卡兰尼克选择了玛丽蒂和帕拉迪诺，为他们提供了几天的卧室，更重要的是，他们回国后通往硅谷的生命线。事实上，Marietti 说，六个月后他回到了加州，在 Kalanick 的厨房餐桌上与几名 YouTube 早期员工谈判 10 万美元的可转换债券。

当时，玛莎佩银行里只有 2000 美元。快进到今天，总部位于旧金山的 Mashape 已经筹集了 2800 万美元的资金，包括由 Andreessen Horowitz 牵头的 1800 万美元的 B 轮融资，早期支持者 Index Ventures 和 CRV 也参与了融资。

正如马里蒂所说，随着时间的推移，这场战斗变得越来越容易打了。

2011 年，新企业联合公司(New Enterprise Associates)领投了 150 万美元的种子资金，然后该公司“艰难地进入了首轮融资”，这是一轮 650 万美元的融资，于 2013 年结束，其中包括来自埃里克·施米特(Eric Schmidt)的创新努力和亚马逊创始人兼首席执行官杰夫·贝索斯的少量资本。

但是 Mashape 的最新一轮“发生得很快，因为我们发现了一个金矿”，他的 25 人公司的 Marietti 说。具体怎么做？该公司已经为自己建立了一种内部技术来帮助导航它几年前开始建立的 API 市场；在某种程度上，它意识到其“类似空中交通管制员”的技术是其业务中最有价值的方面，并开始将其提供给企业。

这是一个明智的转变。虽然像优步这样的早期采用者很早就理解了 API 的力量——还记得优步将其最初的产品与其他公司的地图、支付和通信技术组装在一起——但更传统的公司却没有意识到 API 将成为现代应用架构的中心。

他们现在意识到了 API 的力量。其实大公司也开始一波一波的拥抱这些微服务了。现在他们需要工具来帮助他们的开发团队构建灵活的新软件系统——这就是 Mashape 的用武之地。

在应对这一挑战时，它并不孤单。Apigee 和 Tibco 的 Mashery 是两家争夺相同客户的公司。然而，Mashape 表示，目前有 30 多家公司为其核心产品 Kong 支付了 6 位数的年费。与此同时，Marietti 表示，这家初创公司在去年 12 月实现了现金流转正。
