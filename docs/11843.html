Facebook Messenger 现在允许你私下分享你的实时位置一小时| TechCrunch

Facebook Messenger 希望成为一个线下社交聚会工具，而不仅仅是一个在线聊天应用。今天，它增加了一个新的实时位置功能，让你在一个私人直接或群组消息线程中分享一个小时你在地图上的实时位置。然后，收件人可以看到你开车到达他们那里估计需要多长时间。

该功能的产品经理 Selena Wang 写道:“当你试图与朋友协调时，实时位置非常有用，当你去赴约时，它可以告诉人们你有多近，甚至当你晚上回家时，它可以告诉你的室友你在哪里。”今天，它面向所有 iOS 和 Android Messenger 用户推出。Messenger 现在可以挑战 Foursquare Swarm 等定位应用程序和 Down To Lunch 等 meetup 应用程序。

Live Location 是脸书在 Messenger 地图上的第二次重大尝试，此前第一次尝试因隐私丑闻而在 2015 年被取消。许多用户在每条消息中分享他们瞬间的准确位置，一名哈佛学生发现，通过他构建的显示朋友确切位置的 Chrome 扩展，可以将这些信息刮入活点地图。脸书取消了学生的实习机会，让他关闭了扩展，并改为只允许用户发送他们的一次性当前或未来位置。

静态定位功能仍然可用，但现在也有了实时定位。不过，这一次，你不能默认让它开着。Messenger 的负责人 David Marcus 发布说“我们许多人在 Messenger 上制定计划，包括寻找彼此，或者让朋友和爱人知道我们正在路上。分享你的位置也将帮助你们中的一些人在回家的路上感到更安全。”

要使用实时位置:

  1. 在消息线索中，轻按“位置”按钮或在“更多”菜单中找到它
  2. 在地图上，点击蓝色条开始共享您的实时位置
  3. 收件人将在 60 分钟内看到你在地图上的准确位置，以及你开车到达他们的预计时间
  4. 地图一角的时钟会倒计时，直到您的位置共享过期，您可以随时点击停止共享

当然，电池寿命是定位功能的一个大问题，因为 GPS 会很快耗尽您设备的能量。但是脸书告诉我，它对 Live Location 使用标准量的电池很有信心，Marcus 写道:“我们投入了大量的时间和精力……将电池消耗降至你甚至不用考虑的程度。”尽管它仍然值得关注。

除了活跃的 Now 状态指示器和 Messenger Day 的“谁准备好了？”过滤器，Messenger 正在成为一个功能更加全面的工具，用于与朋友聚会。这可能使它比严格的聊天产品和专用的视觉通信应用程序更有优势。

离线采集仍然是一个很大程度上没有解决的问题。很难知道谁可以一起出去玩，而且离得够近，方便不方便。要想在这个领域胜出，一款产品需要无处不在的社交图、位置共享、用户经常查看的地方，以及用于规划的信息组件。Swarm，下至午餐和其他应用程序的墓地总是错过一个关键的部分。

但正如我去年所写的，Messenger 可能会成功，因为它依赖于脸书身份，并且已经成为人们组织聚会的地方。对于我们所有的花哨应用程序，我们仍然需要一种方式来对抗孤独和面对面的人。记住，现在如果你说“在路上”,你最好不要穿着睡衣在家撒谎。
