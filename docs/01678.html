Sourcery 筹集了 250 万美元来连接厨房和当地食品供应商

在线批发食品供应商总监和支付处理商 Sourcery 今天宣布退出 stealth，并筹集了 250 万美元的种子资金，以将其平台从早期在旧金山的运营扩展到新的市场。

虽然许多厨房经理都坚持从目录中订购，并通过支票、电话或传真支付，但 Sourcery 使这一过程更接近于最终用户从在线送货初创公司订购时获得的东西:一个易于搜索的该地区可用商品及其价格的目录。当你进入订购流程的一部分时，他们的 web 应用程序会处理付款，并对你的订单历史进行全面分析，显示你买了什么，什么时候订购的，付了多少钱，所有这些都可以导出到 QuickBooks。Sourcery 还为那些没有准备好将所有会计数据数字化的企业提供服务，允许客户将数据作为传统发票导出。

Sourcery 的联合创始人兼首席执行官 Na'ama Moran 表示，该公司将批发食品供应商与 Palantir、T2、T4、Airbnb 以及 La Mar、wichcraft 和 Hops Hominy 等餐厅以及当地餐饮公司和果汁制造商联系起来。另一家以旧金山为基地的创业公司 Munchery ，提供熟食上门服务，它通过 Sourcery 处理所有的供应采购，并依靠这项服务将业务扩展到西雅图。虽然她不能透露具体细节，但她也向 TechCrunch 指出，Sourcery 每个月为每个客户处理 2.5 万至 50 万美元的收入，并从中抽取一定比例作为收入。

莫兰说，展望未来，洛杉矶、波特兰和纽约市已经有客户注册了，但还没有得到服务。Sourcery 计划从明年 1 月开始进军这些市场。在那之前，这家初创公司正在努力让已经在等待名单上的 55 家公司加入进来。

Sourcery 的种子轮投资者包括 Palantir 联合创始人乔·朗斯代尔、Yammer 首席技术官亚当·皮索尼、BoxGroup 的大卫·蒂施和亚当·罗森博格、Postini 创始人 Shinya Akamine 以及甲骨文前首席财务官杰夫·爱普斯坦。
