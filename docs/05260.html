Zendesk 以 4500 万美元收购商业智能初创公司 BIME 分析公司

客户支持巨头 Zendesk 以 4500 万美元现金收购法国初创公司 T2 BIME 分析公司 T3。BIME 分析公司是一家采用软件即服务方式的商业智能初创公司。它将来自许多不同来源的数据转化为趋势和见解。这家初创公司从阿尔文资本和商业天使那里筹集了 340 万美元。

这家初创公司提供了几十个数据连接器来聚合来自多个来源的数据——big query、CSV、Excel、Intercom、Google Analytics、Salesforce 等等。你可以为商业智能的目的命名一个数据点，而且 BIME 很可能已经支持它了。

然后，您可以创建交互式仪表板、高级可视化或使用 SQL 查询利用结果。一切都是快速和分布式的，你不必管理服务器来做这些数据操作。

这对 Zendesk 来说是一次重大的收购，因为这家上市公司( NYSE:ZEN )去年报告的收入为 1.3 亿美元。“此次收购预计不会对 Zendesk 的 2015 年收入或非 GAAP 运营亏损产生实质性影响，”该公司在一份新闻稿中表示。

BIME 首席执行官蕾切尔·德拉库尔和首席技术官尼古拉斯·拉斯帕尔将继续为 Zendesk 开发 BIME。Zendesk 将使用 BIME 的产品来推广一种数据驱动的客户支持方法，利用关于客户的一切来提供更好的支持。至于现有的 BIME 客户，Zendesk 将继续开发、销售和支持其产品。

这是法国科技生态系统的又一次重大收购，因为 Crédit Mutuel Arkéa 最近收购了钱罐和支付服务提供商 Leetchi 。我听说其他的法国收购正在进行中。

很高兴看到 Zendesk 收购了一家法国初创公司。虽然 BIME 在美国已经有一个办公室，但团队的大部分成员都在法国的蒙彼利埃。Zendesk 走出自己的舒适区，抢夺这家法国初创公司。
