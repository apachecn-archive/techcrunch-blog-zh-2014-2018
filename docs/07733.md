# 亚马逊起诉卖家购买虚假评论 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/06/01/amazon-sues-sellers-for-buying-fake-reviews/?guccounter=1&guce_referrer=aHR0cHM6Ly93d3cuZ29vZ2xlLmNvbS8&guce_referrer_sig=AQAAAC4Wi3cebvsTpdwp8YhbWhK4JOHHUUBCv78fil-djZ-RzKPr_YvMH1rSnprKu2yl-XMCVz23gxUHFTC4kTMhSTnxKdTTcEIbMH6QTglwbZjy-eo7pEoa-03zLuXjuWgjdHmGTITbtniEm49ztGeL9RRQAFdo3V3qLgN-vIJ9xEVK>

卖家当心——如果你在亚马逊上购买你产品的评论，该公司可能会起诉你。

作为打击其平台上虚假评论的努力的一部分，亚马逊今天起诉了三名卖家，因为他们使用袜子木偶账户发布关于他们产品的虚假评论。过去一年，亚马逊一直在积极起诉它认为不真实的评论者，经常利用[诉讼](https://web.archive.org/web/20230206025132/https://techcrunch.com/2016/04/26/amazon-cracks-down-on-fake-reviews-with-another-lawsuit/)来阻止评论的买卖，但这是它第一次起诉卖家本身。

今天的诉讼针对的是亚马逊声称使用虚假账户给自己的产品留下正面评价的卖家。虚假评论占卖家总评论的 30%到 45%。被告分别是加利福尼亚州的迈克尔·阿巴拉、宾夕法尼亚州的库尔特·鲍尔和一家名为 CCBetter Direct 的中国公司。

亚马逊要求禁止被告在其任何网站上销售产品或访问其服务。诉讼还要求赔偿卖家在亚马逊上获得的利润、律师费以及超过 2.5 万美元的损失。【**更新 6/3，上午 9:22**:亚马逊要求对这些案件进行仲裁，这意味着他们可能会比亚马逊打官司进行得更快，而且不会在公众眼中出现。]

亚马逊表示，自 2015 年初以来，它已经起诉了 1000 多名发布虚假评论以获取现金的人。现在，该公司正在追逐零售商本身。亚马逊表示，它打算消除卖家为其产品购买虚假评论的激励。

“我们的目标是消除卖家滥用评论的动机，并关闭围绕欺诈性评论的生态系统，以换取赔偿，”亚马逊发言人说。

亚马逊采用了许多方法来打击虚假评论，包括暂停卖家和关闭他们的账户。这家在线零售商还在努力开发算法来检测虚假评论，并防止它们出现。去年夏天，它开始对值得信赖的评论进行排名，这样购物者就能首先看到这些评论。

在诉讼和算法之间，亚马逊表示，它在打击虚假评论方面取得了成功。“亚马逊上的绝大多数评论都是真实的，帮助数百万顾客每天做出明智的购买决定，”一位发言人说。
