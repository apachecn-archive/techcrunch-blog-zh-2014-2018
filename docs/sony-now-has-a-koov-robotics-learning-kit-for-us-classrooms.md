# 索尼现在有一个面向美国教室的 Koov 机器人学习套件 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/02/14/sony-now-has-a-koov-robotics-learning-kit-for-us-classrooms/>

去年夏天，索尼在美国试推出了一款基于积木的教育机器人套件，以测试当地的兴趣，索尼认为 Koov 套件的反响热烈，足以将一款成熟的教育产品推向美国市场。 [Koov 教育家 Kid](https://web.archive.org/web/20221207045632/https://www.sony.com/koov) 今天开始接受预购，预计发货日期为 3 月 25 日。

520 美元的价格标签使它成为目前为止最昂贵的以 STEM 为目标的学习编码小发明。(现在*有很多*准教育性的“玩具”供父母选择。)但是，顾名思义，索尼的 Koov 套件是专门为教育工作者和教室使用而设计的，每个套件都支持多个用户。

具体来说，根据索尼的说法，每套 Koov Educator 套件适用于“最多五名学生”——大概是因为孩子们在一起玩得很好，并且公平地分享积木和零件。因此，在这种情况下，定价虽然不便宜，但看起来更合理。

索尼还明确瞄准了“STEAM”学习，首字母缩写中的“A”代表“艺术”，以及更常见的科学、技术、工程和数学组件，这使 Koov 有别于市场上一些不太灵活的学习编码小发明。

虽然其他模块化电子编码套件，如来自 [LittleBits](https://web.archive.org/web/20221207045632/https://beta.techcrunch.com/tag/littlebits/) 和 [Sam Labs](https://web.archive.org/web/20221207045632/https://beta.techcrunch.com/tag/sam-labs/) (仅举两个例子)的套件，也在几乎相同的领域发挥作用。

Koov 系统是为八岁以上的儿童设计的。除了半透明的插接式积木和兼容 Arduino 的电子元件，还有一个基于 Scratch 的拖放编码接口，通过跨平台的配套应用程序将物理创作与数字控制相结合。

教育家套件总共包含 300 多个可连接的模块，加上多个传感器、电机、led 和其他电子元件(完整的清单在这里)。它还包括班级管理软件、与课程相一致的课程计划、面向孩子的分步指南和学生进度报告。

索尼表示，Koov 应用程序通过一个学习课程功能提供了超过 30 小时的教育内容，该公司表示，该功能旨在向学生介绍编码、建筑和设计方面的“关键概念”。与大多数这些 STEM 小发明一样，教育哲学在很大程度上依赖于通过玩耍来学习的理念。

Koov 套件还包括 23 个预先设计、预先编码的“机器人食谱”，以鼓励孩子们立即动手建造。尽管 Koov 系统的更广泛的目标是支持儿童能够设计和建造他们自己的机器人(回到“艺术”元素)——事实上索尼声称有“无数”种方法将它的积木和零件粘在一起。就像乐高积木一样。

它还认为该系统“足够灵活”，可供学生独立使用，同时也提供了支持结构化学习的材料。
