# 为餐馆提供销售服务的 Toast 筹得 3000 万美元 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/01/05/toast-a-point-of-sale-service-for-restaurants-raises-30m/>

当 Chris Comparato 和他以前的同事一起在 Toast 公司工作时，筹集新的资金以继续扩大公司规模几乎是当务之急。

因此，该公司今天表示，它已经在由 Bessemer Venture Partners 牵头的一轮融资中筹集了 3000 万美元，谷歌风险投资公司也参与其中。Comparato 表示，该公司拥有 1000 多名客户，在过去一年左右的时间里，从“数百人”增长到“数千人”。

“总的来说，我们的愿景是通过利用(包括销售点在内的在线服务)，真正创造一个所有不同规模的餐厅都能提供最佳客户体验的世界，”他说。“这意味着为那些希望更灵活地创建订单的男女服务员提供更好的体验，或者为顾客和顾客提供更好的客户体验，我们认为技术可以成为一个推动者。”

这项服务在 Android 平板电脑上运行，但这当然意味着它将与其他销售点服务竞争。这可能包括其他供应商，如 Square 和 Shopkeep。但 Toast 的卖点是，它不仅仅提供销售点，让餐馆老板更容易选择其他附加服务。尽管如此，如果 Square 是一个指标——其具有挑战性的首次公开募股——将销售点服务作为一个关键挂钩可以表明整体业务具有挑战性。

当然，筹集资金并不是 Comparato 唯一考虑的事情。他说，他清单上的第一和第二件事是在美国扩大销售和营销，以及扩大公司的研发工作。他说，这两项都是资本密集型的，因为它们需要增加公司的员工人数，而公司已经有 170 多名员工。

“你可以是一家提供全套服务的小型独立餐厅，你需要一种新的销售点解决方案。也许它从销售点开始，但当你开始权衡你的选择时，你选择了一个包括销售点以及其他集成的开放灵活性的平台，”Comparato 说。“小型运营商在技术选择上受到了打击，他们必须选择销售点解决方案，可能还必须考虑忠诚度计划、在线订购平台和分析能力。”

他说，2016 年的另一个目标是添加一套工具，让 Toast 与其他服务集成。例如，Toast 可能提供一种递送工具，但餐馆老板可能想用其他服务来代替。他说，他们将有望与其他服务整合，而不是强迫餐馆依赖 Toast 的内部服务。
