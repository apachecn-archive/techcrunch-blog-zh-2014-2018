# 尽管第三季度收益好于用户增长乏力、引导乏力，但推特股价仍下跌 8%

下课铃响后，推特宣布了第三季度的财务表现，包括 5.692 亿美元的收入和 0.10 美元的调整后每股收益。市场曾预计这家受欢迎的社交商店将从 5.596 亿美元的收入中获得每股 0.05 美元的收入。

推特的股价跟随其盈利记录大幅下跌。

推特宣布每月有 3.2 亿活跃用户。相比之下，这个数字与去年同期的每月 2.84 亿次活动相比，此前的数字为 3.16 亿次。

对推特似乎长期萎靡不振的用户增长的担忧是投资者的一个主要担忧。如今，该公司未能缓解这些担忧。如果你不明白为什么投资者会像谚语所说的那样把推特股票卖给伐木工人，这就是原因。

事实证明，推特在收入增长方面是个怪物。与去年同期相比，该公司的收入增长了 58%。但用户增长缓慢是未来现金流的固有折扣。你肯定还记得，像这些投资者。

该公司 86%的广告收入来自其服务的移动使用，这是一个强劲的数字。推特是少数几个成功从小屏幕上赚钱的公司之一。

尽管如此，投资者的主要担忧仍有增无减。

展望未来，推特预测本季度其收入将在 6.95 亿至 7.1 亿美元之间。这一事实落后于华尔街[估计的 7 . 397 亿美元巨大而痛苦的赤字。](https://web.archive.org/web/20221006235823/http://finance.yahoo.com/q/ae?s=TWTR+Analyst+Estimates)

按照正常的会计方法，推特在这个季度损失了 1.317 亿美元。投资者长期以来对推特的估值不是基于公认会计准则，而是基于调整后的数据，以考虑到大量的股票薪酬。考虑到公司的其他问题，这种情况可能不会再出现了。
