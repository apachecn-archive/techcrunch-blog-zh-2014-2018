Luminoso 获得 650 万美元 A 轮投资，继续打造云文本分析服务 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2014/07/02/luminoso-lands-6-5m-in-venture-capital-to-keep-building-cloud-text-analytics-service/?ncid=rss

总部位于马萨诸塞州剑桥的文本分析公司 Luminoso 今天宣布获得 650 万美元的首轮融资，由 Acadia Woods 领投，Digital Garage 也参与其中。这使得他们筹集的资金总额达到 800 万美元。

Luminoso 诞生于麻省理工学院媒体实验室的研究，并于 2010 年剥离。在 2012 年获得种子资金后，他们继续开发产品，在云中提供文本分析——这笔钱应该会给他们一个很大的提振，让他们继续建设公司，雇佣更多的人员。

它在竞争中的独特之处在于，他们通过动态自动构建分类，消除了文本分析中最困难的部分。据首席执行官兼联合创始人凯瑟琳·哈瓦西(Catherine Havasi)称，他们建立了数据建模和机器学习，因此这一切都是自动化的，如果他们能够实现这一目标，这是一个非常巧妙的技巧。

根据 Havasi 的说法，这意味着系统建立了自己的单词分类，即使俚语进入词典，Luminoso 也能够将其识别为新单词，并从用户的上下文线索中推断其含义。

“人们谈论产品的方式很有创意，”哈瓦西说。“我们希望给予计算机理解创造性语言的直觉。在媒体实验室，我们建立了一组数据，作为人们如何看待世界的模型，”她解释道

这导致了现代机器学习的发展，但他们更进一步，将其基于云，使客户更容易利用。

她说，到目前为止，最大的用例是围绕市场研究和理解文本流，因为它们可以更好地了解市场上正在发生的事情。虽然她说，它适用于许多使用场景，但作为一家初创公司，重要的是专注于强大的用例，并以此为基础建立公司。

目标当然是帮助人们理解大量的非结构化数据，到目前为止，至少重点是理解客户交互。哈瓦西说，作为概念的证明，他们一直在与索尼合作，从社交媒体收集关于人们对世界杯的反应的信息。

正如《纽约时报》的一篇文章上周在德美比赛中使用这项技术所指出的，这不仅仅是简单的情绪分析，比如人们是否快乐。“它的技术还包括一个知识库，该知识库是建立在麻省理工学院始于 1999 年的一个众包项目基础上的，该项目被称为开放思维常识项目，”文章称。

哈瓦西说，她认为其他文本分析公司，如 Lexalytics 和 T2 Clara bridge 是该公司的主要竞争对手，但她说，她的公司和竞争对手之间的主要区别是，他们使用人来建立分类法，而 Luminoso 是完全自动化的(即使分类法发生变化和演变，它也会继续学习)。

Havasi 说，就分类法的创建和分析而言，这将人的因素排除在外，但不是全部。她认为，当你用这样的技术将人和计算机放在一起时，它会让人们更好地了解他们的客户(或者他们选择使用 Luminoso 技术)，同时自动化那些需要人们花很长时间来构建和处理的困难部分。
