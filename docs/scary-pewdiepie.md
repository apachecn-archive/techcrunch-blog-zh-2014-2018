# YouTube 将于下周首播其原创系列和电影，包括一部新的 PewDiePie 秀

YouTube 将于 2 月 10 日为 [YouTube Red](https://web.archive.org/web/20221007225010/https://www.youtube.com/red) 首播其第一部原创电影和系列。

该网站最近推出的订阅服务 [YouTube Red](https://web.archive.org/web/20221007225010/https://beta.techcrunch.com/2015/10/21/youtube-red/) 背后的主要吸引力一直是无广告观看和原创内容的承诺。不过，到目前为止，还没有原创内容，所以如果你订阅了 Red，你基本上要向谷歌支付 9.99 美元来移除广告(或许还有离线观看)。

不过，从下周开始，YouTube Red 也将最终推出其前三部独家电影，以及无处不在的斯堪的纳维亚视频博客作者 PewDiePie 的一个新系列。

 ****这些看起来都不像是 YouTube 的《纸牌屋》。相反，焦点显然是在知道菲利克斯·阿尔维德·乌尔夫·谢尔贝格和莉莉·辛格的人群中。如果你还不是订户，YouTube Red 上的第一套原创节目可能不会说服你成为订户。

“这些电影背后的多样化、充满活力的创作者已经在 YouTube 上建立了大量的观众，可以与许多有线电视节目相媲美，”YouTube 团队在今天的声明中写道。“通过 YouTube Red Originals，我们希望帮助他们讲述更大、更大胆的故事，让你——他们的粉丝——高兴。YouTube Red 是你的机会，支持你最喜欢的创作者迄今为止只梦想的激情项目。”

谷歌尚未透露有多少人注册了 YouTube Red。然而，仅 PewDiePie 就有超过 4100 万 YouTube 订户，其中一些人可能会订阅 YouTube Red，只是为了更多地看到他(或者他们都会因为“[出卖](https://web.archive.org/web/20221007225010/http://superfame.com/post/pewdiepie-defends-himself-against-redditor-who-calls-him-a-childish-shitty-sellout/)”而讨厌他——这种人口统计数据有点难说)。

YouTube 表示，它将在今年晚些时候向 Red 提供更多内容。
