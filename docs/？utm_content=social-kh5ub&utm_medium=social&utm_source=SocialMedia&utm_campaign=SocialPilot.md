# 人工智能加速初创公司 Xnor.ai 获得 260 万美元资金

Xnor.ai 的承诺和它的技术让我感到兴奋，这种技术大大降低了执行计算机视觉等复杂操作所需的计算能力。似乎我不是唯一的一个:这家刚刚从艾伦人工智能研究所(AI2)正式剥离的公司，已经从其母公司和 Madrona Venture Group 吸引了 260 万美元的种子资金。

你可以在我之前的文章中详细了解产品和过程的细节[，但要点是这样的:像物体和语音识别这样的机器学习模型是出了名的计算量大，这使得它们很难在更小、更弱的设备上实现。Xnor.ai 的研究人员使用一些数学技巧将计算负载减少一两个数量级——这很容易看到好处。](https://web.archive.org/web/20230128093950/https://techcrunch.com/2017/01/19/xnor-ai-frees-ai-from-the-prison-of-the-supercomputer/)

Madrona 的常务董事 Matt McIlwain 在一次新闻发布会上说:“想象一下，如果这种计算方式可以在你手中、手腕上或汽车上的设备上实现，那将会是什么样的情况。”我肯定他们现在都在拼命想象。“过去几年，机器学习和人工智能一直是我们的一个关键投资主题，将图像和语音识别等深度学习功能引入小型设备是一个巨大的挑战，”他在一篇公司博客文章中补充道。

McIlwain 将加入 AI2 首席执行官柳文欢·埃齐奥尼在 Xnor.ai 的董事会；领导最初项目的阿里·法尔哈迪将成为公司的首席执行官，穆罕默德·拉斯特加里是首席技术官。

新公司旨在促进其技术的商业应用(它还不是即插即用的)，但导致它的研究，像 AI2 的其他工作一样，是开源的。
