# 唐纳德·特朗普的密友小罗杰·斯通被推特停职

继深夜在推特上发表针对 CNN 主播的脏话连篇的长篇大论后，唐纳德·特朗普总统的长期密友和助手小罗杰·斯通(Roger Stone Jr .)已被无限期暂停社交媒体平台。

斯通([长得像狄更斯小说中的反派漫画人物)](https://web.archive.org/web/20230307142249/https://www.flickr.com/photos/tom-margie/31954617300/in/photolist-bxhjmy-7nYQjo-7nUWfg-bpvitj-bpvi3q-7RW4y1-7nYR1L-7nUUB6-6qpFeV-6qtPhA-BvUxMt-od8bk9-6qtTsY-6qtQSb-bCqdzg-bpvic1-bpviD7-bCqexg-5Exz8g-7od9HV-6qpKtZ-bpvigq-vdEdsG-bCqeDM-bCqdvn-6qtrRw-bpviNQ-6qpE1D-bpvizu-7h9f54-bCpRvF-7hdcQG-7h9dAz-7oS16K-7h9dkP-6qpicD-7h9dse-7hddmo-bpuweS-6qpTU6-R5qYAc-PYAzgU-QFHLrA-R2LFD5-Rg2RwM-R5rdVp-R5rjRn-Q2osUF-bCqdKg-7hdaXW)很可能是因为 CNN [的报道，特别顾问罗伯特·穆勒对总统竞选团队成员和俄罗斯特工在去年总统选举期间可能勾结的调查中第一次指控被批准。](https://web.archive.org/web/20230307142249/http://www.cnn.com/2017/10/27/politics/first-charges-mueller-investigation/index.html)

在周日发给政治的一封电子邮件中，斯通对禁令表示抗议，并表示他正在咨询律师关于对 Twitter 采取法律行动的事宜。

“这是一种奇怪的做生意的方式，也是技术左派审查和压制保守派声音的系统努力的一部分，”斯通在 POLITICO 引用的一封电子邮件中写道。

斯通电子邮件中的语言比周五晚上斯通对 CNN 主播唐·莱蒙的评论要严肃得多。据报道，斯通在社交媒体平台上称莱蒙为“一坨屎”和“人类粪便”，并写道，主持人必须“面对，羞辱，嘲笑和惩罚”。

![](img/a0c27ff36564d4d82e239c3f9cc8e58a.png)

自 20 世纪 70 年代与理查德·m·尼克森的 T2 总统连任委员会合作以来，斯通一直是共和党的政治顾问，这已经不是他第一次在推特上发表粗俗的攻击了。[他总是这样做](https://web.archive.org/web/20230307142249/https://www.washingtonpost.com/news/the-fix/wp/2017/10/29/cnn-broke-news-in-the-russia-probe-roger-stones-resulting-rant-got-him-yanked-off-twitter/?utm_term=.426fc3880198)。

对于消息服务来说，这个时间可能太长了，[因为用户已经公开指责该服务](https://web.archive.org/web/20230307142249/https://techcrunch.com/2017/10/12/twitters-abuse-problem-is-absolutely-a-failure-of-leadership/)对骚扰的[回应](https://web.archive.org/web/20230307142249/https://techcrunch.com/2017/10/13/twitter-ceo-promises-to-crack-down-on-hate-violence-and-harassment-with-more-aggressive-rules/)及其[武断的执行政策](https://web.archive.org/web/20230307142249/https://techcrunch.com/2017/10/12/twitter-breaks-silence-on-mcgowan-suspension/)。

在发给 Buzzfeed 的一封电子邮件中，Twitter 发言人提到 Twitter 围绕骚扰和虐待的政策是该账户可能被封的潜在原因。

但是不要太担心 Twitter 的责难。斯通至少还有另外两个[账户](https://web.archive.org/web/20230307142249/https://twitter.com/stonecoldtruth)在[上，他仍然在](https://web.archive.org/web/20230307142249/https://twitter.com/STONEFLIK)发微博。也许 Twitter 在检查其执行政策时可能需要考虑的另一个问题。
