# 两位全球投资者将在 Disrupt Berlin 讨论象征性销售

代币销售，也称为 ico，是早期现金的新常态。最初的设想是为支付、客户互动和点对点网络创建新的独特的轨道，现在令牌既是大多数公司不可或缺的一部分，也是资助一个伟大(或糟糕)想法的伟大方式。

在今年的 [Disrupt Berlin](https://web.archive.org/web/20221208054549/https://beta.techcrunch.com/event-info/disrupt-berlin-2017/?utm_medium=speakerpost5top&utm_campaign=disruptsf&utm_source=TC&unii-trigger-open=H9FCDY&ref=speakerpost5top) 大会上，来自 Neufund 的[佐伊·阿达莫维奇](https://web.archive.org/web/20221208054549/https://www.crunchbase.com/person/zoe-adamovicz)和来自 Consensys 的[卡维塔·古普塔将加入我们。这两位都是经验丰富的区块链投资者，拥有数百万可支配的资金，他们将讨论投资者应该如何在监管的暗礁中航行，代币销售如何改变风投与公司互动的方式，以及这些工具在未来将如何变化。](https://web.archive.org/web/20221208054549/https://www.bizjournals.com/newyork/news/2017/09/06/consensys-taps-gupta-to-lead-venture-fund.html)
