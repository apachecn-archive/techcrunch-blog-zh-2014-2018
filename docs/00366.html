SalesPredict 获得 410 万美元首轮融资

SalesPredict 是一家帮助销售团队使用预测分析寻找最佳客户的公司，今天宣布获得 410 万美元的首轮融资。

这笔交易的主要出资方是 Yandex 和 KGC 资本。

SalesPredict 首席执行官亚龙·扎凯(Yaron Zakai)表示，这家成立两年的公司是由他和首席技术官基拉·雷丁斯基(Kira Radinsky)共同创办的。两人之前在微软合作了几年，并希望创办一家公司，帮助销售人员利用数据来帮助他们做出更好的决策。

该产品的工作方式是通过查看过去交易的成功和失败作为起点，教导算法哪些交易最有可能成功，但它不会就此停止。他们还加入其他数据，查看开放网络和维基百科等网站，并使用谷歌和雅虎 API 收集有关个人和公司的信息。最后，他们查看公司员工的公开社交资料，并将其与商业名单和从 Alexa 和 ZoomInfo 购买的数据相结合。

“在一天结束时，我们会全面了解一家公司的人口统计数据，以及每个线索和账户的大量数据，”Zakai-Or 说。他补充说，“我们查看数据，找出决定(每个)客户成功的关键因素。”

根据这些信息，SalesPredict 根据潜在客户转化为销售的可能性给公司从 A 到 D 的等级，A 是最有可能的，D 是最不可能的。然后，它会提供关于向客户推销的最佳方式、他们可能的棘手问题、他们所处的垂直行业以及他们最可能的购买角色的信息，并根据所有这些信息为销售人员提供向最可能的客户销售正确产品的工具。

目前，该产品作为 Salesforce.com 的插件，可在 Salesforce 应用程序交换中获得。Zakai-Or 表示，Marketo 的版本即将推出。

在首轮融资之前，他们于去年 8 月获得了 100 万美元的种子资金。该公司总部位于以色列，但已经在旧金山开设了一个办事处，他们希望利用额外的资金来建立业务。

“第一个目标是扩大销售和营销。在过去的两年中，我们做了大量的工作来使产品成熟，并使其对我们的第一批客户有用。现在我们可以扩大规模，在营销方面做得更多，并获得市场成功的好处，”Zakai-or 告诉我。

当然，SalesPredict 的好处是为其不断增长的销售队伍提供自己产品的使用。
