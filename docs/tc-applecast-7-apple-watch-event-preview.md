# TC AppleCast 7: Apple Watch 活动预告

是时候了:苹果手表活动将于周一举行，我们将从库比蒂诺发布的可穿戴设备中找到所有亟待解决的问题的答案。根据我们的讨论，我们仍然不知道这些软件的价格，以及第三方软件能够提供多少。此外，我们还讨论了苹果可能在舞台上展示的其他东西，包括焕然一新的 Apple TV 或 Thunderbolt 显示器。

就我们目前对 Apple Watch 的了解而言，价格仍然是一大空白，因为有多少细节已经被[最后一刻的泄露和报道](https://web.archive.org/web/20230316064254/https://techcrunch.com/2015/03/06/the-apple-watch-is-time-saved/)填补了。我们也渴望看到苹果选择谁来帮助展示 Apple Watch 上第三方应用的可能性，特别是考虑到目前可用的开发工具的明显局限性。

苹果可能还会有一些惊喜——这是一个由该公司主办的非常罕见的特别活动，没有任何意想不到的事情发生。当然，我们会在现场报道，并且在接下来的几天里，我们还会围绕新发布的内容进行大量跟进和专题报道，敬请关注。
