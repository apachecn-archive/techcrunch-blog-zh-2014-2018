Nordic Web 推出了一项天使基金，旨在寻找该地区不为人知的初创企业

专门研究北欧地区新兴科技公司数据的研究和分析公司 Nordic Web 成立了一只新的天使基金。该基金将投资于丹麦、芬兰、冰岛、挪威和瑞典。

创始人尼尔·穆雷表示，“北欧网络天使基金”计划在未来 12 个月投资 10-15 家北欧初创公司。他声称这将使其成为该地区最活跃的基金之一。

支持这项事业的有限合伙人包括该地区的一些知名人士，如托马斯·麦德森-迈格达尔(23 岁)、琳达·利亚库斯(你好，鲁比)和约翰·布兰德(卡霍特！)以及来自欧洲各地的风险投资者，如马丁·米尼奥(Index Ventures)、克里斯托弗·扬茨(Point Nine Capital)和菲利普·默林(Philipp Moehring)和安迪·钟(Andy Chung)(均来自 AngelList Europe)。

默里说，他已经获得了 50 多位来自不同地区、不同性别和不同经验的 LPs 支持者。完整的名单是这里。

他在一份声明中表示:“通过利用基金投资者的专业技能和知识，我们可以提供无与伦比的支持，帮助早期北欧创业公司，并获得他们背后的整个社区。由于过去五年资本和对北欧的兴趣激增，我们现在开始看到早期企业家离开公司，开始自己创业。然而，由于机构投资者不知道他们的个人资料，他们通常仍不为人所知。该基金发现并支持这些公司。”

米尼奥说:“我已经看到了家族在支持和加速巴黎生态系统中的重要作用，我相信这也能在北欧产生类似的影响。”

默里告诉 TechCrunch，来自 AngelList 的菲利普·默林(Philipp Moehring)帮助筹集了资金，并负责所有的后端和法律事务。“我相信我是欧洲第一批利用这一点的人之一，”他补充道。
