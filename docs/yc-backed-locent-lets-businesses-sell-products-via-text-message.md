# YC 支持的 Locent 让企业通过短信销售产品

推出于 2015 年夏季的 Y Combinator 批次， [Locent](https://web.archive.org/web/20221001010353/https://locent.com/) 是一个完全由短信驱动的电子商务平台。

在线零售商面临的最大挑战之一是引导顾客通过漫长的结账流程。通过为企业提供一个定制的、支持短信的电话号码，Locent 将结账过程变成了一条短信。

以下是零售商如何使用这项服务:在创建一个账户并被分配一个自定义号码后，商店可以使用 Locent 的后端创建不同的商品，包括价格、照片和关键词。

当用户将该关键词发送到一个商店的号码上时，他们会自动获得一个支付表单的链接，以便通过手机完成交易。

![Create listing](img/e5d105149dc25a55ea94cbfe606c9803.png)

虽然 Locent 最终计划以实体和在线零售商为目标，但他们目前专注于帮助品牌和有影响力的人通过现有的营销渠道进行销售。

例如，该公司最近与 Snapchat 名人[肖恩度拉斯](https://web.archive.org/web/20221001010353/http://shonduras.com/)合作，推出了一个支持 Locent 的电话号码，可以回复粉丝订阅 Twitch 频道的链接。在 12 个小时的时间里，这个号码收到了粉丝们 8000 多条短信，他们立即收到了 Twitch 频道链接的回复。

Locent 的联合创始人 Matt Joseph 和 Ryan MacInnes 解释说，他们的服务让任何营销渠道都成为一个点击式销售点。

该公司还在考虑将 Locent 用作捐赠平台，特别是在 2016 年总统候选人开始筹款活动时。例如，政治家可以在一个活动的舞台上给我们他们的现场号码，用户可以发送短信立即收到捐款页面。

虽然该服务建立在 Twilio 和 Stripe 等已知平台的基础上，但 Locent 将这些技术结合在一起，成为一个易于所有企业所有者使用的平台。

为了方便起见，该公司向零售商收取 3%的佣金，这是在 Stripe 的费用(2.9%+30 美分)之外。

Locent 已经从 Y Combinator 和 Chaac Ventures 获得了 63 万美元的种子资金。
