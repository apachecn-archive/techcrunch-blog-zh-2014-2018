阿里巴巴集团领投 2640 万美元的 GPU 数据库提供商 SQream TechCrunch B 轮融资

原文：https://web.archive.org/web/https://techcrunch.com/2018/05/30/alibaba-group-leads-26-4m-series-b-in-gpu-database-provider-sqream/amp/

SQream 首席执行官兼联合创始人阿米·加尔

GPU 数据库开发商 SQream 在获得阿里巴巴集团领投的 2640 万美元 B 轮融资后，将加深对中国的关注。该轮投资者还包括 Hanaco Venture Capital、Sistema.vc、World Trade Ventures、Paradiso Ventures、Glory Ventures 和 Silvertech Ventures。

这家初创公司称，这笔融资是阿里巴巴的战略投资，使其融资总额略高于 4000 万美元。今年早些时候，SQream 和阿里云宣布了一项新协议，将从 10 月开始让阿里云用户访问 GPU 数据库。

在给 TechCrunch 的一份声明中，阿里巴巴数据库业务总监詹超群表示，“阿里云和 SQream 在 2 月份宣布了一项合作，这项投资加深了我们的关系，我们的目标是共同为各类企业提供最佳的云解决方案，帮助他们在这个数字时代取得成功。”

SQream 成立于 2010 年，总部位于特拉维夫。其名为 SQream DB 的 SQL 分析数据库使用 NVIDIA GPUs 中的数千个并行处理核心，使大公司能够更快更便宜地执行大数据分析(SQream 声称其客户可以“以 10%的成本分析多达 20 倍以上的数据，速度高达 100 倍”)。

SQream 联合创始人兼首席执行官 Ami Gal 告诉 TechCrunch，SQream 与 Kinetica 和 MapD 等其他 GPU 数据库的主要区别之一是它能够适应越来越大的数据量。Kinetica 和 MapD 使用内存存储，因此尽管它们可以极快地分析高达约 5tb 的数据，但它们的可扩展性有限。另一方面，创建 SQream 是为了处理高达数百 TB 的数据存储。

该公司的 B 轮资本用于为 SQream DB 增加新功能，并在专注于中国市场和其他地区的同时发展其销售、营销和交付团队。除了阿里云，SQream 在亚洲的其他新客户包括泰国移动运营商 AIS 和印度的企业消息服务 ACL Mobile。
