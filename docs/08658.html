配备触控ID传感器和有机发光二极管迷你屏幕的新款MacBook Pro即将上市TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/08/10/new-macbook-pro-with-touch-id-sensor-and-oled-mini-screen-is-coming-soon/?ncid=rss&utm_source=feedburner&utm_medium=feed&utm_campaign=Feed:+Techcrunch+(TechCrunch)&sr_share=twitter

准备好迎接MacBook Pro的大规模更新吧。苹果公司计划很快推出配备视网膜显示屏的13英寸和15英寸MacBook Pro笔记本电脑。根据来自彭博马克·古尔曼的最新报道，苹果正在制造一款全新的专业笔记本电脑，拥有许多新功能，比如触控ID传感器和键盘上方的有机发光二极管迷你屏幕。

如果你一直在关注MacBook Pro的传言，你应该知道这些传言并不新鲜。但是古尔曼证实了来自KGI证券和 9to5mac 的消息灵通的分析师郭明志已经说过的话:是的，新的MacBook Pro将会是一个大发布。

先说设备的外形。新的MacBook Pro应该更薄更轻，但不是像MacBook Air或12英寸retina MacBook那样的楔形。显示器周围的挡板应该更小，并且设备的整体框架应该更小。

新款MacBook Pro看起来不会像MacBook Air，这对于那些非常依赖现有MacBook Pro上众多端口的人来说是一个好信号——将有不止一个端口。苹果可以改用USB Type-C端口来取代USB端口、MagSafe端口甚至Thunderbolt端口，因为USB-C端口可以支持Thunderbolt协议。也许苹果会放弃HDMI端口和SD卡插槽以节省空间，也许不会。

在引擎盖下，苹果最有可能使用英特尔Skylake处理器，从而提高性能并改善能耗。你可以期待更好的Wi-Fi和蓝牙芯片。

但最重要的增加将是键盘正上方的一个狭窄的有机发光二极管显示屏。它应该用触摸屏取代物理功能键。这可能是这款MacBook Pro最令人惊讶的元素，因为没有人知道苹果会如何使用这一条。你可以在这个屏幕上控制笔记本电脑的基本功能，比如音量和媒体按钮。但苹果也可以利用这个屏幕来显示快捷方式和信息，这取决于你正在使用的应用程序。

苹果还计划在你的Mac电脑中集成一个触控ID传感器。这可以让你不用输入密码就能解锁你的笔记本电脑。更重要的是，触控ID传感器将带有一个安全区域，让你的Mac比以前更安全。它将允许应用程序开发人员将触控ID用于其他用途——例如，密码管理人员可以利用触控ID。最后，这将使苹果支付T2 MAC OS Sierra T3更加安全。

最近，许多人指出MacBook Pro是令人尴尬过时的。上一次重大更新是在2012年6月，为MacBook Pro系列带来了retina显示屏和更薄的设计。

但人们往往会忘记，一体式MacBook Pro是在2008年推出的，也是retina更新的四年前。苹果并不经常更换笔记本电脑，靠的是时不时的内部组件升级。

然而，Skylake处理器应该更早出现在MacBook Pro上。古尔曼表示，苹果不会在9月7日的iPhone发布会上推出新款MacBook Pro。我可以预见苹果会在10月份推出新款笔记本电脑。是时候了。
