配备触控 ID 传感器和有机发光二极管迷你屏幕的新款 MacBook Pro 即将上市 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/08/10/new-macbook-pro-with-touch-id-sensor-and-oled-mini-screen-is-coming-soon/?ncid=rss&utm_source=feedburner&utm_medium=feed&utm_campaign=Feed:+Techcrunch+(TechCrunch)&sr_share=twitter

准备好迎接 MacBook Pro 的大规模更新吧。苹果公司计划很快推出配备视网膜显示屏的 13 英寸和 15 英寸 MacBook Pro 笔记本电脑。根据来自彭博马克·古尔曼的最新报道，苹果正在制造一款全新的专业笔记本电脑，拥有许多新功能，比如触控 ID 传感器和键盘上方的有机发光二极管迷你屏幕。

如果你一直在关注 MacBook Pro 的传言，你应该知道这些传言并不新鲜。但是古尔曼证实了来自 KGI 证券和 9to5mac 的消息灵通的分析师郭明志已经说过的话:是的，新的 MacBook Pro 将会是一个大发布。

先说设备的外形。新的 MacBook Pro 应该更薄更轻，但不是像 MacBook Air 或 12 英寸 retina MacBook 那样的楔形。显示器周围的挡板应该更小，并且设备的整体框架应该更小。

新款 MacBook Pro 看起来不会像 MacBook Air，这对于那些非常依赖现有 MacBook Pro 上众多端口的人来说是一个好信号——将有不止一个端口。苹果可以改用 USB Type-C 端口来取代 USB 端口、MagSafe 端口甚至 Thunderbolt 端口，因为 USB-C 端口可以支持 Thunderbolt 协议。也许苹果会放弃 HDMI 端口和 SD 卡插槽以节省空间，也许不会。

在引擎盖下，苹果最有可能使用英特尔 Skylake 处理器，从而提高性能并改善能耗。你可以期待更好的 Wi-Fi 和蓝牙芯片。

但最重要的增加将是键盘正上方的一个狭窄的有机发光二极管显示屏。它应该用触摸屏取代物理功能键。这可能是这款 MacBook Pro 最令人惊讶的元素，因为没有人知道苹果会如何使用这一条。你可以在这个屏幕上控制笔记本电脑的基本功能，比如音量和媒体按钮。但苹果也可以利用这个屏幕来显示快捷方式和信息，这取决于你正在使用的应用程序。

苹果还计划在你的 Mac 电脑中集成一个触控 ID 传感器。这可以让你不用输入密码就能解锁你的笔记本电脑。更重要的是，触控 ID 传感器将带有一个安全区域，让你的 Mac 比以前更安全。它将允许应用程序开发人员将触控 ID 用于其他用途——例如，密码管理人员可以利用触控 ID。最后，这将使苹果支付 T2 MAC OS Sierra T3 更加安全。

最近，许多人指出 MacBook Pro 是令人尴尬过时的。上一次重大更新是在 2012 年 6 月，为 MacBook Pro 系列带来了 retina 显示屏和更薄的设计。

但人们往往会忘记，一体式 MacBook Pro 是在 2008 年推出的，也是 retina 更新的四年前。苹果并不经常更换笔记本电脑，靠的是时不时的内部组件升级。

然而，Skylake 处理器应该更早出现在 MacBook Pro 上。古尔曼表示，苹果不会在 9 月 7 日的 iPhone 发布会上推出新款 MacBook Pro。我可以预见苹果会在 10 月份推出新款笔记本电脑。是时候了。
