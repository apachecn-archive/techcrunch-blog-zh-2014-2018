# 优步拼车服务 UberPOP 现在在布鲁塞尔被禁

城市交通公司[优步](https://web.archive.org/web/20230404085205/https://www.uber.com/)在 Lyft 扩展到欧洲之前，正在缓慢但稳步地扩展其在欧洲的拼车服务 UberPOP。继在巴黎开设了第一家[之后，该公司最近在布鲁塞尔、柏林和巴塞罗那推出了 UberPOP。但据](https://web.archive.org/web/20230404085205/https://techcrunch.com/2014/02/04/look-out-lyft-uber-is-getting-serious-about-ride-sharing-with-uberpop-in-paris/) [De Tijd](https://web.archive.org/web/20230404085205/http://www.tijd.be/nieuws/ondernemingen_transport/Rechter_legt_taxidienst_Uber_aan_de_ketting.9489881-3087.art) 和 [Tech.eu](https://web.archive.org/web/20230404085205/http://tech.eu/news/brussels-court-bans-uber/) 报道，该公司在布鲁塞尔面临法律反对，因为 UberPOP 现在在布鲁塞尔被禁。每当一辆汽车被发现在其 UberPOP 服务下运营，优步将被罚款 13800 美元(€1 万美元)。

换句话说，该公司可能别无选择，只能停止在布鲁塞尔的运营。提醒一下，传统的黑车服务在布鲁塞尔已经被禁止了，这就是为什么优步选择了低调、监管友好的 UberPOP 服务。

Taxis Verts 在二月份首次对优步提出投诉。该公司抱怨不公平竞争。由于优步可以自己定价，UberPOP 比传统的出租车便宜 20%左右。

提醒一下，有了 UberPOP，每个人都可以成为司机。这是一种更便宜的拼车服务，内置在优步的应用程序中。这样，公司可以允许更小的汽车。

要成为一名司机，你需要一个驾驶执照，并符合年龄要求。你还必须通过背景调查。司机自带车。作为顾客，你可以期待一辆大众 Polo 或一辆雪铁龙 C3(如上图)和瓶装水。

截至写这篇文章时，UberPOP 仍在布鲁塞尔工作。但应该很快就会停止。然后，优步可能会像西雅图一样发起请愿。总之，很可能还没有结束。
