# Teradyne 以 2 . 72 亿美元收购机器人制造商 MiR 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/04/26/teradyne-snatches-up-robot-maker-mir-in-272m-deal/>

[Teradyne](https://web.archive.org/web/20221025230715/http://www.teradyne.com/) ，一家为工业应用提供自动化测试设备的公司，[以令人瞠目的 1.48 亿美元收购了丹麦机器人公司 MiR](https://web.archive.org/web/20221025230715/https://globenewswire.com/news-release/2018/04/25/1486986/0/en/Teradyne-and-Mobile-Industrial-Robots-MiR-Announce-Teradyne-s-Acquisition-of-MiR-Leader-in-Collaborative-Autonomous-Mobile-Industrial-Robots.html) ，在达到业绩目标后，还有 1.24 亿美元摆在桌面上。

[MiR](https://web.archive.org/web/20221025230715/http://www.mobile-industrial-robots.com/en/) ，尽管小写字母“I”代表移动工业机器人，你可能已经猜到了。该公司成立于 2013 年，一直稳步增长，并在 2017 年取得了巨大的成就，在其最新机器人 [MiR200](https://web.archive.org/web/20221025230715/http://www.mobile-industrial-robots.com/en/products/mir200/) 获得客户高度评价后，其收入增长了两倍，达到 1200 万美元。

和平号的机器人是仓库类型的，有轮子的小机器人，可以举起和拉动托盘、箱子等等。它们看起来有点像《星球大战》电影中那些总是被踩在脚下的小家伙。这对 Teradyne 来说是一个自然的选择，特别是后者最近在 2015 年以 3.5 亿美元的价格收购了知名的通用机器人。

测试大量的电子产品和组件可能是一个枯燥的行业，但它是一个蓬勃发展的行业，因为测试速度快的公司出货更快。无论何时，只要在流程中能够提高效率，无论是仓库物流还是在敏感程序中协助专业人员，都可以肯定有公司愿意为此买单。

Teradyne 还指出([机器人报告指出](https://web.archive.org/web/20221025230715/https://www.therobotreport.com/teradyne-acquires-mir-272m/))，两家公司都对机器人以及它们如何互动采取了现代方法，必须由人来训练——机器人专家必须仔细编写这些东西的旧范式不太适用，UR 和 MiR 都有足够的前瞻性来改善这一痛点。

当然，计划是将和平号成功的技术带到全球，希望在更大的规模上重现它的成功。

“我的主要目标是让我们的移动机器人走向全世界，”MiR CSO 和创始人 Niels Jul Jacobsen 在宣布收购的新闻稿中说。“有了 Teradyne 作为所有者，我们将拥有强大的后盾来确保 MiR 在全球市场的持续增长。”
