# 大众汽车和 Mobileye 团队通过众包帮助自动驾驶汽车“看见”

大众和 Mobileye 已经[签署了一项协议](https://web.archive.org/web/20221207093830/https://www.volkswagen-media-services.com/en/detailpage/-/detail/Swarm-data-paves-way-for-autonomous-driving-Volkswagen-and-Mobileye-sign-agreement/view/4560918/7a5bbec13158edd433c6630f5ac445da?p_p_auth=5xCpioZ4)，从 2018 年开始，将 Mobileye 的道路体验管理(REM)地图服务集成到大众品牌的车辆中。供应商 Mobileye 的服务从配备该产品的车辆上获取实时路况报告，以帮助提供智能车队功能，供自动驾驶汽车使用，确保联网智能汽车能够更好地在真实道路上导航。

配备 REM 的汽车可以跟踪来自前置摄像头的信息，包括车道标志和其他道路信息，然后将数据发送到云端，在那里收集数据并反馈到高清导航地图，以确保这些数据在任何时候都尽可能准确地反映情况。Mobileye 和大众汽车打算通过这项协议为 REM 平台上的全行业合作铺平道路，该平台将使用来自一系列制造商汽车的众包数据来构建高清地图，然后帮助改善所有参与者的自动驾驶和高级驾驶辅助功能。

新协议意味着大众车型将在 2018 年开始在新车型中整合 REM 系统，这将导致匿名化的数据收集，不仅可以提供对路况的洞察，还可以提供对停车位能力以及交通和拥堵信息的洞察。

在我试驾德尔福的自动驾驶系统期间，我实际上有机会体验了 [Mobileye 的 REM 系统在拉斯维加斯 CES](https://web.archive.org/web/20221207093830/https://beta.techcrunch.com/2017/01/04/delphis-self-driving-car-takes-us-on-a-stress-free-tour-of-las-vegas/) 上的工作，结果令人印象深刻，尽管很明显很难确定该系统的任何一个组件对其整体熟练程度负责。尽管如此，智能联网汽车协同工作的“智能群体”概念正在加速发展，可能成为我们无人驾驶未来的一个巨大组成部分。
