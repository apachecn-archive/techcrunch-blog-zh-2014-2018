# UberPOOL 现在会建议更好的下车地点，以节省每个人的时间

优步在 UberPOOL 上增加了一项新功能，有望帮助司机和乘客加快行程。UberPOOL 骑手现在将在途中获得应用程序中弹出的下车地点建议，让他们知道什么时候跳下车步行比坐在车流中更容易。

备用下车点将根据有关您的路线和实时交通信息的数据通过算法生成，通知还将向您显示通过采纳建议而不是保持当前路线大约可以节省多少时间。

如果你对优步提供的服务满意，骑手可以点击蓝色小框弹出窗口，显示地图上节省的时间，它会将更新的路线信息发送给拼车司机。优步强调这种行为的好公民的一面，因为参与将帮助你的同伴节省时间。

萨特说“地狱是其他人”，当你掷骰子决定与一群陌生人共乘一辆陌生人驾驶的车时，这句话尤其正确。像这样的功能可以缓解共享乘车中的一些妥协，应该会有所帮助，尽管增加了摩擦，但我很想知道乘客实际上多久会利用这一功能。
