# Jawbone 专利使用智能手机、平板电脑和可穿戴设备测量体脂百分比

设备制造商越来越多地寻找推动健康跟踪市场发展的方法，设备制造商 Jawbone 的一项新专利申请为接下来可能发生的事情提供了线索。Jawbone 最近有一项由 [USPTO](https://web.archive.org/web/20230131011308/http://appft1.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PG01&p=1&u=/netahtml/PTO/srchnum.html&r=1&f=G&l=50&s1=20140121564.PGNR.) 公布的专利申请，描述了一种利用现有设备测量体脂百分比的方法，包括智能手机和平板电脑，或者像 UP24 这样的腕带，使用所述硬件已经携带的传感器。

该专利描述了该系统如何使用显示器、加速度计、陀螺仪、麦克风、GPS 等东西来告诉用户如何最好地放置该设备以测量身体脂肪百分比。它将通过使用机载振动马达广播一个“脉冲”并测量返回的响应来做到这一点；简而言之，手机、平板电脑或其他小工具会晃动你的脂肪，并测量晃动的频率，为你提供身体脂肪百分比的估计值。

Jawbone 的专利将与设备上的一个应用程序协同工作，该应用程序可以构成一个全面的健康跟踪应用程序的一部分，或者是一个专门用于测量身体脂肪百分比的应用程序。这将代表一种由设备跟踪的新方法——移动健康监测平台计算身体脂肪的传统方法涉及使用专用的外部设备，例如 Withings 智能身体分析仪数字秤。

这项专利归功于阿萨·拉斯金，他是 [Massive Health 的创始人，该公司于 2013 年被 Jawbone 收购](https://web.archive.org/web/20230131011308/https://techcrunch.com/2013/02/04/jawbone-acquires-mobile-health-startup-massive-health-for-tens-of-millions/)，目的是培养其专注于设计的人才。
