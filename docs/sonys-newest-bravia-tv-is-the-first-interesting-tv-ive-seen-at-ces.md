# 索尼最新的 Bravia 电视是我在 CES 上看到的第一台有趣的电视

CES 上有这么多该死的电视，很少有一台能引起我的注意——但索尼刚刚宣布了一款新电视，用于他们的旗舰 Bravia 系列，实际上看起来很棒。

这是一台 4K·HDR OLED 电视，索尼称之为 A1E 系列。这款电视采用边对边设计，没有支架，这意味着它直接坐在地上，靠着一条后腿。

它也可以放在桌子上，但我认为它看起来很酷的一部分原因是它直接靠在地板上。它所依靠的后支架是电视处理器和其他重要部件的所在地。

这种边到边的设计意味着电视周围没有扬声器。但索尼表示，声音将来自屏幕本身，而不是仅仅将扬声器放在后面——使用一种称为声学表面的新技术。
