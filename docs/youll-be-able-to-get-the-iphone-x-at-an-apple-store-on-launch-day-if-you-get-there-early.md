# 如果你早到的话，你可以在苹果商店买到 iPhone X

早期报道指出 iPhone X 在发布会上缺货，但是苹果今天早上发布了[一份新闻稿](https://web.archive.org/web/20230316191215/https://www.apple.com/newsroom/2017/10/iphone-x-available-for-pre-order-on-friday-october-27/)[向顾客](https://web.archive.org/web/20230316191215/https://www.macrumors.com/2017/10/24/iphone-x-walk-ins-launch-day/)保证，这款超高端手机实际上将在发布会当天提供给上门顾客。尽管他们被“鼓励早到”换句话说，如果你想要一个，排队。

最后一点似乎让 1000 美元及以上的手机在发布时将相当短缺的故事更加可信。正如我们上周所提到的，该公司目前预计在 11 月 3 日发布之日将有大约 200 万至 300 万台可用。

X 为 iPhone 系列带来了几乎无边框的显示屏，以及包含双光学图像稳定功能的增强型摄像头。Face ID 推出了前置摄像头，带来了用眼神解锁手机的能力，当然还有 [Animoji](https://web.archive.org/web/20230316191215/https://techcrunch.com/2017/09/12/iphone-x-animoji/) 。

缺货被归咎于供应链问题，因为这款手机的新设计带来了一些独特的制造挑战。当然，当这款手机在太平洋时间本周五 12:01 登陆苹果网站时，不那么冲动的消费者被鼓励预购。

这款手机的需求可能会受到许多因素的影响，包括价格和该公司这次错开发布的不同寻常的决定。此时，iPhone 8 已经上市一个月了，可能会对潜在销量造成一定影响，因为该公司已经解决了上述高端设备的生产问题。

iPhone X 正在苹果商店限量发售，这可能是你得到一部 iPhone 的最快方法，当然前提是你的父亲不在 T21 工作。
