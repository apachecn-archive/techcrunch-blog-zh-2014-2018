# Raspberry Pi 的 PC 和 Mac 版 Pixel 为旧电脑注入了新的活力

Pixel 是一个轻量级的操作系统，具有干净的桌面用户界面，由价格实惠、友好的 Raspberry Pi 35 美元计算机背后的基金会于 9 月发布，现在它可以为可能想将旧硬件投入更新服务的 PC 或 Mac 用户提供。基于 Linux 的轻量级操作系统现在以可下载映像的形式出现，您可以刻录到 DVD 或加载到 USB 驱动器上，使您可以在任何具有至少 512 MB RAM 和 x86 处理器的 Mac 或 PC 上直接启动到 Pixel 环境。

这意味着，如果你有一台旧笔记本电脑，比如 OG 塑料 MacBook 或一台老式 ThinkPad，你可以通过一个现代操作系统让它再次运行，完成基本任务，包括通过预装的 Chromium 浏览网页。Pixel 中的其他软件包括一套精选的生产力软件和编程工具，它们都是基于 Debian 开发的，Debian 本身也有大量的免费软件。

Raspberry Pi 在本月发行了一张与*MagPi*杂志一起发行的物理启动 DVD，但是你可以通过[在这里下载镜像](https://web.archive.org/web/20230316041156/http://downloads.raspberrypi.org/pixel_x86/images/pixel_x86-2016-12-13/2016-12-13-pixel-x86-jessie.iso)并将其刻录到你自己的磁盘或驱动器上来非常容易地制作你自己的 DVD。基金会[推荐 Etcher](https://web.archive.org/web/20230316041156/https://etcher.io/) 创建可引导驱动器，事实上，这使得整个过程非常用户友好。

这是一个原型构建，所以预计会遇到一些错误和硬件兼容性问题。Raspberry Pi 表示，现代 MAC 将很难识别该驱动器是否可引导，所以这是一个已知的问题。但该组织表示，他们首先创建这个 Mac 和 PC 版本是很重要的，因为他们认为学校能够在旧硬件上安装操作系统是有用的，因此对于在家使用 Raspberry Pis 的学生来说，这是一个跨家庭和学校的环境，因为他们真的想把 Pixel 变成最好的桌面操作系统，不管硬件如何。

标准的免责声明适用于:您使用这一风险，以及您的旧计算机上的文件的风险。如果你想在本地保存文件，你需要使用 USB 驱动器进行写入。但这听起来真的是一种非常低风险的方式，可以让你可能还在使用的旧硬件起死回生。
