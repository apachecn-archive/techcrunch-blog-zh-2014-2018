# Univision 赢得收购 Gawker Media 的拍卖

Univision 赢得了收购 Gawker Media 网站和业务的拍卖。

Recode 爆料称，收购价格为 1.35 亿美元，将覆盖 Gawker 的所有七个网站。该公司创始人尼克·登顿在给记者的一封电子邮件中证实了这笔交易:

> Gawker 媒体集团今晚同意将我们的业务和流行品牌出售给 Univision，Univision 是美国最大的媒体公司之一，正在迅速组建面向千禧一代和多元文化受众的领先数字媒体集团。我很高兴我们的员工受到了保护，并将在新的所有权下继续工作——从针对公司的法律诉讼中解脱出来。我们不可能找到一个更致力于充满活力的新闻业的收购者。

Univision 以其西班牙语电视网络而闻名，它不是最明显的收购者，但它也拥有 [Fusion](https://web.archive.org/web/20230118114855/https://fusion.net/) ，最近[投资了洋葱](https://web.archive.org/web/20230118114855/http://www.nytimes.com/2016/01/20/business/media/univision-buying-large-stake-in-the-onion.html)。(顺便说一下，Gawker 贴出了一些所谓的[内部截图，这些截图显示 Fusion](https://web.archive.org/web/20230118114855/http://tktk.gawker.com/more-people-work-at-fusion-than-are-reading-its-most-po-1708507965) 的一些流量数据并不突出。)

此次出售是高客在诉讼中失败的结果，诉讼方是摔跤手胡克·霍根·特里·博莱阿，他在高客发布了霍根的性爱录像片段后起诉了该公司。霍根的诉讼部分是由风险投资家彼得·泰尔资助的，他最近为《纽约时报》写了一篇专栏文章，认为这个案子是关于保护隐私和同性恋权利的。

佛罗里达州的一个陪审团支持霍根，判给他 1.4 亿美元。该公司正在对这一决定提出上诉，但法官驳回了 Gawker 推迟付款的请求。因此，该公司于 6 月申请破产并被拍卖。

[Ziff Davis 出价 9000 万美元，基本上为拍卖定下了底价。](https://web.archive.org/web/20230118114855/http://adage.com/article/media/gawker-prepares-auction-90m-opening-bid-ziff-davis/304509/)[据报道，Univision 是唯一的另一个投标人](https://web.archive.org/web/20230118114855/http://www.bloomberg.com/news/articles/2016-08-16/ziff-davis-univision-said-to-be-only-bidders-for-gawker)——little things 也对 Jezebel 提出了 1000 万美元的报价，但这已经过了截止日期。

该出售仍需得到破产法院法官的批准。
