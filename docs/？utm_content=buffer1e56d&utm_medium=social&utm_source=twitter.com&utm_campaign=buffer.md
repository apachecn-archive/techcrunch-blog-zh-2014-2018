# Twitter 收购长期合作伙伴和社交数据提供商 Gnip

Twitter 已经收购了社交数据提供商 Gnip，Gnip 长期以来一直提供对 Twitter 公共数据的访问，并通过一系列产品和合作声明加深了与社交网络的关系。此次收购将导致 Twitter 向 Gnip 的客户群提供更多的数据，与 Gnip 的现有客户关系将保持不变。

Gnip 提供对 Twitter 历史数据和完整 Twitter 数据的访问，但它也提供 API，提供来自其他社交网络的数据，包括 Reddit、Instagram、Tumblr、Bitly 等。Gnip 表示，作为公司的一部分与 Twitter 合作将让它“走得更快、更深”，他们计划扩大支持，以包括不同行业中更广泛的潜在用例。

Twitter 将 Gnip 内部化是有意义的，因为这意味着它对自己内容的货币化有更直接的控制权。Gnip 向其他公司出售其批量数据的访问权限，现在 Twitter 将能够在没有中间人的情况下做同样的事情。这也意味着，在创收方面，Twitter 现在有了更多广告之外的选择——该公司可以更容易地直接向品牌出售关于其网络上已经发生的对话的数据，而不是说服他们为广告空间付费，以试图启动这些对话。对于与 Gnip 在同一领域工作的其他公司来说，这可能是个坏消息，包括 DataSift。

目前看来，Gnip 将继续以自己的品牌运营，办公室和团队肯定会留在他们目前所在的科罗拉多州博尔德市。至于它与其他网络提供商在公共数据方面的关系，Twitter 在博客中宣布，它计划尊重当前的客户合同，但当然，这些提供商可能不会愿意向现在相当于竞争对手的东西提供消防水管。
