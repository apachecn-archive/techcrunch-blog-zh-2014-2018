你现在可以从 HBO 的硅谷租到 Erlich 的 Aviato 汽车

迈克·詹郅的硅谷面临着一个严峻的挑战，因为它必须不断超越自己，以恰当地讽刺真实的硅谷，在那里疯狂的裤子垃圾一直在发生。因此，看到节目中的东西流入真正的硅谷创业公司的世界并不奇怪，就像 Airbnb 汽车创业公司 Turo 今天开始的新促销一样。

对于傻瓜来说，图罗现在出租了埃尔利希·巴赫曼的汽车，这位伪成功的企业家在节目开始前就声名鹊起，这是他的初创公司 Aviato 的一个体面退出。他的 Aviato 品牌的福特 Escape 现在是一个真实的东西，你可以在 Turo 的点对点汽车租赁网络上租到，这辆 2006 年的老爷车每天起价 49 美元。

所以只是澄清一下，因为再次，当涉及到像 HBO 的山谷喜剧这样的事情时，现实和模仿之间的界限非常薄，这是一个你可以实际预订的车辆的真实汽车列表。Turo 上有一个关于“Erlich B .”的真实简介，其中一个人的评分很像商业内幕的 Kif Leswing 。
