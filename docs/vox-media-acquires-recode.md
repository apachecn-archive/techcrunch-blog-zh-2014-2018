# Vox Media 收购 Re/code 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/05/26/vox-media-acquires-recode/>

数字媒体集团 [Vox Media](https://web.archive.org/web/20221219214950/http://www.voxmedia.com/) 收购了由著名科技记者卡拉·斯威舍和沃尔特·莫斯伯格领导的科技行业新闻刊物 [Re/code](https://web.archive.org/web/20221219214950/http://www.recode.net/) 。这条新闻是今天由纽约时报首先报道的。

据报道，此次交易为全股票交易，交易规模的财务细节尚未披露。

Re/code 于 2014 年 1 月由 Swisher 和 Mossberg 以及他们在 AllThingsD 的一些同事推出[，all thingsd 是华尔街日报*的前科技新闻垂直媒体*，经过几个月的谈判后于 2013 年 9 月作为独立实体](https://web.archive.org/web/20221219214950/http://www.theverge.com/2014/1/2/5264506/walt-mossberg-and-kara-swisher-launch-recode-code-conference)[分离出来。据《纽约时报》报道，Re/code 有 44 名全职员工，预计他们都将作为收购的一部分加入 Vox。](https://web.archive.org/web/20221219214950/http://www.theverge.com/2013/9/19/4749982/walt-mossberg-leaving-wall-street-journal-kara-swisher-all-things-d)

同样值得一提的是，Vox Media 拥有 the Verge，因此这笔交易将两家强大的技术出版物——关注点和报道风格完全不同——聚集在一个屋檐下。在接下来的几个月里，看他们如何成长和互补将会非常有趣。

这只是一系列关于科技媒体世界的重大新闻故事中的最新一个，近年来，科技媒体变得更加拥挤和泡沫化，与其覆盖的蓬勃发展的行业相称。资金雄厚的媒体新贵，如 Vox 和 Buzzfeed，正在雇用敏锐的报道团队，寻求独特的硅谷报道，以及该领域稍微年长的参与者，如 TechCrunch、Ars Technica、Wired、VentureBeat 等(如果我没有在这里包括你的出版物，请不要生我的气。我们真的太多了。)与此同时，像《纽约时报》、《华尔街日报》、《彭博》和《美国消费者新闻与商业频道》这样的老牌报纸现在也投入了大量资源报道科技领域。这是一个非常疯狂的时期。

所以，如果你喜欢科技新闻，你很幸运——在可预见的未来，科技新闻似乎不会短缺。
