# 观看约翰·奥利弗关于网络中立的精彩、简明的初级读本

【YouTube https://www.youtube.com/watch?v = fpboearrhyu]

如果你是 TechCrunch 的定期读者，那么你很可能对网络中立性这个问题了如指掌。然而，如果你不是，那你真可耻。开玩笑的。正如 HBO 的《今夜最后一周》主持人(两届嘎吱嘎吱主持人)约翰·奥利弗昨晚在[一个振奋人心的行动号召中所说，“有线电视公司已经发现了美国的伟大真理:如果你想做一些邪恶的事情，就把它放在一些无聊的事情中。”](https://web.archive.org/web/20221207231421/https://www.youtube.com/watch?v=fpbOEoRrHyU&sns=tw)

问题的一部分是术语网络中立性本身和任何关于这个问题的政府听证会。

奥利弗说:“在英语中，只有两个词更让人厌烦，那就是‘以斯汀为特色’。”“我宁愿读托马斯·弗里德曼的书，也不愿听那场听证会。我宁愿听一对码头工人告诉我它做的那个奇怪的梦。”

但是，奥利弗指出，尽管它的名字令人打鼾，网络中立“实际上非常重要。本质上，这意味着所有数据都必须得到平等对待，不管是谁创建的。这就是为什么互联网是一个奇怪的公平竞争环境，初创公司可以取代老牌公司。”通过创建双层系统结束网络中立“将允许大公司通过购买进入快车道，让所有人都在慢车道上。”
