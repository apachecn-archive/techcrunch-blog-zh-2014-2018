Syte.ai 的新 API 让更多在线时尚零售商可以使用可视化搜索技术

原文：https://web.archive.org/web/https://techcrunch.com/2017/12/07/syte-ais-new-api-makes-visual-search-accessible-to-more-online-fashion-retailers/

Syte.ai 的新 API 让更多在线时尚零售商可以使用视觉搜索

今年早些时候，为时尚品牌开发视觉搜索技术的 syte . ai,从包括亚洲顶级科技公司 NHN、Line Corp .和 Naver 在内的投资者那里筹集了 800 万美元。现在，它推出了一个新的 API，使更多的电子商务网站可以使用可视化搜索。

这项名为“面向所有人的视觉搜索”的功能可以在 24 小时内集成到零售网站或应用程序中，让购物者上传保存在手机上的照片，如 Instagram 上的截图，以找到类似的销售产品。它基于与 Syte.ai 针对大型时尚品牌和出版商的搜索工具相同的技术，当顾客将光标悬停在图像的一部分上时，该工具会向顾客显示相关商品(Syte.ai 的一些客户包括马克斯&斯潘塞和美国柯尔百货公司)。

Syte.ai 的联合创始人兼首席营销官 Lihi Pinto Fryman 表示，一旦它将一个品牌的产品提要编入索引，通过添加一行 HTML，不到一天的时间就可以将所有产品的可视化搜索添加到网站的搜索栏中。客户根据可能使用的图像匹配数量支付每月许可费。

Fryman 说，相比之下，其他视觉搜索工具可能需要几周或几个月的时间来实现和训练，以适应品牌的产品提要，而且 Syte.ai 的目标客户通常负担不起，包括向千禧一代和青少年销售的较小零售商。

Facebook Messenger 和 Line 用户可以通过向其聊天机器人 Syte Inspire 发送图像来试用 Syte.ai 的技术。
