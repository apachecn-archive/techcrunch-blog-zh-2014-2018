# 烤架机器人是一个清理你的烤架的机器人

懒惰的烧烤者，欢呼吧。多亏了[烤架机器人](https://web.archive.org/web/20221025222515/http://grillbots.com/)，你可能再也不用清理你的烤架了。

今天在 CES 上，我们发现了 Grillbot，它听起来就像是一个机器人，你可以扑通一声自动清洁你的脏烤架。售价 129 美元，这款机器人有三根可更换的金属刷毛，可以帮助机器人清洁和绕着你的烤架移动。

该设备有一个可充电电池，三个电动马达，一个 LCD 警报和计时器，所以当它做它的事情时，你可以离开它。

现在困难来了。你需要等到你的烤架达到大约 200 度的温度后才能启动吸尘器。由于正常的烤架温度在 400 度左右，你必须等烤架冷却一会儿才能开始清洗。然而，Grillbot 确实有一个警报，如果你把它放在太热的烤架上，它就会发出声音。

最终，Grillbot 不会像好的老肘子油脂一样有效，但可能是给你自己或你生活中任何懒惰的烧烤者的一个整洁的礼物。
