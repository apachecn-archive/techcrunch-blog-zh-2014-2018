# 戒指是真的 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2015/01/04/the-ring-is-real/>

吉田拓郎和他的团队[用](https://web.archive.org/web/20230129235851/https://www.kickstarter.com/projects/1761670738/ring-shortcut-everything/posts)[戒指](https://web.archive.org/web/20230129235851/https://techcrunch.com/2014/02/28/the-ring-input-device-puts-gesture-control-and-home-automation-on-your-finger/)引爆 Kickstarter 已经快一年了，这是一款可穿戴设备，使用简单的类似魔杖的手势来控制家中的小工具。

承诺比比皆是:你只需动动手指就能控制家里的灯。你可以在你的 iPhone 上更换歌曲，打开音乐，甚至拍照。

然后，就像许多硬件项目一样，[Logbar](https://web.archive.org/web/20230129235851/http://logbar.jp/ring/en/)(Ring 背后的公司)遇到了[一些障碍](https://web.archive.org/web/20230129235851/https://www.techinasia.com/ring-kickstarter-wearable-ceo-responds-to-criticism/)。由于团队致力于使设备更小、更(真正)耐磨，制造被推迟，其他延迟使产品比计划落后几个月。事实上，早期支持者在今年 12 月才收到他们的单位。

但尽管如此，在我看来，戒指仍然是一个有吸引力的可穿戴设备。我们在 CES 上见到了吉田，并亲眼看到了这款设备。并不完美，感觉还是有点大(虽然比第一次迭代小很多)。然而，Logbar 团队至少对设计需要结合功能的方式有着坚定的理解。

如今的可穿戴设备很难看。是真的。Apple Watch 可能会削弱这种说法，英特尔的 MICA 肯定会试图反对这种丑陋的颗粒，但 Logbar 的 Ring 标志着第一批公开上市的设备之一，实际上看起来不错。

除此之外，它似乎是有效的。吉田解释说，Ring 需要一个学习曲线，就像在键盘上打字一样。但是一旦你学会了如何使用它，它似乎工作得很好。吉田在他的展示中展示了 100%的手势准确性。

这款戒指现在售价 269 美元，新推出的 Ring Hub 可以让你通过 Wifi 连接你的家用电器(比如灯)，很快将在今年 3 月以约 70 美元的价格上市。
