也是时候了。

Giphy ，四岁的 gif 搜索引擎，将最终开始测试货币化。

据接近该公司的消息人士透露，Giphy 将开始在消息体验中测试赞助商 gif。这意味着搜索 GIF 的用户可以在 messaging 选项卡中获得赞助商 GIF。

这里有一些明显的用例:搜索“周一”或“早上”可能会出现星巴克的 GIF。但是品牌也有机会，特别是电影和电视节目(占 Giphy 内容的很大一部分)，将他们现有的内容融入到人们的信息中。

从结构上来说，这与谷歌对搜索词的处理没有太大区别。如果你搜索“沃尔玛”，你很可能会看到沃尔玛和塔吉特百货的赞助列表。

然而，使用 Giphy，搜索很少是针对特定品牌，而是基于某些行为、反应或情绪。在即将推出的赞助信息产品中，搜索“Wooo”可能会出现有人倒 Jose Cuervo 饮料的 GIF 图片。搜索“干杯”可能会显示人们一起碰杯两个百威啤酒瓶子。(如果网飞不买“chill”的广告，我会非常失望。)

此外，Giphy 上的赞助内容的功能类似于 Snapchat 的赞助面部过滤器。即使它包含品牌内容，但它来自朋友的事实使它值得观看，也更容易与人联系。

同一消息来源还称，Giphy 目前在 API 和网站之间拥有 2 亿每日活跃用户，网站每月活跃用户约为 2.5 亿，Giphy 拥有并运营集成。这意味着 Giphy 的大多数用户每天都在与该服务进行交互。

我的消息来源和 Giphy 都没有确认广告产品的发布时间。也就是说，Giphy 首席运营官亚当莱博森证实，该公司已将重点从增长转向构建广告产品，但没有增加更多关于这些产品可能会是什么样子的细节。

“与 GIPHY 工作室合作，我们已经进行了一年多的创意合作，”Leibsohn 说。“这项工作有助于我们围绕广告产品的外观进行思考和探索。”

我还问 Leibsohn 哪些品牌会率先试用 Giphy 的广告产品。

“我们已经与世界领先的内容、娱乐、技术和广告品牌密切合作，”Leibsohn 说。“当我们准备推出广告产品时，自然会与现有的合作伙伴合作。”

就测试而言，目前还不清楚 Giphy 将首先涉足哪个平台。也不清楚在 Giphy 自己的网站和应用程序之外的第三方平台上提供赞助内容会采取什么样的商业模式。

Slack、iMessage、Facebook Messenger 和 Twitter 都是 Giphy 的大型平台，但它们可能有点太大，无法进行初始测试。请记住，Giphy 是同一家公司，它推出一个又一个独立的应用程序，只是为了了解哪些应用程序适合用户，哪些不适合用户。

在我看来，从战略上讲，Giphy 开始测试赞助内容的最佳地点是 Tinder。Tinder 与 Giphy 的集成获得了很多关注——事实上，去年25 个最受欢迎的 Giphy GIFs 之一爬上了 Tinder 用户的图表。

虽然 Tinder 和 Giphy 是(用户参与)天堂中的一对，但它不像在 Twitter 这样的公共平台上推出广告产品那样高风险，也不像 Slack 的大部分专业环境那样棘手或不可预测。

Tinder 没有立即回应置评请求。

在创收方面，Giphy 有很多选择。除了在消息中销售赞助内容，该公司还看到了 Giphy 网站的大量流量。事实上，吉菲说超过 50%的人来到公司网站是为了浏览。

这也为通过搜索词在网站上销售原生广告提供了机会。

此外，Giphy 目前正在为奥斯卡、金球奖等大型活动“直播 gif”。目前，Giphy 的团队免费做这项工作，但你可以看到他们最终可能会围绕这些活动开始签署许可协议。这还不包括 Giphy 为大型广播和有线电视网制作的 gif 旧内容——想想《满屋》和《宋飞正传》。

虽然许多人认为 P2P 原生内容将是广告未来的重要组成部分，但还不清楚这种成功的版本实际上会是什么样子。但是，在相对较短的时间内，Giphy 已经成为我们沟通方式中根深蒂固的一部分，无论是在 iMessage、Twitter、Tinder 还是 Slack 上。

Giphy 在 2016 年 10 月获得了7200 万美元，使融资总额达到 1.5 亿美元，据报道估值为 6 亿美元。
