# 顶级风险投资人布莱恩·欧玛利谈下一波“点播”浪潮 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/08/27/top-vc-brian-o-malley-on-the-next-on-demand-wave/>

布莱恩·欧玛利擅长他的工作。事实上，他足够优秀，以至于两年前，Accel Partners [将他从他以前的公司 Battery Ventures 中挖了出来，这样他就可以为其团队效力了。](https://web.archive.org/web/20221225211821/https://techcrunch.com/2013/12/04/former-battery-ventures-partner-brian-omalley-joins-accels-early-stage-team/)

正如风险行业观察人士所知，这些举措很少见，而且往往会有回报。两个著名的例子包括红杉资本在 2004 年诱使吉姆·戈茨离开 Accel 此后，戈茨为红杉资本(Sequoia)领导了 T2 利润丰厚的 T3 交易，包括 WhatsApp。同样曾在 Accel 工作的彼得·芬顿，在 2006 年为 Benchmark 做了一个类似的[的出色工作](https://web.archive.org/web/20221225211821/https://techcrunch.com/2014/02/10/peter-fenton-crunchies/),将他挖走。

本周早些时候，我们在 Accel 的旧金山新办公室采访了欧玛利，讨论了转换公司的问题，以及为什么尽管对按需公司的态度突然冷淡，他还是一如既往地看好它们。

今天我们将进行其中的一部分讨论。接下来的周末请继续关注。

**TC:在电池和 Accel 领域，你已经支持了许多满足按需需求的初创公司，比如[hotel night](https://web.archive.org/web/20221225211821/https://www.hoteltonight.com/)，体育票务应用 [Gametime](https://web.archive.org/web/20221225211821/http://gametime.co/) ，以及送餐应用 [Sprig](https://web.archive.org/web/20221225211821/https://www.sprig.com/#/) 。你还对满足最后需求的应用感兴趣吗？**

BO:在 Accel，我们已经做了三个点播公司。我们在小枝上放一小块。我们支持了优步儿童创业公司。我们还投资了 [Din](https://web.archive.org/web/20221225211821/https://www.din.co/faq) 【以前叫做[草料](https://web.archive.org/web/20221225211821/http://women2.com/2014/08/15/food-tech-vets-re-enter-industry-chef-inspired-meal-delivery-service/?hvid=3qLbk)】,这有点像蓝色围裙，但它利用了整个按需基础设施，所以你不需要依赖所有这些包装和 UPS 在全国各地获得食物。

你的意思是 Din 正在抄袭其他按需创业公司？

BO:如果你想一想，优步、Postmates、Sidecar——它们都有 API，所以我们可以利用这样一个事实，即在最后一英里的基础设施上已经投资了数十亿美元，而且这些公司是由 API 驱动的——一种交钥匙资源。当我们可以利用这些人已经拥有的东西时，为什么还要建造呢？

你也提到了包装的重要性。

博:真的很重要。当人们得到几箱干冰和太空时代的包装材料时，过一段时间就会堆积起来，这是流失的一个重要原因。与此同时，我们可以利用人们感到兴奋的食品趋势，即就地取材，吃得更新鲜。

[使用其他创业公司的基础设施]也很有效，因为你想把这些东西放在别人家门口的时间是下午 2 点到 5 点，那时其他人不太忙，所以你可以利用他们一天中的一个较慢的时段。

**TC:你看到或想象的其他类型的“垂直行业”可以建立在别人的物流系统上吗？**

BO:终端市场的范围很广，但 Accel 的一个核心前提是，它不仅仅是建立一个好的业务，而是试图找到一些真正与众不同的东西并取得突破。所以我关注的类别是大规模的类别。介于杂货店和餐馆体验之间的是一个万亿美元的市场。

尽管我们已经看到了很多相关的创业公司和投资，你认为这仍然是一个开放的市场吗？

波:是的。蓝色围裙做得非常好，但是(就收入而言)，他们基本上还是四五家西夫韦杂货店的规模。他们发展迅速，已经筹集了大量资金，因此拥有大量资源，但在杂货支出的大计划中，他们解决的是一个非常狭窄的用例。

就像餐馆和杂货店一样，你将会有不同的公司出现，为不同的用例服务，包括不同的公司成为地区巨头。[想想]像 PigglyWiggly 这样的杂货店，你甚至从未听说过，但它们是价值数十亿美元的生意。

**TC:喜欢地域 [Instacarts](https://web.archive.org/web/20221225211821/https://www.instacart.com/) 。**

博:对，通过利用世界各地的邮局和优步，你可以更快地进入市场。

挑战在于你能建立多好的服务体验，一旦有人成为你的客户，他们成为你终身客户的可能性有多大。但如果你能从成本角度让物流运作起来，让人们喜欢你的服务，机会还是有的。

美国家庭的花钱方式和生活方式仍有很多变化，这些变化还没有被新服务完全满足。

颤抖似乎也是原因之一。它的员工是合同工还是全职员工，你对他们应该是其中之一的广泛争论有什么看法？

BO:我认为要完全弄清楚还需要一段时间，但我不认为我们称之为“按需”服务的所有东西都会自动面临和优步一样的挑战。还有其他一些行业有合同工的传统，包括几十年来一直从事披萨外卖工作的合同工。

优步已经成为一项用户离不开的服务，它的司机也不会流失。我认为，立法的最终结果将是司机和用户(在经济上)受损，而优步仍坐在司机的位子上。
