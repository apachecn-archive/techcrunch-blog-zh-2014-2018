美国交通部禁止所有航班搭载 Galaxy Note 7

原文：https://web.archive.org/web/https://techcrunch.com/2016/10/14/u-s-department-of-transportation-bans-galaxy-note-7-from-all-flights/amp/

图片来源:

在三星正在上演的 Note 7 传奇的最新一幕中，美国交通部向发布了官方的全面禁令，禁止将 Galaxy Note 7 带上飞机。该禁令适用于所有 Note 7 设备，包括随身行李和托运行李，以及禁止任何人将手机随身携带上飞机。这项禁令将于美国东部时间 10 月 15 日中午 12 点开始生效。

美国交通部长安东尼·福克斯在宣布禁令的新闻稿中发表了以下声明:

我们认识到，禁止航空公司使用这些手机会给一些乘客带来不便，但飞机上所有人的安全必须优先考虑。我们正在采取这一额外的步骤，因为即使是一次飞行中的火灾事故也会造成严重的人身伤害，并危及许多人的生命。

Galaxy Note 7 已经被美国消费者产品安全委员会正式召回，所以理论上所有 Note 7 用户都应该归还手机。但这项新的全面禁令意味着乘客甚至不能再将 Note 7 带上飞机(也许是带到他们最初购买它的地方)，如果他们试图这样做，可能会面临罚款，甚至“刑事起诉”，据交通部称。

三星将回应关于如何最好地返还 Note 7s 的请求，发布说明，海关可以在他们的召回支持网站上寻找更多信息，或者通过他们的客户支持号码 1-844-365-6197 直接联系他们，如果飞行禁令让他们陷入困境。

鉴于上周在西南航空公司的航班上起火的替代 Note 7 在开始冒烟时甚至没有通电，随后烧穿了机舱的地毯，禁飞可能是唯一能够保证 Note 7 相关事件不会在未来发生的行动。然而，全面禁令不会给三星的品牌带来任何好处，现在比以往任何时候都更有可能的是，Note 子品牌将无法在这场持续的磨难中幸存下来。

三星发言人向 TechCrunch 提供了以下关于禁令的声明:

三星与运营商一起，正在努力传达美国运输部的新命令，禁止在航班的随身行李和托运行李中携带所有 Galaxy Note7 设备。我们鼓励航空公司直接向乘客发布类似的信息。任何 Galaxy Note7 用户都应该立即访问他们的运营商和零售店，参加美国 Note7 退款和更换计划。我们知道这给你带来不便，但是你的安全仍然是我们的首要任务。
