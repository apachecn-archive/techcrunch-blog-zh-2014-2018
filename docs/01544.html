Windows 9的预览版可能要到10月份才会发布

还记得微软在9月30日举办的即将到来的Windows 9活动吗？它可能不会像人们长期预期的那样，标志着Windows 9技术预览版的实际发布。据保罗·瑟罗特称，这部分代码可能要到10月份才能发布。

所以放下你的笔记本电脑，撅起嘴，因为等待的时间可能比你想象的要长。

作为发布情况报告的一部分，瑟罗特挖掘了大量其他Windows 9主题，最重要的是证实了版本的更新周期确实会很快。下面是保罗[强调原文]:

微软已经创建了一个新的 Windows Insider预览程序，以便用户可以获得更频繁的预览版本，并向公司提供反馈。

一旦你注册了，你就可以使用Windows反馈应用程序来浏览顶级选项菜单(最近使用的应用程序、应用程序和Windows应用商店、硬件和设备、下载和安装、Internet Explorer等)，然后进一步对其进行微调(邮件、地图、消息、电影时刻、音乐等)。).你可以搜索一下，看看是否有其他内部人士给出了类似的反馈，为这些反馈添加更多的细节，或者开始你自己的反馈，根据需要添加文字和截图。

这很好，因为Windows 9将会——我在此冒昧地对微软的产品做一个预测——引发激烈的公众辩论。

9月30日的活动只是微软打算举办的一系列活动中的第一个，在这些活动中，该公司将在不同的用户群中推出新的操作系统。这表现在一个简单的事实上，即将到来的事件，通过公司自己的旗帜，针对企业。想必消费者是下一个目标。

正如我几次提到的，Windows 8的最初时代已经结束了，至少在名称上是这样。有争议的操作系统给Windows带来的大部分东西将会保留，尽管有望在一个更平稳的整合机制下。我们走着瞧。
