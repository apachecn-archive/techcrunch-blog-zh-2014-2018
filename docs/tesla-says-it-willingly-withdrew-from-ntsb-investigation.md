# 特斯拉称自愿退出 NTSB 调查 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/04/12/tesla-says-it-willingly-withdrew-from-ntsb-investigation/>

特斯拉表示，它自愿退出了与美国国家运输安全委员会(National Transportation Safety Board)的党内协议，并补充说，NTSB 更关心“媒体头条，而不是实际促进安全，”特斯拉发言人通过电子邮件告诉 TechCrunch。

“上周，在与 NTSB 的一次谈话中，我们被告知，如果我们在他们 12-24 个月的调查程序完成之前做出补充声明，我们将不再是调查协议的一方，”特斯拉发言人在给 TechCrunch 的一份声明中说。“周二，我们选择退出协议，并发表声明，纠正关于自动驾驶的误导性说法——这些说法似乎是自动驾驶造成了安全问题，而事实恰恰相反。”

在此之前， [NTSB 称其撤销了](https://web.archive.org/web/20221025222028/https://techcrunch.com/2018/04/12/tesla-is-no-longer-working-with-ntsb-in-fatal-model-x-crash-investigation/)特斯拉在涉及特斯拉一辆 Model X 汽车的致命事故调查中的党籍。NTSB 称这样做是因为[特斯拉未经 NTSB 许可，向公众传递了关于调查的信息](https://web.archive.org/web/20221025222028/https://techcrunch.com/2018/03/30/tesla-says-fatal-crash-involved-autopilot/)。

特斯拉接着指出，与自动驾驶汽车相关的死亡事故相比，美国汽车死亡事故的发生率较高。特斯拉表示，每 3.2 亿英里配备自动驾驶的汽车，就有一人死亡，包括已知的行人死亡。特斯拉说，相比之下，所有车辆每行驶 8600 万英里就有一人死亡。

“如果你驾驶一辆配备自动驾驶硬件的特斯拉，你卷入致命事故的可能性会降低 3.7 倍，并且这种情况会继续改善，”这位发言人说。

特斯拉还声称，“在我们与 NTSB 的谈话中很清楚”，它不太关心安全，更关心媒体头条。

这位发言人说，“除其他外，他们违反自己的规定，一再向媒体发布部分不完整的信息，同时他们试图阻止我们说出所有事实。”。“我们认为这是不对的，我们将向国会提出正式投诉。我们还将发布一项信息自由法案要求，以了解他们关注美国最安全的汽车而忽视最不安全的汽车背后的原因。也许这有一个合理的理由，但我们无法想象这可能是什么。”

特斯拉还花时间指出，NTSB 是一个咨询机构，而不是一个监管机构，以及特斯拉如何与美国国家公路交通安全管理局(NHTSA)有着“强大而积极的关系”

“经过 NHTSA 的测试，Model S 和 Model X 不仅在总体上，而且在每个子类别中都获得了五颗星，”特斯拉发言人说。“这是 SUV 唯一一次获得如此好的成绩。此外，在 NHTSA 测试过的所有汽车中，S 型和 X 型被评为受伤概率最低的两款汽车。没有哪家公司比我们更关心安全，证据不言自明。”

当被联系到就特斯拉的声明发表评论时，NTSB 表示坚持其早些时候发布的新闻稿，没有任何补充。与此同时，NHTSA 表示调查仍在进行中。

“该机构正在与当地调查人员联系，符合 NHTSA 对所有机动车辆和设备安全的警惕性监督和权威，”NHTSA 发言人在给 TechCrunch 的一份声明中说。" NHTSA 正在派遣特别事故调查组，并将采取适当的行动."
