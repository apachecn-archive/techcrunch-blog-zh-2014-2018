Matomy Media 以 1760 万美元收购奥地利移动广告平台 MobFox

为了增强其移动能力，以色列的 Matomy Media Group(T1)将以 1760 万美元的现金和股票(分别为 1010 万美元现金和 750 万美元)收购奥地利移动广告专家(T2)mob fox(T3)。

该公司表示，此举将 Matomy 2014 年来自移动相关活动的收入从 7%提高到 20%。

它预计 MobFox 的技术将使其能够在不到五年的时间内设定 50%的收入来自移动活动的目标。MobFox 的伦敦办公室将成为 Matomy 的英国分公司。

本月早些时候，法国广告公司阳狮集团收购了马托米 20%的股份。
