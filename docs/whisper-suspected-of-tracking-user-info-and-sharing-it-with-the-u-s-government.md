# Whisper 否认追踪用户信息并与美国政府分享

据《卫报》报道，匿名社交媒体应用 [Whisper](https://web.archive.org/web/20230403193639/http://whisper.sh/) 可能通过智能手机上的 GPS 或 IP 地址追踪用户的位置，然后将这些信息提交给美国国防部[。](https://web.archive.org/web/20230403193639/http://www.theguardian.com/world/2014/oct/16/-sp-revealed-whisper-app-tracking-users)

《卫报》表示，在与 Whisper 讨论合作伙伴关系的三天行程中，它发现这家总部位于洛杉矶的初创公司可以精确定位 500 米内的用户位置，即使他们中的一些人选择了不被跟踪。据《卫报》报道，估计有 20%的用户选择退出地理定位追踪。

Whisper 允许数百万用户将消息和个人信息叠加到图像上，然后发布到应用程序上供他人查看。其承诺的一部分是匿名给那些想要一个出口张贴秘密或其他事情，他们不觉得自由公开承认。还有一个庞大的军事人员用户群，他们发现该应用程序在表达他们战后的精神和情感健康方面很有用。像这样的指控对许多分享个人信息和关于自己或他人的匿名秘密的人来说是非常令人担忧的。

![url](img/3cc50563094426f976e11b8a924a4d67.png)

《卫报》还声称，Whisper 给了他们一个简单的密码，让他们使用后端分析工具，帮助记者在该公司的大型数据库中找到匿名帖子。这些帖子还包含可以识别这些匿名帖子的街道位置的信息。据《卫报》报道，这包括一封据称来自白宫内部的邮件。《卫报》称，这些信息很容易就被移交给五角大楼和其他政府组织，而不需要任何战斗。

在发给 TechCrunch 的一份书面声明中，Whisper 发言人否认了《卫报》的说法:“Whisper 不会收集或存储任何用户的个人身份信息，并且是匿名的。我们的地理定位数据中没有任何东西可以与个人用户联系在一起，用户的匿名性也不会受到影响。Whisper 不会关注或跟踪用户。《卫报》关于 Whisper 正在收集用户信息并侵犯用户隐私的假设是错误的。”

Whisper 的主编 Neetzan Zimmerman 在 Twitter 上抨击了《卫报》，指责这篇文章撒谎，并表示新闻网站将“后悔”

据报道，Whisper 在得知《卫报》打算发表这些发现后几天就改变了服务条款。目前的条款概述了一项隐私政策，让用户知道该应用程序收集用户数据，如位置:

> 当您下载或使用我们的移动应用程序或服务时，我们可能会要求、访问或跟踪基于位置的信息，例如设备提供的纬度或经度，如果您为应用程序启用了位置服务的话。这样做的一个原因是为了帮助您探索您所在区域的帖子，或者让我们确定有多少帖子来自某个特定区域。然而，允许我们访问或跟踪任何基于位置的数据完全是自愿的。事实上，您可以随时自由选择加入或退出我们基于位置的服务。但是，请记住，即使您禁用了定位服务，我们仍然可以根据您的 IP 地址确定您的城市、州和国家位置(但不是您的确切位置)。
