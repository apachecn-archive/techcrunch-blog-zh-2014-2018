格莱美给了CBS All Access迄今为止第二大注册日| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/01/29/the-grammys-gave-cbs-all-access-its-second-biggest-day-for-signups-yet/

周日晚上播出的第60届格莱美颁奖典礼对哥伦比亚广播公司的流媒体服务CBS All Access来说是一个巨大的推动，CBS All Access在其服务上提供了该事件的直播。根据该网络的说法，其订阅视频点播和直播服务在订户注册和颁奖典礼的独立观众方面创下了新纪录，是有史以来订户注册的第二大天。

订户注册的最大一天是CBS预期的流媒体系列“星际迷航:发现”的首映日，这并不奇怪。

哥伦比亚广播公司拒绝透露总共有多少注册可以归功于格莱美奖，但确实指出，直播的独立观众比去年的电视直播增加了40%以上，格莱美日的哥伦比亚广播公司数字平台流量创下纪录。

哥伦比亚广播公司说，电视直播只能在桌面、网络、移动和电视应用程序的付费墙后进行。

尼尔森还指出，颁奖典礼是今年迄今为止最社会化的电视节目——但是，当然，这只是一月份，所以这不一定是一个重要的里程碑。

然而，哥伦比亚广播公司去年表示，到2017年底，CBS All Access和Showtime的订户总数将超过400万，到2020年将达到800万。今天，哥伦比亚广播公司告诉TechCrunch，它现在有“超过200万”的订户。

在某种程度上，这可能是因为CBS All Access不是插队者这些天观看CBS节目的唯一途径。该网络可在各种直播电视服务上进行流式传输(在某些市场)，例如包括DirecTV Now、PlayStation Vue、Sling TV、YouTube TV、富博TV和Hulu Live TV。

虽然CBS不会透露有多少新用户来到CBS All Access观看格莱美，但有一些其他方法可以衡量颁奖典礼对流媒体服务的影响。

一种方法是通过查看移动应用在应用商店中的排名。

根据Sensor Tower的数据，在格莱美日之前，该应用在前四个周一在美国所有免费iPhone应用中的平均排名约为第325位。自2017年9月以来，该应用程序从未在该榜单上排名第100位以上。

颁奖典礼结束后，这款应用大幅跃升，目前在美国所有免费iPhone应用中排名第89位。

为了让你对这个排名在下载量方面的意义有个大概的了解，一个排名比较接近的例子是Tinder，排名第82位。1月27日周六，排名第82位的应用程序当天的iPhone下载量约为24000次。(当然，在89号，CBS的所有频道都属于这一类，但你可以有个大概的了解。)

我们应该注意到，该应用在格莱美日之前的最高排名实际上并不是新《星际迷航》首映的那一天。这是三天前——显然，人们已经提前做好了准备。这款iPhone应用当天排名第69位。

格莱美没有打破CBS All Access注册人数的新纪录(与《星际迷航》相比)未必是最大的担忧。毕竟，这是一年一次的活动，也就是说，在说服新用户尝试流媒体服务方面，它只能做到这一步。
