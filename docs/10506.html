真正的技术(再次)死了

原文：https://web.archive.org/web/https://techcrunch.com/2017/01/05/the-real-technics-is-dead-again/

Technics刚刚在CES 2017上公布了Grand Class SL-1200GR转盘。它基本上只是他们去年发布的SL-1200g的一个更轻更简化的版本。

黑胶发烧友希望这将是一个预算模型，但事实可能并非如此。Technics尚未发布任何官方定价信息，但 Gizmodo报道价格应该在2000美元左右(相比之下，去年宣布的型号价格为4000美元)。

如果新的“预算”技术甲板真的要花费2000美元，我想可以肯定地说真正的技术已经死了。

更新: 松下已经向TechCrunch证实，价格标签将在2000美元左右，可能会在6月份到货。

地球上的每个DJ都想要Technics SL-1200 MK2。这是世界上最受欢迎的唱盘之一，现在你可以在易贝用不到1000美元买到一个二手的唱盘。

当有大量高质量的二手工艺SL-1200 MK2可用时，为什么还会有人花2000美元买一个新版本呢？

为什么有人会花钱买一个新的工艺转盘，而外面有几十个价格很低的类似选择呢？

松下重新启动了Technics，以吸引无数渴望这个著名转盘品牌回归的DJ。几年前，超过27，000名DJ签署了请愿书，要求恢复生产。

这就很好的解释了为什么松下决定带回技术这个名字，但是从深渊中回来的根本不是技术。

继续努力证明工艺名称的“品牌资产”。事实是，松下用这次重生毁掉了工艺的名字。

这不再是技术了。只是价格太高了。
