# 雅虎问答首次在手机上正式亮相

雅虎问答已经正式在移动端上线。正如 TechCrunch 上个月报道的那样，雅虎代号为 Yahoo Hive 的实验性应用程序已经更名为 Yahoo Answers Now——这一举动表明了雅虎计划将其桌面 Q&A 网站引入应用程序商店。然而，在那个时候，该应用程序仍然需要邀请才能测试——这一限制[于周四](https://web.archive.org/web/20221025223447/https://yahoo.tumblr.com/post/154215434969/introducing-yahoo-answers-now-a-new-app-for-the?soc_src=mail&soc_trk=ma)解除，该应用程序对所有人开放。

获得试用该服务的邀请并不困难——如果你想加入，只需向给定的地址发送电子邮件请求即可。但是额外步骤的存在可能会阻止许多用户注册。

雅虎问答在全球仍有 3 亿月度用户，是雅虎更知名的资产之一，如果仅仅是因为多年来在其网站上发布的低质量、有时令人恐惧、有时令人捧腹的答案。

在媒体行业的诸多苦恼以及网络和社交媒体上流传的“假新闻”和尖酸刻薄的问题中，雅虎问答是一个明确的例子，说明为什么网络上的每个人都不值得拥有平等的发言权。这是众包最好的一面，有时也意味着人性最差的一面。

从《T4》开始，迷因如“巴比是如何形成的？”到[你希望是巨魔在开玩笑的问题](https://web.archive.org/web/20221025223447/https://www.buzzfeed.com/mrloganrhoades/38-yahoo-questions-that-will-destroy-your-faith-in-humanity?utm_term=.ga59oB9Md#.xlnJzBJm6)到如今，一个经常充斥着赤裸裸的种族主义评论的首页，雅虎问答至少可以说正处于一个岌岌可危的境地。它需要整顿自己在移动领域的行为，否则就有可能因违反谷歌和苹果的指导方针而被苹果应用商店除名。

![tumblr_inline_ohutvgnbzj1qhxx5s_500](img/b83e44fa64dbba5572b50308d843fa66.png)

新的应用程序包括几个关键功能，如能够提出与合格回复者匹配的问题，能够以类似 Twitter 的方式关注他人，能够通过“谢谢”和表情按钮、通知表示感谢，并支持图像和链接等丰富的媒体。

这款应用现在可以在 [iOS](https://web.archive.org/web/20221025223447/https://itunes.apple.com/us/app/yahoo-answers-now/id1111749046?mt=8&ign-mpt=uo%3D4) 和 [Android](https://web.archive.org/web/20221025223447/https://play.google.com/store/apps/details?id=com.yahoo.mobile.android.heartbeat) 上使用。
