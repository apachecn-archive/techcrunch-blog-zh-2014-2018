# Whisper 暂停参与卫报访问的编辑团队 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/10/26/whisper-suspends-editorial-team-involved-in-guardian-visit/>

匿名消息应用 [Whisper](https://web.archive.org/web/20230203063014/http://whisper.sh/) 已经暂停其主编[尼赞·齐默尔曼](https://web.archive.org/web/20230203063014/http://en.wikipedia.org/wiki/Neetzan_Zimmerman)和其他工作人员的职务，此前参议院商务委员会主席杰伊·洛克斐勒写信[召集](https://web.archive.org/web/20230203063014/http://images.politico.com/global/2014/10/22/102214_chairman_rockefeller_letter_to_whisper_final_sent__redacted.html) Whisper 高管召开面对面会议，讨论该应用的隐私做法。

洛克菲勒写道:“虽然 Whisper 为用户提供了独特的社交体验，但最近媒体账户中的指控是严重的，用户有权获得透明、公开的隐私政策，并得到公司的遵守。”

《卫报》已经[撰写了](https://web.archive.org/web/20230203063014/http://www.theguardian.com/world/2014/oct/16/-sp-whispers-secret-safest-place-internet-tracks-users) [多篇](https://web.archive.org/web/20230203063014/http://www.theguardian.com/world/2014/oct/16/-sp-whisper-privacy-policy-terms-of-service) [文章](https://web.archive.org/web/20230203063014/http://www.theguardian.com/world/2014/oct/16/-sp-revealed-whisper-app-tracking-users)指责该应用侵犯用户隐私，并报道称，Whisper 的工作人员向《卫报》展示了其数据库如何定位其所在位置 500 米内(距离标准城市街区稍短)的用户，包括那些要求不被跟踪的用户。齐默尔曼在推特上回击说，这些指控是“恶毒的谎言”。然后，他继续在推特上发布了 20 条微博，试图澄清他所说的围绕 Whisper 收集的数据的错误信息。

洛克菲勒参议员周三晚些时候致信 Whisper 首席执行官迈克尔·海沃德，要求与他和他的员工会面，以便海沃德可以解释该应用程序如何跟踪用户。自周三以来，尼赞的 Twitter 账户一直保持沉默，此前他曾就发生的事情发表过几次言论。

海沃德表示，他将继续与公众讨论 Whisper 的内部运作，并于周五发布了一封逐点信函，回应《卫报》关于 Whisper 被传唤到国会山的最新文章。在这篇文章中，海沃德暗示《卫报》从非技术人员那里收集了不正确的信息，包括《耳语》的编辑人员。

Whisper 指责《卫报》引用了不知道自己被记录在案的工作人员的话，其中包括一次这样的对话，编辑人员向两名《卫报》记者展示了他们将如何跟踪一名显然沉迷于性的华盛顿游说者“一辈子”

关于跟踪的来回争论在这里变得有点微妙。据《卫报》报道，Whisper 通过两种方式跟踪位置，一种是 GPS 定位，另一种是使用 IP 地址，为那些没有选择在应用程序上进行位置跟踪的人提供服务。“例如，如果一个用户正在征求未成年人，我们将与国家失踪和被剥削儿童中心分享我们有限的信息，”海沃德写道。

《卫报》声称，在得知《卫报》计划公布其曝光后仅四天，Whisper 就重写了其服务条款，并增加了一个关于隐私的新部分。Whisper 在其新的隐私政策部分包含了对自杀的担忧:

> 耳语是一个安全的地方。在某些情况下，为了更大的利益，我们可能需要披露我们可能拥有的关于您的信息，例如，如果我们看到自杀声明或涉及伤害自己或他人意图的声明，或了解关于失踪儿童的信息，或对社区的威胁。

自杀一直是这个应用上的热门话题，[尤其是在军人中](https://web.archive.org/web/20230203063014/http://www.buzzfeed.com/katienotopoulos/heartbreaking-military-confessions-of-whisper#3s63c86)。

看起来 Whisper 很乐意承认它是如何跟踪用户信息的，但《卫报》推断这是一种常见的做法，超出了 Whisper 只在“极端执法案件”中使用这些信息的声明。

Whisper 早些时候承认，它将某些信息交给国防部和其他政府组织。然而，齐默曼淡化了这些信息是什么，他在推特上说，Whisper 只收集自杀被提及次数的数据，而不是实际的位置信息。

在第一篇文章发表之前，Whisper 的高级副总裁埃里克·耶林(Eric Yellin)在与《卫报》的电子邮件交流中写道，Whisper 使用 IP 地址，“来确定非常近似的位置。”《卫报》称，这让齐默尔曼和耶林在《耳语者》的运作方式上产生了分歧。海沃德声称 Yellin 的意思是为了安全起见，而不是作为一般耳语社区的惯例。

Whisper 此前曾表示，它不会“被动或主动”追踪用户。不过，这有点让人挠头。虽然 Whisper 可能无法跟踪用户的确切地址，但它似乎有能力跟踪 500 米内的用户位置，还可以跟踪那些选择退出 GPS 跟踪的用户的 IP 地址，以便将其交给政府，这将被视为被动和主动跟踪。正如海沃德在信中所写的:

> “当出于编辑目的评估用户声明的真实性时，可以考虑发布的耳语的先前位置。例如，如果一个用户声称自己是一名在西非治疗埃博拉患者的医生，而事实上他从未在西非发布过任何悄悄话，我们的编辑团队就不会发布这个帖子。”

海沃德在信中声称，Whisper 在七天后丢失了 IP 地址信息。然而，这似乎与他在后一段中提到的《卫报》对“无限期”存储用户数据的担忧相矛盾。《卫报》报道称，使用内部绘图工具，信息随时可用，编辑人员可以搜索帖子并验证其真实性，以便在网站上展示它们。

海沃德为这种做法辩护，解释说这是一种允许用户搜索旧帖子的功能，而不是可以识别个人或其位置的功能。这也是一个棘手的论点。《卫报》声称，Whisper 的工作人员可以使用人们的帖子信息来搜索和识别他们。海沃德说，只是帖子，而不是与帖子相关的个人信息被无限期存储。

这仅仅是关于什么构成了信息的被动和主动存储的语义，还是对隐私香肠如何制作的细微误解？《卫报》声称，Whisper 隐瞒了什么吗？《卫报》是一群“危言耸听者”，只是“彻头彻尾的谎言和完全错误”，正如 Whisper 所声称的那样吗？据我们所知，还没有确定 Whisper 高管与洛克菲勒会面并解决此事的日期。但有一点很清楚，如果你想做违法的事情，你绝对不应该在 Whisper 上发布。也就是说，除非你希望政府为此去敲你所有邻居的门。
