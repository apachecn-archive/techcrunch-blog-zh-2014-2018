# 汤米·席尔菲格推出了一个可笑的智能服装系列，奖励你穿上它

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/07/25/tommy-hilfiger-has-launched-a-ridiculous-line-of-smart-clothing-that-rewards-you-for-wearing-it/>

没人要的漂亮衣服又来了。时尚品牌汤米·席尔菲格今天宣布推出一个新系列的[男装](https://web.archive.org/web/20230223153750/https://usa.tommy.com/en/men-tommy-jeans-capsule)和[女装](https://web.archive.org/web/20230223153750/https://usa.tommy.com/en/women-tommy-jeans-capsule)汤米牛仔裤 Xplore，该系列采用智能芯片嵌入式技术。比如说，与谷歌的 Project Jacquard 和[与李维斯](https://web.archive.org/web/20230223153750/https://techcrunch.com/2018/05/14/google-and-levis-connected-jacket-will-let-you-know-when-your-uber-is-here/)的合作不同，它的目标不是在你无法使用手机时提供通话、短信、地图和音乐控制——比如当你骑自行车时。相反，希尔费格的智能服装旨在奖励你穿希尔费格服装的积分。是的，真的。

事情发展到这一步，伙计们。

该系列包括 t 恤、运动衫、帽衫、牛仔裤、夹克、帽子和包，可搭配 Tommy Jeans Xplore(或“Xplore”，如果你使用他们的品牌) [iOS 应用](https://web.archive.org/web/20230223153750/https://itunes.apple.com/us/app/tommy-jeans-xplore/id1393960235?mt=8)蓝牙。一旦配对，用户将在应用程序中竞争挑战以获得积分。你可以根据你穿衣服的频率得到分数。！！)以及在应用程序的地图上四处寻找心形、汤米商标的图标。(?？？)

该公司表示，积分可以转化为奖励，包括礼品卡、签名商品和汤米·席尔菲格档案馆的藏品等。

![](img/42699c8d360f86fdd9b44bddb5b4b7f8.png)

我想给球员发放更多的汤米·席尔菲格·默克是有意义的，因为只有那些愿意花 90 美元买一件智能运动衫来玩营销活动的乐趣的人才是最忠实的——不，是最痴迷的——希尔费格球迷。

但除此之外，汤米的聪明衣服对任何人来说都没什么意义。

尽管它使用了智能技术——比如 embedded Awear Solutions 的蓝牙低能耗智能标签——但该公司实际上并没有在这方面进行创新。充其量，这是一个忠诚度计划，要求客户超支才能加入。

甚至该公司似乎也意识到了该系列的利基吸引力，在其官方声明中表示，其目标是创建一个“品牌大使的微社区”。

是的，微——就像*非常非常非常小。*

然而，该品牌对新思想和新技术的实验并不陌生。但它之前的一些开发并不那么荒谬——比如测试使用[人工智能来预测设计趋势](https://web.archive.org/web/20230223153750/https://news.fitnyc.edu/2018/02/01/students-team-with-ibm-and-tommy-hilfiger-to-utilize-ai-to-elevate-design-manufacturing-and-marketing/)，它的智能手表，或者为残疾人设计的[自适应服装](https://web.archive.org/web/20230223153750/https://www.racked.com/2017/10/17/16489394/tommy-hilfiger-adaptive-clothing)。

为了智能服装而智能服装？

只是没有。
