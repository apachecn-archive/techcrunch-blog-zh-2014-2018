# 脸书的欧洲支付许可证对银行意味着什么

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/01/12/what-facebooks-european-payment-license-could-mean-for-banks/>

12 月初，脸书终于公布了新获得的爱尔兰境外电子货币和支付服务牌照。自从有报道称脸书前阵子申请了资金转移许可证后，关于脸书进入欧洲支付领域的传言就一直在持续。脸书在支付领域野心的另一个明显迹象是雇佣前 [PayPal 总裁 David Marcus](https://web.archive.org/web/20230323202644/https://techcrunch.com/2014/06/09/facebook-poaches-paypal-president-david-marcus-to-run-messenger-maybe-monetize-it-with-payments/) 担任 Facebook Messenger 的负责人。正如马克·扎克伯格[在去年一月](https://web.archive.org/web/20230323202644/http://appleinsider.com/articles/16/01/27/facebooks-zuckerberg-open-to-payments-partnerships-including-apple-pay)所说，“我们将与每一个支付的人合作。”

随着脸书支付已经在美国运营，一个显而易见的策略是在整个欧洲推出这项服务，至于脸书是否会满足于提供一个简单的 P2P 支付服务来为他们的 Messenger 平台创造粘性，或者他们是否也将目标锁定在[5000 多亿美元的全球汇款市场](https://web.archive.org/web/20230323202644/http://gulfnews.com/business/economy/global-remittance-flow-grows-10-77-to-514-billion-in-2012-world-bank-1.1172693)。

如果我们不把支付视为一项孤立的服务，那么脸书作为一家持牌支付服务公司，代表着银行应该密切关注的几种情况。

即将出台的支付服务指令 2 (PSD2)要求银行必须向金融服务的第三方提供商提供支付 API，并允许用户授权许可的第三方 1)发起支付和 2)提取账户信息。

> 作为最强大的数字生态系统，脸书在去中介化零售银行方面处于独特的地位。

随着 Facebook Marketplace 的推出，脸书已经扰乱了分类广告市场，通过 PSD2，脸书有可能成为自己的支付处理器，成为 PISP(支付启动服务提供商)并通过 API 直接连接到银行账户。然后脸书可能会要求消费者允许使用他们的银行信息作为支付方式。一旦您给予许可，脸书将能够安全地访问您的银行账户，并收取他们的付款。

除了节省成本之外，脸书扮演 PISP 角色的解决方案消除了复杂的结账流程，并为回头客提供了[“一键”支付选项](https://web.archive.org/web/20230323202644/https://techcrunch.com/2014/08/19/paypal-rolls-out-one-touch-mobile-checkout-for-apps/)。通过直接从客户的银行账户收款，PSD2 还将[实现更快的支付](https://web.archive.org/web/20230323202644/https://techcrunch.com/2015/07/12/the-future-of-finance-is-in-real-time/)，以及[模糊传统产业之间的界限](https://web.archive.org/web/20230323202644/https://techcrunch.com/2015/04/27/industry-convergence-in-financial-services-is-accelerating/)。

脸书也可能成为 AISP(账户信息服务提供商)。该指令使 AISPs 能够提供一个以上银行账户的汇总视图。AISPs 可以分析消费行为，或将用户在多家银行的账户信息汇总到一个概览中，从而使基于一个账户的传统移动和在线银行解决方案变得过时。

很明显，聊天机器人将对银行业产生重大影响，因为大多数银行服务可以通过简单的聊天请求实现自动化，比如“在我下次发工资之前，我的每日支出限额是多少”或“批准并支付我的未清账单”当一切都已经可以直接在 [Facebook Messenger](https://web.archive.org/web/20230323202644/https://techcrunch.com/2016/07/08/banks-should-not-underestimate-facebook-messenger/) 中获得并且安全时，为什么还要费心登录您的银行应用程序呢？

作为一家获得许可的支付服务公司，脸书还制定了必要的法规来提供结算风险，以符合众筹平台的要求(在某些国家)。尽管这是一个拥挤的领域，整个欧洲有近 300 个替代金融和众筹平台，但脸书在这一领域有着独特的地位，它拥有庞大的用户群和利用用户数据以及账户信息评估风险和信誉的能力。

脸书已经处于一个独特的位置，可以让零售银行去中介化，成为消费者最强大的数字生态系统。脸书强大地位的关键是能够随着用户行为的变化而发展；迄今为止，脸书在这方面表现出色。使脸书成为最强大的数字生态系统之一的是，它基于管理其用户群中每个用户的数字身份，如果做得好，脸书有可能让电话号码、电子邮件地址和银行账号过时。

最终，这可能会让我们所知的银行成为隐形的商品提供商，所有的客户互动——支付、电子商务和日常银行业务——都是通过脸书支付和 Facebook Messenger 等数字生态系统进行的。根据普华永道在[进行的一项研究，68%的银行家担心失去对客户界面的控制。一个监管合规的脸书肯定会让这些担忧变成担忧。](https://web.archive.org/web/20230323202644/https://www.finextra.com/finextra-downloads/newsdocs/catalyst-or-threat.pdf)
