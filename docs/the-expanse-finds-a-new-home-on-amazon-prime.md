# “广阔空间”在亚马逊 Prime 上找到了新家

在 Syfy 在其网络上停止播放该系列后，粉丝们发出了强烈抗议，在杰夫·贝索斯宣布之后，Expanse 将在亚马逊 Prime 上获得第四季。

贝佐斯在国际太空发展会议上透露了这一消息，该节目的演员也在观众中。

该剧根据詹姆斯 S.A 科里的系列丛书改编，目前在 Syfy 网络上播出第三季。这部广受好评的政治阴谋系列片以未来的殖民太阳系为背景，被誉为该网络的最佳剧集之一，但它在 Syfy 上没有获得高收视率，这使得爱尔康娱乐公司本月早些时候开始在不同的网络上购买该剧，为第四季寻找归宿。

亚马逊肯定会吸引更多的观众来观看这场展览，鉴于亚马逊在原创内容方面仍落后于网飞，吸引大量粉丝也有利于该公司的视频流媒体平台。
