Twitter 正在关闭其商业应用 Twitter Dashboard

Twitter 宣布今天将关闭去年夏天推出的名为 Dashboard 的功能，该功能允许企业使用一套工具来跟踪推文、安排帖子、访问分析、监控关于其品牌或其他关键词的推文等等。该公司表示，Dashboard 将于 2017 年 2 月 3 日完全关闭，但就企业未来如何访问类似功能而言，还没有一个可靠的过渡计划。

相反，Twitter 只是说，它“希望”在未来几个月将 Dashboard 的最佳功能带到更广泛的 Twitter 社区。然而，这种含糊不清并不是很有希望。

该产品可能没有一个大的安装基础，但是，这就是为什么它被标记为关闭。根据 App Annie 的数据，Twitter Dashboard 在 iTunes 的“商业”应用类别中排名第 432 位。Sensor Tower 表示，该应用自发布以来在全球范围内被下载了约 4 万次，上个月被下载了约 3000 次。

Dashboard 也与 Twitter Engage 有一些重叠，Twitter Engage 是一个为名人和其他有影响力的人提供的工具，大约在同一时间宣布了，其中也包括分析。

虽然 Twitter 希望为其商业客户提供一款产品是有道理的，但许多更直接处理其社交媒体存在的人已经在使用其他功能更丰富的应用程序，例如 Hootsuite、Buffer 或 Sprout Social。

关闭 Dashboard 的举措既可以在网上使用，也可以在专用的 iOS 应用程序中使用，此前该公司最近几个月一直在进行其他精简。

Twitter 宣布今年秋天将关闭 Vine ,并将只保留一个简单的相机应用。通过将直播功能整合到 Twitter 本身，它也减少了用户安装 Periscope 独立应用的需求。
