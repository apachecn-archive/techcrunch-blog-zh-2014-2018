# Pokémon GO 正式与星巴克合作，开设 7800 家新健身房和神奇宝贝站

Pokémon GO 在美国的下一个大赞助商是 [Starbucks，](https://web.archive.org/web/20221213102459/http://pokemongolive.com/en/post/starbucks/)Starbucks，它将在美国 7，800 家公司经营的星巴克门店开设新的健身房和神奇宝贝站。除了新的游戏内场所，星巴克还制作了 Pokémon GO 版星冰乐以庆祝合作关系。Pokémon GO 以加糖咖啡饮料的形式喝起来是什么味道？我不知道，但我一定会找到的。

这类似于麦当劳今年早些时候从 closer 到 Pokémon GO 的合作伙伴关系，这家快餐店付钱给 Niantic，将日本的 3000 个地点变成健身房和神奇宝贝站。星冰乐的元素为组合增加了一点互惠——虽然麦当劳的交易菜单上没有麦当劳烤鸭。

本周早些时候，Niantic 和 Sprint 披露了一项类似的协议，将 Sprint，Boost Mobile 和“RadioShack 的 Sprint”商店转变为 PokéStops 和健身房，这些商店将遍布美国 10，500 多个地点。
