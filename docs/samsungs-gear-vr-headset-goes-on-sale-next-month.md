# 三星的 Gear VR 耳机下月上市

![Gear_VR_Still](img/2696c8bab1f299d6bd00e86cacfe31ec.png)

三星为那些希望在旅途中尝试虚拟现实的人带来了好消息。在今天早上的开发者大会上，该公司宣布其 [Gear VR 耳机](https://web.archive.org/web/20221001020030/http://www.samsung.com/global/microsite/gearvr/)的创新版将于下月初上市。

这家韩国电子巨头没有给出这款耳机的确切日期或价格，但它重申，第一天将会有 Oculus 的软件准备好加载到 Galaxy Note 4 设备上(目前唯一与三星耳机或 Oculus 移动 SDK 兼容的设备)。这将包括一个 VR 专用的主屏幕，用于在应用程序之间导航，VR 电影院体验和 Oculus 应用程序商店。

三星让那些跟随主题演讲的人到[查看此页面](https://web.archive.org/web/20221001020030/http://www.samsung.com/us/register/see-the-difference-samsung-gear-vr/)进行预注册以购买该设备。这实际上更像是注册接收一堆关于三星设备的垃圾邮件。但是，如果你真的担心提前得到这些东西，你可能不会介意在接下来的几周内收到几封充满该设备图像和视频的电子邮件。
