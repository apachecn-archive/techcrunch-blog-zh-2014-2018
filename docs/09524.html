美国交通部禁止所有航班搭载Galaxy Note 7

原文：https://web.archive.org/web/https://techcrunch.com/2016/10/14/u-s-department-of-transportation-bans-galaxy-note-7-from-all-flights/

美国运输部禁止所有航班搭载Galaxy Note 7

在三星正在进行的Note 7传奇的最新一集中，美国交通部向发布了官方的全面禁令，禁止将Galaxy Note 7带上飞机。该禁令适用于所有Note 7设备，包括随身行李和托运行李，以及禁止任何人将手机随身携带上飞机。这项禁令将于美国东部时间10月15日中午12点开始生效。

美国交通部长安东尼·福克斯在宣布禁令的新闻稿中发表了以下声明:

我们认识到，禁止航空公司使用这些手机会给一些乘客带来不便，但飞机上所有人的安全必须优先考虑。我们正在采取这一额外的步骤，因为即使是一次飞行中的火灾事故也会造成严重的人身伤害，并危及许多人的生命。

Galaxy Note 7已经被美国消费者产品安全委员会正式召回，所以理论上所有Note 7用户都应该归还手机。但这项新的全面禁令意味着乘客甚至不能再将Note 7带上飞机(也许是带到他们最初购买它的地方)，如果他们试图这样做，可能会面临罚款，甚至“刑事起诉”，据交通部称。

三星将回应关于如何最好地返还Note 7s的请求，发布说明，海关可以在他们的召回支持网站上寻找更多信息，或者通过他们的客户支持号码1-844-365-6197直接联系他们，如果飞行禁令让他们陷入困境的话。

鉴于上周在西南航空公司航班上着火的替代Note 7在开始冒烟时甚至没有通电，随后烧穿了机舱的地毯，禁飞可能是唯一能够保证Note 7相关事件未来不会发生的行动方案。然而，全面禁令不会给三星的品牌带来任何好处，现在比以往任何时候都更有可能的是，Note子品牌将无法在这场持续的磨难中幸存下来。

三星发言人向TechCrunch提供了以下关于禁令的声明:

三星与运营商一起，正在努力传达美国运输部的新命令，禁止在航班的随身行李和托运行李中携带所有Galaxy Note7设备。我们鼓励航空公司直接向乘客发布类似的信息。任何Galaxy Note7用户都应该立即访问他们的运营商和零售店，参加美国Note7退款和更换计划。我们知道这给你带来不便，但是你的安全仍然是我们的首要任务。
