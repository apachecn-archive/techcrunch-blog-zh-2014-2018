Stratoscale 收购数据库即服务公司 Tesora

为企业数据中心提供 AWS 兼容环境的初创公司 Stratoscale 已经收购了一家数据库即服务公司 Tesora，该公司运行在 OpenStack 平台之上，允许其客户在 16 个最受欢迎的数据库中选择。

两家公司都拒绝讨论收购的财务细节。据 Crunchbase 报道，自 2010 年成立以来，Tesora 已经筹集了 1270 万美元。

Stratoscale 认为，此次收购将使其能够为客户提供更好的托管数据库服务支持。Tesora 支持 MySQL、MongoDB、Redis、Cassandra、DataStax Enterprise、Couchbase、MariaDB、Oracle、Percona、PostgresSQL 和 DB2 Express 等。

据 Crunchbase 报道，总部位于以色列的 Stratoscale 自 2013 年成立以来已经筹集了 6900 万美元。其标志性产品 Symphony 允许企业在其数据中心的商用服务器上构建类似 AWS 的云基础设施。它的客户包括像 WebEx 和西部数据这样的公司。

Strotoscale 首席执行官兼创始人 Ariel Maislos 告诉我们:“收购 Tesora 是 Stratoscale 的一个机会，有助于在 Stratoscale Symphony 解决方案中向市场推出强大的 DBaaS 产品。“这项服务是 Stratoscale 承诺为企业提供本地 AWS 区域的一部分，随着 Tesora 的收购，Stratoscale 将推出更多高级服务。”

他补充说，Tesora 团队将成为 Stratoscale 的一部分，Tesora 的现有客户将继续得到支持。
