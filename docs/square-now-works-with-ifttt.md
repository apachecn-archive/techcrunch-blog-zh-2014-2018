# Square Now 与 IFTTT 合作

想象一个有两个大圆圈的维恩图:“使用正方形的人”和“使用 IFTTT 的人”。

如果你坐在两者的重叠处——万岁！今天是你的日子。[广场](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/square)现在与[iftt](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/if-this-then-that)进行友谊赛。

如果你想知道 IFTTT 是多少，这是 t1；dr:[iftt](ifttt.com/)(If This Then That 的缩写)让您将在线服务连接在一起，在发生其他事情时自动执行某项操作。

想在每次有人上传你在脸书的照片时收到短信吗？好吧。每当你的[音箱](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/dropbox)发生变化时，你想让你房间的灯变成蓝色吗？很奇怪，但是——好吧！购买一个精美的飞利浦顺化灯泡，将其插入 iftt，打开 iftt 的 Dropbox 频道，您就可以使用了。IFTTT 旨在让互联网上的东西更好地协同工作。

随着 Square 与 IFTTT 的连接，您可以使用 Square 交易——购买、退款或取款/存款——作为其他服务的触发动作。

**您可以，例如:**

*   每次进行超过 X 的交易时得到一个文本提示
*   将其连接到 Evernote，以保持 X 值退款的自动运行列表
*   当一笔大买卖成交时，在 Yammer 上发布一条祝贺信息
*   将所有新付款添加到谷歌文档电子表格的一行

由于 iftt 在从 [Tumblr](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/tumblr) 、到 [Twitter](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/twitter) 、到 iOS/Android/Android Wear、到 [reddit](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/reddit) 、到[易贝](https://web.archive.org/web/20230129221926/http://www.crunchbase.com/organization/ebay)的所有东西上都有可操作的钩子，你可以做各种各样的奇怪的事情——只需要在 iftt 的创作工具上找找适合你的东西。
