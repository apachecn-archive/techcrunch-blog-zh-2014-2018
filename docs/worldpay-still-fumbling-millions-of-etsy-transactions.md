# 3 周后，Worldpay 仍在摸索数百万笔 Etsy 交易

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/07/19/worldpay-still-fumbling-millions-of-etsy-transactions/>

Etsy 用户又度过了糟糕的一周，因为支付处理器 Worldpay 继续引发问题。上周晚些时候，一些 Etsy 用户开始报告称，他们遭遇了重复收费。[重复收费 Etsy 论坛](https://web.archive.org/web/20230213115922/https://www.etsy.com/teams/7720/bugs/discuss/17917356/)已接近 1000 个帖子。

在一份声明中，世界支付承认在试图纠正问题的过程中对 Etsy 用户造成了重复收费。

“与解决此问题相关的持续技术工作导致 Etsy 客户收到有限数量的重复费用。”

[三周前，Etsy 商户开始报告严重的付款处理问题。](https://web.archive.org/web/20230213115922/https://techcrunch.com/2016/07/05/five-days-of-etsy-payment-processing-outages-have-merchants-flipping/)Etsy 臭虫论坛上成千上万的帖子开始涌入。几天后,[发现 Etsy 的支付处理器 Worldpay 对此次停电负有直接责任。](https://web.archive.org/web/20230213115922/https://techcrunch.com/2016/07/06/worldpay-etsys-payments-processor-admits-to-service-outage/)

TechCrunch 还获悉，世界支付公司的一台服务器上的软件更新导致了此次停机。尽管了解了更多关于停电原因的信息，我们仍然不知道该公司离处理所有积压付款有多近。

“不幸的是，我无法量化进展——可以理解的是，这样做会让人们远离解决问题的核心努力，”一位接近世界支付的匿名消息人士表示。

我们当然希望世界支付能够在内部了解自己的进步。该公司似乎正处于受影响的客户数量较少的顶端。

“世界支付意识到我们的一个网关上有一个孤立的问题，导致少数客户延迟结算，”世界支付在回应 Etsy 客户报告的初始问题时说。

当您收到客户的客户发来的 15，000 封令人沮丧的帖子时，您是导致问题的人，您不能继续称问题很小。1%、5%或 10%，这都是一大笔钱。

这些类型的停机在技术公司中很常见，但很少持续超过几个小时或几天。Etsy 客户在处理退款方面已经延迟了几周。

随着时间的推移，该公司似乎慢慢变得更加积极主动，或者至少更加直言不讳。昨日，该公司在其论坛上宣布，将在公司内部成立一个特别支付工作组，以解决客户遗留的问题。该公司已向买方和卖方发出两次收费退款，并要求任何仍受影响的人写信给 Etsy Support。

Etsy 支付产品总监斯蒂芬妮·格罗丁(Stephanie Grodin)表示:“受影响的会员将收到一封包含更多详细信息的电子邮件，并应在三到五天内收到重复付款的退款。”上周，我们制定了一些新的流程来提高我们平台的弹性，从那时起，新的支付已经得到了及时处理。"

Etsy 并不是唯一一个被支付处理器拖下水的 Worldpay 客户。出租车服务和赌博场所也在疯狂中分得一杯羹。这是 Worldpay 正在摸索的数百万美元的其他人的钱。

在这一点上，Etsy 继续使用 Worldpay 进行处理几乎是不负责任的。Etsy 在手工制品电子商务领域面临着激烈的竞争，需要尽可能多的东西向自己的方向突破，以对抗亚马逊和 Ebay 等行业巨头。亚马逊已经推出了类似 Etsy 的手工制作服务，尽管商家成本更高。

自首次公开募股以来，Etsy 一直表现不佳。Etsy 的股票在交易的第一天就从最高点下跌了超过 64%。Etsy 股票在过去五个交易日下跌了近 3.5%。这发生在该公司将于 8 月初公布的第二季度收益之前。

另一方面，目前金融界似乎没有人关心 Worldpay。此前，Worldpay 隶属于苏格兰皇家银行。该公司在几年后被出售给私募股权公司，现在已经上市交易。他们的股票在过去五个交易日上涨了 3%。
