# 由于电影制作人的性虐待自白，YouTube Red 取消了摩根·斯普尔洛克的《超码的我 2》的上映

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/12/18/youtube-red-cancels-release-of-morgan-spurlocks-super-size-me-2-following-filmmakers-sexual-abuse-confessions/>

YouTube 的订阅服务 YouTube Red 正在失去另一个更大的名字。纪录片导演摩根·斯普尔洛克的《超码的我》续集将从 YouTube Red 的 2018 年阵容中撤出，此前斯普尔洛克上周在推特上分享了他最近的性行为不端的自白。

斯普尔洛克周三承认了大学期间的一次性接触，听起来像是约会强奸，以及一名前雇员的性骚扰指控。他[在](https://web.archive.org/web/20221208105736/https://www.hollywoodreporter.com/news/morgan-spurlock-posts-lengthy-sexual-harassment-mea-culpa-i-am-part-problem-1067561)[的推特长文](https://web.archive.org/web/20221208105736/http://www.twitlonger.com/show/n_1sqc244)中发布了这些事件的细节，他还说，“我也是问题的一部分。”

他的坦白现在已经影响了 YouTube Red 迄今为止最大的交易之一。

谷歌的视频服务收到了摩根·斯普尔洛克导演的《超码我 2:神圣的鸡》从今年秋天的多伦多电影节以 350 万美元的价格，在当时被[称为](https://web.archive.org/web/20221208105736/http://deadline.com/2017/09/morgan-spurlock-super-size-me-2-youtube-red-toronto-film-festival-deal-1202164309/)的截止日期，既是“雄心勃勃的”，也是 YouTube 的“最大的电影节电影购买”

YouTube 在 9 月下旬证实了这笔交易，并补充说这部电影最初将于 2018 年在影院上映，然后在 YouTube Red 上独家全球首映。

这部电影本身意味着斯普尔洛克与食品行业斗争的另一个篇章，这一次他开了自己的快餐店。这位电影制作人之前曾与 YouTube Red 合作，于 2016 年发布了他的纪录片“Vlogumentary”，因此像这样的后续交易是有意义的。这被认为是 YouTube Red 的一大胜利，因为这部电影很快也会去圣丹斯电影节。

在他坦白之后，斯普尔洛克从《超码的我 2》的制作公司辞职。好莱坞报道表示，该公司现在已经从圣丹斯电影节撤回了这部电影，该电影原定在圣丹斯电影节首映。斯普尔洛克已经从斯蒂芬妮·苏希蒂格的纪录片《我们知道的魔鬼》中被删除，该片也将于下月在圣丹斯电影节上映。

YouTube 在一份与 THR 分享的声明中证实了撤下《超码我 2》的决定:

“我们同情所有受到摩根·斯普尔洛克最近言论影响的女性。鉴于这种情况，我们决定不在 YouTube Red 上分发超大号 Me 2，”一位发言人说。

斯普尔洛克的坦白是席卷好莱坞、科技行业、媒体等领域的一系列性骚扰和性虐待指控中的最新一起。这也不是第一次影响流媒体服务的发行计划或工作室编程的进一步发展。

例如，路易斯·C·K 的电影《我爱你，爸爸》在上映前就被叫停了，网飞与喜剧演员 T1 断绝了关系，就像与 T2、凯文·史派西、T4、丹尼·马斯特森和 T6 断绝关系一样，这些人都不相信马斯特森的指控者。亚马逊也受到了冲击，[失去了工作室负责人罗伊·普莱斯](https://web.archive.org/web/20221208105736/https://beta.techcrunch.com/2017/10/17/amazon-studios-head-roy-price-steps-down-amid-sexual-harassment-allegations/)和[“透明的”杰弗利·坦鲍。](https://web.archive.org/web/20221208105736/https://beta.techcrunch.com/2017/11/19/jeffrey-tambor-quits-amazons-transparent-after-being-accused-of-sexual-harassment/)

然而，这并不是 YouTube Red 第一次因为创作者的不良行为而失去高知名度的内容。今年早些时候，流行创作者 [PewDiePie 的 YouTube 红色系列被取消](https://web.archive.org/web/20221208105736/https://beta.techcrunch.com/2017/02/14/pewdiepies-youtube-red-series-gets-cancelled-after-vlogger-posts-anti-semitic-content/),因为该视频博客发布了反犹太人的内容。
