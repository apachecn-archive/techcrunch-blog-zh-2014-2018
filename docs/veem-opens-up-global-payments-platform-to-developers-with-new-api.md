# Veem 通过新的 API 向开发者开放全球支付平台

如果你曾经尝试过跨境做生意，你就会知道电汇、等待付款清算银行以及沿途支付一系列费用是多么痛苦。Veem 是一家初创公司，试图通过提供一个平台来简化企业之间的国际资金转移，从而简化中小型企业的所有工作。今天，它宣布以 API 的形式向开发者[开放该功能。](https://web.archive.org/web/20230302124807/https://www.veem.com/developers)

到目前为止，用户可以访问 Veem 网站，或者使用 Quickbooks、Xero 和 Netsuite 中的直接集成，这三个会计平台在 SMB(中小型企业)中很受欢迎。

要使用 Veem，相关各方只需在 Veem 网站上注册创建一个帐户。你只需要一封电子邮件，而不是需要大量的信息来进行转账。你输入收款人的金额和电子邮件，Veem 处理跨境转账，并根据汇率赚钱。

值得注意的是，银行电汇的一般外汇手续费通常在 4%左右。根据交易的不同，Veem 收取 0.5%到 2%的交换费，这使得它成为小企业使用起来更便宜、更方便的选择。

当小企业进行这种国际货币转账时，也需要处理一些规则和条例，Veem 也可以处理这些规则和条例，包括销售单或提货单以及收款银行可能要求的其他细节。

这就是将它开放给除了三个会计软件包之外的其他平台的真正亮点。它使程序员能够通过连接到 Veem API 将国际支付添加到任何应用程序中。

这不仅向开发人员开放了这一功能，极大地简化了实际上非常复杂的事情，还为 Veem 提供了一种在更多应用程序中出现的方式，而无需处理集成本身。它还提供了一种无需用户直接访问 Veem 站点即可使用 Veem 的方法。

该公司不收取开发者连接 Veem 的费用，这使得它成为程序员的一个有吸引力的选择。相反，它继续从汇率费用中赚钱

Veem 于 2014 年推出，已筹集超过 4400 万美元。
