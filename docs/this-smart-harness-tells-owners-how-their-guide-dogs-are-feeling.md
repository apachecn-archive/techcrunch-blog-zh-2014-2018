# 这个智能安全带告诉主人他们的导盲犬感觉如何

北卡罗来纳州立大学的研究人员发明了一种设备，可以通知用户他们的导盲犬是否感到不适或生病。因为狗通常通过它们的呼吸、姿势和心率来表达疾病或压力，所以对于盲人或其他残疾人士来说，当他们感觉不舒服时，很难分辨出来。这个设备可以测量狗的呼吸和心率，并在狗生病或中暑时振动挽具手柄来提醒主人。

北卡罗来纳州立大学计算机科学助理教授大卫·罗伯兹说:“狗主要通过它们的动作和姿势进行交流，这使得盲人很难或根本不可能时刻充分理解他们的狗的需求这种挑战在导盲犬身上尤为明显，它们被培育和训练成外表平静，避免在公共场合引人注意。"

该系统还可以告诉狗什么时候感到紧张或焦虑，这两者都是导盲犬退休的主要因素。该系统会轻轻震动，告知主人狗狗的状态。拇指的一个马达和狗的心跳同步，另一个马达在狗呼吸时振动。当这些马达中的任何一个振动太强烈时，主人可以停下来检查一下。

在马来西亚柔佛州举行的第二届国际动物计算机互动大会上,[项目将出现在一篇题为“盲人训练员对犬类生理的非视觉实时监控”的论文中。](https://web.archive.org/web/20221202042248/http://www.eurekalert.org/pub_releases/2015-11/ncsu-nth111615.php)
