Tapplock One 是一款指纹挂锁，具有可靠的智能技术

原文：https://web.archive.org/web/https://techcrunch.com/2018/03/09/the-tapplock-one-is-a-fingerprint-padlock-with-solid-smarts/?ncid=rss&utm_source=dlvr.it&utm_medium=twitter

苹果的触控 ID 在保护其移动设备的便利性方面迈出了一步，现在挂锁也提供了同样的便利。tap lock One现已发货，配有指纹传感器解锁，以及带有蓝牙解锁备份的配套应用。

Tapplock One 在 2016 年通过众筹活动在 Indiegogo 上开始存在，但现在它已经上市和发货，直接来自 tap lock 和一些零售合作伙伴。该设备具有 IP66 耐候性，可在 14°F 至 140°F 的温度下工作，一次充电的电池寿命长达一年。

Tapplock One 相对于其他智能挂锁的真正优势在于它的多功能性——你可以通过三种不同的方式解锁，包括如上所述的指纹和蓝牙，还可以使用“莫尔斯码”备份模式，即长按或短按电源按钮。这意味着，无论出现什么样的故障，只要电池还有电，你总有办法解决。

该应用程序还支持提供远程访问，这意味着你可以让其他人通过蓝牙解锁。这是一个聪明而方便的功能，尤其是如果你与 Airbnb 的客人或住在你家的任何人共用一个小屋或储物单元。

你可以从 Tapplock One 的重量和手感看出，它的构造非常坚固，事实上，到目前为止，在我的测试中，它很好地保护了我的户外小屋。它经受住了冰雪的考验，仍然可以通过锁面中央的小方形垫读取我注册的指纹。

Tapplock One 的主要弱点可能是其专有的充电器。这对耐用性和暴露在自然环境中的幸存来说是好的，但这是坏的，因为你需要锁附带的电缆，我不得不想象当需要一年一次充电时，你会很难记住你把它放在哪里。

Tapplock One 的售价仅为 99 美元，不会让你倾家荡产，尽管它比传统的钥匙或组合型更贵，但智能功能使它的使用极其方便。我不知道我拥有过多少挂锁，因为在停止使用几个月后我记不起密码，而结束了埋在抽屉里的有用生涯。
