Pixel 2休眠的视觉核心芯片在最新的Android开发者预览版中被激活

可悲的是，自谷歌最新旗舰发布以来，Pixel 2的惊喜并不都是好事。 Visual Core 是更令人愉快的一个，尽管它是一个定制的片上系统(SoC ),只是挂在手机上，并没有真正做什么。

当这款手机上市时，这种组件似乎还没有准备好面向公众消费，但该公司终于准备好让芯片开始赚钱了。

今天发布的Android 8.1 新开发者预览版提供了对Pixel 2/Pixel 2 XL芯片的早期支持。除了生产问题之外，该组件预计将进一步改善手机的摄像头，这些摄像头已经取得了良好的成绩。

据该公司称，Pixel Visual Core拥有8个图像处理单元(IPU)核心和512个算术逻辑单元。该公司表示，使用机器学习，它能够以十分之一的能源将速度提高5倍。对芯片的访问，结合Android相机API，意味着第三方照片应用程序将能够利用系统的快速HDR+。

听起来很棒，对吧？当然，这仍然只是早期的预览，只提供给注册谷歌测试版程序的人。这意味着，在其他事情中，处理早期构建的潜在错误。谷歌不会给我们更多关于何时向公众开放该功能的具体信息，但预计将在12月份随着8.1公开测试版一起发布。

即使该功能上线，它也只对Pixel 2和Pixel 2 XL用户开放——尽管如果它开始在未来的新Android手机中推出，也不要太惊讶。
