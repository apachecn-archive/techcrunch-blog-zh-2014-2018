Zero Motorcycles 首席技术官 Abe Askenazi 谈两轮电动汽车的未来 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/03/23/zero-motorcycles-cto-abe-askenazi-on-the-future-of-two-wheeled-evs/?ncid=rss

电动汽车和公共汽车已经开始接管世界，每天在美国道路上看到特斯拉、普锐斯、博尔特和 Proterra 等电动汽车品牌就是证明。但是两轮电动车仍然很少见。相对于汽车行业的同行，摩托车行业推出全电动和混合动力车型的速度要慢得多。

像哈雷、本田和杜卡迪这样的大品牌目前在市场上没有电动摩托车，尽管客户在取笑电动概念时表示了兴趣。一些初创公司未能制造出能与内燃机车型一较高下的电动自行车。

使命汽车破产。Brammo 将其 Empulse EV 摩托车高调推出，但这家位于俄勒冈州的小型初创公司将其自行车业务出售给了 Polaris。胜利摩托车的母公司北极星最终淘汰了Empulse。Brammo 现在向其他车辆和设备制造商出售动力系统。

不过，幸存者和新来者仍在大量生产电动摩托车。包括美国的 Zero 摩托、 Lightning 摩托和 Alta Motors ，意大利的 Energica Motor 和中国的evolve 摩托。所有这些都有助于唤醒市场需求。根据 ID Tech Ex 的预测，包括摩托车在内的小型电动汽车市场将占电动汽车总销量的 5%左右，到 2027 年每年产生约 350 亿美元的收入。

零摩 CTO Abe Askenazi。

TechCrunch 最近采访了 Zero Motorcycles 的首席技术官 Abe Askenazi，他已经在摩托车行业工作了大约 20 年，了解他对电动摩托车成为骑行者首选的看法。

Askenazi 说，现在，许多 Zero Motorcycles 的客户购买该公司的电动自行车以及市场领先的 V-twins 和 V-4。对他来说，这表明 Zero 的自行车表现与骑手们长期以来的最爱相同或更好。

除此之外，他说，电动自行车很安静，所以骑车人可以听到他们周围发生的事情，不会扰乱和平，他们的邻居对此很欣赏。这种安静是警察部门为他们的车队零购买摩托车的原因之一。他们可以在任何时间骑行，如果他们驾驶电动汽车，可以悄悄地接近嫌疑人。

当然，在世界上的大多数地方，电的成本比汽油低。这对一部分骑行者来说很有吸引力，对那些最环保的摩托车爱好者来说，用太阳能或风力发电为你的自行车提供动力也很有吸引力。

Zero 摩托车的双运动电动版 Zero DS。

Zero's today 生产从轻量级“热棒”到运动越野车型的各种产品，价格从 8495 美元到 15995 美元不等。这些自行车因其加速度、速度、控制和易于阅读的仪器而受到赞赏，但在过去，这些自行车在范围内引起了抱怨。然而，该公司一直在努力改进其电池技术，Zero 2017 年阵容中的一些自行车承诺每次充电可行驶 200 英里。相比之下，就在几年前，Zero 生产的电动摩托车续航里程为 40 英里。

“化学优化让我们走到了这一步。美妙的是，电池是即插即用的。首席技术官说:“当那种能让你行驶 1000 英里的电池到来时，我们就能使用它了。”“虽然这个行业的发展方向并不是更大的电池以在充电之间走得更远，而是更快的充电，这样你就可以去杂货店购物，插上电源，回来后就可以再次出发了。”

CTO 说，随着时间的推移，电池将在电动摩托车上占据更少的重量和空间，这超出了我们今天的想象。这可能会为货物和其他部件留出空间。Zero Motorcycles S 和 DS 自行车已经为骑行者在电池后面提供了一个大的存储空间。“这辆自行车给你带来了很多实用性，”Askenazi 说。他预测，更轻更坚固的材料也将改变设计。
