# Unity CEO 称一半的游戏都是基于 Unity  开发的

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/05/unity-ceo-says-half-of-all-games-are-built-on-unity/>

Unity 首席执行官 John Riccitiello 来到 TechCrunch Disrupt SF，向大家介绍世界上最受欢迎的游戏引擎的最新情况。你可能没有意识到你正在玩的大多数游戏，尤其是手机游戏，都是使用 Unity 构建的。

对于不熟悉游戏引擎的人来说，Riccitiello 一开始就非常清晰地描述了游戏引擎。回到过去，“[游戏开发者]会写出一个游戏程序，里面有很多艺术资产，很多动画，很多声音。但他们还必须编写渲染引擎，编写动画系统，编写声音系统，编写物理系统，”他说。

> 几乎占了所有游戏时间的一半。约翰·里奇泰洛

当你想把你的游戏移植到另一个平台时，你基本上不得不重新开始。Unity 可以在 30 个平台上工作，包括 Windows、iOS、Android、任天堂 Switch、Playstation 4、Oculus Rift 等。Unity 与 Epic 的虚幻引擎竞争，这是堡垒之夜和 PS4 和 Xbox One 上许多游戏背后的游戏引擎。

还有 Valve、亚马逊等不太受欢迎的游戏引擎。AAA 特许经营背后的最大游戏开发商(想想战地或刺客信条)有自己的内部引擎。

但很明显，Unity 已经占据了很大一块市场。根据 Riccitiello 的数据，每个月，人们会下载 20 亿份 Unity 游戏。人们在超过 30 亿台设备上至少尝试了一款 Unity 游戏。

以前 Unity 说[一半的*手机*游戏](https://web.archive.org/web/20230327143844/https://techcrunch.com/2017/05/23/unity-whose-software-powers-half-of-all-new-mobile-games-lands-400-million-from-silver-lake/)在 Unity 上运行。但事实证明，Unity now 也支持游戏机和电脑上的许多游戏。

“这几乎是所有游戏周期的一半。我们有不同的市场份额，这取决于平台。但是超过一半的手机游戏都是 Unity 开发的。Riccitiello 告诉采访者 Lucas Matney:“超过 60%到 70%的虚拟现实或增强现实或任何 XR 平台的机器都是在 Unity 中构建的，这取决于平台。”“然后，为任天堂平台开发的所有游戏中，有一半多一点是在 Unity 中开发的，比 Xbox 和索尼的略少一点。但总的来说超过了一半。”

Unity 如此成功的原因之一是它的定价结构对开发者友好。游戏公司不必将收入分成给 Unity，他们根据订阅情况向 Unity 付费。其他公司有时会要求你签署收入分成协议，以使用他们的引擎。

比数字更重要的是，Riccitiello 认为统一就是通过给创作者提供合适的工具来赋予他们能力。

“团结的动力是一个简单的哲学，即世界是一个有更多创造者的更好的地方，”他说。“我们推动自己将最强大的工具尽可能放在大大小小的创作者手中，让他们能够实现自己的梦想。”
