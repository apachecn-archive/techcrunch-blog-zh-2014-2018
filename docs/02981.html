Target 表示，2013 年至 2014 年，信用卡数据泄露导致其损失 1.62 亿美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/02/25/target-says-credit-card-data-breach-cost-it-162m-in-2013-14/

当谈到数据泄露时，零售商是这些天来最大的目标之一，今天我们有一些关于更高调的攻击之一的成本的细节。 Target 今天表示，它已经在 2013 年和 2014 年计入了 1.62 亿美元与其数据泄露相关的费用，黑客侵入公司网络获取信用卡信息和其他客户数据，影响了部分7000 万客户。

该公司今天发布的第四季度收益中披露的数字包括第四季度的 400 万美元，2014 年的总支出 1.91 亿美元，以及 2013 年的总支出 6100 万美元。Target 表示，2014 年和 2013 年的保险应收账款分别为 4600 万美元和 4400 万美元，部分抵消了上述总额。

这也不包括 Target 因违约后提起的集体诉讼或对其客户声誉的更广泛损害而可能产生的任何费用。一月份，一名联邦法官同意原告继续对该公司提起集体诉讼。

总体目标公布的收入为 218 亿美元，超过了分析师的预期，调整后每股收益为 1.50 美元，超过了预期。该公司还录得 51 亿美元的税前亏损，这与该公司退出加拿大业务有关。在盘前交易中，该公司的股票上涨了 1%多一点，达到每股 77.85 美元。

安全公司 FireEye 昨天发布的一份报告指出，零售商在 2014 年看到了最大的入侵高峰，平均而言，公司仍然需要 200 多天才能发现他们被黑客攻击。塔吉特公司表示，公司花了 12 天时间才弄清楚发生了什么。

像 Target 这样的公司的违规行为，以及最近发生在 Anthem 和索尼的例子，对消费者产生了如此直接的影响，以至于它正在改变关于公司如何认真对待安全的对话，这已经从单纯的 it 问题变成了一个触及公司最高管理层的问题。贝塞麦风险投资公司(Bessemer Venture Partners)的合伙人大卫·考恩(David Cowan)表示:“在(最近的一系列违规事件)发生之前，我得到了 It 安全，诸如此类，但我没有跑得过熊，我只要跑得过你就行了。现在你必须跑得比熊快。它在追你。"

对于塔吉特来说，这导致公司在去年 7 月任命了第一位外部人士首席执行官 Brian Cornell，在Gregg Steinhafel和其他高管离职后。
