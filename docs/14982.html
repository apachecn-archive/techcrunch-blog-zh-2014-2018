Airbnb 着眼于为拥有 100 万以上用户的网站提供会员计划，推出新 API TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/10/16/airbnb-eyes-expansion-with-affiliate-program-for-sites-with-1m-users-new-api/?ncid=rss/

Expedia 和 Priceline 想方设法通过为旅行者提供私人住宅清单来侵占 Airbnb 的空间，而由风险投资支持的 310 亿美元的 Airbnb 也在寻找新的方式来提高自己的流量和预订量。Airbnb 现在推出了一个针对附属合作伙伴的新服务，访客超过 100 万的高流量网站可以嵌入 Airbnb 的房产列表，并在预订时收取佣金。除此之外，Airbnb 现在已经为应用程序开发者打开了一个 API ，让他们可以在后端链接到其他应用程序，并提供一种更简单的方式来管理房源。

这两个步骤凸显了该公司正在寻找为下一阶段的增长开放平台的方法，同时还出现了一系列其他合作伙伴关系的扩张。

几周前，联合办公空间 WeWork 宣布将与 Airbnb 合作，为商务旅客提供联合办公和住宿双重服务。在佛罗里达州，Airbnb 一直在与一家开发商合作，建造一个 Airbnb 品牌的公寓楼——这是该公司可能绕过关于 Airbnb 在私人建筑中出租的一些更棘手的规则的一种方式。

API 和联盟计划都没有正式宣布，但是都被 Matteo Gamba 发现了，他是一名产品设计师，同时还写了关于 Airbnb 的博客。

Airbnb 过去有一个加盟计划，但于 2015 年 12 月终止。从那以后，该公司一直在根据具体情况与数量非常有限的公司合作(比如与日本两个受欢迎的旅游网站达成嵌入列表的协议)，这似乎是新的联盟计划的基础。

“Airbnb 一直与世界各地的合作伙伴保持着附属关系，”该公司发言人在一份声明中告诉 TechCrunch。“2015 年，我们终止了一些不活跃的合作伙伴，但保留了其他合作伙伴。新的登录页面为潜在合作伙伴提供了一个简单、单一的地方，让他们可以找到成为 Airbnb 附属公司所需的信息，并且在 Airbnb 运营的所有国家都可用。”

API——只能通过应用程序获得——也是一个有趣的变化。到目前为止，该公司还没有为第三方集成其列表提供一个广泛可用的 API。如果你环顾四周，你会发现开发者和其他人已经想出了替代的方法，比如使用一个在中间代理或抓取站点的人，来获得类似 API 的信息。(使用像 scraping 这样的技术违反了 Airbnb 的服务条款。)

该 API 的目的不是将 Airbnb 上发布的房源从其围墙花园带到其他网站(正如附属交易将实现的那样)，而是让开发者，特别是让主机管理其房产的应用程序的开发者，在后端与 Airbnb 连接，以便他们可以更容易地管理网站上的房产。

“API 允许开发团队安全地进入新的和现有的 Airbnb 账户，”Airbnb 指出。“您的应用程序用户将能够推送内容、费率和可用性的更新。当 Airbnb 客人预订您的房源时，我们会传回信息和预订细节，允许应用程序构建自定义工作流程，为我们的共享客人创造令人惊叹的体验。”

重新专注于建立联盟合作伙伴网络，以及 API 的推出，都表明 Airbnb 正在寻求与老牌旅游行业进行更密切的合作。例如，联盟营销是旅游行业网站创收的最大方式之一——通过聚集大量其他网站的库存和列表，然后从这些推荐产生的销售中获得一小部分。这也是人们最终预订东西的最常见方式之一。

该联盟网站目前没有列出任何具体的合作伙伴，也没有提供其背后的商业模式的细节，但这里的想法似乎是在人们已经打算购买旅游和相关服务的非 Airbnb 网站上提供 Airbnb 列表。

“赚取佣金，为你的用户提供更多选择，”Airbnb 写道，其中一个加盟选项旨在吸引客人，另一个旨在吸引主人(换句话说，那些会在 Airbnb 上列出房产或其他服务的人)——这是 Airbnb 双边市场的两个部分。

为了让你了解 Airbnb 可能会考虑将什么样的网站作为附属合作伙伴，据一位消息人士称，Airbnb 已经与 Groupon 讨论了附属合作伙伴关系。

(这很有意义:Groupon 本身一直在努力拓展其旅游和本地体验业务，作为其从本地零售商的日常交易中实现多元化的努力的一部分。最近的举措包括与 Viator 合作以增加其本地活动的库存，以及收购 LivingSocial，living social 在自己的日常交易业务暴跌后也重新调整自己以专注于该领域。)

对我来说，综合考虑最新的合作关系，这些举措标志着 Airbnb 的最新增长阶段。该网站最早的概念是，人们在家中提供多余的床和房间，作为住在更昂贵、没有人情味或游客太多的酒店的替代选择。然后发展到当他们不在的时候，人们提供他们的整个(常规的，不是度假的)房子。

很快转移到度假房产和通常不被业主占用的地方，然后代理人似乎同时管理几处房产。虽然 Airbnb 仍然有很多属于前三类的房产和主机，但它需要加速增长，才能真正实现 310 亿美元的估值，理论上，它的市值介于希尔顿和万豪之间。这也是它希望达到的目标之一:通过让房产的列表和管理变得更加容易，让更多的眼球关注它的库存，并迅速将感兴趣的浏览者转化为预订者。
