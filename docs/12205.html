基思·亚历山大将军要来扰乱纽约

作为美国国家安全局(National Security Agency)任职时间最长的局长，基思·亚历山大(Keith Alexander)肯定对网络安全和公司面临的来自老练的民族国家黑客的威胁略知一二。这就是亚历山大从美国国家安全局退休后，于 2014 年创立 IronNet 网络安全公司的原因。

在 IronNet 成立后的三年里，网络安全已经成为一个全国性的重要问题。Alexander 最近在参议院情报特别委员会(Senate Select Committee on Intelligence)就俄罗斯干涉 2016 年选举一事作证，描述了俄罗斯在竞选期间如何利用 T2 的宣传，并就美国应如何反击这些努力提出了建议。

亚历山大还谈到了外国情报监视法案，这是一项有争议的法律，授权某些监视项目，今年将由国会重新授权。“我认为这是最重要的项目，”亚力山大在谈到 FISA 时说。

在 IronNet，Alexander 专注于开发技术来检测和警告企业网络上的可疑行为。他还致力于行业和政府之间的情报共享，以便双方能够更快地检测和应对网络入侵。IronNet 吸引了风险投资界的极大兴趣——到目前为止，它已经筹集了超过 3200 万美元。
