# Twitter 将其黑暗的“夜间模式”主题带到了网络上

Twitter 的深色“夜间模式”界面主题[在 iPhone 和 Android 应用上首次亮相后，现在正在其网络客户端](https://web.archive.org/web/20221007041119/https://twitter.com/TwitterSupport/status/905477696360112129)上推出。黑暗主题将让你更容易地从你的蝙蝠洞里监控关键的实时信息，以帮助确保哥谭市的安全，如果你碰巧是蝙蝠侠的话——它还可以缓解你在晚上或光线较暗时使用 Twitter 网站的眼睛疲劳。

“夜间模式”的外观实际上是深蓝色，而不是黑色或灰色，这可能是更好的选择。这也让我的封面艺术(一张我购买的由了不起的[加拿大漫画家和插画家 Michael Cho](https://web.archive.org/web/20221007041119/http://chodrawings.blogspot.ca/) 绘制的 x 战警 Charles Xavier 的照片)看起来更加吓人，这是我喜欢的。

这可能不是每个人都可以立即使用的，但如果可以，你可以通过点击 Twitter 网络界面右上角的图标，并点击弹出菜单最底部的“夜间模式”开关来启用它。我会推荐它——哪怕只是因为它似乎更适合现代 Twitter。
