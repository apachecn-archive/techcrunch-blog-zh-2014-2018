微软推出新的Azure备份和恢复服务

微软今天宣布了其 Azure Backup 服务的新功能(预览版)。此外，该公司正在为其 Azure Site Recovery 服务推出新功能。

该公司显然正在将这两种服务面向企业，并指出它们将对迁移到混合云部署的公司特别感兴趣。使用Azure Backup，企业可以确保始终备份SharePoint、Exchange、SQL Server、Windows客户端和Hyper-V虚拟机应用程序。目前，这项服务的重点是本地托管的机器，但微软也在为驻留在公司现有数据中心的数据提供Azure Backup的预览版。

同样，企业可以使用站点恢复来保护他们在Azure上的VMware虚拟机和服务器。该服务允许企业远程监控其机器的健康状况，并创建可定制的恢复计划。几乎在默认情况下，虚拟机应该是易于复制的。

这里的不同之处在于，微软的服务重点不在于单个机器或小型虚拟机部署，而是企业规模的数据中心。

如果您的主数据中心发生任何问题，站点恢复可确保您能够快速回切到您的内部基础架构。微软指出，企业也可以使用这项服务来测试其应用程序的新版本，然后使用故障回复中心将它们推向生产。

因为今天是微软的好日子，该公司还将把它的Azure API Management Premium服务推出预览版，并正式推出Azure Active Directory的几项增强功能，包括密码滚动和基于应用的多因素身份验证。
