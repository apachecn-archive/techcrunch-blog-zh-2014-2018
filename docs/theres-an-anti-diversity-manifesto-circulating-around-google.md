# 公司内部流传着一份批评谷歌多元化努力的宣言

据发布备忘录的 Gizmodo 报道，[一位谷歌员工反对公司多元化政策的长篇文章似乎正在公司内部流传。](https://web.archive.org/web/20230325070446/http://gizmodo.com/exclusive-heres-the-full-10-page-anti-diversity-screed-1797564320)

Motherboard first [报道了正在流传的文件](https://web.archive.org/web/20230325070446/https://motherboard.vice.com/en_us/article/kzbm4a/employees-anti-diversity-manifesto-goes-internally-viral-at-google)的存在，谷歌在 Twitter 上谴责了这一行为。在这篇文章中,“宣言”的作者似乎试图证明，技术上的性别差距不是歧视的产物，而是男女之间固有的生理差异。

Gizmodo 发布的备忘录开头写道:“我重视多样性和包容性，不否认性别歧视的存在，也不支持使用刻板印象，”[。“在解决人口代表性的差距时，我们需要研究人口水平分布的差异。如果我们不能对此进行诚实的讨论，那么我们就永远无法真正解决问题。心理安全建立在相互尊重和接受的基础上，但不幸的是，我们羞辱和歪曲的文化不尊重和不接受其回音室之外的任何人。”](https://web.archive.org/web/20230325070446/http://gizmodo.com/exclusive-heres-the-full-10-page-anti-diversity-screed-1797564320)

**更新:**看起来主板[有来自 Danielle Brown](https://web.archive.org/web/20230325070446/https://motherboard.vice.com/en_us/article/vbv54d/google-on-anti-diversity-manifesto-employees-must-feel-safe-sharing-their-opinions) 的内部回应，Danielle Brown 是谷歌多样性、完整性&治理的新副总裁。据 Motherboard 报道，以下是她所说的部分内容:

“建设一个开放、包容的环境意味着培养一种文化，让那些持有不同观点(包括不同政治观点)的人能够放心地分享他们的观点。但这种讨论需要与我们的行为准则、政策和反歧视法中的平等就业原则并行不悖。”

据 Motherboard 称，布朗还表示，该文件“不是我或该公司认可、推广或鼓励的观点”。

随着创业生态系统和优步等公司中性别歧视的故事公之于众，整个硅谷正处于巨大的争议浪潮中。我们联系了谷歌核实这份备忘录，并询问该公司是否计划对此采取任何行动。然而，在 Twitter 上，一些谷歌员工非常愤怒。
