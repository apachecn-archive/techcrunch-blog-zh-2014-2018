微软团队增加了访客访问和对更多开发者工具的支持

微软今天宣布，181 个市场中的 125，000 个组织现在使用团队，这是其 Office 365 用户的松散竞争对手。截至 1 月底，已有 145 个市场的 30，000 家机构采用了这项服务。

除了这一势头更新，该公司还宣布了该服务的一些功能更新。从今天开始，你可以将任何拥有 Azure Active Directory 帐户的人作为来宾添加到团队中。微软表示，目前在自己的商业服务和第三方 Azure Active Directory 应用程序中有 8.7 亿个 Active Directory 用户帐户。这仍然意味着访客访问存在一些障碍，微软计划在不久的将来通过允许团队用户将任何拥有基本 Microsoft 帐户的人添加到团队来降低障碍。

鉴于这是一个企业工具，it 管理员能够控制访客访问并在必要时撤销它就不足为奇了。

团队中的新功能还包括对更受欢迎的机器人构建工具之一 Botkit 的支持，以及与 GitHub、Atlassian 的 JIRA 服务台、核心和软件以及 Bitbucket Cloud 的新集成。这是一个有趣的举动，考虑到 Atlassian 上周推出了自己的竞争对手 Slack 和 Teams，但考虑到许多企业开发人员混合使用 Atlassian 和微软的工具，两家公司在彼此的平台上存在是有意义的。
