# 达芬奇智商是一个赛博朋克蒸发器有一个很好的借鉴

港口上方的天空可能是电视的颜色，调到一个死亡频道，但在大多数黑客巢穴中，通常都是迷幻药和健康小吃。还有什么比真正的赛博朋克音乐更好的方式来分享你最喜欢的精神药物呢？

DaVinci IQ 是一个黑色的平板，用于将草药加热到 230 到 430 华氏度。嵌在表壳中的明亮 LED 灯网格显示电流路径——可在 1 到 4 之间设置的热信号——或温度。一个单独的应用程序让您可以通过为各种情况分配各种配置来进一步控制设置。最后，一个增强模式可以尽快地从草药中获得最大的收益。

这款售价 275 美元的 IQ 看起来像一个未来的电击枪或人工智能子弹，整个包装是独立的，可清洁的。它包括一个可拆卸的可充电电池，通过机身上的微型 USB 端口充电。它有两个顶部——一个平顶和一个看起来像吸管的“液体”尖端。两者都工作得很好，虽然我更喜欢吸管状的。智商也有一个“味道室”，可以容纳干药草。蒸汽穿过干燥的材料，但没有被加热。人们的想法是这种干燥的药草会带来更多的味道。

我并不经常使用包括烟草在内的任何草药，但在过去几个月里，我尝试了许多这种高科技的 vapes，发现这种更有吸引力，味道也稍微多一些。以前的 vapes，如 Pax，以更小的蒸汽羽和更少的味道结束，而 IQ 通常提供足够的蒸汽吸入并最终感受到。

电池充电约 40 分钟，但充电很快。设备完全加热大约需要 30 秒。我特别喜欢将药草推入蒸发器的陶瓷“按钮”。这可以让你更好地收拾东西，也更容易清洁。用过的材料可以用一个小塞子拔出来。

底线呢？我的智商真的很高。今年我已经尝试了很多蒸发器，这个总是能产生最美味的蒸汽。它也是设计最安全的。其他设备有可以滑落的小盖子。智商的顶部和底部牢牢固定，即使它只被强力磁铁吸住。

这个赛博朋克吸烟机的外观、感觉和实用性使它非常独特和实用。这也是我第一次品尝到这种味道，并且感觉很有活力。毕竟，在致命的黑冰层中穿行需要付出精神代价，有东西让我们放松是件好事。
