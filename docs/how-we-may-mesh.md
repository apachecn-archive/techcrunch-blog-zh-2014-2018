# 黑客文化和行为准则之间的战争

您知道开源软件开发领域正在进行一场行为准则战争吗？我知道这听起来很荒谬。行为准则可以归结为:“a)不要做混蛋，b)这就是我们在这些地方对‘混蛋’的定义”。谁会反对呢？然而，这已经引起了激烈的争议——而且是有充分理由的。

在 Model View Culture 的“[新常态:2015 年及以后的行为准则](https://web.archive.org/web/20230328180435/https://modelviewculture.com/pieces/the-new-normal-codes-of-conduct-in-2015-and-beyond)”中可以找到近期历史的快速概要(如果你觉得 MVC 太左翼，[这里是布莱巴特的 take](https://web.archive.org/web/20230328180435/http://www.breitbart.com/tech/2016/01/25/ruby-hackers-in-revolt-after-sjws-attempt-to-impose-politically-correct-code-of-conduct/) ，但是前者有更多实际信息。)简而言之，进步主义者正试图说服开源社区采用行为准则，这样:

> 它们为参与者之间的互动建立了基本规则。它们概述了违法行为的执行机制。它们作为一种信号机制，一种传播组织者或维护者意图的方式，为社区成员的全面参与创造安全的空间，无论种族、宗教、民族、性别、性偏好或身体能力如何。但最重要的是，它们编纂了特定社区的愿望和价值观

一个被广泛采用的 CoC 是[贡献者契约](https://web.archive.org/web/20230328180435/http://contributor-covenant.org/)，其序言指出:

> 精英管理还天真地假设了一个公平的竞争环境…这些因素和更多因素使得对许多人来说，尤其是女性和其他未被充分代表的人来说，为开源做贡献是一个令人畏惧的前景

到目前为止一切顺利！……但是关于这些准则最有争议的是，它们的职权范围实际上延伸到了相关社区之外。

MVC 的那篇文章提到了“所谓的 Opalgate 事件，其中一名 Opal 维护者在 Twitter 上发布的跨性别推文引发了一场关于他继续代表该项目的适当性的激烈讨论。”在真正的开源风格中，这是作为[Github](https://web.archive.org/web/20230328180435/https://github.com/opal/opal/issues/941)上的一个问题提出的——*trans phobic maintainer 应该从项目*中移除——我认为[的回应是](https://web.archive.org/web/20230328180435/https://github.com/opal/opal/issues/941#issuecomment-113222493):

> 维护者的个人观点，只要他们没有以歧视的方式影响项目中接受的贡献，就不应该成为讨论的禁区

总结了这里的争论。社区是否应该接受持有不同观点的人，只要他们不在社区内表达他们的观点？或者他们应该被驱逐，因为他们的观点会对他们的社区工作产生负面影响，或者因为他们的存在让其他人感到不安全？

就个人而言，*两个*答案都让我深感不安。人类是杂乱的、复杂的、矛盾的；人类的互动是平方的；结果是如此的复杂和敏感，以至于他们经常需要在个案的基础上进行判断，而不是通过任何硬性的规则。(参见:精神错乱的“[零容忍](https://web.archive.org/web/20230328180435/http://reason.com/blog/2015/10/05/a-short-sad-history-of-zero-tolerance-sc)”规则。另见:奥柏林拒绝解雇一名将 9/11 归咎于以色列的教授。)

去年，古怪的新反动分子孟子，又名柯蒂斯·雅文，被取消了奇怪循环技术会议的邀请，他将在会上做一个与他古怪的新反动观点无关的技术报告。正如大卫·奥尔巴赫在《候选人名单》中所说:

> 由于雅文演讲的内容不是问题，而是思想家所持有的其他信念，那么任何持有有问题的信念的人现在都要被清除[……]当涉及到像奇怪的循环和一般学术场所这样的聚会时，我们应该尽可能广泛地划定可接受的演讲的界限，因为人们有一个可怕的错误习惯[……]科学的一个优势是它能够……促进那些讨厌彼此讨论任何其他主题的人之间富有成效的对话。

一些——很少——人是不可救药的混蛋，无论他们去哪里，都需要班汉默的迎接。你当然可以证明 Moldbug/Yarvin 和 transphobic Opal maintainer 都属于这一类。但毫无疑问，一个理想的行为准则实际上应该努力“促进那些讨厌彼此讨论任何其他话题的人之间富有成效的对话”，至少在可能的范围内，而不是将任何在任何地方的任何论坛上表达了不可接受的观点的人排除在对话之外。

我想起了梅瑞狄斯·帕特森的经典作品“当书呆子们发生冲突时:我的交集会有古怪的人，否则就是废话。”我在这里给她最后一句话:

> 在成功的第一个迹象出现时，让他们建立的社区从他们手中夺走，现在已经成为古怪屋的一个信号特征。如果它不是一直发生，我们就不会一直提到它。
> 
> 我并不认为这是完全理性的，因为恐惧不是理性的，但它确实解释了当被告知我们的文化已经破碎，必须调整以适应那些通过避开我们而让我们融入他们的文化的人时的反应。
> 
> 当我们得到一些暗示——任何暗示——我们的批评者明白我们文化中的哪些部分是我们不想失去的，以及为什么我们不想失去它们时，我们会开始感觉不那么有防御性。
