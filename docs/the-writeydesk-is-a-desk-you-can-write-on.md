# 写字桌是一张可以写字的桌子

无论你是一个默默无闻的数学天才，喜欢在所有东西上写复杂的方程，还是一个该死的破坏者，你都会发现 WriteyDesk 非常有用。这个测试本质上是一个大白板，你可以在上面写，画，或者画草图，而且是可擦除的。它有白色或“桦木”色，在 Kickstarter 上售价 300 美元(零售时售价 400 美元)。

“但是，家伙写了一张桌子，你可以在上面写字，”你说。"如果你把墨水弄脏了会怎么样？"我不知道，但是你最好小心点。如果你没有接受过水平干擦素描艺术的适当训练，我可以想象你的衬衫袖子和手腕一辈子都会被这个东西弄脏。

这个产品是由制造[将墙壁变成白板](https://web.archive.org/web/20221219155920/https://techcrunch.com/2012/05/17/backed-by-mark-cuban-whiteyboard-launches-v2-of-its-paint-that-turns-walls-into-whiteboards/)的涂料的同一批人制造的。他们得到了马克·库班的支持。

该团队正试图在 Kickstarter 上筹集 25，000 美元，目前进展缓慢，但如果你想在一个兼做白板的工作台上达成协议，这样你也可以在桌子上写晦涩难懂的方程和/或绘制粗糙的图片，那么这就是你要的东西。
