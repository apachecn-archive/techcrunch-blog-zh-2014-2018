# 以下是马克·扎克伯格今天听证会的笔记

脸书的马克·扎克伯格在今天的参议院联合听证会上顺利亮相，回避了大多数问题，同时在五个小时的各种各样但大多温和的提问中保持了足够的耐心。

这位首席执行官避免承认脸书是出版商或垄断者，拒绝承诺任何有意义的立法，并在近五个小时的马拉松式证词中恭敬地向立法者发表讲话。

尽管如此，他还是犯了一个新手的错误。

扎克伯格将他的听证会笔记在座位前打开足够长的时间，以便[美联社摄影师 Andrew Harnik](https://web.archive.org/web/20221230135631/http://www.apimages.com/metadata/Index/Facebook-Privacy-Scandal-Congress/043bf11306c2434bbca255a55d2b87fa/19/0) 拍下清晰可见的高分辨率照片。Twitter 用户和记者在法庭上浏览照片时很快就注意到了这一小小的隐私侵犯的讽刺意味。

大多数笔记都涵盖了我们在听证会期间听到扎克伯格重复的观点，但还有一些更坦率的陈述没有出现。这些笔记也让我们得以一窥脸书所期待的问题。首先，他们预计国会可能会要求他辞职。
