# WhatsApp For the Workplace 的 Cotap 增加了可通过 URL 访问的通用聊天功能 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/12/10/whatsapp-for-the-workplace-cotap-adds-universal-chat-feature-accessible-through-urls-not-the-app/>

由前 Yammer 高管创建的企业消息应用[Cotap](https://web.archive.org/web/20221207013234/http://www.crunchbase.com/organization/cotap)宣称自己是“工作场所的 WhatsApp ”,推出了一项新功能来扩大使用该应用的人群:它现在允许其他人通过 [Cotap.me](https://web.archive.org/web/20221207013234/http://cotap.me/) 上的 URL 向这些用户[发送消息，而无需下载应用本身。](https://web.archive.org/web/20221207013234/http://www.streetinsider.com/Press+Releases/Cotap+Launches+Universal+Chat+Service+Cotap.me/10086008.html)

在此之前，该产品于 10 月份推出了一款桌面应用，使用了 Cotap 的一些相同功能——当同事们不在移动设备上而是在电脑上时，可以使用同事之间的交流。今天的推出有效地让任何人都可以使用该应用的聊天功能，通过个人给出的自定义网址。

在应用程序中添加一个 web 组件意味着那些 Cotap 工作用户现在可以与其他人共享他们的 Cotap IDs 比如，他们可以在电子邮件签名或名片中使用 Skype 或电子邮件地址或手机号码——作为任何人的联系人 ID。

虽然市场上有许多应用程序围绕使用网络界面让人们与本地聊天客户端互动的原则而构建，但其中许多应用程序更多地关注消费者而不是企业用户。作为一家初创公司，Cotap 的部分使命是利用商业应用的消费化趋势，即采用消费应用中更常见的功能，并将它们引入其以企业为中心的应用。这是其中的一个案例。Cotap 说它已经自己开发了这个功能。

![Screen Shot 2014-12-10 at 17.44.56](img/a10dffb9c0d9adac48a16ed192755963.png)

URL 功能背后的想法是，虽然你可能已经在 Cotap 上建立了网络并与同事进行了持续的对话，但 Cotap.me 允许你将第三方引入对话，或者如果你已经在该应用程序上花费了大量时间，则简单地将该平台用作其他人也可以与你聊天的地方。

在与非工作同事互动时，给出 Cotap URL 而不是电话号码也可以帮助人们保护他们的一些隐私——例如，如果员工在客户服务部门工作，不想分享电话号码，但希望客户可以联系到他，这类似于人们使用 Dropbox 或 Google Drive 等服务与第三方共享文档的方式。

“Cotap.me 是一种简单的方式，可以让您的所有业务联系人立即联系到您，而无需分享您的个人电话号码或要求任何人下载单独的应用程序。Cotap 的首席执行官和联合创始人吉姆·帕特森在一份声明中说:“这是一种快速、便捷的方式来开始一场对话或加快解决电子邮件中的问题。“今天，你可以通过 Box、Dropbox 和 Google Drive 的 URL 与同事共享文档，现在，有了 Cotap.me，你可以以同样的方式共享对话。”

在这方面，Cotap 与 Twilio 等服务站在一起，作为企业如何让客户服务代表能够使用他们的手机与客户沟通而不泄露他们自己的个人信息的另一种选择。

(Twilio 通过不同的方式达到了类似的目的:它[为企业提供独特的电话号码](https://web.archive.org/web/20221207013234/https://beta.techcrunch.com/2014/09/18/twilio-adds-low-cost-two-way-mms-picture-messaging-to-its-api-cloud/)，它们可以用这些号码打电话或发短信给业务关系。)

Cotap 已经从 CRV 和 Emergence Capital Partners 筹集了 1550 万美元。帕特森说，它还没有透露用户数量，但很快会有消息。
