# 您现在可以通过 Skype 的移动应用 TechCrunch 使用 PayPal

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/08/02/you-can-now-use-paypal-through-skypes-mobile-app/>

贝宝的合作伙伴不断出现。这家仍在增长的支付巨头今天宣布与 Skype 达成一项新协议，允许全球 22 个国家的用户通过 Skype 移动应用程序的更新版本向其他 Skype 用户汇款。这极大地扩展了 PayPal 的潜在覆盖范围——Skype 应用程序迄今已被下载超过 10 亿次，根据 Skype 母公司微软的数据，截至去年，每月约有 [3 亿](https://web.archive.org/web/20221210002204/http://www.seattletimes.com/business/microsoft/skypes-evolution-at-microsoft-could-hint-at-plans-for-linkedin/)活跃用户。

要使用该功能，您需要安装最新版本的 Skype mobile 应用程序。然后，在与朋友或家人交谈时，你可以选择顶部栏上的“查找”，并选择“汇款”插件。您只需输入要发送的金额，确认付款，然后点击发送即可完成交易。

需要说明的是，该功能是为朋友和家人之间的汇款而设计的，而不是为企业的商品或服务付款。和 PayPal 在其他平台上的点对点支付一样，费率是一样的。也就是说，当你在美国使用 PayPal 余额或借记卡时，发送是免费的。如果你选择使用信用卡，费用是交易金额的 3.4%加 0.30 美元。

这不是 Skype 第一次在其应用程序中提供由 PayPal 支持的点对点支付，不过如果你还记得以前宣布这种集成的时候，你可能会觉得自己有点过时！

事实上，这要追溯到 2007 年，当时易贝还在运营 Skype(2011 年被微软收购之前的 T4)和 PayPal(2015 年剥离之前的 T7)。那年春天， [Skype 在桌面上推出了一项功能](https://web.archive.org/web/20221210002204/http://www.marketwatch.com/story/skype-adds-paypal-payments?siteid=yhoo&dist=yhoo)，允许用户通过 PayPal 互相汇款，这是易贝整合其各种服务的更大计划的一部分。

然而，该公司表示，这是 Skype 首次通过 PayPal 在手机上启用汇款功能。

这是 PayPal 最近几个月宣布的一系列合作伙伴关系中的最新一个，PayPal 正在努力巩固其地位，以应对 Apple Pay 等威胁，甚至是美国银行自己的类似 Venmo 的服务 Zelle T1。它通过与以前的竞争对手合作来做到这一点，[包括苹果](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2017/07/12/paypal-expands-apple-integration-will-become-a-payment-option-in-11-new-markets/)，以及 [Visa](https://web.archive.org/web/20221210002204/http://www.wsj.com/articles/paypal-strikes-partnership-with-visa-1469135090) 和[万事达卡](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2016/09/06/paypal-partners-with-mastercard-for-store-payments/)等等。

该公司最近还宣布与[三星支付](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2017/07/17/paypal-to-become-a-payment-option-in-samsung-pay-including-in-app-online-and-in-store/)、[安卓支付](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2017/04/18/paypal-teams-up-with-android-pay-for-mobile-payments/)和[百度](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2017/07/27/baidu-teams-up-with-paypal/)整合；并延长了与花旗和大通的合作协议。在技术方面，PayPal 推出了[即时银行转账](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2017/06/20/instant-bank-transfers-are-coming-to-paypal-and-venmo/)、 [Siri 集成](https://web.archive.org/web/20221210002204/https://www.paypal.com/stories/us/hey-siri-sending-receiving-money-with-paypal-is-now-easier)、支持支付的 [iMessage](https://web.archive.org/web/20221210002204/http://blog.venmo.com/ios10) 、 [Slack](https://web.archive.org/web/20221210002204/https://beta.techcrunch.com/2017/02/07/paypals-first-payments-bot-is-a-slack-app/) ，以及微软的[Outlook.com](https://web.archive.org/web/20221210002204/https://www.paypal.com/stories/us/make-sending-and-receiving-money-simple-paypal-collaborates-with)。

PayPal 表示，汇款功能将于今天开始向美国和其他地方的 iOS 和 Android Skype 用户推出。

受支持市场的完整列表包括:美国、英国、奥地利、比利时、加拿大、塞浦路斯、爱沙尼亚、芬兰、法国、德国、希腊、爱尔兰、意大利、拉脱维亚、卢森堡、马耳他、荷兰、
葡萄牙、圣马力诺、斯洛伐克、斯洛文尼亚和西班牙。
