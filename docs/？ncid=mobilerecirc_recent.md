# 苹果收购了数字报刊亭，因为它对来自可靠来源的内容加倍下注

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/03/12/apple-acquires-digital-newsstand-texture-as-it-doubles-down-on-content-from-trusted-sources/?ncid=mobilerecirc_recent>

随着围绕假新闻以及脸书等聚合器在传播假新闻中扮演的角色的辩论继续进行，苹果正在进行一项收购，这可能有助于其奠定可信信息提供商的地位。这家 iPhone 制造商正在收购被称为“杂志出版界的网飞”的杂志虚拟报摊 [Texture](https://web.archive.org/web/20230307142251/http://texture.com/) ，读者可以以每月 9.99 美元的价格获得约 200 种杂志。

“我们很高兴 Texture 将加入苹果，以及来自许多世界领先出版商的令人印象深刻的杂志目录，”苹果互联网软件和服务高级副总裁 Eddy Cue 在一份声明中说。“我们致力于从可靠的来源获得高质量的新闻，并允许杂志继续为用户制作设计精美、引人入胜的故事。”

据我们所知，Texture——以前称为 Next Issue——将继续保持原样运营，没有任何变化——这意味着它将继续为 iOS、Android、亚马逊 Kindle Fire 和 Windows 8 和 10 提供应用程序。苹果正在收购整个公司，包括员工，预计交易将很快完成。

“我很高兴 Next Issue Media 及其屡获殊荣的纹理应用程序被苹果收购，”Next Issue Media / Texture 的首席执行官约翰·拉夫林在一份声明中说。“Texture 团队及其目前的所有者 Condé Nast、Hearst、Meredith、Rogers Media 和 KKR 对这一进展感到无比高兴和激动。我们想象不出比这种服务更好的归宿和未来。”

苹果已经在 Texture 专注的出版和媒体领域进行了大量收购，其中包括 Spotify/Pandora 的竞争对手 Beats for Apple Music，以及 [BookLamp](https://web.archive.org/web/20230307142251/https://techcrunch.com/2014/07/25/apple-booklamp/) ，我们在宣布收购消息时将其称为“书籍的潘多拉”。Texture 并不是苹果在杂志出版领域的第一次收购:它在 2014 年[收购了 Prss](https://web.archive.org/web/20230307142251/https://techcrunch.com/2014/09/23/prss-digital-magazine-platform-acquired-by-apple/) ，以帮助[建立苹果新闻](https://web.archive.org/web/20230307142251/https://twitter.com/JochemWijnands/status/973533809965780992)。

这笔交易的财务条款尚未披露，Texture 也从未披露其估值。

在 2015 年更名之前，Texture(当时被称为 Next Issue Media)是康泰纳仕、赫斯特、梅雷迪斯、新闻集团、罗杰斯通信公司和时代公司的合资企业。该公司似乎已经从出版商那里筹集了至少 9000 万美元[、4000 万美元](https://web.archive.org/web/20230307142251/https://nypost.com/2016/06/21/the-netflix-of-magazines-is-about-to-get-a-lot-bigger/)，另外从 KKR 和其他投资者[那里筹集了](https://web.archive.org/web/20230307142251/https://www.crunchbase.com/organization/next-issue-media#section-funding-rounds)[5000 万美元](https://web.archive.org/web/20230307142251/https://www.prnewswire.com/news-releases/next-issue-media-announces-50-million-investment-by-kkr-300003938.html)，其他投资者包括 BuzzFeed、Vox Media 和 Imgur。

纹理，在它下一期之前，从 2010 年就有了。它是在其他一些数字报刊亭上市的时候推出的。

利用应用程序的新普及，以及这些应用程序将成为消费者阅读报纸和杂志的主要方式的信念，出版商还认为，这可能是他们更好地将内容货币化的一个关键方式，此前他们错过了在线内容付费墙和从在线广告中获得巨大利益的机会，而谷歌等大型聚合器在这些领域获得了巨大的回报。

目前还不清楚 Texture 有多少用户。2014 年的一篇文章[估计这一数字为 15 万，而 2016 年的这篇文章](https://web.archive.org/web/20230307142251/https://www.wsj.com/articles/next-issue-a-digital-magazine-service-gets-kkr-funding-1417555373)指出这一数字在“几十万”之间，预计未来几年将增长 50%。该公司还与 Sprint 等第三方达成协议，将该服务与订阅捆绑在一起，作为一种甜头，让消费者选择他们的服务，而不是其他移动运营商的服务。

也不清楚苹果打算如何长期处理纹理。

我们可以看到该产品的一个领域是苹果新闻，苹果已经提供了各种第三方内容的访问。更广泛地说，该公司一直专注于在其他媒体上播放更大规模的优质内容，对音乐、视频和播客进行了大量投资。纹理用阅读材料填充了视觉的范围。

当然，苹果曾经有过自己的“报摊”——特别是它自己的本地应用程序。这项服务从未真正起飞，苹果最终扼杀了该产品，并将其并入苹果新闻。

与众不同的一个原因是，它将本质上把许多杂志纳入一种格式，而不是提供一个本质上不同的杂志和报纸应用程序的市场，从用户和出版商开发者的角度来看，这似乎是最初的报刊亭的一个痛点。

(我不得不说，当时是谁规定了这种格式，这一点从来都不清楚:很可能是苹果和 2011 年报刊亭首次出现时的技术限制，尽管也有其他应用程序可以解决这个问题。同样可能是出版商认为，从零开始建设，类似于他们自己的垂直整合印刷机构，才是前进的方向。)

但从另一个意义上来说，这次收购对苹果这样的公司来说也只是简单的桌面股权。亚马逊去年推出了自己的订阅服务，谷歌当然也通过 Google Play 提供各种报摊，所以这也是为了保持和确保它也能继续提供所有设备所有者越来越想要和期望的东西。

苹果的提示[今天也走上了奥斯汀 SXSW 的舞台](https://web.archive.org/web/20230307142251/https://techcrunch.com/2018/03/12/eddy-cue-explains-apples-original-video-content-strategy/)，他谈到了 Texture 在苹果新闻工作中的地位以及该公司更广泛的媒体和内容战略。
