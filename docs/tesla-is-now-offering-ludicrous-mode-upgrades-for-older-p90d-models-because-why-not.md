# 特斯拉现在为老 P90D 车型提供可笑的模式升级，为什么不呢

想让您的 P90D 快 20%吗？当然，不用担心。特斯拉现在允许在购买时跳过疯狂可笑模式升级的 P90D 购买者以 10，000 美元的价格将其添加到他们的汽车中。该公司声称，升级将使汽车的 1/4 英里时间降至 10.9 秒，0-60 时间降至 2.8 秒。因为，你知道，那些你因为沙丘路的交通而迟到的会议。

订购 P90D 时，买家可以选择添加滑稽模式。到目前为止，如果在购买时没有检查这个选项，买家就会被困在一辆只能在 3.2 秒内完成 0-60 的汽车中。

升级到可笑模式包括交换保险丝和电池组。综合起来，这将最大输出从 1，300 安培增加到 1，500 安培，简单地说，这使快速汽车跑得更快。
