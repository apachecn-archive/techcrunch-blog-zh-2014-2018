# 优步首次推出赞美，让车手给司机具体的积极反馈

优步今天在其应用程序中推出了一项名为“赞美”的新产品功能新系统将提示乘客除了给司机分配的星级之外，还要给出一个具体的、额外的积极反馈，包括“优质服务”或“美妙的音乐”等内容。

新的赞美系统将在收到一个通知时发送给司机，并告诉他们骑手选择了哪一个。预设列表包括九个固定的赞美，这简化了用户端的过程，应该使骑手更有可能实际参与和使用新功能。如果你在一个测试市场，在那里它在一个早期版本中推出，你可能已经熟悉了多项选择增加的反馈选项。

这里有一个积极强化的元素，赞美可以帮助司机知道他们的良好旅行产生了很高的数字评级，使他们更有可能在未来重复这种行为。赞美完全是可有可无的，但在增加优步采用的反馈机制的深度方面可能会有很大帮助，这可能意味着优步方面也有更多有用的数据。

每一条反馈都与司机的特定行为类型相关联，涵盖了一般的服务质量；谈话；导航；娱乐；汽车质量；音乐选择；清洁度；便利设施或额外服务；以及超出标准预期的附加服务。

星级本身本质上是一个量化数据的黑洞——当然，你有一个数字分数，你可以汇总不同地区的数字分数，并查看个人和团体表现随时间变化的趋势，但你真的很难搞清楚是什么影响了这些分数。

毫无疑问，优步在构建关于哪些因素会影响分数的推理模型方面做得很好，但即使是这个简化的附加反馈层，也应该有助于在促成编号评级的合格因素方面大大扩展其视野。

优步在宣布这一消息的博客文章中表示，它将在感恩节前添加这一功能，以此让乘客感谢他们的司机，但这也肯定会加深该公司可以收集的关于乘客偏好和旅行质量的数据类型。
