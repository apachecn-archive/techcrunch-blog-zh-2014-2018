# 三星向美国 Note 7 用户提供 100 美元购买另一部三星手机

在有缺陷的 Galaxy Note 7 的燃烧灾难后，本周早些时候,[迫使三星永久停止生产该设备，韩国电子巨头正在通过为不放弃其品牌的消费者提供更大的信用票据来加强其损害限制策略。](https://web.archive.org/web/20230323181719/https://techcrunch.com/2016/10/11/samsung-kills-the-galaxy-note-7-for-good/)

在一份关于扩大 Note 7 美国退款和换货计划的说明中，三星为那些选择将平板电脑换成任何其他三星设备的人提供最高 100 美元的账单信用，该计划于美国东部时间今天下午 3 点【T2 时间】适用于所有用户，包括那些拥有新替换手机(也曾被爆出起火)的用户。

选择将 Note 7 换成全额退款或任何其他品牌智能手机的美国消费者可以获得更少的 25 美元账单信用——作为一种善意姿态，以应对影响该设备的多次召回的麻烦。但是，如果消费者愿意继续忠于三星，善意会更大。

现阶段，美国以外市场的 Note 7 用户似乎没有获得善意信用——我们已经要求三星澄清它是否在其他地方提供类似的激励措施，并将根据任何回应更新这篇帖子。

本周，三星将其退款计划扩展至 T4、澳洲和亚洲部分地区。而针对 Note 7 的[英国退款和换货计划](https://web.archive.org/web/20230323181719/http://www.samsung.com/uk/note7exchange/)仅指出，选择换货的消费者将获得差价退款。

三星电子美国公司总裁兼首席运营官 Tim Baxter 在一份关于美国退款和换货计划的声明中评论说:“我们感谢我们的消费者、运营商和零售合作伙伴在这些充满挑战的时期承受负担的耐心。我们致力于尽一切努力纠正这种情况。”

在进一步的评论中，他将“客户安全”作为公司的优先事项，并敦促所有 Note 7 用户“关机并利用可用的补救措施”。

有关如何更换 Note 7 的更多信息，美国消费者被要求访问[samsung.com/us/note7recall](https://web.archive.org/web/20230323181719/http://www.samsung.com/us/note7recall/)，以获得运营商和零售商关于如何以及在哪里更换平板手机的具体说明。

目前还不清楚为什么一些 Note 7 设备会爆炸。在第一次召回后，三星在替换 Note 7s 中更换了电池单元，但显然这并没有解决问题，消费者继续报告更换的问题。
