# 美国电话电报公司以近 500 亿美元收购 DirecTV

正如过去几周一直传言的那样，美国电话电报公司刚刚正式宣布了他们收购 DirecTV 的计划。

交易条款总计约为每股 95 美元，总计 485 亿美元。

然而，如果你加上 DirecTV 背后的近 190 亿美元的债务，这笔交易将美国电话电报公司拉回接近 670 亿美元。

似乎 DirecTV 至少在一段时间内将继续作为一家独立公司运营，甚至在交易结束后。他们将总部设在加利福尼亚州的埃尔塞贡多，美国电话电报公司承诺至少在 3 年内提供独立的直播电视服务。

下一步:美国电话电报公司和 DirecTV 必须说服一批监管机构(在美国和拉丁美洲，DirecTV 拥有相当庞大的客户群，大约 20%的收入来自那里)相信这次合并是一个好主意。美国电话电报公司预计这至少需要 12 个月——所以如果你担心这笔交易会对你的服务产生任何负面影响，要知道在这一切开始发生之前，你还有一年左右的时间。
