臭名昭著的约翰·迈克菲在 Def Con 大会上发言时开始说道:“媒体将我描绘成一个疯狂的天才或者一个疯狂的精神病天才——为什么现在要打破这个记录呢？我必须承认:当他疯狂的时候，他像狐狸一样疯狂。最终，尽管他的故事疯狂而引人入胜，但对我来说最有趣的是他奇怪地成为他的观众的象征的方式。

迈克菲的故事，浓缩:在伯利兹拒绝支付数百万美元的贿赂后，他转而向其政府捐赠了大量布满键盘记录器和音频记录器的笔记本电脑，并收集了 17，000 小时的录音，其中包括整个伯利兹内阁与人口走私者、洗钱者和 Sinaloan 毒品卡特尔勾结的证据。因此，他随后被记录在案的法律问题和作为国际逃犯的工作。

现在，很明显，他住在多或少的 run 上，由于一个 17 岁的前情人试图谋杀他而半聋。他说，他和他的妻子不得不每周更换手机，因为他们很快就被卡特尔的宠物黑客提供的“高度复杂”的间谍软件感染，唯一让他活着的是死人开关，在他死后，它会揭示他收集的所有证据。

我不清楚他的整个故事是否真的是字面意思。他在序言中沉思道:“如果你对精神活性化学物质有足够的经验，你就会明白，无论是我想象的还是我经历的，最终都是无关紧要的。”

尽管如此，他还是设法创办了一家总部位于蒙特利尔的初创公司，该公司已经从一个网状路由器转变为一个安卓应用程序，这个安卓应用程序可以让 T2 扫描你其他应用程序的权限，并标记出有问题的应用程序。(这是非常简单的。)就像“我在伯利兹时创立的 17 家不同的公司”一样，它不会引起轰动。但迈克菲作为创始人并不重要，因为我认为，就他的重要性而言，他是一个象征。

上周，我在拉斯维加斯(又名 Crazytown)与黑客/信息安全社区呆在一起，其中很大一部分是古怪的、打破传统的黑衣人，他们坚决反对这个人……尽管他们自己没有错，但他们越来越成为男人，并且不知道如何处理由此产生的生存危机。

大多数身穿黑色衬衫、留着粉红色莫霍克头、纹身浓重的 Def Con 与会者，以及该 Con 的官方材料，都是反政府、反体制、反企业的……然而，与此同时，Def Con 是黑帽子的小兄弟，黑帽子里聚集着大型机构公司的员工和代表，去年，时任美国国家安全局局长的基思·亚历山大将军(General Keith Alexander)发表了主题演讲。

这两家公司都是由杰夫·莫斯(又名黑暗切线)创建的，他在九年前以 1400 万美元的价格出售了黑帽，但仍然密切参与其中，主持并介绍了它的主题演讲。这就像是从菲利普·K·蒂克的 中拿出来的一样，一台扫描仪阴沉地；有一天，Moss/Tangent 主持了一个大型、昂贵的企业安全会议，然后，就在第二天，一个更年轻、愤怒、反叛的黑客暴徒，其公开宣称的精神气质几乎与第一组相反……从未解决这一矛盾。

但我无意挑他的毛病；他实际上代表了整个信息安全行业。

我们生活在一个时代，其特征是每个人都在口袋里携带一台超级计算机，与其他人的超级计算机和大型服务器群无线连接。知道如何渗透、控制和拒绝访问这些设备和网络的人拥有巨大的权力。拥有那种力量却不成为真正的男人是非常困难的。

于是:“信息安全官”变成了 C 级职位。对高级安全专家的需求非常大。到处都是有利可图的出售和加入系统的提议，通常是以一些无害的或客观上仁慈的方式；随着年龄的增长，人们更有可能接受它们。和任何行业一样——但是这个行业的年轻成员认为这里的人是黑这个系统，而不是某一天加入这个系统，即使目的是从内部颠覆/改进它。

因此，迈克菲和他在这里的热情接待。他是 Def Con 集体精神的原型，有好有坏，是一个电影中令人敬畏的叛徒/不法分子/骗子形象。他也是活生生的证据，象征性地证明，你可以在世界上留下你的印记，无论是在技术上还是在一个成功和著名的创业公司，而不用以任何方式出售和公司化；你可以接受生活可能提供的一切，并且仍然保持基本的廉洁。

唉，在我看来，这也是一个故事——虽然基于一些真实的经历——更多的是愿望真实，而不是真实真实。
