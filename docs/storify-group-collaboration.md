# 社交监管平台 Storify 重新设计，支持团队协作 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/10/07/storify-group-collaboration/>

Storify 是一项服务，允许用户整理推文、照片和其他社交媒体帖子，以创建一个连贯的故事，该服务正在进行新的设计。重新设计不仅赋予了产品新的外观，还支持一个重要的新功能——多名记者或其他 Storify 用户可以一起合作撰写报道。

显然，Storify 之前并没有真正支持过这种合作，正如联合创始人伯特·赫尔曼(Burt Herman)向我解释的那样。当然，两个或两个以上的人可以登录同一个账户，开始写同一个故事，然后，[他们实际上可以把对方锁在外面，这样他们就不会因为彼此的工作而浪费掉了](https://web.archive.org/web/20221127021011/https://beta.techcrunch.com/2013/07/01/storify-collaboration/)，但也仅此而已。

赫尔曼说，这是 Storify 第一次被设计成多人同时一起创作一个故事——事实上，一些早期测试者已经用它来报道密苏里州弗格森和香港的抗议活动。例如，Herman 建议，如果您试图创建一个突出某个事件的社交媒体帖子的故事，并且您希望实时更新该故事，那么一个人可能希望监视脸书，而另一个人则可以关注 Twitter。借助 Storify 的实时协作，他们可以随时为故事添加内容。

为了实现这一点，Storify 创建了一个类似于 Google Docs 的界面，多人可以同时处理一个文档的不同部分，不同颜色的光标向您显示您的其他队友在哪里工作。

![storify multiple users](img/829437552481c57c0d303beddfe9c82a.png)

“在某些方面，它比谷歌文档更复杂，因为它不仅仅是文本——还有所有这些类型的多媒体社交媒体格式，”赫尔曼说。

Storify Enterprise 中提供了新的协作功能，以前称为 Storify VIP。Storify Enterprise 的最初用户包括华尔街日报和《环球邮报》。

顺便说一下，距离评论平台 [Livefyre 宣布收购 Storify](https://web.archive.org/web/20221127021011/https://beta.techcrunch.com/2013/09/09/livefyre-acquires-storify/) 刚刚过去一年多。赫尔曼表示，自那以来，最大的变化之一是免费和付费产品之间的区别更加清晰:“现在我们非常喜欢，免费版本是为消费者提供的，企业显然是为专业人士提供的，所以让我们为专业人士提供他们想要的功能。”

他补充说，随着时间的推移，我们可以期待看到 Livefyre 和 Storify 之间的更多集成。
