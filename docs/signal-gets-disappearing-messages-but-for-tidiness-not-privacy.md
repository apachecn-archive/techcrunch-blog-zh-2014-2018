# 信号接收消失的信息——但是是为了整洁，而不是隐私

消息应用 [Signal](https://web.archive.org/web/20221206185600/https://www.crunchbase.com/product/signal-desktop#/entity) 以端到端加密而自豪，隐私是他们的口号——但其新的短暂消息没有被宣传为实现这一目的的工具，也不应该被宣传为实现这一目的的工具。

消失的文本和图片是该应用最新版本的主要新增功能(以及一个肉空间验证工具)。但是，尽管人们曾经认为让 Snapchat 上的图片过期是发送色情或敏感信息的一种方式，但事实是，它对你的安全性几乎没有增加什么，如果有的话。截图、独立摄像头、复制粘贴——很容易让短暂的信息永久化。

相反，开放耳语系统[将这一功能](https://web.archive.org/web/20221206185600/https://whispersystems.org/blog/disappearing-messages/)作为防止数字混乱的一种方式。

> 消失消息是您和您的朋友保持消息历史记录整洁的一种方式。对于所有参与者都希望自动化极简数据卫生的对话，它们是一种协作功能…

这确实是一种更明智的方式来思考这种信息，事实上，这是像米利班德这样的年轻人通常使用的方式。没有人真正关心上周的谈话。就像拉飞奇说的，他们已经过去了！因此，为 [iOS](https://web.archive.org/web/20221206185600/https://itunes.apple.com/us/app/signal-private-messenger/id874139669) 或 [Android](https://web.archive.org/web/20221206185600/https://play.google.com/store/apps/details?id=org.thoughtcrime.securesms) 和 hakuna matata 下载新版应用程序吧。看，我很时髦。
