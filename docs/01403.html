DataFox 提供预测智能即服务，帮助做出更好的投资决策 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2014/09/09/datafox-offers-predictive-intelligence-as-a-service-enabling-better-investment-decisions/?ncid=rss

今天在 2014 年旧金山 TechCrunch Disrupt 上发布的是一家名为 DataFox 的公司，它提供了一个交易智能平台，使任何公司分析师或个人都可以实时访问“大数据”，从而做出更好的投资决策。该公司已经对其基于订阅的平台进行了一年多的测试，有 2000 多名试用用户，今天推出了公共测试版。

该公司于 2013 年 9 月由斯坦福校友 Bastiaan Janmaat(首席执行官)、Ben Trombley(首席技术官)、Mike Dorsey(首席技术官)和 Alden Timme(建筑师)创立——这个团队的背景包括在高盛早期投资集团、计算机科学和统计学领域的经验。

如今，像高盛这样的公司可以聘请分析师，他们从各种来源手动收集信息，将信息输入电子表格或简单的数据库，然后随着时间的推移保持信息的更新。但这需要大量的手工劳动，不是小企业——当然也不是个人——可以轻易复制的。

与此同时，DataFox 正在打造它所谓的“面向大众的分析产品”。作为《就业法案》和其他允许“主街”开始投资私营公司的倡议的支持者，该团队认为，在未来，1.7 亿在其金融投资组合中拥有股票的美国人也将投资于这些私营公司，并将需要工具来帮助他们做出与这些努力相关的更好的决策。

但与此同时，多尔西告诉我们，DataFox 正专注于专业投资者、出版商、风险投资公司和其他参与私募股权的公司。它的出版商客户包括 VentureBeat、华尔街日报、Strictly VC 和 The Information，约占其总客户群的 5%至 10%。投资者和风险资本占另外 45%。如今使用该平台的公司包括 Accel Partners、彭博贝塔公司、Intuit 和谷歌风投——后者既是客户也是投资者。

该公司面临着来自该领域 800 磅重的巨头的竞争，包括彭博、汤森路透、Factset 和 Dun & Bradstreet 等。这些公司经常使用离岸分析师通过不直观的界面手动数字化信息。相反，DataFox 将实时收集信息，不受操作系统或位置的影响。但是该产品也可以与这些解决方案一起使用——或者，客户甚至可以通过其 API 集成来自其他来源的数据以及 DataFox 自己的数据。

至于 DataFox 的工作原理，该解决方案涉及挖掘世界公共信息，并将机器学习应用于公司的在线业务，以跟踪 45 万家私营和上市公司，识别上升或下降的行业，与关键人物建立联系等。该公司声称，所有这些都发生在一个直观的界面上，专业人士，甚至个人，只需几分钟就可以采用。

DataFox 不仅会抓取网页来获取这些数据，它还会跟踪 RSS 源和 YouTube。此外，它还购买公共市场数据和通过其他 API 合作伙伴访问的数据，包括 CrunchBase、AngelList、LinkedIn、Alexa 等。它还通过算法创建自己的数据，以显示新信息——例如，像公司的主要竞争对手一样。数据也相当准确，因为它是实时更新的，DataFox 会自动标记差异。用户还可以通过标记他们知道是错误的东西来帮助改进数据，这也会被考虑在内。

虽然 DataFox 正在削弱老牌公司，但这些竞争对手似乎也支持 DataFox 的创新:Dun & Bradstreet 的首席执行官是 DataFox 的顾问委员会成员，Factset 的其他高管也投资了这家初创公司 178 万美元的种子期。该轮投资者还包括谷歌风险投资公司(Google Ventures)、舍帕洛风险投资公司(Sherpalo Ventures)、贾韦德·卡里姆(Jawed Karim)(YouTube 创始人)、社交杠杆公司(Social Leverage)、Green Visor、Leo Polovets 和斯坦福-StartX 基金。

今天，DataFox 推出了公开测试版，提供三级定价计划，对非营利组织和记者免费。(披露:我已经请求，并相信我可能会收到一个帐户。)个人每月 49 美元，大公司每月 399 美元。还可以定制企业定价。

位于门洛帕克的 DataFox 现在是一个由 9 名全职员工组成的团队。

Judges Q&A

这一环节的评委包括 Tracy Chou (Pinterest)、Josh Elman (Greylock Partners)、Markus Spiering (EyeEm)和 Hunter Walk(home brew)；为简洁起见，我转述了一下。

问:关于数据来源的问题。
答:我们抓取网页、RSS 提要、与 API 合作伙伴建立关系、购买数据并创建我们自己的数据

问:你认为它会完全取代分析师吗？
答:我们授权分析师进行实际分析，而不是进行人工数字运算。分析师们将会超负荷工作。

问:这个分析有多复杂？
答:我们有三位具有机器学习和统计学背景的专家。

问:你拥有哪些专有数据？答:一开始，我们没有专有数据。但是随着我们的发展，随着我们对来自非结构化来源的数据进行结构化，我们已经创造了价值。

问:团队的规模？答:9 人，工程 7 人，商务 2 人
