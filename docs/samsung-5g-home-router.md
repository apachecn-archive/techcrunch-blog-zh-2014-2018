# 三星为家庭展示了一款 5G 路由器

我们知道三星将在今晚的大型移动世界大会开幕式上发布一两款平板电脑，但事实证明，该公司还有其他几个惊喜。这家银河制造商发布了一些 5G 和基础设施公告，包括一款新的 5G 家庭路由器。

除了路由器将实现 1Gbps 的最高数据速率这一事实之外，三星没有提供太多关于该设备的新闻，这是讨论 5G 时经常提到的一个数字。

据该公司称，该公司的 5G 产品目前正在韩国、日本和美国进行商业前测试，“代表着行业采用 5G 之旅的最后一站”。三星 5G 设备的试验将于今年开始，但该公司对何时将实际推出广泛的商业用途保持谨慎。考虑到这项事业最终的规模，这是可以理解的。

路由器将需要放置在用户的窗口，旁边的一个 5G 收音机也宣布在今天的展览。该公司一直在网络部署方面与威瑞森密切合作，该公司的一名代表向三星眨眼示意，希望明年这个时候我们能看到 5G 智能手机。
