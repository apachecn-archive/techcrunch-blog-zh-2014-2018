WordPress.com 母公司收购阿塔维斯特

WordPress.com、WooCommerce、Longreads、Simplenote 和其他一些东西背后的公司 automatic，正在收购位于 T2 的初创公司 Atavist 。

Atavist 一直致力于为独立博客和作家开发内容管理系统。有了阿塔维斯特网站，你可以很容易地用各种媒体撰写和发布故事。

你可能会认为这并不特别具有开创性，因为任何人都可以在 WordPress.com 或 Squarespace 上创建一个网站并做同样的事情。但该公司也允许你创建一个付费墙，并建立一个订阅基础。

许多作家不想处理运营网站的技术细节。这就是为什么 Atavist 给你工具，让你可以专注于你的故事。

阿塔维斯特还经营着一份名为《阿塔维斯特杂志》的出版物。该出版物也加入了 Automattic。目前还不清楚它是会成为 Longreads 的一部分还是会保持独立。

CMS 本身不会继续存在。automatic表示发布平台将整合到 WordPress 中。这是有趣的部分。

虽然 WordPress 可能比 Atavist 更可靠，但这可能意味着 Automattic 希望开始提供订阅和付费墙。你可以想象 WordPress.com 网站提供本地包月服务。

30%的网络运行在 WordPress 上。他们中的许多都是 WordPress 的开源实例，托管在他们自己的服务器上。但许多网站都由 WordPress.com 托管，包括 TechCrunch。

WordPress.com 的订阅对网络来说是个好消息。Medium 突然取消了订阅计划，让许多独立出版物望尘莫及。因此，当涉及到为独立作家提供足够的收入时，很难相信 Medium。

Automattic 可以创建一个无缝的门户来管理多个出版物的订阅。这可能导致更少的广告和更好的内容。
