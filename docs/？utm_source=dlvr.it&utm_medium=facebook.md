# 奥古斯特希望 Airbnb 和 Homeaway 的主人能够使用他们的智能锁

奥古斯特·洛克(August Lock)正在进入家庭共享行业，让 Airbnb 客人的入住过程变得更加容易。

在过去的几个月里，Airbnb 已经尽其所能让入住变得简单明了。例如，该公司开发了一个新工具,让主人在应用程序中拼写出入住说明，所有这些都在一个简单的流程中，以确保客人在指尖就能获得他们需要的所有信息。

但这并没有解决最大的问题:钥匙。

首先，人们通常对结识陌生人不感兴趣，尤其是当他们刚下飞机或刚结束公路旅行时。其次，把那段时间(有时是下班时间)划出来交给 Airbnb 的客人是很烦人的。然后就是复制钥匙或者改装你的智能锁来暂时让陌生人进入。

这就是八月[开始玩](https://web.archive.org/web/20230131001613/https://august.com/host/)的地方。

August 允许 Airbnb 和 Homeaway 主机将他们的账户与 August 关联起来。当客人在家中预订时，August 会生成一个智能代码，该代码会在住宿期间持续有效，让客人可以轻松入住和离开，而无需主人照看整个过程。

客人将通过电子邮件收到他们的 pin 码和说明。

主人只需要一把智能锁和一个智能键盘，就可以让技术来承担重任。为了让主机对这笔交易更感兴趣，August 现在正在开展一项智能主机捆绑销售 25%的交易，其中包括 August 智能锁、August Connect Wi-Fi 桥和 August 智能键盘。

这一点，再加上 Handy 等其他创业服务，应该会让成为 Airbnb 的主人变得像点击几个按钮一样简单。
