# 利基创始人(加上 Vine 明星 Robby Ayala)谈论营销、社交媒体的风险等等

我最近采访了社交营销初创公司 [Niche](https://web.archive.org/web/20221207072733/https://www.niche.co/) 的联合创始人，当时他们[与罗比·阿亚拉(Robby Ayala)一起登上了美国在线总部](https://web.archive.org/web/20221207072733/http://on.aol.com/show/518283063-aol-build/518351419)的舞台，罗比·阿亚拉是该公司的创意合作伙伴，也是他们的常驻明星之一(拥有[280 万粉丝](https://web.archive.org/web/20221207072733/https://vine.co/robbyayala))。

去年秋天，在公司[筹集了一轮小的种子资金](https://web.archive.org/web/20221207072733/https://beta.techcrunch.com/2013/11/22/niche-funding/)之后，我第一次报道了 Niche。该团队建立了一个在线平台，将受欢迎的社交媒体用户与品牌营销人员联系起来，以便他们可以在广告活动中进行合作。

在上面 30 分钟的视频中，联合创始人罗布·菲什曼(Rob Fishman)和达伦·拉赫曼(Darren Lachtman，顺便说一下，他的堂兄是 TechCrunch 作家乔希·康斯坦)讲述了他们如何创建公司，讲述了他们如何说服阿亚拉从法学院退学并加入团队，并展示了一些实际的广告。

我认为，有一点是可以理解的，那就是社交媒体为创造有趣的、有创意的广告提供了机会，但我也很好奇它会给你带来什么样的风险。当被问及如果其创作者之一卷入某种丑闻，利基可能会受到什么影响时，菲什曼承认这是“在这个领域工作的重要组成部分”，特别是因为这些创作者中有许多是青少年。

“我们一开始做的决定之一就是不代表任何人，”他说。“我们不是一家人才机构，我们与我们网络中的任何创作者都没有正式联系，除了我们为他们提供软件，并与他们在一个个活动的基础上合作。因此，对我们来说，当我们网络上的某人犯了一些失礼的行为，或者他们犯了错误，或者他们做了一些不专业的事情，这并不意味着我们要切断与他们的联系，这只是意味着嘿，也许我们不会在不久的将来与他们合作。”

如果你是一个发布了一些似乎让所有人都生气的东西的创作者，Ayala 有一个简单的建议:“删除它。”
