脸书收购 Ozlo 以加强其对话式人工智能的努力

脸书已经购买了查尔斯·乔利的对话式人工智能初创公司 Ozlo。前脸书安卓平台主管乔利将不会回到公司。预计 Ozlo 团队将加入脸书，致力于自然语言处理挑战。

Ozlo 于 2016 年 10 月推出了一款面向消费者的应用。乔利当时告诉我，对话式人工智能领域正在迅速巩固(三星刚刚收购了 Viv )，他很乐意独立于主要科技巨头运营一项服务。随着今天的收购，Ozlo 不再是独立的，对话式人工智能空间变得更加巩固。

今年 3 月，Ozlo 推出了一套 API。该公司的关键区别之一是它的知识图表——它是展示任何智能所必需的关于世界的事实的数据库。Ozlo 将其知识层作为服务出售给开发者。

根据脸书的说法，除了 intent API 和 converse API 之外，知识层将在收购后关闭。对于最初的、容易获得的消费者机器人来说也是如此。

“全世界有 12 亿人使用 Messenger 与他们关心的人和企业联系，”脸书发言人在一份声明中说。“我们很高兴欢迎 Ozlo 团队，因为我们在 Messenger 中构建了由人工智能和机器学习驱动的引人注目的体验。”

目前还不清楚 Ozlo 团队将在脸书研究什么。Ozlo knowledge graph 可以作为脸书 m 的主干找到一个家。大型科技公司最近的一些收购旨在扩大此类信息库的规模。苹果最近收购了 Lattice Data ，以帮助将非结构化数据转换为知识图，可以对其进行推理，从而为用户的问题提供相关答案。

脸书拒绝透露收购 Ozlo 的规模。这家初创公司之前得到了 AME 云风险投资公司和 T2 格雷洛克合伙公司的支持。
