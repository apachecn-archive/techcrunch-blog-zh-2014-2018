Twitter 聊天机器人现在可以包括推你发微博、关注、点击等按钮

Twitter 今天推出了一项新功能，允许企业提示其追随者采取行动，而不仅仅是简单的收藏或转发。该公司正在为企业引入在直接消息对话中添加按钮的功能，旨在鼓励客户做一些事情，如关于该公司 Twitter bot 的推文，访问他们的网站，关注该企业的 Twitter 账户，或与该企业运营的另一个账户开始聊天等。

该功能是 Twitter 最近几个月对其直接消息体验所做的几项增强之一。例如，5 月份，Twitter 宣布了一种让企业运营广告的方式，这种方式旨在将用户带入一种直接的信息体验，用户可以在那里与企业的聊天机器人互动。

Twitter 还推出了一系列直接消息功能，如欢迎消息、快速回复、自定义简介、位置共享等等——这一系列功能似乎旨在挑战 Facebook Messenger 及其自己的商务聊天机器人平台。

虽然脸书聊天机器人经常让你点击项目来浏览他们的互动体验，但 Twitter 的新按钮要简单得多。它们只是被附加到一个公司发送给客户的消息中——也就是说那些由企业机器人所有者发送的消息。

例如，Focus Features 已经实现了新功能，它的机器人可以在推特上发布电影预告片。现在，Twitter 用户可以点击按钮购买电影票，加入公司的奖励计划，或者与人联系，以便获得客户服务问题的答案。在电影《被诱骗者》的发行中，该机器人还邀请用户玩一个小游戏，然后在 Twitter 上分享他们的分数。

Twitter 表示，这些按钮可以被配置为多种用途，这些用途发生在直接消息对话之外，如撰写推文、关注账户或在 Twitter 应用程序中打开网站。

在发布时，企业可以在信息中添加多达三个按钮，并且可以定制这些按钮上的行动号召文本(或表情符号)。

该功能还可以与最近推出的直接信息卡结合使用，后者鼓励用户进入该品牌的 DMs 系统，开始基于聊天机器人的对话。

Twitter 表示，这些新按钮将从今天开始在 iOS、Android 和桌面网络上推出。企业可以通过 Twitter 的开发者应用编程接口来使用这项新功能。
