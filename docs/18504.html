Instagram现在允许您在浏览TechCrunch时进行4人视频聊天

原文：https://web.archive.org/web/https://techcrunch.com/2018/06/26/instagram-group-video-calling/?guccounter=1

Instagram今天发布了对Snapchat、FaceTime和Houseparty的最新攻击。TechCrunch早在三月份就抢先报道了Instagram将推出视频通话功能，该功能于五月份在F8由正式宣布。现在，实际上向iOS和Android上的每个人推出了，允许最多四个朋友通过Instagram Direct一起进行视频通话。

Instagram希望通过feed、故事、消息、直播、IGTV以及现在的视频通话，成为满足其10亿用户社交需求的一站式商店。SimilarWeb在一封电子邮件中告诉TechCrunch，过去两年功能的大规模扩展正在取得回报，据估计，美国用户平均每天在该应用上花费的时间从2017年9月的29分钟增加到了今天的55分钟。花费更多的时间意味着脸书子公司更多的潜在广告浏览量和收入，在2012年以不到10亿美元的价格被收购后，一位彭博分析师刚刚对其估值为1000亿美元。

Instagram视频通话的一个很酷的功能是，你可以最小化窗口，在Instagram的其余部分来回跳动，而不用结束通话。这为你和朋友一起浏览网页提供了新的机会，就像你们一起闲逛一样。更多的朋友可以加入正在进行的Instagram通话，但如果你不想收到更多的通话邀请，可以将他们设为静音。您可以通过在聊天中点击视频按钮给任何您可以直接发送消息的人打电话，被阻止的人不能给你打电话。

以下是Instagram的群组视频通话与其他视频通话相比的优势:

  • insta gram–四路同步浏览
  • snapchat–16路，最多32人通过音频收听
  • face time–32路(将于今年秋天在iOS 12中推出)
  • 派对——每间客房8人，无限平行客房
  • Facebook Messenger–6路，最多50人通过音频收听

Instagram还推出了在F8承诺的另外两个功能。探索页面现在将被分段，以显示各种主题频道，揭示下面的相关内容。此前，Explore的2亿每日用户只能看到与其兴趣相关的流行内容的随机大杂烩，只有一个单独的“你可能喜欢的视频”部分。

现在，用户将在Explore上看到一个水平的频道托盘，包括一个根据算法为你定制的个性化收藏，以及艺术、美容、体育和时尚等内容，具体取决于你经常互动的内容。用户可以在类别之间滑动浏览，然后向上滚动查看更多他们喜欢的帖子。当你打开一个类别时，会出现一个子标签列表，比如当你打开art时，会出现#MoGraph(运动图形)或#Typeface。如果你厌倦了看到一个类别，你可以将其静音。奇怪的是，Instagram已经完全从Explore中剥离了故事，但当被问及时，该团队告诉我们，它计划在不久的将来将故事带回来。

增强的探索页面可以让人们更容易发现新的创作者。扩大这些内容制作者的受众对Instagram至关重要，因为它努力成为他们在竞争中最喜欢的应用。Snapchat缺乏专门的探索板块或其他粉丝群增长机会，这疏远了一些创作者，而新的Instagram话题频道让人想起YouTube的移动趋势页面。

Instagram的新探索频道(左)与YouTube的趋势页面(右)

最后，Instagram正在推出由合作伙伴设计的相机效果，首先是爱莉安娜·格兰德、BuzzFeed、Liz Koshy、Baby Ariel和NBA。如果你在关注这些帐户，你会在Stories相机中看到它们的效果，如果你发现一个朋友在使用你喜欢的帐户，你可以点击“试用”。这为所有提供自己的增强现实和2D滤镜的帐户打开了大门，而不会让Stories相机变得装满你不关心的镜头。

Instagram的新合作伙伴制作的相机特效

奇怪的是，所有这些功能都旨在增加你在Instagram上花费的时间，因为它正准备推出一个 Usage Insights dashboard ，用于跟踪你是否对该应用上瘾。至少视频通话和相机效果促进了积极使用，但Explore肯定会鼓励消极消费，研究表明这可能是不健康的。

这就是Instagram的使命和商业模式与其对用户福祉的承诺之间的摩擦所在。尽管首席执行官凯文·斯特罗姆表示， “任何(花在他的应用上的)时间都应该是积极和有目的的”，他希望Instagram“成为解决方案的一部分”，但该公司通过让人们粘在屏幕上而不是出现在他们的生活中来赚取更多。
