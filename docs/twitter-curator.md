# Twitter 公开发布馆长，这是一款为媒体提供的实时搜索和过滤工具

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/03/31/twitter-curator/>

Twitter 今天上午发布了新产品[策展人](https://web.archive.org/web/20230406224142/https://curator.twitter.com/)，让媒体机构、出版商和广播公司能够实时识别、过滤并在任何屏幕上显示推文和 Vine 视频。这项免费服务在某种程度上是 Storify 的竞争对手，旨在帮助媒体行业的人(很快其他人也会这样做)更好地理解 Twitter 网络上的大量数据，以便为自己的读者和观众突出显示最佳内容。

今年早些时候，该公司在伦敦举行的“新闻:重新连线”会议上公布了“T2”馆长，但是，直到现在，这一职位还没有被广泛使用。该产品仍处于测试阶段，正在由十几个组织进行测试，包括纽约市长办公室和意大利的主要网络 Mediaset。

但从今天开始，任何媒体机构现在都可以获得这项新服务，这将使推文和其他内容整合到他们的电视新闻广播、节目、网站或移动应用程序中变得更快更容易。

通过策展人，Twitter 提供了各种过滤工具，让出版商和广播公司创建简单或高度复杂的查询，帮助他们深入了解特定内容或当前趋势。例如，一家新闻机构可能会选择使用该产品在电视转播期间实时显示观众的反应，或者一家政府机构可能会在直播演讲的同时显示 Twitter feed。

![png;base64525abaa50198a18c](img/0dc1815d678b63899f5603dba98d58f3.png)

虽然最初是为了满足媒体组织的需求而设计的，但策展人也可以用于娱乐环境，例如，在电视节目中显示标签引用，或者在音乐会或音乐节的大屏幕上显示推文。

该服务使得运行一个简单的查询变得很容易，以便获得像被转发最多的内容，或者包含视频和引用给定标签的推文这样的结果。

但让这项服务对其用户来说强大的是它能够实时过滤 Twitter“消防水管”的各种方式，这意味着 Twitter 网络上发生的所有帖子的完整流。使用策展人，公司可以通过一系列因素找到并过滤这些推文，包括关键词、作者(@ handles)、位置、语言、时区、关注者数量、转发或收藏的数量、验证的用户等等。

这允许更复杂的查询。例如，一个组织可以要求查看在特定时区内对某个给定话题做出积极评价的女性。用户也可以在他们的搜索中否定术语，例如，这可以帮助过滤掉无关的内容或脏话。

![twitter-curator1](img/ecc37efe02c309c4c811ca73e693516f.png)

此外，该服务能够通过 Twitter 的视频网络 Vine 搜索相关视频，并可以通过使用过滤器结合适当的关键字来调出流行的直播潜望镜视频。

虽然目前市场上有许多第三方分析公司允许 Twitter 用户从推文中获得见解，但 Twitter 表示，策展人并没有真正专注于这个市场。相反，策展人总经理马克·古奈姆解释说，这项服务旨在实时呈现内容，而不是事后提供分析。他还将其描述为更多的“基线”产品，第三方可以通过在 Twitter 的免费服务上提供设计定制或支持来扩展它。

“这是其他人获得机会的地方，”他说。“我们在那里支持他们，并与他们合作。”与 Twitter 的嵌入功能不同，策展人并不规定推文应该如何出现在网页上或稍后显示的屏幕上。媒体机构自己决定其形式，每个酒店都有其独特之处。

如今，策展人正与许多认证的设计合作伙伴合作，包括 Flowics、Livefyre、ScribbleLIVE、Spredfast 和 [others](https://web.archive.org/web/20230406224142/https://biz.twitter.com/partners/list/certified-products) 。

![primapt_4](img/17ab9ed08e004a0196180b39692fe06f.png)

组织还可以在策展人中就他们的查询进行合作，允许他们一起建立收藏，并邀请那些直接收集新闻的人以外的其他人参与进来。

Ghuneim 指出，“例如，你可以邀请标准和实践部门的人在所有东西上线之前对其进行批准。”“它实际上就是为这个目的而设计的——广播和联合媒体用例，”他说。

作为一款免费产品，它没有计划包含高级功能或整合广告，策展人更多的是通过帮助公司的推文显示给更多的观众，如果不是“登录”的话，来帮助影响 Twitter 的底线。

这个指标是 Twitter 在最近几个月(T1)一直在努力推动的，据估计，大约有 [6 亿](https://web.archive.org/web/20230406224142/https://techcrunch.com/2014/10/27/twitters-next-targets-for-monetization-are-logged-out-and-casual-visitors/)人在注销时登陆 Twitter 页面，而该网络本身声称只有 2.88 亿注册用户。该公司最近与谷歌的搜索交易应该也会对这些数字产生重大影响。通过让更多的推文出现在新闻网站和电视节目上，该公司可以再次向更广泛的受众宣传——现在，通过策展人的方式，跟踪——其推文的可见性和传播。
