# 无人驾驶飞机营救澳大利亚两名游泳者

有一天，他们可能会转而反对我们，但目前，他们仍然是我们的盟友:根据一份新的报告，一架无人机在澳大利亚新南威尔士海岸救起了两名遇险的青少年游泳者。无人驾驶飞机在离海岸半英里的地方发现了两名陷入困境的青少年，然后放下了一个漂浮装置，目的是给他们一些可以抓住的东西(通过 [Verge](https://web.archive.org/web/20230316154423/https://www.theverge.com/2018/1/18/16904802/drone-rescue-australia-video-ocean) )。

这架无人机实际上还不应该拯救任何人——它正在进行一个试点项目，以测试其可行性。但是[悉尼先驱晨报](https://web.archive.org/web/20230316154423/http://www.smh.com.au/nsw/world-first-a-drone-has-been-used-to-rescue-two-swimmers-struggling-in-heavy-surf-20180118-h0kg9m.html?utm_source=twitter&utm_medium=social&utm_campaign=nc&eid=socialn%3Atwi-13omn1677-edtrl-other%3Annn-17%2F02%2F2014-edtrs_socialshare-all-nnn-nnn-vars-o%26sa%3DD%26usg%3DALhdy28zsr6qiq)报道称，当接到游泳者遇到麻烦的电话时，无人驾驶飞机正好在 Ari 附近，并做好了应对的准备。

无人驾驶飞机的飞行员是新南威尔士的一名经验丰富的救生员，他能够在一两分钟内到达游泳者的位置，并放下吊舱，这比直接与真正的血肉救生员反应至少少了几分钟。

这项训练演习旨在让救生员熟悉所谓的“小开膛手”无人机，这是政府计划的一部分，旨在帮助减轻鲨鱼袭击的风险。它拯救游泳者的能力是一个意外，但一个幸运的意外绝对有助于证明它作为 1600 万美元政府项目的一部分的可行性。

还有，这提醒了我们，有时候，无人机其实是好的。
