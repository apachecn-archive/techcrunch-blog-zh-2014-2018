牛角包可以让你使用 iBeacons TechCrunch 在合作空间或咖啡店预订座位

原文：https://web.archive.org/web/http://techcrunch.com/2015/05/03/croissant-lets-you-reserve-a-seat-at-a-co-working-space-or-coffee-shop-using-ibeacons/?ncid=rss

来自本地交易初创公司 Perqy 的三位联合创始人——戴夫·艾德尔、亚当·周和妮莎·加里根——本周末在 TechCrunch Disrupt NY 黑客马拉松上会面，展示他们的项目，该项目使用安装在当地企业，如咖啡店或共同工作空间的 iBeacons，允许客户预订一个位置，然后根据他们在该场所工作的确切分钟数自动收费。

企业主还可以从在线仪表板上实时查看客户地图。

Idell 解释说，黑客的想法是受最近去一家咖啡店的启发，这家咖啡店提供类似的东西，但使用了一种更手动的系统，包括在商店里安装的平板电脑上签到。在那里，顾客可以输入他们的电话和座位号码，向商家提供他们的姓名和位置。

但有了羊角面包，签到将通过面向消费者的移动应用程序自动进行。通过这个应用程序(在 hackathon 演示时尚未完成)，客户可以在共同工作区预订座位，甚至可以从应用程序内的食物和饮料菜单中点菜。

该应用程序与蓝牙信标一起工作，记录顾客的入住和订单(如果适用)，然后在他们离开商店时自动从他们的信用卡中收费。

为了这个周末的黑客目的，团队使用了 Gimball 的 API，Gimball 提供了 iBeacons 来测试；每个都有 50 码的视线。万事达卡的 Simplify Commerce API 也用于在顾客离开指定空间时自动处理他们的卡。

这种系统对于一家咖啡店来说是有意义的，该咖啡店希望为其客户提供在指定时间段内预订桌子作为工作空间的能力，使他们能够在食品和饮料销售的基础上带来额外的收入。与此同时，共同工作空间可以根据用户实际在场的分钟数向他们收费，而不是像现在这样，让他们以半小时或一小时为单位预订时间。

然而，任何希望使用 iBeacon 技术跟踪客户全天活动或他们在商店花费的总时间的企业也有潜力。当然，现在已经有很多公司致力于更大规模地开发这种东西。

羊角面包的创造者说，他们不确定在这次黑客马拉松结束后，他们会在黑客方面做更多的工作，但他们不排除这种可能性。

“可能会与 Perqy 产生一些协同效应，”Idell 指出。"我们得让它再下沉一点。"
