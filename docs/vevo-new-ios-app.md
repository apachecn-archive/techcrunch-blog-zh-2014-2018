# Vevo 发布新的 iOS 应用程序，提供实时信息和实时 Vevo 电视流

音乐视频经销商 [Vevo](https://web.archive.org/web/20220930232252/http://www.vevo.com/) 今天刚刚发布了其 iOS 应用程序的更新，为用户提供了一个新的定期更新的主页，以及访问该公司直播的编程音乐频道。

该公司[在 SXSW](https://web.archive.org/web/20220930232252/https://beta.techcrunch.com/2014/03/09/vevo-sdk/) 向一些记者展示了该应用的新版本，这是继 [Vevo 去年秋天](https://web.archive.org/web/20220930232252/https://beta.techcrunch.com/2013/10/31/vevo-redesign-2013/)推出的对其网站的重大重新设计之后。在那次更新中，Vevo 取消了艺术家图像的旋转传送带，代之以定期更新的特色艺术家、视频和播放列表。

该公司还在其现场 Vevo 电视阵容中增加了两个频道。vevo TV[于一年前推出](https://web.archive.org/web/20220930232252/https://beta.techcrunch.com/2013/03/12/vevo-tv/)，提供连续的、程序化的音乐视频流——有点像 MTV 在被糟糕的真人秀接管之前所做的。

现在，所有这些都被引入 Vevo 的新 iOS 应用程序，该应用程序可在 iPad、iPhone 和 iPod Touch 上使用。该应用还通过一个选项菜单增加了改进的导航功能，点击触摸屏上的“汉堡”按钮即可启动该菜单。

用户可以浏览、搜索和导航不同的视频，即使他们正在观看屏幕上播放的视频的最小化版本。观众还可以轻松地在播放列表中添加或编辑视频，或者通过应用程序内置的动作按钮在社交网络上分享视频。

该应用程序采用了在 SXSW 发布的新[软件开发套件](https://web.archive.org/web/20220930232252/https://beta.techcrunch.com/2014/03/09/vevo-sdk/)。在内部使用的同时，该 SDK 也可供第三方开发者使用，以围绕 Vevo 视频构建他们自己的体验。
