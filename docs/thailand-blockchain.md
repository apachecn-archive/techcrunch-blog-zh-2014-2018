# 泰国正成为区块链的一个重要国家

当美国监管机构仍在试图找出如何看待加密货币时，泰国政府已经在规划自己的[中央银行数字货币](https://web.archive.org/web/20230118055637/https://www.bot.or.th/Thai/PressandSpeeches/Press/News2561/n5461e.pdf)。

这只是泰国在 2018 年成为东南亚最有趣的加密货币和区块链国家之一的众多例子之一。

自今年年初以来，泰国政府对加密货币项目和交易所越来越直言不讳和欢迎。在短短几个月内，泰国监管机构取得了显著进展，从设立加密货币公司许可证到允许交易所和 ico。更重要的是，该国通过为外国区块链公司的运营提供清晰明确的指导方针，吸引了外国公司。这是我们在东南亚看到的一种模式，也是区块链和加密货币初创公司的创始人在考虑全球扩张时应该注意的一种模式。
