零售业的未来对消费者来说不会太好

原文：https://web.archive.org/web/https://techcrunch.com/2015/07/16/the-future-of-retail-wont-be-so-good-for-consumers/

More posts by this contributor

购物——无论是线上还是线下——是现代社会的一大奢侈品。人们可以以较低的价格享受各种选择，并在家中方便地购物，同时仍然可以选择去当地的购物中心或零售商浏览过道，以获得即时满足。

但是消费者不能鱼与熊掌兼得，我们今天所知的零售世界也不能再给他们蛋糕了。

零售业将永远改变

技术正在扼杀传统零售商。受害者将包括那些销售商品品牌类型的产品，如消费电子产品，电器，运动器材和家具，甚至可能包括那些销售消费品。

价格战，加上技术转移，将会淘汰那些跟不上的国家、地区和地方竞争者。随着在线购物在各种市场中占据更大份额，今天的许多供应商将不复存在。看看百思买、史泰博、Radio Shack 和西尔斯等公司的趋势就知道了。

受实体基础设施的拖累，许多今天的店内零售商无法逃脱侵蚀其业务的在线引力，尽管他们进行了投资来适应。

剩下的只会是最大最好的。亚马逊将成功接管在线领域，沃尔玛将成为店内/在线模式的幸存者，而其他在线利基竞争对手则找到了谋生的方法。

过渡阶段

在这种新环境下，以前在当地渠道展示产品的原始设备制造商(OEM)失去了为消费者提供触摸和感觉的能力。

为了解决这一差距，原始设备制造商将追求店中店的概念，在新的“陈列室”环境中并排展示他们的产品，类似于今天在亚洲看到的概念。

消费者可以走进来，浏览 OEM 雇佣的知识丰富的员工提供的许多不同的产品。产品将按照制造商的建议零售价(MSRP)定价，让零售合作伙伴在不影响现有渠道销售的情况下自由定价。

原始设备制造商很高兴，因为他们的产品无论在哪里销售都会被转移，而零售商喜欢在他们的商店里销售商品的触摸和感觉支持。

随着模式的发展，原始设备制造商为直运和当天交付建立了本地物流基础设施，在战略上将他们置于即将到来的巨大转变的位置。

原始设备制造商扰乱

随着原始设备制造商获得直销经验，他们的思维变得更加注重零售。原始设备制造商转向 MAP(制造商的广告价格)的战略举措可能会扰乱竞争市场模式。地图定价要求各地的产品必须以制造商设定的价格出售，这导致所有产品的净价格上涨。

除了零售商支付的批发价，原始设备制造商还可以从直接销售中赚取额外的利润。这一新的利润用于产品促销和进一步控制市场。

苹果、三星、视频游戏机、Beats 和智能手机都是如今定价被原始设备制造商严格控制的例子。

尽管直接购买导致一些销售额流失给了原始设备制造商，但总体效果是，剩余的零售商将获得更高的利润，同时继续凭借广泛的选择、服务和品牌认知度吸引客户。

新零售

由于价格由原始设备制造商设定，消费者别无选择，只能按要求付费。

由于利润率较低，低价品牌或非地图定价产品变得不那么显眼，也不那么容易被零售商推销。

想要进入市场的新零售商必须拥有现有的销量，以便原始设备制造商参与进来；这将很难实现，因为分销由原始设备制造商和零售商控制。这限制了新的竞争。

新的制造商将发现很难推动需求，因为他们缺乏营销能力。为了打入市场，他们必须通过社交/病毒式营销建立牵引力并赢得一席之地，然后才能被主流接受。

正如我们所知，技术已经迅速改变了经济。在 web 1.0 中，竞争力量推动价格下降，商品可获得性上升，这一开始是消费者的一个有希望的好处，最终将演变为我们被困在一个更昂贵、难以破坏的系统中。

最终，这个新世界对消费者来说并不那么好。

零售商能做些什么来改变这种情况？

首先，零售商需要找到并激励有远见的人和技术专家，代表他们推动概念并制定战略。从外部确定和聘用专家将节省时间，避免代价高昂的错误。为了支持这一点，零售商还应该考虑组成联盟来分享知识、降低成本并更快地找到答案。

正如我们所知，技术已经迅速改变了经济。

随着在线玩家尝试并转向当天送达模式，他们以非常近距离和个性化的方式进一步侵入砖块领域，加大了服务和便利性方面的赌注。速度的优势一下子转移到了线上。当地零售商需要通过提供类似的服务来快速应对这一选择。

在线玩家目前还没有能力提供所有产品线的当天送货服务，因此，目前，当地零售商占据上风。零售商应该通过尽快在消费者心目中建立这一地位来利用这一优势。

随着技术不断寻找新的方法来侵占当地业务，当地商店必须通过自己的投资来对抗 ，这些投资专注于订购便利性和 UI 改进。这将是消费者体验的一个重要部分，有助于保持忠诚度和推动重复订单。

此外，零售商需要考虑更好地利用当地基础设施，并在核心区域内开展合作。应该考虑这样一个问题，即在当地有业务和分销的零售商如何才能适应新的方式？

这有助于防范新的竞争，或者提供一个与竞争对手一起发展和合作的机会。同样，有购买力的零售商可以在市场上提供他们的产品。尽管这些物业通常提供利润较低的机会，但随着零售业的发展，它们可以被视为另一种对冲和学习的方式。

此外，需要解决缺乏数据挖掘能力的问题。许多零售商不知道他们的客户是谁，并且受到遗留系统的阻碍。例如，零售商在进行店内销售时，继续在没有具体客户信息的情况下投放本地广告。

只要这个漏洞存在，长期的成功将很难实现，因为那些能够利用与特定消费者相关的数据的公司将会找到一种方法来改善和超越竞争对手。如果他们能够缩小这一差距，他们就会发现一个千载难逢的机会，以一种更加定制化、更具成本效益和更有针对性的方式直接与顾客建立联系。

此外，以一种几乎违反直觉的方式，零售商必须接受向 OEM 地图定价的转变。团结起来，鼓励地图控件提供一个公平的竞争环境，为所有人提供足够的余量。在购买力仍然存在的时候，利用财团来灵活运用购买力，以减缓价格侵蚀。尽管对消费者仍然不利，但这可以成为对冲市场变化的过渡步骤。

最后，无论他们决定做什么，零售商必须紧急行动，快速学习、适应和执行，以避免过时。

未来会怎样？

零售之战远未尘埃落定。需要造就创新，公司需要找到生存之道。当地的幸存者将会是那些碰巧足够快的“幸运”进入正确的模型来稳定正在发生的地震转移的人。

无论发生什么，原始设备制造商在这场游戏中拥有巨大的筹码——正如我们今天所知，他们如何以及何时发挥这一作用将对零售格局产生重大影响。
