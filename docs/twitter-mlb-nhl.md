# Twitter 将直播 MLB 和 NHL TechCrunch 的每周比赛

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/07/25/twitter-mlb-nhl/>

# 推特将直播 MLB 和国家冰球联盟的每周比赛

Twitter 正在扩大其直播体育计划，每周直播美国职业棒球大联盟和国家曲棍球联盟的比赛。

市场外的观众将可以免费观看直播，无论他们是否登录 Twitter。Twitter 还宣布了一个新的夜间体育集锦节目，名为集会，由移动视频公司 [120 Sports](https://web.archive.org/web/20221205114257/http://www.120sports.com/video) 制作。(拉力赛和 NHL 比赛将在美国发售；除了特定的国际地区，MLB 游戏将在全球范围内发售。)

Twitter 表示，它将出售广告来配合这一节目，这是由 Twitter 和 MLB Advanced Media 之间的新伙伴关系实现的。

MLBAM 的业务执行副总裁肯尼·格什(Kenny Gersh)在合作伙伴关系发布会上表示:“长期以来，Twitter 一直是棒球迷和冰球迷的伟大合作伙伴和平台，因为他们每天都在关注他们喜欢的球队和球员。”“我们知道球迷们会接受这种广泛的合作带来的一切——扩大和可靠地接触棒球和曲棍球直播，以及引入 Twitter 的首个每日直播精彩节目，直观地提供给他们积极参与游戏的平台。”

今年早些时候， [Twitter 宣布与 NFL](https://web.archive.org/web/20221205114257/https://beta.techcrunch.com/2016/04/05/twitter-signs-deal-with-nfl-to-live-stream-thursday-night-football/) 达成协议，将直播周四晚上的足球比赛，并通过直播温布尔登，让用户第一次看到了它在[的直播体育体验。](https://web.archive.org/web/20221205114257/https://beta.techcrunch.com/2016/07/06/twitter-started-its-live-coverage-of-wimbledon-today/)
