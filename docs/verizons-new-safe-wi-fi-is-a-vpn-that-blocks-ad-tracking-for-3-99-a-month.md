# 威瑞森新推出的“安全 Wi-Fi”是一种阻止广告跟踪的虚拟专用网，每月收费 3.99 美元

威瑞森推出了一款名为安全 Wi-Fi 的新产品，这种虚拟专用网为登录公共网络的移动用户提供了一个安全屏障。它还被作为一种屏蔽广告的方式推向市场。

那么 [WTF 是一个 VPN](https://web.archive.org/web/20230326172940/https://techcrunch.com/2017/01/01/wtf-is-a-vpn/) 又有什么关系呢？VPN 是虚拟专用网络。它位于您面前的设备和数据中心的服务器之间。可以把它想象成一条隧道，对本地网络上的其他人隐藏你的互联网流量。你当地咖啡店的无线网络可以让广告商和其他邪恶的人追踪你的 IP 地址。VPN 在您和服务器之间提供了一个安全的连接，并隐藏了 IP 地址，防止被人窥探。

安全 Wi-Fi(查看下面的视频)每个账户每月收费 3.99 美元，威瑞森用户可以在 Android 和 iOS 上使用。安全 Wi-Fi 在一个帐户中最多可覆盖 10 台设备。

威瑞森客户可以登录我的威瑞森，进入[产品&应用](https://web.archive.org/web/20230326172940/https://wbillpay.verizonwireless.com/myv/productsapps/)页面，滚动到安全 Wi-Fi，然后点击“立即获取”进行订阅。然后，安全 Wi-Fi 功能会添加到他们的帐户中。客户可以在他们的设备上从谷歌 Play 商店或苹果应用商店下载安全 Wi-Fi 应用，然后按照屏幕上的指示注册一个月的免费推广。

用户可以在安全 Wi-Fi 设置中打开“广告跟踪拦截器”。根据威瑞森关于该功能的常见问题页面，广告追踪拦截器阻止用户在浏览互联网时进行广告网络追踪，并阻止从该设备下载的应用程序生成的广告。在某些情况下，广告将被完全屏蔽，以防止广告追踪器工作。威瑞森说，灰色图像将取代屏幕上的广告。

据该公司称，需要广告跟踪器的网站可能会被屏蔽。

重要的是要记住，VPN 并不能完全消除风险。正如 TechCrunch 记者 Romain Dillet 指出的那样，[风险只是沿着 VPN 隧道向下移动](https://web.archive.org/web/20230326172940/https://techcrunch.com/2018/07/28/how-i-made-my-own-wireguard-vpn-server/)。操作服务器的人可以看到所有未加密的流量。例如，VPN 公司可能会调查客户的浏览习惯，然后卖给广告商。

***更新*** :威瑞森在其常见问题解答中提到，它不会监控客户的互联网浏览历史。
