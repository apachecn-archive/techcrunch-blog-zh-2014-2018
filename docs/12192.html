Tumblr 推出 Cabana，这是一款与朋友一起观看视频的新应用

原文：https://web.archive.org/web/https://techcrunch.com/2017/04/18/tumblr-launches-cabana-a-new-app-for-watching-videos-with-friends/

Tumblr 今天发布了一款名为小屋的新应用，它允许一群朋友一起实时视频聊天和观看视频。这款应用将以 Tumblr 品牌推出，将迎合经常使用 Tumblr 博客服务的用户类型——青少年和年轻成年人——但目前它的设计不是为了与 Tumblr 的产品互操作，也不是为了利用 Tumblr 用户之间的现有关系。

相反，小屋的出现更多的是偶然，而不是战略方向。

今年早些时候，Tumblr 创始人兼首席执行官戴维·卡普在雅虎领导层的场外活动中偶遇了聚沃实验室产品管理总监杰森·李，当时李开复正在演示该产品。

Polyvore Labs 是雅虎内部的一个孵化器——该公司在过去几年里收购了时尚网站 Polyvore 的 Tumblr 和 T11。(雅虎也在被威瑞森收购的过程中，后者拥有 TechCrunch 的母公司美国在线。)实验室团队包括来自 Polyvore 团队的工程师、设计师和产品经理。

它还开发了社交和生活方式领域的其他产品，但在 Cabana 之前，没有一个产品公开上市。

卡普在观看了小屋的演示后解释说:“我立刻就明白了，这是每个上网的人都有的体验。”。他说:“你在网上发现了一些不可思议的东西——最新的搞笑或深刻的病毒视频，或者对你来说意义重大的模糊事物——然后你把你的朋友拖到你的电脑前观看。”

某种程度上，同样的事情也发生在 Tumblr 上——人们发现有趣的内容并与他人分享。

然而，这两种产品的区别在于，Tumblr 本质上是一个分享你兴趣的陌生人网络——可能是动漫、艺术或一些电视节目粉丝——但 Cabana 是为了与一群现实生活中的朋友一起使用。

如果这种共同观看的体验听起来很熟悉，那是因为在这个领域已经有了相当多的行动。

最近获得资助的基于 iMessage 的视频聊天应用 Fam 计划进入群组视频观看领域，但其他公司的现有应用也在这么做。我们最近报道了一个叫做的项目，让我们看看吧！，一个和朋友一起看 YouTube 或者 Twitch 视频的 app。

但说实话，有很多针对团体视频观看的服务，包括凝视、 MyCircleTV 、 ShareTube 、together TV、兔子、 Rave 等等。甚至 YouTube 也在这一领域进行试验，推出了一款名为 Uptime 的应用。

共同观看体验也是较新的直播视频服务吸引力的更广泛部分，如 Twitter 的 Periscope 和脸书直播。

因此，问题并不在于共同观看是否会成为一件事情—这是一件事情——而是，如果这件事情是可以赢得的，那么谁将赢得这件事情。

小屋如何运作

在 Cabana 中，多达六个朋友可以在应用程序中的一个“房间”里闲逛，在那里他们可以分享他们一起实时同步观看的 YouTube 视频。您一次只能加入一个房间，但可以自由离开和重新加入房间。

你可以通过用户名添加朋友，或者从你的手机上传你的地址簿来找到你的朋友，然后创建一个房间，或者加入一个现有的房间。这些房间可以是公共的，也可以是私人的——后者是通过“锁定”房间来实现的——你还可以通过点击“嘿！”

进入房间后，您可以找到锁定房间、翻转相机、静音麦克风、添加好友、搜索视频并与小组成员共享的工具。朋友们甚至可以随时用另一个视频替换当前正在播放的视频。您也可以在对话继续时离开房间浏览您的手机，这会让您进入“暂停”状态

李说，这个应用程序只用了两个月的时间就建成了，旨在提供一种自发的感觉和体验。

李说:“在卡巴纳，我们试图融合闲逛和观看的经验。“我们会在播放视频时将您的脸放在最前面和最中间。我们不再专注于基于文本和表情的对话，而是给你朋友的全脸和各种各样的情感……我们认为这是一个非常独特和有区别的功能，可以让你在分开的时候真实地重现在一起的经历。"

“当杰森演示这个的时候，我觉得很酷，因为它看起来很有人情味……他们捕捉到了我们很多人都有的这种闲逛方式，”卡普补充道。

品牌广告的新家

明确地说，YouTube 的视频可以通过在卡巴纳的 API 整合获得，而不是官方合作。

但假设这被证明很受欢迎，卡巴纳将来可能会扩展到包括其他社交来源的视频。

卡普在 Tumblr 的现场视频方面采取了类似的立场，他认为这是一条可行之路。该公司最初曾考虑建立自己的视频平台和发布工具，但后来决定，当它需要进入实时视频领域时，与合作伙伴，如 YouTube、YouNow 和其他公司合作更有意义。卡巴纳最终也会这样做。

卡普认为，在品牌广告方面，它与 Tumblr 也存在重叠。目前在 Tumblr 上做广告的品牌可能希望通过 Cabana 应用程序接触到类似的年轻人群——13 岁至 18 岁的人群。(如今，Tumblr 超过一半的用户是 13-34 岁的青少年和千禧一代)。

但在发布时，卡巴纳没有广告，也没有关于未来如何整合的明确计划。

虽然 Cabana 是 Tumblr 推出的第一款独立社交应用，但这并不是 Tumblr 第一次尝试扩展到其他移动应用。早在 2012 年，它就推出了 Photoset ，一个将照片发布到社交网站(包括 Tumblr)的独立应用程序。(该应用程序后来被关闭。)

尽管它是一个独立的产品，但它不是 Tumblr 本身的支点。该网站今天仍然活跃，在其平台上有超过 3.4 亿个博客。该公司拒绝分享其活跃用户群，但 2016 年初的一份第三方报告称，Tumblr 在美国有 2320 万用户。Tumblr 在世界各地都有，所以这不是全貌，尽管该报告确实声称，未来几年，它增加的用户数量将少于脸书、Instagram 和 Pinterest 等竞争对手。

Tumblr 的问题是，如今用户在移动设备上联系的方式正在发生变化——我们更有可能发消息和聊天，而不是分享博客帖子。但 Tumblr 声称它已经完成了移动转型，称超过 80%的用户通过移动设备访问其服务。不过，如果成功的话，Cabana 可以帮助 Tumblr 进入视频聊天的新时代。

Cabana 最初在 iOS 上可用，Android 计划在 5 月推出。
