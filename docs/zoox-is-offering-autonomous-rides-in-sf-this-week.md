# Zoox 本周在旧金山提供自动驾驶服务 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/09/12/zoox-is-offering-autonomous-rides-in-sf-this-week/>

# Zoox 本周将在旧金山提供自动驾驶服务

据报道，自动驾驶汽车初创公司 Zoox 的估值为 32 亿美元，作为全球气候行动峰会的一部分，它将于本周提供自动驾驶服务。全球气候行动峰会召集州和地方领导人、科学家、非营利组织和其他人讨论气候行动的机会。

Zoox 将与旧金山的费尔蒙特酒店合作，用其自动驾驶的丰田汉兰达测试车在费尔蒙特和莫斯康中心之间运送精选的峰会嘉宾。Zoox 表示，它正在与费尔蒙特和峰会合作，以确定客人。

“Zoox 的诞生是为了改善我们日益密集的城市的安全、拥堵和污染，”Zoox 联合创始人、总裁兼首席技术官杰西·莱文森在一份新闻稿中说。“我们很自豪能够展示我们最先进的自主技术，与费尔蒙特合作，并展示私营企业对布朗州长和彭博市长在我们家乡举行的全球气候行动峰会的支持。”

这些测试的目标是向客人展示 Zoox 零排放自主技术的潜力，尽管这不是它的完全实现。这一进展紧随 8 月底【《T2》】】Zoox 联合创始人蒂姆·肯特里-克莱从公司离职之后。他被解雇的一个月前，Zoox 完成了由 Grok Ventures 的 Mike Cannon-Brookes 牵头的 5 亿美元融资，使其融资总额达到 8 亿美元。

Zoox 的最终目标是，到 2020 年，以自己的打车服务的形式，在商业上部署自动驾驶汽车。这些汽车本身将是全电动和全自动的。与此同时，优步和 Lyft 等叫车公司也在研究自动驾驶汽车，该领域还有许多其他大型公司。
