# 谷歌的 Android 7.1 开发者预览版将于本月晚些时候发布 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/10/11/the-android-7-1-developer-preview-will-launch-later-this-month/>

谷歌今天宣布，它将继续使用安卓牛轧糖 7.1 的安卓测试程序，这是其移动操作系统最新版本的下一个小更新。

该公司在上周宣布其 Pixel 手机时已经展示了 7.1 的一些功能，但 Pixel 的一些最佳功能，包括对谷歌助手[的全系统支持，将仍然是这些新手机](https://web.archive.org/web/20230131052735/https://techcrunch.com/2016/10/05/pixels-best-features-arent-coming-to-the-new-version-of-android/)的专属。

与 7.0 版本一样，谷歌将继续提供 7.1 测试版的[无线安装](https://web.archive.org/web/20230131052735/https://www.google.com/android/beta)。新的测试版也将支持与 7.0 周期相同的设备:Nexus 5X 和 6P 手机，以及 Pixel C 平板电脑。最终发布的将是 5X、6P 和 Pixel C，以及 Pixel、Pixel XL、Nexus 9、Nexus Player 和 Android One 设备。

[![2016-10-11_1134](img/0719e9cc27e2e12412f85cfad54d3c31.png)](https://web.archive.org/web/20230131052735/https://techcrunch.com/wp-content/uploads/2016/10/2016-10-11_1134.png)

预览版显然是针对开发者的，但如果上一个测试周期是任何迹象，那么 7.1 测试版也将非常稳定(尽管你的里程数可能会有所不同，我不建议在你的主要手机或平板电脑上安装测试版)。如果您注册了 7.0 测试版，并且此后没有注销，您也将自动获得 7.1 测试版。

7.1 有什么新功能？正如预期的那样，它包括对谷歌 Daydream VR 平台的支持和更快的 A/B 系统更新，但谷歌也在今天的公告中强调了几个新功能。开发者将能够直接在启动器中显示他们应用的关键动作。这些操作可以让用户深入到应用程序中去执行某个功能。例如，这将允许消息应用程序为你提供给朋友发短信的快捷方式，或者允许你立即在谷歌地图中找到你最喜欢的位置。

键盘开发人员现在还可以在键盘上添加贴纸、动画 gif 和其他图像(应用程序可以告诉这些键盘它们可以接受哪些图像和其他内容)。应用程序现在还可以引导用户进入一个新的设置屏幕，让他们轻松删除不用的文件并释放存储空间。

至于 Android 7.1 的外观和感觉，这里没有任何重大的震惊。开发者将能够为使用 Pixel 手机(使用新的圆形图标 Pixel 启动器)和其他使用圆形图标的第三方启动器的用户设置他们应用的圆形图标。

在很大程度上，7.1 是牛轧糖的下一个进化版本，顾名思义。
