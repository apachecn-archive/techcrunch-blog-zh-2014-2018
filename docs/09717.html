是时候回顾一下加密货币世界正在发生的事情了！值得称赞的是,永远不会无聊。有些东西看似常青；比特币内战再起；以太坊的成长之痛仍在继续。但是在地平线上，一些奇怪而有趣的事情正在发生。例如，我给你 appcoins、Blockstack 和 Zcash。

Appcoins

忘记比特币吧，appcoin 的忠实信徒们说；忘记以太坊；见鬼，忘了风险投资吧。任何能够找到实现内部加密货币(一种用于支付项目范围内商品或服务的“appcoin ”)的方法的项目都可以利用这一点，首先通过出售“预开采”硬币来众筹价值数百万美元的初始开发成本，然后与那些幸运/有远见的初始用户和投资者分享成功的回报。

听起来很吸引人！它有先例；毕竟，以太坊本身就是这样融资的，许多早期的比特币爱好者从这种热情中致富。你可以看到为什么区块链的狂热者对比特币的发展方向不满意，比如比特币基地的联合创始人 Fred Ehrsam，对这个想法如此兴奋。

但是——完全抛开这在技术上是否合法的问题——像所有的众筹一样，appcoins 面临着令人质疑的激励风险；“预先赚很多钱”比比特币和以太坊那样的“制造一些非常困难和尖端的东西，然后看看里面有没有钱”更有诱惑力。例如，Appcoin 海报儿童 Steemit 引起了如此多的质疑，他们不得不在官方博客上发布“stee MIT 不是庞氏骗局的 7 个理由”。这可不好。

ETA: sooooo，这很尴尬，但不知何故，我最初把一个 Steemit 用户的帖子和他们的官方博客混在了一起。抱歉，我糊涂了。不过，我更大的观点仍然成立……

更重要的是，我们以前来过这里；以太坊不仅仅是第二个大区块链，它是众多后比特币区块链中唯一一个有实际影响的。所谓的“替代硬币”逐渐消失，因为它们有缺陷、不安全、不可信，并且/或者除了事实上的赌博之外没有提供持久的价值。正如一直很有趣的普莱斯顿·伯恩(Preston Byrne)所指出的那样，普莱斯顿·伯恩是一名律师，也是获得许可的区块链初创公司 Monax 的联合创始人。

回到 2014 年，似乎每隔一周就有一个新的热门“比特币 2.0”发布(无论是 Dogecoin、Auroracoin、Maidsafe、SwarmCoin、Bitshares 还是其他)。这些实验大多以失败或缓慢消退而告终。现在，出于只有上帝知道的原因，appcoin 的概念似乎又回到了桌面上

现在情况和 T4 不太一样了。(此外，MaidSafe 并不是真正的加密货币。)如今，appcoins 正被编织成项目，将它们用作内部货币，而不仅仅是为了自身而存在。但是，我们是否正在进入一个由许多独立的“appcoin”区块链组成的世界，还远未明朗。在一些特别有技术野心的案例中——比如以太坊——可能需要一个新的 appcoin /区块链……但是如果没有这种野心，这种方法通常会带有寻找问题解决方案的痛苦味道。Appcoin 概不退换。

块堆栈

我对加密货币最感兴趣的既不是“加密”也不是“货币”部分。让我感兴趣的是，它们是分散的、无许可的服务，不依赖于——因此可以被——任何中央控制者缩减。人们说互联网是去中介化的，但事实上，我们惊人数量的在线活动是通过五大平台进行的:亚马逊、苹果、脸书、谷歌和微软。一些人认为去中介化的想法很有趣，也很有破坏性。

因此，我对去中心化技术产生了兴趣，比如 IPFS 和最近的联合广场风险投资公司/ Y Combinator 的初创公司 Blockstack

一个面向分散式、无服务器应用的新互联网…区块链用于维护跨应用身份系统，将用户 id 安全地映射到用户名、公钥和数据存储 URIs。开发人员不必担心运行服务器、维护数据库或构建用户管理系统

作为一名软件开发人员，这听起来很可爱，但也很像那些听起来很棒的工具/框架中的一个，直到你深入到它们的杂草中，之后你最终意识到它们的技术限制最终会导致比它们解决的问题更多的问题。也许我错了；我希望我是错的。也许我本能地愤世嫉俗，因为这听起来太令人兴奋了。也许我们会看到它的浏览器何时从公测中出现。

据我所知，这种浏览器利用个人用户存储在 Dropbox 或类似服务中的分散数据构建网页，不需要任何中央服务器。此外，它将使用区块链(比特币的)进行域名注册，这是我长期以来一直呼吁的。“顺便说一句，所有这一切也让网站建设变得更加容易，”热情地告诉观察家。作为一个频繁的网站建设者，我仍然持怀疑态度，直到我看到细节。但是我充满希望。

这也是我的感受

Zcash

哪家还刚推出，哪家期待了一段时间。这是另一种加密货币，但不仅仅是另一种加密货币:引用，呃，我自己的话，它

将允许不包含任何关于其发送方或接收方或金额的公开信息的交易——但所有这些仍然可以使用零知识证明来验证，该证明被称为 zk-SNARKs，意为“零知识简洁的非交互式知识论证”

这是非常了不起的。我知道我刚刚花了一段时间对新区块链持怀疑态度，但 Zcash 就像以太坊一样，是一项根本性的技术进步，而不仅仅是使用相同技术的新项目。对一些人来说，真正匿名交易的前景可能听起来令人担忧，但在一个隐私每天都在消失的世界里，这听起来也很像一个令人安心的堡垒。

更平淡无奇的是，引用创始人兼摇滚朋克祖科·威尔科克斯的话

在这篇关于 Zcash 的 IEEE 文章中:

“所有部门都有保护隐私的监管、商业和道德原因，”他说。举一个商业例子:苹果不希望三星能够跟踪其交易并获得有价值的竞争情报。

威尔科克斯的话听起来很鼓舞人心:“我希望能够说我们在那里，推动那场伟大的变革，这场变革开始洗刷令人窒息的低效、腐败和孤立——这场变革释放了数十亿被困在合作边界背后的人的潜力！”令人鼓舞的是，ZCash 为支付了启动前安全审计，这将有望帮助他们避开以太坊遇到的一些障碍。

…当然，再一次，当你进入杂草中时，你总是会发现有连贯论据的怀疑者。这里有一个吹捧它的竞争对手 Monero，它使用环签名而不是 zk-SNARKs 来匿名化交易。

但是关于 ZCash 最怪异和最电影化的事情不是关于相对功效的深奥的争论:它是一个神秘的仪式，它必须被启动，之后神秘的秘密必须被摧毁。不，说真的。这种数学方法不可避免的副作用是(过于简化)当 ZCash 区块链最初建立时，它的发起者有可能保留“秘密密钥”,这将允许他们随时匿名创建和消费新的伪造 ZCash。

ZCash 自己称这些密钥为“T8”有毒废物“T9”，他们报告说，上周他们经历了一个数学和物理上复杂的仪式，在这个仪式中，网络被启动，“有毒废物”被销毁。他们的博客文章解释了这个既超现实又令人敬畏的故事。

这个世界可能想要采用一种从根本上来说需要这种情况发生的货币吗？…我不知道！(这些参数可以被交换出来，如果相信妥协了，但是你怎么能保证下一个组的参数呢？)尽管如此，不管目前颁布的 ZCash 是否会成为一股经济力量，我都看好它的技术；这是向保证隐私迈出的重大而令人印象深刻的下一步，因此，所有–重要的 可替代性。

天哪！

回到加密货币的主流，如果有这样的东西，比特币和以太坊这两大区块链生态系统也非常忙碌。以太坊硬碰硬，再来，对付针对其安全漏洞的攻击者，再来！(早告诉你了)。)比特币社区再次自相残杀！

……此外，除了《骚动与不安》,许多有趣的事情正在发生。麻省理工学院媒体实验室的 Blockcerts“比特币区块链的数字学术证书开放标准”已经发布。在一个充斥着假证件的世界里，这看起来像是可以起飞了。与此同时，随着银行和大公司纷纷涉足区块链，两家独立银行利用区块链和智能合约技术进行的首次全球交易终于完成。不过，如果他们希望跟上当今的变化步伐，他们必须加快步伐。
