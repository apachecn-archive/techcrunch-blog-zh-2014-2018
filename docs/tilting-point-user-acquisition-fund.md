# Tilting Point 将其用户获取基金的年度支出扩大至 1.32 亿美元

移动游戏发行商和营销商 [Tilting Point](https://web.archive.org/web/20221207211022/https://www.tiltingpoint.com/) 正在大幅增加其对用户获取基金的承诺。

该公司[在 2016 年底宣布了一个 1200 万美元的基金](https://web.archive.org/web/20221207211022/https://venturebeat.com/2016/11/29/tilting-point-launches-12-million-user-acquisition-fund-for-mobile-game-devs/)，它说这将帮助开发者在保持独立的同时发展他们的游戏。今天，该公司透露将每年向该基金投入 1.32 亿美元。

首席执行官凯文·塞加拉(Kevin Segalla)表示，随着移动应用商店变得越来越拥挤，“用户获取变得非常复杂”，因此大多数独立开发者“不具备这样的技术和专业知识。”

这就是倾斜点的由来。总裁 Samir El Agili 表示，该公司已经建立了“机器学习技术，以最大限度地优化用户获取。”它喜欢从事处于“十字路口”的游戏，选择一款具有可持续商业模式的稳定游戏，然后通过广告大幅加速其增长。

最初的资金导致了在[颠覆者 Beam 的*星际旅行时间轴*](https://web.archive.org/web/20221207211022/https://www.gamasutra.com/view/pressreleases/312116/Disruptor_Beam__Tilting_Point_Partner_for_Star_Trek_Timelines_UA_Campaign.php) 和 [Nukebox 工作室的*食品卡车厨师*](https://web.archive.org/web/20221207211022/https://www.gamasutra.com/view/pressreleases/311751/NUKEBOX_STUDIOS_PARTNERS_WITH_TILTING_POINT_FOR_MULTIMILLION_DOLLAR_UA_CAMPAIGN_TO_CONTINUE_GROWTH_OF_BREAKOUT_MOBILE_GAME_FOOD_TRUCK_CHEF.php) 的合作。

塞加拉说，鉴于该基金的成功，问题变成了“我们如何在更大的规模上这样做？”由于 CFC Capital (Tilting Point 的大股东)和 Metropolitan Partners Group 承诺的资金，这导致了更大的资金承诺。

需要澄清的是，1.32 亿美元并不是 Tilting Point 给自己的服务贴上的价格标签，而是代表着实际将用于广告的资金。

Segalla 认为，这些交易的结构使得 Tilting Point 的激励措施与开发商的激励措施完全一致。

“这不是我们给他们的贷款，也不是我们寻求的股权，没有持续的收入分成，”他说。相反，该公司押注这笔支出将在与开发商的关系以及由此产生的费用中获得回报:“我们正在做的事情是拿我们自己的资本冒险，因为我们相信我们的营销、技术和团队。”

Tilting Point 表示，它愿意与任何类型的开发者合作，也希望与国际开发者合作。Segalla 预测，该基金每年可以让 Tilting Point 与 20 款新游戏合作，尽管 El Agili 指出，确切的数字将取决于游戏:“事实是，如果马上有两到三款游戏表现非常好，我们就可以开始花很多钱。”
