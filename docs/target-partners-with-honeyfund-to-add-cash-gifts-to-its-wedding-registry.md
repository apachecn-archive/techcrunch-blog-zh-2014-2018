# 锁定 Honeyfund 的合作伙伴，将现金礼物添加到其婚礼登记处

归功于媒体曝光的力量。负责建立 Target 在线注册的团队看到 [Honeyfund](https://web.archive.org/web/20221208040943/https://www.crunchbase.com/organization/honeyfund) 出现在“鲨鱼池”上，并认为将蜜月注册服务集成到 Target 自己的服务中会很酷。如今，这种整合已经在 Target 的网站、移动网络和专用的 Target registry 应用程序中实现。

这家零售商告诉 TechCrunch，与 Honeyfund 合作的决定与它在礼品注册业务中的既定名称以及它在当今消费者中的受欢迎程度有关。

该在线登记处成立于 2006 年，允许朋友和家人向新婚夫妇赠送现金礼物——如旅行资金、蜜月相关的郊游和活动、房屋首付或任何他们喜欢的东西。

![](img/7dce4cfa5ac991c35735c5a0d99f1833.png)

通过 Target partnership 和新推出的 Honeyfund integration，情侣们现在可以从一个目的地注册他们想要的一切，包括实物礼物和现金礼物。从理论上讲，这可能有助于留住那些原本可能选择非传统但越来越受欢迎的现金礼品路线的客户。这也可能对初创公司的礼品登记服务构成挑战，比如佐拉和 T4。

Target 表示，它与 Honeyfund 的交易只是一种合作关系，没有金钱交易，也不构成零售商的财务投资。然而，这并不是 Target 第一次利用第三方技术来增强其面向消费者的产品。例如，公司[去年秋天在其网站和移动应用程序中添加了 Pinterest 的视觉搜索工具](https://web.archive.org/web/20221208040943/https://beta.techcrunch.com/2017/09/25/target-is-adding-pinterests-visual-search-tool-to-its-app-and-website/)。

今天的视觉搜索功能可以根据智能手机拍摄的照片推荐产品添加到注册表中。这适用于各种注册，包括婴儿、周年纪念、婚礼等等。

另一方面，Honeyfund 集成只能在目标婚礼注册中心内部使用。

“随着时间的推移，人们对结婚登记处的期望已经发生了变化，”塔吉特百货公司高级副总裁道恩·布洛克在一份关于新伙伴关系的声明中说。“客人正在寻找完整的注册体验，包括产品和蜜月选项，而 Target 是第一家能够在 Target.com 上提供所有这些服务的零售商。”
