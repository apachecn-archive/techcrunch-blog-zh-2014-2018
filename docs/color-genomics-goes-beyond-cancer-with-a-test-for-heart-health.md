# 彩色基因组学超越癌症，测试心脏健康

在美国，癌症和心脏病是导致死亡的两个主要原因。到目前为止， [Color Genomics](https://web.archive.org/web/20221208124440/https://www.color.com/) 一直专注于测试导致某些癌症更高风险的突变。但是，今天这家成立四年的公司推出了一种新的心血管健康基因检测。

新的彩色遗传性高胆固醇测试将告诉你是否有一种叫做家族性高胆固醇血症(FH)的基因突变，这是一种导致高胆固醇水平导致冠心病的遗传性疾病。

全世界可能有 3400 万人受到这种疾病的影响。大约每五十个高胆固醇人群中就有一个人有这种突变。问题？大多数有基因突变的人直到他们有潜在的致命心脏病发作时才知道自己有基因突变。

像癌症检测一样，早期检测到突变可以预防疾病，提高存活率并降低医疗成本。这就是 Color 希望它的新测试能有所帮助的地方。

“我们从癌症开始，因为它是死亡的主要原因之一，围绕遗传和癌症的科学已经得到了很好的确立，”Color 首席营销官凯蒂·雅各布斯·斯坦顿告诉 TechCrunch。“同样，围绕遗传和心血管疾病有着成熟的科学……鉴于心血管疾病是死亡的主要原因(加上每年超过 1.1 万亿美元的癌症成本)，我们看到了一个机会，可以帮助更多人了解他们患遗传性心血管疾病的风险，并利用遗传数据主动管理他们的心脏健康。”

不像 23andMe 这样的家庭基因测试，你可以通过你的医生来安排这项测试。新客户的测试费用为 249 美元。然而，那些已经通过 Color 癌症测试的人可以额外支付 150 美元购买心血管测试。
