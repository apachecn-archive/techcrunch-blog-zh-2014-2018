# TechShop 迎来新 CEO 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/07/18/techshop-gets-a-new-ceo/>

![shop](img/b59e33ea1053b0a2bdab8aad6889568c.png)

如果你对建筑和制造感兴趣，并且住在大城市附近，你可能听说过 TechShop。就像 DIY 极客的健身房一样，这个想法是他们购买并维护各种昂贵的机器——激光切割机、数控铣床、3D 打印机等——你每月向他们支付费用。

根据 TechShop 创始人吉姆·牛顿(Jim Newton)发给会员和投资者的一封电子邮件，首席执行官马克·哈奇(Mark Hatch)已经辞职，专注于其他项目。Hatch 在 TechShop 工作了大约 7 年。

马克在上述电子邮件中写道:

> “经过深思熟虑和咨询后，我做出了一个最艰难的决定。我决定辞去 TechShop 首席执行官的日常工作，这样我可以最大程度地帮助 TechShop 和创客运动。我得出的结论是，我可以通过将我的日程表扩大到更多的政策、教育、企业演讲、写作和咨询活动，来帮助这场运动更快更好地发展。”

丹·伍兹之前曾帮助创办《MAKE》杂志，并且已经是 TechShop 董事会成员，他将立即担任首席执行官一职
