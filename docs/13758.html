随着脸书将 NLP 引入其更新的 Messenger 平台，Wit.ai 正在关闭 Bot 引擎

Wit.ai 今天早上在一篇博客文章中宣布，它将关闭其 Bot 引擎。这家脸书拥有的公司为自然语言处理开发工具，帮助工程师更快地建立语音和文本聊天机器人，并减少技术经验。

机器人引擎于 2016 年 4 月作为测试版推出。该工具允许开发人员使用共享相似结构的示例对话来训练他们自己的机器人。这些例子可以代表真实的对话，并通过真实的对话日志进行更新，以进行微调。目标是允许训练一个灵活的机器人进行几十次对话，而不是几百万次。

但是这个引擎是为纯文本交互设计的，而且机器人生态系统已经成熟得如此之快，以至于这项技术已经过时了。该团队指出，Messenger 和其他平台已经在文本之外增加了新的交互方式。这一切都是为了更好，但它并没有善待 Bot 引擎的增值。

Wit.ai 表示，有超过 10 万名开发者使用其服务。但是在这个组中，90%的 API 调用都是针对 Wit 的 NLP API 的。Bot 引擎和 Stories UI 将持续到 2018 年 2 月 1 日，以便给开发者时间迁移他们的应用。

无独有偶，脸书今天宣布将自然语言处理工具集成到 Messenger Platform 2.1 中。随着更新，开发者可以从信息中提取信息，如日期、时间、位置、金额、电话号码和电子邮件。然后可以使用 Wit.ai 定制 Messenger 平台的 NLP 集成的功能。
