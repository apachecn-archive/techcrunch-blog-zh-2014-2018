由于黑客利用币安的 API 密钥，比特币价格下跌 10%

欢迎来到加密货币社区的又一个过山车日。短短几分钟内，比特币价格从 10740 美元暴跌至 9690 美元，跌幅达 10.8%。

和往常一样，很难确定到底发生了什么。但是有一家公司的日子特别不好过。加密货币交易所币安已经发现一些不寻常的活动并停止取款。

币安是最大的交易所之一。根据 CoinMarketCap 的数据，就交易量而言，它是顶级加密货币的四大交易所之一。

许多人注意到 Viacoin 在崩溃前发生了一些极不寻常的事情。币安的虚拟货币购买订单大幅增加。短短几分钟内，Viacoin 的市值从 6400 万美元跃升至 1.59 亿美元

币安对此进行了调查，并注意到未经授权的销售订单。“我们正在调查一些用户资金出现问题的报告。我们的团队已经意识到并正在调查这个问题，”该公司在 Reddit 上写道。“到目前为止，唯一确认的受害者已经注册了 API 密钥(用于交易机器人或其他用途)。没有证据表明币安平台受到了损害。”

因此，看起来像是第三方服务或应用程序受到了威胁。该第三方应用的用户依靠 API 密钥来控制币安账户。一个黑客可能开发了一个同时提交订单的机器人(太平洋时间早上 7 点)。

虽然币安团队停止了撤军，但可能为时已晚。也有可能黑客已经在另一家交易所持有大量 Viacoin 股票。黑客可能在操纵币安的价格后不久就卖出了一大堆 Viacoins。

在所有情况下，它再次证明了安全是加密货币的一个大问题。不要把你的硬币存放在交易所。使用硬件钱包或允许您控制私钥的钱包。
