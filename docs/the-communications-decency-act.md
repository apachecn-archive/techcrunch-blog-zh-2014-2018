# 正派可以立法吗？

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/10/08/the-communications-decency-act/>

杰夫·科瑟夫是美国海军学院网络安全法助理教授。所表达的观点仅代表他本人，不代表海军学院或海军部。

More posts by this contributor

你可能从未听说过《通信体面法案》第 230 条。然而，近二十年来，1996 年的联邦法律决定了人们使用互联网的方式。第 230 条为互联网服务提供商、网站和其他在线服务提供了对用户提供的内容的广泛法律豁免权。因此，举例来说，如果一个脸书用户发布了一个关于他邻居的虚假和破坏性的评论，这个邻居就不能成功地对脸书提起诽谤诉讼——这要感谢第 230 条。

如果没有这种法律保护，社交媒体公司和其他网站及应用程序就不太可能允许已经成为现代互联网定义的用户生成内容。访问网站就像阅读杂志或看电视，而不是我们都期待的互动体验。

230 条款长期以来一直受到各种反对者的攻击，这些反对者希望网站和 ISP 为他们用户的内容负责。最近几个月，这些反对者在法庭上取得了一些重大胜利。如果他们继续侵蚀 230 条款，我们所知道的互联网将会变成一个更加封闭和受控的环境。

上个月，犹他州的一名联邦法官允许一家家庭报警公司对 Ripoff Report 的运营商提起诉讼，该网站允许用户发布消费者评论。这起诉讼源于一家竞争对手据称发布的关于这家报警公司的负面评论。法院的结论是，尽管第 230 条规定了广泛的保护，但如果 Ripoff Report“鼓励”了攻击性的用户内容，它可能要对这些内容负责——这是许多其他法院长期以来拒绝接受的结论。

本月，华盛顿最高法院允许对 backpage.com 提起诉讼。原告是三名未成年人，他们声称自己通过 backpage.com 上的广告被出售以获得性服务，导致自己被强奸。法院推断，诉讼可以向前推进，因为 backpage.com 制定了广告内容要求，因此可能有助于广告的创作。

> 最近的法院意见给网站和社交媒体公司带来了很大的不确定性。

我们为什么要关心网站和在线服务是否要对其用户的内容负责，尤其是当这些内容可能导致身体和情感伤害的时候？正如任何看过用户评论的人都会证明的那样，许多推文、帖子和其他未经编辑的评论真的很卑鄙，往往展示了人性最糟糕的一面。我不——也不可能——为大量在线用户内容辩护，尤其是 backpage.com 上的可怕广告。

但是第 230 条正确地将责任放在了创建在线材料的用户身上，而不是作为管道的网站。如果一个人在互联网上发布诽谤性评论或利用未成年人，该用户应该在法庭上承担责任。

自 1996 年以来，法院普遍执行这一原则，并为网站和其他在线服务提供清晰和确定性，以允许用户在线发布评论、照片和其他材料。然而，这些最近的法院意见给网站和社交媒体公司带来了很大的不确定性。

如果法院继续让在线服务为用户生成的内容负责，结果最终将是单向、封闭的互联网，网站、应用和其他平台不允许消费者创建和发布内容。正如电子前沿基金会正确观察到的，backpage.com 的决定表明网站“如果不提供任何互动服务，就是最安全的”

想象一个没有社交媒体、消费者评论和用户论坛的互联网。这样的变化会摧毁网络世界。过去 20 年来，正是互联网的互动特性让它蓬勃发展。不幸的是，法院对联邦法律的曲解会减缓甚至逆转这一进程。

只要人们对用户内容不满，230 条款就会继续在法庭上面临攻击。坚定立场，捍卫在现代互联网发展中发挥了根本作用的法律，符合科技界的最大利益。
