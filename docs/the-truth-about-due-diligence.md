# 尽职调查的真相

> 原文：<https://web.archive.org/web/http://techcrunch.com/2016/05/19/the-truth-about-due-diligence/>

风险投资有一个表达方式。这叫做“哦，妈的”董事会议。奥古斯特资本(August Capital)的普通合伙人大卫·霍尼克(David Hornik)表示:“当你开了一家公司，进行第一次检查后，你会学到很多你希望知道的东西。”。

很容易想象，它们在整个初创企业行业中定期发生。近年来，融资速度一直很快，给投资者比以往任何时候都少的时间来评估他们资助的初创公司。血液检测设备 [Theranos](https://web.archive.org/web/20230328135750/http://www.wsj.com/articles/theranos-voids-two-years-of-edison-blood-test-results-1463616976) 和无线充电公司 [uBeam](https://web.archive.org/web/20230328135750/http://liesandstartuppr.blogspot.com.au/) 等一度声名显赫的公司似乎也在为自己的生命而战，这也引发了许多问题。

《名利场》最近的一篇文章将某些公司的崛起归咎于[硅谷媒体](https://web.archive.org/web/20230328135750/http://www.vanityfair.com/news/2016/05/theranos-silicon-valley-media)(尤其是 TechCrunch)。与此同时，本周早些时候在《Fast Company》上发表的一篇文章暗示，旋转的[文化](https://web.archive.org/web/20230328135750/http://www.fastcompany.com/3059761/inside-silicon-valleys-culture-of-spin)是问题的根源。正如一位创始人告诉媒体的那样，“在硅谷诚实就像是没有服用类固醇的奥运团队中的一员。”

当然，如果不是投资者，我们中没有人可能听说过 Theranos 或 uBeam，他们分别给了这两家公司 6.86 亿美元和 2500 万美元。这些支持者是否过于乐观？他们被骗了吗？他们有注意吗？当我们等着看这些故事如何展开时，很容易摇摇手指，但事实是:尽职调查只进行到此为止。虽然有些人可能认为这是一个科学的过程，将风险公司与不良投资隔离开来，但尽职调查是一个令人惊讶的不完善的过程，有很多局限性。

“如果你在勤奋工作中寻找一个非黑即白的答案，那将是一个失败，”马特·墨菲说，他曾是凯鹏华盈考菲尔德&拜尔斯公司的合伙人，一年前加入门洛风险投资公司担任董事总经理。“你通常在处理灰色阴影。”

**真相会让你自由**

正如您可能想象的那样，尽职调查意味着不同的事情，这取决于初创公司运营的行业及其发展阶段。风投们说，在开始的时候，它通常包括直觉、对市场机会的研究(当有市场需要研究的时候),以及，如果做得好的话，很多关于创始人的电话。

Allegis Capital 的创始人兼董事总经理鲍勃阿克曼(Bob Ackerman)表示:“在创业初期，你面临的挑战是，除了愿景之外，没有任何东西可以推销。

通常情况下，风投都是推荐人，这很简单。霍尼克说:“你通常会被介绍给一群曾与这位企业家共事过的人，你有那一群电话，他们进展顺利，因为企业家已经把你引荐给了那些人。”

如果一个风投真的在做功课，那么这个人会找到创始人的前同事，而这些人不在他们的推荐名单上。

即便如此，还是需要一些字里行间的解读。“人们的自然倾向是说非常好的话，即使有人没有做好工作，”霍尼克说。“当有人告诉我某人‘做得很好’，他们是‘好领导’时，这是一个糟糕的评价。一个很好的参考是，‘这个人真了不起。我会毫不犹豫地再次与她合作。这就是你想要的。"

初创公司发展过程中后期的客户来电也是如此。自然地，风投被赋予了那些倾向于喜欢这家公司的客户的名字。但是“你必须找到证明人，”门罗风险投资公司的墨菲说。如果潜在买家是一名首席安全官，那么你可以与公司尚未接触过的一些 CSO 交谈，向他们解释产品和痛点，并询问他们是否愿意会面，或者，有时，是否有其他竞争对手正在与他们合作。"

尽职调查看起来也因机会而异。社交新闻聚合网站 Nuzzel 的创始人乔纳森·艾布拉姆斯(Jonathan Abrams)是总部位于旧金山的风险公司 Data Collective 基金的有限合伙人，该公司以进行从卫星到抗病毒治疗的广泛押注而自豪。艾布拉姆斯指出，与许多公司一样，Data Collective 有许多专业的博士顾问，帮助他们对合伙人可能也不了解的产品进行尽职调查。

“如果你投资下一个 Snapchat 或 Slack，许多风投可能会对自己的投资充满信心，”艾布拉姆斯说。“当你投资于更难理解的东西时，这种情况就不太可能发生。”

**只是事实，女士**

这并不是说风投不能做得更好。例如，投资者很容易失去对正在发生的事情的了解，特别是当他们正在投资更成熟的初创公司，担任大量董事会成员，和/或不觉得有必要密切关注事情的时候。(一位要求不透露姓名的创始人称他的一位后续投资者为“好时光查理”，暗示该公司没有为公司做很多事情。)

在艰难时期，即使是已经订婚的董事会成员也可能被欺骗。“当你在董事会时，企业家误导你没有问题，”霍尼克说。“当(你)发现一名首席财务官在撒谎或有人在偷窃时，你会问，‘我能做什么？’但事实是，作为董事会成员，你并不是在审计公司。你能做的最好的事情就是和整个公司的人保持密切的关系，这样你就能从各种渠道听到正在发生的事情。如果你不这样做，就增加了你被模糊误导的可能性。"

显然，风投也会进行过多的尽职调查。墨菲说:“你可以做得过分。”“你可以通过与正确或错误的人交谈来说服自己放弃很多事情。很容易在通话之间摇摆不定。很多这种东西都是信仰的飞跃，这就是为什么这个行业如此艰难。”

“你可以理解风投希望避免投资下一个 Theranos，”艾布拉姆斯同意道，他已经从四家公司的风投那里筹集了数百万美元，并亲自投资了大约 50 家初创公司。“但过于规避风险也不好。有时候，一开始就有危险信号的公司最终会赢得最大的胜利。”他指出，优步是一个典型的例子。

然而，投资者犯下的最大罪过就是过于依赖同行的声誉，以此作为尽职调查的一种形式。(并不是。)

Clinkle 是一家支付公司，它在 2013 年获得了 2500 万美元的“种子”融资，被称为硅谷历史上最大的一笔融资——这可能是这种错误思维最引人注目的例子。它的财团包括 Accel Partners、Andreessen Horowitz、Intel 和 Intuit，以及十几位技术名人，包括彼得·泰尔和马克·贝尼奥夫。该公司的产品从未进入市场。相反，到去年春天，Clinkle 已经在很大程度上内爆了。

《华尔街日报》同样[暗示](https://web.archive.org/web/20230328135750/http://www.wsj.com/articles/at-theranos-many-strategies-and-snags-1451259629)是 DFJ 的支持吸引了对该公司的进一步投资，该公司曾支持 Space X、特斯拉汽车和行星实验室，并为 Theranos 开出了第一张支票。明星效应似乎也吸引了明星效应到 uBeam。其支持者包括亿万富翁马克·库班、雅虎首席执行官玛丽莎·梅耶尔、彼得·泰尔创始人基金和安德森·霍洛维茨。

阿克曼说:“我认为(创始人)不一定试图误导任何人。”。但是他们“从其他人——他们的顾问，他们的董事——那里借用信誉，知道如果他们借用这种地位，他们会更容易筹集资金。当资金流动时，你尤其会看到这种行为变得明显。”

墨菲表示:“我很惊讶一些投资者经常会跟在一个大品牌后面，比如，‘嘿，如果这是 KP 和红杉，而且[交易包括]这个合作伙伴和这个行业，我们需要迅速行动。’”他坚持认为聪明的风投更了解这一点。

“如果有一个盲点，你本可以检查出来(却没有)，那你应该感到羞耻，”他补充道。

**每个故事的三个侧面**

当然，风险投资是建立在承担风险的基础上的。这意味着在一个变化比以往任何时候都快的世界里对技术下注。除了围绕欺诈和夸大能力的创始人的明显挑战之外，总有一种危险，即一家公司的技术将在纸面上得到验证，但由于这样或那样的原因，无法在市场上生存。

“环境在变。“人是会变的，”Twitter 前产品总监 Ryan Sarver 说，他两年半前作为合伙人加入了红点风险投资公司。“投资之后，事情有时会发生你无法预料的变化。”

他说，这发生在阿克曼身上。“几年前，我们投资了斯坦福大学的一项技术，根据我们的评估和该技术的开发者以及市场上正在发生的事情，我们认为它确实具有颠覆性。”

阿克曼说他不明白的是“还有多少基础科学有待证明。我们在做开创性的事情，我们并不像我们集体认为的那样接近过程的终点。我们开始假设问题会出现在哪里，我们意识到需要解决的问题比我们预想的要多得多。”

阿克曼说，最终，产品错过了它的窗口。

他对此感到后悔，但他也没有为此自责。这是野兽的天性。

“没有真正的路线图。我们以为我们在 Y 点，但我们在 x 点。我不确定还有其他方法可以发现这一点。”
