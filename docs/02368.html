泰国政府声称可以监控该国 3000 万网络用户

原文：https://web.archive.org/web/https://techcrunch.com/2014/12/23/thailand-line-monitoring-claim/

去年，泰国政府官员要求访问聊天应用 Line，以便他们能够监控该国发生的对话，本周，一名政治家声称他们现在可以这样做。

《国家报》报道 ICT 部长 Pornchai Rujiprapa 在新闻发布会上表示，政府“可以监控泰国人民每天发送的近 4000 万条线路信息”

Line 的一名发言人否认了原始报告的真实性，并告诉 TechCrunch，它没有向泰国政府提供用户信息。完整的声明在这篇文章的底部。

这位部长的说法在泰国颇具启发性，泰国是亚洲少数几个移动通讯应用规模超过脸书的市场之一。Line 声称在泰国有超过 3000 万注册用户，这可能意味着的注册用户比脸书还多。

如果泰国政府正在监控 Line，这看起来像是在补充其正在进行的阻止一系列社交网络上被视为非法的内容的努力。

Pornchai 部长透露，任何用户如果看到违反泰国严格的冒犯君主罪法律(防止诽谤君主制)的信息，应该联系能够“追踪信息来源”的当局。

过去根据冒犯君主罪进行的逮捕与发送短信、脸书信息、互联网论坛和网站有关，也许 Line 很快也会成为其中的一部分。但也有一股新的政治激进主义暗流，泰国当局可能热衷于打击。

泰国陆军总司令(现任总理)通过今年五月的军事政变从民选政府手中夺取了权力。新政府面临许多抗议，最近包括学生借用饥饿游戏电影系列的三个手指敬礼。

军政府威胁要审查用户批评它的各种网站，T2 甚至暂时屏蔽了脸书。今年夏天，官员们试图安排与谷歌和脸书的会面，讨论审查措施，但是没能让两家公司坐到谈判桌前。

从隐私角度来看，当局现在可以自由过滤通过该国顶级社交网络发送的所有信息的说法是一个非常令人担忧的发展，更不用说 Line 自身的安全了。

例如，去年 Line 被发现容易受到中间人攻击，这种攻击可能允许第三方访问通过蜂窝数据发送的信息。Line 修补了这个问题，并且从开始引入了一个加密聊天功能，但是有趣的是，我从未见过任何人在这个功能推出后还在使用它。

政府仍有可能使用自己的系统来监控这项服务，因此不需要 Line 的合作。官员们被认为正在为一个更有效的审查系统规划他们自己的互联网网关，这可能与此有关。

这个项目没有公开宣布任何细节，即使它是真实的，也需要数年时间来建设，但它突显了一种可能性，即泰国可能会与互联网服务提供商、运营商和服务提供商合作，而不是互联网公司自己来监控和治理其网络空间。

以下是 Line 的声明全文:

LINE 已经向[泰国的] ICT(信息和通信技术部)证实，ICT 否认了所谓的 LINE 信息被 ICT 监控的事实。

由于 LINE 对所有信息都进行了加密，从技术上讲，任何第三方都不可能监听 LINE 的对话。

LINE 用户的隐私是我们的首要任务，因此我们不允许其他第三方监控 LINE 消息。
