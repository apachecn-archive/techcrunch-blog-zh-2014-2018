# 适马最新的全画幅广角变焦镜头具有恒定的 F2 光圈

适马有一种新的变焦镜头，可能是街头摄影师的完美随身镜头:一种为全画幅相机设计的 [24-35mm 变焦镜头](https://web.archive.org/web/20221209164926/http://www.sigmaphoto.com/product/24-35mm-f2-dg-hsm-a)，它有一个独特的功能，肯定会增加它的多功能性——一个恒定的大光圈 f2。

对于许多 DSLR 摄影师来说，这种镜头可以取代三种标准的配套镜头，包括 24、28 和 35mm 焦距。以前，如果你想要一个接近 f2 标志的光圈，你必须为每个光圈准备单独的玻璃；适马之前为作物传感器(APS-C)格式的数码单反相机提供了宽变焦，但这款相机也适用于像佳能 5D 系列这样的全画幅相机。

低光圈值意味着你将能够实现出色的背景散焦效果(bokeh)，镜头还包括适马安静的 HSM 自动对焦机制，以及九叶片圆形光圈，以确保你的 bokeh 良好而平滑。它还可以与适马的 USB dock 配合使用，这意味着你可以对它进行编程，以在整个范围内调整自动对焦性能。

适马尚未透露定价或可用性，但根据目前的定价，它可能会在 800 美元至 1000 美元左右。
