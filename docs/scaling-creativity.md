# 失去我的名字，儿童“全栈”个性化图书出版商，从项目 A 风险投资中获得€400 万英镑

总部位于伦敦的“全栈”出版商和儿童内容初创公司“迷失我的名字”为其令人印象深刻的支持者名单增加了一个新的投资者，该名单已经包括谷歌风险投资、格雷克罗夫特和 [Forward Parters](https://web.archive.org/web/20221207012054/https://beta.techcrunch.com/2015/10/11/interview-nic-brisbourne/) 。

这一次，总部位于柏林的 Project A Ventures 向该公司投资了€400 万英镑，据称这是《迷失我的名字》最初的[900 万美元首轮融资](https://web.archive.org/web/20221207012054/https://beta.techcrunch.com/2015/06/25/lost-my-name/)的延伸。

在一次电话中，Lost My Name 联合创始人兼首席执行官 Asi Sharabi 告诉我，基于 Project A 提供的运营支持，决定进一步创办公司——据我所知，该公司实际上在几个月前就关闭了。特别是在商业智能、CRM 和绩效营销领域。

事实上，我听说这家德国风险投资公司已经从伦敦初创公司借调了自己的 100 多人的运营支持团队，致力于建立一个内部商业情报部门。该公司有几位知名的 Rocket Internet 校友担任管理合伙人。项目 A 的创始合伙人弗洛里安·海涅曼也加入了“丢失我的名字”的董事会。

[最初是作为一个附带项目](https://web.archive.org/web/20221207012054/https://beta.techcrunch.com/2015/10/20/interview-asi-sharabi/)成立的，Lost My Name 是最畅销的儿童个性化绘本《失去名字的小男孩/女孩》以及后续作品《不可思议的星际回家之旅》背后的公司，该书以更有趣的方式注入数据，使最终的书及其故事更加个性化。

为了实现这一点，这家初创公司建立了一个定制的电子商务和出版平台，使其能够对出售的每本故事书进行个性化定制，包括从第三方 API(如地图)获取数据，以及使用客户提供的数据，如孩子的名字。它还在世界各地拥有各种印刷合作伙伴，因此 Lost My Name 能够在按需和零库存的基础上运营。

为此，沙拉比将这家初创公司比作 Warby Parker 这样的“垂直整合”公司，从某种意义上说，Lost My Name 控制了其内部业务流程的大部分方面，从创作故事、插图到营销和销售。

然而，在我们的通话中，他告诉我这种情况即将改变，Lost My Name 正在积极寻求与外部创意人员和公司合作，因为它“打开了平台”。“技术和商业模式可以扩展，但创造力不行，”他告诉我。

他尤其希望与富有创造力的开发者建立联系，这些开发者将看到将数据注入发布流程的潜力，从而创造出全新的儿童内容。沙拉比还透露，这家初创公司正在制作个性化的字母海报，因此正在脱离纯粹的图书出版商。
