# 来源:微软考虑以 80 亿美元收购 Slack，将专注于 Skype

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/03/04/source-microsoft-mulled-an-8-billion-bid-for-slack-will-focus-on-skype-instead/>

当 [Slack](https://web.archive.org/web/20230301120343/http://www.slack.com/) 本周早些时候宣布[新的语音和视频服务](https://web.archive.org/web/20230301120343/https://techcrunch.com/2016/03/01/slack-roadmap/)时，这家企业通讯初创公司发出了进军由[微软的 Skype](https://web.archive.org/web/20230301120343/http://www.skype.com/) 等公司主导的领域的信号。但这似乎不是两家公司最近唯一的交集。

TechCrunch 获悉，微软将 Slack 视为一个高达 80 亿美元的潜在收购目标。但围绕出价的内部活动未能赢得支持。我们的消息来源称，微软联合创始人比尔·盖茨和首席执行官塞特亚·纳德拉不相信这个想法，盖茨反而推动给 Skype 添加更多功能，使其在商业市场上更具竞争力。

Slack 吸引新用户的势头——目前它有 230 万日活跃用户，其中 67.5 万人付费——使它成为其他希望在企业协作服务领域领先的公司的竞争威胁。

这并不奇怪，截至去年，Slack 已经有[已经有 8-10 个收购要约](https://web.archive.org/web/20230301120343/https://techcrunch.com/2015/05/28/slack-has-seen-8-10-acquisition-offers-so-far/)。

微软负责 Slack 的人是应用和服务的执行副总裁陆弃。根据他在微软高管档案页面上的介绍，陆兆禧负责微软的所有生产力、通信、教育、搜索和其他信息服务，他还“制定应用和服务集团的愿景、战略和总体方向”，包括微软 Office、Office 365、SharePoint、Exchange、Yammer、Lync、Skype、Bing、Bing Apps、MSN 以及广告平台和业务集团的研发。

据我们所知，卢曾设想以高达 80 亿美元的价格收购 Slack，考虑到本周有关这家初创公司的一些其他消息，这是一个有趣的数字。在 2015 年 4 月宣布的最近一轮 1.6 亿美元融资中，Slack 的估值为 28 亿美元。然而，我们[和其他人](https://web.archive.org/web/20230301120343/http://www.wsj.com/article_email/slack-technologies-seeks-funding-for-business-messaging-app-1456885352-lMyQjAxMTA2ODA5MjgwMzI5Wj)听说 Slack 正在筹集另一轮 1.5 亿至 3 亿美元的资金。我们的消息来源认为这次筹资的估价为 50 亿美元；其他人认为这个数字在 35 亿到 40 亿美元之间[。](https://web.archive.org/web/20230301120343/http://www.bloomberg.com/news/articles/2016-03-02/messaging-startup-slack-seeks-4-billion-valuation-in-funding-round)

你可以想象为什么收购 Slack 的想法会在微软引发支持者和批评者。

一方面，微软在企业协作领域建立并收购了许多其他公司，包括 SharePoint、Yammer、Lync 和 Skype。换句话说，工具已经存在，微软可以建立自己的 Slack 竞争对手，而不必求助于收购初创公司和整合另一家公司。

另一方面，Slack 以微软和其他公司未能做到的方式在企业领域赢得了市场份额。Slack 是一个简单的入职流程，可以让员工之间快速沟通。它的一个关键特点是它与许多其他应用程序的集成方式。使用它的人可以用简短的代码从其他地方快速调用文件，与同事讨论或监控项目的进展，或者可能只是相互娱乐一下。

随着时间的推移，该平台已经扩展到更多基于集成的功能，例如从应用程序中订购[优步](https://web.archive.org/web/20230301120343/https://devblog.uber.com/uber-slack-a-weekend-a-story-of-open-apis/)或 [Lyft](https://web.archive.org/web/20230301120343/https://techcrunch.com/2015/11/03/slack-improves-slash-commands-so-you-can-call-a-lyft-and-more-from-inside-slack/) 。Slack 一直在提升其作为平台的信誉，它投资了一个开发者基金，并在去年[年底](https://web.archive.org/web/20230301120343/https://techcrunch.com/2015/12/15/trophy-emoji/)推出了一个新的应用目录。它于 2 月[日](https://web.archive.org/web/20230301120343/http://slackhq.com/post/139188191125/slack-is-turning-two-to-think-a-year-ago-today)日庆祝了自己的两周岁生日，现在拥有 230 万日活跃用户，67.5 万个付费座位。该公司表示，其免费增值业务模式的年经常性收入超过 6400 万美元。

Skype 本身一直在为其平台添加功能，以增强其消费者和企业使用的功能。它们包括在 iOS 和 Android 上进行群组视频通话，将 Skype for Business 增强到更多平台，甚至将 Skype 与 Slack 进行整合。

“在 Skype，我们致力于打破沟通障碍，让世界说话，”该公司在宣布 Slack 服务的预览版时指出。“这就是为什么我们让 Skype 在多个平台上可用，并继续探索更多方法来帮助您保持联系。”这也可能是为未来更动态的功能打下基础的一种方式。

陆和巴特菲尔德还有一个巧合的重叠之处:两人同时在雅虎工作。巴特菲尔德运营着他联合创立并出售给雅虎的 Flickr 图片服务，而陆是搜索和广告工程的执行副总裁。

微软和 Slack 都拒绝对此事发表评论。
