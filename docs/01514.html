Xpire 的应用程序让你所有的社交媒体变得“短暂”

原文：https://web.archive.org/web/https://techcrunch.com/2014/09/17/xpires-app-makes-all-your-social-media-ephemeral/

达拉斯计算机科学学院的学生杰西·斯陶费尔想要让社交媒体更加私密。在他哥哥的帮助和马克·库班的资金支持下，他最近发布了一款名为 Xpire 的 iOS 应用，让用户在脸书、Twitter 和现在的 Tumblr 上分享“自毁”帖子，并使用各种工具来更好地管理和缩小他们的数字足迹。

该应用于今年夏天早些时候首次推出，现在已经更新为包括一个功能，即让你根据包含的不良内容数量对自己的 Twitter 账户进行排名，以及帮助你管理和删除不需要的关注者的新工具。它还扩大了对脸书和推特之外的支持，还包括“自毁”Tumblr 博客帖子等等。

这实际上是用户在当今更流行的社交媒体平台上重获一些隐私的一种方式。它也是一个相当不错的 Twitter 管理工具，允许你查看和删除多达 3，200 条最近的推文，只需点击一个按钮就可以删除它们，甚至可以通过关键字显示旧推文。

然而，它更大的卖点是能够分享社交网络帖子，这些帖子会在你在应用程序中指定的时间范围内消失，无论是几分钟还是几天。

Stauffer 说，他最初是在看到他的同事在社交媒体上发布有问题的内容，并意识到这会对他们未来申请工作和其他方面产生影响后，才产生了创建这项服务的灵感。此外，他解释说，有时用户会发布长篇大论或“小而无意义”的内容，他们可能不想永远留在这里。时间敏感的内容也是如此，比如“我在商场。谁想出去玩？”也不需要很长的保质期，他说。

最初，他和他的兄弟正在开发一个类似的私人分享应用程序，名为 Bitzy ，它很像 Snapchat，让用户分享自毁的照片和文本。然后他通过电子邮件盲目地联系库班告诉他这个想法，马克真的回复了。经过一番讨论，库班建议，与其发展自己的社交网络，不如整合已有的社交网络。

当然，库班在这方面也有自己的投资要保护。他的最新公司cyber dust——他最近在 TechCrunch Disrupt SF 2014 上谈到的——也是一款私人消息应用。

库班给斯陶费尔的建议可能有风险；正如我们一次又一次看到的那样，在别人的平台上建立自己的公司是危险的，包括最近“脸书的 LinkedIn”branch out 的失败

虽然大型社交网络及其 API 与 Xpire 配合良好，但 Stauffer 可以通过应用内购买销售优质功能来获得可观的收入，正如他打算做的那样。斯托弗给了库班联合创始人的头衔，但从目前的情况来看，他更多的是投资者/顾问，而不是参与日常运营。(Stauffer 通过 CyberDust 联系他，实际上是。)

他说，他自己编写了 Xpire 应用程序的 100%的代码，他的兄弟帮助设计。库班的激进投资公司是这个三个月前申请的唯一投资者。

目前，Xpire 是 iTunes 上的免费下载。
