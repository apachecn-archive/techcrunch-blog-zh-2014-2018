# 克服监管障碍后，优步黑色重新在韩国上市

优步在朝鲜有了好消息，在经历了近一年的不确定性后，该公司在朝鲜重新开放了优步黑色私家车服务。

在发现优步违反当地交通规则后，韩国官员强迫这家美国公司将优步黑人仅限于老年人、残疾人、政府官员和外国人。但是，从今天开始，这些限制已经被取消，任何人都可以搭顺风车。

这要归功于新的法规，该法规还允许 Kakao，韩国最大的聊天应用卡考背后的公司，提供自己的私家车服务:名字并不那么有创意的 Kakao Black。 [Kakao 从 3 月开始提供拼车服务](https://web.archive.org/web/20230130234516/https://techcrunch.com/2015/03/30/kakaotaxi-launch/)，将用户与注册出租车联系起来，但现在它能够提供更多选择——即首尔的 200 名 Kakao 黑人司机组成的车队。

(今年早些时候我访问首尔时，优步的情况就是这样。)

今天的消息对优步来说是积极的一步，但是它的点对点优步 X 服务在三月份迫于政府压力关闭了[后仍然没有在韩国运作。这是它下一步努力解决的问题。](https://web.archive.org/web/20230130234516/https://techcrunch.com/2015/03/06/uberxit/)

“我们希望能够提供更多的产品，所以(在朝鲜重新开放优步 X)是我们希望与政府探讨的事情，”优步的一位发言人告诉 TechCrunch。

作为向更广泛的观众开放优步黑的一部分，优步表示，它正在与当局合作，以帮助在周五和周六这样的挑战时期提供更多的交通选择。这是该市正在运营的“出租车快乐区”的一部分。

首尔市政府对此表示赞同，称优步“对市民来说是一个真正的好处，特别是在一些时间和地点，如周五和周六晚上在江南站和其他繁忙的地方，当其他交通工具不那么容易获得时。”

优步的业务在首尔更受关注，但是优步首席执行官特拉维斯·卡兰尼克在韩国是否受欢迎还没有消息。当局去年威胁卡兰尼克要坐牢，但优步拒绝评论情况是否已经改变。
