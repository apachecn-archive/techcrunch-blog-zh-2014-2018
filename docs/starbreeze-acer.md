# Starbreeze 与宏碁合作开发更好的虚拟现实耳机 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/05/16/starbreeze-acer/>

# Starbreeze 与宏碁合作，寻求打造更好的虚拟现实耳机

在去年 E3 的首次亮相中，StarVR 无疑被更大的名字盖过了风头，但那些成功演示了高端耳机的人似乎留下了深刻的印象。

除此之外， [Starbreeze 的](https://web.archive.org/web/20230120235526/https://www.crunchbase.com/organization/starbreeze)硬件设备提供了比 Oculus 或 HTC 更宽的视野，与这些设备的 110 度成 210 度。这得益于两块四核高清面板，带来了令人印象深刻的 5，120 x 1，440 的组合分辨率(这里 Rift 和 Vive 的分辨率分别为 2，160 x 1，200)。

游戏工作室[本周](https://web.archive.org/web/20230120235526/http://www.businesswire.com/news/home/20160515005023/en/Starbreeze-Acer-Form-Joint-Venture-StarVR-Headset)宣布将[宏碁](https://web.archive.org/web/20230120235526/https://www.crunchbase.com/organization/acer)列为耳机的硬件合作伙伴。根据协议，两家公司将在设计、制造和推广该技术方面进行合作，将 StarVR 带入“专业和基于位置的娱乐市场”，听起来这款耳机似乎将专注于 Rift 和 Vive 客厅人口统计之外的市场。

考虑到高端规格和可能的匹配价格，这真的不是什么大惊喜。

宏碁首席执行官陈以桐补充说，该公司已经为 StarVR 体验准备好了自己的 PC 系列，他在一份声明中解释说，“我们正在将 R&D 的资源投入到 VR 生态系统的多个方面，以获得连贯和高质量的体验，而就在上个月，宏碁宣布了完全适用于 StarVR 的强大台式机和笔记本电脑。”
