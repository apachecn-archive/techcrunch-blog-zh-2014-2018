T-Mobile悄悄透露政府数据需求上升TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/08/27/t-mobile-quietly-reveals-uptick-in-government-data-demands/

T-Mobile透露，它从政府收到的数据需求数量有所上升。

这家手机巨头在8月14日悄悄发布了其2017年透明度报告，显示与去年相比，其回应的总体数据需求数量增加了12%。

报告称，该公司回应了219，377份传票，比2017年增长了11%。这些要求是由联邦机构发布的，不需要任何司法监督。该公司还回应了55，372份法院命令，上升了13%，以及27，203份认股权证，上升了19%。

但是窃听命令的数量——允许警察实时监听电话——比前一年下降了一半。

T-Mobile的一位发言人告诉TechCrunch，这些数字反映了“全面法律需求的典型增长”，并且这些增长“与过去几年一致”。

尽管结果显示了更多的客户数据请求，但透明报告没有说明有多少客户受到了影响。

截至第二季度财报，T-Mobile拥有7700万用户。

几家科技公司开始公布自谷歌2010年发布首份报告以来，它们收到了多少份政府索取客户数据的请求。但直到2013年爱德华·斯诺登(Edward Snowden)的爆料揭露了美国国家安全局(National Security Agency)的大规模监控后，科技公司和电信公司才开始定期发布透明度报告，被视为是为了反驳有关公司帮助政府进行间谍活动的破坏性指控。

两年后的2015年，T-Mobile成为最后一家发布透明度报告的主要手机运营商。

该公司还表示，它回应了执法部门对客户历史手机站点数据的64，266次请求。这些数据成为今年早些时候美国诉卡彭特案的焦点，在该案中，最高法院裁定执法部门必须获得历史手机和位置数据的授权。这一数字预计将在2018年报告年度下降，因为获得法院签署的逮捕令的新门槛更高了。

T-Mobile还表示，它收到了46，395份跟踪客户实时位置的请求，以及4，855份对信号塔转储的授权和命令，警方可以使用这些授权和命令来获取特定时间段内连接到信号塔的所有附近设备的信息。

但2017年收到的国家安全请求数量有所下降。

全年，联邦特工用来秘密获取通话记录的国家安全信函数量和秘密外国情报监视法院批准的命令数量都低于1000项。

科技公司和电信公司在如何秘密报告需要客户数据的机密订单数量方面受到高度限制，只能报告他们收到的请求范围。

自2015年《自由法案》签署成为法律以来，司法部开始允许公司在更窄的范围内报告。
