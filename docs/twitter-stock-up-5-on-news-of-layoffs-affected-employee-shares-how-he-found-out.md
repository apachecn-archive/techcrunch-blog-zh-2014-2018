# 裁员消息传出，Twitter 股价上涨 5%，员工持股受到影响他是如何发现的

今天，Twitter [宣布解雇 8%的员工](https://web.archive.org/web/20221209191518/https://beta.techcrunch.com/2015/10/13/twitter-is-laying-off-8-percent-of-the-company/)，华尔街对此消息做出回应。其股票上涨超过 5%，表明投资者也认为这家社交网络公司是时候瘦身了。

![Screen Shot 2015-10-13 at 8.28.18 AM](img/63bb2f17e42bd5115465370711f2bd8d.png)

受影响的前鸥群成员开始闭门分享他们的故事，但这位在家工作的先生公开分享了他的经历。他显然是被这个消息惊醒的，无法查看自己的公司邮件，但在此之前，他收到了一个怪异的 GitHub 通知:

虽然我们不知道受影响的各方提供了多少遣散费，但据说该公司提供了极其软着陆。当然，这并不能掩盖一个事实，即 300 多人早上醒来时得知他们已经失业了。大多数时候，在公司的土地上，离职协议的一部分包括不以任何方式谈论条款或贬低公司。

前 Twitter 员工已经开始让那些受影响的人知道他们有自己的位置:

https://twitter.com/rk/status/653945157960757248

裁员的心理可以以几种不同的方式推动 Twitter。一方面，公司的其他人可以将此视为稳定的迹象，因为他们出于某种原因留在各自的团队而感到欣慰。这可能会成为 Twitter 多年来最好的作品。

另一方面，一些人可能会将此视为一个信号，如果产品采用率不能很快提高，8%可能只是冰山一角。这可能会导致人才流失，就像我们听说的那样[已经发生了](https://web.archive.org/web/20221209191518/https://beta.techcrunch.com/2015/09/11/the-nest-is-being-robbed/)。当然，现实可能介于两者之间。
