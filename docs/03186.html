微软正在开发将安卓手机转换成Windows 10的软件

原文：https://web.archive.org/web/http://techcrunch.com/2015/03/17/microsoft-android-rom/?ncid=rss

微软今天发布了一条有趣的消息，证实Windows 10将于今年夏天上市。

在其公告接近尾声时，这家总部位于雷德蒙德的公司无意中透露，它正在用小米旗舰智能手机Mi 4的“高级用户”测试Windows 10。小米强调，这一举措不是合作关系，而只是协助试验，这是一个有趣的举措，因为它再次显示了微软新的“平台不可知”的方法。

微软和小米都没有提供正在测试的Windows 10软件的具体细节，但TechCrunch从消息人士那里了解到，它实际上超越了Android，将小米手机变成了一款具有微软服务的Windows 10设备。(该公司希望这会让安卓用户眼花缭乱，转而使用安卓系统。)

也就是说，该软件不提供双引导选项，而微软过去曾在印度推出过。这是一个基于Windows的rom，它的操作很像Cyanogen的软件——一家微软被错误地与投资于T7的公司联系在一起——以及由中国的腾讯和百度等公司开发的其他定制ROM。

因此，ROM的设计超越了微软的Android应用程序，并在Android手机上提供了类似原生Windows的体验。

这是一个非常强大的概念，看到微软在中国测试它并不奇怪，因为中国的消费者更倾向于安装rom。中国有更强的定制精神，尤其是对安卓用户而言，因为第三方应用商店是常态，而谷歌Play在中国受到严格限制。

当然，微软有一个宏大的计划，及时让这款软件在更多的设备上可用。该公司告诉The Next Web 将很快宣布全面可用(更新:下面是完整声明)，但它是从小米开始的——几乎可以肯定，因为它有一个强大的用户社区，他们对该公司的每周软件更新提供反馈。小米的接受观众非常适合这样的试点。

尽管这可能是一个好主意，但在中国采用Android ROM并不容易。问问百度就知道了，尽管百度作为“中国的谷歌”拥有巨大的影响力和分销潜力，但由于兴趣不高，最近关闭了其安卓软件项目。

至于小米，人们往往很快就会认为它与谷歌存在分歧，因为小米在中国的MIUI平台上不提供谷歌的服务。小米强调其参与Windows 10并不是合作关系——再加上雨果·巴拉最近解释其不会向中国以外的用户推广其服务套件——这表明这不是战术性的；小米只是给了它的用户另一个ROM来修补。

在美国，人们可能会以一种愤世嫉俗的方式看待这个消息——在美国，对用户有利的事情往往很方便，因为这对企业也有利——但小米确实已经培养了一个规模可观、值得信赖的社区，所以我们倾向于相信它在这种情况下的说辞。
