Checkr 正在为其背景调查 API 筹集 3000 多万美元，Y Combinator 投资 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2015/10/13/checkr-series-b/?ncid=rss

随着优步和 Instacart 等按需业务的快速增长，服务于更广泛生态系统的 B2B 初创公司也在快速增长。在最新的发展中，TechCrunch 从几个消息来源那里获悉并证实了Checkr——通过 API 为优步、Instacart 和 Handy 等公司以及 Zenefits 和 Weebly 等其他快速增长的初创公司进行背景调查和审查潜在雇员——正在筹集至少 3000 万美元的 B 轮融资，估值超过 2.5 亿美元。

这次加薪同样值得注意的是谁是幕后黑手。TechCrunch 了解到，这一轮融资包括 Y Combinator，这是一家知名的创业加速器，Checkr 是 2014 年夏季班的一部分。

YC 实际上是这轮投资的领军者——它的第一笔领投来自其持续性基金。

YC 尚未正式宣布持续性基金的规模，但有传言称其规模在10 亿美元到数十亿美元之间。YC 也没有透露它将如何运作的太多细节，尽管在 7 月，YC 总裁 Sam Altman 指出YC 将按比例投资价值低于 2.5 亿美元的初创公司(省略了 YC 在初创公司价值超过 2.5 亿美元的情况下会怎么做的细节)。

预计将参与此轮投资的其他人包括该公司的两位早期投资者大卫·斯佩克特(David Spector)和埃拉德·吉尔(Elad Gil)。随着这轮融资的结束，可能会有更多的风投加入这个名单。

在最近一轮之前，Checkr 已经从一长串天使投资者以及谷歌风投和 Accel 等大型风投那里筹集了超过 900 万美元的资金，其中 Accel 领导了该公司的上一轮融资。

Checkr 由 Daniel Yanisse 和 Jonathan Perichon 共同创立，它填补了初创公司世界的一个漏洞，特别是那些快速招聘合同工的初创公司。

简而言之，该公司提供了一种更快、更便宜、更简单的方式来审查潜在的员工，包括社会安全号码验证、地址历史、国家和县级犯罪记录以及驾驶历史等数据。企业可以通过 Checkr 的在线表格进行检查；或者，通过 API，它可以将 Checkr check 集成到招聘系统和其他入职软件中，如 Workable 和 Zenefits。

Checkr 的业务按每人、每份报告定价 25 美元、35 美元，并根据具体要求定制价格，这些要求可能包括对机动车辆的更深入检查、国际搜索和药物筛选。

据我们所知，Checkr 吸引投资者有几个原因。首先也是最重要的事实是，Checkr 的业务在当前的创业市场上受到了很大的关注和增长，但也有很大的潜力从更广泛的业务领域赢得更多交易，这些业务需要对临时工和永久工进行背景调查。

“背景调查市场不会消失，它将存在很长一段时间。他们只是让它变得更容易、更便宜，”一位消息人士说。

这并不是说 Checkr 是唯一一个试图解决这个问题的公司。竞争对手包括美国的古德伊尔和伦敦的 T2。也许是为了帮助避开竞争对手，并跟上它的增长，我们知道 Checkr 的新资金将用于业务发展和雇用更多的人才。

Checkr、Y Combinator 和其他投资者拒绝对此事发表评论。
