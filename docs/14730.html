阿里巴巴牵头对开源数据库初创公司 Maria db TechCrunch 投资 2700 万美元

原文：https://web.archive.org/web/https://techcrunch.com/2017/09/29/alibaba-mariadb/

阿里巴巴在 2017 年一直在推动其云计算业务，现在它正准备对该领域的一家西方初创公司进行首笔重大投资。

据了解谈判情况的消息人士透露，中国电子商务巨头已同意领导€对 MariaDB 的 2290 万欧元(2700 万美元)投资，Maria db 是欧洲公司，开发网络上最受欢迎的开源数据库服务器之一。交易尚未完成，但 MariaDB 的股东本周批准后，交易即将完成。

阿里巴巴和 MariaDB 都没有回应置评请求。

TechCrunch 了解到，阿里巴巴出资约 2000 万€，其余资金来自现有的支持者。该交易对 MariaDB 的估值约为 3 亿€(3.54 亿美元)，阿里巴巴的首席工程师 T2·冯玉(T3)将加入这家初创公司的董事会。

这意味着它在 5 月份从欧洲投资银行为€筹集了 2500 万英镑(当时价值 2700 万美元)时获得的 2 亿至 2.5 亿美元的估值大幅升值。我们的消息来源表明，阿里巴巴做生意的意愿基本上使 MariaDB 能够选择自己的估值。

MariaDB 最为人所知的是运行数据库管理系统 MySQL 最受欢迎的替代品。两者都是开源产品，但是有些人对 MySQL 持谨慎态度，因为它是由甲骨文公司拥有的，甲骨文公司是一家大公司，收购了太阳微系统公司。

阿里巴巴的云计算业务是其增长最快的部门之一，过去一年一直保持三位数的收入增长。

我们在今年早些时候写道，它正在努力与业界最大的玩家竞争，如 AWS、微软 Azure 和谷歌云。虽然还有一段路要走，但迹象是有希望的。云业务今年的年化收入达到 10 亿美元，最近一个季度的客户数量超过了 100 万。该部门很可能在未来几个季度达到盈亏平衡，尽管它在阿里巴巴总收入中的占比仍不到 5%。

云只是阿里巴巴重点发展的一个领域，它希望创造新的收入来源，以减少对其核心中国商务业务的依赖，尽管这仍然非常有利可图。

该公司已经在其他领域进行了大量投资——支持印度的独角兽公司 Paytm ，以及东南亚的 Lazada 和Tokopedia——那么为什么不看看更广泛的云/基础设施行业的交易来推进其战略呢？

在投资了像云存储提供商 Qiniu 和大数据公司 Dt Dream 、这样的中国公司之后，加入了微软的开源社区，甚至在中国直接与 MariaDB 合作，这将是阿里巴巴在海外地盘上最引人注目的云交易。

MariaDB 原名 SkySQL，由于其在行业中的存在而对阿里巴巴具有吸引力。它在瑞典和美国设有办事处，并声称其数据库拥有约 1200 万全球用户，其中一些较大的公司包括 Booking.com、惠普、维珍移动和维基百科。其解决方案用于私有、公共和混合云部署，并且是 Red Hat、Ubuntu 和 SUSE 等许多 Linux 发行版的默认配置，这进一步增加了 6000 万用户。

SkySQL 最初成立于 2009 年，2013 年与 Monty Program Ab 合并。2008 年，Michael 'Monty' Widenius 将他的前公司 MySQL 以 10 亿美元的价格出售给 Sun Microsystems(现为甲骨文公司所有),随后创建了 Monty Program。由于担心 Oracle 可能会使用 MySQL，MariaDB 后来将 MySQL 分叉。
