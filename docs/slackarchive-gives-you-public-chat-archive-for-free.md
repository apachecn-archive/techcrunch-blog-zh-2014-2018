# SlackArchive 免费为您提供公共聊天存档

有很多原因可以让你选择 Slack 作为公共或半公共项目的消息传递平台。它可以是一个开源项目，一个本地社区项目或者仅仅是一个基于兴趣的项目。但是有可能你的社区越来越大，你想获得无限的搜索聊天记录。

然而，Slack 的免费层仅存档最后 10，000 条消息。但是每用户每年 80 美元，Slack 可能成为一种昂贵的软件即服务。认识一下 [SlackArchive](https://web.archive.org/web/20221007140732/http://slackarchive.io/) ，这是一项免费服务，让你免费存档你所有的信息。

本质上，SlackArchive 是一个添加到你的 Slack 中的机器人。这个名为 ArchiveBot 的机器人将记录这个频道中的对话，用于聊天存档。

然后，您可以在公司网站上访问您的档案。像在 Slack 上一样，你可以在多个频道上搜索，在很短的时间内找到信息。您也可以为特定邮件创建永久链接。

最后，因为 SlackArchive 也是一个 bot，所以可以直接在 Slack 内查询搜索历史。几个 bot 命令就可以得到结果，而不必切换到另一个窗口。

那么你为什么要使用 SlackArchive 呢？许多公共项目，比如开源项目，都依赖于留言板、Google Groups 和 IRC。虽然留言板和 Google Groups 提供了一个很好的搜索界面，但它们不能像 Slack 那样提供直接的、即时消息式的体验。

这也是为什么很多项目想用 Slack 的原因。许多人已经使用 Slack 与他们的同事交谈，切换到 Slack 进行其他项目似乎是一个自然的举动。

SlackArchive 不提供私人聊天记录，以避免与 Slack 的[高级计划](https://web.archive.org/web/20221007140732/https://slack.com/pricing)直接竞争。这项服务是专门为公共项目定制的。我觉得它有潜力一劳永逸地取代旧论坛。
