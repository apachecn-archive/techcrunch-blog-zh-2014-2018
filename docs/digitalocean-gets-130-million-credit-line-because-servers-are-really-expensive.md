# DigitalOcean 获得 1.3 亿美元的信用额度，因为服务器非常昂贵

建立一个像数字海洋这样便宜的云托管平台需要一大笔钱(T2)。该公司已经为[筹集了超过 1.2 亿美元](https://web.archive.org/web/20230405201930/https://techcrunch.com/2015/07/08/digitalocean-raises-83m-series-b-round-led-by-access-industries/)，并且还不时地获得信贷以推动其增长。

尽管名字容易让人误解，但云是由服务器组成的。购买服务器和租用数据中心空间一点也不便宜。

随着 700，000 名用户始终在 12 个数据中心启动虚拟专用服务器，该公司必须管理大量机架柜。

数字海洋用户总共启动了 1300 万台云服务器。当然，自那以后，许多服务器都被关闭了，但这让我们对数字海洋的规模有了一个很好的了解。

这个新的信用额度意味着两件事。首先，DigitalOcean 仍在大力投资基础设施。随着新服务器取代现有服务器，现有用户可能会获得性能升级。或者，数字海洋可能会扩大其产品供应。

第二，虽然 DigitalOcean 是小开发者的一个很好的平台，但与云托管巨头，如亚马逊网络服务和微软 Azure，争夺更大的客户是一场永无止境的艰苦战斗。如果你想逗这些巨头开心，你需要现金。

但很高兴看到这家初创公司设法找到了愿意在今天的创业环境中提供大量贷款的银行。如果他们不打算在某个时候拿回他们的钱，他们是不会给钱的。
