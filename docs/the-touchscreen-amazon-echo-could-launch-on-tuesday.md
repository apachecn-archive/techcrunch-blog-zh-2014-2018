# 触摸屏亚马逊 Echo 可能于周二推出

亚马逊最早可能于明天推出带触摸屏的 Echo。这个消息来自《华尔街日报》援引知情人士的话。新型号将配备 7 英寸触摸屏，显示口头问题的答案，并具有基于互联网的电话功能。

[上周，几个网站发布了一款设备的图片](https://web.archive.org/web/20221207113435/https://beta.techcrunch.com/2017/05/05/this-could-be-amazons-echo-with-a-built-in-display/)，他们声称这是即将推出的 Echo 触摸屏扬声器。反应不一。一些人称赞触摸屏增加的功能，而另一些人抨击这款设备，说它很丑，看起来过时了。不管是哪种情况，这样一款具备上述功能的设备都会让亚马逊超越其主要竞争对手谷歌 Home。

回声是一个奇妙的装置。我家里有四个。但这款设备缺少一些功能，而且屏幕不足有时会成为一个限制因素。例如，很难通过语音浏览音乐目录。添加触摸屏可能对一些用户有所帮助。

《华尔街日报》称，下一代 Echo 将能够进行电话和视频通话。由于在中心位置放置回音很常见——就像有人用电话一样——增加这些功能可以帮助吸引一些潜在买家购买该设备。

然而，无头设备上没有屏幕也是一个挑战。应用开发者现在必须决定开发一个可视化用户界面是否有意义。正如布莱恩[昨天](https://web.archive.org/web/20221207113435/https://beta.techcrunch.com/2017/05/06/amazon-echo-world-domination/)指出的，第三方一直是亚马逊战略的关键，这一新设备很可能也是如此。

凭借 Echo，亚马逊在市场上占据了主导地位。eMarketer 进行的一项研究表明，Echo 品牌扬声器的制造商今年在美国拥有 70.6%的语音扬声器用户。这远远领先于 Google Home 的 23.8%和其他较小的公司，如联想、LG、哈曼卡顿和美泰，它们加起来只占用户的 5.6%。亚马逊现在才刚刚站稳脚跟。

亚马逊[刚刚发布了 Echo Look](https://web.archive.org/web/20221207113435/https://beta.techcrunch.com/2017/04/26/amazons-new-echo-look-has-a-built-in-camera-for-style-selfies/) ，这是一款内置网络摄像头的语音助手，旨在帮助人们挑选服装。在加入 Echo 家族后，这款外观在市场上推出了四款 Echo，如果《华尔街日报》的报道属实，第五款也即将推出。
