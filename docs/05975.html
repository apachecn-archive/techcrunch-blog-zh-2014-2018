我一直想赚很多钱，让人们关注我，做很多令人兴奋的事情。我只是不知道怎么做。

我的许多朋友都是自己公司的创始人，他们告诉我他们小时候是如何展现企业家精神的——他们卖背包里的糖果，在夏天做景观美化生意，等等。他们创造了价值，并且很早就知道了努力工作的好处。

但那不是我。我在曼哈顿海滩中学的电脑教室用 Applesoft BASIC 创作了一个赛马模拟游戏，并运行了一个小型赌博操作。我是说，谁能怪我呢？在课间和午餐时间，老师们完全不去管那堂课，好像他们不明白怎么会有孩子在玩那些大型计算打字机时惹上麻烦。

然而，赚几美元去买源源不断的雅达利游戏，以及学习如何从大一点的八年级学生那里收集，并没有教会我任何关于建立一家科技公司的事情。当我决定搬到旧金山时，我完全没有头绪。

同样糟糕的是我对时间的感觉。当我进入就业市场时，互联网泡沫已经破裂，导致许多聪明人失业，一波风险投资公司破产。很多人离开湾区，回到他们来的地方，但我太笨了，不知道人们做出的逻辑决定。

我工作的折扣经纪公司给了我们两周的培训，教我们如何在圣地亚哥的一个呼叫中心进行网上股票交易。就在那时，我有了一个绝妙的主意(只是后来我改了几次，这个主意才变得绝妙)。

负责监督我们培训的呼叫中心经理年近三十，每天脸上都挂着得意的笑容。“我正在教你的这个呼叫中心系统的成本是每人 3 万美元，”他提到了好几次。他傻笑了很多，我鄙视他。

我突然想到，如果我能制造更便宜的呼叫中心软件，我就能创建自己的软件公司，并对假笑者进行报复。

我一定是走运了，因为此后不久，我的大学室友，我给她起了个绰号“The Fro”(我给每个我深爱的人起绰号)，打电话告诉我(吹牛)，他工作的呼叫中心软件初创公司被思科收购了。

他承认，作为一名后期员工，他并没有取得太多成就，但他对未来有一种品味，并鼓励我飞往波士顿，讨论创建我们自己的创业公司。“毕竟，”他说，“你很擅长卖狗屎。”

我不确定这是不是赞美，但我买了机票。

在讨论了一个周末的梦想和人类情感后，我们彼此都很兴奋，决定下周一辞职。我提前两周通知了。

要相信，即使你做了一些愚蠢的事情，比如毫无头绪地辞职，你也会想出办法的。

然后奇怪的事情发生了。那个星期一，Fro 一整天都不接我的电话。 周三 他终于打电话给我，不好意思地说:“布鲁。你知道吗，兄弟？我认为现在辞职不是个好主意。说真的。真对不起。”

我的银行账户里有 5000 美元，我真的想过花其中的 1000 美元飞回波士顿打他的脸。我反而挂了电话，一拳打在墙上。也许有几次。

然后我给他回了电话，尽可能平静地问他，他是否知道他新收购的初创公司里还有谁想和我一起创建一家软件 公司。我提醒他我很擅长卖狗屎。他把我介绍给“图特”，原来，他正和我一起去太浩湖滑雪。

当我们在天堂滑雪场相遇时，我发现了一个明显聪明的家伙，他不喜欢公司生活，对在加利福尼亚州可能不完全合法的活动有着无拘无束的感情。但是，嘿，我们都有问题。

他向我解释说，他和 Fro 曾在一家初创公司工作过，这家公司为呼叫中心开发了软件，这是一个古老的行业，他们通过在公司的名字前加一个. com 进行了彻底改革。

因为我现在是一个企业家(失业)，我告诉他我们应该重新创建那个类的软件，他同意了。值得称赞的是，也让我有点吃惊的是，他给老板发了一封电子邮件，并在当晚辞职。几周后，他卖掉了波士顿的房子，在海湾地区买了一套新房子。

但使他成为如此冒险和杰出的程序员的同时，也使他有时成为一个超级混蛋。一无所知 一无所知关于软件公司，我的很多问题都遇到了他自己的一个问题，他最喜欢的问题是，“你是他妈的白痴吗？”

有那么多令人沮丧的夜晚，我常常会漫不经心地一遍又一遍地看我最喜欢的电影《肖申克的救赎》。也许看到安迪·杜弗兰忍受这么多狗屎让我对自己的处境感觉更好。

显然，我需要帮助(治疗)，所以我问我在折扣经纪公司的前老板，他是否知道任何天使投资者。

谢天谢地，他把我介绍给了他最喜欢的日内交易客户，并向他解释说我是一个有前途的企业家，图特是一个了不起的工程师，他绝对是。我尽可能热情地向投资者解释说，我将建立一个成功的软件 公司或者不惜一切代价，我会自我保护，所以…

当日交易者指着他办公室墙上一架小型喷气式飞机的照片说:“我想让你帮我买下它。”然后他当场给我们开了一张 15 万美元的支票，我们正式得到了资助。

然后我们雇了“机器”(2 号工程师)，我和他一起搬进了旧金山双峰社区的一所公寓，主要是为了让我可以日夜骚扰他，让他编程。当我的两个工程师为我们的呼叫中心软件开发测试版时，我开始打电话要钱，并给呼叫中心的老板打电话，推销并承诺更便宜的东西。

被拒绝了那么多次，我想拒绝自己。

在打了 2000 多个陌生电话后，我终于在另一条线上听到了一个愿意接受的声音。乔在犹他州的普罗沃拥有一个小而成功的呼叫中心。他说我虚张声势，并说他知道我们还没有任何坚实的东西，但他相信我们可以把它建成——但它必须是我向他报价的一半。

这是我早期学到的关于企业家精神的一课，或者可能是美国本身的一课。在多次被拒绝后，我想拒绝自己，但后来找到了一个人，他会寄给我一张 4 万美元的支票，即使是在经济衰退期间，因为他们自己就是敢于冒险的企业主。

这让我意识到美国对企业家来说是一个神奇的地方。即使是无知的人。

有了这 4 万美元的支票，我们从折扣经纪公司雇佣了我以前的老板，从已被思科收购的呼叫中心初创公司雇佣了一名销售代表，我们五个人去了核桃溪的一间小办公室工作。

我们开始运送戴尔机器，里面装有 Dialogic 电话板，装载了我们的呼叫中心软件。这并不性感，但足以支撑我们雇佣更多的人并开始成长。

后来有一天，出于某种奇怪的原因，我又顿悟了。

与其构建软件加载到便宜的服务器上像其他人一样出售，为什么不构建一个我们自己托管的“巨型服务器”并让人们通过网络租用功能？

我的前老板告诉我，“不，那是愚蠢的，那是杀鸡取卵。”

工程师告诉我，“不，这将是一个很难建造的东西。”他们给我发来了各种知名分析师的研究报告，显示“ASP”(被“SaaS”取代之前的流行词)终究不是一个有利可图的商业模式。

我说，“伙计们，还有什么选择？你想在下半辈子里做低成本的领导者，去推销这些破玩意儿吗？”我指着我们小办公室里堆积的越来越多的戴尔盒子。

与他们激烈争论了一个月，我意识到按需软件服务是无利可图的，因为每个人都试图采用传统的客户端-服务器(T2)软件(T3)并自己托管它。

很少有人考虑多租户和 IP 语音。

他们没有想过要构建真正多租户的东西，也就是说要构建软件，这种软件是专为处理多个客户和账户的服务而设计的。如果我们能够成功，这将是我们的一个重要优势，因为要购买的硬件要少得多，管理客户帐户也容易得多。

但是还有另一个问题。与我们的 15 万美元相比，这个领域的少数其他创业公司总共筹集了超过 1 亿美元百万美元，他们正在领先。就像，遥遥领先。

后来有一天，机器给了我一个 IP 语音电话盒来玩。我们都喜欢小玩意，我们觉得能够通过互联网免费打长途电话太酷了。

这时，我突然想到了 Eureka 2.0 我们的竞争对手拥有单租户解决方案，需要大量的硬件和运营支持来为大规模客户提供服务，但他们也不得不将客户的长途电话费翻倍。

因为一个 SaaS 呼叫中心产品的每个呼叫中心代理必须有三个正在进行的电话连接(相比之下，传统的本地模式只有两个)，这消耗了客户迁移到 SaaS 后的大部分成本节约。基于 IP 语音的服务将给我们带来另一个巨大的市场优势，因为客户实际上可以降低他们的电话账单。

很少有人考虑多租户和 IP 语音，但我终于开始购买线索。我很幸运，机器 Tooter 和我的前老板最终以他们巨大的超能力聚集在我的视野周围。直到今天，我都很感激他们每一个人。

对我来说，那“决定命运的一天”是在一家西夫韦杂货店(在那里我经常担心我的借记卡被退回)，当时机器在我的黑莓手机上打电话给我。

“伙计，伙计，伙计。猜猜我是从哪里打来的？”机器问。

“呃，我们的公寓？”我在收银台排队的时候开始看《人物》杂志的封面。布列塔尼·斯皮尔斯在和一个新的人约会。

“是啊，”机器兴奋地说，“但这是我第一次用我们的 IP 电话打电话。听起来真他妈的不错，对吧？”

我跪下，真的被击倒了。我知道我们仍然面临着如此多的障碍，但在那一刻，也是有史以来第一次，我感到无比自信，我们会战胜它。竞争者，怀疑者，所有人。我们要摧毁这个行业，然后按照我们自己的形象重塑它。

这就是我们曲棍球棒的曲线是如何开始的。我们最终找到了一种方法来交付一款性能提高 10 倍、价格降低 5 倍的产品。

在 Safeway Express 收银台排队时，我第一次接到了 IP 语音电话，这让我想起了《肖申克的救赎》中的一个场景，主角在最终挣脱之前，爬过了数英里长的污水管道。

此后不久，我们正式推出(建立了一个新网站)“虚拟呼叫中心”，我们的业务以越来越快的速度增长。我们的竞争对手突然对我们的价格如此之低感到震惊，他们中的大多数人都没有回复我之前问他们是否可以在城里给他们买午餐的电子邮件。

我们有一个很棒的办公空间，除了我父亲，每个人都印象深刻。

具有讽刺意味的是，我们最终成为了低成本的领导者，这是我们可以做到的，因为我们的内在成本低得多。我们从信任我们的大公司那里获得了大量的风险投资，最后，我们有了一个很棒的办公空间，除了我的父亲——他一直问我为什么还没有成为医生。

第一年我们的收入是 0 美元，第二年是 90 万美元，第三年是 300 万美元。在我离开之前，我带领公司实现了每年 1000 万美元的经常性收入。事实是，公司已经成为它自己的实体，不再需要我了。

今天，我命名为 Five9 的公司在纳斯达克股票市场上市，并雇佣了数百名加州湾区的员工，其首次公开募股使公司在上市首日的估值达到了 3 . 5 亿美元，1200 万美元。每当我打电话给一家使用我的软件的公司的呼叫中心时，我都会情不自禁地微笑。直到我等的时间超过两分钟。

我不确定我是否对其他企业家有什么明智的建议，但从这次经历中，我确实学到了一个重要的真理。也就是说，你不必是常春藤联盟的毕业生(我不是)或者有很多钱(我们不是)。你只需要相信自己，反对所有合理的逻辑，尽管这听起来很老套。

要相信，即使你做了一些愚蠢的事情，比如毫无头绪地辞职，你也会想出办法的。相信在忍受了多年监狱般的艰苦创业生活后，你会从数英里长的污水管道中走出来，感受雨水冲刷你脸上的粪便。

你会带着满满一袋子钱消失在墨西哥的海滩上，和你的同谋见面，喝点玛格丽特酒。
