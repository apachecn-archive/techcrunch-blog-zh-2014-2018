苹果不认为政府入侵是 iPhone 的主要安全威胁，然而 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/04/15/apple-doesnt-consider-government-intrusion-a-primary-iphone-security-threat-yet/?ncid=rss

尽管苹果公司与美国联邦调查局就解锁圣贝纳迪诺枪击案枪手之一的 iPhone 进行了高调的唇枪舌剑，但安全工程师在一次记者会上反驳了苹果公司是政府对手的说法。

苹果高级工程师认为，在设计 iPhone 安全时，政府入侵不是他们的主要威胁模式，并表示他们更喜欢专注于抵御黑客。

工程师们还表示，苹果对联邦调查局的抵制并非出于阻碍恐怖主义调查的动机，而是为了捍卫其保护用户免受非政府威胁的能力。

苹果最近重组了其内部安全团队，该团队管理运输产品的安全方面，对苹果自己的设备进行威胁测试，并充当一种过滤系统，将安全置于其工作的核心。鉴于苹果首席执行官蒂姆·库克(Tim Cook)关于安全是苹果战略关键的强烈声明，这并不令人震惊。

在加州圣贝纳迪诺内陆地区中心发生枪击案、造成 14 人死亡后，苹果 iPhone 的安全功能受到了高度审查。美国联邦调查局试图迫使苹果公司设计定制软件，帮助解锁属于 Syed Farook 的 iPhone，但后来在第三方提供另一种进入手机的方法后放弃了案件。从司法部到曼哈顿地区检察官办公室的执法官员都认为，苹果在加密客户数据方面走得太远了，将调查人员与犯罪入侵者一起关在了门外。

但苹果工程师对科技巨头的安全功能使犯罪分子能够逃避执法的理论提出质疑，称数据安全对整个社会的安全至关重要。苹果高管还指出，在数字时代，执法官员可以利用许多其他调查途径——从手机信号塔、社交媒体帖子和信息附带的交易元数据中收集的位置数据。工程师们的言论呼应了苹果公司为回应联邦调查局的要求而发布的 Q & A，该公司在其中呼吁美国政府成为网络安全的国际领导者。

苹果在其问答中表示，政府应该“组建一个委员会或其他专家小组，研究情报、技术和公民自由，以讨论执法、国家安全、隐私和个人自由的影响。”

工程师们今天回顾了该公司的安全白皮书中强调的功能，向记者解释了苹果如何保护其客户的数据，并强调苹果严格的设计理念并不局限于 iPhone 光滑的玫瑰金外观——它也融入了设备的安全性。

特别是，苹果强调了其从芯片层面为 iPhone 构建安全性的独特能力——尽管其他智能手机制造商有时会外包芯片生产，但苹果喜欢将一切都留在内部。其最新的手机配备了 Secure Enclave，这是手机硬件的一部分，作为芯片的一部分，管理用于加密设备的密钥。

苹果还强调了消费者在保护 iPhone 方面的作用，强调触控 ID 和 iCloud 的双重认证等功能是用户保护自己的设备和数据免受窥探的方法。正如苹果之前强调的，在引入触控 ID 之前，苹果发现只有 49%的客户使用密码保护他们的手机。但苹果工程师表示，在引入触控 id 后，密码使用率跃升至 89%(用户需要设置密码才能使用触控 ID 功能)。

尽管苹果从一开始就致力于在 iPhone 中内置加密功能——它在 iMessage 的最早版本中引入了端到端加密，并通过 Secure Enclave 加强了设备加密——但 iPhone 的安全功能只是在最近几年才开始在苹果的营销中发挥重要作用。

消费者对加密和安全的兴趣在后斯诺登时代有所上升，并在圣贝纳迪诺袭击事件后飙升，这影响了苹果公司更公开地谈论其安全的设计和实施。这也意味着，对于苹果来说，现在比以往任何时候都更有必要确保媒体和公众在涉及其安全流程的技术和政策细节时消息灵通。

当下一个圣贝纳迪诺案件发生时，苹果需要确保公众理解“不仅仅是一部 iPhone ”场景的含义。

值得考虑的一件事是，包括苹果在内的任何科技公司，不将政府入侵视为其威胁模式的一部分还能持续多久。如上所述，苹果的工程师们目前没有这样做，但是任何一家管理着大量用户信息(或者以设备的形式制造这些商店)的科技公司都必须至少考虑“govtOS”向量。

相关消息，苹果公司今天宣布，将在纽约刑事案件中反对解锁 iPhone。

对抗政府解锁手机的要求将苹果置于一个艰难的境地——如果调查人员继续要求苹果修改其 iOS 以允许解密，该公司最终将不得不决定是否要进一步提高其安全性，并使自己能够拒绝政府的所有数据请求。

这不是苹果想要做的事情——工程师们表示，他们不想被视为政府的敌人，在 iPhone 和 iCloud 等服务中内置更严格的加密可能也意味着放弃一些对苹果品牌至关重要的设计和简单性——但可能很快就会将政府纳入苹果的威胁模型，与黑客并列。

由于苹果引领了智能手机创新行业，它也可能引领安全创新。在圣贝纳迪诺案件中，硅谷广泛支持苹果反对为美国联邦调查局(FBI)建立一个特殊的操作系统，被苹果称为 govtOS。随着苹果继续提高用户的安全性，其他科技公司很可能会效仿苹果。正如工程师今天所说，数据安全是一个不断发展的目标。
