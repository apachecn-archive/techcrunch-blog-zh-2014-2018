虽然控制移动支付的战争在过去几年中被广泛报道，但似乎大多数观察家对这场战术战争的理解较少。

这是一个冲突的故事，一个闭门讨价还价的故事，一个大卫对抗歌利亚的故事，一个巨大的财富危在旦夕的故事。许多人观看和等待 PayPal 和谷歌与网络寡头(Visa，Mastercard，Amex)的斗争，却对一个曲折的结局感到惊讶。

PayPal 尝试离线

对于 PayPal 来说，智能手机革命提供了它一直等待的线下销售的切入点。以前仅限于通过网络浏览器使用的 PayPal 钱包，有可能进入每个消费者的口袋(就像手机一样)。2010 年，PayPal 抓住了这个机会，开始了一系列战略举措，将他们的在线支付网络扩展到 3 万亿美元的线下支付世界。

他们与一些早期的大型线下商户(如家得宝)签约，并开始与几家销售点供应商合作，以集成到他们的嵌入式商户卡终端中。最重要的是，他们开始与主要的支付处理商(像 First Data 这样的公司，他们承担着管理网络支付流程的重任)进行谈判，以使 PayPal 成为美国各地安装的 1000 多万台商户卡终端的一种支付形式

然而，这一雄心勃勃的战略遇到了许多预期的挑战，推出过程缓慢得令人痛苦。事实证明，与支付处理器合作具有挑战性。他们从贝宝无法维系的关系中寻求经济利益，他们对网络的忠诚正受到考验。

如果这还不够，融入商家销售点的替代途径也同样困难。对于一个相对较新的进入者来说，这是一场艰苦的战斗，再次尝试依靠网络铺设的既定轨道。

在网上世界，贝宝的崛起同样复杂，但明星们围绕着他们的价值主张和网络目标。在 20 世纪 90 年代末，易贝有数亿美元的电子商务交易(正在走向数十亿美元)，这些交易是用支票和汇票结算的。

贝宝提供了廉价和容易地将易贝卖家加入卡网络的手段，在这个过程中，潜在的数十亿美元的支付量转移到了卡网络。贝宝对网络有好处，他们合作了。贝宝对易贝也有好处，因为他们通过加快支付周期来加快交易速度，所以易贝收购了贝宝。

谷歌也想尝试拥有移动交易

像网络和贝宝一样，谷歌在 2010 年认识到了线下支付的价值，以及随着智能手机的爆炸式发展，新进入者扮演重要角色的潜力。谷歌想尝试成为这里的主要玩家，但原因不同。对谷歌来说，一切都在于数据。

通过坐在消费者线下支付支出的中间，他们将有一个窗口来了解消费者在哪里消费，以及这些消费是如何受到广告影响的。谷歌将第一次能够像他们已经能够在网络上做的那样，从广告曝光到消费者支出“闭环”,从而占领更多的广告市场。特别是，他们可以向 CPG 品牌出售更多广告，这是一个巨大的、以前未被谷歌渗透的广告市场

2011 年，谷歌开创了 NFC 支付和安全令牌化的愿景(稍后将详细介绍这些技术)。这是一个美丽的概念，但像贝宝一样，他们也遇到了执行的挑战。他们需要与大型移动运营商合作，这些运营商对拥有线下支付有自己的愿景。而且，他们来得早；在整个解决方案所依赖的终端升级之前。谷歌钱包虽然是正确的愿景，但却停滞不前。

网络努力拥有移动支付

与此同时，网络公司正忙于研究如何确保移动电话对他们来说是一个扩张的机会，而不是一个威胁。他们执行的策略在概念上很简单，但执行起来很复杂。该计划是做他们最擅长的事情:定义标准和规则，通过这些标准和规则，他们现有的网络可以扩展到手机取代实体银行卡的世界。他们下注的两个关键标准是:

  1. NFC(近场通信)是一种协议，通过该协议，数据将从手机无线传输到商户卡终端。NFC 是大约十年前网络所支持的标准，但由于缺乏商家支持和消费者价值主张而未能获得支持。敲一张卡并不比刷卡简单多少。现在是 NFC 第二步行动的时候了。

  2. 通过“安全元件”进行令牌化是一种将卡号存储在手机上并通过 NFC 传输到商户卡终端的方法。简而言之，它是一种手段，可以(I)将银行卡号存储在移动电话的安全芯片上，以及(ii)分发代表该卡号的令牌，从而产生比塑料银行卡更安全的系统，并且可以由第三方移动电话制造商制造和分发。

网络的策略随之而来，因为他们把所有的重量(和机会)放在这两个协议后面。他们的第一步是找出如何让商户卡终端支持 NFC，而实现这一点的方法是，再次定义和立法标准。

2006 年，欧洲强制要求使用“芯片和密码”安全卡(又名 EMV)。这些卡基于早在 1994 年开发的 EMV 标准，提供了额外的安全层。它们在世界各地被广泛采用，并被证明能够降低欺诈成本。在美国，推动采用 EMV 的努力已经停止。

在世界其他地方，信用卡欺诈并不被认为是一个大问题(这是在家得宝和塔吉特安全漏洞之前)，转移到“芯片和密码”将需要升级全美所有的商业卡终端，估计花费超过 60 亿美元。

然后，主街受到 2008 年经济衰退的打击，人们对强制进行昂贵的升级更没兴趣了。然而，电视网有另一个理由搬到 EMV。也许是巧合，也许不是，当商家升级他们的终端以支持 EMV 时，他们也将获得 NFC 功能，这是非常确定的。

从 2013 年开始，这些网络再次合作定义并随后发布了标记化工作的标准。许多业内人士告诉我，这是任何需要网络间合作的标准从概念到最终发布的最快速度。

电视网开始行动了。2011 年和 2012 年，这些网络合作宣布授权将美国支付网络升级到 EMV。所有商户将被要求升级支付终端以支持 EMV，否则将承担欺诈交易的责任。方便的是，对 NFC 的支持将是这次升级的副产品。

实现安全移动支付的下一步是获得对令牌化的支持。从 2013 年开始，这些网络再次合作定义并随后发布了标记化工作的标准。许多业内人士告诉我，这是任何需要网络间合作的标准从概念到最终发布的最快速度。

随着关键标准和技术的到位，以及银行和商家的加入，网络现在需要一个主要的移动电话协议来吸引消费者并使其全部工作。进入苹果——完美的分销合作伙伴——一个将不同的各方聚集在一起并创造消费者需求的大师。

胜利属于网络

2014 年秋天，ApplePay 推出，正如他们所说，剩下的就是历史了。移动支付不会在一夜之间起飞，在苹果的铺垫下，谷歌钱包现在已经准备好占领 Android 离线支付。但是，经过 5 年的小冲突，很明显，移动支付的第一次胜利属于网络，让 PayPal 再战一天。

谷歌和贝宝的下一场战争

对谷歌来说，未来几年将是快速跟进执行。他们需要借鉴电视网和苹果公司制定的技术和交易结构。他们不太可能获得苹果能够提取的经济份额，但如果他们能够利用这些数据做广告，那也没什么。

对于 PayPal 来说，情况要复杂得多。2015 年秋季，PayPal 将成为一家独立的上市公司，在许多方面，如果它不能在移动操作系统即将成为消费者首选钱包的世界中做出正确的战略举措，它将会失去更多。对于在线支付(PayPal 的全部业务)，PayPal 将需要为消费者开发一个价值主张，这个价值主张大于(1)隐私:购物时不需要分享你的卡号，以及(2)便利:你所有的卡都是数字存储的，你不需要掏出你的实体钱包。对于这些主张，苹果和谷歌现在比平价更好。这些创新可能会围绕元商务服务，如商家忠诚度/激励、支出管理和消费者对消费者。无论如何，需要加快创新的步伐来捍卫已经建立起来的帝国。

另一个有趣的问题是，PayPal 是否会在更长时间内保持独立公司的地位，或者 PayPal 的分拆是否会为其在未来几年被收购做好准备。很容易认为，对于一个主要网络来说，这是一笔极其宝贵的资产，它拥有数字第一的全球足迹，比任何现有公司都更灵活的技术堆栈，与数百万消费者和商家的直接关系，以及巨大的直接借记网络。

这将是正在进行的战争中有趣的第二场战斗。
