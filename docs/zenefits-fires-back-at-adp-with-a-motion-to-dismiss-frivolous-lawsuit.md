# Zenefits 以驳回“无聊”诉讼的动议回击 ADP

又来了。价值数十亿美元的云人力资源软件初创公司 Zenefits 提交了一份动议，要求驳回来自 [ADP 的所谓“无聊的”诽谤诉讼。](https://web.archive.org/web/20221006182054/http://www.adp.com/)

薪资服务提供商 ADP 最近关闭了 Zenefits 对运行薪资所需的客户信息的访问，声称这家初创公司正在以未经授权的方式收集这些信息，并带来了安全风险。Zenefits 在[的一篇博客文章](https://web.archive.org/web/20221006182054/http://blog.zenefits.com/adp/)中回击说，ADP 实际上是在关闭访问，因为它开发了一种与 Zenefits 竞争的服务。

ADP 对这一指控提出异议，并对 Zenefits 及其首席执行官帕克·康拉德提起诉讼。

该文件认为，ADP 违反了加利福尼亚州的[反 SLAPP(反对公众参与的战略诉讼](https://web.archive.org/web/20221006182054/http://www.arilaw.com/blog/2014/12/californias-anti-slapp-statute-protecting-first-amendment-right-to-free-speech-2/)),该法规保护个人的言论自由，并提供了寻求驳回诉讼的方法，如果诉讼可能侵犯了他们的请愿权或言论自由权。

“我以前见过这个故事，”Yammer 创始人兼 Zenefits 首席运营官·大卫·萨克斯在电话中告诉 TechCrunch。“当一家大型市场的现任者感到受到竞争产品的威胁时，现任者会尽可能地设置法律障碍。”

萨克斯在 Yammer 工作期间没有面临这种考验，但他提到了优步、Airbnb 和特斯拉，作为年轻科技公司的例子，当老牌企业实体感到竞争威胁时，它们也面临类似的考验。

“旧经济就是告诉人们他们不能有自己的选择，而这正是 ADP 所做的。当我们抱怨 ADP 采取这些行动时，ADP 起诉我们诽谤，”萨克斯说。

Zenefits 认为，ADP 是一个根深蒂固的现任者，它正在诉诸琐碎的诉讼，并传播“恐惧、不确定性和怀疑”，试图压制和阻止一个破坏性的新竞争对手。

“我们与 ADP 的竞争没有问题。然而，我们确实对一家财富 500 强公司利用其律师大军阻止我们向客户讲述真相，并为他们使用任何人来帮助管理他们的工资，福利和人力资源的权利而斗争有异议。我们呼吁法院迅速驳回这一无聊的诉讼，“康拉德说。

我们已经联系了 ADP，请其对驳回动议发表评论，但目前尚未收到回复。
