# 丰田向优步投资 5 亿美元 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2018/08/27/toyota-invests-500-million-into-uber/>

丰田和优步正在合作将按需自动叫车服务推向市场，这笔交易包括这家日本汽车制造商的 5 亿美元投资。

两家公司表示，根据协议，丰田 Sienna 小型货车将配备优步的自动驾驶技术，然后部署在这家叫车公司的网络上。

该交易最初由《华尔街日报》报道，随后由 TechCrunch 证实了新的细节，这是一项不同寻常的交易，因为第三方——尚未命名的车队运营商——将拥有并运营大规模生产的自动驾驶汽车。这些公司表示，试点规模的部署将于 2021 年在优步共乘网络上开始。

首席执行官 Dara Khosrowshahi 在周一下午发布的一份声明中指出，这是优步的首次此类交易。这也应该有助于进一步改善优步作为一家鲁莽的、现在就请求原谅的初创公司的形象，特别是在 3 月份发生致命的自动驾驶汽车事故之后。

Khosrowshahi 说:“优步的先进技术、丰田对安全的承诺以及其著名的制造实力使这一合作伙伴关系非常合适。“我期待看到我们的团队一起完成的事情。”

这些公司称之为“Autono-MaaS”车队，这是一个行话术语，意思是自主移动即服务。

丰田(及其研究部门丰田研究所)对自动驾驶汽车的部署策略与竞争对手不同。该公司此前表示，计划对 autonomy 采取双重方法，称之为“守护者”和“司机”，两者都使用相同的技术堆栈。

丰田的想法是开发全自动汽车，为老龄人口和残疾人服务，并致力于开发可以在辅助和全自动之间切换的常规量产汽车技术。这种“守护者”技术将在后台默默运行。

TRI 于 2017 年 3 月首次推出了第一代自动驾驶汽车。其 Platform 2.1 车辆在几个月后发布，具有由硅谷初创公司 Luminar 开发的光测距和探测雷达。

根据这项新协议，优步的自动驾驶系统和丰田“卫士”自动安全支持系统都将集成到这些 Autono-MaaS 车辆中。

丰田还将使用其联网汽车的核心信息基础设施，这被称为移动服务平台，或 MSPF。

丰田研究院首席执行官 Gill Pratt 博士表示:“优步的自动驾驶系统和丰田的 guardian 系统将独立监控车辆环境和实时情况，增强自动驾驶和车辆的整体车辆安全。

丰田已经与优步建立了关系，尽管没有新协议下的那么密切。丰田在 1 月份的 CES 上宣布，它正在与亚马逊、打车公司优步和滴滴、汽车制造商马自达和必胜客合作开发一种电动自动穿梭巴士，可用于运送人员或包裹。商业联盟的建立是为了在近期内专注于新的电子调色板概念车的开发。
