# YouTube 也删除了亚历克斯·琼斯

又一张社交媒体多米诺骨牌因信息战争而倒下。今天早上苹果和脸书被封杀后，谷歌紧随其后，终止了亚历克斯·琼斯的页面，理由是“违反了 YouTube 的社区指导方针”，根据现在增色的一个亮红色条显示。四面楚歌，阴谋兜售主机的 Infowars 页面，直到最近还拥有 240 万订户，已经从网站和搜索结果中删除。

YouTube 是首批对琼斯进行处罚的网站之一。早在 7 月，该网站发布了反对 Infowars 的罢工，理由是违反了危害儿童和仇恨言论政策。在此过程中，有四个视频被删除，主持人被禁止直播 90 天。

YouTube 当时写道:“我们有反对危害儿童和仇恨言论的长期政策。”。“我们始终如一地根据视频内容实施我们的政策，而不考虑演讲者或渠道。我们也有明确的三振出局政策，当他们在三个月内受到三次打击时，我们就会终止渠道。”

YouTube 尚未具体说明导致终止的第二次罢工，但该公司的一名发言人向 TechCrunch 证实，担忧再次集中在仇恨言论和骚扰上。

一位发言人告诉 TechCrunch:“所有用户在注册使用 YouTube 时，都同意遵守我们的服务条款和社区准则。”。“当用户反复违反这些政策时，比如我们反对仇恨言论和骚扰的政策，或者我们禁止规避强制措施的条款，我们会终止他们的帐户。”

上周，Spotify 也因违反公司条款删除了琼斯的播客。
