# WeWork 与 Chase 合作，为会员提供折扣商务服务 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/05/12/wework-partners-with-chase-to-offer-discounted-payments-services-to-members/>

价值 50 亿美元的纽约科技宠儿[WeWork](https://web.archive.org/web/20221005201451/https://www.crunchbase.com/organization/wework)今天正式宣布[与大通合作](https://web.archive.org/web/20221005201451/https://www.wework.com/chase)，为 we work 会员提供大通商业服务(称为 Paymentech)的折扣价和优质客户服务。

为自己的企业建立商务服务的会员可以从他们当前的支付提供商那里拿到任何三张发票，WeWork 将与 Chase 合作，击败它。该公司不会得到更多关于折扣大小的具体信息，主要是因为它是基于各种算法和因素确定的。

Chase Paymentech 目前为 Square 和 Apple Pay 等支付平台提供支持。

除了更便宜的电子商务支付处理，WeWork 成员还将获得与 Chase Paymentech 客户服务代表的高级访问权限，WeWork Adam Neumann 表示，成员可以在一小时内与 Chase 员工通电话，而不是 48 小时。

[摩根大通对 WeWork 的 C 轮和 D 轮投资](https://web.archive.org/web/20221005201451/https://www.crunchbase.com/organization/wework)，正是在这里埋下了这种合作关系的种子。

或许更有趣的是，这只是我们在合作空间上提供的一长串服务中的又一个。

例如，WeWork 会员在 Rackspace 云托管上可以获得高达 2000 美元/月(12 个月)的折扣。亚马逊网络服务是免费加入的，并向 WeWork 成员提供一年的 AWS 积分，总计节省约 5000 美元。ZipCar、General Assembly、TriNet、美国电话电报公司和 WBMason 也为 WeWork 会员提供折扣。

这是可能的，原因有几个。

首先，WeWork 不是代表其大约 300 名公司员工谈判，而是代表 25000 名会员。

“让美国企业界兴奋的是，我们的任何一个 WeWork 成员都可能代表下一个谷歌或脸书，”首席执行官亚当·纽曼说。“他们希望尽早成为其中的一部分，不仅仅是成为这些创业公司增长的一部分，而是向那些更符合千禧一代的公司学习。”

根据 Neumann 的说法，WeWork 成员在 WeWork 的第一年仅在服务上就可以节省高达 25，000 美元。

Neumann 说，我们应该期待在未来看到更多的 WeWork 会员服务推出。
