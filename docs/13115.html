特斯拉的 Model X 获得 NHTSA 的全面五星安全评级

美国国家公路交通安全管理局(National Highway Traffic Safety Administration)授予特斯拉的 Model X 有史以来最高的 SUV 安全评级。Model X 在 NHTSA 测试的每个类别和子类别中都获得了五星评级，使用的是政府机构自己的独立测试流程。这也使得根据 NHTSA 的排名系统，在对乘客的整体伤害概率方面，Model X 仅次于 Model S。

这种安全性能在很大程度上归功于特斯拉汽车的独特设计，这归功于其电动动力系统。底部安装的电池组分布在车辆的底部，与传统的燃油喷射发动机 SUV 相比，Model X 的重心更低，分布更广。较低的重心极大地有助于防止翻车，这通常是 SUV 的一个大弱点，因为 SUV 在道路上的位置较高。

NHTSA 的评级系统预测，Model X 中的乘客有大约 93%的机会从严重的撞车事故中逃脱，以更相关的术语来表达高评级。这对特斯拉来说是一个很高的荣誉，尤其是与 Model S 的声誉相呼应(尽管在消费者报告中有一些在这一点上的倒退与第二代硬件汽车上的自动制动功能有关，这些功能现在正在通过更新获得)。

这意味着 Model 3 将有很大的发展空间，但如果特斯拉能够凭借这款定于下月开始交付的汽车实现 NHTSA 三连胜，它应该真的有助于确保特斯拉作为核心竞争价值的整体安全声誉。
