# Tapdaq 筹集 140 万美元，由 Balderton 牵头，用于开发友好型移动广告交换

由社区驱动的移动广告交易所 Tapdaq 已经筹集了由 Balderton Capital 牵头的 140 万美元的种子资金。该公司还拉拢了 AdMob EMEA 公司的 Russell Buckley 担任新董事长。

Tapdaq 是 Ted Nash 的创意，Ted Nash 是新一代非常年轻的企业家之一，几乎从出生就开始上网。十几岁时，他创建了匿名八卦论坛【LittleGossip.com ，比这个秘密应用早了好几年。

Tapdaq 是一个移动应用的广告交换平台，面向那些希望在彼此的应用中做广告的独立开发者。该平台通过运营自己的货币“daq ”,避开了需要开发者花费现金的问题。因此，在互利的驱动下，P2P 交易所因其高度有机的本质而在开发者中获得了牵引力。

开发者通过推广其他开发者的 app，带动用户使用，从而赚取 daq。每个开发者为他们的应用程序设定自己的价格，允许他们货比三家，寻找其他相关的应用程序进行推广。
