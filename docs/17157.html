Sketchfab 允许您将 3D 模型导入您最喜爱的 3D 软件 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/03/21/sketchfab-lets-you-import-3d-models-into-your-favorite-3d-software/?via=indexdotco

Sketchfab 已经建立了一个巨大的 3D 模型库。他们中的一些可以在一个创造性的公共许可证下获得，其他的可以购买。这家初创公司现在想帮助你处理所有这些内容。现在，您可以下载 3D 模型并将其导入您喜爱的 3D 应用程序。

在幕后，Sketchfab 已经建立了一个下载 API 和一些与流行的 3D 工具的集成。之后你会在第三方 app 里找到一个 Sketchfab 搜索栏。你可以键入几个关键词，按质量过滤，预览，然后将 3D 模型下载到你当前的作品中。

Sketchfab 使用 glTF 格式。最终，这家初创公司将允许你存储自己的 3D 模型，这样你就可以从所有支持 Sketchfab 的应用程序中访问它们。你也可以在那里找到你购买的模型。

Sketchfab 为 Unity、Unreal 和 Godot 开发了插件。游戏开发者会喜欢这种集成，尤其是如果你想在构建自己的 3D 模型之前快速构建一个原型。

您还可以在 Torch3D、Minsight、Spatial stories、Selerio、StellarX、Holobeam、AnimVR、Plattar、Sketchbox3D 和 Looking Glass 中使用 Sketchfab。

该公司还致力于整合亚马逊苏美尔、Blender、Substance、Modo、4D 影院、Octane、HighFidelity、Aframe、Houdini、Sansar、Mimesys、The Wave VR、Masterpiece VR、Artomatix、Flipside VR、Blend Media、NormalVR、Vectary、Streem 和 Meta glasses。

如你所见，Sketchfab 涵盖了广泛的 3D 软件，以及 VR 和 AR 平台。Sketchfab 上有超过 250 万个 3D 文件。因此，提供一个下载 API 是有意义的，它就像消息应用程序中的 Giphy 集成一样工作。
