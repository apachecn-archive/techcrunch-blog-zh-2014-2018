# Spring 对每个订单都实行免费送货

最近完成了[2500 万美元 B 轮融资](https://web.archive.org/web/20221002192534/https://beta.techcrunch.com/2015/04/16/spring-the-shoppable-instagram-closes-25m-in-series-b/)的市场购物应用 Spring 宣布，用户现在将获得每单免费送货。

Spring 是一个移动应用程序，提供类似 Instagram 的购物体验，你可以关注你最喜欢的零售商，然后在应用程序中轻轻一扫就可以购买。

由于 Spring 上的每一件商品都是直接从设计师那里购买的，传统上不同品牌的运输政策是不同的。今天的公告改变了这一点，现在所有商品都可以享受标准的免费送货服务。

虽然该公司正在凑钱补贴品牌的一些成本，但 Spring 的首席执行官 Alan Tisch 指出，在测试中，免费送货为零售商带来了更高的转化率。

Tisch 还解释说，这是该公司提供最佳客户体验的更广泛努力的一部分。Spring 最近推出了内部礼宾团队，帮助客户解决问题，即使这些问题只与品牌有关。

现在，Spring 推出了来自 750 多个品牌的近 35，000 种独特商品。此外，该公司表示，其回头客率大约是电子商务平台典型回头客率的 2.5 倍。

免费送货立即可用，你可以在 [iOS](https://web.archive.org/web/20221002192534/https://itunes.apple.com/us/app/spring-go-shopping/id899156093?mt=8) 和 [Android](https://web.archive.org/web/20221002192534/https://play.google.com/store/apps/details?id=com.jellolabs.spring.android&referrer=ts%3D9d9c38ba-0e33-11e5-9f16-002590a28fc7) 应用商店下载 Spring。
