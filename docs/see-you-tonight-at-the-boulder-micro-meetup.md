# 今晚在博尔德的小型聚会上见

提醒一下，今晚 7 点我将在科罗拉多州博尔德市百老汇的新兴城市举行一场小型聚会。

你将有两分钟时间面对两位当地评委，然后是两分钟的提问时间。没有幻灯片，没有道具，没有服装(好吧，也许是服装。)

新兴城市位于百老汇 2060 号地下一层。要到达那里，把车停在云杉街和百老汇街附近，它和 Zeal Optics 和看不见的豆子在同一栋大楼里。最近的公共停车场在停车库或云杉路上的公共停车场；晚上 7 点以后，路边收费表和停车场免费开放。一个街区外的松树街和周围的街道上有三个小时的免费停车场。

你可以在这里回复[。名额有限，请早点报名。](https://web.archive.org/web/20230129062330/https://www.eventbrite.com/e/boulder-micro-meetup-tickets-33008825285)

比赛结束后，我们将在[波西米亚风格的 Biergarten](https://web.archive.org/web/20230129062330/http://bohemianbiergarten.com/) 跳舞唱歌，他们将在后面的房间里唱卡拉 ok，提到 Boomtown 或 TechCrunch，并获得所有饮料的欢乐时光价格
