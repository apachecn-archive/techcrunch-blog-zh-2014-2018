# Yandex 介绍了 Alice，一个类似 Alexa 的助手，会说俄语

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/10/10/yandex-introduces-alice-a-alexa-like-assistant-that-speaks-russian/>

Yandex 在美国几乎不为人知，但自成立以来的 20 年里，它已经成为俄罗斯的谷歌(以及亚马逊和优步)。该公司在其祖国控制着大约 65%的搜索引擎市场份额，并且已经扩展到几乎所有其他可以想象的软件类别，所以智能助手是一个显而易见的事情。

该公司一直在[使用某种形式的人工智能](https://web.archive.org/web/20230326023123/https://techcrunch.com/2016/10/12/yandex-woos-smartphone-makers-with-ai-browser-and-launcher-partnership-programs/)来完成一些不同的任务，包括产品推荐，但[爱丽丝](https://web.archive.org/web/20230326023123/https://yandex.com/company/blog/say-privet-to-alice-yandex-s-intelligent-assistant/)标志着该类别更加外向。这款助手类似于俄罗斯的 Alexa 或 Siri，可以提供该公司的各种产品，包括天气、新闻、地图等。

Yandex 机器智能和研究部门的负责人米沙·比连科(Misha Bilenko)告诉 TechCrunch，这项技术是该公司一段时间以来一直在研究的技术的高潮，包括该应用程序中已经存在的语音搜索功能。

“在过去的一年里，我们一直在用一个包括对话管理在内的新系统将这一切联系在一起，”他说。“实际上，如果你有一个高质量的搜索引擎，你已经能够很好地回答问题了。”

该公司声称该助手“展示了接近人类水平的语音识别准确性”，这似乎意味着它能够回答更随意的查询，并根据上下文线索获取信息，即

> 例如，当用户问爱丽丝，“莫斯科的天气如何？”然后用俚语提问，“那彼得呢？”Alice 理解其意图，并提供了圣彼得堡的天气预报。

只有，你知道，在俄罗斯。对于智能助手来说，这是一个棘手的问题——尽管许多英语公司，如谷歌和苹果，已经在自己的产品中很好地解决了这个问题。即便如此，这仍然可能使爱丽丝成为同类中最有能力的俄语助手。

其中一个不同点是一个名为“聊天”的功能，旨在使助手更具对话性。在这一点上，它基本上是一个新奇的东西，但它是一个让整个事情更加逼真的尝试。

有趣的是，爱丽丝的声音是由塔季扬娜·什托娃提供的，她在俄语版的《T2》中为斯嘉丽·约翰逊配音。这应该有助于实现人们对助手的任何一种不久的将来的反乌托邦幻想。

比连科表示，鉴于互联网让一切变得糟糕的先例，该公司将不得不对聊天功能保持警惕，例如[微软的 Tay AI](https://web.archive.org/web/20230326023123/https://techcrunch.com/2016/03/30/you-are-too-fast-please-take-a-rest/) ，它很快就变成了一堆热气腾腾的种族主义垃圾。

Yandex 移动应用程序现在可以使用该功能，并将在移动浏览器中使用。该公司不愿证实其正在开发类似于 Echo/Google Home 的助手的传言，尽管比连科补充说，“这些产品都没有在俄罗斯市场上销售，所以我们不认真考虑一下就太傻了。”
