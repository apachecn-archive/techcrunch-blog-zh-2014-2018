OS X 的下一个版本可以叫做“苹果电脑”

原文：https://web.archive.org/web/https://techcrunch.com/2016/04/17/the-next-version-of-os-x-could-be-called-macos/

OS X 的下一个版本可以叫做“苹果电脑”

这是一个小的品牌变化，但它会有很大的意义。许多不同的迹象表明，苹果放弃了“OS X”这个名字，转而使用“MacOS”，或者可能是没有大写字母的“macOS”。

菲尔·席勒全球营销公司的苹果 SVP 去年在 WWDC 接受约翰格鲁伯采访时首次暗示了这一变化。然后一个配置文件提到了 macOS 而不是 OS X。最后，苹果自己网站上的一个页面说 macOS 而不是 OS X 来描述苹果的环境倡议。

那是一大堆烟。TechCrunch 也听说这可能会在 6 月的 WWDC 上宣布。

这并不是苹果第一次为 Mac 重塑其操作系统。当 Mac OS 9 的继任者首次公布时，它被称为 Mac OS X，苹果在 2012 年去掉了“Mac”部分。

当时，苹果希望将设备与软件元素区分开来。随着 iOS 在 iPhone 和 iPads 上运行，iPhone OS 于 2010 年 6 月成为 iOS。

但是，从那以后，苹果发布了两个基于 iOS 的新平台。苹果电视现在运行 tvOS，苹果手表运行 watchOS。你能看到趋势，对吗？

与苹果的其他平台相比，OS X 现在似乎是一个过时的品牌。类似地，在名字中保留“十”意味着 OS X 的版本号已经被保留了 15 年。现实有所不同，因为苹果每年都会发布 Mac 操作系统的主要版本。

但让我相信 OS X 正在被淘汰的原因要明显得多。苹果将于 6 月在 WWDC 发布 iOS 10。开发者、记者和苹果公司本身都将谈论 iOS 10 几个月。

iOS 10 和 OS X 的名字听起来太相似了，尤其是当你听到有人谈论这些操作系统的时候。将 OS X 更名为 MacOS 将统一苹果的操作系统命名方案，但这将是一个非常受欢迎的变化，让大家在谈论新的操作系统时保持一致。
