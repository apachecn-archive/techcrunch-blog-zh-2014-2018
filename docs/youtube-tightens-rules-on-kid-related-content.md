# YouTube 收紧儿童相关内容的规定

YouTube 是网络上最好和最差内容的奇怪大杂烩，它正在打击与儿童视频相关的滥用和错误。针对儿童或在视频中使用儿童的玩笑者和道德问题创作者将更难获得观众和资金。

孩子们观看一系列算法建议视频的倾向导致一些 YouTubers 试图将他们的内容插入指定的儿童友好流。例如，把粉红猪小妹放在他们的视频中，让它被认为是友好的动物，然后让粉红猪小妹参观屠宰场或其他地方。如果你是一个无聊的青少年，这很有趣，但如果你是 YouTube 的广告部，就不好笑了。

满足于孩子，但不满足于孩子也是一个问题。我不知道为什么 1.5 亿人要看一个小女孩通过一张充满鲜血的嘴哭两分钟，我也不想知道。但这可能不是你想为小家伙排队买的东西。

为了应对这种“增长趋势”，谷歌正在用几种方式来为儿童友好的内容做准备:

*   **更严格的内容控制。**现在警告:“含有未成年人的内容可能会危及儿童，即使这不是上传者的意图”(这将是[满嘴鲜血的人](https://web.archive.org/web/20230308214454/https://techcrunch.com/2017/11/17/youtube-terminates-exploitive-kids-channel-toyfreaks-among-broader-tightening-of-its-endangerment-policies/))和“含有家庭娱乐角色但包含成熟主题或成人幽默的内容。”

[![](img/272917811232ce8a084d7ee3bcdfc1ef.png)](https://web.archive.org/web/20230308214454/https://techcrunch.com/wp-content/uploads/2016/10/fellowkids.gif)

*   消除儿童色情的东西。粉红猪小妹去火腿工厂不会得到任何广告费。
*   **对孩子的视频进行攻击性评论巡逻。**YouTube 的评论部分是，[借用 Penny Arcade 的话](https://web.archive.org/web/20230308214454/https://www.penny-arcade.com/comic/2003/01/27)，“就像桶的底部有它自己的*桶，还有另一个底部，然后滴在一片病鼠的海洋上，然后跳进一个肮脏的下水道。”被发现对其中的孩子有不恰当的(“令人憎恶的”)评论的视频将被完全关闭。*
*   **更好地理解上下文。** *瑞克&莫蒂*也许是一部动画片，但你可能不想让你 7 岁的孩子看。同样，爱探险的朵拉的性感角色扮演(为什么？)其实不算教育内容。因此，YouTube 正在“增加我们合作的专家数量，并将我们在这一领域合作的受信任的旗手数量增加一倍”，以更好地跟踪这类事情。

希望这能避免一些父母不得不回答一些尴尬的问题，比如屠宰方法或者为什么香蕉需要吊床。
