# Slack 赢得了上升最快的创业公司

Slack 正在向右上方生长，就像这样一个快速攀爬者昨晚在 [TechCrunch 第九届年度颁奖典礼上赢得了上升最快的初创公司奖](https://web.archive.org/web/20230125061723/https://techcrunch.com/2016/02/08/and-the-winners-of-the-9th-annual-crunchies-are/)。Slack 还声称拥有比其他任何初创公司都更加多样化的员工队伍，并明智地选择派出四名工程师上台领奖，他们都是黑人和女性。

Slack 派出的获得上升最快创业奖的女性之一埃里卡·贝克(Erika Baker)也是包容性多样性奖的亚军。我们在获奖后与所有四位女性在后台进行了交谈，看看她们希望自己在舞台上的存在会如何激励更多女性和少数族裔担任工程角色。
