专利钓鱼者| TechCrunch

流行的观点是“不要喂巨魔”。但随着一个新项目的推出，Cloudflare 走得更远了——该公司试图通过剥夺他们的收入来饿死他们。

针对科技公司的专利诉讼相当常见，大多数公司只会与对手和解，而不会资助昂贵、漫长的法律辩护。因为公司很快解决，专利诉讼已经成为诉讼当事人赚钱的有利可图的方式。巨魔可以以低至一美元的价格购买技术专利，然后利用这些专利从资金雄厚的公司那里榨取和解金。

但是，当一家名为 Blackbird Technologies 的公司上个月起诉 Cloudflare(以及 Fastly，一个云平台)侵犯其 1998 年的一项专利(涉及代理系统错误消息)时，Cloudflare 决定反击。内容交付网络正在启动 Jengo 项目，这是一个奖金项目，旨在使黑鸟拥有的 70 多项专利无效。

Jengo 项目提供 50，000 美元奖金，用于研究“现有技术”，即专利中涵盖的技术的使用，表明该技术在专利申请前已被广泛使用。现有技术的证据可以用来削弱专利侵权索赔。该基金中的 2 万美元将用于使 Cloudflare 和 Fastly 诉讼中的专利无效，而另外 3 万美元将用于使 Blackbird 的其他专利无效。

“自 2014 年 9 月以来，黑鸟对公司提起了 107 起诉讼，表明它将利用其专利起诉生产公司，”Cloudflare 首席执行官马修·普林斯在的博客帖子中写道。“专利钓鱼者经常瞄准更新、更具创新性的公司，因此调查和研究黑鸟专利是否存在现有技术，并阻止他们对我们或其他公司提起更多诉讼，这一点很重要。”

Project Jengo 诉讼中产生的研究可能有助于 Cloudflare 在法律战中取得胜利(与其他面临专利诉讼的公司不同，Cloudflare 没有和解)。

“面对滋扰，公司和解。我们不打算在任何时候和解，”Cloudflare 总法律顾问 Doug Kramer 告诉 TechCrunch。“通过和解，我们正在解决一个更大的问题。组织需要站出来。”

除了启动 Jengo 项目和打官司，Cloudflare 还在争夺 Blackbird 的领导权。这家公司是由律师创建的，并不制造任何自己的技术；相反，它收购专利，然后就此提起诉讼。Cloudflare 认为这是不道德的，并对黑鸟律师提起诉讼。

TechCrunch 联系了 Blackbird 进行评论，如果公司有回应，将会更新。更新:“黑鸟科技已经审查了 Cloudflare 对我们的指控。这些指控完全没有根据，我们将积极为自己辩护，”黑鸟首席执行官温迪·韦兰德告诉 TechCrunch。
