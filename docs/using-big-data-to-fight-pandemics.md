# 使用大数据抗击流行病 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/11/08/using-big-data-to-fight-pandemics/>

**编者按:** *努里娅 奥利弗是 [Telefonica](https://web.archive.org/web/20221208161645/http://www.telefonica.com/en/home/jsp/home.jsp) 的科学总监，他正在研究如何利用大数据来帮助抗击埃博拉和禽流感等流行病。*

去年，[我在 WIRED 2013](https://web.archive.org/web/20221208161645/http://www.wired.co.uk/news/archive/2013-10/17/nuria-oliver) 上做了一个演讲，讲的是如何利用匿名和聚合的手机数据来了解和抗击传染病的传播。我描述了几年前我们在我的研究团队中进行的一项研究，在 2009 年春天 H1N1 流感爆发期间，我们分析了来自墨西哥的聚合和匿名移动数据。

由于移动电话的广泛使用以及匿名和汇总数据的力量，我们能够量化墨西哥政府采取的措施对人口流动性以及疾病传播的影响。我们以及卡罗林斯卡学院和哈佛大学等机构的研究人员展示了如何利用大规模移动数据分析为社会带来巨大好处。

我一点也不知道，今天我们将抗击我们历史上最严重的埃博拉疫情，已经有近 5000 人死亡，13000 多人感染。不幸的是，在疫情疫情爆发几个月后，我们现在才开始协调努力，分析手机网络数据，以及这些数据对埃博拉病毒传播的影响。

可以理解的是，人们的努力都集中在其他地方。本周在釜山举行的国际电联全权代表会议上，国际电信联盟(ITU)、全球移动通信协会(GSMA)和互联网协会(ISOC)宣布他们将联手抗击埃博拉病毒。这种团结是向前迈出的重要一步，但我真的希望与 GSMA、联合国全球脉搏和许多其他数据科学家一起，确保我们，最重要的是非洲移动运营商，抓住这个机会，真正利用现有数据的潜力。

当然，移动数据分析不能直接帮助地面医生和护士的英勇工作，但在规划资源分配或了解不同移动性遏制措施的有效性时，它可能会被证明非常有帮助。

流动性是导致埃博拉等人传人传染病传播的关键因素之一。因此，了解和量化受埃博拉病毒影响地区的人口流动对于遏制疫情至关重要。而人口流动恰恰是可以利用大规模匿名化移动数据进行分析和预测的特征之一。

此外，手机信号发射塔在特定时间段内的活动水平可以被视为该信号发射塔所服务的地理区域内的人口数量的代表。对受埃博拉影响地区的信号塔活动水平的变化进行建模，将有助于深入了解疫情导致的人口变化。

虽然这些数据远非完美，但它提供了有价值的信息，否则收集起来会非常昂贵和耗时。

可以理解的是，特别是在西非，人们可能会担心对隐私的影响。好消息是，由一系列学术团队进行的广泛研究表明，分析人类移动模式和保护隐私是可能的。所有数据通常使用最先进的加密算法进行匿名处理。此外，数据通常由授权人员在高度安全和受保护的环境(例如移动运营商所在地)中进行分析。没有进行任何分析来识别个人。此外，相关援助机构或政府机构只能获得由此产生的非敏感性综合分析(如人口流动估计、综合统计……)。

技术上的困难也不应该成为障碍，因为有大量的工作说明了如何进行这种类型的分析。此外，包括我们在内的一批技术高超的数据科学家，以及 ITU、ISOC、GSMA 和 United Nations Global Pulse 等组织的大力支持，随时准备并愿意在这一过程中为非洲运营商提供帮助，特别是确保所有数据处理都以道德和匿名的方式进行，始终遵守当地的数据隐私法。

产生积极影响和帮助拯救生命的潜力是巨大的。我真心希望我们能尽快找到一种方法来实现大数据对社会公益的全部潜力。这是一个我们不能错过的机会。
