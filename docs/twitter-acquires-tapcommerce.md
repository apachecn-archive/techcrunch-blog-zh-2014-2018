# Twitter 确认收购移动广告重定向初创公司 TapCommerce

Twitter 刚刚宣布收购移动广告初创公司 TapCommerce。

[Re/code 的 Ina Fried 今天下午早些时候透露了关于收购的消息](https://web.archive.org/web/20230323184738/http://recode.net/2014/06/30/twitter-scoops-up-adtech-firm-tap-commerce-for-around-100-million/)，报道称交易价格为 1 亿美元。(Twitter 拒绝提供价格)。

TapCommerce 帮助移动企业重新定位广告，即根据之前的用户活动定位广告——这在桌面网络广告中很常见，但由于缺乏 cookies，在移动设备上并不常见。(联合创始人兼首席执行官 [Brian Long 表示](https://web.archive.org/web/20230323184738/https://techcrunch.com/2013/06/10/tapcommerce-funding/) TapCommerce 能够利用“大量数据和复杂的统计分析”来重新定位。)例如，一个电子商务应用程序可以使用重新定位的广告来说服一个流失的用户回来，也许是通过促进新的销售和促销活动。

看起来 TapCommerce 的技术可以为 Twitter 提供改善[今天宣布的](https://web.archive.org/web/20230323184738/https://techcrunch.com/2014/06/30/twitter-rolls-out-app-install-and-engagement-ads-and-new-click-pricing-globally/)移动应用安装和参与广告的方法。([脸书也一直在推出这方面的工具](https://web.archive.org/web/20230323184738/https://techcrunch.com/2013/10/01/facebook-mobile-app-ads-calls-to-action/)。)

在一篇博文[中，Twitter 暗示此次收购是其更广泛的移动营销战略的一部分，该战略始于去年](https://web.archive.org/web/20230323184738/https://blog.twitter.com/2014/welcoming-tapcommerce-to-the-flock)[对 MoPub](https://web.archive.org/web/20230323184738/https://techcrunch.com/2013/09/09/twitter-said-to-acquire-mopub/) 的收购。(本月早些时候，该公司还宣布收购移动广告初创公司 Namo Media 。)在最近的收益报告中，该公司称其 80%的广告收入来自手机。

Twitter 全球在线销售副总裁理查德阿尔方西(Richard Alfonsi)写道:

> 与 TapCommerce 团队合作，Twitter 将能够为移动应用营销人员提供更强大的应用再参与能力、用于实时程序化购买的工具和托管服务解决方案，以及更好的衡量能力。结合我们的其他广告解决方案，广告主将能够在整个用户生命周期内，从通过应用安装获得新用户，到吸引已经在设备上安装了广告主应用的现有用户，推动 Twitter 内外移动消费者的转化率和投资回报。如果你是一个日常移动用户，我们预计这将意味着在你使用的应用程序中有更好、更相关的广告。

当被问及是什么让 TapCommerce 脱颖而出时，Alfonsi 告诉我，他对 TapCommerce 对应用重新参与的关注、对程序化广告购买的支持以及现有客户名单印象深刻，他说现有客户名单与 Twitter 的客户名单有很大重叠。顾名思义，该公司最早的广告客户来自电子商务领域，但 Alfonsi 表示，该公司现在也有游戏、旅游和生活消费。([tap commerce 团队表示](https://web.archive.org/web/20230323184738/http://tapcommerce.tumblr.com/post/90380536007/tapcommerce-joins-the-twitter-family)该交易不会影响其向现有客户提供的服务。)

“人们已经谈论[移动重定向]有一段时间了，但我认为它现在真的开始变成现实，”阿尔方西说。

他补充说，我们可能会在 Twitter(或其他运行 MoPub 广告的应用程序)上使用 TapCommerce 技术，“现在说还为时过早”:“我们仍处于基本的团队整合阶段，向他们展示浴室等诸如此类的东西，但我们开始考虑产品路线图，我们对未来的发展感到兴奋。”
