Spotify 增强其免费等级| TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/04/24/spotify-beefs-up-its-free-tier-as-apple-music-gains-traction/

今天在纽约的 Gramercy 剧院，Spotify 的首席 R&D 官 Gustav sder strm 宣布了 Spotify 移动应用程序的全新免费版本。

通过利用他们在机器学习方面的投资，Spotify 的新免费层向用户即时推荐音乐。也就是说，免费层总是限制用户洗牌。有了新版本，用户可以点播他们想要的任何歌曲，想听多少次就听多少次，只要这些歌曲出现在 15 个个性化发现播放列表中的一个，如 Daily Mix，Discover Weekly，Release Radar 或 Today's Top Hits。

总的来说，Spotify 向用户提供了大约 750 首歌曲(> 40 小时的音乐)，供用户点播。

Spotify 还将基于现有的用户自制播放列表在免费移动版本中进行推荐，从这些播放列表上的歌曲到播放列表本身的名称。该公司称之为“辅助播放列表”，这实质上意味着每次你搜索一首歌曲添加到播放列表中，Spotify 也会提供类似的推荐。

最后，Spotify 内置了一种低数据模式(称为 data saver ),可以减少高达 75%的数据消耗。过去，Spotify 不允许免费离线收听，这意味着如果用户需要保存数据，他们会在某种程度上受限于 wifi。

有了新的数据消费系统，它可以提前缓存音乐，通过 3G 传输，用户实际上可以通过无线数据听更多的音乐。除了利用 3G，Spotify 还优化了流媒体本身和应用程序(包括图像和其他 UI 元素)，以节省数据和电力。

然而，需要明确的是，免费的 Spotify 服务仍然不允许任何离线收听。

尽管如此，广告仍将在 Spotify 的免费层上播放。这是该公司战略的一部分，不仅为免费层提供资金，还将用户转化为高级用户。

2014 年，Spotify 将其免费层引入移动设备，允许用户在 shuffle 上收听他们的播放列表和广告。这是 Spotify 免费层增长的一大部分。事实上，如今 Spotify 拥有 9000 万免费用户。其中许多人转化为付费用户——该公司现在有 7000 万付费用户。

“如果你是在听音乐约会，你不会想要广告出现，”Spotify 的全球创作服务负责人特洛伊·卡特说。

自 2014 年以来，该公司一直专注于移动业务，尤其是在涉及高端移动播放器的领域。

Spotify 建立在三个层次之上:无处不在、个性化和免费增值。索德斯特罗姆解释说，Spotify 认为自己是 90 年代的广播电台，在那里，伟大音乐的发现得到广告的支持，并驱使人们去唱片店。

Spotify 的免费层代表 Spotify 的广播电台，是 Spotify 整体战略的关键部分，因为像苹果音乐这样的付费服务继续增长。
