# Twitter 链接温布尔登网球公开赛、动漫展报道等直播交易

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/06/30/twitter-inks-live-stream-deals-for-wimbledon-comic-con-coverage-and-more/>

Twitter 正在继续发展其直播业务，在过去一周左右的时间里宣布了一系列新的交易，包括将于 7 月在圣地亚哥动漫展(San Diego Comic-Con)现场直播的合作伙伴关系，以及来自温布尔登的幕后行动和新闻，以及其他一些努力。

你可能还记得，去年，在 Twitter 宣布斥资 1000 万美元直播 NFL 周四晚上的足球比赛之后，温布尔登是 Twitter 网络上出现的第一项直播体育运动。当时，Twitter 并没有大力推广温布尔登的报道，相反，它被视为对直播如何在 Twitter 平台上运行的测试。

时代变了。自那以来的一年里，Twitter 已经为其提供了一系列直播体育、体育相关节目、[新闻](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/05/01/twitter-will-offer-247-streaming-video-news-in-partnership-with-bloomberg/)、[音乐会](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/05/01/twitter-stream-concerts/)和其他活动，包括来自 [NFL、](https://web.archive.org/web/20230316070808/https://techcrunch.com/2016/07/25/twitter-mlb-nhl/)、 [NBA、](https://web.archive.org/web/20230316070808/https://techcrunch.com/2016/07/19/twitter-will-stream-two-nba-shows/)、 [NHL、](https://web.archive.org/web/20230316070808/https://techcrunch.com/2016/07/25/twitter-mlb-nhl/)、 [NLL、](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/03/15/twitter-adds-lacrosse-to-its-live-streamed-sports-lineup/)、[大学体育、](https://web.archive.org/web/20230316070808/https://techcrunch.com/2016/07/14/twitter-signs-a-live-streaming-deal-to-bring-pac-12-university-sports-to-its-service/)，甚至[电子竞技](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/03/02/twitter-will-live-stream-1500-hours-of-esports-including-original-content/)和[更多](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/05/01/twitter-announces-new-live-shows-from-the-wnba-buzzfeed-live-nation-and-more/)的流媒体。

这一次，Twitter 的温布尔登协议是与全英俱乐部[而不是 ESPN](https://web.archive.org/web/20230316070808/https://techcrunch.com/2016/07/06/twitter-started-its-live-coverage-of-wimbledon-today/)达成的，该俱乐部与 Twitter 合作，在赛事期间直播温布尔登频道。这种报道将包括日常内容，如新闻和采访，幕后镜头，以及比赛中的“精选动作”。(换句话说，你不能看完整场比赛，直播)。

这笔交易是 Twitter 在过去几周宣布的几笔交易之一。另一个值得注意的合作伙伴是 IGN，它将通过 comiccon.twitter.com 直播 2017 年圣地亚哥动漫展这一大型娱乐活动。Twitter 表示，从 7 月 19 日到 7 月 22 日，这家媒体公司将在展厅进行长达 13 小时的直播报道。

这将包括来自 ABC、AMC、DC、狮门影业、漫威、网飞、Starz、TBS 和其他媒体的采访，包括 IGN 主持人和特邀嘉宾在节目前后的现场评论。报道还将增加预告片、幕后镜头、对演员和制片人的采访、与会者穿的 cosplay 等等。

这笔交易代表着 Twitter 和 IGN 之间关系的扩大，最近他们利用社交网络广播了洛杉矶 2017 年电子娱乐博览会(E3)的报道，该活动以电子竞技为重点，在 T2 举行

最近的其他交易也将为 Twitter 带来更多的体育项目，因为该公司正在抢购更多的小众节目，如加拿大足球联盟([【CFL】](https://web.archive.org/web/20230316070808/http://twitter.com/cfl))、阿拉伯世界的[俱乐部间足球](https://web.archive.org/web/20230316070808/http://ac.twitter.com/)，以及全国女子曲棍球联盟([【NWHL】](https://web.archive.org/web/20230316070808/http://nwhl.twitter.com/))。

今年夏天早些时候，Twitter 直播了其他重大事件，例如詹姆斯·科米通过彭博在国会作证和爱莉安娜·格兰德的 One Love Manchester 慈善音乐会。

尽管对直播的持续关注有助于 Twitter 更好地履行其使命，成为一个你可以了解正在发生什么的网络*现在*，但在获得人们实际上想要观看的旗舰报道方面，它也面临着来自直播对手和其他主要科技公司的激烈竞争。(加拿大足球不是。)

例如，今年早些时候， [Twitter 输给了亚马逊](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/04/04/amazon-will-live-stream-10-nfl-games-this-season-replacing-twitter/)，据报道，[为转播权支付了 5000 万美元，比 Twitter 去年支付的费用高出五倍。这导致了一些对 Twitter 在直播领域的努力的批评，专家们说，它不会有足够的资金来跟上今天的主要参与者——它将在更好的直播交易中被出价超过，也就是说，被留下来收拾残局。](https://web.archive.org/web/20230316070808/https://www.wsj.com/articles/nfl-and-amazon-reach-one-year-streaming-deal-for-about-50-million-1491347701)

然而，作为对输给亚马逊的回应，Twitter [与 NFL](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/05/11/twitter-announces-a-new-deal-for-year-round-nfl-content-that-includes-live-video-but-no-games/) 合作了自己的直播视频协议，包括新闻和集锦，但不包括游戏。这可能不是最好的 NFL 内容，但它确实符合 Twitter 关注新闻的声誉。毕竟，这里是新闻突发、被讨论的地方，也是大量记者通常出没的地方。

尽管如此，Twitter 最近的一些声明确实表明，它在体育长尾和其他缺乏主流吸引力的小型活动上押了双倍赌注。公司[在 5 月](https://web.archive.org/web/20230316070808/https://techcrunch.com/2017/05/11/twitter-announces-a-new-deal-for-year-round-nfl-content-that-includes-live-video-but-no-games/)表示，它有 200 个优质直播视频合作伙伴，这似乎支持了它正在做大量“小土豆”交易的假设。

但是，尽管你可以认为单个交易无关紧要，但累积效应——或者至少 Twitter 希望如此——是人们将开始考虑查看 Twitter，看看有没有东西正在直播。如果 Twitter 能够在没有 NFL 比赛这样大的交易的情况下获得直播视频的知名度，这可能会长期帮助提升其其他指标——如注册用户、广告美元等。
