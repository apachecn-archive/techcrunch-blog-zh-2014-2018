Atomo 模块化电子系统就像电子产品的乐高积木

关于 Raspberry Pi 和 Arduino 的最困难的事情之一是弄清楚把所有的部分粘在哪里。虽然这两个系统单独工作都很好——你只需要一块板和一个互联网连接就可以获得很多乐趣——但添加一些像打印机和屏幕这样的小东西来制作有趣的项目也很有趣。这就是 Atomo 出现在的原因。

这种模块化套件来自总部位于香港的制造商乔纳森布福德，并得到了主要制造商 Arrow Electronics 的认证。

“Atomo 是 Arduino 的替代品，也是 Raspberry Pi 的配件，”布福德说。他希望用 Raspberry Pis 更容易地构建更复杂的项目，而不需要大量的电子专业知识。

有许多模块，包括 I/O 板、网络扩展板，甚至更大项目的电源。例如，下面的项目有四个 I/O 适配器和一个电源都连接到一个 Raspberry Pi。这意味着你可以用机器人做一些真正有趣的事情，甚至可以用一大块奶酪大小的包装进行水培。

该套件的早期价格为 39 美元，将于 6 月份上市。你可以购买更多的套装，并根据需要混合搭配。

“我们已经使所有控制器与 Raspberry Pi 上的 HAT 连接器兼容。这允许您在 Pi 上编程并更新控制器。或者直接把 Atomo 当模块化帽子用。布福德写道:“这非常适合 ROS 机器人或任何有处理或接口 Pi 但需要更多功率、IO 或实时控制的系统。”

就我而言，欢迎我们的树莓派和 Arduino 兼容机器人霸主。
