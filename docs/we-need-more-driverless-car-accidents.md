# 我们需要更多无人驾驶汽车事故 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/05/23/we-need-more-driverless-car-accidents/>

肖恩·杜布拉瓦茨撰稿人

More posts by this contributor

今年早些时候，一辆无人驾驶汽车在加州城市街道上发生了一起交通事故。没有人在这场小车祸中受伤，但这起事故确实标志着我们在通往无人驾驶汽车的道路上迈出了令人难以置信的一大步。这听起来可能违反直觉，但这次事故显示了自主技术在如此短的时间内已经走了多远。

这实际上不是传统意义上的意外——几周前实施的有意的软件更改可能是一个促成因素。如果幸运的话，我们将继续加快现实世界的实验，以及更多事故发生的可能性。

这起事件[部分是由一个微妙的软件更新引起的](https://web.archive.org/web/20221202032550/http://www.theverge.com/2016/2/29/11134344/google-self-driving-car-crash-report)，该软件更新在几周前在所有谷歌自动驾驶汽车上实施，使它们能够“拥抱车道的最右侧”，这是一种常见的社会规范，允许其他司机从左侧超车。根据[事故报告](https://web.archive.org/web/20221202032550/https://www.dmv.ca.gov/portal/wcm/connect/3946fbb8-e04e-4d52-8f80-b33948df34b2/Google+Auto+LLC+02.14.16.pdf?MOD=AJPERES)，谷歌自动驾驶汽车在车道内移动以绕过道路上的障碍物，这时它与一辆从后面驶来的公共汽车发生了接触。撞击发生时，汽车的行驶速度低于每小时两英里，而公交车的速度大约为每小时 15 英里。

谷歌正在测试这种拥抱车道右侧的新行为，这一事实表明，这项技术已经超越了简单地遵循道路规则，而是实际上更“像人类一样”驾驶——符合驾驶的社交元素。

像这样的事故是至关重要的学习练习。谷歌的无人驾驶汽车每天行驶超过 10，000 英里，此外还有每天 300 万英里的计算机模拟驾驶。但是这些现实生活中的测试是至关重要的。

> 完全了解我们的世界的唯一方法是通过冒险来探索它。

无人驾驶汽车不仅会改变方向盘后面的人或物——它们有可能改变我们开车的地方，进而改变我们的通勤、度假计划以及我们与家人和朋友的联系方式。无人驾驶汽车的定位是永远改变我们生活的世界……但在这之前，他们必须从根本上全面了解这个世界。而要完全了解我们的世界，唯一的方法就是通过冒险去探索它。

创新的历史是建立在推进前人工作的基础上的。从林德伯格 1927 年单人不间断飞越大西洋到查克·叶格 1947 年突破音障，我们一直以进步的名义揭开未知的神秘面纱。虽然在同一车道上移动可能看起来像是一个微小的细节，但像林德伯格和耶格尔在他们之前一样的自动驾驶汽车开发商正在从根本上改变交通运输。

技术的微妙变化将推动我们前进。从安全气囊到自动雨刷，从车载诊断系统到防撞系统，我们的汽车看起来与 30 年前有了很大的不同，这要归功于长期以来的渐进式创新。

创新成指数增长，但需要风险。第一批七名美国宇航员是军事试飞员。当他们在 1959 年被选中时，还没有人接近离开地球的大气层。当时，没有人知道这七个人中是否有人会成功。

这些创新者推动了我们所知的可能的前沿。当时我们在推动发现和探索，但我们也在与我们的冷战对手苏联进行太空竞赛。无数美国人冒着生命危险，一些首批进入太空的美国人做出了最大的牺牲，赢得了这场竞赛。

毫无疑问:今天，我们发现自己处于一场更加重要的竞赛中。全球每年有超过 100 万人死于交通事故，几乎所有这些死亡都是由人为错误造成的。自动驾驶汽车最有希望根除地球上最致命的力量之一，但首先我们必须突破极限。

自动驾驶汽车需要学会以正确的方式在正确的程度上具有侵略性。其中大部分可以在计算机模拟中确定，但有些必须在开放的道路上确定，那里的障碍物是动态的和复杂的。

一路上会有意外。从这些事件中，我们将获得宝贵的情报，这将推动我们进一步沿着创新曲线前进。

希望我们还没有看到上一次自动驾驶汽车事故；如果/当它们发生时，希望每个人都能看到它们的本来面目:推动我们跨越未跨越的边界的宝贵步骤。
