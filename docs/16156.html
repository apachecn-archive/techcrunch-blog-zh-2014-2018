Equifax泄露、 WannaCry 、 NotPetya 、NSA泄密以及更多网络事件——2017年对于黑客来说无疑是忙碌的一年，这再次说明了创新的网络安全解决方案在对抗网络威胁方面是多么重要。

在网络安全投资方面，仅次于美国，2017年是以色列网络安全初创公司又一个出色的一年，数十家公司成立，打破了融资记录，并实现了稳健的退出。2017年的数据还表明，以色列网络安全行业正在走向成熟，因为我们看到资金转向后期公司。

资本多，创业少

2017年，我们见证了60家新成立的网络安全初创公司在以色列涌现，比2016年成立的83家公司减少了28%。相反，2017年种子轮平均同比增长16%，从285万美元增长到330万美元。这是以色列连续第四年在种子阶段增加融资规模——随着我们加大对优秀网络安全企业家的投资，我们正在观察和推动这一趋势。

有人可能会认为，网络安全初创公司数量的减少是一个令人担忧的信号，警告这个行业正在衰落。我们认为这是一个行业成熟的积极迹象。网络安全是一个拥挤的空间，成千上万的公司在其中运营。CISOs每天都被数十种解决方案轰炸，每一种都承诺阻止下一次大的攻击。鉴于这一动态，“我也是”网络安全公司越来越难获得资金，因为投资者正在寻找更具差异化和更广泛的解决方案，以满足客户日益复杂的需求。

那些成功筹集资金的人往往会传达一个更宏伟的愿景，同时致力于打造需要更多资金的稳健产品。结果是越来越少的创业公司得到了更多的资本支持。对于希望建立可持续发展公司的企业家、支持这些想法的投资者以及需要更复杂解决方案的客户来说，这是一个积极的发展。

 

更年轻的团队和更多的女性创始人

2017年的数据显示，女性创始人的比例大幅上升。虽然仍然是男性占主导地位的领域，但2017年新成立的网络安全团队中有15%的创始人是女性，比前一年的5%有所增加。

与2016年一样，由经验丰富的团队(那些拥有十年以上管理或创业经验的团队)创办的创业公司和由经验不太丰富的企业家创办的公司几乎平分秋色。我们确实看到了由IDF毕业生领导的团队略有增加，这些创始人利用他们的相关军事经验在退伍后不久建立了网络安全公司。Axonius 就是这样一个例子，它是由IDF的精英情报单位8200的三名毕业生创建的，他们正在建立一个可见性和控制平台，以保护企业网络上的资产。

资金多，轮次少

纵观2017年以色列网络安全筹款，我们看到了一个熟悉的趋势，即更少的公司筹集了更多的资金。各阶段的以色列网络安全公司今年筹集了超过8.47亿美元，比2016年的6.89亿美元增长了23%。

进一步细分，种子和A轮的总体融资分别下降了14%和46%，而后期阶段的融资则大幅增加，B轮增长了218%，增长了165%。此外，以色列网络安全公司的投资轮次从2016年的72轮减少到2017年的63轮。

正如《T4》此前在TechCrunch 上报道的那样，融资轮次的减少和资本在各阶段的分布符合风险投资的全球趋势。过去几年，科技公司的风险交易数量有所下降。下降的主要原因是早期投资的下降，随后几轮的融资和数量水平仍然很大。

这在一定程度上是由风险投资公司投资于后期机会和积极跟进有潜力引领市场的公司所推动的。我们认为，以色列网络安全生态系统中也存在同样的动态，像 Deep Instinct 、德米斯图、perimerx、 Twistlock 和 Karamba Security 这样的公司筹集了大量B轮融资，像 SentinelOne 和 Cybereason 这样的公司今年也筹集了大量增长资本。

2017年网络安全趋势

2017年获得资助最多的网络安全领域包括网络安全、移动安全和漏洞与风险管理等传统IT类别。另一个突出的类别是物联网安全，随着新公司的出现和成熟公司的发展势头，该类别在各个阶段都有投资。

智能设备在日常生活中的激增催生了一个不断增长的物联网安全公司生态系统，在该行业中创建了子类别，专注于智能家居保护、保护联网和自动驾驶汽车以及医疗设备专用解决方案等特定用例。医疗设备保护是今年新出现的一个类别，我们已经看到了几家初创公司，包括 Medigate ，它们专注于帮助医疗保健组织保护自己免受越来越多的有针对性的攻击。

2017年网络安全退出

以色列网络安全公司2017年退出约13亿美元(不包括IPO)，平均退出估值1.3亿美元。2017年退出的网络安全公司的平均融资额略高于1700万美元，一家公司被收购平均需要5.5年。将这些数字与今年退出的以色列企业软件公司的数字进行比较，网络安全公司在各个领域的表现都更好——它们筹集的资金更少，获得的估值更高，退出的速度也更快。

今年突出的网络安全退出包括 Fireglass 和 Skycure (被赛门铁克收购)、 Hexadite (被微软收购)、 LightCyber (被帕洛阿尔托网络公司收购)和 Argus网络安全(被大陆集团收购)。

虽然2017年M&A的退出市场肯定是健康的，但也值得一提的是 ForeScout 以超过8亿美元的价格上市，这是以色列生态系统有能力产生大型独立网络安全公司的一个有意义的证据。

以色列网络安全生态系统的持续增长

2017年的全球网络安全入侵事件说明了创新在信息安全和防御中的持续作用。展望2018年，我们相信以色列的初创公司将继续利用当地巨大的人才库来构建面向全球市场的全面解决方案。

随着本地行业的成熟，我们预计最近的趋势将在2018年继续，创业公司的数量将减少，而大量资本将涌入后续几轮以推动增长和扩张。

以色列网络安全初创公司生态系统的不断成熟和发展将很快在今年1月在特拉维夫举行的以色列网络技术大会(Cybertech Israel)上得到充分展示，这是美国以外最大的网络技术年度会议。
