脸书开源了支持其纸质应用 TechCrunch 的动画引擎

原文：https://web.archive.org/web/https://techcrunch.com/2014/04/28/facebook-open-sources-the-animation-engine-that-powers-its-paper-app/amp/

脸书今天宣布开源 Pop 动画引擎，该引擎为其纸质应用的大部分用户界面提供动力。该代码现在可以在 GitHub 的 BSD 许可下获得。

造纸的历史与脸书于 2011 年收购的 Push Pop Press 的历史紧密交织在一起。得益于阿尔·戈尔的《我们的选择》的互动电子书版本，该公司在数字出版领域迅速成名。Push Pop Press 联合创始人 Mike Matas 和 Kimon Tsinteris 去了脸书，将他们的交互设计知识应用到脸书自己的产品中，结果(经过一些弯路)是一篇论文。

对于纸张，Tsinteris 在今天的博客中写道，公司决定使用动态而不是静态动画。Apple 通过其核心动画框架，使开发人员可以轻松使用线性、渐进、渐出和渐进渐出动画。Tsinteris 认为，用户现在期望一种“现实的、无处不在的物理体验”，这种体验将核心动画与捕捉手势速度结合起来，以更紧密地跟随用户的意图。简而言之，这就是 Pop 背后的目标。

Pop 在苹果的四个核心静态动画中增加了三个新的原语:spring、decay 和 custom。添加“弹簧”可以让元素弹跳，就像脸书的纸质应用一样。另一方面，“衰减”使运动缓慢停止，这完全取决于用户的输入。

正如 Tsingeris 指出的，该框架也被设计成可扩展的，所以开发者也可以插入他们自己的代码。为了使开发人员的生活更容易，Pop API 是严格模仿苹果自己的核心动画 API 的，因此只需做一些小的改动就可以在两者之间切换。

Pop 加入了脸书日益壮大的 iOS 开源工具库，包括 KVOController 、 Shimmer 和 Tweaks 。当然，这只是脸书整个开源库的一小部分，其中还包括类似于最初在 FriendFeed 开发的 HipHop 虚拟机、开放计算项目和 Tornado Python web 框架。
