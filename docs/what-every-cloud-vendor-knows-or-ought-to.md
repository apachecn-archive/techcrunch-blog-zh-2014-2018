# 每个云供应商都知道(或应该知道)的事情 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/05/15/what-every-cloud-vendor-knows-or-ought-to/>

上周，当我参加 Gartner 门户、内容和协作峰会时，当我与供应商，尤其是依赖订阅收入的云供应商交谈时，我注意到了一个巨大的变化。他们总是告诉我，他们必须帮助他们的客户取得成功，否则他们可以在合同结束时继续前进。

这使得销售的权力从卖方转移到买方，卖方正在采取措施确保他们的客户满意并成功使用他们的产品。

当然，确保客户成功一直是任何供应商的最大利益所在，但在订阅模式时代，这种观念变得更加明显。虽然从一个云供应商转移到另一个云供应商可能不是一件简单的事情，但淘汰和替换糟糕的内部软件可能比以前更容易，云供应商显然认识到了这一点。

最终结果是更多的牵手来确保成功，并设立首席客户官这样的高管头衔来表明他们对此有多认真。

事实上，上周当我与 Workshare 的首席执行官 Anthony Foy 交谈时，他说必须与客户合作，以确保他们以最佳方式使用该工具，与其他系统集成，并普遍利用该工具为他们提供的功能。work share 是一家帮助律师事务所和其他公司管理、保护和协作文件的公司。

为此，他的公司设立了一个名为首席客户官的 C 级职位，并提供两种类型的客户支持:一种是被动回答客户的询问，但他们也有一个客户成功部门，主动寻找与客户接触并帮助他们成功的方法。

“这是关于变革管理和有意识地决定建立客户参与流程，”Foy 解释道。

云存储和协作初创公司 Box 的高级副总裁兼企业总经理 Whitney Bouck 表示，一段时间以来，她一直在说，像她这样的云公司必须与客户建立合作伙伴关系，以确保客户的成功，因为尽管转换供应商通常需要付出一定的代价，但这比以前低得多，因此转换的障碍也比以前低得多。

布克说，虽然他们在 Box 发布 c 级职位时很小心，但他们确实有一位负责客户成功的高级副总裁乔恩·赫斯坦。布克告诉我，Box 为每个拥有 50 个以上许可证的账户分配了一个免费的客户成功经理。

他们还提供各种级别的咨询服务，帮助客户解决更复杂的集成和使用问题。

“供应商有巨大的责任确保客户满意和成功，”布克告诉我。

高级技术主管 David Goldberg 上周在 Gartner PCC 峰会上的一个关于向云过渡的会议上发言，他鼓励客户与其云供应商建立合作伙伴关系，并让他们知道他们想从产品中得到什么和需要什么。

“帮助他们了解您的需求并推广特色。如果你对自己的需求足够迫切，你就能更快地推动这些变化，”他解释道。

记住[云销售不一定要通过 IT](https://web.archive.org/web/20221006020611/https://beta.techcrunch.com/2014/05/05/as-cloud-upends-traditional-it-sales-channel-software-companies-eye-end-users/) 也很重要，云供应商认识到这种变化的销售动态。因此， [Scott Kuport 和 Preethis Kasireddy 在最近的 Andreessen Horowitz 博客文章](https://web.archive.org/web/20221006020611/http://a16z.com/2014/05/13/understanding-saas-valuation-primer/)中指出，对良好的客户支持的需求更加明显。

他们在博客中写道:“在许多情况下，这些部门与 SaaS 客户支持代表的关系甚至比他们与自己的 IT 部门更密切。”。

值得注意的是，Kupport 和 Kasireddy 在他们的博客中写道，更换供应商可能不是一件小事，因为它不再属于 It 领域。他们写道:“由于 SaaS 的使用是在部门层面，一家公司的用户往往比传统软件产品的用户多得多，这使得转换成本更高。”

因此，在销售发生之前很久，客户就会关注这些供应商对他们的支持，如果他们不能提供足够的帮助，他们甚至可能永远也不会进行销售。

这就是为什么每个云供应商都知道客户支持是获得和留住客户的关键，而对于云供应商来说，一旦获得客户，留住他们对他们的业务模式至关重要。
