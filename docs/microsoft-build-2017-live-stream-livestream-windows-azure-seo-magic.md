# 在此观看 Microsoft Build 2017 主题演讲直播

微软将于本周举行年度构建开发者大会，该公司将于今天上午以其首次主题演讲拉开此次活动的序幕。你可以在这里看直播。

主题演讲定于西海岸上午 8 点在**开始，东海岸上午 11 点，伦敦下午 4 点和巴黎下午 5 点开始。**

这是一次开发者大会，所以你不应该期待新的硬件设备。该公司刚刚发布了 [Surface 笔记本电脑](https://web.archive.org/web/20230326102020/https://techcrunch.com/2017/05/02/the-999-surface-laptop-is-microsofts-expensive-answer-to-chromebooks/)，并即将更新 [Surface Pro](https://web.archive.org/web/20230326102020/https://techcrunch.com/2017/05/05/microsoft-teases-new-surface-event-in-shanghai-on-may-23rd/) 。这就是为什么 Build 可能会专注于 Windows 10、Azure 等所有东西。

第 1 天通常更关注消费者技术，而第 2 天的主题演讲通常围绕开发人员工具和云。我们预计今年也将如此，这两天我们都将在地面上。你可以在 TechCrunch 上关注我们所有的报道[。](https://web.archive.org/web/20230326102020/https://techcrunch.com/events/microsoft-build-2017/)
