# 创业工作室 eFounders 筹集了 530 万美元，因为它无法获得足够的 SaaS

欧洲初创公司 studio [eFounders](https://web.archive.org/web/20221226063742/http://efounders.co/) 正在从企业家和家族办公室筹集额外的 530 万美元(€500 万)。eFounders 一直在支持一些非常有前途的软件即服务初创公司，在初创公司能够独立完成伟大事业之前的第一年左右，它提供人力和财力资源。作为交换，创业工作室获得初始股权的很大一部分。

事实上，我已经介绍了很多这样的公司，因为它们往往对特定的企业问题有聪明的解决方案。投资组合创业公司有 [Mailjet](https://web.archive.org/web/20221226063742/https://techcrunch.com/2015/07/15/mailjet-raises-another-11-million-to-scale-its-unified-marketing-and-transactional-email-service/) ，TextMaster，[提](https://web.archive.org/web/20221226063742/https://techcrunch.com/2013/12/18/mention-becomes-a-full-fledged-media-monitoring-tool-with-buffer-integration/)，[前端](https://web.archive.org/web/20221226063742/https://techcrunch.com/2016/05/26/front-grabs-10-million-to-bring-email-to-the-next-level/)， [Aircall](https://web.archive.org/web/20221226063742/https://techcrunch.com/2016/09/19/aircall-raised-8-million-for-its-cloud-phone-system-for-teams/) ，illustrio， [Spendesk](https://web.archive.org/web/20221226063742/https://techcrunch.com/2016/06/15/spendesk-makes-expenses-and-company-cards-suck-less/) 和 Forestadmin。

随着今天的融资轮，这家初创工作室计划继续全速前进。我们的目标是在未来几年内创建八家新的创业公司。每年四家新创公司听起来是一个雄心勃勃的目标，所以让我们看看 eFounders 模式是否可以扩展。

eFounders 的投资者包括 Stupeflix 首席执行官尼古拉斯·斯蒂格曼(Nicolas Steegmann)、Train 船长创始人让-丹尼尔·海底平顶山(Jean-Daniel)、Resto-in 和 Stuart 创始人克莱门特·伯努瓦(Clément Benoit)等数十人。这也是建立一个商业天使网络的好方法，他们将能够投资未来的 eFounders 创业公司。

我记得早期的 eFounders，当时工作室仍在学习创建软件即服务创业公司的最佳方式。虽然专注于这个利基市场很有趣，但这是一段漫长而坎坷的旅程。

但我最近对 eFounders 目前的节奏印象深刻。虽然有些项目比其他项目更艰难，但 eFounders 一直在尝试许多不同的事情，并复制不同创业公司的最佳做法，经常同时管理多个创业公司。现在让我们看看他们有什么。
