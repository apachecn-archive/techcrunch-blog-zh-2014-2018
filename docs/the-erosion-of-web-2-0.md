# Web 2.0 的侵蚀

## 我们是如何迷失方向的...又找到了

现在去想象似乎有些古怪，但网络最初的愿景并不是信息高速公路。相反，它是一份只给我们提供我们想要的新闻的报纸。这是 20 世纪 90 年代末提出的核心论点，比尔·盖茨等思想家预言了这一点——比尔·盖茨期待一部[美丽、定制的《前路》](https://web.archive.org/web/20230313203918/https://www.amazon.com/Road-Ahead-Bill-Gates/dp/0670859133)——克利夫·斯托尔只看到了[的蛇油](https://web.archive.org/web/20230313203918/https://www.amazon.com/Silicon-Snake-Oil-Thoughts-Information/dp/0385419945/ref=sr_1_1?s=books&ie=UTF8&qid=1528200648&sr=1-1&keywords=silicon+snake+oil)。当时，这是那些思想家认为可能的互联网最引人注目的用途。这一概念——我们将被一个蜂巢大脑所宠溺，这个大脑旨在当我们需要知道它时向我们展示我们需要知道的东西——继续快速发展，直到它被用户生成内容(UGC)的概念所取代。UGC 是一场相关的运动，它摧毁了看门人，几乎摧毁了网络世界的礼仪。

这就是 Web 2.0 的弧线:从新闻组或 IRC 中的一对一对话走向全球报纸。此外，这创造了一百万个一对多的对话，目标是定制的粉丝、支持者和更多情况下的巨魔受众。这种变化给了我们今天所拥有的:一个破碎的棱镜，它将人类折射成除了黑色或白色以外的任何颜色。UGC 曾经是一个伟大的想法，即任何人都可以像摇滚明星一样受欢迎，但现在它变成了一个不可逆转的免费游戏，迫使品牌和广告商重新思考如何接触受众。毕竟，在一个 UGC 网站上，宝洁公司在星巴克里某人对穆斯林的种族主义咆哮旁边做 Downy 织物柔顺剂的广告并不好玩。

然而，硅谷采纳了这些概念，并建立了自我表达的货币化粪坑。脸书、Instagram、YouTube 和 Twitter 是愤怒文化的最大受益者，其持续更新带来的眼球推动了它们的进一步增长。这些网站是 Web 2.0 最黑暗的缩影，是一支箭，击中了我们最深处、最珍贵的机构，让我们丧失了善良和远见。

因此，当广告商面临随机仇恨言论的直接货币化或客户隐私的侵蚀时，他们会选择后者。脸书创造了长相相似的受众，让广告商在非常精细的层面上向人类的某个子集销售，这一举动不断地从一个网站到另一个网站向我们发送相同的鞋类广告，直到我们都确定自己疯了。在进一步拯救我们的理智的幌子下，我们邀请永远在线的麦克风进入我们的家，它们可以观察我们的听力和浏览习惯，并向我们推销它们。我们向 Ancestry 和 23andMe 这样的公司放弃了我们的 DNA，这个决定人类可能很快就会后悔。我们与每个人分享一切，怀着巨大的希望，希望我们进化成人类——网络人——会让我们成为人类。

这并没有发生。

于是钟摆又摆了回来。GDPR 虽然没有牙齿，但它给每一个曾经攻击你的电子邮件或在网上跟踪你的垃圾邮件发送者敲响了警钟。此外，苹果即将在 Safari 中推出的 [cookie 控制软件应该会让那些无处不在的广告消失，迫使广告商向无差别的人群而不是单一个人销售。在一个由互联网作为所有知识(正确或不正确)的字体的具体化和一条基于网络的政治眼镜蛇的起源所定义的时代，这显然是冰冷的安慰。但这是一个开始。](https://web.archive.org/web/20230313203918/https://techcrunch.com/2017/09/15/apple-defends-new-ad-tracking-prevention-measures-in-safari/)

我们目前正处于技术的间歇期，这是一个由诚挚的友情和“去他妈的系统”朋克摇滚 X 世代烘烤而成的蛋糕，但点缀着千禧年的实用主义和对手工艺术的渴望。随着我们走出 UGC 和 Web 2.0 时代，我们将看到旧的方式被抛弃，旧的模式被打破，旧的隐私侵犯被逆转。虽然我不会说区块链会拯救我们所有人，但普遍的加密和完全的数据控制将为真正控制我们的个人生活以及开始基于研究的最低收入铺平道路。我们应该能够向出价最高的人出售我们的观点、想法，甚至我们的 DNA，一旦贪婪的 Web 2.0 秃鹫被赶走，我们将发现自己处于一个有趣的新世界。

作为一名科技汽车人，我确信我们正朝着正确的方向前进。然而，我们正在经历在克林顿和传真机时代我们谁也想象不到的转折，而且还会有更多的转折。然而，幸运的是，我们正在走出上次的大滑坡。
