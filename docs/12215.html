微软将关闭Wunderlist，转而支持其新应用To-Do TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/04/19/microsoft-to-shut-down-wunderlist-in-favor-of-its-new-app-to-do/?ncid=rss

2015年，微软收购了广受欢迎的移动待办列表应用Wunderlist，现在，随着新应用“To-Do”的发布，它正在为用户准备它的最终消亡，今天宣布。微软表示，新应用由Wunderlist背后的团队开发，并将在未来几个月内引入该应用最受欢迎的元素。该公司还补充说，它不会关闭Wunderlist，直到它有信心“将Wunderlist的最佳内容纳入待办事项”

如果你希望Wunderlist能得到某种形式的缓刑，微软清楚地表明了它即将到来的灭亡。在一篇博客文章中，它白纸黑字地陈述了自己的计划:“我们将让神童退役，”如是说。

与此同时，微软正在鼓励Wunderlist用户进行转换，提供一个导入器，将你的列表和待办事项从Wunderlist导入到待办事项中，这些项目现在可以在其他微软产品中使用，如Exchange和Outlook。

微软的待办事项计划之前被泄露，当时该公司被发现正在测试一个新的应用程序，然后是代号为Cheshire项目的。

在泄露事件发生时，这是一个相当简单的待办事项应用程序，允许用户跨平台创建列表、添加项目、设置提醒和同步列表。唯一有趣的功能是，它能够提供添加到列表中的任务建议——这已经转化为待办事项的“智能建议”功能。

根据微软的公告，待办事项会在一个名为“我的一天”的屏幕中启动，该屏幕提供了一个需要在今天完成的项目列表。您可以通过访问“智能建议”功能(点击灯泡图标即可获得)来进一步定制该列表，该功能将提供您可能想要添加到列表中的内容的建议。

例如，如果你昨天没有完成某件事，但也许应该完成，这可以作为一个任务建议出现。微软解释说，通过这个功能，你还可以查看前一天的待办事项，以及查看到期的任务和其他即将到来的任务。

更重要的是——至少对微软来说——待办事项是为了与微软Office集成而构建的。这款应用是基于Office 365开发的，它的第一次集成是在Microsoft Outlook中。您的Outlook任务将与待办事项同步，因此您可以跨设备访问和管理它们。

为了增加安全性，待办事项的数据在传输和存放时也会加密。今天，it管理员可以通过Office 365管理中心启用To-Do的预览版，消费者可以在 iPhone 、 Android 、 Windows 和 web 上使用。

ZDNet向微软澄清，这款应用并不像最初关于发布的博客文章所暗示的那样，仅仅是针对Office 365用户的。相反，该公司证实，任何拥有微软账户的人都可以使用待办事项，即使他们没有Office 365计划。

然而，待办事项应用程序目前还不能满足所有神童用户的需求。正如许多用户意识到的那样，一些平台尚不支持待办事项，包括Mac、iPad和Android平板电脑。列表共享也不可用。但微软表示，这些将随着与微软服务的其他集成及时推出。

一名微软员工在原始博客帖子的评论部分甚至对用户要求亚马逊Alexa集成表现出了一些热情，这可能很有趣。他们指出，对工作账户的支持将在未来几周内出现在网上的待办事项中，并给出了iPad应用程序的时间框架为“未来几个月”
