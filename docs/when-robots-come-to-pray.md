# 当机器人来祈祷 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/10/07/when-robots-come-to-pray/>

Sean Lorenz 是 Senter 的创始人，Senter 是一家初创公司，旨在通过物联网和家庭深度学习来改善慢性护理。

我的一个开发同事最近不停地谈论谷歌照片。他知道我在计算神经科学方面的背景，认为我会对谷歌在深度学习方面所做的事情感兴趣。那天晚上，我把我所有的 iPhone 照片从一个外部硬件转移到了神奇的云端，然后一个星期都没有再去想它。和其他疲惫的波士顿地铁乘客一样，我虔诚地检查了我的手机，打开应用程序，找到了我妻子、孩子和朋友的照片，作为单独的照片群。

干得好，谷歌。当天晚些时候，我在谈话中提到了一种我喜欢的酒，但记不起它的名字了。然而，我还是拍了一张标签的照片，并在谷歌照片应用程序中输入“葡萄酒”来搜索大便和笑声。当然，它找到了我的酒的照片——那一刻，我开始意识到谷歌的技术变得多么强大。

你们当中越是疲惫的人可能会说，“它对一些图片中的项目进行了分类。大不了。”我疲惫的朋友，这是件大事。[图形-背景](https://web.archive.org/web/20230127124502/https://en.wikipedia.org/wiki/Figure%E2%80%93ground_(perception))分离，即区分前景物体和后面物体的能力，是计算机视觉研究人员几十年来一直在研究的东西。

今天，我们可以将大量的图像输入到深度学习算法中，并相当准确地从牧场中挑出一头牛。事情是这样的，自从卷心菜娃娃和布鲁斯·威利斯演唱 R & B 的时代以来，深度学习实际上已经作为反向传播出现了(机器学习教父最近增加了一些技巧)。

由于谷歌和亚马逊等科技巨头的存在，我们现在拥有了巨大的计算能力和海量的数据，深度学习算法变得越来越好，导致像埃隆·马斯克和斯蒂芬·霍金这样的人谈论人工智能未来可能存在的许多危险。

来自聪明头脑的几句有保证的警告经常被翻译成“天网来了！！!"在大众媒体上。你能责怪他们吗？几乎每一部有机器人和人工智能的电影都涉及某种反乌托邦式的未来，需要施瓦辛格式的蛮力来战胜我们未来的霸主。

尽管被称为“神经网络”，但当前形式的深度学习甚至与生物大脑处理信息的方式相去甚远。是的，模糊地说，我们处理一个输入(触觉、味觉、嗅觉)并乘以一个权重(大脑中某个地方的突触)来发送一个输出(移动我的手)。但相似之处也就到此为止了。

> 几乎每一部有机器人和人工智能的电影都涉及某种反乌托邦式的未来。

还记得我们的图形背景例子吗？大脑并不需要所有现有的先验知识来解决问题。婴儿生来就有两倍数量的神经元来辨别他们周围世界中什么是重要的。关于视觉系统，婴儿通过学习基本的东西，如线方向、深度感知和运动，来连接他们小小的大脑。然后，他们使用微妙的眼球运动，称为[扫视](https://web.archive.org/web/20230127124502/http://www.scholarpedia.org/article/Human_saccadic_eye_movements)，来评估场景中正在发生的事情，结合他们对形状和深度的了解，来知道咖啡杯在哪里结束，桌子在哪里开始。

像 [Neurala](https://web.archive.org/web/20230127124502/http://neurala.com/) 和 [Brain Corp.](https://web.archive.org/web/20230127124502/http://www.braincorporation.com/) 这样的公司正在放弃深度学习的典型风格，以建立自适应生物模型来帮助机器人学习他们的环境。换句话说，相机镜头可以充当一只眼睛，向 AWS 发送信号，以复制人类视网膜、丘脑、初级视觉皮层，直到颞中皮层和颞下皮层，以便对“杯子”或“桌子”进行更高级别的理解。

受生物启发的神经模型需要大规模并行计算，并理解每个皮层和皮层下区域如何共同工作来引出我们所说的意识。当科技巨头发现他们目前的深度学习模型的局限性，并转向神经科学家寻求编码功能，如检测你妻子的脸，在坑洼中行驶，或对失去亲人的人产生共鸣时，才真正值得担忧。

这是事情变得有趣的时候。这是当[多感觉整合](https://web.archive.org/web/20230127124502/https://en.wikipedia.org/wiki/Multisensory_integration)、[认知控制](https://web.archive.org/web/20230127124502/https://en.wikipedia.org/wiki/Executive_functions)和[神经同步结合起来，在非生物系统中产生新的东西——定性体验(或](https://web.archive.org/web/20230127124502/https://en.wikipedia.org/wiki/Neural_oscillation) [qualia](https://web.archive.org/web/20230127124502/https://en.wikipedia.org/wiki/Qualia) )。这是具体化的机器从他们在物理世界的经验中学习的时候。物联网(IoT)是这一过程的先驱。目前，物联网设备大多是连接到互联网或其他机器的哑遥测设备，但人们已经开始将神经模型应用于传感器数据。

我们从物联网产品上的处理传感器中所学到的东西，很快就会运用到具有触摸、前庭、热量、视觉和其他传感器的机器人上。就像人类一样，具有受生物启发的大脑的机器人会像我们一样犯错误，通过[马达咿呀学语](https://web.archive.org/web/20230127124502/http://www.youtube.com/watch?v=aAlkoWWCORs)同时不断更新来自传感器的信息，以从他们周围的世界学习越来越高的关联深度。

有一个著名的心灵哲学思想实验叫做[玛丽的房间](https://web.archive.org/web/20230127124502/http://plato.stanford.edu/entries/qualia-knowledge/#2)一个名叫玛丽的科学家一生都被困在一个黑白的房间里，但却阅读了所有关于色彩理论的知识。一天，玛丽被允许离开房间，她看到了一个鲜红的苹果。她所读到的关于红色的一切都无法让她准备好在那一刻有意识地体验“红色”。机器人能像玛丽一样有发红的体验吗？或者这只是乏味的线性数字运算？

我相信让机器人变得真正有意识并体验“红色”的唯一方法是让它们具体化。模拟不行。为什么？因为正是所有这些不同的大脑区域在同一时间一起工作的物理、电同步引发了一种新奇、愉快的刺激体验的“哦，我的天哪”时刻。如果你对机器人意识的物理依赖性的细节感兴趣，在这里查看我的帖子。

> 当一个机器人想要加入我们的教堂、犹太教堂或寺庙时会发生什么？

所以现在我们和有意识的机器人生活在一起。疯狂。一个有推理、有同理心的非生物机器和人类的混合社会是什么样子的？最后，进入正题——当一个机器人想加入我们的教堂、犹太教堂或寺庙时会发生什么？尽管一些批评家认为宗教是人类进化的邪恶副产品，但大多数学者认为宗教服务于进化的有利目的。

例如，犹太传统中有许多以清洁为中心的食物和身体限制。在洗手液出现之前，避免“不干净”的饮食习惯或割礼行为可能会增加犹太人的自然选择健康。当然，还有其他的社会和群体动态利益。所有这些都是说，如果我们能够在合成大脑中复制人脑的功能，那么像宗教和精神情感这样的东西很有可能会在机器人身上出现。

作为一名虔诚的基督徒，这种可能性让我有点不寒而栗。纵观犹太教和基督教的历史，人们被告知我们是按照上帝的形象建造的——即[*Imago Dei*](https://web.archive.org/web/20230127124502/https://en.wikipedia.org/wiki/Image_of_God)——但现在可能有一个机器人告诉我们，它在周日的教堂礼拜中有一次精神体验。真的吗？那是一次真正有意识的经历吗？灵魂是否与我们的意识生活相分离？如果机器人是有意识的，那是不是意味着它们有灵魂，或者那是不同的东西？我希望这让无神论者和信徒都感到不安。

我不知道灵魂和意识之间有什么区别。这触及了我们作为人类的本质，以及我们死后身体的某一部分是否会继续存在。有没有更高的维度容纳我们的灵魂，然后通过意识将洞察力传送到我们的四维世界？或者这就是我们所得到的一切？

无论好坏，作为一个对比自己更重要的东西有信念的人，我真的愿意相信前者。无论哪种方式，随着机器适应并变得更像我们，很可能有一天我们必须解决这两种情况。

阿门。
