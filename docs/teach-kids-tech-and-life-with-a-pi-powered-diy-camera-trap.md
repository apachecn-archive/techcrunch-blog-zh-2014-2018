# 用 Pi 驱动的 DIY 相机陷阱教孩子科技和生活

如今，号称能帮助教孩子编程的产品并不缺乏，无论是卡诺的 DIY 电脑套装(T1)，还是 T2 的机器乌龟棋类游戏(T3)，或是 T4 的任何一种可编程机器人(T5、T6、T7)。旨在让孩子们在室内摆弄和学习的扭杆玩具风靡一时。

英国初创公司 [Naturebytes](https://web.archive.org/web/20230129092631/http://naturebytes.org/) 给这个科技加教育的领域带来了不同的侧重点。它旨在将黑客技术与激发孩子们在户外的好奇心的使命结合起来。鉴于今年儿童词典已经抛弃了与自然相关的词汇，如橡子和毛茛，并增加了科技术语，如宽带和“剪切和粘贴”，可能有迹象表明，科技有可能成为“数字原住民”的全部消费。

Naturebytes 刚刚启动了一个 [Kickstarter](https://web.archive.org/web/20230129092631/https://www.kickstarter.com/projects/279364224/naturebytes-wildlife-cam-kit-digital-making-for-wi) 用于拍摄野生动物照片的树莓 Pi 驱动的相机捕捉套件。这个想法是为了启发孩子们关于电子和编码可以做什么，同时也给他们学习和体验自然的欲望。将技术放在户外也可能是让孩子们感兴趣的一种方式，否则他们可能会喜欢在户外跑来跑去。此外，随着物联网的兴起，在自然环境中嵌入技术是我们将会看到的更多东西。

防风雨的相机陷阱套件设计用于抵御恶劣天气。它装有一个 Raspberry Pi 模型、一个微处理器(也可以使用其他更强大的 Pi)、电池组、Pi cam 和一个红外传感器——这种设置可以对任何经过的野生动物进行运动敏感的照片(和视频)拍摄。相机以套件的形式提供，因此孩子们可以将它们放在一起，并在学习过程中学习。还有扩展功能的空间——例如，该套件可以通过 Wi-Fi 链接升级，以自动上传野生动物照片。

除了众筹这些硬件套件，Naturebytes 还打算建立一个网络平台，用户可以分享他们用相机拍摄的图像，并参与公民科学项目。对于已经拥有 Pi 和 Pi cam 并希望自己 3D 打印相机外壳的更高级的制造商来说，套件起价为 45 英镑。除了圆周率之外，其他都包括在内的套件的价格提高到 85 英镑，或者所有零件的价格提高到 95 英镑。

在撰写本文时，该团队已经筹集了超过 10%的众筹目标，他们的活动还剩一个月。所以还不错。他们的目标是总认捐额约为 45，000 美元，以便生产和运输他们的第一批工具包——预计运输时间为 12 月。
