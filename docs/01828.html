Twitter 推出了 Fabric，这是一个开发人员在其平台上构建应用的工具包

在今天早上 Twitter 的飞行会议上，迪克·科斯特罗推出了 Fabric，这是一套针对开发者的 API，旨在鼓励他们在 Twitter 平台上构建应用和服务。

在引入新 API 的过程中，科斯特洛表示，Twitter“不想从什么对 Twitter 最有利的角度出发，而是从什么能帮助(开发者)最高效的角度出发。”

Fabric 建立在 Twitter 的社交平台上，但也结合了 Twitter 被收购方 Crashlytics 的崩溃报告软件和被收购方 MoPub 的社交定位功能。但这不仅仅是制作让你登录 Twitter 的应用程序，这样他们就可以在你的用户面前推送广告。Twitter 移动平台总监 Jeff Seibert 展示了一个类似 TestFlight 的解决方案，用于向测试人员分发预发布版本，而不需要创建帐户，允许您发送带有特定链接的特定版本。

当然，Twitter 社交平台并没有被完全忽视。现在有一个原生的移动 Twitter SDK，允许开发人员创建 Twitter 客户端，显示来自该服务的丰富媒体，并在移动应用程序中集成通用 Twitter 登录。该解决方案还帮助开发者整合 MoPub，通过最高竞价网络路由广告来增加收入。
