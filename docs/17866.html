Verisart将区块链认证带入全球艺术品拍卖市场

大约三年前，韦里萨带着一个不同寻常的想法出现在舞台上。它将利用区块链的不变性来创建对艺术品市场的认证。从字面上验证艺术品是它所声称的。虽然这个想法看起来不错，但执行起来可能会很困难。但创始人兼首席执行官罗伯特·诺顿(Robert Norton)之前曾试图将发展缓慢的传统艺术市场带入21世纪，他坚持下来了，现在已经达成了三笔重大交易，第一笔是在拍卖市场。

Verisart现在推出与 Paddle8 合作开发的“P8Pass”，将区块链认证和鉴定服务引入拍卖市场。

P8Pass将在paddle8.com举办的所有策划和义卖活动中提供给买家、委托人和合作伙伴。每张P8通行证都包含详细的出处信息，并作为比特币区块链上的唯一指纹。

Paddle8联合创始人兼总裁Alexander Gilkes表示，P8Pass“有可能大大降低艺术品和收藏品交易的交易成本，并通过纳入新的买家来扩大市场，这些买家受到P8Pass区块链产品提供的认证流程简单性和交易安全性的鼓励，这些风险传统上是富裕消费者远离艺术品交易的主要原因。”

诺顿说:“与Paddle8的合作以及Native SA的战略支持进一步扩大了我们作为领先艺术家和画廊以及现在拍卖合作伙伴的区块链认证标准首选供应商的平台覆盖范围。”

Verisart一直很忙。它与 ArtSystems 合作，将区块链认证带到全球领先的画廊，使其成为第一个集成此类基于区块链的证书的艺术品库存管理平台。它还与DAC合作推出了名为“青铜链”的青铜雕塑标志。这结合了使用一个标志邮票与Verisart的区块链。

其他玩家现在正在进入这个不断增长的市场。 Codex Protocol 是一家也将art放在区块链的新公司。
