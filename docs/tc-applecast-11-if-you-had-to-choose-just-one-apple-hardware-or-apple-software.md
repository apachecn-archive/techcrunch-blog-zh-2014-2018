# TC AppleCast 11:如果你只能选择一个——苹果硬件还是苹果软件？

在本周的节目中，我们既要讨论蒂姆·库克的长篇简介，也要讨论史蒂夫·乔布斯的全传。我们还报道了一些具体的公告，包括[React](https://web.archive.org/web/20230129061729/https://techcrunch.com/2015/03/26/facebook-open-sources-react-native/)的开源，脸书(可能比 UIKit 更好)的 UIKit 替代品，以及[苹果](https://web.archive.org/web/20230129061729/https://techcrunch.com/2015/03/24/apple-acquires-durable-database-company-foundationdb/)收购数据库公司 FoundationDB，以及这可能意味着什么。

但是等等，还没完呢！我们还应对了本周浮出水面的 4 英寸 iPhone 6c 的新传言，并关注了苹果申请的智能包装新专利可能带来的可能性，该专利允许用户在将设备从盒子中取出或与销售人员交谈之前设置、保护甚至购买设备。

最后，我们问一个由约翰·格鲁伯(John Gruber)在 2012 年的一次思维实验中首次提出(或者至少是最引人注目地提出)的问题:在所有其他因素都相同的情况下，被迫做出选择，你会选择运行其他(Android/Windows 软件)的苹果硬件，还是运行 Windows/Android 硬件的苹果软件？[无论如何请告诉我们](https://web.archive.org/web/20230129061729/mailto:darrell@beta.techcrunch.com)，并在下面找出我们的选择。
