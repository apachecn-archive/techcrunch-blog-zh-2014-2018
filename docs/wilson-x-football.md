# 威尔逊的联网足球是一块 200 美元的智能猪皮

我觉得我身体的一部分(大约 75%的我，在这一点上)开始抱怨在他的时代扔旧足球已经足够有趣了，我们没有必要，嗯，把游戏游戏化。但是我在骗谁呢？我们也玩电子游戏。嘿，如果你能让扔球变得更有趣，那你就更有力量了。

今年早些时候，体育用品巨头威尔逊向 select press 展示了其联网足球。像去年的篮球一样，[威尔逊 X 联网足球](https://web.archive.org/web/20221005195900/http://www.wilson.com/en-us/football/footballs/wilson_x)或多或少是一个标准的运动球，内置传感器以增强体验。

板上的电子设备记录了关键的足球投掷数据，如距离、速度、旋转速度、螺旋以及球员握球的表现。这些信息自然会通过蓝牙传输到移动设备上。还有一个低功耗的板载电池，应该可以持续大约 500 小时的播放，这显然意味着 20 万次。

游戏化真正开始于应用程序，它将玩家与朋友和更大的排行榜进行排名。现在[开始接受预购](https://web.archive.org/web/20221005195900/http://www.prnewswire.com/news-releases/wilson-sporting-goods-co-introduces-the-wilson-x-connected-football-300310019.html)，价格高达 200 美元。它将在一个月后上市，这并非巧合，而是 2016 年 NFL 赛季的开始。
