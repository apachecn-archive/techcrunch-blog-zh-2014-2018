在虚拟助手的战斗中，微软的Cortana刚刚得分。公司今天推出了新功能，这将允许助理提醒你关于你没有明确告诉她跟踪的事情。具体来说，Cortana现在能够通过弹出提醒建议来帮助你记住你在电子邮件中承诺要做的事情。

这项建议提醒功能于一年前首次发布，是与微软研究部门联合开发的。它的工作归功于机器学习，其中包括让Cortana扫描你的电子邮件，以便及时产生任务建议——当然，只有在用户允许的情况下才能这样做。

微软此前表示，扫描本身是在本地进行的。只有当用户同意设置提醒时，提醒的详细信息才会发送到微软的服务器。

Cortana不仅能找到你已经指定了特定时间的句子(比如下午6点)，还能理解我们做出时间承诺时使用的语言。例如，它可以提取像“一天结束”这样的短语，甚至像“EOW”这样的缩写

微软表示，要使用该功能，您只需要像往常一样继续发送电子邮件。Cortana会在看到你许下承诺后马上介入。当您指定截止日期后，Cortana会在截止日期前提醒您，并将其保存在行动中心。微软指出，其他建议的提醒也会出现在Cortana主页上。

Windows Insider计划的用户已经能够在今天的公开发布之前测试这些主动提醒。该公司表示，在试用期内，微软增加了一些增强功能。例如，它现在链接到提醒邮件，并在截止日期前发送通知。它还能更好地理解你的电子邮件，因此建议也更准确。

Windows 10的建议提醒现在已经在美国上线，iOS和Android支持将在“未来几周”推出。目前支持Outlook.com和Office 365工作和学校电子邮件地址，但将来会添加其他电子邮件服务。
