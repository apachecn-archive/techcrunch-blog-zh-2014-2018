# Twitter 为品牌提供了一种推广聊天机器人的新方式

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/05/23/twitter-is-giving-brands-a-new-way-to-promote-their-chatbots/>

Twitter 已经找到了自己的方式来参与日益增长的聊天机器人趋势。该公司今天宣布了一项面向广告商的新功能，允许 Twitter 上的企业推广广告，旨在通过直接消息传递将消费者带入个性化体验——包括与聊天机器人的互动。例如，一个品牌正在运行一个机器人，在你回答关于场合或口味的问题后，它可以帮助你推荐鸡尾酒。这些互动通过一种推广的“直接消息卡”开始，这种新的可定制的商业卡被称为“直接消息卡”。

鸡尾酒机器人来自烈酒品牌 Patrón Tequila，该品牌使用新的卡片风格推出了自己的推广推特活动，从而拉开了推特新功能的序幕。

![](img/45fe2379a63b96dbce1c782b0443d0b3.png)

直接消息卡可以使用图像或视频以及多达四个行动号召按钮进行定制，旨在鼓励 Twitter 用户进入某个品牌的 DMs。(不好意思。)

这些卡片不是要把人们推向帮助解决客户服务问题的机器人，或鼓励从有问题的品牌购买，这是许多脸书聊天机器人的重点。相反，它们是让人们通过私人信息体验与品牌互动，这意味着有趣，而不是交易。

另一个可以使用这张卡的例子是[温迪今年早些时候推出的](https://web.archive.org/web/20230329061655/https://blog.twitter.com/2017/marchmadness-is-happening-on-twitter)三月疯狂支架生成器。在这种情况下，自动化的体验让球迷通过 DM 创建他们的支架，然后在整个锦标赛中再次查看他们的表现以及赢得的奖品。

利用这种新形式的企业还可以在消费者与有问题的机器人互动后，通过推文提示来提示消费者重新分享体验。

![](img/915c4260bf32d6941077d0ceef6ed048.png)

消费者是否会大规模积极接纳机器人仍有待观察。

这一领域的许多早期尝试都没有达到消费者的期望。例如，脸书的第一批聊天机器人[通过伪装成人来惹恼人们](https://web.archive.org/web/20230329061655/https://techcrunch.com/2017/03/29/facebook-group-bots/)——让客户感觉他们正在与基于消息的呼叫中心电话菜单版本进行交互。他们的购物或信息共享体验通常并不比访问企业自己的网站更容易。

尽管聊天机器人的未来不确定，但许多公司已经投资了这个领域。微软去年为 Skype 发布了[机器人框架和聊天机器人，最近又为 LinkedIn](https://web.archive.org/web/20230329061655/https://techcrunch.com/2016/03/30/microsoft-is-bringing-bots-to-skype-and-everywhere-else/) 增加了对[聊天机器人的支持；除了脸书的](https://web.archive.org/web/20230329061655/https://techcrunch.com/2017/01/19/linkedin-desktop-update/) [Messenger](https://web.archive.org/web/20230329061655/https://techcrunch.com/2017/03/29/facebook-group-bots/) 之外的消息应用也在增加机器人，比如 [Telegram 新增加的对购物机器人的支持](https://web.archive.org/web/20230329061655/https://techcrunch.com/2017/05/18/telegram-launches-chatbot-payments/)；谷歌正在通过其聊天机器人分析平台 Chatbase 进军聊天机器人领域的后端。

Twitter——尽管有许多自动发推特的机器人——似乎只是在用聊天机器人试水，看看这是否是品牌想要采用的东西，并想通过 Twitter 广告进行推广。

相反，该公司早期在品牌和 DMs 方面的努力更侧重于客户支持互动这一更大的业务。该公司已经推出了一些功能，如[欢迎信息](https://web.archive.org/web/20230329061655/https://techcrunch.com/2016/11/01/twitter-unveils-new-options-for-customer-service/)、[快速回复](https://web.archive.org/web/20230329061655/https://blog.twitter.com/2016/speed-up-customer-service-with-quick-replies-welcome-messages-in-direct-messages)、[自定义配置文件](https://web.archive.org/web/20230329061655/https://techcrunch.com/2017/02/22/twitter-debuts-custom-profiles-for-businesses-so-people-dont-think-theyre-talking-to-bots/)、[位置共享](https://web.archive.org/web/20230329061655/https://techcrunch.com/2017/04/03/twitter-adds-location-sharing-for-business-offering-customer-support-over-direct-messages/)，以及其他工具，让企业通过自动化来增强他们的客户支持互动。但像欢迎信息或快速回复这样的事情并不真正被认为是机器人。

新的直接消息卡现在进入测试阶段，只对 Twitter 广告客户开放。
