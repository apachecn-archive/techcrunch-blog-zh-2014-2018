区块链是新的 Linux，而不是新的互联网

原文：https://web.archive.org/web/https://techcrunch.com/2017/05/28/double-double-cryptocoin-bubble/?utm_source=Venture

加密货币的蓬勃发展令人难以置信。去年，比特币价格上涨了 7 倍，达到 2500 美元。三周前，领导以太坊首次发布的令人敬畏的 Vinay Gupta 发表了一篇题为“100 美元的以太意味着什么？“从那以后翻了一倍。随着两大硬币的出现，太多的替代硬币价值暴涨。 ICOs 一手抓着钱一手抓着二头肌。到底是怎么回事？

( eta :在我第一次写那篇文章后的 48 小时内，这些价格大幅下跌，但今年仍然上涨了很多。)

正在发生的事情是，某种诱人的叙事已经占据了主导地位。这种说法的最极端版本是，今天的加密货币就像 1996 年的互联网:不仅仅是新技术，而是一种被大多数人轻视或忽视的全新技术，在过去几年中，它的力量和影响力缓慢而微妙地增长，并将以令人震惊的速度和巨大的影响爆发出全球相关性和重要性。

(以免你认为我夸大了这一点，我有一天得到了一个公关宣传，字面上是这样开始的:“区块链的 1996 年互联网时刻到了，”作为吹捧一个 3300 万美元的 ICO 的序言。嘿，朋友之间 3300 万美元算什么？现在几乎可以肯定我们正处于一个加密硬币泡沫中。)

我理解这个故事的吸引力。我不是区块链怀疑论者。我已经着迷地写了六年关于加密货币的文章了。多年来，我一直在吹捧和赞美区块链的力量，他们如何有潜力让互联网再次去中心化和无许可化，并赋予我们所有人对自己数据的权力。我是像比特币这样无许可货币的忠实信徒。我称以太坊的首次推出为历史性的一天。

但我不禁要看看加密货币今天的状态，想知道实际价值在哪里。我不是指投机者的金融价值；我指的是对用户的效用价值。因为如果没有人想实际使用区块链协议和项目，那些应该反映其价值的令牌最终…嗯…毫无价值。

尽管内部纷争不断，但比特币作为无需许可的全球货币非常有用，有理由成为全球储备和结算货币。它的匿名后代如 ZCash 为最初的比特币命题增加了价值。(同样，由于前面提到的持续冲突，莱特币现在在技术上领先于比特币。)以太坊作为开发者的平台是非常成功的。

但尽管如此，在比特币推出 8 年后，中本聪仍然是唯一一个建造了一个区块链的创造者，相当多的普通人实际上想使用这个。(以太坊很棒，维塔利克·布特林和古普塔一样，是一个不折不扣的梦想家，但它仍然是开发者的工具/解决方案/平台。)似乎没有其他基于区块链的软件创新有任何真正的风险——坚持得到普遍认可，更不用说普遍使用了。

恕我直言，另一位真正的信徒弗雷德·威尔逊(Fred Wilson)——显然，这是一种巨大的尊重——在这种巨大的繁荣中，他引用了“稀有的佩佩卡”作为一个有趣的现代区块链应用程序的主要例子。我的意思是，如果这是艺术的状态…

也许我错了；也许稀有佩佩会成为下一个 Pokémon GO。但另一方面，也许在区块链空间，投机与实际价值的比率从未如此之高，这就是说很多。

一些人认为这项技术如此神奇，如此具有革命性，如果投入足够的资金，杀手级应用和协议就会出现。这简直不能再倒退了。我不反对代币销售，但它们应该遵循“如果你造出足够好的东西，投资者就会蜂拥而至”，而不是“如果足够多的投资者涌向我们，我们就会造出足够好的东西。”

一个可靠的团队致力于一个有趣的项目，但没有达到产品市场的契合度，应该能够筹集到几百万美元——或者，如果你喜欢，几千个比特币——然后，一旦他们的成功被证明，他们可能会出售另一批现在更有价值的代币。但是几乎没有任何用户，几乎没有任何技术的项目，能筹集到几千万？这有点像由蛇油制成的泡沫……很可能会引来美国证券交易委员会严厉而无情的打击。

那个诱人的故事！96 年的互联网！我知道。但是听我说完。也许认为今天的区块链就像 1996 年的互联网是完全错误的。当然，所有的类比都有缺陷，但它们是有用的，它们是我们思考的方式——也许这里还有另一个更准确、更有说服力的类比。

我提出一个相反的说法。我告诉你，今天的区块链不像 1996 年的互联网；它们更像 1996 年的 Linux 。这绝不是挖苦——但是，如果是真的，这对于那些希望从区块链应用程序和协议的主流使用中获利的人来说，有点像丧钟。

与当今密封的、围墙花园式的、堆栈式的互联网相比，去中心化的区块链解决方案要民主得多，在技术上也更有吸引力。类似地，开源 Linux 比当时统治计算机领域的微软和苹果操作系统更加民主，在技术上更有吸引力。但是除了一小撮黑客之外，没有人使用它。它太笨重了；太复杂；太反直觉；需要跨越太多的障碍 Linux 的肮脏秘密是，事实上，主流解决方案对大多数人来说都很好。

听起来熟悉吗？今天，有很多工作都集中在区块链索引的分散分布式存储上； Storj 、 Sia 、 Blockstack 等。这是令人惊叹的、开创性的工作……但为什么一个已经习惯使用 Box 或 Dropbox 的普通人会转向 Storj 或 Blockstack 呢？集中式解决方案对他们来说很好，而且，因为是集中式的，他们知道出问题时应该找谁。尤其是块堆栈，它不仅仅是“仅仅”存储…但是它为普通用户解决了什么令人信服的难题呢？

与 Linux 的相似之处是惊人的。Linux 比当时的替代品便宜得多，功能也强大得多。这似乎令人难以置信，难以置信的破坏性。尼尔·斯蒂芬森将 20 世纪 90 年代的操作系统比作汽车。窗户是一辆嘎嘎作响的柠檬旅行车；MacOS 是一辆密封的大众甲壳虫。然后，不可思议地… 不可思议地…还有:

Linux，它就在隔壁，而它根本不是一个企业。它是一堆房车、蒙古包、帐篷和网格状圆顶建筑，设置在一个场地上，并根据共识组织起来。住在那里的人正在制造坦克。这些不是老式的铸铁苏联坦克；这些更像美国陆军的 M1 坦克，由太空时代的材料制成，从一端到另一端都塞满了尖端技术。但它们比陆军坦克更好。它们经过了改装，永远不会抛锚，足够轻便和灵活，可以在普通街道上行驶，而且不会比一辆微型汽车消耗更多的燃料。这些坦克正在现场以惊人的速度被生产出来，大量的坦克沿着路边排列，钥匙插在点火器上。任何想要的人都可以爬进一辆车，免费把它开走。

顾客不分昼夜成群结队地来到这个十字路口。90%的人会直接去最大的经销商那里购买旅行车……他们甚至不会看其他经销商一眼。

我告诉你，就像昨天的普通消费者不会使用 Linux 一样，今天的普通消费者也不会使用比特币和其他区块链应用，即使比特币和其他建立在区块链基础上的应用在技术上和政治上都很惊人。)我认为“这是消费者广泛使用比特币、Ripple、Stellar、ZCash、分散以太应用等的一年。]”可能类似于“[Ubuntu | Debian | Slackware | Red Hat |等的年份”。]在桌面上。”

请注意:这不是一个不屑一顾的类比，或者以任何方式低估了该技术的潜在最终重要性！T2 有 20 亿台活跃的安卓设备，每一台都运行 Linux 内核。当它们与服务器(也称为“云”)通信时，它们与巨大的、仓库大小的数据中心通信，这些数据中心充斥着无数的 Linux 机器。Linux 是非常重要和有影响力的。大多数现代计算可以说是 Linux 到 Linux 的。

很容易想象区块链和加密货币会有类似的未来。引用我朋友 Shannon 的话:“它(区块链理工大学)看起来肯定会有一个类似 Linux 的采用过程:当它在关键的幕后应用程序中获得主导地位时，将会有一系列注定失败的尝试来使它成为商业上可行的消费产品。”

但是如果你 1996 年的投资主题是普通人将在未来十年内集体采用 Linux 这看起来一点也不疯狂——那么你将陷入一个巨大的伤害世界。Linux 并没有因为普通人使用它而变得重要。相反，它成为推动下一波互联网的商品基础设施。

很容易想象几十个分散的区块链的交织网络如何以及为什么能够在一年又一年的时间里慢慢成为类似类别的关键基础设施:作为储备/结算货币，作为当今金融行业大量产品的替代品，作为名称空间(如域名)，作为分布式存储系统的幕后实现，等等。…而普通人仍然幸福地意识不到他们的存在。甚至很容易想象它们被商品化了。以太坊气成本太高吗？没问题；只要把你的分布式系统切换到另一个更便宜的区块链就行了。

所以不要告诉我这就像 1996 年的互联网，不是没有令人信服的证据。相反，当加密货币的价格开始跟踪基于其区块链的应用和协议的潜在价值时，请叫醒我。因为在此期间，由于缺乏这种价值，我很抱歉地说，相反，我们似乎在谈论分散的数字郁金香。

披露，因为这似乎是必要的:我大多避免任何加密货币的任何金融利益，无论是隐含的还是明确的，长期的还是短期的，这样我就可以不带偏见地写它们了。不过，我确实拥有一枚比特币，这是我几年前购买的，因为我在为一家(现已倒闭的)基于比特币的公司提供咨询时，觉得不拥有任何比特币很傻。
