# 特斯拉承诺不会起诉任何使用其技术的人

根据首席执行官埃隆·马斯克在公司博客上的一篇帖子，特斯拉已经开源了其电动汽车技术专利。

马斯克解释说，该公司最大的目标之一是推动整个电动汽车市场向前发展，如果它让其他人更难进入这个领域，它就无法完成这个目标:

> 特斯拉汽车公司的创立是为了加速可持续交通的到来。如果我们为制造引人注目的电动汽车扫清了道路，但随后却埋下知识产权的地雷来阻止他人，我们的行为就与这一目标背道而驰。特斯拉不会对任何真诚希望使用我们技术的人提起专利诉讼。

就在几天前，马斯克[透露，他希望与其他汽车制造商分享特斯拉超级充电站](https://web.archive.org/web/20230215191351/https://techcrunch.com/2014/06/09/tesla-wants-to-open-its-supercharger-standard-to-other-electric-car-makers/)背后的技术，以创建一个每个电动汽车车主都知道他们在旅行时可以依赖的标准。

这是一个有趣的策略。帮助他人在性能和里程上迎头赶上，使得特斯拉对那些专门购买电动汽车市场的人来说不再是一个有吸引力的选择。

但如果电动汽车全面改善，可能会扩大考虑购买电动汽车的人群。凭借其行业领先设计的声誉，特斯拉在吸收这一增长的重要部分方面处于有利地位。

此举让人想起了 Twitter 的创新者专利协议，[早在 2012 年](https://web.archive.org/web/20230215191351/https://techcrunch.com/2012/04/17/twitter-announces-innovators-patent-agreement-gives-ip-control-to-engineers-and-designers/)宣布。Twitter 承诺，未经专利实际发明人的许可，它不会利用自己的专利起诉竞争对手。它只会保留专利，以确保不会被拥有大量专利的人起诉。

在这个时候，似乎每个人都在起诉对方，试图在竞争中获得某种优势，很高兴看到有影响力的公司决定退出竞争。
