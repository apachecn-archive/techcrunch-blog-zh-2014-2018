# 越南新网络安全法引发对限制言论自由的担忧 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2018/06/12/vietnams-new-cyber-security-law-draws-concern-for-restricting-free-speech/>

在越南政府通过一项法律，承诺对网上言论自由进行更严格的限制后，包括谷歌、脸书和推特在内的大型科技公司表达了主要担忧。

本周通过的新法规加强了政府审查互联网的立场，招致国际特赦组织的谴责，称其在越南“没有给人们自由言论的安全空间”。[亚洲互联网联盟(AIC)](https://web.archive.org/web/20221205111249/https://www.aicasia.org/)——一个代表脸书、谷歌、Twitter、LinkedIn、Line 和其他公司的组织——进一步警告说，这将损害该国数字经济的发展。

据报道，新的网络安全法的要点之一是禁止互联网用户组织或训练他人从事反国家目的，传播虚假信息，破坏民族国家的成就或团结[。](https://web.archive.org/web/20221205111249/https://e.vnexpress.net/news/news/vietnam-says-cybersecurity-law-needed-to-ensure-national-security-3762377.html)

“这一决定对越南的言论自由具有潜在的破坏性后果。大赦国际在一份声明中补充道:“在这个国家极度压抑的氛围中，网络空间是一个相对的避难所，人们可以去那里分享想法和意见，而不用担心受到当局的指责。”。

互联网审查对越南来说并不新鲜，但这部法律增加了该国采取行动的潜力。在过去的一年里，随着一系列逮捕事件的发生，人们的关注度已经很高了。在这些事件中，博客作者因在网上讨论环境问题、政治问题和其他问题而被监禁。

除了限制言论自由，网络法还对外国互联网公司施加压力，这些公司现在将被要求在越南境内运营一个当地办事处并存储用户信息。目前，就谷歌和脸书而言，越南用户的数据存储在新加坡和香港等海外地点。

谷歌和脸书都拒绝置评，但他们是 AIC 的一部分，该机构发表声明谴责新法律。

AIC 在一份声明中写道:“数据本地化的规定，对影响言论自由的内容的控制，以及当地办公室的要求，无疑会阻碍美国第四次工业革命实现 GDP 和就业增长的雄心。”。

“不幸的是，这些条款将严重限制越南的数字经济，抑制外国投资环境，损害当地企业和中小企业在越南国内外蓬勃发展的机会，”该组织补充说。

越南人民也表达了对新法律的不满。彭博报道称，周日投票前发生了示威游行。
