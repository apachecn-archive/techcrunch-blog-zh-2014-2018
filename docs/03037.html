YC 支持的 GitLab 就像 GitHub，但对于企业来说，你知道，就像 GitHub Enterprise

GitLab 称自己是开源的 T2 GitHub T3，公司可以在他们自己的服务器上运行。像 GitHub 一样，它提供了一个开源的 git 库。也和 GitHub 一样，有一个猫生物吉祥物。

“但由于某些地理原因，有时你希望你的集线器在你自己的服务器上受到保护，在你自己的防火墙后面，”GitLab Sytse Sijbrandij 的荷兰联合创始人在电话中告诉我，他和他的联合创始人 Dmitriy Zaporozhets 在加州山景城的公寓里。当然，GitHub 的企业版也提供这个功能。

我不是程序员，所以如果我遗漏了什么，请原谅我，但是除了 GitHub 已经存在了更长时间，因此有一个更大的社区，这两者似乎很难区分。就在它的主页上，GitLab 甚至声称它“比 GitHub 更好”，它不得不为写了一篇博文解释两者之间的区别。

“更好是因为 GitLab 版本更容易缩放和修改，”Sijbrandij 向我解释道。例如，你可以在企业版许可证上对 GitLab 进行修改，或者安装不同类型的网络服务器。你也可以用 GitHub Enterprise 做这件事。

我向程序员求助。在 Reddit 、 Quora 和 Product Hunt 上有几十个关于 GitLab vs GitHub 的讨论。一个朋友给我看了 Slack 上的一个私人讨论，也讨论了这两个问题。

支持 GitLab 的共同点是，它可以很好地与 DigitalOcean 服务器集成，并且是更便宜的选择。那些支持 GitHub 的人说，从长远来看，GitHub 更有利于支持和社区，而且价格也不会高太多。但是双方似乎都同意这两者之间有很强的相似性。

GitLab 的起价为每位用户每年 19.90 美元。数字海洋最受欢迎的计划是每月 10 美元。GitHub 不会在其网站上分享企业的前期定价信息。然而，来自 GitHub 用户的 Quora 帖子指出 GitHub 的企业定价是每月 21 美元。

GitLab 声称 NASA、SpaceX、O'Reilly Media、美国电话电报公司、Comcast 和 IBM 是 GitLab 的用户，并表示自 2011 年推出以来，该软件已被下载了 50 万次。

更新: GitLab 已经收购了开源主机和企业 Git 管理的竞争对手 Gitorious。

“在 Gitorious，我们看到越来越多的组织采用 GitLab，”Gitorious 的首席执行官在 GitLab 博客上表示。“由于来自内部客户的收入减少，经营免费的 Gitorious.org 不再是可持续的。”

GitLab 计划在 6 月 1 日关闭 Gitorious 网站。
