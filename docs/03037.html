YC支持的GitLab就像GitHub，但对于企业来说，你知道，就像GitHub Enterprise

GitLab称自己是开源的T2 GitHub T3，公司可以在他们自己的服务器上运行。像GitHub一样，它提供了一个开源的git库。也和GitHub一样，有一个猫生物吉祥物。

“但由于某些地理原因，有时你希望你的集线器在你自己的服务器上受到保护，在你自己的防火墙后面，”GitLab Sytse Sijbrandij的荷兰联合创始人在电话中告诉我，他和他的联合创始人Dmitriy Zaporozhets在加州山景城的公寓里。当然，GitHub的企业版也提供这个功能。

我不是程序员，所以如果我遗漏了什么，请原谅我，但是除了GitHub已经存在了更长时间，因此有一个更大的社区，这两者似乎很难区分。就在它的主页上，GitLab甚至声称它“比GitHub更好”，它不得不为写了一篇博文解释两者之间的区别。

“更好是因为GitLab版本更容易缩放和修改，”Sijbrandij向我解释道。例如，你可以在企业版许可证上对GitLab进行修改，或者安装不同类型的网络服务器。你也可以用GitHub Enterprise做这件事。

我向程序员求助。在 Reddit 、 Quora 和 Product Hunt 上有几十个关于GitLab vs GitHub的讨论。一个朋友给我看了Slack上的一个私人讨论，也讨论了这两个问题。

支持GitLab的共同点是，它可以很好地与 DigitalOcean 服务器集成，并且是更便宜的选择。那些支持GitHub的人说，从长远来看，GitHub更有利于支持和社区，而且价格也不会高太多。但是双方似乎都同意这两者之间有很强的相似性。

GitLab的起价为每位用户每年19.90美元。数字海洋最受欢迎的计划是每月10美元。GitHub不会在其网站上分享企业的前期定价信息。然而，来自GitHub用户的Quora帖子指出GitHub的企业定价是每月21美元。

GitLab声称NASA、SpaceX、O'Reilly Media、美国电话电报公司、Comcast和IBM是GitLab的用户，并表示自2011年推出以来，该软件已被下载了50万次。

更新: GitLab已经收购了开源主机和企业Git管理的竞争对手Gitorious。

“在Gitorious，我们看到越来越多的组织采用GitLab，”Gitorious的首席执行官在GitLab博客上表示。“由于来自内部客户的收入减少，经营免费的Gitorious.org不再是可持续的。”

GitLab计划在6月1日关闭Gitorious网站。
