哥伦比亚广播公司在其独立流媒体服务CBS All Access TechCrunch中增加了来自CBSN的24/7新闻

原文：https://web.archive.org/web/https://techcrunch.com/2017/08/01/cbs-adds-247-news-from-cbsn-to-its-standalone-streaming-service-cbs-all-access/?utm_source=shootr&utm_medium=referral

哥伦比亚广播公司(CBS)面向脐带切割者的流媒体服务 CBS All Access ，正在通过CBSN向其订阅服务添加直播、锚定新闻报道和原始报道，从而完善其节目阵容。24/7新闻频道将于8月3日开始通过CBS.com向CBS All Access的台式电脑和笔记本电脑用户提供，并能够通过其他CBS All Access平台观看，包括移动设备和连接的电视，将于今年秋季推出。

哥伦比亚广播公司的在线新闻服务，CBSN，于2014年11月首次推出，目标是增加哥伦比亚广播公司新闻内容向数字平台的分发。该网络指出，随着脐带切割占据主导地位，这项服务继续增长，与去年同期相比，第二季度的流量增长了38%。

鉴于其新闻报道已经可以通过CBSNews.com获得，以及CBS新闻应用程序用于移动和联网电视设备，CBSN被加入该网络更大的流媒体服务是有道理的。毕竟，你不应该为了看CBSN的新闻而在哥伦比亚广播公司的应用程序之间切换。

在CBSN加入之前，CBS All Access有许多其他新闻节目，包括通过其附属机构的直播电视节目，为其服务带来当地和全国的新闻报道。它还播放新闻节目，如“哥伦比亚广播公司今晨”和“哥伦比亚广播公司晚间新闻”，并提供点播访问其节目，如“60分钟”和“48小时”

除此之外，CBSN还将提供现场新闻和原创报道，包括最近在黄金时间播出的夏季系列节目《CBSN:在任务中》，该节目于7月31日在CBS电视网推出。

将CBSN整合到流媒体服务的举动也正好赶在9月24日周日CBS All Access的旗舰电视剧《星际迷航:发现》的首播日期之前

这部备受期待的新剧旨在利用《星际迷航》的粉丝来推动人们对独立流媒体服务的兴趣，否则在一个高质量的原创节目可以通过网飞、亚马逊、Hulu、HBO和其他网站提供给有线电视用户的时代，这种服务更难销售。

然而，随着CBSN的加入，CBS All Access变得更加引人注目，因为它利用了许多脐带切割者现有阵容中缺失的组件之一。除非你也订阅了Sling TV或DirecTV Now等直播电视服务，否则你很难像以前的有线电视用户那样收看新闻节目。

虽然像Sling这样的直播电视服务有更强大的频道选择，但CBS All Access是一种负担得起的方式，只需访问当地直播新闻、晚间新闻，现在还可以访问24/7新闻。

如果你直接从CBS购买，CBS流媒体服务本身的费用为每月6美元或每月10美元，分为广告支持和无广告两个等级。通过iOS设备获得的订阅收入又增加了1美元，因为哥伦比亚广播公司旨在弥补苹果公司削减的订阅收入。

除了CBSN与CBS All Access的整合，该公司还宣布了一项分销协议，这将使CBSN首次成为多渠道视频分销商。

专注于体育的流媒体服务fuboTV最近增加了非体育频道，将CBSN作为一项更大交易的一部分，提供CBS频道，包括CBS、CBS体育网、CW和Pop。
