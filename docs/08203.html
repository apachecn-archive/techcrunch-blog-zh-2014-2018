为引起人的注意时所发的声音...分享你的网飞密码还是可以的 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/07/11/psst-its-still-okay-to-share-your-netflix-password/?ncid=rss

联邦上诉法院本月发布了一项裁决，根据《计算机欺诈和滥用法案》,密码共享属于联邦犯罪。这让许多美国人犹豫了，因为共享网飞和 HBO GO 等在线服务的密码是这些天相当常见的活动——现在，由于这一裁决，这违反了联邦法律。但是现在还不要惊慌:事实证明，网飞仍然不介意你分享你的密码。

如果该公司不打算使用联邦法律起诉他们的客户，你可以继续分享密码，而不用担心你很快就会发现自己陷入法律困境…至少现在是这样。

当然，如果它后来决定这种行为不再酷，事情可能会改变，这要感谢这个裁决树立的可怕先例。

话虽如此，我们从那些熟悉网飞在账户共享上的立场的人那里了解到，该公司并没有改变其之前声明的立场。也就是说，账户共享基本上还是没问题的——尤其是在家庭成员之间，他们使用共享作为购买升级账户的替代方式。

网飞首席执行官雷德·哈斯汀斯在今年早些时候的 CES 上澄清了流媒体服务对账户共享的看法，据我们所知，同样的观点今天仍然适用。

黑斯廷斯当时说:“我们喜欢人们分享网飞，不管是两个人睡在一张沙发上，还是 10 个人睡在一张沙发上。”。“这是一件积极的事情，而不是消极的事情。”

他后来还补充说，该公司将账户共享视为其更广泛的用户获取战略的第一步。换句话说，孩子分享父母的账户，最终搬出家门，自己订阅。

黑斯廷斯在 1 月份的 CES 上向记者解释说:“随着孩子们在生活中不断前进，他们喜欢控制自己的生活，因为他们有了收入，我们看到他们单独订阅。”“这真的不是问题。”

这位首席执行官当时没有对非家庭成员共享账户发表评论，但似乎该公司对消费者友好的立场——目前——首先专注于让人们沉迷于网飞，希望他们以后会订阅。

网飞拒绝置评。

与此同时，HBO 的立场有点模糊。该公司目前运营两项流媒体服务——面向有线电视和卫星电视用户的认证服务 HBO GO 和面向脐带切割者的 HBO NOW。2014 年，首席执行官 Richard Plepler 表示 HBO 将账户分享视为“下一代观众的绝佳营销工具”

然而，这些评论是在 HBO NOW 的 2015 年开播之前发表的。鉴于今天的 HBO GO moochers 可能是 HBO NOW 的潜在订户，该公司可能不再对分享如此“冷静”。

但它会走得更远以至于起诉它的用户吗？大概不会。这对打造其品牌不利。另外，HBO NOW 仍然是一项相对较新的服务，也是 HBO 试图扩展的服务。

HBO 告诉我们，它对裁决不予置评。

公司更有可能限制信息流，而不是起诉用户

然而，如果任何一家公司最终决定是时候打击账户共享，他们不太可能在法庭上起诉用户，而是会考虑技术解决方案。他们的软件可以通过多种方式阻止账户共享，或者至少让账户共享变得更加困难，比如使用地理定位或限制并发流媒体会话。

事实上，两家公司今天都已经对并发流媒体进行了限制——网飞的基本账户一次只允许一个流媒体。它的标准账户限制你两个。

有趣的是，HBO NOW对允许多少个并发流含糊其辞，称其“类似于 HBO GO”但看一眼 HBO GO FAQ 也没有提供具体的允许流数量，只说: “出于安全原因，对并发流的数量有限制。如果您共享您的 HBO GO 凭据，您可能会由于太多的活动流而无法观看内容。”

该公司此前曾表示，它将允许在最多三台设备上播放流媒体，但常见问题的措辞表明，它可能不会严格执行这一规定。

无论如何，如果你参与账户共享，你可能不会被联邦起诉。你知道，现在。
