# Rokos 核心操作系统把你的树莓皮变成了比特币节点

Rokos Core T1 现在是第四个版本，是 Raspberry Pis 的磁盘映像，可以将你的单板电脑变成一个完整的 T2 比特币 T3 节点。该系统将允许你持有一个比特币钱包，并通过网络挖掘、发送和接收比特币。

要明白，如果没有专门的硬件，这个东西本质上是积极浪费电力和/或持有比特币的方式。然而，因为它是一个完整的 BTC 节点，所以你在学习一点采矿知识的同时，也帮了比特币世界一个忙。

然而，最棒的是，Rokos 拥有完整的 GUI 和点击式界面。这意味着那些不想使用命令行的人可以玩比特币。例如，早期的系统使用 [CGMiner](https://web.archive.org/web/20221004202448/https://github.com/bitmaintech/cgminer) ，这是一个功能齐全但很难的采矿系统，而 [21 的采矿钻机](https://web.archive.org/web/20221004202448/https://beta.techcrunch.com/2015/11/18/mining-and-learning-with-the-21-bitcoin-computer/)对新来者来说是令人困惑的。如果比特币挖矿不能盈利，它至少应该有趣和/或容易。

所以下载你的磁盘镜像，烧掉它，开始走向加密货币霸权之路吧。聪会感谢你的。
