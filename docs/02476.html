Lizard Squad 的“Lizard Stresser”服务运行在被入侵的路由器上

安全专家 Brian Krebs 分析了 Lizard Stresser，这是一个由所谓的 Lizard Squad 黑客集体创建的攻击工具，并被吹捧为 webmins 的测试，这些 web mins 需要了解他们的服务在胁迫下会发生什么。他的发现？攻击计算机的网络实际上由不安全的和被入侵的家用路由器组成。

这是用于在圣诞节假期关闭 Playstation Network 和 Xbox Live 的网络。在一组安全研究人员的帮助下， Krebs 发现蜥蜴小组控制了一个由被黑的路由器和其他商业服务器组成的大型僵尸网络。

写道克雷布斯:

￼
将易受攻击的系统转化为 stresser 机器人的恶意代码是俄罗斯安全公司 Dr. Web 在 11 月首次记录的一种相当原始的恶意软件的变种，但该恶意软件本身似乎可以追溯到 2014 年初(谷歌的 Chrome 浏览器应该会自动翻译该页面；对其他人来说，谷歌翻译的网络博士论文在这里。僵尸网络不完全是由家用路由器组成的；一些受感染的主机似乎是大学和公司的商用路由器，无疑还涉及其他设备。僵尸网络中路由器的优势可能与僵尸网络传播和扫描新的潜在主机的方式有关。但是恶意软件没有理由不能传播到由 Linux 操作系统驱动的广泛设备，包括桌面服务器和联网摄像头。

他还提出了一些很好的建议来确保路由器的安全性，包括更改默认密码——这是显而易见的——并使用 OpenDNS 来防止恶意网络呼叫到您的路由器。

这意味着，这个将 Xbox 和索尼黑客行为称为营销噱头的团队，本质上是将被黑客入侵的机器作为一种服务出售。

简而言之，数百个，甚至数千个被入侵的路由器正被用来攻击世界各地的服务器，不管是好是坏，这个消息应该让我们暂停一下。如果有的话，我们可能都应该向前支付，并为他们保护我们不太懂技术的朋友的路由器，以准备进一步的恶意软件攻击。
