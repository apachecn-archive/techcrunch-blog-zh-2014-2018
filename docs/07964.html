微软将 SharePoint 引入 iOS TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/06/21/microsoft-brings-sharepoint-to-ios/?ncid=rss

微软今天早上宣布为 SharePoint 用户推出新的移动应用，旨在将公司基于 SharePoint 的内部网门户及其内容带到用户的智能手机和平板电脑上。微软表示，该应用最初在 iOS 上可用，这意味着它可以在 iPhone 和 iPad 上运行，但它将在年底前登陆 Android 和 Windows 平台。

公司上个月表示这样一个应用程序正在工作中，当时它详细说明了它对 SharePoint 未来的计划。超过 200，000 个组织使用 SharePoint，SharePoint 通常是“内部网”的同义词，内部网是企业在防火墙后运行的内部门户，员工在这里共享新闻、文件、项目协作等。然而，缺少的是一种更现代、移动优先的方法，无论企业的最终用户使用哪种设备，都可以使用 SharePoint 内容。

[gallery ids="1341017，1341016，1341015，1341014"]

随着新的 SharePoint 应用程序的推出，微软宣称它正在把“内部网放进你的口袋里”这意味着用户可以通过移动设备访问公司新闻和公告、人员、网站、内容等。该应用程序可以在内部或混合场景中与 Office 365 中的 SharePoint Online 或 SharePoint Server 2013 或 2016 配合使用。

该应用程序具有一个站点选项卡，允许您访问您经常访问的 SharePoint 站点，因此您可以查看最近的活动、文件，并访问这些目标上的这些文件、列表、页面和其他内容。

该公司指出，该应用程序还可以与其他微软移动应用程序配合使用。比如你在 app 中点击一个 Office 文档，它就会启动相应的 Office 手机 app；当您在团队网站上查看文档库时，您将被带到用于 iOS 的 OneDrive 移动应用。

“链接”选项卡允许您查看由 SharePoint 管理员管理的公司网站和门户的完整列表，而“人员”选项卡允许您浏览同事列表以及查看他们的个人资料。

搜索是另一个关键功能，因为它允许你通过关键字定位资源，并通过网站、文件和人员过滤结果，以及由微软的机器学习技术 Microsoft Graph 提供的“内容推荐”。

尽管可以公开发布而不是测试版，但微软表示仍在接受对新应用的反馈，可以通过应用内 Shake、 UserVoice 和 Twitter 帖子提交给 @SharePoint 和 @Office365 。
