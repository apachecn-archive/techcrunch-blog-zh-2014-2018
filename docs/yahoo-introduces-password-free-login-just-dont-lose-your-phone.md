# 雅虎推出免密码登录——只是不要丢失你的手机

雅虎希望结束你对记忆密码的依赖——或者创造可以被猜测或黑客攻击的垃圾密码——在它[推出](https://web.archive.org/web/20230316161016/http://yahoo.tumblr.com/post/113708272894/a-new-simple-way-to-log-in)一个新的“按需”系统，当你需要登录时发送一次性密码。

这种新方法旨在提高安全性，减少你的雅虎账户被黑客攻击的可能性。在某些方面，它做到了这一点。无数人在各种服务上重复使用记忆深刻的密码，包括他们的电子邮件账户。它们不仅通常在本质上相当容易被黑客攻击([随机密码是首选](https://web.archive.org/web/20230316161016/https://techcrunch.com/2015/01/20/this-list-of-2014s-worst-passwords-including-123456-is-embarrassing/)，而且它们本身就不安全，因为如果/当被破解时，它们会打开你的大部分数字身份，或者你的*整个*在线状态。

按需密码在你登录后是不可用的，它是为了消除密码链/潜在的多米诺效应而设计的，因为它们是特定于你的雅虎账户的。但是，有一个相当重要的警告:如果你丢失了你的手机，拥有它的人有一张进入你的电子邮件的票。

在某些情况下，如果你在锁屏时收到短信通知，即使你的手机被锁定，按需密码也会显示出来。因此，如果你丢失了它，捡到它的人甚至不需要知道你的密码，只要他们知道你的 ID 就可以进入你的雅虎账户。

一个更好的方法可能是两步密码，在输入您的主密码后，您将通过短信设置一个临时密码。这是基本的双因素身份认证。雅虎告诉 CNET 这是“消除密码的第一步”，所以我们~~希望~~期待在这方面会有更多的进展。

雅虎关注移动安全是正确的，这是一个普通互联网用户长期不知道最佳实践的领域。[甚至那些更精明的人也被抓了出来](https://web.archive.org/web/20230316161016/http://www.theverge.com/a/anatomy-of-a-hack)。最好的方法仍然是密码管理器，如 1Password 或 LastPass，并意识到风险。

在相关新闻中，该公司还展示了其端到端电子邮件加密插件的早期一瞥，该插件[正在西南偏南与谷歌](https://web.archive.org/web/20230316161016/https://techcrunch.com/2014/12/17/googles-end-to-end-email-encryption-tool-gets-closer-to-launch/)合作。
