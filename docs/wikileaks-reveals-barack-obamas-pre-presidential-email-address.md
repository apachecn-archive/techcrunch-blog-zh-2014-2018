# 维基解密披露巴拉克·奥巴马当选总统前的电子邮件地址

巴拉克·奥巴马的收件箱。维基解密在克林顿竞选团队负责人约翰·波德斯塔的电子邮件中揭露了许多秘密和不那么秘密的东西，其中包括总统的个人电子邮件地址。

它现在就在那里，如果它仍然活跃，可能会立即被关闭，所以我认为重复地址是…bobama@ameritech.net 没有什么坏处。我发去查看的一封邮件(对不起，总统先生)没有立即退回。

如果你期待一些有趣或不寻常的事情，对不起。Ameritech 是贝尔家族的一员，负责美国的一大块粮仓，包括奥巴马的家乡伊利诺伊州。它在 1999 年被整合到 SBC，然后在 2006 年回到美国电话电报公司，但是客户显然需要保留他们的旧电子邮件地址。

在被黑客攻击的电子邮件所涵盖的时期内，有几封电子邮件被发送给了博巴马，但他只回复了一次，作为对波德斯塔提出的一个人事问题的跟进。他的签名显示是从美国电话电报公司无线的黑莓手机上发出的。

像其他公众人物一样，奥巴马可能至少有六个电子邮件地址。考虑到对总统通信的限制，这个可能已经多年没有使用了，或者它可能是非官方查询和非正式聊天的首选联系点。很难说；如果总统回复我，我会更新这个帖子。
