# Wingify 为没有编码经验的人推出新的 A/B 测试平台 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/06/25/wingify-launches-a-new-ab-testing-platform-for-people-without-coding-experience/>

网站和应用程序的 A/B 测试通常是开发人员和 IT 团队的领域。然而，随着可视化网站优化器(VWO) 的[重新推出，](https://web.archive.org/web/20221006175914/https://vwo.com/) [Wingify](https://web.archive.org/web/20221006175914/http://wingify.com/) 希望让更多的人可以使用 A/B 测试，即使他们没有 HTML 或编码经验。这意味着公司可以向营销团队和产品经理开放其产品的 A/B 测试，让他们立即看到什么可行或不可行，并允许他们立即对网站和原生 iOS 应用进行更改。

总部位于德里的 Wingify 的产品目前被 75 个国家的 3700 个品牌使用，包括微软、通用电气和 Rackspace。

为了使用 VWO，营销人员在他们的网站上添加一小段代码，就像他们使用谷歌分析一样。之后，他们可以使用 VWO 的平台进行 A/B 和多元测试，并对他们的网站或应用程序进行更改。

创始人兼首席执行官 Paras Chopra 解释说:“用户可以点击更改页面标题，添加或更改按钮，拖放更改布局，发布新版本的网页，以衡量其与默认版本的网页相比的有效性。

该平台还让营销团队更容易进行 A/B 测试，提供了一个包含 300 多条测试建议的库，称为“创意工厂”，然后生成关于他们网站或应用程序中哪些功能正在工作的实时报告，Chopra 表示，这可以增加销售和转化率。

通过提供不需要编码或 HTML 专业知识的 A/B 测试，Wingify 希望进一步区别于其主要竞争对手，包括 Optimizely 和 Adobe Target。

“VWO 的巨大优势在于它专注于提供一种代码和 IT 免费资源，任何营销人员都可以立即获取和使用，并获得一套超越 A/B 测试的用户研究、热图和个性化工具。此外，VWO 还是唯一一个将可视化和无代码 web A/B 测试技术引入 web 和原生 iOS 应用的解决方案，”乔普拉说。

“最后，VWO 的独特之处在于，它允许对 A/B 测试和结果进行任意细分，以找到开展个性化营销活动的有趣机会。这是唯一的解决方案，让营销人员能够深入研究数据，专注于特定的标准，以找出什么在起作用，而不必事先定义那些‘桶’。”
