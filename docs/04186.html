有了新的预算工具，AWS 让开发者更容易管理成本

亚马逊今天宣布两个新工具让开发者更容易控制他们在 AWS 云计算平台上的开支。

第一个工具叫做预算，允许 AWS 用户为他们的 AWS 成本定义一个月预算。顾名思义，这意味着您现在可以为所有 AWS 支出设置预算，或者仅为 EC2 服务设置特定预算。然后，当您接近超出每月预算时，或者当您的预测成本超过 100%时，AWS 将向您发送警报。

此外，AWS 今天为其成本浏览器服务推出了一个新工具，试图预测未来三个月的每月成本。该服务可以查看总体数据，但更有趣的是，它可以查看特定的服务、标签、可用区域、购买选项和 API 操作。考虑到在给定的一个月里你使用 AWS 的方式可能会有一些变化，该服务还会显示其预测的置信区间。

估计 AWS 的成本是一门神秘的艺术，亚马逊的精细定价结构让它变得更加复杂。你在 AWS 上托管的应用程序越复杂，就越难计算出在亚马逊的服务上运行它的成本(这也使得很难将 AWS 的成本与其他云平台进行比较)。这些新服务有望使跟踪 AWS 成本变得更容易，而不必求助于第三方工具。
