# 扎克伯格 2017 年的挑战是会见并倾听所有 50 个州的人们

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/01/03/zuckerberg-challenge/>

在围绕假新闻、审查和偏见的批评和指控持续了一年之后，脸书的首席执行官马克·扎克伯格正在利用他的年度挑战来教育自己，了解美国各地用户的需求和问题。

这可能有助于脸书成为一个更好的平台，让每个人的声音和想法都可以表达出来，而不仅仅是沿海地区或从事技术工作的人。这一点现在尤其重要，因为技术可能会取代低技能工人，而总统选举已经导致了人口的两极分化。

扎克伯格写道:“我的工作是连接世界，让每个人都有发言权。今年我想亲自听到更多这样的声音。”[更新:扎克伯格现在[邀请](https://web.archive.org/web/20230406143822/https://www.facebook.com/zuck/posts/10103387479341041?pnref=story)人加入他的[“旅行挑战”](https://web.archive.org/web/20230406143822/https://www.facebook.com/groups/YearofTravel/)，尝试参观 12 个新地方，并在每个地方见一些人。]

![Zuckerberg Crowd](img/fd1c2e2c8e54efef825f607305823f7c.png)

在过去几年的挑战中，扎克伯格已经跑了 365 英里，[学会了普通话](https://web.archive.org/web/20230406143822/https://techcrunch.com/2014/10/23/zuckerberg-speaks-chinese-internet-soils-itself/)，[读了 25 本书](https://web.archive.org/web/20230406143822/https://techcrunch.com/2015/01/04/mark-zuckerberg-starts-a-book-club-becomes-this-generations-oprah/)，[为他的房子建造了一个个人人工智能](https://web.archive.org/web/20230406143822/https://techcrunch.com/2016/12/20/watch-mark-zuckerbergs-morgan-freeman-voiced-jarvis-ai-in-action/)。

你可以点击阅读[他在脸书的全部帖子，其中包括:](https://web.archive.org/web/20230406143822/https://www.facebook.com/zuck/posts/10103385178272401?notif_t=notify_me&notif_id=1483479845570349)

> “我个人对 2017 年的挑战是在年底前拜访并会见美国每个州的人……在经历了动荡的去年之后，我对这一挑战的希望是走出去，与更多的人谈论他们如何生活、工作和思考未来。
> 
> 几十年来，技术和全球化让我们变得更有生产力，联系更紧密。这带来了很多好处，但对很多人来说，这也让生活变得更具挑战性。这造成了我有生以来从未有过的分裂感。我们需要找到一种方法来改变这个游戏，让它适合所有人。"

最近的一份白宫报告称，虽然机器人、人工智能和自动驾驶汽车等技术将提高生产率和效率，但也可能导致数百万低技能工作岗位的流失。这些工作可能更集中在美国中部的一些州，像扎克伯格这样的科技领袖可能没有去过或认识很多人。

![15073487_10103271812333651_5131664792175687007_n](img/aedb4d376fc11d0f1c025195bf36e743.png)

[扎克伯格希望脸书成为一个城市广场](https://web.archive.org/web/20230406143822/https://techcrunch.com/2016/12/21/fbonc/)，而不仅仅是一家传统媒体或科技公司，这意味着听取所有选民的意见。脸书固有的媒体特征，比如选择人们所看内容的算法和内容政策，意味着该公司必须加倍努力，在保持文明的同时，保持思想的公平竞争

这一承诺与竞选之旅的相似之处也可能引发人们对扎克伯格是否认真考虑涉足政治的猜测。首席执行官[提交文件](https://web.archive.org/web/20230406143822/https://www.bloomberg.com/news/articles/2016-12-08/facebook-s-investors-criticize-marc-andreessen-for-conflict-of-interest?hn=1)要求脸书董事会可能允许他[在政府](https://web.archive.org/web/20230406143822/https://techcrunch.com/2016/12/08/andreessen-in-hot-water-for-texts-he-sent-zuckerberg/)工作，同时保留对脸书的控制权。[更正:正如我们最初根据彭博报道的那样，扎克伯格在政府中的服务不一定限于两年，因为有例外情况允许无限期服务于政府，这取决于他的股票所有权或董事会的批准。]

通过挑战自己，通过与全国各地的人联系来增加自己的同理心，扎克伯格将有望建立一个可以更好地帮助他们适应未来的社交网络。
