# 与大富翁先生 Equifax 参议院听证会的英雄——谈论公司的贪婪🎩💸

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/10/04/the-monopoly-man-interview-equifax-forced-arbitration/>

对美国来说，这是特别黑暗的一周，一缕希望之光闪耀着，它的单片眼镜在严酷的媒体闪光灯下勇敢地闪烁着。富有的 Pennybags 大叔出场了，他是一只桌游肥猫，更广为人知的名字是“大富翁”，他在今天的参议院听证会上高调亮相关于 [Equifax 黑客](https://web.archive.org/web/20220930173252/https://beta.techcrunch.com/tag/equifax/)。

垄断先生坐在被普遍讨厌的前 Equifax 首席执行官理查德·史密斯的后面，若有所思地听着，不时地调整他的单片眼镜，捻着他的白胡须，用百元大钞擦着他的额头，就像任何人可能做的那样。迫切需要胜利的社交媒体失去了理智。

TechCrunch 采访了神秘的垄断者，现已确认为倡导组织[公众公民](https://web.archive.org/web/20220930173252/https://www.citizen.org/)的 Amanda Werner，讲述了参议院露面背后的原因。(作为作者，我必须披露，沃纳实际上是我的朋友，老实说，这是我见过的最好的事情。)

当被问及他们是如何获得如此好的特技座位时，沃纳解释说，他们组织的一名实习生实际上在早上 7 点的公开听证会上排队，以确保这个位置。

沃纳告诉 TechCrunch:“我就坐在许多 Equifax 公关人员和高管的旁边，比如理查德·史密斯。”。"在整个听证会上，我遭到了很多白眼。"尽管如此，沃纳说，“我们肯定得到了一些委员会工作人员的默默支持。”

因为大富翁先生的角色不是沃纳的第一次表演，所以沃纳通过安检没有任何困难。

“我们有穿鲨鱼装的人来扮演高利贷者，”沃纳解释道。"那里的安全人员通常尊重言论自由的权利。"

也许是最搞笑的时刻，在听证会后，沃纳实际上尾随史密斯，给这位前 Equifax 首席执行官送去了一大笔钱，这意味着该公司刚刚从国税局获得了 725 万美元的合同。“他前面有新闻摄像机……[而且]我在后面追着他，”沃纳说，他可能正确地认为这些镜头会成为晚间新闻。如果你目睹了这一切，请用 GIF 图片告诉我们。

沃纳希望每个在 Equifax 愤怒浪潮中的人都会打电话给他们的代表，为消费者金融保护局限制强制仲裁的规定辩护。这种仲裁通常对消费者非常不利，阻止个人对公司提起诉讼，而是“将纠纷推入秘密的仲裁程序，这种程序被操纵以有利于金融公司，并向监管机构隐瞒不法行为，”正如 [Public Citizen 解释的那样。](https://web.archive.org/web/20220930173252/https://www.citizen.org/media/press-releases/forced-arbitration-%E2%80%98get-out-jail-free%E2%80%99-card-banks-cheat-customers)

在 Equifax 黑客攻击之后，当消费者在该公司的服务条款中注意到这一点时，强制仲裁出现了，Equifax[以其粗略的类似 Equifax 的方式澄清了这场争议。](https://web.archive.org/web/20220930173252/https://beta.techcrunch.com/2017/09/08/equifax-says-it-wont-bar-consumers-from-joining-breach-related-lawsuits/)

沃纳解释说，强制仲裁对消费者有更广泛的重大影响。当涉及到不正当的商业或隐私行为时，个人诉讼可能会起到“煤矿中的金丝雀”的作用，但如果诉讼在强制仲裁下被迫隐藏，公众可能需要数年时间才能发现它，就像富国银行假账丑闻一样。沃纳说:“这让消费者无法讲述他们的故事。”。“因为那些消费者是被迫进行秘密仲裁的，所以我们三年都没有发现这件事。”

总的来说，我们很高兴与这位冉冉升起的互联网明星谈论消费者保护，尽管由于沃纳相当的政策悟性，角色中的笑声没有我们希望的那么热情。

当被问及 Monopoly 先生是否受到 Equifax 数据泄露的影响时，Werner 指出他并不担心他的财务前景中的身份盗窃。"垄断者实际上只是把他的财富保存在黄金中."
