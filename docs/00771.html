苹果将停止开发 Aperture，并将用户转移到 OS X 的照片

随着今年晚些时候 OS X Yosemite 的发布，苹果将停止开发其“专业”照片编辑应用 Aperture。该程序的用户将被转移到 Photos，这是一款在 WWDC 主题演讲期间推出的新应用，将于明年发布。

“照片”集成了许多以前在 Aperture 中发现的高级照片编辑功能，并将在新的 OS X 上取代 iPhoto。

“随着新照片应用程序和 iCloud 照片库的引入，使你能够安全地将所有照片储存在 iCloud 中，并从任何地方访问它们，Aperture 不会有新的发展，”苹果发言人告诉 TechCrunch。“明年 OS X 版照片上市时，用户将能够将其现有的 Aperture 资料库迁移到 OS X 版照片。”

苹果表示，它将为 Aperture 提供兼容性更新，允许它在 OS X Yosemite 上运行，但不会继续开发它。 Adobe 表示将加倍支持 Lightroom，并为苹果用户提供一种迁移方式:

简而言之，我们正在加倍投资 Lightroom 和新的 Creative Cloud 摄影计划，您可以期待在未来几周、几个月和几年内看到桌面、网络和设备工作流快速创新的丰富路线图。我们还继续积极投资 iOS 和 OSX 平台，并致力于帮助感兴趣的 iPhoto 和 Aperture 客户迁移到我们跨桌面、设备和网络工作流程的丰富解决方案。

苹果还将提供更新路径，帮助用户从 Aperture 和 iPhoto 过渡到 OS X 的照片。苹果表示，这并不意味着不再为 pro 客户提供 Final Cut Pro 和 Logic 等其他应用程序，这些应用程序将继续存在。

作为 Lightroom 的长期用户和前专业摄影师，这个消息一点也不让我感到惊讶。Lightroom 在吸引 Aperture 客户方面做得非常出色，尤其是在 Aperture 长期萎靡不振，没有大量重大更新的情况下。现在，苹果正在通过将其照片应用程序统一为一个产品来正式实现这一转变。与 Aperture 不同，照片集成了苹果新的 iCloud 照片库，这应该会让拥有多台苹果设备的人更加方便。

如果苹果停止在 iOS 上使用 iPhoto，转而使用 one photos 应用程序，我也不会感到惊讶，因为 iOS 8 将为 stock 应用程序引入新的编辑工具。但我不知道这是否会发生；这只是一种直觉。
