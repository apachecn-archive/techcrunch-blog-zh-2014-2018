突破实验室支持4家“重新编程自然”的创业公司

原文：https://web.archive.org/web/https://techcrunch.com/2016/10/11/breakout-labs-backs-four-startups-that-are-reprogramming-nature/?ncid=rss

据创始人兼管理合伙人呼唤·菲什伯恩称，一家支持非常早期的高科技和科学初创公司的风险基金 Breakout Labs 已经投资了4项新交易。

该基金是泰尔基金会的一部分，通常向每个初创公司投资35万美元，没有任何附加条件。如果一家初创公司的研发没有像希望的那样导致商业化，Breakout Labs不会寻求任何退款，并将它投入的现金算作赠款。

如果一家公司足够成功，可以继续筹集外部风险资本，Breakout Labs的早期赠款可以折价换取股票。

如果Breakout Labs公司被收购或上市，为投资者带来回报，该基金将把所有回报用于其他先进技术和科学创业公司的未来拨款。菲什伯恩称这种方法为“第一英里”，资助。

Breakout Labs还为新支持的初创公司创始人支付费用，让他们前往旧金山参加季度会议，以及一年一度的“拆箱日”，创始人可以在这里展示他们的研究，并与潜在的战略合作伙伴和风险投资者建立联系。

菲什伯恩说，如果迄今为止它所支持的公司中有一个统一的主题，那就是“重新编程”自然的想法。“我们可以从自然界中已经存在的东西中重新编程、构建和升级我们需要的东西，而不是被动地依赖大自然来提供给我们，”菲什伯恩说。

迄今为止，突破实验室已经投资了生物技术、数据和材料科学以及能源领域的30家公司。其中七家公司已经进行了首轮融资，两家公司已经完成了首轮融资。

突破实验室投资的四家新创公司是:

  • 阿齐特拉——这家总部位于康涅狄格州法明顿的初创公司正在开发一种“分泌治疗性蛋白质的皮肤安全细菌重组菌株”当一个人患有湿疹或葡萄球菌感染等问题时，这些蛋白质从皮肤的微生物群中缺失。
  • Opus 12——总部位于加州伯克利，在产生污染的地方捕获碳排放，并将二氧化碳转化为甲烷、乙烯和乙醇等化学物质，这些气体用于制造塑料、表面活性剂、清洁剂和燃料，与柴油和汽油相比，这些燃料燃烧起来相对清洁。
  • 这家位于加州帕萨迪纳的公司是加州理工学院美国宇航局喷气推进实验室的一个分支，该公司正在开发在某些情况下可以替代锂电池的技术。具体来说，Seatrec制造了一种用相变材料包装的电池。当电池周围的温度变化约10度时，这些电池就会膨胀和收缩，产生高压，然后储存起来用于驱动发电机。Seatrec系统可以驱动无人水下航行器，称为浮子和滑翔机，用于海洋研究。
  • 酵母化学——加州另一家初创公司，酵母化学正在开发微生物和工艺，将糖和其他可再生原料转化为有价值的化学品，而不会排放温室气体。该公司旨在可持续地生产制造服装、汽车塑料和食品包装所需的化学品，这些化学品目前都是使用石油制成的。
