# 这是三星 Galaxy S8

看哪。三星 Galaxy S8 [在](https://web.archive.org/web/20221207151022/https://beta.techcrunch.com/2017/02/26/samsung-just-proved-it-doesnt-need-the-worlds-largest-mobile-conference-anymore/)前一个月你就应该看到了。这张图片来自专业泄密者埃文·布拉斯，他对大部分手机泄密事件负有责任。

这里的图像显示了重新设计的 Galaxy 手机。它没有物理按钮，但仍有 Galaxy S7 edge 的环绕式屏幕。边框比过去的型号更小，只占设备顶部和底部的一小部分。

泄露并没有透露任何关于软件或手机外壳的信息。但不用担心，这款手机的发布还有一个月的时间。也有足够的时间泄露这些信息。

三星处境艰难，这次泄露的时间很有意思。今天是世界上最大的手机贸易展世界移动通信大会的最后一天。这是三星用来公布最后三款 Galaxy 手机的发布会。但今年，三星只发布了两款平板电脑，并宣布将于 3 月下旬在纽约举行 S8 发布会。

MWC 2017 仍然是一场高质量的展会，但缺少了通常由三星引起的一点兴奋。

随着这一泄漏，三星跳回到新闻周期的顶端，使其他公司如诺基亚、黑莓和 LG [安静下来，这些公司用他们自己的手机引起了轰动](https://web.archive.org/web/20221207151022/https://beta.techcrunch.com/2017/02/27/mwc-day-zero/)。
