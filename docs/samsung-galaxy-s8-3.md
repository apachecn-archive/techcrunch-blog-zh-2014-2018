# 我们认为自己对三星 Galaxy S8 的了解

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/03/25/samsung-galaxy-s8-3/>

下周在纽约，三星将最终发布 Galaxy S8。这款手机在很大程度上取决于它——不仅仅是因为它是业内最大的品牌之一的品牌定义旗舰，或者因为该公司似乎已经推迟了相当长的时间，因为它一直在努力做好事情。下周的 Unpacked 活动将对三星产生更广泛的影响，这是自去年 T2 Galaxy Note 惨败给许多消费者留下严重品牌创伤以来的第一次重大手机发布。

三星已经高兴地炒作了很长时间的公告。事实上，在另一场平淡无奇的世界移动通信大会新闻发布会上，发布会的日期本身就是一个重大的启示[,这场新闻发布会的焦点是两款新平板电脑。在接下来的几周里，我们对 S8 的印象已经相当完整了，这是通过该公司的官方声明和一些看似合法的泄漏来实现的，这些泄漏表明，无论是有意还是无意，S8 都没有很好地控制住局面。](https://web.archive.org/web/20230128070839/https://techcrunch.com/2017/02/26/samsung-mwc-press-conference/)

预计周三发布会的开场将再次呈现出介于忧郁和兴奋之间的基调，这可能是 Note 7 最后一次公开道歉，同时还会对该公司最近实施的严格安全标准进行更彻底的赞扬。虽然其他公司可能会放慢脚步，但这从来不是三星的方式。预计这将是该公司的又一款“厨房水槽万能”设备。

有一点我们可以肯定的是，三星加入了越来越多的手机制造商选择自己的智能助手的行列——这一事实无疑让谷歌感到不快，因为这家安卓制造商希望将自己的助手推广到更多手机上。上周，三星详细介绍了 Bixby 的大量信息，证实了我们对 Siri/Alexa/Assistant 竞争对手的一些了解。

作为去年收购 Viv 的产物，人工智能承诺了一些有趣的下一代功能，如手机上其他应用程序的上下文感知，以及对其他被认为“支持 Bixby”的应用程序的大量第三方支持。该公司还承诺从学习过程中“消除摩擦”，以帮助切换界面“无缝和直观”。

至于硬件，至少自多年前发布第一款 Note 设备以来，该公司一直在推行其“越大越好”的叙事。相应地，传言指向的不是一款，而是两款大手机——Galaxy S8 和 Galaxy S8+，分别为 5.7 英寸和 6.2 英寸。就像我说的，中庸不是三星的 DNA，即使是在去年的 s 风暴之后。至少，据说 S8 的分辨率为 2960 x 1440，比 S7 的 2560 x 1440 有了明显的提高。

在设计方面，三星正在继续寻求消除世界上的边框，我们看到的泄露图像似乎让该公司非常接近这一目标，接近真正的边对边显示器的梦想，这在一定程度上归功于一些圆形的屏幕侧面。与此同时，三星的标准长方形 home 键看起来完全消失了，这似乎表明指纹解锁已经转移到了设备的背面，就像谷歌 Pixel 等手机一样。

规格自然会全面提高。高通的 Snapdragon 835，呃，平台似乎是可能的，因为三星参与了它的创建。据说该公司还在发布前囤积芯片。S8 和 S8+的电池价格分别为 3000 和 3500 毫安时。虽然再次，在谣言中有一些回旋的余地，但不要期望该公司在这方面太疯狂，然而，它使安全成为其电池谈话的主要焦点。

大多数泄漏似乎指向三星这次抓住耳机插孔不放，但对此双方都有传言。与此同时，这款手机的发布时间定在了 4 月 21 日，有[黑色、灰色和银色](https://web.archive.org/web/20230128070839/https://techcrunch.com/2017/03/20/the-samsung-galaxy-s8-will-come-in-three-colors/)。

在赢回消费者(和空乘人员)方面，该公司仍有工作要做，对他们来说，这个名字已经成为安全问题的代名词。但值得庆幸的是，对于三星来说，记忆通常与升级周期一样长。该公司的财务状况一直很好，非常感谢，感谢强大的零部件业务。假设下周事情进展顺利，三星应该没有太多关于其公众形象的担忧(即使它[在幕后与其他问题作斗争](https://web.archive.org/web/20230128070839/https://techcrunch.com/2017/02/16/samsung-vice-chairman-arrested-on-bribery-charges/))。

鉴于该公司过去提供的东西，至少三星下周应该会有很多精彩的展示。
