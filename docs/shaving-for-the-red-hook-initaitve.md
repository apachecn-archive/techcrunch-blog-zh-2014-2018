# 为红钩计划刮胡子

我们现在在布鲁克林的红钩镇参加 TechCrunch Disrupt NY 节目，乔恩·谢伯的胡子失控了。为红钩计划剃光头。

使用[这一 GoFundMe 页面](https://web.archive.org/web/20230130232939/https://www.gofundme.com/23jty6k/)为该事业捐款，或者[直接](https://web.archive.org/web/20230130232939/https://donatenow.networkforgood.org/1439252)向红钩计划捐款。

乔恩是一个温柔的人，也是一个优秀的作家。作为一名高级编辑，他的任务是组织和管理 Crunch Network。这是一项庞大的工作，显然消耗了乔恩太多的生命，以至于他没有时间刮胡子。

对于 Disrupt NY，我们与红钩计划(Red Hook Initiative)合作，这是一个社区中心，其使命是让红钩青年“充满灵感、富有弹性和健康，并将自己视为生活、社区和社会的共同创造者。”RHI 的数字管家计划旨在帮助年轻人做好在 IT 行业工作的准备，该计划的参与者在展会上做志愿者，并参加黑客马拉松。

RHI 是一项有意义的事业，我们愿意帮助更多的人。如果筹集到 1000 美元，这个项目就可以支付一次职业培训的费用。如果我们筹到 1000 美元，我们将在本周三的 Disrupt 电视直播节目上剃掉乔恩的胡子。
