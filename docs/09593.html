Weebly 被黑，4300 万份凭证被盗

原文：https://web.archive.org/web/https://techcrunch.com/2016/10/20/weebly-hacked-43-million-credentials-stolen/

Weebly 被黑，4300 万张凭证被盗

据数据泄露通知网站 LeakedSource 称，网络设计平台 Weebly 在 2 月份遭到黑客攻击。超过 4300 万个账户的用户名和密码被盗，尽管密码是用强哈希算法 bcrypt 保护的。

Weebly 在给客户的一封电子邮件中表示，用户的 IP 地址也遭到了破坏。

Weebly 在给用户的通知中说:“我们不认为有任何客户网站被不当访问。”“该公司还表示，它不存储信用卡信息，因此欺诈性收费不太可能发生。

LeakedSource 表示，它从一个匿名来源收到了 Weebly 数据库，并通知了 Weebly 这一违规行为。除了客户通知邮件之外，LeakedSource 声称正在发布密码重置——但是，如果你是 Weebly 用户，并且你没有收到密码重置，你可能还是想更改你的密码。

与此同时，LeakedSource 还发现了 Foursquare 的数据，称 2013 年 12 月有 2250 万个账户被入侵。这家社交媒体公司对调查结果提出质疑，声称电子邮件地址只是与 Foursquare 的公开数据进行了交叉引用。这些数据包括电子邮件、用户名、脸书和 Twitter IDs，可能是从 Foursquare 的 API 或搜索中搜集来的。

“我们已经做了内部调查，没有违规行为发生，”公司发言人在一份声明中说。

这只是一系列兆带宽中的最新一个。雅虎最近透露，5 亿用户的数据被盗，今年 Dropbox、MySpace 和 Tumblr 的漏洞都被曝光。
