# 特斯拉售价 3.5 万美元的 Model 3 轿车将在两年内投产，明年 3 月接受预订

它正在发生！你，是的你(还有我！)，也许很快就能买得起特斯拉了。今天，马斯克在 Twitter 上分享了特斯拉的圣杯，即[低成本 Model 3 轿车](https://web.archive.org/web/20230130231558/https://techcrunch.com/2014/07/15/teslas-35000-car-will-be-called-the-model-iii/)，将于明年 3 月正式发布(同时接受预订)。

我们[从去年开始就知道了它](https://web.archive.org/web/20230130231558/https://techcrunch.com/2014/07/15/teslas-35000-car-will-be-called-the-model-iii/)，但是现在我们知道什么时候应该保持警惕。

当时，马斯克说“[Model 3]将与路上的任何其他汽车都不同。但是，“在某种程度上，这真的很有用，只是感觉不像一个奇怪的手机。“不管那意味着什么，我想要它。

如果你预定了一辆性感的特斯拉 SUV(Model X)，[你现在就可以开始定制了](https://web.archive.org/web/20230130231558/https://techcrunch.com/2015/09/01/reserved-teslas-top-line-model-x-suv-you-can-start-customizing-it-now/)。当我们谈论特斯拉的时候，如果你错过了， [Model S 也打破了一些消费者满意度记录](https://web.archive.org/web/20230130231558/https://techcrunch.com/2015/08/27/teslas-model-s-p85d-just-broke-consumer-reports-ratings-system-scoring-103-out-of-100/)。

对于我们其他人来说，现在就开始为 Model 3 攒钱吧。
