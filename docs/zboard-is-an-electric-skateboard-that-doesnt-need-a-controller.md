# ZBoard 是一款不需要控制器的电动滑板 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/01/10/zboard-is-an-electric-skateboard-that-doesnt-need-a-controller/>

毫无疑问，电动滑板效率不高，也不好玩。但大多数现代电动滑板，如[助推板](https://web.archive.org/web/20230203223412/https://boostedboards.com/)和[内侧 M1](https://web.archive.org/web/20230203223412/https://inboardtechnology.com/) 都是由通过蓝牙连接到板上的手持控制器操作的。但是位于洛杉矶的初创公司 [ZBoard](https://web.archive.org/web/20230203223412/http://www.zboardshop.com/) 专注于避开控制器、可以由电路板本身的传感器操作的电路板。

[我们上一次关注 ZBoard 是在两年前](https://web.archive.org/web/20230203223412/https://techcrunch.com/2015/01/06/zboards-new-foot-triggered-electric-skateboards-go-far-but-feel-unstable/)当时我们乘坐了他们的一款 V1 板。虽然我们对该范围印象深刻，但我们认为脚控加速器在城市环境中骑起来有点太不稳定。

因此，上周在 CES 上，我们检查了他们的第二个版本的板，ZBoard 2。它有两种型号——续航里程为 16 英里、最高时速为 20 英里的蓝色，售价 1299 美元；续航里程为 24 英里、最高时速为 20 英里的珍珠，售价 1499 美元。

这两个版本都包括一个皮带驱动的电机，集成的头灯和尾灯，当然还有两个压力垫，供你的脚向前和向后移动板。

起初，习惯控制冲浪板有点尴尬，特别是如果你通常骑的是依靠手持控制器的助力冲浪板之类的东西。但是经过 20 分钟的骑行后，肯定会变得更容易，并且可能会在几周的骑行后变得完全自然。当然，基于踏板的控制不如手持控制器准确，但一旦你习惯了，它实际上会感觉更自然。抛弃控制器实际上让我感觉更像是在玩一个普通的滑板。

没有控制器还有其他一些好处，比如控制器和电路板之间不会失去连接(这种情况很少见，但确实会发生)，也不需要给另一个设备充电。

最终，如果你想要一个电动滑板，但不喜欢手持控制器，ZBoard 是一个不错的选择。查看上面的视频，看看我们在 CES 2017 上试用它。
