# Bolstr 包是小工具爱好者最好的朋友

如今，我们很少在 TC 上谈论包包(除了在[包包周](https://web.archive.org/web/20221224192609/https://techcrunch.com/tag/bag-week/)期间)，但这里有一个众筹项目，我认为对我们这些拥有多部手机、携带笔记本电脑的电脑迷来说是完美的。称之为[bolst](https://web.archive.org/web/20221224192609/https://www.kickstarter.com/projects/jayyoo/bolstr-the-ultimate-edc-bag-minimal-and-perfectly)，这个美国制造的包足够小，不引人注目，也不够花哨，不能算作一个真正的 murse。

由克利夫兰本地人 Jay Yoo 创作，薄背为您的所有小配件提供口袋，包括一个隐藏的平板电脑和文件后袋。

斜挎包甚至包括一条腰带，可以像皮套一样系在腰上，这样你就可以完成各种动作，比如“跑过机场”和“跳起来接住掉落的啤酒”

柳上了 Cleveland.com[的专题节目](https://web.archive.org/web/20221224192609/http://www.cleveland.com/business/index.ssf/2014/08/cleveland_entrepreneurs_modern.html)，他将自己的设计哲学描述为“少即是多”

他说:“我们大多数人都不想背着一个超大的笔记本电脑包，系上一个腰包，或者拎一个时髦的男士钱包，只是为了每天多带几样东西。”

在旅途中度过多年后，Yoo 对旅行服装市场有了深入的了解，并想创造一些很酷的东西。他的第一个项目，一件翻领上有 iPod 控制的夹克，失败了，但这个项目已经超支了 67，000 美元，还有 14 天的时间。单个袋子的起价为 45 美元。它最终的零售价为 75 美元。

它看起来像是一个非常了解这个领域的人创造的有趣产品，不到 50 美元，它可能就是取代你破旧的百事品牌腰包的东西(看着你，保罗叔叔)。
