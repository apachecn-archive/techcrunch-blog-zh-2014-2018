# “大声喊出来”旨在成为与朋友分享照片的最简单方式

> 原文：<https://web.archive.org/web/https://techcrunch.com/2014/09/27/shoutout-app-launch/>

“大声喊出来”是一款新的 iOS 应用，提供联合创始人兼首席执行官 Philip Golbraikh 所说的“视觉短信”

基本上，这个想法是为了创造一个环境，让人们不担心他们照片的质量，而是将它们视为“你的信息的视觉表达”——这就是我正在做的，这是我在街上看到的一些很酷的东西，等等。

这并不是说没有任何东西阻止人们从[在其他地方分享蹩脚的照片](https://web.archive.org/web/20230304043246/https://twitter.com/anthonyha/status/515677336235438080)，也有其他类似的应用，最著名的是 [Snapchat](https://web.archive.org/web/20230304043246/https://www.snapchat.com/) 。但是,“大声喊出来”不仅强调了照片作为交流的理念，它还尽可能地简化了这种交流。

当你打开应用程序时，它会立即显示你的 iPhone 相机的视图。在屏幕底部的三分之一，你还可以看到图标显示你在应用程序中的所有连接，按照你的排名。如果你想拍照，只需点击朋友的图标，照片就会同步拍摄并分享。(您也可以在“相机胶卷”中存储一份副本。)

“这比发短信还要少，”设计师加勒特·皮克(之前是 Draw Something 的设计师)说。

在过去的几天里，我一直在玩“大声喊出来”,它真的很简单，尽管我认为我需要更多的时间，需要更多的朋友来使用这个应用程序，以完全融入随时发送随机照片的精神。

你还可以在拍照前添加滤镜，将联系人分组，这样你就可以一次给一堆朋友发一张照片。当你在接收端时，照片在应用程序中显示为一叠照片，你可以轻扫而过——没有时间限制，但一旦你看了照片，它就消失了。

这款应用还类似于短信，你只需要某人的用户名就可以开始给他们发送图片；他们不必同意与你联系。另一方面，如果你真的很讨厌(或者更糟)，他们总是可以阻止你。

今年早些时候，Golbraikh、Peek 和联合创始人/首席技术官 David Alson 的团队发布了一款名为 Sneeky 的匿名照片分享应用[。虽然 Sneeky 仍然可以在 App Store 中使用，但 Golbraikh 表示，该公司(由 ENIAC Ventures 支持)现在专注于“大声喊出来”。](https://web.archive.org/web/20230304043246/https://techcrunch.com/2014/03/18/draw-something-designer-debuts-sneeky-an-anonymous-social-app-which-focuses-on-photos/)
