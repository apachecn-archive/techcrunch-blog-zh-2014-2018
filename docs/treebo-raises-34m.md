# 印度 Treebo 为其经济型酒店网络 TechCrunch 融资 3400 万美元

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/08/31/treebo-raises-34m/>

Treebo Hotels ，印度新兴经济型酒店网络中最杰出的成员之一，已经完成了 3400 万美元的 C 轮融资。

这笔交易由两家香港私募股权公司 Ward Ferry Management 和 Karst Peak Capital 牵头，现有投资者 SAIF 合伙公司(Bernstein Partners)、Matrix Partners India 和贝塔斯曼印度投资公司(Bertelsmann India Investments)参与了投资。在这笔交易之前，成立两年的 Treebo 已经筹集了 2300 万美元，包括[去年 1700 万美元的 B 轮融资](https://web.archive.org/web/20221028183308/https://beta.techcrunch.com/2016/07/22/indias-treebo-hotels-raises-17-million-for-its-budget-hotel-network/)。

像 Treebo 和 OYO 这样的初创公司已经从软银和红杉这样的投资者那里筹集了超过 2.5 亿美元的资金，它们已经崛起，为印度各地酒店质量千差万别的问题提供了一种科技化的解决方案。

他们为客人提出了一个基本标准，包括干净的床单、床上用品、免费 WiFi 等，价格范围为每晚 1000-3000 印度卢比，约合 15-50 美元。此外，他们使用移动应用程序作为虚拟礼宾服务，帮助办理入住手续、订单、回答问题等。

这个想法是在没有标准的地方设立标准。在西方环境下，这听起来可能不那么令人印象深刻，但任何在印度或亚洲其他地区旅行过的人都可以证明这些保证可以让他们安心。

Treebo 声称在印度 50 多个城市提供近 300 家酒店(覆盖超过 6000 间客房)。与 OYO 不同，它没有涉足海外。联合创始人 Rahul Chaudhary 去年告诉 TechCrunch，印度的机会是一个足够大的挑战。他估计经济型酒店占全国酒店市场的 65 %,这意味着每年大约 150-200 亿美元的支出。

该公司确实在方法上与竞争对手略有不同。Treebo 采用完全特许经营模式，这意味着它独家经营酒店，与酒店所有者合作，提供其品牌、客户获取渠道、管理软件等。作为交换，它需要削减高达 40%的收入。

OYO 和其他公司运营部分库存，允许酒店的一些房间不在列表中，或者通过其他平台提供。然而，它们已经开始向与树波相同的方向移动。今年早些时候， [OYO 说它有 30，000 间客房处于满负荷状态](https://web.archive.org/web/20221028183308/https://www.oyorooms.com/officialoyoblog/2017/07/28/oyo-report-card-fy-2015-2017-2)，但不清楚这占其总负荷的比例是多少。

Treebo 表示，它计划将这笔新资金用于改善客户体验、营销、技术和扩张。今年，该公司开始利用电视广告来吸引新观众，知名演员伊尔凡·可汗是该公司的形象大使。由于这笔新的现金储备，我们可能会期待更多同样的事情发生。
