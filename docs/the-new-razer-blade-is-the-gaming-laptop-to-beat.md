# 全新 Razer Blade 是击败 TechCrunch 的游戏笔记本电脑

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/02/25/the-new-razer-blade-is-the-gaming-laptop-to-beat/>

本月早些时候， [Razer](https://web.archive.org/web/20221016110436/http://www.razerzone.com/) 推出了[其超高分辨率游戏笔记本电脑的第二代](https://web.archive.org/web/20221016110436/https://beta.techcrunch.com/2015/02/03/these-go-to-eleven/)，Blade。我已经玩了几个星期，可以确认它的硬件符合疯狂的 QHD+屏幕。

像它的前辈一样，今年的 Razer Blade 采用了 14 英寸、3200 x 1800 像素的屏幕。它从大多数角度看起来都很漂亮，可以变得非常明亮，如果你是 Windows 8 用户中的一员，并且实际上利用了触摸友好菜单，它还有一个触摸屏。照片和视频确实会在显示器上弹出——这不太像 4K，但考虑到它已经超过了无法识别合理距离内的像素的程度，任何人都不会错过这一点。

当然，真正重要的是，新的刀片式服务器是否能够在面板的原生分辨率下以体面的设置处理游戏。升级到 Geforce 970M 芯片组对刀片式服务器很有帮助——我能够在许多游戏中将图形速度提高到 Ultra，并保持每秒 45-60 帧，只需通过设置来关闭抗锯齿或最高的运动模糊设置。

在新的 GPU 和通常为大型笔记本电脑保留的移动 Core i7 之间，Blade 不仅仅是一台好的游戏机。如果你是创意型的，它也是照片和视频编辑的好工具。同样，如果您打算使用它来完成这两个任务，该屏幕会使软件包更加完整。

除了屏幕，你可以对大多数游戏笔记本电脑的性能说同样的话，因为高端硬件是该类别的先决条件。因此，值得讨论一下刀片式服务器如何在大多数竞争对手拥有相同规格的领域脱颖而出。

这款外壳的铝制外壳在外观上极具侵略性，但在内部却很优雅。键盘、触控板和扬声器的布局与 MacBook Pro 相似。按键之间有一个舒适的空间，当你要删除时，你不会不小心按下电源按钮。触控板对于 Windows 笔记本电脑来说非常出色，它的手势支持不会滞后，也不会误认为是向上。触控板下方的左右鼠标按钮不会提供非常令人满意的点击，但如果你在这个东西上玩游戏，你可能会使用单独的鼠标。

虽然当你把它从盒子里拿出来时，黑色的铝看起来很棒，但你手上的油确实会在刀片表面突出。每次我们想要拍摄原始照片时，都要花几分钟时间来清理笔记本电脑，所以有些人可能会觉得这很烦人。

在外壳大多数人会看到的部分，你发现两个凸起和一个亮绿色的雷蛇标志。如果你非常害羞，要意识到它会在一堆平面和苹果商标中引起注意。与笔记本电脑的内部一样，请做好准备，在大多数光照条件下都能看到刀片式服务器外部的指纹。

在笔记本电脑的底部，有长橡胶条保持刀片稍微抬高。这为两个风扇留出了空间，使热空气远离硬件。在正常使用过程中，Blade 基本上是安静的——但在过去几年的游戏中，打开图形，它们会变得相当吵。至于“meh”扬声器，我不指望许多游戏玩家会注意到，因为购买 2200 美元游戏笔记本电脑的人可能会有像样的 3D 音频或多人通信耳机。

在笔记本电脑上玩游戏总是需要在性能和便利性之间进行权衡。你可以以更低的价格买到一台在游戏性能上绝对压倒刀锋的台式电脑，而在笔记本电脑中，你可以买到一台更重、更丑、规格更好的电脑，从而节省几百美元。但是有人在乎性能*和*审美。对于这些游戏玩家来说，Blade 是最好的折中选择。
