成百上千的工程师和设计师聚在一起想出了一些很酷，很整洁，很棒的东西。唯一的条件是他们只有 24 小时来完成他们的项目。他们中的一些人是第一次参加我们的活动，而其他人是常客。他们中的一些人睡在角落里的地板上，而另一些人喝得太多了。

## 《士师记》

**Sabela Garcia 单面山，Next Media Accelerator(下一个媒体加速器)项目经理** Sabela Garcia 单面山在汉堡 next media accelerator (NMA)工作，致力于为早期媒体初创公司提供一个在欧洲快速发展和扩大规模的地方，首先进入德国市场。作为项目经理，她代表 NMA 在德国和国外为欧洲和以色列的初创公司传播项目的利益，寻找项目的潜在申请人，并与媒体高管和投资者交往。一旦团队来到德国，她还会指导他们，帮助他们达到目标。

蒂娜·埃戈夫，首席采购官，线索
蒂娜·埃戈夫领导海盗船(又名产品团队)。作为 Clue 的首席运营官，她目前正在定义全球产品战略，并为全球女性健康领域排名第一的移动应用 Clue 组建产品、增长和盈利团队。在加入 Clue 之前，她曾绕道咨询 ThoughtWorks，在 Podio 担任高级产品经理，创办了自己的初创公司，并以失败告终，还出售当代艺术作品。她还撰写了关于“工作的未来”的博客，并协调了汉堡极客网(Hamburg Geekettes)，这是一个面向科技和初创公司女性的网络。

**Susanne Kaiser，just.social 首席技术官**
作为 Just Software 的首席技术官，Susanne Kaiser 负责 JUST SOCIAL 的软件开发——为组织内的协作和交流提供应用程序。她拥有计算机科学的背景，以及超过 15 年的技术和创业专长。她也是国际科技舞台上的公共演讲者。

Diksha Dutta 是 ElektroCotoure 的首席沟通官和独立专栏作家。她在印度以特写记者的身份开始职业生涯，后来逐渐涉足核心新闻报道，并密切跟踪私募股权/风险投资/初创企业领域。在媒体行业担任全职商业记者的六年时间里，她曾与《金融快报》和印度新闻集团等机构合作。自 2015 年以来，她一直在为《赫芬顿邮报》和《Entrepreneur.com》等出版物独立撰写关于初创公司的文章。目前，她正在与 Bloomsbury India 合作撰写一本关于印度创业生态系统的书，该书汇集了全国 50 多位企业家的精彩故事，将于明年出版。她选了几篇关于创业的专栏文章，汇编成了一本名为《奇迹》(WunderNova)的电子书，在亚马逊上可以买到。

**Ina Wagner，CODE 大学学生关系负责人**
Ina Wagner 是建立 CODE 的团队的一员，CODE 是一所新成立的国家认可的数字产品开发应用科学大学，于 2017 年 10 月在柏林开始他们的学习计划。她负责学生旅程的每一步——从申请和录取到他们获得学士学位后成为真正的数字先锋。在加入 CODE 之前，她是汉堡一家数字产品代理公司 deepblue networks 的人力资源主管和数字主管，这两个角色使她能够支持人才发挥他们的全部潜力。Ina 还共同创立了 Code+Design Initiative，组织编码营和其他活动来支持青少年选择科技职业。
