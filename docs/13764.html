关于特斯拉 Model 3 TechCrunch 你想知道的一切

原文：https://web.archive.org/web/https://techcrunch.com/2017/07/28/everything-you-want-to-know-about-the-tesla-model-3/?ncid=mobilenavtrend

特斯拉 Model 3 现在是一个已知的数量——我们有关于上市时可用的不同套件选项的信息，以及内部和外部尺寸、汽车性能和客舱舒适度的信息。特斯拉首席执行官埃隆·马斯克(Elon Musk)表示，关键是要以“几乎相同的成本”制造“最好的汽车”，这在理论上是一个强有力的证明。

正如之前宣布的那样，Model 3 的起价为 35，000 美元，但现在我们确切地知道它会给你带来什么。这是新款电动汽车的基本配置:

  • 220 英里的范围
  • 使用增压器充电 30 分钟，可恢复 130 英里的行驶里程
  • 家用 240 伏 32A 充电器每小时可行驶 30 英里
  • 0-60 英里/小时仅需 5.6 秒
  • 最高时速 130 英里

如果再多花 9000 美元，你就可以升级到远程电池版本，税前或抵税总额为 44000 美元。其中包括以下提升规格:

  • 310 英里的范围
  • 通过增压器充电仅 30 分钟就可行驶 170 英里
  • 在家里用 240 伏 40A 充电器每小时充电 37 英里
  • 0-60 英里/小时仅需 5.1 秒
  • 最高时速 140 英里

两款车的内饰相同，都有一个 15 英寸的触摸屏，可以访问虚拟仪表板、媒体和气候控制。

该车还标配了许多便利功能，包括:

  • 车载导航
  • Wi-Fi 和 LTE 连接
  • 通过 Tesla 应用程序实现无钥匙进入(车辆没有钥匙或遥控钥匙)
  • 语音控制
  • 备用摄像机
  • 后排座椅可折叠 60/40 度(非完全放平),以增加货物
  • 在弱光下自动变暗的后视镜
  • 双区气候控制
  • 带有两个 USB 端口的前控制台

其他选项包括一个高级升级包，它以 5，000 美元的价格推出了许多表面和功能改进，包括:

  • 加热座椅和更好的客舱材料，包括“开孔木质装饰”
  • 两个后置 USB 端口
  • 电动可调前排座椅、转向柱和侧镜(非常酷)
  • 超重低音扬声器和顶级环绕音响系统
  • 有色玻璃屋顶
  • 自动变暗的电动折叠加热侧镜
  • 两部智能手机的前置坞站
  • LED 雾灯

该车还拥有所有必要的传感器，以提供增强的自动驾驶功能，这是另一个 5000 美元的升级，包括速度匹配，车道保持，自动车道切换和自我停车。硬件也提供了全自动驾驶的潜力，一旦它变得可用，这又增加了 3000 美元的成本。该车总共包括七个摄像头、前向雷达和十二个超声波传感器。

其他付费升级包括基本黑色之外的颜色选择，包括金属银、深蓝和“午夜银”(非常好看的石板灰)，以及珍珠白和红色。这些都是 1000 美元的升级，然后你也可以花 1500 美元从 18 英寸的标准车轮升级到 19 英寸的运动版本。

总而言之，一辆完全升级的 Model 3 税前零售价格为 59，500 美元，考虑到你所得到的，这仍然是一笔相当合理的交易，特别是与可比的气体燃料汽车相比。

基本型号 3 的发货时间将晚于升级电池组的版本，升级电池组将在未来三个月内专门向首批客户发货。如果你一直在等着看 Model 3 在实际细节规格方面提供了什么，现在是时候预订了，如果你觉得有兴趣的话——这条线可能会延伸到 2019 年，并且只会继续增长。
