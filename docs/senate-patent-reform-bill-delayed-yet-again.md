# 参议院专利改革法案再次被推迟

一项旨在遏制滥用美国专利制度的两党参议院法案今天再次受挫，参议院司法委员会主席帕特里克·莱希(D-Vt。)宣布，对《专利透明度和改进法案》的审议将第四次推迟。

参议员莱希在一份声明中表示，他将在即将到来的休会后分发该法案的新版本。这位参议员预计，“司法委员会将在回来的第一周考虑这项立法。”

虽然延迟可能会挫败那些支持最快可能改革的人，但该法案的通过似乎不会遇到预期的障碍。这位来自佛蒙特州的参议员声称，该法案已经“原则上获得了广泛的两党一致同意。”

目前卡住的是细节。正如之前报道的那样，双方的症结在于罚款的[问题。共和党人希望法律要求专利侵权诉讼中的败诉方支付对方的法律费用。一个合理的想法，当然。民主党人似乎担心，一些确实有价值的诉讼可能不会被受理，如果一个甚至合理的诉讼失败，可能会产生更大的法律费用。](https://web.archive.org/web/20230131072510/http://thehill.com/blogs/hillicon-valley/technology/202886-overnight-tech-leahy-hopeful-for-tuesday-patent-markup)

此外，将费用转移给败诉方的想法可能会导致个人和资本不足的公司不追求合理的索赔，因为大公司会有更多的资源，因此可能会在小公司获胜后倾倒巨额费用。据报道称，妥协看起来很接近，但是难以捉摸。

两周的休会期可以提供足够的时间把事情安排好。鉴于该法案是由两党共同发起的，除非出现复杂情况，否则该法案可能会在 5 月初获得委员会通过。
