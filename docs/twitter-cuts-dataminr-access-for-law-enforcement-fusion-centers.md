# 推特切断了执法融合中心的数据访问

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/12/15/twitter-cuts-dataminr-access-for-law-enforcement-fusion-centers/>

# 推特切断了执法融合中心的数据访问

面对来自美国公民自由联盟的压力，推特宣布将限制以前提供给国内监控中心的数据访问。推特的[长期政策](https://web.archive.org/web/20221127015702/https://blog.twitter.com/2016/developer-policies-to-protect-people-s-voices-on-twitter)是禁止使用其数据进行监控，但是它部分拥有的社交媒体监控公司 Dataminr，已经多次接受政府合同，允许地方和联邦执法机构不受限制地访问基于推特原始数据的新闻提醒。

美国公民自由联盟(ACLU)通过[公共记录请求](https://web.archive.org/web/20221127015702/http://www.aclunc.org/docs/20160315_dataminr_email_to_jric.pdf)发现了 Dataminr 与融合中心的合作，本周 Twitter 在一封[信](https://web.archive.org/web/20221127015702/http://www.aclunc.org/docs/20161212_twitter_letter_to_aclu.pdf)中回应称，将指示 Dataminr 切断美国所有 [77 个融合中心](https://web.archive.org/web/20221127015702/https://www.dhs.gov/fusion-center-locations-and-contact-information)的接入。与 Dataminr 签约的融合中心“可以搜索数十亿条实时和历史公共推特，然后潜在地与联邦政府共享信息，”加利福尼亚美国公民自由联盟(ACLU)技术和公民自由政策主管 Nicole Ozer 说。

Dataminr 质疑 Ozer 关于访问历史推特的观点，声称它从未向任何客户提供推特历史档案或大量推特数据的“直接访问”。Dataminr 表示，它利用推特丰富的数据来制作突发新闻警报，但并未与其任何客户共享“消防软管”数据。

推特表示，过去一年，Dataminr 对其产品进行了改进，更加关注针对媒体公司和第一反应者的突发新闻警报，而 Dataminr 表示，突发新闻一直是其核心产品。据该公司称，地理空间分析实验已经被放弃，比如美国公民自由联盟(ACL)发现的实验。

“我们共同致力于开发一款专注的 Dataminr 突发新闻警报产品，旨在让第一反应者尽早了解新闻和事件。我们从这种针对性的 Dataminr 警报产品中看到了公共利益，”Twitter 全球公共政策副总裁科林·克劳尔(Colin Crowell)写道。

Dataminr 在一份声明中表示，它正在与 Twitter 合作，以保护隐私和公民自由，它仍将向执法部门提供有限版本的产品，供第一反应使用。

“我们为那些支持第一反应任务的人提供我们产品的有限版本，该版本根据公共推文提供量身定制的突发新闻警报。Datatminr 的产品不向任何政府客户提供自己的直接消防软管访问或导出数据的功能；搜索原始历史推文档案或定位或描述用户的能力；进行地理空间分析；或任何形式的监视，”Dataminr 发言人说。
