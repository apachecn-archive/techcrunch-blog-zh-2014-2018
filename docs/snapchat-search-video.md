# 试用 Snapchat 新的通用搜索功能

Snapchat 最近在 iOS 上发布了其[重新设计的 app](https://web.archive.org/web/20230404031554/https://techcrunch.com/2017/01/12/snapchat-search/)[；它现在有通用搜索功能，所以我决定试一试——你可以在上面的视频中看我摸索。](https://web.archive.org/web/20230404031554/https://techcrunch.com/2017/01/23/snapchats-big-redesign-featuring-universal-search-and-more-just-hit-ios/)

搜索栏的功能类似于搜索栏——如果你在寻找另一个用户、群聊或 Discover publisher，你只需输入它们。它还引入了快速聊天推荐，所以你可能根本不需要进行搜索。

这看起来可能是一个相对较小的改进，但对 Snapchat 来说，这似乎是一个重要的举措，它并没有对其以相机为中心的界面进行太多改变——这种界面可能会对像我这样的老年人产生一些威胁。因此，在该公司准备[传闻中的首次公开募股](https://web.archive.org/web/20230404031554/https://techcrunch.com/2016/11/15/snapchat-has-reportedly-filed-confidentially-for-its-massive-ipo/)之际，这可能是使该应用程序更加用户友好的一步。
