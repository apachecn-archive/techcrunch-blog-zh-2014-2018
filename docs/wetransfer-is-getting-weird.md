# WeTransfer 越来越奇怪了…

如果你是一家欧洲初创公司，与像 [Box](https://web.archive.org/web/20221025222136/https://www.box.com/home) 和 [Dropbox](https://web.archive.org/web/20221025222136/https://www.dropbox.com/business/landing-t61fl-v2?_tk=sem_b_goog&_camp=&_kw=dropbox%7Ce&_ad=245892100926%7C1t1%7Cc&tk=sem_b_goog&_camp=&_kw=dropbox%7Ce&_ad=245892100926%7C1t1%7Cc&gclid=CjwKCAjw5ZPcBRBkEiwA-avvk4VLvA2zE2OL5-Q9HjPbRSLMPfUQ59hdwqgGSGjL33ORl0EYYLvvgBoCeoYQAvD_BwE) 这样的公司竞争，并希望在美国这样的国际市场引起轰动，你会怎么做？

好吧，如果你是荷兰初创公司 [WeTransfer](https://web.archive.org/web/20221025222136/https://wetransfer.com/) (大约三年前，它筹集了整整 2500 万美元，在美国市场掀起一阵风暴)，你会感到奇怪。真的真的前卫级诡异。

hipsterati 的最新序曲是该公司与 Krule 国王合作的三个视频集(我为它鼓掌，只是因为它让我可以在网站上写 Krule 国王)。

这是第一个来自这位([碧昂斯](https://web.archive.org/web/20221025222136/http://www.mtv.com/news/1711211/king-krule-beyonce-earl-sweatshirt/)-和- [泰勒-创作者](https://web.archive.org/web/20221025222136/https://twitter.com/tylerthecreator/status/239099324422832129?lang=en)-和- [纽约客](https://web.archive.org/web/20221025222136/https://www.newyorker.com/magazine/2013/09/23/heir-to-the-throne)认可的)艺术家和文件传输和存储服务合作的视频。

视频

在 [WePresent“平台”](https://web.archive.org/web/20221025222136/https://wepresent.wetransfer.com/story/king-krule-ccwade-biscuit-town/)(在我那个时代，我们会称之为“网络杂志”)上，Krule 与导演和创作团队讨论了创建视频的过程——他将为所有后续版本讨论这个过程。

该系列的第一个视频由 Krule 的长期合作者迈克尔和帕拉伊克·莫里西执导，他们在视频 cc 的名义下工作。韦德。

King Krule collab 并不是 WeTransfer 第一次利用一些文化宝藏赚钱。该公司已经与麦克斯威尼的故事合作，由雪莉·奥里亚和爱丽丝·索拉·金撰写名为《清洁 T21》的故事。

不管这些对 Kool Kidz 世界的进军是否是该公司相对较新的首席执行官 Gordon Willoughby(前亚马逊首席执行官)带来的战略转变的结果，它们都非常棒。(至少，从我们几年来第一次写 WeTransfer 的意义上来说。)

我不能说 WeTransfer 的文件共享服务明显比 Box 或 Dropbox 好还是差，但他们的潮人信誉是不可否认的。指向你，我们的传递者。指向你。
