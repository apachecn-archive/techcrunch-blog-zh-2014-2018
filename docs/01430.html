Path 将打造一款 Apple Watch 应用，但戴夫·莫林不会谈论那些收购传闻

移动社交网络应用 Path 成立于四年前，旨在帮助用户与最亲近的人联系和交流。但这家初创公司正寻求拓展新的应用和体验，包括为苹果即将推出的可穿戴设备开发一些东西。

在 Disrupt SF today，Morin 承认他昨天参加了一年一度的苹果 9 月发布会，并首次看到了即将推出的新产品。他特别兴奋的一款设备是苹果手表，该公司希望“明年初”上市

莫林指出，看时间是最古老的技术之一，但苹果已经找到了一种围绕它创造新体验的方法。他称赞这家消费电子公司“牢牢抓住了用户体验”,并称赞苹果内置在手机中的软件和操作系统。

他期待为它开发一个应用程序。Morin 指出，根据他所看到的，该公司已经做了很好的工作，使开发者更容易为设备建立体验。

目前还不清楚这将是什么样的应用程序，但 Path 目前正在采取多应用程序的开发方法，这要归功于不久前推出的专用消息应用程序，以补充其核心的社交网络体验。该公司将继续推出新的应用程序，并更新其现有的产品组合，因为它试图设计基于通用后端基础设施的新体验。

这种方法的核心原则之一是，人们正在寻找更多可以启动并做好一件事情的应用程序，而不是一个具有许多功能但需要多次点击才能到达每个功能的应用程序。消息服务就是如此，今年早些时候，它作为一款独立产品从其核心应用中脱颖而出。

“我们认为应用程序是新功能，”莫林说。“在移动领域，人们真的很喜欢单一使用案例的体验。他们希望接触应用程序用例的摩擦小。”

虽然莫林热衷于赞美苹果，但他拒绝谈论关于这家消费电子公司将收购他的初创公司的传言。“我们不对谣言或猜测发表评论，”当被问及该报道时，莫兰回答说。
