# 优步首席执行官表示 2019 年是首次公开募股的目标 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/11/09/uber-ceo-says-2019-is-the-target-for-ipo/>

# 优步首席执行官表示，2019 年是 IPO 的目标

优步首席执行官达拉·科斯罗萨西(Dara Khosrowshahi)今天出席了《纽约时报》的 Dealbook 会议，打车公司的高管给出了优步上市计划的时间表。Khosrowshahi 在创始人特拉维斯·卡兰尼克今年早些时候辞职后接任首席执行官，他早些时候表示，优步的首次公开募股是他计划的一部分，将在未来 18 至 36 个月内发生，但现在我们有了更明确的目标，首席执行官将 2019 年定为具体年份。

Khosrowshahi 在活动中指出，在保持私有化方面，优步“拥有作为上市公司的所有劣势，聚光灯对准我们，没有任何优势”，这标志着卡兰尼克的观点明显不同，卡兰尼克以希望优步尽可能长时间保持私有化而闻名，[认为没有必要紧急进入公开市场](https://web.archive.org/web/20221007193153/http://www.businessinsider.com/uber-is-about-to-raise-a-giant-round-at-a-65-billion-valuation-heres-how-thats-even-possible-2015-11)。

Khosrowshahi 还表示，上市计划不会影响软银的交易，他证实交易正在进行中，并表示有信心很快会发生。这位优步首席执行官表示，软银“对上市没有特别的兴趣”，这意味着融资过程独立于其 IPO 战略。
