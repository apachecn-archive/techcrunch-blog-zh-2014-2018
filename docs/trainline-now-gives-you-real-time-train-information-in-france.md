# Trainline 现在为您提供法国的实时列车信息

Trainline 的[欧洲服务](https://web.archive.org/web/20221208131959/https://www.trainline.eu/)(以前被称为 Train 队长)正在成为你购买法国火车的一站式商店。该公司已经与 SNCF 合作，以获取关于火车、延误和你当前位置的实时数据。你必须使用 SNCF 的[官方应用](https://web.archive.org/web/20221208131959/https://itunes.apple.com/fr/app/sncf/id945434433?l=en&mt=8)才能获得这类信息。

如果你以前使用过 Trainline 订票，你知道该应用程序可以提醒你告诉你你的火车的站台。但是没有办法检查你的火车是否被延迟或取消。你不得不看广告牌，即使你的手机可能比火车站的广告牌更强大。

或者，SNCF 的应用程序显示这种信息，但不允许你订票——你必须使用 SNCF 航次或 Trainline 购买门票。但是没有人愿意处理两个应用程序。

SNCF 现在有了一个[官方 API](https://web.archive.org/web/20221208131959/https://data.sncf.com/) ，并且发布了大量的[公开数据](https://web.archive.org/web/20221208131959/https://ressources.data.sncf.com/explore/?sort=modified)。前数字部长 Axelle Lemaire 强迫公共管理部门和像 SNCF 这样的上市公司在开放数据许可下共享关键数据。因此，Trainline 将这些 API 和数据用于自己的应用程序。

如果你的火车是准时的，你仍然会看到不同之处，因为你可以看到下一站，你要停多长时间，以及你现在的位置。该公司表示，它希望显示其他国家的实时信息。

其他消息，Audrey Détrie 现在是 Trainline 法国、比利时、荷兰和卢森堡的区域经理。这是一个全新的职位。Détrie 将宣传 Trainline 并代表公司。
