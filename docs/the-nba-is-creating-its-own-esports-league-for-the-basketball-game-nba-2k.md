# NBA 正在为篮球游戏 NBA 2K 创建自己的电子竞技联盟

电子竞技的流行主要集中在玩英雄联盟、Dota 和反恐精英等游戏的联盟，这些游戏实际上都与体育无关。

但模仿真实体育运动的电子竞技游戏也非常受欢迎——例如，Madden NFL 和 FIFA 足球系列是 Xbox 和 Playstation 上最受欢迎的游戏。

所以今天 [NBA](https://web.archive.org/web/20221207040144/http://nba.com/) 宣布他们将与 [Take-Two 互动软件](https://web.archive.org/web/20221207040144/http://take2games.com/)合作，以 [NBA 2K](https://web.archive.org/web/20221207040144/https://www.nba2k.com/) 为中心创建一个新的电子竞技联盟——这是一款由 Take-Two 制作的流行篮球游戏。

该联盟将于 2018 年启动，由 NBA 特许经营的球队组成。虽然还没有球队做出承诺，但联盟专员亚当·席尔瓦[告诉华尔街日报](https://web.archive.org/web/20221207040144/https://www.wsj.com/articles/nba-take-two-to-create-professional-videogame-league-1486616461)所有 30 个联盟都“表示有兴趣”组建球队。

每支球队将有五名球员，以真正的联盟形式进行比赛，包括常规赛、季后赛和冠军赛。

这是美国职业体育联盟首次创建自己的电子竞技联盟。这也很有意思，因为这是 NBA 让电竞的重心回归到实际体育的一个举措。虽然联盟及其特许经营商总体上看好电子竞技(多个 NBA 球队已经[收购](https://web.archive.org/web/20221207040144/https://beta.techcrunch.com/2016/09/26/76ers-get-into-esports-with-purchase-of-dignitas-and-apex-pro-gaming-teams/)或投资电子竞技队)，但重点一直是与实际运动无关的游戏。

例如，10 月份的《英雄联盟》世界锦标赛在洛杉矶的斯台普斯中心(Staples Center)座无虚席，这无疑是电子竞技的一个好兆头。那么唯一的问题(特别是对于像 NBA 这样的职业体育联盟)是，英雄联盟是一个具有神话般转折的在线战斗游戏——本质上是像 Madden 或 NBA 2K 这样的体育游戏的对立面。

因此，通过创建他们自己的专注于 NBA 2K 的电子竞技联盟，联盟可以确保篮球等实际运动仍然是游戏和电子竞技不可或缺的一部分。

联盟表示，比赛可能会在网上和电视上直播，冠军赛可能会在 NBA 赛场举行。
