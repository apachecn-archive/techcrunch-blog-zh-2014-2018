# 你的霍尼韦尔泡泡数是多少？

这个月，我在西非和旧金山湾区度过，这引发了许多文化冲击，让我想到了过滤泡沫。我担心今天的技术会强化我们的本能，混淆什么是*熟悉的*和什么是*正常的*……这导致了扭曲的认知、错误的决定和不必要的冲突。生活在泡沫中是可以的，但是不知道你生活在泡沫中是不可以的。

但同时，我是一名工程师，这意味着前面的整个段落已经感觉太抽象和笨拙了。什么是泡沫？一个人如何区分什么是熟悉的，什么是正常的？你怎么可能测量和量化这些呢？

…很方便，我有了这个问题的答案，[灵感来自我的朋友](https://web.archive.org/web/20230404191413/https://twitter.com/hypatiadotca/status/447112185743822850)[利·霍尼韦尔](https://web.archive.org/web/20230404191413/https://twitter.com/hypatiadotca)，我在此以她的名字命名霍尼韦尔泡泡计数。这确实是一个非常简单的算法:

1.  去你最活跃的社交网络。*(对我来说:[推特](https://web.archive.org/web/20230404191413/https://twitter.com/rezendi)。)*
2.  在你积极关注的人中，与你性别不同的人占多大比例？*(对我来说:98/251，39%。不算太坏。)*
3.  在你积极关注的人群中，有百分之多少——用一句精彩的加拿大话来说——是可见少数群体的成员，或者是与你不同的可见少数群体的成员？*(对我来说:35/251，14%。嗯。)*
4.  在你积极关注的人群中，有百分之多少的人不是你所居住的国家的居民？*(对我来说:80/251，32%。比我想象的要少得多，但我想还不算太糟。)*

现在把这些百分比加起来。瞧啊。我的霍尼韦尔泡泡数是 39+14+32 = 85。

你的总分超过 80 分了吗，他说，选择这个数字是出于真实有效的原因，而完全不是为了让自己的分数看起来不错而随意挑选出来的？那好，干得好！你可能没有生活在一个狭窄的泡沫里，这个泡沫每天都在极大地扭曲你的世界观和你所有的本能和信念。

是否低于 80？是*比 80 低很多*吗？这些数字中有低于 20 的吗？那么你可能有泡沫问题。

理论上，互联网应该把我们所有人拉近，慢慢消除我们的分歧。今天早些时候，我加了一位塞内加尔出租车司机为好友(在脸书，不幸的是我的泡沫计数)，这种持续的联系在十年前是不可思议的。然而，挑选那些偶尔出现的生动例子太容易了，而对现代技术让我们无意识地从世界的大部分地区退出，并把我们几乎所有的时间花在我们一生都在其中度过的同一种社交圈子中的事实视而不见；不骗你，我看着上面那个 *14%* 我退缩了。

(另见阿尼尔·达什的《[我没有转发男人的那一年](https://web.archive.org/web/20230404191413/https://medium.com/the-web-we-make/79403a7eade1)))

但是衡量一个问题往往是解决问题的第一步。可以把霍尼韦尔气泡计数看作是过滤气泡的 Fitbit。不要误会，我不是假装这是对如何衡量跨圈社交的认真分析和/或考虑；事实上，我并没有假装它是一个异想天开、易于计算、名字吸引人的指标。但我怀疑任何一丝自我意识总比没有强。我想今年我会努力让我的 HBC 达到至少 100 分。如果你计算了你的，并对它的贫乏感到震惊，也许可以考虑做同样的事情。
