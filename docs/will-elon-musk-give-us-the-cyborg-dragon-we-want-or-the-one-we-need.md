# 埃隆·马斯克会给我们想要的电子人龙，还是我们需要的那个？

现在是 2018 年。我们需要一个愚蠢的机器人龙。你知道，我也知道。但是只有一个近乎超级恶棍的亿万富翁有足够的勇气来建造这该死的东西。埃隆·马斯克(Elon Musk)肯定没有太多空闲时间，也没有一辈子都花不完的钱，但他终于造出了这个该死的东西。我们绝对需要和想要的东西。那个机器人龙的事情。

Kanye West 可能会和[呆伯特家伙](https://web.archive.org/web/20221208142128/https://twitter.com/kanyewest/status/988456148905832448)一起开着他的特斯拉到处闲逛，还会和某个叫 Peter Teal 的[纨绔子弟会面，他让他的造币厂创造了完美的蓝/绿混合动力车，但 Elon Musk 是个实干家。看看这些帽子和喷火器，他对社会做出了贡献。](https://web.archive.org/web/20221208142128/https://twitter.com/danprimack/status/989227946509570048)

当然，埃隆·马斯克说他将制造一个机器人龙，这很可能是某种*满洲候选人*风格的暗语，将世界上所有的 x 型车转换成 Geo Metros。更有可能是跟[这条龙](https://web.archive.org/web/20221208142128/https://techcrunch.com/2018/01/13/spacexs-dragon-capsule-returns-from-the-international-space-station/)有关。或者更有可能是某个燃烧的人做的交易。但是听着，这是艰难的一周。

唯一让我从床上爬起来的想法是，在大西洋深处的某个地方，他正在创造某种机器龙的混合体，它最终会成为道恩·强森的朋友。 我真的很需要这个，伙计们。
