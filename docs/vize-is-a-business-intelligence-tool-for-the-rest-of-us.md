# Vize 是我们其他人的商业智能工具 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/02/16/vize-is-a-business-intelligence-tool-for-the-rest-of-us/>

很可能你现在的公司每秒钟都在收集大量关于你的客户的数据。这是一座金矿，它经常被闲置在角落里。Vize 是一款商业智能工具，可以让你更好地了解你的业务，并充分利用你的数据。该公司目前正在参与 [Y Combinator 的](https://web.archive.org/web/20221208031859/https://www.crunchbase.com/organization/y-combinator)冬季批次。

“Vize 的主要用例是数据分析和数据发现，”联合创始人首席运营官·努尔丁·拉哈尔告诉我。“我们让用户在没有任何编码技能的情况下完成许多不同的任务—准备、清理、扩展、组合、可视化、聚合、计算、建模您的数据，或者对您的数据进行‘假设’分析……”

所以 Vize 并没有直接与最复杂的商业智能工具竞争。这是一个可视化工具，让您开始使用商业智能。这都是关于可视化探索大数据集。

而且 Vize 在处理数据方面也是相当全能的。您可以上传任何静态文件，如 CSV、XLS 或 JSON 文件。您也可以直接将 Vize 连接到您的 SQL 数据库。这家初创公司将很快增加对 NoSQL 数据库的支持。

在将 Vize 连接到您的数据之后，启动程序允许您可视化您的数据并构建基本的仪表板。然后，您可以用鼠标与图表进行交互。如果你选择了数据的一个子集，它将会在其他可视化中实时突出显示。在很多方面，这感觉更像是在谷歌地图上移动，而不是操纵数据。它看起来是这样的:

说到用例，Lakehal 给了我几个例子。例如，工业公司使用 Vize 快速处理来自传感器的数据，电信公司使用它分析日志，金融公司使用它分析市场数据。

在很多情况下，Vize 计划取代 Excel。虽然 Excel 在许多情况下很棒，但它不一定是这项工作的最佳工具。Vize 每个用户每年花费 8000 美元。公司[已经筹集了](https://web.archive.org/web/20221208031859/https://www.crunchbase.com/organization/vize-software)不到 90 万美元。
