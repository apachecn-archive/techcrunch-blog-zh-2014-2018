Redis实验室为其内存NoSQL数据库服务筹集了1400万美元TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/07/21/redis-labs-raises-14m-for-its-in-memory-nosql-database-services/?ncid=rss

开源Redis数据结构商店背后的公司Redis Labs今天宣布，它已经筹集了1400万美元的C轮融资，由贝恩资本风险投资和卡梅尔风险投资牵头。该公司的现有投资者，包括硅谷银行和Tamar Ventures管理合伙人Zohar Gilon也参与了此轮投资。

在这一轮融资中，Redis Labs已经筹集了总计4200万美元的资金。

该公司表示，它计划利用新的资金来扩大其销售和营销努力，因为它试图在企业中获得更强的立足点。

这家成立于2011年的公司今天还宣布，今年上半年它已经赢得了600家新的企业客户。它的客户现在横跨一系列垂直行业，包括Groupon、TD Bank、威瑞森、HipChat、DBS、Ring Central、Menards、Twitch和flydubai等。

Redis很好地利用了不断增长的内存数据存储市场(它现在也提供了基于闪存的Redis，这允许在SSD而不是更昂贵的RAM上运行Redis)。随着越来越多的企业关注物联网用例以及如何充分利用实时分析，访问快速数据存储显然也变得越来越重要。

Redis Labs首席执行官兼联合创始人Ofer Bengal在今天的声明中表示:“现在，企业比以往任何时候都更需要能够快速处理大量数据的解决方案。“我们成立了 Redis 实验室，其使命是将 Redis 的力量扩展到需要简化管理大而复杂数据的企业。随着越来越多的企业和开发者认识到 Redis 的价值，这一C轮投资使 Redis 实验室能够进一步创新并提供高性能的 Redis 解决方案。”

像其他基于开源的企业公司一样，Redis免费提供核心数据库技术。除此之外，它还提供许多服务，包括完全托管的Redis Cloud服务和对Redis Labs企业集群的支持，以及其他服务。
