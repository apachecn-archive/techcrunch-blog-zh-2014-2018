# TC 制造商:Kinova 机器人公司向残疾人伸出援助之手

总部位于蒙特利尔的 Kinova Robotics 的成立部分是因为一个人——创始人的叔叔张博，一个残疾的发明家，他用热狗钳、一些微型开关和一些复杂的电子设备为他的轮椅创造了一个机械手。这个早期的尝试虽然很原始，但却是一系列令人惊叹——而且非常有用——的机器人手臂的基础。

Kinova 的首席执行官 Charles Deguire 带着 [TC 制造商](https://web.archive.org/web/20230128094812/https://techcrunch.com/tag/tc-makers)参观了他在城外的小工厂，在那里他和他的团队正在尽可能快地制造机械臂。这些手臂适合标准轮椅，允许人们自己进食，操纵物体，过独立的生活。目前，该团队在世界各地销售武器，目前在荷兰有超过 150 名用户使用张博喂自己。

这些机械臂重量轻、成本低，他们正在自己的工厂里制造整个模块化系统。他们每天可以制造大约五个机器人，目前正在研究一种新的模型，以使东西更轻。研究人员还将该系统用于机器学习和机器人研究。

Kinova 是一家令人惊叹的公司，它做着令人惊叹的事情:它以令人惊叹的方式向残疾人伸出援助之手。
