# Splunk 以 1.9 亿美元收购安全初创公司 Caspida

Splunk 今天晚上宣布以 1.9 亿美元收购了帕洛阿尔托的初创公司 [Caspida](https://web.archive.org/web/20221205131150/http://caspida.com/about) ，该公司使用机器学习技术来帮助识别来自公司内外的网络安全威胁。该公司报告称，交易已经完成。

Splunk 帮助公司使用数据科学技术和自动化来理解来自 IT 系统的机器数据的冲击。据 Splunk 安全市场的宋海燕·SVP 称，其中一部分是不断增长的安全业务，该业务在最近一个季度占公司收入的三分之一。

该公司有一款名为 [Splunk 企业安全应用](https://web.archive.org/web/20221205131150/http://www.splunk.com/en_us/solutions/solution-areas/security-and-fraud/splunk-app-for-enterprise-security.html)的安全产品。通过这次收购，Splunk 为其安全武器库添加了一个新工具，以增强使用 Caspida 开发的机器学习技术定位威胁的能力。

像所有人一样，Splunk 在过去的一年里目睹了越来越多的违规事件，其客户一直在寻求更好的安全检测工具来帮助应对这些威胁，其中许多都是通过泄露的凭据来使用的。这种攻击很难用寻找签名或规则的传统安全技术来检测。如果有人使用有效凭证从前门进入，则没有规则或模式。他们看起来像一个有效的用户，宋解释说。

这就是为什么您需要应用数据科学来更密切地观察每个使用系统的人的使用情况，即使他们看起来是具有适当凭证的有效用户。Splunk 的一般方法是一种数据科学驱动的解决方案，它提供自动检测，并使用机器学习来随着时间的推移变得更加智能。

这是 Caspida 为 Splunk 提供的最后一块，填补了其安全工具包中的一个大漏洞。这两家公司在硅谷相距不远，35 名 Caspida 员工将立即加入 Splunk。

Caspida 于 2014 年推出，并于去年年底推出了首款产品。它有付费客户，即使它成为 Splunk 团队的一部分，它也将继续服务他们。宋指出，Caspida 现在拥有 Splunk 销售组织和客户基础的支持，它在 Splunk 内部的增长速度应该会比它自己快得多。届时，Caspida 团队将更加全面地融入 Splunk

根据 Crunchbase 的数据，Caspida 自推出以来已经筹集了 1150 万美元。1.9 亿美元的标价似乎能给投资者带来丰厚的回报。
