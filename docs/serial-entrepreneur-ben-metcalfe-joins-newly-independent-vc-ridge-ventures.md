# 连续创业家本·梅特卡夫加入新独立的风险投资公司

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/10/20/serial-entrepreneur-ben-metcalfe-joins-newly-independent-vc-ridge-ventures/>

Ridge Ventures 是旧 IDG Ventures USA 的新化身，作为一家新的独立风险投资公司，它在通往未来的道路上雇用了一名重要的新员工。它的第一任负责人是曾在 WP Engine 和优步工作的[本·梅特卡夫](https://web.archive.org/web/20221207212515/https://twitter.com/dotBen)。该公司表示，作为一名企业家、天使投资人、软件工程师、产品开发人员和运营商，他拥有非常丰富的经验。

梅特卡夫是创办 BBC 新闻在线的早期技术团队的一员，共同创立并扩展了 WP Engine——一家成功的 SaaS 初创公司，拥有 70，000 名客户和超过 500，000 个网站依赖于它——他为优步开发了无障碍产品。

作为投资者，他还实施了战略营销计划，评估了技术解决方案和商业模式。在 WP Engine，他帮助公司实现了数千万美元的收入，同时为产品、销售、营销和合作伙伴关系做出了贡献。梅特卡夫是名副其实的万事通。

他对 TechCrunch 说:“我一直是一名技术专家和操作人员。所以对我来说这是一个有趣的举动。如果我看我的职业生涯，感觉是一场用户的持久战。来自 BBC 数据，优步，聋人和重听司机，WP 引擎和民主化出版。所以对我来说，为用户而战也可以是坐在桌子的另一边，希望改善创始人的体验。这就是我加入 Ridge Ventures 的原因。”

梅特卡夫说，他想在“一家我能做出更大贡献的小公司，而不是一家更难的大公司。”

“我喜欢 B2B Saas，因为它们总是值得投资的有趣公司。我是一名软件工程师，做过产品和公共政策方面的工作，由于这种不同的经历，我对很多事情都感兴趣。”

Ridge Ventures 在一份声明中表示:“本提供了我们欢迎和重视的背景和新鲜视角:他比大多数投资者更了解当今创始人的需求，利用他在没有正规大学背景的情况下学习技术和拓展业务的时间。他还帮助扩大 Ridge 在年轻、雄心勃勃的创始人中的人脉，这样我们就可以继续与最优秀、最聪明的人合作。”
