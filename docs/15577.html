Stitch Fix挑战赔率，自平淡无奇的首次亮相以来飙升超过50% TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2017/11/27/stitch-fix-defies-odds-soars-over-50-since-lackluster-debut/?ncid=rss

自 Stitch Fix 在股票市场首次亮相以来，它已经上涨了近54%。这对时尚造型行业来说是一个惊人的壮举，开局不利，但很快扭转了局面，在交易的第三天开始获得动力，今天飙升24 ，更好地被称为“网络星期一”

对于那些没有观看这场备受期待的IPO的人来说，Stitch Fix最初将其发行定价为15美元，低于其18美元至20美元的拟议价格区间，即使在缩减其发行规模后也是如此。该股首日收于15.15美元，低于16.90美元的开盘价。第二天，它“破发”，以低于15美元的发行价收盘。所有这些都远低于该公司去年对其回购估价的22.61美元。

然而，上周二，Stitch Fix改变了方向，从此一帆风顺。乐观情绪的一种可能性是，投资者预测电子商务尤其是这项业务将迎来一个大的假期季度。亚马逊的股票本周也上涨了。

吉恩·蒙斯特曾是Piper Jaffray的分析师，现在是Loup Ventures的风险投资家，他说“可能是几个大买家，图表开始工作，人们觉得他们错过了什么。”

他认为这主要归结为简单的供给和需求。Stitch Fix做了所谓的“低浮存”，这意味着它使可供出售的股票数量相对较少——这是新上市公司的常见情况。

不过，尽管IPO后的几天经常会出现波动，但统计数据显示，Stitch Fix的情况非常不寻常。明斯特说，他“不记得什么时候发生过这样的事情。”

根据Dealogic的数据，首日交易持平，或在IPO价格上下1%以内的美国上市公司，在IPO后一个月平均仅上涨1.6%。相比之下，同期其他新上市公司的涨幅为16%。

这种相关性在更长时间内持续存在，首日表现平平的公司比上市首日上涨10.8%。其他美国上市公司从首次公开募股到接下来的12个月期间平均增长20%。

这就是为什么在第一天会有这么多的审查“流行”，这个术语用来描述预期在第一天发生的收益。投资银行家建议公司将首次公开募股的价格定在低于其预测的首日表现15%到30%的水平，有时被称为“折扣”,以鼓励新投资者购买股票。这也是为了防止股票在接下来的几周或几个月下跌。

由于“pop”的重要性，银行家们经常行使所谓的“绿鞋期权”，他们可以在最初30天内以IPO价格回购股票以保全面子。看来他们在第一天就这样做了。在这个案例中，银行家们被允许购买 多达120万股的股票。

虽然现在宣布Stitch Fix取得长期成功还为时过早，但它的短期表现不容乐观。事实上，IPO ETF复兴资本(Renaissance Capital)的经理凯瑟琳·史密斯(Kathleen Smith)称这种转变“罕见”。她说，通常情况下，“打破你的IPO价格会留下印记。”

Stitch Fix的最大优势在于，它建立了一个快速增长的时尚业务，而且几乎没有资本。据报道，该公司从Benchmark Capital、Lightspeed Venture Partners和Baseline Ventures筹集了4200万美元。

在短短六年时间里，联合创始人兼首席执行官卡特里娜·莱克(Katrina Lake)建立了一家年收入9.771亿美元的公司。根据纳斯达克的数据，她也是有史以来最年轻的上市女性。

Stitch Fix已经在一些年实现了盈利，尽管它在2017年出现了59.4万美元的小幅亏损。去年，Stitch Fix的净收入为3320万美元。

Stitch Fix已经成为越来越多的先试后买时尚行业的领导者。该公司的算法和人类造型师的组合有助于确定哪些服装应该发送给常客。

当然，也总有看跌一只股票的理由。在这种情况下，一些IPO投资者在Blue围裙的股市暴跌中受到了伤害，起初可能对任何看起来像认购交易的东西都很警惕。(了解IPO价格的人与典型的散户投资者是不同的群体。这是一个由机构投资者和对冲基金投资者组成的排他性团体，他们被期望坚守岗位以保持股票稳定)。

我们还被告知，一些投资者最初担心联合创始人兼首席执行官卡特里娜·莱克(Katrina Lake)计划在IPO时出售股票，这种做法有时被解读为内部人士对公司的长期表现不太有信心的信号。

莱克在缩减IPO规模后取消了这些计划，她说她认为公司的估值应该超过15美元。

其他担忧包括该公司短暂的“锁定”期，根据股票表现，35%的内部股票可能在90天后有资格出售。公司通常会让员工等到第180天，因为抛售可能会导致股价下跌。此外，首席财务官保罗·易(Paul Yee)到公司才五个月，公司的首席运营官迈克·史密斯(Mike Smith)也是10月份才刚刚升职。我们听说这吓坏了一些IPO买家。

也有人担心增长减速和客户获取成本高。

不管怎样，看起来Stitch Fix的早期收益是一份“圣诞礼物”，史密斯说，他称之为“看到股票实现这些复苏令人鼓舞。”
