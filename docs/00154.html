货币云，一个基于 API 的资金转移者，又募集了 1000 万美元 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2014/04/15/the-currency-cloud-an-api-based-money-mover-raises-another-10m/

有消息称脸书一直在与英国的金融服务初创公司讨论推出电子资金服务，其中一家为这些初创公司提供支付服务的公司已经筹集了一轮资金。货币云提供了一个面向终端用户的公司用于跨货币转账的 API，已经从现有投资者 Atlas Venture 、 Anthemis Group 、opinion Capital和 XAnge Private Equity 那里筹集了 1000 万美元的 B 轮融资。硅谷银行也提供了一笔未具体说明的额外资金。

Currency Cloud 的首席执行官迈克·拉文(Mike Laven)表示，这笔资金将用于在国际上扩展总部位于伦敦、以欧洲为主的业务，特别是进入美国

目前，这家初创公司每月处理约 4 亿美元的支付，支付处理目前以每年 12 倍的速度增长。

然而，这方面的利润相当小:去年(2013 年)的收入总额为 300 万美元，他表示，他预计今年的收入将增长 150%-200%(换句话说，收入将达到 900 万美元。像许多其他金融服务创业公司一样，如 Stripe，另一个基于 B2B/API 的游戏，这个游戏是规模经济之一。

货币云是做什么的？该公司有效地与银行竞争，从金融服务公司赢得业务，这些公司的业务是为他人转移和兑换货币，无论是个人还是其他企业。银行目前控制着大约 85%的国际支付转移。

像许多寻求颠覆传统玩家的创业公司一样，货币云的优势在于它更小，因此在增加新功能和适应与他人合作方面更快；以及使用基于 API 的技术来集成和处理在线业务。

“我们理念的一部分是非常有利于创业，”拉文说。“从这个意义上来说，我们会与其他合作公司交流，看看我们如何才能在整个市场上实现全球化。”他说，货币云做的一部分是投资自动化和对账，作为 API 的一部分，这是它区别于银行的另一种方式。

它的客户包括昨天在脸书报告中被点名的所有三家公司——transfer wise、 Moni Technologies 和Azimo——以及支付方面的 Payoneer 和 Kantox 等其他公司；像 Sofort 、zipcard这样的电子商务提供商；甚至还有像 Fidor Bank 和 MedBank 这样的网上银行。

拉文告诉我，虽然他的公司肯定接触过脸书接触过的公司，但就目前而言，脸书似乎很乐意与外包合作伙伴合作提供服务，而不想自己承担任何相同的职能——也就是说，成为货币云这样的公司的直接客户。

“我们不是脸书对话的一部分，”Laven 说。

尽管在线交易领域另一家基于 API 的初创公司 Stripe 最近也开始采取措施扩大其国际范围，并且也将自己定位为商家减少与银行交易的一种方式，但 Laven 将他的公司所做的描述为对这家总部位于加州的支付初创公司的“补充”。

“他们站在消费者和商家之间，而我们是 B2B，在商家和其他商家之间，”他说。“或者，在 TransferWise 或 Azimo 的情况下，我们与他们打交道，而不是他们的客户。”

自 2012 年以来，最新一轮货币云筹集的资金总额达到 1900 万美元。
