在收益之前，微软的开发者前景喜忧参半

Developer Economics最近的一份报告详细描述了微软Windows Phone和Windows 8平台的复杂情况。微软将于今天盘后公布其收益，这使得调查结果更加有趣。微软将如何框定其在吸引开发者注意力方面的进展，其笔记会与既定数字一致吗？

报告中的关键图表跟踪了使用每个平台的开发人员的百分比，这是一个被称为“移动平台思想份额”的指标Windows Phone显示出持续的增长，而Windows 8在接受调查的开发者中的受欢迎程度有所下降。

Windows Phone的“思想共享”:

  • 2013年第三季度:21%
  • Q1 2014年:26%
  • 2014年第三季度:28%

Windows 8的“思想共享”:

  • 2013年第三季度:20%
  • Q1 2014年:21%
  • 2014年第三季度:18%

对于Windows Phone来说，增速较慢，但仍是实实在在的增长。对于Windows 8，向后。

微软正致力于更好地统一其平台和开发者体验，以鼓励应用程序创建者在其基于Windows的平台上进行开发，但这些努力仍处于开发阶段。在某种程度上，我们可以看到Windows Phone开发者推动Windows应用开发的势头。更奇怪的事情发生了。

微软将在今天盘后公布收益。苹果也是。期待烟火。
