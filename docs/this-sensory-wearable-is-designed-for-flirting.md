# 这种感官可穿戴设备是为调情而设计的

如今，当人们在公共场合闲逛时，他们更有可能盯着智能手机屏幕看，这款羽毛状可穿戴设备背后的伦敦设计师担心，与路过的陌生人交换调情目光的机会正在被设计出日常生活。或者，让我们诚实地说，翻译成对 Tinder 等人的可货币化的打击。

他们对智能手机窃取我们的存在和周边视觉的回应是一种叫做 [Ripple](https://web.archive.org/web/20221209205434/https://www.feelripple.com/#home-section) 的感觉设备，它代表佩戴者寻求赞赏的目光。尽管，考虑到佩戴者会在肩膀上戴着看起来像两只巨大颤抖的银尖海葵，它们完全有可能获得比平均数量更多的侧眼。然而，谁知道时尚的未来会是什么样子呢？更别说约会的未来了…

该项目由伦敦帝国理工学院和皇家艺术学院创新设计工程联合双硕士课程的四名设计师完成，将传感器和计算机视觉技术与一系列细长的突起相结合，当通过一对板载摄像头检测到相互吸引时，这些突起就会颤动。

他们写道:“Ripple 是你身体未来约会的可穿戴延伸，它可以计算房间里谁被你吸引。”“当它找到某人时，它会给你感官上的反馈，反映出你遇到某个特别的人时的兴奋感。如果吸引力是相互的，那么它的触角会随着他们的注视而移动，放大两个人之间的诱惑语言。”

这种原型设备通过向佩戴者背部发送波纹状的感觉来告知佩戴者他们正在被注视。此时，佩戴者可以转动身体来确定房间里谁在偷窥他们。当该设备检测到他们正在看着看着他们的人时，他们会收到一个胸部轻拍以确认。如果他们继续寻找，涟漪会继续荡漾。
