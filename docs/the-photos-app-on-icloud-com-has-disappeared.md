# iCloud.com 上的照片应用已经消失了

iCloud.com 上照片应用的测试版已经消失，[几个](https://web.archive.org/web/20221207144358/http://9to5mac.com/2015/01/02/photos-app-icloud-missing/) [站点](https://web.archive.org/web/20221207144358/http://www.imore.com/photos-section-disappears-icloudcom)今早[报道](https://web.archive.org/web/20221207144358/http://appleinsider.com/articles/15/01/02/icloudcom-photos-beta-disappears-as-users-continue-to-await-os-x-iphoto-replacement)，我们也通过测试确认。此前，该应用允许苹果用户通过网络登录，在 PC 或 Mac 上访问他们的 iCloud 照片库。这款应用不仅在 iCloud.com 主页上不见了——直接访问[网址](https://web.archive.org/web/20221207144358/http://icloud.com/#photos)也会显示一条错误信息*“无法加载照片。”*

在错误出现之前，该应用程序首先尝试加载一分钟，然后向用户提供向苹果报告问题的选项，或者只需单击“确定”来消除消息。

目前还不清楚失踪是否与故障有关，或者照片应用程序本身即将发生的变化——可能是在 2014 年 WWDC 期间承诺的 OS X 即将发布的照片应用程序之前。[苹果一直在开发](https://web.archive.org/web/20221207144358/https://beta.techcrunch.com/2014/06/27/apple-to-cease-development-of-aperture-and-transition-users-to-photos-for-os-x/)一款新的照片应用，该应用将集成 Aperture 中的许多高级编辑功能，Aperture 现已停产。新的照片应用程序也将取代 OS X Yosemite 上的 iPhoto。许多人现在猜测，照片应用在 iCloud.com 的消失与这种预期的转变有关。

然而，[其他人报告说](https://web.archive.org/web/20221207144358/http://9to5mac.com/2015/01/02/photos-app-icloud-missing/)他们已经好几天无法访问网络上的照片应用程序了。这使得该应用的消失看起来更像是一个正在被纠正的小故障，而不是与新版本相关的变化——因为向新服务的过渡肯定不会涉及长达一周的中断，对吗？

我们还听到一些人说照片应用程序现在可以工作了，这也表明这是一次宕机，而不是计划内宕机。

苹果公司尚未就该应用的消失发表任何官方声明，尽管我们已经联系了该公司，希望该公司能提供一份声明或解释。

**更新**:美国东部时间下午 3:20——照片应用已经回归，仍然带有“beta”标签。
