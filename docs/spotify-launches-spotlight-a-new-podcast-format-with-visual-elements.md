# Spotify 推出“Spotlight”，这是一种带有视觉元素的新播客格式

Spotify 一直在努力发展其播客业务，现在提供了一个大型节目目录(包括我们自己的[原创内容](https://web.archive.org/web/20230103014946/https://open.spotify.com/show/0oVMB8XsjNmr1eY8xd3sPO?si=Aev40KkDSci0rOYDiZg2Kg)，你一定要看看)。今天，它推出了一种新的“多媒体格式”，为播客(以及有声读物、新闻片段和其他音频)添加了视觉元素，但为了简单起见，我们姑且称之为“播客”。

Spotlight 的推出将展示 Gimlet Media、Crooked Media、Cheddar、BuzzFeed News 和其他网站的内容，还将包括 Spotify 自己创作的原创系列，如 RISE、Secret Genius 和 Spotify 与各种艺术家的单曲系列。

Spotify 的原创内容一开始并不出彩(单曲是个例外)，但这种新形式可能有助于推动它的发展。这也可能有助于 Spotify 将其播客产品与苹果播客等传统领导者区分开来，苹果播客在整体市场份额方面仍占主导地位。

Spotlight 将提供播放列表格式的内容，这些视觉元素可以在使用 Spotify 应用程序的设备上使用。这有点像一些创作者花时间构建的富媒体 MP3 播客，它具有动态变化的专辑插图，以及嵌入的链接、章节标记等。然而，并非每个播客播放器都支持这些功能，所以播客制作人很少会花时间内置这些功能。除了音频，Spotify 的聚光灯还将包括照片、视频和文本，并首先向美国用户推出。

这不要与 Spotify 在 2013 年推出的同名功能“Spotlight”混淆，Spotify 也使用播放列表，但它是一种非常不同的东西，旨在提供音乐和艺术家评论。

Spotify 对播客的实验很有趣，但接受和采用将是真正的考验。我们希望他们能真正做到，因为播客格式可以做一些创新，以帮助它达到更广泛的受众。
