# Unbound 为女性性健康募集了 270 万美元 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/12/22/unbound-raises-2-7-million-for-womens-sexual-wellness/>

女性性健康创业公司 Unbound 最近从 Founders Fund、Slow Ventures、Arena Ventures、SoGal Ventures 等公司筹集了 270 万美元。由 SexTech 妇女组织的 Polly Rodriguez 创立的[unbounded](https://web.archive.org/web/20230131160505/https://unboundbabes.com/)旨在赋予女性拥有自己性的权利。

自由的面包和黄油有三层:与性健康、健康和快乐相关的内容，个人性玩具产品和一个由策划产品组成的[季度盒子](https://web.archive.org/web/20230131160505/https://unboundbabes.com/products/unbound-quarterly-subscription)。Unbound 最初是一家像振动器和润滑剂这样的性用品的第三方销售商。最近几个月，它一直在开发和销售自己的[内部产品](https://web.archive.org/web/20230131160505/https://unboundbabes.com/collections/the-babe-pack)。未来的目标是完全脱离第三方转售业务。

“从销售第三方产品到主导自己的产品的转变并不总是最顺利的，”她说。"在接下来的一年里，最理想的情况是，大部分产品都将在室内生产."

目前，在 Unbound 上出售的大部分是振动器。该公司还销售捆绑产品，如捆绑盒、g 点盒、扭结盒和更年期盒。到今年年底，Unbound 预计将实现约 200 万美元的销售额。

在某个时候，Unbound 还将扩展到实体领域，更加以体验为导向。罗德里格斯设想自由成为类似计划生育 2.0 的东西，在那里人们可以进行乳房 x 光检查、子宫颈抹片检查和其他“女性害怕的事情”。

“女性的性健康和健康可以成为我们不害怕的事情，”她说。

在与癌症和辐射引起的更年期进行斗争后，罗德里格斯决定创办这家公司。在那段时间里，她说她想确保自己能掌控自己的身体和性。

但她说，购买润滑油的经历让她“感到窘迫”。"那是一次可怕的经历，让我难以忘怀。"

除了赋予女性权力和解决女性特有的问题，罗德里格斯还希望围绕女性健康和性教育公众。
