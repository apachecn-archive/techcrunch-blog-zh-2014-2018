Twilio 将很快推出 Flex，这是一款专门的联络中心解决方案 TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/02/16/twilio-will-soon-launch-flex-a-dedicated-contact-center-solution/?guccounter=1

TechCrunch 获悉，Twilio 的参与云，即其用于建立新客户体验的产品套件，即将获得新功能。根据我们今天下午收到的消息，该公司计划在 3 月份的企业连接会议上推出针对企业的完整联系中心解决方案的测试版。当被联系确认时，Twilio 没有发表评论。

随着该产品的推出，Twilio 可能会与一些向企业销售联络中心解决方案的现有客户展开竞争。在我们看到的一份关于即将推出这款新产品的内部电子邮件中，该公司显然旨在避免这种印象，但这很可能是因为它担心这个市场上目前的玩家会如何看待这一举动，这些玩家可能自己也在使用 Twilio 的一些服务。

到目前为止，Twilio 将其各种 API 定位为开发新的呼叫中心解决方案的基础。有了 Flex，它现在基本上将这些捆绑在一起，使开发人员更容易构建这些服务。

我们的理解是，这项服务目前的名称 Twilio Flex(尽管在发布日期之前可能会有所变化)将追随该公司大多数产品的脚步，因为它将强调开发人员的体验。例如，它将允许系统集成商在 Flex 的基础上构建定制的联络中心解决方案。

Twilio Flex 将为这些组织的劳动力管理和劳动力优化套件(即所有常见的联络中心产品，如电话录音、代理教练、语音分析等)提供基本的构建模块，以增强通信体验、单点登录和集成点。)，以及与其后台员工调度系统的集成。

顾名思义，Twilio 将这项服务定位为一个非常可定制的解决方案，尽管这也意味着需要客户进行一些额外的集成工作才能使其工作。然而，Twilio 认为，正是这种定制将使企业优化他们的联络中心。

根据我们的消息来源，该公告目前定于 3 月 12 日，即奥兰多企业连接会议的第一天，该会议主要关注联系和呼叫中心市场。
