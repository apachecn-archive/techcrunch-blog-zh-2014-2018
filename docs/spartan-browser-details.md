# Windows 10 可以同时搭载 IE11 和 Spartan

昨天，微软[宣布【Windows 10 将包括一个新的浏览器——目前代号为“斯巴达”不过，当时它很少透露细节，今天早上它在](https://web.archive.org/web/20221231010001/https://techcrunch.com/2015/01/21/microsofts-browser-with-a-halo-codename/)[贴出了一份更详细的描述](https://web.archive.org/web/20221231010001/http://blogs.msdn.com/b/ie/archive/2015/01/22/project-spartan-and-the-windows-10-january-preview-build.aspx)，纠正了这个问题。

例如，我们现在知道的是，微软将继续为那些仍然需要 Active X 等一些遗留功能的人提供 Internet Explorer 10。Windows 10 上的 IE 将支持微软的新旧渲染引擎，Spartan 在遇到遗留网站时也可以依靠旧的 IE 引擎。

这听起来很令人困惑，但是 Spartan 将会是默认的浏览器，它将会有一个更新的界面和许多 IE 所没有的改进。Spartan 也是一款通用应用，旨在适用于从智能手机到台式机的所有 Windows 10 设备。很有可能，大多数普通用户不会在 Windows 10 上看到旧的 IE，微软只是为那些真正仍然需要它的企业用户保留它。

![2015-01-22_1022](img/868c64a2a01afc4b510e0426ed4cad1e.png)

目前，尚不清楚 IE 是否仍将随 Windows 10 一起发布，或者是否将是一个单独的下载。不过，Windows 10 肯定会支持这两者。默认安装两个微软浏览器看起来相当混乱(但是我们在这里谈论的是微软)。

那么是什么让斯巴达与众不同呢？微软表示，重点将放在互操作性和支持现代网络标准上。虽然下一个技术预览版不会采用 Spartan，但它将使用其渲染引擎。自从上一次发布预览版以来，微软已经对其进行了许多更改，并更新了内置的开发人员工具。

微软尚未发布任何关于该浏览器的性能细节。在这个版本中，这似乎不是重点。相反，它更多的是方便的功能，如新的无干扰阅读模式和 Cortana 集成，以及标注网页的能力。
