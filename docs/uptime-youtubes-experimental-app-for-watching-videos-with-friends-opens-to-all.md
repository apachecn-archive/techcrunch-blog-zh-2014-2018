# YouTube 用于与朋友一起观看视频的实验性应用 Uptime 向所有人开放 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/06/26/uptime-youtubes-experimental-app-for-watching-videos-with-friends-opens-to-all/>

YouTube 的实验性应用程序 [Uptime](https://web.archive.org/web/20230111064015/https://uptime.area120.com/) ，可以让你和朋友一起观看视频，同时做出反应和评论，现在已经向所有用户开放。该应用于今年三月首次在 T2 推出，来自谷歌的内部孵化器 Area 120，作为一种测试观看 YouTube 的更具互动性和社会性的方式。

然而，它最初需要一个邀请才能使用。几天前，这项要求被取消了。

Uptime 让你可以直接在应用程序中与朋友一起观看视频，而不是独自观看 YouTube 视频，然后通过聊天或社交网络的链接与朋友分享你喜欢的视频。朋友们可以实时与你一起观看，或者他们可以稍后加入，看看其他人在观看时对播放的视频的反应，即使你独自观看，也能给正常运行时间带来生动和互动的感觉。

这些反应——观看视频时点击的表情符号——受到了其他直播视频服务的启发，例如脸书直播或 Twitter 的 Periscope。有笑脸，惊讶的脸，心形的眼睛，愤怒的脸，鬼脸，和哭泣的脸。你也可以点击视频来留下“火花”，但这些只会显示给你实时观看的人。

![](img/4ba262eb1acef34b7b8cc7294f52dcc7.png)

自今年早些时候推出以来，Uptime 增加了新功能，包括脸书朋友查找功能，支持观看音乐视频，并改进了主屏幕，使其更容易找到值得观看的内容。

然而，提供共同观看体验的想法并不是 Uptime 独有的。这是时下的热门话题，许多应用程序正在进入这一领域，包括 Tumblr 的 [Cabana](https://web.archive.org/web/20230111064015/https://techcrunch.com/2017/04/18/tumblr-launches-cabana-a-new-app-for-watching-videos-with-friends/) 、[Let ' s Watch](https://web.archive.org/web/20230111064015/https://techcrunch.com/2017/04/14/lets-watch-it-allows-friends-to-watch-react-to-videos-together-on-imessage/)、 [Fam](https://web.archive.org/web/20230111064015/https://itunes.apple.com/us/app/fam-group-video-calling-for-imessage/id1185328193?mt=8) 等。甚至 Skype 也表示，这是它计划在未来推出的一项功能，从支持 YouTube 开始。

正常运行时间实验现在是从谷歌的 120 区出现的几个实验之一，它让有创业精神的谷歌员工尝试新想法。该计划已经推出了其他项目，包括个人造型师应用[裁缝](https://web.archive.org/web/20230111064015/https://itunes.apple.com/us/app/tailor-personal-stylist-shopping-assistant/id1165603432?mt=8)，语音信使[超声波](https://web.archive.org/web/20230111064015/https://itunes.apple.com/us/app/supersonic-fun-voice-messenger/id1178838430?mt=8)，以及学习编码应用[蚱蜢](https://web.archive.org/web/20230111064015/https://play.google.com/store/apps/details?id=com.area120.grasshopper)；[另一个 app](https://web.archive.org/web/20230111064015/https://techcrunch.com/2017/04/26/appointments-is-the-latest-app-from-googles-internal-incubator-area-120/) ，一个叫预约的沙龙预约工具，还没有公开推出。

然而，Area 120 应用程序在应用商店中并没有打上谷歌的商标，谷歌也没有在推广方面提供太多帮助。相反，这些应用一般都是由记者[发现](https://web.archive.org/web/20230111064015/https://techcrunch.com/2017/04/26/appointments-is-the-latest-app-from-googles-internal-incubator-area-120/)的，没有任何正式的发布声明。

由于这个事实，Uptime 还没有机会获得很多用户，因为它以前需要邀请才能测试。它在上周悄悄地取消了邀请代码，这意味着现在任何人都可以下载和使用该应用程序。尽管存在这些障碍，这款应用还是设法在 App Store 的排行榜上短暂占据了一席之地——在娱乐类别的巅峰时期排在第 403 位，此后一直下滑。这听起来不起眼，但对于一个基本上处于预发布模式、没有广泛开放的应用程序来说，它的排名值得一提。
