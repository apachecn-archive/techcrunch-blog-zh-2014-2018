一个智能键盘专业版可以让你轻松地弹钢琴

虽然ONE Smart Keyboard Pro没有甜美的演示曲也不能在没有帮助的情况下演奏吠叫狗铃儿响叮当，但它可以教你或你的孩子如何弹钢琴。优雅的键盘有88个加重键，模拟真正的机械钢琴，并连接到您的手机，以便您可以按照自己的节奏学习演奏。

Keyboard Pro售价799美元，本质上是一款紧凑型教学键盘。它可以通过一根奇怪形状的USB B线连接到你的iOS或Android设备，一旦它与应用程序配对，你就可以播放简单的歌曲——比如绿色袖子——和更复杂的乐谱。这种键盘是有重量的，但不是渐进的，这意味着每个键提供相同的阻力，这对于一些更有经验的玩家来说是很重要的。此外，您可以连接USB电缆并将键盘连接到电脑，以将其用作MIDI控制器。

同样，这是一个非常简单的键盘。除了教你怎么玩之外，它没做什么，毕竟，这是我们大多数人所需要的。因为它没有卡西欧那种花哨的功能，而且大多数智能都在应用程序本身，所以对大多数人来说有点难以接受。然而，如果你想学习，这个方法很有效。

这款更大更全的One智能键盘提供了优质的工艺和设计。除了键盘前面的电源按钮和音量之外，整个系统出奇地稀疏。有一个延音踏板的输入和几个耳机输出插孔，仅此而已。不要指望用这个键盘来挑选乐器或变调。一旦你启动应用程序，你就可以访问教学练习和游戏，让你在LED键盘上跟着歌曲和音阶跑。最后，你可以花3.99美元左右购买活页乐谱，这样你就可以在上面学习演奏。对于那些想演奏一点古典音乐的人来说，还有免费的活页乐谱。

我发现整个系统非常有用，我的孩子们一旦知道如何放慢音乐速度，就马上开始学习小曲。没有什么能像人类老师一样教你如何弹钢琴——这个应用程序没有足够的智能来根据你的技能进行调整——但它相当于买一本自学钢琴的书，然后坐在奶奶的旧竖琴前。我对键盘的质量特别满意。这些年来，我已经有了一些MIDI键盘，包括卡西欧和雅马哈的型号，这一个与那些相当。正如我之前提到的，教学功能是这里的主要吸引力，因为使用这种开箱即用的键盘，你几乎无法做其他事情。然而，如果这就是你在键盘上寻找的东西，并且你不想采样身体的噪音，以便你可以在学校的才艺表演上演奏放屁的克莱尔月光，这可能是你的模型。
