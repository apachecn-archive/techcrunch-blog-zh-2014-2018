# 这种纹身般的展示是由一种新的超薄保护性“电子皮肤”实现的

这听起来像是《星际迷航》里的东西:一个比创可贴薄的贴片，你把它贴在手臂上，几分钟后，它会随着心率、血糖等指标亮起——几天后就会脱落。这是东京大学研究人员的工作目标，他们发明了一种超薄的“电子皮肤”，可以保护柔性电子器件，并使类似皮肤显示器的东西成为可能。

当然，电子产品以前就已经贴在皮肤上，或者用导电墨水画出[，但是即使是只有一毫米厚的基板，其柔韧性也有限。由乔雄·萨默亚和横田友之领导的研究将厚度降低了一个数量级。](https://web.archive.org/web/20221206115345/https://beta.techcrunch.com/2015/11/23/chaotic-moon-explores-biometric-tattoos-for-medicine-and-the-military/)

电子皮肤，也许是未知的，可以被描述为智能塑料包装。分层的亚氧化硅和聚对二甲苯封装在一个超薄衬底上，团队在其上安装了类似的薄而柔韧的有机发光二极管显示器和光敏二极管。整个装置会随着皮肤伸展和皱缩，但密封良好，可以在失效前持续几天——以前的(和更厚的)装置只能持续几个小时。

上面显示的显示就是这个概念的证明:它们需要电极，而不是由体热或柔性电池供电，当然还没有真正阅读任何东西。

“如果我们的展示能够附着在我们的身体上，甚至显示我们的情绪、压力或不安程度，这个世界会是什么样子？”萨默亚在随该报纸发布的《T2》新闻发布会上问道，该报纸于周五在科学进展发布于《T4》。

可能很恐怖。但也有大量合法的医疗和消费应用程序，像这样的研究将我们推向了这些应用程序。
