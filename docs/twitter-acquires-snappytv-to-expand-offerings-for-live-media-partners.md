# Twitter 收购 SnappyTV，为直播媒体合作伙伴提供更多服务

![snappytv-twitter-top](img/4df38e46db85d20d472532c53bcad5ce.png) Twitter [今天](https://web.archive.org/web/20230403225651/https://blog.twitter.com/2014/snappytv-is-joining-the-flock)宣布将收购 SnappyTV，这是一项近实时剪辑、编辑和分享直播视频的服务。Twitter 已经表示，它与 SnappyTV 及其一系列广播和媒体合作伙伴合作，这将有助于它继续投资于这些合作伙伴关系，并支持这一重要的客户群体。

SnappyTV 帮助 Twitter 快速、即时地提供大型体育赛事、红毯庆典和其他实时全球媒体活动的关键、高度可分享时刻的视频。这种合作关系使得这种内容可以直接嵌入推文中，这是帮助 Twitter 在其网络上捕捉和放大这一时刻的关键。

在 SnappyTV 网站的一篇[博客文章中，这家成立四年的初创公司背后的团队讲述了它如何与 Twitter Amplify 合作，以及如何通过有机增长向 Twitter 流提供直播和突发事件的剪辑。该公司表示，它将继续在 Twitter 的保护伞下维护和建设其平台，并表示有了 Twitter 的资源，它将能够发展和改善其产品。](https://web.archive.org/web/20230403225651/http://blog.snappytv.com/?p=2101)

SnappyTV 的客户包括福克斯、CW、纳斯卡、美国公开赛和 us、TechCrunch。正如其大多数客户一样，我们使用该服务来快速转变我们直播中的特定片段(如 TechCrunch Disrupt conference live stream ),以嵌入 Twitter、我们的博客和其他地方的帖子中。在后端，SnappyTV 提供分析，告诉你你的观点来自哪里，并帮助衡量你分享的剪辑的社交参与度。

对于 Twitter 来说，这是为了扩展它可以向直播内容制作者提供的工具集，这些制作者经常转向网络作为社交网络，可以团结和扩大分散的受众。Twitter 的 Deb Roy 是 Twitter 的首席媒体科学家，在去年伦敦的一次会议上，他讨论了 Twitter 与直播合作伙伴合作背后的[战略，SnappyTV 正好符合这些计划。](https://web.archive.org/web/20230403225651/https://techcrunch.com/2013/10/17/bluefins-deb-roy-sees-twitter-as-the-shared-semiotic-bridge-for-all-live-experiences/)
