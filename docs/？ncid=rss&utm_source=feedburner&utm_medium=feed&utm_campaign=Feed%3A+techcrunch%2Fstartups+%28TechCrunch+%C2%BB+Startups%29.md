# 触摸手术将手术训练带入增强现实

对外科医生进行特定手术程序的培训既昂贵又困难。总部位于伦敦的[触摸外科](https://web.archive.org/web/20221003052242/https://www.touchsurgery.com/)已经为在手机或平板电脑上完成的外科手术创建了 200 多个培训项目。在国际消费电子展上，它宣布支持一种新型的深度沉浸式外科手术培训，并有可能在手术室里提供帮助——在 [DAQRI](https://web.archive.org/web/20221003052242/https://daqri.com/) 和 [HoloLens](https://web.archive.org/web/20221003052242/https://www.microsoft.com/microsoft-hololens/en-us) 上。

“我们相信与外科医生合作，优化和扩大一流的手术程序，为全球患者提供培训和更安全的手术，”Touch Surgery 的首席执行官 Jean Nehme 博士告诉我。“迄今为止，我们的产出一直是通过移动设备，主要用于培训。2017 年，我们已经使我们的管道能够交付给增强现实平台。”

该公司本周在拉斯维加斯的消费电子展(CES)上首次展示了其增强现实内容。

[![touch-surgery-movemet](img/a4274265e9233fa554baad5faa4be17b.png)](https://web.archive.org/web/20221003052242/https://beta.techcrunch.com/wp-content/uploads/2017/01/touch-surgery-movemet.gif) 真正的机会是将触摸手术的培训内容从相对无菌的场景——能够在手机上点击和滑动手术模拟——转移到手术室。很容易想象，一个外科医生拿起手术刀和十几个其他锋利的器械，可能想要对他们即将开始的手术进行实时覆盖。

这听起来有点未来派，但我仍然相信外科医生在做手术时戴着全息眼镜。一方面，也许这减少了复杂手术中的错误数量，但另一方面，也许想到你的外科医生可能不太清楚他们在做什么会让人有点不安。此外，鉴于本文右侧的 GIF 循环，我开始对我使用的“一方面”习语产生一些严重的怀疑。

增强现实用于手术室仍处于早期阶段，但如果它使手术更安全、更快速或并发症更少，很难否认它的潜在用途。我说，带上全息透镜。
