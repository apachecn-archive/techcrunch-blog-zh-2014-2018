Windows 10将于7月29日上市

微软6月初宣布，Windows 10将于7月29日(T1)发布，距离现在不到两个月。MS的下一个重大更新旨在成为一个统一的平台，将在移动设备、桌面设备、平板电脑甚至Xbox One游戏主机之间提供连续性，并将为Windows 7和8用户提供第一年的免费升级。为了帮助确保轻松更新(可能也是为了尝试立即获得大量Windows 10用户)，微软还使用了一个“预订”系统，允许用户注册以获得更新可用时的通知，并安排在你想要的时候安装。

Windows 10的一个大的新功能是Cortana，所以这里有一条该公司准备的来自其虚拟助手ant的消息来宣布这一消息。

你还得到什么？我很高兴你问了这个问题——下面快速浏览一下亮点:

  • 微软Edge:Internet Explorer的继任者，围绕极简主义和协作工具设计，并集成了Cortana。
  • 内置Word，Excel，PowerPoint。
  • Xbox Live和Xbox app，用于录制游戏、与Xbox好友互动以及将Xbox游戏传输到桌面。
  • Windows Continuum，让你可以在多个Windows 10设备之间顺利切换，让你像使用PC一样使用手机和外部输入附件。
  • Windows Hello，一种新的登录方法，使用面部，虹膜或指纹识别，无需密码即可登录，具体取决于硬件支持。

微软正在尽一切努力使过渡顺利，比如确保它能与你现有的应用程序兼容。Windows 8是一个尴尬的一代，所以这里希望跳过的数字能够满足每个人的需求。
