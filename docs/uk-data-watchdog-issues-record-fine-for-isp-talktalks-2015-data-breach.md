# 英国数据监管机构对 ISP TalkTalk 的 2015 年数据泄露处以创纪录的罚款 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/10/06/uk-data-watchdog-issues-record-fine-for-isp-talktalks-2015-data-breach/>

2015 年，英国 ISP TalkTalk 的一次数据泄露导致该公司被美国国家数据保护局(national data protection agency)罚款 40 万英镑——这是 ICO 开出的创纪录罚款，尽管比目前的最高罚款额少了 10 万英镑。

问题中的漏洞可以追溯到 2015 年 10 月，当时黑客从其网站上窃取了近 157，000 个 TalkTalk 客户账户的数据。不久后，警方逮捕了两名涉嫌参与黑客攻击的十几岁男孩，但调查仍在进行中。据英国广播公司报道，总共有六人被捕。

虽然数据泄露的规模没有最初想象的那么大，银行账户或部分信用卡信息被窃取的 TalkTalk 客户的数量也更少(数万)，但黑客能够轻易穿透 ISP 的安全系统并窃取敏感数据，这导致了广泛的谴责。

黑客使用了一个针对易受攻击网页的 SQL 注入，TalkTalk 在收购另一家英国 ISP Tiscali 后接管了这些网页。同年，针对同一漏洞的两次早期攻击显然被 TalkTalk 忽略了。

在开出罚单后，信息专员 Elizabeth Denham 在[的一份声明中评论说:“TalkTalk 未能实施最基本的网络安全措施，这使得黑客可以轻而易举地侵入 TalkTalk 的系统。](https://web.archive.org/web/20221226032708/https://ico.org.uk/about-the-ico/news-and-events/news-and-blogs/2016/10/talktalk-gets-record-400-000-fine-for-failing-to-prevent-october-2015-attack/)

“是的，黑客行为是错误的，但这不是公司放弃安全义务的借口。TalkTalk 应该也本可以做得更多来保护它的客户信息。事实并非如此，我们已经采取了行动。”

在有关罚款金额的进一步评论中，她补充道:“今天创纪录的罚款是对其他人的警告，网络安全不是一个 IT 问题，而是一个董事会问题。公司必须勤奋和警惕。他们必须这样做，因为根据法律他们有责任，但他们也必须这样做，因为他们对他们的客户有责任。”

当新的[通用数据保护指令](https://web.archive.org/web/20221226032708/https://techcrunch.com/2015/12/16/gdpr-agreed/)于 2018 年 5 月生效时，欧洲的数据违规处罚将大幅增加。到那时，最高罚款将上升到一家公司全球营业额的百分之四(或€2000 万，如果更多)——想法是迫使公司优先考虑保护客户数据。

根据新的欧洲法律，大规模收集客户数据或处理敏感数据的公司还将被要求任命数据保护官，该官员将有义务向相关国家 DPA 报告任何数据泄露事件。《GDPR》让企业有责任迅速报告数据泄露事件，在大多数情况下，会在意识到这一点的 72 小时内报告。

虽然不清楚英国打算在离开欧盟后如何谈判遵守欧洲法律，但在今年夏天英国退出欧盟投票后，任何在欧洲有客户的英国公司仍需要遵守 GDPR。随着新制度的生效，在欧洲做生意的公司的安全支出似乎很可能会增加，以减轻其底线和声誉的通胀风险。

此外，特别是在互联网服务提供商方面，即将出台的英国法律将规定互联网服务提供商有义务[收集和存储所有客户全年的网站访问数据](https://web.archive.org/web/20221226032708/https://techcrunch.com/2015/11/04/cementing-uk-surveillance-state/)——根据有争议的[调查权力法案](https://web.archive.org/web/20221226032708/https://techcrunch.com/tag/investigatory-powers-bill/)，作为政府试图扩大国家安全机构和警察权力的一部分。

如果这一要求如期在今年通过成为法律，鉴于法律要求他们存储额外的敏感数据，英国互联网服务提供商很可能成为黑客更具吸引力的目标，正如该法案的[批评者](https://web.archive.org/web/20221226032708/https://techcrunch.com/2016/06/27/uk-surveillance-bills-logging-of-web-activity-a-huge-risk-to-privacy-peers-warn/)所警告的那样——[包括前英国 ICO](https://web.archive.org/web/20221226032708/https://techcrunch.com/2016/01/12/uk-surveillance-bill-a-risk-to-data-security-and-privacy-says-ico/) 。
