# Scaleway 现在提供非常便宜的类似 VPS 的服务器，每月 3.40 美元

Online.net 的 [Scaleway](https://web.archive.org/web/20230306024655/https://www.scaleway.com/) 是云托管世界中一只有趣的野兽。该公司没有建立一个虚拟云托管基础设施来直接与亚马逊网络服务、[数字海洋](https://web.archive.org/web/20230306024655/https://techcrunch.com/2014/03/06/digitalocean-raises-37-million-from-andreessen-horowitz/)和其他 VPS 提供商竞争，而是设计了自己的基于 ARM 的服务器。

这就是为什么该公司能把价格压得这么低。你现在可以以每月 3.40 美元(€2.99 美元)的价格获得 2GB 内存和 50GB 存储的裸机 SSD 服务器，这比 Scaleway 之前每月 9.99 美元的€价格便宜 70%。

该公司[在 Twitter](https://web.archive.org/web/20230306024655/https://twitter.com/online_fr/status/638751557014679552) 上表示，新的定价也适用于现有用户。提醒一下，由于 ARM v7 芯片组最初是为智能手机设计的，因此很容易使用非常少的电力、冷却和空间来运行其中的许多芯片组。Scaleway 设法将 912 台独立的计算机挤在一个服务器机架中。

该公司还保持了两个世界的优势——专用服务器具有虚拟化的灵活性，因为您可以获得 4 个专用 ARM 内核、一个专用 IP 和 200 兆位/秒的未计量带宽。与许多流行的 VPS 提供商相反，你不会与其他用户分享你的 CPU 原始能力。

除此之外，它的工作方式和其他人差不多。只需点击几下，您就可以[启动一个实例并](https://web.archive.org/web/20230306024655/https://www.scaleway.com/imagehub/)安装一个发行版或应用程序。其中一些应用已经可以用于 ARM 架构，但其中一些必须由 Scaleway 的团队移植才能在这些服务器上运行。你已经可以找到所有主要的发行版，从 Debian 到 Ubuntu 和 Fedora。

只需几次点击，你就可以添加更多的存储空间或与亚马逊 S3 整合。该公司只有一种服务器型号。如果您需要更多的 RAM 或 CPU 能力，您应该用现有的映像启动一个新的服务器，并使它与您的其他服务器一起工作。所有的服务器目前都托管在伊利亚特位于法国的数据中心。

起初，Scaleway 是一项[技术成果](https://web.archive.org/web/20230306024655/https://techcrunch.com/2014/11/13/online-labs-designed-its-own-arm-servers-to-take-on-aws-digitalocean/)。然后，当谈到云托管时，它成为了一个有前途的替代品。现在，凭借这种新的激进定价策略，它可能会成为一个强有力的竞争者。
