# 在移民禁令抗议中，一些科技高管正在匹配美国公民自由联盟的捐款

在一项禁止来自七个国家的难民和签证持有人入境的行政命令颁布后，科技高管们向美国公民自由联盟(ACLU)提供了相应的捐款。

由于科技公司的多名高管和首席执行官公开反对所谓的移民禁令，一些人选择在推特上提供与个人捐款匹配的服务。例如，早期的推特投资者克里斯·萨卡(Chris Sacca)就率先发起了这一趋势，并提出将捐款与那些直接传递信息或用收据回应的人相匹配。

此外，[谷歌创建了一个 200 万美元的“危机基金”](https://web.archive.org/web/20230320220333/https://techcrunch.com/2017/01/29/google-us-immigration-fund/)，员工最多可捐赠 200 万美元。根据该报告，该基金用于四个组织:美国公民自由联盟、移民资源中心、国际救援委员会和慈善团。谷歌高管们分别为这一事业捐款，尽管报告没有具体提到是谁。谷歌发言人证实了该基金的存在。

美国公民自由联盟(ACLU)对行政命令提出质疑，并赢得了紧急居留许可，允许持有效签证在美国入境(或过境)的人入境并居留。暂缓令是由纽约的一家联邦法院下达的。

该行政命令发布后，周六爆发了抗议活动，向科技行业的知名人士和高管施压，要求他们站出来反对移民禁令。而且，在很大程度上，T2 最大公司的高管们已经在内部或公开场合做出了某种声明。该订单尤其对谷歌和优步等公司提出了挑战，这些公司可能有在国外工作的员工，他们可能没有能力按照订单回到国内。

例如，优步表示，它将[设立一个 300 万美元的国防法律基金，以支付受禁令影响的司机的法律、移民和翻译费用。Lyft 首席执行官洛根·格林表示，该公司将在四年内向美国公民自由联盟捐赠 100 万美元，并强烈反对这一命令。](https://web.archive.org/web/20230320220333/https://techcrunch.com/2017/01/29/uber-apologizes-for-confusion-at-jfk-during-immigration-protest/)

对于硅谷的百万富翁和亿万富翁来说，这些数万美元的增量可能看起来微不足道(Sacca 是其中最高的)。但与此同时，它给了这些高管和社区的杰出成员一个以某种可衡量的方式出来反对禁令的途径，而且它显然成功地让人们向美国公民自由联盟捐款。
