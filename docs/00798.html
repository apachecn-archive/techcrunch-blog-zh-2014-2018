美国国家档案馆将所有藏品上传至维基共享资源

自从 2010 年国家档案和记录管理局启动开放政府计划以来，它越来越多地将内容上传到维基百科进行数字化，并获得更广泛的馆藏。但是，一直以来，将它的数字化馆藏上传到维基共享资源是一个次要的项目。现在，正如本月早些时候发布的 2014 开放政府计划中提到的，根据路标的说法，该项目是 NARA 的核心部分。

NARA 的数字内容专家 Dominic McDevitt-Parks 表示，将联邦政府的“档案保管员”上传到维基媒体的目的是向公众提供更广泛的服务。

2012 年，NARA 进行了试验，并上传了 10 万张数字图像到维基共享。这使得维基百科的编辑可以将这些文件整合到项目和文章中。

“2013 财年，介绍我们记录的 4，000 篇维基百科文章的浏览量超过了 10 亿次。根据《2014 年开放政府计划》，在接下来的两年里，我们将努力增加维基共享资源上可用的国家档案馆记录的数量，这将进一步推进我们的战略目标，即“让访问成为现实”，并扩大公众对我们记录的再利用。

通过提供对他们的藏品的访问，奈良也通过维基百科得到了更好的展示，因为它的内容到达了更多的观众。奈良目录中任何被数字化的物品都将被上传到维基媒体。

任何人都可以通过 Wikimedia Commons 上传文件，但麦克德维特-帕克斯表示，没有简单的拖放功能来上传批量文件，这是 NARA 必须应对的第一个障碍。

他们一直在使用维基媒体公共 API 开发一个更强大的上传脚本，计划在今年年底完成。

他说，就上传量而言，不担心维基媒体共享服务器会承受任何类型的压力，他说 NARA 还没有就维基媒体超载进行任何讨论。

这些上传的内容从平凡的联邦机构记录到二战照片和总统肖像。

维基媒体华盛顿分部负责几个项目，其中许多是与 NARA 和国会图书馆合作的。分会是独立的组织，并不直接隶属于维基媒体基金会。

维基媒体特区分部主席詹姆斯·黑尔说，几名维基百科志愿者一直在与 NARA 合作这个项目，将他们的收藏数字化。

“与维基媒体社区的联系一直是他们的优先事项……特别是自 2011 年以来，我们的关系开始有所发展，”黑尔说。

由于 NARA 只是世界上许多向维基媒体上传文件的机构之一，Hare 说维基媒体基金会有数兆字节的存储空间，应该能够毫无问题地处理上传。随着筹款工作的继续，空间将永远可用。

黑尔说:“联邦政府内部有一项命令，要求向公众提供数据，并发布关于政府所做事情的信息，特别是与维基百科和 WMDC 合作是一种很好的方式，因为与其把你的信息放在某个不知名的门户网站上，你必须把它放在人们真正阅读它的地方。”。
