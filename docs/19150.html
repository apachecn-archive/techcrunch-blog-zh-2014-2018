最近几个月，让用户买卖数万美元的区块链收藏猫的 CryptoKitties 时尚已经相当稳定了。这并不是说 CryptoKitties 没有产生大量的模仿者。不幸的是，CryptoKitties 的巨大受欢迎程度不太可能重现，至少不会被希望从区块链的 crypto collectables 的新奇事物中赚钱的克隆人重现。

CryptoKitties 的遗产仍在开发中，但大多数人都认为该项目提高了人们对区块链令牌新用途的认识(并吸引了开发人才)。特别是，CryptoKitties 引入了不可替代令牌(NFT)的概念，这可能会影响到加密货币以外的领域。

NFT 是唯一的区块链令牌，可以转让给其他人，类似于加密货币(如比特币和以太坊)，但它们通常不能被另一个等值的令牌取代，这是因为每个 NFT 都有自己唯一的令牌标识符(通常还有相关的参考元数据)。

今天，大多数 NFT 被用于区块链的收藏游戏；然而，对非功能性测试用例的探索才刚刚开始。本文简要讨论了 NFTs 的起源，探讨了区块链生态系统中几种类型的 NFTs，并强调了 NFT 开发者面临的一些潜在的法律障碍。

收藏来源

实物收藏品交易出现在 19 世纪 60 年代，出现了第一张棒球交易卡。从那时起，实物收藏品主导了收藏品市场。实物收藏品的所有权很简单:当收藏者购买实物收藏品时，收藏者拥有完全的所有权，可以随意出售或交易收藏品。收藏家只是被禁止侵犯收藏品的潜在知识产权。因此，收藏家可以购买，出售或交易，但不能复制，实物收藏品。T9】

在 20 世纪 90 年代末和 21 世纪初，游戏中的收藏品或虚拟资产出现了。虚拟资产是那些只能在虚拟世界中找到的商品。像实物收藏品一样，虚拟资产也有一个强劲的市场。然而，与实体收藏品不同，在虚拟世界中找到虚拟资产的游戏玩家并不拥有它们:大型游戏开发商和出版商通常通过合同条款向游戏玩家许可虚拟资产。因此，虚拟资产的交易被限制在授权的游戏内交易或通过 G2G 这样的拍卖网站进行的非法交易。

随着 ERC-721 令牌标准的出现，新的数字收藏品(如 CryptoKitties)或令牌化商品改变了收藏品所有权的讨论。具体来说，NFT 正在彻底改变数字资产所有权。NFT 允许对令牌化商品的完全所有权:收藏者永久拥有他们的令牌化商品，并且可以随意购买、出售和交易他们的令牌化商品。因此，令牌化商品是物理收藏品和虚拟资产之间的独特混合体，因为它们提供了数字商品的完全所有权。尽管在实践中，NFT 所有者对其象征性商品拥有产权，但尚不清楚美国财产法是否承认所有者的产权。T3】

进化:ERC-721 和外来代币

ERC-721

第一个也是最流行的 NFT 是 ERC-721 标准，主要是由于 CryptoKitties 的成功。开发商正在试验 ERC-721，从数字收藏品到证券化投资产品。ERC-721 基于令牌的可收藏游戏的开发以及随后投资者投机兴趣的爆发，使几个重要的法律问题受到关注，包括与所有权、隐私和资金转移有关的问题。

在像 CryptoKitties 这样的游戏中，令牌的所有权是明确的，因为每个 CryptoKitty 都是新开发的知识产权。然而，如果大型游戏开发商和发行商采用 NFTs，我们可能会很快回到虚拟商品的老路，即代币持有者被视为被许可人，而不是所有者，因此无法将虚拟商品转让或出售给平台外的其他人。

ERC-721 基于令牌的游戏通常使用“令牌元数据”功能来创建各种特征的排列和组合(例如，密码小猫的繁殖)。根据应用，该参考数据可能存储在令牌所有者控制之外的中央数据库或缓存文件夹中，这可能导致所有权争议，特别是当参考数据和令牌的知识产权属于不同的所有者时。T9】

非功能性金融交易可以包括可能要求 NFT 企业向金融犯罪执法网注册的功能，因为接受非功能性金融交易的钱包也可以接受 ERC-20 可替代令牌。NFT 企业需要特别注意如何构建代币以及存放代币的钱包的功能。

此外，一个令牌化的资产交换可能有资格作为一个投资合同，从而证券。例如，假设在可以使用令牌化资产的应用程序或游戏开发出来之前，应用程序开发人员开发了一个令牌化资产，比如一个独特的数字武器。然后，应用程序开发人员决定出售令牌化的资产以获得资金，从而为应用程序本身的开发增加利润。该交易可能被视为投资合同，因此是未注册证券。T3】

ERC-721 可收集的令牌目前主导着 NFT 的景观，但我们预计这一趋势将随着新的和更强大的用例的出现而改变。其他外来令牌标准的发展也可能蚕食 ERC-721 的市场份额；然而，NFT 标准的多样化应该有助于支持 NFT 生态系统。

外来代币

更奇特的令牌标准，如 ERC-420 和 ERC-998，提供了对 NFT 和准 NFT 潜力的见解。符合这些外来令牌标准的令牌将具有独特的功能，使它们非常适合各种使用情形。例如，在游戏中，符合 ERC-998 标准的代币可以代表携带消耗品配给(ERC-20 代币)和不可替换武器(ERC-721 代币)的角色，或者在商业中，这些代币的类别可以用于分级会员计划或创建证券产品市场。T9】

虽然可能不是最政治正确的使用 NFTs，“稀有的 Pepes”，或基于 Pepe the Frog 的稀有数字艺术，允许用户为“Pepecash”创建、交易和出售他们的艺术作品。稀有 Pepes 利用符合 ERC-420 标准的令牌(“dank 标准”)，该令牌展示了如何使用 NFT 来建立数字艺术品的限量版。这种类型的准 NFT 也可以用来提供多个系列的有限用户投资产品。T11】

外来令牌刚刚出现，我们希望在未来几年看到更多的标准和用例频繁进入市场。T13】

结论

从棒球卡的卑微起源，NFTs 提供了许多与实物收藏品和虚拟资产相同的功能，是这个行业的下一个进化飞跃。除了数字收藏品之外，NFT 还有许多潜在的用途，从艺术品到证券化投资工具。尽管它们仍处于非常早期的阶段，但我们预计 NFT 将有一个光明的未来。
