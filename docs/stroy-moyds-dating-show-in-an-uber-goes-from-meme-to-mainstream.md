# Stroy Moyd 的优步相亲节目从迷因走向主流 

> 原文：<https://web.archive.org/web/https://techcrunch.com/2016/12/31/stroy-moyds-dating-show-in-an-uber-goes-from-meme-to-mainstream/>

多年来，总部位于奥克兰的单口喜剧演员斯特罗伊·莫伊德一直兼职为优步和 Lyft 开车，以赚取额外收入。

这对创意产业的任何人来说都是一个熟悉的故事——尽管 Moyd 在[顶级俱乐部](https://web.archive.org/web/20221208233721/https://www.eventbrite.com/e/downtown-comedy-night-hellafunny-tickets-30307112389)如[妙语](https://web.archive.org/web/20221208233721/http://www.punchlinecomedyclub.com/)和[科布斯](https://web.archive.org/web/20221208233721/http://cobbscomedyclub.com/)担任主角，并在奥克兰[福克斯剧院](https://web.archive.org/web/20221208233721/http://thefoxoakland.com/)为大卫·查普尔等偶像开场，但喜剧并不总能支付他的账单。

在旧金山为几家高速增长的科技公司工作给了 Moyd 一个快乐的副作用，给他的行为带来了大量科技色彩的素材。

他的内容帮助他获得了旧金山风险投资家和初创公司创始人喜欢去的俱乐部 [The Battery](https://web.archive.org/web/20221208233721/https://www.thebatterysf.com/) 的会员资格，他有时会在那里为会员观众表演。

但 2016 年，在将他的普锐斯变成不仅仅是一辆出租车之后，Moyd 终于在网上获得了一些全国知名度。他把它变成了一个游戏节目的场地，有点像是现金出租车和约会游戏的混合体。

在一个他自己制作的名为“Rideshare the Love”的视频剪辑中，Moyd 为孤独的心们安排了一场类似快速约会的活动，同时开车送他们去目的地。

他说，他通过在个人网络和粉丝邮件列表上发布消息，以及偶尔在 Craigslist 上发布广告来安排约会。朋友们会告诉他们的单身伙伴，“嘿，我叫你搭车，车在路上了。”然后用约会游戏给他们一个惊喜。

两个竞争者竞争成为被选中的人。一个单身女人或男人戴着眼罩或眼罩坐着，问追求者一些有趣的问题，永远看不到他们长什么样。

几轮过后，但在第一站之前，被蒙住眼睛的一方必须选择他或她更喜欢和谁约会。如果他们愿意，莫伊德会马上开车送他们去第一次约会。

但是，当然，有一个转折。如果被选中的人感受不到爱怎么办？他或她可以拒绝约会，并带着现金奖励离开。

今年早些时候，Moyd 的约会秀视频出现在世界之星 Hip Hop 上，并在网上疯传。现在，迷因已经准备好成为主流。[藻奈美娱乐](https://web.archive.org/web/20221208233721/http://monamient.com/)已经与这部漫画的制作公司海拉滑稽公司签约，并将于 2017 年通过网络、有线电视或在线发行的方式进行购物旅行。

由电视制片人[莫娜·斯科特·杨](https://web.archive.org/web/20221208233721/http://monamient.com/)创立的人才管理公司因其与格莱美获奖嘻哈偶像米西·埃利奥特的合作而闻名，真人秀特许经营包括[爱&嘻哈](https://web.archive.org/web/20221208233721/http://www.vh1.com/shows/love-and-hip-hop)和[金钱、权力&尊重](https://web.archive.org/web/20221208233721/http://www.wetv.com/shows/money-power-respect)。
