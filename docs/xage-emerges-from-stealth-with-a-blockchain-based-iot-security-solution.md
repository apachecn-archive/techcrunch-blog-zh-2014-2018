# Xage 凭借总部位于区块链的物联网安全解决方案脱颖而出

为工业物联网中的无数设备提供资源并以安全的方式相互通信，将是未来几年企业面临的巨大技术挑战之一。Xage(prounounded Zage)今天带着一个基于区块链的安全解决方案悄然出现，它可以帮助简化这一过程。

该公司还宣布 [Duncan Greatwood](https://web.archive.org/web/20230225042820/https://www.bloomberg.com/research/stocks/private/person.asp?personId=239580&privcapId=25409704) 已经加入公司担任首席执行官。巨木是一位经验丰富的企业家，[在 2013 年将托普斯卖给了苹果](https://web.archive.org/web/20230225042820/https://techcrunch.com/2013/12/02/apple-buys-topsy-for-a-reported-200m-could-use-social-signals-to-bolster-app-store-relevance/)，在 2008 年将 PostPath 卖给了思科。这些退出让他可以自由挑选他想做的项目，他喜欢从技术角度看他在 Xage 看到的东西。

“这是一个变革浪潮正在席卷整个行业的领域。安全性是这项创新的基础要素，”Greatwood 告诉 TechCrunch。

他表示，Xage 正在为物联网构建一个安全架构，该架构采用区块链，并将其与其他功能相结合，为设备的运行创造一个安全的环境。如果区块链的核心是一种信任机制，那么它可以让公司相信他们的物联网设备不会受到威胁。Xage 认为区块链是这个问题的完美解决方案。

他们通过在区块链上建立一个由人、机器和应用程序组成的可信网络来做到这一点，这在这些不同的实体之间创建了一个不可辩驳的连接，并防止任何没有得到明确许可的人获得访问权限。

“区块链就像一个分布式、冗余的防篡改数据存储库。它与从云中推送或本地配置的策略相连接。[安全]结构使设备、人工智能和人能够相互通信，并控制信息流，”他解释道。

Greatwood 表示，这有助于解决一个巨大的物联网安全挑战，因为当一切都可以与一切交谈时，固有的巨大风险。他说:“在边缘与许多设备的任何对任何通信[对于安全来说是最糟糕的情况，因为你正在创造最大的攻击面。”](https://web.archive.org/web/20230225042820/https://techcrunch.com/2017/08/03/edge-computing-could-push-the-cloud-to-the-fringe/)

但是，他说，Xage 的区块链方法反过来说，因为参与越多，就越安全。“参与者越多，安全就越有保障，冗余就越多，就越难攻击系统，打破区块链要建立的共识，”他说。

最终部署的是一个安全架构，一组网关和工业边缘的客户端设备，它们之间形成了区块链该公司[正在与 IBM 合作 Hyperledger Fabric 项目](https://web.archive.org/web/20230225042820/https://techcrunch.com/2017/03/19/ibm-unveils-blockchain-as-a-service-based-on-open-source-hyperledger-fabric-technology/)来构建他们的区块链以及一些以太坊技术。

该产品现已上市。该公司位于帕洛阿尔托，目前有 20 名员工。他们的早期客户包括 ABB 和 Itron，Itron 正在使用该技术提供智能电表。
