谷歌正式将 WebVR 引入 Chrome

虚拟现实正在走向网络。

在手机上使用虚拟现实一直是一个非常深思熟虑的决定，你可以将手机插入耳机，浏览内部菜单和主屏幕，并从耳机中启动应用程序和体验。现在，谷歌正在通过将 WebVR 引入 Chrome，让人们更容易自发地发现 VR 内容。

以前 Chrome 缺乏对 WebVR 的移动浏览器支持，这意味着浏览器无法收集任何关于手机位置和方向或输入控制器的数据。有了这些数据，Chrome 就有能力提供相当丰富的体验。用户可以导航到一个特定的网页，点击 WebVR 体验，并可以选择用手指(或桌面上的鼠标)导航，或将手机扔进他们的 Daydream 视图，以获得更身临其境的体验。

去年 12 月，谷歌在 Chrome 的测试版中推出了对 WebVR API 的支持，但现在这一更新正在广泛进行。

WebVR 允许公司和内容创作者深入研究创建 VR 体验的世界，而不必在原生 VR 应用程序中托管这些体验。Chrome 的这一更新不仅允许网站托管用户可以观看的 360 度视频，还支持使用 Daydream 耳机和控制器实现完整的体验。

在谷歌的博客文章中，该公司强调了一些早期合作伙伴，包括 SketchFab、Matterport 和 PlayCanvas。

目前，这种扩展仅限于谷歌 Daydream 耳机的用户以及不戴耳机在手机或桌面上观看 VR 内容的人。谷歌表示，他们计划很快增加对其他耳机的支持，包括他们自己的 Cardboard 平台。鉴于这仅适用于 Daydream，iPhone 用户显然不包括在内，但一旦谷歌推出完全的 Cardboard 支持，iOS 可能会看到一些爱。
