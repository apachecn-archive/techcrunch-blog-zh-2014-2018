# 随着 IO 的推出，秘密创始人回归匿名出版

Secret 的联合创始人大卫·拜托(David Byttow)表示，他关闭的初创公司将在某种程度上回归，他在 11 月份也发了同样多的推文。周四上午，Byttow 推出了复活的匿名发布应用程序 [IO](https://web.archive.org/web/20230323141144/https://bold.io/) ，这是一种不同的方法，与 Twitter 等移动社交应用程序相比，这种方法与媒体更有共同点(与他在 6 月推出的初创公司 [Bold](https://web.archive.org/web/20230323141144/https://techcrunch.com/2016/06/07/bold/) ，这是一个面向企业的长篇内容发布平台，实际上为 IO 提供了主干 URL)。

IO 是 Byttow [在 11 月告诉我们的 Secret 的伪复活](https://web.archive.org/web/20230323141144/https://techcrunch.com/2016/11/12/secret-is-coming-back/)的出现部分是因为“当前社交媒体产品的负面影响必须得到解决”，在上次美国大选结果出来后，他觉得这一必要性尤为迫切。IO 宣称的使命是实现“真实出版”，Byttow 的意思是，他希望用户可以选择匿名、使用假名或以真实身份出版，从而更容易分享真实的想法和感受。

根据 Byttow 在 [Product Hunt](https://web.archive.org/web/20230323141144/https://www.producthunt.com/posts/io-publish-freely#1) 上的说法，周四的发布会是非常早的——它是“粗糙的边缘”。但是轻量级客户端已经感觉到它尽可能地减少了你和网络发布之间的摩擦。从截图可以看出。你输入你想要的，可选地添加一个 Twitter 账户，命名或保留空白，点击发布，你会得到一个像这样的可共享链接。

我这个作家喜欢这种线条清晰、动态格式的大胆、单步出版，但这种吸引力将在多大程度上占据主导地位还有待观察。此外，Secret 最初的发布充满了关于使用其平台进行欺凌的问题；这个似乎不太倾向于这种特殊的用途，如果只是因为它缺少位置的成分，但匿名总是伴随着它自己的问题。

当你自愿创建一个临时编辑访问时，IO 会分配一个随机 cookie，编辑器支持降价、导出和图像，以及编写“助手”，包括帮助你的散文，甚至通过环境声音让你进入正确的顶部空间。IO 上目前可用内容的完整分类是[这里有](https://web.archive.org/web/20230323141144/https://bold.io/about-io-2016-12-06)。
