# 美联社报道，美国创建“古巴推特”煽动异议，向杰克·多西寻求资助

> 原文：<https://web.archive.org/web/http://techcrunch.com/2014/04/03/u-s-created-cuban-twitter-to-stir-dissent-sought-funding-from-jack-dorsey-ap-reports/>

根据美联社今天一篇令人瞠目结舌的新[报道，美国政府基于手机短信创建了自己的 Twitter 版本，以帮助削弱古巴政府。该报告详细描述了一个技术承包商团队在古巴建设和启动一个信息网络的努力，该网络将避开该国对信息流的严格监控。](https://web.archive.org/web/20230130004052/http://bigstory.ap.org/article/us-secretly-created-cuban-twitter-stir-unrest)

这个网络被称为“ZunZuneo”，这是古巴俚语，意思是蜂鸟之歌，它的设计本质上是一个所谓的“古巴推特”，可以在没有网络的情况下运行，最初使用安全的内容建立受众，如谈论体育、音乐和穿越该地区的极端天气系统。一旦有了足够多的定期活跃用户，计划就是打开批评统治权力的内容，并在美联社获得的文件中意图激发被称为“聪明的暴民”的政治行动活动。

ZunZuneo 在最受欢迎的时候，其网络上有大约 4 万名古巴人，该网络从未让任何人知道它与美国政府或其承包商有牵连。这是故意的，美联社的文件援引说，没有提到美国的参与对服务和任务的成功“绝对至关重要”。这可能是因为他们希望由此产生的政治激进主义看起来是自发的、草根性质的。

负责创建“古巴推特”的政府机构实际上是美国国际开发署，它并没有否认其参与，也没有否认其参与保密的努力。美国国际开发署的目的是“帮助人们行使他们的基本权利和自由，让他们获得改善生活和与外部世界联系的工具，”该机构的一位发言人告诉美联社，并声称隐瞒其参与只是为了保护相关公民。

然而，批评者质疑该计划的整体合法性(他们认为，这需要总统授权才能合法进行，目前还不清楚是否获得了授权)，当然，古巴用户也有风险，他们不知道这是美国政府的一个项目。

最终，项目的发展超出了政府承包商的控制范围，他们意识到他们需要退出以隐藏他们的角色。有一次，美国国际开发署甚至去找 Twitter 的联合创始人杰克·多西为 ZunZuneo 寻求资金，这是它独立并成为合法企业的计划的一部分，条件是新的管理层保持其激发政治变革的最初雄心。

这些计划失败后，ZunZuneo 于 2012 年年中关闭，美国国际开发署称这是该计划资金耗尽的结果。鉴于它变得如此流行，当它变暗时，用户显然感到困惑和失望。

这是一个奇怪的故事，关于一个政府机构如何利用 Twitter 的剧本来寻求煽动革命的种子，Twitter 已经成为表达政治异议和在世界各地政治紧张的国家组织起来的关键工具，包括埃及、乌克兰和土耳其。也许最奇怪的部分是，它遇到了一个典型的创业问题，导致了它的最终结局:如何有效地扩展。
