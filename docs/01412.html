Nest 最终与 Control4、快思聪、RTI 和 URC 家庭自动化系统合作

今天，在 2014 年 CEDIA 世博会上，Nest 让世界各地的每一个家用自动装置安装者和零售商都感到高兴:Nest 终于与家庭自动化、Control4、快思聪、远程技术公司(RTI)和通用遥控器(URC)的主要参与者合作。这些是昂贵的高级家庭自动化系统，到目前为止，还没有正式支持流行的 Nest Learning 恒温器。

通过这一举措，拥有昂贵的家庭自动化系统的消费者可以在他们的豪宅墙壁上悬挂一个光滑的 Nest 恒温器，结束了使用陈旧的 HVAC 控制垫的第一个世界问题。

这种整合来自 Nest 最近宣布的“与 Nest 合作”开发者计划，该计划允许其他供应商将 Nest 支持构建到其产品中。随着 Nest 的集成，谷歌 Nest 新购买的 Dropcam 无线网络摄像头现在也与上述公司的系统兼容。

Nest 的创始人兼工程副总裁马特·罗杰斯在今天发布的一份声明中说:“我们与惠而浦、Jawbone 和 LIFX 等消费品牌一起推出了与 Nest 合作的计划，形成了有意义的联系，如当能源价格高时知道不运行的电器或当你的烟雾报警器响起时闪烁的灯泡。”“今天，我们很高兴将 Nest 应用于整个家庭自动化。专业安装人员是 Nest 的主要市场合作伙伴，我们致力于让我们的产品与他们每天信赖和安装的家庭自动化系统相连接。”

这是一个漫长的过程，但时间表也是为了在成功推出消费产品后支持其他平台。公司在 2013 年宣布了它的开发者计划，在今年早些时候宣布了它与 Nest 计划的第一个合作伙伴。

Matt Rogers 在去年的 CEDIA 博览会上说:“自从我们在 2011 年推出以来，开发人员社区对 Nest 开发 API 的需求一直很稳定。“虽然我们一直想创建一个 Nest 开发者计划，但我们的首要任务是打造一个优秀的产品、客户体验和团队。我们已经定义了什么是 Nest 体验。现在我们正准备打开我们的大门。”

现在大门敞开了。
