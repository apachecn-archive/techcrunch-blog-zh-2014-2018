# 凭借来自知名天使投资者的 1000 万美元，Globality 将致力于重建全球贸易

> 原文：<https://web.archive.org/web/https://techcrunch.com/2015/09/22/with-10m-from-high-profile-angels-globality-will-aim-to-restructure-global-trade/>

总部位于门洛帕克的超级隐形初创公司 Globality 已经从一群高调的天使投资人那里筹集了 1000 万美元，以建立一个模糊的全球贸易新生态系统。

这笔资金来自 14 位天使投资人，包括阿尔·戈尔、雪莉·桑德伯格、前 J.C. Penney 首席执行官罗恩·约翰逊和雅虎首席财务官肯·戈德曼。

Globality 的使命声明既雄心勃勃又含糊不清。据联合创始人乔尔·凯悦(Joel Hyatt)称，Globality 正在建立人工智能技术，使中小企业能够在推动全球经济方面发挥更大的作用。

哦，同时也将恢复美国梦。

“美国有一种传统观念，认为全球化和技术已经抹杀了美国梦，”Hyatt 说，他指的是上个世纪消失的制造业高薪蓝领工作。

“如果有任何经济体应该能够利用全球化和技术的好处，那一定是美国经济，”Hyatt 继续说道。

那么，全球化究竟打算如何解决美国的经济生存危机呢？他们没告诉我们。

凯悦会说:

> “我们将为全球贸易提供便利，参与出口商品和服务的公司数量将远远超过目前的数量，这对于美国经济和国内生产总值至关重要。90%的出口来自 1%的公司，这些大型工业集团和国际服务公司。”

目前，由 Hyatt 和 Lior Delgo 成立的 14 人团队计划在 2016 年第二季度公开发布，他们的前两家创业公司[被雅虎](https://web.archive.org/web/20230227002611/http://www.cnet.com/news/yahoo-buys-travel-company/)和[微软](https://web.archive.org/web/20230227002611/https://www.crunchbase.com/organization/videosurf)收购。Hyatt 表示，他预计到明年年底，员工人数会增加到几百人。

“这是一个创新的新解决方案，旨在解决全球贸易中一些真正重大的问题和摩擦，这些问题和摩擦阻碍了许多许多公司成为全球经济的参与者，”凯悦重申。“这不是一个小的或迭代的改进——我们正在构建根本不存在的东西。”
