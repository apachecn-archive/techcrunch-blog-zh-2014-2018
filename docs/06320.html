电报学院训练营教有色人种如何编码TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2016/01/30/telegraph-academy-bootcamp-teaches-people-of-color-how-to-code/

科技公司里从事工程工作的有色人种都到哪里去了？总部位于加州伯克利的coding bootcamp Telegraph Academy计划在未来几个月内搬迁到奥克兰，它希望这样一来，科技行业就没有借口不雇佣黑人和拉丁美洲人担任技术职务。

电报学院创始人阿尔布雷·布朗告诉TechCrunch说:“首先，我们希望在一个种族多元化的城市，拥有强大的有色人种社区，他们会对此感到兴奋。”“我们名字的由来是因为我们发现加利福尼亚州的奥克兰是美国种族最多样化的城市之一，奥克兰最古老的街道是Telegraph Ave。其次，我们做了大量的研究，发现市场上没有人提供我们的产品，短短三个月的专业软件工程职业。能够在如此独特的地方建造一座通往山谷的桥梁，对我们来说是一个巨大的机会。”

电报学院的编码训练营是久负盛名的 Hack Reactor 网络的一部分，该网络有99%的招聘率，并与30多家科技公司建立了合作关系。电报学院是在布朗于2014年6月经历Hack Reactor后开始的，最终觉得该计划可能对旧金山湾区的有色人种有益，但更加强调有色人种试图成为更好的工程师的感受。

“当想到为有色人种提供训练营体验时，我想从优秀的基线开始，”布朗说。“这是我最想提供的。我们还想在此基础上发展自己的风格，比如面向有色人种的领导力和品牌建设。”

在电报学院第二次训练营的第一天，我参观了它在伯克利的总部。这一天以布朗的欢迎演讲开始，在演讲中，他给了11名学生一个诚实的，没有废话的未来三个月的想法。

“如果你能想出你做过的最困难的事情，乘以1000，你可能还差得远，”布朗在第一天告诉学生。他后来说，“你的工作就是让你一直尽可能地不舒服，并推动你掌握一些东西——一个你从未想过的概念——并享受这种不舒服的过程。”

通过Telegraph Academy，学生可以获得设备、每周一天12小时的黑客空间、讲座、促进协作、求职支持、解决方案演练等。在第一周(上周)，有28场讲座。

“这将是一场尽可能精彩的表演，”布朗告诉学生们。

接下来是递归部分。尽管我对递归一无所知——然而在那天之前，只有我一个人听说过这个词——但当老师开始谈论可字符串化对象和JSON时，我觉得我已经完全被弄糊涂了。

电报学院的第一批学员开始于2015年6月。布朗告诉我，从那以后，他意识到对电报学院的毕业生来说，有“一点文化冲击”,因为科技行业并不主要是黑人和拉丁美洲人。

布朗说，这就是为什么电报学院已经开始提供领导力发展研讨会，学生们每周参加两小时的研讨会，讨论他们在工作场所的差异，并致力于“打破种族主义和微妙的微观侵略”。“我们希望让他们能够很好地处理微攻击，并将它们分解成建设性的批评。”领导力发展研讨会名为“变革型领导力”，是选修课，但我们强烈建议每位通过电报学院的学生都参加。通过编码训练营和领导力发展研讨会，Telegraph Academy的目标是让学生为在微软、Intuit、Slack、SolarCity、脸书、谷歌、LinkedIn等科技公司工作做好准备。

不幸的是，很多这样的办公室不仅设在有很多白人的城市，IBTimes本月早些时候报道，而且这些公司也雇佣了很多白人。看看最近来自 Dropbox 、苹果和谷歌的一些多样性报告就知道了。尽管优步即将搬进奥克兰——根据2014年人口普查数据，这座城市有34.5%的白人、28%的黑人和25.4%的西班牙裔人——引发了一些争议，优步表示，它致力于成为一个好邻居，实现平等，抗击流离失所，支持当地经济。优步一直在奥克兰进行“倾听之旅”，会见当地组织，看看独角兽在奥克兰着陆后如何成为真正的合作伙伴。优步全球多元化和包容性负责人达明·胡珀·坎贝尔亲自会见了电报学院，并向他们概述了优步在多元化、包容性和社区参与方面的目标和战略。

“我们真的对他的想法感到兴奋，并看到优步在提高技术多样性方面采取主动，”布朗说。“这与我们为有色人种拓宽机会的努力有着明显的协同作用。”

布朗说，归根结底，硅谷不应该仅限于旧金山和半岛。洛杉矶地区有硅海滩，凤凰城有硅沙漠，全国其他城市甚至全世界都有其他创新生态系统。

“技术正在传播，训练营模式确保尽可能多的人有机会接受教育，让他们进入其中，”布朗说。"否则，我们将把我们的城市经济置于危险之中."
