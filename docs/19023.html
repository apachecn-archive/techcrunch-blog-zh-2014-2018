亚马逊推出Alexa Auto SDK，将其语音助手引入更多汽车TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2018/08/09/amazon-launches-an-alexa-auto-sdk-to-bring-its-voice-assistant-to-more-cars/?utm_source=dlvr.it&utm_medium=twitter

亚马逊今天早上宣布为开发者推出一个工具包，允许他们将Alexa集成到汽车的信息娱乐系统中。“ Alexa Auto SDK 现在可以在 GitHub 上使用，它包括了所有的Alexa核心功能，如流媒体、智能家居控制、天气报告，以及对Alexa数万种第三方技能的支持。亚马逊表示，它还将为汽车用户增加新功能，如导航和搜索。

源代码和函数库将采用C++和Java语言，允许车辆处理音频输入和触发器，然后与Alexa服务连接，并处理Alexa交互。

此外，亚马逊正在提供各种示例应用、构建脚本和文档，支持ARM和x86处理器架构上的Android和QNX操作系统。

目前，SDK将允许来自亚马逊音乐、iHeartRadio和Audible的流媒体，并将允许客户通过说出联系人的姓名或电话号码来拨打电话。这些将通过车辆中的本地呼叫服务启动。

此外，当客户指定地址或兴趣点时，或者如果他们取消导航，它可以接入本地的逐路段导航系统。

本地搜索功能允许客户搜索餐馆、电影院、杂货店、酒店和其他企业，并导航到相应位置。

无论如何，这不是Alexa第一次涉足汽车领域。亚马逊一直在与福特、宝马、T4、雷克萨斯和丰田等汽车制造商合作，他们已经将语音助手集成到精选车辆中。Alexa也可以通过各种附加设备安装在老款汽车上，例如来自 Anker 、 Muse(说音乐)、 Garmin 和 Logitech 的设备。

通过这个SDK，亚马逊向尚未与亚马逊建立关系的其他汽车开发人员开放了语音助手。
