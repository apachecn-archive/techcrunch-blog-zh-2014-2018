Twitterrific 回归 Mac，成为一个可定制的 Twitter 客户端，但价格是 Tweetbot TechCrunch 的两倍

原文：https://web.archive.org/web/https://techcrunch.com/2017/10/10/twitterrific-returns-to-mac-as-a-customizable-twitter-client-but-at-twice-the-price-of-tweetbot/?ncid=rss&utm_source=dlvr.it&utm_medium=gplus

在今年早些时候通过 Kickstarter 活动筹集了超过 10 万美元后，Iconfactory 的团队重新推出了更受欢迎的第三方 Twitter 客户端之一，用于 Mac 的 twit terrious。几年前，当 Iconfactory 将重点转移到移动设备时，Kickstarter 实际上已经被放弃了，它的目的是在投入开发人员资源开发 Mac 应用程序之前，建立对 Mac 应用程序的需求。

今天，twit terrorial 5 for Mac已经上线，包括许多功能，如黑暗模式、主题、支持多个时间轴和账户、与 iOS 版本同步、可定制字体和字体大小等。

该客户端还考虑了 macOS 集成，包括对通知中心、Retina 显示屏、内置共享、全屏模式和其他功能的支持。

Twitter 的长期用户会记得，曾经有一段时间，这个领域充斥着第三方 Twitter 客户端，包括 Twitterrific、Tweetbot 等等。但是在 2010 年 Twitter 收购 tweet ie成为其移动应用的基础后不久，它在 2011 年彻底修改了其 API 服务条款，实际上是告诉开发者停止开发他们自己的第三方 Twitter 应用。

Twitter 随后以 4000 万美元收购了 TweetDeck以避免被 UberMedia 收购。但是这款应用在最近几个月并没有得到太多的关注，因为 Twitter 更关注移动和网络，而不是桌面。

Twitterrific Mac 应用程序于十年前首次发布，但在 2013 年停止更新，因为团队将资金转移到了移动设备上。

然而，正如 Kickstarter 所证明的那样，仍然有一小部分忠实的人对 Twitterrific Mac 应用程序感兴趣。

但 Iconfactory 的价格是竞争对手 Mac 第三方客户端 Tweetbot 的两倍，它的软件定价吸引了多年来对 Twitterrific 的喜爱一直没有消退的小众观众。(它的新应用售价 19.99 美元，相比之下，Tweetbot 的售价仅为 9.99 美元。)

也就是说，这款应用提供了许多 Mac 用户会喜欢的功能，除了支持多个窗格等标准项目，还可以访问 Twitter 的主要功能，如提及、DMs、收藏夹、转发、引用推文、热门话题、保存的搜索、列表等。

它还可以更容易地跟踪连锁推文和“推文风暴”，同时添加 Twitter 本身没有的按需功能，例如编辑推文的功能。(需要明确的是，Twitter 不允许原生编辑推文，所以 Twitterrific 的实现是一种变通方法——它删除了你需要用你编辑过的新推文进行更正的推文。尽管如此，它还是很方便。)

不同类型的推文在 Twitterrific 中也有颜色编码，让你可以快速区分普通推文与回复、DMs 和你自己的更新。该应用程序也非常可定制，支持明暗主题，能够控制字体类型和大小，以及你想看到哪些窗口和不想看到哪些内容。

对于拥有 twitteelliant iOS 版本的用户，这款应用可以让你在不同设备间的阅读位置保持同步，还能让你的静音和消音效果达到较轻版本的静音。

这款应用还提供了一种无广告体验，不会像 Twitter 上的“你不在的时候”那样杂乱无章，也不会被其他人的推文打断。(Twitter 目前并不通过 API 提供广告服务，所以 Twitter 很棒这样的第三方应用程序不会显示广告。但这种情况可能会改变。)

Kickstarter 的支持者们以较低的价格获得了 Mac 新应用 twitteetally，而且在某些情况下，他们还能进行测试。不过，在 Twitter bot 和免费的 Tweetdeck 应用程序的竞争下，这款售价 19.99 美元的产品在其他主流 Mac 用户看来可能更难卖出去。
