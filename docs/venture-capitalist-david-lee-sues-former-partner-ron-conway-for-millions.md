# 风险投资家李尚义起诉前合伙人罗恩·康韦索赔数百万

> 原文：<https://web.archive.org/web/https://techcrunch.com/2017/11/05/venture-capitalist-david-lee-sues-former-partner-ron-conway-for-millions/>

根据上周在马林县高级法院提起的诉讼，罗恩·康韦和他的前 SV 天使合伙人李尚义之间的法律诉讼关系到数百万美元，该诉讼最早由 Axios 报道。

李提起诉讼，声称康威没有向他支付之前承诺的费用和在 Airbnb、Pinterest 和 Snapchat 等公司投资的约定股份。

李开复是他和康威一起创立的 SV Angel 公司的前执行合伙人，但根据《财富》杂志上一篇详细描述他离开的文章，两年前他离开了洛杉矶，这在当时被描述为“个人决定”,要花更多的时间和家人在一起。

然而，这次分手似乎并不那么友好。事实上，据说康威在对 LPs 说李安缺乏参与和搬到洛杉矶时批评了李安。

康威还认为，他的前合伙人通过一种被称为费用豁免的技术从 SV 天使基金中获利，从而支付给自己的薪酬远远超过该基金 25 万美元的年薪上限。费用豁免是基金经理用来弥补预期投资成本的一种相当常见的方式。然而，这个概念对康威来说显然是陌生的。

现在，经营一家名为 Refactor Capital 的风险投资公司的李(Lee)声称，康威欠他 350 万美元的违约费和未支付的费用，这些费用是两人在李离开前达成的。此外，李开复希望获得声明性救济，以确保他在公司投资的未来退出能够获得报酬，这些投资总额可能在 1500 万美元至 2000 万美元之间。

然而，康威告诉 TechCrunch，李在两人解决问题之前就提起了诉讼。

康威在一份声明中说:“最近，在遵守一些和解协议条款方面出现了争议。”。“在我们解决分歧之前，大卫提起诉讼，坚持索要 SV Angel 认为他无权得到的钱。”

“非常令人失望的是，大卫试图向 SV 天使施压，要求其支付不属于他的钱，而 SV 天使基金打算大力抵制大卫的无理要求，”康威说。

当被问及评论时，李开复告诉 TechCrunch:“当我离开 SV Angel 时，罗恩·康韦和我就我的离开达成了协议。我们同意的许多条款都是罗恩提出的。我希望罗恩没有恶意。我只想让他履行我们签署的协议。”。
