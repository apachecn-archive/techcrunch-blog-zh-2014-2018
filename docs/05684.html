<p class = " dropcap ">虚拟现实是一种现实…从某种意义上说，硬件、软件和内容明星终于联合起来，使平台不仅仅是铁杆开发者、游戏玩家和业余爱好者可以接触到。

总有一批早期采用者承担重任，脸书的Oculus一直在这样做。自从被脸书收购以来，Oculus一直致力于将虚拟现实带给更多的人，它与三星的独特合作关系推动它朝着这一目标前进。在收购日，脸书首席执行官马克·扎克伯格转述了Oculus的使命以及它如何与脸书的使命完美契合:

Oculus的使命是让你体验不可能。他们的技术开启了全新体验的可能性。

直到我尝试了三星Gear VR的第一个版本，我才被卖了，它需要一台三星Galaxy Note 4才能运行。它被称为“创新者”版，并没有完全针对消费者。虽然我不是一个重度游戏玩家，因为如果PlayStation或Xbox游戏不是Madden或NBA2K，我不会连续玩几个小时，但我从中获得了足够的享受，可以看到脸书和Oculus通过三星的目标是什么。

在本月早些时候开始预售之后，今天是Gear VR的第三代产品的真正消费者模型上架的日子。

Gear VR售价99美元，需要三星的Galaxy S6、S6 Edge、S6 Edge+或Note 5才能运行。这不仅是Oculus让VR更接近大众的一种方式，也让大众对他们的设备更感兴趣。如果你想买一部Android设备，并且对Gear VR感兴趣，现在可能是时候认真对待他们的手机了。

我已经测试了新的三星Gear VR大约一周了，我将分享我的体验，有些很好，有些不值一提，但总体而言，虚拟现实作为一个整体是一个巨大的飞跃。

“人们谈论VR，认为它是孤立的，”Oculus移动副总裁Max Cohen告诉我社交环境中的VR。“但是你可以去你没有办法去的地方旅行。照片和视频是社交。在Oculus，我们谈论与任何人在任何地方体验任何事情。”

这就是我们开始的地方。

“齿轮”

要怎么做才能让这东西动起来？好吧，我告诉你。顶级的Oculus虚拟现实体验可以让你花费数千美元，包括一台可以运行它的个人电脑和一个很难找到的开发者工具包，从技术上讲已经售罄，这将让你在Craigslist上逛上几个星期，就像我在找到我的之前一样。

幸运的是，三星Gear VR的价格在你的掌握之中。你对手机的选择取决于你的运营商是谁，你是把它作为一条新线路添加进去还是直接购买手机。一部非合约手机的价格可以超过700美元。

我一直在用三星S6 Edge+ 测试Gear VR，虽然这不是“手机评论”，但重要的是要注意，这是一款强大的设备——强大到足以流畅地运行消费者就绪的VR体验。请注意，如果你打算将它作为日常手机，你可能不喜欢三星的Android版本，这是一款有趣的设备。我主要是在Gear VR中使用它，所以我没有太多其他日常手机使用情况可以报告。

这两样东西是唯一的要求，但是你肯定会想要给自己准备三样东西:一个控制器，一副耳机和一个装你的装备的包。出于测试目的，我得到了Steelseries的 Stratus XL控制器(59.99美元)和无线耳机上的三星Level(229美元)。控制器很棒，价格也很合理。它通过蓝牙工作，不需要真正的设置。三星的耳机不是我喜欢的。耐用吗？是的。但是你可能已经有一些好的耳机了。我只是建议你使用耳挂式，以获得完全的沉浸式效果。如果你试图单独使用脱离手机声音的Gear VR，你只能获得一半的体验。

“虚拟现实”

Gear VR本身就很轻……准确的说是0.62磅。Gear VR的前两个“创新者”版本是一磅，但感觉更重。请注意，你将感受到的大部分重量来自手机，但即使这样也没什么大不了的。你有两条带子可以把它固定在你可爱的头上，它们真的可以把东西固定住。

我戴着眼镜，所以额外的海绵泡沫给了我需要的空间和支撑，让我一次戴这个东西超过45分钟，这是我在以前的版本中经常遇到的问题。

“Oculus一直致力于舒适的虚拟现实体验，如果不舒适，我们不想给人们，”科恩告诉我。

内部传感器包括陀螺仪和加速度计，让你的头部在某种意义上成为“控制器”。当你环顾你探索的新世界，就好像你在那里。这些传感器可以做到这一点。

这是一个骗局，因为φ38，FOV 96，x6.7光学镜头欺骗你的大脑，让你以为你在沉浸式360度3D体验中看到了什么。保持镜头清洁是关键，除了手机的玻璃，因为手机正在被Gear VR上的镜头解读。弄点灰尘进去，你会注意到它，它会让你心烦意乱。

手机连接microUSB。你可能需要几次尝试才能正确插入手机，因为Gear VR可以处理四种不同的手机。会有一些拉环滑动和折断的情况发生，但一旦它被固定，你就知道了。如果你戴上它的时候看不到正确的图像，那可能是因为手机没有扣好。很容易修理。

从外部来看，触控板得到了升级。它曾经是一个平坦的白色表面，没有任何触感。你会不小心进入应用程序，这让我很沮丧。这个新的触控板感觉好多了，因为毕竟……你头上顶着Gear VR的时候是看不到的。并不是所有的体验都需要控制器，但一切都需要触控板。巨大的升级。

最后，同样重要的是，这个东西是白色的。或者三星所说的“霜冻”。对于一款你和你周围的人可能需要一些时间来适应的便携式产品，我会期待一些其他的选择。我绝对会摇滚一个橙色耳机。

Oculus和三星当初是怎么组队的？几年前的CES上，Oculus的科恩告诉我。

科恩说:“基本上，三星知道他们有这些令人惊叹的屏幕，颜色很棒，可以做到低持久性。“我们致力于解决分歧，约翰·卡马克说他喜欢在约束下工作。他认为，移动领域的发展可能会比其他领域快得多。

于是这场VR天堂里的比赛。

“内容”

正如我说过几次的那样，消费者版Gear VR的内容比硬件更重要。我的意思是，不要误会我的意思，三星和Oculus已经做了一项了不起的工作，将过去只能放在硬核电脑上的东西缩小到可以从一个房间跑到另一个房间玩的东西，但对于没有经验可供参考的人来说，你能做的就是让这个东西变得神奇。

Oculus商店里有大量的游戏。射击游戏似乎是时下流行的类型，但我希望随着时间的推移，随着更多的人习惯于沉浸式体验的想法，这种情况会发生变化。不过，有些游戏我称之为“轻玩”，比如Oculus的Arcade beta，它让你步入一个充满站立机器的Arcade，包括刺猬索尼克和吃豆人。

VR卡丁车 是目前为止我最喜欢的游戏，无论是个人，还是当朋友想尝试Gear VR的时候。这就像马里奥赛车，但当然你是在驾驶座上。把你的头转向左边，你在向左看。对吗？一样。向上？没错。开车时回头看看，当你看到一辆车悄悄靠近你时，你可能会感到有点头晕。这几乎就像在一个真正的卡丁车。你肯定需要这个控制器。如果你能用你的大脑控制更多一点会很酷，但是有一种模式，你可以瞄准一个司机并向他们发射一些东西，这需要你用你的大脑来排列东西。超级好玩，到目前为止我已经玩了至少10个小时了。

如果你对射击非常感兴趣，那么来自Climax工作室的 强盗六:齐射 就像是旗舰游戏体验。你用你的头作为控制器来瞄准向坦克和船只发射的大炮和迫击炮。它可能会变得迷失方向，尤其是在换枪的时候，但这真他妈的有趣。是的，你周围的人会想知道你到底在看什么，因为你的头随着炮塔一起旋转。第一个VR相关的伤害什么时候会发生在我身上？我想很快。颈部扭伤。它需要一个名字。VRlash？这是“轻松游戏”,但是伙计，它会让人上瘾，你可以不断地从一级到另一级，所有的东西都开始向你飞来。

陆地的尽头 真正利用了虚拟现实真正的360度性质。我强烈建议你给自己找一把转椅，不要让路过的人看到。沙发做不到这一点，如果你站着，你可能会摔倒，特别是因为你正在爬山，沿途解锁路径。你必须绕过那座山的一侧，是的，这需要你用所有能想到的方式去寻找…上、下、左、右以及其间的一切。这是一部分困惑，一部分探索和完全的乐趣。在这部电影中，你会失去。

欧米茄代理 是让自己习惯处于360度VR环境的绝佳方法。大概是我开着VR卡丁车逛完之后建议玩的第二个游戏。你戴着一个喷气背包，有一系列的任务来测试你如何灵活地指向你的头和一起使用控制器。习惯玩是需要时间的，但是值得。是的，这是一个你可以用猎枪射击的游戏，但这是我有过的让“飞行”体验感觉正常的最好体验。我从飞机上跳了下来，所以当你的降落伞打开时，感觉非常像在四处漂流。

其他几个值得关注的游戏是《枪战》、《斯珀2》、《寻找怪物》、《安萨尔战争2》和《走进死亡》。同样，根据你的口味和你能处理的虚拟现实的类型，每个人都有一些东西。

不喜欢游戏？没问题。这是我认为三星和Oculus能够接触到大众的地方。有很多没有游戏元素的体验，但是玩起来很有趣。想看视频？Oculus Video拥有Vimeo和Twitch的9000部电影，以及20世纪福克斯和狮门影业的70部长篇电影。还有一个网飞的应用程序，让你可以访问他们的整个目录。

无论你在哪里，你都会在你的“沙发”上欣赏电影，因为该应用程序模拟了坐在红色大沙发上看大电视的体验。当你在机场停留时，这可能会很酷。它会挡住你周围的一切。三星的牛奶虚拟现实应用程序有几个值得一试的体验，包括潜水。你所要做的就是启动它，当摄像机在你周围游动时，环顾四周。

三星沉浸式产品和虚拟现实副总裁兼总经理尼克·迪卡洛(Nick DiCarlo)告诉我:“当我们回想过去两三年我们是如何谈论虚拟现实的，很多人都没有听说过或尝试过它。”。“我们去了百思买和其他活动，让人们参与进来。我们开始对话了。虚拟现实对很多不同的人都有帮助。”

也有迷你体验，有时作为营销活动。《侏罗纪世界》就有这样一次经历，虽然我不想毁了你，但一只大屁股恐龙对“你”感兴趣了，你想不想被吓到是你自己的选择。你还可以看到比尔·克林顿总统讨论东非的局势。全部身临其境，就像你真的在那里。

有很多东西是免费的。大多数游戏都低于8美元。这当然胜过60美元的PlayStation游戏，尽管是完全不同的体验。从第一天起，你就一定会从为你提供的内容中获得回报。

Oculus作为一个平台吸引了脸书。在这个话题上，让开发者为平台创造，科恩告诉我:

我们试图发展一个健康的生态系统。我们从不与我们的开发者竞争。我们永远不会禁止来自竞争商店的应用程序。从第一方的角度来看，我们玩很多东西，看看什么东西有腿。

科恩还告诉了我Oculus商店的一个名为概念的区域，那里有来自尚未体验过Oculus paces的开发者的游戏。把它想象成一个所有类型事物的沙箱。今天发布的。

“体验”

处于虚拟现实中很容易占用你一天的大部分时间。尤其是当你刚刚开始的时候。然而，随着时间一天天地过去，当你有时间的时候，你会发现自己时不时地尝试一下。那种认为这不是一种社交体验的想法有点用词不当，因为我觉得给朋友戴上护目镜并告诉他们沉浸在周围的世界中很有趣。

有了Oculus背后的脸书，你可以打赌社交体验会越来越好。想象一下，浏览你的新闻订阅，看到一些遥远的土地…比如你妈妈在新泽西的房子。在360度虚拟现实中。有点酷。现在，我们必须满足于Oculus Social的Alpha(非常Alpha)版本。还没有太多，但是我被告知，随着发展，事情正在快速发展。

在你能忍受的范围内，放轻松。如果你很容易头晕，就试试像网飞电影那样轻松的东西。习惯暂时屏蔽掉你周围的一切。没有人评判你，也没有人根据你有多“酷”给你打分。

“瘦骨嶙峋”

这是给你的吗？是的，当你戴上耳机时，你可能看起来像个大书呆子。是的，人们可能会评价你。但是除非你亲身体验过某种身临其境的感觉，否则很难做出决定。

我强烈建议拿起一个谷歌纸板，它不是“真正的虚拟现实”，但会让你感受到走进一个你周围的世界是什么样子。试试看视频，看看照片。如果你对它感兴趣，那么Gear VR是显而易见的下一步。是的，你需要购买三星手机的事实听起来很痛苦。但它胜过昂贵的个人电脑。当你想到任天堂Wii、PlayStation和Xbox时，你会为游戏机、配件和游戏支付数百美元。

说到Wii，我认为三星Gear VR相当类似于Wii让通常不玩游戏的人玩游戏的体验。我妈妈以前经常玩Wii网球。她从未想过自己是一名游戏玩家。如果你想和你的朋友和家人玩得开心，你可以通过Gear VR体验虚拟现实带来的同样兴奋。

不过，Gear VR的虚拟现实体验并非没有缺点。根据你手机的充电情况，游戏有时会有起伏感。你可能不得不重新调整耳机的方向，因为当你开始的时候，你最终会面对一个完全不同的方向，事情会变得混乱。

不过，这都是关于学习的。你不必害怕测试它，因为它不会轻易被打破。如果是这样的话，你不需要更换成千上万美元的设备。

虽然脸书和Oculus表示，他们的消费产品Oculus Rift应该会在2016年初上市，但不会是这样的移动体验。你可以打开这个东西，把它戴在头上，然后在几分钟内检查出一些东西。没有准备也没有大惊小怪。

无论如何，脸书和Oculus将长期合作，科恩告诉我自收购以来两者之间的关系如何:

脸书在各方面都给予了帮助。他们放手了。我们控制着愿景，但脸书一直在那里提供工程方面、技术方面和财务方面的资源。

这是脸书真正信奉的东西。这是对脸书的热情。

虚拟现实和内容消费的“未来”就在今天，而Gear VR是了解“未来”的窗口。
