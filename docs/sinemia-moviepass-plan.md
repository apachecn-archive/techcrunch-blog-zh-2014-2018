# Sinemia 再次瞄准 MoviePass，推出新的 9.99 美元计划

Sinemia 继续开展活动，利用 MoviePass 高调的斗争，赢得更知名的电影票订阅服务的客户。今天，它宣布了一项定价为每月 9.99 美元的新计划。

毕竟，MoviePass 最近宣布,[将把每月订阅价格保持在 9.95 美元](https://web.archive.org/web/20221206151956/https://techcrunch.com/2018/08/06/moviepass-keeps-plan-at-10-but-limits-subscribers-to-three-movies-a-month/)，但限制订阅者每月只能看三部电影(额外门票有折扣)。

新的 Sinemia tier 还包括每月三张门票，但它还有一个额外的好处，即允许订户购买任何 2D 非 IMAX 屏幕的门票，并提前购买这些门票。相比之下，MoviePass 每天轮流播放电影，它要求订户在放映的同一天到电影院买票。

就在几周前， [Sinemia 宣布了一项推荐给朋友的计划](https://web.archive.org/web/20221206151956/https://www.forbes.com/sites/jamos/2018/08/16/sinemia-tosses-rival-moviepass-subscribers-a-life-preserver/#2ab332ff6957)，奖励那些说服朋友离开其他订阅服务的用户。该公司毫不掩饰其特别针对 MoviePass 的事实——在今天的声明中，它描述新计划是“与 MoviePass 的最新计划相匹配的”。"

Sinemia 还提供各种其他选择，从每月 3.99 美元的一张票到 14.99 美元的三张票，可以观看 IMAX 和 3D 电影。
