Opera 收购 SurfEasy，为其浏览器软件 TechCrunch 添加安全 VPN 服务

原文：https://web.archive.org/web/http://techcrunch.com/2015/03/19/opera-buys-surfeasy-to-add-secure-vpn-services-to-its-browser-software/?ncid=rss

Opera 是一套软件的制造商，这套软件用于在约 3 . 5 亿消费者使用的 T2 移动设备和 T4 桌面设备上浏览网页，Opera 又进行了一次收购，以扩大其为用户提供的服务。它已经收购了 SurfEasy，这是一个虚拟专用网(VPN)应用程序的制造商，可以让用户更安全地浏览网页。

这是 Opera 第一次以安全为重点的收购，是在消费者不仅对简单廉价的互联网浏览方式需求日益增长的背景下进行的，Opera 目前已经进入了这个市场，而且对更私人的方式也有需求。

“越来越多的互联网用户开始质疑他们是否可以信任互联网，并积极为他们的手机或电脑寻找安全的解决方案，”奥斯陆 Opera 软件公司的首席执行官拉尔斯·布瓦森在一份声明中说。

这其中隐含着通过 NSA 披露提出的问题，以及消费者越来越意识到他们的数据中有多少被跟踪并用于更多的商业目的。

除此之外，对于 Opera 来说，这也是一次有趣的收购。如今，该公司因在国际(即美国以外)市场拥有强大的用户基础而闻名，尤其是在发展中市场，在那里，它与 international 的脸书和其他公司一起努力让更多的消费者联网。

碰巧的是，这些市场也是 VPN 使用蓬勃发展的地方，用户采取安全隧道技术不仅有助于保护他们的浏览免受政府的窥探，还可以帮助他们访问否则可能会被地理围栏和阻止查看的内容——无论是在直接的审查播放中，还是因为版权许可。Opera 增加安全 VPN 服务将有助于它继续在这些市场获得用户和发展品牌。

该公司告诉我，Opera 计划暂时保持 SurfEasy 产品的可操作性。

其中包括 Windows、Mac、Android、iPhone 和 iPad 设备的免费增值应用。SurfEasy 还提供基于硬件的产品，surf easy Private bros wer T1，这是一款在 u 盘上销售的 VPN 产品，面向那些使用各种可能与其他人共享的设备工作的人。

Opera 说:“SurfEasy 是一个公认的消费者品牌，是在线隐私保护方面客户评价最好的服务之一，所以在这个时候，SurfEasy 的产品将保持活跃和不变。”“然而，由于这笔交易，该公司现在将被称为 SurfEasy，一家歌剧公司’"

更长期的计划是将 SurfEasy 的一些功能整合到 Opera 自己的应用程序中，包括压缩移动设备浏览数据的 Opera Max，内容推荐的 Opera Coast 和 Opera 的网络浏览器。但是 Opera 还没有对这些产品何时上市发表评论。

“现在，我们不猜测未来发布的时间或它们将采取什么形式，但我们期待将这些整合到联合产品中，这将扩大 Opera 的产品组合，”该发言人说。

在此之前，Opera 已经试图在声明其对网络隐私问题的立场上走在前面。在它的原则中，它说它只处理个人数据“出于 Opera Software 的服务客观上合理的目的，并在基本尊重隐私权的情况下进行处理。”但尽管如此，该公司还通过其 Opera Mediaworks 子公司提供一套广告服务，有人可能会认为这可能与隐私理念背道而驰。

这是 SurfEasy 收购如何为 Opera 服务的另一个有趣角度。通过向用户提供付费 VPN 服务以及免费浏览和数据压缩技术，Opera 可能会继续从这些用户身上获得收入，即使它无法再通过广告赚钱。

两家公司没有透露交易条款，但“Opera 和 SurfEasy 都对协议感到满意，并期待合作，”一位发言人告诉我。目前还不清楚 SurfEasy 有多少用户，但它一直在增长，在过去的 12 个月里增加了 600 万新用户。

然而很明显，surf easy——2011 年在多伦多成立，从那以后只进行了一轮融资——需要更大的规模来发展业务。

“在过去的几年里，围绕在线隐私问题的对话和意识显著增强。SurfEasy 创始人兼首席执行官克里斯·休斯顿(Chris Houston)在一份声明中表示:“我们现在都非常清楚，无论你使用何种网络或设备访问网络，我们的数据都容易受到黑客攻击、监控和审查。”。

他继续说道:“通过与拥有超过 3.5 亿用户的全球互联网公司 Opera 合作，我们能够加速实现我们的愿景，为用户提供简单的应用程序，让他们重新掌控自己的在线隐私和自由。”。
